Business-Service-Konfigurationen bearbeiten

  1. Wenn dies noch nicht geschehen ist, klicken Sie zum Erstellen einer neuen Session auf Erstellen oder zum Aufrufen einer vorhandenen Session auf Bearbeiten. Weitere Informationen finden Sie unter Change Center verwenden.

    2. Sie müssen keine Session eingeben, wenn Sie lediglich Details anzeigen möchten.

  2. Suchen Sie nach dem Business-Service, den Sie anzeigen oder bearbeiten möchten. Die Suche ist unter Business-Services suchen beschrieben.
  3. Klicken Sie auf den Business-Service-Namen. Auf der Seite Business-Service anzeigen - Konfigurationsdetails werden Konfigurationsinformationen für den gewählten Business-Service angezeigt.
  4. Gehen Sie wir folgt vor, um Einstellungen anzuzeigen oder zu ändern:
    • Klicken Sie auf das Symbol Bearbeiten neben dem Namen der Kategorie, deren Eigenschaften Sie anzeigen oder bearbeiten möchten (Beispiel: Allgemeine Konfiguration, Transportkonfiguration usw.). Welche Seiten Sie bearbeiten können, hängt davon ab, welche Seiten Sie während des Erstellens des Business-Service konfiguriert haben.

      • Um eine Liste aller solcher Seiten anzuzeigen, gehen Sie zur Seite Business-Service anzeigen - Konfigurationsdetails.

    • Klicken Sie auf Bearbeiten unten auf der Seite, um die Seite Business-Services erstellen/bearbeiten - Allgemeine Konfiguration anzuzeigen. Dies ist die erste Seite in der Seitenabfolge für das Konfigurieren dieses Business-Service.
  5. Setzen Sie das Anzeigen oder Bearbeiten fort, wie unter Business-Services Erstellen und konfigurieren beschrieben.
  6. Klicken Sie auf der Seite Business-Service bearbeiten - Zusammenfassung auf Speichern, um die Updates in der aktuellen Session festzuschreiben.
  7. Um die Session zu beenden und die Konfiguration für die Laufzeit bereitzustellen, klicken Sie unter Change Center auf Aktivieren.
