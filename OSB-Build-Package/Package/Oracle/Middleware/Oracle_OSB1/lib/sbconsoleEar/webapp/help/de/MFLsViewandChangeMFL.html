MFLs bearbeiten

  1. Wenn Sie es noch nicht getan haben, klicken Sie auf Erstellen, um eine neue Session zu erstellen, oder klicken Sie auf Bearbeiten, um eine vorhandene Session einzugeben. Siehe Change Center verwenden.
  2. Suchen Sie die MFL-Datei wie in MFLs finden beschrieben.
  3. Klicken Sie auf den MFL-Dateinamen. Die Seite MFL-Ressourcendetails zeigt die Informationen an, die in Tabelle 15-2 dargestellt sind.

    4. Tabelle 15-2 MFL-Ressourcendetails 
    Eigenschaft
    Beschreibung
    Zuletzt geändert von
    Der Anwender, der diese MFL-Datei erstellt oder in die Konfiguration importiert hat.
    Zuletzt geändert am
    Datum und Zeit, wann der Benutzer diese MFL-Datei erstellt oder in die Konfiguration importiert hat. Klicken Sie auf das Datums- und Uhrzeitlink, um die Änderungshistorie dieser Ressource anzuzeigen. Siehe Seite "Änderungshistorie anzeigen".
    Referenzen
    Die Anzahl von Objekten, auf die diese MFL-Datei verweist. Wenn solche Verweise vorhanden sind, klicken Sie auf den numerischen Link, um eine Liste der Objekte anzuzeigen. Siehe Verweise auf Ressourcen anzeigen.
    Referenziert von
    Die Anzahl von Objekten, die auf diese MFL-Datei verweisen. Wenn solche Verweise vorhanden sind, klicken Sie auf den numerischen Link, um eine Liste der Objekte anzuzeigen. Wenn Sie beispielsweise diese MFL-Datei als den Anforderungs- oder Antwortnachrichtentyp für einen Messaging Business-Service oder Proxy-Service gewählt haben, wird der Business-Service oder Proxy-Service als Referenz aufgelistet, wenn Sie auf den Link klicken. Siehe Referenzen auf Ressourcen anzeigen.
    Beschreibung
    Eine Beschreibung der MFL-Datei, sofern vorhanden.
    MFL-Text
    Der Text für diese MFL.

  4. Wenn Sie eine Änderung an den Feldern machen möchten, klicken Sie auf Bearbeiten. Beschreibungen der Felder finden Sie unter MFLs hinzufügen.

    5. Sie können das Feld Ressourcenname nicht ändern.

  5. Klicken Sie auf Speichern, um die Aktualisierungen in der aktuellen Session zu speichern.
  6. Um die Session zu beenden und die Konfiguration für die Laufzeit bereitzustellen, klicken Sie unter Change Center auf Aktivieren.
