Alert-Ziel

Eine Alert-Zielressource erfasst eine Liste derjenigen Empfänger, die Alert-Benachrichtigungen von Oracle Service Bus erhalten können. In einem typischen Systemüberwachungskontext sind Alerts, die von Oracle Service Bus generiert wurden, für eine begrenzte Anzahl Benutzer signifikant.

Sie können jede Alert-Zielressource in Oracle Service Bus so konfigurieren, dass sie entsprechend dem vorgegebenen Kontext eine Gruppe von Empfängern enthält. Alert-Ziele werden von Alert-Aktionen, die im Nachrichtenfluss konfiguriert werden, sowie von SLA-Alert-Regeln verwendet. Ein Alert-Ziel kann einen oder mehrere der folgenden Zieltypen einschließen: Konsole (Standard), Berichtsdatenstrom, SNMP-Trap, E-Mail, JMS-Queue oder JMS-Topic.

Bei E-Mail und JMS-Zielen kann eine Zielressource entweder eine E-Mail-Adressliste oder JMS-URIs enthalten.

Sie können Alert-Ziele in allen Alert-Konfigurationen für Services wiederverwenden. Siehe Alert-Aktionen hinzufügen.
