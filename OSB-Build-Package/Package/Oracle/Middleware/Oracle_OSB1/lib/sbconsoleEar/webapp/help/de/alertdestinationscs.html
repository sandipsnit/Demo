E-Mail-Empfänger hinzufügen

Bevor Sie eine E-Mail-Adresse hinzufügen, müssen Sie einen SMTP-Server konfigurieren (siehe SMTP-Server hinzufügen) oder eine JavaMail-Session in WebLogic Server (siehe "Zugriff auf JavaMail konfigurieren" in Oracle Fusion Middleware Oracle WebLogic Server Administration Console Online Help unter http://download.oracle.com/docs/cd/E15523_01/apirefs.1111/e13952/taskhelp/mail/CreateMailSessions.html). Sind dort keine SMTP-Serverressourcen oder JavaMail-Sessions verfügbar oder konfiguriert, können Sie keinen E-Mail-Empfänger konfigurieren.

  1. Wählen Sie Alert über E-Mail senden aus der Dropdown-Liste, und klicken Sie auf Hinzufügen, um die Seite E-Mail-Empfänger hinzufügen anzuzeigen.
  2. Geben Sie im Feld Mail-Empfänger einen E-Mail-Empfänger in folgendem Format ein: mailto:username@hostname

    3. Dieses Feld ist erforderlich.

    Sie können mehrere E-Mail-Empfänger angeben, wenn Sie Benutzernamen und Hostnamen in einer durch Komma getrennten Liste eingeben. Zum Beispiel: mailto:username@hostname [,username_1@hostname_1]...[,username_n@hostname_n]

    Nur beim ersten E-Mail-Empfänger muss das Präfix "mailto:" vorangestellt werden.

  3. Wählen Sie im Feld SMTP-Server den Namen des SMTP-Servers für die ausgehende Mail aus. Dieses Feld ist nicht erforderlich, wenn im nächsten Schritt eine Mailsession gewählt wird.
  4. Wählen Sie im Feld Mailsession eine verfügbare Mailsession. Dieses Feld ist nicht erforderlich, wenn im vorigen Schritt ein SMTP-Server gewählt wurde.
  5. Geben Sie im Feld Name des Absenders einen Absender an, damit eine Benachrichtigung erfolgen kann. Dieses Feld ist optional.
  6. Geben Sie im Feld Absenderadresse eine gültige E-Mail-Adresse an. Dieses Feld ist erforderlich, wenn für das Feld "Name des Absenders" ein Wert angegeben wurde.
  7. Geben Sie im Feld Antwortname einen Namen an, an den eine Antwort gerichtet werden kann. Dieses Feld ist optional.
  8. Geben Sie im Feld Antwortadresse eine E-Mail-Adresse an, an die eine Antwort gesendet werden kann. Dieses Feld ist erforderlich, wenn für das Feld "Antwortname" ein Wert angegeben wurde.
  9. Geben Sie im Feld Verbindungs-Timeout die Sekunden an, die eine Verbindung auf eine Antwort von einem Server warten muss, bevor die Wartezeit überschritten ist. Der Standardwert ist 0.
  10. Geben Sie im Feld Anforderungscodierung einen Wert für die Zeichensatzcodierung ein. Der Standardcodierungswert ist iso-8859-1.
  11. Klicken Sie auf Speichern. Der E-Mail-Empfänger wird der Liste der E-Mail-Empfänger für diese Benachrichtigung hinzugefügt und in der aktuellen Session gespeichert.
