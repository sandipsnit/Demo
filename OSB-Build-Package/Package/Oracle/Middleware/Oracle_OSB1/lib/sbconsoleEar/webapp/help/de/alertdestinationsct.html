JMS-Ziele hinzufügen

  1. Wählen Sie Alert an ein JMS-Ziel senden aus der Dropdown-Liste, und klicken Sie auf Hinzufügen, um die Seite JMS-Ziele hinzufügen anzuzeigen.
  2. Geben Sie im Feld "Ziel-URI" einen JMS-Ziel-URI im folgenden Format ein: jms://host:port/factoryJndiName/destJndiName

    3. Dieses Feld ist erforderlich.

    Hinweis: Während bei Oracle WebLogic Server Vorwärtsschrägstriche in JNDI-Namen (Beispiel: "myqueues/myqueue") zulässig sind, stören JNDI-Namen mit Vorwärtsschrägstrichen das URI-Format, das für Oracle Service Bus erforderlich ist, und Sie können diese Namen nicht verwenden. Um dieses Problem zu umgehen, definieren Sie einen JMS-Fremdserver, und referenzieren Sie diesen Fremdserver im URI. Weitere Informationen finden Sie unter "Configure foreign servers" in der Oracle Fusion Middleware Oracle WebLogic Server Administration Console Online Help unter http://www.oracle.com/pls/as111170/lookup?id=WLACH.

  3. Wählen Sie im Feld Zieltyp den Wert Queue oder Topic.
  4. Wählen Sie im Feld Nachrichtentyp den Wert Byte oder Text.
  5. Geben Sie im Feld Anforderungscodierung einen Wert für die Zeichensatzcodierung ein. Der Standardwert ist UTF-8.
  6. Klicken Sie auf Speichern. Das JMS-Ziel wird der Liste mit JMS-Zielen für dieses Alert-Ziel hinzugefügt und in der aktuellen Session gespeichert.
