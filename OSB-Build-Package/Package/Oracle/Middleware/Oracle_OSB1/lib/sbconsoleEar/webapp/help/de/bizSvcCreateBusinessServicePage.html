Einen Business-Service erstellen/bearbeiten - Seitenverweis

Erstellen Sie einen Business-Service, indem Sie Business-Service aus der Dropdown-Liste Ressource erstellen auf der Seite Projekt/Ordner-Ansicht auswählen. Wenn Sie diese Option wählen, wird zunächst eine Reihe von Seiten angezeigt, um Business-Service zu konfigurieren und hinzuzufügen. Die angezeigten Seiten variieren, je nachdem, welche Optionen Sie gewählt haben. Diese Seiten sind:

  • Seite "Allgemeine Konfiguration"
  • Seite "Nachrichtentyp-Konfiguration"
  • Seite "Transportkonfiguration"
  • Seite "Protokoll-spezifische Transportkonfiguration"
  • Seite "SOAP-Binding-Konfiguration"
  • Seite "Nachrichtenübermittlungskonfiguration"
  • Seite "Zusammenfassung"
  • Seite "Business-Service Policys"
  • Seite "Business-Service-Sicherheit"
