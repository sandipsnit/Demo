Seite "FTP-Transportkonfiguration"

Die Seite Business-Service erstellen/bearbeiten – FTP-Transportkonfiguration ist eine von diversen Seiten, auf denen Sie Business-Services erstellen und bearbeiten können. Diese Funktionen sind unter Business-Services erstellen und konfigurieren und Business-Service-Konfigurationen bearbeiten genauer beschrieben.

Auf dieser Seite können Sie Transporteinstellungen für Business-Services mit FTP-Transportprotokoll konfigurieren. In Tabelle 18-9 ist die Verwendung dieser Seite beschrieben.

Tabelle 18-9 Seite "Business-Services erstellen/bearbeiten – FTP-Transportkonfiguration"
Option
Verwendung
Benutzer-Authentifizierung
Wenn der Benutzer des FTP-Servers anonym ist, wählen Sie Anonym. Wenn es sich beim Benutzer des FTP-Servers um einen extern konfigurierten Account handelt, wählen Sie Externer Benutzer.
Identity (E-Mail-ID)
Dieses Feld ist nur dann verfügbar, wenn die Option Benutzer-Authentifizierung auf Anonym gesetzt wurde.
Geben Sie die E-Mail-ID des anonymen Benutzers ein.
Service-Account
Dieses Feld ist nur dann verfügbar, wenn die Option Benutzer-Authentifizierung auf Externer Benutzer gesetzt wurde.
Geben Sie den Service-Account des externen Benutzers ein.

Weitere Informationen:

  • Service-Accounts
  • Service-Accounts hinzufügen
Timeout
Geben Sie das Socket-Timeout-Intervall vor dem Trennen der Verbindung in Sekunden ein. Der Standardwert ist 60 Sekunden.
Präfix für den Zieldateinamen
Geben Sie ein Präfix für den Dateinamen ein, unter dem die Datei auf dem Remote-Server gespeichert wird.
Geben Sie in dieses Feld kein * ein. Dieses Zeichen verursacht eine Laufzeit-Exception.
Suffix für den Zieldateinamen
Geben Sie ein Suffix für den Dateinamen ein, unter dem die Datei auf dem Remote-Server gespeichert wird.
Geben Sie in dieses Feld kein * ein. Dieses Zeichen verursacht eine Laufzeit-Exception.
Transfermodus
Wählen Sie für den Transfermodus ASCII oder Binär aus.
Anforderungscodierung
Übernehmen Sie die Standardeinstellung UTF-8 als Zeichensatzcodierung für Anforderungen in FTP-Transporten, oder geben Sie eine andere Zeichensatzcodierung ein.
