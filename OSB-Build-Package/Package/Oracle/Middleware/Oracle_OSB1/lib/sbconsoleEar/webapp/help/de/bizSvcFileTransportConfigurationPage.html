Seite "Dateitransportkonfiguration"

Die Seite Business-Services erstellen/bearbeiten - Dateitransportkonfiguration gehört zu einer Reihe von Seiten zum Erstellen und Bearbeiten von Business-Services. Diese Aktionen sind unter Business-Services erstellen und bearbeiten und unter Business-Service-Konfigurationen bearbeiten beschrieben.

Sie verwenden diese Seite, um Transporteinstellungen für einen Business-Service mithilfe des Dateitransportprotokolls zu konfigurieren. Tabelle 18-7 liefert eine Beschreibung zur Verwendung dieser Seite.

Tabelle 18-7 Business-Service erstellen und bearbeiten - Seite "Dateitransportkonfiguration" 
Option
Verwendung
Präfix
Geben Sie ein Präfix ein, das dem Dateinamen vorangestellt wird.
Geben Sie in dieses Feld kein * ein. Dieses Zeichen verursacht eine Laufzeit-Exception.
Suffix
Geben Sie ein Suffix ein, das dem Dateinamen angehängt wird. Dies ist ein obligatorisches Feld.
Geben Sie in dieses Feld kein * ein. Dieses Zeichen verursacht eine Laufzeit-Exception.
Anforderungscodierung
Akzeptieren Sie den Standard utf-8 als Zeichensatz für die Codierung von Anforderungen in Dateitransporten, oder geben Sie eine andere Zeichensatzcodierung ein.

Wenn Sie fertig sind

Klicken Sie auf Weiter, um auf der nächsten Seite mit der Konfiguration dieses Service fortzufahren, oder klicken Sie auf Letzte, um die Konfiguration auf der Seite Business-Service erstellen/bearbeiten – Übersicht zu prüfen und zu speichern.
