Seite "Flow-Transportkonfiguration"

Die Seite Business-Service erstellen/bearbeiten – Flow-Transportkonfiguration ist eine von diversen Seiten, auf denen Sie Business-Services erstellen und bearbeiten können. Diese Funktionen sind unter Business-Services erstellen und konfigurieren und Business-Service-Konfigurationen bearbeiten genauer beschrieben.

Mit Business-Services mit Flow-Transport können Sie auf Split Joins zugreifen. Weitere Informationen zu Split Joins finden Sie unter "Improving Service Performance with Split-Join" im Oracle Fusion Middleware Developer's Guide for Oracle Service Bus unter http://www.oracle.com/pls/as111170/lookup?id=OSBAG291.

Auf dieser Seite können Sie Transporteinstellungen für Business-Services mit Flow-Transportprotokoll konfigurieren. In Tabelle 18-8 ist die Verwendung dieser Seite beschrieben.

Tabelle 18-8 Seite "Business-Services erstellen/bearbeiten – Flow-Transportkonfiguration"
Option
Verwendung
Timeout
Geben Sie ein, wie viele Sekunden auf eine Antwort gewartet werden soll. Dieser Wert wird bei "Nur Antwort"-Vorgängen (einseitigen Vorgängen) ignoriert.

Wenn Sie fertig sind

Klicken Sie auf Weiter, um auf der nächsten Seite mit der Konfiguration dieses Service fortzufahren, oder klicken Sie auf Letzte, um die Konfiguration auf der Seite Business-Service erstellen/bearbeiten – Übersicht zu prüfen und zu speichern.
