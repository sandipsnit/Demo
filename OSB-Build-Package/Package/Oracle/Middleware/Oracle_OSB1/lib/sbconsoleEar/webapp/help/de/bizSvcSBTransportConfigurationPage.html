Seite "SB-Transportkonfiguration"

Die Seite Business-Service erstellen/bearbeiten - SB-Transportkonfiguration ist eine von mehreren Seiten für das Erstellen und Bearbeiten von Business-Services. Weitere Informationen hierzu finden Sie unter Business-Services erstellen und konfigurieren und unter Business-Service-Konfigurationen bearbeiten.

Weitere Informationen finden Sie unter "SB Transport" im Oracle Fusion Middleware Developer's Guide for Oracle Service Bus unter http://www.oracle.com/pls/as111170/lookup?id=OSBDV997.

Auf dieser Seite können Sie Transporteinstellungen für einen Business-Service mit dem SB-(Service Bus-)Transportprotokoll konfigurieren. In Tabelle 18-15 wird die Verwendung der Seite beschrieben.

Tabelle 18-15 Seite "Business Service erstellen/bearbeiten - SB-Konfiguration"
Option
Verwendung
Dispatch Policy
Wählen Sie die Instanz von WebLogic Server Work Manager, die Sie für die Dispatch Policy für diesen Endpunkt verwenden möchten. Wenn kein anderer Work Manager vorhanden ist, wird der Standard-Work Manager verwendet. Weitere Informationen zu Work Managern finden Sie unter "Using Work Managers to Optimize Scheduled Work" in Oracle Fusion Middleware Configuring Server Environments for Oracle WebLogic Server unter http://www.oracle.com/pls/as111170/lookup?id=CNFGD112.
Timeout
Der Zeitraum in Sekunden, nach dem es zu einem Timeout des Service kommt.
Hinweis: Das Timeout wird ignoriert, wenn die Quality of Service "Genau einmal" ist.
Service-Account
Klicken Sie auf Durchsuchen, und wählen Sie aus der angezeigten Liste einen Service-Account. Wenn Sie keinen Service-Account angeben, wird ein anonymes Subject verwendet.
