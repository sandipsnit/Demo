Seite "Zusammenfassung"

Die Seite Business-Service erstellen/bearbeiten - Übersicht ist eine Seite von mehreren, auf denen Business-Services erstellt und bearbeitet werden können, wie in Business-Services erstellen und konfigurieren und Business-Service-Konfigurationen bearbeiten beschrieben.

Verwenden Sie diese Seite, um die Konfigurationseinstellungen für einen Business-Service anzuzeigen oder zu ändern, bevor Sie sie speichern.

Um die Einstellungen anzuzeigen oder zu ändern, klicken Sie in der Zeile der entsprechenden Kategorie (z.B. Allgemeine Konfiguration, Transportkonfiguration usw.) auf Bearbeiten. Welche Seiten Sie bearbeiten können, hängt davon ab, welche Seite Sie beim Erstellen des Business-Services konfiguriert haben. Folgende Liste zeigt alle Seiten an:

  • Allgemeine Konfiguration
  • Seite "Nachrichtentyp-Konfiguration"
  • Seite "Transportkonfiguration"
  • Seite "Protokoll-spezifische Transportkonfiguration"
  • Seite "Business-Service Policys"
  • Seite "Business-Service-Sicherheit"
