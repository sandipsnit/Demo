Business-Services löschen

  1. Wenn dies noch nicht geschehen ist, klicken Sie zum Erstellen einer neuen Session auf Erstellen oder zum Aufrufen einer vorhandenen Session auf Bearbeiten. Weitere Informationen finden Sie unter Change Center verwenden.
  2. Wählen Sie Ressourcenbrowser > Business-Services, um die Seite Business-Services - Überblick anzuzeigen.
  3. Klicken Sie in der Zeile des zu löschenden Business-Service auf das Symbol Löschen. Der Business-Service wird in der aktuellen Session gelöscht. Das Symbol Warnung wegen Löschvorgang wird angezeigt, wenn andere Ressourcen diese Ressource referenzieren. Sie können die Ressource mit einer Warnbestätigung löschen. Dies kann jedoch zu Konflikten aufgrund von nicht aufgelösten Referenzen zur gelöschten Ressource führen.
  4. Um die Session zu beenden und die Konfiguration für die Laufzeit bereitzustellen, klicken Sie unter Change Center auf Aktivieren.
