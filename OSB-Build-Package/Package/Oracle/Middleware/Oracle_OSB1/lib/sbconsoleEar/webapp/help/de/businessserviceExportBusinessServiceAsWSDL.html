Mit Business-Service verknüpfte WSDL exportieren

Sie können die WSDL eines WSDL-basierten Business-Service exportieren, um die WSDL in einem externen Tool wie IDE anzuzeigen oder zu verändern. Die WSDL wird als JAR-Datei exportiert.

Beachten Sie, dass diese Funktion sich von der Funktion "Ressourcen exportieren" im Modul "Systemadministration", die Sie verwenden, um Ressourcen zwischen zwei Domains zu verschieben und zwischenzuspeichern, unterscheidet. SieheRessourcen exportieren.

Bevor Sie beginnen

Sie können eine WSDL nur exportieren, wenn Sie sich außerhalb einer Session befinden. Siehe Change Center verwenden.

So exportieren Sie eine WSDL:

  1. Führen Sie eine der folgenden Aktionen aus:
    • Wählen Sie Ressourcenbrowser > Business-Services, um die Seite Zusammenfassung der Business-Services anzuzeigen.
    • Wählen Sie Project Explorer, wählen Sie dann das Projekt oder den Ordner, die den Business-Service enthalten, den Sie als WSDL exportieren möchten. Die Seite Projekt-/Ordneransicht wird angezeigt.
  2. Klicken Sie auf das Symbol WSDL exportieren in der Zeile des Business-Service, dessen WSDL Sie exportieren möchten. Ein Dialogfeld fordert Sie auf, die exportierte JAR-Datei zu öffnen oder zu speichern.
  3. Klicken Sie im Dialogfeld auf Öffnen, um die Datei zu öffnen, oder klicken Sie auf Speichern, um sie zu speichern.
