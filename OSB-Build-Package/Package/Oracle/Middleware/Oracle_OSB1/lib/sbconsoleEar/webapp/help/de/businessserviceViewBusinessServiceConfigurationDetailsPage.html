Seite "Business-Service anzeigen – Konfigurationsdetails"

Auf der Seite Business-Service anzeigen – Konfigurationsdetails werden die Konfigurationsdetails eines Business-Service angezeigt. In Tabelle 18-22 sind alle auf dieser Seite angezeigten Eigenschaften beschrieben. (Die angezeigten Eigenschaften unterscheiden sich je nach den Details des Business-Service.)

Die auf dieser Seite angezeigten Kategorien entsprechen den Seiten Business-Service erstellen/bearbeiten, auf denen Sie Business-Service-Konfigurationen erstellen und bearbeiten können. Eine genaue Beschreibung finden Sie unter Einen Business-Service erstellen/bearbeiten – Seitenverweis.

Klicken Sie neben einem beliebigen Kategorienamen auf den Link Bearbeiten, um die entsprechende Konfigurationsseite anzuzeigen.
