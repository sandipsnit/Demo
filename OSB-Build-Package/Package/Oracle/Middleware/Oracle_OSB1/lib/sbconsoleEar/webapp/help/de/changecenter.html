Change Center

Das Change Center unterstützt die Zusammenarbeit im Team bei der Konfiguration von Services und Metadaten in Oracle Service Bus. Jeder Benutzer arbeitet in einer Sandbox-Session, bis er bereit ist, die Arbeitskonfiguration in die Kernkonfiguration des Bus einzuchecken. Das Change Center umfasst mehrere Ebenen der Rückgängigmachung sowie Transparenz über Konflikte, während mehrere Benutzer an der Konfiguration arbeiten.

Die meisten Seiten im Change Center zeigen verschiedene Informationen an, je nachdem, ob Sie sich innerhalb einer Session (wenn Sie beispielsweise im Change Center auf Bearbeiten geklickt haben) oder außerhalb einer Session befinden. Wenn Sie sich innerhalb einer Session befinden, listet beispielsweise die Seite Konfigurationsänderungen anzeigen, die geöffnet wird, wenn Sie im Change Center auf Änderungen anzeigen klicken, alle in dieser Session vorgenommenen Änderungen auf. Außerhalb einer Session listet die Seite alle Sessionaktivierungen auf.

Sessiondaten
Konflikte anzeigen
Nur innerhalb der Session: Dieser Link zeigt die Anzahl der Konflikte, die in einer Session auftreten. Sie können alle Konflikte zwischen Änderungen der aktuellen Session und solchen aus anderen Sessions in der Oracle Service Bus Administration-Konsole anzeigen und beheben. Ein Konflikt tritt auf, wenn eine Ressource in der aktuellen Session geändert wurde, die bereits in einer anderen Session geändert und aktiviert wurde. Zwei Änderungen, die von zwei Sessions an derselben Ressource vorgenommen wurden, verursachen erst dann einen Konflikt, wenn eine der Sessions aktiviert wird.
Auf dieser Seite werden auch semantische Fehler für Ressourcen in der Session angezeigt.
Änderungen anzeigen
Innerhalb der Session: Zeigt die Konfigurationsänderungen an, die Sie während der aktuellen Session vorgenommen haben.
Außerhalb der Session: Zeigt die Konfigurationsänderungen an, die von früheren Sessionaktivierungen verursacht wurden.
Änderungen rückgängig machen
Innerhalb der Session: Macht eine Änderung rückgängig, die Sie während der aktuellen Session vorgenommen haben.
Außerhalb der Session: Macht das Aktivieren einer Session rückgängig.
Alle Sessions anzeigen
Innerhalb und außerhalb einer Session: Zeigt alle vorhandenen Sessions innerhalb der Oracle Service Bus Administration-Konsole an.
Sessions aktivieren
Nur innerhalb der Session : Aktiviert die Session.
Task-Details anzeigen
Innerhalb der Session: Zeigt die Details einer bestimmten Änderung an, die Sie während der aktuellen Session vorgenommen haben.
Außerhalb der Session: Zeigt die Details bestimmter Änderungen an, die von einer früheren Sessionaktivierung durchgeführt wurden.
