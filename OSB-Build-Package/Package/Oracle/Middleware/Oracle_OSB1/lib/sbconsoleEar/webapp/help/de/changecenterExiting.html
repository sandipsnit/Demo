Sessions verlassen

Klicken Sie unter Change Center auf Beenden, wenn Sie diese Session verlassen möchten. Die Session wird jedoch nicht beendet.

Sie können unter Change Center auf Bearbeiten klicken, um zur Session zurückzukehren und weitere Änderungen vorzunehmen. Dies ist auch gültig, wenn Sie auf Abmelden klicken, um sich von der Oracle Service Bus Administration-Konsole abzumelden, oder wenn Sie Ihren Browser schließen. Die Session und alle vorgenommenen Änderungen bleiben bestehen, auch wenn Sie sich von der Oracle Service Bus Administration-Konsole abmelden und der Server neu gestartet wird.

Die Session endet erst, wenn Sie aktiviert wurde. Siehe Sessions aktivieren.
