Historie der Sessionaktivierung löschen

Sie können über einen bestimmten Zeitraum aktivierte Sessions, begrenzt durch Start- und Enddatum, endgültig löschen. Diese Aktion kann nur außerhalb einer Session durchgeführt werden.

Achtung: Das Löschen der Historie der Sessionaktivierung beinhaltet das Löschen von Daten, die "Undo" über mehrere Ebenen aktivieren. Anders ausgedrückt, wenn Sie die Historie der Sessionaktivierung für einen bestimmten Zeitraum löschen, werden Sie Sessions, die in dieser Periode aktiviert wurden, nicht rückgängig machen können.
  1. Stellen Sie sicher, dass Sie sich nicht in einer aktiven Session befinden, und klicken Sie auf Änderungen anzeigen.
  2. Klicken Sie auf der Seite Konfigurationsänderungen anzeigen auf Nach Datum löschen. Die Seite Historie der Sessionaktivierung nach Datum löschen wird angezeigt.
  3. Geben Sie die Anfangs- und Enddaten in die entsprechenden Felder ein, im Format: MM/TT/JJJJ.

    4. Die Historie der Sessionaktivierung für alle Sessions zwischen den angegebenen Daten wird von Oracle Service Bus gelöscht.
