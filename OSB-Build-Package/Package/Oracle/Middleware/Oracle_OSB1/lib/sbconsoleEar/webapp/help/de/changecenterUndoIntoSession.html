Vorgänge in einer Session rückgängig machen

Wenn beim Rückgängigmachen einer Sessionaktivierung semantische Fehler auftreten, wird der Undo-Vorgang verhindert. In diesem Fall gibt es allerdings eine Alternative. Sie können die Sessionaktivierung rückgängig machen und die Änderungen in eine neue Session einfügen. Anschließend können Sie die semantischen Fehler beheben und die Session aktivieren. Außerdem können Sie diese Funktion zum Rückgängigmachen der Sessionaktivierung nutzen, um die Auswirkungen eines solchen Undo-Vorgangs zu untersuchen. Sie können alle daraus resultierenden Änderungen prüfen und dann entscheiden, ob Sie den Undo-Vorgang tatsächlich ausführen möchten. Wenn Sie sich dafür entscheiden, können Sie die Session aktivieren.
