Alle Sessions anzeigen

Zeigen Sie auf dieser Seite alle vorhandenen Sessions innerhalb der Oracle Service Bus Administration-Konsole an. Sie können diese Sessions anzeigen, wenn Sie sich gerade in oder außerhalb einer Session befinden.

Sie können nur alle Sessions anzeigen, wenn Sie die Administratorrolle verwenden. Weitere Informationen finden Sie unter "Configuring Administrative Security" im Oracle Fusion Middleware Developer's Guide for Oracle Service Busunter http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539.

  1. Klicken Sie auf Alle Sessions anzeigen unter Change Center. Für jede vorhandene Session zeigt die Seite Alle Sessions anzeigen die Information an, die in Tabelle 2-5 dargestellt ist.

    2. Tabelle 2-5 Sessiondetails
    Eigenschaft
    Beschreibung
    Sessionname
    Der Name der Session. Der Name ist ein Link zu der Session.
    Benutzer
    Der Name des Benutzers, der die Session erstellt hat.
    Erstellzeit
    Erstelldatum und -zeit der Session.
    Zuletzt geändert
    Datum und Zeit, wann während der letzten Session eine Änderung gemacht wurde.
    Anzahl der geänderten Objekte
    Die Anzahl der Objekte, die während der Session geändert wurden.

  2. Klicken Sie zum Wechseln zu einer anderen Session auf den Link in der Spalte Sessionname für diese Session.
Hinweis: Ein Benutzer, der mit mehreren Browsern angemeldet ist, wird nicht unterstützt. Diese Anmeldung verursacht unvorhersehbares Verhalten in der Oracle Service Bus Administration-Konsole.
