Task-Details anzeigen

Verwenden Sie diese Seite, wenn Sie in der Session sind, um Details einer bestimmten Änderung, die Sie in der aktuellen Session gemacht haben, anzuzeigen, und zeigen Sie Details zu einer bestimmten Session an, die aktiviert wurden, wenn Sie außerhalb einer Session sind.

  1. Wählen Sie Änderungen anzeigen unter Change Center.
  2. Klicken Sie auf der Seite Konfigurationsänderungen anzeigen auf den Namen der Task in der Spalte Task. Die Seite Task-Details zeigt die in Tabelle 2-4 dargestellten Informationen an.

    3. Tabelle 2-4 Task-Details
    Eigenschaft
    Beschreibung
    Task
    Die Art der Task, die Sie ausgeführt haben.
    • Erstellen
    • Aktualisieren
    • Löschen
    • Umbenennen
    • Verschieben
    Ressourcentyp
    Die Oracle Service Bus-Ressourcentypen, die Teil der Task waren.
    Ressource
    Der Name und der Pfad der Ressource. Der Pfad ist der Name des Projekts und des Ordners, in denen die Ressource gespeichert ist.

  3. Klicken Sie auf Zurück, um zur Seite Konfigurationsänderungen anzeigen zurückzukehren.
