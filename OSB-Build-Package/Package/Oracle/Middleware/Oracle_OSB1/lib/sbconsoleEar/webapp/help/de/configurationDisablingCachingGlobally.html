Ergebnis-Caching global deaktivieren

Wenn Sie das Ergebnis-Caching global deaktivieren, löscht Oracle Service Bus den gesamten Cache und entfernt dabei gecachte Ergebnisse für alle Business-Services, deren Ergebnis-Caching aktiviert ist. Die Standardeinstellung für Ergebnis-Caching lautet "aktiviert".

So deaktivieren Sie das Ergebnis-Caching auf Domainebene:

  1. Falls noch nicht erfolgt, klicken Sie auf Erstellen, um eine neue Session zu erstellen; oder klicken Sie auf Bearbeiten, um eine vorhandene Session einzugeben. Siehe Change Center verwenden.
  2. Wählen Sie Vorgänge > Globale Einstellungen.
  3. Deaktivieren Sie das Kontrollkästchen Ergebnis-Caching aktivieren.
    4. Hinweis: Diese Einstellung setzt jedes für einzelne Business-Services konfiguriertes Ergebnis-Caching außer Kraft.
  4. Klicken Sie auf Aktualisieren. Die Konfiguration wird in der aktuellen Session gespeichert.
  5. Klicken Sie zu einem beliebigen Zeitpunkt während der Session auf Zurücksetzen, um die Änderungen zu verwerfen und die Seite mit den aktuell gespeicherten Einstellungen zu aktualisieren.
  6. Um die Session zu beenden und die Konfiguration für die Laufzeit bereitzustellen, klicken Sie unter Change Center auf Aktivieren.
