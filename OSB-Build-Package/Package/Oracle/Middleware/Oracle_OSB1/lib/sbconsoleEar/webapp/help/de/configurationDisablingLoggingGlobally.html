Logging global deaktivieren

So können Sie die Logging-Ausgabe für alle Proxy-Services auf Domainebene ausschalten:

  1. Wenn dies noch nicht geschehen ist, klicken Sie zum Erstellen einer neuen Session auf Erstellen oder zum Aufrufen einer vorhandenen Session auf Bearbeiten. Weitere Informationen finden Sie unter Change Center verwenden.
  2. Wählen Sie Vorgänge > Globale Einstellungen.
  3. Deaktivieren Sie das Kontrollkästchen Logging aktivieren, um sämtliche Log-Aktionen in der Pipeline für alle Proxy-Services zu stoppen.
    4. Hinweis: Mit dieser Option werden alle Einstellungen unter Logging außer Kraft gesetzt, die Sie für bestimmte Proxy-Services wählen können. Weitere Informationen finden Sie unter Vorgangseinstellungen für Proxy-Services konfigurieren.
  4. Klicken Sie auf Aktualisieren. Die Logging-Konfiguration wird in der aktuellen Session gespeichert.
  5. Klicken Sie zu einem beliebigen Zeitpunkt während der Session auf Zurücksetzen, um die Änderungen zu verwerfen und die Seite mit den aktuell gespeicherten Einstellungen zu aktualisieren.
  6. Um die Session zu beenden und die Konfiguration für die Laufzeit bereitzustellen, klicken Sie unter Change Center auf Aktivieren.
