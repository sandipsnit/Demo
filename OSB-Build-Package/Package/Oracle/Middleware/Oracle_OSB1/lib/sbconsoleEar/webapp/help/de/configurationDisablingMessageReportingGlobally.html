Nachrichten-Reporting global deaktivieren

Für Proxy-Services: So schalten Sie Nachrichten-Reporting für alle Proxy-Services auf der Domainebene aus:

  1. Wenn dies noch nicht geschehen ist, klicken Sie zum Erstellen einer neuen Session auf Erstellen oder zum Aufrufen einer vorhandenen Session auf Bearbeiten. Weitere Informationen finden Sie unter Change Center verwenden.
  2. Wählen Sie Vorgänge > Globale Einstellungen.
  3. Deaktivieren Sie das Kontrollkästchen Reporting aktivieren, um Pipeline Reporting-Aktionen für alle Proxy-Services zu stoppen.
    4. Hinweis: Diese Option überschreibt alle Einstellungen in Erstellen von Nachrichtenberichten, die Sie für bestimmte Proxy-Services angeben können. Siehe Vorgangseinstellungen für Proxy-Services konfigurieren.
  4. Klicken Sie auf Aktualisieren. Die Konfiguration für Nachrichten-Reporting wird in der aktuellen Session gespeichert.
  5. Klicken Sie zu einem beliebigen Zeitpunkt während der Session auf Zurücksetzen, um die Änderungen zu verwerfen und die Seite mit den aktuell gespeicherten Einstellungen zu aktualisieren.
  6. Um die Session zu beenden und die Konfiguration für die Laufzeit bereitzustellen, klicken Sie unter Change Center auf Aktivieren.
