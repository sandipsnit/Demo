SLA-Alerts global deaktivieren

So deaktivieren Sie SLA-Alerts für alle Services auf Domainebene:

  1. Wenn dies noch nicht geschehen ist, klicken Sie zum Erstellen einer neuen Session auf Erstellen oder zum Aufrufen einer vorhandenen Session auf Bearbeiten. Weitere Informationen finden Sie unter Change Center verwenden.
  2. Wählen Sie Vorgänge > Globale Einstellungen.
  3. Deaktivieren Sie das Kontrollkästchen SLA-Alerts aktivieren, damit das System das Auswerten der Alert-Regeln für alle Services in Ihrer Konfiguration beendet.
    4. Hinweis: Diese Option setzt SLA-Alerts, die Sie für spezielle Business und Proxy-Services wählen können, außer Kraft. Siehe Vorgangseinstellungen für Business-Services konfigurieren und Vorgangseinstellungen für Proxy-Services konfigurieren.
  4. Klicken Sie auf Aktualisieren. Die SLA-Alert-Konfiguration wird in der aktuellen Session gespeichert.
  5. Klicken Sie zu einem beliebigen Zeitpunkt während der Session auf Zurücksetzen, um die Änderungen zu verwerfen und die Seite mit den aktuell gespeicherten Einstellungen zu aktualisieren.
  6. Um die Session zu beenden und die Konfiguration für die Laufzeit bereitzustellen, klicken Sie unter Change Center auf Aktivieren.
