Ergebnis-Caching global aktivieren

Wenn Sie Business-Services aufrufen, deren Ergebnisse sich selten ändern, wird durch Ergebnis-Caching die Business-Service-Performance verbessert, wenn die gecachten Ergebnisse zum Client zurückgegeben werden, statt dass auf die Ergebnisse eines Service-Aufrufs gewartet wird.

Bevor Sie Ergebnis-Caching für einzelne Business-Services konfigurieren, müssen Sie das Ergebnis-Caching zuerst mithilfe der folgenden Schritte global aktivieren. Die Standardeinstellung für Ergebnis-Caching ist aktiviert.

  1. Falls noch nicht erfolgt, klicken Sie auf Erstellen, um eine neue Session zu erstellen; oder klicken Sie auf Bearbeiten, um eine vorhandene Session einzugeben. Siehe Change Center verwenden.
  2. Wählen Sie Vorgänge > Globale Einstellungen.
  3. Aktivieren Sie das Kontrollkästchen Ergebnis-Caching aktivieren.
  4. Klicken Sie auf Aktualisieren, um alle Einstellungen auf dieser Seite zu speichern.
  5. Klicken Sie zu einem beliebigen Zeitpunkt während der Session auf Zurücksetzen, um die Änderungen zu verwerfen und die Seite mit den aktuell gespeicherten Einstellungen zu aktualisieren.
  6. Um die Session zu beenden und die Konfiguration für die Laufzeit bereitzustellen, klicken Sie unter Change Center auf Aktivieren.

Weitere Informationen zur Konfiguration des Ergebnis-Caching für Business-Services finden Sie auf der Seite Nachrichtenübermittlung konfigurieren.
