Globale Überwachung von Services aktivieren

So aktivieren Sie das Überwachen für alle Services auf Domainebene:

  1. Wenn dies noch nicht geschehen ist, klicken Sie zum Erstellen einer neuen Session auf Erstellen oder zum Aufrufen einer vorhandenen Session auf Bearbeiten. Weitere Informationen finden Sie unter Change Center verwenden.
  2. Wählen Sie Vorgänge > Globale Einstellungen.
  3. Aktivieren Sie das Kontrollkästchen Überwachung aktivieren, damit das System das Erfassen von Überwachungsstatistiken für alle Services startet, deren Überwachung auf Service-Ebene aktiviert ist.
  4. Klicken Sie auf Aktualisieren, um alle Einstellungen auf dieser Seite zu speichern.
  5. Klicken Sie zu einem beliebigen Zeitpunkt während der Session auf Zurücksetzen, um die Änderungen zu verwerfen und die Seite mit den aktuell gespeicherten Einstellungen zu aktualisieren.
  6. Um die Session zu beenden und die Konfiguration für die Laufzeit bereitzustellen, klicken Sie unter Change Center auf Aktivieren.
