Alert-Ziele suchen

  1. Wählen Sie Vorgänge > Smart Search.
  2. Wählen Sie aus der Dropdown-Liste Typ die Option Alert-Ziele, und klicken Sie dann auf Suchen. Auf der Seite Übersicht der Alert-Ziele werden die in Tabelle 26-5 beschriebenen Informationen angezeigt.
  3. So schränken Sie die Anzahl der Objekte in der Liste ein oder suchen nach bestimmten Alert-Zielen:
    • Klicken Sie auf das Symbol Öffnen, um zusätzliche Suchfilter anzuzeigen. Wenn Sie mehrere Suchkriterien angeben, werden nur Ergebnisse angezeigt, die allen Kriterien entsprechen.
    • Filtern Sie nach Name und Pfad des Alert-Ziels. Geben Sie in die Felder Name und Pfad den Namen und den Pfad des Suchziels ein, und klicken Sie dann auf Suchen. Der Pfad ist der Projektname und der Name des Ordners, in dem sich das Alert-Ziel befindet. Verwenden Sie die Zeichen * und ? als Platzhalter, um eine allgemeinere Suche durchzuführen.
    • Filtern Sie nach dem Ziel des Alert-Ziels. Wählen Sie aus dem Kombinationsfeld Ziel ein oder mehrere Ziele für das Alert-Ziel aus: SNMP-Trap, Reporting, Alert Logging, E-Mail oder JMS. Klicken Sie anschließend auf Suchen. Es werden nur Alert-Ziele mit mindestens einem der gewählten Ziele angezeigt. Wählen Sie "Keine Ziele für Alert-Ziel", um die Filterkriterien für Alert-Ziele zu ignorieren.
    • Filtern Sie nach Suchmuster. Geben Sie in das Textfeld Suchmuster (beliebige Zeichenfolge) eine beliebige Zeichenfolge ein, und klicken Sie auf Suchen. Es werden die Felder Beschreibung der Alert-Ziele und bestimmte detaillierte Felder von E-Mail- und JMS-Zielen nach dieser Zeichenfolge durchsucht. Wenn die Zeichenfolge in Alert-Ziel-Feldern vorhanden ist, werden die entsprechenden Alert-Ziele angezeigt.

      • Tabelle 26-5 Zusammenfassende Informationen zu Alert-Zielen 
      Eigenschaft
      Beschreibung
      Name
      Der Name der Alert-Zielressource. Dieses Feld ist ein Link zur Seite Alert-Ziel anzeigen – Konfiguration. Weitere Informationen finden Sie unter Alert-Ziele suchen.
      Pfad
      Das mit dem Alert-Ziel verbundene Projekt. Wenn das Alert-Ziel in einem Ordner gespeichert ist, wird dieser Ordner ebenfalls angezeigt. Der Pfad hat folgendes Format:
      project-name/root-folder/ . . ./parent-folder
      Der Pfad ist ein Link zum entsprechenden Pfad im Project Explorer.
      Optionen
      Klicken Sie auf das Symbol Löschen, um ein bestimmtes Alert-Ziel zu löschen. Wenn andere Ressourcen diese Ressource referenzieren, wird das Symbol Warnung wegen Löschvorgang angezeigt. Sie können die Ressource löschen, indem Sie den Löschvorgang bestätigen. Dies kann aufgrund von nicht aufgelösten Referenzen mit der gelöschten Ressource zu Konflikten führen. Weitere Informationen finden Sie unter Alert-Ziele löschen.

Auf dieser Seite können Sie auch folgende Aufgaben ausführen:

  • Entfernen Sie den Suchfilter, und zeigen Sie alle Alert-Ziele an, indem Sie auf Alle anzeigen klicken.
  • Entfernen Sie ein bestimmtes Alert-Ziel, das nicht von anderen Ressourcen referenziert wird, indem Sie das entsprechende Kontrollkästchen aktivieren und auf Löschen klicken.
  • Passen Sie die Anzeige der Informationen zu den Vorgangseinstellungen an, indem Sie auf das Symbol Tabellen-Customizer klicken. Weitere Informationen finden Sie unter Tabellenansichten anpassen.
