SLA-Alert-Regeln suchen

  1. Wählen Sie Vorgänge > Smart Search.
  2. Wählen Sie aus der Dropdown-Liste Typ die Option SLA-Alerts, klicken Sie dann auf Suchen. Auf der Seite Übersicht der SLA-Alert-Regeln finden Sie die Informationen aus Tabelle 26-6.
  3. So beschränken Sie die Anzahl der Elemente in der Liste oder suchen nach speziellen SLA-Alert-Regeln:
    • Klicken Sie auf das Symbol Öffnen, um zusätzliche Suchfilter anzuzeigen. Wenn Sie mehrere Suchkriterien angeben, werden nur Ergebnisse angezeigt, die allen Kriterien entsprechen.
    • Nach Alert-Regelnamen filtern. Geben Sie im Feld Name den Namen des gesuchten Ziels oder Platzhalterzeichen ein (mit den Platzhaltern * und ? können Sie eine allgemeinere Suche durchführen), dann klicken Sie auf Suchen.

      • Wenden Sie diese Suchmethode bevorzugt an, wenn der Name der Alert-Regel in allen Projekten und Pfaden einzigartig ist.

    • Nach übergeordnetem Service-Pfad filtern. Geben Sie im Feld Pfad den Pfad des übergeordneten Service an, der auch der Projektname und der Name des Ordners ist, in dem der übergeordnete Service abgelegt ist. Sie können die Platzhalter * und ? für eine allgemeinere Suche einsetzen. Alle Alert-Regeln, deren übergeordnete Services in diesem Pfad gespeichert sind, werden angezeigt.

      • Das Format für das Feld Pfad ist folgendermaßen:
      project-name/root-folder/ . . ./parent-folder

      Wenn ein Service sich direkt unter dem Projekt befindet, sieht das Format so aus: project-name

    • Nach übergeordnetem Service filtern. Klicken Sie im Feld Übergeordneter Service auf Durchsuchen, um die Seite Service wählen aufzurufen. Wählen Sie den übergeordneten Service aus der Liste, und klicken Sie auf Weiterleiten. Klicken Sie auf Löschen, um das Suchkriterium Übergeordneter Service zu entfernen, oder klicken Sie auf Suchen.
      • Hinweis: Die Auswahl, die Sie im Feld Übergeordneter Service treffen, schränkt Ihre Wahlmöglichkeiten im Feld Service-Typ ein.
    • Nach übergeordnetem Service-Typ filtern. Wählen Sie aus der Dropdown-Liste Service-Typ entweder die Option "Proxy-Services" oder "Business-Services", und klicken Sie dann auf Suchen. Mit der Wahl "Alle Services" ignorieren Sie das Filterkriterium "Übergeordneter Service-Typ".
      • Hinweis: Die Auswahl, die Sie im Feld Service-Typ treffen, schränkt Ihre Wahlmöglichkeiten im Feld Übergeordneter Service ein.
    • Nach Regelstatus filtern. Wählen Sie aus der Dropdown-Liste Regelstatus den Status der Alert-Regel: "Aktiviert" oder "Deaktiviert". Klicken Sie dann auf Suchen. Mit der Wahl "Alle" ignorieren Sie das Filterkriterium "Regelstatus".
    • Nach Schweregrad filtern. Schränken Sie über die Dropdown-Liste Schweregrad die SLA-Alert-Regeln auf die gewünschte Stufe ein. Klicken Sie dann auf Suchen.

      • Aktivieren Sie das Kontrollkästchen oder höher, um die Suche auf den gewünschten Schweregrad oder höher zu beschränken (von der weitreichendsten bis zur restriktivsten Stufe): Normal, Warnung, Unbedeutend, Kritisch und Schwerwiegend.

      Tabelle 26-6 SLA-Alert-Regeln - Zusammenfassende Informationen 
      Eigenschaft
      Beschreibung
      Name
      Name, der dieser Alert-Regeln zugewiesen wurde. Der Name ist ein Link zur Seite Alert-Regelkonfiguration. Siehe Alert-Regelkonfiguration anzeigen.
      SLA-Status
      Status der Alert-Regel: aktiviert oder deaktiviert.
      Beschreibung
      Hinweis: Dieses Feld ist standardmäßig ausgeblendet.
      Beschreibung der Alert-Regel.
      Service-Name
      Name des übergeordneten Service. Der Name ist ein Link zur Seite SLA-Alert-Regeln. Siehe Alert-Regeln anzeigen.
      Pfad
      Projekt, das mit dem übergeordneten Service der Alert-Regel verknüpft ist. Wenn der übergeordnete Service der Alert-Regel in einem Projektordner abgelegt ist, wird dieser Ordner auch aufgelistet. Der Pfad wird in folgendem Format angezeigt:
      project-name/root-folder/ . . ./parent-folder
      Der Pfad ist ein Link zum entsprechenden Pfad im Project Explorer.
      Schweregrad
      Schweregrad des Alerts, der von dieser Regel ausgelöst wird: Normal, Warnung, Unbedeutend, Gravierend, Kritisch oder Schwerwiegend.
      Aggr. intervall
      Dauer des Aggregationsintervalls in Stunden und Minuten.
      Ablaufdatum
      Datum, an dem diese Alert-Regel nicht mehr in Kraft ist.
      Verarbeitung stoppen
      Hinweis: Dieses Feld ist standardmäßig ausgeblendet.
      Zeigt "Ja" oder "Nein" an.
      Häufigkeit
      Häufigkeit dieses Alerts:
      • Jedes Mal
      • Einmal benachrichtigen

Auf dieser Seite können Sie auch folgende Aufgaben ausführen:

  • Wenn Sie den Suchfilter entfernen und alle SLA-Alert-Regeln anzeigen möchten, klicken Sie auf Alle anzeigen.
  • Klicken Sie auf Zurücksetzen, um Ihre Änderungen zu verwerfen und die Seite mit den aktuell gespeicherten Einstellungen und denselben Suchkriterien zu aktualisieren.
  • Wenn Sie aus einer Session heraus eine SLA-Alert-Regel aktivieren oder deaktivieren möchten, aktivieren oder deaktivieren Sie das Kontrollkästchen neben der Regel, und klicken Sie auf Aktualisieren.
  • Passen Sie die Anzeige der Informationen zu den Vorgangseinstellungen an, indem Sie auf das Symbol Tabellen-Customizer klicken. Weitere Informationen finden Sie unter Tabellenansichten anpassen.
