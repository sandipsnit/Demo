Vorgangseinstellungen suchen und aktualisieren

Auf der Seite Smart Search können Sie ganz einfach nach Proxy-Services, Business-Services, Alert-Zielen und SLA-Alert-Regeln suchen. Sie können auf der Seite Smart Search außerdem Service-spezifische Vorgangseinstellungen festlegen, wobei jedoch die folgenden Ausnahmen gelten: Sie können kein Aggregationsintervall festlegen und keine Alerting- oder Logging-Schweregradstufe angeben. Weitere Informationen finden Sie unter Vorgangseinstellungen für Proxy-Services konfigurieren und Vorgangseinstellungen für Business-Services konfigurieren.

Die Laufzeitauswirkungen der Einstellungen auf Service-Ebene hängen von den entsprechenden globalen Einstellungen ab. Sie müssen sowohl die globalen Einstellungen als auch die Einstellungen auf Service-Ebene aktivieren, damit der Service zur Laufzeit vollständig aktiviert ist. Zudem muss der Status des Service aktiviert sein. Weitere Informationen finden Sie unter Globale Einstellungen aktivieren.

Vorgangseinstellungen können nur aus einer Session heraus aktiviert und deaktiviert werden.
