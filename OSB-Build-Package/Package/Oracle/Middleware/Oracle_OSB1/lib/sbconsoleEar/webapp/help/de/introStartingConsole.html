Oracle Service Bus Administration-Konsole starten

  1. Starten Sie Oracle Service Bus mithilfe der folgenden Methoden:
    • Wählen Sie folgende Einträge aus dem Start-Menü von Windows aus:

      • Start > Alle Programme > Oracle WebLogic > Benutzerprojekte > domain_name > Server für Oracle Service Bus-Domain starten

      domain_name entspricht hier dem Namen, dem Sie Ihrer Oracle Service Bus-Domain bei der Erstellung zugewiesen haben. Benutzerprojekte ist das Standardverzeichnis, in dem Ihre Domain standardmäßig erstellt wird.

    • Rufen Sie den Befehl startWebLogic vom Stammverzeichnis der Oracle Service Bus-Domain auf: startWebLogic.cmd oder startWebLogic.sh

      • Ein Befehlsfenster mit den Statusinformationen zu WebLogic Server wird angezeigt.

      Wenn Sie den Server im Produktionsmodus gestartet haben, müssen Sie zum Starten des Servers einen gültigen Benutzernamen und ein Kennwort eingeben.

  2. Wenn der Server gestartet wird, geben Sie folgenden URL in Ihrem Browser ein:

    3. http://hostname:port/sbconsole

    host stellt hier den Namen des Rechners dar, auf dem der WebLogic Server ausgeführt wird. port entspricht der Port-Nummer.

    Beispiel: Wenn WebLogic Server auf Ihrem lokalen Rechner anhand der Standard-Port-Konfiguration ausgeführt wird, geben Sie folgenden URL in Ihrem Browser ein:

    http://localhost:7001/sbconsole

    Die Anmeldeseite für die Oracle Service Bus Administration-Konsole wird angezeigt.

  3. Geben Sie den Benutzernamen und das Kennwort an, das Sie während der Installation angegeben haben.
    4. Hinweis: Der Benutzername und das Kennwort für die Beispieldomain von Oracle Service Bus lauten "weblogic / welcome1".
  4. Klicken Sie auf Anmelden.

Zur Abmeldung klicken Sie im Banner oben rechts in der Oracle Service Bus Administration-Konsole auf Abmelden.
