JCA-Bindings

Oracle SOA Suite ermöglicht die Integration von Backend Enterprise Information Systems (EIS) mithilfe von J2EE Connector Architecture (JCARAWAdaptern. Mit Oracle JDeveloper und anderen unterstützten JCA-Entwicklungsumgebungen können Sie die Adapter konfigurieren und JCA-Adapterressourcen (.JCA-Dateien und abstrakte WSDLs) für die Backend-Integration erstellen.

Sie können außerdem mit JCA-kompatiblen EIS-Komponenten über Oracle Service Bus (OSB) kommunizieren, der die JCA-Ressourcen in Form von JCA-Binding-Ressourcen verbraucht. Mit JCA Bindings generieren Sie Proxy und Business-Services, die den Oracle Service Bus JCA-Transport für die Kommunikation mit den EIS-Anwendungen über JCA-Adapter benutzen.

Dieses Kapitel, das beschreibt, wie Sie JCA-Bindings in Oracle Service Bus hinzufügen, bearbeiten, suchen und löschen, enthält die folgenden Themen:

  • JCA-Bindings suchen
  • JCA-Bindings hinzufügen
  • JCA-Bindings bearbeiten
  • JCA-Bindings löschen

Weitere Informationen zum JCA-Transport finden Sie unter "JMS-Transport" im Oracle Fusion Middleware Developer's Guide for Oracle Service Bus auf der Website http://www.oracle.com/pls/as111170/lookup?id=OSBDV910.

Weitere Informationen zu Oracle SOA Suite JCA-Adaptern finden Sie im Oracle Fusion Middleware User's Guide for Technology Adapters auf der Website http://www.oracle.com/pls/as111170/lookup?id=TKADP.
