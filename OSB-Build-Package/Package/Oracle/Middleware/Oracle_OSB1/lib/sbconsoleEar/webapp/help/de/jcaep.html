JCA-Bindings löschen

Basiert ein beliebiger Business-Service oder Proxy-Service auf JCA-Binding, entfernen Sie JCA-Binding aus dem Business-Service oder Proxy-Service, bevor Sie das JCA-Binding löschen.

  1. Wenn dies noch nicht geschehen ist, klicken Sie zum Erstellen einer neuen Session auf Erstellen oder zum Aufrufen einer vorhandenen Session auf Bearbeiten. Weitere Informationen finden Sie unter Change Center verwenden.
  2. Wählen Sie Ressourcenbrowser > JCA-Bindings.
  3. Klicken Sie auf das Symbol Löschen im Feld Optionen des JCA-Bindings, welches Sie löschen wollen.

    4. Das JCA-Binding ist in der aktuellen Session gelöscht. Wenn ein Business-Service oder Proxy-Service für die Verwendung von JCA-Binding konfiguriert wurde, zeigt eine Löschwarnungan, dass Sie die JCA-Binding mit einer Warnungsbestätigung löschen können. Das könnte zu Konflikten aufgrund ungelöster Referenzen vom Service auf das gelöschte JCA-Binding führen.

  4. Um die Session zu beenden und die Konfiguration für die Laufzeit bereitzustellen, klicken Sie unter Change Center auf Aktivieren.
