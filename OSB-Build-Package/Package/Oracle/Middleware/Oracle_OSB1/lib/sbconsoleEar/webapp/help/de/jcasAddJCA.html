JCA-Bindings hinzufügen

Die einfachste Methode zum Erstellen von JCA-Bindings in Oracle Service Bus besteht darin, einen Bulk-Import für die erstellten JCA-Ressourcen (.jca-Datei, zugehörige WSDL und TopLink- oder EclipseLink-Mapping-Datei) durchzuführen. Beim Bulk-Import generiert Oracle Service Bus automatisch ein JCA-Binding aus .jca-Dateien, WSDL-Ressourcen aus den zugehörigen JCA-WSDLs und XML-Dokumentressourcen aus Mapping-Dateien und behält die Abhängigkeiten zwischen den Dateien bei.

Weitere Informationen zum Importieren finden Sie unter Ressourcen aus einer Zip-Datei laden, unter Ressourcen aus einem URL ladenund unter Ressourcen importieren.

Nachdem Sie die JCA-Ressourcen importiert haben, können Sie Proxy oder Business-Services daraus erstellen. Weitere Informationen hierzu finden Sie unter Proxy-Service aus einer JCA-Binding-Ressource generieren und unter Business-Service aus einer JCA-Binding-Ressource generieren.

Wenn Sie keinen Bulk-Import der JCA-Ressourcen durchführen, können Sie JCA-Bindings manuell erstellen. Gehen Sie dazu wie folgt vor:

  1. Erstellen Sie in Oracle JDeveloper eine JCA-Service-WSDL, eine zugehörige JCA-Datei (.jca) und, falls nötig, andere Ressourcen wie TopLink- oder EclipseLink-Mapping-Dateien und XSLT-Dateien.

    2. Weitere Informationen finden Sie im Oracle Fusion Middleware User's Guide for Technology Adapters unter http://www.oracle.com/pls/as111170/lookup?id=TKADP.

  2. Wenn dies noch nicht geschehen ist, klicken Sie zum Erstellen einer neuen Session in der OSB-Administrationskonsole auf Erstellen oder zum Aufrufen einer vorhandenen Session auf Bearbeiten. Weitere Informationen finden Sie unter Change Center verwenden.
  3. Erstellen oder importieren Sie in der OSB-Administrationskonsole die JCA-Ressourcen, die Sie in Oracle JDeveloper erstellt haben, darunter die WSDL- XML- und XSLT-Ressourcen.
  4. Erstellen Sie das JCA-Binding.
    1. Wählen Sie Project Explorer, und wählen Sie das Projekt- oder Ordnerverzeichnis für das JCA-Binding. Die Seite Projekt-/Ordneransicht wird angezeigt.
    2. Wählen Sie im Feld Ressource erstellen die Option JCA-Binding, um die Seite Neue JCA-Binding-Ressource erstellen anzuzeigen.
    3. Klicken Sie im Feld JCA-Datei auf Durchsuchen, um die JCA-Datei auszuwählen.

      4. Das Feld "Ressourcenname" wird automatisch mit dem Dateinamen (ohne Dateierweiterung) aufgefüllt.

      Anstatt eine Datei auszuwählen, können Sie den Content einer JCA-Datei kopieren und in den Textbereich des Feldes "JCA-Datei" einfügen.

    4. Geben Sie im Feld Ressourcenname den Namen des JCA-Bindings ein oder ändern Sie diesen. JCA-Binding-Namen müssen eindeutig sein.

      Befolgen Sie die Benennungseinschränkungen für Ressourcen bei der Namensgebung.
    5. Geben Sie im Feld Ressourcenbeschreibung eine Beschreibung für dieses JCA-Binding ein.
    6. Klicken Sie auf Speichern. Das JCA-Binding wird in der aktuellen Session gespeichert.

      7. Eine Validierungswarnung wird angezeigt, wenn das JCA-Binding noch keine Referenzen auf erforderliche Ressourcen (beispielsweise eine WSDL und eine Mapping-Datei) aufweist.

  5. Klicken Sie auf den Namen des neuen JCA-Bindings, um es auf der Seite JCA-Binding anzeigen anzuzeigen.
  6. Klicken Sie auf Referenzen bearbeiten.
  7. Klicken Sie auf der Seite Referenz einer JCA-Binding-Ressource bearbeiten auf Durchsuchen, um die JCA-Binding-Referenzressourcen (Beispiel: die WSDL, eine TopLink-/EclipseLink-Mapping-Datei, eine XSLT-Datei) zu wählen.
  8. Klicken Sie auf Speichern.
  9. Um die Session zu beenden und die Konfiguration für die Laufzeit bereitzustellen, klicken Sie unter Change Center auf Aktivieren.

Nachdem Sie JCA-Binding-Ressourcen hinzugefügt und die Abhängigkeiten der Ressourcen (unter Zuweisung einer WSDL) konfiguriert haben, oder wenn Sie JCA-Ressourcen importiert haben, die ihre Abhängigkeiten beibehalten, können Sie Proxy und Business-Services aus JCA-Bindings generieren. Weitere Informationen finden Sie unter Proxy-Service aus einer JCA-Binding-Ressource generieren und unter Business-Service aus einer JCA-Binding-Ressource generieren.
