JCA-Bindings suchen

Auf der Seite Übersicht der JCA-Bindings wird eine Liste der JCA-Bindings angezeigt, die Sie zur Oracle Service Bus-Domain hinzugefügt haben.

  1. Führen Sie eine der folgenden Aktionen aus:
    • Wählen Sie Project Explorer, um die Seite "Projektansicht" oder die Seite "Projekt-/Ordneransicht" anzuzeigen. Navigieren Sie dann durch die Projekte und Ordner, um das JCA-Binding zu suchen.
    • Wählen Sie Ressourcenbrowser > JCA-Bindings. Auf der Seite Übersicht der JCA-Bindings werden die in Tabelle 12-1 dargestellten Informationen angezeigt. Eine ausführliche Beschreibung der Eigenschaften finden Sie unter JCA-Bindings suchen.
  2. So suchen Sie ein bestimmtes JCA-Binding:
    • Um die Anzahl der Elemente in der Liste einzuschränken, können Sie nach Namen und Pfad filtern, bevor Sie auf Suchen klicken.

      • Der Pfad ist der Projektname und der Name der Ordner, in denen die Ressource gespeichert ist.

    • Um die Suchfilter zu entfernen und alle JCA-Bindings anzuzeigen, klicken Sie auf Alle anzeigen.

      • Tabelle 12-1 JCA-Binding-Informationen
      Eigenschaft
      Beschreibung
      JCA-Binding-Name
      Der eindeutige Name, der dem JCA-Binding zugewiesen ist. Wenn Sie auf den Namen klicken, wird die Seite JCA-Binding-Details anzeigen angezeigt. Weitere Informationen finden Sie unter JCA-Bindings bearbeiten.
      Pfad
      Der Projektname und der Name des Ordners, in dem das JCA-Binding gespeichert ist. Wenn Sie auf den Namen klicken, wird das Projekt oder der Ordner angezeigt, in dem diese Ressource enthalten ist. Weitere Informationen finden Sie unter Ressourcennamen mit Projekten und Ordnern angeben.
      Aktion
      Enthält ein Symbol "Generieren", mit dem Sie einen Oracle Service Bus-WSDL- und -JCA-Service aus dem JCA-Binding generieren können.
      Optionen
      Enthält ein Symbol Löschen. Wenn ein Business-Service oder Proxy-Service für die Verwendung eines JCA-Bindings konfiguriert wurde, gibt das Symbol Warnung wegen Löschvorgangan, dass Sie das JCA-Binding mit einer Warnungsbestätigung löschen können. Dadurch können Konflikte aufgrund nicht aufgelöster Referenzen vom Service zum gelöschten JCA-Binding entstehen. Weitere Informationen finden Sie unter JCA-Bindings löschen.
