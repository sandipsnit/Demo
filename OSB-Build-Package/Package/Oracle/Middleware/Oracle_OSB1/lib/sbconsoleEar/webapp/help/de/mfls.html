MFLs

Ein MFL-Dokument (Message Format Language) ist ein spezielles XML-Dokument zur Beschreibung von Binärdaten. Es handelt sich um eine eigene Oracle-Sprache zum Festlegen von Regeln, um formatierte Binärdaten in XML-Daten zu konvertieren. Ein MFL-Dokument stimmt mit der Dokumenttypdefinitionsdatei mfl.dtd überein, die Elemente und Attribute zur Beschreibung der einzelnen Datenfelder sowie Feldgruppierungen (Gruppen), Wiederholungen und Aggregation enthält.

Wenn Sie Business oder Proxy-Services vom Typ Messaging Service erstellen, können Sie MFL-Typen als Anforderungs- beziehungsweise Antwortnachrichtentyp für den Service bestimmen.

Sie verwenden Oracle Format Builder zum Erstellen von MFLs. Wenn Sie die Hierarchie eines binären Datensatzes, das Layout von Feldern und die Gruppierung von Feldern und Gruppen festlegen, werden diese Daten als MFL-Dokument gespeichert, das Sie anschließend zum Ausführen von Laufzeitübersetzungen verwenden können. MFL-Dokumente können ebenfalls in Format Builder zum Generieren der entsprechenden DTD-Dateien eingesetzt werden, die das Contentmodell beschreiben. Weitere Informationen zur Verwendung von Format Builder finden Sie unter "Format Builder" im Oracle Fusion Middleware Developer's Guide for Oracle Service Bus auf der Website http://www.oracle.com/pls/as111170/lookup?id=OSBDV763.
