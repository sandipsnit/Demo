Ansicht der Einträge der Domain-Log-Datei anpassen

  1. Klicken Sie auf Diese Tabelle anpassen. Zusätzliche Felder werden angezeigt.

    2. Sie können jederzeit auf den Link Diese Tabelle anpassen klicken, um die Tabelle zu schließen und die ursprünglichen Einstellungen wiederherzustellen.

  2. Führen Sie in den Feldern Filter folgende Aktionen aus:
    1. Wählen Sie im Feld Zeit das Zeitintervall, für das Sie Log-Einträge anzeigen möchten.
    2. Geben Sie im Feld Startzeit eine Startzeit im Format MM/TT/JJJJ HH:MM:SS ein. Beispiel: Geben Sie 10/25/06 08:39:48 ein. Verwenden Sie die Felder Startzeit und Endzeit, um ein Zeitfenster für das Anzeigen der Log-Einträge anzugeben.
    3. Geben Sie im Feld Endzeit eine Endzeit im Format MM/TT/JJJJ HH:MM:SS ein. Beispiel: Geben Sie 10/25/06 13:20:51 ein. Verwenden Sie die Felder Startzeit und Endzeit, um ein Zeitfenster für das Anzeigen der Log-Einträge anzugeben.
    4. Geben Sie im Textfeld Kriterien ein Filterkriterium (Textzeichenfolge) ein. Das Filterkriterium wird als Zeichenfolge in der WLDF-Abfragesprache angegeben. Die Abfragesprache unterstützt die Booleschen Operatoren AND, OR und NOT sowie relationale Operatoren. Weitere Informationen zur Abfragesyntax finden Sie unter "WLDF Query Language" in Oracle Fusion Middleware Configuring and Using the Diagnostics Framework for Oracle WebLogic Server auf der Website http://www.oracle.com/pls/as111170/lookup?id=WLDFC316.
  3. Wählen Sie im Feld Spalten die anzuzeigenden Spalten.
    1. Wählen Sie einen Spaltennamen im Feld Verfügbar.
    2. Klicken Sie auf den Pfeil, um den Spaltennamen in das Feld Gewählt zu verschieben.
    3. Wiederholen Sie die Schritte, bis Sie alle anzuzeigenden Spaltennamen im Feld Gewählt aufgelistet haben.

      4. Verwenden Sie im Feld Gewählt den nach unten und den nach oben weisenden Pfeil, um die Spaltennamen wunschgemäß anzuordnen.

  4. Wählen Sie im Feld Anzahl der pro Seite angezeigten Zeilen die Anzahl der Log-Einträge, die auf einer Seite angezeigt werden sollen.
  5. Wählen Sie im Feld Maximal zurückgegebene Ergebnisse die Höchstanzahl der Log-Einträge, die insgesamt angezeigt werden sollen, oder wählen Sie Alle anzeigen, um alle Einträge anzuzeigen.
  6. Führen Sie einen der folgenden Schritte aus:
    • Klicken Sie auf Anwenden, um die neuen Einstellungen zu speichern.
    • Klicken Sie auf Zurücksetzen, um die Änderungen zu verwerfen und die ursprünglichen Einstellungen beizubehalten.
