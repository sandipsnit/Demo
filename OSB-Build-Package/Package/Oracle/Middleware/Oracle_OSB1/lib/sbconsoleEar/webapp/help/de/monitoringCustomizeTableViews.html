Tabellenansichten anpassen

Mit dem Tabellen-Customizer können Sie Tabellendaten entsprechend den angegebenen Einstellungen anzeigen.

Beachten Sie, dass die Standardsortierreihenfolge für eine Tabelle durch die erste Spalte in der Tabelle festgelegt wird.

  1. Klicken Sie auf das Symbol Tabellen-Customizer. Es werden zusätzliche Felder angezeigt.

    2. Sie können jederzeit auf Abbrechen klicken, um diese Tabelle zu schließen und die ursprünglichen Einstellungen beizubehalten.

  2. Wählen Sie im Feld Spalten die anzuzeigenden Spalten.
    1. Wählen Sie im Feld Verfügbare Spalten einen Spaltennamen.
    2. Klicken Sie auf den Pfeil, um diesen Spaltennamen in das Feld Gewählte Spalten zu verschieben.

      3. Eine erforderliche Spalte wird durch ein Sternchen gekennzeichnet.

    3. Wiederholen Sie diesen Vorgang, bis die Liste alle Spaltennamen enthält, die im Feld Gewählte Spalten angezeigt werden sollen.

      4. Mit den Pfeilen nach oben und unten können Sie die Reihenfolge der Spaltennamen im Feld Gewählte Spalten gegebenenfalls ändern.

  3. Um Spalten aus der Anzeige zu entfernen, verschieben Sie diese Spaltennamen aus dem Feld Gewählte Spalten in das Feld Verfügbare Spalten.
  4. Wählen Sie in der Dropdown-Liste Zeilen die Anzahl der Zeilen, die Sie auf einer einzelnen Seite anzeigen möchten.
  5. Führen Sie einen der folgenden Schritte aus:
    • Um diese Einstellungen zu speichern und den Tabellen-Customizer zu schließen, klicken Sie auf Anwenden.
    • Klicken Sie auf Zurücksetzen, um die Änderungen zu verwerfen und die ursprünglichen Einstellungen beizubehalten.
