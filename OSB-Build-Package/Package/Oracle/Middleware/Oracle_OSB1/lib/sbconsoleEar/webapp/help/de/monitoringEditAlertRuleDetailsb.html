Alert-Regeln bearbeiten

  1. Wenn dies noch nicht geschehen ist, klicken Sie zum Erstellen einer neuen Session auf Erstellen oder zum Aufrufen einer vorhandenen Session auf Bearbeiten. Weitere Informationen finden Sie unter Change Center verwenden.
  2. Suchen Sie die gewünschte Alert-Regel. Weitere Informationen finden Sie unter Alert-Regeln suchen.
  3. Klicken Sie auf den Namen der Alert-Regel.
  4. Wenn Sie die Felder auf den Konfigurationsseiten ändern möchten, klicken Sie auf der entsprechenden Seite auf das Symbol Bearbeiten oder unten auf der Seite auf die Schaltfläche Bearbeiten. Eine genauere Beschreibung der Seiten und Felder finden Sie unter Alert-Regeln erstellen und bearbeiten.
  5. Führen Sie einen der folgenden Schritte aus:
    • Klicken Sie auf Weiter, um zur nächsten Seite zu gelangen.
    • Klicken Sie auf Zurück, um zur vorherigen Seite zurückzukehren.
    • Klicken Sie auf Letzte, um zur Übersicht zu gelangen.
    • Klicken Sie auf Speichern, um die Aktualisierungen in der aktuellen Session zu speichern.
    • Wenn Sie die Änderungen verwerfen und zur Seite SLA-Alert-Regeln zurückkehren möchten, klicken Sie auf Abbrechen.
  6. Um die Session zu beenden und die Konfiguration für die Laufzeit bereitzustellen, klicken Sie unter Change Center auf Aktivieren.
