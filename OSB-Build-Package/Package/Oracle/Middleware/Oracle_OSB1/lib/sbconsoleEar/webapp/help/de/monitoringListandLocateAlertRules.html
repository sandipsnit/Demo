Alert-Regeln anzeigen

Auf dieser Seite können Sie die Alert-Regeldaten aus Tabelle 25-19 anzeigen.

Tabelle 25-19 Alert-Regeldaten 
Eigenschaft
Beschreibung
Name
Der Name, der dieser Alert-Regel zugeordnet ist. Der Name ist ein Link zur Seite Alert-Regel - Allgemeine Konfiguration. Siehe Alert-Regel-Konfiguration anzeigen.
SLA-Status
Status der Alert-Regel: aktiviert oder deaktiviert.
Beschreibung
Dieses Feld ist standardmäßig ausgeblendet.
Beschreibung der Alert-Regel.
Schweregrad
Schweregrad des Alerts, der von dieser Regel ausgelöst wird: Normal, Warnung, Unbedeutend, Gravierend, Kritisch oder Schwerwiegend.
Aggr. intervall
Dauer des Aggregationsintervalls in Stunden und Minuten.
Ablaufdatum
Datum, an dem diese Alert-Regel nicht mehr in Kraft ist.
Verarbeitung stoppen
Zeigt "Ja" oder "Nein" an.
Häufigkeit
Häufigkeit dieses Alerts:
  • Jedes Mal
  • Einmal benachrichtigen
Optionen
Diese Spalte enthält folgende Funktionen:
  • Klicken Sie auf die Pfeile, um die Alert-Regeln neu zu sortieren. Sie können einzelne Alert-Regeln nach oben oder nach unten verschieben. Dies ist allerdings nur möglich, wenn mehr als eine Regel für diesen Service konfiguriert wurde.
  • Klicken Sie auf das Symbol Umbenennen, um eine Alert-Regel umzubenennen.
  • Klicken Sie auf das Symbol Löschen, um eine bestimmte Alert-Regel zu löschen. Siehe Alert-Regeln löschen.
