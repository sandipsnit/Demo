Alerts löschen

Auf dieser Seite können Sie die angegebenen SLA- oder Pipeline Alerts dauerhaft löschen.

Hinweis: Diese Aktion kann nicht rückgängig gemacht werden.
  1. Wählen Sie Vorgänge > Dashboard.
  2. Wählen Sie die Registerkarte SLA-Alerts oder Pipeline Alerts, und klicken Sie auf den Link Erweiterte Alert-Historie.
  3. Klicken Sie rechts oben auf der Seite Erweiterte Alert-Historie auf SLA-Alert-Historie löschen oder auf Pipeline Alert-Historie löschen.
  4. Führen Sie einen der folgenden Schritte aus, die in Tabelle 25-3 dargestellt sind.

    5. Tabelle 25-3 Alerts löschen – Optionen
    Ziel
    Vorgehensweise
    • Alle SLA-Alerts löschen
    • Alle Pipeline Alerts löschen
    1. Wählen Sie Alle Alerts zurücksetzen.
    2. Klicken Sie auf Löschen.
    • SLA-Alerts innerhalb eines bestimmten Zeitrahmens löschen
    • Pipeline Alerts innerhalb eines bestimmten Zeitrahmens löschen
    1. Wählen Sie Alerts löschen von... Bis...
    2. Geben Sie im Feld Löschen von einen Monat, einen Tag, ein Jahr und eine Uhrzeit ein.
    3. Geben Sie im Feld Löschen bis einen Monat, einen Tag, ein Jahr und eine Uhrzeit ein.
    4. Klicken Sie auf Löschen.

Hinweis: Wenn der Server nicht verfügbar ist, von dem die Alerts gelöscht werden sollen, speichert das Logging Framework einen Datensatz des Löschvorgangs und fragt den Server alle 30 Minuten ab, um dessen Verfügbarkeit zu prüfen. Wenn das Logging Framework die Verfügbarkeit des Servers feststellt, werden die Alerts gelöscht.
