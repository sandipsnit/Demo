Statistiken für Services zurücksetzen

Sie können die Seite Service-Integrität verwenden, um die Überwachungsstatistiken aller Services in Ihrer Konfiguration zurückzusetzen. Sie können die Statistiken zurücksetzen, unabhängig davon, ob Sie sich in einer Session befinden oder nicht.

  1. Wählen Sie Vorgänge > Dashboard > Service-Integrität.
  2. Wählen Sie im Feld Statistik anzeigen Seit der letzten Rücksetzung. Die Ansicht "Seit der letzten Rücksetzung" zeigt die Metriken in einer fortlaufenden Zahl an.
  3. Wenn Sie die Überwachungsstatistik für einen bestimmten Service zurücksetzen möchten, klicken Sie auf das Symbol Zurücksetzen in der Spalte Aktion für diesen Service.

    4. Mit Ihrer Bestätigung löscht das System dann alle Überwachungsstatistiken, die für den Service gesammelt wurden, seit Sie die Statistik zum letzten Mal zurückgesetzt haben. Das System löscht jedoch keine Statistiken, die für den Service innerhalb des Aktuellen Aggregationsintervalls gesammelt wurden. Außerdem beginnt das System sofort Überwachungsstatistiken für den Service zu sammeln, nachdem Sie erneut auf das Symbol Zurücksetzen geklickt haben.

  4. Wenn Sie für alle Services die Überwachungsstatistiken zurücksetzen möchten, klicken Sie auf den Link Alle Statistiken zurücksetzen auf der Seite oben.
    5. Hinweis: Klicken auf Alle Statistiken zurücksetzen setzt die Statistik für alle Services zurück, gleich ob sie auf der Seite angezeigt werden oder nicht.

    Mit Ihrer Bestätigung löscht das System alle Überwachungsstatistiken, die für alle Services in Ihrer Konfiguration gesammelt wurden, seit Sie die Statistik zum letzten Mal zurückgesetzt haben. Das System löscht jedoch keine Statistiken, die für den Service innerhalb des Aktuellen Aggregationsintervalls für einen beliebigen Service gesammelt wurden. Außerdem beginnt das System sofort Überwachungsstatistiken für den Service zu sammeln, nachdem Sie erneut auf das Symbol Alle Statistiken zurücksetzen geklickt haben.
