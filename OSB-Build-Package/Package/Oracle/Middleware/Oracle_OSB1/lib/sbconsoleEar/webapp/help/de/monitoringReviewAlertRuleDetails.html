Alert-Regelkonfiguration überprüfen

Verwenden Sie diese Seite, um die Konfigurationsdaten zu überprüfen, die Sie für diese Alert-Regel eingegeben haben. Gegebenenfalls können Sie auf Bearbeiten klicken, um Änderungen an der Konfiguration vorzunehmen, bevor Sie die Alert-Regel speichern.

Führen Sie einen der folgenden Schritte aus:

  • Wenn Sie eine Änderung an einer der Konfigurationsseiten machen wollen, klicken Sie auf das Symbol Bearbeiten für die entsprechende Seite.
  • Klicken Sie auf Zurück, um zur vorherigen Seite zurückzukehren.
  • Klicken Sie auf Speichern, um den Alert zu erstellen. Die Alert-Regel wird in der aktuellen Session erstellt und gespeichert.
  • Klicken Sie auf Abbrechen, wenn Sie Änderungen verwerfen möchten.
  • Um die Session zu beenden und die Konfiguration für die Laufzeit bereitzustellen, klicken Sie unter Change Center auf Aktivieren.
