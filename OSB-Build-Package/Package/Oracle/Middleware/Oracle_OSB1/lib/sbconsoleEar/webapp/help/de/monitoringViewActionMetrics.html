Aktionsmetriken anzeigen

Verwenden Sie diese Seite, um die Service-Überwachungsinformation anzuzeigen, die in Tabelle 25-10 dargestellt ist, sowie die Aktionsmetriken für Proxy-Services, die über Pipelines verfügen, dargestellt in Tabelle 25-14.

Tabelle 25-14 Aktionsmetriken
Eigenschaft
Beschreibung
Knoten
Eine hierarchische Anzeige der verschachtelten Aktionen.
Anzahl
Wenn Sie Aktuelles Aggregationsintervall im Feld Statistik anzeigen gewählt haben, die Anzahl der Aufrufe innerhalb der Periode des aktuellen Aggregationsintervalls.
Wenn Sie Seit der letzten Rücksetzung im Feld Statistik anzeigen gewählt haben, die Anzahl der Aufrufe innerhalb der Periode seit der letzten Rücksetzung der Statistik.
Beachten Sie, dass diese sich von der Anzahl der Nachrichten unterscheidet, da eine Aktion nicht mehrmals pro Session aufgerufen werden kann.
Durchschnittliche Dauer
Wenn Sie Aktuelles Aggregationsintervall im Feld Statistik anzeigen gewählt haben, die durchschnittliche Ausführungszeit pro Aufruf innerhalb der Periode des aktuellen Aggregationsintervalls.
Wenn Sie Seit der letzten Rücksetzung im Feld Statistik anzeigengewählt haben, die durchschnittliche Ausführungszeit pro Aufruf innerhalb der Periode seit der letzten Rücksetzung der Statistik.
Gesamtzeit
Wenn Sie Aktuelles Aggregationsintervall im Feld Statistik anzeigen gewählt haben, die gesamte Ausführungszeit innerhalb der Periode des aktuellen Aggregationsintervalls.
Wenn Sie Seit der letzten Rücksetzung im Feld Statistik anzeigen gewählt haben, die gesamte Ausführungszeit innerhalb der Periode seit der letzten Rücksetzung der Statistik.
