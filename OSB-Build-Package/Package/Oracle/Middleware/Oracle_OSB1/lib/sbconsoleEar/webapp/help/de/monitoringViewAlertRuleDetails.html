Alert-Regelkonfiguration anzeigen

Verwenden Sie diese Site, um Alert-Regelkonfigurationsdetails anzuzeigen, die in Tabelle 25-22 dargestellt sind.

Tabelle 25-22 Alert-Regel - Allgemeine Konfiguration 
Eigenschaft
Beschreibung
Regelname
Der Name der Alert-Regel.
Regelbeschreibung
Beschreibung der Alert-Regel.
Alert-Zusammenfassung
Eine kurze Beschreibung der Alert-Regel. Dieser Text wird auch als Betreffzeile in einer E-Mail verwendet, wenn diese Alert-Regel mit einer Alert-Benachrichtigung per E-Mail konfiguriert ist.
Alert-Ziel
Der Pfad und der Name der Alert-Zielressource, die die Zieladresse für Alert-Benachrichtigungen enthält. Das Feld ist ein Link zu der Seite Alert-Ziel anzeigen - Allgemeine Konfiguration. Siehe Alert-Ziele bearbeiten.
Startzeit (HH:MM)
Die Startzeit für diese Alert-Regel.
Endzeit (HH:MM)
Die Endzeit für diese Alert-Regel.
Regelablaufdatum (MM/TT/JJJJ)
Das Datum, ab wann diese Alert-Regel nicht mehr gilt.
Regel aktiviert
Zeigt an, ob die Alert-Regel aktiviert oder deaktiviert ist.
Alert-Schweregrad
Der Schweregrad des ausgelösten Alerts als Ergebnis der Alert-Regel:
  • Normal
  • Warnung
  • Unbedeutend
  • Gravierend
  • Kritisch
  • Schwerwiegend
Alert-Häufigkeit
Die Häufigkeit des Alerts.
  • Jedes Mal
  • Einmal benachrichtigen
Verarbeitung von mehr Regeln anhalten
Ob der Alert das Verarbeiten von mehr Regeln anhält oder nicht.

Eigenschaft
Beschreibung
Bedingungsausdruck

Eine Liste von einer oder mehreren Bedingungen, die diese Alert-Regel beherrschen.
