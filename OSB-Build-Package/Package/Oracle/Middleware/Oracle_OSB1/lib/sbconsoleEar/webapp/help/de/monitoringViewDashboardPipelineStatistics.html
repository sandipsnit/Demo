Pipeline Alerts anzeigen

Auf der Seite Pipeline Alerts können Sie eine Übersicht der Pipeline Alerts anzeigen und detaillierte Informationen zur Alert-Historie abrufen, die in Tabelle 25-2 näher beschrieben sind.

Auf der Seite Pipeline Alerts wird Folgendes angezeigt:

  • Die Registerkarten SLA-Alerts, Service-Integrität und Serverintegrität

    • Im Header der Registerkarte "Pipeline Alerts" zeigt ein Zahlenindikator die Anzahl der neuen Alerts an, die seit der letzten Anzeige der Alerts oder während des letzten Dashboard-Aktualisierungszeitraums aufgetreten sind (je nachdem, welches Ereignis am kürzesten zurückliegt). Die Dashboard-Aktualisierungsrate ist standardmäßig auf Keine Aktualisierung gesetzt. Informationen zum Ändern des Intervalls für die Dashboard-Aktualisierungsrate finden Sie unter Benutzervoreinstellungen festlegen.

Achtung: Wenn Sie einen anderen Wert als den Standardwert Keine Aktualisierung als Aktualisierungsrate für das Dashboard wählen, aktualisiert der Browser das Dashboard in regelmäßigen Abständen. Dadurch wird ein Timeout der Session verhindert, selbst wenn Sie nicht an der Oracle Service Bus Administration-Konsole arbeiten.
Achtung: Oracle empfiehlt, die Dashboard-Aktualisierungsrate mit Vorsicht anzuwenden. Lassen Sie eine Session nie unbeaufsichtigt. Während die Session aktiv ist, können Dritte ohne erneute Authentifizierung über Ihren Rechner Zugriff darauf erhalten.
  • Ein Kreisdiagramm der Services mit Pipeline Alerts der letzten 30 Minuten (dies ist das Standardintervall der Alert-Historie). Informationen zum Ändern des Zeitintervalls für die Anzeige von historischen Alert-Daten finden Sie unter Benutzervoreinstellungen festlegen.

    • Die Anzahl der Pipeline Alerts auf der Seite entspricht dem Kreisdiagramm, das eine Aufgliederung der Alerts nach Schweregradstufe während desselben Intervalls darstellt. Sie können im Kreisdiagramm auf den Bereich für einen Alert-Schweregrad klicken, um weitere Details anzuzeigen.

  • Die zehn ersten Services mit Pipeline Alerts werden in aufsteigender Reihenfolge der Anzahl der Alerts angezeigt, die von diesen Services ausgegeben werden. Jeder Service-Name ist ein Link zu den Service-Überwachungsdetails. Weitere Informationen finden Sie unter ###Service-Metriken anzeigen.
  • Felder, die entsprechend Ihren Einstellungen für die Alert-Tabelle Alert-Informationen anzeigen. Beispiel: Zeitstempel, Alert-Übersicht, Alert-Schweregrad, Aktion sowie die Namen der Services und die mit dem Alert verknüpften Service-Typen. Weitere Informationen zu diesen Feldern finden Sie in Tabelle 25-2.

    • Das Symbol Anmerkung neben einer Alert-Übersicht weist auf Anmerkungen hin. Wenn Sie mit dem Mauszeiger auf das Symbol zeigen, wird der Anmerkungstext angezeigt. Klicken Sie auf das Symbol, um den gesamten Anmerkungstext anzuzeigen.

  • Das Symbol Tabellen-Customizer. Siehe Tabellenansichten anpassen.
  • Ein Link zur Seite Erweiterte Alert-Historie
