SLA-Alerts anzeigen

Verwenden Sie die Seite SLA-Alerts , um eine Zusammenfassung der SLA-Alerts anzuzeigen und um auf detaillierte Alert-Historieninformation zuzugreifen, wie in Tabelle 25-1 dargestellt.

Die Seite SLA-Alerts enthält Folgendes

  • Registerkarten zum Anzeigen von Informationen zu Pipeline-Alerts, Service-Integrität und Server-Integrität.

    • Auf dem Registerkarten-Header von SLA-Alerts gibt eine Zahlenanzeige die Anzahl der neuen Alerts an, die aufgetreten sind, seit Sie sie zuletzt angezeigt haben oder seit der letzten Dashboard-Aktualisierungsperiode, je nachdem was aktueller ist. Standardmäßig ist die Dashboard-Aktualisierungsrate Keine Aktualisierung. Wie Sie das Intervall der Dashboard-Aktualisierungsrate ändern können, ist unter Benutzervoreinstellungen einrichten beschrieben.

Achtung: Wenn Sie einen anderen Wert als den Standardwert Keine Aktualisierung als Aktualisierungsrate für das Dashboard wählen, aktualisiert der Browser das Dashboard in regelmäßigen Abständen. Dadurch wird ein Timeout der Session verhindert, selbst wenn Sie nicht an der Oracle Service Bus Administration-Konsole arbeiten.
Achtung: Oracle empfiehlt, die Dashboard-Aktualisierungsrate mit Vorsicht anzuwenden. Lassen Sie eine Session nie unbeaufsichtigt. Während die Session aktiv ist, können Dritte ohne erneute Authentifizierung über Ihren Rechner Zugriff darauf erhalten.
  • Ein Kreisdiagramm der Services mit SLA-Alerts der letzten 30 Minuten, (dies ist das Standardintervall der Alert-Historie). Informationen darüber, wie dieses Zeitintervall für die Anzeige der historischen Alert-Daten geändert wird, finden Sie unter Benutzervoreinstellungen einrichten.

    • Die Anzahl der SLA-Alerts auf der Seite stimmt mit dem Kreisdiagramm überein, das die Analyse der Alerts nach Schweregrad anzeigt, über das gleiche Zeitintervall. Sie können auf den Bereich im Diagramm klicken, der dem Alert-Schweregrad entspricht, um mehr Details anzuzeigen.

  • Die Top-10-Services mit SLA-Alerts werden in absteigender Reihenfolge der Alert-Anzahl angezeigt, die von diesen Services ausgegeben wurden. Jeder Service-Name ist ein Link zu den Service-Überwachungsdetails. Siehe Service-Metriken anzeigen.
  • Felder, die Alert-Information in Anhängigkeit von den benutzerdefinierten Einstellungen für die Alert-Tabelle anzeigen. Offlinesetzen Zeitstempel, Alert-Name, Alert-Schweregrad, Aktion und die Namen des Service und der Service-Typ, der dem Alert zugeordnet ist. Mehr Informationen zu diesen Feldern finden Sie in Tabelle 25-1.

    • Das Symbol Anmerkung neben dem Alert-Namen zeigt Anmerkungen an. Wird der Mauszeiger über das Symbol gebracht, erfolgt eine Vorschau des Anmerkungstexts. Wenn Sie auf das Symbol klicken, können Sie den gesamten Text der Anmerkung anzeigen.

  • Das Symbol Tabellen-Customizer. Siehe Tabellenansichten anpassen.
  • Ein Link zu der Seite Erweiterte Alert-Historie. Siehe Alerts finden.
