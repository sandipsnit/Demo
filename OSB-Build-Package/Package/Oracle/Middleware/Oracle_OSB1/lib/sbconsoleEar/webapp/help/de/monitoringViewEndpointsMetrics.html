Metriken für Business-Service-Endpunkt-URIs anzeigen

Sie verwenden diese Seite, um die Service-Überwachungsinformationen anzuzeigen, die in Tabelle 25-10 aufgelistet sind, sowie die Metriken der Endpunkt-URIs für Business-Services aus Tabelle 25-15. Weitere Informationen finden Sie unter "Managing Endpoint URIs for Business-Services" im Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus auf der Website http://www.oracle.com/pls/as111170/lookup?id=OSBAG595.

Tabelle 25-15 Metriken der Endpunkt-URIs
Eigenschaft
Beschreibung
Endpunkt-URI
Endpunkt-URI, der mit diesem Business-Service verknüpft ist.
Anzahl Nachrichten
Wenn Sie im Feld Statistiken anzeigen die Option Aktuelles Aggregationsintervall gewählt haben, ist dies die Gesamtanzahl der von diesem Endpunkt-URI innerhalb des aktuellen Aggregationsintervalls ausgeführten Nachrichten.
Wenn Sie im Feld Statistiken anzeigen die Option Seit der letzten Rücksetzung gewählt haben, ist dies die Gesamtanzahl der von diesem Endpunkt-URI innerhalb des Zeitraums seit der letzten Statistikrücksetzung ausgeführten Nachrichten.
Fehleranzahl
Wenn Sie im Feld Statistiken anzeigen die Option Aktuelles Aggregationsintervall gewählt haben, ist dies die Anzahl der von diesem Endpunkt-URI innerhalb des Zeitraums des aktuellen Aggregationsintervalls ausgeführten Nachrichten mit Fehlern.
Wenn Sie im Feld Statistiken anzeigen die Option Seit der letzten Rücksetzung gewählt haben, ist dies die Anzahl der von diesem Endpunkt-URI innerhalb des Zeitraums seit der letzten Statistikrücksetzung ausgeführten Nachrichten mit Fehlern.
Min. Antwortzeit
Wenn Sie im Feld Statistiken anzeigen die Option Aktuelles Aggregationsintervall gewählt haben, ist dies die minimale Antwortzeit, die dieser Endpunkt-URI zum Ausführen von Nachrichten innerhalb des aktuellen Aggregationsintervalls benötigt hat.
Wenn Sie im Feld Statistiken anzeigen die Option Seit der letzten Rücksetzung gewählt haben, ist dies die minimale Antwortzeit, die dieser Endpunkt-URI zum Ausführen von Nachrichten innerhalb des Zeitraums seit der letzten Statistikrücksetzung benötigt hat.
Max. Antwortzeit
Wenn Sie im Feld Statistiken anzeigen die Option Aktuelles Aggregationsintervall gewählt haben, ist dies die maximale Antwortzeit, die dieser Endpunkt-URI zum Ausführen von Nachrichten innerhalb des aktuellen Aggregationsintervalls benötigt hat.
Wenn Sie im Feld Statistiken anzeigen die Option Seit der letzten Rücksetzung gewählt haben, ist dies die maximale Antwortzeit, die dieser Endpunkt-URI zum Ausführen von Nachrichten innerhalb des Zeitraums seit der letzten Statistikrücksetzung benötigt hat.
Durchschn. Antwortzeit
Wenn Sie im Feld Statistiken anzeigen die Option Aktuelles Aggregationsintervall gewählt haben, ist dies die durchschnittliche Antwortzeit, die dieser Endpunkt-URI zum Ausführen von Nachrichten innerhalb des aktuellen Aggregationsintervalls benötigt hat.
Wenn Sie im Feld Statistiken anzeigen die Option Seit der letzten Rücksetzung gewählt haben, ist dies die durchschnittliche Antwortzeit, die dieser Endpunkt-URI zum Ausführen von Nachrichten innerhalb des Zeitraums seit der letzten Statistikrücksetzung benötigt hat.
Status
Status des Business-Service-Endpunkt-URI.
Für einzelne Serverdomains oder für einen einzelnen Managed Server in einem Cluster: "Online" oder "Offline".
Für Statistiken auf Cluster-Ebene: "Online", "Offline" oder "Teilweise", wenn dieser bestimmte URI bei mindestens einem Managed Server im Cluster offline ist.
Aktion
Klicken Sie auf das Symbol Endpunkt online markieren, um einen URI mit Status "Offline" als "Online" zu markieren. Diese Aktion ist nur aktiviert, wenn der URI den Status "Offline" oder "Teilweise" hat.
Um diese Aktion zu aktivieren, muss ein Business-Service zum Offlinesetzen von URIs und seine Service- und globalen Überwachungsvorgangseinstellungen aktiviert sein. Siehe Vorgangseinstellungen für Business-Services konfigurieren.
Hinweis: Wenn die Aktion erfolgreich ist, wirkt sie unmittelbar. Die Umsetzung aber, dass der Endpunkt-URI-Status jetzt online ist, dauert einen Überwachungsaggregationszyklus lang, welcher bis zu eineinhalb Minuten dauern kann.
