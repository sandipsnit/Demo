Vorgangsmetriken für WSDL-basierte Services anzeigen

Sie verwenden diese Seite, um die Service-Überwachungsinformationen anzuzeigen, die in der Tabelle 25-10 aufgelistet sind, sowie die Vorgangsmetriken für WSDL-basierte Services aus Tabelle 25-12.

Tabelle 25-12 Vorgangsmetriken
Eigenschaft
Beschreibung
Vorgangsname
Name des mit diesem Service verknüpften Vorgangs.
Anzahl Nachrichten
Wenn Sie im Feld Statistiken anzeigen die Option Aktuelles Aggregationsintervall gewählt haben, ist dies die Gesamtanzahl Nachrichten, die dieser Vorgang innerhalb des aktuellen Aggregationsintervalls ausgeführt hat.
Wenn Sie im Feld Statistiken anzeigen die Option Seit der letzten Rücksetzung gewählt haben, ist dies die Gesamtanzahl Nachrichten, die dieser Vorgang innerhalb des Zeitraums seit der letzten Statistikrücksetzung ausgeführt hat.
