Benutzerdefinierte WS-Policys hinzufügen

Sie können entweder eine XML-Datei importieren, die Ihre WS-Policy enthält, oder Sie schreiben die WS-Policy direkt in die Oracle Service Bus Administration-Konsole.

  1. Wenn dies noch nicht geschehen ist, klicken Sie zum Erstellen einer neuen Session auf Erstellen oder zum Aufrufen einer vorhandenen Session auf Bearbeiten. Weitere Informationen finden Sie unter Change Center verwenden.
  2. Wählen Sie Project Explorer und dann ein Projekt oder einen Ordner, in dem Sie die WS-Policy hinzufügen möchten. Die Seite Projektansicht/Ordneransicht wird angezeigt.
  3. Wählen Sie aus der Dropdown-Liste Ressource erstellen den Wert WS-Policy unter Schnittstelle, um die Seite Neue WS-Policy erstellen anzuzeigen.
  4. Geben Sie im Feld Ressourcenname einen eindeutigen Namen für diese WS-Policy ein.

    Befolgen Sie die Benennungsbeschränkungen für Ressourcen bei der Namensgebung.
  5. Im Feld Ressourcenbeschreibung geben Sie eine Beschreibung dieser WS-Policy ein.
  6. Führen Sie im Feld Policy einen der folgenden Schritte aus:
    • Klicken Sie auf Durchsuchen, wenn Sie eine XML-Datei suchen und importieren möchten, in der Ihre benutzerdefinierte WS-Policy enthalten ist.
    • Geben Sie Text für die neue WS-Policy ein.
    • Kopieren Sie den Text einer vorhandenen WS-Policy, und fügen Sie ihn in dieses Feld ein:
  7. Klicken Sie auf Speichern. Die WS-Policy wird in der aktuellen Session gespeichert.
  8. Um die Session zu beenden und die Konfiguration für die Laufzeit bereitzustellen, klicken Sie unter Change Center auf Aktivieren.
