Benutzerdefinierte WS-Policys löschen

Wenn ein Business oder Proxy-Service für die Verwendung der WS-Policy konfiguriert wurde, entfernen Sie die WS-Policy aus dem Business oder Proxy-Service.

  1. Wenn dies noch nicht geschehen ist, klicken Sie zum Erstellen einer neuen Session auf Erstellen oder zum Aufrufen einer vorhandenen Session auf Bearbeiten. Weitere Informationen finden Sie unter Change Center verwenden.
  2. Wählen Sie Ressourcenbrowser > WS-Policys.
  3. Klicken Sie auf der Symbol Löschen im Feld Optionen der WS-Policy, die gelöscht werden soll.

    4. Die WS-Policy wird in der aktuellen Session gelöscht. Wenn ein Business oder Proxy-Service für die Verwendung einer WS-Policy konfiguriert wurde, zeigt das Symbol Warnung wegen Löschvorgang an, dass Sie die Ressource mit einer Warnbestätigung löschen können. Dies kann jedoch zu Konflikten aufgrund von nicht aufgelösten Referenzen vom Service zur gelöschten WS-Policy führen.

  4. Um die Session zu beenden und die Konfiguration für die Laufzeit bereitzustellen, klicken Sie unter Change Center auf Aktivieren.
