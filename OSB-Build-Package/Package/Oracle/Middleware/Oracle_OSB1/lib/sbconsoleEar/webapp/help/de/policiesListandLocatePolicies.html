Benutzerdefinierte WS-Policys suchen

Auf der Seite Zusammenfassung der WS-Policys werden die benutzerdefinierten Web Service Policys (WS-Policys) angezeigt, die Sie der aktuellen Oracle Service Bus-Domain hinzugefügt haben. In der Oracle Service Bus Administration-Konsole werden keine Oracle Service Bus-WS-Policys angezeigt, die vorkonfiguriert und in allen Oracle Service Bus-Domains verfügbar sind.

  1. Führen Sie eine der folgenden Aktionen aus:
    • Wählen Sie Project Explorer, um die Seite "Projektansicht" oder die Seite "Projekt-/Ordneransicht" anzuzeigen. Durchsuchen Sie dann die Projekte und Ordner nach der WS-Policy.
    • Wählen Sie Ressourcenbrowser > WS-Policys. Auf der Seite Zusammenfassung der WS-Policys werden die in Tabelle 11-1 beschriebenen Informationen angezeigt. Eine detailliertere Beschreibung der Eigenschaften finden Sie unter Benutzerdefinierte WS-Policys suchen.
  2. Wenn Sie nach einer benutzerdefinierten WS-Policy suchen möchten, geben Sie einen Teil des Namens oder den gesamten Namen der WS-Policy in das Feld Name ein. Sie können auch einen Teil des Projektnamens oder den gesamten Projektamen oder Ordner der WS-Policy in das Feld Pfad eingeben. Klicken Sie anschließend auf Suchen.

    3. Klicken Sie auf Alle anzeigen, um die Suchfilter zu deaktivieren und alle WS-Policys anzuzeigen.

    Tabelle 11-1 Informationen zu WS-Policys 
    Eigenschaft
    Beschreibung
    WS-Policy-Name
    Der eindeutige Name der WS-Policy. Klicken Sie auf den Namen, um die Seite WS-Policy-Details anzeigen anzuzeigen. Weitere Informationen finden Sie unter Benutzerdefinierte WS-Policys bearbeiten.
    Pfad
    Der Projektname und der Name des Ordners, in dem die WS-Policy gespeichert ist. Klicken Sie auf den Namen, um das Projekt oder den Ordner anzuzeigen, das bzw. der diese Ressource enthält. Weitere Informationen finden Sie unter Ressourcennamen mit Projekten und Ordner angeben.
    Optionen
    Enthält ein Symbol Löschen. Wenn ein Business-Service oder ein Proxy-Service für die Verwendung einer WS-Policy konfiguriert wurde, weist ein Symbol Warnung wegen Löschvorgang darauf hin, dass Sie die WS-Policy löschen können, indem Sie den Löschvorgang bestätigen. Dies kann aufgrund nicht aufgelöster Referenzen zwischen dem Service und der gelöschten WS-Policy zu Konflikten führen. Weitere Informationen finden Sie unter Benutzerdefinierte WS-Policys löschen
