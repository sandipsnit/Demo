Benutzerdefinierte WS-Policys bearbeiten

Auf der Seite WS-Policy-Details anzeigen können Sie Details einer bestimmten benutzerdefinierten Web Service Policy (WS-Policy) anzeigen und bearbeiten. In der Oracle Service Bus Administration-Konsole werden keine schreibgeschützten Oracle Service Bus-WS-Policys angezeigt.

  1. Wenn dies noch nicht geschehen ist, klicken Sie zum Erstellen einer neuen Session auf Erstellen oder zum Aufrufen einer vorhandenen Session auf Bearbeiten. Weitere Informationen finden Sie unter Change Center verwenden.
  2. Suchen Sie wie unter Benutzerdefinierte WS-Policys suchen beschrieben die WS-Policy.
  3. Klicken Sie auf den Namen der WS-Policy. Auf der Seite WS-Policy-Details anzeigen werden die in Tabelle 11-2 beschriebenen Informationen angezeigt.

    4. Tabelle 11-2 WS-Policy-Details 
    Eigenschaft
    Beschreibung
    Zuletzt geändert von
    Der Benutzer, der diese WS-Policy erstellt oder in die Konfiguration importiert hat.
    Zuletzt geändert am
    Datum und Uhrzeit des Zeitpunkts, an dem der Benutzer die WS-Policy erstellt oder in die Konfiguration importiert hat. Klicken Sie auf den Link mit Datum und Uhrzeit, um die Änderungshistorie dieser Ressource anzuzeigen. Weitere Informationen finden Sie unter Seite "Änderungshistorie anzeigen".
    Referenzen
    Die Anzahl der Objekte, die diese WS-Policy referenziert. Wenn solche Referenzen vorhanden sind, klicken Sie auf den Link mit der Anzahl, um eine Liste mit allen Objekten anzuzeigen. Weitere Informationen finden Sie unter Referenzen auf Ressourcen anzeigen.
    Referenziert von
    Die Anzahl der Objekte, die diese WS-Policy referenzieren. Wenn solche Referenzen vorhanden sind, klicken Sie auf den Link mit der Anzahl, um eine Liste mit allen Objekten anzuzeigen. Weitere Informationen finden Sie unter Referenzen auf Ressourcen anzeigen.
    Beschreibung
    Eine Beschreibung der WS-Policy, sofern vorhanden.
    Policy
    Der Text dieser WS-Policy.

  4. Wenn Sie die Felder ändern möchten, klicken Sie auf Bearbeiten. Eine Beschreibung der Felder finden Sie unter Benutzerdefinierte WS-Policys hinzufügen.

    5. Sie können das Feld Ressourcenname nicht ändern.

  5. Klicken Sie auf Speichern, um die Aktualisierungen in der aktuellen Session zu speichern.
  6. Um die Session zu beenden und die Konfiguration für die Laufzeit bereitzustellen, klicken Sie unter Change Center auf Aktivieren.
