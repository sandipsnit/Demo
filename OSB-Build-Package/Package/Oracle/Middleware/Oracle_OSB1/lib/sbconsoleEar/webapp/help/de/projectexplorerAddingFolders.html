Ordner hinzufügen

  1. Wenn dies noch nicht geschehen ist, klicken Sie zum Erstellen einer neuen Session auf Erstellen oder zum Aufrufen einer vorhandenen Session auf Bearbeiten. Weitere Informationen finden Sie unter Change Center verwenden.
  2. Wählen Sie zunächst Project Explorer und dann ein Projekt oder einen Ordner, dem Sie den neuen Ordner hinzufügen möchten.
  3. Geben Sie auf der Seite Projekt-/Ordneransicht im Feld Neuen Ordnernamen eingeben einen eindeutigen Namen für den Ordner ein. Siehe Projekte und Ordner benennen.
  4. Klicken Sie auf Ordner hinzufügen. Der neue Ordner wird in der aktuellen Session gespeichert.
  5. Um die Session zu beenden und die Konfiguration für die Laufzeit bereitzustellen, klicken Sie unter Change Center auf Aktivieren.
