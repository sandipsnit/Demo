Projekte clonen

Es gibt drei Möglichkeiten, um ein Projekt zu clonen (kopieren).

  • Clonen Sie das Projekt mit einem neuen Namen auf Root-Ebene. In diesem Fall sind das Originalprojekt und sein Clone als Peer-Projekte vorhanden. Das geclonte Projekt enthält die gleichen Ordner und Ressourcen wie das Original (wenigstens so lange, bis Sie Quelle oder Clone verändern). Anweisungen finden Sie unter Clonen eines Projekts auf Root-Ebene.
  • Clonen Sie das Projekt, und verwenden Sie den Namen eines vorhandenen Projekts (ein anderer als das Quellprojekt). In diesem Fall werden die Ordner und Ressourcen vom Quellprojekt mit den Ordnern und den Ressourcen des Projekts, dessen Name dem Clone zugewiesen wurde, vermischt. Das Originalprojekt bleibt unverändert. Anweisungen finden Sie unter Clonen eines Projekts auf Root-Ebene.
  • Clonen Sie das Projekt an einem neuen Speicherort, in einem vorhanden Projekt oder Projektordner. In diesem Fall wird das geclonte Projekt an einen Ordner am neuen Speicherort zurückgestuft. Es enthält die gleichen Ordner und Ressourcen wie das Originalprojekt.

    • Wenn Sie ein Projekt mit dem gleichen Namen und am gleichen Speicherort wie ein vorhandener Ordner clonen, wird der Inhalt (Unterordner und Ressourcen) des Projekts mit dem Inhalt des Zielordners gemischt.

    In jedem Fall bleibt das Originalprojekt unverändert.

    Anweisungen finden Sie unter Ein Projekt clonen und den Clone in einen Ordner zurückstufen.

Projekt auf Root-Ebene clonen
  1. Wenn dies noch nicht geschehen ist, klicken Sie zum Erstellen einer neuen Session auf Erstellen oder zum Aufrufen einer vorhandenen Session auf Bearbeiten. Weitere Informationen finden Sie unter Change Center verwenden.
  2. Wählen Sie Project Explorer > Projekte, um die Seite Projektansicht anzuzeigen.
  3. Klicken Sie auf das Symbol Projekt clonen in der Zeile des Projekts, das Sie clonen möchten. Die Seite Projekt clonen wird mit dem aktuellen Speicherort des markierten Projekts angezeigt.
  4. Geben Sie im Feld Neuer Name einen Name für das Clone ein. Siehe Projekte und Ordner benennen.

    5. Verwenden Sie einen eindeutigen Namen, um das Clone als neues Projekt zu erstellen, oder verwenden Sie den Namen eines vorhandenen Projekts, um den Clone mit diesem vorhandenen Projekt zu mischen.

  5. Behalten Sie im Fenster Speicherort den Standard-Root-Speicherort bei, wenn Sie ein Projekt auf der Root-Ebene clonen.
  6. Klicken Sie auf Projekt clonen. Das Projekt wird in der aktuellen Session geclont.
  7. Um die Session zu beenden und die Konfiguration für die Laufzeit bereitzustellen, klicken Sie unter Change Center auf Aktivieren.
Ein Projekt clonen und das Clone in einen Ordner zurückstufen
  1. Wenn dies noch nicht geschehen ist, klicken Sie zum Erstellen einer neuen Session auf Erstellen oder zum Aufrufen einer vorhandenen Session auf Bearbeiten. Weitere Informationen finden Sie unter Change Center verwenden.
  2. Wählen Sie Project Explorer > Projekte, um die Seite Projektansicht anzuzeigen.
  3. Klicken Sie auf das Symbol Projekt clonen in der Zeile des Projekts, das Sie clonen möchten. Die Seite Projekt clonen wird mit dem aktuellen Speicherort des markierten Projekts angezeigt.
  4. Im Feld Neuer Name geben Sie optional einen neuen Namen für den Ordner ein. Siehe Projekte und Ordner benennen.

    5. Verwenden Sie einen eindeutigen Namen, um einen neuen Ordner zu erstellen, oder verwenden Sie den Namen eines vorhandenen Ordners, um den Clone mit dem vorhandenen Ordner zu mischen (an dem Speicherort, den Sie im nächsten Schritt wählen werden.

  5. Wählen Sie im Fenster Speicherort ein Projekt oder einen Ordner, klicken Sie dann auf Projekt clonen. Das geclonte Projekt wird am neuen Speicherort in einen Ordner konvertiert und in der aktuellen Session gespeichert.
  6. Um die Session zu beenden und die Konfiguration für die Laufzeit bereitzustellen, klicken Sie unter Change Center auf Aktivieren.
