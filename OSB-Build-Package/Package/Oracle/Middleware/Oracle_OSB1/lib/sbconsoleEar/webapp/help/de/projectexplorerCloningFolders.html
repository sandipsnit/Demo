Ordner clonen

  1. Wenn dies noch nicht geschehen ist, klicken Sie zum Erstellen einer neuen Session auf Erstellen oder zum Aufrufen einer vorhandenen Session auf Bearbeiten. Weitere Informationen finden Sie unter Change Center verwenden.
  2. Wählen Sie Project Explorer, navigieren Sie zum übergeordneten Projekt oder Ordner, das bzw. der den zu clonenden Ordner enthält. Die Seite "Projekt-/Ordneransicht" wird angezeigt.
  3. Klicken Sie in der Zeile mit dem zu clonenden Ordner auf das Symbol Ordner clonen. Die Seite Ordner clonen wird angezeigt, und der aktuelle Speicherort des Ordners ist markiert.
  4. Sie können den Ordner im selben Speicherort oder in einem anderen Speicherort clonen oder den Ordner in ein Projekt aktualisieren. Gehen Sie dazu wie folgt vor:
    • Wenn Sie den Ordner im selben Speicherort clonen möchten, geben Sie einen neuen Namen in das Feld Neuer Name ein. Weitere Informationen finden Sie unter Projekte und Ordner benennen.

      • Beim Clonen im selben Speicherort wählen Sie im Bereich Speicherort entweder gar keinen Speicherort (und übernehmen damit den aktuellen Speicherort), oder Sie wählen explizit das aktuelle Projekt und den aktuellen Ordner.

    • Wenn Sie einen Ordner in einem anderen Speicherort clonen möchten, können Sie optional einen neuen Namen in das Feld Neuer Name eingeben und wählen im Bereich Speicherort ein Projekt und/oder einen Ordner. Weitere Informationen finden Sie unter Projekte und Ordner benennen.

      • Wenn Sie den Namen eines vorhandenen Ordners verwenden, wird der Inhalt des vorhandenen Ordners mit dem Inhalt des Zielordners zusammengeführt.

    • Wenn Sie den Ordner in ein Projekt aktualisieren möchten, wählen Sie das Root-Verzeichnis aus.
  5. Klicken Sie auf Ordner clonen. Der Ordner wird in der aktuellen Session geclont.

    6. Beim Clonen eines Ordners werden alle Ordner und Ressourcen innerhalb dieses Ordners rekursiv geclont.

  6. Um die Session zu beenden und die Konfiguration für die Laufzeit bereitzustellen, klicken Sie unter Change Center auf Aktivieren.
