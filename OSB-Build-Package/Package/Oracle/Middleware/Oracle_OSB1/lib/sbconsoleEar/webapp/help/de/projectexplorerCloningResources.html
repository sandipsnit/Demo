Ressourcen clonen

  1. Wenn dies noch nicht geschehen ist, klicken Sie zum Erstellen einer neuen Session auf Erstellen oder zum Aufrufen einer vorhandenen Session auf Bearbeiten. Weitere Informationen finden Sie unter Change Center verwenden.
  2. Wählen Sie Project Explorer und dann das Projekt oder den Ordner, mit der zu clonenden Ressource. Die Seite "Projekt-/Ordneransicht" wird angezeigt.
  3. Klicken Sie in der Zeile mit der zu clonenden Ressource auf das Symbol Ressource clonen. Die Seite Ressource clonen wird angezeigt, und der aktuelle Speicherort der Ressource ist markiert.
  4. Sie können die Ressource im selben Speicherort oder in einem anderen Speicherort clonen:
    • Wenn Sie die Ressource im selben Speicherort clonen möchten, geben Sie im Feld Neuer Ressourcenname einen neuen Namen ein. Siehe Benennungsbeschränkungen für Ressourcen.

      • Beim Clonen im selben Speicherort wählen Sie im Bereich Speicherort entweder gar keinen Speicherort (und übernehmen damit den aktuellen Speicherort), oder Sie wählen explizit das aktuelle Projekt und den aktuellen Ordner.

    • Wenn Sie die Ressource in einem anderen Speicherort clonen möchten, können Sie optional einen neuen Namen im Feld Neuer Ressourcenname eingeben und wählen im Bereich Speicherort ein Projekt und/oder einen Ordner aus.
  5. Klicken Sie auf Ressource clonen. Die Ressource wird in der aktuellen Sitzung geclont.

    6. Wenn Sie einen Business-Service clonen, der aus der UDDI-Registry importiert wurde, wird die Zuordnung des geclonten Service zur Registry aufgehoben.

  6. Um die Session zu beenden und die Konfiguration für die Laufzeit bereitzustellen, klicken Sie unter Change Center auf Aktivieren.
