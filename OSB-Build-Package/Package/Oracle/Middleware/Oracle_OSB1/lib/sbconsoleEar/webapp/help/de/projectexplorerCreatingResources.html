Ressourcen erstellen

  1. Wenn dies noch nicht geschehen ist, klicken Sie zum Erstellen einer neuen Session auf Erstellen oder zum Aufrufen einer vorhandenen Session auf Bearbeiten. Weitere Informationen finden Sie unter Change Center verwenden.
  2. Wählen Sie Project Explorer, und wählen Sie das Projekt oder den Ordner, in dem Sie die Ressource erstellen möchten.
  3. Wählen Sie auf der Seite "Projekt-/Ordneransicht" in der Dropdown-Liste Ressource erstellen den Typ der zu erstellenden Ressource aus. Je nach dem gewählten Ressourcentyp werden unterschiedliche Seiten angezeigt.
  4. Befolgen Sie die Anweisungen zum Hinzufügen von Daten zu den einzelnen Seiten, um die Ressource zu erstellen. Weitere Informationen zum Erstellen der einzelnen Ressourcentypen finden Sie im entsprechenden Thema (siehe Liste in Tabelle 3-3).

  5. Wenn Sie die Ressource erstellen, wird sie in der aktuellen Session erstellt.
  6. Um die Session zu beenden und die Konfiguration für die Laufzeit bereitzustellen, klicken Sie unter Change Center auf Aktivieren.
