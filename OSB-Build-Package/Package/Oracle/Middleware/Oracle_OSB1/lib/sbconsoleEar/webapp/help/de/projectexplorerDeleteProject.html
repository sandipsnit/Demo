Projekte löschen

  1. Wenn dies noch nicht geschehen ist, klicken Sie zum Erstellen einer neuen Session auf Erstellen oder zum Aufrufen einer vorhandenen Session auf Bearbeiten. Weitere Informationen finden Sie unter Change Center verwenden.
  2. Wählen Sie Project Explorer > Projekte, um die Seite Projektansicht anzuzeigen.
  3. Klicken Sie auf das Symbol Projekt löschen in der Zeile des Projekts, das Sie löschen möchten. Das Projekt wird in der aktuellen Session gelöscht.

    4. Wenn Sie ein Projekt löschen, werden auch alle Ressourcen des Projekts gelöscht. Falls Ressourcen dieses Projektes von Ressourcen eines anderen Projektes referenziert werden, können Sie das Projekt nur mit einer Warnbestätigung löschen. Dies kann zu Konflikten aufgrund von ungelösten Referenzen auf die gelöschte Ressource führen.

    Bei diesen Ressourcen wird anstelle des Löschen-Symbols das Symbol für eine Warnung wegen Löschvorgang angezeigt.

  4. Um die Session zu beenden und die Konfiguration für die Laufzeit bereitzustellen, klicken Sie unter Change Center auf Aktivieren.
