WSDL exportieren

Über die Seite Projekt-/Ordneransicht exportieren Sie eine WSDL, die mit einem Proxy oder Business-Service verknüpft ist. Die WSDL wird als JAR-Datei exportiert. Sie können eine WSDL nur außerhalb einer Session exportieren. Siehe Change Center verwenden.

Verwenden Sie die Funktion zum Exportieren einer WSDL, um die WSDL für externe Tools, wie IDE, nutzbar zu machen. Beachten Sie, dass es sich hierbei um eine andere Funktion als "Ressourcen exportieren" des Systemverwaltungsmoduls handelt. Jene verwenden Sie, um Ressourcen zwischen zwei Domains zu verschieben und zwischenzuspeichern.

  1. Klicken Sie im linken Navigationsfenster unter Project Explorer auf ein Projekt oder einen Ordner. Die Seite Projekt-/Ordneransicht wird angezeigt.
  2. Klicken Sie auf das Symbol WSDL exportieren in der Zeile mit dem Proxy-Service, dessen WSDL Sie exportieren möchten. Das Dialogfeld Herunterladen von Dateien wird angezeigt.
  3. Klicken Sie im Dialogfeld Herunterladen von Dateien auf Öffnen, um die JAR-Datei zu öffnen, oder klicken Sie auf Speichern, um die JAR-Datei auf Ihrem Desktop zu speichern.
