WSDL generieren

Eine generierte WSDL ist eine WSDL-Ressource, die Oracle Service Bus für einen Service generiert, der nicht mit einer WSDL-Ressource begonnen hat, aber mit einer WSDL beschrieben werden kann. Offlinesetzen ist eine WSDL die von einem EJB-basierten Service erstellt wird, eine generierte WSDL. Weitere Informationen dazu finden Sie unter "Wie WSDL in Oracle Service Bus verwendet wird" im Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus unter http://www.oracle.com/pls/as111170/lookup?id=OSBAG114.

Verwenden Sie die Seite Projekt-/Ordneransicht , um eine WSDL, die mit einem EJB- oder Flow-Transport-typisierten Business-Service verknüpft ist, zu generieren.

  1. Klicken Sie im linken Navigationsfenster unter Project Explorer auf ein Projekt oder einen Ordner. Die Seite Projekt-/Ordneransicht wird angezeigt.
  2. Klicken Sie auf das Symbol WSDL generieren in der Zeile des Business-Service, dessen WSDL Sie generieren möchten. Das Dialogfeld WSDL-Ressource generieren wird angezeigt, wobei der aktuelle Speicherort (Projektname und der Name des Ordners, in dem der Business-Service gespeichert ist) markiert ist.
  3. Geben Sie im Dialogfeld WSDL-Ressource generieren einen Neuen WSDL-Ressourcennamen und ein Verzeichnis für die WSDL ein. Klicken Sie dann auf WSDL generieren.

    Befolgen Sie die Benennungseinschränkungen für Ressourcen bei der Namensgebung.

Wenn Sie fertig sind

Fahren Sie unter WSDLs bearbeiten fort, die generierte WSDL anzuzeigen oder zu ändern.
