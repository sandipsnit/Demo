Seite "Projektansicht"

Die Seite Projektansicht listet die in der aktuellen Domain definierten Projekte auf. Tabelle 3-1 beschreibt die Optionen und die Steuerelemente auf der Seite.

Table 3-1 Seite "Projektansicht" - Optionen und Steuerelemente
Option oder Steuerelement
Beschreibung
Neuen Projektnamen eingeben
Wenn Sie ein neues Projekt erstellen möchten, geben Sie im Textfeld einen Namen ein und klicken auf Projekt hinzufügen. Siehe Projekte hinzufügen.
Siehe auch Projekte und Ordner benennen.
Name
Klicken Sie auf einen Projektnamen, um die Seite Projekt-/Ordneransicht für das Projekt anzuzeigen.
Optionen
Sie haben für jedes aufgelistete Projekt folgende Möglichkeiten:
  • Klicken Sie auf das Symbol Projekt umbenennen, um das Projekt umzubenennen. Siehe Projekte umbenennen.
  • Klicken Sie auf das Symbol Projekt verschieben, um das Projekt zu verschieben. Siehe Projekte verschieben (Konvertierung in einen Ordner).
  • Klicken Sie auf das Symbol Projekte clonen, um das Projekt zu clonen. Siehe Projekte clonen.
  • Klicken Sie auf das Symbol Projekt löschen, um das Projekt zu löschen. Eine Löschwarnung wird angezeigt, wenn andere Ressourcen auf diese Ressource verweisen. Sie können die Ressource mit einer Warnbestätigung löschen. Dies kann zu Konflikten aufgrund ungelöster Referenzen auf die gelöschte Ressource führen. Siehe Projekte löschen.
Wenn Sie eine dieser Aktionen durchführen, wird das Ergebnis in der aktuellen Session gespeichert. Zum Beenden der Session und Bereitstellen der Konfiguration zur Laufzeit klicken Sie auf Aktivieren unter Change Center.
