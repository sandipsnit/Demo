Seite "Ressourcen Laden – Ressourcen aus URL laden"

Auf der Seite Ressourcen Laden – Ressourcen aus URL laden können Sie Ressourcen aus einer Datei im Web importieren. Weitere Informationen finden Sie unter Ressourcen aus einem URL laden.

In Tabelle 3-5 sind die Optionen und Steuerelemente auf der Seite beschrieben.

Tabelle 3-5 Ressourcen laden – Ressourcen aus URL laden: Optionen und Steuerelemente 
Optionen und Steuerelemente
Beschreibung
URL/Pfad
Geben Sie den URL der Datei ein.
Um eine lokale Ressource anzugeben, können Sie das Datei-Protokoll mit einem Dateinamen verwenden, der vollständig durch den Pfad angegeben ist. Beispiel: file:///c:/osbresources/ForeachAction.jar
Ressourcenname
Geben Sie einen Namen für die Ressource ein.
Ressourcentyp
Wählen Sie den Ressourcentyp aus der Liste aus.
