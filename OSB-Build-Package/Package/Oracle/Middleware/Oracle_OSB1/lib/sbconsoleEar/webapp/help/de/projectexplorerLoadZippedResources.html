Optionen und Steuerelemente
Beschreibung
ZIP-Datei wählen
Geben Sie den vollständigen Namen einer zu importierenden ZIP-Datei an, oder klicken Sie auf Durchsuchen, um die Datei auszuwählen.
Ressourcentyp und Dateierweiterung
Jede Datei in der Zip-Datei ist eine mögliche Ressource, und die Dateierweiterung ist im Allgemeinen der Weg, den Dateityp zu identifizieren. Oracle Service Bus definiert eine Standarderweiterung für jede Ressource, aber wenn Sie eine Ressource mit zusätzlichen Erweiterungen beschreiben, ermöglicht Ihnen diese Zuordnung, diese Erweiterungen für den Loader zu deklarieren.
Nachdem Sie die Erweiterungszuordnung der ZIP-Datei definiert haben, scannt das Oracle Service Bus-Ladeprogramm den Inhalt der ZIP-Datei und versucht, jeder Datei einen Ressourcentyp zuzuweisen. Eine Datei ohne Erweiterung oder eine Datei, für die in der Zuordnung keine Erweiterung definiert ist, wird als unbekannt betrachtet und automatisch vom Ladevorgang ausgeschlossen.
Bei bekannten Dateien wird als Name der Ressource der Name der Datei, jedoch ohne Erweiterung, übernommen. Diese Ordnerstruktur der Zip-Datei wird im Zielordner der Oracle Service Bus Administration-Konsole erneut erstellt.
