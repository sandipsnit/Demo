Ordner verschieben oder upgraden

  1. Wenn dies noch nicht geschehen ist, klicken Sie zum Erstellen einer neuen Session auf Erstellen oder zum Aufrufen einer vorhandenen Session auf Bearbeiten. Weitere Informationen finden Sie unter Change Center verwenden.
  2. Wählen Sie Project Explorer, und wählen Sie das Projekt oder den Ordner, in dem der Ordner enthalten ist, den Sie verschieben oder upgraden möchten. Die Seite "Projekt-/Ordneransicht" wird angezeigt.
  3. Klicken Sie in der Zeile der Ordners, den Sie verschieben oder upgraden möchten, auf das Symbol Ordner verschieben. Die Seite Ordner verschieben wird mit dem aktuellen Speicherort des markierten Ordners angezeigt.
  4. Legen Sie fest, ob Sie den Ordner an einen neuen Speicherort unter einem anderen Projekt verschieben oder den Ordner zu einem Projekt upgraden möchten:
    • Um den Ordner an einen neuen Speicherort zu verschieben, wählen Sie im Bereich Speicherort ein Projekt oder einen Ordner aus, in das oder in den der Ordner verschoben werden soll.
    • Wenn Sie den Ordner in ein Projekt aktualisieren möchten, wählen Sie das Root-Verzeichnis aus.
  5. Klicken Sie auf Weiterleiten. Der Ordner (oder das Projekt, sofern es von einem Ordner upgegradet wurde) wird in der aktuellen Session am neuen Speicherort gespeichert.

    6. Wenn Sie einen Ordner verschieben oder upgraden und an diesem Speicherort bereits ein Ordner mit demselben Namen vorhanden ist, wird der Inhalt des verschobenen oder upgegradeten Ordners mit dem Inhalt des vorhandenen Ordners oder Projekts zusammengeführt.

  6. Um die Session zu beenden und die Konfiguration für die Laufzeit bereitzustellen, klicken Sie unter Change Center auf Aktivieren.
