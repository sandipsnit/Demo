Ressourcen umbenennen

  1. Wenn dies noch nicht geschehen ist, klicken Sie zum Erstellen einer neuen Session auf Erstellen oder zum Aufrufen einer vorhandenen Session auf Bearbeiten. Weitere Informationen finden Sie unter Change Center verwenden.
  2. Wählen Sie Project Explorer und dann das Projekt oder den Ordner, in dem sich die Ressource für die Umbenennung befindet. Die Seite Projektansicht/Ordneransicht wird angezeigt.
  3. Klicken Sie in der Zeile der Ressource, die umbenannt werden soll, auf das Symbol Umbenennen. Daraufhin wird die Seite Umbenennen angezeigt.
  4. Geben Sie im Feld Neuer Ressourcenname einen neuen Namen für die Ressource ein. Siehe Benennungseinschränkungen für Ressourcen.
  5. Klicken Sie auf Umbenennen.
    6. Hinweis: Wenn Sie einen Business-Service umbenennen, der aus der UDDI-Registry importiert wurde, wird der Service von dieser Registry getrennt
  6. Um die Session zu beenden und die Konfiguration für die Laufzeit bereitzustellen, klicken Sie unter Change Center auf Aktivieren.
