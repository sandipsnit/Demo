Seite "Ressourcen laden - Geladene Ressourcen prüfen"

Auf der Seite Ressourcen laden - Geladene Ressourcen prüfen werden Details zu Ressourcen angezeigt, die aus einer ZIP- oder einer anderen Datei importiert wurden, wie es unter Ressourcen aus einer ZIP-Datei laden oder Ressourcen aus URL laden beschrieben ist.

Tabelle 3-6 beschreibt die Optionen und Steuerelemente auf der Seite.

Tabelle 3-6 Seite "Ressourcen laden - Geladene Ressourcen prüfen - Optionen und Steuerelemente" 
Optionen und Steuerelemente
Beschreibung
Dateiname
Dateiname der Ressource, einschließlich Dateierweiterung.
Name
Name der Ressource.
Pfad
Pfad der Ressource, der Projekt und Ordner ist, in dem die Ressource abgelegt ist.
Ressourcentyp
Listet die unterstützten Ressourcentypen auf.

Ein Warn-Symbol neben dem Dateinamen gibt an, dass der Ressourcentyp unbekannt ist. (Eine Datei ohne Dateierweiterung oder eine Datei ohne eine in der Zuordnung definierte Dateierweiterung gilt als unbekannt.) Dateien unbekanntes Typs können nicht importiert werden.
