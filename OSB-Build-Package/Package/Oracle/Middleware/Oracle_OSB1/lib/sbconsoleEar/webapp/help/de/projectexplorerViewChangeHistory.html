Seite "Änderungshistorie anzeigen"

Auf der Seite Änderungshistorie anzeigen werden die 10 zuletzt an einer Ressource durchgeführten Änderungen für alle Sessionaktivierungen angezeigt.

In Tabelle 3-9 werden die Optionen und Steuerelemente auf der Seite beschrieben.

Tabelle 3-9 Seite "Änderungshistorie anzeigen" – Optionen und Steuerelemente
Option oder Steuerelement
Beschreibung
Task
Der Typ der ausgeführten Task:
  • Erstellen
  • Aktualisieren
  • Clonen
  • Umbenennen
  • Verschieben
Benutzer
Der Benutzer, der diese Ressource erstellt oder bearbeitet hat.
Ausführungszeit
Das Datum und die Uhrzeit, zu der der Benutzer diese Ressource erstellt oder bearbeitet hat.
