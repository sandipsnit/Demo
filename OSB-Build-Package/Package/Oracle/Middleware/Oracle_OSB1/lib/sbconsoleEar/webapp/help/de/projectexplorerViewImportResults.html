Seite "Ressourcen laden - Ergebnis importieren"

Auf der Seite Ressourcen laden - Ergebnis importieren werden die Ergebnisse des Ladens von Ressourcen aus einer importierten Zip-Datei oder aus einem URL angezeigt.

Je nachdem, ob Sie Ressourcen aus einer Zip-Datei oder aus einem URL geladen haben, müssen Sie die Schritte im Abschnitt Ressourcen aus einer Zip-Datei laden oder im Abschnitt Ressourcen aus einem URL laden ausführen. Stellen Sie sicher, dass Sie die Schritte im Abschnitt Seite "Ressourcen laden - Geladene Ressourcen prüfen ausgeführt haben.

Die Seite Ergebnis importieren wird angezeigt, die folgende Informationen enthält:

Tabelle 3-7 Seite "Projekt-/Ordneransicht – Optionen und Steuerelemente
Option oder Steuerelement
Beschreibung
Status
Der Status der Ressource, zum Beispiel, ob sie erfolgreich importiert wurde oder nicht.
Eine WSDL kann vorübergehend als ungültig markiert werden, bis alle Abhängigkeiten importiert wurden. Nach Abschluss der Importaktion werden die als ungültig markierten Dateien allerdings vom Konfigurations-Service erneut geprüft.
Name
Name der Ressource.
Pfad
Pfad der Ressource, der Projekt und Ordner ist, in dem die Ressource abgelegt ist.
Ressourcentyp
Zeigt den importierten Ressourcentyp an.
Fehlermeldung
Zeigt die Details der Fehlermeldung an, wenn die Ressource nicht erfolgreich importiert wurde.
