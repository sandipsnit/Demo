Name
Klicken Sie auf einen Ordnernamen, um die Seite Projekt-/Ordnername für diesen Ordner anzuzeigen.
Optionen
Bei allen angezeigten Ordnern können Sie folgende Aufgaben ausführen:
  • Klicken Sie auf das Symbol Ordner umbenennen, um den Ordner umzubenennen. Weitere Informationen finden Sie unter Ordner umbenennen.
  • Klicken Sie auf das Symbol Upgrade zu Projekt oder Verschieben von Ordner, um den Ordner zu verschieben oder in ein Projekt zu aktualisieren. Weitere Informationen finden Sie unter Ordner verschieben oder upgraden.
  • Klicken Sie auf das Symbol Ordner clonen, um den Ordner zu clonen. Weitere Informationen finden Sie unter Ordner clonen.
  • Klicken Sie auf das Symbol Ordner löschen, um einen Ordner zu löschen. Wenn andere Ressourcen diese Ressource referenzieren, wird das Symbol Warnung wegen Löschvorgang angezeigt. Sie können die Ressource löschen, indem Sie den Löschvorgang bestätigen. Dies kann aufgrund von nicht aufgelösten Referenzen mit der gelöschten Ressource zu Konflikten führen. Weitere Informationen finden Sie unter Ordner löschen.
Wenn Sie eine dieser Aktionen durchführen, wird das Ergebnis in der aktuellen Session gespeichert. Zum Beenden der Session und Bereitstellen der Konfiguration zur Laufzeit klicken Sie auf Aktivieren unter Change Center.
Löschen
Wenn Sie Ordner löschen möchten, aktivieren Sie die entsprechenden Kontrollkästchen neben den Namen der zu löschenden Ordner, und klicken Sie auf Löschen.

Ressource erstellen
Wenn Sie eine neue Ressource für dieses Projekt erstellen möchten, wählen Sie aus der Liste Ressource erstellen einen Ressourcentyp. Weitere Informationen finden Sie unter Ressourcen erstellen.
Name
Klicken Sie auf einen Ressourcennamen, um Details der Ressource anzuzeigen.
Ressourcentyp
Der Typ dieser Ressource.
Aktionen
Die für eine Ressource verfügbaren Aktionen hängen vom Typ der Ressource ab:
  • Das Symbol Nachrichtenfluss bearbeiten wird für Proxy-Services angezeigt. Klicken Sie auf dieses Symbol, um die Nachricht eines Proxy-Service zu bearbeiten. Weitere Informationen finden Sie unter Nachrichtenfluss erstellen, bearbeiten und anzeigen.
  • Das Symbol Testkonsole starten wird für Business-Services, Proxy-Services, XQuery-Transformationen, XSL-Transformationen und MFL-Dateien angezeigt. Klicken Sie auf dieses Symbol, um die Testkonsole aufzurufen, mit der Sie das Design von Services und Transformationen validieren und testen. Bei Services können Sie die Testkonsole nur zur Laufzeit verwenden, das heißt, wenn die Session aktiviert ist. Bei Transformationen können Sie die Testkonsole sowohl innerhalb als auch außerhalb einer Session verwenden. Weitere Informationen finden Sie unter Services testen und unter Transformationen testen.
  • Das Symbol WSDL exportieren wird bei WSDL-basierten Proxy-Services angezeigt. Klicken Sie auf dieses Symbol, um WSDL zu exportieren und später in externen Tools wie IDE anzeigen und bearbeiten zu können.

    • Hinweis: Dieser Vorgang unterscheidet sich von der Funktion "Ressourcen exportieren" in der Systemverwaltung, die Sie verwenden, um Ressourcen zwischen zwei Domains zu verschieben und bereitzustellen. Weitere Informationen finden Sie unter WSDL exportieren.

  • Das Symbol WSDL generieren wird für Transport-Business-Services wie EJB und Flow angezeigt. Klicken Sie auf das Symbol, um WSDL zu generieren und später anzuzeigen und zu ändern. Weitere Informationen finden Sie unter WSDL generieren.
Optionen
In der Spalte Optionen werden die folgenden Symbole angezeigt:
  • Klicken Sie auf das Symbol Ressource umbenennen, um die Ressource umzubenennen. Weitere Informationen finden Sie unter Ressourcen umbenennen.
  • Klicken Sie auf das Symbol Ressource verschieben, um die Ressource zu verschieben. Weitere Informationen finden Sie unter Ressourcen verschieben.
  • Klicken Sie auf das Symbol Ressource clonen, um die Ressource zu clonen. Weitere Informationen finden Sie unter Ressourcen clonen.
  • Klicken Sie auf das Symbol Löschen, um die Ressource zu löschen. Wenn andere Ressourcen diese Ressource referenzieren, wird das Symbol Warnung wegen Löschvorgang angezeigt. Sie können die Ressource löschen, indem Sie den Löschvorgang bestätigen. Dies kann aufgrund von nicht aufgelösten Referenzen mit der gelöschten Ressource zu Konflikten führen.
Wenn Sie eine dieser Aktionen durchführen, wird das Ergebnis in der aktuellen Session gespeichert. Zum Beenden der Session und Bereitstellen der Konfiguration zur Laufzeit klicken Sie auf Aktivieren unter Change Center.
