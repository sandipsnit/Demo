Service-Schlüsselprovider suchen

  1. Führen Sie eine der folgenden Aktionen aus:
    • Wählen Sie Project Explorer, um die Seite "Projektansicht" oder die Seite "Projekt-/Ordneransicht" anzuzeigen. Durchsuchen Sie dann die Projekte und Ordner nach dem Service-Schlüsselprovider.
    • Wählen Sie Ressourcenbrowser > Service-Schlüsselprovider. Auf der Seite Übersicht der Service-Schlüsselprovider werden die in Tabelle 17-1 beschriebenen Informationen angezeigt.
  2. Wenn Sie nach einem Service-Schlüsselprovider suchen möchten, geben Sie einen Teil des Providernamens oder den gesamten Namen im Feld Name ein. Sie können auch einen Teil des Provider-Projektnamens oder den gesamten Projektamen und den Ordner im Feld Pfad eingeben. Klicken Sie anschließend auf Suchen.

    3. Klicken Sie auf Alle anzeigen, um die Suchfilter zu deaktivieren und alle Service-Schlüsselprovider anzuzeigen.

    Tabelle 17-1 Informationen zum Service-Schlüsselprovider 
    Eigenschaft
    Beschreibung
    Name
    Der eindeutige Name für den Service-Schlüsselprovider. Klicken Sie auf den Namen, um die Seite Details des Service-Schlüsselproviders anzeigen anzuzeigen. Weitere Informationen finden Sie unter Service-Schlüsselprovider bearbeiten.
    Pfad
    Der Projektname und der Name des Ordners, in dem der Service-Schlüsselprovider gespeichert ist. Klicken Sie auf den Namen, um das Projekt oder den Ordner anzuzeigen, das bzw. der diese Ressource enthält. Weitere Informationen finden Sie unter Ressourcennamen mit Projekten und Ordner angeben.
