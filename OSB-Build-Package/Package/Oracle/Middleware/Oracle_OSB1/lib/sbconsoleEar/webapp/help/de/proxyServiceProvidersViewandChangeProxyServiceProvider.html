Service-Schlüsselprovider bearbeiten

Zeigen Sie über die Seite Details des Service-Schlüsselproviders anzeigen die Details eines bestimmten Service-Schlüsselproviders an, und ändern Sie diese gegebenenfalls.

  1. Wenn dies noch nicht geschehen ist, klicken Sie zum Erstellen einer neuen Session auf Erstellen oder zum Aufrufen einer vorhandenen Session auf Bearbeiten. Weitere Informationen finden Sie unter Change Center verwenden.
  2. Ermitteln Sie den Service-Schlüsselprovider, wie in Service-Schlüsselprovider suchen beschrieben.
  3. Klicken Sie auf den Namen des Service-Schlüsselproviders. Auf der Seite Details des Service-Schlüsselproviders anzeigen werden die Informationen der Tabelle 17-3 angezeigt.

    4. Tabelle 17-3 Service-Schlüsselproviderdetails 
    Eigenschaft
    Beschreibung
    Name des Service-Schlüsselproviders
    Name dieses Service-Schlüsselproviders.
    Zuletzt geändert von
    Benutzer, der diesen Service-Schlüsselprovider erstellt oder in die Konfiguration importiert hat.
    Zuletzt geändert am
    Datum und Uhrzeit, als der Benutzer diesen Service-Schlüsselprovider erstellt oder in die Konfiguration importiert hat. Klicken Sie auf den Link für Datum und Uhrzeit, um die Änderungshistorie zu dieser Ressource anzuzeigen. Siehe die Seite Änderungshistorie anzeigen.
    Referenzen
    Anzahl Objekte, die dieser Service-Schlüsselprovider referenziert. Falls Referenzen vorhanden sind, klicken Sie auf den Zahlenlink, um eine Liste der Objekte anzuzeigen. Siehe Referenzen auf Ressourcen anzeigen.
    Referenziert von
    Anzahl Objekte, die diesen Service-Schlüsselprovider referenzieren. Falls Referenzen vorhanden sind, klicken Sie auf den Zahlenlink, um eine Liste der Objekte anzuzeigen. Beispiel: Wenn Sie diesen Service-Schlüsselprovider als den Service-Schlüsselprovider für einen bestimmten Proxy-Service gewählt haben, wird der Proxy-Service als Referenz aufgelistet, sobald Sie auf den Link klicken. Siehe Referenzen auf Ressourcen anzeigen.
    Beschreibung
    Beschreibung dieses Service-Schlüsselproviders, falls vorhanden.

  4. Wenn Sie die Felder ändern möchten, klicken Sie auf Bearbeiten. Eine detaillierte Beschreibung der Felder finden Sie unter Service-Schlüsselprovider hinzufügen.
  5. Klicken Sie auf Speichern, um die Aktualisierungen in der aktuellen Session zu speichern.
  6. Um die Session zu beenden und die Konfiguration für die Laufzeit bereitzustellen, klicken Sie unter Change Center auf Aktivieren.
