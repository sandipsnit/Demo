Alert-Aktionen hinzufügen

Mit Alert-Aktionen generieren Sie nachrichtenkontextbasierte Alerts in einer Pipeline, die an ein Alert-Ziel gesendet werden sollen. Anders als SLA-Alerts sind von Alert-Aktionen generierte Benachrichtigungen in erster Linie für Geschäftszwecke oder zur Fehlermeldung gedacht, nicht aber für das Überwachen der Systemintegrität. Alert-Ziele müssen vor diesem Hintergrund konfiguriert und gewählt werden. Weitere Informationen finden Sie unter Alert-Ziele.

Wenn "Pipeline Alerting" nicht für den Service oder auf Domainebene aktiviert ist, wird die konfigurierte Alert-Aktion während der Nachrichtenverarbeitung umgangen.

Bevor Sie beginnen

In diesen Anweisungen wird davon ausgegangen, dass Sie bereits einen Nachrichtenfluss auf der Seite Nachrichtenfluss bearbeiten bearbeiten, wie in Nachrichtenfluss erstellen, bearbeiten und anzeigen erläutert.

So fügen Sie eine Alert-Aktion hinzu

  1. Navigieren Sie an die Stelle, an der Sie die Aktion hinzufügen möchten. Eine Beschreibung hierzu finden Sie unter Aktionen im Nachrichtenfluss hinzufügen und bearbeiten.
  2. Klicken Sie auf das entsprechende Symbol, und wählen Sie Aktion hinzufügen > Reporting > Alert.
  3. Klicken Sie auf <Ziel>. Die Seite Alert-Ziel wählen wird angezeigt. Wählen Sie das erforderliche Alert-Ziel aus der Liste, und klicken Sie auf Weiterleiten.

    4. Standardmäßig wird der Alert zur Oracle Service Bus Administration-Konsole geleitet.

  4. Klicken Sie auf <Ausdruck>. Die Seite XQuery-Ausdruckseditor wird angezeigt. Geben Sie den Nachrichtenkontext ein, der der Alert-Nachricht über XQuery-Ausdrücke bei Kontextvariablen hinzugefügt werden soll. Siehe Inline XQuery- und XPath-Ausdrücke erstellen und bearbeiten.
  5. Geben Sie im Feld Alert-Übersicht eine kurze Beschreibung des Alerts ein. Diese wird zur Betreffzeile bei einer E-Mail-Benachrichtigung. Sie darf nicht mehr als 80 Zeichen enthalten. Wird keine Beschreibung eingegeben, erscheint eine vordefinierte Betreffzeile mit folgendem Wortlaut "Oracle Service Bus Alert".
  6. Wählen Sie in der Dropdown-Liste Schweregrad einen Schweregrad für den Alert aus. Wählen Sie aus den Optionen: Normal, Warnung, Unbedeutend, Gravierend, Kritisch und Schwerwiegend.

Wenn Sie fertig sind

Wenn Sie die Konfiguration dieser Aktion abgeschlossen haben, konfigurieren Sie weitere Aktionen, oder speichern Sie Ihre Konfiguration, wie unter Aktionen in Nachrichtenflüssen hinzufügen und bearbeiten beschrieben.
