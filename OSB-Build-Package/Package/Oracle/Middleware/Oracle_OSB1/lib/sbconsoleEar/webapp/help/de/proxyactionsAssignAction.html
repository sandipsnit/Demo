Zuweisungsaktionen hinzufügen

Verwenden Sie die Zuweisungsaktion, um das Ergebnis eines XQuery-Ausdrucks einer Kontextvariablen zuzuweisen.

Bevor Sie beginnen

In diesen Anweisungen wird davon ausgegangen, dass Sie bereits einen Nachrichtenfluss auf der Seite Nachrichtenfluss bearbeiten bearbeiten, wie in Nachrichtenfluss erstellen, bearbeiten und anzeigen erläutert.

So fügen Sie eine Zuweisungsaktion hinzu

  1. Navigieren Sie an die Stelle, an der Sie die Aktion hinzufügen möchten. Eine Beschreibung hierzu finden Sie unter Aktionen im Nachrichtenfluss hinzufügen und bearbeiten.
  2. Klicken Sie auf das entsprechende Symbol, und wählen Sie dann Aktion hinzufügen > Nachrichtenverarbeitung > Zuweisen.
  3. Klicken Sie auf <Ausdruck>. Die Seite XQuery-Ausdruckseditor wird angezeigt. Der XQuery-Ausdruck wird verwendet, um die Daten zu erstellen, die der genannten Variablen zugewiesen werden. Siehe Inline XQuery- und XPath-Ausdrücke erstellen und bearbeiten.
  4. Wenn Sie mit der Bearbeitung des Ausdrucks fertig sind, geben Sie eine Kontextvariable im Feld "Variable" ein. Weitere Informationen zu Kontextvariablen finden Sie in "Inbound and Outbound Variables" unter http://www.oracle.com/pls/as111170/lookup?id=OSBAG325 sowie "Constructing Messages to Dispatch" unter http://www.oracle.com/pls/as111170/lookup?id=OSBAG348 im Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus.

Wenn Sie fertig sind

Wenn Sie die Konfiguration dieser Aktion abgeschlossen haben, konfigurieren Sie weitere Aktionen, oder speichern Sie Ihre Konfiguration, wie unter Aktionen in Nachrichtenflüssen hinzufügen und bearbeiten beschrieben.
