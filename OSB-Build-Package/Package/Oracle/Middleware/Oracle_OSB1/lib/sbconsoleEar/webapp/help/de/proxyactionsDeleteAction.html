Löschaktionen hinzufügen

Mit der Aktion "Löschen" können Sie eine Kontextvariable oder eine Gruppe von Knoten löschen, die durch einen XPath-Ausdruck festgelegt wird. Die Aktion "Löschen" ist eine von mehreren Update-Aktionen.

Bevor Sie beginnen

In diesen Anweisungen wird davon ausgegangen, dass Sie bereits einen Nachrichtenfluss auf der Seite Nachrichtenfluss bearbeiten bearbeiten, wie in Nachrichtenfluss erstellen, bearbeiten und anzeigen erläutert.

So fügen Sie eine Aktion "Löschen" hinzu:

  1. Navigieren Sie an die Stelle, an der Sie die Aktion hinzufügen möchten. Eine Beschreibung hierzu finden Sie unter Aktionen im Nachrichtenfluss hinzufügen und bearbeiten.
  2. Um eine Kontextvariable zu löschen, aktivieren Sie das Optionsfeld Variable. Geben Sie anschließend im Feld Variable den Namen der Kontextvariablen ein. Weitere Informationen zu Kontextvariablen finden Sie in ###Anhang A, Nachrichtenkontext.

    3. Alternativ können Sie mit einem XPath-Ausdruck alle markierten Knoten löschen. Aktivieren Sie hierzu das Optionsfeld XPath, und klicken Sie auf <XPath>. Die Seite XPath-Ausdruckseditor wird angezeigt. Weitere Informationen finden Sie unter Inline XQuery- und XPath-Ausdrücke erstellen und bearbeiten. Nachdem Sie den Ausdruck gespeichert haben, geben Sie im Feld Variable eine Kontextvariable ein. Weitere Informationen zu Kontextvariablen finden Sie in ###Anhang A, Nachrichtenkontext.

Wenn Sie fertig sind

Wenn Sie die Konfiguration dieser Aktion abgeschlossen haben, konfigurieren Sie weitere Aktionen, oder speichern Sie Ihre Konfiguration, wie unter Aktionen in Nachrichtenflüssen hinzufügen und bearbeiten beschrieben.
