Einfügeaktionen hinzufügen

Verwenden Sie die Einfügeaktion, um das Ergebnis eines XQuery-Ausdrucks an einem bestimmten Ort relativ zu den von einem XPath-Ausdruck gewählten Knoten einzufügen. Die Einfügeaktion ist eine von mehreren Aktualisierungsaktionen.

Bevor Sie beginnen

In diesen Anweisungen wird davon ausgegangen, dass Sie bereits einen Nachrichtenfluss auf der Seite Nachrichtenfluss bearbeiten bearbeiten, wie in Nachrichtenfluss erstellen, bearbeiten und anzeigen erläutert.

So fügen Sie eine Einfügeaktion hinzu:

  1. Navigieren Sie an die Stelle, an der Sie die Aktion hinzufügen möchten. Eine Beschreibung hierzu finden Sie unter Aktionen im Nachrichtenfluss hinzufügen und bearbeiten.
  2. Klicken Sie auf das entsprechende Symbol, danach klicken Sie auf Aktion hinzufügen > Nachrichtenverarbeitung > Einfügen.
  3. Klicken Sie auf <Expression>, um einen XQuery-Ausdruck zu bearbeiten. Mithilfe des XQuery-Ausdrucks werden die Daten erstellt, die an einem bestimmten Speicherort in einer benannten Variable eingefügt werden. Die Seite XQuery-Ausdruckseditor wird angezeigt. Siehe Inline-XQuery- und XPath-Ausdrücke erstellen und bearbeiten.
  4. Wenn Sie den Ausdruck fertig bearbeitet haben, wählen Sie den relativen Speicherort aus der Dropdown-Liste aus. Mit dem relativen Speicherort wird gesteuert, wo Daten relativ zum Ergebnis des XPath-Ausdrucks eingefügt werden:
    • Vor: Als gleichrangiges Objekt vor jedem Element oder Attribut, das vom XPath-Ausdruck gewählt wurde
    • Nach: Als gleichrangiges Objekt nach jedem Element oder Attribut, das vom XPath-Ausdruck gewählt wurde
    • Als erstes untergeordnetes Element von: Als erstes untergeordnetes Element eines jeden Elements, das vom XPath-Ausdruck identifiziert wird. Ein Fehler tritt auf, wenn das Ergebnis von XPath Attribute zurückgibt.
    • Als letztes untergeordnetes Element von: Als letztes untergeordnetes Element eines jeden Elements, das vom XPath-Ausdruck identifiziert wird. Ein Fehler tritt auf, wenn das Ergebnis von XPath Attribute zurückgibt.
  5. Klicken Sie auf XPath. Die Seite XPath-Ausdruckseditor wird angezeigt. Siehe Inline-XQuery- und XPath-Ausdrücke erstellen und bearbeiten.

    6. Gültige Konfigurationen sind auch jene, in denen:

    • sowohl XQuery- als auch XPath-Ausdrücke Elemente zurückgeben
    • sowohl XQuery- als auch XPath-Ausdrücke Attribute zurückgeben. In diesem Fall muss der XQuery-Ausdruck Attribute zurückgeben.
  6. Wenn Sie den XPath-Ausdruck fertig bearbeitet haben, geben Sie eine Kontextvariable in das Variablenfeld ein. XPath prüft den Inhalt dieser Variablen. Weitere Informationen zu Kontextvariablen finden Sie unter ###Anhang A, Nachrichtenkontext.

Wenn Sie fertig sind

Wenn Sie die Konfiguration dieser Aktion abgeschlossen haben, konfigurieren Sie weitere Aktionen, oder speichern Sie Ihre Konfiguration, wie unter Aktionen in Nachrichtenflüssen hinzufügen und bearbeiten beschrieben.
