MFL-Transformationsaktionen hinzufügen

Verwenden Sie die MFL-(Message Format Language-)Transformationsaktion, um den Nachrichteninhalt in der Nachrichten-Pipeline von XML zu Nicht-XML oder umgekehrt zu konvertieren. Eine MFL ist ein spezialisiertes XML-Dokument, das verwendet wird, um das Layout der binären Daten zu beschreiben. Es ist eine Oracle-eigene Sprache, die verwendet wird, um Regeln zu definieren, um formatierte binäre Daten in XML-Daten zu transformieren oder umgekehrt. Siehe MFLs.

Bevor Sie beginnen

In diesen Anweisungen wird davon ausgegangen, dass Sie bereits einen Nachrichtenfluss auf der Seite Nachrichtenfluss bearbeiten bearbeiten, wie in Nachrichtenfluss erstellen, bearbeiten und anzeigen erläutert.

Eine MFL-Transformationsaktion hinzufügen

  1. Navigieren Sie an die Stelle, an der Sie die Aktion hinzufügen möchten. Eine Beschreibung hierzu finden Sie unter Aktionen im Nachrichtenfluss hinzufügen und bearbeiten.
  2. Klicken Sie auf das entsprechende Symbol, und wählen Sie Aktion hinzufügen > Nachrichtenverarbeitung > MF-Transformation
  3. Wählen Sie aus der Dropdown-Liste MFL-Transformation anwenden entweder XML zu Nicht-XML oder Nicht-XML zu XML, je nach Ihren Bedürfnissen.
  4. Klicken Sie auf <Expression>. Verwenden Sie den XQuery-Ausdruckseditor, geben Sie die Variable an, für die die MFL-Transformationsaktion ausgeführt werden soll. Dies muss bei der Transformation zu XML eine Text- oder Binäreingabe sein, und XML bei der Transformation zu Nicht-XML. Binärer Content im Nachrichtenkontext wird repräsentiert durch das binäre Content-XML-Element. Diese XML soll das Ergebnis des XQuery-Ausdrucks sein, wenn die Eingabe binär sein muss. Siehe Inline-XQuery und XPath-Ausdrücke erstellen und bearbeiten.
  5. Wählen Sie eine der folgenden Optionen:
    • MFL-Ressource: Klicken Sie auf den Link <resource>. Die Seite MFL wählen wird angezeigt. Wählen Sie die statische MFL-Ressource, die die MFL-Transformationsaktion ausführen wird.
    • MFL-Ressource von: Klicken Sie auf den Link Ausdruck. Die Seite XQuery-Ausdruckseditor wird angezeigt. Verwenden Sie den XQuery-Ausdruckseditor, um einen XQuery-Ausdruck zu erstellen oder zu bearbeiten, der dynamisch eine MFL-Ressource spezifiziert, die im Format project/folder/MFLresourcename die Transformationsaktion ausführt. Siehe Inline XQuery- und XPath-Ausdrücke erstellen und bearbeiten.
  6. Geben Sie im Feld Zuweisen zu Variable den Namen der Variablen ein, zu der das Ergebnis der Transformationsaktion zugewiesen werden soll. Das Ergebnis wird ein binäres Content-XML-Element sein.

Wenn Sie fertig sind

Wenn Sie die Konfiguration dieser Aktion abgeschlossen haben, konfigurieren Sie weitere Aktionen, oder speichern Sie Ihre Konfiguration, wie unter Aktionen in Nachrichtenflüssen hinzufügen und bearbeiten beschrieben.
