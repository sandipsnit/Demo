Publishing-Aktionen hinzufügen

Mit einer Publishing-Aktion definieren Sie einen statisch angegebenen Ziel-Service für eine Nachricht und konfigurieren, wie die Nachricht gepackt und an den Service gesendet wird.

Weitere Informationen zum Veröffentlichungsverhalten finden Sie unter "Performing Transformations in Message Flows" im Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus auf der Website http://www.oracle.com/pls/as111170/lookup?id=OSBAG205.

  1. Navigieren Sie an die Stelle, an der Sie die Aktion hinzufügen möchten. Eine Beschreibung hierzu finden Sie unter Aktionen im Nachrichtenfluss hinzufügen und bearbeiten.
  2. Wählen Sie Aktion hinzufügen > Kommunikation > Publizieren.
  3. Klicken Sie auf <Service>. Die Seite Service wählen wird angezeigt.
  4. Wählen Sie einen Service aus der Liste, und klicken Sie dann auf Weiterleiten. Dies ist der Ziel-Service für die Nachricht.
  5. Falls der Service definierte Vorgänge enthält, können Sie ein Vorgang angeben, der dadurch aufgerufen wird, dass er aus der Liste Vorgang gewählt wird.
  6. Um einen Ausgangsvorgang in einen Eingangsvorgang zu verwandeln, aktivieren Sie das Kontrollkästchen Eingangsvorgang für Ausgang verwenden.
  7. Für die Konfiguration, wie die Nachricht gepackt und an den Service gesendet wird, klicken Sie im Feld Anforderungsaktionen auf Aktion hinzufügen. Wählen Sie dann eine Aktion, die Sie mit dem Service verknüpfen möchten. Sie können mehr als eine Aktion hinzufügen. Siehe Aktionen in Nachrichtenflüssen hinzufügen und bearbeiten.

Wenn Sie fertig sind

Wenn Sie die Konfiguration dieser Aktion abgeschlossen haben, konfigurieren Sie weitere Aktionen, oder speichern Sie Ihre Konfiguration, wie unter Aktionen in Nachrichtenflüssen hinzufügen und bearbeiten beschrieben.
