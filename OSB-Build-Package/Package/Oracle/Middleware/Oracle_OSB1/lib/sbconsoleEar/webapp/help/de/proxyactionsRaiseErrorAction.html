Fehlerausgabe-Aktionen hinzufügen

Verwenden Sie die Auslösefehleraktion, um eine Exception mit einem bestimmten Fehlercode (einer Zeichenfolge) und einer Beschreibung auszulösen.

Bevor Sie beginnen

In diesen Anweisungen wird davon ausgegangen, dass Sie bereits einen Nachrichtenfluss auf der Seite Nachrichtenfluss bearbeiten bearbeiten, wie in Nachrichtenfluss erstellen, bearbeiten und anzeigen erläutert.

Aktion Auslösefehler hinzufügen

  1. Navigieren Sie an die Stelle, an der Sie die Aktion hinzufügen möchten. Eine Beschreibung hierzu finden Sie unter Aktionen im Nachrichtenfluss hinzufügen und bearbeiten.
  2. Klicken Sie auf das entsprechende Symbol, und wählen SieAktion hinzufügen > Flusssteuerung > Auslösefehler .
  3. Geben Sie im Feld Fehlercode den Fehlercode ein, den Sie auslösen möchten.
  4. Geben Sie im Feld Fehlermeldung eine Beschreibung des Fehlercodes ein.

Wenn Sie fertig sind

Wenn Sie die Konfiguration dieser Aktion abgeschlossen haben, konfigurieren Sie weitere Aktionen, oder speichern Sie Ihre Konfiguration, wie unter Aktionen in Nachrichtenflüssen hinzufügen und bearbeiten beschrieben.

Transaktionen

Wenn ein Service transaktional ist, bricht die Aktion Auslösefehler die Transaktion in der Anforderung (asynchron) oder in entweder der Anforderung oder der Antwort (synchron) ab. Offlinesetzen können Sie eine Introspektion von Nachrichten durchführen und Bedingungen festlegen, unter denen die Aktion "Auslösefehler" auftreten soll, auch wenn kein SOAP-Fehler auftritt, und "Auslösefehler" dazu führt, dass die Transaktion abgebrochen wird
