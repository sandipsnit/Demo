Umbenennungsaktionen hinzufügen

Verwenden Sie die Umbenennungsaktion, um die von einem XPath-Ausdruck gewählten Elemente umzubenennen, ohne den Inhalt des Elements zu ändern. Die Umbenennungsaktion ist eine von mehreren Aktualisierungsaktionen.

Bevor Sie beginnen

In diesen Anweisungen wird davon ausgegangen, dass Sie bereits einen Nachrichtenfluss auf der Seite Nachrichtenfluss bearbeiten bearbeiten, wie in Nachrichtenfluss erstellen, bearbeiten und anzeigen erläutert.

So fügen Sie eine Umbenennungsaktion hinzu

  1. Navigieren Sie an die Stelle, an der Sie die Aktion hinzufügen möchten. Eine Beschreibung hierzu finden Sie unter Aktionen im Nachrichtenfluss hinzufügen und bearbeiten.
  2. Klicken Sie auf das entsprechende Symbol, danach klicken Sie auf Aktion hinzufügen > Nachrichtenverarbeitung > Umbenennen.
  3. Klicken Sie auf <XPath>. Die Seite XPath-Ausdruckseditor wird angezeigt. Mit dem XQuery-Ausdruckseditor können Sie die Daten (in der benannten Variablen) angeben, die umbenannt werden sollen. Siehe Inline-XQuery- und XPath-Ausdrücke erstellen und bearbeiten .
  4. Im Feld Variable geben Sie die Kontextvariable ein, die das Element für die Umbenennung enthält. Weitere Informationen zu Kontextvariablen finden Sie unter ###Anhang A, Nachrichtenkontext..
  5. Führen Sie einen der folgenden Schritte aus:
    • Um das gewählte Element mit einem lokalen Namen umzubenennen, klicken Sie zuerst auf das erste Optionsfeld localname, dann geben Sie einen lokalen Namen in das Feld localname ein.
    • Um das gewählte Element mit einem Namespace umzubenennen, klicken Sie zuerst auf das erste Optionsfeld namespace, dann geben Sie einen Namespace in das Feld namespace ein.
    • Um das gewählte Element mit einem lokalen Namen und einem Namespace umzubenennen, klicken Sie zuerst auf das Optionsfeld localname und namespace, dann geben Sie einen lokalen Namen und einen Namespace in die Felder localname und namespace ein.

Wenn Sie fertig sind

Wenn Sie die Konfiguration dieser Aktion abgeschlossen haben, konfigurieren Sie weitere Aktionen, oder speichern Sie Ihre Konfiguration, wie unter Aktionen in Nachrichtenflüssen hinzufügen und bearbeiten beschrieben.
