Ersetzungsaktionen hinzufügen

Mit einer Ersetzungsaktion ersetzen Sie einen Knoten oder Knotencontent, der durch einen XPath-Ausdruck festgelegt wurde. Der Knoten oder sein Content werden durch den Wert ersetzt, der von einem XQuery-Ausdruck zurückgegeben wird.

Eine Ersetzungsaktion kann zum Ersetzen einfacher Werte, Elemente und selbst Attribute verwendet werden. Ein XQuery-Ausdruck, der keinen Wert zurückgibt, entspricht dem Löschen oder Leeren der identifizierten Knoten, je nachdem, ob die Aktion den gesamten Knoten oder nur den Knotencontent ersetzt.

Die Ersetzungsaktion ist Teil der Aktualisierungsaktionen.

Bevor Sie beginnen

In diesen Anweisungen wird davon ausgegangen, dass Sie bereits einen Nachrichtenfluss auf der Seite Nachrichtenfluss bearbeiten bearbeiten, wie in Nachrichtenfluss erstellen, bearbeiten und anzeigen erläutert.

So fügen Sie eine Ersetzungsaktion hinzu

  1. Navigieren Sie an die Stelle, an der Sie die Aktion hinzufügen möchten. Eine Beschreibung hierzu finden Sie unter Aktionen im Nachrichtenfluss hinzufügen und bearbeiten.
  2. Klicken Sie auf das entsprechende Symbol, und wählen Sie Aktion hinzufügen > Nachrichtenverarbeitung > Ersetzen.
  3. Klicken Sie auf <XPath>. Die Seite XPath-Ausdruckseditor wird angezeigt. Mit einem XPath-Ausdruck geben Sie die Daten (in der benannten Variablen) an, die ersetzt werden sollen. Siehe Eingebettete XQuery- und XPath-Ausdrücke erstellen und bearbeiten.
  4. Wenn Sie die Bearbeitung eines XPath-Ausdrucks abgeschlossen haben, geben Sie im Feld Variable eine Kontextvariable ein. Siehe Anhang A, Nachrichtenkontext.
  5. Klicken Sie auf <Ausdruck>. Die Seite XQuery-Ausdruckseditor wird angezeigt. Mit einen XQuery-Ausdruck erstellen Sie die Daten, die diejenigen Daten ersetzen sollen, die in der benannten Variablen durch den XPath angegeben wurden. Siehe Eingebettet XPath- und XQuery-Ausdrücke erstellen und bearbeiten.
  6. Wenn Sie die Bearbeitung des XQuery-Ausdrucks abgeschlossen haben, wählen Sie eine der folgenden Optionen:
    • Ganzen Knoten ersetzen - damit geben Sie an, dass alle vom XPath-Ausdruck gewählten Knoten zusammen mit deren Kontext ersetzt werden sollen, wobei Sie zuvor den XPath-Ausdruck definiert haben.
    • Knoteninhalt ersetzen - damit geben Sie an, dass nur der Content, nicht der Knoten selbst ersetzt werden soll.
      • Hinweis: Wenn Sie die Option Knoteninhalt ersetzen wählen und das Feld XPath leer lassen, ist dieses Vorgehen effektiver, als die Option Ganzen Knoten ersetzen zu wählen und den XPath auf ./* zu setzen.

Wenn Sie fertig sind

Wenn Sie die Konfiguration dieser Aktion abgeschlossen haben, konfigurieren Sie weitere Aktionen, oder speichern Sie Ihre Konfiguration, wie unter Aktionen in Nachrichtenflüssen hinzufügen und bearbeiten beschrieben.
