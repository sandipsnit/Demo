Routing-Aktionen zu Route-Knoten hinzufügen

Legen Sie einen Ziel-Service für die Nachricht fest, und konfigurieren Sie, wie die Nachricht an den Service geleitet wird.

Hierbei handelt es sich um eine abschließende Aktion. Das heißt, Sie können nach dieser keine weiteren Aktionen hinzufügen. Diese Aktion kann allerdings Anforderungs- und Antwortaktionen beinhalten.

Weitere Informationen zum Routing finden Sie unter "Modeling Message Flow in Oracle Service Bus" im Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus auf der Website http://www.oracle.com/pls/as111170/lookup?id=OSBAG181.

Bevor Sie beginnen

In diesen Anweisungen wird davon ausgegangen, dass Sie bereits einen Nachrichtenfluss auf der Seite Nachrichtenfluss bearbeiten bearbeiten, wie in Nachrichtenfluss erstellen, bearbeiten und anzeigen erläutert.

So fügen Sie eine Routing-Aktion zu einem Route-Knoten hinzu

  1. Navigieren Sie an die Stelle, an der Sie die Aktion hinzufügen möchten. Eine Beschreibung hierzu finden Sie unter Aktionen im Nachrichtenfluss hinzufügen und bearbeiten.
  2. Klicken Sie auf das Symbol Route-Knoten und anschließend auf Route bearbeiten. Die Seite Stage-Konfiguration bearbeiten wird angezeigt.
  3. Klicken Sie auf das Symbol Aktion hinzufügen, dann wählen Sie Kommunikation > Routing.
  4. Klicken Sie auf Service. Der Service-Browser wird angezeigt.
  5. Wählen Sie einen Service aus der Liste aus, klicken Sie dann auf Weiterleiten. Der Service wird statt des Standardlinks angezeigt.
  6. Wenn Ausgangs- und Eingangsvorgang identisch sein sollen, aktivieren Sie das Kontrollkästchen Eingangsvorgang für Ausgang verwenden.
  7. Klicken Sie im Feld Anforderungsaktionen auf Aktion hinzufügen, um eine Aktion hinzuzufügen, dann wählen Sie eine Aktion aus, die Sie mit diesem Service verknüpfen möchten. Sie können mehr als eine Aktion hinzufügen. Weitere Informationen darüber, welche Typen von Aktionen Sie hinzufügen können, finden Sie in der Aktionstabelle unter Aktionen im Nachrichtenfluss hinzufügen und bearbeiten.
  8. Klicken Sie im Feld Antwortaktionen auf Aktion hinzufügen, um eine Aktion hinzuzufügen, wählen Sie dann eine Aktion aus, die Sie mit diesem Service verknüpfen möchten. Sie können mehr als eine Aktion hinzufügen. Weitere Informationen darüber, welche Typen von Aktionen Sie hinzufügen können, finden Sie in der Aktionstabelle unter Aktionen im Nachrichtenfluss hinzufügen und bearbeiten.
  9. Klicken Sie auf Speichern.
  10. Setzen Sie die Erstellung des Nachrichtenflusses auf der Seite Nachrichtenfluss bearbeiten fort, wie in Nachrichtenfluss erstellen, bearbeiten und anzeigen erläutert.
  11. Klicken Sie auf Speichern, um die Aktualisierungen in der aktuellen Session zu speichern.
  12. Um die Session zu beenden und die Konfiguration für die Laufzeit bereitzustellen, klicken Sie unter Change Center auf Aktivieren.
