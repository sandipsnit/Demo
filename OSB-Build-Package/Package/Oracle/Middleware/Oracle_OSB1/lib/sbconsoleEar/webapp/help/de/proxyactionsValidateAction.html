Validierungsaktionen hinzufügen

Verwenden Sie eine Validierungsaktion, um Elemente, die von einem XPath-Ausdruck gewählt wurden, anhand eines XML-Schemaelements oder einer WSDL-Ressource zu validieren. Sie können nur globale Elemente validieren. Oracle Service Bus unterstützt die Validierung mithilfe lokaler Element nicht.

Bevor Sie beginnen

In diesen Anweisungen wird davon ausgegangen, dass Sie bereits einen Nachrichtenfluss auf der Seite Nachrichtenfluss bearbeiten bearbeiten, wie in Nachrichtenfluss erstellen, bearbeiten und anzeigen erläutert.

So fügen Sie eine Validierungsaktion hinzu

  1. Navigieren Sie an die Stelle, an der Sie die Aktion hinzufügen möchten. Eine Beschreibung hierzu finden Sie unter Aktionen im Nachrichtenfluss hinzufügen und bearbeiten.
  2. Klicken Sie auf das entsprechende Symbol, danach klicken Sie auf Aktion hinzufügen > Nachrichtenverarbeitung > Validieren.
  3. Klicken Sie auf XPath, um einen XPath-Ausdruck zu erstellen, der die zu validierenden Elemente angibt. Siehe Inline-XQuery- und XPath-Ausdrücke erstellen und bearbeiten. Wenn den Ausdruck im XPath-Ausdruckseditor fertig erstellt haben, klicken Sie auf Speichern, um den Ausdruck auf der Seite Stage-Konfiguration bearbeiten einzufügen.
  4. Geben Sie im Feld Variable den Namen der Variablen ein, die das zu validierende Element enthalten soll.
  5. Klicken Sie auf Ressource, dann wählen Sie WSDL oder Schema.
  6. Führen Sie im WSDL-Browser oder im XML-Schema-Browser folgende Schritte durch:
    1. Wählen Sie das WSDL- oder XML-Schema.
    2. Wählen Sie WSDL- oder XML-Schematyp oder -element.
    3. Klicken Sie auf Weiterleiten.
  7. Um das Ergebnis dieser Validierung zu speichern (ein Boolesches Ergebnis), wählen Sie Validierungsergebnis speichern in Variable, dann geben Sie den Namen der Variablen ein, in der Sie das Ergebnis speichern möchten.

    8. Alternativ dazu wählen Sie bei Validierungsfehlern Fehler auslösen, um einen Fehler auszulösen, falls die Validierung anhand des WSDL- oder XML-Schemaelements nicht erfolgreich ist.

Wenn Sie fertig sind

Wenn Sie die Konfiguration dieser Aktion abgeschlossen haben, konfigurieren Sie weitere Aktionen, oder speichern Sie Ihre Konfiguration, wie unter Aktionen in Nachrichtenflüssen hinzufügen und bearbeiten beschrieben.
