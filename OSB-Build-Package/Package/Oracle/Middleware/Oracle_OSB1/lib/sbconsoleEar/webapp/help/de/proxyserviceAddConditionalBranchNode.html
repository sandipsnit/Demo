Bedingte Verzweigungsknoten zum Nachrichtenfluss hinzufügen

Ein Verzweigungsknoten ermöglicht das Verarbeiten in genau einem von mehreren Pfaden. Die Verzweigung wird durch eine XPath-basierte Switch-Tabelle gesteuert. Jeder Zweig in der Tabelle gibt einen Zustand an (z.B. <500), der den Nachrichtenfluss abwärts anhand eines einzelnen XPath-Ausdrucks geprüft wird (z.B. ./ns: PurchaseOrder/ns:totalCost in $body). Die Tatsache, welcher Zustand zuerst eine Entsprechung findet, bestimmt, welcher Verzweigung gefolgt wird. Findet kein Verzweigungszustand eine Entsprechung, wird der Standardverzweigung gefolgt. Ein Verzweigungsknoten kann mehrere Nachfolger im Nachrichtenfluss haben: einen für jede Verzweigung, einschließlich der Standardverzweigung.

Wenn der Proxy-Service nicht auf WSDL basiert und er mehrere Dokumenttypen als Eingaben erhält, sollten Sie die Verwendung eines bedingten Verzweigungsknotens in Betracht ziehen.

Die bedingte Verzweigung wird durch eine Lookup-Tabelle gesteuert, bei der jede Verzweigung mit einer einfachen, aber eindeutigen Zeichenfolge gekennzeichnet wird. Eine Variable im Nachrichtenkontext wird als Lookup-Variable für diesen Knoten festgelegt, und zur Laufzeit wird mit diesem Wert bestimmt, welcher Verzweigung gefolgt wird. Wenn keine Verzweigung mit dem Wert der Lookup-Variablen übereinstimmt, wird der Standardverzweigung gefolgt. Sie sollten den Proxy-Service so konzipieren, dass der Wert der Lookup-Variablen eingestellt wird, bevor der Verzweigungsknoten erreicht wird.

Bevor Sie beginnen

In diesen Anweisungen wird davon ausgegangen, dass Sie bereits einen Nachrichtenfluss auf der Seite Nachrichtenfluss bearbeiten bearbeiten, wie in Nachrichtenfluss erstellen, bearbeiten und anzeigen erläutert.

So fügen Sie eine bedingte Verzweigung zu einem Nachrichtenfluss hinzu:

  1. Klicken Sie auf das Symbol Pipeline-Paarknoten oder Verzweigungsknoten und anschließend auf Bedingten Verzweigungsknoten erstellen. Der bedingte Verzweigungsknoten wird hinzugefügt, und alle nach dem eingefügten Verzweigungsknoten vorhandenen Knoten werden zur Standardverzweigung des neuen bedingten Verzweigungsknoten verschoben.
  2. Um den Standardnamen zu ändern und eine Beschreibung für den Verzweigungsknoten hinzuzufügen, führen Sie folgende Schritte durch:
    1. Klicken Sie auf das Symbol Bedingte Verzweigung und anschließend auf Name und Kommentare bearbeiten.
    2. Ändern Sie den Namen und die Beschreibung nach Wunsch.
    3. Klicken Sie auf Speichern.
  3. Um weitere Verzweigungsdefinitionen hinzuzufügen, klicken Sie auf das Symbol Bedingte Verzweigung und anschließend auf Verzweigung bearbeiten. Die Seite Verzweigungsknoten bearbeiten wird angezeigt.
  4. Gehen Sie folgendermaßen vor:
    1. Klicken Sie im Feld gewählter Pfad auf Bearbeiten, um einen XPath-Ausdruck für die Pfadangabe hinzuzufügen. Siehe Inline-XQuery- und XPath-Ausdrücke erstellen und bearbeiten.
    2. Geben Sie im Feld Variable eine Kontextvariable ein.
    3. Wählen Sie im Feld Operator einen Vergleichsoperator aus.
    4. Geben Sie im Feld Wert einen Wert für die Verzweigung ein.
    5. Geben Sie im Feld Beschriftung eine Beschriftung für die Verzweigung ein.
  5. Klicken Sie optional unter Optionen auf:
    • Neue Verzweigung hinzufügen, um eine neue Verzweigungsdefinition zu diesem Verzweigungsknoten hinzuzufügen.
    • Diese Verzweigung löschen, um eine Verzweigungsdefinition zu löschen.
    • Verzweigung nach oben verschieben oder Verzweigung nach unten verschieben, um die Positionen der Verzweigungsdefinitionen zu ändern. Diese Option wird nur angezeigt, wenn mehr als eine Verzweigungsdefinition vorhanden ist.
  6. Klicken Sie auf Speichern, um die Aktualisierungen in der aktuellen Session zu speichern.
  7. Setzen Sie die Erstellung des Nachrichtenflusses auf der Seite Nachrichtenfluss bearbeiten fort, wie in Nachrichtenfluss erstellen, bearbeiten und anzeigen erläutert.
  8. Klicken Sie auf Speichern, um die Aktualisierungen in der aktuellen Session zu speichern.
  9. Um die Session zu beenden und die Konfiguration für die Laufzeit bereitzustellen, klicken Sie unter Change Center auf Aktivieren.
