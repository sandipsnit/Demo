Pipeline Error Handler hinzufügen

Bevor Sie beginnen

Bei diesen Anweisungen wird vorausgesetzt, dass Sie bereits einen Nachrichtenfluss auf der Seite "Nachrichtenfluss bearbeiten" bearbeiten. Weitere Informationen finden Sie unter Nachrichtenfluss erstellen, bearbeiten und anzeigen.

Außerdem wird vorausgesetzt, dass Sie einen Pipeline-Paarknoten erstellt haben. Weitere Informationen finden Sie unter Pipeline-Paarknoten zu Nachrichtenfluss hinzufügen.

So fügen Sie einen Pipeline Error Handler hinzu:

  1. Navigieren Sie zu dem Pipeline-Paarknoten mit der Pipeline, zu der Sie einen Error Handler hinzufügen möchten. Falls das Pipeline-Paar noch nicht erweitert ist, klicken Sie auf das Pluszeichen neben dem Symbol, um die Pipelines anzuzeigen.
  2. Klicken Sie auf das Symbol Anforderungs-Pipeline oder auf das Symbol Antwort-Pipeline, und klicken Sie auf Pipeline Error Handler hinzufügen. Die Seite Error Handler bearbeiten wird angezeigt.
  3. Klicken Sie auf das Symbol Error Handler und anschließend auf Stage hinzufügen.
  4. Klicken Sie auf das Symbol Stage und dann auf Stage bearbeiten. Die Seite Stage-Konfiguration bearbeiten wird angezeigt.
  5. Klicken Sie auf Aktion hinzufügen, und wählen Sie die gewünschte Aktion.

    6. Ein Error Handler ist eine Pipeline und wird daher wie jede andere Pipeline konfiguriert. Beispiel: Mit der Aktion Publishing können Sie Fehlerbenachrichtigungen an andere Services senden, mit der Aktion Zuweisen können Sie die Kontextvariablen ändern und so weiter. Weitere Informationen zum Typ der hinzuzufügenden Aktion finden Sie in der entsprechenden Prozedur unter Aktionen im Nachrichtenfluss hinzufügen und bearbeiten. Die Möglichkeiten zum Aneinanderketten von Aktionen sind uneingeschränkt.

    Drei häufig verwendete Fehleraktionen sind Auslösefehler, Antworten und Wiederaufnehmen.

  6. Sie können auf der Seite Stage-Konfiguration bearbeiten weitere Aktionen hinzufügen und andere Bearbeitungen vornehmen.
  7. Um die Aktualisierungen in der aktuellen Session festzuschreiben, klicken Sie auf der Seite Stage-Konfiguration bearbeiten auf Speichern.
  8. Um die Aktualisierungen in der aktuellen Session festzuschreiben, klicken Sie auf der Seite Error Handler bearbeiten auf Speichern.

Wenn Sie fertig sind

Fahren Sie auf der Seite Nachrichtenfluss bearbeiten mit dem Erstellen des Nachrichtenflusses fort. Weitere Informationen finden Sie unter Nachrichtenfluss erstellen, bearbeiten und anzeigen.
