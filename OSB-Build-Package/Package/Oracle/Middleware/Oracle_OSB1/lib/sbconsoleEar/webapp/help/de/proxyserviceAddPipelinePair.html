Pipeline-Paarknoten zu Nachrichtenfluss hinzufügen

Nachrichtenfluss kann Null oder mehr Pipeline-Paarknoten enthalten: Anforderungs- und Antwort-Pipelines für den Proxy-Service (oder für die Vorgänge auf dem Service). Für Stages, Pipelines und Proxy-Services können Error Handler-Pipelines definiert werden. Pipelines können einen oder mehrere Stages enthalten, die wiederum Aktionen beinhalten.

Bevor Sie beginnen

In diesen Anweisungen wird davon ausgegangen, dass Sie bereits einen Nachrichtenfluss auf der Seite Nachrichtenfluss bearbeiten bearbeiten, wie in Nachrichtenfluss erstellen, bearbeiten und anzeigen erläutert.

So fügen Sie einen Pipeline-Paarknoten zu einem Nachrichtenfluss hinzu

  1. Klicken Sie auf das Symbol Proxy-Service und dann auf Pipeline-Paar hinzufügen.
  2. So ändern Sie den Standardnamen und fügen eine Beschreibung für den Pipeline-Paarknoten hinzu:
    1. Klicken Sie auf das Symbol Pipeline-Paarknoten und dann auf Bearbeiten Name und Anmerkung.
    2. Ändern Sie den Namen und die Beschreibung nach Wunsch.
    3. Klicken Sie auf Speichern.
      4. Hinweis: Wenn Sie einen Pipeline- oder einen Route-Knoten umbenennen, kann es sein, dass die Anzahl der Nachrichten, die auf der Seite Dashboard im Modul Überwachung angezeigt wird, nicht mit der von anderen Komponenten korreliert, weil die Pipeline-Zähler auf Null gesetzt wurden. Der Grund dafür ist, dass Oracle Service Bus das Umbenennen als Löschen und Neuerstellen behandelt. Die Anzahl sollte nach einem Zeitintervall, das dem Service-Überwachungsintervall entspricht, wieder korrelieren.
  3. Wie Sie der Pipeline Stages hinzufügen können, finden Sie unter Pipelines Stages hinzufügen
  4. So fügen Sie in der Pipeline Stages hinzu. Siehe Aktionen in Nachrichtenfluss hinzufügen und bearbeiten
  5. Setzen Sie die Erstellung des Nachrichtenflusses auf der Seite Nachrichtenfluss bearbeiten fort, wie in Nachrichtenfluss erstellen, bearbeiten und anzeigen erläutert.
  6. Klicken Sie auf Speichern, um die Aktualisierungen in der aktuellen Session zu speichern.
  7. Um die Session zu beenden und die Konfiguration für die Laufzeit bereitzustellen, klicken Sie unter Change Center auf Aktivieren.
