Route-Knoten zum Nachrichtenfluss hinzufügen

Bevor Sie beginnen

In diesen Anweisungen wird davon ausgegangen, dass Sie bereits einen Nachrichtenfluss auf der Seite Nachrichtenfluss bearbeiten bearbeiten, wie in Nachrichtenfluss erstellen, bearbeiten und anzeigen erläutert.

So fügen Sie einen Route-Knoten zu einem Nachrichtenfluss hinzu:

  1. Klicken Sie auf das Symbol Pipeline-Paarknoten und anschließend auf Route hinzufügen.
  2. Um den Standardnamen zu ändern und eine Beschreibung für den Route-Knoten hinzuzufügen, klicken Sie auf das Symbol Route-Knoten und anschließend auf Name und Kommentare bearbeiten. Ändern Sie Name und Beschreibung nach Bedarf, und klicken Sie dann auf Speichern.
    3. Hinweis: Wenn Sie einen Pipeline- oder einen Route-Knoten umbenennen, kann es sein, dass die Anzahl der Nachrichten, die auf der Seite Dashboard im Modul Überwachung angezeigt wird, nicht mit der von anderen Komponenten korreliert, weil die Pipeline-Zähler auf Null gesetzt wurden. Der Grund dafür ist, dass Oracle Service Bus das Umbenennen als Löschen und Neuerstellen behandelt. Die Anzahl sollte nach einem Zeitintervall, das dem Service-Überwachungsintervall entspricht, wieder korrelieren.
  3. Um Aktionen zu einem Route-Knoten hinzuzufügen, klicken Sie auf das Symbol Route-Knoten und anschließend auf Route bearbeiten. Die Seite Nachrichtenfluss bearbeiten wird angezeigt. Weitere Informationen zu den Aktionen, die zu Route-Knoten hinzugefügt werden können, finden Sie in folgenden Abschnitten:
    • Wenn-Dann-Aktionen hinzufügen
    • Dynamisches Routing zu Route-Knoten hinzufügen
    • Routing-Aktionen zu Route-Knoten hinzufügen
    • Routing-Tabellen zu Route-Knoten hinzufügen
    • Route-Knoten-Error Handler hinzufügen
  4. Setzen Sie die Erstellung des Nachrichtenflusses auf der Seite Nachrichtenfluss bearbeiten fort, wie in Nachrichtenfluss erstellen, bearbeiten und anzeigen erläutert.
  5. Klicken Sie auf Speichern, um die Aktualisierungen in der aktuellen Session zu speichern.
  6. Um die Session zu beenden und die Konfiguration für die Laufzeit bereitzustellen, klicken Sie unter Change Center auf Aktivieren.
