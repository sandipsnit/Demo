Stages zu Pipelines hinzufügen

Bevor Sie beginnen

In diesen Anweisungen wird davon ausgegangen, dass Sie bereits einen Nachrichtenfluss auf der Seite Nachrichtenfluss bearbeiten bearbeiten, wie in Nachrichtenfluss erstellen, bearbeiten und anzeigen erläutert.

So fügen Sie eine Stage zu einer Pipeline hinzu:

  1. Falls erforderlich, klicken Sie auf das Pluszeichen links neben dem Symbol Pipeline-Paarknoten, um es zu erweitern. Ein Pipeline-Paar enthält eine Anforderungs-Pipeline und eine Antwort-Pipeline.
  2. Klicken Sie auf die Pipeline, der Sie die Stage hinzufügen möchten, und anschließend auf Stage hinzufügen.
  3. Um den Standardnamen zu ändern und eine Beschreibung für die Stage hinzuzufügen, führen Sie folgende Schritte durch:
    1. Klicken Sie auf das Symbol Stage und anschließend auf Name und Kommentare bearbeiten.
    2. Ändern Sie den Namen und die Beschreibung nach Wunsch.
    3. Klicken Sie auf Speichern.
  4. Um Aktionen zur Stage hinzuzufügen, klicken Sie auf das Symbol Stage und anschließend auf Stage bearbeiten. Siehe Aktionen im Nachrichtenfluss hinzufügen und bearbeiten.
  5. Um eine Fehlerbehandlung zur Stage hinzuzufügen, klicken Sie auf das Symbol Stage und anschließend auf Stage Error Handler hinzufügen. Siehe Stage Error Handler hinzufügen. Die Seite Nachrichtenfluss bearbeiten wird angezeigt.
  6. Setzen Sie die Erstellung des Nachrichtenflusses fort, wie in Nachrichtenfluss erstellen, bearbeiten und anzeigen erläutert.
  7. Klicken Sie auf Speichern, um die Aktualisierungen in der aktuellen Session zu speichern.
  8. Um die Session zu beenden und die Konfiguration für die Laufzeit bereitzustellen, klicken Sie unter Change Center auf Aktivieren.
