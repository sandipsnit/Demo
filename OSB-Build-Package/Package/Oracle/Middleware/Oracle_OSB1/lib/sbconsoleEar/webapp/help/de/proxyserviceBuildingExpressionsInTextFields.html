Ausdrücke in den Textfeldern des Editor-Workspace erstellen

Der XQuery-Ausdruckseditor, der XQuery-Bedingungseditor und der XPath-Ausdruckseditor verfügen alle über Textfelder, in denen Sie Ausdrücke erstellen können, indem Sie Elemente direkt eintippen oder von den Paletten einfügen.

Bevor Sie beginnen

Bei diesen Anweisungen wird davon ausgegangen, dass Sie einen Ausdruck im XQuery-Ausdruckseditor, XQuery-Bedingungseditor oder im XPath-Ausdruckseditor erstellen oder bearbeiten, wie in Inline-XQuery- und XPath-Ausdrücke erstellen und bearbeiten erläutert.

So erstellen Sie einen Ausdruck in einem Textfeld:

  1. Zeigen Sie das Fenster mit dem Textfeld an. Führen Sie je nach Editor einen der folgenden Schritte aus:
    • Wählen Sie im XQuery-Ausdruckseditor den Wert XQuery-Text (unter der Workspace-Schaltflächenleiste), sofern dieser noch nicht gewählt ist.
    • Wählen Sie im XQuery-Bedingungseditor den Wert Text (unter der Workspace-Schaltflächenleiste), sofern dieser noch nicht gewählt ist.
    • Im XPath-Ausdruckseditor brauchen Sie nichts auszuwählen, da keine Optionen zum Auswählen anderer Tools vorhanden sind.
      • Hinweis: Wenn Sie einen der oben stehenden Links wählen, wird ein Textfeld angezeigt, in dem Sie einen kompletten Ausdruck für den jeweiligen Kontext erstellen können. Die in diesem Thema beschriebenen Tools und Techniken können überall angewendet werden, wo sich Textfelder in Editoren befinden, beispielsweise wenn Sie Variablen aus den importierten Ressourcen mit dem Inline-Ausdruck verbinden, wie in Externe XQuery-Ressourcen an eingebettete Inline-XQuerys binden und Externe XSLT-Ressourcen an Inline-XQuerys binden erläutert.
  2. Wenn Sie möchten, können Sie einen Ausdruck oder ein Ausdrucksfragment in das Feld eintippen oder einfügen. Wenn Sie auf diese Weise einen kompletten Ausdruck erstellen, machen Sie unten mit Schritt 7 weiter. Ansonsten gehen Sie zum nächsten Schritt.
  3. Wählen Sie die Palette mit den Elementen aus, die Sie zum Ausdruck hinzufügen möchten, und suchen Sie das hinzuzufügende Element, wie in Tabelle 22-1 unten beschrieben.

    Siehe auch:
    • XQuery-Transformationen
    • Weitere Informationen finden Sie unter "XQuery Implementation" im Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus unter http://www.oracle.com/pls/as111170/lookup?id=OSBAG444.
    Variablenstrukturen
    Zeigt Variablen und ihren Content als Bäume an, die für die Visualisierung hilfreich sind.
    So zeigen Sie eine Variablenstruktur und ihren Content an
    • Wählen Sie den Namen der Struktur aus der Dropdown-Liste im oberen Bereich der Palette. Die Liste zeigt die vordefinierten Nachrichtenkontextvariablen (Anhänge, Text, Header, Ausgehend und Eingehend) sowie benutzerdefinierte Strukturen, die nach Typ (XML-Typ, Service-Schnittstelle und Einfacher Typ) organisiert sind.
    • Maximieren oder minimieren Sie die Knoten im Baum, indem Sie auf das Plus- (+) oder das Minuszeichen (-) klicken.
    Sie können auch eigene Variablenstrukturen definieren. Siehe Variablenstrukturen in den XQuery-Editoren erstellen.
    Variablenstrukturen erstellen keine Variablen. Variablen werden zur Laufzeit als das Ziel der Aktion "Zuweisen" in der Stage erstellt.
    Wenn Sie ein Element aus einem Variablenstrukturbaum in das Textfeld einfügen, wird es als XPath-Ausdruck eingefügt, der den Pfad beschreibt.
    Siehe auch XQuery-Transformationen.

  4. Fügen Sie das gewünschte Element mit einer der unten in Tabelle 22-2 beschriebenen Methoden in das Textfeld ein.

    5. Tabelle 22-2 Verschiedene Arten, Elemente in die Editor-Textfelder einzufügen 
    In Palette
    Vorgehensweise
    Palette Namespace-Definitionen
    Sie können einen Namespace standardmäßig mit der Maus oder der Tastatur wählen, kopieren und einfügen, zum Beispiel:
    1. Wählen Sie die gesamte Zeichenfolge des Namespace (oder seine Abkürzung, sofern vorhanden), indem Sie den Mauszeiger über die Zeichenfolge ziehen.
    2. Drücken Sie STRG-C, um die Zeichenfolge zu kopieren.
    3. Klicken Sie an die Stelle des Textfeldes, an der Sie den Namespace einfügen möchten.
    4. Drücken Sie STRG-V, um die Zeichenfolge einzufügen.
    Palette XQuery-Funktionen oder Palette Variablenstrukturen
    Ziehen Sie ein Element aus der Palette in das Textfeld.
    Hinweis: Das Ziehen von der Palette zum Workspace wird nur in Internet Explorer unterstützt.
    Palette XQuery-Funktionen oder Palette Variablenstrukturen
    1. Klicken Sie auf ein Element in der Palette. Das Element wird im Fenster Property Inspector angezeigt:
      • Funktionen werden mit Platzhaltern für die Werte angezeigt, die Sie angeben müssen.
      • Variablen und deren Attribute werden als XPath-Ausdrücke angezeigt.
    2. Klicken Sie auf die Stelle im Textfeld, an der Sie das Element einfügen möchten.
    3. Klicken Sie auf Eigenschaft kopieren, um das Element an der gewählten Stelle im Textfeld einzufügen.
    Palette XQuery-Funktionen oder Palette Variablenstrukturen
    1. Klicken Sie auf ein Element in der Palette. Das Element wird im Fenster Property Inspector angezeigt.
    2. Mit den Standardaktionen für Tastatur oder Maus wählen Sie das Element im Property Inspector aus und kopieren es.
    3. Wählen Sie eine Stelle im Textfeld aus, und fügen Sie das Element mit den Standardaktionen für Tastatur oder Maus in das Textfeld ein.

  5. Arbeiten Sie weiter mit den Drag-and-Drop-Funktionen, um den gewünschten Ausdruck zu erstellen.
  6. Bearbeiten Sie den Ausdruck im Textfeld nach Bedarf.
  7. Führen Sie optional einen oder beide der folgenden Schritte aus:
    • Klicken Sie auf Validieren. Eine Nachricht wird angezeigt, wenn der Ausdruck erfolgreich validiert wurde.
    • Um den Ausdruck zu testen, klicken Sie auf Testen. Siehe XQuery-Tests durchführen.
  8. Klicken Sie auf Speichern, um den Editor zu schließen und den Ausdruck in die Aktion einzufügen.

Wenn Sie fertig sind

Setzen Sie die Konfiguration der Aktion fort, wie in Aktionen im Nachrichtenfluss hinzufügen und bearbeiten erläutert.
