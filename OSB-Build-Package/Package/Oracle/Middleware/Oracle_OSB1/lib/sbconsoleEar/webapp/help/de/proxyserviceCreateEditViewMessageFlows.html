Nachrichtenfluss erstellen, bearbeiten und anzeigen

  1. Wenn dies noch nicht geschehen ist, klicken Sie zum Erstellen einer neuen Session auf Erstellen oder zum Aufrufen einer vorhandenen Session auf Bearbeiten. Weitere Informationen finden Sie unter Change Center verwenden.
  2. Suchen Sie den Proxy-Service, dessen Nachrichtenfluss Sie anzeigen oder bearbeiten möchten. Führen Sie eine der folgenden Aktionen aus:
    • Wählen Sie Project Explorer , um die Seite Projektansicht oder die Seite Projekt-/Ordneransicht anzuzeigen. Navigieren Sie dann durch die Projekte und Ordner, um den Proxy-Service zu finden.
    • Wählen Sie Ressourcenbrowser > Proxy-Services, um die Seite Zusammenfassung der Proxy-Services anzuzeigen.
  3. Klicken Sie auf das Symbol Nachrichtenfluss bearbeiten in der Zeile des Proxy-Service, dessen Nachrichtenfluss Sie anzeigen oder bearbeiten möchten.
    • Wenn für den gewählten Proxy-Service noch kein Nachrichtenfluss erstellt wurde, wird die Seite Nachrichtenfluss bearbeiten mit einem einzigen Symbol auf der Seite angezeigt, dem Symbol Proxy-Service. Dies ist ein Startknoten für den Proxy-Service. Klicken Sie auf dieses Symbol, um den Nachrichtenfluss zu konstruieren.
    • Wenn der Proxy-Service bereits über Nachrichtenfluss verfügt, enthält die Seite eine grafische Darstellung des Flusses. Klicken Sie auf die Symbole, um die Teile des Nachrichtenflusses anzuzeigen oder zu bearbeiten.

      • Siehe Seite "Nachrichtenfluss bearbeiten". Dort finden Sie weitere Informationen darüber, wie Sie Nachrichtenfluss auf der Seite aufbauen.

  4. Klicken Sie auf Speichern, um die Aktualisierungen in der aktuellen Session zu speichern.
  5. Um die Session zu beenden und die Konfiguration für die Laufzeit bereitzustellen, klicken Sie unter Change Center auf Aktivieren.
