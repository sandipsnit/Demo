Seite "SB-Transportkonfiguration"

Die Seite Proxy-Service erstellen/bearbeiten – SB-Transportkonfiguration ist eine von diversen Seiten, auf denen Sie Proxy-Services erstellen und bearbeiten können. Diese Funktionen sind unter Proxy-Services erstellen und konfigurieren und Proxy-Service-Konfigurationen bearbeiten genauer beschrieben.

Weitere Informationen finden Sie unter "SB Transport" im Oracle Fusion Middleware Developer's Guide for Oracle Service Bus unter http://www.oracle.com/pls/as111170/lookup?id=OSBDV997.

Auf dieser Seite können Sie Transporteinstellungen für Proxy-Services mit dem SB-(Service Bus-)Dateitransportprotokoll konfigurieren. In Tabelle 19-12 ist die Verwendung dieser Seite beschrieben.

Tabelle 19-12 Seite "Proxy-Service erstellen/bearbeiten – SB-Transportkonfiguration 
Option
Verwendung
Dispatch Policy
Wählen Sie eine Dispatch Policy für diesen Endpunkt, oder verwenden Sie die Standard-Dispatch Policy.
Die Dispatch Policy bezieht sich auf die Instanz des WLS-Work Managers, der vom Service-Endpunkt zur Verarbeitung von Anforderungen verwendet werden soll. Weitere Informationen zu Work Manager finden Sie unter "Using Work Managers to Optimize Scheduled Work" in Oracle Fusion Middleware Configuring Server Environments for Oracle WebLogic Server unter http://www.oracle.com/pls/as111170/lookup?id=CNFGD112.
SSL verwenden
Wenn diese Option gewählt ist, müssen Anforderungen über eine SSL-Verbindung gesendet werden.
Nicht gesicherte Verbindungen sind jedoch nicht verboten. Der Administrator muss alle nicht gesicherten Protokolle auf dem Server (z.B. t3 oder http) schließen, um gesicherte Clientverbindungen zu erzwingen.
