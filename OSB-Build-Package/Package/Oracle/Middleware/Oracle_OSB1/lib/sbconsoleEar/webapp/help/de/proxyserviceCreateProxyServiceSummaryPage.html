Seite "Zusammenfassung"

Die Seite Proxy-Service erstellen/bearbeiten - Übersicht gehört zu einer Reihe von Seiten zum Erstellen und Bearbeiten von Proxy-Services. Diese Aktionen sind in Proxy-Services erstellen und konfigurieren und in Proxy-Service-Konfigurationen bearbeiten beschrieben.

Sie verwenden diese Seite, um die Konfigurationseinstellungen für einen Proxy-Service vor dem Speichern anzuzeigen oder zu bearbeiten.

Wenn Sie die Einstellungen bearbeiten möchten, klicken Sie in der entsprechenden Kategoriezeile auf Bearbeiten (Beispiel: Allgemeine Konfiguration, Transportkonfiguration und so weiter). Sie können nur diejenigen Seiten bearbeiten, die Sie während des Erstellens des Proxy-Service konfiguriert haben. Die folgende Liste nennt alle Seiten:
