Namespaces zur Verwendung in Inline-Ausdrücken erstellen

Die Palette Namespace-Definitionen umfasst eine Liste von Standard-Namespaces. Sie können aber auch neue Namespaces definieren.

Bevor Sie beginnen

Bei diesen Anweisungen wird davon ausgegangen, dass Sie einen Ausdruck im XQuery-Ausdruckseditor, XQuery-Bedingungseditor oder im XPath-Ausdruckseditor erstellen oder bearbeiten, wie in Inline-XQuery- und XPath-Ausdrücke erstellen und bearbeiten erläutert.

So erstellen und verwenden Sie Namespaces in einem Inline-Ausdruck:

  1. Wählen Sie Namespace-Definitionen. Die Palette Namespace-Definitionen umfasst eine Liste mit Standard-Namespaces sowie Listen mit Variablen-Namespaces und benutzerdefinierten Namespaces, sofern vorhanden.
  2. So können Sie einen Benutzer-Namespace definieren und hinzufügen:
    1. Klicken Sie auf Namespace hinzufügen.
    2. Geben Sie im Feld Präfix eine eindeutige ID für den Namespace ein. Sie dürfen dasselbe Präfix nicht mehrmals verwenden.
    3. Geben Sie im Feld URI einen URL für diesen Namespace im Format http://url/.../ oder einen URN im Format uddi:server: ein.
    4. Klicken Sie auf Hinzufügen, um den Namespace zur Liste Benutzerdefinierte Namespaces hinzuzufügen.
    5. Kopieren Sie den benutzerdefinierten Namespace, und fügen Sie ihn in den XQuery-Ausdruck, in die XQuery-Bedingung oder in XPath ein. Weitere Informationen hierzu finden Sie unter Inline XQuery und XPath-Ausdrücke erstellen und bearbeiten.

Wenn Sie fertig sind

Fahren Sie gemäß der Beschreibung unter Inline XQuery und XPath-Ausdrücke erstellen und bearbeiten fort.
