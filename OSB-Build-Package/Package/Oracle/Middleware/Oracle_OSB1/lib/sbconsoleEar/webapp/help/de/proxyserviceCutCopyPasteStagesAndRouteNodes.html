Stages und Route-Knoten ausschneiden, kopieren und einfügen

Sie können Stages und Route-Knoten ausschneiden, kopieren und einfügen

  • Klicken Sie zum Ausschneiden von Stages oder Route-Knoten auf sein Symbol, und wählen Sie Ausschneiden oder Kopieren.
  • Wenn Sie eine Stage einfügen möchten, die Sie aus einem anderen Pipeline-Paar im Nachrichtenfluss dieses Proxy-Service oder vom Nachrichtenfluss eines anderen Proxy-Service ausgeschnitten oder kopiert haben, gehen Sie wie folgt vor:
    • Klicken Sie auf das Symbol Pipeline anfordern oder Antwort-Pipeline und dann auf Stage einfügen.
    • Klicken Sie auf das Symbol Stage in einer Pipeline, klicken Sie dann auf Einfügen.
  • Wenn Sie einen Route-Knoten einfügen wollen, den Sie aus dem Nachrichtenfluss eines anderen Proxy-Service ausgeschnitten oder kopiert haben, klicken Sie auf das Symbol Pipeline-Paarknoten für das Pipeline-Paar, klicken Sie dann auf Route einfügen.
