Variablenstrukturen in den XQuery-Editoren erstellen

In der Palette Variablenstrukturen in den XQuery- und XPath-Editoren werden grafische Darstellungen des Contents von Variablen angezeigt. Dazu gehören standardmäßig die vordefinierten Nachrichtenkontextvariablen attachments, body, header, outbound und inbound.

Jeder Zuordnungseintrag für die Variablenstruktur verfügt über eine Beschriftung und ordnet eine Variable oder einen Variablenpfad einer oder mehreren Strukturen zu. Der Geltungsbereich dieser Zuordnungen ist ein Stage- oder ein Route-Knoten.

Sie können auch eigene Variablenstrukturen festlegen, die auf Folgendem basieren:

  • XML-Typen, darunter:
    • Schemaelemente
    • WSDL-Elemente
    • Schematypen
    • WSDL-Typen
    • MFLs
  • Service-Schnittstellen
  • Einfache Typen (Zeichenfolge oder beliebige XML)

Sie können diese Funktion direkt für alle benutzerdefinierten Variablen verwenden sowie für $inbound, $outbound und $fault. Sie können diese Funktion jedoch nicht direkt verwenden, um auf XML-Anhänge in $attachments, auf Header in $header oder auf Dokumente und RPC-Parameter in $body zuzugreifen. Hierbei gilt jedoch eine Ausnahme: Sie können die Funktion direkt verwenden, um auf Dokumente und Parameter in $body für Anforderungsnachrichten zuzugreifen, die von einem WSDL-Proxy-Service empfangen wurden.

Bevor Sie beginnen

Bei diesen Anweisungen wird davon ausgegangen, dass Sie einen Ausdruck im XQuery-Ausdruckseditor, XQuery-Bedingungseditor oder im XPath-Ausdruckseditor erstellen oder bearbeiten, wie in Inline-XQuery- und XPath-Ausdrücke erstellen und bearbeiten erläutert.

So erstellen Sie eine Variablenstruktur:

  1. Wählen Sie Variablenstrukturen.
  2. Klicken Sie in der Palette Variablenstrukturen auf Neue Struktur hinzufügen.
  3. Fahren Sie mit einem der unten in Tabelle 22-3 beschriebenen Schritte fort.

    4. Tabelle 22-3 Neue Variablenstruktur erstellen 
    Ziel
    Führen Sie diese Schritte aus...
    Variablenstruktur erstellen, die eine Variable einem XML-Schematyp zuordnet
    1. Wählen Sie oben in der Palette Variablenstruktur die Option XML-Typen, wenn sie nicht bereits gewählt ist.
    2. Geben Sie im Feld Strukturlabel einen Anzeigenamen für die zu erstellende Variable ein. Mit diesem Anzeigenamen können Sie der Struktur einen aussagekräftigen Namen geben, damit Sie sie zum Zeitpunkt des Entwurfs erkennen können, der aber keine Auswirkungen auf die Laufzeit hat.
    3. Geben Sie im Feld Strukturpfad den Pfad der Variablenstruktur zur Laufzeit ein. Der Pfad muss mit $ beginnen.
    4. Aktivieren Sie unterhalb des Feldes Typ das gewünschte Optionsfeld, und wählen Sie dann Schematyp.
    5. Klicken Sie auf Durchsuchen. Daraufhin·wird·der·XML-Schemabrowser angezeigt. Wählen Sie aus der Liste ein XML-Schema und aus dem Bereich "Definitionen" einen XML-Schematyp aus, und klicken Sie auf Weiterleiten.
    6. Klicken Sie auf Hinzufügen, um die Variablenstruktur zu erstellen.
    Variablenstruktur erstellen, die eine Variable einem WSDL-Typ zuordnet
    1. Wählen Sie oben in der Palette Variablenstruktur die Option XML-Typen, wenn sie nicht bereits gewählt ist.
    2. Geben Sie im Feld Strukturlabel einen Anzeigenamen für die zu erstellende Variable ein. Mit diesem Anzeigenamen können Sie der Struktur einen aussagekräftigen Namen geben, damit Sie sie zum Zeitpunkt des Entwurfs erkennen können, der aber keine Auswirkungen auf die Laufzeit hat.
    3. Geben Sie im Feld Strukturpfad den Pfad der Variablenstruktur zur Laufzeit ein.
    4. Aktivieren Sie unterhalb des Feldes Typ das gewünschte Optionsfeld, und wählen Sie dann WSDL-Type.
    5. Klicken Sie auf Durchsuchen. Daraufhin wird der WSDL-Browser angezeigt. Wählen Sie aus der Liste der WSDLs eine WSDL und im Bereich "Definitionen" einen WSDL-Typ aus, und klicken Sie auf Weiterleiten.
    6. Klicken Sie auf Hinzufügen, um die Variablenstruktur zu erstellen.

    3. Geben Sie im Feld Strukturpfad den Pfad der Variablenstruktur zur Laufzeit ein.

    5. Klicken Sie auf Durchsuchen. Daraufhin·wird·der·XML-Schemabrowser angezeigt. Wählen Sie aus der Liste ein XML-Schema und aus dem Bereich "Definitionen" einen XML-Schematyp aus, und klicken Sie auf Weiterleiten.
    6. Klicken Sie auf Hinzufügen, um die Variablenstruktur zu erstellen.

    1. Wählen Sie oben in der Palette Variablenstruktur die Option XML-Typ, wenn sie nicht bereits gewählt ist.
    2. Geben Sie im Feld Strukturlabel einen Anzeigenamen für die zu erstellende Variable ein. Mit diesem Anzeigenamen können Sie der Struktur einen aussagekräftigen Namen geben, damit Sie sie zum Zeitpunkt des Entwurfs erkennen können, der aber keine Auswirkungen auf die Laufzeit hat.

    4. Aktivieren Sie unterhalb des Feldes Typ das gewünschte Optionsfeld, und wählen Sie dann WSDL-Element.

    Variablenstruktur erstellen, die eine Variable einem untergeordneten Element zuordnet
    1. Wählen Sie oben in der Palette Variablenstruktur die Option XML-Typ, wenn sie nicht bereits gewählt ist.
    2. Geben Sie im Feld Strukturlabel einen Anzeigenamen für die zu erstellende Variable ein. Mit diesem Anzeigenamen können Sie der Struktur einen aussagekräftigen Namen geben, damit Sie sie zum Zeitpunkt des Entwurfs erkennen können, der aber keine Auswirkungen auf die Laufzeit hat.
    3. Geben Sie im Feld Strukturpfad den Pfad der Variablenstruktur zur Laufzeit ein.
    4. Wählen Sie unter dem Feld Typ den zu erstellenden Variablentyp aus:

      5. Wenn Sie ein XML-Schemaelement oder eine WSDL-Elementvariable erstellen möchten, aktivieren Sie das Optionsfeld der entsprechenden Option, und wählen Sie anschließend Schemaelement oder WSDL-Element.

      Wenn Sie eine MFL-Variable erstellen möchten, aktivieren Sie das Optionsfeld der entsprechenden Option, und wählen Sie anschließend MFL.

    5. Klicken Sie bei XML-Schema, WSDL oder MFL auf Durchsuchen, um ein Objekt aus der vom Browser angezeigten Liste auszuwählen. Klicken Sie anschließend auf Weiterleiten. Beispiel: Wählen Sie eine MFL aus der Liste der MFLs, und klicken Sie auf Weiterleiten.
    6. Aktivieren Sie das Kontrollkästchen Als untergeordnetes Element festlegen, um das Element als untergeordnetes Element der Struktur festzulegen, die Sie gerade erstellen.
    7. Klicken Sie auf Hinzufügen, um die Variablenstruktur zu erstellen.
    Variablenstruktur erstellen, die eine MFL-Ressource verwendet
    1. Wählen Sie oben in der Palette Variablenstruktur die Option XML-Typ, wenn sie nicht bereits gewählt ist.
    2. Geben Sie im Feld Strukturlabel einen Anzeigenamen für die zu erstellende Variable ein. Mit diesem Anzeigenamen können Sie der Struktur einen aussagekräftigen Namen geben, damit Sie sie zum Zeitpunkt des Entwurfs erkennen können, der aber keine Auswirkungen auf die Laufzeit hat.
    3. Geben Sie im Feld Strukturpfad den Pfad der Variablenstruktur zur Laufzeit ein.
    4. Aktivieren Sie unterhalb des Feldes Typ das gewünschte Optionsfeld, und klicken Sie auf Durchsuchen. Daraufhin wird der MFL-Browser angezeigt.
    5. Wählen Sie aus der Liste der MFLs eine MFL aus, und klicken Sie auf Weiterleiten.
    6. Klicken Sie auf Hinzufügen, um die Variablenstruktur zu erstellen.
    Variablenstruktur für Service-Schnittstellen erstellen
    1. Wählen Sie oben in der Palette Variablenstruktur die Option Service-Schnittstelle.
    2. Geben Sie im Feld Strukturlabel einen Anzeigenamen für die zu erstellende Variable ein. Mit diesem Anzeigenamen können Sie der Struktur einen aussagekräftigen Namen geben, damit Sie sie zum Zeitpunkt des Entwurfs erkennen können, der aber keine Auswirkungen auf die Laufzeit hat.
    3. Im Feld Strukturpfad ist bereits der Standardwert $body festgelegt. Dieses Feld kann nicht geändert werden.
    4. Wählen Sie im Feld WSDL-basierter Service das Symbol Service-Browser, wählen Sie in der vom Service-Browser angezeigten Service-Liste einen Service aus, und klicken Sie auf Weiterleiten.

      5. Der gewählte Service wird im Feld "WSDL-basierter Service" angezeigt.

    5. Wählen Sie im Feld Vorgang einen Vorgang aus, oder wählen Sie Keine, damit kein Vorgang eingeschlossen wird.
    6. Klicken Sie auf Hinzufügen, um die Variable zu erstellen.
    Einfache Variablenstruktur erstellen
    1. Wählen Sie oben in der Palette Variablenstruktur die Option Einfacher Typ.
    2. Geben Sie im Feld Strukturlabel einen Anzeigenamen für die zu erstellende Variable ein. Mit diesem Anzeigenamen können Sie der Struktur einen aussagekräftigen Namen geben, damit Sie sie zum Zeitpunkt des Entwurfs erkennen können, der aber keine Auswirkungen auf die Laufzeit hat.
    3. Geben Sie im Feld Strukturname einen Namen für die zu erstellende Variablenstruktur ein.
    4. Wählen Sie unter dem Feld Typ entweder Zeichenfolge oder Beliebiges XML.
    5. Klicken Sie auf Hinzufügen, um die Variable zu erstellen.
