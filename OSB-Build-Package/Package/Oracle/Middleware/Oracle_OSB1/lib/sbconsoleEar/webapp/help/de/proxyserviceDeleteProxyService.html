Proxy-Services löschen

Wenn Sie einen Proxy-Service löschen, werden alle ACLs, auf die der Proxy verweist, aus dem von Oracle Service Bus gesteuerten Repository sowie vom entsprechenden Autorisierungsprovider gelöscht.

  1. Wenn dies noch nicht geschehen ist, klicken Sie zum Erstellen einer neuen Session auf Erstellen oder zum Aufrufen einer vorhandenen Session auf Bearbeiten. Weitere Informationen finden Sie unter Change Center verwenden.
  2. Wählen Sie Ressourcenbrowser > Proxy-Services, um die Seite Proxy-Services - Überblick anzuzeigen.
  3. Klicken·Sie·in·der·Zeile·des·zu·löschenden·Proxy-Service·auf·das·Symbol·Löschen.· Der·Proxy-Service·wird·in·der·aktuellen·Session·gelöscht.· Wenn andere Ressourcen diese Ressource referenzieren, wird das Symbol Warnung wegen Löschvorgang angezeigt. Sie können die Ressource löschen, indem Sie den Löschvorgang bestätigen. Dies kann aufgrund von nicht aufgelösten Referenzen mit der gelöschten Ressource zu Konflikten führen.
  4. Um die Session zu beenden und die Konfiguration für die Laufzeit bereitzustellen, klicken Sie unter Change Center auf Aktivieren.
