Binding von Dynamic XQuery-Ausdrücken an Inline XQueries

Die Seite XQuery-Ausdruckseditor ermöglicht Ihnen, einen dynamischen XQuery-Ausdruck anzugeben, dessen Auswertung bei Laufzeit den Namen einer vorregistrierten XQuery-Ressource ergibt. Mehr Informationen über diesen Editor finden Sie unter Inline XQuery- und XPath-Ausdrücke erstellen und bearbeiten.

So definieren Sie einen dynamischen XQuery-Ausdruck

  1. Wählen Sie die Option Dynamische XQuery.
  2. Geben Sie im Bereich Ausdruck für XQuery-Ressource eingeben den XQuery-Ausdruck ein, dessen Auswertung bei Laufzeit den Namen einer vorregistrierten XQuery-Ressource ergibt.

    3. Nachfolgend die Syntax für die XQuery-Ressource (steht für den vollen Namen der Ressource)

    Project/folder1/folder2/XQueryResourceName

  3. Im Bereich Variablen mit XQuery-Vorlage für benutzerdefinierte Variablen binden geben Sie die Eingabeparameter für die Transformation an.
    • Klicken Sie auf "Durchsuchen" im Feld XQuery-Vorlage wählen, um eine vorhandene registrierte Ressource auszuwählen, die als Vorlage für die Form der Abfrage dient (Nummer und Namen der Variablen). Nach der Auswahl einer Vorlage werden die Variablen im Bereich Variablen binden angezeigt. Beachten Sie, dass die Vorlage nicht mit der Konfiguration besteht. Stattdessen dient die Vorlage als Schnellstart, um Ihnen dabei zu helfen, die Variablen für die Abfrage festzulegen.
    • Geben Sie im Feld benutzerdefinierte Variable hinzufügen einen Variablennamen ein, und klicken Sie auf "Hinzufügen". Geben Sie für jede unter Variablenname aufgelistete Variable einen XQuery-Ausdruck ein, der ihr zugewiesen wird. Sie müssen für jeden Parameter eine Zuweisung definieren. Wenn beispielsweise eine XQuery-Transformation zwei Eingabeparameter mit den Namen one und two hat, hat das Feld Variablenname zwei Labels -one und two. Ein Textfeld, in das der XQuery-Ausdruck eingegeben wird, ist mit jedem Label verknüpft.

      • Folgende XQuery-Ausdrücke sind Beispiel für gültige Eingaben in die Variablenfelder:

      $body/*[1]

      $body/po:PurchaseOrder

  4. Wenn Sie fertig sind, fahren Sie mit einer der folgenden Aufgaben fort.
    • Klicken Sie auf Validieren. Eine Nachricht wird angezeigt, wenn der Ausdruck erfolgreich validiert wurde.
    • Klicken Sie auf Test. Siehe XQuery-Testen.
    • Speichern oder verwerfen Sie Ihre Änderungen.
