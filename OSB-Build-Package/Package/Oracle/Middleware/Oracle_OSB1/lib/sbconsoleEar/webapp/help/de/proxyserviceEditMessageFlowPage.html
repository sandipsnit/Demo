Seite "Nachrichtenfluss bearbeiten"

Verwenden Sie die Seite Nachrichtenfluss bearbeiten , um einen Nachrichtenfluss für einen Proxy-Service zu erstellen.

Im linken Fenster der Seite Nachrichtenfluss bearbeiten wird eine Baumansicht der Knoten und Objekte im Nachrichtenfluss angezeigt. Wenn die Einzelheiten eines Objekts auf einer eigenen Seite definiert wurden, können Sie auf den Namen des Objekts klicken, um die verbundene Seite anzuzeigen.

Im rechten Fenster gibt es ein Feld, in dem Sie den Nachrichtenfluss erstellen können. Wenn noch kein Nachrichtenfluss definiert wurde, enthält das Fenster ein einzelnesProxy-Service-Symbol, das den Startknoten für den Service kennzeichnet. Klicken Sie auf das Symbol, um Pipeline-Paarknoten, Route-Knoten, bedingte Verzweigungsknoten, Vorgangsverzweigungen und Error Handling für den Service hinzuzufügen.

Wenn Sie der Seite Objekte hinzufügen, werden auf der Seite Symbole angezeigt, die die Objekte darstellen. Die Beziehungen unter den Objekten werden mit Linien und Verbindungsfeldern angezeigt. Klicken Sie auf ein Symbol auf der Seite Nachrichtenfluss bearbeiten , um ein Menü der Aktionen anzuzeigen, die Sie für dieses Objekt ausführen können. Die Optionen, die im Menü verfügbar sind, können je nach Kontext abweichen. Siehe Tabelle 20-1, sie enthält eine komplette Liste der Symbole und Optionen.
