Seite "Stage-Konfiguration bearbeiten"

Auf der Seite Stage-Konfiguration bearbeiten können Sie Aktionen zu Pipeline Stages, Error Handler Stages und Route-Knoten in einem Nachrichtenfluss hinzufügen.

  • Wenn auf der Seite Stage-Konfiguration bearbeiten noch keine Definitionen vorliegen, wird das Symbol Aktion hinzufügen als einziges Objekt angezeigt. Klicken Sie zum Starten auf dieses Symbol.
  • Wenn bereits eine Stage oder ein Route-Knoten konfiguriert wurde, werden die für diese Stage oder diesen Route-Knoten definierten Aktionen und Objekte auf der Seite angezeigt. Sie können die vorhandenen Aktionen gegebenenfalls bearbeiten oder auf eines der Symbole für Aktionen klicken, um weitere Aktionen zur Stage hinzuzufügen.

Anweisungen zum Arbeiten mit allen Arten von Aktionen, die Sie zu einer Stage hinzufügen können, finden Sie unter Aktionen im Nachrichtenfluss hinzufügen und bearbeiten.
