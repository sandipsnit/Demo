XQuery-Vergleichsausdruck mit Builder Option eingeben

Bevor Sie beginnen

Diese Anweisungen gehen davon aus, dass Sie einen XQuery-Bedingungsausdruck im XQuery-Bedingungseditor erstellen oder bearbeiten, wie in Inline XQuery- und XPath-Ausdrücke erstellen und bearbeiten beschrieben.

So geben Sie einen XQuery-Vergleichsausdruck ein:

  1. Wählen Sie im XQuery-Bedingungseditor Builder (zu finden unter der Arbeitsflächenschaltfläche), wenn dies nicht bereits eine gewählte Option ist.
  2. Wählen Sie im Feld Expression Builder sofern noch nicht gewählt Vergleichsausdruck.
  3. Geben Sie im Feld Operand eine Kontextvariable, Namespace-Definition oder XQuery-Funktion ein.

    4. Um den Operanden zu erstellen, können Sie XQuery-Funktionen aus der XQuery-Funktions-Palette und Namespaces aus der Namespace-Definitions-Palette einfügen. Siehe Schritt 3 und Schritt 4 in Ausdrücke in den Textfeldern des Editor Workspace erstellen.

  4. Wählen Sie in der Liste Operator einen Vergleichsoperator aus.
  5. Geben Sie im Feld Wert Text oder eine Kontextvariable ein.

    6. Sie müssen Text in Anführungszeichen eingeben - zum Beispiel ist "true" gültig, aber true nicht.

  6. Klicken Sie auf Hinzufügen. Der Text den Sie eingegeben haben, wird im Fenster Ausdrücke angezeigt.
  7. Wiederholen Sie die Schritte 3-6, um zusätzliche Bedingungen zu erstellen. Jede Bedingung wird am Ende der Bedingungsliste hinzugefügt.

    8. Beachten Sie Folgendes, wenn Sie mehrere Bedingungen verwenden:

    • Wenn Sie zusätzliche Bedingungen erstellen, vergewissern Sie sich, dass die Optionen And oder Or im Feld Konjunktion gewählt wurden.
    • Sie können eine Bedingung wählen und auf den Pfeil Nach oben klicken, damit diese in der Liste der Bedingungen nach oben verschoben wird, oder Sie klicken auf den Pfeil Nach unten, um sie nach unten zu verschieben. Sie können auch auf das Symbol Bearbeiten klicken, um eine Bedingung zu aktualisieren, oder Sie klicken auf Löschen, um sie zu löschen.
    • Unäre Ausdrücke können in der Gesamtdefinition einer Bedingung mit Vergleichsausdrücken vermischt werden.
  8. Führen Sie optional einen oder beide der folgenden Schritte aus:
    • Klicken Sie auf Validieren. Eine Nachricht wird angezeigt, wenn der Ausdruck erfolgreich validiert wurde.
    • Um den Ausdruck zu testen, klicken Sie auf Testen. Siehe XQuery-Tests durchführen.
  9. Klicken Sie auf Speichern, um den Editor zu schließen und den Ausdruck in die Aktion einzufügen.

Wenn Sie fertig sind

Setzen Sie die Konfiguration der Aktion fort, wie in Aktionen im Nachrichtenfluss hinzufügen und bearbeiten erläutert.
