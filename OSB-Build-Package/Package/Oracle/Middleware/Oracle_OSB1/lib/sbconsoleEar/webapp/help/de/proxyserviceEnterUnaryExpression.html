Unäre Ausdrücke mit der Option "Builder" eingeben

Bevor Sie beginnen

Diese Anweisungen gehen davon aus, dass Sie einen XQuery-Bedingungsausdruck im XQuery-Bedingungseditor erstellen oder bearbeiten, wie in Inline XQuery- und XPath-Ausdrücke erstellen und bearbeiten beschrieben.

So geben Sie einen XQuery-Vergleichsausdruck ein:

  1. Wählen Sie im XQuery-Bedingungseditor Builder (zu finden unter der Arbeitsflächenschaltfläche), wenn dies nicht bereits eine gewählte Option ist.
  2. Wählen Sie im Feld Expression Builder den Eintrag Unärer Ausdruck, wenn er noch nicht gewählt ist.
  3. Aktivieren Sie das Kontrollkästchen Nicht, um einen negativen Ausdruck zu erstellen, oder lassen Sie das Kontrollkästchen deaktiviert.
  4. Geben Sie im Feld Ausdruck eine Kontextvariable, eine Namespace-Definition oder eine XQuery-Funktion ein.

    5. Zum Erstellen des Ausdrucks können Sie XQuery-Funktionen aus der Palette XQuery-Funktionen und Namespaces aus der Palette Namespace-Definitionen einfügen. Weitere Informationen finden Sie unter Schritt 3 und Schritt 4 in Ausdrücke in den Textfeldern des Editor Workspace erstellen.

  5. Klicken Sie auf Hinzufügen, um den Text dem Bereich Ausdrücke hinzuzufügen.
  6. Wiederholen Sie die Schritte 3 bis 5, um weitere Bedingungen zu erstellen. Jede zusätzliche Bedingung wird am Ende der Liste mit Bedingungen hinzugefügt.

    7. Beachten Sie beim Erstellen von unären Ausdrücken folgende Hinweise:

    • Wenn Sie zusätzliche Bedingungen erstellen, vergewissern Sie sich, dass die Optionen And oder Or im Feld Konjunktion gewählt wurden.
    • Sie können eine Bedingung wählen und auf den Pfeil Nach oben klicken, um sie in der Liste nach oben zu verschieben, oder auf den Pfeil Nach unten klicken, um sie in der Liste nach unten zu verschieben. Klicken Sie auf das Symbol Bearbeiten, um die Bedingung zu aktualisieren, oder auf das Symbol Löschen, um sie zu löschen.
    • Unäre Ausdrücke können in der Gesamtdefinition einer Bedingung mit Vergleichsausdrücken vermischt werden.
  7. Führen Sie optional einen oder beide der folgenden Schritte aus:
    • Klicken Sie auf Validieren. Eine Nachricht wird angezeigt, wenn der Ausdruck erfolgreich validiert wurde.
    • Um den Ausdruck zu testen, klicken Sie auf Testen. Siehe XQuery-Tests durchführen.
  8. Klicken Sie auf Speichern, um den Editor zu schließen und den Ausdruck in die Aktion einzufügen.

Wenn Sie fertig sind

Setzen Sie die Konfiguration der Aktion fort, wie in Aktionen im Nachrichtenfluss hinzufügen und bearbeiten erläutert.
