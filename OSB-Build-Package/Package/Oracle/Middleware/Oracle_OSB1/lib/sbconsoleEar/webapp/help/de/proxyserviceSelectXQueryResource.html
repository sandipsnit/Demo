Externe XQuery-Ressourcen an eingebettete XQuerys binden

Sie können externe XQuery-Ressourcen an XQuery-Ausdrücke binden, sodass diese als Teil einer Aktion inline ausgeführt werden.

Bevor Sie beginnen

Bei diesen Anweisungen wird davon ausgegangen, dass Sie einen Ausdruck im XQuery-Ausdruckseditor, XQuery-Bedingungseditor oder im XPath-Ausdruckseditor erstellen oder bearbeiten, wie in Inline-XQuery- und XPath-Ausdrücke erstellen und bearbeiten erläutert.

So binden Sie eine XQuery-Ressource an einen eingebetteten Ausdruck:

  1. Klicken Sie auf Variablenstruktur.
  2. Wählen Sie im Workspace (unter der Schaltflächenleiste) die Option XQuery-Ressourcen.
  3. Klicken Sie im Feld 1. Auszuführende XQuery-Ressource wählen auf Durchsuchen.
  4. Wählen Sie im Feld XQuery-Browser das mit der zu verwendenden XQuery verknüpfte Optionsfeld. Klicken Sie dann auf Weiterleiten.
  5. Legen Sie im Feld 2. Variablen binden die Eingabeparameter für die Transformation fest. Geben Sie für jede Variable in der Liste Variablenname einen XQuery-Ausdruck an, der mit der Variablen verknüpft werden soll. Für jeden Parameter müssen Sie zudem eine Zuordnung festlegen. Beispiel: Wenn eine XQuery-Transformation zwei Eingabeparameter folgenden Namens one und two hat, hat auch das Feld Variablenname die beiden Label one und two. Jedem Label ist ein Textfeld zugeordnet, in das Sie den XQuery-Ausdruck eingeben.

    6. Die folgenden XQuery-Ausdrücke sind Beispiele für gültige Eingabe in dieses Feld:

    $body/*[1]

    $body/po:PurchaseOrder

    Hinweis: Der folgenden Variablenname ist keine gültige Eingabe in diesem Feld und verursacht eine Exception:
    body.
  6. Wenn Sie fertig sind, fahren Sie mit einer der folgenden Aufgaben fort.
    • Klicken Sie auf Validieren. Eine Nachricht wird angezeigt, wenn der Ausdruck erfolgreich validiert wurde.
    • Klicken Sie auf Test. Siehe XQuery-Testen.
    • Speichern oder verwerfen Sie Ihre Änderungen.
