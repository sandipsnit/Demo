Externe XSLT-Ressourcen an Inline XQuerys binden

Auf der Seite XQuery-Ausdruckseditor können Sie eine XSLT-Ressource für die Ausführung wählen. Weitere Informationen zu diesem Editor finden Sie unter Inline XQuery und XPath-Ausdrücke erstellen und bearbeiten.

So wählen Sie eine XSLT-Ressource für die Ausführung:

  1. Wählen Sie die Option XSLT-Ressourcen.
  2. Klicken Sie unter dem Feld Wählen Sie eine auszuführende XSLT-Ressource auf das Symbol XSLT-Browser.
  3. Aktivieren Sie im XSLT-Browser das Optionsfeld, das der auszuführenden XSLT zugeordnet ist, und klicken Sie auf Weiterleiten.
  4. Unter dem Feld Bindevariablen werden ein Label und ein entsprechendes Textfeld für die einzelnen Eingabeparameter der Transformation angezeigt. Jedes Label entspricht dem Namen eines Parameters, und in den einzelnen Textfeldern können Sie einen XQuery-Ausdruck definieren, der dem Parameter zugeordnet werden soll. Sie müssen eine Zuordnung für jeden Parameter definieren. Beispiel: Wenn eine XSL-Transformation die beiden Eingabeparameter one und two aufweist, hat das Feld Variablenzuordnung zwei Labels, nämlich one und two. Dabei ist jedem Label ein Textfeld zugeordnet, in dem der XQuery-Ausdruck eingegeben werden kann. Neben der Zuordnung für die einzelnen Eingabevariablen müssen Sie auch einen XQuery-Ausdruck für das Eingabedokument zur Transformation angeben. Die Zuordnung wird in dem Textfeld mit dem Label Eingabedokument angegeben.

    5. Die folgenden XQuery-Ausdrücke sind Beispiele für gültige Eingabe in dieses Feld:

    $body/*[1]

    $body/po:PurchaseOrder

    Hinweis: Der folgenden Variablenname ist keine gültige Eingabe in diesem Feld und verursacht eine Exception:
    body.
  5. Fahren Sie mit einer der folgenden Tasks fort:
    • Namespaces zur Verwendung in Inline-Ausdrücken erstellen
    • Variablenstrukturen in den XQuery-Editoren erstellen
    • Ausdrücke in den Textfeldern des Editor Workspace erstellen
    • Externe XQuery-Ressourcen an eingebettete XQuerys binden
    • Klicken Sie auf Validieren. Eine Nachricht wird angezeigt, wenn der Ausdruck erfolgreich validiert wurde.
    • Klicken Sie auf Test. Siehe XQuery-Testen.
