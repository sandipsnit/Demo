Seite "Proxy-Service anzeigen - Konfigurationsdetails"

Auf der Seite Proxy-Service anzeigen - Konfigurationsdetails werden die Konfigurationsdetails zu einem Proxy-Service angezeigt. In Tabelle 19-19 werden alle Eigenschaften beschrieben, die auf dieser Seite angezeigt werden können. (Die Eigenschaften können je nach den Details zum Proxy-Service unterschiedlich sein.)

Die Kategorien auf dieser Seite entsprechen den Seiten Proxy-Service erstellen/bearbeiten, die zum Erstellen und Bearbeiten von Proxy-Service-Konfigurationen verwendet werden. Diese Konfigurationen werden unter Seite "Proxy-Service erstellen/bearbeiten - Referenz" beschrieben.

Klicken Sie auf den Link Bearbeiten neben den einzelnen Kategorienamen, um die zugehörige Konfigurationsseite anzuzeigen.
