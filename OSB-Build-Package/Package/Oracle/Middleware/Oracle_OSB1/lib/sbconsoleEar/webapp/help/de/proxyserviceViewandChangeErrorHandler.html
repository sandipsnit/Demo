Error Handler bearbeiten

Bevor Sie beginnen

Bei diesen Anweisungen wird vorausgesetzt, dass Sie bereits einen Nachrichtenfluss auf der Seite "Nachrichtenfluss bearbeiten" bearbeiten. Weitere Informationen finden Sie unter Nachrichtenfluss erstellen, bearbeiten und anzeigen.

So zeigen Sie Error Handler an und bearbeiten diese

  1. Führen Sie einen der folgenden Schritte aus:

    2. Tabelle 23-1 Error Handler anzeigen und bearbeiten 
    Ziel
    Führen Sie diesen Schritt durch:
    Error Handler des Proxy-Service-Nachrichtenflusses anzeigen und bearbeiten
    Klicken Sie auf das Symbol Proxy-Service. Klicken Sie dann auf Service Error Handler bearbeiten. Die Seite Error Handler bearbeiten wird angezeigt. Siehe Proxy-Service Error Handler hinzufügen.
    Pipeline Error Handler anzeigen und ändern
    Klicken Sie entweder auf das Symbol Anforderungs-Pipeline oder auf Antwort-Pipeline. Klicken Sie dann auf Pipeline Error Handler bearbeiten. Die Seite Error Handler bearbeiten wird angezeigt. Siehe Pipeline Error Handler hinzufügen.
    Route-Knoten-Error Handler anzeigen und ändern
    Klicken Sie auf das Symbol Route-Knoten. Klicken Sie dann auf Route Error Handler bearbeiten. Die Seite Error Handler bearbeiten wird angezeigt. Siehe Route-Knoten-Error Handler hinzufügen.
    Stage Error Handler anzeigen und ändern
    Klicken Sie auf das Symbol Stage. Klicken Sie dann auf Stage Error Handler bearbeiten. Die Seite Error Handler bearbeiten wird angezeigt. Siehe Stage Error Handler hinzufügen.

Wenn Sie fertig sind

Fahren Sie auf der Seite Nachrichtenfluss bearbeiten mit dem Erstellen des Nachrichtenflusses fort. Weitere Informationen finden Sie unter Nachrichtenfluss erstellen, bearbeiten und anzeigen.
