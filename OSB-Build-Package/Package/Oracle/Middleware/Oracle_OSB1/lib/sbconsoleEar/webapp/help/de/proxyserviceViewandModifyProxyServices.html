Proxy-Service-Konfigurationen bearbeiten

Dieser Abschnitt beschreibt das Bearbeiten von Proxy-Service-Konfigurationen.

  1. Wenn dies noch nicht geschehen ist, klicken Sie zum Erstellen einer neuen Session auf Erstellen oder zum Aufrufen einer vorhandenen Session auf Bearbeiten. Weitere Informationen finden Sie unter Change Center verwenden.
  2. Suchen Sie den Proxy-Service, den Sie anzeigen oder bearbeiten möchten. Siehe Proxy-Service suchen.
  3. Klicken Sie auf den Proxy-Service-Namen. Die Seite Proxy-Service anzeigen - Konfigurationsdetails zeigt Konfigurationsinformationen für den gewählten Proxy-Service an.
  4. Gehen Sie wir folgt vor, um Einstellungen anzuzeigen oder zu ändern:
    • Klicken Sie auf das Symbol Bearbeiten neben dem Namen der Kategorie, deren Eigenschaften Sie anzeigen oder bearbeiten möchten (zum Beispiel Allgemeine Konfiguration, Transport-Konfiguration). Ob Sie die Seiten bearbeiten können hängt davon ab, was für Seiten Sie bei der Erstellung des Proxy-Service konfiguriert haben.

      • Eine Liste sämtlicher Seiten finden Sie unter Proxy-Service anzeigen - Seite "Konfigurationsdetails".

    • Klicken Sie auf Bearbeiten unten auf der Seite, um die Seite Proxy-Service erstellen/bearbeiten - Allgemeine Konfiguration anzuzeigen, welche die erste Seite einer Reihe von Seiten für die Konfiguration dieses Proxy-Service ist.
  5. Fahren Sie mit Anzeigen und Bearbeiten fort, wie unter Proxy-Service erstellen und konfigurieren beschrieben.
  6. Auf der Seite Proxy-Service bearbeiten - Zusammenfassung klicken Sie auf Speichern, um die Aktualisierungen in der aktuellen Session festzuschreiben.
  7. Um die Session zu beenden und die Konfiguration für die Laufzeit bereitzustellen, klicken Sie unter Change Center auf Aktivieren.
