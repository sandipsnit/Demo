Berichtmeldungen anzeigen

Verwenden Sie die Seite Nachrichtenberichte - Zusammenfassung , um Nachrichtenberichtinformation anzuzeigen, wie in Tabelle 27-2 dargestellt. Die Seite zeigt bis zu 100 Nachrichten an.

  1. So beschränken Sie die Anzahl der Elemente in der Liste oder finden eine bestimmte Nachricht.
    • Klicken Sie auf das Symbol Öffnen, um zusätzliche Suchfilter anzuzeigen. Wenn Sie mehrere Suchkriterien angeben, werden nur Ergebnisse angezeigt, die allen Kriterien entsprechen.
    • Führen Sie zum Filtern der Parameter einen oder mehrere der folgenden Schritte aus, dargestellt in Tabelle 27-1.

      • Tabelle 27-1 Nachrichtenfilterparameter 
      Ziel
      Vorgehensweise
      Filtern nach Zeitparametern
      Einstellen eines Zeitraums:
      1. Aktivieren Sie das Aktionsfeld Nachrichten von... An... .
      2. Sie können im Feld Nachrichten von ein Datum und die Zeit im Format MM/TT/JJ HH:MM:SS AM|PM eingeben. Offlinesetzen können Sie eingeben 10/10/07 12:45:00 AM.
      3. Im Feld Nachricht an geben Sie Datum und Zeit im Format MM/TT/JJ HH:MM:SS AM|PM ein.
      Oder Sie wählen Parameter für die aktuellsten Nachrichten aus:
      1. Aktivieren Sie das Aktionsfeld In den letzten.
      2. Geben Sie im Feld In den letzten die Anzahl der Tage ein, und geben Sie dann Stunden und Minuten an.
      Hinweis: Standardmäßig werden Nachrichtenberichte der letzten 30 Minuten angezeigt.
      Nach Service-Name filtern
      Geben Sie im Feld Eingehender Service-Name den Service-Namen an.
      Filtern nach Fehlercode
      Geben Sie im Feld Fehlercode einen Fehlercode ein.
      Filtern nach Berichtsindex
      Geben Sie im Feld Berichtsindex die Berichtsindizes ein. Es handelt sich dabei um die Schlüsselwertpaare, die Sie definiert haben, als Sie die Berichtaktion für diesen Proxy-Service konfiguriert haben. Dies ist eine Zeichensatzsuche.

    • Sie können Platzhalterzeichen in Service-Name-, Fehlercode- und Berichtindex-Feldsuchen verwenden.
      • Wenn Sie nur einen Asterisk (*) eingeben, werden alle Nachrichten zurückgegeben. Wenn Sie zum Beispiel , * im Feld Fehlercode eingeben, werden Nachrichten mit und ohne Fehlercodes angezeigt.
      • Im Fehlercode-Feld nur gelten strikte Sucheinschränkungen. Wenn Sie beispielsweise Nachrichten mit dem Fehlercode BEA12345 suchen, können Sie *, BE* oder BEA12345 eingeben.
  2. Klicken Sie auf Suchen. Die Seite zeigt Nachrichten an, die unter die von Ihnen festgelegten Filterparameter fallen. Die maximale Anzahl von Nachrichten, die beim Filtern der Nachrichten angezeigt wird, ist 1000.
  3. Klicken Sie auf Alle anzeigen, um die Suchfilter zu entfernen und alle Nachrichten anzuzeigen.
  4. Sie können eine Nachricht auch wie folgt suchen:
    • Sortieren Sie die Nachrichtenliste erneut. Klicken Sie auf einen unterstrichenen Spaltennamen. Pfeile, die nach oben oder unten zeigen, geben die Sortierrichtung an. Klicken Sie auf den Spaltennamen, um die Sortierung zu ändern.
    • Rollen Sie durch die Seiten. Verwenden Sie die Seitensteuerung über oder unter der Tabelle. Gehen Sie über Auswahl der Seitennummer auf eine Seite, oder verwenden Sie die Pfeiltasten, um zur nächsten, vorigen, ersten oder letzten Seite zu gelangen.

      • Tabelle 27-2 Nachrichten (Berichterstellung) Zusammenfassungsdetails 
      Eigenschaft
      Beschreibung
      Berichtsindex
      Ein Link auf die Seite Nachrichtendetails anzeigen. Siehe Nachrichtendetails anzeigen.
      Berichtindizes sind Schlüsselwertpaare, die Sie verwenden, um Schlüssel-IDs von einer Nachrichtenkontextvariablen zu extrahieren. Schlüssel sind für das Filtern von Nachrichten praktisch. Sie verwenden die Berichtaktion, um Schlüsselwertpaare hinzuzufügen. Siehe Aktionen im Nachrichtenfluss hinzufügen und bearbeiten.
      DB-Zeitstempel
      Datum und Zeit, wann die Nachricht der Datenbank hinzugefügt wurde.
      Eingehender Service
      Der der Nachricht zugewiesene eingehende Service. Der Service ist ein Link auf die Seite Proxy-Service anzeigen - Konfigurationsdetails. Siehe Proxy-Service-Konfigurationen bearbeiten.
      Fehlercode
      Der der Nachricht zugewiesene Fehlercode, sofern vorhanden.
      Sie geben bei der Konfiguration der Auslösefehleraktion Fehlercodes an. Siehe "Umgang mit Fehlern im Nachrichtenfluss" im Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus unter http://www.oracle.com/pls/as111170/lookup?id=OSBAG232 und Proxy-Service Error-Handler hinzufügen.
