Nachrichten löschen

Sie verwenden diese Seite, um gewählte Nachrichten aus dem Berichtsdatenspeicher zu löschen.

Das Löschen von Nachrichten erfolgt im Hintergrund. Auf diese Weise wird die Seite Nachrichtenberichtsübersicht nicht gesperrt, solange der Löschvorgang läuft. Sie können daher einen Löschvorgang vornehmen, der länger dauert. Die Dauer hängt davon ab, wie viele Nachrichten gelöscht werden müssen. Wenn Sie jedoch Berichtsnachrichten filtern, verlangsamt dies das Löschen. Auf der Seite Nachrichtenberichtsübersicht können falsche Daten angezeigt werden, da einige Daten noch nicht gelöscht wurden.

Das Feature zum Löschen von Nachrichten kann während der Entwurfs- und Testphase des Projekts hilfreich sein. In der Production-Umgebung empfiehlt Oracle jedoch eine Verwaltung der Daten in der Datenbank (einschließlich Löschen) durch den Datenbankadministrator mithilfe von Datenbankverwaltungstools.

  1. Wählen Sie Vorgänge > Nachrichten löschen.
  2. Führen Sie einen der folgenden Schritte aus Tabelle 27-9 aus.

    3. Tabelle 27-9 Nachrichtenlöschoptionen 
    Ziel
    Vorgehensweise
    Alle Meldungen löschen
    1. Wählen Sie das Optionsfeld Alle Meldungen löschen.
    2. Klicken Sie auf Löschen.
    Nachrichten innerhalb eines bestimmten Zeitrahmens löschen
    1. Wählen Sie das Optionsfeld Nachrichten löschen von ... bis ... .
    2. Geben Sie im Feld Nachrichten löschen von einen Monat, einen Tag, ein Jahr und eine Uhrzeit im Format MM/TT/JJ HH:MM:SS ein. Beispiel: 10/10/07 12:45:00 AM.
    3. Geben Sie im Feld Nachrichten löschen bis einen Monat, einen Tag, ein Jahr und eine Uhrzeit im Format MM/TT/JJ HH:MM:SS ein.
    4. Klicken Sie auf Löschen.
