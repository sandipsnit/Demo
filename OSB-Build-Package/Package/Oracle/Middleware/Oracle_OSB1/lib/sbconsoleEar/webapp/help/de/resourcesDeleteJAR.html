JARs löschen

  1. Wenn dies noch nicht geschehen ist, klicken Sie zum Erstellen einer neuen Session auf Erstellen oder zum Aufrufen einer vorhandenen Session auf Bearbeiten. Weitere Informationen finden Sie unter Change Center verwenden.
  2. Wählen Sie Ressourcenbrowser > JARs.
  3. Klicken Sie auf das Symbol Löschen im Feld Optionen der zu löschenden JAR . Die JAR wird in der aktuellen Session gelöscht. Eine Löschwarnung wird angezeigt, wenn andere Ressourcen auf diese Ressource verweisen. Sie können die Ressource mit einer Warnbestätigung löschen. Dies kann zu Konflikten aufgrund ungelöster Referenzen auf die gelöschte Ressource führen.
  4. Um die Session zu beenden und die Konfiguration für die Laufzeit bereitzustellen, klicken Sie unter Change Center auf Aktivieren.
