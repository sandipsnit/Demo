Alert-Ziele bearbeiten

  1. Wenn dies noch nicht geschehen ist, klicken Sie zum Erstellen einer neuen Session auf Erstellen oder zum Aufrufen einer vorhandenen Session auf Bearbeiten. Weitere Informationen finden Sie unter Change Center verwenden.
  2. Suchen Sie das Alert-Ziel wie unter ###Alert-Ziele ermitteln oder ###Alert-Ziele suchen beschrieben.
  3. Klicken Sie auf den Namen des Alert-Ziels. Auf der Seite Alert-Ziel anzeigen – Konfiguration werden die in Tabelle 5-2 beschriebenen Informationen angezeigt.

    4. Tabelle 5-2 Alert-Ziel – Konfigurationsdetails 
    Eigenschaft
    Beschreibung
    Zuletzt geändert von
    Der Benutzer, der das Alert-Ziel erstellt oder bearbeitet hat
    Zuletzt geändert am
    Datum und Uhrzeit des Zeitpunkts, an dem der Benutzer dieses Alert-Ziel erstellt oder bearbeitet hat. Klicken Sie auf den Link mit Datum und Uhrzeit, um die Änderungshistorie dieser Ressource anzuzeigen. Weitere Informationen finden Sie unter Seite "Änderungshistorie anzeigen".
    Referenzen
    Die Anzahl der Objekte, die dieses Alert-Ziel referenziert. Wenn solche Referenzen vorhanden sind, klicken Sie auf den Link mit der Anzahl, um eine Liste mit allen Objekten anzuzeigen. Weitere Informationen finden Sie unter Referenzen auf Ressourcen anzeigen.
    Referenziert von
    Die Anzahl der Objekte, die dieses Alert-Ziel referenzieren. Wenn solche Referenzen vorhanden sind, klicken Sie auf den Link mit der Anzahl, um eine Liste mit allen Objekten anzuzeigen. Weitere Informationen finden Sie unter Referenzen auf Ressourcen anzeigen.
    Beschreibung
    Eine Beschreibung des Alert-Ziels, sofern vorhanden.
    SNMP-Trap
    "Ja" oder "Nein" gibt an, ob die Option "SNMP-Trap" bei der Konfiguration dieses Alerts als Ziel für den Alert festgelegt wurde oder nicht.
    Berichte
    "Ja" oder "Nein" gibt an, ob die Option "Reporting" bei der Konfiguration dieses Alerts als Ziel für den Alert festgelegt wurde oder nicht.
    Alert Logging
    "Ja" oder "Nein" gibt an, ob Alert Logging aktiviert ist oder nicht.
    E-Mail und JMS
    Es wird eine Liste mit E-Mail-Empfängern oder JMS-Zielen oder beiden angezeigt, sofern diese konfiguriert wurden.

  4. Sie können die Einstellungen für SNMP-Trap, Reporting und Alert Logging ändern.
  5. Wenn Sie einen E-Mail-Empfänger oder ein JMS-Ziel löschen möchten, klicken Sie unten auf der Seite auf Bearbeiten.
    1. Wenn Sie den E-Mail-Empfänger ändern möchten, klicken Sie unter E-Mail unter Optionen auf das Symbol Bearbeiten für den zu bearbeitenden E-Mail-Empfänger. Auf der Seite E-Mail-Empfänger bearbeiten können Sie Informationen zum E-Mail-Empfänger ändern. Weitere Informationen finden Sie unter E-Mail-Empfänger hinzufügen.
    2. Wenn Sie das JMS-Ziel ändern möchten, klicken Sie unter JMS unter Optionen auf das Symbol Bearbeiten für das zu bearbeitende JMS-Ziel. Auf der Seite JMS-Ziel bearbeiten können Sie Informationen zum JMS-Ziel ändern. Weitere Informationen finden Sie unter E-Mail-Empfänger hinzufügen.
    3. Wenn Sie ein(en) oder alle E-Mail-Empfänger oder JMS-Ziele löschen möchten, klicken Sie unter Optionen auf das Symbol Löschen für die zu löschenden E-Mail-Empfänger oder JMS-Ziele.
  6. Klicken Sie auf Speichern, um die Aktualisierungen in der aktuellen Session zu speichern.
  7. Um die Session zu beenden und die Konfiguration für die Laufzeit bereitzustellen, klicken Sie unter Change Center auf Aktivieren.
