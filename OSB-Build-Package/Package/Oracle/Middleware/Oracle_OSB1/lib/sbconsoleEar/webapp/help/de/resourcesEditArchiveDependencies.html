JAR-Abhängigkeiten bearbeiten

In Oracle Service Bus können JAR-Ressourcen Klassen enthalten, die von anderen Klassen in anderen JARs abhängen. Fügen Sie über die Seite JAR-Abhängigkeiten bearbeiten referenzierte JAR-Ressourcen hinzu, oder entfernen Sie diese.

  1. Wenn dies noch nicht geschehen ist, klicken Sie zum Erstellen einer neuen Session auf Erstellen oder zum Aufrufen einer vorhandenen Session auf Bearbeiten. Weitere Informationen finden Sie unter Change Center verwenden.
  2. Suchen Sie die JAR, wie in JARs suchen beschrieben.
  3. Klicken Sie auf den Namen der JAR.

    4. Auf der Seite JAR-Ressource anzeigen - JAR-Konfiguration gibt eine numerische Anzeige in der Registerkarte Abhängigkeiten die Anzahl der aktualisierten JARs an, die für diese JAR-Ressource konfiguriert wurden, beziehungsweise eine null, wenn keine JARs vorhanden sind.

  4. Öffnen Sie die Registerkarte Abhängigkeiten, um die referenzierten JARs anzuzeigen oder JARs hinzuzufügen.

    5. Die Seite der JAR-Abhängigkeiten enthält Informationen, die in Tabelle 7-4 aufgeführt sind.

    Tabelle 7-4 Informationen zu JAR-Abhängigkeiten 
    Eigenschaft
    Beschreibung
    Name
    Name der referenzierten JAR. Der Name ist ein Link zur Seite JAR-Ressource anzeigen - JAR-Konfiguration. Siehe JARs bearbeiten.
    Pfad
    Der Pfad ist der Projektname und der Name des Ordners, in dem die JAR abgelegt ist.

  5. Um referenzierte JARs hinzuzufügen oder zu löschen, klicken Sie auf Bearbeiten.

    6. Die Seite JAR-Abhängigkeiten bearbeiten enthält Informationen, die in Tabelle 7-5 aufgeführt sind.

    Tabelle 7-5 Details zu JAR-Abhängigkeiten
    Eigenschaft
    Beschreibung
    Manifest-Classpath
    Classpath-Einträge in der JAR-Manifest-Datei.
    Name
    Name und Pfad der referenzierten JAR-Ressource.
    Optionen
    • Mit dem Nach-oben- und dem Nach-unten-Pfeil können Sie die Prioritätenreihenfolge der referenzierten JARs ändern.
    • Klicken Sie auf das Symbol Löschen, um eine referenzierte JAR zu löschen.

  6. Klicken Sie auf JARs hinzufügen.
  7. Suchen oder wählen Sie im JARs-Browser referenzierte JAR-Ressourcen, und klicken Sie auf Weiterleiten.

    8. Beispiel: Wenn Sie die JAR-Ressource A bearbeiten und die JAR-Ressourcen B und C hinzufügen, referenziert (hängt) JAR A (ab von) JAR B und referenziert (hängt) JAR A (ab von) JAR C.

    Mit den Pfeilen in der Spalte Optionen ordnen Sie die Hierarchie der referenzierten JARs neu an oder entfernen diese mithilfe des Symbols Löschen.

  8. Klicken Sie auf Speichern, um die Aktualisierungen in der aktuellen Session zu speichern.
  9. Um die Session zu beenden und die Konfiguration für die Laufzeit bereitzustellen, klicken Sie unter Change Center auf Aktivieren.
