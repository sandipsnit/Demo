Split Joins bearbeiten

  1. Wenn Sie es noch nicht getan haben, klicken Sie auf Erstellen, um eine neue Session zu erstellen, oder klicken Sie auf Bearbeiten, um eine vorhandene Session einzugeben. Siehe Change Center verwenden.
  2. Suchen Sie den Split Join, wie in Split Joins erläutert.
  3. Klicken Sie auf den Namen des Split Joins.

    4. Auf der Seite Split Join anzeigen - Konfiguration werden die Informationen aus der Tabelle 9-3 angezeigt.

    Tabelle 9-3 Split Join - Details 
    Eigenschaft
    Beschreibung
    Zuletzt geändert von
    Der Benutzer, der diesen Split Join erstellt oder in die Konfiguration importiert hat.
    Zuletzt geändert am
    Das Datum oder die Uhrzeit, zu der der Benutzer diesen Split Join erstellt oder in die Konfiguration importiert hat. Klicken Sie auf den Link von Datum und Uhrzeit, um die Änderungshistorie dieser Ressource anzuzeigen. Siehe Seite "Änderungshistorie anzeigen".
    Referenzen
    Die Anzahl der Objekte, die dieser Split Join referenziert. Wenn solche Referenzen vorhanden sind, klicken Sie auf den Link mit der Anzahl, um eine Liste der Objekte anzuzeigen. Siehe Referenzen auf Ressourcen anzeigen.
    Referenziert von
    Die Anzahl der Objekte, die diesen Split Join referenzieren. Wenn solche Referenzen vorhanden sind, klicken Sie auf den Link mit der Anzahl, um eine Liste der Objekte anzuzeigen. Siehe Referenzen auf Ressourcen anzeigen.
    Beschreibung
    Eine Beschreibung des Split Joins, sofern vorhanden.

  4. Sie können in einem Split Join folgende Vorgangseinstellungen vornehmen:

    5. Tabelle 9-4 Split Join-Vorgangseinstellungen
    Eigenschaft
    Beschreibung
    Logs
    Wenn Sie Log-Informationen in einem Split Join erfassen möchten, wählen Sie Logging aktivieren und anschließend die gewünschte Logging-Ebene.
    Ausführungs-Tracing
    Wenn Sie Laufzeit-Tracing aktivieren möchten, wählen Sie Aktiviert.

  5. Um die Aktualisierungen in der aktuellen Session festzuschreiben, klicken Sie auf Aktualisieren.
  6. Um die Session zu beenden und die Konfiguration für die Laufzeit bereitzustellen, klicken Sie unter Change Center auf Aktivieren.
