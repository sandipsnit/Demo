MQ-Verbindungen bearbeiten

Dieser Abschnitt beschreibt, wie Sie vorhandene MQ-Verbindungsressourcen bearbeiten.

  1. Wenn Sie es noch nicht getan haben, klicken Sie auf Erstellen, um eine neue Session zu erstellen, oder klicken Sie auf Bearbeiten, um eine vorhandene Session einzugeben. Siehe Change Center verwenden.
  2. Suchen Sie die MQ-Verbindungsressource wie unter MQ-Verbindungen suchen beschrieben.
  3. Klicken Sie auf den Namen der MQ-Verbindungsressource.

    4. Die Seite MQ-Verbindungen - Konfiguration zeigt die Informationen an, die in Tabelle 8-3 dargestellt sind.

    Tabelle 8-3 MQ-Verbindungsdetails 
    Eigenschaft
    Beschreibung
    Zuletzt geändert von
    Der Anwender, der diese MQ-Verbindungsressource erstellt oder in die Konfiguration importiert hat.
    Zuletzt geändert am
    Datum und Zeit, wann der Benutzer diese MQ-Verbindungsressource erstellt oder in die Konfiguration importiert hat. Klicken Sie auf den Datums- und Uhrzeitlink, um die Änderungshistorie dieser Ressource anzuzeigen. Siehe Seite "Änderungshistorie anzeigen".
    Referenzen
    Die Anzahl von Objekten, auf die diese MQ-Verbindungsressource verweist. Wenn solche Verweise vorhanden sind, klicken Sie auf den numerischen Link, um eine Liste der Objekte anzuzeigen. Siehe Verweise auf Ressourcen anzeigen.
    Referenziert von
    Die Anzahl von Objekten, die auf diese MQ-Verbindungsressource verweisen. Wenn solche Verweise vorhanden sind, klicken Sie auf den numerischen Link, um eine Liste der Objekte anzuzeigen. Siehe Verweise auf Ressourcen anzeigen.
    Beschreibung
    Eine Beschreibung der MQ-Verbindungsressource, sofern vorhanden.

  4. Wenn Sie eine Änderung an den Feldern vornehmen möchten, klicken Sie auf Bearbeiten. Beschreibungen der Felder finden Sie unter MQ-Verbindungen hinzufügen.

    5. Sie können das Feld Ressourcenname nicht ändern.

  5. Klicken Sie auf Speichern, um die Aktualisierungen in der aktuellen Session zu speichern.
  6. Um die Session zu beenden und die Konfiguration für die Laufzeit bereitzustellen, klicken Sie unter Change Center auf Aktivieren.
