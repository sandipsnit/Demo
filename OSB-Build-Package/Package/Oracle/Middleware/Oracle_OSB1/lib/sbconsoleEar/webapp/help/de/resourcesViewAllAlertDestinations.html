Alert-Ziele suchen

Auf der Seite Übersicht der Alert-Ziele finden Sie eine Liste der Alert-Ziele. Falls Sie bis zu diesem Zeitpunkt noch kein Alert-Ziel erstellt haben und dies jetzt tun möchten, erstellen Sie im Project Explorer eine neue Alert-Zielressource. Weitere Informationen dazu finden Sie unter Alert-Ziele hinzufügen.

  1. Führen Sie eine der folgenden Aktionen aus:
    • Wählen Sie Project Explorer, um die Seite Projektansicht oder die Seite Projekt-/Ordneransicht anzuzeigen. Navigieren Sie dann durch die Projekte und Ordner, um nach dem Alert-Ziel zu suchen.
    • Wählen Sie Ressourcenbrowser > Alert-Ziele. Die Seite Übersicht der Alert-Ziele wird angezeigt und listet die Informationen aus Tabelle 5-1 auf.
  2. Wenn Sie die Anzahl der Elemente in der Liste beschränken möchten, können Sie nach Name, Pfad oder beidem filtern. Geben Sie in den Feldern Name und Pfad den Namen und den Pfad der Suchziele ein, und klicken Sie dann auf Suchen.

    3. Der Pfad ist der Projektname und der Name des Ordners, in dem das Alert-Ziel abgelegt ist.

    Klicken Sie auf Alle anzeigen, um die Suchfilter zu entfernen und alle Alert-Ziele anzuzeigen.

    Tabelle 5-1 Details zu Alert-Zielen 
    Eigenschaft
    Beschreibung
    Name
    Name der Alert-Zielressource. Der Name ist ein Link zur Seite Alert-Ziel anzeigen - Konfiguration. Siehe Alert-Ziele bearbeiten.
    Pfad
    Der Pfad ist der Projektname und der Name des Ordners, in dem das Alert-Ziel abgelegt ist. Der Pfad ist ein Link zum Projekt oder zum Ordner, das bzw. der diese Ressource enthält. Siehe Ressourcennamen mit Projekten und Ordnern angeben.
    Optionen
    Klicken Sie auf das Symbol Löschen, um ein bestimmtes Alert-Ziel zu löschen. Siehe Alert-Ziele löschen.
