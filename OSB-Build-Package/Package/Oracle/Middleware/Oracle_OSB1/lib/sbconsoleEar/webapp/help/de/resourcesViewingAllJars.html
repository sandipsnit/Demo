JAR-Dateien suchen

  1. Führen Sie eine der folgenden Aktionen aus:
    • Wählen Sie Project Explorer, um die Seite "Projektansicht" oder die Seite "Projekt-/Ordneransicht" anzuzeigen. Durchsuchen Sie dann die Projekte und Ordner nach dem JAR-Archiv.
    • Wählen Sie Ressourcenbrowser > JAR-Dateien. Auf der Seite Übersicht der JAR-Dateien werden die in Tabelle 7-1 beschriebenen Informationen angezeigt. Eine detailliertere Beschreibung der Eigenschaften finden Sie unter JARs bearbeiten.
  2. Sie können die Anzahl der Elemente in der Liste einschränken oder nach bestimmten JARs suchen, indem Sie die JARs nach Name und Pfad filtern. Geben Sie in den Feldern Name und Pfad den Namen und Pfad des Suchziels bzw. der Suchziele ein, und klicken Sie auf Suchen.

    3. Der Pfad ist der Projektname und der Name des Ordners, in dem die JAR abgelegt ist.

    Diese Suche unterstützt Platzhalterzeichen. Beispiel: Sie können im Feld Name "j*" eingeben, damit eine Liste mit JARs angezeigt wird, die mit "j" beginnen.

    Klicken Sie auf Alle anzeigen, um die Suchfilter zu deaktivieren und alle JARs anzuzeigen.

    Tabelle 7-1 JAR-Details
    Eigenschaft
    Beschreibung
    JAR-Name
    Der eindeutige Name der JAR-Datei. Der Name ist ein Link zur Seite JAR-Ressource anzeigen – JAR-Konfiguration. Weitere Informationen finden Sie unter JARs bearbeiten.
    Pfad
    Der Pfad ist der Projektname und der Name des Ordners, in dem die JAR-Datei gespeichert ist. Er ist ein Link zu dem Projekt oder Ordner, in dem die Ressource gespeichert ist. Weitere Informationen finden Sie unter Ressourcennamen mit Projekten und Ordner angeben.
    Anzahl Klassen
    Die Anzahl der in einem JAR enthaltenen Klassen.
    Größe
    Die Größe des JAR in Bytes.
    Optionen
    Klicken Sie auf das Symbol Löschen, um eine bestimmten JAR-Datei zu löschen. Weitere Informationen finden Sie unter JAR-Dateien löschen.
