MQ-Verbindungen suchen

  1. Führen Sie eine der folgenden Aktionen aus:
    • Wählen Sie Project Explorer, um die Seite Projektansicht oder die Seite Projekt-/Ordneransicht anzuzeigen. Navigieren Sie dann durch die Projekte und Ordner, um nach der MQ-Verbindungsressource zu suchen.
    • Wählen Sie Ressourcenbrowser > MQ-Verbindungen. Die Seite Übersicht der MQ-Verbindungen wird angezeigt und listet die Informationen auf, die in Table 8-1 aufgeführt sind. Eine detaillierte Beschreibung der Eigenschaften finden Sie unter MQ-Verbindungen bearbeiten.
  2. Wenn Sie die Anzahl der Elemente in der Liste beschränken möchten, können Sie nach Name, Pfad oder beidem filtern. Geben Sie in den Feldern Name und Pfad den Namen und den Pfad der Suchziele ein, und klicken Sie dann auf Suchen.
  3. Klicken Sie auf das Symbol Öffnen, um zusätzliche Suchfilter anzuzeigen. Wenn Sie mehrere Suchkriterien angeben, werden nur Ergebnisse angezeigt, die allen Kriterien entsprechen.
    • Filtern nach Name des Queue Managers. Geben Sie eine beliebige Zeichenfolge ein, einschließlich Platzhalter. Klicken Sie dann auf Suchen.
    • Filtern nach Queue Manager-Kanalname. Geben Sie eine beliebige Zeichenfolge ein, einschließlich Platzhalter. Klicken Sie dann auf Suchen.
    • Klicken Sie auf Alle anzeigen, um die Suchfilter zu entfernen und alle MQ-Verbindungsressourcen anzuzeigen.

      • Tabelle 8-1 Details zu MQ-Verbindungen 
      Eigenschaft
      Beschreibung
      MQ-Verbindungsname
      Name, der der MQ-Verbindungsressource zugeordnet ist. Der Name ist ein Link auf die Seite MQ-Verbindungskonfiguration. Siehe MQ-Verbindungen bearbeiten.
      Pfad
      Der Pfad ist der Projektname und der Name des Ordners, in dem die MQ-Verbindungsressource abgelegt ist. Er ist ein Link zum Projekt oder zum Ordner, das bzw. der diese Ressource enthält. Siehe Ressourcennamen mit Projekten und Ordnern angeben.
      Optionen
      Klicken Sie auf das Symbol Löschen, um eine bestimmte MQ-Verbindungsressource zu löschen. Siehe MQ-Verbindungen löschen.
