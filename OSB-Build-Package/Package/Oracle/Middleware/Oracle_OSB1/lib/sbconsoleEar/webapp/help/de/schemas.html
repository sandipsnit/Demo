XML-Schemas

Schemas beschreiben Typen für primitive oder strukturierte Daten. XML-Schemas sind ein XML-Vokabular, das die Regeln beschreibt, denen XML-Business-Daten folgen müssen. XML-Schemas geben die Struktur der Dokumente an, sowie die Daten und den Typ jedes Elements und Attributs, das in dem Dokument enthalten ist.

Sie verwenden XML-Schemas als Verweise für WSDL-Ressourcen. Sie können auch XML-Schemas im Nachrichtenfluss der Proxy-Services verwenden, um ein Element zu validieren, das mit einem XPath-Ausdruck angegeben wird. Siehe Aktionen validieren.
