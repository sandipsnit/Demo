XML-Schemas hinzufügen

  1. Wenn dies noch nicht geschehen ist, klicken Sie zum Erstellen einer neuen Session auf Erstellen oder zum Aufrufen einer vorhandenen Session auf Bearbeiten. Weitere Informationen finden Sie unter Change Center verwenden.
  2. Wählen Sie Project Explorer, wählen Sie dann das Projekt oder den Ordner, zu dem Sie das neue XML-Schema hinzufügen möchten. Die Seite Projekt-/Ordneransicht wird angezeigt.
  3. Wählen Sie aus der Dropdown-ListeRessource erstellen XML-Schema aus, um die Seite Neue XML-Schemaressource erstellen anzuzeigen.
  4. Geben Sie im Feld Ressourcenname einen eindeutigen Namen für dieses XML-Schema ein. Dies ist ein obligatorisches Feld.

    Befolgen Sie die Benennungseinschränkungen für Ressourcen bei der Namensgebung.
  5. Geben Sie im Feld Ressourcenbeschreibung eine Beschreibung für das XML-Schema ein.
  6. Führen Sie im Feld XML-Schema einen der folgenden Schritte aus:
    • Geben Sie Text für das neue XML-Schema ein.
    • Klicken Sie aufDurchsuchen, um ein vorhandenes XML-Schema zu finden und zu importieren.
    • Sie können Text von einem vorhandenen XML-Schema in dieses Feld kopieren und einfügen.

      • Dieses Feld ist erforderlich.

  7. Klicken Sie auf Speichern. Das neue XML-Schema wird in der aktuellen Session gespeichert.

    8. Wenn Sie auf Speichern klicken, zeigt das System an, wenn es ungelöste Verweise für das neue XML-Schema gibt. Siehe Ungelöste XML-Schema-Referenzen anzeigen.

  8. Um die Session zu beenden und die Konfiguration für die Laufzeit bereitzustellen, klicken Sie unter Change Center auf Aktivieren.
