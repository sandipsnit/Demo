Nicht aufgelöste XML-Schemareferenzen auflösen

Gehen Sie auf die Seite Referenzen von XML-Schemaressourcen bearbeiten, um nicht aufgelöste XML-Schemareferenzen aufzulösen, indem Sie die Zuordnung für XML-Schemareferenzen, wie XML-Schemaimporte, konfigurieren.

  1. Wenn dies noch nicht geschehen ist, klicken Sie zum Erstellen einer neuen Session auf Erstellen oder zum Aufrufen einer vorhandenen Session auf Bearbeiten. Weitere Informationen finden Sie unter Change Center verwenden.
  2. Klicken Sie auf der Seite Übersicht der XML-Schemas in der Spalte XML-Schemaname auf den entsprechenden XML-Schemanamen, um Details der nicht aufgelösten Referenz anzuzeigen. Die Seite XML-Schemadetails wird angezeigt.
  3. Klicken Sie auf Referenzen bearbeiten, um die Seite Referenzen einer XML-Schemaressource bearbeiten anzuzeigen.
  4. Wählen Sie im Feld Ressourcentyp die Option XML-Schema.
  5. Klicken Sie auf Durchsuchen. Der XML-Schemabrowser wird angezeigt.
  6. Wählen Sie im XML-Schemabrowser ein XML-Schema. Wählen Sie dann eine Definition im Bereich Definitionen.
  7. Klicken Sie auf Weiterleiten. Das von Ihnen gewählte XML-Schema wird im Feld Ressourcenname angezeigt.
  8. Klicken Sie auf Speichern, um die Referenz aufzulösen.
  9. Klicken Sie auf der Seite XML-Schemadetails auf OK. Der Status des XML-Schemas wird als Gültig angezeigt.
