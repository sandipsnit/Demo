Gruppen hinzufügen

  1. Melden Sie sich bei der Oracle Service Bus Administration-Konsole als Benutzer mit WebLogic Server Admin-Berechtigung an. Nur Benutzer in der Admin-Rolle können die Sicherheitskonfigurationsdaten ändern. Siehe "Configuring Administrative Security" im Oracle Fusion Middleware Developer's Guide for Oracle Service Bus unter http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539.
  2. Wählen Sie Sicherheitskonfiguration > Gruppen, um die Seite Gruppenzusammenfassung anzuzeigen.
  3. Klicken Sie auf Neue hinzufügen.

    4. Sie können eine Gruppe von innerhalb oder von außerhalb einer Session hinzufügen.

  4. Geben Sie im Feld Gruppenname einen eindeutigen Namen ein. Sie können keine Leer- oder Sonderzeichen eingeben. Dieses Feld ist ein Pflichtfeld.
  5. Wählen·Sie·im·Feld·Authentifizierungsprovider den Authentifizierungsprovider aus.
  6. Wählen Sie im Feld Gruppenmitgliedschaft eine Gruppe aus, zu der diese Gruppe gehören soll.
    1. Wählen Sie eine Gruppe aus dem Feld Verfügbare Gruppen aus.
    2. Klicken Sie auf den Pfeil, um die Gruppe in das Feld Aktuelle Gruppen zu verschieben.
  7. Klicken Sie auf Speichern, um die Gruppe zu erstellen.

    8. Die Gruppe wird in der Oracle Service Bus Administration-Konsole gespeichert und steht dann allen Sessions unmittelbar zur Verfügung. Wenn Sie sich beim Hinzufügen einer Gruppe in einer Session befinden und diese Session dann beenden, löscht die Oracle Service Bus Administration-Konsole die neue Gruppe nicht.
