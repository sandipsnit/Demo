Gruppen bearbeiten

Auf der Seite Gruppendetails anzeigen können Sie Details zu einer bestimmten Gruppe anzeigen und ändern.

  1. Melden Sie sich bei der Oracle Service Bus Administration-Konsole als Benutzer mit WebLogic Server Admin-Berechtigung an. Nur Benutzer in der Admin-Rolle können die Sicherheitskonfigurationsdaten ändern. Siehe "Configuring Administrative Security" im Oracle Fusion Middleware Developer's Guide for Oracle Service Bus unter http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539.
  2. Suchen Sie die Gruppe. Weitere Informationen finden Sie unter Gruppen suchen.
  3. Klicken Sie auf den Gruppennamen. Auf der Seite Gruppendetails anzeigen werden die in Tabelle 24-4 dargestellten Informationen angezeigt.

    4. Tabelle 24-4 Gruppendetails
    Eigenschaft
    Beschreibung
    Gruppenname
    Der Name dieser Gruppe.
    Authentication Provider
    Der Authentication Provider, der diese Gruppendefinition enthält.
    Gruppen
    Die Gruppe, zu der diese Gruppe gehört.

  4. Zum Bearbeiten der Gruppendetails klicken Sie auf Neu konfigurieren, um die Seite Gruppendetails bearbeiten anzuzeigen.

    5. Sie können die Gruppendetails innerhalb oder außerhalb einer Session bearbeiten.

  5. Nehmen Sie die entsprechenden Änderungen im Feld Gruppenmitgliedschaft vor. Beschreibungen zum Feld finden Sie unter Gruppen hinzufügen.

    6. Das Feld Gruppenname darf nicht geändert werden.

  6. Um die Gruppe zu aktualisieren, klicken Sie auf Änderungen speichern.

    7. Die Gruppendetails werden in der Oracle Service Bus-Administrationskonsole aktualisiert, und das Update ist sofort in allen Sessions verfügbar. Wenn Sie sich beim Aktualisieren der Gruppe in einer Session befinden und die Session dann verwerfen, werden die Updates in der Oracle Service Bus-Administrationskonsole nicht gelöscht.
