Rollen hinzufügen

  1. Melden Sie sich bei der Oracle Service Bus Administration-Konsole als Benutzer mit WebLogic Server Admin-Berechtigung an. Nur Benutzer in der Admin-Rolle können die Sicherheitskonfigurationsdaten ändern. Siehe "Configuring Administrative Security" im Oracle Fusion Middleware Developer's Guide for Oracle Service Bus unter http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539.
  2. Wählen Sie Sicherheitskonfiguration > Rollen, um die Seite Globale Rollen zu öffnen.
  3. Klicken Sie auf Neu.

    4. Sie können innerhalb und außerhalb der Session Rollen hinzufügen.

  4. Geben Sie im Feld Rollenname einen eindeutigen Namen ein. Dieses Feld ist ein Pflichtfeld.

    5. Stellen Sie sicher, dass Sie im Namen der Sicherheitsrolle keine Leerzeichen oder die Zeichen < > verwenden. Bei Namen von Sicherheitsrollen muss die Groß- und Kleinschreibung beachtet werden. Laut Oracle-Konvention müssen die Namen von Sicherheitsrollen im Singular sein.

  5. Klicken Sie auf OK, um die Rolle zu erstellen.

    6. Die Rolle wird in der Oracle Service Bus Administration-Konsole gespeichert und ist sofort für alle Sessions verfügbar. Wenn Sie sich beim Hinzufügen gerade in einer Session befinden und die Session dann beenden, wird die neue Rolle in der Oracle Service Bus Administration-Konsole nicht gelöscht.

    Wenn Sie auf OK klicken, um die Rolle zu erstellen, müssen Sie im nächsten Schritt die Bedingungen definieren, unter denen die Rolle angewendet wird.

  6. Fahren Sie mit Rollenbedingungen definieren fort.
