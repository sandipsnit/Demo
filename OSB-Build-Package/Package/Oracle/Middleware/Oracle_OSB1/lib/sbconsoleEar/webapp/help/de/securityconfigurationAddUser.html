Benutzer hinzufügen

  1. Melden Sie sich bei der Oracle Service Bus Administration-Konsole als Benutzer mit WebLogic Server Admin-Berechtigung an. Nur Benutzer in der Admin-Rolle können die Sicherheitskonfigurationsdaten ändern. Siehe "Configuring Administrative Security" im Oracle Fusion Middleware Developer's Guide for Oracle Service Bus unter http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539.
  2. Wählen Sie Sicherheitskonfiguration, um die Seite Benutzerzusammenfassung anzuzeigen.
  3. Klicken Sie auf Neue hinzufügen, um die Seite Neuen Benutzer erstellen - Allgemeine Konfiguration anzuzeigen.

    4. Sie können einen Benutzer innerhalb oder außerhalb einer Session hinzufügen.

  4. Geben Sie im Feld Benutzername einen eindeutigen Namen ein. Dieses Feld ist obligatorisch.
  5. Geben Sie im Feld Kennwort ein Kennwort ein. Dieses Feld ist obligatorisch.
    6. Hinweis: Authentication Provider können eine Mindestlänge für das Kennwort festlegen. Für einen Benutzer im WebLogic-Authentication Provider beträgt die standardmäßige Kennwortmindestlänge 8 Zeichen. Sie können diese Einstellung in der WebLogic Server-Administrationskonsole anpassen. (Der WebLogic-Authentication Provider wird in der Standardsicherheits-Realm mit dem Namen "DefaultAuthenticator" konfiguriert.)
  6. Geben Sie im Feld Kennwort bestätigen dasselbe Kennwort wie im Feld Kennwort ein. Dieses Feld ist obligatorisch.
  7. Wählen Sie im Feld Authentication Provider den Authentication Provider für diesen Benutzer aus.

    8. Wenn in der Sicherheits-Realm mehrere Authentication Provider konfiguriert sind, werden sie in der Liste angezeigt. Wählen Sie die Datenbank des Authentication Providers aus, in der Informationen zum neuen Benutzer gespeichert werden sollen.

  8. Wählen Sie im Feld Gruppenmitgliedschaft eine Gruppe für diesen Benutzer aus.
    1. Wählen Sie eine Gruppe aus dem Feld Verfügbare Gruppen aus.
    2. Klicken Sie auf den Pfeil, um die Gruppe in das Feld Aktuelle Gruppen zu verschieben.
  9. Um den Benutzer zu erstellen, klicken Sie auf Speichern.

    10. Der Benutzer wird in der Oracle Service Bus Administration-Konsole gespeichert und ist sofort in allen Sessions verfügbar. Wenn Sie sich beim Hinzufügen gerade in einer Session befinden und die Session beenden, wird der neue Benutzer in der Oracle Service Bus Administration-Konsole nicht gelöscht.
