Policy-Bedingungen hinzufügen

  1. Greifen Sie über den Policy-Editor auf eine Zugriffskontroll-Policy zu. Siehe Zugriffs-Policys auf Transportebene bearbeiten oder Zugriffs-Policys auf Nachrichtenebene bearbeiten.
  2. Klicken Sie im Policy-Editor unter Policy-Bedingungen auf Bedingung hinzufügen.

    3. Folgender Prompt wird angezeigt:

    Wählen Sie das Prädikat, das Sie als neue Bedingung verwenden möchten

  3. Wählen Sie ein Prädikat aus der Liste.
  4. Klicken Sie auf Weiter. Führen Sie, je nach Ihrer Auswahl eines Bedingungsprädikats, einen der folgenden Schritte aus, wie in Tabelle 24-12 dargestellt.
