Gruppen löschen

Auf der Seite Gruppenzusammenfassung können Sie eine gewählte Gruppe oder mehrere Gruppen löschen.

  1. Melden Sie sich bei der Oracle Service Bus Administration-Konsole als Benutzer mit WebLogic Server Admin-Berechtigung an. Nur Benutzer in der Admin-Rolle können die Sicherheitskonfigurationsdaten ändern. Siehe "Configuring Administrative Security" im Oracle Fusion Middleware Developer's Guide for Oracle Service Bus unter http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539.
  2. Wählen Sie Sicherheitskonfiguration, um die Seite Gruppenzusammenfassung anzuzeigen.
  3. Wählen Sie die zu löschende Gruppe. Gegebenenfalls können Sie mehrere Gruppen wählen.

    4. Sie können Gruppen innerhalb oder außerhalb einer Session löschen.

  4. Klicken Sie auf Löschen. Eine Meldung wird angezeigt, in der Sie aufgefordert werden zu bestätigen, dass Sie die Gruppe löschen möchten.
  5. Um die Gruppe zu löschen, klicken Sie auf OK.

    6. Die Gruppe wird in der Oracle Service Bus Administration-Konsole gelöscht. Wenn Sie sich beim Löschen gerade in einer Session befinden und die Session beenden, bleibt die Gruppe in der Oracle Service Bus Administration-Konsole gelöscht.

  6. Alternativ können Sie in der Spalte Optionen der zu löschenden Gruppe auf das Symbol Löschen klicken.
