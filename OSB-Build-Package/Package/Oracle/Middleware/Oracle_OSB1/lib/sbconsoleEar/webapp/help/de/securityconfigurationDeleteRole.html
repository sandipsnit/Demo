Rollen löschen

  1. Melden Sie sich bei der Oracle Service Bus Administration-Konsole als Benutzer mit WebLogic Server Admin-Berechtigung an. Nur Benutzer in der Admin-Rolle können die Sicherheitskonfigurationsdaten ändern. Siehe "Configuring Administrative Security" im Oracle Fusion Middleware Developer's Guide for Oracle Service Bus unter http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539.
  2. Wählen Sie Sicherheitskonfiguration > Rollen, um die Seite Globale Rollen zu öffnen.
  3. Wählen Sie die zu löschende Rolle aus. Sie können bei Bedarf mehrere Rollen wählen.

    4. Sie können innerhalb und außerhalb der Session Rollen löschen.

  4. Klicken Sie auf Löschen. In einer Meldung werden Sie aufgefordert, den Löschvorgang zu bestätigen.
  5. Klicken Sie auf OK, um die Rolle zu löschen.

    6. Die Rolle wird in der Oracle Service Bus Administration-Konsole gelöscht. Wenn Sie sich beim Löschen gerade in einer Session befinden und die Session beenden, bleibt die Rolle in der Oracle Service Bus Administration-Konsole gelöscht.
