Benutzer löschen

Auf der Seite Benutzerzusammenfassung können Sie einen gewählten Benutzer oder mehrere Benutzer löschen.

  1. Melden Sie sich bei der Oracle Service Bus Administration-Konsole als Benutzer mit WebLogic Server Admin-Berechtigung an. Nur Benutzer in der Admin-Rolle können die Sicherheitskonfigurationsdaten ändern. Siehe "Configuring Administrative Security" im Oracle Fusion Middleware Developer's Guide for Oracle Service Bus unter http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539.
  2. Wählen Sie Sicherheitskonfiguration, um die Seite Benutzerzusammenfassung anzuzeigen.
  3. Wählen Sie den zu löschenden Benutzer aus. Sie können bei Bedarf mehrere Benutzer wählen.

    4. Sie können innerhalb und außerhalb der Session Benutzer löschen.

  4. Klicken Sie auf Löschen. In einer Meldung werden Sie aufgefordert, den Löschvorgang zu bestätigen.
  5. Klicken Sie auf OK, um den Benutzer zu löschen.

    6. Der Benutzer wird in der Oracle Service Bus Administration-Konsole gelöscht. Wenn Sie sich beim Löschen gerade in einer Session befinden und die Session dann beenden, bleibt der Benutzer in der Oracle Service Bus Administration-Konsole gelöscht.

  6. Sie können auch in der Spalte Optionen des zu löschenden Benutzers auf das Symbol Löschen klicken.
