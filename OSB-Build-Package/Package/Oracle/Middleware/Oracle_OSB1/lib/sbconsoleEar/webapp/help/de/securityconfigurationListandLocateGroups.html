Gruppen suchen

  1. Wählen Sie Sicherheitskonfiguration > Gruppen. Auf der Seite Gruppenzusammenfassung werden die in Tabelle 24-3 dargestellten Informationen angezeigt. Eine ausführlichere Beschreibung der Eigenschaften finden Sie unter Gruppen bearbeiten.
  2. Um die Anzahl der Elemente in der Liste einzuschränken oder eine bestimmte Gruppe zu suchen, können Sie nach Gruppennamen filtern. Geben Sie den Gruppennamen ganz oder teilweise im Feld Name ein, und klicken Sie auf Suchen.

    3. Sie können das Sternchen (*) als Platzhalterzeichen verwenden. (Andere Platzhalterzeichen werden nicht unterstützt.)

    Um die Suchfilter zu entfernen und alle Gruppen anzuzeigen, klicken Sie auf Alle anzeigen.

    Tabelle 24-3 Gruppeninformationen 
    Eigenschaft
    Beschreibung
    Gruppenname
    Der Name der Gruppe. Der Name ist ein Link zur Seite Gruppendetails anzeigen. Weitere Informationen finden Sie unter Gruppen bearbeiten.
    Gruppenmitgliedschaft
    Die Gruppe, zu der diese Gruppe gehört. Der Name ist ein Link zur Seite Gruppendetails anzeigen. Weitere Informationen finden Sie unter Gruppen bearbeiten.
    Authentication Provider
    Der Authentication Provider, der diese Gruppendefinition enthält.
    Löschen
    Klicken Sie auf das Symbol Löschen, um eine bestimmte Gruppe zu löschen. Weitere Informationen finden Sie unter Gruppen löschen.
