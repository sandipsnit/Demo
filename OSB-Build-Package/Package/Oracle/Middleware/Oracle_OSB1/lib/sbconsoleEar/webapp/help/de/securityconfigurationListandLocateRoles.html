Rollen suchen

  1. Wählen Sie Sicherheitskonfiguration> Rollen. Die Seite Globale Rollen zeigt die Informationen an, die in Tabelle 24-5 dargestellt sind. Eine detailliertere Beschreibung der Eigenschaften finden Sie unter Rollen bearbeiten.
  2. Wenn Sie die Anzahl der Elemente in der Liste begrenzen möchten oder eine bestimmte Rolle suchen, durchsuchen Sie die Seiten. Verwenden Sie die Seitensteuerung über oder unter der Tabelle.

    3. Tabelle 24-5 Rolleninformation 
    Eigenschaft
    Beschreibung
    Rollenname
    Der Name der Rolle. Der Name ist ein Link zu der Seite Rollendetails anzeigen. Siehe Rollen bearbeiten.
    Providername
    Der Authentication Provider für diese Gruppe.
