Benutzer suchen

  1. Wählen Sie Sicherheitskonfiguration > Benutzer. Auf der Seite Benutzerzusammenfassung werden die Informationen angezeigt, die in der Tabelle 24-1 aufgeführt sind. Eine detaillierte Beschreibung der Eigenschaften finden Sie unter Benutzer bearbeiten.
  2. Um die Anzahl der Elemente in der Liste zu beschränken oder nach einem bestimmten Benutzer zu suchen, können Sie nach Benutzername filtern. Geben Sie den Benutzernamen ganz oder teilweise in das Feld Name ein, und klicken Sie auf Suchen.

    3. Sie können das Sternchen (*) als Platzhalterzeichen verwenden. (Andere Platzhalterzeichen werden nicht unterstützt.)

    Klicken Sie auf Alle anzeigen, um die Suchfilter zu entfernen und alle Benutzer anzuzeigen.

    Tabelle 24-1 Benutzerinformationen 
    Eigenschaft
    Beschreibung
    Benutzername
    Name, der dem Benutzer zugewiesen ist. Der Name ist ein Link zur Seite Benutzerdetails anzeigen. Siehe Benutzer bearbeiten.
    Gruppenmitgliedschaft
    Name der Gruppe, der der Benutzer angehört. Der Name ist ein Link zur Seite Gruppendetails anzeigen. Siehe Gruppen bearbeiten.
    Authentication Provider
    Authentification-Provider für diesen Benutzer.
    Optionen
    Klicken Sie auf das Symbol Löschen, um einen bestimmten Benutzer zu löschen. Siehe Benutzer löschen.
