Rollen bearbeiten

Gehen Sie zur Seite Rollendetails anzeigen, um Details einer bestimmten Rolle anzuzeigen oder zu ändern.

  1. Melden Sie sich bei der Oracle Service Bus Administration-Konsole als Benutzer mit WebLogic Server Admin-Berechtigung an. Nur Benutzer in der Admin-Rolle können die Sicherheitskonfigurationsdaten ändern. Siehe "Configuring Administrative Security" im Oracle Fusion Middleware Developer's Guide for Oracle Service Bus unter http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539.
  2. Suchen Sie nach der Rolle. Siehe Rollen suchen.
  3. Klicken Sie auf den Rollennamen. Auf der Seite Rollendetails anzeigen werden die Informationen angezeigt, die in Tabelle 24-8 aufgelistet sind.

    4. Tabelle 24-8 Rollendetails 
    Eigenschaft
    Beschreibung
    Name
    Name der Rolle.
    Rollenbedingungen
    Bedingungen, die die Mitgliedschaft zu dieser Rolle festlegen.

    Sie können die Rollendetails innerhalb und außerhalb der Session ändern.

  4. Führen Sie einen der folgenden Schritte aus, wie in Tabelle 24-9 beschrieben.

    5. Tabelle 24-9 Rollenoptionen 
    Ziel
    Führen Sie diesen Schritt durch:
    Reihenfolge der gewählten Ausdrücke ändern.
    Klicken Sie auf Nach oben verschieben oder Nach unten verschieben.
    Integrieren Sie Rollenbedingungen, bzw. heben Sie die Integration auf, und wechseln Sie die markierten Anweisungen and und or zwischen Ausdrücken.
    Klicken Sie auf Kombinieren oder Kombination aufheben.
    Machen Sie eine Bedingung negativ. Beispiel: NICHT Gruppenoperatoren schließt die Gruppe "Operators" aus der Rolle aus.
    Klicken Sie auf Negieren.
    gewählten Ausdruck löschen.
    Klicken Sie auf Entfernen.

  5. Klicken Sie auf Speichern.

    6. Die Rolle wird in der Oracle Service Bus-Administrationskonsole aktualisiert, und das Update ist sofort in allen Sessions verfügbar. Wenn Sie sich beim Aktualisieren der Rolle in einer Session befinden und die Session dann verwerfen, werden die Updates in der Oracle Service Bus-Administrationskonsole nicht gelöscht.
