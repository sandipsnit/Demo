Benutzer bearbeiten

Auf der Seite Benutzerdetails anzeigen können Sie Details zu einem bestimmten Benutzer anzeigen und ändern.

  1. Melden Sie sich bei der Oracle Service Bus Administration-Konsole als Benutzer mit WebLogic Server Admin-Berechtigung an. Nur Benutzer in der Admin-Rolle können die Sicherheitskonfigurationsdaten ändern. Siehe "Configuring Administrative Security" im Oracle Fusion Middleware Developer's Guide for Oracle Service Bus unter http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539.
  2. Suchen Sie den Benutzer. Weitere Informationen finden Sie unter Benutzer suchen.
  3. Klicken Sie auf den Benutzernamen. Auf der Seite Benutzerdetails anzeigen werden die in Tabelle 24-2 dargestellten Informationen angezeigt.

    4. Tabelle 24-2 Benutzerdetails
    Eigenschaft
    Beschreibung
    Benutzername
    Der Name dieses Benutzers.
    Authentication Provider
    Der Authentication Provider, der diese Benutzerdefinition enthält.
    Gruppenmitgliedschaft
    Der Name der Gruppe, zu der dieser Benutzer gehört.

  4. Zum Bearbeiten der Benutzerdetails klicken Sie auf Neu konfigurieren, um die Seite Benutzerdetails bearbeiten anzuzeigen.

    5. Sie können die Benutzerdetails innerhalb oder außerhalb einer Session bearbeiten.

  5. Nehmen Sie die entsprechenden Änderungen in den Felder Neues Kennwort, Kennwort bestätigen und Gruppenmitgliedschaft vor. Beschreibungen zu den Feldern finden Sie unter Benutzer hinzufügen.

    6. Das Feld Benutzername darf nicht geändert werden.

  6. Um den Benutzer zu aktualisieren, klicken Sie auf Änderungen speichern.

    7. Die Benutzerdetails werden in der Oracle Service Bus-Administrationskonsole aktualisiert, und das Update ist sofort in allen Sessions verfügbar. Wenn Sie sich beim Aktualisieren des Benutzers gerade in einer Session befinden und die Session dann verwerfen, werden die Updates in der Oracle Service Bus Administrationskonsole nicht gelöscht.
