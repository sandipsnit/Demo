Rollenbedingungen definieren

  1. Klicken Sie auf der Seite Globale Rollen auf den Namen der neuen globalen Rolle, um die Seite Globale Rollenbedingungen anzuzeigen.
  2. Klicken Sie unter Rollenbedingungen auf Bedingung hinzufügen.

    3. Folgender Prompt wird angezeigt:

    Wählen Sie das Prädikat, das Sie als neue Bedingung verwenden möchten

  3. Wählen Sie ein Prädikat aus dem Listenfeld. Normalerweise wählen Sie Gruppe. Wenn eine Sicherheitsrolle auf der Grundlage einer Gruppe erstellt wird, kann die Sicherheitsrolle allen Gruppenmitgliedern (also mehreren Benutzern) zugewiesen werden.
  4. Klicken Sie auf Weiter. Führen Sie je nach gewähltem Bedingungsprädikat einen der in Tabelle 24-6 beschriebenen Schritte aus.

    5. Tabelle 24-6 Bedingungsprädikatoptionen 
    Bedingungsprädikat
    Vorgehensweise
    Wenn Sie Gruppe gewählt haben, geben Sie ein oder mehrere Argumente ein, um die Gruppe zu definieren, denen diese Rolle zugewiesen werden soll.
    1. Geben Sie im Feld Gruppenargumentname ein Argument ein, das die Gruppe definiert.
    2. Klicken Sie auf Hinzufügen.
    3. Wiederholen Sie gegebenenfalls die Schritte 1 und 2, bis Sie alle Argumente hinzugefügt haben. Sie können auf Entfernen klicken, um die Argumente aus der Liste zu entfernen.
    4. Klicken Sie auf Beenden.
    Wenn Sie Benutzer gewählt haben, geben Sie ein oder mehrere Argumente ein, um den Benutzer oder die Benutzer zu definieren, denen diese Rolle zugewiesen werden soll.
    1. Geben Sie im Feld Benutzerargumentname ein Argument ein, das den Benutzer definiert.
    2. Klicken Sie auf Hinzufügen.
    3. Wiederholen Sie gegebenenfalls die Schritte 1 und 2, bis Sie alle Argumente hinzugefügt haben. Sie können auf Entfernen klicken, um die Argumente aus der Liste zu entfernen.

    Wenn Sie ein Prädikat mit Zeitbegrenzung gewählt haben (Beispiel: Zugriff erfolgt zwischen x und y Uhr), wählen Sie Start- und Endzeit sowie einen GMT Offset.
    1. Geben Sie im Feld Startzeit die früheste zulässige Zeit im Format hh:mm:ss AM|PM ein. Beispiel: 12:45:00 AM.
    2. Geben Sie im Feld Endzeit die späteste zulässige Zeit im Format hh:mm:ss AM|PM ein. Beispiel: 12:45:00 AM.
    3. Geben Sie im Feld GMT-Offset die Stunden ein, die die Zeit der GMT voraus ist. Verwenden Sie dazu das Format GMT+hh:mm. Geben Sie im Format GMT-hh:mm die Stunden ein, die die Zeit hinter der GMT zurückliegt. Beispiel: Die Eastern Standard Time in den USA ist GMT-5:00.
    4. Klicken Sie auf Beenden.
    Wenn Sie Kontextelement definiert aktiviert haben, geben Sie den Namen eines Kontextelements ein.
    1. Geben Sie im Feld Kontextelementname den Namen eines Kontextelements ein.
    2. Klicken Sie auf Beenden.
    Wenn Sie Wert des Kontextelements entspricht einer numerischen Konstante, Wert des Kontextelements ist größer als eine numerische Konstante oder Wert des Kontextelements ist kleiner als eine numerische Konstante gewählt haben, geben Sie den Namen eines Kontextelements und einen numerischen Wert ein, mit dem der Name verglichen werden soll.
    1. Geben Sie im Feld Kontextelementname den Namen eines Kontextelements ein, dessen Wert geprüft werden muss.
    2. Geben Sie im Feld Numerischer Wert einen numerischen Wert ein.
    3. Klicken Sie auf Beenden.
    Wenn Sie Wert des Kontextelements entspricht einem Zeichenfolgenwert gewählt haben, geben Sie den Namen eines Kontextelements und einen Zeichenfolgenwert ein, mit dem der Name verglichen werden soll.
    1. Geben Sie im Feld Kontextelementname den Namen eines Kontextelements ein, dessen Wert geprüft werden muss.
    2. Geben Sie im Feld Zeichenfolgenwert den Zeichenfolgewert ein, der verglichen werden soll.
    3. Klicken Sie auf Beenden.
    Wenn Sie ein Prädikat mit Zeitbegrenzung gewählt haben. Beispiel: Zugriff erfolgt voroder Zugriff erfolgt nach
    1. Geben Sie im Feld Datum ein Datum im Format MM/TT/JJ ein. Beispiel: 1/1/04. Sie können zusätzlich die Uhrzeit im Format hh:mm:ss AM|PM hinzufügen. Beispiel: 1/1/04 12:45:00 AM.
    2. Klicken Sie auf Beenden.
    Wenn Sie das Prädikat mit Zeitbegrenzung Zugriff erfolgt an bestimmten Tagen der Woche gewählt haben, wählen Sie den Wochentag und ein GMT-Offset
    1. Geben Sie im Feld Wochentag den Wochentag ein.
    2. Geben Sie im Feld GMT-Offset die Stunden ein, die die Zeit der GMT voraus ist. Verwenden Sie dazu das Format GMT+hh:mm. Geben Sie im Format GMT-hh:mm die Stunden ein, die die Zeit hinter der GMT zurückliegt. Beispiel: Die Eastern Standard Time in den USA ist GMT-5:00.
    3. Klicken Sie auf Beenden.
    Wenn Sie ein Prädikat mit Zeitbegrenzung gewählt haben, wie Zugriff erfolgt an einem bestimmten Tag des Monats, Zugriff erfolgt vor einem bestimmten Tag des Monats oder Zugriff erfolgt nach einem bestimmten Tag des Monats
    1. Geben Sie in das Feld "Tag des Monats" die Ordinalzahl des Tages im aktuellen Monat mit einem Wert zwischen -31 und 31 an. Negative Werte zählen vom Ende des Monats rückwärts, sodass der letzte Tag des Monats als -1 angegeben wird. 0 steht für den Tag vor dem ersten Tag des Monats.
    2. Geben Sie im Feld GMT-Offset die Stunden ein, die die Zeit der GMT voraus ist. Verwenden Sie dazu das Format GMT+hh:mm. Geben Sie im Format GMT-hh:mm die Stunden ein, die die Zeit hinter der GMT zurückliegt. Beispiel: Die Eastern Standard Time in den USA ist GMT-5:00.
    3. Klicken Sie auf Beenden.

  5. Wiederholen Sie bei Bedarf diese Schritte, um Ausdrücke hinzuzufügen, die auf verschiedenen Rollenbedingungen basieren. Im Bereich Rollenbedingungen können Sie die in Tabelle 24-7 beschriebenen Schritte ausführen, um die Ausdrücke zu ändern.
