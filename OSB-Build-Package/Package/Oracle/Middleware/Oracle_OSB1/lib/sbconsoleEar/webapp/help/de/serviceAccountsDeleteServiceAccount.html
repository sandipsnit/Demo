Service-Accounts löschen

Wenn Sie einen Service-Account löschen, werden der Benutzername, das Kennwort sowie die im Service-Account enthaltenen Zuordnungsdaten von lokalem Benutzer zu Remote-Benutzer ebenfalls gelöscht.

  1. Wenn dies noch nicht geschehen ist, klicken Sie zum Erstellen einer neuen Session auf Erstellen oder zum Aufrufen einer vorhandenen Session auf Bearbeiten. Weitere Informationen finden Sie unter Change Center verwenden.
  2. Wenn ein Business-Service oder Proxy-Service für die Verwendung des Service-Accounts konfiguriert ist, entfernen Sie den Service-Account aus dem Business-Service oder Proxy-Service.

    3. Weitere Informationen finden Sie unter Business-Service-Konfigurationen bearbeiten oder unter Proxy-Service-Konfigurationen bearbeiten.

  3. Wählen Sie Ressourcenbrowser > Service-Accounts, um die Seite Übersicht der Service-Accounts anzuzeigen.
  4. Klicken Sie im Feld Optionen des zu löschenden Service-Accounts auf das Symbol Löschen.

    5. Der Service-Account wird in der aktuellen Session gelöscht. Wenn ein Business-Service oder Proxy-Service für die Verwendung eines Service-Accounts konfiguriert wurde, gibt das Symbol Warnung wegen Löschvorgang an, dass Sie den Service-Account mit einer Warnungsbestätigung löschen können. Dadurch können Konflikte aufgrund nicht aufgelöster Referenzen vom Service zum gelöschten Service-Account entstehen.

  5. Um die Session zu beenden und die Konfiguration für die Laufzeit bereitzustellen, klicken Sie unter Change Center auf Aktivieren.
