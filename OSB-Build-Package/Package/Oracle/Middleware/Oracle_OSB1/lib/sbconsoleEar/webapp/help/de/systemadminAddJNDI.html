JNDI-Provider hinzufügen

Unter Oracle Service Bus werden JNDI-Providerressourcen als Systemressourcen hinzugefügt, konfiguriert und verfügbar gemacht. Sie müssen in einer aktiven Session sein.

  1. Wenn dies noch nicht geschehen ist, klicken Sie zum Erstellen einer neuen Session auf Erstellen oder zum Aufrufen einer vorhandenen Session auf Bearbeiten. Weitere Informationen finden Sie unter Change Center verwenden.
  2. Wählen Sie Systemadministration > JNDI-Provider.
  3. Klicken Sie auf Hinzufügen.
  4. Geben Sie im Feld Name einen Namen für die JNDI-Providerressource ein. Dies ist ein obligatorisches Feld.

    Befolgen Sie die Benennungseinschränkungen für Ressourcen bei der Namensgebung.
  5. Geben Sie im Feld Beschreibung eine kurze Beschreibung für den JNDI-Provider ein.
  6. Halten Sie die Option JNDI-Cache "Aktiviert" (Standard), oder wählen Sie "Deaktiviert" aus.

    7. Bei Aktivierung werden JNDI-Kontext und JNDI-Objekte lokal gecacht, dadurch wird die Performance bei Objekt-Lookup verbessert. Oracle empfiehlt, dass Sie das JNDI-Cache aktiviert halten.

  7. Im URL-Feld Provider geben Sie den URL für den JNDI-Provider in folgendem Format ein: protocol://host:port

    8. Sie können jedes Protokoll verwenden, zum Beispiel http, https, t3, t3s, iiop, iiops.

    Dieses Feld ist erforderlich.

  8. Bei Clustern sollte der JNDI-Provider-URL mit einer Komma-getrennten Liste der Managed Server in folgendem Format konfiguriert sein:

    9. protocol://<hostname>:<ms1port>, <hostname>:<ms2port>

  9. Wenn der Zugriff auf einen Ziel-JNDI-Provider einen Benutzernamen und ein Kennwort erfordert, dann geben Sie im Feld Benutzername einen Benutzernamen ein und das entsprechende Kennwort in den Feldern Kennwort und Kennwort bestätigen.

    10. Diese Felder sind optional, sie sind nur erforderlich, wenn der JNDI-Baum gesichert ist.

  10. Klicken Sie auf Speichern, um die JNDI-Providerressource in der aktuellen Session zu erstellen und zu speichern.
  11. Um die Session zu beenden und die Konfiguration für die Laufzeit bereitzustellen, klicken Sie unter Change Center auf Aktivieren.
