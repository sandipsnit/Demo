Status "automatischer Import" verwenden

Auf der Seite Status automatischer Import können Sie Änderungen an einem Service mit den in der Registry vorhandenen synchronisieren. Nach jeder Änderung eines Service in der Registry liefert Oracle Service Bus eine Änderungsbenachrichtigung an die Seite Status automatischer Import, auf der alle nicht synchronen Services aufgelistet werden. Sie können den Service daraufhin in der Oracle Service Bus Administration-Konsole mit dem entsprechenden Service in der UDDI-Registry synchronisieren.

Wenn Sie eine Registry konfigurieren, können Sie die Option Automatischen Import aktivieren wählen, mit der alle importierten Services mit der UDDI-Registry synchronisiert werden. Jeder importierte Service, bei dem diese Option aktiviert ist, wird automatisch mit der UDDI-Registry synchron gehalten. Siehe UDDI-Registrys konfigurieren.

Wenn ein Fehler während der automatischen Synchronisierung auftritt, wird dieser auf der Seite Status automatischer Import gemeldet, wo Sie ihn manuell aktualisieren können.

  1. Wenn dies noch nicht geschehen ist, klicken Sie zum Erstellen einer neuen Session auf Erstellen oder zum Aufrufen einer vorhandenen Session auf Bearbeiten. Weitere Informationen finden Sie unter Change Center verwenden.
  2. Wählen Sie Systemadministration > Status automatischer Import.
  3. Auf der Seite Status automatischer Import wählen Sie den Service-Typ, der in der Dropdown-Liste Ansicht angezeigt werden soll.
  4. Klicken Sie auf die Kontrollkästchen neben den Proxy-Services, die mit den entsprechenden Services in der Registry synchronisiert werden sollen.
  5. Klicken Sie auf Synchronisieren.
  6. Klicken Sie auf der nächsten Seite auf Fertig stellen.
