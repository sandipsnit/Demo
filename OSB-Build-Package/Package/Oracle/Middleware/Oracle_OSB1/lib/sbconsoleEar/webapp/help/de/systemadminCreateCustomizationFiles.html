Anpassungsdateien erstellen

Auf der Seite Anpassungsdatei erstellen können Sie Anpassungsdateien für mehrere gewählte Ressourcen oder Projekte auf einfache Weise generieren. Sie können diese Datei als Ausgangspunkt für gewünschte Änderungen verwenden, indem Sie die tatsächlichen Werte für eine Umgebung angeben.

Mithilfe von Anpassungsdateien können Sie Änderungen an Umgebungswerten und an Referenzen in Ressourcen vornehmen. Anpassungsdateien können Anpassungen für alle Umgebungswerte in der von Ihnen gewählten Ressource umfassen, einschließlich komplexer Umgebungswerte, die innerhalb der Klasse EnvValueTypes definiert wurden. Außerdem ist ein Referenzanpassungstyp vorhanden, mit dem Ressourcenreferenzen innerhalb von Ressourcen mit Abhängigkeiten geändert werden können.

Das Anpassungsschema (Customization.xsd), das die Anpassungstypen beschreibt, ist an folgender Stelle Ihrer Oracle Service Bus-Installation verfügbar:

OSB_ORACLE_HOME\modules\com.bea.common.configfwk_version.jar.

  1. Wählen Sie Systemadministration > Anpassungsdatei erstellen.

    2. Auf der Seite Anpassungsdatei erstellen wird eine Liste der Objekte in Ihrer Konfiguration angezeigt. Auch Name und Typ eines jeden Objekts werden angezeigt.

  2. Wählen Sie die Projekte oder Ressourcen, die Sie in die Anpassungsdatei aufnehmen möchten.
    1. Erweitern Sie die Projektordner. Name und Typ einer jeden Ressource im Projekt werden angezeigt.
    2. Aktivieren Sie die Kontrollkästchen der Projekte oder Ressourcen, die Sie in die Anpassungsdatei aufnehmen möchten.
    3. Heben Sie die Auswahl der Kontrollkästchen für die Projekte oder Ressourcen auf, die nicht in die Anpassungsdatei aufgenommen werden sollen.
  3. Klicken Sie auf Datei erstellen, um eine Anpassungs-XML-Datei zu erstellen.
  4. Klicken Sie im Dialogfeld Herunterladen von Dateien auf Öffnen, um die Anpassungsdatei zu öffnen, oder klicken Sie auf Speichern, um die XML-Datei auf Ihrem Desktop zu speichern.
