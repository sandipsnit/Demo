Standard-UDDI-Konfiguration einrichten

Sie können eine der konfigurierten Registrys als Standard-UDDI-Registry für die Domain festlegen. Siehe UDDI-Registrys konfigurieren.

Um die Funktion des automatischen Publishings verwenden zu können, müssen Sie eine Standard-Registry einrichten. Siehe ###Automatisches Publishing verwenden.

  1. Wenn dies noch nicht geschehen ist, klicken Sie zum Erstellen einer neuen Session auf Erstellen oder zum Aufrufen einer vorhandenen Session auf Bearbeiten. Weitere Informationen finden Sie unter Change Center verwenden.
  2. Wählen Sie Systemadministration > UDDI-Registrys.
  3. Klicken Sie im Fenster der UDDI-Standardkonfiguration auf Standard wählen.
  4. Wählen Sie aus der Liste Standard-Registry-Name den Namen der Registry, die Sie als Standard-Registry einrichten möchten.
  5. Um eine Standard-Business-Entität einzurichten, wählen Sie eine Entität aus der Liste Business-Entität.
  6. Klicken Sie auf Als Standard speichern.
  7. Um die Standard-Registry zu ändern, klicken Sie auf Auswahl löschen, dann wiederholen Sie die oben genannten Schritte.
