JNDI-Provider löschen

  1. Wenn dies noch nicht geschehen ist, klicken Sie zum Erstellen einer neuen Session auf Erstellen oder zum Aufrufen einer vorhandenen Session auf Bearbeiten. Weitere Informationen finden Sie unter Change Center verwenden.
  2. Wählen Sie Systemadministration > JNDI-Provider.
  3. Wählen Sie auf der Seite Übersicht der JNDI-Provider einen JNDI-Provider. Sie können auch mit der Option Suchen nach einem bestimmten JNDI-Provider suchen. Siehe JNDI-Provider anzeigen.
  4. Klicken Sie im Feld Optionen des zu löschenden JNDI-Providers auf das Symbol Löschen. Das Symbol Warnung wegen Löschvorgang wird angezeigt, wenn andere Ressourcen diese Ressource referenzieren. Sie können die Ressource mit einer Warnbestätigung löschen. Dies kann jedoch zu Konflikten aufgrund von nicht aufgelösten Referenzen zur gelöschten Ressource führen.
  5. Um die Session zu beenden und die Konfiguration für die Laufzeit bereitzustellen, klicken Sie unter Change Center auf Aktivieren.
