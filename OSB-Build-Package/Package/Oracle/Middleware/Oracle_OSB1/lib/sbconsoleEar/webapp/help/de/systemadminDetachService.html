Zuordnung von Services aufheben

Wenn Sie nicht möchten, dass der Service in der Oracle Service Bus Administration-Konsole mit dem entsprechenden Service in der Registry synchronisiert wird, können Sie die Synchronisierung verhindern, indem Sie die Zuordnung des Service zur Registry aufheben. Hierfür stehen folgende Methoden zur Verfügung:

  • Zuordnung eines Service durch Bearbeiten der Konfiguration aufheben
  • Zuordnung eines Service über die Seite "Status automatischer Import" aufheben
Zuordnung eines Service durch Bearbeiten der Konfiguration aufheben
  1. Wenn dies noch nicht geschehen ist, klicken Sie zum Erstellen einer neuen Session auf Erstellen oder zum Aufrufen einer vorhandenen Session auf Bearbeiten. Weitere Informationen finden Sie unter Change Center verwenden.
  2. Wählen Sie Project Explorer.
  3. Erweitern Sie den Projektordner, und wählen Sie den Business-Service, dessen Zuordnung vom entsprechenden Service in der Registry Sie aufheben möchten.
  4. Klicken Sie unten auf der Seite Business-Service anzeigen auf Bearbeiten.
  5. Aktivieren Sie die Option Zuordnung zur Registry aufheben. Die Zuordnung des Business-Service vom entsprechenden Service in der UDDI-Registry wird aufgehoben.
Zuordnung eines Service über die Seite "Status automatischer Import" aufheben

Sie können diese Option nur dann verwenden, wenn Sie über die Änderung auf der Seite Status des automatischen Imports benachrichtigt werden.

  1. Aktivieren Sie auf der Seite Systemverwaltung > Status des automatischen Imports die Kontrollkästchen neben den Business-Services, deren Zuordnung zu den entsprechenden Services in der UDDI-Registry Sie aufheben möchten.

    2. Services werden auf dieser Seite nur dann angezeigt, wenn eine Änderung am ursprünglichen Service in der Registry vorgenommen wurde. Nicht jeder Service ist auf dieser Seite verfügbar.

  2. Klicken Sie auf Trennen.
