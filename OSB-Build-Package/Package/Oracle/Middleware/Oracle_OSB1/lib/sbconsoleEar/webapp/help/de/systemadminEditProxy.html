Proxyserver bearbeiten

Sie verwenden diese Seite, um die Details eines Proxyservers anzuzeigen und gegebenenfalls dessen Konfiguration zu ändern.

  1. Wenn dies noch nicht geschehen ist, klicken Sie zum Erstellen einer neuen Session auf Erstellen oder zum Aufrufen einer vorhandenen Session auf Bearbeiten. Weitere Informationen finden Sie unter Change Center verwenden.
  2. Suchen Sie den Proxyserver. Siehe Proxyserver suchen.
  3. Klicken Sie auf den Namen des Proxyservers.

    4. Auf der Seite Proxyserver anzeigen - Konfiguration werden die Informationen angezeigt, die in Tabelle 30-6 aufgelistet sind.

    Tabelle 30-6 Proxyserverinformationen 
    Eigenschaft
    Beschreibung
    Zuletzt geändert von
    Der Benutzer, der diese Proxyserverressource erstellt oder geändert hat.
    Zuletzt geändert am
    Datum und Uhrzeit, als der Benutzer diese Proxyserverressource erstellt hat. Klicken Sie auf den Link für Datum und Uhrzeit, um die Änderungshistorie zu dieser Ressource anzuzeigen. Siehe die Seite Änderungshistorie anzeigen.
    Referenzen
    Anzahl Objekte, die dieser Proxyserver referenziert. Falls Referenzen vorhanden sind, klicken Sie auf den Zahlenlink, um eine Liste der Referenzen anzuzeigen. Siehe Referenzen auf Ressourcen anzeigen.
    Referenziert von
    Anzahl Objekte, die auf diesen Proxyserver referenzieren. Falls Referenzen vorhanden sind, klicken Sie auf den Zahlenlink, um eine Liste der Objekte anzuzeigen. Siehe Referenzen auf Ressourcen anzeigen.
    Beschreibung
    Beschreibung dieses Proxyservers, falls vorhanden.
    Host-Port-Parameter
    Details zu jedem Proxyserver in der Proxyserverressource, einschließlich der folgenden Informationen:
    • Serverhost: Hostname oder IP-Adresse des Proxyservers.

      • Der Serverhostname für den Oracle Service Bus-Proxyserver muss identisch sein mit dem Serverhostnamen des tatsächlichen Proxyservers.

    • Klartext-Port: Klartext-Port-Nummer des Proxyservers.
    • SSL-Port: SSL-Port-Nummer des Proxyservers.
    Benutzername
    Der für die Proxyauthentifizierung verwendete Benutzername.

  4. Klicken Sie auf Bearbeiten, um die Seite Proxyserver bearbeiten - [Servername] anzuzeigen.

    5. Das Feld Name kann nicht bearbeitet werden.

  5. Im Feld Beschreibung können Sie die Kurzbeschreibung des Proxyservers bearbeiten.
  6. Im Bereich Host-Port-Parameter können Sie einen neuen Proxyserver hinzufügen, indem Sie die folgenden Informationen eingeben:
    1. Geben Sie im Feld Serverhost den Hostnamen oder die IP-Adresse des Proxyservers ein. Dies ist ein Pflichtfeld.

      2. Der Serverhostname für den Oracle Service Bus-Proxyserver muss identisch sein mit dem Serverhostnamen des tatsächlichen Proxyservers.

    2. Geben Sie im Feld Klartext-Port die Klartext-Port-Nummer des Proxyservers ein.
    3. Geben Sie im Feld SSL-Port die SSL-Port-Nummer des Proxyservers ein. Sie müssen entweder einen Klartext oder die SSL-Port-Nummer eingeben.
    4. Klicken Sie auf Hinzufügen.

      5. Sie können mehrere Proxyserver für jede Proxyserverressource eingeben. So kann Oracle Service Bus Lastverteilung ausführen und Fault-Toleranz-Features für die Proxyserverressource bereitstellen.

      Sie können auch eine vorhandene Proxyserverkonfiguration bearbeiten oder löschen. Gehen Sie dazu folgendermaßen vor:

    • Klicken Sie auf das Symbol Bearbeiten im Feld Optionen desjenigen Proxyservers, den Sie bearbeiten möchten. Nehmen Sie Änderungen in den Feldern Serverhost, Klartext-Port und SSL-Port vor. Klicken Sie dann auf Aktualisieren.
    • Klicken Sie auf das Symbol Löschen im Feld Optionen desjenigen Proxyservers, den Sie löschen möchten.
  7. Sie können die Felder Benutzername und Kennwort, die für die Proxyauthentifizierung verwendet werden, bearbeiten.
  8. Klicken Sie auf Speichern, um die Aktualisierungen in der aktuellen Session zu speichern.
  9. Um die Session zu beenden und die Konfiguration für die Laufzeit bereitzustellen, klicken Sie unter Change Center auf Aktivieren.
