SMTP-Server bearbeiten

Auf dieser Seite können Sie die Details eines SMTP-Servers anzeigen und die Konfiguration nach Bedarf bearbeiten.

  1. Wenn dies noch nicht geschehen ist, klicken Sie zum Erstellen einer neuen Session auf Erstellen oder zum Aufrufen einer vorhandenen Session auf Bearbeiten. Weitere Informationen finden Sie unter Change Center verwenden.
  2. Suchen Sie den SMTP-Server. Siehe SMTP-Server suchen.
  3. Klicken Sie auf den Namen des SMTP-Servers.

    4. Auf der Seite SMTP-Server anzeigen - Konfiguration werden die Informationen aus der Tabelle 30-4 angezeigt.

    Tabelle 30-4 SMTP-Server-Informationen 
    Eigenschaft
    Beschreibung
    Zuletzt geändert von
    Der Benutzer, der diese SMTP-Server-Ressource erstellt oder überarbeitet hat.
    Zuletzt geändert am
    Das Datum oder die Uhrzeit, zu der der Benutzer diese SMTP-Server-Ressource erstellt oder überarbeitet hat. Klicken Sie auf den Link von Datum und Uhrzeit, um die Änderungshistorie dieser Ressource anzuzeigen. Siehe Seite "Änderungshistorie anzeigen".
    Referenzen
    Die Anzahl der Objekte, die dieser SMTP-Server referenziert. Wenn solche Referenzen vorhanden sind, klicken Sie auf den Link mit der Anzahl, um eine Liste der Referenzen anzuzeigen. Siehe Referenzen auf Ressourcen anzeigen.
    Referenziert von
    Die Anzahl der Objekte, die diesen SMTP-Server referenzieren. Wenn solche Referenzen vorhanden sind, klicken Sie auf den Link mit der Anzahl, um eine Liste der Objekte anzuzeigen. Siehe Referenzen auf Ressourcen anzeigen.
    Beschreibung
    Die Beschreibung dieses SMTP-Servers, sofern angegeben.
    Server-URL
    Der URL, der zum Speicherort des SMTP-Servers zeigt.
    Port-Nummer
    Die Port-Nummer dieses SMTP-Servers. Standardmäßig ist dies Port 25.
    Benutzername
    Der Benutzername, mit dem auf einen sicheren SMTP-Server zugegriffen wird.

  4. Klicken Sie auf Bearbeiten, um Seite SMTP-Server bearbeiten - [Servername] anzuzeigen.

    5. Das Feld Name kann nicht bearbeitet werden.

  5. Im Feld Beschreibung können Sie die Kurzbeschreibung des SMTP-Servers bearbeiten.
  6. Im Feld Server-URL können Sie den URL des SMTP-Servers bearbeiten.
  7. Im Feld Port-Nummer können Sie die Port-Nummer bearbeiten.
  8. Sie können die Felder Benutzer und Kennwort bearbeiten.
  9. Klicken Sie auf Speichern, um die Aktualisierungen in der aktuellen Session zu speichern.
  10. Um die Session zu beenden und die Konfiguration für die Laufzeit bereitzustellen, klicken Sie unter Change Center auf Aktivieren.

Wenn Sie eine SMTP-Ressource umbenennen, wird der neue Name automatisch in allen E-Mail-Business-Services aktualisiert, die die SMTP-Ressource im Endpunkt-URI referenzieren.
