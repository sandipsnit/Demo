Anpassungsdateien ausführen

Verwenden Sie diese Seite, um eine XML-Anpassungsdatei auszuführen, die zuvor auf dem System gespeichert wurde. Anpassungsdateien liefern einen bequemen Weg, die Umgebung bei Deployment zu verändern, von der Entwicklung zum Staging, vom Staging zur Produktion oder während der Designzeit.

Sie müssen in einer Session sein, um eine Anpassungsdatei auszuführen.

  1. Wenn dies noch nicht geschehen ist, klicken Sie zum Erstellen einer neuen Session auf Erstellen oder zum Aufrufen einer vorhandenen Session auf Bearbeiten. Weitere Informationen finden Sie unter Change Center verwenden.
  2. Wählen Sie Systemadministration > Anpassungsdatei ausführen.
  3. Wählen Sie eine auszuführende Anpassungsdatei.
    1. Klicken Sie im Feld Dateiname auf Durchsuchen, um das Verzeichnis zu suchen, in dem die Anpassungsdatei gespeichert ist.
    2. Wählen Sie die XML-Datei aus, und klicken Sie auf Öffnen.
  4. Klicken Sie auf Weiter. Folgende Anpassungsdetails werden angezeigt, dargestellt in der Tabelle 31-2.

    5. Tabelle 31-2 Anpassungsdetails
    Eigenschaft
    Beschreibung
    Vorgang
    • Suchen und ersetzen - Ändern Sie Umgebungsvariablen, indem Sie Zeichenfolgen-Ersetzungen durchführen.
    • Referenzen zuordnen - Weisen Sie vorhandene Referenzen zu, die in der Ressource oder anderen Referenzen gefunden wurden.
    • Umgebungsvariable festlegen - Ändern Sie Umgebungsvariablen für Ressourcen
    Anwenden für
    Name des Service, Projekts, Ordners oder der anzupassenden Ressource.
    Der Name ist ein Link zu der Seite Konfigurationsdetails des Service oder der Ressource oder zum Projekt oder Ordner im Project Explorer.
    Beschreibung
    Eine Beschreibung für jede Anpassung.

  5. Damit Sie nur Services, Projekte oder Ordner anpassen, die in der aktuellen Session geändert wurden, aktivieren Sie das Kontrollkästchen Nur in der aktuellen Session geänderte Elemente.

    6. Wenn Sie die Anpassung auf Ressourcen beschränken, die in der aktuellen Session geändert wurden, wird die Spalte Anwenden auf mit den in der aktuellen Session geänderten Ressourcen aktualisiert. Wenn Sie die Anpassung auf ein Projekt oder einen Ordner beschränken, werden die in diesem Projekt oder Ordner in der aktuellen Session geänderten Ressourcen in der Spalte Anwenden auf angezeigt.

  6. Klicken Sie auf Ausführen, um die Aktualisierungen in der aktuellen Session festzuschreiben.

    7. Wenn Sie in der Session Anpassungsdetails anzeigen möchten, wählen Sie Änderungen anzeigen unter Change Center, und klicken Sie auf die Aufgabe Anpassung. Siehe Konfigurationsänderungen anzeigen.

  7. Um die Session zu beenden und die Konfiguration für die Laufzeit bereitzustellen, klicken Sie unter Change Center auf Aktivieren.
