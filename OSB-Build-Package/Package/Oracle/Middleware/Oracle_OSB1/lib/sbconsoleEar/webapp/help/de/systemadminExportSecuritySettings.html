Exportsicherheitseinstellungen angeben

Wenn Ihr Export einen Service-Account oder einen Service-Schlüsselprovider enthält, wird die Seite Ressourcen exportieren - Sicherheitseinstellungen angezeigt.

Oracle Service Bus exportiert für jeden Service-Account den Benutzernamen und das Kennwort oder die Zuordnung zwischen lokalem und Remote-Benutzer (je nachdem, welche Daten im Service-Account gespeichert waren). Für jeden Service-Schlüsselprovider exportiert Oracle Service Bus den Alias für das Schlüsselpaar-Binding vom Provider der PKI-Zugangsdatenzuordnungen. Private Schlüssel, Zertifikate oder andere Daten aus den Key Stores werden nicht exportiert. Key Store-Daten müssen mit den Tools des Key Store-Lieferanten exportiert werden.

Wenn Sie möchten, dass Oracle Service Bus die Benutzernamen- und Kennwortdaten in den Ressourcen von Service-Account, Service-Schlüsselprovider, UDDI-Registry, JNDI-und SMTP-Provider vor dem Export in eine JAR-Datei verschlüsselt, führen Sie folgende Schritte aus:

  1. Aktivieren Sie das Kontrollkästchen Sensible Daten schützen.
  2. Geben Sie ein Kennwort ein, und bestätigen Sie es.

    3. Wenn Sie oder eine andere Person versuchen, diese JAR-Datei zu importieren, importiert Oracle Service Bus die Ressourcen mit den verschlüsselten Benutzernamen- und Kennwortdaten nur, wenn Sie dieses Kennwort angeben. Alle nicht verschlüsselten Ressourcen in der JAR-Datei können ohne Angabe des Kennwortes importiert werden.

  3. Klicken Sie auf die Schaltfläche Export beenden.
  4. Wenn Sie auf "Export" oder "Export beenden" klicken, wird das Popup-Fenster Konfigurationsdaten werden verarbeitet... geöffnet. Warten Sie, bis sich dieses Fenster wieder schließt, bevor Sie andere Aktionen in der Oracle Service Bus Administration-Konsole durchführen. Das Fenster wird automatisch geschlossen, wenn das Dialogfeld Herunterladen von Dateien angezeigt wird.
  5. Klicken Sie im Dialogfeld Herunterladen von Dateien auf Öffnen, um die exportierte JAR-Datei zu öffnen, oder klicken Sie auf Speichern, um die JAR-Datei auf Ihrem Desktop zu speichern.
