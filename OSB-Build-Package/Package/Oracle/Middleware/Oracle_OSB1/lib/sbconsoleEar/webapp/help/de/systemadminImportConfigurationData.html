Ressourcen importieren

Auf der Seite Ressourcen importieren können Sie die JAR-Datei eines vollständigen Projekts oder einer Ressource importieren, die zuvor aus einer anderen Oracle Service Bus-Domain exportiert wurde. Sie können nur einen Teil der exportierten Daten importieren. Wenn die Ressource bereits im importierenden System vorhanden ist, wird sie aktualisiert. Andernfalls wird sie erstellt. Ressourcen werden nur für das Löschen geplant, wenn es sich bei der importierten JAR-Datei um eine JAR-Datei des vollständigen Projekts handelt und in demselben Projekt im importierenden System Ressourcen enthalten sind, die in der importierten JAR-Datei nicht vorhanden sind. Es werden keine Ressourcen aus anderen Projekten gelöscht.

Sie können Ressourcen auf Ressourcenebene importieren, auch wenn sie auf Projektebene exportiert wurden. Beispiel: Auch wenn Ressourcen in einer JAR-Datei des vollständigen Projekts exportiert wurden, können Sie die Auswahl für diese Ressourcen beim Importieren aufheben.

Beim Exportieren einer Konfiguration können Sie keine Benutzer, Gruppen, Rollen oder Zertifikate exportieren. Aus diesem Grund müssen Sie diese Objekte erneut erstellen, wenn Sie eine exportierte Konfiguration importieren.

Hinweis: Weitere Informationen zum Importieren von Ressourcen, die nicht Bestandteil einer Oracle Service Bus-Konfiguration sind, finden Sie unter Ressourcen aus einer Zip-Datei laden und unter Ressourcen aus einem URL laden.
  1. Wenn dies noch nicht geschehen ist, klicken Sie zum Erstellen einer neuen Session auf Erstellen oder zum Aufrufen einer vorhandenen Session auf Bearbeiten. Weitere Informationen finden Sie unter Change Center verwenden.
  2. Wählen Sie Systemverwaltung > Ressourcen importieren.
  3. Wählen Sie die JAR-Datei eines exportierten Projekts oder einer exportierten Ressource.
    1. Klicken Sie im Feld Dateiname auf Durchsuchen, um das Verzeichnis zu suchen, in dem die JAR-Datei gespeichert ist.
    2. Wählen Sie die JAR-Datei, und klicken Sie auf Öffnen.
  4. Klicken Sie auf Weiter. Fahren Sie mit dem Abschnitt Zu importierende Projekte und Ressourcen wählen fort.
