Sicherheitseinstellungen für den Import festlegen

Wenn die JAR-Datei Ressourcen mit Benutzername- und Kennwortdaten enthält, die beim Export verschlüsselt wurden, wird die Seite Ressourcen importieren – Sicherheitseinstellungen angezeigt.

  1. Geben Sie das Kennwort ein, das für die Verschlüsselung der Daten verwendet wurde. (Weitere Informationen finden Sie unter Ressourcen exportieren).

    2. Wenn Sie das Kennwort, das für die Verschlüsselung der Ressourcendaten verwendet wurde, nicht kennen, klicken Sie auf die Schaltfläche Zurück, und deaktivieren Sie das Kontrollkästchen neben der Ressource. Sie können alle anderen nicht verschlüsselten Ressourcen importieren, ohne das Kennwort zu kennen.

  2. Klicken Sie auf Import beenden.

    3. Das Popup-Fenster Konfigurationsdaten werden verarbeitet wird angezeigt. Warten Sie, bis dieses Fenster geschlossen wird, bevor Sie weitere Schritte in der Oracle Service Bus Administration-Konsole ausführen. Das Fenster wird automatisch geschlossen, wenn das System mit dem Importieren der Objekte fertig ist. Anschließend wird auf der Seite Ressourcen importieren eine Nachricht angezeigt, die Sie darüber informiert, dass Sie die Daten erfolgreich bereitgestellt haben.
