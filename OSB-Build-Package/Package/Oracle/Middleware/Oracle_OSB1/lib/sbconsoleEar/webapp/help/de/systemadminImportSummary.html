Importzusammenfassung prüfen

In der Tabelle 28-3 wird die Importzusammenfassung angezeigt.

Tabelle 28-Importzusammenfassungsinformation 
Eigenschaft
Beschreibung
Status
Ob die Ressource erfolgreich importiert wurde oder nicht.
Eine WSDL kann vorübergehend als ungültig markiert werden, bis alle Abhängigkeiten importiert wurden. Nach Abschluss der Importaktion werden die als ungültig markierten Dateien allerdings vom Konfigurations-Service erneut geprüft.
Name
Name der Ressource.
Pfad
Der Pfad der Ressource. Der Pfad ist der Name des Projekts und des Ordners, in dem die Ressource gespeichert ist.
Typ
Zeigt die Oracle Service Bus-Ressourcentypen an.
Diagnosemeldung
Zeigt gegebenenfalls Fehler oder Informationen für diese Ressource an.

  1. Klicken Sie auf Weitere importieren.

    2. Sie können eine weitere JAR-Datei importieren, oder Sie klicken auf Änderungen anzeigen unter Change Center, um die Importdetails anzuzeigen. Siehe Konfigurationsänderungen anzeigen.

  2. Wenn Sie fertig sind, klicken Sie auf Aktivieren unter Change Center.
