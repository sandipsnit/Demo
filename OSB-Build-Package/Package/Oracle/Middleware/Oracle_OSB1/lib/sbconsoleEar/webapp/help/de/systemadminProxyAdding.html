Proxyserver hinzufügen

Über die Seite Übersicht der Proxyserver können Sie Proxyserverressourcen hinzufügen und konfigurieren und diese in Oracle Service Bus als Systemressource bereitstellen. Die Session, in der Sie sich befinden, muss aktiv sein, um Proxyserverressourcen zu konfigurieren bzw. neu zu konfigurieren.

  1. Wenn dies noch nicht geschehen ist, klicken Sie zum Erstellen einer neuen Session auf Erstellen oder zum Aufrufen einer vorhandenen Session auf Bearbeiten. Weitere Informationen finden Sie unter Change Center verwenden.
  2. Wählen Sie Systemadministration > Proxyserver.
  3. Klicken Sie auf Hinzufügen.
  4. Geben Sie auf der Seite Name einen Namen für die Proxyserverressource ein. Dies ist ein obligatorisches Feld.

    Befolgen Sie die Benennungseinschränkungen für Ressourcen bei der Namensgebung.
  5. Geben Sie im Feld Beschreibung eine Kurzbeschreibung der Proxyserverressource ein.
  6. Geben Sie im Bereich Host-Port-Parameter folgende Informationen ein.
    1. Geben Sie im Feld Serverhost den Hostnamen oder die IP-Adresse des Proxyservers ein. Dies ist ein Pflichtfeld.

      2. Der Serverhostname für den Oracle Service Bus-Proxyserver muss identisch sein mit dem Serverhostnamen des tatsächlichen Proxyservers.

    2. Geben Sie im Feld Klartext-Port die Klartext-Port-Nummer des Proxyservers ein.
    3. Geben Sie im Feld SSL-Port die SSL-Port-Nummer des Proxyservers ein. Sie müssen entweder einen Klartext oder die SSL-Port-Nummer eingeben.
    4. Klicken Sie auf Hinzufügen.

      5. Sie können mehrere Proxyserver für jede Proxyserverressource eingeben. So kann Oracle Service Bus Lastverteilung ausführen und Fault-Toleranz-Features für die Proxyserverressource bereitstellen.

  7. Wenn der Proxyserver eine Proxyauthentifizierung vornimmt, geben Sie einen Benutzernamen im Feld Benutzername und das zugehörige Kennwort in den Feldern Kennwort und Kennwort bestätigen ein.

    8. Dies sind fakultative Felder. Sie werden nur zu Pflichtfeldern, wenn der Proxyserver gesichert ist.

  8. Klicken Sie auf Speichern, um die Proxyserverressource in der aktuellen Session zu erstellen und zu speichern.
  9. Um die Session zu beenden und die Konfiguration für die Laufzeit bereitzustellen, klicken Sie unter Change Center auf Aktivieren.
