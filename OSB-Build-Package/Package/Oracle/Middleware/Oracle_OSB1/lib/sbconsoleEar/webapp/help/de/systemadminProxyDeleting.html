Proxyserver löschen

  1. Wenn dies noch nicht geschehen ist, klicken Sie zum Erstellen einer neuen Session auf Erstellen oder zum Aufrufen einer vorhandenen Session auf Bearbeiten. Weitere Informationen finden Sie unter Change Center verwenden.
  2. Wählen Sie Systemadministration > Proxyserver.
  3. Wählen Sie auf der Seite Übersicht der Proxyserver einen Proxyserver aus. Weitere Informationen finden Sie unter SMTP-Server suchen.
  4. Klicken Sie im Feld Optionen des zu löschenden Proxyservers auf das Symbol Löschen. Der Proxyserver wird in der aktuellen Session gelöscht. Wenn andere Ressourcen diese Ressource referenzieren, wird das Symbol Warnung wegen Löschvorgang angezeigt. Sie können die Ressource löschen, indem Sie den Löschvorgang bestätigen. Dies kann aufgrund von nicht aufgelösten Referenzen mit der gelöschten Ressource zu Konflikten führen.
  5. Um die Session zu beenden und die Konfiguration für die Laufzeit bereitzustellen, klicken Sie unter Change Center auf Aktivieren.
