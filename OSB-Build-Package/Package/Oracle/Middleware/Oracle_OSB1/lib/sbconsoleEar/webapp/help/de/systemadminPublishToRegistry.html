Proxy-Services in einer UDDI-Registry publizieren

Sie können einen Service in einer Registry publizieren, damit andere Organisationen den Service finden und verwenden können. Alle in Oracle Service Bus entwickelten Proxy-Services können in einer UDDI-Registry publiziert werden. Sie können die Business-Entität wählen, unter der der Service publiziert werden soll, und Sie können mehrere Services gleichzeitig publizieren.

Vor dem Publizieren in einer Registry müssen Sie über einen Account bei dieser Registry verfügen. Oracle Service Bus unterstützt die Interoperabilität mit UDDI-Registrys, die mit Version 3.0 kompatibel sind. Die in diesem Abschnitt beschriebene Konfiguration setzt voraus, dass Sie die Oracle Service Registry verwenden.

Informationen zum Einrichten eines Benutzer-Accounts in der Oracle Service Registry finden Sie in den Produktinformationen der Oracle Service Registry unter http://www.oracle.com/technology/products/soa/registry/index.html.

Sie können die Veröffentlichung eines Service in einer Registry bei der Installation der Oracle Service Registry aufheben.

  1. Wählen Sie Systemverwaltung > In UDDI publizieren.

    2. Sie können nur außerhalb einer Session publizieren. Beenden Sie die aktuelle Session, um auf die Liste mit Registrys zuzugreifen. Alle Registrys, auf die Sie Zugriff haben, werden in einer Dropdown-Liste mit gewählten Registrys angezeigt.

  2. Wählen Sie unter Publishing Registry-Name den Namen der Registry aus, in der Sie publizieren möchten, und klicken Sie auf Weiter.

    3. Wenn in Oracle Service Bus eine einzige Registry konfiguriert ist, wird die anfängliche Seite zum Wählen der Registry nicht angezeigt, da es sich hierbei um die Standard-Registry handelt. Die Seite zum Wählen der Registry wird nur angezeigt, wenn mindestens zwei Registrys definiert sind.

    Die Seite In UDDI publizieren – Einzelne Services wählen und publizieren wird mit einer Liste mit Oracle Service Bus-Proxy-Services und ihren Speicherorten im Projektordner angezeigt. Beginnend mit dem obersten Projektordner werden die Details für Name, Typ und Beschreibung für alle Services angezeigt.

  3. Wählen Sie die zu publizierenden Services aus.
    1. Erweitern Sie den Ordner Projekt, um die definierten Proxy-Services anzuzeigen. Der Ordner und seine Inhalte sind standardmäßig für die Publikation markiert.
    2. Sie können einzelne Elemente zum Publizieren wählen.
  4. Wählen Sie aus der Dropdown-Liste Services in Business-Entität publizieren die Business-Entität aus, unter der Sie den Service in der Registry hinzufügen möchten. Dies ist die Entität, unter der der Service in der Registry klassifiziert wird.
  5. Klicken Sie auf Publizieren, um die Services zu publizieren.

    6. In einer Bestätigungsmeldung wird angezeigt, dass der Service erfolgreich publiziert wurde.

  6. Klicken Sie auf Weitere publizieren, um zur Seite In UDDI publizieren – Registry wählen zurückzukehren.
