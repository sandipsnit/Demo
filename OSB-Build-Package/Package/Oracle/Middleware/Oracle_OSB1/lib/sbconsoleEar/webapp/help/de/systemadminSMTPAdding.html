SMTP-Server hinzufügen

Auf der Seite Zusammenfassung SMTP-Server können Sie SMTP-Server-Ressourcen hinzufügen, konfigurieren und sie in Oracle Service Bus als Systemressource zur Verfügung stellen. Sie müssen sich in einer aktiven Session befinden, um SMTP-Server-Ressourcen konfigurieren oder neu konfigurieren zu können.

  1. Wenn dies noch nicht geschehen ist, klicken Sie zum Erstellen einer neuen Session auf Erstellen oder zum Aufrufen einer vorhandenen Session auf Bearbeiten. Weitere Informationen finden Sie unter Change Center verwenden.
  2. Wählen Sie Systemadministration > SMTP-Server.
  3. Klicken Sie auf Hinzufügen.
  4. Geben Sie im Feld Name einen Namen für die SMTP-Serverressource ein. Dies ist ein obligatorisches Feld.

    Befolgen Sie die Benennungseinschränkungen für Ressourcen bei der Namensgebung.
  5. Geben Sie im Feld Beschreibung eine Kurzbeschreibung für die SMTP-Server-Ressource ein.
  6. Geben Sie im Feld Server-URL den URL ein, der zum SMTP-Server zeigt. In dieses Feld muss ein Wert eingegeben werden.
  7. Geben Sie im Feld Port-Nummer eine Port-Nummer für den SMTP-Server ein (Standard-Port ist 25). In dieses Feld muss ein Wert eingegeben werden.
  8. Wenn für den Zugriff auf den Ziel-SMTP-Server Benutzername und Kennwort benötigt werden, geben Sie im Feld Benutzername einen Benutzernamen und das zugehörige Kennwort in die Felder Kennwort und Kennwort bestätigen ein.

    9. Eine Eingabe in diese Felder ist optional, solange der SMTP-Server nicht gesichert ist.

  9. Klicken Sie auf Speichern, um die SMTP-Server-Ressource in der aktuellen Session zu erstellen und zu speichern.
  10. Um die Session zu beenden und die Konfiguration für die Laufzeit bereitzustellen, klicken Sie unter Change Center auf Aktivieren.
