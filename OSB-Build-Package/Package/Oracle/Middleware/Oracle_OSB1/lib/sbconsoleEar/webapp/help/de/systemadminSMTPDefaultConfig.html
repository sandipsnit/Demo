Standard-SMTP-Server konfigurieren

Sie können einen der konfigurierten SMTP-Server als Standardserver für die Domain festlegen. Siehe SMTP-Server hinzufügen.

  1. Wenn dies noch nicht geschehen ist, klicken Sie zum Erstellen einer neuen Session auf Erstellen oder zum Aufrufen einer vorhandenen Session auf Bearbeiten. Weitere Informationen finden Sie unter Change Center verwenden.
  2. Wählen Sie Systemadministration > SMTP-Server.
  3. Klicken Sie im Bereich "Standard-SMTP-Server" auf Standard wählen.
  4. Wählen Sie aus der Dropdown-Liste Standard-SMTP-Server einen SMTP-Server.
  5. Klicken Sie auf Als Standard speichern.

    6. Die Seite Zusammenfassung SMTP-Server wird mit dem gewählten Server als Standard-SMTP-Server für die Domain angezeigt.

  6. Um die Session zu beenden und die Konfiguration für die Laufzeit bereitzustellen, klicken Sie unter Change Center auf Aktivieren.
