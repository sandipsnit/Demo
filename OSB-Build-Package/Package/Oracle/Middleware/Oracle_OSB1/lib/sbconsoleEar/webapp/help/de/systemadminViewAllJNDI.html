JNDI-Provider anzeigen

Sie verwenden diese Seite, um eine Liste der JNDI-Provider und die Informationen aus Tabelle 30-1 anzuzeigen.

Außerdem können Sie nach JNDI-Providern suchen, die Sie zuvor konfiguriert haben.

  1. So beschränken Sie die Anzahl Elemente in der Liste oder suchen nach speziellen JNDI-Providern:
    • Klicken Sie auf das Symbol Öffnen, um zusätzliche Suchfilter anzuzeigen. Wenn Sie mehrere Suchkriterien angeben, werden nur Ergebnisse angezeigt, die allen Kriterien entsprechen.
    • Geben Sie den Namen der JNDI-Provider ein, nach denen Sie im Feld Name suchen möchten. Klicken Sie auf Suchen, um nach einem bestimmten Eintrag zu suchen.

      • Sie können Platzhalter (? für ein Zeichen; * für mehrere Zeichen) für eine allgemeinere Suche eingeben.

    • Geben Sie den Provider-URL des gesuchten JNDI-Providers ein, und klicken Sie auf Suchen.
  2. Klicken Sie auf Alle anzeigen, um die Suchfilter zu entfernen und alle JNDI-Providerressourcen anzuzeigen, die für eine Verwendung mit Oracle Service Bus konfiguriert wurden.

    3. Tabelle 30-1 zeigt die Konfigurationseinstellungen der UDDI-Registry.

    Tabelle 30-1 JNDI-Providerinformationen 
    Eigenschaft
    Beschreibung
    Name
    Eindeutiger Name dieses JNDI-Providers. Der Name ist ein Link zur Seite JNDI-Provider anzeigen - Konfiguration. Siehe JNDI-Provider bearbeiten.
    Provider-URL
    URL des JNDI-Providers. Siehe JNDI-Provider hinzufügen.
    JNDI-Cache
    Aktiviert oder Deaktiviert. Ist diese Option aktiviert, werden der JNDI-Kontext und JNDI-Objekte lokal gecacht, wodurch die Performance beim Objekt-Lookup verbessert wird. Oracle empfiehlt, den JNDI-Cache zu aktivieren.
    Aktionen
    • Klicken Sie auf das Symbol Zurücksetzen, um den JNDI-Kontext zurückzusetzen, damit die JNDI-Verbindung sowie die lokal gecachten Objekte verworfen werden.
    • Klicken Sie auf das Symbol Testkonsole starten, um herauszufinden, ob der JNDI-Provider erfolgreich eine Verbindung herstellen und einen JNDI-Kontext abrufen kann.
    Optionen
    • Klicken Sie auf das Symbol Umbenennen, um den JNDI-Provider umzubenennen.
    • Klicken Sie auf das Symbol Löschen, um einen bestimmten JNDI-Provider zu löschen. Siehe JNDI-Provider löschen.
