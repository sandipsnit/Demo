Proxyserver anzeigen

Auf dieser Seite können Sie eine Liste der Proxyserver und der Daten aus Tabelle 30-5 anzeigen.

Proxyserverressourcen werden bei der Konfiguration des HTTP-Transports für Business-Services verwendet. Weitere Informationen finden Sie auf der Seite HTTP-Transportkonfiguration.

Tabelle 30-5 zeigt die Proxyserverinformationen.

Tabelle 30-5 Proxyserverinformationen.   
Eigenschaft
Beschreibung
Name
Der eindeutige Name, der dieser Proxyserverressource zugeordnet ist. Der Name ist ein Link zur Seite Proxyserverkonfiguration. Siehe Proxyserver bearbeiten.
Optionen
  • Klicken Sie auf das Symbol Umbenennen, um diese Proxyserverressource umzubenennen.
  • Klicken Sie auf das Symbol Löschen, um einen bestimmten Proxyserver zu löschen. Siehe Proxyserver löschen.
