SMTP-Server anzeigen

Auf dieser Seite können Sie einen Standard-SMTP-Server festlegen (siehe Standard-SMTP-Server konfigurieren) und eine Liste mit SMTP-Servern anzeigen.

SMTP-Serverressourcen werden beim Konfigurieren von Alert-Zielen und Business-Services, die auf E-Mail-Transport basieren, verwendet. Weitere Informationen finden Sie in Alert-Ziele hinzufügen und Seite "Einen Business-Service erstellen/bearbeiten – E-Mail-Transportkonfiguration".

In Tabelle 30-3 werden Informationen zum SMTP-Server angezeigt.

Tabelle 30-3 Informationen zum SMTP-Server 
Eigenschaft
Beschreibung
Name
Der dieser SMTP-Serverressource zugewiesene eindeutige Name. Der Name ist ein Link zur Seite SMTP-Server anzeigen – Konfiguration. Weitere Informationen finden Sie in SMTP-Server bearbeiten.
Server-URL
Der URL, der auf den SMTP-Server verweist. Weitere Informationen finden Sie in SMTP-Server hinzufügen.
Optionen
  • Klicken Sie auf das Symbol Umbenennen, um diese SMTP-Serverressource umzubenennen.
  • Klicken Sie auf das Symbol Löschen, um einen bestimmten SMTP-Server zu löschen. Weitere Informationen finden Sie unter SMTP-Server löschen.
