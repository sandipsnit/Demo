Verwendung der Transporteinstellungen in der Testkonsole zur Laufzeit

Unter Proxy-Service-Testdaten konfigurieren und Business-Service-Testdaten konfigurieren wird beschrieben, wie Sie die Werte der Transport-Header, Transportmetadaten und transportbezogenen Sicherheitsdaten für ausgehende Anforderungen beim Testen von Proxy-Services oder Business-Services in der Testkonsole konfigurieren. Einige der Spezifikationen, die Sie in der Testkonsole festlegen können, werden jedoch zur Laufzeit nicht berücksichtigt. Das heißt, die Werte von bestimmten Headern oder Metadaten werden überschrieben oder von Oracle Service Bus zur Laufzeit ignoriert, wenn der Test ausgeführt wird. Die Header und Metadaten, für die Einschränkungen bestehen, sind in Tabelle 32-10 beschrieben.
