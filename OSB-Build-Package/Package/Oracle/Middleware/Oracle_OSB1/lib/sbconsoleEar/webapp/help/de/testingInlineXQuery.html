XQuery-Tests durchführen

Mit den folgenden Editoren können Sie eine Aktion im Nachrichtenfluss bearbeiten und testen: XQuery-Ausdruckseditor, XQuery-Bedingungseditor und XPath-Ausdruckseditor. Beim XQuery-Ausdruckseditor und -Bedingungseditor werden die Tests in der gleichen Form ausgeführt. Beim XPath-Ausdruckseditor ist das Szenario allerdings anders, weil nur eine Eingabe berücksichtigt wird.

Hinweis: Damit die XQuery-Tests funktionieren, müssen Sie die Popup-Blocker in Ihrem Browser deaktivieren. Wenn Sie Symbolleisten im Internet Explorer-Browser haben, müssen Sie möglicherweise die Popup-Blocker im Menü Optionen sowie für alle Symbolleisten deaktivieren, die zum Blockieren konfiguriert sind.

Dieser Abschnitt enthält die folgenden Themen:

  • XQuery-Ausdruckseditor und XQuery-Bedingungseditor verwenden
  • XPath-Ausdruckseditor verwenden
