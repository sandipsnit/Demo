Transformationen testen

Sie können Transformationen nach dem Aktivieren einer Session oder während einer Session testen, um sicherzustellen, dass die Ressourcen mit dem erwarteten Verhalten ausgeführt werden. Sie müssen die Session aktivieren, damit die Tests zur Laufzeit durchgeführt werden. Andernfalls werden die Tests mit Ihren lokalen Änderungen zur Entwurfszeit durchgeführt.

Sie können folgende Transformationen testen:

  • MFL-Transformationen testen
  • XSLT-Transformationen testen
  • XQuery-Transformationen testen
