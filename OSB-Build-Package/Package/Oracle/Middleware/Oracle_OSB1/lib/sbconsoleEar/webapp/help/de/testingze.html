XQuery-Transformationen testen

XQuery-Zuordnungen können XML-zu-XML-, XML-zu-Nicht-XML-, und Nicht-XML-zu-XML-Zuordnungen beschreiben. Eine XQuery-Transformation kann mehrere Eingaben verarbeiten und gibt eine Ausgabe zurück. Jede Eingabe entspricht einer externen XQuery-Variablen, die in der XQuery-Ressource deklariert ist. Bei dem Wert einer XQuery-Eingabevariablen kann es sich um einen einfachen primitiven Datentyp (Zeichenfolge, Ganzzahl, Datum und so weiter), ein XML-Dokument oder eine Sequenz dieser Typen handeln. Bei dem Ausgabewert kann es sich um einen einfachen primitiven Wert (Zeichenfolge, Ganzzahl, Datum und so weiter), ein XML-Dokument oder eine Sequenz dieser Typen handeln.

Hinweis: Die Testkonsole unterstützt keine Sequenzen bei der Eingabe.

Dieser Abschnitt enthält ein Eingabefeld für jede externe XQuery-Variable.
<param_name> ([] als XML)
param_name ist der Name einer XQuery-Variablen in der XQuery-Ressource.
In der Testkonsole wird ein einzeiliges Bearbeitungsfeld angezeigt, wenn es sich um einen einfachen Typ handelt. Bei XML-Daten wird ein mehrzeiliges Bearbeitungsfeld angezeigt.
Eine Kombinationseingabe (<param_name> ([] as XML)) wird verwendet, wenn für die Variable kein Typ angegeben ist. Sie müssen den Variablentyp deklarieren. Aktivieren Sie das Kontrollkästchen, um einen Parameter des Typs "XML" anzugeben.
Eine XML-Eingabe kann dateibasiert oder textbasiert sein. Ein Verweis auf eine Datei für die Eingabe hat Vorrang vor einer Texteingabe. Suchen und markieren Sie die Datei, die Sie im Test verwenden möchten.
Eingaben in der Testkonsole werden basierend auf dem Typ angezeigt, um verständlicher zu machen, welchen Datentyp Sie eingeben müssen. Erfolgt keine Eingabe, wird der Standardtyp "XML" verwendet.

Sie können die Entwurfszeit oder die Laufzeit testen.

  1. Klicken Sie auf Aktivieren, wenn Sie die Laufzeit testen möchten. Aktivieren Sie die Session nicht, um die Entwurfszeit zu testen.
  2. Wählen Sie Project Explorer > XQuerys, um die Seite Übersicht der XQuerys anzuzeigen.
  3. Klicken Sie unter Aktionen auf das Symbol Testkonsole starten, das mit der zu testenden Ressource verknüpft ist. Auf der Testkonsole wird die Seite Ressource testen geöffnet.
  4. Konfigurieren Sie die Testdaten für die Ressource im Abschnitt Variablen der Seite. Weitere Informationen finden Sie in Tabelle 32-7.
  5. Klicken Sie auf Ausführen:. Auf der Seite Ressource testen werden die Ergebnisse angezeigt.
  6. Klicken Sie auf Zurück, um erneut zu prüfen. Sie können die Testkonsole schließen, ändern und die Ressource erneut prüfen.
