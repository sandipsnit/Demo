UDDI-Registrys bearbeiten

  1. Wenn dies noch nicht geschehen ist, klicken Sie zum Erstellen einer neuen Session auf Erstellen oder zum Aufrufen einer vorhandenen Session auf Bearbeiten. Weitere Informationen finden Sie unter Change Center verwenden.
  2. Wählen Sie Systemadministration > UDDI-Registrys.
  3. Klicken Sie im Fenster "UDDI-Registrys" in der Liste der verfügbaren Registrys auf den Namen der Registry, die Sie bearbeiten möchten.

    4. Sie können ebenfalls nach einer bestimmten Registry über die Option Suchen suchen. Siehe Konfigurierte UDDI-Registrys anzeigen.

    Auf der Seite UDDI-Registry-Konfiguration werden die Konfigurationseigenschaften angezeigt, die sowohl die Registry als auch die folgenden Registry-spezifischen Informationen festlegen, die in Tabelle 29-2 aufgelistet sind.

    Tabelle 29-2 UDDI-Registry-Informationen 
    Eigenschaft
    Beschreibung
    Zuletzt geändert von
    Benutzer, der die letzte Konfigurationsänderung an dieser UDDI-Registry-Ressource vorgenommen hat.
    Zuletzt geändert am
    Datum und Uhrzeit der letzten Konfigurationsänderung. Klicken Sie auf den Link für Datum und Uhrzeit, um die Änderungshistorie zu dieser Ressource anzuzeigen. Siehe die Seite Änderungshistorie anzeigen.
    Referenzen
    Anzahl Objekte, auf die diese UDDI-Registry referenziert. Falls Referenzen vorhanden sind, klicken Sie auf den Zahlenlink, um eine Liste der Referenzen anzuzeigen. Siehe Referenzen auf Ressourcen anzeigen.
    Referenziert von
    Anzahl Objekte, die diese UDDI-Registry referenzieren. Falls Referenzen vorhanden sind, klicken Sie auf den Zahlenlink, um eine Liste der Objekte anzuzeigen. Siehe Referenzen auf Ressourcen anzeigen.
    Beschreibung
    Beschreibung der Registry.

  4. Klicken Sie unten auf der Seite auf Bearbeiten.
  5. Bearbeiten Sie auf der Seite UDDI-Konfiguration - Registry bearbeiten die Registry-Konfigurationsparameter.

    6. Diese Eigenschaften sind in Tabelle 29-1 beschrieben. Sie können den Namen der Registry nicht bearbeiten.

  6. Klicken Sie auf Speichern, um die Aktualisierungen in der aktuellen Session zu speichern.
  7. Um die Session zu beenden und die Konfiguration für die Laufzeit bereitzustellen, klicken Sie unter Change Center auf Aktivieren.
