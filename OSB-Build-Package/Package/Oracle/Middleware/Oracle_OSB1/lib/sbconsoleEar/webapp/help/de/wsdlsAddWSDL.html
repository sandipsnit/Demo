WSDLs hinzufügen

  1. Wenn dies noch nicht geschehen ist, klicken Sie zum Erstellen einer neuen Session auf Erstellen oder zum Aufrufen einer vorhandenen Session auf Bearbeiten. Weitere Informationen finden Sie unter Change Center verwenden.
  2. Wählen Sie Project Explorer und dann ein Projekt oder einen Ordner, in dem Sie die WSDL hinzufügen möchten. Die Seite Projektansicht/Ordneransicht wird angezeigt.
  3. Wählen Sie von der Dropdown-Liste Ressource erstellen den Wert WSDL unter Schnittstelle, um die Seite Neue WSDL-Ressource erstellen anzuzeigen.
  4. Geben Sie im Feld Ressourcenname einen eindeutigen Namen für diese WSDL ein. Dies ist ein obligatorisches Feld.

    Befolgen Sie die Benennungseinschränkungen für Ressourcen bei der Namensgebung.
  5. Im Feld Ressourcenbeschreibung geben Sie eine Beschreibung dieser WSDL ein.
  6. Führen Sie im Feld WSDL einen der folgenden Schritte aus:
    • Geben Sie Text für die neue WSDL ein.
    • Klicken Sie auf Durchsuchen, wenn Sie eine vorhandene WSDL suchen und importieren möchten.
    • Kopieren Sie den Text einer vorhandenen WSDL, und fügen Sie ihn in dieses Feld ein:

      • Dieses Feld ist erforderlich.

  7. Klicken Sie auf Speichern. Die neue WSDL wird in der aktuellen Session gespeichert.

    8. Wenn Sie auf Speichern klicken, werden alle nicht aufgelösten Referenzen für die neue WSDL vom System angezeigt. Siehe Nicht aufgelöste WSDL-Referenzen anzeigen.

  8. Um die Session zu beenden und die Konfiguration für die Laufzeit bereitzustellen, klicken Sie unter Change Center auf Aktivieren.
