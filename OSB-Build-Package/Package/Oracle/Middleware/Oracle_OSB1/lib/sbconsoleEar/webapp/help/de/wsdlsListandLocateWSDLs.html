WSDLs suchen

  1. Führen Sie eine der folgenden Aktionen aus:
    • Wählen Sie Project Explorer, um die Seite "Projektansicht" oder die Seite "Projekt-/Ordneransicht" anzuzeigen. Navigieren Sie dann durch die Projekte und Ordner, um die WSDL zu suchen.
    • Wählen Sie Ressourcenbrowser > WSDLs. Auf der Seite Übersicht der WSDLs werden die in Tabelle 6-1 dargestellten Informationen angezeigt. Eine ausführlichere Beschreibung der Eigenschaften finden Sie unter WSDLs bearbeiten.
  2. So finden Sie eine bestimmte WSDL:
    • Klicken Sie auf das Symbol Öffnen, um zusätzliche Suchfilter anzuzeigen. Wenn Sie mehrere Suchkriterien angeben, werden nur Ergebnisse angezeigt, die allen Kriterien entsprechen.
    • Um die Anzahl der Elemente in der Liste einzuschränken, können Sie nach Namen, Pfad und Namespace filtern. Geben Sie in den Feldern Name, Pfad und Namespace den Namen, den Pfad und den Namespace für die Suchziele ein, und klicken Sie auf Suchen.

      • Der Pfad ist der Projektname und der Name des Ordners, in dem die WSDL gespeichert ist.

    • Um die Suchfilter zu entfernen und alle WSDLs anzuzeigen, klicken Sie auf Alle anzeigen.

      • Tabelle 6-1 WSDL-Details
      Eigenschaft
      Beschreibung
      WSDL-Name
      Der eindeutige Name, der der WSDL zugeordnet ist. Der Name ist ein Link zur Seite WSDL-Details anzeigen. Weitere Informationen finden Sie unter WSDLs bearbeiten.
      Pfad
      Der Pfad ist der Projektname und der Name des Ordners, in dem die WSDL gespeichert ist. Er ist ein Link zu dem Projekt oder dem Ordner, in dem diese Ressource enthalten ist. Weitere Informationen finden Sie unter Ressourcennamen mit Projekten und Ordnern angeben.
      WSDL-Namespace
      Die Adresse der WSDL. Sie wird in folgendem Format angezeigt:
      http://example.com/example/example
      Optionen
      Klicken Sie auf das Symbol Löschen, um eine bestimmte WSDL zu löschen. Weitere Informationen finden Sie unter WSDLs löschen.
