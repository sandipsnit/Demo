Nicht aufgelöste WSDL-Referenzen anzeigen

Auf der Seite Nicht aufgelöste WSDL-Referenzen anzeigen werden die in Tabelle 6-5 beschriebenen Informationen angezeigt.

Tabelle 6-5 Details von nicht aufgelösten WSDL-Referenzen 
Eigenschaft
Beschreibung
WSDL-Name
Der der WSDL-Ressource zugewiesene Name. Klicken Sie auf den Namen einer bestimmten WSDL-Ressource, um Details dieser Ressource anzuzeigen.
Namespace
Die Adresse der WSDL. Sie wird in folgendem Format angezeigt:
http://example.com/example/example

So finden Sie eine bestimmte Referenz:

  • Sortieren Sie die Liste neu. Klicken Sie auf einen unterstrichenen Spaltennamen. Nach oben und nach unten gerichtete Pfeile geben die auf- oder absteigende Sortierreihenfolge an. Klicken Sie auf den Spaltennamen, um die Sortierreihenfolge zu ändern.
  • Rollen Sie durch die Seiten. Verwenden Sie die Seitensteuerung über oder unter der Tabelle. Gehen Sie über Auswahl der Seitennummer auf eine Seite, oder verwenden Sie die Pfeiltasten, um zur nächsten, vorigen, ersten oder letzten Seite zu gelangen.
