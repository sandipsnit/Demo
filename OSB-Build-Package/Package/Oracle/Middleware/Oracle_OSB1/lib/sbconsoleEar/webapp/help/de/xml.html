XML-Dokumente

Mit XML-Dokumentressourcen speichern Sie XML-Dateien für die Verwendung in Proxy oder Business-Service-Konfigurationen. Beispiel: Sie können XML-Dokumentressourcen für TopLink- oder EclipseLink-Mapping-Dateien erstellen, die in JCA-Proxy oder Business-Services erforderlich sind, um mit JCA-kompatiblen Systemen zu kommunizieren.

Weitere Informationen zu JCA-Services finden Sie in Kapitel 12, JCA-Bindings.

Dieser Abschnitt enthält die folgenden Themen:

  • XML-Dokumente suchen
  • XML-Dokumente hinzufügen
  • XML-Dokumente bearbeiten
  • XML-Dokumente löschen
