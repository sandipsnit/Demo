XML-Dokumente hinzufügen

Die einfachste Methode zum Erstellen von XML-Dokumenten in Oracle Service Bus besteht darin, einen Bulk-Import durchzuführen. Beispiel: Wenn Sie einen Bulk-Import für die erstellten JCA-Ressourcen (.jca-Datei, zugehörige WSDL und TopLink- oder EclipseLink-Mapping-Datei) durchführen, generiert Oracle Service Bus automatisch XML-Dokumentressourcen aus Mapping-Dateien und behält die Abhängigkeiten zwischen den Ressourcendateien bei.

Weitere Informationen zum Importieren finden Sie unter Ressourcen aus einer Zip-Datei laden, unter Ressourcen aus einem URL ladenund unter Ressourcen importieren.

Wenn Sie keinen Bulk-Import durchführen, können Sie XML-Dokumente manuell erstellen. Gehen Sie dazu wie folgt vor:

  1. Wenn dies noch nicht geschehen ist, klicken Sie zum Erstellen einer neuen Session auf Erstellen oder zum Aufrufen einer vorhandenen Session auf Bearbeiten. Weitere Informationen finden Sie unter Change Center verwenden.
  2. Wählen Sie Project Explorer, und wählen Sie ein Projekt oder einen Ordner, in dem das XML-Dokument hinzugefügt werden soll. Die Seite Projekt-/Ordneransicht wird angezeigt.
  3. Wählen Sie im Feld Ressource erstellen die Option XML-Dokument, um die Seite Neues XML-Dokument erstellen anzuzeigen.
  4. Klicken Sie im Feld XML-Datei auf Durchsuchen, und wählen Sie die zu verwendende XML-Datei.

    5. Das Feld "Ressourcenname" wird automatisch mit dem Dateinamen (ohne Dateierweiterung) aufgefüllt.

    Anstatt eine Datei auszuwählen, können Sie den Inhalt einer XML-Datei kopieren und in den Textbereich des Feldes "XML-Datei" einfügen.

  5. Geben Sie im Feld Ressourcenname den Namen des XML-Dokuments ein, oder ändern Sie ihn. XML-Dokumentnamen müssen eindeutig sein.

    Befolgen Sie die Benennungseinschränkungen für Ressourcen bei der Namensgebung.
  6. Geben Sie im Feld Ressourcenbeschreibung eine Beschreibung für das XML-Dokument ein.
  7. Klicken Sie auf Speichern. Das neue XML-Dokument wird in der aktuellen Session gespeichert.
  8. Um die Session zu beenden und die Konfiguration für die Laufzeit bereitzustellen, klicken Sie unter Change Center auf Aktivieren.
