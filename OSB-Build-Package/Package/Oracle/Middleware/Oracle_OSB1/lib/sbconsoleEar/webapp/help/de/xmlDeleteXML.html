XML-Dokumente löschen

Falls andere Ressourcen (zum Beispiel JCA-Bindings) auf das zu löschende XML-Dokument verweisen, müssen Sie diese Referenzen entfernen, bevor Sie das XML-Dokument löschen.

  1. Wenn dies noch nicht geschehen ist, klicken Sie zum Erstellen einer neuen Session auf Erstellen oder zum Aufrufen einer vorhandenen Session auf Bearbeiten. Weitere Informationen finden Sie unter Change Center verwenden.
  2. Wählen Sie Ressourcenbrowser > XML-Dokumente.
  3. Klicken Sie im Feld Optionen des zu löschenden XML-Dokuments auf das Symbol Löschen. Das XML-Dokument wird in der aktuellen Session gelöscht. Wenn andere Ressourcen auf diese Ressource verweisen, wird ein Symbol Warnung wegen Löschvorgang angezeigt. Sie können die Ressource mit einer Warnungsbestätigung löschen. Dadurch entstehen möglicherweise Konflikte aufgrund nicht aufgelöster Referenzen auf die gelöschte Ressource.
  4. Um die Session zu beenden und die Konfiguration für die Laufzeit bereitzustellen, klicken Sie in Change Center auf Aktivieren.
