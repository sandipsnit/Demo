XML-Dokumente suchen

Auf der Seite Übersicht über XML-Dokumente werden die XML-Dokumente angezeigt, die Sie der Oracle Service Bus-Domain hinzugefügt haben.

  1. Führen Sie eine der folgenden Aktionen aus:
    • Wählen Sie Project Explorer, um die Seite "Projektansicht" oder die Seite "Projekt-/Ordneransicht" anzuzeigen. Durchsuchen Sie dann die Projekte und Ordner nach dem XML-Dokument.
    • Wählen Sie Ressourcenbrowser > XML-Dokumente. Auf der Seite Übersicht über XML-Dokumente werden die in Tabelle 13-1 beschriebenen Informationen angezeigt. Eine detailliertere Beschreibung der Eigenschaften finden Sie unter XML-Dokumente bearbeiten.
  2. So suchen Sie ein bestimmtes XML-Dokument:
    • Wenn Sie die Anzahl der Elemente in der Liste einschränken möchten, können Sie nach Name und Pfad filtern, bevor Sie auf Suchen klicken.

      • Der Pfad ist der Projektname und der Name der Ordner, in denen die Ressource gespeichert ist.

    • Klicken Sie auf Alle anzeigen, um die Suchfilter zu deaktivieren und alle XML-Dokumente anzuzeigen.

      • Tabelle 13-1 Details der XML-Dokumente 
      Eigenschaft
      Beschreibung
      XML-Dokument
      Der eindeutige Name des XML-Dokuments. Der Name ist ein Link zur Seite XML-Dokument anzeigen. Weitere Informationen finden Sie unter XML-Dokumente bearbeiten.
      Pfad
      Der Pfad ist der Projektname und der Name des Ordners, in dem das XML-Dokument gespeichert ist. Er ist ein Link zu dem Projekt oder Ordner, in dem die Ressource gespeichert ist. Weitere Informationen finden Sie unter Ressourcennamen mit Projekten und Ordner angeben.
      Optionen
      Klicken Sie auf das Symbol Löschen, um ein bestimmtes XML-Dokument zu löschen. Weitere Informationen finden Sie unter XML-Dokumente löschen.
