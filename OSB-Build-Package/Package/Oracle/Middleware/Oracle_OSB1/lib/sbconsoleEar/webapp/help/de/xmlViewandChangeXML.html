XML-Dokumente bearbeiten

Auf der Seite XML-Dokument bearbeiten können Sie Details eines JCA-Bindings anzeigen und ändern.

  1. Wenn dies noch nicht geschehen ist, klicken Sie zum Erstellen einer neuen Session auf Erstellen oder zum Aufrufen einer vorhandenen Session auf Bearbeiten. Weitere Informationen finden Sie unter Change Center verwenden.
  2. Suchen Sie wie unter XML-Dokumente suchen beschrieben nach dem XML-Dokument.
  3. Klicken Sie auf den Namen des XML-Dokuments. Auf der Seite XML-Dokument anzeigen werden die in Tabelle 13-2 beschriebenen Informationen angezeigt.

    4. Tabelle 13-2 Details der XML-Dokumente 
    Eigenschaft
    Beschreibung
    Zuletzt geändert von
    Der Benutzer, der dieses XML-Dokument erstellt oder in die Konfiguration importiert hat
    Zuletzt geändert am
    Datum und Uhrzeit des Zeitpunkts, an dem der Benutzer das XML-Dokument erstellt oder in die Konfiguration importiert hat. Klicken Sie auf den Link mit Datum und Uhrzeit, um die Änderungshistorie dieser Ressource anzuzeigen. Weitere Informationen finden Sie unter Seite "Änderungshistorie anzeigen".
    Referenzen
    Die Anzahl der Objekte, die dieses XML-Dokument referenziert. Wenn solche Referenzen vorhanden sind, klicken Sie auf den Link mit der Anzahl, um eine Liste mit allen Objekten anzuzeigen. Weitere Informationen finden Sie unter Referenzen auf Ressourcen anzeigen.
    Referenziert von
    Die Anzahl der Objekte, die dieses XML-Dokument referenzieren. Wenn solche Referenzen vorhanden sind, klicken Sie auf den Link mit der Anzahl, um eine Liste mit allen Objekten anzuzeigen. Beispiel: Wenn die XML-Datei eine TopLink-Datei ist, die von einem JCA-Binding referenziert wird, dann wird das JCA-Binding als Referenz angezeigt, wenn Sie auf den Link klicken. Weitere Informationen finden Sie unter Referenzen auf Ressourcen anzeigen.
    Beschreibung
    Eine Beschreibung des XML-Dokuments, sofern vorhanden.

    Die schreibgeschützte Ansicht des XML-Dokuments wird standardmäßig angezeigt.

  4. Wenn Sie das XML-Dokument bearbeiten möchten, klicken Sie auf Bearbeiten. Weitere Informationen zu Feldbeschreibungen finden Sie in XML-Dokumente hinzufügen.
  5. Nehmen Sie die gewünschten Änderungen vor, und klicken Sie auf Speichern, um die Aktualisierungen in der aktuellen Session zu speichern.
  6. Um die Session zu beenden und die Konfiguration für die Laufzeit bereitzustellen, klicken Sie unter Change Center auf Aktivieren.
