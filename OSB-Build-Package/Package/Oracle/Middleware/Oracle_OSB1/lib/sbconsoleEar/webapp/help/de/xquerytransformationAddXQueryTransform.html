XQuery-Transformationen hinzufügen

  1. Wenn dies noch nicht geschehen ist, klicken Sie zum Erstellen einer neuen Session auf Erstellen oder zum Aufrufen einer vorhandenen Session auf Bearbeiten. Weitere Informationen finden Sie unter Change Center verwenden.
  2. Wählen Sie Project Explorer, um die Seite Projektansicht oder Projekt-/Ordneransicht anzuzeigen.
  3. Navigieren Sie zu dem Projekt oder Ordner, in dem Sie die XQuery-Transformation hinzufügen möchten.
  4. Wählen Sie aus der Liste Ressourcen erstellen XQuery. Die Seite Eine neue XQuery erstellen wird angezeigt.
  5. Geben Sie im Feld Ressourcenname einen eindeutigen Namen für die XQuery-Transformation ein.

    Befolgen Sie die Benennungseinschränkungen für Ressourcen bei der Namensgebung.
  6. Geben Sie im Feld Ressourcenbeschreibung eine Beschreibung für die XQuery-Transformation an.
  7. Führen Sie im Feld XQuery einen der folgenden Schritte aus:
    • Geben Sie Text für die neue XQuery-Transformation ein.
    • Klicken Sie auf Durchsuchen , um eine vorhandene XQuery-Transformation zu suchen und zu importieren.
    • Sie können Text von einer vorhandenen XQuery-Transformation in dieses Feld kopieren und einfügen.
  8. Klicken Sie auf Speichern, um die Aktualisierungen in der aktuellen Session zu speichern.
  9. Zum Beenden der Session und für eine Bereitstellung der Konfiguration zur Laufzeit, klicken Sie auf Aktivieren unter Change Center.
