XQuery-Transformationen bearbeiten

  1. Wenn dies noch nicht geschehen ist, klicken Sie zum Erstellen einer neuen Session auf Erstellen oder zum Aufrufen einer vorhandenen Session auf Bearbeiten. Weitere Informationen finden Sie unter Change Center verwenden.
  2. Suchen Sie nach der XQuery-Transformation, wie in XQuery-Transformationen suchen beschrieben.
  3. Klicken Sie auf den Namen der XQuery-Transformation. Auf der Seite Details zur XQuery-Transformation anzeigen werden die Informationen angezeigt, die in Tabelle 10-2 aufgelistet sind.

    4. Tabelle 10-2 XQuery-Transformationsressource - Details 
    Eigenschaft
    Beschreibung
    Zuletzt geändert von
    Benutzer, der diese XQuery-Transformation erstellt oder in die Konfiguration importiert hat.
    Zuletzt geändert am
    Datum und Uhrzeit, als der Benutzer diese XQuery-Transformation erstellt oder in die Konfiguration importiert hat. Klicken Sie auf den Link für Datum und Uhrzeit, um die Änderungshistorie zu dieser Ressource anzuzeigen. Siehe die Seite Änderungshistorie anzeigen.
    Referenzen
    Anzahl Objekte, die diese XQuery-Transformation referenziert. Falls Referenzen vorhanden sind, klicken Sie auf den Zahlenlink, um eine Liste der Objekte anzuzeigen. Siehe Referenzen auf Ressourcen anzeigen.
    Referenziert von
    Anzahl Objekte, die auf diese XQuery-Transformation referenzieren. Falls Referenzen vorhanden sind, klicken Sie auf den Zahlenlink, um eine Liste der Objekte anzuzeigen. Beispiel: Wenn Sie diese XQuery-Transformation einer Variablen im Nachrichtenfluss eines bestimmten Proxy-Service zugeordnet haben, wird der Proxy-Service als Referenz aufgelistet, sobald Sie auf den Link klicken. Siehe Referenzen auf Ressourcen anzeigen.
    Beschreibung
    Beschreibung der XQuery-Transformation, falls vorhanden.
    XQuery
    Volltext der XQuery-Transformation.
    XQuery-Variablen
    XQuery-Variablen, die für diese XQuery-Transformation gewählt wurden.

  4. Wenn Sie die Felder ändern möchten, klicken Sie auf Bearbeiten. Eine detaillierte Beschreibung der Felder finden Sie unter XQuery-Transformationen hinzufügen.

    5. Sie können das Feld Ressourcenname nicht ändern.

  5. Klicken Sie auf Speichern, um die Aktualisierungen in der aktuellen Session zu speichern.
  6. Um die Session zu beenden und die Konfiguration für die Laufzeit bereitzustellen, klicken Sie unter Change Center auf Aktivieren.
