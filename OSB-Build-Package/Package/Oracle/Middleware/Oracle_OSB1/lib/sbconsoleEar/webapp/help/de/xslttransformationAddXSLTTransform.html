XSL-Transformationen hinzufügen

  1. Wenn dies noch nicht geschehen ist, klicken Sie zum Erstellen einer neuen Session auf Erstellen oder zum Aufrufen einer vorhandenen Session auf Bearbeiten. Weitere Informationen finden Sie unter Change Center verwenden.
  2. Wählen Sie Project Explorer, um die Seite Projektansicht oder Projekt-/Ordneransicht anzuzeigen.
  3. Navigieren Sie zu dem Projekt oder zu einem Ordner, in dem Sie die XSL-Transformation hinzufügen möchten.
  4. Wählen Sie aus der Liste Ressource erstellen die Option XSLT, um die Seite Neue XSLT erstellen anzuzeigen.
  5. Geben Sie im Feld Ressourcenname einen eindeutigen Namen für die XSL-Transformation ein.

    Befolgen Sie die Benennungseinschränkungen für Ressourcen bei der Namensgebung.
  6. Geben Sie im Feld Ressourcenbeschreibung eine Beschreibung für die XSL-Transformation ein.
  7. Führen Sie im Feld XSLT einen der folgenden Schritte aus:
    • Geben Sie den Text für die neue XSL-Transformation ein.
    • Klicken Sie auf Durchsuchen, um eine vorhandene XSL-Transformation zu suchen und zu importieren.
    • Kopieren Sie den Text aus einer vorhandenen XSL-Transformation, und kopieren Sie ihn in dieses Feld.
  8. Klicken Sie auf Speichern, um die Aktualisierungen in der aktuellen Session zu speichern.
  9. Um die Session zu beenden und die Konfiguration für die Laufzeit bereitzustellen, klicken Sie unter Change Center auf Aktivieren.
