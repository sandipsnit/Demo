XSL-Transformationen suchen

  1. Führen Sie eine der folgenden Aktionen aus:
    • Wählen Sie Project Explorer, um die Seite Projektansicht oder die Seiten Projekt-/Ordneransicht anzuzeigen. Navigieren Sie dann durch die Projekte und Ordner, um die XSL-Transformation zu finden.
    • Wählen SieRessourcenbrowser > XSLTs. Die Seite Zusammenfassung von XSLTs zeigt die Informationen an, die in Tabelle 14-1 dargestellt sind.
  2. Um die Anzahl der Elemente in der Liste zu beschränken, können Sie nach Name, Pfad oder beidem filtern. Geben Sie in den Feldern Name und Pfad unter Suche den Namen und/oder den Pfad des oder der Ziele ein, und klicken Sie dann auf die Schaltfläche Suche.

    3. Der Pfad ist der Projektname und der Name des Ordners, in dem XSLT gespeichert ist.

    Die Platzhalter * und ? sind zulässig. Die Suche beachtet die Groß- /Kleinschreibung.

    Klicken Sie auf Alle anzeigen, um alle XSLTs in der Domain anzuzeigen. So werden Suchparameter der vorigen Suche entfernt.

    Tabelle 14-1 XSL-Transformationsdetails 
    Eigenschaft
    Beschreibung
    XSLT-Name
    Der Name, der der XSL-Transformation zugeordnet wurde. Klicken Sie auf den Namen, um die Seite XSL-Transformationsdetails anzeigen anzuzeigen.
    Siehe XSL-Transformationen bearbeiten.
    Pfad
    Der Pfad ist der Projektname und der Name des Ordners, in dem die XSL-Transformation gespeichert ist, zum Beispiel: MyProject/MyFolder/MyResource.
    Klicken Sie auf den Pfad einer XSLT, um die Seite Projekt-/Ordneransicht anzuzeigen, die sie enthält.
    Aktionen
    Klicken Sie auf das Symbol Testkonsole starten, um die Testkonsole aufzurufen, die Sie zum Validieren und Testen des Designs Ihrer Services und Transformationen verwenden. Für die XSL-Transformation können Sie die Testkonsole verwenden, gleich, ob Sie innerhalb oder außerhalb einer Session sind. Mehr darüber unter Transformationen testen.
    Optionen
    Wenn dies noch nicht geschehen ist, klicken Sie zum Erstellen einer neuen Session auf Erstellen oder zum Aufrufen einer vorhandenen Session auf Bearbeiten. Weitere Informationen finden Sie unter Change Center verwenden.
    Klicken Sie auf das Symbol Löschen, um die XSLT zu löschen. Eine Löschwarnung wird angezeigt, wenn andere Ressourcen auf diese Ressource verweisen. Sie können die Ressource mit einer Warnbestätigung löschen. Dies kann zu Konflikten aufgrund ungelöster Referenzen auf die gelöschte Ressource führen. Weitere Informationen unter Eine XSL-Transformation löschen.
