Nicht aufgelöste XLS-Transformationsreferenzen auflösen

  1. Wenn dies noch nicht geschehen ist, klicken Sie zum Erstellen einer neuen Session auf Erstellen oder zum Aufrufen einer vorhandenen Session auf Bearbeiten. Weitere Informationen finden Sie unter Change Center verwenden.
  2. Wählen Sie Ressourcenbrowser > XSLTs, um die Seite Übersicht der XSLTs anzuzeigen.
  3. Klicken Sie in der Spalte XSLT-Name auf den entsprechenden XSLT-Namen, um Details der nicht aufgelösten Referenz anzuzeigen. Daraufhin wird die Seite Details der XSL-Transformation anzeigen angezeigt.
  4. Klicken Sie auf Referenzen bearbeiten, um die Seite Referenzen einer XSL-Transformationsressource bearbeiten anzuzeigen.
  5. Wählen Sie im Feld Ressourcentyp den Eintrag XSLT.
  6. Klicken Sie auf Durchsuchen. Der XSLT-Browser wird angezeigt.
  7. Wählen Sie im XSLT-Browser eine XSLT, und wählen Sie dann im Bereich Abhängigkeiten eine Abhängigkeit.
  8. Klicken Sie auf Weiterleiten. Die gewählte XSLT wird im Feld Ressourcenname angezeigt.
  9. Klicken Sie auf Speichern. Die Seite XSLT-Detailsanzeigen wird angezeigt.
  10. Klicken Sie auf OK. Der Status der XSL-Transformation wird als Gültig angezeigt.
  11. Um die Session zu beenden und die Konfiguration für die Laufzeit bereitzustellen, klicken Sie unter Change Center auf Aktivieren.
