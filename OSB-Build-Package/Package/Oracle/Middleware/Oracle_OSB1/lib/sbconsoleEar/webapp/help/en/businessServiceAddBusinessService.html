Creating and Configuring Business Services

This section describes how to create and configure business services.

For JCA services, you can generate business services from JCA Bindings, as described in Generating a Business Service from a JCA Binding Resource, or from Oracle Enterprise Repository, as described in Generating a Business Service from Oracle Enterprise Repository.

  1. If you have not already done so, click Create to create a new session or click Edit to enter an existing session. See Using the Change Center.
  2. Select Project Explorer, then select the project or folder to which you want to add the business service.
  3. On the Project/Folder View page, select Business Service from the Create Resource drop-down list.
  4. On the Create/Edit a Business Service - General Configuration page, provide a name for the service and select the type of service to create.

    Follow the Resource Naming Restrictions for naming guidance.
  5. Click Next. The pages that follow depend on the choices you made on the first page. Enter the appropriate information on each of the subsequent pages, until you reach the summary page, then click Save to save the service in the current session.

    6. For detailed instructions on completing each page, see the following:

    • General Configuration page
    • Message Type Configuration page
    • Transport Configuration page
    • Protocol-Specific Transport Configuration page
    • SOAP Binding Configuration page
    • Message Handling Configuration Page
    • Summary page
    • Business Service Policies page
    • Business Service Security page

If a business service is created from a WSDL that includes WS-Policy attachments, the policies (read-only) are displayed on the Protocol-Specific Transport Configuration page. If any of the service's WS-Policies specifies authentication, then you must select a service account. A proxy service that routes to this business service will use this service account to authenticate to the business service.

Generating a Business Service from a JCA Binding Resource

You can generate a JCA business service from an outbound JCA Binding resource in Oracle Service Bus. For more information on JCA Binding resources, see Chapter 12, JCA Bindings.

To generate a JCA business service from a JCA Binding:

  1. In the Oracle Service Bus Administration Console, click Create or Edit in the Change Center if you area not already in Create or Edit mode.
  2. In the Resource Browser, click JCA Bindings.
  3. Locate the JCA Binding from which you want to generate a service, and click the Action icon.
  4. In the window that appears, confirm the name of the WSDL and the service you want to generate, select the location for these new resources, and click Generate.

    5. Oracle Service Bus generates the service and its corresponding WSDL.

  5. Modify any other configuration details on the generated service as appropriate, such as the Endpoint URI.

Generating a Business Service from Oracle Enterprise Repository

You can generate a business service from Oracle Enterprise Repository using the Oracle Service Bus development environment. For more information, see "Generating a Business Service from Oracle Enterprise Repository" in the Oracle Fusion Middleware Developer's Guide for Oracle Service Bus at http://www.oracle.com/pls/as111170/lookup?id=OSBDV1797.
