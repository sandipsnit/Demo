Monitoring

When you create a business or proxy service, monitoring is disabled by default for that service. To learn how to enable monitoring for proxy services, see Configuring Operational Settings for Proxy Services; for business services, see Configuring Operational Settings for Business Services.

To enable or disable monitoring of all services that have individually been enabled or disabled for monitoring, use the Enable Monitoring option on the Operations > Global Settings page. See Enabling Global Settings.

To learn more about monitoring at run time, see "Monitoring Oracle Service Bus at Run Time" in the Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus at http://www.oracle.com/pls/as111170/lookup?id=OSBAG472.
