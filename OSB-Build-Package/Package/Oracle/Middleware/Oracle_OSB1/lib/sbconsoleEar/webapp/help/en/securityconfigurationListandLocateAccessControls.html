Locating Access Control Policies

The Security page provides a link to the access control policies for a proxy service in the current Oracle Service Bus domain.

This page lists does not list proxy services that you have created in session but have not not yet activated. If you want to edit access control policies for a new proxy service, first activate the session in which you created the proxy service.

  1. If you want to locate the access control policies for a new proxy service, activate the session in which you created the proxy service.
  2. In the Access Control column select the name of the proxy service from Transport Access Control, or the name of the proxy service or a particular operation from Message Access Control.
