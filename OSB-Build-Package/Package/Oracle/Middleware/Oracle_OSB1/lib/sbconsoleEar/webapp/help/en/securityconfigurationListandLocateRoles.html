Locating Roles

  1. Select Security Configuration > Roles. The Global Roles page displays the information shown in Table 24-5. For a more detailed description of the properties, see Editing Roles.
  2. To restrict the number of items in the list or locate a specific role, scroll through the pages. Use the page controls above or below the table.

    3. Table 24-5 Role Information 
    Property
    Description
    Role Name
    The name of the role. The name is a link to the View Role Details page. See Editing Roles.
    Provider Name
    The authentication provider for this group.
