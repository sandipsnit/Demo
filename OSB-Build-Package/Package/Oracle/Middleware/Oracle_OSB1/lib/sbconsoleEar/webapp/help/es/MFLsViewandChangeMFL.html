Edición de MFL

  1. Si aún no lo ha hecho, haga clic en Crear para crear una nueva sesión o en Editar para acceder a una sesión existente. Consulte Uso del Centro de Cambios.
  2. Localice el archivo MFL, como se describe en Localización de MFL.
  3. Haga clic en el nombre del archivo MFL. La página Ver Detalles de Recurso MFL muestra la información de la Tabla 15-2.

    4. Tabla 15-2 Detalles de Recurso MFL 
    Propiedad
    Descripción
    Modificado por
    Usuario que ha creado este archivo MFL o lo ha importado a la configuración.
    Fecha de Modificación
    Fecha y hora a la que el usuario ha creado este archivo MFL o lo ha importado a la configuración. Haga clic en el enlace de fecha y hora para ver el historial de cambios de este recurso. Consulte Página Ver Historial de Cambios.
    Referencias
    Número de objetos a los que hace referencia este archivo MFL. Si existen estas referencias, haga clic en el enlace numérico para ver una lista de los objetos. Consulte Visualización de Referencias a Recursos.
    Con Referencia de
    Número de objetos que hacen referencia a este archivo MFL. Si existen estas referencias, haga clic en el enlace numérico para ver una lista de los objetos. Por ejemplo, si ha seleccionado este archivo MFL como tipo de mensaje de solicitud o respuesta para un servicio de negocio de mensajes o de proxy, el servicio de negocio o de proxy se enumera como referencia cuando se hace clic en el enlace. Consulte Visualización de Referencias a Recursos.
    Descripción
    Descripción de este archivo MFL, si existe.
    Texto MFL
    Texto de este MFL.

  4. Para realizar un cambio en los campos, haga clic en Editar. Para ver las descripciones de los campos, consulte Adición de MFL.

    5. No puede cambiar el campo Nombre del Recurso.

  5. Haga clic en Guardar para confirmar las actualizaciones en la sesión actual.
  6. Para terminar la sesión y desplegar la configuración en tiempo de ejecución, haga clic en Activar en Centro de Cambios.
