Adición de Destinos de JMS

  1. Seleccione Enviar Alerta a un Destino de JMS de la lista desplegable y haga clic en Agregar para mostrar la página Agregar Destino de JMS.
  2. En el campo URI de Destino, introduzca un URI de destino de JMS con el formato: jms://host:puerto/Nombre_Jndi_fábrica/Nombre_Jndi_destino

    3. Este campo es necesario.

    Nota: aunque Oracle WebLogic Server permite el uso de barras inclinadas en los nombres de JNDI (por ejemplo, "miscolas/micola"), los nombres de JNDI con barras inclinadas interfieren en el formato de URI que necesita Oracle Service Bus, por lo que no se pueden utilizar esos nombres. Para solucionar este problema, defina un servidor ajeno de JMS y haga referencia a él en el URI. Para obtener más información, consulte la sección sobre configuración de servidores ajenos en Oracle Fusion Middleware Oracle WebLogic Server Administration Console Online Help (Ayuda en Pantalla de la Consola de Administración de Oracle Fusion Middleware Oracle WebLogic Server), en http://www.oracle.com/pls/as111170/lookup?id=WLACH.

  3. En el campo Tipo de Destino, seleccione Cola o Tema.
  4. En el campo Tipo de Mensaje, seleccione Bytes o Texto.
  5. En el campo Codificación de Solicitud, introduzca un valor de codificación de juego de caracteres. El valor de codificación por defecto es UTF-8.
  6. Haga clic en Guardar. El destino de JMS se agrega a la lista de destinos de JMS que se incluye para este destino de alerta y se guarda en la sesión actual.
