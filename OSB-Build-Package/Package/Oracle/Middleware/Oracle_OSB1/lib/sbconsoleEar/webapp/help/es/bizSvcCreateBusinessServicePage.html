Crear/Editar un Servicio de Negocio: Referencia de Página

Para crear un servicio de negocio, seleccione Servicio de Negocio en la lista desplegable Crear Recurso de la página Vista de Proyectos/Carpetas. Al seleccionar esa opción, se muestra la primera de una serie de páginas para configurar y agregar servicios de negocio. Las páginas mostradas varían en función de las opciones seleccionadas. Las páginas son:

  • Página Configuración General
  • Página Configuración de Tipo de Mensaje
  • Página Configuración de Transporte
  • Página Configuración de Transporte Específica del Protocolo
  • Página Configuración de Enlace de SOAP
  • Página Configuración de Manejo de Mensajes
  • Página Resumen
  • Página Políticas de Servicio de Negocio
  • Página Seguridad de Servicio de Negocio
