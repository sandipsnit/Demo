Página Configuración de Transporte de DSP

La página Crear/Editar un Servicio de Negocio: Configuración de Transporte de DSP es una de una serie de páginas para la creación y edición de servicios de negocio, como se describe en Creación y Configuración de Servicios de Negocio y Edición de Configuraciones de Servicio de Negocio.

Para obtener más información, consulte la sección sobre transporte de DSP y Oracle Data Service Integrator en Oracle Fusion Middleware Developer's Guide for Oracle Service Bus (Guía del Desarrollador de Oracle Fusion Middleware para Oracle Service Bus) en http://www.oracle.com/pls/as111170/lookup?id=OSBDV1229.

Utilice esta página para configurar el transporte de un servicio de negocio mediante el protocolo de transporte de DSP (Oracle Data Service Integrator). En la Tabla 18-4 se describe el uso de esta página.

Tabla 18-4 Crear/Editar un Servicio de Negocio: Página Configuración de Transporte de DSP 
Opción
Para crear o editar...
Nivel de Depuración
Especifique una de las siguientes opciones:
  • 0: Sin información de depuración
  • 1: Información sobre el mensaje de solicitud
  • 3: Información sobre el mensaje de solicitud y el mensaje de respuesta
Cuenta de Servicios
Haga clic en Examinar y seleccione una cuenta de servicio de la lista mostrada. Si no se especifica ninguna, se utiliza un sujeto anónimo.

Para obtener más información:

  • Cuentas de Servicio
  • Adición de Cuentas de Servicio
Política de Distribución
Seleccione la instancia del gestor de trabajos de WebLogic Server que desea utilizar para la política de distribución de este punto final. Se utiliza el gestor de trabajos por defecto si no existe ningún otro.
El gestor de trabajos se utiliza para enviar el mensaje de respuesta para el procesamiento de respuesta. Para obtener más información sobre los gestores de trabajo, consulte la sección sobre el uso de gestores de trabajos para optimizar el trabajo programado en Oracle Fusion Middleware Configuring Server Environments for Oracle WebLogic Server (Configuración de Entornos de Servidor de Oracle Fusion Middleware para Oracle WebLogic Server) en http://www.oracle.com/pls/as111170/lookup?id=CNFGD112.

Después de terminar

Haga clic en Siguiente para seguir configurando este servicio en la siguiente página; o haga clic en Último para revisar y guardar esta configuración en Crear/Editar un Servicio de Negocio: Página Resumen.
