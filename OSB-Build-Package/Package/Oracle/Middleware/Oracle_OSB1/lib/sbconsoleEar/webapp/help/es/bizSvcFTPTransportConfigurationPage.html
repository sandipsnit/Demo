Página Configuración de Transporte de FTP

La página Crear/Editar un Servicio de Negocio: Configuración de Transporte de FTP es una de una serie de páginas para la creación y edición de servicios de negocio, como se describe en Creación y Configuración de Servicios de Negocio y Edición de Configuraciones de Servicio de Negocio.

Utilice esta página para configurar el transporte de un servicio de negocio mediante el protocolo de transporte de FTP. En la Tabla 18-9 se describe el uso de esta página.

Tabla 18-9 Crear/Editar un Servicio de Negocio: Página Configuración de Transporte de FTP 
Opción
Para crear o editar...
Autenticación de Usuario
Seleccione anónimo si el usuario del servidor FTP es anónimo o usuario externo si el usuario del servidor FTP es una cuenta configurada externamente.
Identidad (Identificador de Correo Electrónico)
Este campo está disponible si la opción Autenticación de Usuario se define en anónimo.
Introduzca el identificador de correo para el usuario anónimo.
Cuenta de Servicios
Este campo está disponible si la opción Autenticación de Usuario se define en usuario externo.
Introduzca la cuenta de servicio para el usuario externo.

Para obtener más información:

  • Cuentas de Servicio
  • Adición de Cuentas de Servicio
Timeout
Introduzca el intervalo de timeout del socket, en segundos, antes de que se cierre la conexión. El valor por defecto es 60 segundos.
Prefijo para el Nombre de Archivo de Destino
Introduzca un prefijo para el nombre del archivo que se almacena en el servidor remoto.
No introduzca * en este campo. Este carácter causa un excepción en tiempo de ejecución.
Sufijo para el Nombre de Archivo de Destino
Introduzca un sufijo para el nombre del archivo que se almacena en el servidor remoto.
No introduzca * en este campo. Este carácter causa un excepción en tiempo de ejecución.
Modo de Transferencia
Seleccione ASCII o binario como modo de transferencia.
Codificación de Solicitud
Acepte el valor por defecto UTF-8 como codificación del juego de caracteres para las solicitudes de transportes de FTP o introduzca otra diferente.

Después de Terminar

Haga clic en Siguiente para seguir configurando este servicio en la siguiente página; o haga clic en Último para revisar y guardar esta configuración en Crear/Editar un Servicio de Negocio: Página Resumen.
