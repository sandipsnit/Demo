Página Configuración de Transporte de Archivo

La página Crear/Editar un Servicio de Negocio: Configuración de Transporte de Archivo es una de una serie de páginas para la creación y edición de servicios de negocio, como se describe en Creación y Configuración de Servicios de Negocio y Edición de Configuraciones de Servicio de Negocio.

Utilice esta página para configurar el transporte de un servicio de negocio mediante el protocolo de transporte de archivo. En la Tabla 18-7 se describe el uso de esta página.

Tabla 18-7 Crear/Editar un Servicio de Negocio: Página Configuración de Transporte de Archivo 
Opción
Para crear o editar...
Prefijo
Introduzca un prefijo que se antepondrá al nombre de archivo.
No introduzca * en este campo. Este carácter causa un excepción en tiempo de ejecución.
Sufijo
Introduzca un sufijo que se agregará al nombre de archivo. Éste es un campo necesario.
No introduzca * en este campo. Este carácter causa un excepción en tiempo de ejecución.
Codificación de Solicitud
Acepte el valor por defecto utf-8 como codificación del juego de caracteres para las solicitudes de transportes de archivo o introduzca otra diferente.

Después de Terminar

Haga clic en Siguiente para seguir configurando este servicio en la siguiente página; o haga clic en Último para revisar y guardar esta configuración en Crear/Editar un Servicio de Negocio: Página Resumen.
