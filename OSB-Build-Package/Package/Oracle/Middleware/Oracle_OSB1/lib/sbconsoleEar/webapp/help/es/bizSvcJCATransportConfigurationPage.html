Página Configuración de Transporte de JCA

Utilice esta página para configurar el transporte mediante el protocolo de transporte de JCA. Para obtener más información sobre el uso del transporte de JCA, consulte la sección sobre transporte de JCA en Oracle Fusion Middleware Developer's Guide for Oracle Service Bus (Guía del Desarrollador de Oracle Fusion Middleware para Oracle Service Bus) en http://www.oracle.com/pls/as111170/lookup?id=OSBDV910 y Generación de un Servicio de Negocio a partir de un Recurso de Enlace de JCA.

Archivo JCA
Haga clic en Examinar para seleccionar un enlace de JCA. El enlace de JCA define diferentes aspectos del servicio, como los detalles sobre el adaptador utilizado, un enlace a WSDL y un archivo de asignación de TopLink o EclipseLink, así como las propiedades de especificación de activación/interacción necesarias para el servicio.
Después de seleccionar un enlace de JCA válido, el resto de los campos de configuración de transporte están disponibles.
Para obtener más información sobre enlaces de JCA, consulte Chapter 12, JCA Bindings (Capítulo 12, Enlaces de JCA).
Nombre del Adaptador
Valor de sólo lectura que muestra el nombre del adaptador que utilizará el servicio de JCA.
Tipo de Adaptador
Valor de sólo lectura que muestra el tipo de adaptador.
Política de Distribución
Seleccione la instancia del gestor de trabajos de WebLogic Server que desea utilizar para la política de distribución de este punto final. Se utiliza el gestor de trabajos por defecto si no existe ningún otro.
Para obtener más información sobre los gestores de trabajo, consulte la sección sobre el uso de gestores de trabajos para optimizar el trabajo programado en Oracle Fusion Middleware Configuring Server Environments for Oracle WebLogic Server (Configuración de Entornos de Servidor de Oracle Fusion Middleware para Oracle WebLogic Server) en http://www.oracle.com/pls/as111170/lookup?id=CNFGD112.
Cuenta de Servicio de JNDI
Cuenta de Servicio de JNDI es para la seguridad de contexto de JNDI, se utiliza para acceder a la fábrica de conexiones gestionada por el adaptador EIS. Haga clic en Examinar y seleccione una cuenta de servicio. Si no se especifica ninguna, se utiliza un sujeto anónimo.
Para los servicios de negocio de JCA, no existe ninguna restricción en cuanto al tipo de cuenta de servicio de JNDI que se puede configurar, como estática o de traspaso, pero en tiempo de ejecución se debe poder acceder a un nombre de usuario y contraseña. Los servicios de proxy de JCA sólo pueden utilizar cuentas de servicio de JNDI estáticas.
Para obtener más información sobre las cuentas de servicio de JNDI, consulte la sección sobre seguridad del capítulo sobre transporte de JCA en Oracle Fusion Middleware Developer's Guide for Oracle Service Bus (Guía del Desarrollador de Oracle Fusion Middleware para Oracle Service Bus) en http://www.oracle.com/pls/as111170/lookup?id=OSBDV916.
Propiedades de Punto Final
Este campo permite asignar valores a las propiedades de punto final como los reintentos para el tipo de adaptador que utiliza el servicio.
Para obtener una lista de las propiedades de punto final de JCA soportadas, consulte la sección sobre propiedades de punto final en Oracle Fusion Middleware Developer's Guide for Oracle Service Bus (Guía del Desarrollador de Oracle Fusion Middleware para Oracle Service Bus) en http://www.oracle.com/pls/as111170/lookup?id=OSBDV1841.
Propiedades Dinámicas de Punto Final
Esta opción permite transferir parámetros de solicitud a los servicios compatibles con JCA. Por ejemplo, puede utilizar una propiedad dinámica de punto final para transferir los parámetros de consulta de base de datos al adaptador JCA de Oracle para Database.
Para obtener más información sobre la consulta con parámetros, consulte la sección sobre el adaptador JCA de Oracle para Database en Oracle Fusion Middleware User's Guide for Technology Adapters (Guía del Usuario de Oracle Fusion Middleware para Technology Adapters) en http://www.oracle.com/pls/as111170/lookup?id=TKADP2117.
Introduzca un par nombre/valor para cada propiedad dinámica de punto final que desea proporcionar. La clave de la propiedad de punto final coincide con el nombre del parámetro de consulta.
Usar Siempre Configuración de Archivo JCA
Esta opción determina si las propiedades de especificación de activación (servicios de proxy) y las propiedades de especificación de interacción (servicios de negocio) se utilizan siempre desde el archivo JCA.
Si está seleccionada esta opción (por defecto), el transporte de JCA interactúa con el marco de JCA mediante propiedades de especificación de activación/interacción en el archivo JCA.
Si no se selecciona, puede sustituir las propiedades de especificación de activación/interacción.
Para conocer el impacto del nuevo despliegue de esta opción, consulte la sección sobre el nuevo despliegue de punto final en el capítulo sobre transporte de JCA en Oracle Fusion Middleware Developer's Guide for Oracle Service Bus (Guía del Desarrollador de Oracle Fusion Middleware para Oracle Service Bus) en http://www.oracle.com/pls/as111170/lookup?id=OSBDV940.
Nombre de la Operación
Muestra un nombre de sólo lectura de la operación de WSDL seleccionada. Una operación puede tener sus propias propiedades de especificación de activación/interacción, mostradas en el campo Propiedades de Especificación de Activación/Interacción.
Propiedades de Especificación de Activación/Interacción
Propiedades de Especificación de Activación es el nombre del campo para los servicios de proxy; Propiedades de Especificación de Interacción es el nombre del campo para los servicios de negocio.
Si este servicio es un servicio entrante llamado por una aplicación de EIS, este campo muestra las propiedades de especificación de activación para la operación entrante de JCA mostrada en el campo Nombre de la Operación.
Para sustituir las propiedades de especificación de activación/interacción, no seleccione Usar Siempre Configuración de Archivo JCA.
Nota: Para los adaptadores de Oracle Adapter Suite, las propiedades de especificación de activación/interacción siempre se muestran como de sólo lectura. Los adaptadores de Oracle Adapter Suite almacenan su propias configuraciones, que se deben cambiar con las herramientas de gestión de Oracle Adapter Suite.
Propiedades de la Conexión (legado)
Para los servicios de JCA de legado que utilizan propiedades de conexión de modo no gestionado (anticuadas en esta versión), consulte las opciones de configuración de conexiones en http://download.oracle.com/docs/cd/E13159_01/osb/docs10gr3/jcatransport/transport.html#wp1105451.

Para obtener más información sobre las propiedades de especificación de activación/interacción y punto final, consulte Oracle Fusion Middleware User's Guide for Technology Adapters (Guía del Usuario de Oracle Fusion Middleware para Technology Adapters) en http://www.oracle.com/pls/as111170/lookup?id=TKADP.
