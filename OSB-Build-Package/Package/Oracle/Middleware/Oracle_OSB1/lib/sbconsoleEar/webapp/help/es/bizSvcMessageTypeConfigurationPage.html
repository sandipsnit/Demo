Página Configuración de Tipo de Mensaje

La página Crear/Editar un Servicio de Negocio: Configuración de Tipo de Mensaje es una de una serie de páginas para la creación y edición de servicios de negocio, como se describe en Creación y Configuración de Servicios de Negocio y Edición de Configuraciones de Servicio de Negocio.

Utilice esta página para configurar tipos de mensajes para un servicio de negocio cuyo tipo es Servicio de Mensajes.

La definición de enlace para los servicios de mensajes consiste en la configuración del tipo de contenido de los mensajes que se intercambian. No es necesario que el tipo de contenido de la respuesta sea el mismo que el de la solicitud; por lo tanto, la respuesta se configura por separado (por ejemplo, el servicio podría aceptar un mensaje en MFL y devolver un acuse de recibo en XML).

Nota: Los servicios de negocio de transporte de correo electrónico, archivo, FTP o SFTP cuyo tipo es Servicio de Mensajes sólo soportan los mensajes unidireccionales; la opción Tipo de Mensaje de Respuesta debe estar definida en ninguno. Si selecciona una opción que no sea ninguno, los protocolos de archivo, ftp o sftp no estarán disponibles en la página Configuración de Transporte.

Tipo de Mensaje de Solicitud
Seleccione un tipo de mensaje para el mensaje de solicitud:
  • Ninguno: Seleccione esta opción si no hay ningún mensaje de solicitud (ejemplo HTTP GET).
  • Binario: Seleccione esta opción si el tipo de contenido del mensaje es desconocido o no importa.
  • Texto: Seleccione esta opción si el mensaje se puede restringir a texto.
  • MFL: Seleccione esta opción si el mensaje es un documento binario que cumple con la definición MFL. Sólo puede configurar un archivo MFL.

    • Para MFL, puede hacer clic en Examinar para seleccionar un MFL del explorador de archivos MFL y, a continuación, haga clic en Enviar.

  • XML: Seleccione esta opción si el mensaje es un documento XML. Para proporcionar algún tipo de información, puede declarar el tipo de esquema XML del documento XML intercambiado.
  • Java: Seleccione esta opción si se va a enviar un objeto Java en la solicitud. El transporte de JMS se utiliza para los mensajes con objetos Java. Para obtener más información, consulte la sección sobre envío y recepción de objetos Java en los mensajes en Oracle Fusion Middleware Developer's Guide for Oracle Service Bus (Guía del Desarrollador de Oracle Fusion Middleware para Oracle Service Bus) en http://www.oracle.com/pls/as111170/lookup?id=OSBDV1874.
Tipo de Mensaje de Respuesta
Seleccione un tipo de mensaje para el mensaje de respuesta:
  • Ninguno: Seleccione esta opción si no hay ningún mensaje de respuesta.
  • Binario: Seleccione esta opción si el tipo de contenido del mensaje es desconocido o no importa.
  • Texto: Seleccione esta opción si el mensaje se puede restringir a texto.
  • MFL: Seleccione esta opción si el mensaje es un documento binario que cumple con la definición MFL. Sólo puede configurar un archivo MFL.

    • Para MFL, puede hacer clic en Examinar para seleccionar un MFL del explorador de archivos MFL y, a continuación, haga clic en Enviar.

  • XML: Seleccione esta opción si el mensaje es un documento XML. Para proporcionar algún tipo de información, puede declarar el tipo de esquema XML del documento XML intercambiado.
  • Java: Seleccione esta opción si se va a recibir un objeto Java en la respuesta. El transporte de JMS se utiliza para los mensajes con objetos Java. Para obtener más información, consulte la sección sobre envío y recepción de objetos Java en los mensajes en Oracle Fusion Middleware Developer's Guide for Oracle Service Bus (Guía del Desarrollador de Oracle Fusion Middleware para Oracle Service Bus) en http://www.oracle.com/pls/as111170/lookup?id=OSBDV1874.
