Página Configuración de Transporte de SB

La página Crear/Editar un Servicio de Negocio: Configuración de Transporte de SB es una de una serie de páginas para la creación y edición de servicios de negocio, como se describe en Creación y Configuración de Servicios de Negocio y Edición de Configuraciones de Servicio de Negocio.

Para obtener más información, consulte la sección sobre transporte de SB en Oracle Fusion Middleware Developer's Guide for Oracle Service Bus (Guía del Desarrollador de Oracle Fusion Middleware para Oracle Service Bus) en http://www.oracle.com/pls/as111170/lookup?id=OSBDV997.

Utilice esta página para configurar el transporte de un servicio de negocio mediante el protocolo de transporte de SB (Service Bus). En la Tabla 18-15 se describe el uso de esta página.

Tabla 18-15 Crear/Editar un Servicio de Negocio: Página Configuración de Transporte de SB 
Opción
Para crear o editar...
Política de Distribución
Seleccione la instancia del gestor de trabajos de WebLogic Server que desea utilizar para la política de distribución de este punto final. Se utiliza el gestor de trabajos por defecto si no existe otro. Para obtener más información sobre los gestores de trabajo, consulte la sección sobre uso de gestores de trabajos para optimizar el trabajo programado en Oracle Fusion Middleware Configuring Server Environments for Oracle WebLogic Server (Configuración de Entornos de Servidor de Oracle Fusion Middleware para Oracle WebLogic Server) en http://www.oracle.com/pls/as111170/lookup?id=CNFGD112.
Timeout
Cantidad de tiempo, en segundos, que tarda en servicio en sufrir un timeout.
Nota: El timeout se ignora si la calidad del servicio es Una Sola Vez.
Cuenta de Servicios
Haga clic en Examinar y seleccione una cuenta de servicio de la lista mostrada. Si no se especifica ninguna, se utiliza un sujeto anónimo.
