Página Configuración de Enlace de SOAP

La página Crear/Editar un Servicio de Negocio: Configuración de Enlace de SOAP es una de una serie de páginas para la creación y edición de servicios de negocio, como se describe en Creación y Configuración de Servicios de Negocio y Edición de Configuraciones de Servicio de Negocio.

Utilice esta página para configurar el enlace de SOAP para un servicio de negocio basado en WSDL.

Seleccione Aplicar Cumplimiento de WS-I para especificar si el servicio debe cumplir el perfil básico definido por la organización de interoperabilidad de servicios web. Esta opción sólo está disponible para servicios de SOAP 1.1

Cuando un servicio se marca como compatible con WS-I, se realizan comprobaciones en los mensajes enviados a y desde ese servicio.

Después de Terminar

Haga clic en Siguiente para seguir configurando este servicio en la siguiente página; o haga clic en Último para revisar y guardar esta configuración en Crear/Editar un Servicio de Negocio: Página Resumen.
