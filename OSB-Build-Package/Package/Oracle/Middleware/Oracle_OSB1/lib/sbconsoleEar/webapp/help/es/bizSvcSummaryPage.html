Página Resumen

La página Crear/Editar un Servicio de Negocio: Resumen es una de una serie de páginas para la creación y edición de servicios de negocio, como se describe en Creación y Configuración de Servicios de Negocio y Edición de Configuraciones de Servicio de Negocio.

Utilice esta página para ver o modificar la configuración de un servicio de negocio antes de guardarla.

Para ver o modificar la configuración, haga clic en Editar en la fila de la categoría adecuada (por ejemplo, Configuración General, Configuración de Transporte, etc.). Las páginas que puede editar dependen de las páginas configuradas al crear el servicio de negocio. En la siguiente lista se muestran todas las páginas:

  • Página Configuración General
  • Página Configuración de Tipo de Mensaje
  • Página Configuración de Transporte
  • Página Configuración de Transporte Específica del Protocolo
  • Página Políticas de Servicio de Negocio
  • Página Seguridad de Servicio de Negocio
