Página Configuración de Transporte de Tuxedo

La página Crear/Editar un Servicio de Negocio: Configuración de Transporte de Tuxedo es una de una serie de páginas para la creación y edición de servicios de negocio, como se describe en Creación y Configuración de Servicios de Negocio y Edición de Configuraciones de Servicio de Negocio.

Para obtener más información, consulte la sección sobre transporte de Tuxedo en Oracle Fusion Middleware Developer's Guide for Oracle Service Bus (Guía del Desarrollador de Oracle Fusion Middleware para Oracle Service Bus) en http://www.oracle.com/pls/as111170/lookup?id=OSBDV1200.

Utilice esta página para configurar el transporte de un servicio de negocio mediante el protocolo de transporte de Tuxedo. En la Tabla 18-18 se describe el uso de esta página.

Tabla 18-18 Crear/Editar un Servicio de Negocio: Página Configuración de Transporte de Tuxedo (Continuación)
Opción
Para crear o editar...
Clases de Tablas de Campo
Introduzca el nombre de las clases que describan el buffer de FML/FML32 recibido. Se utilizan para las rutinas de conversión de FML/FML32 a XML para asignar nombres de campo a nombres de elemento. Es una lista separada por espacios de nombres de clase completamente cualificados.
Clases de Vista
Introduzca el nombre de las clases que describan el buffer de VIEW/VIEW32 recibido o enviado. Se utilizan para las rutinas de conversión de VIEW a XML o VIEW32 a XML para asignar nombres de campo a nombres de elemento. Es una lista separada por espacios de nombres de clase completamente cualificados.
Jar de Clases
Seleccione un recurso de JAR que contenga un archivo JAR con las clases FML/FML32 o VIEW/VIEW32 necesarias para esta operación de punto final.
Puntos de Acceso Remoto
Seleccione un punto de acceso remoto de la lista desplegable asociada a la importación. La lista contiene los puntos de acceso remoto configurados en WTC. No se puede crear un servicio de negocio si no hay ningún punto de acceso remoto asociado.
Si no existe ningún punto de acceso remoto o para crear uno nuevo, seleccione Nuevo. Introduzca el nombre del punto de acceso y la dirección de red correspondientes en los campos adyacentes. Tras la validación del punto final, se agrega el punto de acceso a la configuración de WTC para cada servidor de WTC. Si no existe ningún servidor de WTC, se crea uno.
Si se ha especificado más de un URI, habrá un campo de punto de acceso remoto por URI y el URI se muestra para fines informativos. Si existe más de un URI, cada uno necesita un punto de acceso remoto diferente. Si el URI especificado ya corresponde a un recurso de WTC existente, se muestra el punto de acceso remoto correspondiente, pero no se puede modificar.
Puntos de Acceso Local
Este campo aparece sólo si selecciona Nuevo en el campo Punto de Acceso Remoto.
En la lista desplegable, seleccione el punto de acceso local que desea asociar al nuevo punto de acceso remoto creado. Si no existe ninguno o para crear uno nuevo, seleccione Nuevo. Introduzca el nombre del punto de acceso local y la dirección de red local correspondientes en los campos adyacentes.
Tipo de Buffer de Solicitud
Seleccione el tipo de buffer que recibirá el servicio de Tuxedo remoto.
Subtipo de Buffer de Solicitud
Esta opción se activa si el valor del campo anterior Tipo de Buffer de Solicitud es VIEW o VIEW32. Introduzca el subtipo de buffer al que se asociará el buffer de solicitud.
¿Respuesta Obligatoria?
Active esta casilla de control para indicar una llamada bidireccional. Si no se activa, se llama al elemento tpcall subyacente con el indicador TPNOREPLY y se envía una respuesta nula de forma asíncrona.
¿Suspender Transacción?
Active esta casilla de control para suspender la transacción, si existe. Resulta útil cuando el servicio remoto no soporta transacciones.
Política de Distribución
Seleccione la instancia del gestor de trabajos de WebLogic Server que desea utilizar para la política de distribución de este punto final. Se utiliza el gestor de trabajos por defecto si no existe ningún otro.
Se utiliza el gestor de trabajos para enviar una respuesta nula de forma asíncrona en el caso de una llamada unidireccional. Para obtener más información sobre los gestores de trabajo, consulte la sección sobre uso de gestores de trabajos para optimizar el trabajo programado en Oracle Fusion Middleware Configuring Server Environments for Oracle WebLogic Server (Configuración de Entornos de Servidor de Oracle Fusion Middleware para Oracle WebLogic Server) en http://www.oracle.com/pls/as111170/lookup?id=CNFGD112.
Codificación de Solicitud
Especifique una codificación de juego de caracteres para las solicitudes de los transportes de Tuxedo.
Codificación de Respuesta
Especifique una codificación de juego de caracteres para las respuestas de los transportes de Tuxedo.
Timeout
Especifique el tiempo máximo (en segundos) que el tiempo de ejecución del transporte espera respuestas; un valor entero mayor o igual que 0. Si no se especifica o se define en cero (valor por defecto), las respuestas sufrirán un timeout en BLOCKTIME, el número máximo de segundos que permite el punto de acceso local de WTC para una llamada de bloqueo.
Estilo de Transformación
Seleccione una de las siguientes opciones:
  • Ninguno: (Valor por defecto) Puede que no se respete el orden de los campos.
  • Ordenado: Los campos se presentan con todas sus incidencias en el orden correcto.
  • Agrupado y Ordenado: Si los campos tienen una estructura lógica de registros, los campos se ordenan por incidencia y se agrupan por registro.
