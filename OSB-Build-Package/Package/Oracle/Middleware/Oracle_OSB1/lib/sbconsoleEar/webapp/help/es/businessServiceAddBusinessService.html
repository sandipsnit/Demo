Creación y Configuración de Servicios de Negocio

En esta sección se describe la forma de crear y configurar servicios de negocio.

Para los servicios de JCA, puede generar servicios de negocio a partir de enlaces de JCA, como se describe en Generación de un Servicio de Negocio a partir de un Recurso de Enlace de JCA o desde Oracle Enterprise Repository, como se describe en Generación de un Servicio de Negocio desde Oracle Enterprise Repository.

  1. Si aún no lo ha hecho, haga clic en Crear para crear una nueva sesión o en Editar para acceder a una sesión existente. Consulte Uso del Centro de Cambios.
  2. Seleccione Explorador de Proyectos y elija el proyecto o la carpeta conde desea agregar el servicio de negocio.
  3. En la página Vista de Proyectos/Carpetas, seleccione Servicio de Negocio en la lista desplegable Crear Recurso.
  4. En Crear/Editar un Servicio de Negocio- página Configuración General, proporciona un nombre para el servicio y seleccione el tipo de servicio que desea crear.

    Siga las Restricciones de Nomenclatura de Recursos para obtener ayuda.
  5. Haga clic en Siguiente. Las páginas que aparecen a continuación dependen de las opciones seleccionadas en la primera página. Introduzca la información adecuada en cada página hasta que llegue a la página de resumen, a continuación, haga clic en Guardar para guardar el servicio en la sesión actual.

    6. Para obtener instrucciones detalladas para rellenar cada página, consulte:

    • Página Configuración General
    • Página Configuración de Tipo de Mensaje
    • Página Configuración de Transporte
    • Página Configuración de Transporte Específica del Protocolo
    • Página Configuración de Enlace de SOAP
    • Página Configuración de Manejo de Mensajes
    • Página Resumen
    • Página Políticas de Servicio de Negocio
    • Página Seguridad de Servicio de Negocio

Si se crea un servicio de negocio a partir de un WSDL que incluye anexos de política de servicios web, se muestran las políticas (sólo lectura) en la página Configuración de Transporte Específica del Protocolo. Si alguna de las políticas de servicios web especifica autenticación, debe seleccionar una cuenta de servicio. Un servicio de proxy que se direcciona a este servicio de negocio utilizará esta cuenta de servicio para autenticar el servicio de negocio.

Generación de un Servicio de Negocio a partir de un Recurso de Enlace de JCA

Puede generar un servicio de negocio de JCA a partir de un recurso de enlace de JCA saliente en Oracle Service Bus. Para obtener más información sobre los recursos de enlace de JCA, consulte Chapter 12, JCA Bindings (Capítulo 12, Enlaces de JCA).

Para generar un servicio de negocio a partir de un enlace de JCA:

  1. En la consola de administración de Oracle Service Bus, haga clic en Crear o Editar en el Centro de Cambios si el área no se encuentra en el modo de creación o edición.
  2. En el Explorador de Recursos, haga clic en Enlaces de JCA.
  3. Localice el enlace de JCA desde el que desea generar un servicio y haga clic en el icono Acción.
  4. En la ventana que aparece, confirme el nombre de WSDL y el servicio que desea generar, seleccione la ubicación de estos nuevos recursos y haga clic en Generar.

    5. Oracle Service Bus genera el servicio y su correspondiente WSDL.

  5. Realice las modificaciones de configuración que necesite en el servicio generado, como el URI de punto final.

Generación de un Servicio de Negocio desde Oracle Enterprise Repository

Puede generar un servicio de negocio desde Oracle Enterprise Repository mediante el entorno de desarrollo de Oracle Service Bus. Para obtener más información, consulte la sección sobre generación de un servicio de negocio desde Oracle Enterprise Repository en Oracle Fusion Middleware Developer's Guide for Oracle Service Bus (Guía del Desarrollador de Oracle Fusion Middleware para Oracle Service Bus) en http://www.oracle.com/pls/as111170/lookup?id=OSBDV1797.
