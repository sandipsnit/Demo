Exportación de un WSDL Asociado a un Servicio de Negocio

Puede exportar el WSDL de un servicio de negocio basado en WSDL, para ver o modificar el WSDL en una herramienta externa como IDE. El WSDL se exporta como archivo JAR.

Tenga en cuenta que es diferente a la funcionalidad Exportar Recursos del módulo Administración del Sistema, que se utiliza para mover y crear etapas de los recursos entre dos dominios. Consulte Exportación de Recursos.

Antes de Empezar

Sólo puede exportar un WSDL si está fuera de una sesión. Consulte Uso del Centro de Cambios.

Para exportar un WSDL:

  1. Realice una de las siguientes acciones:
    • Seleccione Explorador de Recursos > Servicios de Negocio para mostrar la página Resumen de Servicios de Negocio.
    • Seleccione Explorador de Proyectos y elija el proyecto o la carpeta que contiene el servicio de negocio que desea exportar como WSDL. Aparece la página Vista de Proyectos/Carpetas.
  2. Haga clic en el icono Exportar WSDL en la fila del servicio de negocio cuyo WSDL desea exportar. Un cuadro de diálogo le solicitará que abra o guarde el archivo JAR exportado.
  3. En el cuadro de diálogo, haga clic en Abrir para abrir el archivo o en Guardar para guardarlo.
