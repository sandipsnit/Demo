Página Ver Servicio de Negocio: Detalles de Configuración

La página Ver Servicio de Negocio: Detalles de Configuración muestra los detalles de la configuración de un servicio de negocio. En la Tabla 18-22 se describen todas las propiedades que pueden aparecer en esta página. (Las propiedades varían según los detalles del servicio de negocio.)

Las categorías mostradas en esta página corresponden a las páginas Crear/Editar un Servicio de Negocio utilizadas para la creación y edición de configuraciones de servicio de negocio, como se describe en Crear/Editar un Servicio de Negocio: Referencia de Página.

Haga clic en el enlace Editar junto al nombre de alguna categoría para mostrar la página de configuración asociada.

Tabla 18-22 Página Ver Servicio de Negocio: Detalles de Configuración 
Propiedades
Descripción
Modificado por
Usuario que ha creado este servicio o lo ha importado a la configuración.
Fecha de Modificación
Fecha y hora a la que el usuario ha creado este servicio o lo ha importado a la configuración. Haga clic en el enlace de fecha y hora para ver el historial de cambios de este recurso. Consulte Página Ver Historial de Cambios.
Referencias
Número de objetos a los que hace referencia este servicio de negocio. Si existen estas referencias, haga clic en el enlace numérico para ver una lista de los objetos. Para obtener más información, consulte Visualización de Referencias a Recursos.
Con Referencia de
Número de objetos que hacen referencia a este servicio de negocio. Si existen estas referencias, haga clic en el enlace numérico para ver una lista de los objetos. Para obtener más información, consulte Visualización de Referencias a Recursos.
Descripción
