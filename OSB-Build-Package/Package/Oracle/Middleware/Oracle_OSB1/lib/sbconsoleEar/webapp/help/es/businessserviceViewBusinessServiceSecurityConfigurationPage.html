Página Seguridad de Servicio de Negocio

La página Ver Servicio de Negocio: Seguridad es una de una serie de páginas para la edición de servicios de negocio, como se describe en Creación y Configuración de Servicios de Negocio y Página Ver Servicio de Negocio: Detalles de Configuración.

Los campos disponibles en esta página dependen del uso de las políticas del servicio de negocio. Por ejemplo, si el servicio utiliza políticas de OWSM (recomendado), aparece Sustituciones de Políticas en la página Seguridad. Para obtener más información, consulte la sección sobre protección de Oracle Service Bus con Oracle Web Services Manager en Oracle Fusion Middleware Developer's Guide for Oracle Service Bus (Guía del Desarrollador de Oracle Fusion Middleware para Oracle Service Bus) en http://www.oracle.com/pls/as111170/lookup?id=OSBDV1681.

Se muestran (en línea) las políticas de WLS 9.2 enlazadas al servicio en el WSDL efectivo. Las políticas abstractas se procesan previamente antes de ponerse en línea. Las políticas de OWSM se enlazan por referencia, no se ponen en línea en el WSDL efectivo.

Utilice esta página para configurar la seguridad de un servicio de negocio que tiene una política basada en WSDL o que utiliza enlaces de política personalizada. En la Tabla 18-24 se describe el uso de esta página.
