Depuración del Historial de Activación de Sesión

Puede depurar las sesiones activadas en un período concreto, delimitado por las fechas de inicio y finalización. Esta acción sólo se puede realizar fuera de una sesión.

Atención: La depuración del historial de activación de sesión implica la supresión de datos que permite varios niveles para deshacer. Es decir, si depura el historial de activación de sesión de un determinado período, no podrá deshacer las sesiones activadas durante ese período.
  1. Asegúrese de que no está en una sesión activa y haga clic en Ver Cambios.
  2. En la página Ver Cambios de Configuración, haga clic en Depurar por Fecha. Aparece la página Depurar Historial de Activación de Sesión por Fecha.
  3. Introduzca las fechas de inicio y finalización en sus respectivos campos, con el formato: MM/DD/AAAA.

    4. Se suprimirá el historial de activación de sesión de todas las sesiones entre las fechas especificadas de Oracle Service Bus.
