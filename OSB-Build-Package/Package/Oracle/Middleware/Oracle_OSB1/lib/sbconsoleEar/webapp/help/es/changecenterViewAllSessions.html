Visualización de Todas las Sesiones

Utilice esta página para ver todas las sesiones existentes en la consola de administración de Oracle Service Bus. Puede verlas tanto dentro como fuera de una sesión.

Puede ver todas las sesiones sólo si utiliza el rol Administrador. Para obtener más información, consulte la sección sobre configuración de seguridad administrativa en Oracle Fusion Middleware Developer's Guide for Oracle Service Bus (Guía del Desarrollador de Oracle Fusion Middleware para Oracle Service Bus) en http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539.

  1. Haga clic en Ver Todas las Sesiones en el Centro de Cambios. Para cada sesión existente, la página Ver Todas las Sesiones muestra la información de la Tabla 2-5.

    2. Tabla 2-5 Detalles de la Sesión
    Propiedad
    Descripción
    Nombre de Sesión
    Nombre de la sesión. El nombre es un enlace a la sesión.
    Usuario
    Nombre del usuario que creó la sesión.
    Hora de Creación
    Fecha y hora en las que se creó la sesión.
    Fecha de Modificación
    Fecha y hora a las que se realizó el último cambio durante la sesión.
    Nº de Objetos Modificados
    Número de objetos que se han cambiado durante la sesión.

  2. Para cambiar a otra sesión, haga clic en el enlace de la columna Nombre de Sesión correspondiente.
Nota: No se admite el mismo usuario conectado con varios exploradores. Esto causa un comportamiento impredecible en la consola de administración de Oracle Service Bus.
