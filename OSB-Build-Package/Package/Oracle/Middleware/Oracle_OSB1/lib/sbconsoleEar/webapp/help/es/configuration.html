Configuración

Utilice el módulo Configuración para localizar y gestionar los servicios de proxy, servicios de negocio, destinos de alertas, reglas de alertas de SLA y el rastreo en tiempo de ejecución para los servicios de proxy.

Puede especificar la configuración operativa de todos los servicios, a nivel de servicio y global, y utilizar la configuración global para activar y desactivar la supervisión, alertas de SLA y, sólo para los servicios de proxy, los informes de mensajes de pipeline y el registro de mensajes de pipeline.
