Desactivación de la Supervisión Global de los Servicios

Para desactivar la supervisión de todos los servicios de nivel de dominio:

Nota: Si desactiva la supervisión de todos los servicios, también se suprimen todas las estadísticas recopiladas hasta el momento para esos servicios. Estas estadísticas no se pueden restaurar con la función Deshacer de la sesión. Puede utilizar Deshacer para activar la supervisión de nuevo para los servicios, pero la supresión de las estadísticas es irreversible.
  1. Si aún no lo ha hecho, haga clic en Crear para crear una nueva sesión o en Editar para acceder a una sesión existente. Consulte Uso del Centro de Cambios.
  2. Seleccione Operaciones > Configuración Global.
  3. Desactive la casilla de control Activar Supervisión para que el sistema deje de recopilar estadísticas de supervisión para todos los servicios de la configuración.
    4. Nota: Esta opción sustituye a la opción Activar Supervisión de Servicio que puede seleccionar para servicios de negocio y de proxy concretos. Consulte Configuración Operativa de Servicios de Negocio y Configuración Operativa de Servicios de Proxy.
    Nota: Esta opción también sustituye a la opción Activar Alertas de SLA. Si desactiva la supervisión a nivel global, las alertas de SLA estarán desactivadas en tiempo de ejecución también, incluso si está activada la casilla de control Activar Alertas de SLA correspondiente. Consulte Activación Global de Alertas de SLA.
  4. Haga clic en Actualizar. La configuración de supervisión se guarda en la sesión actual.
  5. Como alternativa, haga clic en Restablecer en cualquier momento durante la sesión para desechar los cambios y refrescar la página con la configuración almacenada actualmente.
  6. Para terminar la sesión y desplegar la configuración en tiempo de ejecución, haga clic en Activar en Centro de Cambios.
