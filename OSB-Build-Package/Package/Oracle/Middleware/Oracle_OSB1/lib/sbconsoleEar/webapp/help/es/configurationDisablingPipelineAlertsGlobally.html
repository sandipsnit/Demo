Desactivación Global de Alertas de Pipeline

Para los servicios de proxy, para desactivar las alertas de pipeline de todos los servicios de proxy de nivel de dominio:

  1. Si aún no lo ha hecho, haga clic en Crear para crear una nueva sesión o en Editar para acceder a una sesión existente. Consulte Uso del Centro de Cambios.
  2. Seleccione Operaciones > Configuración Global.
  3. Desactive la casilla de control Activar Alertas de Pipeline para que el sistema deje de ejecutar las acciones de alerta de pipeline para los servicios de proxy.
    4. Nota: Esta opción sustituye a la configuración de Alertas de Pipeline que puede seleccionar para servicios de proxy concretos. Consulte Configuración Operativa de Servicios de Proxy.
  4. Haga clic en Actualizar. La configuración de alertas de pipeline se guarda en la sesión actual.
  5. Como alternativa, haga clic en Restablecer en cualquier momento durante la sesión para desechar los cambios y refrescar la página con la configuración almacenada actualmente.
  6. Para terminar la sesión y desplegar la configuración en tiempo de ejecución, haga clic en Activar en Centro de Cambios.
