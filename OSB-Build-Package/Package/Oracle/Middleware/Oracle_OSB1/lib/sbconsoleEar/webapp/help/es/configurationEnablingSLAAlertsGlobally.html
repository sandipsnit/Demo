Activación Global de Alertas de SLA

Aunque puede configurar las alertas de SLA de forma independiente a la supervisión, existe una interacción entre las dos configuraciones en tiempo de ejecución. Si se activa la supervisión global, las alertas de SLA se pueden activar o desactivar. Sin embargo, si se desactiva la supervisión global, las alertas de SLA estarán desactivadas porque las condiciones de las reglas de alertas de SLA dependen de las estadísticas de supervisión que se evalúan.

Para activar las alertas de SLA de todos los servicios a nivel de dominio:

  1. Si aún no lo ha hecho, haga clic en Crear para crear una nueva sesión o en Editar para acceder a una sesión existente. Consulte Uso del Centro de Cambios.
  2. Seleccione Operaciones > Configuración Global.
  3. Active la casilla de control Activar Alertas de SLA para que el sistema empiece a evaluar las reglas de alertas para todos los servicios de la configuración.
  4. Haga clic en Actualizar para guardar toda la configuración de esta página.
  5. Como alternativa, haga clic en Restablecer en cualquier momento durante la sesión para desechar los cambios y refrescar la página con la configuración almacenada actualmente.
  6. Para terminar la sesión y desplegar la configuración en tiempo de ejecución, haga clic en Activar en Centro de Cambios.
