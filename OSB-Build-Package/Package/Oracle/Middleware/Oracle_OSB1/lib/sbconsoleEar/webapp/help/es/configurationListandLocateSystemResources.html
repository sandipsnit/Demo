Búsqueda y Actualización de Configuración Operativa

Utilice la página Búsqueda Inteligente para localizar fácilmente los servicios de proxy, servicios de negocio, destinos de alertas y reglas de alertas de SLA. Además, puede utilizar la página Búsqueda Inteligente para especificar la configuración operativa específica del servicio, con las siguientes excepciones: No puede definir un intervalo de agregación ni especificar un nivel de gravedad de alerta o registro. Para obtener más información, consulte Configuración Operativa de Servicios de Proxy y Configuración Operativa de Servicios de Negocio.

Los efectos en tiempo de ejecución de la configuración a nivel de servicio dependen de su configuración global correspondiente. Debe activar tanto la configuración global como a nivel de servicio para que un servicio esté completamente activado en tiempo de ejecución. También se debe activar el estado del servicio. Consulte Activación de la Configuración Global.

Puede activar o desactivar la configuración operativa sólo desde una sesión.
