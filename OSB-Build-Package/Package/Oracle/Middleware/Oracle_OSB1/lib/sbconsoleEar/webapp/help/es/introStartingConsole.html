Inicio de la Consola de Administración de Oracle Service Bus

  1. Inicie Oracle Service Bus mediante uno de los siguientes métodos:
    • En el menú Inicio de Windows, seleccione:

      • Inicio > Todos los programas > Oracle WebLogic > Proyectos de Usuario > nombre_dominio > Iniciar Servidor para Dominio de Oracle Service Bus.

      Donde nombre_dominio representa el nombre asignado al dominio de Oracle Service Bus cuando lo creó. Proyectos de Usuario es la ubicación por defecto en la que se crea el dominio por defecto.

    • Invoque el comando startWebLogic desde la raíz del dominio de Oracle Service Bus: startWebLogic.cmd o startWebLogic.sh

      • Aparece una ventana de comandos con información de estado sobre WebLogic Server.

      Si ha iniciado el servidor en modo de producción, debe introducir un nombre de usuario y contraseña válidos para iniciar el servidor.

  2. Cuando se inicie el servidor, introduzca la siguiente URL en el explorador:

    3. http://nombre_host:puerto/sbconsole

    donde host representa el nombre de la máquina en la que se ejecuta WebLogic Server y puerto representa el número de puerto.

    Por ejemplo, si WebLogic Server se ejecuta en la máquina local con la configuración de puerto por defecto, introduzca la siguiente URL en el explorador:

    http://localhost:7001/sbconsole

    Aparece la página de conexión a la consola de administración de Oracle Service Bus.

  3. Introduzca el nombre de usuario y contraseña especificados durante el proceso de instalación.
    4. Nota: El nombre de usuario y la contraseña del dominio Examples de Oracle Service Bus es weblogic / welcome1.
  4. Haga clic en Conectar.

Para desconectarse, haga clic en Desconectar en el rótulo cerca de la parte superior del panel derecho de la consola de administración de Oracle Service Bus.
