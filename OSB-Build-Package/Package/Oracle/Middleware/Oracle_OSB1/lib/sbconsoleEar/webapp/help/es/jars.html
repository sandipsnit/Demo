JAR

Un archivo JAR (del inglés, Java ARchive) es un archivo comprimido que contiene un juego de clases Java. Se utiliza para almacenar clases Java compiladas y los metadatos asociados que pueden formar un programa. Un archivo JAR actúa como una biblioteca de programas que se puede llamar para los elementos de código Java (de esta forma, un solo enlace de compilación proporciona acceso a varios elementos, en lugar de necesitar enlaces para cada elemento individual). Para obtener más información sobre archivos JAR, consulte http://java.sun.com/docs/books/tutorial/deployment/jar/

Los recursos JAR se utilizan en Service Bus en:

  • Acciones de llamada Java
  • Servicios de negocio basados en EJB
  • Servicios de negocio y de proxy basados en Tuxedo
