Enlaces de JCA

Oracle SOA Suite permite la integración con Enterprise Information Systems (EIS) de backend mediante adaptadores J2EE Connector Architecture (JCA). Mediante Oracle JDeveloper y otros entornos de desarrollo de JCA soportados, puede configurar los adaptadores y crear los recursos del adaptador de JCA (archivos .jca y WSDL abstractos) utilizados para la integración de backend.

Puede comunicarse con los componentes de EIS compatibles con JCA mediante Oracle Service Bus (OSB), que permite consumir los recursos de JCA en forma de recursos de enlace de JCA. Con los enlaces de JCA, puede generar servicios de proxy y de negocio que utilicen el transporte de JCA de Oracle Service Bus para la comunicación con las aplicaciones de EIS mediante los adaptadores de JCA.

Este capítulo, en el que se describe cómo agregar, editar, localizar y suprimir enlaces de JCA en Oracle Service Bus, contiene los siguientes temas:

  • Localización de Enlaces de JCA
  • Adición de Enlaces de JCA
  • Edición de Enlaces de JCA
  • Supresión de Enlaces de JCA

Para obtener más información sobre el transporte·de·JCA, consulte la sección sobre transporte de JCA en Oracle Fusion Middleware Developer's Guide for Oracle Service Bus (Guía del Desarrollador de Oracle Fusion Middleware para Oracle Service Bus) en http://www.oracle.com/pls/as111170/lookup?id=OSBDV910.

Para obtener más información sobre los adaptadores de JCA de Oracle SOA Suite, consulte Oracle Fusion Middleware User's Guide for Technology Adapters (Guía del Usuario de Oracle Fusion Middleware para Technology Adapters) en http://www.oracle.com/pls/as111170/lookup?id=TKADP.
