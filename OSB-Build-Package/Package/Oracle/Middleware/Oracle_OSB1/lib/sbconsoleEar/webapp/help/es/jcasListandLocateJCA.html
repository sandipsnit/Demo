Localización de Enlaces de JCA

En la página Resumen de Enlaces de JCA aparecen los enlaces de JCA que ha agregado al dominio de Oracle Service Bus.

  1. Realice una de las siguientes acciones:
    • Seleccione Explorador de Proyectospara que aparezca la página Vista de proyectos o la página Vista de Proyectos/Carpetas. A continuación, navegue por los proyectos o carpetas para buscar el enlace de JCA.
    • Seleccione Explorador de Recursos > Enlaces de JCA. La página Resumen de Enlaces de JCA muestra la información que aparece en la Tabla 12-1. Para obtener una descripción más detallada de las propiedades, consulte Localización de Enlaces de JCA.
  2. Para buscar un enlace de JCA concreto:
    • Para limitar el número de elementos de la lista, puede filtrar por nombre y ruta de acceso antes de hacer clic en Buscar.

      • La ruta de acceso es el nombre del proyecto y el nombre de las carpetas donde reside el recurso.

    • Haga clic en Ver Todo para eliminar los filtros de búsqueda y mostrar todos los enlaces de JCA.

      • Tabla 12-1 Información de Enlace de JCA 
      Propiedad
      Descripción
      Nombre de Enlace de JCA
      Nombre único asignado al enlace de JCA. Haga clic en el nombre para ver la página Ver Detalles de Enlace de JCA. Consulte Edición de Enlaces de JCA.
      Ruta de Acceso
      Nombre del proyecto y de la carpeta donde reside el enlace de JCA. Haga clic en el nombre para ver el proyecto o la carpeta que contiene este recurso. Consulte Cualificación de Nombres de Recurso con Proyectos y Carpetas.
      Acción
      Contiene el icono Generar que permite generar un WSDL y servicio de JCA de Oracle Service Bus a partir del enlace de JCA.
      Opciones
      Contiene el icono Suprimir. Si se ha configurado un servicio de negocio o de proxy para utilizar un enlace de JCA, un icono de advertencia de supresión indica que puede suprimir el enlace de JCA con una confirmación de advertencia. Esto puede causar conflictos debido a referencias no resueltas del servicio al enlace de JCA suprimido. Consulte Supresión de Enlaces de JCA.
