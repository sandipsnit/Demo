Edición de Enlaces de JCA

Utilice la página Ver Detalles de Enlace de JCA para ver y cambiar los detalles de un enlace de JCA.

  1. Si aún no lo ha hecho, haga clic en Crear para crear una nueva sesión o en Editar para acceder a una sesión existente. Consulte Uso del Centro de Cambios.
  2. Localice el enlace de JCA, como se describe en Localización de Enlaces de JCA.
  3. Haga clic en el nombre del enlace de JCA. La página Ver un Enlace de JCA muestra la información de la Tabla 12-2.

    4. Tabla 12-2 Detalles de Enlace de JCA 
    Propiedad
    Descripción
    Modificado por
    Usuario que ha creado este enlace de JCA o lo ha importado a la configuración.
    Fecha de Modificación
    Fecha y hora a la que el usuario ha creado este enlace de JCA o lo ha importado a la configuración. Haga clic en el enlace de fecha y hora para ver el historial de cambios de este recurso. Consulte Página Ver Historial de Cambios.
    Referencias
    Proporciona un enlace al recurso de WSDL al que hace referencia el enlace de JCA. Consulte Visualización de Referencias a Recursos.
    Con Referencia de
    Número de objetos que hacen referencia a este enlace de JCA. Si existen estas referencias, haga clic en el enlace numérico para ver una lista de los objetos. Consulte Visualización de Referencias a Recursos.
    Icono Acción
    Con el Centro de Cambios en modo de edición, permite generar un WSDL y servicio de JCA de Oracle Service Bus a partir del enlace de JCA.
    Archivo JCA
    Muestra una vista de sólo lectura del enlace de JCA.
    WSDL Abstracto
    Muestra el WSDL asociado al enlace de JCA. Si el enlace de JCA no hace referencia a un WSDL, haga clic en Editar Referencias para hacer referencia al WSDL asociado.
    XML de TopLink
    Muestra el archivo de asignación de TopLink o EclipseLink necesario del enlace de JCA. Si el recurso del enlace de JCA no hace referencia a un archivo de asignación necesario, haga clic en Editar Referencias para seleccionar el archivo de asignación necesario. Este campo sólo aparece si el archivo JCA declara un recurso XML.
    XSLT
    Muestra el archivo XSLT necesario del enlace de JCA. Si el recurso del enlace de JCA no hace referencia a un recurso XSLT necesario, haga clic en Editar Referencias para seleccionar el archivo XSLT necesario. Este campo sólo aparece si el archivo JCA declara un recurso XSLT, por ejemplo, para un adaptador de socket.

  4. Para modificar la descripción o el archivo JCA, haga clic en Editar.

    5. No puede cambiar el nombre del recurso.

  5. Haga clic en Guardar para confirmar las actualizaciones en la sesión actual.
  6. Para cambiar las referencias a los recursos necesarios del enlace de JCA, o bien, para corregir las referencias no resueltas, haga clic en Editar Referencias y seleccione los recursos adecuados.
  7. Para terminar la sesión y desplegar la configuración en tiempo de ejecución, haga clic en Activar en Centro de Cambios.
