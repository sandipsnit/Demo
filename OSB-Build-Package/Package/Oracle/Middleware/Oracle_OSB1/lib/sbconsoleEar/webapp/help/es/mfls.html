MFL

Un documento MFL (Lenguaje de Formato de Mensajes) es un documento XML especializado utilizado para describir el diseño de los datos binarios. Es un lenguaje propiedad de Oracle que se utiliza para definir reglas que transforman los datos con formato binario en datos XML. Un documento MFL cumple con mfl.dtd, que incluye elementos y atributos utilizados para describir cada campo de datos, así como agrupaciones de campos (grupos), repetición y agregación.

Al crear un servicio de negocio o de proxy del tipo servicio de mensajes, puede seleccionar tipos MFL como tipo de mensaje de solicitud o de respuesta del servicio.

Utilice Oracle Format Builder para crear MFL. Al definir la jerarquía de un registro binario, el diseño de los campos y la agrupación de campos y grupos, la información se guarda como documento MFL que se puede utilizar para realizar traducciones en tiempo de ejecución. También se puede utilizar un documento MFL en Format Builder para generar el DTD correspondiente que describe su modelo de contenido. Para obtener más información sobre el uso de Format Builder, consulte "Format Builder" en Oracle Fusion Middleware Developer's Guide for Oracle Service Bus (Guía del Desarrollador de Oracle Fusion Middleware para Oracle Service Bus) en http://www.oracle.com/pls/as111170/lookup?id=OSBDV763.
