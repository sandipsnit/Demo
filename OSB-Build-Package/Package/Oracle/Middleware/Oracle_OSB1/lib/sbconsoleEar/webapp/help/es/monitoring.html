Supervisión

Al crear un servicio de negocio o de proxy, la supervisión está desactivada por defecto para ese servicio. Para obtener más información sobre la activación de la supervisión para servicios de proxy, consulte Configuración Operativa de Servicios de Proxy; para los servicios de negocio, consulte Configuración Operativa de Servicios de Negocio.

Para activar o desactivar la supervisión de todos los servicios que se han activado o desactivado de forma individual para la supervisión, utilice la opción Activar Supervisión de la página Operaciones > Configuración Global. Consulte Activación de la Configuración Global.

Para obtener más información sobre la supervisión en tiempo de ejecución, consulte la sección sobre supervisión de Oracle Service Bus en tiempo de ejecución en Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus (Guía del Administrador de Oracle Fusion Middleware para Oracle Service Bus) en http://www.oracle.com/pls/as111170/lookup?id=OSBAG472.
