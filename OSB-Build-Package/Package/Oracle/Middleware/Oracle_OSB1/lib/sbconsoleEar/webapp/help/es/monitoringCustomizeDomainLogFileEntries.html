Personalización de la Vista de las Entradas del Archivo Log del Dominio

  1. Haga clic en Personalizar Tabla. Aparecen campos adicionales.

    2. Puede hacer clic en el enlace Personalizar Tabla en cualquier momento para cerrar esta tabla y mantener la configuración original.

  2. En los campos Filtro, haga lo siguiente:
    1. En el campo Tiempo, seleccione el intervalo de tiempo del que desea ver las entradas del log.
    2. En el campo Hora de Inicio, introduzca una hora de inicio con el formato MM/DD/AAAA HH:MM:SS. Por ejemplo, introduzca 10/25/06 08:39:48. Puede utilizar los campos Hora de Inicio y Hora de Finalización para especificar el período para el que desea ver las entradas del log.
    3. En el campo Hora de Finalización, introduzca una hora de finalización con el formato MM/DD/AAAA HH:MM:SS. Por ejemplo, introduzca 10/25/06 13:20:51. Puede utilizar los campos Hora de Inicio y Hora de Finalización para especificar el período para el que desea ver las entradas del log.
    4. En el cuadro de texto Criterios, especifique los criterios de filtro (una cadena de texto). Los criterios de filtro se especifican como cadena en el lenguaje de consulta WLDF. Este lenguaje de consulta soporta operadores booleanos: AND, OR y NOT, y operadores relacionales. Para obtener más información sobre la sintaxis de consulta, consulte la sección sobre lenguaje de consulta WLDF en Oracle Fusion Middleware Configuring and Using the Diagnostics Framework for Oracle WebLogic Server (Configuración y Uso de Diagnostics Framework de Oracle Fusion Middleware para Oracle WebLogic Server) en http://www.oracle.com/pls/as111170/lookup?id=WLDFC316.
  3. En el campo Columnas, seleccione las columnas que desea mostrar:
    1. Seleccione un nombre de columna del campo Disponibles.
    2. Haga clic en la flecha para mover este nombre de columna al campo Seleccionadas.
    3. Repita la operación hasta que aparezcan todos los nombres de columna deseados en el campo Seleccionadas.

      4. En el campo Seleccionadas, puede utilizar las flechas Arriba y Abajo para cambiar el orden de las columnas como desee.

  4. En el campo Número de Filas Mostradas por Página, seleccione el número de entradas de log que desea mostrar en una sola página.
  5. En el campo Máximo de Resultados Devueltos, seleccione el número máximo de entradas de log que desea mostrar en total o elija Mostrar Todo para que aparezcan todas.
  6. Realice una de las siguientes acciones:
    • Para guardar la nueva configuración, haga clic en Aplicar.
    • Para desechar sus cambios y mantener la configuración original, haga clic en Restablecer.
