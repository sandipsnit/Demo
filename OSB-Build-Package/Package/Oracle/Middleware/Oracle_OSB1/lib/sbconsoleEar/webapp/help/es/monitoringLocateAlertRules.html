Localización de Reglas de Alertas

  1. Seleccione Explorador de Recursos > Servicios de Negocio o Servicios de Proxy.
  2. Haga clic en el nombre del servicio.
  3. Seleccione el separador Reglas de Alertas de SLA.
  4. Para localizar una determinada regla de alerta para este servicio, desplácese por las páginas. Utilice los controles de página situados sobre y debajo de la tabla. Para ir a una página, seleccione su número o utilice los botones de flecha para ir a la página siguiente, anterior, primera o última.

También puede utilizar esta página para realizar las siguientes operaciones:

  • Para configurar la supervisión de este servicio, seleccione el separador Configuración Operativa. Consulte Configuración Operativa de Servicios de Proxy y Configuración Operativa de Servicios de Negocio.
  • Para agregar una nueva regla de alerta, haga clic en Agregar Nuevo. Consulte Creación y Edición de Reglas de Alertas.
