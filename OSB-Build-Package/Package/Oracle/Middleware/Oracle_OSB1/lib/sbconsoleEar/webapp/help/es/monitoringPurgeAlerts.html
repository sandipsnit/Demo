Depuración de Alertas

Utilice esta página para suprimir de forma permanente las alertas de SLA o pipeline especificadas.

Nota: Esta acción no se puede deshacer.
  1. Seleccione Operaciones > Panel de Control.
  2. Seleccione el separador Alertas de SLA o Alertas de Pipeline y, a continuación, haga clic en el enlace Historial de Alertas Ampliado.
  3. En la parte superior derecha de la página Historial de Alertas Ampliado, haga clic en Depurar Historial de Alertas de SLA o Depurar Historial de Alertas de Pipeline.
  4. Realice uno de los pasos a continuación mostrados en la Tabla 25-3.

    5. Tabla 25-3 Opciones de Depuración de Alertas 
    Para...
    Realice estos pasos...
    • Depurar todas las alertas de SLA
    • Depurar todas las alertas de pipeline
    1. Seleccione Depurar Todas las Alertas.
    2. Haga clic en Depurar.
    • Depurar las alertas de SLA de un período especificado
    • Depurar las alertas de pipeline de un período especificado
    1. Seleccione Depurar Alertas desde... Hasta...
    2. En el campo Depurar desde, introduzca un mes, día, año y hora.
    3. En el campo Depurar hasta, introduzca un mes, día, año y hora.
    4. Haga clic en Depurar.

Nota: Si no está disponible el servidor desde el que se están depurando las alertas, el marco de registro conserva un registro de la depuración y sondea el servidor cada 30 minutos para comprobar su disponibilidad. Cuando el marco de registro detecta que el servidor está disponible, depura las alertas.
