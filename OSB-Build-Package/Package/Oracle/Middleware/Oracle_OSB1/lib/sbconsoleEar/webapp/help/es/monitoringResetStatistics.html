Restablecimiento de Estadísticas para Servicios

Puede utilizar la página Estado de Servicio para restablecer las estadísticas de supervisión de todos los servicios de la configuración. Puede restablecer las estadísticas tanto si está en una sesión como si no.

  1. Seleccione Operaciones > Panel de Control > Estado de Servicio.
  2. En el campo Mostrar Estadísticas, seleccione Desde Último Restablecimiento. Esta vista muestra un recuento continuo de las métricas.
  3. Para restablecer las estadísticas de supervisión de un determinado servicio, haga clic en el icono Restablecer de la columna Acción de ese servicio.

    4. Cuando confirma que desea realizar esta acción, el sistema suprime todas las estadísticas de supervisión recopiladas para el servicio desde que se restableció por última vez las estadísticas. Sin embargo, el sistema no suprime las estadísticas recopiladas durante el intervalo de agregación actual para el servicio. Además, después de hacer clic en el icono Restablecer, el sistema empieza inmediatamente a recopilar estadísticas de supervisión para el servicio de nuevo.

  4. Para restablecer las estadísticas de supervisión de todos los servicios, haga clic en el enlace Restablecer Todas las Estadísticas al principio de la página.
    5. Nota: Al hacer clic en Restablecer Todas las Estadísticas se restablecen las estadísticas de todos los servicios supervisados tanto si aparecen en la página como si no.

    Cuando confirma que desea restablecer las estadísticas, el sistema suprime todas las estadísticas de supervisión recopiladas para todos los servicios de la configuración desde que se restableció por última vez las estadísticas. El sistema no suprime las estadísticas recopiladas durante el intervalo de agregación actual para ninguno de los servicios. Además, después de hacer clic en Restablecer Todas las Estadísticas, el sistema empieza inmediatamente a recopilar estadísticas de supervisión para los servicios de nuevo.
