Revisión de la Configuración de Reglas de Alertas

Utilice esta página para revisar los datos de configuración introducidos para esta regla de alerta. Si es necesario, puede hacer clic en Editar para realizar cambios en la configuración antes de guardar la regla de alerta.

Realice una de las siguientes acciones:

  • Para realizar un cambio en una de las páginas de configuración, haga clic en el icono Editar de la página adecuada.
  • Para volver a la página anterior, haga clic en Anterior.
  • Para crear la alerta, haga clic en Guardar. Se creará la regla de alerta y se guardará en la sesión actual.
  • Para desechar los cambios, haga clic en Cancelar.
  • Para terminar la sesión y desplegar la configuración en tiempo de ejecución, haga clic en Activar en Centro de Cambios.
