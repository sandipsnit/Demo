Visualización de Detalles de Archivos Log de Dominio

Utilice esta página para ver los detalles de las entradas del archivo log de dominio mostrados en la Tabla 25-18.

Tabla 25-18 Detalles de Log de Dominio 
Propiedad
Descripción
Mensaje
Descripción del evento o condición.
Fecha
Muestra la hora y fecha en que se originó el mensaje, en un formato específico de la configuración regional. La Java Virtual Machine (JVM) que ejecuta cada instancia de WebLogic Server consulta el sistema operativo de la computadora host para obtener información sobre la zona horaria local y el formato.
Subsistema
Indica el subsistema de WebLogic Server que ha sido el origen del mensaje; por ejemplo, un contenedor Enterprise Java Bean (EJB) o Java Messaging Service (JMS).
Identificador de Mensaje
Identificador único de seis dígitos.
Todos los identificadores de mensaje que generan los mensajes del sistema WebLogic Server empiezan por BEA- y están incluidos en un rango numérico de 0-499999.
Gravedad
Indica el grado de impacto o la seriedad del evento notificado por el mensaje:
  • Alerta: Un servicio concreto está inutilizable mientras que otras partes del sistema siguen funcionando. No es posible la recuperación automática; es necesaria la atención inmediata del administrador para resolver el problema.
  • Crítico: Se ha producido un error del sistema o servicio. Se puede recuperar el sistema pero puede haber una pérdida momentánea o una degradación permanente del servicio.
  • Emergencia: El servidor está inutilizable. Esta gravedad indica un fallo grave del sistema.
  • Error: Se ha producido un error de usuario. El sistema o la aplicación puede manejar el error sin interrupciones y con una degradación limitada del servicio.
  • Información: Se utiliza para informar de operaciones normales; mensaje informativo de bajo nivel.
  • Aviso: Mensaje informativo con un nivel mayor de importancia.
  • Advertencia: Se ha producido una operación o configuración sospechosa pero puede que no afecte al funcionamiento normal.
Máquina
Nombre DNS de la computadora que aloja la instancia del servidor.
Servidor
Nombre de la instancia de WebLogic Server en la que se ha generado el mensaje.
Thread
Identificador que JVM asigna al thread en el que se ha originado el mensaje.
Identificador de Usuario
Identificador de usuario con el que se ha ejecutado el evento asociado.
Identificador de Transacción
Sólo presente para mensajes registrados dentro del contexto de una transacción.
Identificador de Contexto
Información de contexto para correlacionar mensajes procedentes de una solicitud o una aplicación concreta.
Detalles
Descripción del evento o condición.
Causa
Causa del mensaje.
Acción
Acción que se debe realizar.
