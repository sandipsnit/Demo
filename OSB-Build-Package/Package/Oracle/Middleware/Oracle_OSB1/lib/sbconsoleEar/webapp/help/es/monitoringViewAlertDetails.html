Visualización de Detalles de Alerta

Esta página muestra los detalles de alertas concretas incluida la información mostrada en la Tabla 25-4.

Tabla 25-4 Detalles de Alerta 
Propiedad
Descripción
Nombre de Alerta
Sólo para alertas de SLA:
Nombre asignado a la alerta. El nombre es un enlace a la página Configuración de Regla de Alerta. Consulte Visualización de Configuraciones de Reglas de Alertas.
Resumen de Alertas
Sólo para alertas de pipeline:
Descripción breve de la acción de alerta de pipeline o alerta de Oracle Service Bus (valor por defecto proporcionado para las acciones de alertas de pipeline diseñadas sin texto de resumen de alerta).
Descripción
Descripción de la regla de alerta.
Registro de Hora
  • Para alertas de pipeline: Fecha y hora en que se produjo la alerta.
  • Para alertas de SLA: Fecha y hora en que la regla de alerta se evaluó por última vez en verdadera.
Gravedad
Gravedad definida por el usuario de la alerta.
  • Fatal
  • Crítica
  • Mayor
  • Menor
  • Advertencia
  • Normal
Servicio
Nombre y ruta de acceso del servicio asociado a esta alerta. El campo es un enlace a la página Detalles de Supervisión del Servicio (sólo si se está supervisando activamente el servicio). Consulte Visualización de Métricas de Servicio.
Tipo de Servicio
Tipo de servicio principal: Servicio de proxy o servicio de negocio.
Sólo los servicios de proxy pueden definir alertas de pipeline, por lo tanto, para las alertas de pipeline, todos los servicios principales son servicios de proxy.
Servidor
Nombre del servidor en el que se ha generado esta alerta.
Sólo para alertas de SLA: Aparece N/A.
Anotación
Cuadro de texto en el que puede introducir notas para esta alerta. Aparece un icono Anotación en el Historia del Alertas para las alertas con anotaciones.
