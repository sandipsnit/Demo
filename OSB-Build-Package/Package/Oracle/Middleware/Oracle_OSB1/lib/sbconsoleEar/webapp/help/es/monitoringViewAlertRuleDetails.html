Nombre de la Regla

Resumen de Alertas
Descripción breve de la regla de alerta. Este texto también se utiliza como línea de asunto de un mensaje de correo electrónico si se configura esta regla de alerta con una notificación de alerta de correo electrónico.
Destino de Alerta
Nombre y ruta de acceso del recurso de destino de alerta que contiene la dirección de destino para las notificaciones de alerta. Este campo es un enlace a la página Ver Destino de Alerta: Configuración General. Consulte Edición de Destinos de Alertas.
Hora de Inicio (HH:MM)
Hora de inicio de esta regla de alerta.
Hora de Finalización (HH:MM)
Hora de finalización de esta regla de alerta.
Fecha de Caducidad de la Regla (MM/DD/AAAA)
Fecha en la que esta regla de alerta deja de aplicarse.
Regla Activada
Indica si esta regla de alerta está activada o no.
Gravedad de Alerta
Gravedad de la alerta emitida como resultado de esta regla de alerta:
  • Normal
  • Advertencia
  • Menor
  • Mayor
  • Crítica
  • Fatal
Frecuencia de Alerta
Frecuencia de la alerta:
  • Cada Vez
  • Notificar una Vez
Parar Procesamiento de Más Reglas
Indica si la alerta deja de procesar más reglas.
