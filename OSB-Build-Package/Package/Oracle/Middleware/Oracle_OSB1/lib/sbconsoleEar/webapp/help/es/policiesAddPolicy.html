Adición de Políticas de Servicios Web Personalizadas

Puede importar un archivo XML que contenga la política de servicio web o escribir la política de servicio web directamente en la consola de administración de Oracle Service Bus.

  1. Si aún no lo ha hecho, haga clic en Crear para crear una nueva sesión o en Editar para acceder a una sesión existente. Consulte Uso del Centro de Cambios.
  2. Seleccione Explorador de Proyectos y elija el proyecto o carpeta donde desea agregar la política de servicio web. Aparece la página Vista de Proyectos/Carpetas.
  3. En la lista desplegable Crear Recurso, seleccione la opción Política de Servicios Web en Interfaz para mostrar la página Crear una Nueva Política de Servicios Web.
  4. En el campo Nombre del Recurso , introduzca un nombre único para estas WS-Policy.

    Siga las Restricciones de Nomenclatura de Recursos para obtener ayuda con la nomenclatura.
  5. En el campo Descripción del Recurso, introduzca una descripción para esta política de servicio web.
  6. En el campo Política, realice una de las siguientes acciones:
    • Haga clic en Examinar para buscar e importar un archivo XML que contenga la política de servicio web personalizada.
    • Introduzca el texto para la nueva política de servicio web.
    • Copie y pegue el texto de una política de servicio web existente en este campo.
  7. Haga clic en Guardar. La nueva política de servicio web se guarda en la sesión actual.
  8. Para terminar la sesión y desplegar la configuración en tiempo de ejecución, haga clic en Activar en Centro de Cambios.
