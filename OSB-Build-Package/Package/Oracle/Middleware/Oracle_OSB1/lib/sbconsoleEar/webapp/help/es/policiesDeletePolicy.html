Supresión de Políticas de Servicios Web Personalizadas

Si algún servicio de negocio o de proxy está configurado para utilizar la política de servicio web, elimine la política de estos servicios.

  1. Si aún no lo ha hecho, haga clic en Crear para crear una nueva sesión o en Editar para acceder a una sesión existente. Consulte Uso del Centro de Cambios.
  2. Seleccione Explorador de Recursos > Políticas de Servicios Web.
  3. Haga clic en el icono Suprimir del campo Opciones de la política de servicio web que desea suprimir.

    4. La política de servicio web se suprime de la sesión actual. Si se ha configurado un servicio de negocio o de proxy para utilizar una política de servicio web, un icono de advertencia de supresión indica que puede suprimir la política de servicio web con una confirmación de advertencia. Esto puede causar conflictos debido a referencias no resueltas del servicio a la política de servicio web suprimida.

  4. Para terminar la sesión y desplegar la configuración en tiempo de ejecución, haga clic en Activar en Centro de Cambios.
