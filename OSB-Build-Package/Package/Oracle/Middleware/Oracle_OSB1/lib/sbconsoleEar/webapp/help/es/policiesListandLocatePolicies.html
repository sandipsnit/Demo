Búsqueda de Políticas de Servicios Web Personalizadas

La página Resumen de Políticas de Servicios Web muestra las políticas de servicios web personalizadas que ha agregado al dominio actual de Oracle Service Bus. La consola de administración de Oracle Service Bus no muestra las políticas de servicios web de Oracle Service Bus, que están preempaquetadas y disponibles para todos los dominios de Oracle Service Bus.

  1. Realice una de las siguientes acciones:
    • Seleccione Explorador de Proyectospara que aparezca la página Vista de proyectos o la página Vista de Proyectos/Carpetas. A continuación, navegue por los proyectos o carpetas para buscar la política de servicios web.
    • Seleccione Explorador de Recursos > Políticas de Servicios Web. La página Resumen de Políticas de Servicios Web muestra la información que aparece en la Tabla 11-1. Para obtener una descripción más detallada de las propiedades, consulte Búsqueda de Políticas de Servicios Web Personalizadas.
  2. Para·buscar·una·política·de·servicio·web·personalizada,·introduzca·parte·o·todo·el·nombre·de·la·política de servicio web en el campo Nombre. También puede introducir parte o todo el nombre de proyecto y carpeta de la política de servicio web en los campos Ruta de Acceso. Haga clic en Buscar.

    3. Haga clic en Ver Todo para eliminar los filtros de búsqueda y mostrar todas las políticas de servicios web.

    Tabla 11-1 Información de Política de Servicio Web 
    Propiedad
    Descripción
    Nombre de Política de Servicios Web
    Nombre único asignado a la política de servicios web. Haga clic en el nombre para ver la página Ver Detalles de Política de Servicios Web. Consulte Edición de Políticas de Servicios Web Personalizadas.
    Ruta de Acceso
    Nombre del proyecto y de la carpeta donde reside la política de servicio web. Haga clic en el nombre para ver el proyecto o la carpeta que contiene este recurso. Consulte Cualificación de Nombres de Recurso con Proyectos y Carpetas.
    Opciones
    Contiene el icono Suprimir. Si se ha configurado un servicio de negocio o de proxy para utilizar una política de servicio web, un icono de advertencia de supresión indica que puede suprimir la política de servicio web con una confirmación de advertencia. Esto puede causar conflictos debido a referencias no resueltas del servicio a la política de servicio web suprimida. Consulte Supresión de Políticas de Servicios Web Personalizadas.
