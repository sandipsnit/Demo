Edición de Políticas de Servicios Web Personalizadas

Utilice la página Ver Detalles de Política de Servicios Web para ver y cambiar los detalles de una política de servicio web personalizada. La consola de administración de Oracle Service Bus no muestra las políticas de servicios web de Oracle Service Bus, que son de sólo lectura.

  1. Si aún no lo ha hecho, haga clic en Crear para crear una nueva sesión o en Editar para acceder a una sesión existente. Consulte Uso del Centro de Cambios.
  2. Busque la política de servicio web, como se describe en Búsqueda de Políticas de Servicios Web Personalizadas.
  3. Haga clic en el nombre de la política de servicio web. La página Ver Detalles de Política de Servicios Web muestra la información que se ve en la Tabla 11-2.

    4. Tabla 11-2 Detalles de Política de Servicios Web 
    Propiedad
    Descripción
    Modificado por
    Usuario que ha creado esta política de servicio web o la ha importado a la configuración.
    Fecha de Modificación
    Fecha y hora a la que el usuario ha creado esta política de servicio web o la ha importado a la configuración. Haga clic en el enlace de fecha y hora para ver el historial de cambios de este recurso. Consulte Página Ver Historial de Cambios.
    Referencias
    Número de objetos a los que hace referencia esta política de servicio web. Si existen estas referencias, haga clic en el enlace numérico para ver una lista de los objetos. Consulte Visualización de Referencias a Recursos.
    Con Referencia de
    Número de objetos que hacen referencia a esta política de servicio web. Si existen estas referencias, haga clic en el enlace numérico para ver una lista de los objetos. Consulte Visualización de Referencias a Recursos.
    Descripción
    Descripción de la política de servicio web, si existe.
    Política
    Texto de esta política de servicio web.

  4. Para realizar un cambio en los campos, haga clic en Editar. Para ver las descripciones de los campos, consulte Adición de Políticas de Servicios Web Personalizadas.

    5. No puede cambiar el campo Nombre del Recurso.

  5. Haga clic en Guardar para confirmar las actualizaciones en la sesión actual.
  6. Para terminar la sesión y desplegar la configuración en tiempo de ejecución, haga clic en Activar en Centro de Cambios.
