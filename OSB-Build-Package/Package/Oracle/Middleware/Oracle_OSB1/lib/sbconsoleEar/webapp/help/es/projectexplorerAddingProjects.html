Adición de Proyectos

  1. Si aún no lo ha hecho, haga clic en Crear para crear una nueva sesión o en Editar para acceder a una sesión existente. Consulte Uso del Centro de Cambios.
  2. Seleccione Explorador de Proyectos > Proyectos para mostrar la página Vista de Proyectos.
  3. En el campo Introducir Nombre de Nuevo Proyecto, introduzca un nombre único para el proyecto y haga clic en Agregar Proyecto. Consulte Asignación de Nombre a Proyectos y Carpetas. El nuevo proyecto se guarda en la sesión actual.
  4. Para terminar la sesión y desplegar la configuración en tiempo de ejecución, haga clic en Activar en Centro de Cambios.
