Clonación de Proyectos

Hay tres formas de clonar (copiar) un proyecto:

  • Clonar el proyecto con un nuevo nombre a nivel de raíz. En este caso, el proyecto original y su clon existen como proyectos iguales. El proyecto clonado contiene las mismas carpetas y recursos que el original (al menos hasta que modifique el original o el clon). Para obtener instrucciones, consulte Clonación de un Proyecto a Nivel de Raíz.
  • Clonar proyecto con el nombre de un proyecto existente (que no sea el proyecto de origen). En este caso, las carpetas y los recursos del proyecto de origen se fusionan con las carpetas y recursos del proyecto con el nombre asignado al clon. El proyecto original se mantiene sin cambios. Para obtener instrucciones, consulte Clonación de un Proyecto a Nivel de Raíz.
  • Clonar el proyecto en una nueva ubicación, en un proyecto o carpeta de proyecto existente. En este caso, el proyecto clonado baja de nivel a carpeta en la nueva ubicación. Contiene las mismas carpetas y recursos que el proyecto original.

    • Si clona un proyecto con el mismo nombre y en la misma ubicación que una carpeta existente, el contenido (subcarpetas y recursos) del proyecto se fusiona con el contenido de la carpeta de destino.

    En cualquier caso, el proyecto original permanece sin cambios.

    Para obtener instrucciones, consulte Clonación de un Proyecto y Bajada de Nivel del Clon a Carpeta.

Clonación de un Proyecto a Nivel de Raíz
  1. Si aún no lo ha hecho, haga clic en Crear para crear una nueva sesión o en Editar para acceder a una sesión existente. Consulte Uso del Centro de Cambios.
  2. Seleccione Explorador de Proyectos > Proyectos para mostrar la página Vista de Proyectos.
  3. Haga clic en el icono Clonar Proyecto en la fila del proyecto que desee clonar. Aparece la página Clonar Proyecto con la ubicación actual del proyecto resaltado.
  4. En el campo Nombre Nuevo, introduzca un nombre para el clon. Consulte Asignación de Nombre a Proyectos y Carpetas.

    5. Utilice un nombre único para crear el clon como nuevo proyecto o use el nombre de un proyecto existente para fusionar el clon con ese proyecto.

  5. En el panel Ubicación, mantenga la ubicación raíz por defecto al clonar un proyecto a nivel de raíz.
  6. Haga clic en Clonar Proyecto. El proyecto se clona en la sesión actual.
  7. Para terminar la sesión y desplegar la configuración en tiempo de ejecución, haga clic en Activar en Centro de Cambios.
Clonación de un Proyecto y Bajada de Nivel del Clon a Carpeta
  1. Si aún no lo ha hecho, haga clic en Crear para crear una nueva sesión o en Editar para acceder a una sesión existente. Consulte Uso del Centro de Cambios.
  2. Seleccione Explorador de Proyectos > Proyectos para mostrar la página Vista de Proyectos.
  3. Haga clic en el icono Clonar Proyecto en la fila del proyecto que desee clonar. Aparece la página Clonar Proyecto con la ubicación actual del proyecto resaltado.
  4. En el campo Nombre Nuevo, puede introducir un nombre nuevo para la carpeta. Consulte Asignación de Nombre a Proyectos y Carpetas.

    5. Utilice un nombre único para crear una nueva carpeta o use el nombre de una carpeta existente para fusionar el clon con la carpeta existente (en la ubicación seleccionada en el siguiente paso).

  5. En el panel Ubicación, seleccione un proyecto o carpeta y, a continuación, haga clic en Clonar Proyecto. El proyecto clonado se convierte en carpeta en la nueva ubicación y se guarda en la sesión actual.
  6. Para terminar la sesión y desplegar la configuración en tiempo de ejecución, haga clic en Activar en Centro de Cambios.
