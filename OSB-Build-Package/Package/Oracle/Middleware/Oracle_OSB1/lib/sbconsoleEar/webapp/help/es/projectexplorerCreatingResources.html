Creación de Recursos

  1. Si aún no lo ha hecho, haga clic en Crear para crear una nueva sesión o en Editar para acceder a una sesión existente. Consulte Uso del Centro de Cambios.
  2. Seleccione Explorador de Proyectos y elija el proyecto o la carpeta donde desea crear el recurso.
  3. En la lista desplegable Crear Recurso de la página Vista de Proyectos/Carpetas, seleccione el tipo de recurso que desea crear. Se muestran diferentes páginas, según el tipo de recurso seleccionado.
  4. Siga las peticiones de datos para agregar información a cada página para crear el recurso. Para obtener más información sobre la creación de cada clase de recurso, consulte el tema correspondiente, como se muestra en la Tabla 3-3.

  6. Para terminar la sesión y desplegar la configuración en tiempo de ejecución, haga clic en Activar en Centro de Cambios.
