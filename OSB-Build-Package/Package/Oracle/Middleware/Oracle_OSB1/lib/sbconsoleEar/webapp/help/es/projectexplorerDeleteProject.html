Supresión de Proyectos

  1. Si aún no lo ha hecho, haga clic en Crear para crear una nueva sesión o en Editar para acceder a una sesión existente. Consulte Uso del Centro de Cambios.
  2. Seleccione Explorador de Proyectos > Proyectos para mostrar la página Vista de Proyectos.
  3. Haga clic en el icono Suprimir Proyecto en la fila del proyecto que desee suprimir. El proyecto se suprime en la sesión actual.

    4. Al suprimir un proyecto, se suprimen todos los recursos que contiene. Si otros recursos de un proyecto diferente hacen referencia a algún recurso de este proyecto, puede suprimir el proyecto pero con una confirmación de advertencia. Esto puede causar conflictos debido a referencias no resueltas al recurso suprimido.

    En lugar del icono Suprimir, aparece un icono de advertencia de supresión para estos recursos.

  4. Para terminar la sesión y desplegar la configuración en tiempo de ejecución, haga clic en Activar en Centro de Cambios.
