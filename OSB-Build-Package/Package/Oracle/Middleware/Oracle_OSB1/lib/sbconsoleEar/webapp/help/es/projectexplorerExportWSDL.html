Exportación de WSDL

Utilice la página Vista de Proyectos/Carpetas para exportar un WSDL asociado a un servicio de proxy o negocio. WSDL se exporta como archivo JAR. Sólo puede exportar WSDL desde fuera de una sesión. Consulte Uso del Centro de Cambios.

Utilice la función Exportar WSDL para que WSDL esté disponible para su consumo en herramientas externas como IDE. Tenga en cuenta que es diferente a la funcionalidad Exportar Recursos del módulo Administración del Sistema, que se utiliza para mover y crear etapas de los recursos entre dos dominios.

  1. En el panel de navegación izquierdo, haga clic en una proyecto o carpeta del Explorador de Proyectos. Aparece la página Vista de Proyectos/Carpetas.
  2. Haga clic en el icono Exportar WSDL en la fila del servicio de proxy cuyo WSDL desea exportar. Aparece el cuadro de diálogo Descarga de Archivo.
  3. En el cuadro de diálogo Descarga de Archivo, haga clic en Abrir para abrir el archivo JAR o en Guardar para guardar el archivo JAR en el escritorio.
