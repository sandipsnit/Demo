Generación de un WSDL

Un WSDL generado es un recurso de WSDL que genera Oracle Service Bus para un servicio que no se ha iniciado con un recurso de WSDL pero que se puede describir con un WSDL. Por ejemplo, un WSDL creado a partir de un servicio basado en EJB es un WSDL generado. Para obtener más información, consulte la sección sobre cómo utilizar WSDL en Oracle Service Bus en Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus (Guía del Administrador de Oracle Fusion Middleware para Oracle Service Bus) en http://www.oracle.com/pls/as111170/lookup?id=OSBAG114.

Utilice la página Vista de Proyectos/Carpetas para generar un WSDL asociado a un servicio de negocio de tipo de transporte de EJB o flujo.

  1. En el panel de navegación izquierdo, haga clic en una proyecto o carpeta del Explorador de Proyectos. Aparece la página Vista de Proyectos/Carpetas.
  2. Haga clic en el icono Generar WSDL en la fila del servicio de negocio cuyo WSDL desea generar. Aparece el cuadro de diálogo Generar Recurso de WSDL con la ubicación actual (nombre de proyecto y nombre de la carpeta en la que reside el servicio de negocio) resaltada.
  3. En el cuadro de diálogo Generar Recurso de WSDL, especifique un nuevo nombre de recurso de WSDL y la ubicación del WSDL, a continuación, haga clic en Generar WSDL.

    Siga las Restricciones de Nomenclatura de Recursos para obtener ayuda con la nomenclatura.

Después de Terminar

Continúe en Edición de WSDL para ver o modificar el WSDL generado.
