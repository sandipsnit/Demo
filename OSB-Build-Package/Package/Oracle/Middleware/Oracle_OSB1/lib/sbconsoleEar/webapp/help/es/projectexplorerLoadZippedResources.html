Seleccionar Archivo Zip
Introduzca el nombre completamente cualificado del archivo Zip que desea importar o haga clic en Examinar para seleccionarlo.
Tipo de Recurso y Extensión de Archivo
Cada archivo del archivo Zip es un posible recurso y la extensión de archivo es la forma más común de identificar el tipo de archivo. Oracle Service Bus define una extensión por defecto para cada recurso, pero si dispone de extensiones adicionales para describir esos recursos, esta asignación permite declarar esas extensiones en el cargador.
Después de definir el archivo Zip y la asignación de extensiones, el cargador de Oracle Service Bus explora el contenido del archivo Zip e intenta asociar un tipo de recurso a cada archivo. Los archivos sin extensiones o sin extensiones definidas en la asignación se consideran desconocidos y se excluyen automáticamente de la carga.
En el caso de los archivos conocidos, el nombre del recurso es el nombre del archivo sin la extensión. La estructura de carpetas del archivo ZIP se vuelve a crear en la carpeta de destino de la consola de administración de Oracle Service Bus.
