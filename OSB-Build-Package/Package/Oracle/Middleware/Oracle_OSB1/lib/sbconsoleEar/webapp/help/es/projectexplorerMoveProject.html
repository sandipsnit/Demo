Movimiento de Proyectos (Conversión en Carpeta)

Al "mover" un proyecto, éste se convierte en carpeta en la nueva ubicación.

  1. Si aún no lo ha hecho, haga clic en Crear para crear una nueva sesión o en Editar para acceder a una sesión existente. Consulte Uso del Centro de Cambios.
  2. Seleccione Explorador de Proyectos > Proyectos para mostrar la página Vista de Proyectos.
  3. Haga clic en el icono Mover Proyecto en la fila del proyecto que desee mover. Aparece la página Mover Proyecto con la ubicación actual del proyecto resaltado.
  4. En el panel Ubicación, seleccione un proyecto o carpeta al que mover el proyecto. El proyecto se convierte en carpeta en la nueva ubicación y se guarda en la sesión actual.

    5. Si mueve un proyecto a una nueva ubicación y ya existe una carpeta con el mismo nombre en esa ubicación, el contenido del proyecto se fusiona con el contenido de la carpeta de existente.

  5. Para terminar la sesión y desplegar la configuración en tiempo de ejecución, haga clic en Activar en Centro de Cambios.
