Cambio de Nombre de Proyectos

  1. Si aún no lo ha hecho, haga clic en Crear para crear una nueva sesión o en Editar para acceder a una sesión existente. Consulte Uso del Centro de Cambios.
  2. Seleccione Explorador de Proyectos > Proyectos para mostrar la página Vista de Proyectos.
  3. Haga clic en el icono Cambiar Nombre de Proyecto en la fila del proyecto que desee cambiar de nombre. Aparece la página Cambiar Nombre de Proyecto.
  4. En el campo Nuevo Nombre de Proyecto, introduzca un nombre nuevo para el proyecto. Consulte Asignación de Nombre a Proyectos y Carpetas.

    5. Si utiliza el nombre de un proyecto existente, se fusionan los recursos de ambos proyectos con ese nombre. Por ejemplo, si existe el proyecto MiProyecto con los recursos A y B, y el proyecto EsteProyecto con los recursos C y D. Al cambiar el nombre de MiProyecto por EsteProyecto, el nuevo EsteProyecto tendrá los recursos A, B, C y D.

  5. Haga clic en Cambiar Nombre. El proyecto cambia de nombre en la sesión actual.
  6. Para terminar la sesión y desplegar la configuración en tiempo de ejecución, haga clic en Activar en Centro de Cambios.
