Página Cargar Recursos: Revisar Recursos Cargados

La página Cargar Recursos: Revisar Recursos Cargados muestra detalles sobre los recursos importados de un archivo Zip u otro archivo, como se describe en Carga de Recursos de un Archivo Zip o Carga de Recursos de una URL.

En la Tabla 3-6 se describen las opciones y los controles de la página.

Tabla 3-6 Página Cargar Recursos: Revisar Recursos Cargados: Opciones y Controles 
Opciones y Controles
Descripción
Nombre de Archivo
Nombre de archivo del recurso, incluida la extensión de archivo.
Nombre
Nombre del recurso.
Ruta de Acceso
Ruta de acceso del recurso, que es el proyecto y la carpeta donde reside el recurso.
Tipo de Recurso
Muestra los tipos de recursos soportados.

Un icono de advertencia junto a un nombre de archivo indica que el tipo de recurso es desconocido. (Los archivos sin extensiones o sin extensiones definidas en la asignación se consideran desconocidos.) No se pueden importar los archivos de tipo desconocido.
