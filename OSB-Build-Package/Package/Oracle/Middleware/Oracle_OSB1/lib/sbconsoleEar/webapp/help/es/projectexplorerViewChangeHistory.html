Página Ver Historial de Cambios

La página Ver Historial de Cambios muestra los diez cambios más recientes realizados en un recurso, en todas las activaciones de sesión.

En la Tabla 3-9 se describen las opciones y los controles de la página.

Tabla 3-9 Página Historial de Cambios: Opciones y Controles 
Opción o Control
Descripción
Tarea
Tipo de tarea realizada:
  • Crear
  • Actualizar
  • Clonar
  • Cambiar Nombre
  • Mover
Usuario
Usuario que ha creado o editado este recurso.
Tiempo de Ejecución
Fecha y hora en que el usuario ha creado o editado este recurso.
