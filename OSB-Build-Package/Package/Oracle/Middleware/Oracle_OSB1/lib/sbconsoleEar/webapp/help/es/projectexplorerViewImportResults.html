Página Cargar Recursos: Resultado de Importación

La página Cargar Recursos: Resultado de Importación muestra los resultados de la carga de recursos desde un archivo Zip importado o una URL.

Dependiendo de si se han cargado los recursos desde un archivo Zip o una URL, asegúrese de realizar los pasos de Carga de Recursos de un Archivo Zip o Carga de Recursos de una URL, así como los pasos de la página Cargar Recursos: Revisar Recursos Cargados.

Aparece la página Resultado de Importación, que muestra la siguiente información.

Tabla 3-7 Página Vista de Proyectos/Carpetas: Opciones y Controles 
Opción o Control
Descripción
Estado
Estado del recurso, por ejemplo, si se ha importado correctamente.
Es posible que se marque un WSDL como no válido de forma temporal hasta que se importen todas sus dependencias. Sin embargo, cuando termina la acción de importación, el servicio de configuración vuelve a validar los archivos marcados como no válidos.
Nombre
Nombre del recurso.
Ruta de Acceso
Ruta de acceso del recurso, que es el proyecto y la carpeta donde reside el recurso.
Tipo de Recurso
Muestra el tipo de recurso importado.
Mensaje de Error
Muestra los detalles del mensaje de error si el recurso no se ha importado correctamente.
