Página Ver Referencias

Dependiendo de si ha hecho clic en Referencias o Con Referencia de en la página Vista de Proyectos/Carpetas, la página Ver Referencias muestra una lista de:

  • Los recursos fuera de un proyecto o carpeta actual a los que hacen referencia los recursos de dentro del proyecto o carpeta.
  • Los recursos fuera de un proyecto o carpeta actual que hacen referencia a los recursos de dentro del proyecto o carpeta.

En la Tabla 3-8 se describen las opciones y los controles de la página.

Tabla 3-8 Página Ver Referencias: Opciones y Controles 
Opción o Control
Descripción
Nombre
Haga clic en el nombre del recurso para ver sus detalles.
Tipo de Recurso
Muestra Proyecto, Carpeta o cualquiera de los tipos de recursos soportados por Oracle Service Bus.
