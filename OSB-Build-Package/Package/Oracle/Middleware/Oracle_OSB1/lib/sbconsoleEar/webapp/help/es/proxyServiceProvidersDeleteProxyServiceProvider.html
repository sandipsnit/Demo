Supresión de Proveedores de Claves de Servicio

Al suprimir un proveedor de claves de servicio, Oracle Service Bus también suprime el alias asociado a enlaces de par de claves del proveedor de asignación de credenciales PKI. Oracle Service Bus no suprime el par de clave-certificado asociado del almacén de claves.

  1. Si aún no lo ha hecho, haga clic en Crear para crear una nueva sesión o en Editar para acceder a una sesión existente. Consulte Uso del Centro de Cambios.
  2. Si se configura cualquier servicio de proxy para utilizar el proveedor de claves de servicio, elimine el proveedor de claves de servicio del servicio de proxy. Consulte Edición de Configuraciones de Servicio de Proxy.
  3. Seleccione Explorador de Recursos > Proveedores de Claves de Servicio para mostrar la página Resumen de Proveedores de Claves de Servicio.
  4. Haga clic en el icono Suprimir en el campo Opciones del proveedor de claves de servicio que desea suprimir. El proveedor de claves de servicio se suprime de la sesión actual. Si se ha configurado un servicio de negocio o de proxy para utilizar una cuenta de servicio, un icono Advertencia de Supresión indica que puede suprimir el proveedor de claves de servicio con una confirmación de advertencia. Esto puede causar conflictos debido a referencias no resueltas del servicio al proveedor de claves de servicio suprimido.
  5. Para terminar la sesión y desplegar la configuración en tiempo de ejecución, haga clic en Activar en Centro de Cambios.
