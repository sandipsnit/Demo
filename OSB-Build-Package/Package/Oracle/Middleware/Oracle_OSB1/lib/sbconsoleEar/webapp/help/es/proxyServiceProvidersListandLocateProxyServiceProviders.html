Localización de Proveedores de Claves de Servicio

  1. Realice una de las siguientes acciones:
    • Seleccione Explorador de Proyectos para que aparezca la página Vista de Proyectos o la página Vista de Proyectos/Carpetas. A continuación, navegue por los proyectos o carpetas para buscar el proveedor de claves de servicio.
    • Seleccione Explorador de Recursos > Proveedores de Claves de Servicio. La página Resumen de Proveedores de Claves de Servicio muestra la información que se ve en la Tabla 17-1.
  2. Para buscar un proveedor de claves de servicio, introduzca el nombre del proveedor, o parte de él, en el campo Nombre. También puede introducir el nombre del proyecto o la carpeta del proveedor, o parte de él, en los campos Ruta de Acceso. Haga clic en Buscar.

    3. Haga clic en Ver Todo para eliminar los filtros de búsqueda y mostrar todos los proveedores de claves de servicio.

    Tabla 17-1 Información de Proveedores de Claves de Servicio 
    Propiedad
    Descripción
    Nombre
    Un nombre único para el proveedor de claves de servicio. Haga clic en el nombre para ver la página Ver Detalles del Proveedor de Claves de Servicio. Consulte Edición de Proveedores de Claves de Servicio.
    Ruta de Acceso
    Nombre del proyecto y de la carpeta donde reside el proveedor de claves de servicio. Haga clic en el nombre para ver el proyecto o la carpeta que contiene este recurso. Consulte Cualificación de Nombres de Recurso con Proyectos y Carpetas.
