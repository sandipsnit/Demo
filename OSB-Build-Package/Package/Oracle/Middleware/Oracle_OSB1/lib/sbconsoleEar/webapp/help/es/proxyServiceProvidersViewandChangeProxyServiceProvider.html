Edición de Proveedores de Claves de Servicio

Utilice la página Ver Detalles del Proveedor de Claves de Servicio para ver y cambiar los detalles de un proveedor de claves de servicio concreto.

  1. Si aún no lo ha hecho, haga clic en Crear para crear una nueva sesión o en Editar para acceder a una sesión existente. Consulte Uso del Centro de Cambios.
  2. Localice el proveedor de claves de servicio tal y como se describe en Localización de Proveedores de Claves de Servicio.
  3. Haga clic en el nombre del proveedor de claves de servicio. La página Ver Detalles del Proveedor de Claves de Servicio muestra la información que se ve en la Tabla 17-3.

    4. Tabla 17-3 Detalles del Proveedor de Claves de Servicio 
    Propiedad
    Descripción
    Nombre de Proveedor de Claves de Servicio
    Nombre de este proveedor de claves de servicio.
    Modificado por
    Usuario que ha creado este proveedor de claves de servicio o lo ha importado a la configuración.
    Fecha de Modificación
    Fecha y hora a la que el usuario ha creado este proveedor de claves de servicio o lo ha importado a la configuración. Haga clic en el enlace de fecha y hora para ver el historial de cambios de este recurso. Consulte Página Ver Historial de Cambios.
    Referencias
    Número de objetos a los que hace referencia este proveedor de claves de servicio. Si existen estas referencias, haga clic en el enlace numérico para ver una lista de los objetos. Consulte Visualización de Referencias a Recursos.
    Con Referencia de
    Número de objetos que hacen referencia a este proveedor de claves de servicio. Si existen estas referencias, haga clic en el enlace numérico para ver una lista con los objetos. Por ejemplo, si ha seleccionado este proveedor de claves de servicio como proveedor de servicio para un servicio de proxy concreto, el servicio de proxy aparecerá como una referencia cuando se haga clic en el enlace. Consulte Visualización de Referencias a Recursos.
    Descripción
    Descripción de este proveedor de claves de servicio, si existe.

  4. Para hacer cambios en los campos, haga clic en Editar. Consulte Adición de Proveedores de Claves de Servicio para ver las descripciones de los campos.
  5. Haga clic en Guardar para confirmar las actualizaciones en la sesión actual.
  6. Para terminar la sesión y desplegar la configuración en tiempo de ejecución, haga clic en Activar en Centro de Cambios.
