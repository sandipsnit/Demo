Adición de Acciones de Asignación

Utilice esta acción para asignar el resultado de una expresión XQuery a una variable de contexto.

Antes de Empezar

En estas instrucciones se supone que ya está editando un flujo de mensajes en la página Editar Flujo de Mensajes, como se explica en Creación, Edición y Visualización de Flujos de Mensajes.

Para agregar una acción de asignación

  1. Navegue a donde desea agregar la acción, como se describe en Adición y Edición de Acciones en Flujos de Mensajes.
  2. Haga clic en el icono adecuado y, a continuación, seleccione Agregar una Acción > Procesamiento de Mensajes > Asignar.
  3. Haga clic en <Expresión>. Aparecerá la página Editor de Expresiones XQuery. La expresión XQuery se utiliza para crear los datos que se asignarán a la variable con nombre. Consulte Creación y Edición de Expresiones XQuery y XPath en Línea.
  4. Cuando termine de editar la expresión, introduzca una variable de contexto en el campo de variable. Para obtener más información sobre las variables de contexto, consulte la sección sobre variables entrantes y salientes en http://www.oracle.com/pls/as111170/lookup?id=OSBAG325 y sobre creación de mensajes para distribución en http://www.oracle.com/pls/as111170/lookup?id=OSBAG348 en la Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus (Guía del Administrador de Oracle Fusion Middleware para Oracle Service Bus).

Después de Terminar

Cuando termine de configurar esta acción, siga configurando otras acciones o guarde la configuración, como se describe en Adición y Edición de Acciones en Flujos de Mensajes.
