Adición de Acciones de Supresión

Utilice esta acción para suprimir una variable de contexto o un juego de nodos especificado por una expresión XPath. La acción de supresión forma parte de un juego de acciones de actualización.

Antes de Empezar

En estas instrucciones se supone que ya está editando un flujo de mensajes en la página Editar Flujo de Mensajes, como se explica en Creación, Edición y Visualización de Flujos de Mensajes.

Para agregar una acción de supresión

  1. Navegue a donde desea agregar la acción, como se describe en Adición y Edición de Acciones en Flujos de Mensajes.
  2. Para suprimir una variable de contexto, seleccione el botón de radio Variable e introduzca el nombre de una variable de contexto en el campo Variable. Para obtener más información sobre las variables de contexto consulte el Apéndice A, Contexto de Mensaje.

    3. Como alternativa, para suprimir todos los nodos seleccionados por una expresión XPath, seleccione el botón de radio XPath y haga clic en <XPath>. Aparece la página Editor de Expresiones XPath. Consulte Creación y Edición de Expresiones XQuery y XPath en Línea. Después de guardar la expresión, introduzca una variable de contexto en el campo Variable. Para obtener más información sobre las variables de contexto consulte el Apéndice A, Contexto de Mensaje.

Después de Terminar

Cuando termine de configurar esta acción, siga configurando otras acciones o guarde la configuración, como se describe en Adición y Edición de Acciones en Flujos de Mensajes.
