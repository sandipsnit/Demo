Adición de Acciones de Inserción

Utilice esta acción para insertar el resultado de una expresión XQuery en un lugar identificado relativo a los nodos seleccionados por una expresión XPath. La acción de inserción forma parte de un juego de acciones de actualización.

Antes de Empezar

En estas instrucciones se supone que ya está editando un flujo de mensajes en la página Editar Flujo de Mensajes, como se explica en Creación, Edición y Visualización de Flujos de Mensajes.

Para agregar una acción de inserción

  1. Navegue a donde desea agregar la acción, como se describe en Adición y Edición de Acciones en Flujos de Mensajes.
  2. Haga clic en el icono adecuado y, a continuación, seleccione Agregar una Acción > Procesamiento de Mensajes > Insertar.
  3. Haga clic en <Expresión> para editar una expresión XQuery. La expresión XQuery se utiliza para crear los datos que se insertarán en la ubicación especificada de la variable con nombre. Aparecerá la página Editor de Expresiones XQuery. Consulte Creación y Edición de Expresiones XQuery y XPath en Línea.
  4. Cuando termine de editar la expresión, seleccione la ubicación relativa de la lista desplegable. La ubicación relativa se utiliza para controlar dónde se realiza la inserción en relación con el resultado de la expresión XPath:
    • antes de: Como hermano antes de cada elemento o atributo seleccionado por la expresión XPath.
    • después de: Como hermano después de cada elemento o atributo seleccionado por la expresión XPath.
    • como primer secundario de: Como primer secundario de cada elemento identificado por la expresión XPath. Se produce un error si el resultado de XPath devuelve atributos.
    • como último secundario de: Como último secundario de cada elemento identificado por la expresión XPath. Se produce un error si XPath devuelve atributos.
  5. Haga clic en <XPath>. Aparecerá la página Editor de Expresiones XPath. Consulte Creación y Edición de Expresiones XQuery y XPath en Línea.

    6. Las configuraciones válidas son aquellas en las que:

    • Tanto la expresión XQuery como XPath devuelven elementos.
    • Tanto la expresión XQuery como XPath devuelven atributos, en cuyo caso, la expresión XQuery debe devolver atributos.
  6. Cuando termine de editar la expresión XPath, introduzca una variable de contexto en el campo de variable. XPath evalúa el contenido de esta variable. Para obtener más información sobre las variables de contexto consulte el Apéndice A, Contexto de Mensaje.

Después de Terminar

Cuando termine de configurar esta acción, siga configurando otras acciones o guarde la configuración, como se describe en Adición y Edición de Acciones en Flujos de Mensajes.
