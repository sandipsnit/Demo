Adición de Acciones de Transformación de MFL

Utilice la acción de transformación de MFL (Lenguaje de Formato de Mensajes) para convertir el contenido del mensaje de XML a no XML, o viceversa, en el pipeline de mensaje. Un MFL es un documento XML especializado utilizado para describir el diseño de los datos binarios. Es un lenguaje propiedad de Oracle que se utiliza para definir reglas que transforman los datos con formato binario en datos XML, o viceversa. Consulte MFL.

Antes de Empezar

En estas instrucciones se supone que ya está editando un flujo de mensajes en la página Editar Flujo de Mensajes, como se explica en Creación, Edición y Visualización de Flujos de Mensajes.

Para agregar una acción de transformación de MFL

  1. Navegue a donde desea agregar la acción, como se describe en Adición y Edición de Acciones en Flujos de Mensajes.
  2. Haga clic en el icono adecuado y, a continuación, seleccione Agregar una Acción > Procesamiento de Mensajes > Transformación de MFL.
  3. En la lista desplegable Aplicar Transformación de MFL, seleccione De XML a no XML o De no XML a XML, según sus necesidades.
  4. Haga clic en <Expresión>. Con el Editor de Expresiones XQuery, especifique la variable en la que se va a realizar la acción de transformación de MFL. Esta entrada debe ser de texto o binaria si se va a transformar en XML, y XML si se va a transformar en no XML. El contenido binario del contexto de mensaje se representa con un elemento XML de contenido binario. Este XML debe ser el resultado de la expresión Xquery cuando la entrada debe ser binaria. Consulte Creación y Edición de Expresiones XQuery y XPath en Línea.
  5. Seleccione una de las opciones siguientes:
    • Recurso MFL: Haga clic en el enlace <recurso>. Aparece la página Seleccionar MFL. Seleccione el recurso MFL estático que realizará la acción de transformación de MFL.
    • Recurso MFL de: Haga clic en el enlace Expresión. Aparece la página Editor de Expresiones XQuery. Con el editor, cree o edite una expresión XQuery para especificar de forma dinámica un recurso MFL que realizará la acción de transformación, con el formato proyecto/carpeta/nombre_recurso_MFL. Consulte Creación y Edición de Expresiones XQuery y XPath en Línea.
  6. En el campo Asignar a Variable, introduzca el nombre de la variable a la que se va a asignar el resultado de esta acción de transformación. El resultado será un elemento XML de contenido binario.

Después de Terminar

Cuando termine de configurar esta acción, siga configurando otras acciones o guarde la configuración, como se describe en Adición y Edición de Acciones en Flujos de Mensajes.
