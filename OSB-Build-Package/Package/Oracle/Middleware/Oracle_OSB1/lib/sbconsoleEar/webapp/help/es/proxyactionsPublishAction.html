Adición de Acciones de Publicación

Utilice esta acción para identificar un servicio de destino especificado de forma estática para un mensaje y configurar cómo se empaqueta y envía el mensaje a dicho servicio.

Para obtener más información sobre el comportamiento de publicación, consulte la sección sobre la realización de transformaciones en los flujos de mensajes en Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus (Guía del Administrador de Oracle Fusion Middleware para Oracle Service Bus), en http://www.oracle.com/pls/as111170/lookup?id=OSBAG205.

  1. Navegue a donde desea agregar la acción, como se describe en Adición y Edición de Acciones en Flujos de Mensajes.
  2. Seleccione Agregar una Acción > Comunicación > Publicar.
  3. Haga clic en <Servicio>. Aparecerá la página Seleccionar Servicio.
  4. Seleccione un servicio de la lista y, a continuación, haga clic en Enviar. Éste es el servicio de destino para el mensaje.
  5. Si el servicio tiene operaciones definidas, puede especificar la operación que se va a llamar seleccionándola en la lista Operación.
  6. Para que la operación saliente sea la misma que la entrante, active la casilla de control Utilice la operación entrante para la saliente.
  7. Para configurar cómo se empaqueta el mensaje y se envía al servicio, en el campo Acciones de Solicitud, haga clic en Agregar una Acción. A continuación, seleccione una acción para asociarla al servicio. Puede agregar más de una acción. Consulte Adición y Edición de Acciones en Flujos de Mensajes.

Después de Terminar

Cuando termine de configurar esta acción, siga configurando otras acciones o guarde la configuración, como se describe en Adición y Edición de Acciones en Flujos de Mensajes.
