Adición de Acciones de Cambio de Nombre

Utilice la acción de cambio de nombre para cambiar el nombre de elementos seleccionados por una expresión XPath sin modificar el contenido del elemento. La acción de cambio de nombre es una de un juego de acciones de actualización.

Antes de Empezar

En estas instrucciones se supone que ya está editando un flujo de mensajes en la página Editar Flujo de Mensajes, como se explica en Creación, Edición y Visualización de Flujos de Mensajes.

Para agregar una acción de cambio de nombre

  1. Navegue a donde desea agregar la acción, como se describe en Adición y Edición de Acciones en Flujos de Mensajes.
  2. Haga clic en el icono adecuado y, a continuación, seleccione Agregar una Acción > Procesamiento de Mensajes > Cambiar Nombre.
  3. Haga clic en <XPath>. Aparecerá la página Editor de Expresiones XPath. Cree una expresión XPath para especificar los datos (en la variable con nombre) a los que se cambiara el nombre. Consulte Creación y Edición de Expresiones XQuery y XPath en Línea.
  4. En el campo Variable, introduzca la variable de contexto que contiene el elemento al que desea cambiar el nombre. Para obtener más información sobre las variables de contexto, consulte el Apéndice A, Contexto de Mensaje.
  5. Realice una de las siguientes acciones:
    • Para cambiar el nombre de elementos seleccionados con un nombre local, seleccione el primer botón de radio de nombre local y, a continuación, introduzca un nombre local en el campo de nombre local.
    • Para cambiar el nombre de elementos seleccionados con un espacio de nombres, seleccione el primer botón de radio de espacio de nombres y, a continuación, introduzca un espacio de nombres en el campo de espacio de nombres.
    • Para cambiar el nombre de elementos seleccionados con un nombre local y espacio de nombres, seleccione el botón de radio de nombre local y espacio de nombres y, a continuación, introduzca un nombre local y espacio de nombres en el campo de nombre local y espacio de nombres.

Después de Terminar

Cuando termine de configurar esta acción, siga configurando otras acciones o guarde la configuración, como se describe en Adición y Edición de Acciones en Flujos de Mensajes.
