Adición de Acciones de Sustitución

Utilice una acción de sustitución para sustituir un nodo o el contenido de un nodo especificado por una expresión XPath. El nodo o su contenido se sustituye por el valor devuelto por una expresión XQuery.

Una acción se sustitución se puede utilizar para sustituir valores simples, elementos e incluso atributos. Una expresión XQuery que no devuelve nada equivale a suprimir los nodos identificados o a convertirlos en vacíos, según si la acción es sustituir los nodos completos o sólo su contenido.

La acción se sustitución es una de un juego de acciones de actualización.

Antes de Empezar

En estas instrucciones se supone que ya está editando un flujo de mensajes en la página Editar Flujo de Mensajes, como se explica en Creación, Edición y Visualización de Flujos de Mensajes.

Para agregar una acción de sustitución

  1. Navegue a donde desea agregar la acción, como se describe en Adición y Edición de Acciones en Flujos de Mensajes.
  2. Haga clic en el icono adecuado y, a continuación, seleccione Agregar una Acción > Procesamiento de Mensajes > Sustituir.
  3. Haga clic en <XPath>. Aparecerá la página Editor de Expresiones XPath. La expresión XPath se utiliza para especificar los datos (en la variable con nombre) que se sustituirán. Consulte Creación y Edición de Expresiones XQuery y XPath en Línea.
  4. Cuando acabe de editar la expresión XPath, introduzca una variable de contexto en el campo de variable. Consulte el Apéndice A, Contexto de Mensaje.
  5. Haga clic en <Expresión>. Aparecerá la página Editor de Expresiones XQuery. La expresión XQuery se utiliza para crear los datos que sustituyen a los datos expresados por el XPath en la variable con nombre. Consulte Creación y Edición de Expresiones XQuery y XPath en Línea.
  6. Cuando acabe de editar la expresión XQuery, seleccione una de las opciones:
    • Sustituir nodo completo: Para especificar que los nodos seleccionados por la expresión XPath definida se sustituyan junto con todo su contenido.
    • Sustituir contenido del nodo: Para especificar que el nodo no se sustituya; sólo se sustituye el contenido.
      • Nota: Seleccionar la opción Sustituir contenido del nodo y dejar el campo XPath en blanco es más eficaz que seleccionar la opción Sustituir nodo completo y definir XPath en ./*.

Después de Terminar

Cuando termine de configurar esta acción, siga configurando otras acciones o guarde la configuración, como se describe en Adición y Edición de Acciones en Flujos de Mensajes.
