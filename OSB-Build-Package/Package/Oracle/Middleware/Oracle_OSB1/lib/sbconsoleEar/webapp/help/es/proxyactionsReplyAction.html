Adición de Acciones de Respuesta

Utilice la acción de respuesta para especificar el envío de una respuesta inmediata al invocador.

La acción de respuesta se puede utilizar en la solicitud, respuesta o pipeline de errores. Puede configurarla como resultado a una respuesta con una acción correcta o un fallo. En caso de respuesta con fallo en la que el transporte entrante es HTTP, la acción de respuesta especifica que se envía una respuesta inmediata al invocador.

Antes de Empezar

En estas instrucciones se supone que ya está editando un flujo de mensajes en la página Editar Flujo de Mensajes, como se explica en Creación, Edición y Visualización de Flujos de Mensajes.

Para agregar una acción de respuesta

  1. Navegue a donde desea agregar la acción, como se describe en Adición y Edición de Acciones en Flujos de Mensajes.
  2. Haga clic en el icono adecuado y, a continuación, seleccione Agregar una Acción > Control de Flujo > Responder.
  3. Seleccione Correctamente para responder que el mensaje ha sido correcto, o bien seleccione Con Fallo para responder que el mensaje tiene un fallo.

    4. Responder Con Fallo hará que se aborte una transacción, si se ha iniciado con Oracle Service Bus.

Después de Terminar

Cuando termine de configurar esta acción, siga configurando otras acciones o guarde la configuración, como se describe en Adición y Edición de Acciones en Flujos de Mensajes.
