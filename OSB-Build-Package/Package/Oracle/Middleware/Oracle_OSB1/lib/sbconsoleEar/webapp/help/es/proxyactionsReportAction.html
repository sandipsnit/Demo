Adición de Acciones de Informe

Utilice la acción de informe para activar los informes de mensajes para un servicio de proxy.

Antes de Empezar

En estas instrucciones se supone que ya está editando un flujo de mensajes en la página Editar Flujo de Mensajes, como se explica en Creación, Edición y Visualización de Flujos de Mensajes.

Para agregar una acción de informe

  1. Navegue a donde desea agregar la acción, como se describe en Adición y Edición de Acciones en Flujos de Mensajes.
  2. Haga clic en el icono adecuado y, a continuación, seleccione Agregar una Acción > Control de Flujo > Informe.
  3. Haga clic en <Expresión>. Aparecerá la página Editor de Expresiones XQuery. Consulte Creación y Edición de Expresiones XQuery y XPath en Línea. La expresión XQuery se utiliza para crear los datos de los que se informará al panel de control de Oracle Service Bus.
  4. Cuando acabe de editar la expresión XQuery, haga clic en Agregar Clave. Se mostrarán dos campos: Un campo Nombre de la Clave y un campo Valor de Clave, que incluye un enlace de XPath en el que puede hacer clic para editar una expresión XPath y un campo de variable en el que puede introducir una variable de contexto.

    5. Utilice los pares de valores de clave para extraer identificadores de clave de cualquier variable de contexto de mensaje o carga útil de mensaje, e ignorar el resto del mensaje. Las claves son una forma adecuada de identificar un mensaje. Se muestran como índices de informes en el módulo Informes. Consulte Visualización de Mensajes de Informes y Visualización de Detalles de Mensaje.

    1. Introduzca un nombre de clave en el campo Nombre de la Clave.
    2. Haga clic en <XPath>. Aparecerá la página Editar Expresión XPath. Consulte Creación y Edición de Expresiones XQuery y XPath en Línea.
    3. Introduzca una variable de contexto en el campo de variable. Para obtener más información sobre las variables de contexto consulte el Apéndice A, Contexto de Mensaje.
    4. Para agregar más valores de clave, haga clic en el icono Clave y, a continuación, seleccione Agregar Clave. Para suprimir una clave, haga clic en el icono Clave y, a continuación, seleccione Suprimir esta Clave.

Por ejemplo, considere una acción de informe configurada en un manejador de errores de una etapa. La acción informa del contenido de la variable de contexto fault en caso de error. La acción de informe está configurada de la siguiente forma:

  • Nombre de la clave = errorCode
  • Valor de la clave = ./ctx:errorCode en la variable fault

Cada vez que se ejecuta esta acción en tiempo de ejecución, se muestra un mensaje mediante el flujo de datos de informe. En la siguiente tabla se muestran los resultados después de ejecutar dos veces la acción de informe.

Después de Terminar

Cuando termine de configurar esta acción, siga configurando otras acciones o guarde la configuración, como se describe en Adición y Edición de Acciones en Flujos de Mensajes.
