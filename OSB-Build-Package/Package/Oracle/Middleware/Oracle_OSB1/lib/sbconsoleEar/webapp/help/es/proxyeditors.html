Servicios de Proxy: Editores de XQuery y XPath

En el flujo de mensajes de un servicio de proxy, puede asignar expresiones XQuery a variables de contexto de mensaje, asigne acciones If... Then... basadas en el resultado booleano de una expresión XQuery, inserte el resultado de una expresión XQuery en un lugar identificado con respecto a una expresión XPath, especifique el contexto de mensaje que desea registrar mediante expresiones XQuery en variables de contexto, etc.

El Editor de Expresiones XQuery, el Editor de Condición de XQuery y el Editor de Expresiones XPath están disponibles en el contexto adecuado de los flujos de mensajes para construir el tipo de expresión llamada en el contexto.
