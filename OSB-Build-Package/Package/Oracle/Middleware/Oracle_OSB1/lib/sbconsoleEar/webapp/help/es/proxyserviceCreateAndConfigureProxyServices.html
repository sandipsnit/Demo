Creación y Configuración de Servicios de Proxy

En esta sección se describe la forma de crear y configurar servicios de proxy.

Para los servicios de JCA, puede generar servicios de proxy a partir de enlaces de JCA, como se describe en Generación de un Servicio de Proxy a partir de un Recurso de Enlace de JCA.

  1. Si aún no lo ha hecho, haga clic en Crear para crear una nueva sesión o en Editar para acceder a una sesión existente. Consulte Uso del Centro de Cambios.
  2. Seleccione Explorador de Proyectos y elija el proyecto o la carpeta donde desea agregar el servicio de proxy.
  3. En la página Vista de Proyectos/Carpetas, seleccione Servicio de Proxy en la lista desplegable Crear Recurso.
  4. En la página Configuración General, proporcione un nombre para el servicio y seleccione el tipo de servicio que desea crear.

    Siga las Restricciones de Nomenclatura de Recursos para obtener ayuda con la nomenclatura.
  5. Haga clic en Siguiente. Las páginas que aparecen a continuación dependen de las opciones seleccionadas en la primera página. Introduzca la información adecuada en cada página hasta que llegue a la página de resumen, a continuación, haga clic en Guardar para guardar el servicio en la sesión actual.

    6. Para obtener instrucciones detalladas para rellenar cada página, consulte:

    • Página Configuración General
    • Página Configuración de Tipo de Mensaje
    • Página Configuración de Transporte
    • Páginas de Configuración de Transporte Específica del Protocolo
    • Página Configuración de Selección de Operaciones
    • Página Manejo de Mensajes
    • Página Resumen
    • Página Política de Servicio de Proxy
    • Página Seguridad de Servicio de Proxy
  6. Para terminar la sesión y desplegar la configuración en tiempo de ejecución, haga clic en Activar en Centro de Cambios.
  7. Después de crear un servicio de proxy, el siguiente paso es configurar su flujo de mensajes. El flujo de mensajes define la implantación de un servicio de proxy. Para obtener más información, consulte Creación, Edición y Visualización de Flujos de Mensajes.

Generación de un Servicio de Proxy a partir de un Recurso de Enlace de JCA

Puede generar un servicio de proxy de JCA a partir de un recurso de enlace de JCA entrante en Oracle Service Bus. Para obtener más información sobre los recursos de enlace de JCA, consulte Chapter 12, JCA Bindings (Capítulo 12, Enlaces de JCA).

Para generar un servicio de proxy a partir de un enlace de JCA:

  1. En la consola de administración de Oracle Service Bus, haga clic en Crear o Editar en el Centro de Cambios si el área no se encuentra en el modo de creación o edición.
  2. En el Explorador de Recursos, haga clic en Enlaces de JCA.
  3. Localice el enlace de JCA desde el que desea generar un servicio y haga clic en el icono Acción.
  4. En la ventana que aparece, confirme el nombre de WSDL y el servicio que desea generar, seleccione la ubicación de estos nuevos recursos y haga clic en Generar.

    5. Oracle Service Bus genera el servicio y su correspondiente WSDL.

  5. Realice las modificaciones de configuración que necesite en el servicio generado, como el URI de punto final.
