Creación, Edición y Visualización de Flujos de Mensajes

  1. Si aún no lo ha hecho, haga clic en Crear para crear una nueva sesión o en Editar para acceder a una sesión existente. Consulte Uso del Centro de Cambios.
  2. Localice el servicio de proxy cuyo flujo de mensajes desea ver o editar. Realice una de las acciones siguientes:
    • Seleccione Explorador de Proyectos para que aparezca la página Vista de Proyectos o la página Vista de Proyectos/Carpetas. A continuación, navegue por los proyectos o carpetas para buscar el servicio de proxy.
    • Seleccione Explorador de Recursos > Servicios de Proxy para mostrar la página Resumen de Servicios de Proxy.
  3. Haga clic en el icono Editar Flujo de Mensajes en la fila del servicio de proxy cuyo flujo de mensajes desea ver o editar.
    • Si aún no se ha creado ningún flujo de mensajes para el servicio de proxy seleccionado, aparecerá la página Editar Flujo de Mensajes con un único icono en la página, el icono Servicio de Proxy. Éste es el nodo de inicio del servicio de proxy. Haga clic en este icono para empezar a construir el flujo de mensajes.
    • Si el servicio de proxy ya tiene un flujo de mensajes, la página contiene una representación gráfica del flujo. Haga clic en los iconos para ver o editar las partes del flujo de mensajes.

      • Consulte Página Editar Flujo de Mensajes para obtener más información sobre lo que hacer para construir un flujo de mensajes en dicha página.

  4. Haga clic en Guardar para confirmar las actualizaciones en la sesión actual.
  5. Para terminar la sesión y desplegar la configuración en tiempo de ejecución, haga clic en Activar en Centro de Cambios.
