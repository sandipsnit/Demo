Página Configuración de Transporte de SB

La página Crear/Editar un Servicio de Proxy: Configuración de Transporte de SB es una de una serie de páginas para la creación y edición de servicios de proxy, como se describe en Creación y Configuración de Servicios de Proxy y Edición de Configuraciones de Servicio de Proxy.

Para obtener más información, consulte la sección sobre transporte de SB en Oracle Fusion Middleware Developer's Guide for Oracle Service Bus (Guía del Desarrollador de Oracle Fusion Middleware para Oracle Service Bus) en http://www.oracle.com/pls/as111170/lookup?id=OSBDV997.

Utilice esta página para configurar el transporte de un servicio de proxy mediante el protocolo de transporte de SB (Service Bus). En la Tabla 19-12 se describe el uso de esta página.

Tabla 19-12 Página Crear/Editar un Servicio de Proxy: Configuración de Transporte de SB

Política de Distribución
Seleccione una política de distribución para este punto final o utilice la política de distribución por defecto.
La política de distribución hace referencia a la instancia del gestor de trabajos de WLS que desea utilizar para el punto final de servicio para procesar la solicitud. Para obtener más información sobre los gestores de trabajos, consulte la sección sobre el uso de gestores de trabajos para optimizar el trabajo programado en Oracle Fusion Middleware Configuring Server Environments for Oracle WebLogic Server (Configuración de Entornos de Servidor de Oracle Fusion Middleware para Oracle WebLogic Server) en http://www.oracle.com/pls/as111170/lookup?id=CNFGD112.
Usar SSL
Si se especifica, las solicitudes se deben enviar a través de una conexión SSL.
Sin embargo, las conexiones no seguras no están prohibidas. El administrador debe cerrar todos los protocolos no seguros en el servidor (por ejemplo, t3 o http) para forzar estrictamente conexiones de cliente seguras.
