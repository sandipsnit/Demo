Página Resumen

La página Crear/Editar un Servicio de Proxy: Resumen es una de una serie de páginas para la creación y edición de servicios de proxy, como se describe en Creación y Configuración de Servicios de Proxy y Edición de Configuraciones de Servicio de Proxy.

Utilice esta página para ver o modificar la configuración de un servicio de proxy antes de guardarla.

Para ver o modificar la configuración, haga clic en Editar en la fila de la categoría adecuada (por ejemplo, Configuración General, Configuración de Transporte, etc.). Las páginas que puede editar dependen de las páginas configuradas al crear el servicio de proxy. En la siguiente lista se muestran todas las páginas:

  • Página Configuración General
  • Página Configuración de Tipo de Mensaje
  • Página Configuración de Transporte
  • Páginas de Configuración de Transporte Específica del Protocolo
  • Página Configuración de Transporte de Correo Electrónico
  • Página Configuración de Transporte de Archivo
  • Página Configuración de Transporte de FTP
  • Página Configuración de Transporte de HTTP
  • Página Configuración de Transporte de JCA
  • Página Configuración de Transporte de JEJB
  • Página Configuración de Transporte de JMS
  • Página Configuración de Transporte de MQ
  • Página Configuración de Transporte de SB
  • Página Configuración de Transporte de SFTP
  • Página Configuración de Transporte de Tuxedo
  • Página Configuración de Transporte de WS
  • Página Configuración de Selección de Operaciones
  • Página Manejo de Mensajes
