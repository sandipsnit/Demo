Páginas de Configuración de Transporte Específica del Protocolo

La página Crear/Editar un Servicio de Proxy: Configuración de Tipo de Transporte es una de una serie de páginas para la creación y edición de servicios de proxy, como se describe en Creación y Configuración de Servicios de Proxy y Edición de Configuraciones de Servicio de Proxy.

Esta página difiere según qué tipo de transporte seleccionado para configurar. Puede ser:

  • Página Configuración de Transporte de Correo Electrónico
  • Página Configuración de Transporte de Archivo
  • Página Configuración de Transporte de FTP
  • Página Configuración de Transporte de HTTP
  • Página Configuración de Transporte de JCA
  • Página Configuración de Transporte de JEJB
  • Página Configuración de Transporte de JMS
  • Página Configuración de Transporte de MQ
  • Página Configuración de Transporte de SB
  • Página Configuración de Transporte de SFTP
  • Página Configuración de Transporte de Tuxedo
  • Página Configuración de Transporte de WS
