Creación de Estructuras de Variables en los Editores de XQuery.

La paleta Estructuras de Variables de los editores de XQuery y XPath muestran representaciones gráficas del contenido de las variables. Incluye por defecto las variables de contexto de mensaje incorporadas attachments, body, header, outbound e inbound.

Cada entrada de asignación de la estructura de variable tiene una etiqueta y asigna una variable o ruta de acceso de la variable a una o más estructuras. El ámbito de estas asignaciones es una etapa o nodo de direccionamiento.

También puede declarar sus propias estructuras de variables, según lo siguiente:

  • Tipos XML, que incluyen:
    • Elementos del esquema
    • Elementos WSDL
    • Tipos de esquemas
    • Tipos WSDL
    • MFL
  • Interfaces de servicio
  • Tipos simples (cadena o cualquier XML)

Puede utilizar esta función directamente para todas las variables definidas por el usuario, así como $inbound, $outbound y $fault. Sin embargo, no puede utilizarla directamente para acceder a anexos XML de $attachments, cabeceras de $header o documentos y parámetros de RPC de $body, con una excepción; puede utilizarla directamente para acceder a documentos y parámetros de $body para mensajes de solicitud recibidos por un servicio de proxy WSDL.

Antes de Empezar

En estas instrucciones se supone que está creando o editando una expresión en el Editor de Expresiones XQuery, Editor de Condición de XQuery o Editor de Expresiones XPath, como se describe en Creación y Edición de Expresiones XQuery y XPath en Línea.

Para crear una estructura de variables:

  1. SeleccioneEstructuras de Variables.
  2. En la paleta Estructuras de Variables, haga clic en Agregar Nueva Estructura.
  3. Siga con cualquiera de las tareas indicadas en la Tabla 22-3 a continuación.

    Crear un estructura de variables que asigne una variable a un tipo de esquema XML
    1. Seleccione Tipo XML en la parte superior de la paleta Estructuras de Variables, si aún no se ha seleccionado.
    2. En el campo Etiqueta de la Estructura, introduzca un nombre mostrado para la variable que desea crear. Este nombre mostrado permite dar un nombre significativo a la estructura para que pueda reconocerla en el momento del diseño, pero no tiene ningún impacto en tiempo de ejecución.

    Crear un estructura de variables que asigne una variable a un elemento secundario
    1. En la parte superior de la paleta Estructuras de Variables, seleccione Tipo XML, si aún no se ha seleccionado.
    2. En el campo Etiqueta de la Estructura, introduzca un nombre mostrado para la variable que desea crear. Este nombre mostrado permite dar un nombre significativo a la estructura para que pueda reconocerla en el momento del diseño, pero no tiene ningún impacto en tiempo de ejecución.

    4. En el campo Tipo, seleccione el tipo de variable que desea crear:

      5. Para crear una variable de elemento de esquema XML o de elemento WSDL, seleccione el botón de radio asociado a esta opción y, a continuación Elemento de Esquema o Elemento WSDL.

      Para crear una variable MFL, seleccione el botón de radio asociado a esta opción y, a continuación MFL.

    5. Para el esquema XML, WSDL o MFL, haga clic en Examinar para seleccionar un objeto de la lista que muestra el explorador y, a continuación, haga clic en Enviar. Por ejemplo, seleccione un MFL de la lista correspondiente y, a continuación, haga clic en Enviar.
    6. Active la casilla de control Definir como Secundario para definir el elemento como secundario de la estructura que se está creando.
    7. Haga clic en Agregar para crear la estructura de variables.
    Crear una estructura de variables que utilice un recurso MFL
    1. En la parte superior de la paleta Estructuras de Variables, seleccione Tipo XML, si aún no se ha seleccionado.
    2. En el campo Etiqueta de la Estructura, introduzca un nombre mostrado para la variable que desea crear. Este nombre mostrado permite dar un nombre significativo a la estructura para que pueda reconocerla en el momento del diseño, pero no tiene ningún impacto en tiempo de ejecución.
    3. En el campo Ruta de Acceso de la estructura, introduzca la ruta de acceso de la estructura de variables en tiempo de ejecución.
    4. En el campo Tipo, seleccione el botón de radio adecuado y, a continuación, haga clic en Examinar. Aparecerá Explorador de MFL.
    5. Seleccione un MFL de la lista correspondiente y, a continuación, haga clic en Enviar.
    6. Haga clic en Agregar para crear la estructura de variables.
    Crear una estructura de variables de interfaz de servicio
    1. En la parte superior de la paleta Estructuras de Variables, seleccione Interfaz de Servicio.
    2. En el campo Etiqueta de la Estructura, introduzca un nombre mostrado para la variable que desea crear. Este nombre mostrado permite dar un nombre significativo a la estructura para que pueda reconocerla en el momento del diseño, pero no tiene ningún impacto en tiempo de ejecución.
    3. En el campo Ruta de Acceso de la Estructura, el valor por defecto ya está definido como $body. No puede cambiar este campo.
    4. En el campo Servicio Basado en WSDL, seleccione el icono Explorador de Servicio, seleccione un servicio de la lista que muestra el Explorador de Servicio y, a continuación, haga clic en Enviar.

      5. El servicio seleccionado se muestra en el campo Servicio Basado en WSDL.

    5. En el campo Operación, seleccione una operación o Ninguna para no incluir ninguna operación.
    6. Haga clic en Agregar para crear la variable.
    Crear una estructura de variables simple
    1. En la parte superior de la paleta Estructuras de Variables, seleccione Tipo Simple.
    2. En el campo Etiqueta de la Estructura, introduzca un nombre mostrado para la variable que desea crear. Este nombre mostrado permite dar un nombre significativo a la estructura para que pueda reconocerla en el momento del diseño, pero no tiene ningún impacto en tiempo de ejecución.
    3. En el campo Nombre de la Estructura, introduzca un nombre para la estructura de variables que desea crear.
    4. En el campo Tipo, seleccione Cadena o Cualquier XML.
    5. Haga clic en Agregar para crear la variable.
