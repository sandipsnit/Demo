Supresión de Servicios de Proxy

La supresión de un servicio de proxy suprime todas las ACL a las que hace referencia el proxy desde el repositorio controlado por Oracle Service Bus, así como desde el proveedor de autorización adecuado.

  1. Si aún no lo ha hecho, haga clic en Crear para crear una nueva sesión o en Editar para acceder a una sesión existente. Consulte Uso del Centro de Cambios.
  2. Seleccione Explorador de Recursos > Servicios de Proxy para mostrar la página Resumen de Servicios de Proxy.
  3. Haga clic en el icono Suprimir de la fila del servicio de proxy que desea suprimir. El servicio de proxy se suprime en la sesión actual. Aparece un icono Advertencia de Supresión si hay otros recursos que hacen referencia a éste. Puede suprimirlo confirmando la advertencia. Esto puede causar conflictos debido a referencias no resueltas al recurso suprimido.
  4. Para terminar la sesión y desplegar la configuración en tiempo de ejecución, haga clic en Activar en Centro de Cambios.
