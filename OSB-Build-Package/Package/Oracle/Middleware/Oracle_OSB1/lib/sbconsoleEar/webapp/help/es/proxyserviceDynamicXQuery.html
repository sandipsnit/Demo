Enlace de Expresiones XQuery Dinámicas a XQueries en Líneas

La página Editor de Expresiones XQuery permite especificar una expresión XQuery dinámica que evalúa en tiempo de ejecución el nombre de un recurso de XQuery registrado anteriormente. Para obtener más información sobre este editor, consulte Creación y Edición de Expresiones XQuery y XPath en Línea.

Para definir una expresión XQuery dinámica

  1. Seleccione la opción XQuery Dinámica.
  2. En el área Introducir Expresión para Recurso de XQuery, introduzca la expresión XQuery que evaluará en tiempo de ejecución el nombre de un recurso de XQuery registrado anteriormente.

    3. A continuación, se muestra la sintaxis del recurso de XQuery (que representa el nombre completo del recurso):

    Project/folder1/folder2/XQueryResourceName

  3. En el área Variables de Enlace mediante Plantilla de XQuery o Variables Personalizadas, defina los parámetros de entrada de la transformación.
    • Haga clic en Examinar en el campo Seleccionar Plantilla de XQuery para seleccionar un recurso registrado existente que sirva de plantilla para la forma de la consulta (el número y nombre de las variables). Después de seleccionar una plantilla, las variables aparecen en el área Variables de Enlace. Tenga en cuenta que la plantilla no se conserva con la configuración. En su lugar, la plantilla sirve como un inicio rápido para ayudarle a especificar las variables de la consulta.
    • Escriba un nombre de variable en el campo Agregar Variable Personalizada y haga clic en Agregar. Para cada variable mostrada en Nombre de la Variable, introduzca una expresión XQuery para asignársela. Debe definir una asignación para cada parámetro. Por ejemplo, si una transformación de XQuery tiene dos parámetros de entrada denominados uno y dos, el campo Nombre de la Variable tiene dos etiquetas: uno y dos. Se asocia a cada etiqueta un cuadro de texto, en el que se introduce la expresión XQuery.

      • Las siguientes expresiones XQuery son ejemplos de entrada válida para los campos de variables:

      $body/*[1]

      $body/po:PurchaseOrder

  4. Después de terminar, continúe con cualquiera de las tareas siguientes.
    • Haga clic en Validar. Aparecerá un mensaje si la expresión se valida correctamente.
    • Haga clic en Probar. Consulte Realización de Pruebas de XQuery.
    • Guarde o deseche los cambios.
