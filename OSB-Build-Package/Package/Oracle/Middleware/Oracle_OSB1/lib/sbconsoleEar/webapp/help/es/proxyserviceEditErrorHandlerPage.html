Página Editar Manejador de Errores

Puede configurar el manejo de errores en el flujo de mensajes, pipeline, nodo de direccionamiento y nivel de etapa.

Configure manejadores de errores en la página Editar Manejador de Errores. Siempre debe agregar al menos una etapa a la página para especificar cómo funcionará el manejador de errores. Consulte los siguientes temas:

  • Adición de Manejadores de Errores de Servicio de Proxy
  • Adición de Manejadores de Errores de Pipeline
  • Adición de Manejadores de Errores de Etapa
  • Adición de Manejadores de Errores de Nodo de Direccionamiento
  • Edición de Manejadores de Errores
  • Sección sobre manejo de errores en flujos de mensajes en Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus (Guía del Administrador de Oracle Fusion Middleware para Oracle Service Bus) en http://www.oracle.com/pls/as111170/lookup?id=OSBAG232.
