Página Editar Flujo de Mensajes

Utilice la página Editar Flujo de Mensajes para construir un flujo de mensajes para un servicio de proxy.

El panel de navegación izquierdo de la página Editar Flujo de Mensajes muestra una vista de árbol de los nodos y objetos del flujo de mensajes. Cuando se definen los detalles de un objeto en una página independiente, puede hacer clic en el nombre del objeto para visualizar la página asociada.

El panel derecho proporciona un campo en el que construir el flujo de mensajes. Cuando aún no se ha definido un flujo de mensajes, el panel incluye un único icono Servicio de Proxy que indica el nodo de inicio del servicio. Haga clic en el icono para agregar nodos de pares del pipeline, nodos de direccionamiento, ramas condicionales, ramas operativas y manejo de errores para el servicio.

Al agregar objetos a la página, se muestran iconos en la página para representar los objetos. Las relaciones entre los objetos se muestran con líneas y cuadros delimitadores. Haga clic en un icono de la página Editar Flujo de Mensajes para visualizar un menú de las acciones que puede realizar en dicho objeto. Las opciones disponibles en el menú puede diferir según el contexto. Consulte la Tabla 20-1 para obtener una lista completa de iconos y opciones.
