Página Editar Configuración de Etapa

Utilice la página Editar Configuración de Etapa para agregar acciones a etapas de pipeline, etapas de manejador de errores y nodos de direccionamiento en un flujo de mensajes.

  • Cuando aún no se ha definido nada en la página Editar Configuración de Etapa, el único objeto mostrado es el icono Agregar un Acción. Haga clic en ese icono para empezar.
  • Cuando ya se ha configurado una etapa o un nodo de direccionamiento, las acciones y objetos definidos para dicha etapa o nodo de direccionamiento aparecen en la página. Edite las acciones existentes, como corresponda, o haga clic en cualquiera de los iconos que representan acciones para agregar más acciones a la etapa.

Consulte Adición y Edición de Acciones en Flujos de Mensajes para obtener instrucciones sobre el trabajo con todos los tipos de acciones que puede agregar a una etapa.
