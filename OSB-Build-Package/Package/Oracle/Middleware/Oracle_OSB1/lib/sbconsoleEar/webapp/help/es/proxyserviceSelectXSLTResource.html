Enlace de Recursos de XSLT Externos a XQueries en Línea

La página Editor de Expresiones XQuery permite seleccionar un recurso de XSLT para su ejecución. Para obtener más información sobre este editor, consulte Creación y Edición de Expresiones XQuery y XPath en Línea.

Para Seleccionar un Recurso de XSLT para su Ejecución

  1. Seleccione la opción Recursos de XSLT.
  2. En el campo Seleccionar el Recurso de XSLT para su Ejecución, seleccione el icono Explorador de XSLT.
  3. En el Explorador de XSLT, seleccione el botón de radio asociado al XSLT que desea ejecutar y, a continuación, haga clic en Ejecutar.
  4. En el campo Variables Ligadas, se muestran una etiqueta y un cuadro de texto correspondiente para cada parámetro de entrada de la transformación. Cada etiqueta corresponde al nombre de un parámetro y cada cuadro de texto es para definir una expresión XQuery que asignar al parámetro. Debe definir una asignación para cada parámetro. Por ejemplo, si una transformación XSL tiene dos parámetros de entrada denominados uno y dos, el campo Asignación de Variable tiene dos etiquetas (uno y dos) con un cuadro de texto asociado a cada una en el que se introduce la expresión XQuery. Además de la asignación de cualquier variable de entrada, debe especificar una expresión XQuery para el documento de entrada de la transformación. La asignación se especifica en el cuadro de texto con la etiqueta Documento de Entrada.

    5. Las siguientes expresiones XQuery son ejemplos de entrada válida para este campo:

    $body/*[1]

    $body/po:PurchaseOrder

    Nota: El siguiente nombre de la variable no es una entrada válida para este campo y su resultado es una excepción:
    body
  5. Continúe con cualquiera de las tareas siguientes:
    • Creación de Espacios de Nombres que Utilizar en Expresiones en Línea.
    • Creación de Estructuras de Variables en los Editores de XQuery.
    • Creación de Expresiones en los Campos de Texto del Espacio de Trabajo del Editor.
    • Enlace de Recursos de XQuery Externos a XQueries en Línea.
    • Haga clic en Validar. Aparecerá un mensaje si la expresión se valida correctamente.
    • Haga clic en Probar. Consulte Realización de Pruebas de XQuery.
