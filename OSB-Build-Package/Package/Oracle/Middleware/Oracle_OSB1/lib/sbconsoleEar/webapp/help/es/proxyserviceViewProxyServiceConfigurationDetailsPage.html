Página Ver Servicio de Proxy: Detalles de Configuración

La página Ver Servicio de Proxy: Detalles de Configuración muestra los detalles de la configuración de un servicio de proxy. En la Tabla 19-19 se describen todas las propiedades que pueden aparecer en esta página. (Las propiedades varían según los detalles del servicio de proxy.)

Las categorías mostradas en esta página corresponden a las páginas Crear/Editar un Servicio de Proxy utilizadas para la creación y edición de configuraciones de servicio de proxy, como se describe en Crear/Editar un Servicio de Proxy: Referencia de Página.

Haga clic en el enlace Editar junto al nombre de alguna categoría para mostrar la página de configuración asociada.
