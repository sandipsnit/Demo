Edición de Configuraciones de Servicio de Proxy

En esta sección se describe cómo editar configuraciones de servicio de proxy.

  1. Si aún no lo ha hecho, haga clic en Crear para crear una nueva sesión o en Editar para acceder a una sesión existente. Consulte Uso del Centro de Cambios.
  2. Localice el servicio de proxy que desee ver o editar. Consulte Localización de Servicios de Proxy.
  3. Haga clic en el nombre del servicio de proxy. La página Ver Servicio de Proxy: Detalles de Configuración muestra la información de la configuración del servicio de proxy seleccionado.
  4. Para ver o modificar la configuración, realice una de las acciones siguientes:
    • Haga clic en el icono Editar junto al nombre de la categoría cuyas propiedades desea ver o editar (por ejemplo, Configuración General, Configuración de Transporte, etc.). Las páginas que puede editar dependen de las páginas configuradas al crear el servicio de proxy.

      • Para consultar la lista de todas las páginas, consulte la página Ver Servicio de Proxy: Detalles de Configuración.

    • Haga clic en Editar al final de la página para mostrar la página Crear/Editar un Servicio de Proxy: Configuración General, que es la primera página de la secuencia de páginas para la configuración de este servicio de proxy.
  5. Siga viendo o editando, como se describe en Creación y Configuración de Servicios de Proxy.
  6. En la página Editar un Servicio de Proxy: Resumen, haga clic en Guardar para confirmar las actualizaciones en la sesión actual.
  7. Para terminar la sesión y desplegar la configuración en tiempo de ejecución, haga clic en Activar en Centro de Cambios.
