Depuración de Mensajes

Utilice esta página para depurar los mensajes seleccionados del almacén de datos de informes.

La depuración de mensajes se producen en segundo plano para que no se bloquee la página Resumen de Informe de Mensajes cuando se produzca la depuración. Por lo tanto, puede realizar una depuración que tarda un poco en terminar; la duración depende del número de mensajes que sea necesario depurar. Sin embargo, si filtra mensajes de informe, el proceso de depuración se ralentiza; la página Resumen de Informe de Menajes puede mostrar datos incorrectos ya que algunos de los datos no se han depurado aún.

La función Depurar Mensajes puede resultar útil durante las fases de diseño y prueba del proyecto. Sin embargo, en un entorno de producción, Oracle recomienda que el administrador de la base de datos maneje la gestión de los datos de la base de datos (incluida la depuración) con herramientas de gestión de base de datos.

  1. Seleccione Operaciones > Depurar Mensajes.
  2. Realice uno de los pasos siguientes mostrados en la Tabla 27-9.

    3. Tabla 27-9 Opciones de Depuración de Mensajes 
    Para...
    Realice estos pasos...
    Depurar todos los mensajes
    1. Seleccione el botón de radio Depurar Todos los Mensajes.
    2. Haga clic en Depurar.
    Depurar mensajes de un período especificado
    1. Seleccione el botón de radio Depurar Mensajes desde... Depurar Mensajes hasta...
    2. En el campo Depurar Mensajes desde, introduzca un mes, día, año y hora con el formato MM/DD/AA HH:MM:SS. Por ejemplo, puede introducir 10/10/07 12:45:00 AM.
    3. En el campo Depurar Mensajes hasta, introduzca un mes, día, año y hora con el formato MM/DD/AA HH:MM:SS.
    4. Haga clic en Depurar.
