Visualización de Detalles de Mensajes

Utilice esta página para ver los detalles de un mensaje concreto.

  1. Localice el mensaje. Consulte Visualización de Mensajes de Información.
  2. Haga clic en el índice de informes. La página Ver Detalles de Mensajes muestra la información de Configuración General que se muestra en la Tabla 27-3.

    3. Tabla 27-3 Parámetros de Configuración de Mensajes 
    Propiedad
    Descripción
    Identificador de Mensaje
    Identificación única para este mensaje.
    Registro de Hora de Base de Datos
    Fecha y hora en que se registró el mensaje en la base de datos.
    Hora en el Momento de Realizar el Registro
    Fecha y hora en que se registró el mensaje.
    Nombre del Servidor
    Nombre del servidor en el que se ha generado este mensaje.
    Estado
    Estado del pipeline en el que se ha generado este mensaje.
    SOLICITUD: Indica que la acción de informe se ejecuta en un pipeline de solicitud.
    RESPUESTA: Indica que la acción de informe se ejecuta en un pipeline de respuesta.
    ERROR: La acción se ejecuta en el manejador de errores de nivel de servicio.
    Nombre de Nodo
    Nombre de nodo del pipeline en el que se ha generado este mensaje.
    Nombre de Pipeline
    Nombre del pipeline en el que se ha generado este mensaje.
    Nombre de Etapa
    Nombre de la etapa en la que se ha generado este mensaje.

    La página muestra la información de Servicio Entrante que se muestra en la Tabla 27-4.

    Tabla 27-4 Detalles del Servicio Entrante 
    Propiedad
    Descripción
    Nombre
    Servicio de proxy entrante asociado a este mensaje. Se trata de un enlace a la página Ver Servicio de Proxy: Detalles de Configuración. Consulte Edición de Configuraciones de Servicio de Proxy.
    URI
    URI asociado al servicio de proxy.
    Operación
    Operación entrante asociada a este mensaje.

    La página muestra la información de Servicio Saliente que se muestra en la Tabla 27-5.

    Tabla 27-5 Detalles del Servicio Saliente 
    Propiedad
    Descripción
    Nombre
    Servicio de negocio saliente asociado a este mensaje. Se trata de un enlace a la página Ver Servicio de Negocio: Detalles de Configuración. Consulte Edición de Configuraciones de Servicio de Negocio.
    URI
    URI al punto final del servicio de negocio saliente.
    Operación
    Nombre de la operación llamada en el servicio saliente.

    La página muestra la información de Índice de Informes que se muestra en la Tabla 27-6.

    Tabla 27-6 Detalles de Índice de Informes 
    Propiedad
    Descripción
    Texto del Índice del Informe
    Los índices de informes son pares de valores de clave que se utilizan para extraer identificadores de clave de una variable de contexto de mensaje. Las claves son una forma cómoda para filtrar mensajes. Utilice la acción de informe para agregar pares de valores de clave. Consulte Adición y Edición de Acciones en Flujos de Mensajes.

    La página muestra la información de Fallo que se muestra en la Tabla 27-7.

    Tabla 27-7 Detalles de la Información de Fallo 
    Propiedad
    Descripción
    Código de Error
    Código de error asociado al mensaje, si lo hay.
    Especifique códigos de error al configurar la acción de emisión de error. Consulte la sección sobre manejo de errores en flujos de mensajes en Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus (Guía del Administrador de Oracle Fusion Middleware para Oracle Service Bus) en http://www.oracle.com/pls/as111170/lookup?id=OSBAG232 y Adición de Manejadores de Errores de Servicio de Proxy.
    Motivo
    Motivo del código de error.
    Detalles
    Detalles del fallo asociados al código de error. Estos detalles, si los hay, suelen ser un rastreo de pila de dónde se ha producido un fallo concreto y se pueden truncar debido a una limitación de tamaño de la base de datos. El límite es de 2048 caracteres.

    La página muestra la información de Cuerpo de Informe que se muestra en la Tabla 27-8.

    Tabla 27-8 Detalles del Cuerpo de Informe 
    Propiedad
    Descripción
    Texto del Cuerpo del Informe
    Este enlace abre otro explorador que muestra los detalles del cuerpo del informe. Se utiliza una expresión en la acción de informe para capturar el texto del cuerpo del informe. Consulte el procedimiento de la acción de informe en Adición y Edición de Acciones en Flujos de Mensajes.

  3. Haga clic en Aceptar cuando haya terminado de ver los detalles de un mensaje concreto.
