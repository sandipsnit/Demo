Adición de Destinos de Alertas

Los destinos de las alertas son direcciones de destino donde se reciben las notificaciones de alertas en Oracle Service Bus. Puede crear recursos de destino de alertas en la página Crear Destino de Alerta.

  1. Si aún no lo ha hecho, haga clic en Crear para crear una nueva sesión o en Editar para acceder a una sesión existente. Consulte Uso del Centro de Cambios.
  2. Seleccione Explorador de Proyectos y seleccione a continuación el proyecto o la carpeta a donde desee agregar el destino de alertas. Se muestra la página Vista de Carpetas/Proyectos.
  3. Desde la lista desplegable Crear Recurso, seleccione Destino de Alerta para mostrar la página Crear Destino de Alerta.
  4. En el campo Nombre del Recurso, introduzca un nombre único para este destino de alerta. Es un campo requerido.

    Siga las Restricciones de Nomenclatura de Recursos para obtener ayuda con la nomenclatura.
  5. En el campo Descripción del Recurso, introduzca una descripción para el destino de alertas.
  6. Seleccione alguno de los destinos a continuación, o todos, para incluirlos en este recurso de destino de alertas.
    7. Nota: Las alertas se agregan en la consola de administración de Oracle Service Bus en tiempo de ejecución, donde puede verlas en Operaciones > Panel de Control.
    • Interrupción SNMP: Si especifica Interrupción SNMP, las alertas se enviarán como interrupciones SNMP y se podrán procesar mediante cualquier sistema de supervisión de empresas (ESM) de terceros.
    • Informes: Si especifica Informes, las alertas se enviarán al módulo de informes de Oracle Service Bus, donde se podrán capturar mediante un proveedor de informes personalizado, el cual podrá desarrollar Oracle Service Bus con el proveedor de las API de generación de informes. Esto permite a terceros recibir y procesar alertas en código Java personalizado.
    • Registro de Alertas: Si selecciona Sí, las alertas que se envíen a este destino de alertas se registrarán en el registro de alertas.
  7. Agregue un destinatario de correo electrónico a la definición del recurso de destino de alertas. Consulte Adición de Destinatarios de Correo Electrónico.
  8. Agregue un destino de JMS a la definición del recurso de destino de alertas. Consulte Adición de Destinos de JMS.
  9. Haga clic Guardar. Se guardará el nuevo destino de alertas en la sesión actual.
  10. Para terminar la sesión y desplegar la configuración en tiempo de ejecución, haga clic en Activar en Centro de Cambios.
