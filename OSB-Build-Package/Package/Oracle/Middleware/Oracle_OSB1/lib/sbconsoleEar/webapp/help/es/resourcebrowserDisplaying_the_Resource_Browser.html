Uso del Explorador de Recursos

El Explorador de Recursos proporciona herramientas para localizar recursos y realizar acciones simples en ellos, como por ejemplo mostrarlos, editarlos, exportarlos y suprimirlos.

  1. Para mostrar el Explorador de Recursos, seleccione Explorador de Recursos en el panel de navegación de la izquierda.
  2. En el lado izquierdo de la página, haga clic en el nombre del tipo de recurso que quiera localizar. La Tabla 3-10 muestra los elementos que puede seleccionar y las tareas asociadas con la página que se muestra para cada una.
