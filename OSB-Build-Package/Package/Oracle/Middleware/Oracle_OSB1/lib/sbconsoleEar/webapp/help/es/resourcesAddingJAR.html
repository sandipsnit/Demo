Adición de JAR

  1. Si aún no lo ha hecho, haga clic en Crear para crear una nueva sesión o en Editar para acceder a una sesión existente. Consulte Uso del Centro de Cambios.
  2. Seleccione Explorador de Proyectos y seleccione a continuación el proyecto o la carpeta a donde desee agregar el recurso JAR. Se muestra la página Vista de Carpetas/Proyectos.
  3. En la lista desplegable Crear Recurso, seleccione JAR bajo Utilidad para abrir la página Crear un Nuevo Recurso JAR.
  4. En el campo Nombre del Recurso, introduzca un nombre único para este JAR. Es un campo requerido.

    Siga las Restricciones de Nomenclatura de Recursos para obtener ayuda con la nomenclatura.
  5. En el campo Descripción del Recurso, introduzca una descripción para el recurso JAR.
  6. En el campo Archivo de Origen JAR, realice una de las acciones siguientes:
    • Introduzca la ruta de acceso a la ubicación del recurso JAR.
    • Haga clic en Examinar para localizar un recurso JAR existente e importarlo.

      • Este campo es necesario.

  7. Haga clic en Guardar. Se creará el recurso JAR y se guardará en la sesión actual.

    8. Si el recurso JAR se carga sin problemas, podrá ver su contenido. Para obtener información sobre cómo ver detalles de los recursos JAR, consulte Edición de Recursos JAR.

  8. Para terminar la sesión y desplegar la configuración en tiempo de ejecución, haga clic en Activar en Centro de Cambios.
