Edición de Divisiones-Uniones

  1. Si aún no lo ha hecho, haga clic en Crear para crear una nueva sesión o en Editar para acceder a una sesión existente. Consulte Uso del Centro de Cambios.
  2. Localice la división-unión tal y como se describe en Localización de Divisiones-Uniones.
  3. Haga clic en el nombre de la división-unión.

    4. La página Ver División-Unión - Configuración muestra la información que se ve en la Tabla 9-3.

    Tabla 9-3 Detalles de Divisiones-Uniones 
    Propiedad
    Descripción
    Modificado por
    El usuario que creó la división-unión o que la importó a la configuración.
    Fecha de Modificación
    La fecha y la hora en la que el usuario creó esta división-unión o la importó a la configuración. Haga clic en el enlace de la fecha y la hora para ver el historial de cambios del recurso. Consulte Página Ver Historial de Cambios.
    Referencias
    El número de objetos al que hace referencia esta división-unión. Si existen tales referencias, haga clic en el enlace numérico para ver una lista con los objetos. Consulte Visualización de Referencias a Recursos.
    Con Referencia de
    El número de objetos que hacen referencia a esta división-unión. Si existen tales referencias, haga clic en el enlace numérico para ver una lista con los objetos. Consulte Visualización de Referencias a Recursos.
    Descripción
    La descripción de esta división-unión, de haberla.

  4. Puede aplicar la siguiente configuración operativa en una división-unión:

    5. Tabla 9-4 Configuración Operativa de División-Unión
    Propiedad
    Descripción
    Logs
    Si desea capturar información de registro en una división-unión, seleccione Activar Registro y, a continuación, seleccione el nivel de registro que desee capturar.
    Rastreo de Ejecución
    Para activar el rastreo en tiempo de ejecución, seleccione Activado.

  5. Haga clic en Actualizar para confirmar las actualizaciones en la sesión actual.
  6. Para terminar la sesión y desplegar la configuración en tiempo de ejecución, haga clic en Activar en Centro de Cambios.
