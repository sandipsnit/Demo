Edición de Recursos JAR

Utilice la página Ver un Recurso JAR - Configuración para actualizar archivos JAR. Actualizar un archivo JAR conlleva fundamentalmente hacer que el recurso apunte a una nueva versión de JAR, ya que no se puede editar un JAR con Oracle Service Bus.

  1. Si aún no lo ha hecho, haga clic en Crear para crear una nueva sesión o en Editar para acceder a una sesión existente. Consulte Uso del Centro de Cambios.
  2. Localice el recurso JAR tal y como se describe en Localización de Recursos JAR.
  3. Haga clic en el nombre del archivo JAR. La página Ver un Recurso JAR - Configuración de JAR muestra la información que se ve en la Tabla 7-2.

    4. Tabla 7-2 Información de Recursos JAR 
    Propiedad
    Descripción
    Modificado por
    El usuario que importó este JAR a la configuración.
    Fecha de Modificación
    La fecha y la hora en la que el usuario importó este JAR a la configuración. Haga clic en el enlace de la fecha y la hora para ver el historial de cambios del recurso. Consulte Página Ver Historial de Cambios.
    Referencias
    El número de recursos JAR al que hace referencia este recurso JAR. Si existen tales referencias, haga clic en el enlace numérico para ver una lista con los recursos JAR. Consulte Visualización de Referencias a Recursos.
    Con Referencia de
    El número de objetos que hacen referencia a este recurso JAR. Si existen tales referencias, haga clic en el enlace numérico para ver una lista con los objetos. Consulte Visualización de Referencias a Recursos.
    Descripción
    La descripción del recurso JAR, de haberla.

    La configuración JAR se muestra por defecto e incluye la información que se ve en la Tabla 7-3.

    Tabla 7-3 Detalles de Configuración JAR
    Propiedad
    Descripción
    Configuración JAR
    La lista de clases e interfaces en este JAR.
    Haga clic en el signo (+) que hay junto al nombre de una clase Java para mostrar una lista con los métodos que incluye la clase.

  4. Para hacer cambios en los campos, haga clic en Actualizar.

    5. El único campo que puede editar es Descripción. No puede editar archivos JAR en Oracle Service Bus. Para actualizar el archivo JAR, deberá seleccionar un nuevo JAR para utilizarlo. Para saber cómo hacerlo, consulte Adición de Recursos JAR.

  5. Haga clic en Guardar para confirmar las actualizaciones en la sesión actual.
  6. Para terminar la sesión y desplegar la configuración en tiempo de ejecución, haga clic en Activar en Centro de Cambios.
Nota: Si cambia el JAR utilizado por un servicio de negocio de EJB (dirige el recurso JAR a una nueva versión del JAR), necesitará volver a desplegar el EJB. Después, tendrá que editar todos los servicios de EJB que utilicen este JAR y volver a seleccionar el recurso JAR, guardar y activar. De este modo se volverá a empaquetar el servicio de negocio de EJB para utilizar el nuevo JAR.
Nota: Sin embargo, las acciones de llamada de Java y los servicios basados en Tuxedo sí detectarán automáticamente el nuevo JAR.
