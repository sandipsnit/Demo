Edición de Conexiones de MQ

En esta sección se describe cómo editar recursos de conexión de MQ que ya existen.

  1. Si aún no lo ha hecho, haga clic en Crear para crear una nueva sesión o en Editar para acceder a una sesión existente. Consulte Uso del Centro de Cambios.
  2. Localice el recurso de conexión de MQ tal y como se describe en Localización de Conexiones de MQ.
  3. Haga clic en el nombre del recursos de conexión de MQ.

    4. La página Ver Conexión de MQ - Configuración muestra la información que se ve en la Tabla 8-3.

    Tabla 8-3 Detalles de Conexión de MQ 
    Propiedad
    Descripción
    Modificado por
    El usuario que creó el recurso de conexión de MQ o que lo importó a la configuración.
    Fecha de Modificación
    La fecha y la hora en la que el usuario creó este recurso de conexión de MQ o lo importó a la configuración. Haga clic en el enlace de la fecha y la hora para ver el historial de cambios del recurso. Consulte Página Ver Historial de Cambios.
    Referencias
    El número de objetos al que hace referencia este recurso de conexión de MQ. Si existen tales referencias, haga clic en el enlace numérico para ver una lista con los objetos. Consulte Visualización de Referencias a Recursos.
    Con Referencia de
    El número de objetos que hacen referencia a este recurso de conexión de MQ. Si existen tales referencias, haga clic en el enlace numérico para ver una lista con los objetos. Consulte Visualización de Referencias a Recursos.
    Descripción
    Una descripción del recurso de conexión de MQ, de haberla.

  4. Para hacer un cambio en los campos, haga clic en Editar. Para ver una descripción de los distintos campos, consulte Adición de Conexiones de MQ.

    5. No puede cambiar el campo Nombre del Recurso.

  5. Haga clic en Guardar para confirmar las actualizaciones en la sesión actual.
  6. Para terminar la sesión y desplegar la configuración en tiempo de ejecución, haga clic en Activar en Centro de Cambios.
