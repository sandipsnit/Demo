Esquemas XML

Los esquemas describen tipos para datos primitivos o estructurados. Los esquemas XML corresponden a un lenguaje XML que describe las reglas que han de seguir los datos de negocio XML. Los esquemas XML especifican la estructura de los documentos, así como el tipo de dato de cada elemento y de cada atributo incluido en el documento.

Los esquemas XML se utilizan como referencias para los recursos WSDL. También puede usar esquemas XML en flujos de mensajes de servicios de proxy para validar un elemento especificado con una expresión Pat. Consulte Adición de Acciones de Validación.
