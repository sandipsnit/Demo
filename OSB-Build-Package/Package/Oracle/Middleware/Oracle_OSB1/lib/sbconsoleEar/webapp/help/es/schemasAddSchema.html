Adición de Esquemas XML

  1. Si aún no lo ha hecho, haga clic en Crear para crear una nueva sesión o en Editar para acceder a una sesión existente. Consulte Uso del Centro de Cambios.
  2. Seleccione Explorador de Proyectos y seleccione a continuación el proyecto o la carpeta donde desee agregar el nuevo esquema XML. Se muestra la página Vista de Carpetas/Proyectos.
  3. Desde la lista desplegable Crear Recurso, seleccione Esquema XML para mostrar la página Crear un Nuevo Recurso de Esquema XML.
  4. En el campo Nombre del Recurso, introduzca un nombre único para este esquema XML. Es un campo requerido.

    Siga las Restricciones de Nomenclatura de Recursos para obtener ayuda con la nomenclatura.
  5. En el campo Descripción del Recurso, introduzca una descripción para el esquema XML.
  6. En el campo Esquema XML, realice una de las siguientes acciones:
    • Introduzca el texto del nuevo esquema XML.
    • Haga clic en Examinar para localizar un esquema XML existente e importarlo.
    • Copie texto de un esquema XML existente y péguelo en este campo.

      • Este campo es necesario.

  7. Haga clic Guardar. Se guardará el nuevo esquema XML en la sesión actual.

    8. Al hacer clic en Guardar, el sistema mostrará las referencias cruzadas no resueltas asociadas al nuevo esquema XML, si es que las hay. Consulte Visualización de Referencias de Esquemas XML No Resueltas.

  8. Para terminar la sesión y desplegar la configuración en tiempo de ejecución, haga clic en Activar en Centro de Cambios.
