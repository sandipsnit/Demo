Supresión de Esquemas XML

  1. Si aún no lo ha hecho, haga clic en Crear para crear una nueva sesión o en Editar para acceder a una sesión existente. Consulte Uso del Centro de Cambios.
  2. Seleccione Explorador de Recursos > Esquemas XML.
  3. Haga clic en el icono de Suprimir del campo Opciones del esquema XML que desea suprimir. El esquema XML se suprimirá en la sesión actual. Aparecerá un icono de advertencia de supresión si hay otros recursos que hagan referencia a éste. Podrá suprimirlo confirmando la advertencia. Esto puede causar conflictos por referencias no resueltas al recurso suprimido.
  4. Para terminar la sesión y desplegar la configuración en tiempo de ejecución, haga clic en Activar en Centro de Cambios.
