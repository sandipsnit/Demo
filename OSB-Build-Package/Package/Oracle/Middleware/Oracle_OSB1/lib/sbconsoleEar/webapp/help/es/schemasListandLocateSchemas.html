Localización de Esquemas XML

  1. Realice una de las siguientes acciones:
    • Seleccione Explorador de Proyectospara abrir la página Vista de proyectos o la página Vista de Proyectos/Carpetas. A continuación, navegue por los proyectos o carpetas para buscar el esquema XML.
    • Seleccione Explorador de Recursos > Esquemas XML. La página Resumen de Esquemas XML muestra la información que se ve en la Tabla 4-1. Para ver una descripción más detallada de las distintas propiedades, consulte Edición de Esquemas XML.
  2. Para localizar un esquema XML concreto:
    • Haga clic en el icono Abrir para mostrar filtros de búsqueda adicionales. Si especifica varios criterios de búsqueda, sólo se devolverán los resultados que coincidan con todos los criterios.
    • Para limitar el número de elementos de la lista, puede filtrar por nombre, ruta de acceso y espacio de nombres. En los campos Nombre, Ruta de Acceso y Espacio de Nombres, introduzca el nombre, la ruta de acceso y el espacio de nombres de los destinos de búsqueda y haga clic en Buscar.

      • La ruta de acceso es el nombre del proyecto y el nombre de la carpeta donde reside el esquema XML.

    • Haga clic en Ver Todo para eliminar los filtros de búsqueda y ver todos los esquemas XML.

      • Tabla 4-1 Información de Esquemas XML
      Propiedad
      Descripción
      Nombre del Esquema XML
      El nombre único que se ha asignado al esquema XML. El nombre funciona como enlace a la página Detalles de Esquema XML. Consulte Edición de Esquemas XML.
      Ruta de Acceso
      La ruta de acceso corresponde al nombre del proyecto y al nombre de la carpeta en la que reside el esquema XML. Se trata de un enlace al proyecto o a la carpeta que contiene el recurso. Consulte Cualificación de Nombres de Recursos Mediante Proyectos y Carpetas.
      Espacio de Nombres de Esquema XML
      El espacio de nombres que se utiliza para cualificar cualquier definición incluida en el esquema XML.
      Opciones
      Haga clic en el icono de Suprimir para suprimir un esquema XML concreto. Consulte Supresión de Esquemas XML.
