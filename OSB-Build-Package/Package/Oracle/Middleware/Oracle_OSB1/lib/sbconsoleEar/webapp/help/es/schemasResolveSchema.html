Resolución de Referencias de Esquemas XML No Resueltas

Utilice la página Editar las Referencias de un Recurso de Esquema XML para resolver las referencias de esquemas XML no resueltas mediante la configuración de asignaciones de referencias de esquemas XML, como importaciones de esquemas XML.

  1. Si aún no lo ha hecho, haga clic en Crear para crear una nueva sesión o en Editar para acceder a una sesión existente. Consulte Uso del Centro de Cambios.
  2. En la página Resumen de Esquemas XML, en la columna Nombre del Esquema XML, haga clic en el nombre del esquema XML que le interese para ver los detalles de la referencia no resuelta. Se muestra la página Ver Detalles de Esquema XML.
  3. Haga clic en Editar Referencias para abrir la página Editar las Referencias de un Recurso de Esquema XML.
  4. En el campo Tipo de Recurso, seleccione Esquema XML.
  5. Haga clic en Explorar. Se abrirá el Explorador de Esquemas XML.
  6. En el Explorador de Esquemas XML, seleccione un esquema XML y, a continuación, seleccione una definición en el panel Definiciones.
  7. Haga clic en Enviar. El esquema XML que seleccionó aparecerá en el campo Nombre del Recurso.
  8. Haga clic en Guardar para resolver la referencia.
  9. En la página Detalles de Esquema XML, haga clic en Aceptar. El estado del esquema XML se mostrará como Válido.
