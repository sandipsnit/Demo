Visualización de Referencias de Esquemas XML No Resueltas

Para cada referencia de un esquema XML sin resolver, esta página muestra la información que se ve en la Tabla 4-5.

Tabla 4-5 Detalles de Esquemas XML No Resueltos
Propiedad
Descripción
Nombre del Esquema
El nombre único que se ha asignado al esquema XML. Haga clic en el nombre de un esquema XML concreto para ver sus detalles. ConsulteEdición de Esquemas XML.
Espacio de Nombres
El espacio de nombres que se utiliza para cualificar cualquier definición incluida en el esquema XML.

Para localizar una referencia concreta:

  • Reordene la lista. Haga clic en el nombre de la columna que aparece subrayado. Las flechas en orden ascendente o descendente indican el orden de visualización. Haga clic en el nombre de la columna para cambiar el orden.
  • Desplácese por las páginas. Utilice los controles de página situados sobre y debajo de la tabla. Para ir a una página, seleccione su número o utilice los botones de flecha para ir a la página siguiente, anterior, primera o última.
