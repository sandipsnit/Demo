Configuración de Seguridad

El módulo Configuración de Seguridad se puede utilizar para crear y modificar datos de seguridad y usarlos en el sistema de seguridad administrativa y de seguridad del transporte entrante de Oracle Service Bus.

El sistema de seguridad en el nivel de transporte entrante y el de seguridad en el nivel de mensajes utilizan los datos de usuario, de grupo y de rol para autenticar las solicitudes entrantes de los clientes. Este sistema de seguridad aplica políticas de control de acceso para determinar qué usuarios autenticados pueden utilizar los servicios de negocio y los servicios de proxy.

El sistema de seguridad administrativa utiliza los datos de usuario, de grupo y de rol para determinar qué usuarios autenticados pueden crear o modificar los datos de configuración de Oracle Service Bus o supervisar el rendimiento de Oracle Service Bus. Para obtener más información, consulte Descripción de Usuarios, Grupos, Roles de Seguridad y Políticas. Consulte también la sección sobre configuración de la seguridad administrativa en Oracle Fusion Middleware Developer's Guide for Oracle Service Bus (Guía del Desarrollador de Oracle Fusion Middleware para Oracle Service Bus) en http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539.

Nota: Cuando se exporta una configuración, no se pueden exportar usuarios, grupos, ni roles, ya que estos objetos se encuentran en almacenes del proveedor de seguridad. Tendrá que crear estos objetos de nuevo cuando importe la configuración exportada, o bien utilizar las herramientas de WebLogic Server (si dispone de ellas) para exportarlos e importarlos.
