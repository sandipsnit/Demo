Adición de Grupos

  1. Conéctese a la consola de administración de Oracle Service Bus como usuario con privilegios de administrador de WebLogic Server. Sólo los usuarios con el rol de administrador podrán modificar los datos de la configuración de seguridad. Consulte la sección sobre configuración de la seguridad administrativa en Oracle Fusion Middleware Developer's Guide for Oracle Service Bus (Guía del Desarrollador de Oracle Fusion Middleware para Oracle Service Bus) en http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539.
  2. Seleccione Configuración de Seguridad > Grupos para abrir la página Resumen de Grupos.
  3. Haga clic en Agregar Nuevo.

    4. Puede agregar un grupo desde dentro o desde fuera de una sesión.

  4. En el campo Nombre del Grupo, introduzca un nombre único. Tenga en cuenta que se pueden incluir espacios o caracteres especiales. Este campo es de inclusión obligatoria.
  5. En el campo Proveedor de Autenticación, seleccione el proveedor de autenticación.
  6. En el campo Grupos de Pertenencia, seleccione un grupo al que pertenezca el grupo.
    1. Seleccione un grupo en el campo Grupos Disponibles.
    2. Haga clic en la flecha para desplazar el grupo al campo Grupos Actuales.
  7. Haga clic en Guardar para crear el grupo.

    8. La consola de administración de Oracle Service Bus guarda el grupo y éste queda inmediatamente disponible para todas las sesiones. Si el usuario se encuentra en una sesión al agregar el grupo y después desecha la sesión, la consola de administración de Oracle Service Bus no suprime el nuevo grupo.
