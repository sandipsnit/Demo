Edición de Grupos

Utilice la página Ver Detalles de Grupo para ver los detalles de un grupo concreto y cambiarlos.

  1. Conéctese a la consola de administración de Oracle Service Bus como usuario con privilegios de administrador de WebLogic Server. Sólo los usuarios con el rol de administrador podrán modificar los datos de la configuración de seguridad. Consulte la sección sobre configuración de la seguridad administrativa en Oracle Fusion Middleware Developer's Guide for Oracle Service Bus (Guía del Desarrollador de Oracle Fusion Middleware para Oracle Service Bus) en http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539.
  2. Localice el grupo. Consulte Localización de Grupos.
  3. Haga clic en el nombre del grupo. La página Ver Detalles de Grupo muestra la información que se ve en la Tabla 24-4.

    4. Tabla 24-4 Detalles de Grupos
    Propiedad
    Descripción
    Nombre del Grupo
    Nombre de este grupo.
    Proveedor de Autenticación
    El proveedor de autenticación que contiene esta definición de grupo.
    Grupos
    El grupo al que pertenece el grupo.

  4. Para editar los detalles del grupo, haga clic en Volver a Configurar para abrir la página Editar Detalles del Grupo.

    5. Puede editar los detalles de un grupo desde dentro o desde fuera de una sesión.

  5. Realice los cambios oportunos en el campo Grupos de Pertenencia. Consulte Adición de Grupos para ver descripciones de los distintos campos.

    6. El campo Nombre del Grupo no se puede cambiar.

  6. Haga clic en Guardar Cambios para actualizar el grupo.

    7. La consola de administración de Oracle Service Bus actualiza los detalles del grupo y la actualización queda inmediatamente disponible en todas las sesiones. Si el usuario se encuentra en una sesión al actualizar el grupo y después desecha la sesión, la consola de administración de Oracle Service Bus no suprime las actualizaciones.
