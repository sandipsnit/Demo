Adición de Usuarios

  1. Conéctese a la consola de administración de Oracle Service Bus como usuario con privilegios de administrador de WebLogic Server. Sólo los usuarios con el rol de administrador podrán modificar los datos de la configuración de seguridad. Consulte la sección sobre configuración de la seguridad administrativa en Oracle Fusion Middleware Developer's Guide for Oracle Service Bus (Guía del Desarrollador de Oracle Fusion Middleware para Oracle Service Bus) en http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539.
  2. Seleccione Configuración de Seguridad para abrir la página Resumen de Usuarios.
  3. Haga clic en Agregar Nuevo para ver la página Crear un Nuevo Usuario - Configuración General.

    4. Puede agregar un usuario desde dentro o desde fuera de una sesión.

  4. En el campo Usuario, introduzca un nombre único. Este campo es de inclusión obligatoria.
  5. En el campo Contraseña, escriba una contraseña. Este campo es de inclusión obligatoria.
    6. Nota: Los proveedores de autenticación pueden imponer una longitud mínima para las contraseñas. Si los usuarios se han definido en el proveedor de autenticación de WebLogic, la longitud mínima por defecto es de 8 caracteres. Puede personalizar este valor mediante la consola de administración de WebLogic Server. (El proveedor de autenticación de WebLogic se configura en el dominio de seguridad por defecto con el nombre de DefaultAuthenticator.)
  6. En el campo Confirmar Contraseña, escriba la misma contraseña que escribió en el campo Contraseña. Este campo es de inclusión obligatoria.
  7. En el campo Proveedor de Autenticación, seleccione el proveedor de autenticación para el usuario.

    8. Si se configuran varios proveedores de autenticación en el dominio de seguridad, aparecerán en la lista. Seleccione qué base de datos del proveedor de autenticación debe almacenar la información para el nuevo usuario.

  8. En el campo Grupos de Pertenencia, seleccione un grupo para el usuario.
    1. Seleccione un grupo en el campo Grupos Disponibles.
    2. Haga clic en la flecha para desplazar el grupo al campo Grupos Actuales.
  9. Haga clic en Guardar para crear el usuario.

    10. La consola de administración de Oracle Service Bus guarda el usuario y éste queda inmediatamente disponible para todas las sesiones. Si el usuario se encuentra en una sesión al agregar el usuario y después desecha la sesión, la consola de administración de Oracle Service Bus no suprime el nuevo usuario.
