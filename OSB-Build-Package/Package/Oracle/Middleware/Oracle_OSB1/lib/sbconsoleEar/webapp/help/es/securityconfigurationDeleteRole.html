Supresión de Roles

  1. Conéctese a la consola de administración de Oracle Service Bus como usuario con privilegios de administrador de WebLogic Server. Sólo los usuarios con el rol de administrador podrán modificar los datos de la configuración de seguridad. Consulte la sección sobre configuración de la seguridad administrativa en Oracle Fusion Middleware Developer's Guide for Oracle Service Bus (Guía del Desarrollador de Oracle Fusion Middleware para Oracle Service Bus) en http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539.
  2. Seleccione Configuración de Seguridad > Roles para mostrar la página Roles Globales.
  3. Seleccione el rol que quiera suprimir. Puede seleccionar varios roles si lo estima oportuno.

    4. Puede suprimir roles desde dentro y desde fuera de una sesión.

  4. Haga clic en Suprimir. Aparecerá un mensaje que le solicitará que confirme que desea suprimir el rol.
  5. Para suprimir el rol, haga clic en Aceptar.

    6. La consola de administración de Oracle Service Bus suprime el rol. Si el usuario se encuentra en una sesión al suprimir el rol y después desecha la sesión, la consola de administración de Oracle Service Bus no anula la supresión del rol.
