Supresión de Usuarios

Utilice la página Resumen de Usuarios para suprimir un usuario que haya seleccionado o varios de ellos.

  1. Conéctese a la consola de administración de Oracle Service Bus como usuario con privilegios de administrador de WebLogic Server. Sólo los usuarios con el rol de administrador podrán modificar los datos de la configuración de seguridad. Consulte la sección sobre configuración de la seguridad administrativa en Oracle Fusion Middleware Developer's Guide for Oracle Service Bus (Guía del Desarrollador de Oracle Fusion Middleware para Oracle Service Bus) en http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539.
  2. Seleccione Configuración de Seguridad para abrir la página Resumen de Usuarios.
  3. Seleccione el usuario que quiera suprimir. Puede seleccionar varios usuarios si lo estima oportuno.

    4. Puede suprimir un usuario desde dentro o desde fuera de una sesión.

  4. Haga clic en Suprimir. Aparecerá un mensaje que le solicitará que confirme que desea suprimir el usuario.
  5. Para suprimir el usuario, haga clic en Aceptar.

    6. La consola de administración de Oracle Service Bus suprimirá el usuario. Si se encuentra en una sesión al suprimir el usuario y después desecha la sesión, la consola de administración de Oracle Service Bus no suprimirá las actualizaciones.

  6. Otra opción es hacer clic en el icono de Suprimir en la columna Opciones correspondiente al usuario que quiera suprimir.
