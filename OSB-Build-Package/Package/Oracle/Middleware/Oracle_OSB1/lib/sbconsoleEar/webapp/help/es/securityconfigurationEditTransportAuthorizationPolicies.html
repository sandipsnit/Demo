Edición de Políticas de Acceso de Nivel de Transporte

Utilice la página VerDetalles de Política para editar la política de control de acceso en el nivel de transporte de un servicio de proxy. La página muestra la información que se ve en la Tabla 24-10.

Tabla 24-10 Detalles de Políticas
Propiedad
Descripción
Nombre del Servicio de Proxy
Muestra el nombre del servicio de proxy para el cual se seleccionó Control de Acceso a Transporte en la página Seguridad.
Proveedores
Muestra los proveedores de autorización configurados para el dominio de seguridad.
Condiciones de Política
Muestra las condiciones que determinan para qué usuarios procesará solicitudes el servicio de proxy.

  1. Conéctese a la consola de administración de Oracle Service Bus como usuario con privilegios de administrador de WebLogic Server. Sólo los usuarios con el rol de administrador podrán modificar los datos de la configuración de seguridad. Consulte la sección sobre configuración de la seguridad administrativa en Oracle Fusion Middleware Developer's Guide for Oracle Service Bus (Guía del Desarrollador de Oracle Fusion Middleware para Oracle Service Bus) en http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539.
  2. Localice la política de acceso. Consulte Ubicación de Políticas de Control de Acceso.
  3. En la página Seguridad, en la columna Control de Acceso a Transporte, haga clic en el nombre del servicio de proxy. Se muestra el editor de políticas.

    4. Puede editar políticas de control de acceso desde dentro de una sesión.

  4. En el campo Proveedores de Autorización, seleccione un proveedor de autorización. Oracle recomienda seleccionar XACMLAuthorizer.
    5. Nota: Oracle Service Bus ha ya no soporta el proveedor de autorización por defecto de WebLogic. En lugar de éste, Oracle recomienda utilizar el proveedor de autorización XACML de WebLogic.
  5. Agregue condiciones a las políticas. Consulte Adición de Condiciones de Políticas.
  6. Cuando termine de introducir las condiciones en la sección Condiciones de Política, haga clic en Guardar.
