Localización de Grupos

  1. Seleccione Configuración de Seguridad > Grupos. La página Resumen de Grupos muestra la información que se ve en la Tabla 24-3. Para ver una descripción más detallada de las distintas propiedades, consulte Edición de Grupos.
  2. Para restringir el número de elementos en la lista o localizar un grupo concreto, puede optar por aplicar un filtro según nombres de grupo. Escriba el nombre del grupo o parte de él en el campo Nombre y, a continuación, haga clic en Buscar.

    3. Puede utilizar el carácter comodín de asterisco (*). (No se soportan otros caracteres comodín.)

    Haga clic en Ver Todo para eliminar los filtros de búsqueda y mostrar todos los grupos.

    Tabla 24-3 Información de Grupos 
    Propiedad
    Descripción
    Nombre del Grupo
    El nombre del grupo. El nombre funciona como un enlace a la página Ver Detalles de Grupo. Consulte Edición de Grupos.
    Grupos de Pertenencia
    El grupo al que pertenece el grupo. El nombre funciona como un enlace a la página Ver Detalles de Grupo. Consulte Edición de Grupos.
    Proveedor de Autenticación
    El proveedor de autenticación que contiene esta definición de grupo.
    Suprimir
    Haga clic en el icono Suprimir para suprimir un grupo concreto. Consulte Supresión de Grupos.
