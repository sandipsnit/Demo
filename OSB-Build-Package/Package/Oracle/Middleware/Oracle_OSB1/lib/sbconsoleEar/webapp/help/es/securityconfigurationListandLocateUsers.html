Localización de Usuarios

  1. Seleccione Configuración de Seguridad > Usuarios. La página Resumen de Usuarios muestra la información que se ve en la Tabla 24-1. Para ver una descripción más detallada de las distintas propiedades, consulte Edición de Usuarios.
  2. Para restringir el número de elementos en la lista o localizar a un usuario concreto, puede optar por aplicar un filtro según nombres de usuario. Escriba el nombre del usuario o parte de él en el campo Nombre y, a continuación, haga clic en Buscar.

    3. Puede utilizar el carácter comodín de asterisco (*). (No se soportan otros caracteres comodín.)

    Haga clic en Ver Todo para eliminar los filtros de búsqueda y mostrar todos los usuarios.

    Tabla 24-1 Información de Usuarios
    Propiedad
    Descripción
    Nombre de Usuario
    El nombre único asignado al usuario. El nombre funciona como un enlace a la página Ver Detalles de Usuario. Consulte Edición de Usuarios.
    Grupos de Pertenencia
    El nombre del grupo al que pertenece el usuario. El nombre funciona como un enlace a la página Ver Detalles de Grupo. Consulte Edición de Grupos.
    Proveedor de Autenticación
    El proveedor de autenticación para el usuario.
    Opciones
    Haga clic en el icono de Suprimir para suprimir un usuario concreto. Consulte Supresión de Usuarios.
