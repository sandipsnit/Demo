Edición de Usuarios

Utilice la página Ver Detalles de Usuario para ver los detalles de un usuario concreto y cambiarlos.

  1. Conéctese a la consola de administración de Oracle Service Bus como usuario con privilegios de administrador de WebLogic Server. Sólo los usuarios con el rol de administrador podrán modificar los datos de la configuración de seguridad. Consulte la sección sobre configuración de la seguridad administrativa en Oracle Fusion Middleware Developer's Guide for Oracle Service Bus (Guía del Desarrollador de Oracle Fusion Middleware para Oracle Service Bus) en http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539.
  2. Localice al usuario. Consulte Localización de Usuarios.
  3. Haga clic en el nombre del usuario. La página Ver Detalles de Usuario muestra la información que se ve en la Tabla 24-2.

    4. Tabla 24-2 Detalles de Usuarios
    Propiedad
    Descripción
    Nombre de Usuario
    El nombre del usuario.
    Proveedor de Autenticación
    El proveedor de autenticación que contiene esta definición de usuario.
    Grupos de Pertenencia
    Nombre del grupo al que pertenece el usuario.

  4. Para editar los detalles del usuario, haga clic en Volver a Configurar para abrir la página Editar Detalles de Usuario.

    5. Puede editar los detalles de un usuario desde dentro o desde fuera de una sesión.

  5. Realice los cambios oportunos en los campos Contraseña Nueva, Confirmar Contraseña y Grupos de Pertenencia. Consulte Adición de Usuarios para ver las descripciones de los distintos campos.

    6. El campo Usuario no puede cambiar.

  6. Haga clic en Guardar Cambios para actualizar el usuario.

    7. La consola de administración de Oracle Service Bus actualiza los detalles del usuario y la actualización queda inmediatamente disponible en todas las sesiones. Si el usuario se encuentra en una sesión al actualizar el usuario y después desecha la sesión, la consola de administración de Oracle Service Bus no suprime las actualizaciones.
