Supresión de Cuentas de Servicio

Cuando se suprime una cuenta de servicio, también se suprime el nombre de usuario, la contraseña o los datos de asignación de usuario local a usuario remoto que contiene la cuenta de servicio.

  1. Si aún no lo ha hecho, haga clic en Crear para crear una nueva sesión o en Editar para acceder a una sesión existente. Consulte Uso del Centro de Cambios.
  2. Si hay algún servicio de negocio o de proxy configurado para usar la cuenta de servicio, elimine la cuenta de servicio del servicio de negocio o del servicio de proxy.

    3. Consulte Edición de Configuraciones de Servicio de Negocio o Edición de Configuraciones de Servicio de Proxy.

  3. Seleccione Explorador de Recursos > Cuentas de Servicio para mostrar la página Resumen de Cuentas de Servicio.
  4. Haga clic en el icono de Suprimir del campo Opciones correspondiente a la cuenta de servicio que desea suprimir.

    5. La cuenta de servicio se suprime de la sesión actual. Si se ha configurado un servicio de negocio o de proxy para utilizar una cuenta de servicio, un icono de advertencia de supresión indicará que puede suprimir la cuenta de servicio con una confirmación de advertencia. Esto puede causar conflictos debido a referencias no resueltas del servicio a la cuenta de servicio suprimida.

  5. Para terminar la sesión y desplegar la configuración en tiempo de ejecución, haga clic en Activar en Centro de Cambios.
