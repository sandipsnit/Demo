Divisiones-Uniones

Las divisiones-uniones le permiten enviar llamadas a mensajes en paralelo (además de secuencialmente) y agregar las respuestas. Para obtener más información acerca de las divisiones-uniones, consulte la sección sobre mejora del rendimiento del servicio con divisiones-uniones en Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus (Guía del Administrador de Oracle Fusion Middleware para Oracle Service Bus) en http://www.oracle.com/pls/as111170/lookup?id=OSBAG291. Consulte también la sección sobre cómo trabajar con divisiones-uniones en http://www.oracle.com/pls/as111170/lookup?id=OSBDV193 y la sección sobre la referencia de la interfaz de usuario de las divisiones-uniones en http://www.oracle.com/pls/as111170/lookup?id=OSBDV573 en la Oracle Fusion Middleware Developer's Guide for Oracle Service Bus (Guía del Desarrollador de Oracle Fusion Middleware para Oracle Service Bus).

A las divisiones-uniones se accede a través de los servicios de negocio de tipo de transporte de flujo. Para obtener más información, consulte la página Configuración de Transporte de Flujo.
