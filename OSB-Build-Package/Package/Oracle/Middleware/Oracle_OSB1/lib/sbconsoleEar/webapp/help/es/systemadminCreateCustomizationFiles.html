Creación de Archivos de Personalización

La página Crear Archivo de Personalización permite generar cómodamente un archivo de personalización para el juego de recursos o proyectos que seleccione el usuario. Podrá utilizar este archivo como punto de inicio para hacer las modificaciones oportunas especificando los valores reales para el entorno.

Puede utilizar archivos de personalización para cambiar valores de entorno y para cambiar referencias dentro de los recursos. Los archivos de personalización pueden incluir personalizaciones para todos los valores de entorno dentro de los recursos que se seleccionen, incluidos tipos de valores de entornos complejos definidos en la clase EnvValueTypes. Además, estos archivos incluyen un tipo de personalización de referencias para cambiar referencias a recursos dentro de recursos con dependencias.

El esquema de personalización (Customization.xsd), que describe los tipos de personalización disponibles en la siguiente ubicación de la instalación de Oracle Service Bus:

OSB_ORACLE_HOME\modules\com.bea.common.configfwk_version.jar.

  1. Seleccione Administración del Sistema > Crear Archivo de Personalización.

    2. La página Crear Archivo de Personalización muestra la lista de objetos en la configuración. Se muestra el nombre y el tipo de cada objeto.

  2. Seleccione los proyectos o los recursos que desee incluir en el archivo de personalización.
    1. Amplíe las carpetas de proyectos. Se muestra el nombre y el tipo de cada recurso contenido en el proyecto.
    2. Seleccione la casilla de control asociada con los proyectos o los recursos que quiera incluir en el archivo de personalización.
    3. Desactive la casilla de control asociada con los proyectos o los recursos que no quiera incluir en el archivo de personalización.
  3. Haga clic en Crear Archivo para crear un archivo XML de personalización.
  4. En el cuadro de diálogo Descarga de Archivo, haga clic en Abrir para abrir el archivo de personalización, o haga clic en Guardar para guardar el archivo XML en el escritorio.
