Establecimiento de una Configuración de UDDI por Defecto

Puede diseñar uno de los registros configurados como el registro UDDI por defecto para el dominio. Consulte Configuración de Registros UDDI.

Para utilizar la función de publicación automática, primero deberá establecer un registro por defecto. Consulte Uso de Publicación Automática.

  1. Si aún no lo ha hecho, haga clic en Crear para crear una nueva sesión o en Editar para acceder a una sesión existente. Consulte Uso del Centro de Cambios.
  2. Seleccione Administración del Sistema > Registros UDDI.
  3. En el panel Configuración por Defecto de UDDI, haga clic en Seleccionar Valor por Defecto.
  4. Desde la lista Nombre de Registro por Defecto, seleccione el nombre del registro que quiera establecer como registro por defecto.
  5. Para establecer una entidad de negocio por defecto, elija una entidad en la lista Entidad de Negocio.
  6. Haga clic en Definir Valor por Defecto.
  7. Para cambiar el registro por defecto, haga clic en Limpiar Selección y repita los pasos anteriores.
