Edición de Proveedores de JNDI

Utilice esta página para ver los detalles de un proveedor de JNDI y, si fuera necesario, editar la configuración. Además podrá restablecer la caché del proveedor de JNDI y probar la conexión con el proveedor de JNDI.

  1. Si aún no lo ha hecho, haga clic en Crear para crear una nueva sesión o en Editar para acceder a una sesión existente. Consulte Uso del Centro de Cambios.
  2. Localice al proveedor de JNDI. Consulte Visualización de Proveedores de JNDI.
  3. Haga clic en el nombre del proveedor de JNDI.

    4. La página Ver Proveedor de JNDI - Configuración muestra la información del proveedor de JNDI que se ve en la Tabla 30-2.

    Tabla 30-2 Información de Proveedor de JNDI
    Propiedad
    Descripción
    Modificado por
    El usuario que ha creado o modificado este recurso de proveedor de JNDI, o lo ha importado a la configuración.
    Fecha de Modificación
    La fecha y hora a la que el usuario ha creado o modificado este recurso de proveedor de JNDI, o lo ha importado a la configuración. Haga clic en el enlace de fecha y hora para ver el historial de cambios de este recurso. Consulte Página Ver Historial de Cambios.
    Referencias
    El número de objetos a los que hace referencia este proveedor de JNDI. Si existen estas referencias, haga clic en el enlace numérico para ver una lista de las referencias. Consulte Visualización de Referencias a Recursos.
    Con Referencia de
    El número de objetos que hacen referencia a este proveedor de JNDI. Si existen estas referencias, haga clic en el enlace numérico para ver una lista de los objetos. Consulte Visualización de Referencias a Recursos.
    Descripción
    Descripción del proveedor de JNDI.
    Caché de JNDI
    Activado o Desactivado. Cuando está activado, el contexto de JNDI y los objetos de JNDI se almacenan en la caché local, lo cual mejora el rendimiento al buscar objetos. Oracle recomienda dejar activada la caché de JNDI.
    URL de Proveedor
    La URL del proveedor de JNDI.
    Timeout de Solicitud de JNDI
    El timeout para solicitudes de JNDI en milésimas de segundo. El valor por defecto de cero (0) implica que no hay timeout.
    Usuario y Contraseña
    El nombre de usuario que se utiliza para acceder a un proveedor de JNDI protegido, con capacidad para cambiar la contraseña.

  4. Haga clic en Editar para mostrar la página Editar Proveedor de JNDI - [nombre del proveedor].

    5. El campo Nombre no se puede editar.

  5. En el campo Descripción podrá editar la descripción corta del proveedor de JNDI.
  6. En el campo URL de Proveedor podrá editar la URL del proveedor de JNDI.
  7. Puede editar los campos Usuario y Contraseña.

    8. Si se ha configurado el proveedor de JNDI con un nombre de usuario y una contraseña, el valor de Usuario no se podrá editar. Puede editar los campos Contraseña Nueva y Confirmar Contraseña. Si el proveedor de JNDI no se ha configurado con un nombre de usuario y una contraseña, entonces el campo Usuario se podrá editar y podrá introducir un nuevo nombre de usuario.

  8. Haga clic en Guardar para confirmar las actualizaciones en la sesión actual.
  9. Para terminar la sesión y desplegar la configuración en tiempo de ejecución, haga clic en Activar en Centro de Cambios.
