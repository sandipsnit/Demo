Edición de Servidores Proxy

Utilice esta página para ver los detalles de un servidor proxy y, si fuera necesario, editar su configuración.

  1. Si aún no lo ha hecho, haga clic en Crear para crear una nueva sesión o en Editar para acceder a una sesión existente. Consulte Uso del Centro de Cambios.
  2. Localice el servidor proxy. Consulte Localización de Servidores Proxy.
  3. Haga clic en el nombre del servidor proxy.

    4. La página Ver Servidores Proxy - Configuración muestra la información que se ve en la Tabla 30-6.

    Tabla 30-6 Información de Servidores Proxy
    Propiedad
    Descripción
    Modificado por
    El usuario que creó o modificó este recurso de servidor proxy.
    Fecha de Modificación
    La fecha y la hora en la que el usuario creó o modificó este recurso de servidor proxy. Haga clic en el enlace de la fecha y la hora para ver el historial de cambios del recurso. Consulte Página Ver Historial de Cambios.
    Referencias
    Número de objetos a los que hace referencia este servidor proxy. Si existen estas referencias, haga clic en el enlace numérico para ver una lista de ellas. Consulte Visualización de Referencias a Recursos.
    Con Referencia de
    El número de objetos que hacen referencia a este servidor proxy. Si existen estas referencias, haga clic en el enlace numérico para ver una lista de los objetos. Consulte Visualización de Referencias a Recursos.
    Descripción
    La descripción de este servidor proxy, si la hay.
    Parámetros de Host-Port
    Detalles acerca de cada servidor proxy en el recurso de servidor proxy, con la siguiente información:
    • Host de Servidor: Nombre del host o dirección IP del servidor proxy.

      • El nombre de host de servidor del servidor proxy de Oracle Service Bus debe ser idéntico al nombre de host de servidor del servidor proxy real.

    • Puerto sin Cifrado: Número de puerto del servidor proxy sin cifrar.
    • Puerto SSL: Número de puerto SSL del servidor proxy.
    Nombre de Usuario
    El nombre de usuario utilizado para la autenticación de proxy.

  4. Haga clic en Editar para mostrar la página Editar Servidor Proxy - [nombre de servidor].

    5. El campo Nombre no se puede editar.

  5. En el campo Descripción podrá editar la descripción corta del servidor proxy.
  6. En la sección Parámetros de Host-Port, podrá agregar un nuevo servidor proxy introduciendo la información siguiente:
    1. En el campo Host de Servidor, introduzca el nombre del host o la dirección IP del servidor proxy. Este campo es de inclusión obligatoria.

      2. El nombre de host de servidor del servidor proxy de Oracle Service Bus debe ser idéntico al nombre de host de servidor del servidor proxy real.

    2. En el campo Puerto sin Cifrado, introduzca el número de puerto sin cifrar del servidor proxy.
    3. En el campo Puerto SSL, introduzca el número de puerto SSL del servidor proxy. Debe introducir texto sin cifrar o un número de puerto SSL.
    4. Haga clic en Agregar.

      5. Puede configurar varios servidores proxy para cada recurso de servidor proxy. Esto permite a Oracle Service Bus realizar un equilibrio de carga y ofrecer función de tolerancia ante fallos para el recurso de servidor proxy.

      También puede editar o suprimir una configuración de servidor proxy que ya exista haciendo lo siguiente:

    • Haga clic en el icono de Editar del campo Opciones correspondiente al servidor proxy que quiera editar. Edite el campo Host de Servidor, el campo Puerto sin Cifrado y el campo Puerto SSL y, a continuación, haga clic en Actualizar.
    • Haga clic en el icono de Suprimir del campo Opciones correspondiente al servidor proxy que desea suprimir.
  7. Si fuera necesario, podrá editar los campos Usuario y Contraseña utilizados para realizar la autenticación de proxy.
  8. Haga clic en Guardar para confirmar las actualizaciones en la sesión actual.
  9. Para terminar la sesión y desplegar la configuración en tiempo de ejecución, haga clic en Activar en Centro de Cambios.
