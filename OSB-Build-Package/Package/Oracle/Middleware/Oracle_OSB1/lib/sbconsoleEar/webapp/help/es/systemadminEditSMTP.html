Edición de Servidores SMTP

Utilice esta página para ver los detalles de un servidor SMTP y, si fuera necesario, editar su configuración.

  1. Si aún no lo ha hecho, haga clic en Crear para crear una nueva sesión o en Editar para acceder a una sesión existente. Consulte Uso del Centro de Cambios.
  2. Localice el servidor SMTP. Consulte Localización de Servidores SMTP.
  3. Haga clic en el nombre del servidor SMTP.

    4. La página Ver Servidores SMTP - Configuración muestra la información que se ve en la Tabla 30-4.

    Tabla 30-4 Información de Servidores SMTP
    Propiedad
    Descripción
    Modificado por
    El usuario que creó o modificó este recurso de servidor SMTP.
    Fecha de Modificación
    La fecha y la hora en la que el usuario creó o modificó este recurso de servidor SMTP. Haga clic en el enlace de la fecha y la hora para ver el historial de cambios del recurso. Consulte Página Ver Historial de Cambios.
    Referencias
    Número de objetos a los que hace referencia este servidor SMTP. Si existen estas referencias, haga clic en el enlace numérico para ver una lista de ellas. Consulte Visualización de Referencias a Recursos.
    Con Referencia de
    El número de objetos que hacen referencia a este servidor SMTP. Si existen estas referencias, haga clic en el enlace numérico para ver una lista de los objetos. Consulte Visualización de Referencias a Recursos.
    Descripción
    La descripción de este servidor SMTP, de haberla.
    URL del Servidor
    La URL que apunta a la ubicación del servidor SMTP.
    Número de Puerto
    El número de puerto para el servidor SMTP. Este puerto es el número 25 por defecto.
    Nombre de Usuario
    El nombre de usuario para acceder a un servidor SMTP protegido.

  4. Haga clic en Editar para mostrar la página Editar Servidor SMTP - [nombre de servidor].

    5. El campo Nombre no se puede editar.

  5. En el campo Descripción podrá editar la descripción corta del servidor SMTP.
  6. En el campo URL del Servidor podrá editar la URL del servidor SMTP.
  7. En el campo Número de Puerto podrá editar el número de puerto.
  8. Puede editar los campos Usuario y Contraseña.
  9. Haga clic en Guardar para confirmar las actualizaciones en la sesión actual.
  10. Para terminar la sesión y desplegar la configuración en tiempo de ejecución, haga clic en Activar en Centro de Cambios.

Si cambia el nombre de un recurso SMTP, el nombre nuevo se actualiza automáticamente en los servicios de negocio de correo electrónico que hacen referencia al recurso SMTP en el URI de punto final.
