Ejecución de Archivos de Personalización

Utilice esta página para ejecutar un archivo XML de personalización que se haya guardado previamente en el sistema. Los archivos de personalización permiten modificar cómodamente el entorno durante el despliegue, desde la etapa de desarrollo al almacenamiento en zona intermedia, desde el almacenamiento en zona intermedia a la producción, o durante la etapa de diseño.

Debe estar en una sesión para poder ejecutar un archivo de personalización.

  1. Si aún no lo ha hecho, haga clic en Crear para crear una nueva sesión o en Editar para acceder a una sesión existente. Consulte Uso del Centro de Cambios.
  2. Seleccione Administración del Sistema > Ejecutar Archivo de Personalización.
  3. Seleccione un archivo de personalización para ejecutarlo.
    1. En el campo Nombre de Archivo, haga clic en Examinar para localizar el directorio en el que está almacenado el archivo de personalización.
    2. Seleccione el archivo XML y, a continuación, haga clic en Abrir.
  4. Haga clic en Siguiente. Se muestran los detalles de personalización, los cuales se pueden ver en la Tabla 31-2.

    5. Tabla 31-2 Detalles de Personalización
    Propiedad
    Descripción
    Operación
    • Buscar y Sustituir: Modifique los valores del entorno realizando sustituciones de cadenas.
    • Asignar Referencias: Asigna referencias existentes encontradas en recursos a otras referencias.
    • Definir Valores de Entorno: Modifica valores de entorno para recursos.
    Aplicar a
    Nombre del servicio, proyecto, carpeta o recurso que personalizar.
    El nombre funciona como un enlace a la página Detalles de Configuración del servicio o del recurso, o al proyecto o la carpeta en el Explorador de Proyectos.
    Descripción
    Una descripción de cada personalización.

  5. Para personalizar sólo recursos, servicios, proyectos o carpetas que se hayan cambiado en la sesión actual, seleccione la casilla de control Sólo Elementos Cambiados en la Sesión Actual.

    6. Si limita las personalizaciones a sólo los recursos que se hayan modificado en la sesión actual, la columna Aplicar a se actualizará con los recursos que se hayan modificado en la sesión actual. Si limita las actualizaciones de un proyecto o una carpeta, entonces los recursos modificados en la sesión actual dentro del proyecto o la carpeta, se mostrarán en la columna Aplicar a.

  6. Haga clic en Ejecutar para confirmar las actualizaciones en la sesión actual.

    7. Para ver los detalles de personalización, desde dentro de la sesión, seleccione Ver Cambios bajo Centro de Cambios y, a continuación, haga clic en la tarea Personalización. Consulte Visualización de Cambios de Configuración.

  7. Para terminar la sesión y desplegar la configuración en tiempo de ejecución, haga clic en Activar en Centro de Cambios.
