Exportación de Recursos

Sírvase de la página Exportar Recursos para aplicar las funciones Exportar Proyectos o Exportar Recursos. Si selecciona exportar un proyecto concreto, se podrán seleccionar uno o varios proyectos para exportarlos. Seguirá pudiendo ampliar los proyectos para ver carpetas y recursos, si bien sólo podrá seleccionar en el nivel de proyecto. Incluir Dependencias no se puede seleccionar.

Puede exportar proyectos y recursos tanto si está trabajando desde dentro de una sesión como desde fuera. Si exporta desde dentro de una sesión, los recursos serán recursos de la sesión y la configuración podría quedar incompleta o presentar conflictos. Si exporta desde fuera de una sesión, los recursos serán recursos de núcleo activados.

Oracle Service Bus no puede exportar usuarios, grupos o roles que se hayan creado en el módulo Configuración de Seguridad de la consola de administración de Oracle Service Bus. Tampoco puede exportar asignaciones de credenciales u otros datos de proveedor de seguridad que se hayan creado en la consola de administración de WebLogic Server. En su lugar, utilice la consola de administración de WebLogic Server para exportar estos datos. Consulte la sección sobre migración de datos de seguridad en Oracle Fusion Middleware Securing Oracle WebLogic Server (Protección de Oracle Fusion Middleware para Oracle WebLogic Server) en http://www.oracle.com/pls/as111170/lookup?id=SECMG319.

  1. Seleccione Administración del Sistema > Exportar Recursos.

    2. La página Exportar Recursos muestra la lista de objetos presentes en la configuración. Se verá el nombre, el tipo, la hora de la última modificación y las referencias de cada objeto.

    En cuanto a los servicios, el registro de la hora de la última modificación refleja qué modificaciones se han realizado en la definición del servicio, así como en su transporte o en la configuración del pipeline. No obstante, no refleja los cambios realizados en las alertas definidas para el servicio en cuestión.

  2. Seleccione Exportar Proyectos o Exportar Recursos.
    3. Nota: Exportar proyectos puede hacer que se supriman recursos al importar el archivo JAR del proyecto completo. Para obtener más información, consulte Importación de Recursos.
  3. Seleccione los proyectos o los recursos que desee exportar. Si ha seleccionado Exportar Proyectos, no podrá seleccionar casillas de control de recursos individuales.
    1. Amplíe las carpetas de proyecto. Verá el nombre, el tipo, la hora de la última modificación y las referencias de cada recurso incluido en el proyecto.
    2. Seleccione las casillas de control asociadas con los proyectos o los recursos que desee exportar.
    3. Desactive las casillas de control asociadas con los proyectos o los recursos que no desee exportar.
  4. Si ha seleccionado Exportar Recursos, active o desactive la casilla de control Incluir Dependencias. Utilice esta opción para exportar cualquier otro recurso al que haga referencia este recurso.
  5. Haga clic en Exportar para crear un archivo JAR y exportarlo.

Después de Terminar

Si procede, continúe con Especificación de la Configuración de Seguridad en la Exportación.
