Búsqueda y Sustitución de Valores de Entorno

Utilice la página Buscar y Sustituir para buscar los valores de entorno que difieren entre los dominios. Los valores de entorno son determinados campos predefinidos en los datos de configuración cuyos valores es muy probable que cambien cuando se mueve la configuración de un dominio a otro (por ejemplo, de prueba a producción). Los valores de entorno representan entidades como URL, URI, nombres de archivo y directorio, nombres de servidor, correos electrónicos, etcétera. Además, los valores de entorno se pueden encontrar en destinos de alerta, servicios proxy, servicios de negocio, recursos de servidor SMTP y proveedor de JNDI, y entradas de registro UDDI.

Esta página funciona de forma diferente según se esté o no dentro de una sesión. Si está en una sesión, podrá buscar y sustituir valores de entorno. No obstante, si está fuera de una sesión, sólo podrá buscar valores de entorno; la opción Sustituir Todo aparecerá desactivada.

  1. Si aún no lo ha hecho, haga clic en Crear para crear una nueva sesión o en Editar para acceder a una sesión existente. Consulte Uso del Centro de Cambios.
  2. Seleccione Administración del Sistema > Buscar y Sustituir.
  3. En el campo Buscar Valor, introduzca el valor de entorno que desee localizar.
  4. Para mostrar una lista de valores de entorno en la configuración que contenga el valor introducido, haga clic en Buscar.

    5. Aparece una lista con los valores relevantes, con las columnas Propietario, Tipo y Valor de Entorno.

  5. Para localizar sólo los elementos que hayan cambiado en la sesión actual, active la casilla de control Sólo Elementos Cambiados en la Sesión Actual.
  6. Seleccione el tipo de variable que quiera buscar en la lista desplegable Buscar en Tipo de Variable, tal y como se describe en la Tabla 31-1.

  7. Para localizar valores de entorno que se encuentran en un proyecto concreto, seleccione el nombre del proyecto en la lista desplegable Ubicado en el Proyecto.
  8. En el campo Sustituir por, introduzca el nuevo valor de entorno.
  9. Para sustituir el valor de entorno original con el nuevo valor, haga clic en Sustituir Todo.

    10. Todas las apariciones del valor de entorno que se haya introducido en el campo Buscar Valor se sustituirán con el valor de entorno que se haya introducido en el campo Sustituir por en la sesión actual.

    Nota: Algunos valores de entorno son objetos XML complejos que no se pueden encontrar y sustituir con la consola de administración de Oracle Service Bus. A pesar de ello, podrá definir estos valores de entorno directamente mediante ALSBConfigurationMBean desde un script. Para obtener información detallada acerca de ALSBConfigurationMBean, consulte el documento de referencia de la API de Oracle Service Bus. Además de definir estos valores a través de la API, podrá definir valores de entorno de tipo complejo mediante archivos de personalización. Consulte Ejecución de Archivos de Personalización.
  10. Para terminar la sesión y desplegar la configuración en tiempo de ejecución, haga clic en Activar en Centro de Cambios.
