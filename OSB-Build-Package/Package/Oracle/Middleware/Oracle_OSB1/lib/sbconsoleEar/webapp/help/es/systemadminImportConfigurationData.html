Importación de Recursos

Utilice la página Importar Recursos para importar un archivo JAR de proyecto íntegro o un archivo JAR de recurso que se haya exportado previamente de otro dominio de Oracle Service Bus. Puede elegir importar sólo un subjuego de los datos exportados. Si el recurso ya existe en el sistema de importación, se actualizará. Si no, se creará. Los recursos sólo se programan para suprimirse cuando el recurso JAR que se está importando es un JAR de proyecto completo y cuando hay recursos en el mismo proyecto dentro del sistema de importación que no están presenten en el archivo JAR importado. No se suprimirán los recursos que se encuentren en otros proyectos.

Puede importar recursos en el nivel de recurso aunque se exportaran en el nivel de proyecto. Por ejemplo, incluso si los recursos del sistema se exportaran en un archivo JAR de proyecto completo, podrá anular su selección al importarlos.

No se pueden exportar usuarios, grupos, roles o certificados cuando se exporta una configuración. Así pues, deberá crear estos objetos de nuevo cuando importe una configuración exportada.

Nota: Para obtener información acerca de cómo importar recursos que no formen parte de una configuración de Oracle Service Bus, consulte Carga de Recursos de un Archivo Zip y Carga de Recursos de una Dirección URL.
  1. Si aún no lo ha hecho, haga clic en Crear para crear una nueva sesión o en Editar para acceder a una sesión existente. Consulte Uso del Centro de Cambios.
  2. Seleccione Administración del Sistema > Importar Recursos.
  3. Seleccione un archivo JAR de recurso o un proyecto exportado.
    1. En el campo Nombre de Archivo, haga clic en Examinar para localizar el directorio en el que está almacenado el archivo JAR.
    2. Seleccione el archivo JAR y, a continuación, haga clic en Abrir.
  4. Haga clic en Siguiente. Continúe con Selección de Proyectos y Recursos que Importar.
