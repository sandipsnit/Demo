Especificación de la Configuración de Seguridad en la Importación

Si el archivo JAR contiene recursos con datos de nombre de usuario y contraseña que se cifraron al exportarse, aparecerá la páginaImportar Recursos: Configuración de Seguridad.

  1. Introduzca la contraseña que se utilizó para cifrar los datos (consulte Exportación de Recursos).

    2. Si desconoce la contraseña que se utilizó para cifrar los datos del recurso, haga clic en el botón Anterior y desactive la casilla de control que hay junto al recurso. Podrá importar cualquier otro recurso no cifrado sin saber la contraseña.

  2. Haga clic en Terminar Importación.

    3. Aparece la ventana emergente Procesando Datos de Configuración. Espere a que esta ventana se cierre antes de realizar ninguna otra acción en la consola de administración de Oracle Service Bus. La ventana se cerrará automáticamente cuando el sistema termine de importar los objetos. Aparecerá la página Importar Recursos, donde podrá apreciar un mensaje indicando que se han desplegado correctamente los datos.
