Importación de Cuentas de Servicio o Proveedores de Claves de Servicio

Nota: En las versiones anteriores de Oracle Service Bus, los proveedores de claves de servicio se denominaban proveedores de servicios de proxy.

Si el archivo JAR se creó con AquaLogic Service Bus 3.0 o posterior y contiene cuentas de servicio o proveedores de claves de servicio, podrá importar estos recursos, junto con los nombres de usuario, las contraseñas, las asignaciones de usuario local a usuario remoto y los alias de enlaces de par de claves que contengan.

Para cada proveedor de claves de servicio, Oracle Service Bus importa el alias de enlace de par de claves al proveedor de asignaciones de credenciales PKI. Si estos datos estaban cifrados en el momento de exportarse, deberá proporcionar la contraseña que se utilizó para cifrarlos. Si desconoce la contraseña, podrá importar el resto de los recursos no cifrados.

Si importa una cuenta de servicio o un proveedor de claves de servicio y en el dominio ya existe un recurso con el mismo nombre, el recurso importado sobrescribirá el que ya existe en el dominio, incluso aunque el que ya está presente en el dominio se haya modificado durante la sesión actual, a menos que especifique que se mantenga la configuración de seguridad en el proceso de importación. Consulte el paso 5 en Selección de Proyectos y Recursos que Importar.
