Adición de Servidores Proxy

Utilice la página Resumen de Servidores Proxy para agregar y configurar recursos de servidor proxy y hacer que queden disponibles en Oracle Service Bus como un recurso del sistema. Deberá encontrarse en una sesión activa para poder configurar o volver a configurar recursos de servidor proxy.

  1. Si aún no lo ha hecho, haga clic en Crear para crear una nueva sesión o en Editar para acceder a una sesión existente. Consulte Uso del Centro de Cambios.
  2. Seleccione Administración del Sistema > Servidores Proxy.
  3. Haga clic en Agregar.
  4. En el campo Nombre, introduzca un nombre para el recurso del servidor proxy. Es un campo requerido.

    Siga las Restricciones de Nomenclatura de Recursos para obtener ayuda con la nomenclatura.
  5. En el campo Descripción, introduzca una descripción corta para el recurso de servidor proxy.
  6. En la sección Parámetros de Host-Port, introduzca la siguiente información:
    1. En el campo Host de Servidor, introduzca el nombre del host o la dirección IP del servidor proxy. Este campo es de inclusión obligatoria.

      2. El nombre de host de servidor del servidor proxy de Oracle Service Bus debe ser idéntico al nombre de host de servidor del servidor proxy real.

    2. En el campo Puerto sin Cifrado, introduzca el número de puerto sin cifrar del servidor proxy.
    3. En el campo Puerto SSL, introduzca el número de puerto SSL del servidor proxy. Debe introducir texto sin cifrar o un número de puerto SSL.
    4. Haga clic en Agregar.

      5. Puede configurar varios servidores proxy para cada recurso de servidor proxy. Esto permite a Oracle Service Bus realizar un equilibrio de carga y ofrecer función de tolerancia ante fallos para el recurso de servidor proxy.

  7. Si el servidor proxy realiza una autenticación de proxy, introduzca un nombre de usuario en el campo Usuario y la contraseña asociada en los campos Contraseña y Confirmar Contraseña.

    8. Estos campos son opcionales y sólo son de inclusión obligatoria cuando el servidor proxy está protegido.

  8. Haga clic en Guardar para crear y guardar el recurso de servidor proxy en la sesión actual.
  9. Para terminar la sesión y desplegar la configuración en tiempo de ejecución, haga clic en Activar en Centro de Cambios.
