Supresión de Servidores Proxy

  1. Si aún no lo ha hecho, haga clic en Crear para crear una nueva sesión o en Editar para acceder a una sesión existente. Consulte Uso del Centro de Cambios.
  2. Seleccione Administración del Sistema > Servidores Proxy.
  3. En la página Resumen de Servidores Proxy, seleccione un servidor proxy. Consulte Localización de Servidores SMTP.
  4. Haga clic en el icono de Suprimir en el campo Opciones del recurso de servidor proxy que desee suprimir. El recurso de servidor proxy se suprimirá en la sección actual. Si hubiera otros recursos que hicieran referencia a este recurso, aparecerá un icono de advertencia de supresión. Puede suprimir el recurso confirmando la advertencia. En este caso, se podrían generar conflictos por la presencia de referencias no resueltas al recurso suprimido.
  5. Para terminar la sesión y desplegar la configuración en tiempo de ejecución, haga clic en Activar en Centro de Cambios.
