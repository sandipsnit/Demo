Visualización de Registros UDDI Configurados

Utilice esta página para designar un registro UDDI por defecto (consulte Establecimiento de una Configuración de UDDI por Defecto) y ver los registros UDDI. Además, podrá buscar registros que haya configurado antes.

  1. En el panel Registros UDDI, introduzca el nombre del registro que quiera encontrar en el campo Nombre, o bien introduzca caracteres comodín; utilice * en el lugar de varios caracteres y ? en el lugar un solo carácter. La búsqueda devolverá todos los registros que cumplan con los criterios especificados para el nombre.
  2. Haga clic en Buscar.
  3. Haga clic en Ver Todo para eliminar los filtros de búsqueda y ver todos los registros configurados para Oracle Service Bus.

    4. La Tabla 29-1 muestra valores de configuración de los registros UDDI.

    Tabla 29-1 Valores de Configuración de Registros UDDI   
    Propiedad
    Descripción
    Nombre del Registro
    El nombre del registro. El nombre asignado a un registro en el momento de crearse. No podrá editar el nombre de los registros una vez guardados.
    URL de Consulta
    La URL del punto final de la API de consulta que se utiliza para localizar e importar servicios.
    URL de Publicación
    La URL del punto final de la API de publicación que se utiliza para publicar servicios.
    URL de Seguridad
    La URL del punto final de la API de seguridad que se utiliza para obtener un elemento de autenticación para poder publicar en el registro.
    URL de Suscripción
    La URL del punto final de la API de suscripción que se utiliza para suscribirse a cambios en el registro, para crear suscripciones al registro y para recibir cambios en servicios que se hayan importado.
    Nombre de Usuario
    El nombre de usuario necesario para autenticar al usuario.
    Contraseña (Confirmar Contraseña)
    La contraseña necesaria para autenticar al usuario.
    Cargar tModels en el Registro
    Carga los tModels en el registro seleccionado. Esta opción sólo se tiene que seleccionar una vez por registro.
    Activar Importación Automática
    Sincroniza automáticamente servicios con el registro UDDI.
