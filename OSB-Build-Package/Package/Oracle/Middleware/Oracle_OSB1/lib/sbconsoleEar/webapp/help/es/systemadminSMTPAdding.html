Adición de Servidores SMTP

Utilice la página Resumen de Servidores SMTP para agregar y configurar recursos de servidor SMTP y hacer que queden disponibles en Oracle Service Bus como un recurso del sistema. Deberá encontrarse en una sesión activa para poder configurar o volver a configurar recursos de servidor SMTP.

  1. Si aún no lo ha hecho, haga clic en Crear para crear una nueva sesión o en Editar para acceder a una sesión existente. Consulte Uso del Centro de Cambios.
  2. Seleccione Administración del Sistema > Servidores SMTP.
  3. Haga clic en Agregar.
  4. En el campo Nombre, introduzca un nombre para el recurso del servidor SMTP. Es un campo requerido.

    Siga las Restricciones de Nomenclatura de Recursos para obtener ayuda con la nomenclatura.
  5. En el campo Descripción, introduzca una descripción corta para el recurso de servidor SMTP.
  6. En el campo URL del Servidor, introduzca la URL que apunta al servidor SMTP Server. Este campo es de inclusión obligatoria.
  7. En el campo Número de Puerto, introduzca un número de puerto para el servidor SMTP (el puerto por defecto es 25). Este campo es de inclusión obligatoria.
  8. Si para acceder al servidor SMTP de destino hace falta indicar un nombre de usuario y una contraseña, escriba un nombre de usuario en el campo Usuario y la contraseña asociada en los campos Contraseña y Confirmar Contraseña.

    9. Estos campos son opcionales y sólo son de inclusión obligatoria cuando el servidor SMTP está protegido.

  9. Haga clic en Guardar para crear y guardar el recurso de servidor SMTP en la sesión actual.
  10. Para terminar la sesión y desplegar la configuración en tiempo de ejecución, haga clic en Activar en Centro de Cambios.
