Configuración de un Servidor SMTP por Defecto

Puede diseñar uno de los servidores SMTP configurados como el servidor por defecto para el dominio. Consulte Adición de Servidores SMTP.

  1. Si aún no lo ha hecho, haga clic en Crear para crear una nueva sesión o en Editar para acceder a una sesión existente. Consulte Uso del Centro de Cambios.
  2. Seleccione Administración del Sistema > Servidores SMTP.
  3. En el panel Servidor SMTP por Defecto, haga clic en Seleccionar Valor por Defecto.
  4. Desde la lista desplegable Servidor SMTP por Defecto, seleccione un servidor SMTP.
  5. Haga clic en Definir Valor por Defecto.

    6. Aparece la página Resumen de Servidores SMTP con el servidor SMTP seleccionado como el servidor SMTP por defecto para el dominio.

  6. Para terminar la sesión y desplegar la configuración en tiempo de ejecución, haga clic en Activar en Centro de Cambios.
