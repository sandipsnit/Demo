Visualización de Proveedores de JNDI

Utilice esta página para ver una lista de proveedores de JNDI, junto con la información que se ve en la Tabla 30-1.

También podrá buscar proveedores de JNDI que haya configurado antes.

  1. Para limitar el número de elementos de la lista o para buscar proveedores de JNDI concretos:
    • Haga clic en el icono Abrir para mostrar filtros de búsqueda adicionales. Si especifica varios criterios de búsqueda, sólo se devolverán los resultados que coincidan con todos los criterios.
    • Introduzca el nombre del proveedor de JNDI que quiera encontrar en el campo Nombre y, a continuación, haga clic en Buscar para buscar una entrada concreta.

      • Puede introducir caracteres comodín (? en el lugar de un único carácter; * en el lugar de varios caracteres) para realizar búsquedas más generales.

    • Introduzca el valor de URL de Proveedor correspondiente al proveedor de JNDI que quiera encontrar y, a continuación, haga clic en Buscar.
  2. Haga clic en Ver Todo para eliminar los filtros de búsqueda y ver todos los recursos de proveedor de JNDI configurados para Oracle Service Bus.

    3. La Tabla 30-1 muestra valores de configuración de los registros UDDI.

    Tabla 30-1 Información de Proveedores de JNDI
    Propiedad
    Descripción
    Nombre
    El nombre único asignado a este proveedor de JNDI. El nombre funciona como un enlace a la página Ver Proveedor de JNDI - Configuración. Consulte Edición de Proveedores de JNDI.
    URL de Proveedor
    La URL del proveedor de JNDI. Consulte Adición de Proveedores de JNDI.
    Caché de JNDI
    Activado o Desactivado. Cuando está activado, el contexto de JNDI y los objetos de JNDI se almacenan en la caché local, lo cual mejora el rendimiento al buscar objetos. Oracle recomienda dejar activada la caché de JNDI.
    Acciones
    • Haga clic en el icono de Restablecer para restablecer el contexto JNDI y desechar la conexión JNDI y los objetos almacenados en la caché local.
    • Haga clic en el icono de Iniciar Consola de Prueba para probar si el proveedor de JNDI puede establecer correctamente una conexión y obtener un contexto JNDI.
    Opciones
    • Haga clic en el icono de Cambiar Nombre para cambiar el nombre del proveedor de JNDI.
    • Haga clic en el icono de Suprimir para suprimir un proveedor de JNDI específico. Consulte Supresión de Proveedores de JNDI.
