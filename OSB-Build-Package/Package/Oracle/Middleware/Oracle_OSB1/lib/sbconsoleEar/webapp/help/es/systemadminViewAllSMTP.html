Visualización de Servidores SMTP

Utilice esta página para designar un servidor SMTP por defecto (consulte Configuración de un Servidor SMTP por Defecto) y para ver una lista de servidores SMTP.

Los recursos de servidor SMTP se utilizan cuando se están configurando recursos de destino de alertas y servicios de negocio basados en transporte de correo electrónico. Consulte Adición de Destinos de Alerta y Página Crear/Editar un Servicio de Negocio - Configuración de Transporte de Correo Electrónico.

La Tabla 30-3 muestra información de los servidores SMTP.

Tabla 30-3 Información de Servidores SMTP
Propiedad
Descripción
Nombre
El nombre único asignado a este recurso de servidor SMTP. El nombre funciona como un enlace a la página Ver Servidor SMTP - Configuración. Consulte Edición de Servidores SMTP.
URL del Servidor
La URL que apunta al servidor SMTP. Consulte Adición de Servidores SMTP.
Opciones
  • Haga clic en el icono de Cambiar Nombre para cambiar el nombre del recurso de servidor SMTP.
  • Haga clic en el icono de Suprimir para suprimir un servidor SMTP concreto. Consulte Supresión de Servidores SMTP.
