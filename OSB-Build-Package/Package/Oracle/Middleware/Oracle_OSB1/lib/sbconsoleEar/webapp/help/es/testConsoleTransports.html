Uso de la Configuración de Transporte por Parte del Tiempo de Ejecución en la Consola de Prueba

Configuración de Datos de Prueba de Servicios de Proxy y Configuración de Datos de Prueba de Servicios de Negocio describen cómo configurar los valores de las cabeceras de transporte, los metadatos de transporte y los datos de seguridad relacionados con el transporte para las solicitudes salientes al realizar pruebas de servicios de proxy o de negocio en la consola de prueba. Debe tener en cuenta que algunas de las especificaciones que se hacen en la consola de prueba no se aplican en el tiempo de ejecución. Esto implica que los valores de algunas cabeceras o metadatos se sobrescribirán o que Oracle Service Bus no los tendrá en cuenta en el tiempo de ejecución cuando se ejecute la prueba. Las cabeceras y los metadatos sujetos a limitaciones se describen en la Tabla 32-10.
