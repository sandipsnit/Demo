Realización de Prueba de XQuery

Puede editar y probar una acción en el flujo de mensajes mediante los siguientes editores: Editor de Expresiones XQuery, Editor de Condición de XQuery y Editor de Expresiones XPath. La prueba utiliza el mismo formulario tanto para el Editor de Expresiones como para el Editor de Condiciones XQuery. Sin embargo, el caso es diferente para el Editor de Expresiones XPath porque sólo requiere una entrada.

Nota: Debe desactivar los bloqueos de ventanas emergentes en su explorador para que funcione la prueba de XQuery. Si dispone de barras de herramientas en el explorador Internet Explorer, puede que necesite desactivar los bloqueos de ventanas emergentes desde el menú Opciones, así como para todas las barras de herramientas que estén configuradas para bloquearlas.
