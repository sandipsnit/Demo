Visualización de Resultados de la Prueba de Servicios de Proxy

En la Tabla 32-2 se describen los resultados de la prueba de servicios de proxy. El rastreo sólo está activado si ha seleccionado tanto la opción Llamada Directa como Incluir Rastreo.

Tabla 32-2 Resultados de la Prueba de Servicios de Proxy 
Sección
Descripción
Nombre del Servicio de Proxy
Nombre del servicio de proxy que se está probando.
Acciones de la Consola de Prueba
Atrás muestra la página anterior del navegador.
Cerrar cierra la consola de prueba.
Documento de Solicitud
Mensaje de solicitud enviado al servicio de proxy por la consola de prueba.
Esta sección está reducida inicialmente si la consola de prueba no modificó el mensaje de solicitud. Esta sección está ampliada inicialmente para los servicios SOAP configurados mediante el separador Pantalla o si se ha aplicado WS-Security.
Si se ha aplicado WS-Security, esta sección incluirá dos mensajes de SOAP: El primer mensaje es un mensaje no cifrado; el segundo es el mensaje de SOAP seguro.
Documento de Respuesta
Respuesta al mensaje.
Para un servicio SOAP con una respuesta WS-Security, esta sección contiene dos mensajes de SOAP. El primer mensajes es el mensaje seguro que recibe el cliente. El segundo mensaje de SOAP corresponde al mensaje no cifrado.
Metadatos de Respuesta
Metadatos devueltos con la respuesta al mensaje.
Rastreo de Servicios de Proxy
El rastreo muestra el estado del mensaje cuando se transfiere a través del sistema. Cuando no está seleccionada la opción Llamada Directa, no se realiza el rastreo. Para obtener más información sobre el rastreo, consulte Rastreo de Servicios de Proxy.
