Acciones de la Consola de Prueba
Haga clic en Atrás para mostrar la página anterior del explorador.
Haga clic en Cerrar para cerrar la ventana de la consola de prueba.
Documento de Solicitud
Mensaje de solicitud enviado al servicio de negocio por la consola de prueba.
Esta sección está reducida inicialmente si la consola de prueba no modificó el mensaje de solicitud. Esta sección está ampliada inicialmente para los servicios SOAP configurados mediante el separador Pantalla o si se ha aplicado WS-Security.
Si se ha aplicado WS-Security, esta sección incluirá dos mensajes de SOAP. El primer mensaje es el mensaje texto no cifrado. El segundo es el mensaje de SOAP seguro.
Documento de Respuesta
Respuesta al mensaje.
Para un servicio SOAP con una respuesta WS-Security, esta sección incluirá dos mensajes de SOAP. El primer mensajes es el mensaje seguro tal y como lo recibe el cliente. El segundo mensaje de SOAP corresponde al mensaje no cifrado.
Metadatos de Respuesta
Metadatos devueltos con la respuesta al mensaje.

Nota: El mensaje de SOAP seguro aparece impreso con sangría; por ejemplo, con espacios en blanco adicionales. Este mensaje de SOAP no puede utilizarse siempre como información literal, ya que los espacios en blanco pueden afectar a la semántica del documento. Por ejemplo, las firmas digitales son sensibles a los espacios en blanco y pueden resultar no válidas.
