Prueba de transformaciones de MFL

Un documento MFL (Lenguaje de Formato de Mensajes) es un documento XML especializado que se utiliza para describir el formato de los datos binarios. Los recursos MFL soportan dos transformaciones. Cada transformación acepta sólo una entrada y proporciona una sola salida: De XML a binario; o de binario a XML.

En la Tabla 32-5 se describen las opciones de configuración de MFL.

Configuración de Transformación de MFL
Entrada XML
Necesaria para transformaciones de XML a binario:
El esquema XML se puede inferir para el documento MFL. Automáticamente se introduce un documento XML de ejemplo en el campo de texto.
La entrada XML puede basarse en archivo o en texto. La referencia a un archivo para la entrada tiene prioridad sobre la entrada de texto. Busque y seleccione el archivo que desee utilizar en la prueba.
Entrada Binaria
Necesaria para transformaciones de binario a XML:
La entrada binaria puede basarse en archivo o en texto. La referencia a un archivo para la entrada tiene prioridad sobre la entrada de texto. Busque y seleccione el archivo que desee utilizar en la prueba.

Puede probar el tiempo de diseño o el tiempo de ejecución.

  1. Haga clic en Activar si desea probar el tiempo de ejecución. No active la sesión para probar el tiempo de diseño.
  2. Seleccione Explorador de Recursos > MFL para mostrar la página Resumen de Archivos MFL.
  3. En Acciones, haga clic en el icono Iniciar Consola de Prueba asociado al recurso que desee probar. La consola de prueba abre la página Prueba de Recursos.
  4. Seleccione las transformaciones que desee probar.
  5. Configure los datos de prueba para los recursos de MFL. Para obtener más información, consulte la Tabla 32-5.
  6. Haga clic en Ejecutar. La página Prueba de Recursos mostrará los resultados.
  7. Para repetir la prueba, haga clic en Atrás. Puede cerrar la consola de prueba, modificar y volver a probar el recurso.
