Prueba de Transformaciones de XQuery

Los mapas de XQuery pueden describir asignaciones de XML a XML, de XML a no XML y de no XML a XML. Una transformación de XQuery puede tomar varias entradas y devolver una salida. Cada entrada corresponde a una variable externa de XQuery declarada en el recurso de XQuery. El valor de una variable de entrada de XQuery puede ser un valor primitivo (cadena, entero, fecha, etc.), un documento XML o una secuencia de estos tipos. El valor de salida puede ser un valor primitivo (cadena, entero, fecha, etc.) un documento XML o una secuencia de estos tipos.

Nota: la consola de prueba no soporta secuencias en la entrada.

Variables
Esta sección contiene un campo de entrada para cada una de las variables externas de XQuery.
<nombre_parámetro> ([] como XML)
nombre_parámetro es un nombre de variable de XQuery en el recurso de XQuery.
En la consola de prueba aparece un cuadro de edición de una sola línea si es de tipo simple. Si los datos son XML aparece un cuadro de edición de varias líneas.
Se utiliza una entrada combinada (<nombre_parámetro> ([] como XML)) cuando es una variable sin tipo. Debe declarar el tipo de variable. Seleccione la casilla de control para identificar un parámetro de tipo XML.
Una entrada XML puede basarse en archivo o en texto. La referencia a un archivo para la entrada tiene prioridad sobre la entrada de texto. Busque y seleccione el archivo que desee utilizar en la prueba.
La entrada en la consola de prueba se presenta según el tipo para facilitar la comprensión del tipo de dato que debe introducir. Cuando es sin tipo, el tipo por defecto es XML.

Puede probar el tiempo de diseño o el tiempo de ejecución.

  1. Haga clic en Activar si desea probar el tiempo de ejecución. No active la sesión para probar el tiempo de diseño.
  2. Seleccione Explorador de Proyectos > XQueries para mostrar la página Resumen de XQueries.
  3. En Acciones, haga clic en el icono Iniciar Consola de Prueba asociado al recurso que desee probar. La consola de prueba abre la página Prueba de Recursos.
  4. Configure los datos de prueba para el recurso en la sección Variables de la página. Para obtener más información, consulte la Tabla 32-7.
  5. Haga clic en Ejecutar. La página Prueba de Recursos mostrará los resultados.
  6. Para repetir la prueba, haga clic en Atrás. Puede cerrar la consola de prueba, modificar y volver a probar el recurso.
