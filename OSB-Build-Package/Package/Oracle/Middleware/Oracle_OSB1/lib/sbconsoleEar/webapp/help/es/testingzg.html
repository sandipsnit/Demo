Uso del Editor de Expresiones XPath

Utilice expresiones XPath para seleccionar un subjuego de una variable de contexto de mensaje XML. Puede utilizar la consola de prueba del Editor de Expresiones XPath para probar la definición de la expresión XPath. La expresión XPath toma un solo documento XML como entrada y genera como salida una secuencia de documentos XML, tipos primitivos o ambos.

En la Tabla 32-9 se describen las opciones de configuración de las expresiones XPath.

Variables
Esta sección contiene un solo campo de entrada que corresponde al documento XML en el que se prueba esta expresión XPath.
 
La entrada XML puede basarse en archivo o en texto. La referencia a un archivo para la entrada tiene prioridad sobre la entrada de texto. Busque y seleccione el archivo que desee utilizar en la prueba.

  1. Acceda a la consola de prueba cuando edite una acción en el flujo de mensajes de un pipeline. Para acceder al Editor de Expresiones XPath, consulte Creación y Edición de Expresiones XQuery y XPath en Línea.
  2. Configure los datos de prueba para la expresión XPath en la sección Variables de la página. Para obtener más información, consulte la Tabla 32-9.
  3. Haga clic en Ejecutar. La página de prueba muestra los resultados.
  4. Una vez completada una prueba, puede utilizar el botón Atrás para ejecutar una nueva prueba. Para ejecutar otra prueba después de hacer cambios en la expresión XPath, debe cerrar y volver a abrir la consola de prueba para que se apliquen los cambios.
