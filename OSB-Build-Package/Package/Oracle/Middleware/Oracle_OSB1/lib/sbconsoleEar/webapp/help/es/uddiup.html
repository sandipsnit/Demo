Edición de Registros UDDI

  1. Si aún no lo ha hecho, haga clic en Crear para crear una nueva sesión o en Editar para acceder a una sesión existente. Consulte Uso del Centro de Cambios.
  2. Seleccione Administración del Sistema > Registros UDDI.
  3. En el panel Registros UDDI, en la lista de registros disponibles, haga clic en el nombre del registro que desea editar.

    4. También puede buscar un registro concreto utilizando la opción Buscar. Consulte Visualización de Registros UDDI Configurados.

    La página Configuración de Registro UDDI muestra las propiedades de configuración que definen el registro y la siguiente información específica del registro, como se muestra en la Tabla 29-2.

    Tabla 29-2 Información de Registro UDDI 
    Propiedad
    Descripción
    Modificado por
    Usuario que realizó el último cambio de configuración en este recurso de registro UDDI
    Fecha de Modificación
    Fecha y hora a la que se realizó el último cambio de configuración. Haga clic en el enlace de fecha y hora para ver el historial de cambios de este recurso. Consulte Página Ver Historial de Cambios.
    Referencias
    Número de objetos a los que hace referencia este registro UDDI. Si existen estas referencias, haga clic en el enlace numérico para ver una lista de las referencias. Consulte Visualización de Referencias a Recursos.
    Con Referencia de
    Número de objetos que hacen referencia a este registro UDDI. Si existen estas referencias, haga clic en el enlace numérico para ver una lista de los objetos. Consulte Visualización de Referencias a Recursos.
    Descripción
    Descripción del registro.

  4. En la parte inferior de la página, haga clic en Editar.
  5. En la página Configuración de UDDI-Editar Registro, edite los parámetros de configuración del registro.

    6. Estas propiedades se describen en la Tabla 29-1. No puede editar el nombre del registro.

  6. Haga clic en Guardar para confirmar las actualizaciones en la sesión actual.
  7. Para terminar la sesión y desplegar la configuración en tiempo de ejecución, haga clic en Activar en Centro de Cambios.
