WSDL

Un WSDL (Lenguaje de Definición de Servicio Web) es la descripción formal de un servicio web; en Oracle Service Bus, describe un servicio de proxy o un servicio de negocio. Se utiliza un WSDL para describir lo que puede hacer un servicio web, dónde reside y cómo llamarlo.

Puede basar los servicios SOAP y XML en un recurso WSDL existente. Hay disponible un documento WSDL para servicios de proxy y negocio de cualquier transporte. Consulte Crear/Editar un Servicio de Proxy: Página Configuración General y Creación y Configuración de Servicios de Negocio.

Para obtener más información, consulte la sección sobre cómo utilizar WSDL en Oracle Service Bus en Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus (Guía del Administrador de Oracle Fusion Middleware para Oracle Service Bus) en http://www.oracle.com/pls/as111170/lookup?id=OSBAG114.
