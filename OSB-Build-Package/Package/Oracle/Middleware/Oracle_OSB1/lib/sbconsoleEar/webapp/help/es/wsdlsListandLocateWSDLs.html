Localización de WSDL

  1. Realice una de las siguientes acciones:
    • Seleccione Explorador de Proyectos para que aparezca la página Vista de Proyectos o la página Vista de Proyectos/Carpetas. A continuación, navegue por los proyectos y carpetas para buscar el WSDL.
    • Seleccione Explorador de Recursos > WSDL. La página Resumen de WSDL muestra la información que aparece en la Tabla 6-1. Para obtener una descripción más detallada de las propiedades, consulte Edición de WSDL.
  2. Para localizar un WSDL concreto:
    • Haga clic en el icono Abrir para mostrar filtros de búsqueda adicionales. Si especifica varios criterios de búsqueda, sólo se devolverán los resultados que coincidan con todos los criterios.
    • Para limitar el número de elementos de la lista, puede filtrar por nombre, ruta de acceso y espacio de nombres. En los campos Nombre, Ruta de Acceso y Espacio de Nombres, introduzca el nombre, la ruta de acceso y el espacio de nombres de los destinos de búsqueda y haga clic en Buscar.

      • La ruta de acceso es el nombre del proyecto y el nombre de la carpeta donde reside el WSDL.

    • Haga clic en Ver Todo para eliminar los filtros de búsqueda y mostrar todos los WSDL.

      • Tabla 6-1 Detalles de WSDL
      Propiedad
      Descripción
      Nombre de WSDL
      Nombre único asignado al WSDL. El nombre es un enlace a la página Ver Detalles de WSDL. Consulte Edición de WSDL.
      Ruta de Acceso
      La ruta de acceso es el nombre del proyecto y de la carpeta donde reside el WSDL. Es un enlace al proyecto o a la carpeta que contiene este recurso. Consulte Cualificación de Nombres de Recurso con Proyectos y Carpetas.
      Espacio de Nombres de WSDL
      Dirección del WSDL. Se muestra con el siguiente formato:
      http://ejemplo.com/ejemplo/ejemplo
      Opciones
      Haga clic en el icono Suprimir para suprimir un WSDL concreto. Consulte Supresión de WSDL.
