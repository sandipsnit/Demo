Edición de WSDL

  1. Si aún no lo ha hecho, haga clic en Crear para crear una nueva sesión o en Editar para acceder a una sesión existente. Consulte Uso del Centro de Cambios.
  2. Localice el WSDL, tal y como se describe en Localización de WSDL.
  3. Haga clic en el nombre del WSDL. La página Ver Detalles de WSDL muestra la información que aparece en la Tabla 6-2, Tabla 6-3 y Tabla 6-4.

    4. Tabla 6-2 Detalles de WSDL 
    Propiedad
    Descripción
    Modificado por
    Usuario que ha creado este WSDL o lo ha importado a la configuración.
    Fecha de Modificación
    Fecha y hora a la que el usuario ha creado este WSDL o lo ha importado a la configuración. Haga clic en el enlace de fecha y hora para ver el historial de cambios de este recurso. Consulte Página Ver Historial de Cambios.
    Referencias
    Número de objetos a los que hace referencia este WSDL. Si existen estas referencias, haga clic en el enlace numérico para ver una lista de los objetos. Consulte Visualización de Referencias a Recursos.
    Con Referencia de
    Número de objetos que hacen referencia a este WSDL. Si existen estas referencias, haga clic en el enlace numérico para ver una lista de los objetos. Por ejemplo, si selecciona este puerto o enlace de WSDL como tipo de servicio para un servicio de negocio o de proxy concreto, el servicio de negocio o de proxy se enumera como referencia cuando se hace clic en el enlace. Consulte Visualización de Referencias a Recursos.
    Descripción
    Descripción de este WSDL, si existe.

    Se muestra por defecto la vista estructural del WSDL, que incluye la información que aparece en la Tabla 6-3.

    Tabla 6-3 Detalles de la Estructura de WSDL 
    Propiedad
    Descripción
    Espacio de Nombres de Destino
    Espacio de nombres utilizado para cualificar cualquiera de las definiciones que se incluyen en el WSDL.
    Definiciones de WSDL
    Atributos y grupos asociados al WSDL:
    • Tipos de Puerto de WSDL
    • Enlaces de WSDL
    • Puertos de WSDL
    • Tipos de Esquemas XML
    • Elementos de Esquema XML
    • Importaciones de WSDL
    • Importaciones de Esquemas XML
    • Referencias de Políticas de Servicios Web
    • Referencias de Políticas de Servicios Web Implícitas
    Estado de WSDL
    Estado del WSDL:
    • Válido

      • Todas las ubicaciones de los esquemas XML o los WSDL incluidos por el WSDL actual están especificadas y son válidas. Asimismo, todas las ubicaciones de los esquemas XML o los WSDL anidados (los esquemas XML o los WSDL incluidos posteriormente por los WSDL o esquemas XML incluidos) están especificadas y son válidas.

    • No Válido

      • Una o varias de las ubicaciones de los esquemas XML o los WSDL incluidos por el WSDL actual no se han especificado o no son válidas. Además, es posible que no se haya especificado o no sea válida la ubicación de uno o varios esquemas XML o WSDL anidados (esquemas XML o WSDL que a su vez incluyen los WSDL o esquemas XML incluidos). Para resolver un esquema XML o WSDL no válido, haga clic en Editar Referencias. Consulte Resolución de Referencias de WSDL No Resueltas.

  4. Haga clic en Vista de Texto para mostrar una vista de texto de los detalles de WSDL. La vista de texto muestra la información que aparece en la Tabla 6-4.

    5. Tabla 6-4 Detalles de Texto de WSDL 
    Propiedad
    Descripción
    Espacio de Nombres de Destino
    Espacio de nombres utilizado para cualificar cualquiera de las definiciones que se incluyen en el WSDL.
    Texto
    Texto del WSDL.

  5. Para hacer un cambio en los campos, haga clic en Editar. Consulte Adición de WSDL para obtener descripciones de los campos.
  6. Realice las ediciones adecuadas.
  7. Haga clic en Guardar para confirmar las actualizaciones en la sesión actual.
  8. Para terminar la sesión y desplegar la configuración en tiempo de ejecución, haga clic en Activar en Centro de Cambios.
Nota: La página Ver Detalles de WSDL puede incluir un botón Devoluciones de Llamada Separadas. Este botón sólo aparece si el WSDL tiene tipos de puertos con operaciones de devolución de llamada que pueden reescribirse. Si hace clic en este botón, se reescribe el WSDL.
