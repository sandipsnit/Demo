Documentos XML

Utilice recursos de documento XML para almacenar archivos XML a fin de utilizarlos en las configuraciones de servicios de proxy o de negocio. Por ejemplo, cree recursos de documento XML para los archivos de asignación TopLink o EclipseLink necesarios en los servicios de proxy JCA o de negocio que se comunican con sistemas compatibles con JCA.

Para obtener más información sobre los servicios de JCA, consulte Chapter 12, JCA Bindings (Capítulo 12, Enlaces de JCA).
