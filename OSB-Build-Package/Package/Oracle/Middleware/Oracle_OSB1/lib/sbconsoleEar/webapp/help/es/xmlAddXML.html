Adición de Documentos XML

La forma más sencilla de crear documentos XML en Oracle Service Bus es la importación en bloque. Por ejemplo, si importa en bloque los recursos JCA creados (archivo .jca, WSDL asociado y archivo de asignación TopLink o EclipseLink), Oracle Service Bus genera automáticamente recursos de documento XML a partir de archivos de asignación y mantiene las dependencias entre los archivos de recursos.

Para obtener más información sobre la importación, consulte Carga de Recursos de un Archivo Zip, Carga de Recursos de una URL e Importación de Recursos.

Si no realiza importaciones en bloque, puede crear documentos XML de forma manual, como se describe en los siguientes pasos:

  1. Si aún no lo ha hecho, haga clic en Crear para crear una nueva sesión o en Editar para acceder a una sesión existente. Consulte Uso del Centro de Cambios.
  2. Seleccione Explorador de Proyectos y elija el proyecto o la carpeta donde desee agregar el documento XML. Aparece la página Vista de Proyectos/Carpetas.
  3. En el campo Crear Recurso, seleccione Documento XML para mostrar la página Crear un Nuevo Documento XML.
  4. En el campo Archivo XML, haga clic en Examinar y seleccione el archivo XML que desee utilizar.

    5. El campo Nombre del Recurso se rellena automáticamente con el nombre del archivo (menos la extensión del archivo).

    En lugar de seleccionar un archivo, puede copiar y pegar el contenido de un archivo XML en el área de texto del campo Archivo XML.

  5. En el campo Nombre del Recurso, introduzca o cambie el nombre del documento XML. Los nombres de los documentos deben ser únicos.

    Siga las Restricciones de Nomenclatura de Recursos para obtener ayuda con la nomenclatura.
  6. En el campo Descripción del Recurso, introduzca una descripción para el documento XML.
  7. Haga clic en Guardar. El nuevo documento XML se guarda en la sesión actual.
  8. Para terminar la sesión y desplegar la configuración en tiempo de ejecución, haga clic en Activar en Centro de Cambios.
