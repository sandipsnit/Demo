Edición de Documentos XML

Utilice la página Editar un Documento XML para ver y cambiar los detalles de un enlace de JCA.

  1. Si aún no lo ha hecho, haga clic en Crear para crear una nueva sesión o en Editar para acceder a una sesión existente. Consulte Uso del Centro de Cambios.
  2. Localice el documento XML, como se describe en Localización de Documentos XML.
  3. Haga clic en el nombre del documento XML. Aparece la página Ver Documento XML con la información que se muestra en la Tabla 13-2.

    4. Tabla 13-2 Detalles de Documentos XML 
    Propiedad
    Descripción
    Modificado por
    Usuario que ha creado este documento XML o lo ha importado a la configuración.
    Fecha de Modificación
    Fecha y hora a la que el usuario ha creado este documento XML o lo ha importado a la configuración. Haga clic en el enlace de fecha y hora para ver el historial de cambios de este recurso. Consulte Página Ver Historial de Cambios.
    Referencias
    Número de objetos a los que hace referencia este documento XML. Si existen estas referencias, haga clic en el enlace numérico para ver una lista de los objetos. Consulte Visualización de Referencias a Recursos.
    Con Referencia de
    Número de objetos que hacen referencia a este documento XML. Si existen esas referencias, haga clic en el enlace numérico para ver una lista de los objetos. Por ejemplo, si el archivo XML es un archivo TopLink al que hace referencia un enlace de JCA, este enlace de JCA se enumera como referencia cuando se hace clic en el enlace. Consulte Visualización de Referencias a Recursos.
    Descripción
    Descripción de este documento XML, si existe uno.

    Aparece por defecto la vista de sólo lectura del documento XML.

  4. Para editar el documento XML, haga clic en Editar. Consulte Adición de Documentos XML sobre las descripciones de campos.
  5. Realice las ediciones necesarias y haga clic en Guardar para confirmar las actualizaciones en la sesión actual.
  6. Para terminar la sesión y desplegar la configuración en tiempo de ejecución, haga clic en Activar en Centro de Cambios.
