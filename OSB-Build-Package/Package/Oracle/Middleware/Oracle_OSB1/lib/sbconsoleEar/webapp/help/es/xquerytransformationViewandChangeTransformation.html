Edición de Transformaciones de XQuery

  1. Si aún no lo ha hecho, haga clic en Crear para crear una nueva sesión o en Editar para acceder a una sesión existente. Consulte Uso del Centro de Cambios.
  2. Localice la trasformación de XQuery, como se describe en Búsqueda de Transformaciones de XQuery.
  3. Haga clic en el nombre de la transformación de XQuery. Aparecerá la página Ver Detalles de Transformación de XQuery con la información que se muestra en la Tabla 10-2.

    4. Tabla 10-2 Detalles de Recurso de Transformación de XQuery
    Propiedad
    Descripción
    Modificado por
    Usuario que ha creado esta transformación de XQuery o la ha importado a la configuración.
    Fecha de Modificación
    Fecha y hora a la que el usuario ha creado esta transformación de XQuery o la ha importado a la configuración. Haga clic en el enlace de fecha y hora para ver el historial de cambios de este recurso. Consulte Página Ver Historial de Cambios.
    Referencias
    Número de objetos a los que hace referencia esta transformación de XQuery. Si existen estas referencias, haga clic en el enlace numérico para ver una lista de los objetos. Para obtener más información, consulte Visualización de Referencias a Recursos.
    Con Referencia de
    Número de objetos que hacen referencia a esta transformación de XQuery. Si existen estas referencias, haga clic en el enlace numérico para ver una lista de los objetos. Por ejemplo, si asigna esta transformación de XQuery a una variable en el flujo de mensajes de un servicio de proxy concreto, este servicio de proxy aparece como referencia al hacer clic en el enlace. Para obtener más información, consulte Visualización de Referencias a Recursos.
    Descripción
    Descripción de esta transformación de XQuery, si existe una.
    XQuery
    Texto completo de la transformación de XQuery.
    Variables de XQuery
    Variables de XQuery seleccionadas para esta transformación de XQuery.

  4. Para hacer un cambio en los campos, haga clic en Editar. Consulte Adición de Transformaciones de XQuery para obtener descripciones de los campos.

    5. No puede cambiar el campo Nombre del Recurso.

  5. Haga clic en Guardar para confirmar las actualizaciones en la sesión actual.
  6. Para terminar la sesión y desplegar la configuración en tiempo de ejecución, haga clic en Activar en Centro de Cambios.
