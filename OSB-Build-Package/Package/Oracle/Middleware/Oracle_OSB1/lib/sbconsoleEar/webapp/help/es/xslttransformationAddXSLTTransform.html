Adición de una Transformación XSL

  1. Si aún no lo ha hecho, haga clic en Crear para crear una nueva sesión o en Editar para acceder a una sesión existente. Consulte Uso del Centro de Cambios.
  2. Seleccione Explorador de Proyectos para mostrar la página Vista de proyectos o la página Vista de Proyectos/Carpetas.
  3. Navegue hasta el proyecto o carpeta a la que desee agregar la transformación XSL.
  4. En la lista Crear Recurso, seleccione XSLT para mostrar la página Crear Nuevo XSLT.
  5. En el campo Nombre del Recurso, introduzca un nombre único para esta transformación XSL.

    Siga las Restricciones de Nomenclatura de Recursos para obtener ayuda con la nomenclatura.
  6. En el campo Descripción del Recurso, introduzca una descripción para la transformación XSL.
  7. En el campo XSLT, realice una de las siguientes acciones:
    • Introduzca texto para la nueva transformación XSL.
    • Haga clic en Examinar para localizar e importar una transformación XSL existente.
    • Copie y pegue texto en este campo desde una transformación XSL existente.
  8. Haga clic en Guardar para confirmar las actualizaciones en la sesión actual.
  9. Para terminar la sesión y desplegar la configuración en tiempo de ejecución, haga clic en Activar en Centro de Cambios.
