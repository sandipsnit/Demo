Localización de Transformaciones XSL

  1. Realice una de las siguientes acciones:
    • Seleccione Explorador de Proyectos para que aparezca la página Vista de Proyectos o la página Vista de Proyectos/Carpetas. A continuación, navegue por los proyectos o carpetas para buscar la transformación XSL.
    • Seleccione Explorador de Recursos > XSLT. Aparece la página Resumen de XSLT con la información que se muestra en la Tabla 14-1.
  2. Para limitar el número de elementos de la lista, puede filtrar por nombre, ruta de acceso o ambos. En los campos Nombre y Ruta de Acceso debajo de Buscar, introduzca el nombre y/o la ruta de acceso de los destinos y haga clic en el botón Buscar.

    3. La ruta de acceso es el nombre del proyecto y el nombre de la carpeta donde reside el XSLT.

    Se permiten los comodines * y ?. La búsqueda es sensible a mayúsculas/minúsculas.

    Haga clic en Ver Todo para mostrar todos los XSLT del dominio. Se borran los parámetros de búsquedas anteriores.

    Tabla 14-1 Detalles de Transformación XSL 
    Propiedad
    Descripción
    Nombre de XSLT
    Nombre asignado a la transformación XSL. Haga clic en él para mostrar la página Ver Detalles de Transformación XSL.
    Consulte Edición de Transformaciones XSL.
    Ruta de Acceso
    La ruta de acceso es el nombre del proyecto y el nombre de la carpeta donde reside la transformación XSL; por ejemplo, MiProyecto/MiCarpeta/MiRecurso.
    Haga clic en la ruta de acceso de un XSLT para mostrar la página Vista de Proyectos/Carpetas que lo contiene.
    Acciones
    Haga clic en el icono Iniciar Consola de Prueba para llamar a la consola de prueba, que se utiliza para validar y probar el diseño de los servicios y las transformaciones. Para las transformaciones XSL, puede utilizar la consola de prueba tanto dentro como fuera de una sesión. Para obtener más información, consulte Prueba de Transformaciones.
    Opciones
    Si aún no lo ha hecho, haga clic en Crear para crear una nueva sesión o en Editar para acceder a una sesión existente. Consulte Uso del Centro de Cambios.
    Haga clic en el icono Suprimir para suprimir el XSLT. Aparece un icono de advertencia de supresión si hay otros recursos que hacen referencia a éste. Puede suprimirlo confirmando la advertencia. Esto puede causar conflictos debido a referencias no resueltas al recurso suprimido. Para obtener más información, consulte Supresión de una Transformación XSL.
