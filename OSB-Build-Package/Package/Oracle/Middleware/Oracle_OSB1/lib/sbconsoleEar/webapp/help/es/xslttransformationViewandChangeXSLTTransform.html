Edición de Transformaciones XSL

  1. Si aún no lo ha hecho, haga clic en Crear para crear una nueva sesión o en Editar para acceder a una sesión existente. Consulte Uso del Centro de Cambios.
  2. Localice la transformación XSL, como se describe en Localización de Transformaciones XSL.
  3. Haga clic en el nombre de la transformación XSL. La página Ver Detalles de Transformación XSL muestra por defecto la vista estructural del XSLT, que incluye la información que se detalla en la Tabla 14-2.

    4. Tabla 14-2 Detalles del Recurso de Transformación XSL 
    Propiedad
    Descripción
    Modificado por
    Usuario que ha creado esta transformación XSL o la ha importado a la configuración.
    Fecha de Modificación
    Fecha y hora a la que el usuario ha creado esta transformación XSL o lo ha importado a la configuración. Haga clic en el enlace de fecha y hora para ver el historial de cambios de este recurso. Consulte Página Ver Historial de Cambios.
    Referencias
    Número de objetos a los que hace referencia esta transformación XSL. Si existen estas referencias, haga clic en el enlace numérico para ver una lista de los objetos. Para obtener más información, consulte Visualización de Referencias a Recursos.
    Con Referencia de
    Número de objetos que hacen referencia a esta transformación XSL. Si existen estas referencias, haga clic en el enlace numérico para ver una lista de los objetos. Por ejemplo, si asigna esta transformación XSL a una variable en el flujo de mensajes de un servicio de proxy concreto, este servicio de proxy aparece como referencia al hacer clic en el enlace. Para obtener más información, consulte Visualización de Referencias a Recursos.
    Descripción
    Descripción de esta transformación XSL.
    Espacio de Nombres de Destino
    Espacio de nombres utilizado para cualificar cualquiera de las definiciones que se incluyen en la transformación XSL.
    Dependencias de XSLT
    Atributos y grupos asociados al XSLT:
    • Ubicación del XSLT
    Estado
    Estado de la transformación XSL:
    • Válido

      • Todas las ubicaciones de las transformaciones XSL incluidas por la transformación XSL actual están especificadas y son válidas. Asimismo, todas las ubicaciones de las transformaciones XSL anidadas (las transformaciones XSL incluidas posteriormente por las transformaciones XSL incluidas) están especificadas y son válidas.

    • No Válido

      • Una o varias de las ubicaciones de las transformaciones XSL incluidas por la transformación XSL actual no se han especificado o no son válidas. Además, es posible que no se hayan especificado o no sean válidas una o varias de las transformaciones XSL anidadas (transformaciones XSL a su vez incluidas por las transformaciones XSL incluidas). Para resolver una transformación XSL no válida, haga clic en Editar Referencias.Para obtener más información, consulte Resolución de Referencias a Transformaciones XSL No Resueltas.

  4. Haga clic en Vista de Texto para mostrar una vista de texto de los detalles de la transformación XSL. La vista de texto muestra la información que aparece en la Tabla 14-3.

    5. Tabla 14-3 Detalles de Texto de la Transformación XSL
    Propiedad
    Descripción
    Espacio de Nombres de Destino
    Espacio de nombres utilizado para cualificar cualquiera de las dependencias que se incluyen en la transformación XSL.
    XSLT
    Texto completo de la transformación XSL.

  5. Para hacer un cambio en los campos, haga clic en Editar. Consulte Adición de Transformaciones XSL para obtener descripciones de los campos.

    6. No puede cambiar el campo Nombre del Recurso.

  6. Haga clic en Guardar para confirmar las actualizaciones en la sesión actual.
