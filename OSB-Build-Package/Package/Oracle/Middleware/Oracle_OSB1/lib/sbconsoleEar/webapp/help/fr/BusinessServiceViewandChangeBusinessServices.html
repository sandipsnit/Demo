Modification des configurations des services fonctionnels

  1. Si vous ne l'avez pas déjà fait, cliquez sur Créer afin de créer une session ou cliquez sur Modifier pour accéder à une session existante. Reportez-vous à Utilisation du centre des modifications.

    2. Vous ne devez pas obligatoirement ouvrir une session si vous souhaitez seulement visualiser les détails.

  2. Localisez le service fonctionnel à visualiser ou à modifier, comme l'explique la rubrique Localisation de services fonctionnels.
  3. Cliquez sur le nom du service fonctionnel. La page Visualiser un service fonctionnel - Détails de configuration affiche les informations de configuration du service fonctionnel sélectionné.
  4. Pour visualiser ou modifier les paramètres, utilisez l'une des deux méthodes suivantes :
    • Cliquez sur l'icône Modifier à côté du nom de la catégorie dont vous souhaitez visualiser ou modifier les propriétés (par exemple, Configuration générale, Configuration du transport, etc.). Les pages que vous pouvez modifier dépendent des pages que vous avez configurées au moment de la création du service fonctionnel.

      • Pour obtenir la liste de l'ensemble de ces pages, reportez-vous à la page Visualiser un service fonctionnel - Détails de configuration.

    • Cliquez sur Modifier en bas de la page pour afficher la page Créer/Modifier un service fonctionnel - Configuration générale, qui est la première de la série de pages destinées à la configuration de ce service fonctionnel.
  5. Poursuivez pour visualiser ou modifier les propriétés, comme le décrit la rubrique Création et configuration de services fonctionnels.
  6. Sur la page Modifier un service fonctionnel - Récapitulatif, cliquez sur Enregistrer pour valider les mises à jour de la session en cours.
  7. Pour terminer la session et déployer la configuration lors de l'exécution, cliquez sur Activer sous Centre des modifications.
