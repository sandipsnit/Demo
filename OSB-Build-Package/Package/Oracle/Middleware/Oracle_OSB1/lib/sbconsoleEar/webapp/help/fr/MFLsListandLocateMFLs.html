Localisation des fichiers MFL

  1. Effectuez l'une des opérations suivantes :
    • Sélectionnez Explorateur de projets pour afficher la page Vue des projets ou la page Vue du projet/Vue de dossier. Naviguez ensuite dans les projets et les dossiers pour rechercher le fichier MFL.
    • Sélectionnez Navigateur Web de ressources > MFL. La page Récapitulatif des fichiers MFL affiche les informations répertoriées dans le tableau 15-1. Pour obtenir une description plus détaillée des propriétés, reportez-vous à Modification de fichiers MFL.
  2. Pour limiter le nombre d'éléments dans la liste, vous pouvez filtrer par nom, chemin, ou les deux. Dans les champs Nom et Chemin, saisissez le nom et le chemin des cibles de recherche, puis cliquez sur Rechercher.

    3. Le chemin est constitué du nom de projet et du nom du dossier dans lequel le fichier MFL réside.

    Cliquez sur Tout visualiser pour enlever les filtres de recherche et afficher tous les fichiers MFL.

    Tableau 15-1 Détails des fichiers MFL
    Propriété
    Description
    Nom du fichier MFL
    Nom unique affecté au fichier MFL. Ce nom est un lien permettant d'accéder à la page Visualiser les détails de ressource MFL. Reportez-vous à Modification de fichiers MFL.
    Chemin
    Le chemin est constitué du nom de projet et du nom du dossier dans lequel le fichier MFL réside. Il s'agit d'un lien permettant d'accéder au projet ou au dossier contenant cette ressource. Reportez-vous à Qualification des noms de ressource à l'aide de projets et de dossiers.
    Actions
    Cliquez sur l'icône Lancer la console de test pour appeler la console de test, qui permet de valider et tester la conception de vos services et transformations. Pour les fichiers MFL, vous pouvez utiliser cette console que vous soyez dans une session ou en dehors d'une session. Reportez-vous à Test des transformations.
    Options
    Cliquez sur l'icône Supprimer pour supprimer un fichier MFL spécifique. Reportez-vous à Suppression de fichiers MFL.
