Modification de fichiers MFL

  1. Si ce n'est pas déjà fait, cliquez sur Créer pour créer une session ou sur Modifier pour indiquer une session existante. Reportez-vous à Utilisation du centre des modifications.
  2. Localisez le fichier MFL, comme décrit dans Localisation des fichiers MFL.
  3. Cliquez sur le nom de fichier MFL. La page Visualiser les détails de ressource MFL affiche les informations répertoriées dans le tableau 15-2.

    4. Tableau 15-2 Détails des ressources MFL  
    Propriété
    Description
    Dernière modification par
    Utilisateur ayant créé ce fichier MFL ou l'ayant importé dans la configuration.
    Dernière modification le
    Date et heure auxquelles l'utilisateur a créé le fichier MFL ou l'a importé dans la configuration. Cliquez sur le lien de date et d'heure pour visualiser l'historique des modifications apportées à cette ressource. Reportez-vous à Page Visualiser l'historique des modifications.
    Références
    Nombre d'objets référencés par le fichier MFL. Lorsque des références existent, cliquez sur le lien numérique pour visualiser la liste des objets en question. Reportez-vous à Visualisation des références aux ressources.
    Référencé par
    Nombre d'objets référençant le fichier MFL. Lorsque des références existent, cliquez sur le lien numérique pour visualiser la liste des objets en question. Par exemple, si vous sélectionnez le fichier MFL comme type de message de demande ou de réponse d'un service proxy ou fonctionnel de messagerie, ce dernier est répertorié comme référence lorsque vous cliquez sur ce lien. Reportez-vous à Visualisation des références aux ressources.
    Description
    Description du fichier MFL, le cas échéant.
    Texte MFL
    Texte du fichier MFL.

  4. Pour modifier les champs, cliquez sur Modifier. Pour obtenir la description de ces champs, reportez-vous à Ajout de fichiers MFL.

    5. Vous ne pouvez pas modifier le champ Nom de la ressource.

  5. Cliquez sur Enregistrer pour valider les mises à jour dans la session en cours.
  6. Pour terminer la session et déployer la configuration lors de l'exécution, cliquez sur Activer sous Centre des modifications.
