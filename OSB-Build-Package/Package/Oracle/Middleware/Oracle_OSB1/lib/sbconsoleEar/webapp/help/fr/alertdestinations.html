Destinations d'alertes

Une ressource de destination d'alertes capture la liste des destinataires pouvant recevoir les notifications d'alerte d'Oracle Service Bus. Dans des contextes de surveillance du système standard, les alertes générées par Oracle Service Bus ne sont significatives que pour un ensemble restreint d'utilisateurs.

Dans Oracle Service Bus, vous pouvez faire en sorte que chaque ressource de destination d'alertes inclue un ensemble de destinataires en fonction d'un contexte donné. Les destinations d'alertes sont utilisées par les actions d'alerte configurées dans le flux de messages et par les règles d'alerte de contrat de niveau de service. Une destination d'alertes peut inclure les types de destination suivants : console (valeur par défaut), flux de données de rapport, interruption SNMP, courrier électronique, file d'attente JMS ou rubrique JMS.

Dans le cas des destinations de courrier électronique et JMS, une ressource de destination peut inclure une liste d'adresses électroniques ou d'URI JMS, respectivement.

Vous pouvez réutiliser les destinations d'alertes dans plusieurs configurations d'alerte pour les services. Reportez-vous à Ajout d'actions d'alerte.
