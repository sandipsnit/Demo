Suppression de destinations d'alertes

Pour supprimer une destination d'alertes, procédez comme suit :

  1. Si vous ne l'avez pas déjà fait, cliquez sur Créer afin de créer une session ou cliquez sur Modifier pour accéder à une session existante. Reportez-vous à Utilisation du centre des modifications.
  2. Sélectionnez Navigateur Web de ressources > Destinations d'alertes pour afficher la page Récapitulatif des destinations d'alertes.
  3. Cliquez sur l'icône Supprimer du champ Options de la destination d'alertes à supprimer. Celle-ci est supprimée de la session en cours. Si cette ressource est référencée par d'autres, l'icône Avertissement relatif à la suppression apparaît. Vous pouvez supprimer la ressource en confirmant l'opération. Toutefois, des conflits peuvent survenir, les références à la ressource supprimée n'étant pas résolues.

    4. Pour supprimer des destinataires de courrier électronique ou des destinations JMS spécifiques configurés dans une destination d'alertes, reportez-vous à Modification de destinations d'alertes.

  4. Pour terminer la session et déployer la configuration lors de l'exécution, cliquez sur Activer sous Centre des modifications.
