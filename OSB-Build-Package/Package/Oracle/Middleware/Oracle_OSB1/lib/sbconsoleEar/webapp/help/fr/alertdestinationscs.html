Ajout de destinataires de courrier électronique

Avant d'ajouter une destination de courrier électronique, vous devez configurer un serveur SMTP (reportez-vous à Ajout de serveurs SMTP) ou une session JavaMail dans WebLogic Server (reportez-vous à "Configure access to JavaMail" dans l'aide en ligne d'Oracle Fusion Middleware Oracle WebLogic Server Administration Console, à l'adresse suivante : http://download.oracle.com/docs/cd/E15523_01/apirefs.1111/e13952/taskhelp/mail/CreateMailSessions.html). Si aucune ressource de serveur SMTP, ni aucune session JavaMail n'est disponible, vous ne pourrez pas configurer de destinataire de courrier électronique.

  1. Sélectionnez Envoyer une alerte par courrier électronique dans la liste déroulante et cliquez sur Ajouter pour afficher la page Ajouter un destinataire de courrier électronique.
  2. Dans le champ Destinataires du courrier électronique, indiquez un destinataire de courrier électronique au format suivant : mailto:username@hostname.

    3. Ce champ est obligatoire.

    Vous pouvez indiquer plusieurs destinataires de courrier électronique en saisissant les noms utilisateur et noms d'hôte sous la forme d'une liste d'éléments séparés par des virgules. Exemple : mailto:username@hostname [,username_1@hostname_1]...[,username_n@hostname_n].

    Seul le premier destinataire doit être précédé de la mention "mailto:".

  3. Dans le champ Serveur SMTP, sélectionnez le nom du serveur SMTP utilisé pour les courriers électroniques sortants. Ce champ n'est pas obligatoire si une session de messagerie est sélectionnée à l'étape suivante.
  4. Dans le champ Session de messagerie, sélectionnez une session de messagerie disponible. Ce champ n'est pas obligatoire si un serveur SMTP est sélectionné à l'étape précédente.
  5. Dans le champ Nom de l'émetteur, saisissez le nom de l'expéditeur de la notification d'alerte. Ce champ est facultatif.
  6. Dans le champ Adresse de l'émetteur, saisissez une adresse électronique valide. Ce champ est obligatoire si vous indiquez une valeur dans le champ Nom de l'émetteur.
  7. Dans le champ Nom de réponse aux messages, saisissez le nom de l'utilisateur auquel envoyer la réponse. Ce champ est facultatif.
  8. Dans le champ Adresse de réponse aux messages, saisissez l'adresse électronique à laquelle envoyer la réponse. Ce champ est obligatoire si vous indiquez une valeur dans le champ Nom de réponse aux messages.
  9. Dans le champ Temporisation de la connexion, indiquez le temps d'attente, en secondes, d'une réponse du serveur avant temporisation de la connexion. La valeur par défaut est 0.
  10. Dans le champ Encodage de la demande, indiquez la valeur d'encodage du jeu de caractères. La valeur par défaut est ISO-8859-1.
  11. Cliquez sur Enregistrer. Le destinataire est ajouté à la liste des destinataires de courrier électronique pour cette destination d'alertes et il est enregistré dans la session en cours.
