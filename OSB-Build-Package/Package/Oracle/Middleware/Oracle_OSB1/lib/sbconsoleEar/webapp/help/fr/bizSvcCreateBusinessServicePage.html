Créer/Modifier un service fonctionnel - Référence de page

Pour créer un service fonctionnel, sélectionnez Service fonctionnel dans la liste déroulante Créer une ressource sur la page Vue du projet/Vue de dossier. Lorsque vous sélectionnez cette option, un ensemble de pages permettant de configurer et d'ajouter des services fonctionnels apparaît. Les pages affichées dépendent des options sélectionnées au fur et à mesure. Voici la liste des pages susceptibles d'apparaître :

  • Page Configuration générale
  • Page Configuration du type de message
  • Page Configuration du transport
  • Page Protocole - Configuration du transport
  • Page Configuration de la liaison SOAP
  • Page Configuration de la gestion des messages
  • Page Récapitulatif
  • Page Stratégies de service fonctionnel
  • Page Sécurité de service fonctionnel
