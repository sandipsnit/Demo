Page DSP Transport Configuration

La page Créer/Modifier un service fonctionnel - DSP Transport Configuration est une des pages permettant de créer et modifier des services fonctionnels, comme indiqué dans Création et configuration de services fonctionnels et Modification des configurations des services fonctionnels.

Pour plus d'informations, reportez-vous à "DSP and Oracle Data Service Integrator Transport" dans le manuel Oracle Fusion Middleware Developer's Guide for Oracle Service Bus à l'adresse suivante : http://www.oracle.com/pls/as111170/lookup?id=OSBDV1229.

Utilisez cette page pour configurer les paramètres de transport d'un service fonctionnel utilisant le protocole DSP (Oracle Data Service Integrator). Le tableau 18-4 explique comment utiliser cette page.

Tableau 18-4 Page Créer/Modifier un service fonctionnel - DSP Transport Configuration
Option
Procédure
Niveau de débogage
Indiquez l'une des options suivantes :
  • 0 : aucune information de débogage
  • 1 : informations sur le message de demande
  • 3 : informations sur les messages de demande et de réponse
Compte de service
Cliquez sur Parcourir et sélectionnez un compte de service dans la liste affichée. Si vous n'indiquez aucun compte de service, un sujet anonyme est utilisé.

Pour plus d'informations :

  • Comptes de service
  • Ajout de comptes de service
Stratégie de répartition
Sélectionnez l'instance du gestionnaire de travaux WebLogic Server à utiliser dans la stratégie de répartition de cette adresse. Le gestionnaire de travaux par défaut est utilisé s'il n'en existe pas d'autre.
Le gestionnaire de travaux permet d'envoyer le message de réponse pour le traitement de cette dernière. Pour plus d'informations sur les gestionnaires de travaux, reportez-vous à "Using Work Managers to Optimize Scheduled Work" dans le manuel Oracle Fusion Middleware Configuring Server Environments for Oracle WebLogic Server à l'adresse suivante : http://www.oracle.com/pls/as111170/lookup?id=CNFGD112.

Après avoir fini

Cliquez sur Suivant pour poursuivre la configuration du service sur la page suivante ou cliquez sur Dernier pour vérifier et enregistrer cette configuration sur la page Créer/Modifier un service fonctionnel - Récapitulatif.
