Page FTP - Configuration du transport

La page Créer/Modifier un service fonctionnel - FTP - Configuration du transport est une des pages permettant de créer et modifier des services fonctionnels, comme indiqué dans Création et configuration de services fonctionnels et Modification des configurations des services fonctionnels.

Utilisez cette page pour configurer les paramètres de transport d'un service fonctionnel utilisant le protocole de transport FTP. Le tableau 18-9 explique comment utiliser cette page.

Tableau 18-9 Page Créer/Modifier un service fonctionnel - FTP - Configuration du transport 
Option
Procédure
Authentification utilisateur
Sélectionnez anonyme si l'utilisateur du serveur FTP est anonyme ou utilisateur externe s'il s'agit d'un compte configuré en externe.
Identité (adresse électronique)
Ce champ n'est disponible que si vous définissez l'option Authentification utilisateur sur anonyme.
Entrez l'adresse électronique de l'utilisateur anonyme.
Compte de service
Ce champ n'est disponible que si vous définissez l'option Authentification utilisateur sur utilisateur externe.
Entrez le compte de service de l'utilisateur externe.

Pour plus d'informations :

  • Comptes de service
  • Ajout de comptes de service
Temporisation
Indiquez l'intervalle de temporisation de socket, en secondes, au bout duquel la connexion est supprimée. La valeur par défaut est de 60 secondes.
Préfixe pour le nom du fichier de destination
Indiquez le préfixe à ajouter au nom de fichier utilisé pour stocker le fichier sur le serveur distant.
N'entrez pas le caractère * dans ce champ. Il entraîne une exception d'exécution.
Suffixe pour le nom du fichier de destination
Indiquez le suffixe à ajouter au nom de fichier utilisé pour stocker le fichier sur le serveur distant.
N'entrez pas le caractère * dans ce champ. Il entraîne une exception d'exécution.
Mode de transfert
Sélectionnez le mode de transfert ASCII ou binaire.
Encodage de la demande
Acceptez l'encodage de jeu de caractères par défaut, UTF-8, pour les demandes dans les transports FTP ou indiquez un encodage de jeu de caractères différent.

Après avoir fini

Cliquez sur Suivant pour poursuivre la configuration du service sur la page suivante ou cliquez sur Dernier pour vérifier et enregistrer cette configuration sur la page Créer/Modifier un service fonctionnel - Récapitulatif.
