Page Fichier - Configuration du transport

La page Créer/Modifier un service fonctionnel - Fichier - Configuration du transport est une des pages permettant de créer et modifier des services fonctionnels, comme indiqué dans Création et configuration de services fonctionnels et Modification des configurations des services fonctionnels.

Utilisez cette page pour configurer les paramètres de transport d'un service fonctionnel utilisant le protocole de transport de fichier. Le tableau 18-7 explique comment utiliser cette page.

Tableau 18-7 Page Créer/Modifier un service fonctionnel - Fichier - Configuration du transport
Option
Procédure
Préfixe
Indiquez le préfixe à ajouter devant le nom de fichier.
N'entrez pas le caractère * dans ce champ. Il entraîne une exception d'exécution.
Suffixe
Indiquez le suffixe à ajouter à la fin du nom de fichier. Ce champ est obligatoire.
N'entrez pas le caractère * dans ce champ. Il entraîne une exception d'exécution.
Encodage de la demande
Acceptez l'encodage de jeu de caractères par défaut, UTF-8, pour les demandes dans les transports de fichier ou indiquez un encodage de jeu de caractères différent.

Après avoir fini

Cliquez sur Suivant pour poursuivre la configuration du service sur la page suivante ou cliquez sur Dernier pour vérifier et enregistrer cette configuration sur la page Créer/Modifier un service fonctionnel - Récapitulatif.
