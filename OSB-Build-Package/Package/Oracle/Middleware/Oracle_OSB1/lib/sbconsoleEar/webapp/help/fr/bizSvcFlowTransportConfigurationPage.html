Page Flux - Configuration du transport

La page Créer/Modifier un service fonctionnel - Flux - Configuration du transport est une des pages permettant de créer et modifier des services fonctionnels, comme indiqué dans Création et configuration de services fonctionnels et Modification des configurations des services fonctionnels.

Les services fonctionnels de type transport de flux permettent d'accéder aux ressources de Fractionnement-Fusion. Pour plus d'informations sur les ressources de Fractionnement-Fusion, reportez-vous à "Improving Service Performance with Split-Join" dans le manuel Oracle Fusion Middleware Developer's Guide for Oracle Service Bus à l'adresse suivante : http://www.oracle.com/pls/as111170/lookup?id=OSBAG291.

Utilisez cette page pour configurer les paramètres de transport d'un service fonctionnel utilisant le protocole de transport de flux. Le tableau 18-8 explique comment utiliser cette page.
