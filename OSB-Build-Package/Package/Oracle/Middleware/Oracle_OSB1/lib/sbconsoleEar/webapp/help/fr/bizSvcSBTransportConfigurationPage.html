Page SB - Configuration du transport

La page Créer/Modifier un service fonctionnel - SB - Configuration du transport est l'une des pages destinées à la création et à la modification de services fonctionnels, décrites dans les rubriques Création et configuration de services fonctionnels et Modification des configurations des services fonctionnels.

Pour plus d'informations, reportez-vous à SB Transport dans le manuel Oracle Fusion Middleware Developer's Guide for Oracle Service Bus, à l'adresse suivante : http://www.oracle.com/pls/as111170/lookup?id=OSBDV997.

Cette page permet de configurer les paramètres de transport pour un service fonctionnel utilisant le protocole de transport SB (Service Bus). Le tableau 18-15 explique comment l'utiliser.

Tableau 18-15 Page Créer/Modifier un service fonctionnel - SB - Configuration du transport 
Option
Procédure
Stratégie de répartition
Sélectionnez l'instance du gestionnaire de travaux WebLogic Server à utiliser dans la stratégie de répartition de cette adresse. Le gestionnaire de travaux par défaut est utilisé s'il n'en existe pas d'autre. Pour plus d'informations sur les gestionnaires de travaux, reportez-vous à "Using Work Managers to Optimize Scheduled Work" dans le manuel Oracle Fusion Middleware Configuring Server Environments for Oracle WebLogic Server à l'adresse suivante : http://www.oracle.com/pls/as111170/lookup?id=CNFGD112.
Temporisation
Durée, en secondes, nécessaire à la temporisation du service.
Remarque : La temporisation n'est pas prise en compte lorsque la qualité de service a pour valeur Une seule fois.
Compte de service
Cliquez sur Parcourir et sélectionnez un compte de service dans la liste affichée. Si vous n'indiquez aucun compte de service, un sujet anonyme est utilisé.
