Page Configuration de la liaison SOAP

La page Créer/Modifier un service fonctionnel - Configuration de la liaison SOAP est l'une des pages destinées à la création et à la modification de services fonctionnels, décrites dans les rubriques Création et configuration de services fonctionnels et Modification des configurations des services fonctionnels.

Cette page permet de configurer la liaison SOAP pour un service fonctionnel basé sur une ressource WSDL.

Sélectionnez Imposer la conformité WS-I pour indiquer si le service doit se conformer au profil de base défini par la Web Services Interoperability Organization (WS-I). Cette option est disponible pour les services SOAP 1.1 uniquement.

Lorsqu'un service est conforme à la WS-I, des vérifications sont réalisées sur les messages reçus et envoyés par ce service.

Après avoir fini

Cliquez sur Suivant pour poursuivre la configuration du service sur la page suivante ou cliquez sur Dernier pour vérifier et enregistrer cette configuration sur la page Créer/Modifier un service fonctionnel - Récapitulatif.
