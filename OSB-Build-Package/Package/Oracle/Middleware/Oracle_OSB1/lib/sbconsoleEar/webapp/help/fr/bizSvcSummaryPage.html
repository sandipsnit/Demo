Page Récapitulatif

La page Créer/Modifier un service fonctionnel - Récapitulatif est l'une des pages destinées à la création et à la modification de services fonctionnels, comme décrit dans Création et configuration de services fonctionnels et Modification des configurations des services fonctionnels.

Cette page permet de visualiser ou de modifier les paramètres de configuration d'un service fonctionnel avant de l'enregistrer.

Pour visualiser ou modifier les paramètres, cliquez sur Modifier dans la ligne de la catégorie appropriée (par exemple, Configuration générale, Configuration du transport, etc.). Les pages que vous pouvez modifier dépendent des pages que vous avez configurées au moment de la création du service fonctionnel. La liste suivante affiche toutes les pages :

  • Configuration générale, page
  • Page Configuration du type de message
  • Page Configuration du transport
  • Page Protocole - Configuration du transport
  • Page Stratégies de service fonctionnel
  • Page Sécurité de service fonctionnel
