Création et configuration de services fonctionnels

Cette section explique comment créer et configurer des services fonctionnels.

Pour les services JCA, vous pouvez générer des services fonctionnels à partir de liaisons JCA (reportez-vous à Génération d'un service fonctionnel à partir d'une ressource de liaison JCA) ou depuis Oracle Enterprise Repository (reportez-vous à Génération d'un service fonctionnel à partir d'Oracle Enterprise Repository).

  1. Si vous ne l'avez pas déjà fait, cliquez sur Créer afin de créer une session ou cliquez sur Modifier pour accéder à une session existante. Reportez-vous à Utilisation du centre des modifications.
  2. Sélectionnez Explorateur de projets, puis sélectionnez le projet ou le dossier à ajouter au service fonctionnel.
  3. Sur la page Vue du projet/Vue de dossier, sélectionnez Service fonctionnel dans la liste déroulante Créer une ressource.
  4. Sur la page Créer/Modifier un service fonctionnel - Configuration générale, indiquez le nom du service et sélectionnez le type de service à créer.

    Pour obtenir des conseils sur la dénomination, suivez les restrictions concernant la dénomination des ressources.
  5. Cliquez sur Suivant. Les pages suivantes dépendent des choix que vous avez faits sur la première page. Entrez les informations appropriées sur chacune des pages suivantes jusqu'à accéder à la page récapitulative, puis cliquez sur Enregistrer pour enregistrer le service de la session en cours.

    6. Pour obtenir des instructions détaillées afin de remplir chaque page, reportez-vous aux rubriques suivantes :

    • Configuration générale, page
    • Page Configuration du type de message
    • Page Configuration du transport
    • Page Protocole - Configuration du transport
    • Page Configuration de la liaison SOAP
    • Page Configuration de la gestion des messages
    • Page Récapitulatif
    • Page Stratégies de service fonctionnel
    • Page Sécurité de service fonctionnel

Lorsqu'un service est créé à partir d'une ressource WSDL contenant des pièces jointes de stratégie WS, les stratégies (en lecture seule) apparaissent sur la page Protocole - Configuration du transport. Si l'une des stratégies WS du service indique une authentification, vous devez sélectionner un compte de service. Un service proxy qui achemine les données vers ce service fonctionnel utilisera ce compte de service pour s'authentifier auprès du service fonctionnel.

Génération d'un service fonctionnel à partir d'une ressource de liaison JCA

Vous pouvez générer un service fonctionnel JCA à partir d'une ressource de liaison JCA sortante dans Oracle Service Bus. Pour plus d'informations sur les ressources de liaison JCA, reportez-vous au Chapitre 12, Liaisons JCA.

Pour générer un service fonctionnel JCA à partir d'une liaison JCA :

  1. Dans la console d'administration d'Oracle Service Bus, cliquez sur Créer ou sur Modifier dans le centre des modifications si vous n'êtes pas encore en mode Créer ou Modifier.
  2. Dans le navigateur Web de ressources, cliquez sur Liaisons JCA.
  3. Localisez la liaison JCA à partir de laquelle vous souhaitez générer un service et cliquez sur l'icône Action.
  4. Dans la fenêtre qui apparaît, confirmez le nom de la ressource WSDL et le service que vous souhaitez générer, sélectionnez l'emplacement de ces nouvelles ressources et cliquez sur Générer.

    5. Oracle Service Bus génère le service et la ressource WSDL correspondante.

  5. Procédez aux modifications nécessaires de tous les autres détails de configuration du service généré, tels que l'URI endpoint.

Génération d'un service fonctionnel à partir d'Oracle Enterprise Repository

Vous pouvez générer un service fonctionnel à partir d'Oracle Enterprise Repository en utilisant l'environnement de développement d'Oracle Service Bus. Pour plus d'informations, reportez-vous à "Generating a Business Service from Oracle Enterprise Repository" dans le manuel Oracle Fusion Middleware Developer's Guide for Oracle Service Bus, à l'adresse suivante : http://www.oracle.com/pls/as111170/lookup?id=OSBDV1797.
