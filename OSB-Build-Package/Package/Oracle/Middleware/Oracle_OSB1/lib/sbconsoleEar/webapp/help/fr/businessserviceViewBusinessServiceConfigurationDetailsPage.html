Page Visualiser un service fonctionnel - Détails de configuration

La page Visualiser un service fonctionnel - Détails de configuration affiche les détails de configuration d'un service fonctionnel. Le tableau 18-22 décrit toutes les propriétés qui peuvent apparaître sur cette page. (Les propriétés varient selon les détails du service fonctionnel.)

Les catégories répertoriées sur cette page correspondent aux pages Créer/Modifier un service fonctionnel utilisées pour créer et modifier les configurations des services fonctionnels, comme le décrit la rubrique Créer/Modifier un service fonctionnel - Référence de page.

Cliquez sur le lien Modifier à côté de n'importe quel nom de catégorie pour afficher la page de configuration associée.

Tableau 18-22 Page Visualiser un service fonctionnel - Détails de configuration
