Page Sécurité de service fonctionnel

La page Visualiser un service fonctionnel - Configuration de la sécurité est l'une des pages destinées à la création et à la modification de services fonctionnels, décrites dans les rubriques Création et configuration de services fonctionnels et dans la page Visualiser un service fonctionnel- Détails de configuration.

Les champs disponibles sur cette page dépendent des stratégies utilisées dans le service fonctionnel. Par exemple, si le service utilise des stratégies OWSM (recommandé), l'option Remplacements de stratégie apparaît sur la page de sécurité. Pour plus d'informations, reportez-vous à Securing Oracle Service Bus with Oracle Web Services Manager dans le manuel Oracle Fusion Middleware Developer's Guide for Oracle Service Bus, à l'adresse suivante : http://www.oracle.com/pls/as111170/lookup?id=OSBDV1681.

Les stratégies WLS 9.2 liées au service sont exposées (en mode Inline) dans le WSDL en vigueur. Les stratégies de type Abstract sont pré-traitées avant d'être mises en mode Inline. Les stratégies OWSM ne sont pas mises en mode Inline dans le WSDL en vigueur mais liées par référence.

Cette page permet de configurer les paramètres de sécurité d'un service fonctionnel associé à une stratégie de type WSDL ou utilisant des liaisons de stratégie personnalisées. Le tableau 18-24 explique comment l'utiliser.

Compte de service
Cliquez sur Parcourir et sélectionnez un compte de service dans la liste affichée. Si vous n'indiquez aucun compte de service, un sujet anonyme est utilisé.
Remarque : cette option n'est pas disponible pour les services utilisant les stratégies d'Oracle Web Services Manager car les informations d'identification et de connexion de l'utilisateur sont fournies par des clés de remplacement.

Remplacements de stratégie
Pour les stratégies OWSM, indiquez tous les remplacements autorisés de votre choix.
Pour plus d'informations, reportez-vous au tableau 18-23 et à Securing Oracle Service Bus with Oracle Web Services Manager dans le manuel Oracle Fusion Middleware Developer's Guide for Oracle Service Bus, à l'adresse suivante : http://www.oracle.com/pls/as111170/lookup?id=OSBDV1681.

Après avoir fini

Cliquez sur Mettre à jour pour enregistrer cette configuration ou sur Réinitialiser pour annuler vos modifications.
