Centre des modifications

Le centre des modifications permet à l'équipe de collaborer lorsque les services et les métadonnées sont configurés dans Oracle Service Bus. Chaque utilisateur travaille dans une session de modèle d'environnement restreint jusqu'à ce que l'utilisateur soit prêt à réinsérer la configuration de travail à la configuration centrale du bus. Le centre des modifications propose différents niveaux d'annulation et permet de voir les conflits, étant donné que plusieurs utilisateurs travaillent sur la configuration.

La plupart des pages du centre de modifications affichent des informations différentes si vous êtes dans une session (par exemple, si vous avez cliqué sur Modifier dans le centre des modifications) ou en dehors d'une session. Par exemple, dans une session, la page Visualiser les modifications de configuration qui apparaît lorsque vous cliquez sur Visualiser les modifications dans le centre des modifications répertorie toutes les modifications apportées au cours de cette session ; en dehors d'une session, la page répertorie toutes les activations de session.

Le tableau 2-1 présente les tâches du centre des modifications et les informations de session correspondantes.

Tableau 2-1 Tâches du centre des modifications et informations de session 
Tâches
Informations sur la session
Visualiser les conflits
Dans une session uniquement : ce lien affiche le nombre de conflits survenus dans une session. Visualisez et résolvez tous les conflits entre les modifications de la session en cours et celles effectuées dans toutes les autres sessions au sein de la console d'administration d'Oracle Service Bus. Un conflit survient si une ressource modifiée dans la session en cours a déjà été modifiée et activée par une autre session. Deux modifications de la même ressource par deux sessions ne créent pas de conflit tant qu'aucune des sessions n'est activée.
Cette page fait également apparaître les erreurs sémantiques des ressources de la session.
Visualiser les modifications
Dans une session : visualisez les modifications de configuration effectuées pendant la session en cours.
En dehors d'une session : visualisez les modifications de configuration dues à l'activation de précédentes sessions.
Annuler les modifications
Dans une session : permet d'annuler une modification effectuée durant la session en cours.
En dehors de la session : permet d'annuler l'activation d'une session.
Visualiser toutes les sessions
Dans une session ou en dehors d'une session : permet de visualiser toutes les sessions existantes dans la console d'administration d'Oracle Service Bus.
Activer les sessions
Dans une session uniquement : permet d'activer la session.
Voir les détails de la tâche
Dans une session : permet de visualiser les détails d'une modification précise effectuée dans la session en cours.
En dehors de la session : permet de visualiser les détails de modifications précises effectuées lors de l'activation d'une précédente session.
