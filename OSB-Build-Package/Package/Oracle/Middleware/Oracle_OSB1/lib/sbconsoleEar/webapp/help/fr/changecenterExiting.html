Sessions existantes

Cliquez sur Quitter sous Centre des modifications à n'importe quel moment pour quitter la session. Cependant, la session n'expire pas.

Vous pouvez cliquer sur Modifier sous Centre des modifications pour revenir à la session et poursuivre les modifications. Ce comportement s'applique également si vous cliquez sur Déconnexion pour vous déconnecter de la console d'administration d'Oracle Service Bus ou si vous fermez le navigateur. La session et toutes les modifications apportées dans celle-ci persistent même si vous vous déconnectez de la console d'administration d'Oracle Service Bus ou si le serveur est redémarré.

La session expire uniquement après avoir été activée. Reportez-vous à Activation de sessions.
