Purge de l'historique d'activation des sessions

Vous pouvez purger les sessions activées durant une période précise, délimitée par une date de début et de fin. Cette action peut être effectuée uniquement en dehors d'une session.

Attention : La purge de l'historique d'activation des sessions implique une suppression de données qui active plusieurs niveaux d'annulation. En d'autres termes, si vous purgez l'historique d'activation des sessions pour une période déterminée, vous ne pourrez pas annuler les sessions activées durant cette période.
  1. Assurez-vous que vous n'êtes pas dans une session active et cliquez sur Visualiser les modifications.
  2. Sur la page Visualiser les modifications de configuration, cliquez sur Purger par date. La page Purger l'historique d'activation des sessions par date apparaît.
  3. Entrez les dates de début et de fin dans leurs champs respectifs au format : MM/DD/YYYY.

    4. L'historique d'activation des sessions pour toutes les sessions comprises entre les dates indiquées sera supprimé d'Oracle Service Bus.
