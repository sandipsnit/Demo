Annulation dans une session

En cas d'erreurs sémantiques dues à l'annulation de l'activation d'une session, vous ne pouvez pas effectuer l'annulation. Cependant, il existe alors une alternative. Vous pouvez annuler l'activation de la session et effectuer les modifications dans une nouvelle session. Vous pouvez ensuite résoudre les erreurs sémantiques et activer la session. Vous pouvez également utiliser cette fonction d'annulation dans une session pour explorer les ramifications de l'annulation de l'activation d'une session. Vous pouvez examiner toutes les modifications effectuées et décider si vous souhaitez réellement annuler. Si c'est le cas, vous pouvez activer la session.
