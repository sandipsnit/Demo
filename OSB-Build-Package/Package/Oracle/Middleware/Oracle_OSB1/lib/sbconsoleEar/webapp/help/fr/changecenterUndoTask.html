Annulation des tâches

Utilisez la page Visualiser les modifications de configuration pour annuler des tâches effectuées lors de la configuration d'Oracle Service Bus pendant la session en cours et pour annuler les activations de sessions en dehors d'une session. Oracle Service Bus permet d'annuler différents niveaux d'activation de session, limités uniquement par vos ressources système. Pour plus d'informations, reportez-vous à Ordre d'annulation des tâches.

Remarque : En cas de mise à niveau d'Oracle Service Bus version 2.1 à la version 2.5, vous ne pourrez pas annuler les sessions activées avant la mise à niveau. Les sessions activées après la mise à niveau pourront être annulées dans l'historique d'activation des sessions.
Annulation d'une tâche dans une session
  1. Assurez-vous que vous êtes dans une session.
  2. Cliquez sur Visualiser les modifications en dessous du centre des modifications.
  3. Dans la colonne Options d'une tâche précise, cliquez sur l'icône Annuler.

    4. La tâche est annulée.

    Vous pouvez annuler toutes les modifications de la session en cours. Toutefois, si vous changez d'avis par la suite, vous pouvez annuler l'annulation. Si vous changez à nouveau d'avis, vous pouvez annuler l'annulation ayant annulé l'annulation précédente, et ainsi de suite.

Annulation d'une session précédemment activée
  1. Assurez-vous que vous êtes en dehors d'une session.
  2. Cliquez sur Visualiser les modifications en dessous du centre des modifications.
  3. Effectuez l'une des opérations suivantes :
    • Dans la colonne Options d'une tâche précise, cliquez sur l'icône Annuler. L'activation de la session est annulée.
    • Dans la colonne Options d'une tâche précise, sélectionnez l'icône Annuler la tâche dans la session. Une session est créée et pré-remplie avec les tâches nécessaires pour annuler les modifications activées dans la session. Pour plus d'informations, reportez-vous à Annulation dans une session.
