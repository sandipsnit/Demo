Utilisation du centre des modifications

Le module Centre des modifications sert de point de départ pour utiliser la console d'administration d'Oracle Service Bus en vue de modifier la configuration. Pour modifier votre configuration à l'aide de la console d'administration d'Oracle Service Bus, vous devez utiliser le centre des modifications pour démarrer une session.

  1. Cliquez sur Modifier ou sur Créer pour démarrer une session. Le nom de la session apparaît sous Centre des modifications.
  2. Apportez les modifications nécessaires sur la page de la console d'administration d'Oracle Service Bus correspondante.
  3. Cliquez sur Enregistrer sur chaque page où vous effectuez une modification. Toutes les modifications effectuées dans la session en cours sont enregistrées.

    4. Cliquez sur Annuler sous Centre des modifications à tout moment pendant la session pour annuler les modifications effectuées jusqu'à présent dans la session en cours.

  4. Après avoir effectué toutes vos modifications, cliquez sur Activer sous Centre des modifications. Reportez-vous à Activation de sessions.
