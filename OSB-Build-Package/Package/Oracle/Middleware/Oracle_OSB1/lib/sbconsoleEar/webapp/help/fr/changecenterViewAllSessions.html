Visualisation de toutes les sessions

Cette page permet de visualiser toutes les sessions existantes dans la console d'administration d'Oracle Service Bus. Vous pouvez visualiser ces sessions si vous êtes dans une session en cours ou en dehors d'une session.

Vous pouvez visualiser toutes les sessions uniquement si vous utilisez le rôle Administrateur. Pour plus d'informations, reportez-vous à Configuring Administrative Security dans le manuel Oracle Fusion Middleware Developer's Guide for Oracle Service Bus, à l'adresse suivante : http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539.

  1. Cliquez sur Visualiser toutes les sessions sous Centre des modifications. Pour chaque session existante, la page Visualiser toutes les sessions affiche les informations présentées dans le tableau 2-5.

    2. Tableau 2-5 Détails des sessions
    Propriété
    Description
    Nom de la session
    Nom de la session. Le nom est un lien vers la session.
    Utilisateur
    Nom du créateur de la session.
    Heure de création
    Date et heure de création de la session.
    Dernière modification
    Date et heure de la dernière modification pendant la session.
    Nombre d'objets modifiés
    Nombre d'objets modifiés pendant la session.

  2. Pour accéder à une autre session, cliquez sur le lien dans la colonne Nom de session de cette session.
Remarque : Un même utilisateur connecté à partir de différents navigateurs n'est pas pris en charge. Cela entraîne un comportement imprévisible de la console d'administration d'Oracle Service Bus.
