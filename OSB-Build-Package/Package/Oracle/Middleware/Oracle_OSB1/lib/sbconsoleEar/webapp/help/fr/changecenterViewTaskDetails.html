Visualisation des détails de tâche

Cette page permet de visualiser les détails d'une modification en particulier, apportée dans la session en cours, ainsi que les détails de modifications précises apportées dans les sessions activées, si vous êtes en dehors d'une session.

  1. Sélectionnez Visualiser les modifications sous Centre des modifications.
  2. Sur la page Visualiser les modifications de configuration, cliquez sur le nom de la tâche dans la colonne Tâche. La page Détails de la tâche affiche les informations présentées dans le tableau 2-4.

    3. Tableau 2-4 Détails de la tâche
    Propriété
    Description
    Tâche
    Type de la tâche réalisée.
    • Créer
    • Mettre à jour
    • Supprimer
    • Renommer
    • Déplacer
    Type de ressource
    Types de ressource Oracle Service Bus faisant partie de la tâche.
    Ressource
    Nom et chemin de la ressource. Le chemin est le nom du projet et du dossier dans lesquels réside la ressource.

  3. Cliquez sur Précédent pour revenir à la page Visualiser les modifications de configuration.
