Désactivation globale de la mise en mémoire cache des résultats

Lorsque vous désactivez de manière globale la mise en mémoire cache des résultats, Oracle Service Bus vide l'ensemble du cache, en supprimant les résultats en mémoire cache de tous les services fonctionnels pour lesquels cette fonction était activée. Le paramètre par défaut de mise en mémoire cache des résultats est activé.

Pour désactiver la mise en mémoire cache des résultats au niveau du domaine :

  1. Si cela n'est pas encore fait, cliquez sur Créer pour créer une session ou cliquez sur Modifier pour accéder à une session existante. Reportez-vous à Utilisation du centre des modifications.
  2. Sélectionnez Opérations > Paramètres globaux.
  3. Désélectionnez la case à cocher Activer la mise en mémoire cache des résultats.
    4. Remarque : Ce paramètre remplace toutes les mises en mémoire cache des résultats configurées pour chaque service fonctionnel.
  4. Cliquez sur Mettre à jour. La configuration est enregistrée dans la session en cours.
  5. Vous pouvez également cliquer sur Réinitialiser à n'importe quel moment pendant la session pour annuler vos modifications et actualiser la page avec les paramètres actuellement stockés.
  6. Pour terminer la session et déployer la configuration lors de l'exécution, cliquez sur Activer sous Centre des modifications.
