Désactivation de la surveillance globale des services

Pour désactiver la surveillance de tous les services au niveau du domaine :

Remarque : Lorsque vous désactivez la surveillance de tous les services, l'ensemble des statistiques collectées jusqu'à présent pour ces services sont également supprimées. Il est impossible de les restaurer en utilisant la fonction Annuler de la session. Cette fonction sert à réactiver la surveillance pour les services, mais la suppression des statistiques est irréversible.
  1. Si vous ne l'avez pas déjà fait, cliquez sur Créer afin de créer une session ou cliquez sur Modifier pour accéder à une session existante. Reportez-vous à Utilisation du centre des modifications.
  2. Sélectionnez Opérations > Paramètres globaux.
  3. Désélectionnez la case à cocher Activer la surveillance pour que le système arrête de collecter les statistiques de surveillance correspondant à tous les services de votre configuration.
    4. Remarque : Cette option remplace l'option Activer la surveillance de service que vous pouvez sélectionner pour des services fonctionnels et proxy spécifiques. Reportez-vous à Configuration de paramètres opérationnels pour les services fonctionnels et à Configuration des paramètres opérationnels des services proxy.
    Remarque : Cette option remplace également l'option Activer les alertes de contrat de niveau de service. Lorsque vous désactivez la surveillance au niveau global, les alertes de contrat de niveau de service (SLA) seront également désactivées lors de l'exécution, même si la case Activer les alertes de contrat de niveau de service correspondante est cochée. Reportez-vous à Activation globale des alertes de contrat de niveau de service.
  4. Cliquez sur Mettre à jour. La configuration de surveillance est enregistrée dans la session en cours.
  5. Vous pouvez également cliquer sur Réinitialiser à n'importe quel moment pendant la session pour annuler vos modifications et actualiser la page avec les paramètres actuellement stockés.
  6. Pour terminer la session et déployer la configuration lors de l'exécution, cliquez sur Activer sous Centre des modifications.
