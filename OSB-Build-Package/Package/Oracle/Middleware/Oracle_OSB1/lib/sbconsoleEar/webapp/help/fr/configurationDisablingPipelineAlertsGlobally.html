Désactivation globale des alertes de pipeline

En ce qui concerne les services proxy, pour désactiver les alertes de pipeline pour tous les services proxy au niveau du domaine :

  1. Si vous ne l'avez pas déjà fait, cliquez sur Créer afin de créer une session ou cliquez sur Modifier pour accéder à une session existante. Reportez-vous à Utilisation du centre des modifications.
  2. Sélectionnez Opérations > Paramètres globaux.
  3. Désélectionnez la case à cocher Activer les alertes de pipeline pour que le système arrête d'exécuter les actions d'alerte de pipeline pour les services proxy.
    4. Remarque : Cette option remplace tous les paramètres Alertes de pipeline que vous pouvez sélectionner pour des services proxy précis. Reportez-vous à Configuration des paramètres opérationnels des services proxy.
  4. Cliquez sur Mettre à jour. La configuration des alertes de pipeline est enregistrée dans la session en cours.
  5. Vous pouvez également cliquer sur Réinitialiser à n'importe quel moment pendant la session pour annuler vos modifications et actualiser la page avec les paramètres actuellement stockés.
  6. Pour terminer la session et déployer la configuration lors de l'exécution, cliquez sur Activer sous Centre des modifications.
