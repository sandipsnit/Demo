Désactivation globale des alertes SLA

Pour désactiver les alertes de contrat de niveau de service (SLA) pour tous les services au niveau du domaine :

  1. Si vous ne l'avez pas déjà fait, cliquez sur Créer afin de créer une session ou cliquez sur Modifier pour accéder à une session existante. Reportez-vous à Utilisation du centre des modifications.
  2. Sélectionnez Opérations > Paramètres globaux.
  3. Désélectionnez la case à cocher Activer les alertes de contrat de niveau de service pour que le système arrête d'évaluer les règles d'alerte pour tous les services dans votre configuration.
    4. Remarque : Cette option remplace tous les paramètres Alertes de contrat de niveau de service que vous pouvez sélectionner pour des services fonctionnels ou proxy précis. Reportez-vous à Configuration de paramètres opérationnels pour les services fonctionnels et Configuration des paramètres opérationnels des services proxy.
  4. Cliquez sur Mettre à jour. La configuration des alertes de contrat de niveau de service (SLA) est enregistrée dans la session en cours.
  5. Vous pouvez également cliquer sur Réinitialiser à n'importe quel moment pendant la session pour annuler vos modifications et actualiser la page avec les paramètres actuellement stockés.
  6. Pour terminer la session et déployer la configuration lors de l'exécution, cliquez sur Activer sous Centre des modifications.
