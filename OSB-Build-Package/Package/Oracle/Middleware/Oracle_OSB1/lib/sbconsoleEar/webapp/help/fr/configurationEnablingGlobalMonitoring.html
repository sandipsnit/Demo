Activation de la surveillance globale des services

Pour activer la surveillance de tous les services du domaine, procédez comme suit :

  1. Si vous ne l'avez pas déjà fait, cliquez sur Créer afin de créer une session ou cliquez sur Modifier pour accéder à une session existante. Reportez-vous à Utilisation du centre des modifications.
  2. Sélectionnez Opérations > Paramètres globaux.
  3. Cochez la case Activer la surveillance pour que le système commence à collecter des statistiques de surveillance pour tous les services dont la surveillance est activée au niveau service.
  4. Cliquez sur Mettre à jour pour enregistrer tous les paramètres sur cette page.
  5. Vous pouvez également cliquer sur Réinitialiser à n'importe quel moment pendant la session pour annuler vos modifications et actualiser la page avec les paramètres actuellement stockés.
  6. Pour terminer la session et déployer la configuration lors de l'exécution, cliquez sur Activer sous Centre des modifications.
