Activation globale des alertes de contrat de niveau de service

Bien que vous puissiez configurer les alertes de contrat de niveau de service indépendamment de la surveillance, il existe une interaction entre ces deux paramètres à l'exécution. En effet, si la surveillance globale est activée, vous pouvez activer ou désactiver les alertes de contrat de niveau de service. Toutefois, si elle est désactivée, ces alertes seront en réalité désactivées car les conditions de règle d'alerte de contrat de niveau de service dépendent des statistiques de surveillance en cours d'évaluation.

Pour activer les alertes de contrat de niveau de service pour tous les services du domaine, procédez comme suit :

  1. Si vous ne l'avez pas déjà fait, cliquez sur Créer afin de créer une session ou cliquez sur Modifier pour accéder à une session existante. Reportez-vous à Utilisation du centre des modifications.
  2. Sélectionnez Opérations > Paramètres globaux.
  3. Cochez la case Activer les alertes de contrat de niveau de service de sorte que le système commence à évaluer les règles d'alerte pour tous les services de votre configuration.
  4. Cliquez sur Mettre à jour pour enregistrer tous les paramètres sur cette page.
  5. Vous pouvez également cliquer sur Réinitialiser à n'importe quel moment pendant la session pour annuler vos modifications et actualiser la page avec les paramètres actuellement stockés.
  6. Pour terminer la session et déployer la configuration lors de l'exécution, cliquez sur Activer sous Centre des modifications.
