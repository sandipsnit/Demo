Recherche de destinations d'alertes

  1. Sélectionnez Opérations > Recherche intelligente.
  2. Dans la liste déroulante Type, sélectionnez Destinations d'alertes, puis cliquez sur Rechercher. La page Récapitulatif des destinations d'alertes affiche alors les informations répertoriées dans le tableau 26-5.
  3. Pour limiter le nombre d'éléments dans la liste ou localiser des destinations d'alertes spécifiques, procédez comme suit :
    • Cliquez sur l'icône Open pour afficher les autres filtres de recherche. Si vous indiquez plusieurs critères de recherche, seuls les résultats correspondant à tous les critères seront renvoyés.
    • Filtrer en fonction du nom et du chemin de destination d'alerte. Dans les champs Nom et Chemin, saisissez le nom et le chemin de la cible de recherche, puis cliquez sur Rechercher. Le chemin est constitué du nom de projet et du nom du dossier dans lequel la destination d'alerte réside. Vous pouvez utiliser les caractères génériques * et ? afin d'effectuer une recherche plus générale.
    • Filtrer par cible de destination d'alerte. Dans la zone à sélections multiples Cible, sélectionnez au moins une cible de destination d'alerte : Interruption SNMP, Génération de rapports, Journalisation d'alertes, Courrier électronique ou JMS. Puis, cliquez sur Rechercher. Seules les destinations d'alertes ayant au moins une des cibles sélectionnées seront affichées. Pour ne pas prendre en compte les critères de filtre de destination d'alerte, ne sélectionnez aucune cible de destination d'alerte.
    • Filtrer par modèle de recherche. Dans la zone de texte Modèle de recherche (n'importe quelle chaîne), indiquez une chaîne, puis cliquez sur Rechercher. Le système utilise cette chaîne pour effectuer une recherche dans tous les champs Description des destinations d'alertes, ainsi que dans les champs détaillés spécifiques des destinations de courrier électronique et JMS. Si la chaîne apparaît dans un des champs de destination d'alerte, les destinations d'alertes correspondant aux critères de recherche sont affichées.

      • Tableau 26-5 Informations récapitulatives sur les destinations d'alertes
      Propriété
      Description
      Nom
      Nom de la ressource de destination d'alerte. Ce champ est un lien permettant d'accéder à la page Visualiser la destination d'alerte - Configuration. Reportez-vous à Localisation de destinations d'alertes.
      Chemin
      Projet associé à la destination d'alerte. Si cette dernière réside dans un dossier de projet, ce dossier est également répertorié. Le chemin est affiché au format suivant :
      project-name/root-folder/ . . ./parent-folder
      Ce chemin est un lien permettant d'accéder au chemin correspondant dans l'explorateur de projets.
      Options
      Cliquez sur l'icône Supprimer pour supprimer une destination d'alerte spécifique. Lorsque cette ressource est référencée par d'autres, l'icône Avertissement relatif à la suppression apparaît. Vous pouvez supprimer la ressource à condition de confirmer l'opération lorsqu'un avertissement s'affiche. Toutefois, des conflits peuvent survenir, les références à la ressource supprimée n'étant pas résolues. Reportez-vous à Suppression de destinations d'alertes.

Cette page vous permet également d'effectuer les tâches suivantes :

  • Pour enlever le filtre de recherche et afficher toutes les destinations d'alerte, cliquez sur Tout visualiser.
  • Pour supprimer une destination d'alerte spécifique si elle n'est pas référencée par d'autres ressources, cochez la case en regard de celle-ci et cliquez sur Supprimer.
  • Pour personnaliser votre vue des informations sur les paramètres opérationnels, cliquez sur l'icône Outil de personnalisation de table. Reportez-vous à Personnalisation des vues tabulaires.
