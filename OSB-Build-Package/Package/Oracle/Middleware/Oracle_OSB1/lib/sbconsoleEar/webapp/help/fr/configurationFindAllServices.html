Recherche de tous les services (services proxy et fonctionnels)

  1. Sélectionnez Opérations > Recherche intelligente.
  2. Dans la liste déroulante Type, sélectionnez Tous les services, puis cliquez sur Rechercher. La page Récapitulatif de tous les services affiche les informations répertoriées dans le tableau 26-1.
  3. Pour limiter le nombre d'éléments dans la liste ou localiser des services spécifiques, procédez comme suit :
    • Cliquez sur l'icône Open pour afficher les autres filtres de recherche. Si vous indiquez plusieurs critères de recherche, seuls les résultats correspondant à tous les critères seront renvoyés.
    • Filtrer par nom de service. Dans le champ Nom, saisissez le nom de la cible de recherche ou entrez des caractères génériques (utilisez les caractères génériques * et ? pour exécuter une recherche plus générale), puis cliquez sur Rechercher.

      • Cette méthode de recherche est conseillée si le nom du service est unique dans tous les projets et chemins.

    • Filtrer par chemin de service. Dans le champ Chemin, entrez le chemin de la cible de recherche, c'est-à-dire le nom de projet et le nom du dossier dans lequel les services résident. Vous pouvez utiliser les caractères génériques * et ? pour effectuer une recherche plus générale. Tous les services qui résident dans ce chemin sont affichés.

      • Le format du champ Chemin est le suivant :
      project-name/root-folder/ . . ./parent-folder

      Si un service se trouve directement sous le projet, le format est le suivant : project-name

    • Filtrer par nom et chemin de service. Cette méthode de recherche est conseillée si plusieurs services portant le même nom résident dans différents chemins.
    • Filtrer par état de service. Dans la liste déroulante Etat, sélectionnez l'état du service (Activé ou Désactivé), puis cliquez sur Rechercher. Sélectionnez Tout afin que les critères de filtre d'état de service ne soient pas pris en compte.
    • Filtrer par surveillance de service. Dans la liste déroulante Surveillance, sélectionnez le statut de surveillance du service (Activé ou Désactivé), puis cliquez sur Rechercher. Sélectionnez Tout afin que les critères de filtre de surveillance de service ne soient pas pris en compte.
    • Filtrer par paramètre Alertes de contrat de niveau de service du service. Dans la liste déroulante Alertes de contrat de niveau de service, sélectionnez le paramètre d'alertes de contrat de niveau de service pour le service concerné (Activé ou Désactivé), puis cliquez sur Rechercher. Sélectionnez Tout afin que les critères de filtre d'alerte de contrat de niveau de service ne soient pas pris en compte.

      • Tableau 26-1 Informations récapitulatives sur les services 
      Propriété
      Description
      Nom
      Nom affecté au service. Ce nom est un lien permettant d'accéder à la page Paramètres opérationnels. Reportez-vous à Configuration des paramètres opérationnels des services proxy ou Configuration de paramètres opérationnels pour les services fonctionnels.
      Chemin
      Projet associé au service. Si le service réside dans un dossier de projet, ce dossier est également répertorié. Le chemin est affiché au format suivant :
      project-name/root-folder/ . . ./parent-folder
      Ce chemin est un lien permettant d'accéder au chemin correspondant dans l'explorateur de projets.
      Type
      Type du service parent : service proxy ou service fonctionnel.
      Etat
      Etat du service : Activé ou Désactivé.
      Surveillance
      Statut de surveillance du service : Activé ou Désactivé.
      Alertes de contrat de niveau de service
      Statut des alertes de contrat de niveau de service : Activé ou Désactivé ; niveau à partir duquel les alertes sont activées : Normal (N), Avertissement (A), Mineur (Mn), Majeur (Mj), Critique (C) ou Fatal (F).
      Alertes de pipeline
      Pour les services proxy uniquement :
      Statut des alertes de pipeline : Activé ou Désactivé ; niveau à partir duquel les alertes sont activées : Normal (N), Avertissement (A), Mineur (Mn), Majeur (Mj), Critique (C) ou Fatal (F).
      Rapports
      Pour les services proxy uniquement :
      Statut de la génération de rapports sur les messages du service : Activé ou Désactivé.
      Journaux
      Pour les services proxy et ressources de Fractionnement-Fusion uniquement :
      Statut de journalisation : Activé ou Désactivé ; niveau à partir duquel la journalisation est activée : Débogage (D), Informations (I), Avertissement (A) ou Erreur (E).
      Fonction de trace
      Pour les services proxy et ressources de Fractionnement-Fusion uniquement :
      Statut de la fonction de trace d'exécution du service : Activé ou Désactivé.
      Mettre en mémoire cache
      Pour les services fonctionnels uniquement. Permet d'indiquer si la mise en mémoire cache des résultats est activée ou désactivée.
      Actions
      Pour les services proxy : l'icône Modifier le flux de messages est un lien permettant d'accéder au pipeline de ce service proxy.

Cette page vous permet également d'effectuer les tâches suivantes :

  • Pour enlever le filtre de recherche et afficher tous les services, cliquez sur Tout visualiser.
  • Pour annuler vos modifications et actualiser la page avec les paramètres actuellement stockés en utilisant les mêmes critères de recherche, cliquez sur Réinitialiser.
  • A partir d'une session, pour activer ou désactiver des paramètres de niveau service, cochez ou désélectionnez la case en regard des paramètres concernés et cliquez sur Mettre à jour.
    • Remarque : L'effet à l'exécution des paramètres de niveau service dépend des paramètres globaux associés. Vous devez activer à la fois les paramètres globaux et ceux définis au niveau du service pour que ce dernier soit entièrement activé à l'exécution. En outre, les alertes de contrat de niveau de service dépendent de l'état de la surveillance globale et la surveillance de niveau service. Toutes deux doivent être activées pour que les alertes de contrat de niveau de service soient activées à l'exécution. Reportez-vous à Activation des paramètres globaux.
    Remarque : De plus, l'état du service doit également être activé.
  • Pour personnaliser votre vue des informations sur les paramètres opérationnels, cliquez sur l'icône Outil de personnalisation de table. Reportez-vous à Personnalisation des vues tabulaires.
