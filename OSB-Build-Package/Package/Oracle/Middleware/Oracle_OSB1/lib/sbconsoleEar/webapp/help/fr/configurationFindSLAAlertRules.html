Recherche de règles d'alerte de contrat de niveau de service

  1. Sélectionnez Opérations > Recherche intelligente.
  2. Dans la liste déroulante Type, sélectionnez Alertes de contrat de niveau de service, puis cliquez sur Rechercher. La page Récapitulatif des règles d'alerte SLA affiche les informations répertoriées dans le tableau 26-6.
  3. Pour limiter le nombre d'éléments dans la liste ou localiser des règles d'alerte de contrat de niveau de service spécifiques, procédez comme suit :
    • Cliquez sur l'icône Open pour afficher les autres filtres de recherche. Si vous indiquez plusieurs critères de recherche, seuls les résultats correspondant à tous les critères seront renvoyés.
    • Filtrer par nom de règle d'alerte. Dans le champ Nom, saisissez le nom de la cible de recherche ou entrez des caractères génériques (utilisez les caractères génériques * et ? pour effectuer une recherche plus générale), puis cliquez sur Rechercher.

      • Cette méthode de recherche est conseillée si le nom de la règle d'alerte est unique dans tous les projets et chemins.

    • Filtrer par chemin de service parent. Dans le champ Chemin, entrez le chemin du service parent, c'est-à-dire le nom de projet et le nom du dossier dans lequel le service parent réside. Vous pouvez utiliser les caractères génériques * et ? pour exécuter une recherche plus générale. Toutes les règles d'alerte dont les services parent résident dans ce chemin sont affichées.

      • Le format du champ Chemin est le suivant :
      project-name/root-folder/ . . ./parent-folder

      Si un service se trouve directement sous le projet, le format est le suivant : project-name

    • Filtrer par service parent. Dans le champ Service parent, cliquez sur Parcourir pour afficher la page Sélectionner un service. Sélectionnez le service parent dans la liste et cliquez sur Soumettre. Cliquez sur Effacer pour enlever les critères de filtre de service parent ou cliquez sur Rechercher.
      • Remarque : L'élément sélectionné dans le champ Service parent limite vos choix dans le champ Type de service.
    • Filtrer par type de service parent. Dans la liste déroulante Type de service, sélectionnez Services proxy ou Services fonctionnels, puis cliquez sur Rechercher. Sélectionnez Tout afin que les critères de filtre de type de service parent ne soient pas pris en compte.
      • Remarque : L'élément sélectionné dans le champ Type de service limite vos choix dans le champ Service parent.
    • Filtrer par état de la règle. Dans la liste déroulante Etat de la règle, sélectionnez l'état de la règle d'alerte (Activé ou Désactivé), puis cliquez sur Rechercher. Sélectionnez Tout afin que les critères de filtre d'état de la règle ne soient pas pris en compte.
    • Filtrer par gravité. Dans la liste déroulante Gravité, sélectionnez une valeur afin de limiter les règles d'alerte de contrat de niveau de service au niveau indiqué, puis cliquez sur Rechercher.

      • Cochez la case ou supérieur pour limiter votre recherche au niveau de gravité indiqué ou supérieur (les niveaux sont affichés du plus général au plus restrictif) : Normal, Avertissement, Mineur, Critique et Fatal.

Cette page vous permet également d'effectuer les tâches suivantes :

  • Pour enlever le filtre de recherche et afficher toutes les règles d'alerte de contrat de niveau de service, cliquez sur Tout visualiser.
  • Pour annuler vos modifications et actualiser la page avec les paramètres actuellement stockés en utilisant les mêmes critères de recherche, cliquez sur Réinitialiser.
  • A partir d'une session, pour activer ou désactiver une règle d'alerte de contrat de niveau de service, cochez ou désélectionnez la case en regard de la règle concernée et cliquez sur Mettre à jour.
  • Pour personnaliser votre vue des informations sur les paramètres opérationnels, cliquez sur l'icône Outil de personnalisation de table. Reportez-vous à Personnalisation des vues tabulaires.
