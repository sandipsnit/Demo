Démarrage de la console d'administration d'Oracle Service Bus

  1. Démarrez Oracle Service Bus via l'une des méthodes suivantes :
    • Dans le menu Démarrer de Windows, sélectionnez :

      • Démarrer > Tous les programmes > Oracle WebLogic > Projets utilisateur > domain_name > Démarrer le serveur pour le domaine Oracle Service Bus.

      Où domain_name représente le nom affecté à votre domaine Oracle Service Bus lors de sa création. Projets utilisateur désigne l'emplacement par défaut dans lequel votre domaine est créé.

    • Appelez la commande startWebLogic à la racine du domaine Oracle Service Bus : startWebLogic.cmd ou startWebLogic.sh

      • Une fenêtre de commande apparaît. Elle contient les informations de statut de WebLogic Server.

      Si vous avez lancé le serveur en mode de production, vous devez saisir un nom utilisateur et un mot de passe valides pour pouvoir le démarrer.

  2. Lorsque le serveur démarre, entrez l'URL suivante dans votre navigateur Web :

    3. http://hostname:port/sbconsole

    où host représente le nom de l'ordinateur sur lequel WebLogic Server est en cours d'exécution et où port désigne le numéro de port.

    Par exemple, si WebLogic Server est en cours d'exécution sur votre ordinateur local et qu'il utilise la configuration de port par défaut, saisissez l'URL suivante dans votre navigateur Web :

    http://localhost:7001/sbconsole

    La page de connexion à la console d'administration d'Oracle Service Bus apparaît.

  3. Saisissez les nom utilisateur et mot de passe que vous avez indiqués lors du processus d'installation.
    4. Remarque : Les nom utilisateur et mot de passe du domaine d'exemples d'Oracle Service Bus sont weblogic / welcome1.
  4. Cliquez sur Connexion.

Pour vous déconnecter, cliquez sur Se déconnecter dans la bannière au-dessus du panneau droit de la console d'administration d'Oracle Service Bus.
