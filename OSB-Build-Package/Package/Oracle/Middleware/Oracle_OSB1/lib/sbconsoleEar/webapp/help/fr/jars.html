fichiers JAR

Un fichier JAR (Java ARchive) est un fichier compressé contenant un ensemble de classes Java. Il permet de stocker les classes Java compilées et les métadonnées associées dont un programme peut être constitué. Un fichier JAR agit comme une bibliothèque de programmes pouvant être appelée pour les éléments de code Java (un seul lien de compilation donne ainsi accès à plusieurs éléments, au lieu d'exiger des liaisons pour chaque élément). Pour plus d'informations sur les fichiers JAR, reportez-vous à l'adresse suivante : http://java.sun.com/docs/books/tutorial/deployment/jar/

Les ressources JAR dans Oracle Service Bus sont utilisées dans :

  • les actions d'appel Java,
  • les services fonctionnels basés sur EJB,
  • les services fonctionnels et proxy basés sur Tuxedo.
