Suppression des liaisons JCA

Si un service fonctionnel ou un service proxy est basé sur la liaison JCA, vous devez enlever cette dernière des services pour pouvoir la supprimer.

  1. Si vous ne l'avez pas déjà fait, cliquez sur Créer afin de créer une session ou cliquez sur Modifier pour accéder à une session existante. Reportez-vous à Utilisation du centre des modifications.
  2. Sélectionnez Navigateur Web de ressources > Liaisons JCA.
  3. Cliquez sur l'icône Supprimer dans le champ Options de la liaison JCA à supprimer.

    4. La liaison JCA est supprimée dans la session en cours. Si un service fonctionnel ou un service proxy a été configuré de façon à utiliser une liaison JCA, une icône Avertissement relatif à la suppression indique que vous pouvez supprimer cette liaison, à condition de confirmer l'opération lorsqu'un avertissement s'affiche. Toutefois, des conflits peuvent survenir, les références du service à la liaison JCA supprimée n'étant pas résolues.

  4. Pour terminer la session et déployer la configuration lors de l'exécution, cliquez sur Activer sous Centre des modifications.
