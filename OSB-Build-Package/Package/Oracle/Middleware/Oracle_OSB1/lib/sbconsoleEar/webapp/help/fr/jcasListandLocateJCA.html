Localisation des liaisons JCA

La page Récapitulatif des liaisons JCA répertorie les liaisons JCA que vous avez ajoutées au domaine Oracle Service Bus.

  1. Effectuez l'une des opérations suivantes :
    • Sélectionnez Explorateur de projets pour afficher la page Vue des projets ou la page Vue du projet/Vue de dossier. Naviguez ensuite dans les projets et les dossiers pour rechercher la liaison JCA.
    • Sélectionnez Navigateur Web de ressources > Liaisons JCA. La page Récapitulatif des liaisons JCA affiche les informations répertoriées dans le tableau 12-1. Pour obtenir une description plus détaillée des propriétés, reportez-vous à Localisation des liaisons JCA.
  2. Pour localiser une liaison JCA spécifique, procédez comme suit :
    • Pour limiter le nombre d'éléments dans la liste, vous pouvez filtrer par nom et chemin avant de cliquer sur Rechercher.

      • Le chemin est constitué du nom de projet et du nom des dossiers dans lesquels la ressource réside.

    • Cliquez sur Tout visualiser pour enlever les filtres de recherche et afficher toutes les liaisons JCA.

      • Tableau 12-1 Informations sur les liaisons JCA
      Propriété
      Description
      Nom de la liaison JCA
      Nom unique affecté à la liaison JCA. Cliquez sur ce nom pour que la page Visualiser les détails de liaison JCA apparaisse. Reportez-vous à Modification des liaisons JCA.
      Chemin
      Nom de projet et nom du dossier dans lequel la liaison JCA réside. Cliquez sur ce nom pour afficher le projet ou le dossier contenant cette ressource. Reportez-vous à Qualification des noms de ressource à l'aide de projets et de dossiers.
      Action
      Contient une icône Générer qui permet de générer un service WSDL et JCA Oracle Service Bus à partir de la liaison JCA.
      Options
      Contient une icône Supprimer. Si un service fonctionnel ou un service proxy a été configuré de façon à utiliser une liaison JCA, une icône Avertissement relatif à la suppression indique que vous pouvez supprimer cette liaison, à condition de confirmer l'opération lorsqu'un avertissement s'affiche. Toutefois, des conflits peuvent survenir, les références du service à la liaison JCA supprimée n'étant pas résolues. Reportez-vous à Suppression des liaisons JCA.
