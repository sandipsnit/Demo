Fichiers MFL

Un document MFL (Message Format Language) est un document XML spécialisé servant à décrire la mise en page des données binaires. Il s'agit d'un langage propriétaire Oracle permettant de définir les règles de transformation des données binaires formatées en données XML. Un document MFL est conforme au fichier mfl.dtd, qui inclut les éléments et attributs utilisés pour décrire chaque champ de données, ainsi que les regroupements de champs (groupes), la répétition et l'agrégation.

Lorsque vous créez des services fonctionnels ou proxy de type Service de messagerie, vous pouvez sélectionner des types MFL comme type du message de demande ou de réponse du service.

Vous pouvez utiliser le constructeur de format Oracle pour créer des fichiers MFL. Lorsque vous définissez la hiérarchie d'un enregistrement binaire, la mise en page de champs et le regroupement de champs et de groupes, les informations sont enregistrées sous forme de document MFL pouvant ensuite être utilisé pour les traductions à l'exécution. Un document MFL peut également être employé dans le constructeur de format pour générer la DTD correspondante qui décrit son modèle de contenu. Pour plus d'informations sur l'utilisation du constructeur de format, reportez-vous à "Format Builder" dans le manuel Oracle Fusion Middleware Developer's Guide for Oracle Service Bus à l'adresse suivante : http://www.oracle.com/pls/as111170/lookup?id=OSBDV763.
