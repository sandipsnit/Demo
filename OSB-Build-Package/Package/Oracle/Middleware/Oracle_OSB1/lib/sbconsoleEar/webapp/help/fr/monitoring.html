Surveillance

Lorsque vous créez un service proxy ou fonctionnel, la surveillance de ce service est désactivée par défaut. Pour savoir comment activer la surveillance des services proxy, reportez-vous à Configuration des paramètres opérationnels des services proxy ; pour les services fonctionnels, reportez-vous à Configuration de paramètres opérationnels pour les services fonctionnels.

Pour activer ou désactiver la surveillance de tous les services pour lesquels cette fonction a été activée ou désactivée de façon individuelle, utilisez l'option Activer la surveillance sur la page Opérations > Paramètres globaux. Reportez-vous à Activation des paramètres globaux.

Pour en savoir plus sur la surveillance à l'exécution, reportez-vous à "Monitoring Oracle Service Bus at Run Time" dans le manuel Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus à l'adresse suivante : http://www.oracle.com/pls/as111170/lookup?id=OSBAG472.
