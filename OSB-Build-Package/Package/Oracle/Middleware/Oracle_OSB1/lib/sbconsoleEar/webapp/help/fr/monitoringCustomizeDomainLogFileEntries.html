Personnalisation de la vue des entrées de fichier journal de domaine

  1. Cliquez sur Personnaliser cette table. Des champs supplémentaires apparaissent.

    2. Vous pouvez cliquer à tout moment sur le lien Personnaliser cette table pour fermer cette table et conserver les paramètres d'origine.

  2. Dans les champs Filtre, effectuez les opérations suivantes :
    1. Dans le champ Heure, sélectionnez l'intervalle pour lequel vous souhaitez visualiser les entrées de journal.
    2. Dans le champ Date/Heure de début, indiquez une date/heure de début au format MM/DD/YYYY HH:MM:SS. Par exemple, saisissez 10/25/06 08:39:48. Les champs Date/Heure de début et Date/Heure de fin permettent d'indiquer une fenêtre de temps au cours de laquelle vous souhaitez visualiser les entrées de journal.
    3. Dans le champ Date/Heure de fin, indiquez une date/heure de fin au format MM/DD/YYYY HH:MM:SS. Par exemple, saisissez 10/25/06 13:20:51. Les champs Date/Heure de début et Date/Heure de fin permettent d'indiquer une fenêtre de temps au cours de laquelle vous souhaitez visualiser les entrées de journal.
    4. Dans la zone de texte Critères, définissez des critères de filtrage (chaîne de texte). Ces critères sont indiqués sous forme de chaîne dans le langage d'interrogation WLDF. Ce dernier prend en charge les opérateurs booléens AND, OR et NOT, ainsi que les opérateurs relationnels. Pour plus d'informations sur la syntaxe d'interrogation, reportez-vous à "WLDF Query Language" dans le manuel Oracle Fusion Middleware Configuring and Using the Diagnostics Framework for Oracle WebLogic Server à l'adresse suivante : http://www.oracle.com/pls/as111170/lookup?id=WLDFC316.
  3. Dans le champ Colonnes, sélectionnez les colonnes à afficher :
    1. Sélectionnez un nom de colonne dans le champ Disponible.
    2. Cliquez sur la flèche pour déplacer ce nom de colonne dans le champ Choisi.
    3. Répétez cette opération jusqu'à ce que tous les noms de colonne à afficher soient répertoriés dans le champ Choisi.

      4. Dans le champ Choisi, vous pouvez utiliser les flèches vers le haut et vers le bas pour re-trier les noms de colonne, si nécessaire.

  4. Dans le champ Nombre de lignes affichées par page, sélectionnez le nombre d'entrées de journal à afficher sur une seule page.
  5. Dans le champ Nombre maximal de résultats renvoyés, sélectionnez le nombre maximal d'entrées de journal à afficher au total ou sélectionnez Tout afficher pour toutes les afficher.
  6. Effectuez l'une des opérations suivantes :
    • Pour enregistrer les nouveaux paramètres, cliquez sur Appliquer.
    • Pour annuler les modifications et conserver les paramètres d'origine, cliquez sur Réinitialiser.
