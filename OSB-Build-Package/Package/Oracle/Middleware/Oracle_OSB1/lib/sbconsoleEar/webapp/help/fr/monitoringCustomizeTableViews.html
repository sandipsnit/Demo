Personnalisation des vues tabulaires

Utilisez l'outil de personnalisation de table pour afficher les informations sur les tables selon les paramètres que vous avez indiqués.

L'ordre de tri par défaut des tables est déterminé par leur première colonne.

  1. Cliquez sur l'icône Outil de personnalisation de table. Des champs supplémentaires apparaissent.

    2. Vous pouvez cliquer sur Annuler à tout moment pour fermer la table et conserver les paramètres d'origine.

  2. Dans le champ Colonnes, sélectionnez les colonnes à afficher :
    1. Sélectionnez un nom de colonne dans le champ Colonnes disponibles.
    2. Cliquez sur la flèche pour déplacer ce nom de colonne dans le champ Colonnes sélectionnées.

      3. Un astérisque indique une colonne obligatoire.

    3. Répétez cette opération jusqu'à ce que tous les noms de colonne à afficher soient répertoriés dans le champ Colonnes sélectionnées.

      4. Dans le champ Colonnes sélectionnées, vous pouvez utiliser les flèches vers le haut et vers le bas pour re-trier les noms de colonne, si nécessaire.

  3. Pour que certaines colonnes soient masquées, déplacez le nom des colonnes en question du champ Colonnes sélectionnées vers le champ Colonnes disponibles.
  4. Dans la liste déroulante Lignes, sélectionnez le nombre de lignes à afficher sur une seule page.
  5. Effectuez l'une des opérations suivantes :
    • Pour enregistrer ces paramètres et fermer l'outil de personnalisation de table, cliquez sur Appliquer.
    • Pour annuler les modifications et conserver les paramètres d'origine, cliquez sur Réinitialiser.
