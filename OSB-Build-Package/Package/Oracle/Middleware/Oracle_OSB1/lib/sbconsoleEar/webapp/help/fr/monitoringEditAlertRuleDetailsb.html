Modification des règles d'alerte

  1. Si vous ne l'avez pas déjà fait, cliquez sur Créer afin de créer une session ou cliquez sur Modifier pour accéder à une session existante. Reportez-vous à Utilisation du centre des modifications.
  2. Localisez la règle d'alerte. Reportez-vous à Localisation des règles d'alerte.
  3. Cliquez sur le nom de la règle d'alerte.
  4. Pour modifier les champs des pages de configuration, cliquez sur l'icône Modifier de la page appropriée ou sur le bouton Modifier au bas de la page. Pour obtenir la description des pages et des champs, reportez-vous à Création et modification des règles d'alerte.
  5. Exécutez l'une des actions suivantes :
    • Pour accéder à la page suivante, cliquez sur Suivant.
    • Pour revenir à la page précédente, cliquez sur Préc.
    • Pour accéder à la page récapitulative, cliquez sur Dernier.
    • Pour valider les mises à jour dans la session en cours, cliquez sur Enregistrer.
    • Pour ignorer les modifications et revenir à la page Règles d'alerte de contrat de niveau de service, cliquez sur Annuler.
  6. Pour terminer la session et déployer la configuration lors de l'exécution, cliquez sur Activer sous Centre des modifications.
