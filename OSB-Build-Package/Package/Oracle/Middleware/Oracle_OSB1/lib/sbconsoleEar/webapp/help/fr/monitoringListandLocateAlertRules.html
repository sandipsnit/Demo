Visualisation des règles d'alerte

Utilisez cette page pour visualiser les informations sur les règles d'alerte répertoriées dans le tableau 25-19.

Tableau 25-19 Informations sur les règles d'alerte
Propriété
Description
Nom
Nom affecté à cette règle d'alerte. Ce nom est un lien permettant d'accéder à la page Configuration générale des règles d'alerte. Reportez-vous à Visualisation des configurations de règle d'alerte.
Etat SLA
Statut de la règle d'alerte : Activé ou Désactivé.
Description
Ce champ est masqué par défaut.
Description de la règle d'alerte.
Gravité
Gravité de l'alerte qui sera déclenchée par cette règle : Normal, Avertissement, Mineur, Majeur, Critique ou Fatal.
Aggr. Intervalle
Durée de l'intervalle d'agrégation en heures et minutes.
Date d'expiration
Date à laquelle cette règle d'alerte ne sera plus en vigueur.
Arrêter le traitement
Indique Oui ou Non.
Fréquence
Fréquence de l'alerte :
  • A chaque fois
  • Notifier une fois
Options
Cette colonne contient les fonctions suivantes :
  • Cliquez sur les flèches pour re-trier les règles d'alerte. Vous pouvez déplacer chacune des règles d'alerte vers le haut ou vers le bas de la liste. Cette opération n'est possible que lorsque plusieurs règles d'alerte sont configurées pour ce service.
  • Cliquez sur l'icône Renommer pour renommer une règle d'alerte.
  • Cliquez sur l'icône Supprimer pour supprimer une règle d'alerte spécifique. Reportez-vous à Suppression des règles d'alerte.
