Localisation des règles d'alerte

  1. Sélectionnez Navigateur Web de ressources > Services fonctionnels ou Services proxy.
  2. Cliquez sur le nom du service.
  3. Cliquez sur l'onglet Règles d'alerte de contrat de niveau de service.
  4. Pour localiser une règle d'alerte spécifique pour ce service, faites défiler les pages. Utilisez les contrôles de page au-dessus ou au-dessous de la table. Accédez à une page en sélectionnant son numéro ou utilisez les flèches pour accéder à la page suivante ou précédente, à la première ou dernière page.

Cette page vous permet également d'effectuer les tâches suivantes :

  • Pour configurer la surveillance du service, cliquez sur l'onglet Paramètres opérationnels. Reportez-vous à Configuration des paramètres opérationnels des services proxy et Configuration de paramètres opérationnels pour les services fonctionnels.
  • Pour ajouter une nouvelle règle d'alerte, cliquez sur Ajouter une valeur. Reportez-vous à Création et modification des règles d'alerte.
