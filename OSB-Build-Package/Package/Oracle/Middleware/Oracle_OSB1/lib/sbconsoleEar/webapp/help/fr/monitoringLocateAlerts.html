Localisation des alertes

  1. Sélectionnez Opérations > Tableau de bord.
  2. Cliquez sur l'onglet Alertes de contrat de niveau de service ou Alertes de pipeline pour afficher les alertes de contrat de niveau de service ou de pipeline.
  3. Cliquez sur le lien Historique étendu des alertes. Les pages Historique étendu des alertes de contrat de niveau de service et Historique étendu des alertes de pipeline présentent des informations détaillées sur l'historique des alertes.
  4. Pour limiter le nombre d'éléments dans la liste ou localiser des alertes spécifiques, procédez comme suit :
    • Cliquez sur l'icône Open pour afficher les autres filtres de recherche. Si vous indiquez plusieurs critères de recherche, seuls les résultats correspondant à tous les critères seront renvoyés.
    • Filtrer par plage de dates. Effectuez l'une des opérations suivantes :
      • Sélectionnez Tout afin que la plage de dates et les critères de filtre d'intervalle ne soient pas pris en compte et ainsi afficher toutes les alertes.
      • Dans les champs De et A, indiquez les date et heure au format MM/DD/YY HH:MM AM|PM, puis cliquez sur Rechercher. Les alertes générées au cours des date et heure saisies apparaissent.
      • Dans le champ Pour les derniers, indiquez le nombre de jours, d'heures et de minutes, puis cliquez sur Rechercher. Les alertes générées au cours de la période spécifiée apparaissent.
    • Filtrer par gravité de l'alerte. Dans la liste déroulante Gravité de l'alerte, sélectionnez une valeur pour limiter les alertes au niveau de gravité indiqué, puis cliquez sur Rechercher.

      • Cochez la case ou supérieur pour limiter la recherche au niveau de gravité indiqué ou supérieur (les niveaux sont affichés du plus général au plus restrictif) : Normal, Avertissement, Mineur, Majeur, Critique et Fatal.

    • Filtrer par service parent. Cliquez sur Parcourir pour afficher la page Sélectionner un service. Sélectionnez le service parent dans la liste et cliquez sur Soumettre. Cliquez sur Effacer pour enlever les critères de filtre de service parent ou cliquez sur Rechercher.
    • Pour les alertes de contrat de niveau de service uniquement, filtrez par Nom d'alerte (n'importe quelle chaîne). Saisissez une chaîne, y compris des caractères génériques, puis cliquez sur Rechercher. Pour plus de détails, reportez-vous à Langage d'interrogation WebLogic Diagnostics Framework.
    • Pour les alertes de pipeline uniquement, filtrez par Récapitulatif des alertes (n'importe quelle chaîne). Saisissez une chaîne, y compris des caractères génériques, puis cliquez sur Rechercher. Pour plus de détails, reportez-vous à Langage d'interrogation WebLogic Diagnostics Framework. Pour rechercher les actions d'alerte de pipeline conçues sans texte récapitulatif d'alerte, indiquez Alerte Oracle Service Bus.
    • Cliquez sur Tout visualiser pour enlever les filtres de recherche et afficher toutes les alertes.

Utilisez également les pages Historique étendu des alertes de contrat de niveau de service et Historique étendu des alertes de pipeline pour effectuer les opérations suivantes :

  • Pour afficher les données d'alerte dans un graphique à secteurs, sélectionnez Vue de graphique à secteurs.
  • Pour afficher les mêmes données dans un graphique à barres, sélectionnez Vue de graphique à barres.
  • Pour afficher les données au format tabulaire, sélectionnez Vue tabulaire.
  • Pour afficher une version mise à jour des données, cliquez sur Actualiser.
  • Pour supprimer des alertes individuelles, cochez les cases correspondantes et cliquez sur Supprimer au bas de la page.
  • Pour supprimer toutes les alertes ou celles générées au cours d'une plage de dates et d'heures donnée, cliquez sur le lien Purger l'historique des alertes en haut de la page. Reportez-vous à Purge des alertes.

Langage d'interrogation WebLogic Diagnostics Framework

Les alertes sont stockées à l'aide de WebLogic Diagnostics Framework, qui fournit son propre langage d'interrogation, y compris les caractères génériques. Pour filtrer les alertes dans l'historique étendu des alertes, utilisez la syntaxe décrite dans "WLDF Query Language" dans Oracle Fusion Middleware Configuring and Using the Diagnostics Framework for Oracle WebLogic Server à l'adresse suivante : http://www.oracle.com/pls/as111170/lookup?id=WLDFC316.
