Réinitialisation des statistiques des services

La page Etat du service permet de réinitialiser les statistiques de surveillance pour tous les services de votre configuration. Vous pouvez réinitialiser les statistiques que vous soyez connecté à une session ou non.

  1. Sélectionnez Opérations > Tableau de bord > Etat du service.
  2. Dans le champ Afficher les statistiques, sélectionnez Depuis la dernière réinitialisation. La vue Depuis la dernière réinitialisation affiche le nombre de mesures en cours d'exécution.
  3. Pour réinitialiser les statistiques de surveillance d'un service spécifique, cliquez sur l'icône Réinitialiser dans la colonne Action de ce service.

    4. Lorsque vous confirmez cette opération, le système supprime toutes les statistiques de surveillance collectées pour ce service depuis la dernière réinitialisation des statistiques. Toutefois, le système ne supprime pas celles collectées pendant l'intervalle d'agrégation en cours de ce service. En outre, une fois que vous avez cliqué sur l'icône Réinitialiser, le système recommence immédiatement à collecter les statistiques de surveillance du service.

  4. Pour réinitialiser les statistiques de surveillance de tous les services, cliquez sur le lien Réinitialiser toutes les statistiques en haut de la page.
    5. Remarque : Lorsque vous cliquez sur Réinitialiser toutes les statistiques, les statistiques de tous les services surveillés sont réinitialisées qu'elles soient affichées sur la page ou non.

    Lorsque vous confirmez la réinitialisation des statistiques, le système supprime toutes les statistiques de surveillance collectées pour l'ensemble des services de votre configuration depuis la dernière réinitialisation des statistiques. Le système ne supprime pas celles collectées pendant l'intervalle d'agrégation en cours des services. En outre, une fois que vous avez cliqué sur l'icône Réinitialiser toutes les statistiques, le système recommence immédiatement à collecter les statistiques de surveillance des services.
