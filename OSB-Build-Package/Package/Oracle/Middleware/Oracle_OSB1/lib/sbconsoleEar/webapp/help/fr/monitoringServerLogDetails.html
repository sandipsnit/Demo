Visualisation des détails des fichiers journaux de domaine

Utilisez cette page pour visualiser les détails des entrées dans les fichiers journaux de domaine présentées dans le tableau 25-18.

Tableau 25-18 Détails des journaux de domaine
Propriété
Description
Message
Description de l'événement ou de la condition.
Date
Affiche l'heure et la date auxquelles le message a été généré, dans un format propre aux paramètres locaux. La Java Virtual Machine (JVM) qui exécute chaque instance WebLogic Server fait appel au système d'exploitation de l'ordinateur hôte pour obtenir des informations sur les format et fuseau horaire locaux.
Sous-système
Indique le sous-système WebLogic Server à l'origine du message ; par exemple un conteneur Enterprise Java Bean (EJB) ou Java Messaging Service (JMS).
ID de message
Identificateur unique à six chiffres.
Tous les ID de message générés par les messages système WebLogic Server commencent par BEA- et se situent dans une plage numérique comprise entre 0 et 499 999.
Gravité
Indique l'impact ou le niveau de gravité de l'événement signalé par le message.
  • Alerte : un service spécifique présente un état inutilisable tandis que les autres parties du système continuent de fonctionner. La récupération automatique n'est pas possible ; l'administrateur doit intervenir immédiatement pour résoudre le problème.
  • Critique : une erreur système ou de service est survenue. Le système peut effectuer la récupération mais le service peut être perdu temporairement ou définitivement endommagé.
  • Urgence : le serveur présente un état inutilisable. Ce niveau de gravité indique une panne ou une urgence système grave.
  • Erreur : une erreur utilisateur est survenue. Le système ou l'application peut gérer l'erreur sans interruption et avec une dégradation limitée du service.
  • Informations : utilisé pour les opérations standard de génération de rapports ; message d'information de bas niveau.
  • Notification : message d'information avec un niveau d'importance plus élevé.
  • Avertissement : une opération ou configuration suspecte a été détectée mais cela ne risque pas d'influer sur le fonctionnement normal.
Ordinateur
Nom DNS de l'ordinateur hébergeant l'instance de serveur.
Serveur
Nom de l'instance WebLogic Server sur laquelle le message a été généré.
Thread
ID affecté par la Java Virtual Machine (JVM) au thread dont le message provient.
ID utilisateur
ID utilisateur sous lequel l'événement associé a été exécuté.
ID de transaction
Présent uniquement pour les messages journalisés dans le contexte d'une transaction.
ID de contexte
Informations de contexte permettant de mettre en corrélation les messages issus d'une demande ou d'une application particulière.
Information
Description de l'événement ou de la condition.
Cause
Cause du message.
Action
Action devant être entreprise.
