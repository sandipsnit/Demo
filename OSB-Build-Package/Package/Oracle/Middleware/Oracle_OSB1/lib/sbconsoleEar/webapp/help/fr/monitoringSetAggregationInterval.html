Définition de l'intervalle d'agrégation d'un service

Utilisez la page Paramètres opérationnels pour définir l'intervalle d'agrégation d'un service particulier. L'intervalle d'agrégation correspond à la période au cours de laquelle les statistiques agrégées sont calculées en vue d'être affichées dans la console d'administration d'Oracle Service Bus. Par défaut, l'intervalle d'agrégation est défini sur 10 minutes.

  1. Si vous ne l'avez pas déjà fait, cliquez sur Créer afin de créer une session ou cliquez sur Modifier pour accéder à une session existante. Reportez-vous à Utilisation du centre des modifications.
  2. Sélectionnez Navigateur Web de ressources > Services fonctionnels ou Services proxy.
  3. Cliquez sur le nom du service.
  4. Sélectionnez l'onglet Paramètres opérationnels.
  5. Dans le champ Intervalle d'agrégation, sélectionnez la longueur de l'intervalle d'agrégation en heures et en minutes, puis cliquez sur Mettre à jour.

    6. Si vous spécifiez un nombre d'heures supérieur à 1, la valeur par défaut indiquée pour les minutes est toujours 0. Toutefois, si vous spécifiez la valeur 0 ou 1 pour les heures, vous pouvez également indiquer des minutes.

  6. Cliquez sur Mettre à jour pour valider les mises à jour dans la session en cours.
  7. Pour terminer la session et déployer la configuration lors de l'exécution, cliquez sur Activer sous Centre des modifications.
