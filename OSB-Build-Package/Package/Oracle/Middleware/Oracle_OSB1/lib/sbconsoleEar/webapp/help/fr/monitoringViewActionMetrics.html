Visualisation des mesures d'action

Utilisez cette page pour visualiser les informations de surveillance de service, présentées dans le tableau 25-10, et les mesures d'action des services proxy contenant des pipelines, présentées dans le tableau 25-14.

Tableau 25-14 Mesures d'action
Propriété
Description
Noeud
Affichage hiérarchique des actions imbriquées.
Nombre
Si vous avez sélectionné Intervalle d'agrégation en cours dans le champ Afficher les statistiques, il s'agit du nombre d'appels au cours de l'intervalle d'agrégation en cours.
Si vous avez sélectionné Depuis la dernière réinitialisation dans le champ Afficher les statistiques, il s'agit du nombre d'appels effectués au cours de la période écoulée depuis la dernière réinitialisation des statistiques.
Ce nombre est différent du nombre de messages car une action peut être appelée plusieurs fois par message.
Temps moy
Si vous avez sélectionné Intervalle d'agrégation en cours dans le champ Afficher les statistiques, il s'agit du temps moyen d'exécution par appel au cours de l'intervalle d'agrégation en cours.
Si vous avez sélectionné Depuis la dernière réinitialisation dans le champ Afficher les statistiques, il s'agit du temps moyen d'exécution par appel au cours de la période écoulée depuis la dernière réinitialisation des statistiques.
Durée totale
Si vous avez sélectionné Intervalle d'agrégation en cours dans le champ Afficher les statistiques, il s'agit du temps total d'exécution au cours de l'intervalle d'agrégation en cours.
Si vous avez sélectionné Depuis la dernière réinitialisation dans le champ Afficher les statistiques, il s'agit du temps total d'exécution au cours de la période écoulée depuis la dernière réinitialisation des statistiques.
