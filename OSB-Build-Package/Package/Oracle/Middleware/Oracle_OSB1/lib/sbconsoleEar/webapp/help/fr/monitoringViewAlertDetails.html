Visualisation des détails des alertes

Cette page affiche les détails d'alertes spécifiques, y compris les informations répertoriées dans le tableau 25-4.

Tableau 25-4 Détails des alertes
Propriété
Description
Nom d'alerte
Pour les alertes de contrat de niveau de service uniquement :
Nom affecté à l'alerte. Ce nom est un lien permettant d'accéder à la page Configuration des règles d'alerte. Reportez-vous à Visualisation des configurations de règle d'alerte.
Récapitulatif des alertes
Pour les alertes de pipeline uniquement :
Brève description de l'action d'alerte de pipeline ou alerte Oracle Service Bus (valeur par défaut définie pour les actions d'alerte de pipeline conçues sans texte récapitulatif d'alerte).
Description
Description de la règle d'alerte.
Horodatage
  • Pour les alertes de pipeline : date et heure auxquelles l'alerte a été générée.
  • Pour les alertes de contrat de niveau de service : date et heure auxquelles la règle d'alerte a renvoyé la valeur True pour la dernière fois.
Gravité
Gravité (définie par l'utilisateur) de l'alerte.
  • Fatal
  • Critique
  • Majeur
  • Mineur
  • Avertissement
  • Normal
Service
Nom et chemin du service associé à l'alerte. Ce champ est un lien permettant d'accéder à la page Détails de la surveillance des services (uniquement lorsque le service est surveillé de façon active). Reportez-vous à Visualisation des mesures de service.
Type de service
Type du service parent : service proxy ou service fonctionnel.
Seuls les services proxy peuvent définir les alertes de pipeline. Par conséquent, pour les alertes de pipeline, tous les services parent sont des services proxy.
Serveur
Nom du serveur sur lequel l'alerte a été générée.
Pour les alertes de contrat de niveau de service uniquement : N/A apparaît.
Annotation
Zone de texte dans laquelle vous pouvez saisir des remarques sur l'alerte. Une icône Annotation apparaît dans l'historique pour les alertes ayant des annotations.
