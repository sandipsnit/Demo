Visualisation des mesures des URI endpoint de service fonctionnel

Utilisez cette page pour visualiser les informations de surveillance de service, présentées dans le tableau 25-10, et les mesures des URI endpoint pour les services fonctionnels, présentées dans le tableau 25-15. Pour plus d'informations, reportez-vous à "Managing Endpoint URIs for Business Services" dans le manuel Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus à l'adresse suivante : http://www.oracle.com/pls/as111170/lookup?id=OSBAG595.

Tableau 25-15 Mesures des URI endpoint
Propriété
Description
URI endpoint
URI endpoint associé au service fonctionnel.
Nombre de messages
Si vous avez sélectionné Intervalle d'agrégation en cours dans le champ Afficher les statistiques, il s'agit du nombre total de messages exécutés par cet URI endpoint pendant l'intervalle d'agrégation en cours.
Si vous avez sélectionné Depuis la dernière réinitialisation dans le champ Afficher les statistiques, il s'agit du nombre total de messages exécutés par cet URI endpoint au cours de la période écoulée depuis la dernière réinitialisation des statistiques.
Nombre d'erreurs
Si vous avez sélectionné Intervalle d'agrégation en cours dans le champ Afficher les statistiques, il s'agit du nombre de messages présentant des erreurs exécutés par cet URI endpoint pendant l'intervalle d'agrégation en cours.
Si vous avez sélectionné Depuis la dernière réinitialisation dans le champ Afficher les statistiques, il s'agit du nombre de messages présentant des erreurs exécutés par cet URI endpoint au cours de la période écoulée depuis la dernière réinitialisation des statistiques.
Temps de réponse minimal
Si vous avez sélectionné Intervalle d'agrégation en cours dans le champ Afficher les statistiques, il s'agit du temps de réponse minimal nécessaire à cet URI endpoint pour exécuter les messages pendant l'intervalle d'agrégation en cours.
Si vous avez sélectionné Depuis la dernière réinitialisation dans le champ Afficher les statistiques, il s'agit du temps de réponse minimal nécessaire à cet URI endpoint pour exécuter les messages au cours de la période écoulée depuis la dernière réinitialisation des statistiques.
Temps de réponse maximal
Si vous avez sélectionné Intervalle d'agrégation en cours dans le champ Afficher les statistiques, il s'agit du temps de réponse maximal nécessaire à cet URI endpoint pour exécuter les messages pendant l'intervalle d'agrégation en cours.
Si vous avez sélectionné Depuis la dernière réinitialisation dans le champ Afficher les statistiques, il s'agit du temps de réponse maximal nécessaire à cet URI endpoint pour exécuter les messages au cours de la période écoulée depuis la dernière réinitialisation des statistiques.
Temps de réponse moyen
Si vous avez sélectionné Intervalle d'agrégation en cours dans le champ Afficher les statistiques, il s'agit du temps moyen global nécessaire à cet URI endpoint pour exécuter les messages pendant l'intervalle d'agrégation en cours.
Si vous avez sélectionné Depuis la dernière réinitialisation dans le champ Afficher les statistiques, il s'agit du temps moyen global nécessaire à cet URI endpoint pour exécuter les messages au cours de la période écoulée depuis la dernière réinitialisation des statistiques.
Statut
Statut de l'URI endpoint du service fonctionnel.
Pour les domaines à serveur unique ou pour un serveur géré dans un cluster : En ligne ou Hors ligne.
Pour les statistiques de niveau de cluster : En ligne, Hors ligne ou Partiel, lorsqu'un URI particulier est hors ligne sur au moins l'un des serveurs gérés du cluster.
Action
Cliquez sur l'icône Marquer l'adresse comme étant en ligne pour marquer comme étant en ligne un URI hors ligne. Cette action n'est activée que si l'URI présente le statut Hors ligne ou Partiel.
Pour activer cette action, un service fonctionnel doit pouvoir utiliser les URI hors ligne et ses paramètres opérationnels de surveillance de service et de niveau global doivent également être activés. Reportez-vous à Configuration de paramètres opérationnels pour les services fonctionnels.
Remarque : En cas de succès, l'action est immédiate. Toutefois, pour que le nouveau statut (En ligne) de l'URI endpoint soit reflété, un cycle d'agrégation de surveillance est nécessaire. Il peut prendre jusqu'à 1 minute et demie.
