Visualisation des mesures d'opérations pour les services basés sur le WSDL

Utilisez cette page pour visualiser les informations de surveillance de service, présentées dans le tableau 25-10, et les mesures d'opérations des services basés sur le WSDL, présentées dans le tableau 25-12.

Tableau 25-12 Mesures d'opérations
Propriété
Description
Nom d'opération

Si vous avez sélectionné Intervalle d'agrégation en cours dans le champ Afficher les statistiques, il s'agit du nombre total de messages exécutés par cette opération pendant l'intervalle d'agrégation en cours.
