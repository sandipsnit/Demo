Visualisation des fichiers journaux de domaine

Cette page affiche un récapitulatif des entrées de fichier journal de domaine, y compris les informations répertoriées dans le tableau 25-17.

  • Pour personnaliser l'affichage des informations de fichier journal de domaine, reportez-vous à Personnalisation de la vue des entrées de fichier journal de domaine.
  • Pour afficher les détails d'un fichier journal particulier, sélectionnez le bouton d'option associée à l'entrée appropriée, puis cliquez sur Visualiser. Reportez-vous à Visualisation des détails des fichiers journaux de domaine.

    • Tableau 25-17 Informations récapitulatives sur les journaux de domaine
    Propriété
    Description
    Date
    Date à laquelle l'entrée a été consignée dans le journal.
    Sous-système
    Sous-système associé à l'entrée.
    Gravité
    Gravité du message.
    ID de message
    Identification unique du message
    Message
