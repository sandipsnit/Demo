Ajout de stratégies WS personnalisées

Vous pouvez importer un fichier XML contenant votre stratégie WS ou écrire directement la stratégie WS dans la console d'administration d'Oracle Service Bus.

  1. Si vous ne l'avez pas déjà fait, cliquez sur Créer afin de créer une session ou cliquez sur Modifier pour accéder à une session existante. Reportez-vous à Utilisation du centre des modifications.
  2. Sélectionnez Explorateur de projets, puis choisissez le projet ou le dossier auquel ajouter la stratégie WS. La page Vue du projet/Vue de dossier apparaît.
  3. Dans la liste déroulante Créer une ressource, sélectionnez Stratégie WS sous Interface pour afficher la page Créer une stratégie WS.
  4. Dans le champ Nom de la ressource, entrez ou modifiez un nom unique pour cette stratégie WS.

    Pour obtenir des conseils sur la dénomination, suivez les restrictions concernant la dénomination des ressources.
  5. Dans le champ Description de la ressource, entrez la description de cette stratégie WS.
  6. Dans le champ Stratégie, effectuez l'une des opérations suivantes :
    • Cliquez sur Parcourir pour localiser et importer le fichier XML contenant votre stratégie WS personnalisée.
    • Saisissez le texte de la nouvelle stratégie WS.
    • Copiez et collez le texte d'une stratégie WS existante dans ce champ.
  7. Cliquez sur Enregistrer. La stratégie WS est enregistrée dans la session en cours.
  8. Pour terminer la session et déployer la configuration lors de l'exécution, cliquez sur Activer sous Centre des modifications.
