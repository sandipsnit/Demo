Localisation des stratégies WS personnalisées

La page Récapitulatif des stratégies WS répertorie les stratégies de service Web personnalisées (stratégies WS) que vous avez ajoutées au domaine Oracle Service Bus en cours. La console d'administration d'Oracle Service Bus n'affiche pas les stratégies WS Oracle Service Bus qui sont prépackagées et disponibles dans tous les domaines Oracle Service Bus.

  1. Effectuez l'une des opérations suivantes :
    • Sélectionnez Explorateur de projets pour afficher la page Vue des projets ou la page Vue du projet/Vue de dossier. Naviguez ensuite dans les projets et les dossiers pour rechercher la stratégie WS.
    • Sélectionnez Navigateur Web de ressources > Stratégies WS. La page Récapitulatif des stratégies WS affiche les informations répertoriées dans le tableau 11-1. Pour obtenir une description plus détaillée des propriétés, reportez-vous à Localisation des stratégies WS personnalisées.
  2. Pour rechercher une stratégie WS personnalisée, entrez tout ou partie du nom de la stratégie WS dans le champ Nom. Vous pouvez également saisir tout ou partie du nom du projet et du dossier de stratégie WS dans le champ Chemin. Cliquez sur Rechercher.

    3. Cliquez sur Tout visualiser pour enlever les filtres de recherche et afficher toutes les stratégies WS.

    Tableau 11-1 Informations sur les stratégies WS
    Propriété
    Description
    Nom de la stratégie WS
    Nom unique affecté à la stratégie WS. Cliquez sur ce nom pour que la page Visualiser les détails de la stratégie WS apparaisse. Reportez-vous à Modification des stratégies WS personnalisées.
    Chemin
    Nom de projet et nom du dossier dans lequel la stratégie WS réside. Cliquez sur ce nom pour afficher le projet ou le dossier contenant cette ressource. Reportez-vous à Qualification des noms de ressource à l'aide de projets et de dossiers.
    Options
    Contient une icône Supprimer. Si un service fonctionnel ou proxy a été configuré de façon à utiliser une stratégie WS, une icône Avertissement relatif à la suppression indique que vous pouvez supprimer la stratégie WS à condition de confirmer l'opération lorsqu'un avertissement s'affiche. Toutefois, des conflits peuvent survenir, les références du service à la stratégie WS supprimée n'étant pas résolues. Reportez-vous à Suppression de stratégies WS personnalisées.
