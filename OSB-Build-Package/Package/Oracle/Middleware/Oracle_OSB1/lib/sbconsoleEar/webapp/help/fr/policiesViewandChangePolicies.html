Modification des stratégies WS personnalisées

Utilisez la page Visualiser les détails de la stratégie WS pour visualiser et modifier les détails d'une stratégie de service Web (stratégie WS) personnalisée spécifique. La console d'administration d'Oracle Service Bus n'affiche pas les stratégies WS Oracle Service Bus, qui sont en lecture seule.

  1. Si vous ne l'avez pas déjà fait, cliquez sur Créer afin de créer une session ou cliquez sur Modifier pour accéder à une session existante. Reportez-vous à Utilisation du centre des modifications.
  2. Localisez la stratégie WS, comme décrit dans Localisation des stratégies WS personnalisées.
  3. Cliquez sur le nom de la stratégie WS. La page Visualiser les détails de la stratégie WS affiche les informations répertoriées dans le tableau 11-2.

    4. Tableau 11-2 Détails des stratégies WS 
    Propriété
    Description
    Dernière modification par
    Utilisateur ayant créé cette stratégie WS ou l'ayant importée dans la configuration.
    Dernière modification le
    Date et heure auxquelles l'utilisateur a créé la stratégie WS ou l'a importée dans la configuration. Cliquez sur le lien de date et d'heure pour visualiser l'historique des modifications apportées à cette ressource. Reportez-vous à Page Visualiser l'historique des modifications.
    Références
    Nombre d'objets référencés par la stratégie WS. Lorsque des références existent, cliquez sur le lien numérique pour visualiser la liste des objets en question. Reportez-vous à Visualisation des références aux ressources.
    Référencé par
    Nombre d'objets référençant la stratégie WS. Lorsque des références existent, cliquez sur le lien numérique pour visualiser la liste des objets en question. Reportez-vous à Visualisation des références aux ressources.
    Description
    Description de la stratégie WS, le cas échéant.
    Stratégie
    Texte de la stratégie WS.

  4. Pour modifier les champs, cliquez sur Modifier. Pour obtenir la description de ces champs, reportez-vous à Ajout de stratégies WS personnalisées.

    5. Vous ne pouvez pas modifier le champ Nom de la ressource.

  5. Cliquez sur Enregistrer pour valider les mises à jour dans la session en cours.
  6. Pour terminer la session et déployer la configuration lors de l'exécution, cliquez sur Activer sous Centre des modifications.
