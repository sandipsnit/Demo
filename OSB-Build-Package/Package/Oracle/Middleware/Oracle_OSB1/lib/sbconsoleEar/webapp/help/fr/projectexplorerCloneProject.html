Clonage de projets

Vous pouvez cloner (copier) un projet de trois façons :

  • Cloner un projet en lui attribuant un nouveau nom au niveau racine. Dans ce cas, le projet d'origine et son clone sont des projets homologues. Le projet cloné contient les mêmes dossiers et ressources que le projet d'origine (jusqu'à ce que vous modifiiez la source ou le clone). Pour obtenir des instructions, reportez-vous à Clonage d'un projet au niveau racine.
  • Cloner un projet en utilisant le nom d'un projet existant (autre que le projet source). Dans ce cas, les dossiers et ressources du projet source sont fusionnés avec ceux du projet dont le nom est affecté au clone. Le projet d'origine n'est pas modifié. Pour obtenir des instructions, reportez-vous à Clonage d'un projet au niveau racine.
  • Cloner un projet en le plaçant dans un nouvel emplacement, dans un projet ou un dossier de projet existant. Dans ce cas, le projet cloné est rétrogradé vers un dossier du nouvel emplacement. Il contient les mêmes dossiers et ressources que le projet d'origine.

    • Lorsque vous clonez un projet portant le même nom qu'un dossier existant et qui se trouve dans le même emplacement, le contenu (sous-dossiers et ressources) du projet est fusionné avec celui du dossier cible.

    Dans les deux cas, le projet d'origine n'est pas modifié.

    Pour obtenir des instructions, reportez-vous à Clonage d'un projet et rétrogradation du clone vers un dossier.

Clonage d'un projet au niveau racine
  1. Si vous ne l'avez pas déjà fait, cliquez sur Créer afin de créer une session ou cliquez sur Modifier pour accéder à une session existante. Reportez-vous à Utilisation du centre des modifications.
  2. Sélectionnez Explorateur de projets > Projets pour afficher la page Vue des projets.
  3. Cliquez sur l'icône Cloner un projet dans la ligne du projet à cloner. La page Cloner un projet apparaît et l'emplacement en cours du projet est sélectionné.
  4. Dans le champ Nouveau nom, attribuez un nom au clone. Reportez-vous à Affectation de noms aux fichiers et aux dossiers.

    5. Saisissez un nom unique pour créer le clone en tant que nouveau projet ou entrez le nom d'un projet existant pour fusionner le clone avec celui-ci.

  5. Dans le panneau Emplacement, conservez l'emplacement racine par défaut lorsque vous clonez un projet au niveau racine.
  6. Cliquez sur Cloner un projet. Le projet est cloné dans la session en cours.
  7. Pour terminer la session et déployer la configuration lors de l'exécution, cliquez sur Activer sous Centre des modifications.
Clonage d'un projet et rétrogradation du clone vers un dossier
  1. Si vous ne l'avez pas déjà fait, cliquez sur Créer afin de créer une session ou cliquez sur Modifier pour accéder à une session existante. Reportez-vous à Utilisation du centre des modifications.
  2. Sélectionnez Explorateur de projets > Projets pour afficher la page Vue des projets.
  3. Cliquez sur l'icône Cloner un projet dans la ligne du projet à cloner. La page Cloner un projet apparaît et l'emplacement en cours du projet est sélectionné.
  4. Dans le champ Nouveau nom, entrez éventuellement le nouveau nom à attribuer au dossier. Reportez-vous à Affectation de noms aux fichiers et aux dossiers.

    5. Saisissez un nom unique pour créer un dossier ou entrez le nom d'un dossier existant pour fusionner le clone avec celui-ci (dans l'emplacement que vous sélectionnerez à l'étape suivante).

  5. Dans le panneau Emplacement, sélectionnez un projet ou un dossier, puis cliquez sur Cloner un projet. Le projet cloné est converti en dossier dans le nouvel emplacement et il est enregistré dans la session en cours.
  6. Pour terminer la session et déployer la configuration lors de l'exécution, cliquez sur Activer sous Centre des modifications.
