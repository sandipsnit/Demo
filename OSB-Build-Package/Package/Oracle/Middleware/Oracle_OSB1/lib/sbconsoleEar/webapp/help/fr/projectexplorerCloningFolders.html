Clonage de dossiers

  1. Si vous ne l'avez pas déjà fait, cliquez sur Créer afin de créer une session ou cliquez sur Modifier pour accéder à une session existante. Reportez-vous à Utilisation du centre des modifications.
  2. Sélectionnez Explorateur de projets, puis accédez au projet ou au dossier parent contenant le dossier à cloner. La page Vue du projet/Vue de dossier apparaît.
  3. Cliquez sur l'icône Cloner un dossier dans la ligne du dossier à cloner. La page Cloner un dossier apparaît et l'emplacement en cours du dossier est sélectionné.
  4. Clonez le dossier dans le même emplacement ou dans un autre, ou mettez-le à niveau vers un projet, en procédant comme suit :
    • Pour cloner le dossier dans le même emplacement, attribuez-lui un nouveau nom dans le champ Nouveau nom. Reportez-vous à Affectation de noms aux fichiers et aux dossiers.

      • Lorsque vous clonez un élément dans un même emplacement, vous pouvez soit ne sélectionner aucun emplacement dans le panneau Emplacement (ce qui équivaut à sélectionner l'emplacement en cours), soit sélectionner de façon explicite le projet et le dossier en cours.

    • Pour cloner le dossier dans un emplacement différent, entrez éventuellement un nouveau nom dans le champ Nouveau nom, puis sélectionnez un projet et/ou un dossier dans la zone Emplacement. Reportez-vous à Affectation de noms aux fichiers et aux dossiers.

      • Si vous indiquez le même nom que celui d'un dossier existant, le contenu du dossier d'origine est fusionné avec celui du dossier cible.

    • Pour mettre à niveau le dossier vers un projet, sélectionnez l'emplacement racine.
  5. Cliquez sur Cloner un dossier. Le dossier est cloné dans la session en cours.

    6. Lorsque vous clonez un dossier, tous les sous-dossiers et ressources associés sont clonés de façon récursive.

  6. Pour terminer la session et déployer la configuration lors de l'exécution, cliquez sur Activer sous Centre des modifications.
