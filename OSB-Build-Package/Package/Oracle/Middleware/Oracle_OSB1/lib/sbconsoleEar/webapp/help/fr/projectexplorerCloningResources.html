Clonage de ressources

  1. Si vous ne l'avez pas déjà fait, cliquez sur Créer afin de créer une session ou cliquez sur Modifier pour accéder à une session existante. Reportez-vous à Utilisation du centre des modifications.
  2. Sélectionnez Explorateur de projets, puis choisissez le projet ou le dossier contenant la ressource à cloner. La page Vue du projet/Vue de dossier apparaît.
  3. Cliquez sur l'icône Cloner une ressource dans la ligne de la ressource à cloner. La page Cloner une ressource apparaît et l'emplacement en cours de la ressource est sélectionné.
  4. Vous pouvez cloner la ressource dans le même emplacement ou dans un autre, en procédant comme suit :
    • Pour cloner la ressource au même emplacement, attribuez-lui un nouveau nom dans le champ Nouveau nom de la ressource. Reportez-vous aux restrictions concernant la dénomination des ressources.

      • Pour cloner un élément dans un même emplacement, deux possibilités s'offrent à vous : vous pouvez soit ne sélectionner aucun emplacement dans le panneau Emplacement (ce qui équivaut à sélectionner l'emplacement en cours), soit sélectionner de façon explicite le projet et le dossier en cours.

    • Pour cloner la ressource dans un emplacement différent, entrez éventuellement un nouveau nom dans le champ Nouveau nom de la ressource, puis sélectionnez un projet et/ou un dossier dans la zone Emplacement.
  5. Cliquez sur Cloner une ressource. La ressource est clonée dans la session en cours.

    6. Si vous clonez un service fonctionnel importé à partir du registre UDDI, le service cloné est détaché du registre.

  6. Pour terminer la session et déployer la configuration lors de l'exécution, cliquez sur Activer sous Centre des modifications.
