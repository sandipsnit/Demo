Création de ressources

  1. Si vous ne l'avez pas déjà fait, cliquez sur Créer afin de créer une session ou cliquez sur Modifier pour accéder à une session existante. Reportez-vous à Utilisation du centre des modifications.
  2. Sélectionnez Explorateur de projets, puis choisissez le projet ou le dossier dans lequel créer la ressource.
  3. Dans la liste déroulante Créer une ressource de la page Vue du projet/Vue de dossier, sélectionnez le type de ressource à créer. Différentes pages sont affichées en fonction du type de ressource sélectionné.
  4. Suivez les invites sur chaque page pour créer la ressource. Pour plus d'informations sur la création de chaque type de ressource, reportez-vous à la rubrique correspondante, comme indiqué dans le tableau 3-3.

  5. Lorsque vous créez une ressource, celle-ci est créée dans la session en cours.
  6. Pour terminer la session et déployer la configuration lors de l'exécution, cliquez sur Activer sous Centre des modifications.
