Suppression de projets

  1. Si vous ne l'avez pas déjà fait, cliquez sur Créer afin de créer une session ou cliquez sur Modifier pour accéder à une session existante. Reportez-vous à Utilisation du centre des modifications.
  2. Sélectionnez Explorateur de projets > Projets pour afficher la page Vue des projets.
  3. Cliquez sur l'icône Supprimer un projet dans la ligne du projet à supprimer. Celui-ci est supprimé de la session en cours.

    4. Lorsque vous supprimez un projet, toutes les ressources associées le sont également. Si certaines ressources contenues dans ce projet sont référencées par d'autres contenues dans un projet différent, vous pouvez supprimer le projet à condition de confirmer l'opération lorsqu'un avertissement s'affiche. Toutefois, des conflits peuvent survenir, les références à la ressource supprimée n'étant pas résolues.

    Pour ces ressources, l'icône Avertissement relatif à la suppression apparaît à la place de l'icône Supprimer.

  4. Pour terminer la session et déployer la configuration lors de l'exécution, cliquez sur Activer sous Centre des modifications.
