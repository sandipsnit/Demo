Export d'un WSDL

La page Vue du projet/Vue de dossier permet d'exporter un WSDL associé à un service proxy ou fonctionnel. Le WSDL est exporté sous forme de fichier JAR. Vous ne pouvez exporter un WSDL que si vous êtes en dehors d'une session. Reportez-vous à Utilisation du centre des modifications.

Utilisez la fonctionnalité Exporter le WSDL pour que des outils externes, tels que les IDE (Integrated Development Environment), puissent utiliser un WSDL. Cette fonctionnalité est différente de la fonctionnalité Exporter les ressources du module Administration du système, qui permet quant à elle de déplacer et préparer des ressources entre deux domaines.

  1. Dans le panneau de navigation de gauche, cliquez sur un projet ou un dossier à partir de l'explorateur de projets. La page Vue du projet/Vue de dossier apparaît.
  2. Cliquez sur l'icône Exporter le WSDL dans la ligne du service proxy dont vous souhaitez exporter le WSDL. La boîte de dialogue Téléchargement de fichier apparaît.
  3. Dans la boîte de dialogue Téléchargement de fichier, cliquez sur Open pour ouvrir le fichier JAR ou sur Enregistrer pour l'enregistrer sur votre bureau.
