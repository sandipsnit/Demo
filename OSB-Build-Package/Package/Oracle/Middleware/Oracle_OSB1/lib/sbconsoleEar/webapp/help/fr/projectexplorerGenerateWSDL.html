Génération d'un WSDL

Un WSDL généré est une ressource WSDL générée par Oracle Service Bus pour un service n'ayant pas démarré avec une ressource WSDL mais pouvant être décrit à l'aide d'un WSDL. Par exemple, un WSDL créé à partir d'un service EJB est un WSDL généré. Pour plus d'informations, reportez-vous à "How WSDL is Used in Oracle Service Bus" dans le manuel Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus à l'adresse suivante : http://www.oracle.com/pls/as111170/lookup?id=OSBAG114.

Utilisez la page Vue du projet/Vue de dossier pour générer un WSDL associé à un service fonctionnel doté d'un type de transport de flux ou EJB.

  1. Dans le panneau de navigation de gauche, cliquez sur un projet ou un dossier à partir de l'explorateur de projets. La page Vue du projet/Vue de dossier apparaît.
  2. Cliquez sur l'icône Générer le WSDL dans la ligne du service fonctionnel dont vous souhaitez générer le WSDL. La boîte de dialogue Générer la ressource WSDL apparaît et l'emplacement en cours (nom du projet et du dossier où réside le service fonctionnel) est sélectionné.
  3. Dans la boîte de dialogue Générer la ressource WSDL, indiquez une valeur pour Nom de la nouvelle ressource WSDL et l'emplacement du WSDL, puis cliquez sur Générer le WSDL.

    Pour obtenir des conseils sur la dénomination, suivez les restrictions concernant la dénomination des ressources.

Après avoir fini

Suivez la procédure décrite dans Modification de ressources WSDL pour visualiser ou modifier le WSDL généré.
