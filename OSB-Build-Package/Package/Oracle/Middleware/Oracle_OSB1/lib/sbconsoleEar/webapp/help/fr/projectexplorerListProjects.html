Page Vue des projets

La page Vue des projets répertorie les projets définis dans le domaine en cours. Le tableau 3-1 décrit les options et contrôles de cette page.

Tableau 3-1 Page Vue des projets - Options et contrôles
Option ou contrôle
Description
Entrer le nom du nouveau projet
Pour créer un projet, entrez un nom dans ce champ de texte, puis cliquez sur Ajouter un projet. Reportez-vous à Ajout de projets.
Reportez-vous également à Affectation de noms aux fichiers et aux dossiers.
Nom
Cliquez sur le nom d'un projet pour afficher la page Vue du projet/Vue de dossier correspondante.
Options
Pour chaque projet répertorié, vous pouvez effectuer les opérations suivantes :
  • Cliquez sur l'icône Renommer le projet pour renommer le projet. Reportez-vous à Attribution d'un nouveau nom aux projets.
  • Cliquez sur l'icône Déplacer le projet pour déplacer le projet. Reportez-vous à Déplacement de projets (conversion en dossier).
  • Cliquez sur l'icône Cloner un projet pour cloner le projet. Reportez-vous à Clonage de projets.
  • Cliquez sur l'icône Supprimer un projet pour supprimer le projet. Si cette ressource est référencée par d'autres, l'icône Avertissement relatif à la suppression apparaît. Vous pouvez supprimer la ressource à condition de confirmer l'opération lorsqu'un avertissement s'affiche. Toutefois, des conflits peuvent survenir, les références à la ressource supprimée n'étant pas résolues. Reportez-vous à Suppression de projets.
Lorsque vous effectuez une de ces opérations, le résultat est enregistré dans la session en cours. Pour terminer la session et déployer la configuration lors de l'exécution, cliquez sur Activer sous Centre des modifications.
