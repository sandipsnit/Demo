Page Charger des ressources - Charger des ressources à partir d'une URL

La page Charger des ressources - Charger des ressources à partir d'une URL permet d'importer des ressources à partir d'un fichier disponible sur le Web, comme indiqué dans Chargement de ressources à partir d'une URL.

Le tableau 3-5 décrit les options et contrôles de cette page.

Tableau 3-5 Page Charger des ressources : Charger des ressources à partir d'une URL - Options et contrôles
Options et contrôles
Description
URL/Chemin
Entrez l'URL correspondant à l'emplacement du fichier.
Pour spécifier une ressource locale, vous pouvez utiliser le protocole de fichier (file) et entrer un nom de fichier entièrement qualifié par son chemin, par exemple, file:///c:/osbresources/ForeachAction.jar.
Nom de la ressource
Attribuez un nom à la ressource.
Type de ressource
Sélectionnez le type de la ressource dans la liste.
