Chargement de ressources à partir d'une URL

Vous pouvez effectuer un import global des ressources disponibles via une URL ou dans un système de fichiers. Cette fonctionnalité permet d'importer une ressource racine, telle qu'un WSDL, et ses éléments dépendants, tels que d'autres WSDL et des schémas. La correspondance de dépendances est résolue de façon automatique.

  1. Si vous ne l'avez pas déjà fait, cliquez sur Créer afin de créer une session ou cliquez sur Modifier pour accéder à une session existante. Reportez-vous à Utilisation du centre des modifications.
  2. Sélectionnez Explorateur de projets, puis choisissez le projet ou le dossier auquel ajouter les ressources. La page Vue du projet/Vue de dossier apparaît.
  3. Dans la liste déroulante Créer une ressource, sélectionnez Ressources à partir de l'URL. La page Charger des ressources - Charger des ressources à partir d'une URL apparaît.
  4. Dans le champ URL/Chemin, entrez l'URL ou le chemin de la ressource.

    5. Pour spécifier une ressource locale, vous pouvez utiliser le protocole de fichier (file) et entrer un nom de fichier entièrement qualifié par son chemin, par exemple, file:///c:/osbresources/ForeachAction.jar.

  5. Dans le champ Nom de la ressource, entrez le nom de la ressource.
  6. Dans la liste déroulante Type de ressource, sélectionnez le type de la ressource.
  7. Cliquez sur Suivant pour afficher la page Charger des ressources - Vérifier les ressources chargées. Désélectionnez les cases en regard des ressources à exclure de l'import.
  8. Pour importer les ressources sélectionnées, cliquez sur Importer.
  9. La page Charger des ressources - Importer le résultat affiche les ressources importées. Vous pouvez cliquer sur Charger un autre élément pour importer des ressources à partir d'un autre fichier ou effectuer d'autres tâches en sélectionnant une autre page dans le panneau de navigation de gauche.
  10. Pour terminer la session et déployer la configuration lors de l'exécution, cliquez sur Activer sous Centre des modifications.
