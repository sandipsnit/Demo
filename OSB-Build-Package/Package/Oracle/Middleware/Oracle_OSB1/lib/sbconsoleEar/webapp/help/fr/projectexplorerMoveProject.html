Déplacement de projets (conversion en dossier)

Lorsque vous "déplacez" un projet, ce dernier est converti en dossier dans le nouvel emplacement.

  1. Si vous ne l'avez pas déjà fait, cliquez sur Créer afin de créer une session ou cliquez sur Modifier pour accéder à une session existante. Reportez-vous à Utilisation du centre des modifications.
  2. Sélectionnez Explorateur de projets > Projets pour afficher la page Vue des projets.
  3. Cliquez sur l'icône Déplacer le projet dans la ligne du projet à déplacer. La page Déplacer le projet apparaît et l'emplacement en cours du projet est sélectionné.
  4. Sous Emplacement, sélectionnez le projet ou le dossier vers lequel "déplacer" le projet. Ce dernier est converti en dossier dans le nouvel emplacement et il est enregistré dans la session en cours.

    5. Lorsque vous déplacez un projet vers un nouvel emplacement et qu'il porte le même nom qu'un dossier y figurant déjà, le contenu du projet est fusionné avec celui du dossier existant.

  5. Pour terminer la session et déployer la configuration lors de l'exécution, cliquez sur Activer sous Centre des modifications.
