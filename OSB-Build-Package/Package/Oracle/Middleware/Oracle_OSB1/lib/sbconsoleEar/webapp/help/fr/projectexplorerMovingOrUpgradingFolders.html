Déplacement ou mise à niveau de dossiers

  1. Si vous ne l'avez pas déjà fait, cliquez sur Créer afin de créer une session ou cliquez sur Modifier pour accéder à une session existante. Reportez-vous à Utilisation du centre des modifications.
  2. Sélectionnez Explorateur de projets, puis choisissez le projet ou le dossier contenant le dossier à déplacer ou à mettre à niveau. La page Vue du projet/Vue de dossier apparaît.
  3. Cliquez sur l'icône Déplacer le dossier dans la ligne du dossier à déplacer ou à mettre à niveau. La page Déplacer le dossier apparaît et l'emplacement en cours du dossier est sélectionné.
  4. Déterminez si vous souhaitez déplacer le dossier vers un nouvel emplacement sous un projet différent ou mettre à niveau le dossier vers un projet :
    • Pour déplacer le dossier vers un nouvel emplacement, dans le panneau Emplacement, sélectionnez le projet ou le dossier vers lequel déplacer le dossier.
    • Pour mettre à niveau le dossier vers un projet, sélectionnez l'emplacement racine.
  5. Cliquez sur Soumettre. Le dossier (ou le projet, si vous avez mis le dossier à niveau vers un projet) est enregistré dans le nouvel emplacement dans la session en cours.

    6. Lorsque vous déplacez ou mettez à niveau un dossier et qu'un autre dossier portant le même nom figure dans le même emplacement, le contenu du dossier déplacé ou mis à niveau est fusionné avec celui du dossier ou du projet existant.

  6. Pour terminer la session et déployer la configuration lors de l'exécution, cliquez sur Activer sous Centre des modifications.
