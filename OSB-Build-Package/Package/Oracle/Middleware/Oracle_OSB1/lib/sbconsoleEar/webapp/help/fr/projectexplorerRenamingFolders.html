Attribution d'un nouveau nom aux dossiers

  1. Si vous ne l'avez pas déjà fait, cliquez sur Créer afin de créer une session ou cliquez sur Modifier pour accéder à une session existante. Reportez-vous à Utilisation du centre des modifications.
  2. Sélectionnez Explorateur de projets, puis choisissez le projet ou le dossier contenant le dossier à renommer. La page Vue du projet/Vue de dossier apparaît.
  3. Cliquez sur l'icône Renommer le dossier dans la ligne du dossier à renommer. La page Renommer le dossier apparaît.
  4. Dans le champ Nouveau nom du dossier, entrez le nouveau nom à attribuer au dossier. Reportez-vous à Affectation de noms aux fichiers et aux dossiers.
  5. Cliquez sur Renommer. Le dossier renommé est enregistré dans la session en cours.
  6. Pour terminer la session et déployer la configuration lors de l'exécution, cliquez sur Activer sous Centre des modifications.
