Page Charger des ressources - Vérifier les ressources chargées

La page Charger des ressources - Vérifier les ressources chargées affiche des détails relatifs aux ressources importées à partir d'un fichier ZIP ou de tout autre fichier, comme indiqué dans Chargement de ressources à partir d'un fichier ZIP et Chargement de ressources à partir d'une URL.

Le tableau 3-6 décrit les options et contrôles de cette page.

Tableau 3-6 Page Charger des ressources : Vérifier les ressources chargées - Options et contrôles
Options et contrôles
Description
Nom de fichier
Nom de fichier de la ressource, extension comprise.
Nom
Nom de la ressource.
Chemin
Chemin de la ressource, correspondant au projet et au dossier dans lesquels réside la ressource.
Type de ressource
Enumère les types de ressource pris en charge.

Lorsque l'icône Avertissement apparaît en regard d'un nom de fichier, cela signifie que le type de ressource en question est inconnu. (Un fichier ne portant aucune extension ou dont l'extension n'est pas définie dans la correspondance est considéré comme un fichier inconnu.) Les fichiers de type inconnu ne peuvent pas être importés.
