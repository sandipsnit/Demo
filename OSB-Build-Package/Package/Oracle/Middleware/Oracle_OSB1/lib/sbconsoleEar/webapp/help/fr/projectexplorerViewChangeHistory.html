Page Visualiser l'historique des modifications

La page Visualiser l'historique des modifications affiche les dix modifications les plus récentes apportées à une ressource, parmi toutes les activations de session.

Le tableau 3-9 décrit les options et contrôles de cette page.

Tableau 3-9 Page Historique des modifications - Options et contrôles
Option ou contrôle
Description
Tâche
Type de tâche exécuté :
  • Créer
  • Mettre à jour
  • Cloner
  • Renommer
  • Déplacer
Utilisateur
Utilisateur ayant créé ou modifié la ressource.
Temps d'exécution
Date et heure de création ou de modification de la ressource.
