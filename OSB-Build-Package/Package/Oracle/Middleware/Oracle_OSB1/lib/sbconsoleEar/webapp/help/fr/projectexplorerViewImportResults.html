Page Charger des ressources - Importer le résultat

La page Charger des ressources - Importer le résultat affiche les résultats du chargement des ressources à partir d'un fichier ZIP importé ou d'une URL.

Selon que vous avez chargé des ressources à partir d'un fichier ZIP ou d'une URL, assurez-vous que vous avez suivi les étapes décrites dans Chargement de ressources à partir d'un fichier ZIP ou Chargement de ressources à partir d'une URL, et que vous avez respecté la procédure fournie dans Page Charger des ressources - Vérifier les ressources chargées.

La page Importer le résultat apparaît. Elle affiche les informations décrites ci-après.

Tableau 3-7 Page Vue du projet/Vue de dossier - Options et contrôles
Option ou contrôle
Description
Statut
Statut de la ressource (indiquant, par exemple, qu'elle a été importée ou non).
Un fichier WSDL peut temporairement être considéré comme non valide tant que toutes ses dépendances ne sont pas importées. Une fois l'import terminé, le service de configuration valide à nouveau les fichiers marqués comme non valides.
Nom
Nom de la ressource.
Chemin
Chemin de la ressource, correspondant au projet et au dossier dans lesquels réside la ressource.
Type de ressource
Affiche le type de ressource importé.
Message d'erreur
Affiche les détails du message d'erreur qui apparaît lorsque l'import d'une ressource a échoué.
