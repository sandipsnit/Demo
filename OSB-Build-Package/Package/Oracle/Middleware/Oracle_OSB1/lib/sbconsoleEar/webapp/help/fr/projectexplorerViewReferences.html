Page Visualiser les références

Selon que vous avez sélectionné Références ou Référencé par sur la page Vue du projet/Vue de dossier, la page Visualiser les références affiche la liste des éléments suivants :

  • Ressources ne faisant pas partie du projet ou dossier en cours, mais qui sont référencées par les ressources du projet ou du dossier.
  • Ressources ne faisant pas partie du projet ou dossier en cours, mais qui référencent les ressources du projet ou du dossier.

Le tableau 3-8 décrit les options et contrôles de cette page.

Tableau 3-8 Page Références de projet - Options et contrôles
Option ou contrôle
Description
Nom
Cliquez sur le nom d'une ressource pour afficher les détails la concernant.
Type de ressource
Affiche le projet, le dossier ou tout autre type de ressource pris en charge par Oracle Service Bus.
