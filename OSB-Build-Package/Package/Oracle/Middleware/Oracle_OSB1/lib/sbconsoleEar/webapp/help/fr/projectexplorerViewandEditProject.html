Cliquez sur le nombre (si celui-ci est supérieur à 0) figurant dans ce champ pour afficher la liste des ressources ne faisant pas partie du projet, mais qui dépendent des ressources du projet.
Description
Cliquez sur Modifier la description pour ajouter ou modifier la description du projet, puis cliquez sur Soumettre la description pour la mettre à jour.
Dossiers
Entrer le nom du nouveau dossier
Pour créer un dossier dans le projet, entrez un nom dans ce champ de texte, puis cliquez sur Ajouter un dossier. Reportez-vous à Ajout de dossiers.
Reportez-vous également à Affectation de noms aux fichiers et aux dossiers.
Nom
Cliquez sur le nom d'un dossier pour afficher la page Vue du projet/Vue de dossier correspondante.
Options
Pour chaque dossier répertorié, vous pouvez effectuer les opérations suivantes :
  • Cliquez sur l'icône Renommer le dossier pour renommer le dossier. Reportez-vous à Attribution d'un nouveau nom aux dossiers.
  • Cliquez sur l'icône Mettre à niveau vers le projet ou déplacer le dossier pour déplacer le dossier ou le mettre à niveau vers un projet. Reportez-vous à Déplacement ou mise à niveau de dossiers.
  • Cliquez sur l'icône Cloner un dossier pour cloner le dossier. Reportez-vous à Clonage de dossiers.
  • Cliquez sur l'icône Supprimer un dossier pour supprimer le dossier. Si cette ressource est référencée par d'autres, l'icône Avertissement relatif à la suppression apparaît. Vous pouvez supprimer la ressource à condition de confirmer l'opération lorsqu'un avertissement s'affiche. Toutefois, des conflits peuvent survenir, les références à la ressource supprimée n'étant pas résolues. Reportez-vous à Suppression de dossiers.
Lorsque vous effectuez une de ces opérations, le résultat est enregistré dans la session en cours. Pour terminer la session et déployer la configuration lors de l'exécution, cliquez sur Activer sous Centre des modifications.
Supprimer
Pour supprimer des dossiers, cochez les cases en regard de leurs noms, puis cliquez sur Supprimer.
Ressources
Créer une ressource
Pour créer une ressource pour ce projet, sélectionnez un type de ressource dans la liste Créer une ressource. Reportez-vous à Création de ressources.
Nom
Cliquez sur le nom d'une ressource pour afficher des détails la concernant.
Type de ressource
Type de cette ressource.
Actions
Les actions disponibles pour une ressource dépendent de son type, à savoir :
  • L'icône Modifier le flux de messages est disponible pour les services proxy. Cliquez sur cette icône pour modifier le flux de messages d'un service proxy. Reportez-vous à Création, modification et visualisation de flux de messages.
  • L'icône Lancer la console de test est disponible pour les services fonctionnels et proxy, les transformations XQuery et XSL, ainsi que les fichiers MFL. Cliquez sur cette icône pour appeler la console de test, qui permet de valider et tester la conception des services et des transformations. En ce qui concerne les services, vous ne pouvez utiliser la console de test qu'à l'exécution, à savoir lors de l'activation de la session. Quant aux transformations, vous pouvez utiliser la console de test tant à l'intérieur qu'à l'extérieur d'une session. Reportez-vous à Test des services et à Test des transformations.
  • L'icône Exporter le WSDL est disponible pour les services proxy basés sur le WSDL. Cliquez sur cette icône pour exporter un WSDL, afin de le visualiser ou de le modifier dans un outil externe tel qu'un IDE (Integrated Development Environment).

    • Cette fonctionnalité est différente de la fonction Exporter les ressources du module Administration du système, qui permet quant à elle de déplacer et préparer des ressources entre deux domaines. Reportez-vous à Export d'un WSDL.

  • L'icône Générer le WSDL est disponible pour les services fonctionnels dotés d'un type transport, tel que le transport EJB ou le transport de flux. Cliquez sur cette icône pour générer un WSDL, que vous pourrez ensuite visualiser ou modifier. Reportez-vous à Génération d'un WSDL.
Options
La colonne Options affiche les icônes suivantes :
  • Cliquez sur l'icône Renommer la ressource pour renommer la ressource. Reportez-vous à Attribution d'un nouveau nom aux ressources.
  • Cliquez sur l'icône Déplacer la ressource pour déplacer la ressource. Reportez-vous à Déplacement de ressources.
  • Cliquez sur l'icône Cloner une ressource pour cloner la ressource. Reportez-vous à Clonage de ressources
  • Cliquez sur l'icône Supprimer pour supprimer la ressource. Si cette ressource est référencée par d'autres, l'icône Avertissement relatif à la suppression apparaît. Vous pouvez supprimer la ressource à condition de confirmer l'opération lorsqu'un avertissement s'affiche. Toutefois, des conflits peuvent survenir, les références à la ressource supprimée n'étant pas résolues.
Lorsque vous effectuez une de ces opérations, le résultat est enregistré dans la session en cours. Pour terminer la session et déployer la configuration lors de l'exécution, cliquez sur Activer sous Centre des modifications.
