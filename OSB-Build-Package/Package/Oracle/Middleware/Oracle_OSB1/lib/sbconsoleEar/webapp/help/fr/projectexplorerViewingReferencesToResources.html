Visualisation des références aux ressources

Vous pouvez visualiser les références aux ressources tant à l'intérieur qu'à l'extérieur d'une session. Reportez-vous à Utilisation du centre des modifications.

  1. Cliquez sur Références ou Référencé par sur la page de détails d'un projet, d'un dossier ou d'une ressource dans les modules Explorateur de projets ou Navigateur Web de ressources.
    • Cliquez sur Références pour afficher les ressources ne faisant pas partie d'un projet ou d'un dossier, mais qui sont référencées par les ressources du projet ou du dossier.
    • Cliquez sur Référencé par pour afficher les ressources ne faisant pas partie d'un projet ou d'un dossier, mais qui référencent les ressources du projet ou du dossier.

      • La page Visualiser les références apparaît.

  2. Cliquez sur le nom d'une ressource pour afficher les détails la concernant ou cliquez sur Précédent pour revenir à la page précédente.
