Localisation des fournisseurs de clés de service

  1. Effectuez l'une des opérations suivantes :
    • Sélectionnez Explorateur de projets pour afficher la page Vue des projets ou la page Vue du projet/Vue de dossier. Ensuite, parcourez les projets et les dossiers pour trouver le fournisseur de clés de service.
    • Sélectionnez Navigateur Web de ressources > Fournisseurs de clés de service. La page Récapitulatif des fournisseurs de clés de service affiche les informations présentées dans le tableau 17-1.
  2. Pour rechercher un fournisseur de clés de service, entrez tout ou partie de son nom dans le champ Nom. Vous pouvez également saisir, en partie ou en entier, le nom du projet du fournisseur et du dossier dans les champs Chemin. Cliquez sur Rechercher.

    3. Cliquez sur Tout visualiser pour enlever les filtres de recherche et afficher tous les fournisseurs de clés de service.

    Tableau 17-1 Informations relatives au fournisseur de clés de service 
    Propriété
    Description
    Nom
    Nom unique du fournisseur de clés de service. Cliquez sur le nom pour voir la page Visualiser les détails du fournisseur de clés de service. Reportez-vous à Modification des fournisseurs de clés de service.
    Chemin
    Nom de projet et nom du dossier dans lequel réside le fournisseur de clés·de service. Cliquez sur le nom pour visualiser le projet ou le dossier qui contient cette ressource. Reportez-vous à Qualification des noms de ressource à l'aide de projets et de dossiers.
