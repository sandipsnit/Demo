Modification des fournisseurs de clés de service

Utilisez la page Visualiser les détails du fournisseur de clés de service pour visualiser et modifier les détails d'un fournisseur de clés de service spécifique.

  1. Si vous ne l'avez pas déjà fait, cliquez sur Créer afin de créer une session ou cliquez sur Modifier pour accéder à une session existante. Reportez-vous à Utilisation du centre des modifications.
  2. Localisez le fournisseur de clés de service, comme décrit dans Localisation des fournisseurs de clés de service.
  3. Cliquez sur le nom du fournisseur de clés de service. La page Visualiser les détails du fournisseur de clés de service affiche les informations présentées dans le tableau 17-3.

    4. Tableau 17-3 Détails du fournisseur de clés de service
    Propriété
    Description
    Nom du fournisseur de clés de service
    Nom du fournisseur de clés de service.
    Dernière modification par
    Utilisateur ayant créé le fournisseur de clés de service ou l'ayant importé dans la configuration.
    Dernière modification le
    Date et heure auxquelles l'utilisateur a créé le fournisseur de clés·de service ou l'a importé dans la configuration. Cliquez sur le lien de date et d'heure pour visualiser l'historique des modifications apportées à cette ressource. Reportez-vous à Page Visualiser l'historique des modifications.
    Références
    Nombre d'objets référencés par le fournisseur de clés·de service. Lorsque des références existent, cliquez sur le lien numérique pour visualiser la liste des objets en question. Reportez-vous à Visualisation des références aux ressources.
    Référencé par
    Nombre d'objets référencés par le fournisseur de clés de service. Si ces références existent, cliquez sur le lien numérique pour visualiser la liste des objets en question. Par exemple, si vous avez sélectionné le fournisseur de clés de service en tant que fournisseur d'un service proxy spécifique, ce service proxy est listé en tant que référence lorsque vous cliquez sur le lien. Reportez-vous à Visualisation des références aux ressources.
    Description
    Description du fournisseur de clés de service, le cas échéant.

  4. Pour modifier les champs, cliquez sur Modifier. Pour obtenir les descriptions de ces champs, reportez-vous à Ajout de fournisseurs de clés de service.
  5. Cliquez sur Enregistrer pour valider les mises à jour dans la session en cours.
  6. Pour terminer la session et déployer la configuration lors de l'exécution, cliquez sur Activer sous Centre des modifications.
