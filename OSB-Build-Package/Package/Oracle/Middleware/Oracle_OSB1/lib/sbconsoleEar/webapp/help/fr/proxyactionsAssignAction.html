Ajout d'actions d'affectation

L'action d'affectation permet d'affecter le résultat d'une expression XQuery à une variable de contexte.

Avant de commencer

Les instructions suivantes supposent que vous soyez déjà en train de modifier un flux de messages sur la page Modifier le flux de messages, comme indiqué dans Création, modification et visualisation de flux de messages.

Pour ajouter une action d'affectation, procédez comme suit :

  1. Accédez à l'emplacement auquel ajouter l'action, comme indiqué dans Ajout et modification d'actions dans les flux de messages.
  2. Cliquez sur l'icône appropriée, puis sélectionnez Ajouter une action > Traitement des messages > Affecter.
  3. Cliquez sur <Expression>. La page Editeur d'expressions XQuery apparaît. L'expression XQuery permet de créer les données à affecter à la variable nommée. Reportez-vous à Création et modification d'expressions XPath et XQuery imbriquées.
  4. Une fois l'expression modifiée, entrez une variable de contexte dans le champ Variable. Pour plus d'informations sur les variables de contexte, reportez-vous à Inbound and Outbound Variables à l'adresse suivante : http://www.oracle.com/pls/as111170/lookup?id=OSBAG325 et à Constructing Messages to Dispatch à l'adresse suivante : http://www.oracle.com/pls/as111170/lookup?id=OSBAG348 dans le manuel Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus.

Après avoir fini

Une fois la configuration de cette action terminée, vous pouvez configurer d'autres actions ou enregistrer votre configuration, comme indiqué dans Ajout et modification d'actions dans les flux de messages.
