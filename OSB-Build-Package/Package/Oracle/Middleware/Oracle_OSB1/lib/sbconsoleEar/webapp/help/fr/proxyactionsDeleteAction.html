Ajout d'actions de suppression

L'action de suppression permet de supprimer une variable de contexte ou un ensemble de noeuds spécifiés par une expression XPath. L'action de suppression fait partie des actions de mise à jour.

Avant de commencer

Les instructions suivantes supposent que vous soyez déjà en train de modifier un flux de messages sur la page Modifier le flux de messages, comme indiqué dans Création, modification et visualisation de flux de messages.

Pour ajouter une action de suppression, procédez comme suit :

  1. Accédez à l'emplacement auquel ajouter l'action, comme indiqué dans Ajout et modification d'actions dans les flux de messages.
  2. Pour supprimer une variable de contexte, sélectionnez le bouton d'option Variable, puis entrez le nom d'une variable de contexte dans le champ Variable. Pour en savoir plus sur les variables de contexte, reportez-vous à L'annexe A, Contexte de message.

    3. Pour supprimer tous les noeuds spécifiés par une expression XPath, sélectionnez le bouton d'option XPath, puis cliquez sur <XPath>. La page Editeur d'expressions XPath apparaît. Reportez-vous à Création et modification d'expressions XPath et XQuery imbriquées. Une fois l'expression enregistrée, entrez une variable de contexte dans le champ Variable. Pour en savoir plus sur les variables de contexte, reportez-vous à L'annexe A, Contexte de message.

Après avoir fini

Une fois la configuration de cette action terminée, vous pouvez configurer d'autres actions ou enregistrer votre configuration, comme indiqué dans Ajout et modification d'actions dans les flux de messages.
