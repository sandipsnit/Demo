Ajout d'actions de publication dynamique

L'action de publication dynamique permet de publier un message sur un service identifié par une expression XQuery.

Pour plus d'informations sur le comportement de publication, reportez-vous à Performing Transformations in Message Flows dans le manuel Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus à l'adresse http://www.oracle.com/pls/as111170/lookup?id=OSBAG205.

  1. Accédez à l'emplacement auquel ajouter l'action, comme indiqué dans Ajout et modification d'actions dans les flux de messages.
  2. Cliquez sur l'icône appropriée, puis sélectionnez Ajouter une action > Communication > Publication dynamique.
  3. Cliquez sur <Expression>.
  4. Dans l'éditeur d'expressions XQuery, entrez une expression XQuery ou sélectionnez une ressource XQuery dont le résultat est semblable à ce qui suit : 
    5. <ctx:route>

    <ctx:service 
isProxy="false">project/folder/businessservicename</ctx:service>

    <ctx:operation>foo</ctx:operation>

    </ctx:route>
    Remarque : Si un service proxy est appelé, l'attribut isProxy doit être défini sur True. Si un service fonctionnel est appelé, définissez isProxy sur False.
    L'élément operation est facultatif.
  5. Cliquez sur Enregistrer.
  6. Dans le champ Actions de demande, cliquez sur Ajouter une action pour ajouter une action, puis sélectionnez l'action à associer au service. Vous pouvez ajouter plusieurs actions. Pour en savoir plus sur les différents types d'action que vous pouvez ajouter, reportez-vous au tableau dans Ajout et modification d'actions dans les flux de messages.

Après avoir fini

Une fois la configuration de cette action terminée, vous pouvez configurer d'autres actions ou enregistrer votre configuration, comme indiqué dans Ajout et modification d'actions dans les flux de messages.
