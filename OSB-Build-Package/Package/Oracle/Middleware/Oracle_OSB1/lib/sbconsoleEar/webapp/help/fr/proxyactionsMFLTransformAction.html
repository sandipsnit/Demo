Ajout d'actions de transformation MFL

L'action de transformation MFL (Message Format Language) permet de convertir le contenu d'un message au format XML en un format non-XML, et inversement, dans le pipeline de messages. Un fichier MFL est un document XML spécialisé utilisé pour décrire la mise en page des données binaires. Il s'agit d'un langage propriétaire Oracle qui sert à définir les règles permettant de transformer des données binaires formatées en données XML, et inversement. Reportez-vous à Fichiers MFL.

Avant de commencer

Les instructions suivantes supposent que vous soyez déjà en train de modifier un flux de messages sur la page Modifier le flux de messages, comme indiqué dans Création, modification et visualisation de flux de messages.

Pour ajouter une action de transformation MFL, procédez comme suit :

  1. Accédez à l'emplacement auquel ajouter l'action, comme indiqué dans Ajout et modification d'actions dans les flux de messages.
  2. Cliquez sur l'icône appropriée, puis sélectionnez Ajouter une action > Traitement des messages > Transformation MFL.
  3. Dans la liste déroulante Appliquer la transformation MFL, sélectionnez De XML vers non-XML ou De non-XML vers XML, selon vos exigences.
  4. Cliquez sur <Expression>. A l'aide de l'éditeur d'expressions XQuery, spécifiez la variable sur laquelle effectuer l'action de transformation MFL. Il doit s'agir d'un élément texte ou binaire dans le cas d'une transformation vers le format XML et d'un élément XML dans le cas d'une transformation vers le format non-XML. Le contenu binaire du contexte de message est représenté par l'élément XML à contenu binaire. Cet élément XML est le résultat de l'expression XQuery lorsque l'entrée doit être binaire. Reportez-vous à Création et modification d'expressions XPath et XQuery imbriquées.
  5. Sélectionnez l'une des options suivantes :
    • Ressource MFL : cliquez sur le lien <Ressource>. La page Sélectionner une ressource MFL apparaît. Sélectionnez la ressource MFL statique qui exécutera l'action de transformation MFL.
    • Ressource MFL de l'élément : cliquez sur le lien Expression. La page Editeur d'expressions XQuery apparaît. A l'aide de l'éditeur d'expressions XQuery, créez ou modifiez une expression XQuery afin de spécifier de façon dynamique une ressource MFL qui exécutera l'action de transformation, au format project/folder/MFLresourcename. Reportez-vous à Création et modification d'expressions XPath et XQuery imbriquées.
  6. Dans le champ Affecter à la variable, saisissez le nom de la variable à laquelle affecter le résultat de l'action de transformation. Il s'agira d'un élément XML à contenu binaire.

Après avoir fini

Une fois la configuration de cette action terminée, vous pouvez configurer d'autres actions ou enregistrer votre configuration, comme indiqué dans Ajout et modification d'actions dans les flux de messages.
