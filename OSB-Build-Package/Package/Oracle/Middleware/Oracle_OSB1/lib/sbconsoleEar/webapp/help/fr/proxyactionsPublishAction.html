Ajout d'actions de publication

L'action de publication permet d'identifier un service cible spécifié de façon statique pour un message et de configurer la méthode de packaging et d'envoi du message vers ce service.

Pour plus d'informations sur le comportement de publication, reportez-vous à Performing Transformations in Message Flows dans le manuel Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus à l'adresse http://www.oracle.com/pls/as111170/lookup?id=OSBAG205.

  1. Accédez à l'emplacement auquel ajouter l'action, comme indiqué dans Ajout et modification d'actions dans les flux de messages.
  2. Sélectionnez Ajouter une action > Communication > Publication.
  3. Cliquez sur <Service>. La page Sélectionner un service apparaît.
  4. Sélectionnez un service dans la liste, puis cliquez sur Soumettre. Il s'agit du service cible pour le message.
  5. Si des opérations sont définies pour ce service, vous pouvez sélectionner une opération à appeler dans la liste Opération.
  6. Si vous souhaitez que l'opération sortante soit identique à l'opération entrante, cochez la case Utiliser une opération entrante pour l'opération sortante.
  7. Pour configurer la méthode de packaging et d'envoi du message vers le service choisi, dans le champ Actions de demande, cliquez sur Ajouter une action. Sélectionnez ensuite l'action à associer au service. Vous pouvez ajouter plusieurs actions. Reportez-vous à Ajout et modification d'actions dans les flux de messages.

Après avoir fini

Une fois la configuration de cette action terminée, vous pouvez configurer d'autres actions ou enregistrer votre configuration, comme indiqué dans Ajout et modification d'actions dans les flux de messages.
