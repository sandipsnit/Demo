Ajout d'actions de changement de nom

L'action de changement de nom permet de renommer les éléments sélectionnés par une expression XPath sans modifier leur contenu. L'action de changement de nom fait partie des actions de mise à jour.

Avant de commencer

Les instructions suivantes supposent que vous soyez déjà en train de modifier un flux de messages sur la page Modifier le flux de messages, comme indiqué dans Création, modification et visualisation de flux de messages.

Pour ajouter une action de changement de nom, procédez comme suit :

  1. Accédez à l'emplacement auquel ajouter l'action, comme indiqué dans Ajout et modification d'actions dans les flux de messages.
  2. Cliquez sur l'icône appropriée, puis sélectionnez Ajouter une action > Traitement des messages > Renommer.
  3. Cliquez sur <XPath>. La page Editeur d'expressions XPath apparaît. L'expression XPath permet de spécifier les données (dans la variable nommée) à renommer. Reportez-vous à Création et modification d'expressions XPath et XQuery imbriquées.
  4. Dans le champ variable, entrez la variable de contexte qui contient l'élément à renommer. Pour plus d'informations sur les variables de contexte, reportez-vous à L'annexe A, Contexte de message.
  5. Effectuez l'une des opérations suivantes :
    • Pour renommer les éléments sélectionnés avec un nom local, sélectionnez le premier bouton d'option nom local, puis entrez un nom local dans le champ nom local.
    • Pour renommer les éléments sélectionnés avec un espace de noms, sélectionnez le premier bouton d'option espace de noms, puis entrez un espace de noms dans le champ espace de noms.
    • Pour renommer les éléments sélectionnés avec un nom local et un espace de noms, sélectionnez le bouton d'option nom local et espace de noms, puis entrez un nom local et un espace de noms dans les champs nom local et espace de noms.

Après avoir fini

Une fois la configuration de cette action terminée, vous pouvez configurer d'autres actions ou enregistrer votre configuration, comme indiqué dans Ajout et modification d'actions dans les flux de messages.
