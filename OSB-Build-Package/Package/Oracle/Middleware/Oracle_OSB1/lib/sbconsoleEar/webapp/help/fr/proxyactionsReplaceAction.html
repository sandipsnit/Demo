Ajout d'actions de remplacement

L'action de remplacement permet de remplacer un noeud ou le contenu d'un noeud spécifié par une expression XPath. Le noeud ou son contenu est remplacé par la valeur renvoyée par une expression XQuery.

Une action de remplacement peut être utilisée pour remplacer des valeurs, des éléments et des attributs simples. Si une expression XQuery ne renvoie aucune valeur, les noeuds identifiés sont supprimés ou vidés, selon que l'action remplace des noeuds entiers ou uniquement leur contenu.

L'action de remplacement fait partie des actions de mise à jour.

Avant de commencer

Les instructions suivantes supposent que vous soyez déjà en train de modifier un flux de messages sur la page Modifier le flux de messages, comme indiqué dans Création, modification et visualisation de flux de messages.

Pour ajouter une action de remplacement, procédez comme suit :

  1. Accédez à l'emplacement auquel ajouter l'action, comme indiqué dans Ajout et modification d'actions dans les flux de messages.
  2. Cliquez sur l'icône appropriée, puis sélectionnez Ajouter une action > Traitement des messages > Remplacer.
  3. Cliquez sur <XPath>. La page Editeur d'expressions XPath apparaît. L'expression XPath permet de spécifier les données (dans la variable nommée) à remplacer. Reportez-vous à Création et modification d'expressions XPath et XQuery imbriquées.
  4. Une fois l'expression XPath modifiée, entrez une variable de contexte dans le champ variable. Reportez-vous à L'annexe A, Contexte de message.
  5. Cliquez sur <Expression>. La page Editeur d'expressions XQuery apparaît. L'expression XPath permet de spécifier les données (dans la variable nommée) à remplacer. Reportez-vous à Création et modification d'expressions XPath et XQuery imbriquées.
  6. Une fois l'expression XQuery modifiée, sélectionnez l'une des options suivantes :
    • Remplacer tout le noeud : choisissez cette option pour remplacer les noeuds sélectionnés par l'expression XPath définie, ainsi que l'ensemble de leur contenu.
    • Remplacer le contenu du noeud : choisissez cette option pour remplacer uniquement le contenu des noeuds.
      • Remarque : il est plus efficace de sélectionner l'option Remplacer le contenu du noeud et de laisser le champ XPath vide que de sélectionner l'option Remplacer tout le noeud et de définir XPath sur ./*

Après avoir fini

Une fois la configuration de cette action terminée, vous pouvez configurer d'autres actions ou enregistrer votre configuration, comme indiqué dans Ajout et modification d'actions dans les flux de messages.
