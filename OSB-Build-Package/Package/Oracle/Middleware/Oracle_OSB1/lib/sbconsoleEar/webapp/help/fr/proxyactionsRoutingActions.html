Ajout d'actions de routage aux noeuds de routage

Identifiez un service cible pour le message et configurez la méthode de routage du message vers ce service.

Il s'agit d'une action finale ; cela signifie que vous ne pouvez pas ajouter d'autres actions après celle-ci. Cette action peut toutefois inclure des actions de demande et de réponse.

Pour plus d'informations, reportez-vous à "Modeling Message Flow in Oracle Service Bus" dans le manuel Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus à l'adresse suivante : http://www.oracle.com/pls/as111170/lookup?id=OSBAG181.

Avant de commencer

Les instructions suivantes supposent que vous soyez déjà en train de modifier un flux de messages sur la page Modifier le flux de messages, comme indiqué dans Création, modification et visualisation de flux de messages.

Pour ajouter une action de routage à un noeud de routage, procédez comme suit :

  1. Accédez à l'emplacement auquel ajouter l'action, comme indiqué dans Ajout et modification d'actions dans les flux de messages.
  2. Cliquez sur l'icône Noeud de routage, puis sur Modifier le routage. La page Modifier la configuration de la phase apparaît.
  3. Cliquez sur l'icône Ajouter une action, puis sélectionnez Communication > Routage.
  4. Cliquez sur Service. Le navigateur Web de services apparaît.
  5. Sélectionnez un service dans la liste, puis cliquez sur Soumettre. Le service apparaît à la place du lien par défaut.
  6. Si vous souhaitez que l'opération sortante soit identique à l'opération entrante, cochez la case Utiliser une opération entrante pour l'opération sortante.
  7. Dans le champ Actions de demande, cliquez sur Ajouter une action pour ajouter une action, puis sélectionnez l'action à associer au service. Vous pouvez ajouter plusieurs actions. Pour en savoir plus sur les différents types d'action que vous pouvez ajouter, reportez-vous au tableau dans Ajout et modification d'actions dans les flux de messages.
  8. Dans le champ Actions de réponse, cliquez sur Ajouter une action pour ajouter une action, puis sélectionnez l'action à associer au service. Vous pouvez ajouter plusieurs actions. Pour plus d'informations sur les différents types d'action que vous pouvez ajouter, reportez-vous au tableau des actions dans Ajout et modification d'actions dans les flux de messages.
  9. Cliquez sur Enregistrer.
  10. Sur la page Modifier le flux de messages, poursuivez la construction du flux de messages, comme indiqué dans Création, modification et visualisation de flux de messages.
  11. Cliquez sur Enregistrer pour valider les mises à jour dans la session en cours.
  12. Pour terminer la session et déployer la configuration lors de l'exécution, cliquez sur Activer sous Centre des modifications.
