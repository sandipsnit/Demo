Ajout d'actions de non-prise en compte

L'action de non-prise en compte permet d'indiquer, au moment de l'exécution, que l'application de la phase en cours est ignorée et que la phase suivante dans le flux de messages est directement traitée. Cette action ne comporte pas de paramètres et peut être utilisée dans les pipelines de demandes, de réponses ou d'erreurs.

Avant de commencer

Les instructions suivantes supposent que vous soyez déjà en train de modifier un flux de messages sur la page Modifier le flux de messages, comme indiqué dans Création, modification et visualisation de flux de messages.

Pour ajouter une action de non-prise en compte, procédez comme suit :

  1. Accédez à l'emplacement auquel ajouter l'action, comme indiqué dans Ajout et modification d'actions dans les flux de messages.
  2. Cliquez sur l'icône appropriée, puis sélectionnez Ajouter une action > Contrôle de flux > Ignorer.

Après avoir fini

Une fois la configuration de cette action terminée, vous pouvez configurer d'autres actions ou enregistrer votre configuration, comme indiqué dans Ajout et modification d'actions dans les flux de messages.
