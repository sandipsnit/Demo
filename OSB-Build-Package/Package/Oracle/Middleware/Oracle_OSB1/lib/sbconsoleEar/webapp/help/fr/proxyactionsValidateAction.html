Ajout d'actions de validation

L'action de validation permet de valider les éléments sélectionnés par une expression XPath par rapport à un élément de schéma XML ou une ressource WSDL. Vous ne pouvez valider que des éléments globaux ; Oracle Service Bus ne prend pas en charge la validation par rapport à des éléments locaux.

Avant de commencer

Les instructions suivantes supposent que vous soyez déjà en train de modifier un flux de messages sur la page Modifier le flux de messages, comme indiqué dans Création, modification et visualisation de flux de messages.

Pour ajouter une action de validation, procédez comme suit :

  1. Accédez à l'emplacement auquel ajouter l'action, comme indiqué dans Ajout et modification d'actions dans les flux de messages.
  2. Cliquez sur l'icône appropriée, puis sélectionnez Ajouter une action > Traitement des messages > Valider.
  3. Cliquez sur XPath pour construire une expression XPath indiquant les éléments à valider. Reportez-vous à Création et modification d'expressions XPath et XQuery imbriquées. Une fois l'expression créée dans l'éditeur d'expressions XPath, cliquez sur Enregistrer pour insérer l'expression dans la page Modifier la configuration de la phase.
  4. Dans le champ dans la variable, entrez le nom de la variable qui doit contenir l'élément à valider.
  5. Cliquez sur ressource, puis sélectionnez WSDL ou Schéma.
  6. A partir du navigateur Web WSDL ou du navigateur Web de schéma XML, effectuez les opérations suivantes :
    1. Sélectionnez le WSDL ou le schéma XML
    2. Sélectionnez le type ou l'élément de WSDL ou de schéma XML
    3. Cliquez sur Soumettre.
  7. Pour enregistrer le résultat (résultat booléen) de cette validation, sélectionnez Enregistrer le résultat de la validation dans la variable et entrez le nom de la variable dans laquelle enregistrer le résultat.

    8. Pour générer une erreur en cas d'échec de la validation de l'élément par rapport au WSDL ou à l'élément de schéma XML, sélectionnez Générer une erreur lors de l'échec de la validation.

Après avoir fini

Une fois la configuration de cette action terminée, vous pouvez configurer d'autres actions ou enregistrer votre configuration, comme indiqué dans Ajout et modification d'actions dans les flux de messages.
