Services proxy : éditeurs XQuery et XPath

Dans le flux de messages d'un service proxy, vous pouvez affecter des expressions XQuery aux variables de contexte de message, affecter des actions If...Then... en fonction du résultat booléen d'une expression XQuery, insérer le résultat d'une expression XQuery dans un emplacement relatif à une expression XPath et spécifier le contexte de message à journaliser via des expressions XQuery sur des variables de contexte, entre autres.

L'éditeur d'expressions XQuery, l'éditeur de conditions XQuery et l'éditeur d'expressions XPath sont disponibles dans le contexte approprié dans les flux de messages et permettent de construire le type d'expression appelé dans le contexte.
