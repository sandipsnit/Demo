Ajout de phases aux pipelines

Avant de commencer

Les instructions suivantes supposent que vous soyez déjà en train de modifier un flux de messages sur la page Modifier le flux de messages, comme indiqué dans Création, modification et visualisation de flux de messages.

Pour ajouter une phase à un pipeline, procédez comme suit :

  1. Si nécessaire, cliquez sur le signe plus à gauche de l'icône Noeud de paire de pipeline pour développer la zone. Une paire de pipelines contient un pipeline des demandes et un pipeline des réponses.
  2. Cliquez sur le pipeline auquel ajouter la phase, puis sur Ajouter une phase.
  3. Pour modifier le nom par défaut de la phase et ajouter une description, procédez comme suit :
    1. Cliquez sur l'icône Phase, puis sur Modifier le nom et l'annotation.
    2. Modifiez le nom et la description, comme souhaité.
    3. Cliquez sur Enregistrer.
  4. Pour ajouter des actions à la phase, cliquez sur l'icône Phase, puis sur Modifier la phase. Reportez-vous à Ajout et modification d'actions dans les flux de messages.
  5. Pour ajouter un gestionnaire d'erreur à la phase, cliquez sur l'icône Phase, puis sur Ajouter un gestionnaire d'erreur de phase. Reportez-vous à Ajout de gestionnaires d'erreur de phase. La page Modifier le flux de messages apparaît.
  6. Continuez à construire le flux de messages, comme indiqué dans Création, modification et visualisation de flux de messages.
  7. Cliquez sur Enregistrer pour valider les mises à jour dans la session en cours.
  8. Pour terminer la session et déployer la configuration lors de l'exécution, cliquez sur Activer sous Centre des modifications.
