Construction d'expressions dans les champs de texte de l'espace de travail de l'éditeur

L'éditeur d'expressions XQuery, l'éditeur de conditions XQuery et l'éditeur d'expressions XPath présentent tous les trois des champs de texte, dans lesquels vous pouvez créer des expressions en les saisissant directement ou en collant des éléments à partir des palettes.

Avant de commencer

Les instructions suivantes supposent que soyez en train de créer ou modifier une expression via l'éditeur d'expressions XQuery, l'éditeur de conditions XQuery ou l'éditeur d'expressions XPath, comme indiqué dans Création et modification d'expressions XPath et XQuery imbriquées.

Pour créer une expression dans un champ de texte, procédez comme suit :

  1. Affichez le panneau contenant le champ de texte. En fonction de l'éditeur, effectuez les opérations suivantes :
    • Dans l'éditeur d'expressions XQuery, sélectionnez Texte XQuery (sous la barre de boutons de l'espace de travail), si ce n'est pas déjà fait.
    • Dans l'éditeur de conditions XQuery, sélectionnez Texte (sous la barre de boutons de l'espace de travail), si ce n'est pas déjà fait.
    • Dans l'éditeur d'expressions XPath, aucune sélection n'est nécessaire car il n'est pas possible de sélectionner d'autres outils.
      • Remarque : lorsque vous sélectionnez un des liens ci-dessus, un champ de texte apparaît. Il permet de créer une expression complète appropriée au contexte. Cependant, vous pouvez également utiliser les outils et les techniques décrits dans cette rubrique à chaque fois que des champs de texte sont fournis dans les éditeurs, par exemple lors de la liaison de variables à partir de ressources importées vers l'expression imbriquée, comme indiqué dans Liaison de ressources XQuery externes et d'expressions XQuery imbriquées et Liaison de ressources XSLT externes et d'expressions XQuery imbriquées.
  2. Si vous le souhaitez, entrez ou collez une expression ou un fragment d'expression dans le champ. Si vous créez l'expression complète de cette façon, passez directement à l'étape 7 ci-dessous. Sinon, passez à l'étape suivante.
  3. Sélectionnez la palette contenant les éléments à ajouter à l'expression, puis localisez l'élément à ajouter, comme indiqué dans le tableau 22-1 ci-dessous.

    4. Tableau 22-1 Palettes 
    Palette
    Description et utilisation
    Définitions d'espace de noms
    Répertorie les espaces de noms par défaut d'Oracle Service Bus, les espaces de noms de variables et les espaces de noms définis par les utilisateurs. Les abréviations des espaces de noms sont répertoriées lors de leur définition.
    Parcourez les listes pour trouver l'espace de noms souhaité.
    Vous pouvez également définir un espace de noms. Reportez-vous à Création d'espaces de noms à utiliser dans les expressions imbriquées.
    Fonctions XQuery
    Contient un ensemble de fonctions XQuery standard, triées par ordre alphabétique et par type, y compris les fonctions personnalisées que vous avez ajoutées. Développez ou réduisez des noeuds de l'arborescence en cliquant sur le signe plus (+) ou le signe moins (-).
    Voir aussi :
    • Transformations XQuery
    • "XQuery Implementation" dans le manuel Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus à l'adresse suivante : http://www.oracle.com/pls/as111170/lookup?id=OSBAG444.
    Structures de variable
    Affiche les variables et leur contenu sous forme d'arborescences, afin de mieux les visualiser.
    Pour afficher une structure de variable et son contenu, procédez comme suit :
    • Sélectionnez le nom de la structure dans la liste déroulante en haut de la palette. La liste affiche les variables de contexte de message intégrées (attachments, body, header, outbound et inbound), ainsi que toutes les structures définies par l'utilisateur, organisées par type (Type XML, Interface de service et Type simple).
    • Développez ou réduisez les noeuds de l'arborescence en cliquant sur le signe plus (+) ou le signe moins (-).
    Vous pouvez également définir vos propres structures de variable. Reportez-vous à Création de structures de variable dans les éditeurs XQuery.
    Les structures de variable ne créent pas de variables. Les variables sont créées lors de l'exécution en tant que cibles de l'action d'affectation dans la phase.
    Lorsque vous insérez un élément à partir de l'arborescence de structures de variable dans le champ de texte, il est inséré en tant qu'expression XPath décrivant le chemin.
    Reportez-vous à Transformations XQuery.

  4. Collez l'élément souhaité dans le champ de texte à l'aide de l'une des méthodes présentées dans le tableau 22-2 ci-dessous :

    5. Tableau 22-2 Méthodes de collage des éléments dans les champs de texte de l'éditeur 
    Palette
    Procédure
    Palette Définitions d'espace de noms
    Vous pouvez sélectionner, copier et coller un espace de noms de façon standard, à l'aide de votre souris ou de votre clavier, par exemple :
    1. Sélectionnez la chaîne de l'espace de noms (ou son abréviation, le cas échéant) en faisant glisser le pointeur de la souris sur la chaîne.
    2. Appuyez sur Ctrl+C pour copier la chaîne.
    3. Cliquez dans le champ de texte à l'endroit où insérer l'espace de noms.
    4. Appuyez sur Ctrl+V pour coller la chaîne.
    Palette Fonctions XQuery ou Structures de variable
    Faites glisser un élément de la palette vers le champ de texte.
    Remarque : le déplacement d'un élément de la palette vers l'espace de travail n'est pris en charge que dans Internet Explorer.
    Palette Fonctions XQuery ou Structures de variable
    1. Cliquez sur un élément de la palette. L'élément est affiché dans le panneau Gestionnaire de propriétés :
      • Les fonctions sont affichées avec leurs éléments de réserve pour toutes les valeurs que vous devez fournir.
      • Les variables et leurs attributs sont affichés en tant qu'expressions XPath.
    2. Cliquez dans le champ de texte à l'endroit où insérer l'élément
    3. Cliquez sur Copier une propriété pour coller l'élément à l'emplacement sélectionné dans le champ de texte.
    Palette Fonctions XQuery ou Structures de variable
    1. Cliquez sur un élément de la palette. L'élément est affiché dans le panneau Gestionnaire de propriétés.
    2. Sélectionnez et copiez l'élément dans le panneau Gestionnaire de propriétés à l'aide des opérations standard effectuées via le clavier ou la souris.
    3. Sélectionnez un emplacement dans le champ de texte et collez l'élément à l'aide des opérations standard effectuées via le clavier ou la souris.

  5. Faites glisser les fonctions voulues pour créer l'expression souhaitée.
  6. Modifiez l'expression dans le champ de texte, comme souhaité.
  7. Vous pouvez également exécuter une des tâches suivantes ou les deux :
    • Cliquez sur Valider. Un message apparaît si la validation de l'expression a réussi.
    • Cliquez sur Tester pour tester l'expression. Reportez-vous à Exécution du test XQuery.
  8. Cliquez sur Enregistrer pour fermer l'éditeur et insérer l'expression dans l'action.

Après avoir fini

Continuez à configurer l'action, comme indiqué dans Ajout et modification d'actions dans les flux de messages.
