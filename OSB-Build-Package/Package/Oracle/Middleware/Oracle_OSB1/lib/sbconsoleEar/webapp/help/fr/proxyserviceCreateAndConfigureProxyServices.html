Création et configuration de services proxy

Cette section explique comment créer et configurer des services proxy.

En ce qui concerne les services JCA, vous pouvez générer des services proxy à partir des liaisons JCA, comme indiqué dans Génération d'un service proxy à partir d'une ressource de liaison JCA.

  1. Si vous ne l'avez pas déjà fait, cliquez sur Créer afin de créer une session ou cliquez sur Modifier pour accéder à une session existante. Reportez-vous à Utilisation du centre des modifications.
  2. Sélectionnez Explorateur de projets, puis choisissez le projet ou le dossier auquel ajouter le service proxy.
  3. Dans la page Vue du projet/Vue de dossier, sélectionnez Service proxy dans la liste déroulante Créer une ressource.
  4. Dans la page Configuration générale, indiquez le nom du service et sélectionnez le type de service à créer.

    Pour obtenir des conseils sur la dénomination, suivez les restrictions concernant la dénomination des ressources.
  5. Cliquez sur Suivant. Les pages suivantes dépendent des choix que vous avez faits sur la première page. Entrez les informations appropriées sur chacune des pages suivantes jusqu'à accéder à la page récapitulative, puis cliquez sur Enregistrer pour enregistrer le service de la session en cours.

    6. Pour obtenir des instructions détaillées afin de remplir chaque page, reportez-vous aux rubriques suivantes :

    • Configuration générale, page
    • Page Configuration du type de message
    • Page Configuration du transport
    • Page Protocole - Configuration du transport
    • Page Configuration de la sélection d'opération
    • Page Gestion des messages
    • Page Récapitulatif
    • Page Stratégie de service proxy
    • Page Sécurité de service proxy
  6. Pour terminer la session et déployer la configuration lors de l'exécution, cliquez sur Activer sous Centre des modifications.
  7. Une fois le service proxy créé, l'étape suivante consiste à configurer le flux de messages associé. Le flux de messages définit l'implémentation d'un service proxy. Pour plus d'informations, reportez-vous à Création, modification et visualisation de flux de messages.

Génération d'un service proxy à partir d'une ressource de liaison JCA

Vous pouvez générer un service proxy JCA à partir d'une ressource de liaison JCA entrante dans Oracle Service Bus. Pour plus d'informations sur les ressources de liaison JCA, reportez-vous au chapitre 12, Liaisons JCA.

Pour générer un service proxy JCA à partir d'une liaison JCA, procédez comme suit :

  1. Dans la console d'administration d'Oracle Service Bus, cliquez sur Créer ou sur Modifier dans le centre des modifications si vous n'êtes pas encore en mode Créer ou Modifier.
  2. Dans le navigateur Web de ressources, cliquez sur Liaisons JCA.
  3. Localisez la liaison JCA à partir de laquelle vous souhaitez générer un service et cliquez sur l'icône Action.
  4. Dans la fenêtre qui apparaît, confirmez le nom de la ressource WSDL et le service que vous souhaitez générer, sélectionnez l'emplacement de ces nouvelles ressources et cliquez sur Générer.

    5. Oracle Service Bus génère le service et la ressource WSDL correspondante.

  5. Procédez aux modifications nécessaires de tous les autres détails de configuration du service généré, tels que l'URI endpoint.
