Création, modification et visualisation de flux de messages

  1. Si vous ne l'avez pas déjà fait, cliquez sur Créer afin de créer une session ou cliquez sur Modifier pour accéder à une session existante. Reportez-vous à Utilisation du centre des modifications.
  2. Localisez le service proxy dont vous souhaitez visualiser ou modifier le flux de messages. Effectuez l'une des opérations suivantes :
    • Sélectionnez Explorateur de projets pour afficher la page Vue des projets ou la page Vue du projet/Vue de dossier. Parcourez ensuite les projets et dossiers afin d'accéder au service proxy.
    • Sélectionnez Navigateur Web de ressources > Services proxy pour afficher la page Récapitulatif des services proxy.
  3. Cliquez sur l'icône Modifier le flux de messages dans la ligne du service proxy dont vous souhaitez visualiser ou modifier le flux de messages.
    • Si aucun flux de messages n'a encore été créé pour le service proxy sélectionné, la page Modifier le flux de messages affiche uniquement l'icône Service proxy. Il s'agit du noeud de début du service proxy. Cliquez sur cette icône pour commencer à construire le flux de messages.
    • Si le service proxy contient déjà un flux de messages, la page contient une représentation graphique du flux. Cliquez sur les icônes affichées pour visualiser ou modifier les différentes parties du flux de messages.

      • Reportez-vous à Page Modifier le flux de messages pour plus d'informations sur les actions que vous pouvez exécuter pour construire un flux de messages sur cette page.

  4. Cliquez sur Enregistrer pour valider les mises à jour dans la session en cours.
  5. Pour terminer la session et déployer la configuration lors de l'exécution, cliquez sur Activer sous Centre des modifications.
