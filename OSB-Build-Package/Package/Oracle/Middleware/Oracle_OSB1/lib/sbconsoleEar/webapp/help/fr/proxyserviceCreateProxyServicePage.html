Créer/Modifier un service proxy - Référence de page

Créez un service proxy en sélectionnant Service proxy dans la liste déroulante Créer une ressource sur la page Vue du projet/Vue de dossier. Lorsque vous sélectionnez cette option, la première des pages destinées à la configuration et à l'ajout de services proxy apparaît. Les pages affichées dépendent des options sélectionnées au fur et à mesure. Voici la liste des pages susceptibles d'apparaître :

  • Configuration générale, page
  • Page Configuration du type de message
  • Page Configuration du transport
  • Page Protocole - Configuration du transport
  • Page Configuration de la sélection d'opération
  • Page Gestion des messages
  • Page Récapitulatif
  • Page Stratégie de service proxy
  • Page Sécurité de service proxy
