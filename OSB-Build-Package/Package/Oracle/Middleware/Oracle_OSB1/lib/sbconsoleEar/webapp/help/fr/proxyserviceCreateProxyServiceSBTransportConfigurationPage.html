Page SB - Configuration du transport

La page Créer/Modifier un service proxy - SB - Configuration du transport est l'une des pages destinées à la création et à la modification de services proxy, décrites dans les rubriques Création et configuration de services proxy et Modification des configurations des services proxy.

Pour plus d'informations, reportez-vous à SB Transport dans le manuel Oracle Fusion Middleware Developer's Guide for Oracle Service Bus, à l'adresse suivante : http://www.oracle.com/pls/as111170/lookup?id=OSBDV997.

Cette page permet de configurer les paramètres de transport pour un service proxy utilisant le protocole de transport SB (Service Bus). Le tableau 19-12 explique comment l'utiliser.

Tableau 19-12 Page Créer/Modifier un service proxy - SB - Configuration du transport
Option
Procédure
Stratégie de répartition
Sélectionnez une stratégie de répartition pour cette adresse ou utilisez la stratégie de répartition par défaut.
La stratégie de répartition fait référence à l'instance du gestionnaire de travaux WLS que vous souhaitez utiliser pour l'adresse de service pour traiter la demande. Pour obtenir des informations sur les gestionnaires de travaux, reportez-vous à Using Work Managers to Optimize Scheduled Work dans Oracle Fusion Middleware Configuring Server Environments for Oracle WebLogic Server, à l'adresse suivante : http://www.oracle.com/pls/as111170/lookup?id=CNFGD112.
Utiliser SSL
Lorsque cela est indiqué, les demandes doivent être envoyées via une connexion SSL.
Cependant, les connexions non sécurisées ne sont pas interdites. L'administrateur doit fermer tous les protocoles non sécurisés sur le serveur (par exemple, t3 ou http) pour imposer strictement des connexions client sécurisées.
