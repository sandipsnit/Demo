Page Récapitulatif

La page Créer/Modifier un service proxy - Récapitulatif est l'une des pages destinées à la création et à la modification de services proxy, décrites dans les rubriques Création et configuration de services proxy et Modification des configurations des services proxy.

Cette page permet de visualiser ou de modifier les paramètres de configuration d'un service proxy avant de l'enregistrer.

Pour visualiser ou modifier les paramètres, cliquez sur Modifier dans la ligne de la catégorie appropriée (par exemple, Configuration générale, Configuration du transport, etc.). Les pages que vous pouvez modifier dépendent des pages que vous avez configurées au moment de la création du service proxy. La liste suivante affiche toutes les pages :

  • Configuration générale, page
  • Page Configuration du type de message
  • Page Configuration du transport
  • Page Protocole - Configuration du transport
  • Page Courrier électronique - Configuration du transport
  • Page Fichier - Configuration du transport
  • Page FTP - Configuration du transport
  • Page HTTP - Configuration du transport
  • Page JCA - Configuration du transport
  • Page JEJB - Configuration du transport
  • Page JMS - Configuration du transport
  • Page MQ - Configuration du transport
  • Page SB - Configuration du transport
  • Page SFTP - Configuration du transport
  • Page Tuxedo - Configuration du transport
  • Page WS - Configuration du transport
  • Page Configuration de la sélection d'opération
  • Page Gestion des messages
