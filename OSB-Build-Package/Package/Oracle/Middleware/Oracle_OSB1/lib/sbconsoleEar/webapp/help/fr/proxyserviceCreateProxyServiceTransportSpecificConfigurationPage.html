Page Protocole - Configuration du transport

Les pages Créer/Modifier un service proxy - Configuration du type de transport sont destinées à la création et à la modification de services proxy, décrites dans les rubriques Création et configuration de services proxy et Modification des configurations des services proxy.

Cette page est différente selon le type de transport que vous avez choisi de configurer. Il peut s'agir des éléments suivants :

  • Page Courrier électronique - Configuration du transport
  • Page Fichier - Configuration du transport
  • Page FTP - Configuration du transport
  • Page HTTP - Configuration du transport
  • Page JCA - Configuration du transport
  • Page JEJB - Configuration du transport
  • Page JMS - Configuration du transport
  • Page MQ - Configuration du transport
  • Page SB - Configuration du transport
  • Page SFTP - Configuration du transport
  • Page Tuxedo - Configuration du transport
  • Page WS - Configuration du transport
