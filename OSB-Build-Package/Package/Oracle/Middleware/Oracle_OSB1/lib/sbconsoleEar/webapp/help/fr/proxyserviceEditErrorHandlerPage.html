Page Modifier le gestionnaire d'erreur

Vous pouvez configurer la gestion des erreurs au niveau du flux de messages, du pipeline, du noeud de routage et de la phase.

Configurez les gestionnaires d'erreur sur la page Modifier le gestionnaire d'erreur. Vous devez toujours ajouter au moins une phase à la page pour préciser le mode de fonctionnement du gestionnaire d'erreur. Reportez-vous aux rubriques suivantes :

  • Ajout de gestionnaires d'erreur de service proxy
  • Ajout de gestionnaires d'erreur de pipeline
  • Ajout de gestionnaires d'erreur de phase
  • Ajout de gestionnaires d'erreur de noeud de routage
  • Modification des gestionnaires d'erreur
  • "Handling Errors in Message Flows" dans le manuel Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus, à l'adresse suivante : http://www.oracle.com/pls/as111170/lookup?id=OSBAG232.
