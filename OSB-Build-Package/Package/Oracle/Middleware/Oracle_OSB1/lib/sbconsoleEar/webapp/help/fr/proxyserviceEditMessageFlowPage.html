Page Modifier le flux de messages

Utilisez la page Modifier le flux de messages afin de créer un flux de messages pour un service proxy.

Le panneau de navigation à gauche de la page Modifier le flux de messages affiche une vue d'arborescence des noeuds et des objets du flux de messages. Lorsque les détails d'un objet sont définis dans une page séparée, vous pouvez cliquer sur le nom de l'objet pour afficher la page associée.

Le panneau de droite affiche un champ dans lequel vous pouvez créer·un flux de messages. Lorsque aucun flux de messages n'a encore été défini, le panneau inclut une seule icône Service proxy indiquant le noeud de début du service. Cliquez sur l'icône pour ajouter des noeuds de paires de pipelines, des noeuds de routage, des branchements conditionnels, des branchements opérationnels et un gestionnaire d'erreur pour le service.

Lorsque vous ajoutez des objets à la page, des icônes représentant les objets apparaissent sur cette page. Les relations entre les objets sont indiquées par des lignes et des boîtes englobantes. Cliquez sur une icône de·la page Modifier le flux de messages pour afficher le menu des actions que vous pouvez exécuter sur cet objet. Les options disponibles dans ce menu varient selon le contexte.
