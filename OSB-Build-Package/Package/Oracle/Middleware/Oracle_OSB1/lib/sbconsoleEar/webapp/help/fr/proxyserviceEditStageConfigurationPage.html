Page Modifier la configuration de la phase

Utilisez la page Modifier la configuration de la phase pour ajouter des actions aux phases de pipeline, aux phases du gestionnaire d'erreur et aux noeuds de routage dans un flux de messages.

  • Lorsque aucun paramètre n'a été défini sur la page Modifier la configuration de la phase, le seul objet affiché est l'icône Ajouter une action. Cliquez sur cette icône pour commencer.
  • Lorsqu'un noeud de phase ou de routage a·déjà été configuré, les actions et les objets définis pour ce noeud de phase ou de routage apparaissent sur cette page. Modifiez les actions existantes selon vos besoins ou cliquez sur les icônes représentant des actions pour ajouter d'autres actions à la phase.

Reportez-vous à Ajout et modification d'actions dans les flux de messages pour obtenir des instructions sur l'utilisation de tous les types d'actions que vous pouvez ajouter à une phase.
