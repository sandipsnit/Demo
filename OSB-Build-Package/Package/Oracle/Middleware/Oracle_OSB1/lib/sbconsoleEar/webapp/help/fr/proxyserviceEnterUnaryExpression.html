Saisie d'expressions unaires à l'aide de l'option Générateur

Avant de commencer

Ces instructions supposent que vous créez ou que vous modifiez une expression conditionnelle XQuery dans l'éditeur de conditions XQuery, comme décrit dans Création et modification d'expressions XPath et XQuery imbriquées.

Pour entrer une expression de comparaison XQuery, procédez comme suit :

  1. Dans l'éditeur de conditions XQuery, sélectionnez l'option Générateur (située en dessous de la barre de boutons de l'espace de travail), si ce n'est pas déjà fait.
  2. Dans la zone Générateur d'expressions, sélectionnez Expression unaire, si ce n'est pas déjà fait.
  3. Cochez la case NOT pour obtenir une expression négative, ou laissez-la vide.
  4. Entrez une variable de contexte, une définition d'espace de noms ou une fonction XQuery dans le champ Expression.

    5. Pour construire l'expression, vous pouvez coller les fonctions XQuery à partir de la palette Fonctions XQuery et les espaces de noms à partir de la palette Définitions d'espace de noms. Reportez-vous à l'étape 3 et à l'étape 4 dans Construction d'expressions dans les champs de texte de l'espace de travail de l'éditeur.

  5. Cliquez sur Ajouter pour ajouter le texte au panneau Expressions.
  6. Répétez les étapes 3 à 5 pour construire d'autres conditions. Chaque condition est ajoutée à la fin de la liste.

    7. Tenez compte des éléments suivants lorsque vous construisez des expressions unaires :

    • Lorsque vous construisez des expressions supplémentaires, veillez à sélectionner les options AND ou OR dans le champ Conjonction.
    • Vous pouvez sélectionner une condition et cliquer sur la flèche vers le haut pour la faire remonter dans la liste des conditions. Cliquez sur la flèche vers le bas pour la faire descendre. Vous pouvez également cliquer sur l'icône Modifier pour la mettre à jour ou sur l'icône Supprimer pour la supprimer.
    • Des expressions unaires peuvent être mélangées aux expressions de comparaison dans la définition globale d'une condition.
  7. Vous pouvez également exécuter une des tâches suivantes ou les deux :
    • Cliquez sur Valider. Un message apparaît si la validation de l'expression a réussi.
    • Cliquez sur Tester pour tester l'expression. Reportez-vous à Exécution du test XQuery.
  8. Cliquez sur Enregistrer pour fermer l'éditeur et insérer l'expression dans l'action.

Après avoir fini

Continuez à configurer l'action, comme indiqué dans Ajout et modification d'actions dans les flux de messages.
