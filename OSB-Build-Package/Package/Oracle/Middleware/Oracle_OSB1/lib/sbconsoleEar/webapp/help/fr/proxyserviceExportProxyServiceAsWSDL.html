Export d'un WSDL associé à un service proxy

Vous pouvez exporter le WSDL d'un service proxy WSDL, afin de pouvoir le visualiser ou le modifier dans un outil externe tel qu'un IDE. Le WSDL est exporté en tant que fichier JAR.

Cette fonction est différente de la fonction Exporter les ressources du module Administration du système, que vous utilisez pour déplacer et préparer les ressources entre deux domaines. Reportez-vous à l'export de ressources.

Avant de commencer

Vous pouvez exporter une ressource WSDL uniquement lorsque vous êtes en dehors d'une session. Reportez-vous à Utilisation du centre des modifications.

Pour exporter une ressource WSDL

  1. Effectuez l'une des opérations suivantes :
    • Sélectionnez Navigateur Web de ressources > Services proxy pour afficher la page Récapitulatif des services proxy.
    • Sélectionnez Explorateur de projets, puis choisissez·le projet ou le dossier contenant le service proxy que vous souhaitez exporter en tant que WSDL. La page Vue du projet/Vue de dossier apparaît.
  2. Cliquez sur l'icône Exporter le WSDL dans la ligne du service proxy dont vous souhaitez exporter le WSDL. Une boîte de dialogue vous invite à ouvrir ou à enregistrer le fichier JAR exporté.
  3. Dans la boîte de dialogue, cliquez sur Open pour ouvrir le fichier ou sur Enregistrer pour l'enregistrer.
