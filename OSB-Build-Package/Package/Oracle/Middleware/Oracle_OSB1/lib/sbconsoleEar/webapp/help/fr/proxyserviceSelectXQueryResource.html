Liaison de ressources XQuery externes et d'expressions XQuery imbriquées

Vous pouvez lier des ressources XQuery à des expressions XQuery imbriquées de sorte qu'elles soient exécutées en ligne en tant qu'élément d'une action.

Avant de commencer

Les instructions suivantes supposent que soyez en train de créer ou modifier une expression via l'éditeur d'expressions XQuery, l'éditeur de conditions XQuery ou l'éditeur d'expressions XPath, comme indiqué dans Création et modification d'expressions XPath et XQuery imbriquées.

Pour lier une ressource XQuery à une expression imbriquée, procédez comme suit :

  1. Cliquez sur Structures de variable.
  2. Dans l'espace de travail (en dessous de la barre de boutons), sélectionnez Ressources XQuery.
  3. Dans la zone 1. Sélectionner la ressource XQuery à exécuter, cliquez sur Parcourir.
  4. Dans le navigateur XQuery, sélectionnez le bouton d'option associé à la ressource XQuery que vous souhaitez utiliser, puis cliquez sur Soumettre.
  5. Dans la zone 2. Variables attachées (bind variable), définissez les paramètres d'entrée pour la transformation. Pour chaque variable listée sous Nom de la variable, entrez une expression XQuery à mettre en correspondance avec elle. Vous devez définir une correspondance pour chaque paramètre. Par exemple, si une transformation XQuery a deux paramètres d'entrée nommés un et deux, le champ Nom de la variable a deux libellés : un et deux. Une zone de texte est associée à chaque libellé ; elle permet de saisir l'expression XQuery.

    6. Les expressions XQuery suivantes sont des exemples d'entrées valides pour ce champ :

    $body/*[1]

    $body/po:PurchaseOrder

    Remarque : Le nom de variable suivant n'est pas une entrée valide pour ce champ, ce qui entraîne une exception :
    body.
  6. Une fois que vous avez fini, poursuivez avec l'une des tâches suivantes :
    • Cliquez sur Valider. Un message apparaît si la validation de l'expression a réussi.
    • Cliquez sur Tester. Reportez-vous à Exécution du test XQuery.
    • Enregistrez ou annulez vos modifications.
