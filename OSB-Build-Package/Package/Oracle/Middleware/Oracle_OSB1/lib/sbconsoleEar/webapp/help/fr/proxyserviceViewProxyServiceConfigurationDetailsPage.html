Page Visualiser un service proxy : Détails de configuration

La page Visualiser un service proxy : Détails de configuration affiche les détails de configuration d'un service proxy. Le tableau 19-19 décrit toutes les propriétés qui peuvent figurer·sur cette page. (Ces propriétés varient selon les détails du service proxy.)

Les catégories répertoriées sur cette page correspondent aux pages Créer/Modifier un service proxy utilisées pour créer et modifier les configurations des services proxy, comme décrit dans Créer/Modifier un service fonctionnel - Référence de page.

Cliquez sur le lien Modifier en regard de n'importe quel nom de catégorie pour afficher la page de configuration associée.

Tableau 19-19·Visualiser un service proxy : page Détails de configuration

Description
Dernière modification par
L'utilisateur ayant créé ce service proxy·ou l'ayant importé dans la configuration.
