Modification des gestionnaires d'erreur

Avant de commencer

Les instructions suivantes supposent que vous soyez déjà en train de modifier un flux de messages sur la page Modifier le flux de messages, comme indiqué dans Création, modification et visualisation de flux de messages.

Pour visualiser et modifier un gestionnaire d'erreur

  1. Effectuez l'une des opérations suivantes.

    2. Tableau 23-1 Visualisation et modification du gestionnaire d'erreur 
    Opération
    Procédure
    Visualiser et modifier le gestionnaire d'erreur du flux de messages du service proxy
    Cliquez sur l'icône Service proxy, puis cliquez sur Modifier le gestionnaire d'erreur de service. La page Modifier le gestionnaire d'erreur apparaît. Reportez-vous à Ajout de gestionnaires d'erreur de service proxy.
    Visualiser et modifier le gestionnaire d'erreur de pipeline
    Cliquez sur l'icône Pipeline des demandes ou Pipeline des réponses, puis sur Modifier le gestionnaire d'erreur de pipeline. La page Modifier le gestionnaire d'erreur apparaît. Reportez-vous à Ajout de gestionnaires d'erreur de pipeline.
    Visualiser et modifier le gestionnaire d'erreur de noeud de routage
    Cliquez sur l'icône Noeud de routage appropriée, puis sur Modifier le gestionnaire d'erreur de routage. La page Modifier le gestionnaire d'erreur apparaît. Reportez-vous à Ajout de gestionnaires d'erreur de noeud de routage.
    Visualiser et modifier le gestionnaire d'erreur de phase
    Cliquez sur l'icône Phase appropriée, puis sur Modifier le gestionnaire d'erreur de phase. La page Modifier le gestionnaire d'erreur apparaît. Reportez-vous à Ajout de gestionnaires d'erreur de phase.

Après avoir fini

Sur la page Modifier le flux de messages, continuez à construire le flux de messages, comme indiqué dans Création, modification et visualisation de flux de messages.
