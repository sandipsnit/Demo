Modification des configurations des services proxy

Cette section décrit comment modifier les configurations des services proxy.

  1. Si vous ne l'avez pas déjà fait, cliquez sur Créer afin de créer une session ou cliquez sur Modifier pour accéder à une session existante. Reportez-vous à Utilisation du centre des modifications.
  2. Localisez le service proxy à visualiser ou à modifier. Reportez-vous à Localisation des services proxy.
  3. Cliquez sur le nom du service proxy. La page Visualiser un service proxy : Détails de configuration affiche les informations de configuration du service proxy sélectionné.
  4. Pour visualiser ou modifier les paramètres, utilisez l'une des deux méthodes suivantes :
    • Cliquez sur l'icône Modifier en regard du nom de la catégorie dont vous souhaitez visualiser ou modifier les propriétés (par exemple, Configuration générale, Configuration du transport, etc.). Les pages que vous pouvez modifier dépendent de celles que vous avez configurées lors de la création du service proxy.

      • Pour obtenir une liste de l'ensemble de ces pages, reportez-vous à Page Visualiser un service fonctionnel - Détails de configuration.

    • Cliquez sur Modifier en bas de la page pour afficher la page Créer/Modifier un service fonctionnel : Configuration générale, qui est la première d'une série de pages destinées à la configuration de ce service proxy.
  5. Poursuivez pour visualiser ou modifier, comme décrit dans Création et configuration de services proxy.
  6. Sur la page Modifier un service fonctionnel : Récapitulatif, cliquez sur Enregistrer pour valider les mises à jour de la session en cours.
  7. Pour terminer la session et déployer la configuration lors de l'exécution, cliquez sur Activer sous Centre des modifications.
