Services proxy : création et gestion

Les services proxy sont les définitions Oracle Service Bus de services implémentés localement sur WebLogic Server.

Pour plus d'informations, reportez-vous à Configuring Proxy Services and Business Services dans le manuel Oracle Fusion Middleware Developer's Guide for Oracle Service Bus, à l'adresse suivante : http://www.oracle.com/pls/as111170/lookup?id=OSBAG111.
