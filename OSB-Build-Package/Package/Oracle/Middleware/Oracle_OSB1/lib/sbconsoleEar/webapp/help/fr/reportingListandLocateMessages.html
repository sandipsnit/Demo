Visualisation des messages de rapport

Utilisez la page Récapitulatif des rapports sur les messages pour visualiser les informations relatives aux·messages de rapport présentées dans le tableau 27-2. La page affiche jusqu'à 100 messages.

  1. Pour limiter·le nombre d'éléments dans la liste ou localiser des messages en particulier, procédez comme suit :
    • Cliquez sur l'icône Open pour afficher les autres filtres de recherche. Si vous indiquez plusieurs critères de recherche, seuls les résultats correspondant à tous les critères seront renvoyés.
    • Pour définir des paramètres de filtre, exécutez une ou plusieurs des étapes suivantes, présentées dans le tableau 27-1.

      • Tableau 27-1 Paramètres de filtre de messages 
      Opération
      Exécutez les étapes suivantes...
      Filtrer par période
      Définissez une période :
      1. Sélectionnez le bouton d'option Messages de... A...
      2. Dans le champ Messages de, entrez une date et une heure au format MM/DD/YY HH:MM:SS AM|PM. Par exemple, vous pouvez entrer 10/10/07 12:45:00 AM.
      3. Dans le champ Messages jusqu'à, entrez une date et une heure au format MM/DD/YY HH:MM:SS AM|PM.
      Vous pouvez également sélectionner des paramètres pour obtenir les messages les plus récents :
      1. Sélectionnez le bouton d'option Pour les derniers.
      2. Dans le champ Pour les derniers, entrez le nombre de jours, puis indiquez les heures et les minutes.
      Remarque : Par défaut, les rapports de messages des 30 dernières minutes apparaissent.
      Filtrer par nom de service
      Dans le champ Nom du service entrant, entrez le nom du service.
      Filtrer par code d'erreur
      Dans le champ Code d'erreur, entrez un code d'erreur.
      Filtrer par index de rapport
      Dans le champ Index des rapports, entrez les index de rapport. Il s'agit des paires clé-valeur définies au moment de la configuration de l'action de rapport de ce service proxy. Il s'agit d'une recherche par chaîne.

    • La recherche par nom de service, code d'erreur ou champ d'index de rapport prend en charge les caractères génériques.
      • Lorsque vous entrez uniquement un astérisque (*), tous les messages sont renvoyés. Par exemple, si vous entrez * dans le champ de code d'erreur, les messages avec et sans code d'erreur apparaissent.
      • Dans le champ Code d'erreur uniquement, des limitations de recherche strictes sont appliquées. Par exemple, pour trouver des messages contenant le code d'erreur BEA12345, vous pouvez entrer *, BE* ou BEA12345.
  2. Cliquez sur Rechercher. La page affiche les messages qui correspondent aux paramètres du filtre que vous avez défini. Le nombre maximal de messages renvoyé est de 1 000.
  3. Cliquez sur Tout visualiser pour enlever les filtres de recherche et afficher tous les messages.
  4. Pour localiser un message, vous pouvez également procéder comme suit :
    • Retriez la liste des messages. Cliquez sur un nom de colonne souligné. Des flèches vers le haut et vers le bas indiquent l'ordre de tri. Cliquez sur le nom de la colonne pour modifier l'ordre de tri.
    • Faites défiler les pages. Utilisez les contrôles de page au-dessus ou au-dessous de la·table. Accédez à une page en sélectionnant son·numéro ou en utilisant les flèches pour accéder à la page suivante ou précédente, à la première ou à la dernière page.

      • Tableau 27-2 Détails du récapitulatif des messages (rapports)
      Propriété
      Description
      Index de rapport
      Lien vers la page Visualiser les détails des messages. Reportez-vous à Visualisation des détails des messages.
