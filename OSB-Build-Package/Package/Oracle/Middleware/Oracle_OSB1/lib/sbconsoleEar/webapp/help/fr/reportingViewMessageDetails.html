Visualisation des détails des messages

Cette page permet de visualiser les détails d'un message en particulier.

  1. Localisez le message. Reportez-vous à Visualisation des messages de rapport.
  2. Cliquez sur l'index de rapport. La page Visualiser les détails des messages affiche les informations de configuration générale présentées dans le tableau 27-3.

    3. Tableau 27-3 Paramètres de configuration des messages
    Propriété
    Description
    ID de message
    Identificateur unique du message.
    Horodatage de base de données
    Date et heure auxquelles le message a été inscrit·dans la base de données.
    Heure au point de journalisation
    Date et heure auxquelles le message a été journalisé.
    Nom du serveur
    Nom du serveur dans lequel le message a été généré.
    Etat
    Etat du pipeline dans lequel le message a été généré :
    REQUEST : indique que l'action de rapport est exécutée dans un pipeline de demande.
    RESPONSE : indique que l'action de rapport est exécutée dans un pipeline de réponse.
    ERROR : l'action est exécutée dans le gestionnaire d'erreur de niveau service.
    Nom du noeud
    Nom du noeud du pipeline dans lequel le message a été généré.
    Nom du pipeline
    Nom du pipeline dans lequel le message a été généré.
    Nom de la phase
    Nom de la phase dans laquelle le message a été généré.

    Cette page affiche les informations relatives au service entrant, présentées dans le tableau 27-4.

    Tableau 27-4 Détails du service entrant
    Propriété
    Description
    Nom
    Service proxy entrant associé au message. Il s'agit d'un lien vers la page Visualiser un service proxy : Détails de configuration. Reportez-vous à Modification des configurations des services proxy.
    URI
    URI associé au service proxy.
    Opération
    Opération entrante associée au message.

    Cette page affiche les informations relatives au service sortant, présentées dans le tableau 27-5.

    Tableau 27-5 Détails du service sortant
    Propriété
    Description
    Nom
    Service fonctionnel sortant associé au message. Il s'agit d'un lien vers la page Visualiser un service fonctionnel : Détails de configuration. Reportez-vous à Modification des configurations des services fonctionnels.
    URI
    URI de l'adresse du service fonctionnel sortant.
    Opération
    Nom de l'opération appelée sur le service sortant.

    Cette page affiche les informations relatives à l'index de rapport, présentées dans le tableau 27-6.

    Tableau 27-6 Détails de·l'index de rapport
    Propriété
    Description
    Texte de l'index des rapports
    Les index des rapports sont des paires clé-valeur que vous utilisez pour extraire des identificateurs à partir d'une variable de contexte de message. Les clés facilitent le filtrage des messages. Utilisez l'action de rapport pour ajouter des paires clé-valeur. Reportez-vous à Ajout et modification d'actions dans les flux de messages.

    Cette·page affiche les informations d'erreur présentées dans le tableau 27-7.

    Tableau 27-7 Détails des informations d'erreur
    Propriété
    Description
    Code d'erreur
    Code d'erreur associé au message, le cas échéant.
    Vous indiquez des codes d'erreur lorsque vous configurez l'action de génération d'erreur. Reportez-vous à "Handling Errors in Message Flows" dans le manuel Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus, à l'adresse http://www.oracle.com/pls/as111170/lookup?id=OSBAG232, et à Ajout de gestionnaires d'erreur de service proxy.
    Motif
    Motif du code d'erreur.
    Information
    Détails de l'erreur associés au code d'erreur. Lorsqu'ils existent, ces détails sont généralement constitués par une trace de pile désignant l'emplacement où une erreur spécifique est survenue. Ils peuvent être tronqués en raison d'une limite de taille dans la base de données. Cette·limite est de 2 048 caractères.

    Cette page affiche les informations relatives au·corps du rapport, présentées dans le tableau 27-8.

    Tableau 27-8 Détails du corps du rapport
    Propriété
    Description
    Texte du corps du rapport
    Ce lien ouvre un autre navigateur qui affiche les détails du corps du rapport. Dans l'action de rapport, vous utilisez une expression pour capturer le texte du corps du rapport. Reportez-vous à la procédure de l'action de rapport dans Ajout et modification d'actions dans les flux de messages.

  3. Cliquez sur OK lorsque vous avez terminé de consulter·les détails d'un message en particulier.
