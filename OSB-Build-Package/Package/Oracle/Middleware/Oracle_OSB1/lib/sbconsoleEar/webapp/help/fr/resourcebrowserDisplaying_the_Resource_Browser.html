Utilisation du navigateur Web de ressources

Le navigateur Web de ressources fournit des outils permettant de·localiser des ressources et d'effectuer des actions simples sur celles-ci (Afficher, Modifier, Exporter et Supprimer).

  1. Pour afficher le navigateur Web de ressources, sélectionnez Navigateur Web de ressources dans le panneau de navigation de gauche.
  2. Dans la partie gauche de la page, cliquez sur le nom du type de ressource à localiser. Le tableau 3-10 présente les éléments disponibles et les tâches associées à la page affichée pour chacun d'entre eux.
