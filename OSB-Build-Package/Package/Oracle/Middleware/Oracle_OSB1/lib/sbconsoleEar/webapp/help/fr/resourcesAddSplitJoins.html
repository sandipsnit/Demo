Ajout de ressources de Fractionnement-Fusion

Avant de commencer

Les ressources de Fractionnement-Fusion requièrent un fichier .flow généré lors de la création d'une ressource de ce type à l'aide des modules d'extension Oracle Service Bus pour Eclipse ou lors de l'import d'un fichier JAR de configuration de Fractionnement-Fusion dans le module d'extension. Le fichier .flow peut contenir des références à des services proxy, à des services fonctionnels ou à des ressources WSDL externes. Les ressources dont dépend la ressource de Fractionnement-Fusion doivent être présentes sur le serveur avant l'activation de cette dernière.

  1. Si vous ne l'avez pas déjà fait, cliquez sur Créer afin de créer une session ou cliquez sur Modifier pour accéder à une session existante. Reportez-vous à Utilisation du centre des modifications.
  2. Sélectionnez Explorateur de projets, puis choisissez le projet ou le dossier auquel ajouter la ressource de Fractionnement-Fusion. La page Vue du projet/Vue de dossier apparaît.
  3. Dans la liste déroulante Créer une ressource, sélectionnez Fractionnement-Fusion pour afficher la page Créer une ressource de Fractionnement-Fusion.

    4. Spécifiez les paramètres présentés dans le tableau 9-2.

    Tableau 9-2 Paramètres des ressources de Fractionnement-Fusion 
    Propriété
    Procédure
    Nom de la ressource
    Entrez un nom unique pour cette ressource de Fractionnement-Fusion.

    Pour obtenir des conseils sur la dénomination, suivez les restrictions concernant la dénomination des ressources.
    Description de la ressource
    Entrez la description de la ressource de Fractionnement-Fusion.
    Fractionnement-Fusion
    Cliquez sur Parcourir afin de localiser un fichier .flow sur votre système. Ces fichiers se trouvent généralement à l'emplacement MW_HOME\user_projects\workspaces\your_workspace.
    Stratégie de répartition
    Sélectionnez l'instance du gestionnaire de travaux WebLogic Server utilisé pour créer des threads (branchements parallèles) afin d'exécuter la ressource de Fractionnement-Fusion. Le gestionnaire de travaux par défaut est utilisé s'il n'en existe pas d'autre.

  4. Cliquez sur Enregistrer. La ressource de Fractionnement-Fusion est créée et enregistrée dans la session en cours.
    5. Remarque : Avant d'activer une ressource de Fractionnement-Fusion, les ressources dont elle dépend doivent être présentes.
  5. Pour terminer la session et déployer la configuration lors de l'exécution, cliquez sur Activer sous Centre des modifications.
