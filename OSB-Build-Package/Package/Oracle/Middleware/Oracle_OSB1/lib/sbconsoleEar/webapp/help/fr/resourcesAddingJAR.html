Ajout de ressources JAR

  1. Si vous ne l'avez pas déjà fait, cliquez sur Créer afin de créer une session ou cliquez sur Modifier pour accéder à une session existante. Reportez-vous à Utilisation du centre des modifications.
  2. Sélectionnez Explorateur de projets, puis choisissez le projet ou le dossier auquel ajouter le fichier JAR. La page Vue du projet/Vue de dossier apparaît.
  3. Dans la liste déroulante Créer une ressource, sélectionnez JAR sous Utilitaire pour afficher la page Créer une ressource JAR.
  4. Dans le champ Nom de la ressource, entrez ou modifiez un nom unique pour ce fichier JAR. Il s'agit d'un champ obligatoire.

    Pour obtenir des conseils sur la dénomination, suivez les restrictions concernant la dénomination des ressources.
  5. Dans le champ Description de la ressource, entrez la description du fichier JAR.
  6. Dans le champ Fichier source JAR, effectuez l'une des opérations suivantes :
    • Indiquez le chemin d'accès du fichier JAR.
    • Cliquez sur Parcourir pour localiser et importer un fichier JAR existant.

      • Ce champ est obligatoire.

  7. Cliquez sur Enregistrer. La ressource JAR est créée et enregistrée dans la session en cours.

    8. Si aucun problème ne survient pendant le chargement du fichier JAR, vous pourrez en visualiser le contenu. Pour savoir comment visualiser les détails d'un fichier JAR, reportez-vous à Modification de ressources JAR.

  8. Pour terminer la session et déployer la configuration lors de l'exécution, cliquez sur Activer sous Centre des modifications.
