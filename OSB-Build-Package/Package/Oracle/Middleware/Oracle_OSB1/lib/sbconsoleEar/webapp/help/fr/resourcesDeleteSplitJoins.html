Suppression de ressources de Fractionnement-Fusion

  1. Si vous ne l'avez pas déjà fait, cliquez sur Créer afin de créer une session ou cliquez sur Modifier pour accéder à une session existante. Reportez-vous à Utilisation du centre des modifications.
  2. Sélectionnez Navigateur Web de ressources > Opérations Fractionnement-Fusion pour afficher la page Récapitulatif des opérations Fractionnement-Fusion.
  3. Cliquez sur l'icône Supprimer dans le champ Options de la ressource de Fractionnement-Fusion à supprimer. Celle-ci est supprimée de la session en cours. Si cette ressource est référencée par d'autres, l'icône Avertissement relatif à la suppression apparaît. Vous pouvez supprimer la ressource à condition de confirmer l'opération lorsqu'un avertissement s'affiche. Toutefois, des conflits peuvent survenir, les références à la ressource supprimée n'étant pas résolues.
  4. Pour terminer la session et déployer la configuration lors de l'exécution, cliquez sur Activer sous Centre des modifications.
