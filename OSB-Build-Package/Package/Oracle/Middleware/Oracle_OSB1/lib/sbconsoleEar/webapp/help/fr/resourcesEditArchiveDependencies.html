Modification de dépendances JAR

Dans Oracle Service Bus, les ressources JAR peuvent contenir des classes dépendant d'autres classes dans différentes ressources JAR. La page Modifier les dépendances JAR permet d'ajouter ou d'enlever des ressources JAR référencées.

  1. Si ce n'est pas déjà fait, cliquez sur Créer pour créer une session ou sur Modifier pour indiquer une session existante. Reportez-vous à Utilisation du centre des modifications.
  2. Localisez la ressource JAR, comme décrit dans Localisation de fichiers JAR.
  3. Cliquez sur le nom de la ressource JAR.

    4. Sur la page Visualiser une ressource JAR - Configuration JAR, dans l'onglet Dépendances, un indicateur numérique affiche le nombre de ressources JAR référencées étant configurées pour cette ressource JAR, ou le chiffre zéro s'il n'y en a aucune.

  4. Sélectionnez l'onglet Dépendances pour afficher les ressources JAR référencées ou en ajouter.

    5. La page des dépendances de ressources JAR inclut les informations présentées dans le tableau 7-4.

    Tableau 7-4 Informations sur les dépendances JAR 
    Propriété
    Description
    Nom
    Nom de la ressource JAR référencée. Ce nom est un lien permettant d'accéder à la page Visualiser une ressource JAR - Configuration JAR. Reportez-vous à Modification de ressources JAR.
    Chemin
    Le chemin est constitué du nom de projet et du nom du dossier dans lequel la ressource JAR réside.

  5. Pour ajouter ou enlever des ressources JAR référencées, cliquez sur Modifier.

    6. La page Modifier les dépendances JAR inclut les informations présentées dans le tableau 7-5.

    Tableau 7-5 Détails des dépendances JAR
    Propriété
    Description
    Variable d'environnement CLASSPATH manifest
    Entrées de variable d'environnement CLASSPATH dans le fichier manifest JAR.
    Nom
    Nom et chemin de la ressource JAR référencée.
    Options
    • Utilisez les flèches vers le haut et vers le bas pour modifier l'ordre de priorité des ressources JAR référencées.
    • Cliquez sur l'icône Supprimer pour enlever une ressource JAR référencée.

  6. Cliquez sur Ajouter des ressources JAR.
  7. Dans le navigateur Web de ressources JAR, localisez ou sélectionnez les ressources JAR référencées et cliquez sur Soumettre.

    8. Par exemple, si vous modifiez la ressource JAR A et que vous ajoutez les ressources JAR B et C, la ressource JAR A référence la ressource JAR B ainsi que la ressource JAR C (autrement dit, elle en dépend).

    Utilisez les flèches de la colonne Options pour modifier l'ordre de la hiérarchie des ressources JAR référencées, ou cliquez sur l'icône Supprimer pour enlever ces dernières.

  8. Cliquez sur Enregistrer pour valider les mises à jour dans la session en cours.
  9. Pour terminer la session et déployer la configuration lors de l'exécution, cliquez sur Activer sous Centre des modifications.
