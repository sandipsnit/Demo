Modification de ressources de Fractionnement-Fusion

  1. Si ce n'est pas déjà fait, cliquez sur Créer pour créer une session ou sur Modifier pour indiquer une session existante. Reportez-vous à Utilisation du centre des modifications.
  2. Localisez la ressource de Fractionnement-Fusion, comme décrit dans Localisation d'opérations Fractionnement-Fusion.
  3. Cliquez sur le nom de la ressource de Fractionnement-Fusion.

    4. La page Visualiser une ressource de Fractionnement-Fusion - Configuration affiche les informations répertoriées dans le tableau 9-3.

    Tableau 9-3 Détails des ressources de Fractionnement-Fusion 
    Propriété
    Description
    Dernière modification par
    Utilisateur ayant créé la ressource de Fractionnement-Fusion ou l'ayant importée dans la configuration.
    Dernière modification le
    Date et heure auxquelles l'utilisateur a créé la ressource de Fractionnement-Fusion et l'a importée dans la configuration. Cliquez sur le lien de date et d'heure pour visualiser l'historique des modifications apportées à cette ressource. Reportez-vous à Page Visualiser l'historique des modifications.
    Références
    Nombre d'objets référencés par la·ressource de Fractionnement-Fusion. Lorsque des références existent, cliquez sur le lien numérique pour visualiser la liste des objets en question. Reportez-vous à Visualisation des références aux ressources.
    Référencé par
    Nombre d'objets référençant la·ressource de Fractionnement-Fusion. Lorsque des références existent, cliquez sur le lien numérique pour visualiser la liste des objets en question. Reportez-vous à Visualisation des références aux ressources.
    Description
    Description de la ressource de Fractionnement-Fusion, le cas échéant.

  4. Vous pouvez appliquer les paramètres opérationnels suivants à une ressource de Fractionnement-Fusion :

    5. Tableau 9-4 Paramètres opérationnels des ressources de Fractionnement-Fusion
    Propriété
    Description
    Journaux
    Pour capturer des informations de journalisation dans une ressource de Fractionnement-Fusion, sélectionnez Activer la journalisation, puis choisissez le niveau de journalisation à capturer.
    Fonction de trace d'exécution
    Pour activer la fonction de trace d'exécution, sélectionnez Activé.

  5. Cliquez sur Mettre à jour pour valider les mises à jour dans la session en cours.
  6. Pour terminer la session et déployer la configuration lors de l'exécution, cliquez sur Activer sous Centre des modifications.
