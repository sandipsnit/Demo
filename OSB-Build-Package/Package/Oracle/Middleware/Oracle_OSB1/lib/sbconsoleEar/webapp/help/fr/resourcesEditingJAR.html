Modification de ressources JAR

La page Visualiser une ressource JAR - Configuration permet de mettre à jour les ressources JAR. Dans la mesure où vous ne pouvez pas modifier une ressource JAR à l'aide d'Oracle Service Bus, ce type de mise à jour consiste principalement à pointer vers une nouvelle version de la ressource.

  1. Si ce n'est pas déjà fait, cliquez sur Créer pour créer une session ou sur Modifier pour indiquer une session existante. Reportez-vous à Utilisation du centre des modifications.
  2. Localisez la ressource JAR, comme décrit dans Localisation de fichiers JAR.
  3. Cliquez sur le nom de la ressource JAR. La page Visualiser une ressource JAR - Configuration JAR affiche les informations répertoriées dans le tableau 7-2.

    4. Tableau 7-2 Informations sur les ressources JAR 
    Propriété
    Description
    Dernière modification par
    Utilisateur ayant importé la ressource JAR dans la configuration.
    Dernière modification le
    Date et heure auxquelles l'utilisateur a importé la ressource JAR dans la configuration. Cliquez sur le lien de date et d'heure pour visualiser l'historique des modifications apportées à cette ressource. Reportez-vous à Page Visualiser l'historique des modifications.
    Références
    Nombre de ressources JAR référencées par la ressource JAR. Lorsque des références existent, cliquez sur le lien numérique pour visualiser la liste des ressources JAR en question. Reportez-vous à Visualisation des références aux ressources.
    Référencé par
    Nombre d'objets référençant la ressource JAR. Lorsque des références existent, cliquez sur le lien numérique pour visualiser la liste des objets en question. Reportez-vous à Visualisation des références aux ressources.
    Description
    Description de la ressource JAR, le cas échéant.

    La configuration de la ressource JAR est affichée par défaut. Elle comprend les informations répertoriées dans le tableau 7-3.

    Tableau 7-3 Détails de la configuration des ressources JAR
    Propriété
    Description
    Configuration JAR
    Liste des classes et des interfaces de la ressource JAR.
    Cliquez sur le signe (+) en regard du nom d'une classe Java pour afficher la liste des méthodes incluses dans cette classe.

  4. Pour modifier les champs, cliquez sur Mettre à jour.

    5. Le champ Description est le seul que vous pouvez modifier. Vous ne pouvez pas modifier une ressource JAR dans Oracle Service Bus. Pour mettre à jour la ressource JAR, vous devez sélectionner une nouvelle ressource JAR à utiliser. Pour connaître la procédure, reportez-vous à Ajout de ressources JAR.

  5. Cliquez sur Enregistrer pour valider les mises à jour dans la session en cours.
  6. Pour terminer la session et déployer la configuration lors de l'exécution, cliquez sur Activer sous Centre des modifications.
Remarque : Si vous modifiez la ressource JAR utilisée par un service fonctionnel EJB (en pointant vers une nouvelle version de la ressource JAR), vous devez redéployer l'EJB. Vous devez ensuite modifier tous les services EJB utilisant cette ressource JAR, puis sélectionner à nouveau cette dernière, l'enregistrer et l'activer. Le service fonctionnel EJB est à nouveau packagé pour pouvoir utiliser la nouvelle ressource JAR.
Remarque : En revanche, les actions d'appel Java et les services Tuxedo sélectionnent automatiquement la nouvelle ressource JAR.
