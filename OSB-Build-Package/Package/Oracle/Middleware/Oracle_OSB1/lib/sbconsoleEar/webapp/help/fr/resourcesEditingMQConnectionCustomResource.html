Modification de connexions MQ

Cette section explique comment modifier des ressources de connexion MQ existantes.

  1. Si ce n'est pas déjà fait, cliquez sur Créer pour créer une session ou sur Modifier pour indiquer une session existante. Reportez-vous à Utilisation du centre des modifications.
  2. Localisez la ressource de connexion MQ, comme décrit dans Localisation des connexions MQ.
  3. Cliquez sur le nom de la ressource de connexion MQ.

    4. La page Visualiser la connexion MQ : Configuration affiche les informations répertoriées dans le tableau 8-3.

    Tableau 8-3 Détails des connexions MQ 
    Propriété
    Description
    Dernière modification par
    Utilisateur ayant créé la ressource de connexion MQ ou l'ayant importée dans la configuration.
    Dernière modification le
    Date et heure auxquelles l'utilisateur a créé la ressource de connexion MQ ou l'a importée dans la configuration. Cliquez sur le lien de date et d'heure pour visualiser l'historique des modifications apportées à cette ressource. Reportez-vous à Page Visualiser l'historique des modifications.
    Références
    Nombre d'objets référencés par la·ressource de connexion MQ. Lorsque des références existent, cliquez sur le lien numérique pour visualiser la liste des objets en question. Reportez-vous à Visualisation des références aux ressources.
    Référencé par
    Nombre d'objets référençant la·ressource de connexion MQ. Lorsque des références existent, cliquez sur le lien numérique pour visualiser la liste des objets en question. Reportez-vous à Visualisation des références aux ressources.
    Description
    Description de la ressource de connexion MQ, le cas échéant.

  4. Pour modifier les champs, cliquez sur Modifier. Pour obtenir la description de ces champs, reportez-vous à Ajout de connexions MQ.

    5. Vous ne pouvez pas modifier le champ Nom de la ressource.

  5. Cliquez sur Enregistrer pour valider les mises à jour dans la session en cours.
  6. Pour terminer la session et déployer la configuration lors de l'exécution, cliquez sur Activer sous Centre des modifications.
