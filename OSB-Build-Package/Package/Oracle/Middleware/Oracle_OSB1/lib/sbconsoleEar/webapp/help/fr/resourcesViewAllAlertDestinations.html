Localisation de destinations d'alertes

Vous pouvez visualiser la liste des destinations d'alertes sur la page Récapitulatif des destinations d'alertes. Si vous n'avez pas encore créé de destination d'alertes et que vous voulez le faire, créez une nouvelle ressource de destination d'alertes dans l'explorateur de projets. Pour connaître la procédure, reportez-vous à Ajout de destinations d'alertes.

  1. Effectuez l'une des opérations suivantes :
    • Sélectionnez Explorateur de projets pour afficher la page Vue des projets ou la page Vue du projet/Vue de dossier. Parcourez ensuite les projets et dossiers afin d'accéder à la destination d'alertes.
    • Sélectionnez Navigateur Web de ressources > Destinations d'alertes. La page Récapitulatif des destinations d'alertes affiche les informations répertoriées dans le tableau 5-1.
  2. Pour limiter le nombre d'éléments dans la liste, vous pouvez filtrer par nom, chemin, ou les deux. Dans les champs Nom et Chemin, saisissez le nom et le chemin des cibles de recherche, puis cliquez sur Rechercher.

    3. Le chemin est constitué du nom de projet et du nom du dossier dans lequel la destination d'alertes réside.

    Cliquez sur Tout visualiser pour enlever les filtres de recherche et afficher toutes les destinations d'alertes.

    Tableau 5-1 Détails des destinations d'alertes 
    Propriété
    Description
    Nom
    Nom de la ressource de destination d'alertes. Ce nom est un lien permettant d'accéder à la page Visualiser une destination d'alertes : Configuration. Reportez-vous à Modification de destinations d'alertes.
    Chemin
    Le chemin est constitué du nom de projet et du nom du dossier dans lequel la destination d'alertes réside. Il s'agit d'un lien permettant d'accéder au projet ou au dossier contenant cette ressource. Reportez-vous à Qualification des noms de ressource à l'aide de projets et de dossiers.
    Options
    Cliquez sur l'icône Supprimer pour supprimer une destination d'alertes spécifique. Reportez-vous à Suppression de destinations d'alertes.
