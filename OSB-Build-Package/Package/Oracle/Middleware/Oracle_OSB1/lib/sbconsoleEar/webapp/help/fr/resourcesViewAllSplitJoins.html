Localisation d'opérations Fractionnement-Fusion

  1. Effectuez l'une des opérations suivantes :
    • Sélectionnez Explorateur de projets pour afficher la page Vue des projets ou la page Vue du projet/Vue de dossier. Parcourez ensuite les projets et dossiers afin d'accéder à la ressource de Fractionnement-Fusion.
    • Sélectionnez Navigateur Web de ressources > Opérations Fractionnement-Fusion. La page Récapitulatif des opérations Fractionnement-Fusion affiche les informations répertoriées dans le tableau 9-1. Pour obtenir une description plus détaillée des propriétés, reportez-vous à Modification de ressources de Fractionnement-Fusion.
  2. Pour limiter le nombre d'éléments dans la liste, vous pouvez filtrer par nom, chemin, ou les deux. Dans les champs Nom et Chemin, saisissez le nom et le chemin des cibles de recherche, puis cliquez sur Rechercher.
  3. Cliquez sur Tout visualiser pour enlever les filtres de recherche et afficher toutes les ressources de Fractionnement-Fusion.

    4. Tableau 9-1 Détails des ressources de Fractionnement-Fusion 
    Propriété
    Description
    Nom de la ressource de Fractionnement-Fusion
    Nom affecté à la ressource de Fractionnement-Fusion. Ce nom est un lien permettant d'accéder à la page Configuration d'une ressource de Fractionnement-Fusion. Reportez-vous à Modification de ressources de Fractionnement-Fusion.
    Chemin
    Le chemin est constitué du nom de projet et du nom du dossier dans lequel la ressource de Fractionnement-Fusion réside. Il s'agit d'un lien permettant d'accéder au projet ou au dossier contenant cette ressource. Reportez-vous à Qualification des noms de ressource à l'aide de projets et de dossiers.
    Options
    Cliquez sur l'icône Supprimer pour supprimer une ressource de Fractionnement-Fusion spécifique. Reportez-vous à Suppression de ressources de Fractionnement-Fusion.
