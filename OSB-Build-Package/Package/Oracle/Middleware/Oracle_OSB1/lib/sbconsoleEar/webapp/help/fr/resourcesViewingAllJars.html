Localisation de fichiers JAR

  1. Effectuez l'une des opérations suivantes :
    • Sélectionnez Explorateur de projets pour afficher la page Vue des projets ou la page Vue du projet/Vue de dossier. Parcourez ensuite les projets et dossiers afin d'accéder à la ressource JAR.
    • Sélectionnez Navigateur Web de ressources > JAR. La page Récapitulatif des ressources JAR affiche les informations répertoriées dans le tableau 7-1. Pour obtenir une description plus détaillée des propriétés, reportez-vous à Modification de ressources JAR.
  2. Pour limiter le nombre d'éléments dans la liste ou localiser une ressource JAR spécifique, recherchez une ressource JAR à l'aide des filtres de nom et de chemin. Dans les champs Nom et Chemin, saisissez le nom et le chemin des cibles de recherche, puis cliquez sur Rechercher.

    3. Le chemin est constitué du nom de projet et du nom du dossier dans lequel la ressource JAR réside.

    Cette recherche prend en charge les caractères génériques. Vous pouvez ainsi entrer "j*" dans le champ Nom pour obtenir la liste des ressources JAR dont le nom commence par "j".

    Cliquez sur Tout visualiser pour enlever les filtres de recherche et afficher toutes les ressources JAR.

    Tableau 7-1 Détails des ressources JAR
    Propriété
    Description
    Nom JAR
    Nom unique affecté à la ressource JAR. Ce nom est un lien permettant d'accéder à la page Visualiser une ressource JAR - Configuration JAR. Reportez-vous à Modification de ressources JAR.
    Chemin
    Le chemin est constitué du nom de projet et du nom du dossier dans lequel la ressource JAR réside. Il s'agit d'un lien permettant d'accéder au projet ou au dossier contenant cette ressource. Reportez-vous à Qualification des noms de ressource à l'aide de projets et de dossiers.
    Nb de classes
    Nombre de classes que contient la ressource JAR.
    Taille
    Taille de la ressource JAR en octets.
    Options
    Cliquez sur l'icône Supprimer pour supprimer une ressource JAR spécifique. Reportez-vous à Suppression de ressources JAR.
