Localisation des connexions MQ

  1. Effectuez l'une des opérations suivantes :
    • Sélectionnez Explorateur de projets pour afficher la page Vue des projets ou la page Vue du projet/Vue de dossier. Parcourez ensuite les projets et dossiers afin d'accéder à la ressource de connexion MQ.
    • Sélectionnez Navigateur Web de ressources > connexions MQ. La page Récapitulatif des connexions MQ affiche les informations répertoriées dans le tableau 8-1. Pour obtenir une description plus détaillée des propriétés, reportez-vous à Modification de connexions MQ.
  2. Pour limiter le nombre d'éléments dans la liste, vous pouvez filtrer par nom, chemin, ou les deux. Dans les champs Nom et Chemin, saisissez le nom et le chemin des cibles de recherche, puis cliquez sur Rechercher.
  3. Cliquez sur l'icône Open pour afficher les autres filtres de recherche. Si vous indiquez plusieurs critères de recherche, seuls les résultats correspondant à tous les critères seront renvoyés.
    • Filtrez par nom de gestionnaire de files d'attente. Entrez la chaîne de votre choix, qui peut comprendre des caractères génériques, puis cliquez sur Rechercher.
    • Filtrez par nom de canal du gestionnaire de files d'attente MQ. Entrez la chaîne de votre choix, qui peut comprendre des caractères génériques, puis cliquez sur Rechercher.
    • Cliquez sur Tout visualiser pour enlever les filtres de recherche et afficher toutes les ressources de connexion MQ.

      • Tableau 8-1 Détails des connexions MQ 
      Propriété
      Description
      Nom de la connexion MQ
      Nom affecté à la ressource de connexion MQ. Ce nom est un lien permettant d'accéder à la page Configuration de la connexion MQ. Reportez-vous à Modification de connexions MQ.
      Chemin
      Le chemin est constitué du nom de projet et du nom du dossier dans lequel la ressource de connexion MQ réside. Il s'agit d'un lien permettant d'accéder au projet ou au dossier contenant cette ressource. Reportez-vous à Qualification des noms de ressource à l'aide de projets et de dossiers.
      Options
      Cliquez sur l'icône Supprimer pour supprimer une ressource de connexion MQ spécifique. Reportez-vous à Suppression de connexions MQ.
