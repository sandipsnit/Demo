Schémas XML

Les schémas décrivent les types des données primitives ou structurées. Les schémas XML correspondent à un vocabulaire XML qui décrit les règles devant être suivies par les business data XML. Ils spécifient la structure des documents et le type de données de chaque élément et attribut contenus dans le document.

Les schémas XML permettent de référencer les ressources WSDL. Vous pouvez également utiliser les schémas XML dans le flux de messages des services proxy pour valider un élément indiqué à l'aide d'une expression XPath. Reportez-vous à Ajout d'actions de validation.
