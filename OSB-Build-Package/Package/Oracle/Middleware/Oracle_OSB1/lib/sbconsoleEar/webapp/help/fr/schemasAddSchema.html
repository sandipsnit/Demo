Ajout de schémas XML

  1. Si vous ne l'avez pas déjà fait, cliquez sur Créer afin de créer une session ou cliquez sur Modifier pour accéder à une session existante. Reportez-vous à Utilisation du centre des modifications.
  2. Sélectionnez Explorateur de projets, puis choisissez le projet ou le dossier auquel ajouter le nouveau schéma XML. La page Vue du projet/Vue de dossier apparaît.
  3. Dans la liste déroulante Créer une ressource, sélectionnez Schéma XML pour afficher la page Créer une ressource de schéma XML.
  4. Dans le champ Nom de la ressource, entrez un nom unique pour ce schéma XML. Ce champ est obligatoire.

    Pour obtenir des conseils sur la dénomination, suivez les restrictions concernant la dénomination des ressources.
  5. Dans le champ Description de la ressource, entrez la description du schéma XML.
  6. Dans le champ Schéma XML, effectuez l'une des opérations suivantes :
    • Saisissez le texte du nouveau schéma XML.
    • Cliquez sur Parcourir pour localiser et importer un schéma XML existant.
    • Copiez et collez le texte d'un schéma XML existant dans ce champ.

      • Ce champ est obligatoire.

  7. Cliquez sur Enregistrer. Le nouveau schéma XML est enregistré dans la session en cours.

    8. Lorsque vous cliquez sur Enregistrer, le système affiche les références non résolues du nouveau schéma XML, le cas échéant. Reportez-vous à Visualisation des références de schéma XML non résolues.

  8. Pour terminer la session et déployer la configuration lors de l'exécution, cliquez sur Activer sous Centre des modifications.
