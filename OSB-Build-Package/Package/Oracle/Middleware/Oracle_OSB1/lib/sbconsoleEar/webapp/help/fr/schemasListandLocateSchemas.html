Localisation de schémas XML

  1. Effectuez l'une des opérations suivantes :
    • Sélectionnez Explorateur de projets pour afficher la page Vue des projets ou la page Vue du projet/Vue de dossier. Parcourez ensuite les projets et dossiers afin d'accéder au schéma XML.
    • Sélectionnez Navigateur Web de ressources > Schémas XML. La page Récapitulatif des schémas XML affiche les informations répertoriées dans le tableau 4-1. Pour obtenir une description plus détaillée des propriétés, reportez-vous à Modification de schémas XML.
  2. Pour localiser un schéma XML spécifique, procédez comme suit :
    • Cliquez sur l'icône Open pour afficher les autres filtres de recherche. Si vous indiquez plusieurs critères de recherche, seuls les résultats correspondant à tous les critères seront renvoyés.
    • Pour limiter le nombre d'éléments dans la liste, vous pouvez filtrer par nom, chemin et espace de noms. Dans les champs Nom, Chemin et Espace de noms, entrez le nom, le chemin et l'espace de noms des cibles de recherche, puis cliquez sur Rechercher.

      • Le chemin est constitué du nom de projet et du nom du dossier dans lequel le schéma XML réside.

    • Cliquez sur Tout visualiser pour enlever les filtres de recherche et afficher tous les schémas XML.

      • Tableau 4-1 Informations sur les schémas XML 
      Propriété
      Description
      Nom du schéma XML
      Nom unique affecté au schéma XML. Ce nom est un lien permettant d'accéder à la page Détails du schéma XML. Reportez-vous à Modification de schémas XML.
      Chemin
      Le chemin est constitué du nom de projet et du nom du dossier dans lequel le schéma XML réside. Il s'agit d'un lien permettant d'accéder au projet ou au dossier contenant cette ressource. Reportez-vous à Qualification des noms de ressource à l'aide de projets et de dossiers.
      Espace de noms du schéma XML
      Espace de noms utilisé pour qualifier les définitions incluses dans le schéma XML.
      Options
      Cliquez sur l'icône Supprimer pour supprimer un schéma XML spécifique. Reportez-vous à Suppression de schémas XML.
