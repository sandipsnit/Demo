Résolution des références de schéma XML non résolues

La page Modifier les références d'une ressource de schéma XML permet de résoudre les références de schéma XML non résolues via la configuration de la mise en correspondance des références de schéma XML, telles que les imports de schéma XML.

  1. Si vous ne l'avez pas déjà fait, cliquez sur Créer afin de créer une session ou cliquez sur Modifier pour accéder à une session existante. Reportez-vous à Utilisation du centre des modifications.
  2. Sur la page Récapitulatif des schémas XML, dans la colonne Nom du schéma XML, cliquez sur le nom de schéma XML approprié afin de visualiser les détails de la référence non résolue. La page Visualiser les détails du schéma XML apparaît.
  3. Cliquez sur Modifier les références pour afficher la page Modifier les références d'une ressource de schéma XML.
  4. Dans le champ Type de ressource, sélectionnez Schéma XML.
  5. Cliquez sur Parcourir. Le navigateur Web de schémas XML apparaît.
  6. Dans le navigateur Web de schémas XML, sélectionnez un schéma XML, puis choisissez une définition dans le panneau Définitions.
  7. Cliquez sur Soumettre. Le schéma XML que vous avez sélectionné apparaît dans le champ Nom de la ressource.
  8. Cliquez sur Enregistrer pour résoudre la référence.
  9. Sur la page Détails du schéma XML, cliquez sur OK. L'état du schéma XML est défini sur Valide.
