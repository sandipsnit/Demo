Visualisation des références de schéma XML non résolues

Pour chaque référence de schéma XML non résolue, cette page affiche les informations répertoriées dans le tableau 4-5.

Tableau 4-5 Détails des schémas XML non résolus 
Propriété
Description
Nom de schéma
Nom unique affecté au schéma XML. Cliquez sur le nom d'un schéma XML spécifique pour visualiser les détails le concernant. Reportez-vous à Modification de schémas XML.
Espace de noms
Espace de noms utilisé pour qualifier les définitions incluses dans le schéma XML.

Pour localiser une référence spécifique, procédez comme suit :

  • Re-triez la liste. Cliquez sur un nom de colonne souligné. Les flèches vers le haut et vers le bas indiquent l'ordre de tri. Cliquez sur le nom de la colonne pour modifier l'ordre de tri.
  • Faites défiler les pages. Utilisez les contrôles de page au-dessus ou au-dessous de la·table. Accédez à une page en sélectionnant son·numéro ou en utilisant les flèches pour accéder à la page suivante ou précédente, à la première ou à la dernière page.
