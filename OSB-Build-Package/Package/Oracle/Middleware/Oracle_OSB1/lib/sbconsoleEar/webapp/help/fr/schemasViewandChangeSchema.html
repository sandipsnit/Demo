Modification de schémas XML

  1. Si vous ne l'avez pas déjà fait, cliquez sur Créer afin de créer une session ou cliquez sur Modifier pour accéder à une session existante. Reportez-vous à Utilisation du centre des modifications.
  2. Localisez le schéma XML, comme décrit dans Localisation de schémas XML.
  3. Cliquez sur le nom du schéma XML. La page Visualiser les détails du schéma XML affiche les informations répertoriées dans les tableaux 4-2, 4-3 et 4-4.

    4. Tableau 4-2 Détails des schémas XML 
    Propriété
    Description
    Dernière modification par
    Utilisateur ayant créé le schéma XML ou l'ayant importé dans la configuration.
    Dernière modification le
    Date et heure auxquelles l'utilisateur a créé le schéma XML ou l'a importé dans la configuration. Cliquez sur le lien de date et d'heure pour visualiser l'historique des modifications apportées à cette ressource. Reportez-vous à Page Visualiser l'historique des modifications.
    Références
    Nombre d'objets référencés par le schéma XML. Lorsque des références existent, cliquez sur le lien numérique pour visualiser la liste des objets en question. Reportez-vous à Visualisation des références aux ressources.
    Référencé par
    Nombre d'objets référençant le schéma XML. Lorsque des références existent, cliquez sur le lien numérique pour visualiser la liste des objets en question. Par exemple, si ce schéma XML est référencé par un WSDL spécifique, ce dernier est répertorié comme référence lorsque vous cliquez sur ce lien. Reportez-vous à Visualisation des références aux ressources.
    Description
    Description du schéma XML, le cas échéant.

    La vue structurelle du schéma XML est affichée par défaut. Elle comprend les informations répertoriées dans le tableau 4-3.

    Tableau 4-3 Détails de la structure XML 
    Propriété
    Description
    Espace de noms cible
    Espace de noms utilisé pour qualifier les définitions incluses dans le schéma XML.
    Définitions de schéma XML
    Les attributs et les groupes associés au schéma XML sont les suivants :
    • Types de schéma XML
    • Eléments de schéma XML
    • Imports du schéma XML
    • Inclusions/redéfinitions du schéma XML
    Etat du schéma
    Statut du schéma XML :
    • Valide
      Le schéma XML est valide si tous les emplacements des schémas XML inclus dans le schéma XML en cours sont indiqués. En outre, tous les emplacements des schémas XML imbriqués (schémas XML inclus par la suite dans les schémas XML) doivent être spécifiés.
    • Non valide
      Le schéma XML n'est pas valide si un des emplacements du schéma XML n'est pas indiqué. En outre, il se peut que l'emplacement d'un schéma XML imbriqué (schéma XML inclus par la suite dans les schémas XML) ne soit pas spécifié. Pour résoudre un schéma XML non valide, cliquez sur Modifier les références. Reportez-vous à Résolution des références de schéma XML non résolues.

  4. Cliquez sur Vue textuelle pour afficher une vue textuelle des détails du schéma XML. La vue textuelle comprend les informations répertoriées dans le tableau 4-4.

    5. Tableau 4-4 Détails textuels des schémas XML
    Propriété
    Description
    Espace de noms cible
    Espace de noms utilisé pour qualifier les définitions incluses dans le schéma XML.
    Schéma XML
    Texte du schéma XML.

  5. Pour modifier les champs, cliquez sur Modifier. Pour obtenir la description de ces champs, reportez-vous à Ajout de schémas XML.
  6. Effectuez les modifications souhaitées.
  7. Cliquez sur Enregistrer pour valider les mises à jour dans la session en cours.
  8. Pour terminer la session et déployer la configuration lors de l'exécution, cliquez sur Activer sous Centre des modifications.
