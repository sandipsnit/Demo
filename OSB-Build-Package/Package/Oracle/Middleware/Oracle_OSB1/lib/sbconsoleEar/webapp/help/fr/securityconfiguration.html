Configuration de la sécurité

Le module Configuration de la sécurité permet de créer et de modifier les données de sécurité utilisées dans la sécurité entrante et la sécurité d'administration d'Oracle Service Bus.

La sécurité entrante au niveau du transport et du message utilise les données de l'utilisateur, du groupe et du rôle pour authentifier les demandes client entrantes. Elle applique des stratégies de contrôle d'accès pour déterminer les utilisateurs authentifiés étant autorisés à utiliser les services proxy et les services fonctionnels.

La sécurité d'administration utilise les données de l'utilisateur, du groupe et du rôle pour déterminer les utilisateurs authentifiés étant autorisés à créer ou modifier les données de configuration Oracle Service Bus, ou à surveiller les performances. Pour plus d'informations, reportez-vous à Présentation des utilisateurs, des groupes, des rôles de sécurité et des stratégies. Reportez-vous également à "Configuring Administrative Security" dans le manuel Oracle Fusion Middleware Developer's Guide for Oracle Service Bus à l'adresse suivante : http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539.

Remarque : vous ne pouvez pas exporter d'utilisateurs, de groupes ou de rôles lorsque vous exportez une configuration car ces objets se trouvent dans des emplacements de stockage de fournisseur de sécurité. Vous devez recréer ces objets lors de l'import de la configuration exportée ou utiliser, s'ils sont disponibles, les outils de WebLogic Server pour les exporter et les importer.
