Ajout de groupes

  1. Connectez-vous à la console d'administration d'Oracle Service Bus en tant qu'utilisateur disposant des privilèges d'administration de WebLogic Server. Seuls les utilisateurs disposant du rôle d'administration peuvent modifier les données de configuration de sécurité. Reportez-vous à "Configuring Administrative Security" dans le manuel Oracle Fusion Middleware Developer's Guide for Oracle Service Bus à l'adresse suivante : http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539.
  2. Sélectionnez Configuration de la sécurité > Groupes pour afficher la page Récapitulatif des groupes.
  3. Cliquez sur Ajouter une valeur.

    4. Vous pouvez ajouter un groupe que vous soyez ou non connecté à une session.

  4. Dans le champ Nom du groupe, entrez un nom unique. Vous ne pouvez pas saisir d'espaces ou de caractères spéciaux. Ce champ est obligatoire.
  5. Dans le champ Fournisseur d'authentification, sélectionnez le fournisseur d'authentification.
  6. Dans le champ Appartenance à des groupes, sélectionnez le groupe auquel ce groupe appartient.
    1. Sélectionnez un groupe dans le champ Groupes disponibles.
    2. Cliquez sur la flèche pour déplacer le groupe dans le champ Groupes en cours.
  7. Cliquez sur Enregistrer pour créer le groupe.

    8. La console d'administration d'Oracle Service Bus enregistre le groupe, qui devient immédiatement disponible dans toutes les sessions. Si vous supprimez la session dans laquelle vous vous trouviez pendant l'ajout du groupe, la console d'administration d'Oracle Service Bus ne supprime pas le nouveau groupe.
