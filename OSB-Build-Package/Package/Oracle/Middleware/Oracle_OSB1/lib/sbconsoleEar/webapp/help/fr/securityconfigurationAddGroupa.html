Modification de groupes

La page Visualiser les détails de groupe permet de visualiser et de modifier les détails d'un groupe spécifique.

  1. Connectez-vous à la console d'administration d'Oracle Service Bus en tant qu'utilisateur disposant des privilèges d'administration de WebLogic Server. Seuls les utilisateurs disposant du rôle d'administration peuvent modifier les données de configuration de sécurité. Reportez-vous à "Configuring Administrative Security" dans le manuel Oracle Fusion Middleware Developer's Guide for Oracle Service Bus à l'adresse suivante : http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539.
  2. Localisez le groupe. Reportez-vous à Localisation de groupes.
  3. Cliquez sur le nom du groupe. La page Visualiser les détails de groupe affiche les informations répertoriées dans le tableau 24-4.

    4. Tableau 24-4 Détails des groupes 
    Propriété
    Description
    Nom du groupe
    Nom du groupe.
    Fournisseur d'authentification
    Fournisseur d'authentification contenant la définition du groupe.
    Groupes
    Groupe auquel le groupe appartient.

  4. Pour modifier les détails du groupe, cliquez sur Reconfigurer afin d'afficher la page Modifier les détails de groupe.

    5. Vous pouvez modifier les détails du groupe que vous soyez ou non connecté à une session.

  5. Apportez les modifications souhaitées au champ Appartenance à des groupes. Pour obtenir la description de ce champ, reportez-vous à Ajout de groupes.

    6. Vous ne pouvez pas modifier le champ Nom du groupe.

  6. Cliquez sur Enregistrer les modifications pour mettre à jour le groupe.

    7. La console d'administration Oracle Service Bus met immédiatement à jour les détails du groupe dans toutes les sessions. Si vous supprimez la session dans laquelle vous vous trouviez pendant la mise à jour du groupe, la console d'administration d'Oracle Service Bus ne supprime pas les mises à jour.
