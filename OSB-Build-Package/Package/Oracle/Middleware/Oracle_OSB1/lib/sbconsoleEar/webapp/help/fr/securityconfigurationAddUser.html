Ajout d'utilisateurs

  1. Connectez-vous à la console d'administration d'Oracle Service Bus en tant qu'utilisateur disposant des privilèges d'administration de WebLogic Server. Seuls les utilisateurs disposant du rôle d'administration peuvent modifier les données de configuration de sécurité. Reportez-vous à "Configuring Administrative Security" dans le manuel Oracle Fusion Middleware Developer's Guide for Oracle Service Bus à l'adresse suivante : http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539.
  2. Sélectionnez Configuration de la sécurité pour afficher la page Récapitulatif des utilisateurs.
  3. Cliquez sur Ajouter une valeur pour afficher la page Créer un utilisateur - Configuration générale.

    4. Vous pouvez ajouter un utilisateur que vous soyez ou non connecté à une session.

  4. Dans le champ Nom utilisateur, entrez un nom unique. Ce champ est obligatoire.
  5. Dans le champ Mot de passe, entrez un mot de passe. Ce champ est obligatoire.
    6. Remarque : les fournisseurs d'authentification peuvent imposer une longueur minimale pour les mots de passe. Lorsqu'ils sont définis par l'utilisateur via le fournisseur d'authentification WebLogic, la longueur minimale par défaut des mots de passe est de 8 caractères. Vous pouvez personnaliser ce paramètre à l'aide de la console d'administration WebLogic Server. (Le fournisseur d'authentification WebLogic est configuré dans le domaine de sécurité sous le nom DefaultAuthenticator.)
  6. Dans le champ Confirmer le mot de passe, entrez le même mot de passe que celui que vous avez saisi dans le champ Mot de passe. Ce champ est obligatoire.
  7. Dans le champ Fournisseur d'authentification, sélectionnez le fournisseur d'authentification de l'utilisateur.

    8. Si plusieurs fournisseurs d'authentification sont configurés dans le domaine de sécurité, ils apparaissent dans la liste. Sélectionnez la base de données du fournisseur d'authentification dans laquelle stocker les informations relatives au nouvel utilisateur.

  8. Dans le champ Appartenance à des groupes, sélectionnez le groupe de l'utilisateur.
    1. Sélectionnez un groupe dans le champ Groupes disponibles.
    2. Cliquez sur la flèche pour déplacer le groupe dans le champ Groupes en cours.
  9. Cliquez sur Enregistrer pour créer l'utilisateur.

    10. La console d'administration d'Oracle Service Bus enregistre l'utilisateur, qui devient immédiatement disponible dans toutes les sessions. Si vous supprimez la session dans laquelle vous vous trouviez pendant l'ajout de l'utilisateur, la console d'administration d'Oracle Service Bus ne supprime pas le nouvel utilisateur.
