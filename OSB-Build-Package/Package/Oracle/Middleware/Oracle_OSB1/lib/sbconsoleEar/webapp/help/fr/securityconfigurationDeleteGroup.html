Suppression de groupes

La page Récapitulatif des groupes permet de supprimer le groupe sélectionné ou plusieurs groupes.

  1. Connectez-vous à la console d'administration d'Oracle Service Bus en tant qu'utilisateur disposant des privilèges d'administration de WebLogic Server. Seuls les utilisateurs disposant du rôle d'administration peuvent modifier les données de configuration de sécurité. Reportez-vous à "Configuring Administrative Security" dans le manuel Oracle Fusion Middleware Developer's Guide for Oracle Service Bus à l'adresse suivante : http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539.
  2. Sélectionnez Configuration de la sécurité pour afficher la page Récapitulatif des groupes.
  3. Sélectionnez le groupe à supprimer. Vous pouvez sélectionner plusieurs groupes, si nécessaire.

    4. Vous pouvez supprimer des groupes que vous soyez ou non connecté à une session.

  4. Cliquez sur Supprimer. Un message vous invitant à confirmer la suppression du groupe apparaît.
  5. Pour supprimer le groupe, cliquez sur OK.

    6. La console d'administration d'Oracle Service Bus supprime le groupe. Si vous supprimez la session dans laquelle vous vous trouviez pendant la suppression du groupe, la console d'administration d'Oracle Service Bus n'annule pas la suppression du groupe.

  6. Vous pouvez également cliquer sur l'icône Supprimer dans la colonne Options du groupe à supprimer.
