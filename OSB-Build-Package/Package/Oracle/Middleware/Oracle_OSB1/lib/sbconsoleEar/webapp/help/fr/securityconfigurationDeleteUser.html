Suppression d’utilisateurs

La page Récapitulatif des utilisateurs permet de supprimer l'utilisateur sélectionné ou plusieurs utilisateurs.

  1. Connectez-vous à la console d'administration d'Oracle Service Bus en tant qu'utilisateur disposant des privilèges d'administration de WebLogic Server. Seuls les utilisateurs disposant du rôle d'administration peuvent modifier les données de configuration de sécurité. Reportez-vous à "Configuring Administrative Security" dans le manuel Oracle Fusion Middleware Developer's Guide for Oracle Service Bus à l'adresse suivante : http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539.
  2. Sélectionnez Configuration de la sécurité pour afficher la page Récapitulatif des utilisateurs.
  3. Sélectionnez l'utilisateur à supprimer. Vous pouvez sélectionner plusieurs utilisateurs, si nécessaire.

    4. Vous pouvez supprimer un utilisateur que vous soyez ou non connecté à une session.

  4. Cliquez sur Supprimer. Un message vous invitant à confirmer la suppression de l'utilisateur apparaît.
  5. Pour supprimer l'utilisateur, cliquez sur OK.

    6. La console d'administration d'Oracle Service Bus supprime l'utilisateur. Si vous supprimez la session dans laquelle vous vous trouviez pendant la suppression de l'utilisateur, la console d'administration d'Oracle Service Bus ne supprime pas la mise à jour.

  6. Vous pouvez également cliquer sur l'icône Supprimer dans la colonne Options de l'utilisateur à supprimer.
