Localisation de stratégies de contrôle d'accès

Sur la page Sécurité, un lien vous permet d'accéder aux stratégies de contrôle d'accès d'un service proxy du domaine Oracle Service Bus en cours.

Cette page énumère les services proxy que vous avez créés dans la session mais que vous n'avez pas encore activés. Pour modifier les stratégies de contrôle d'accès d'un nouveau service proxy, vous devez d'abord activer la session dans laquelle vous avez créé le service proxy.

  1. Pour localiser les stratégies de contrôle d'accès d'un nouveau service proxy, vous devez d'abord activer la session dans laquelle vous avez créé le service proxy.
  2. Dans la colonne Contrôle d'accès, sélectionnez le nom du service proxy dans Contrôle d'accès du transport ou le nom du service proxy ou d'une opération spécifique dans Contrôle d'accès des messages.
