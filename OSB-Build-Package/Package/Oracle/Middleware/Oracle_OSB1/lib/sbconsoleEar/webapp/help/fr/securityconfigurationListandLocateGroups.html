Localisation de groupes

  1. Sélectionnez Configuration de la sécurité > Groupes. La page Récapitulatif des groupes affiche les informations répertoriées dans le tableau 24-3. Pour obtenir une description plus détaillée des propriétés, reportez-vous à Modification de groupes.
  2. Pour limiter le nombre d'éléments dans la liste ou localiser un groupe spécifique, vous pouvez filtrer par nom de groupe. Entrez une partie ou la totalité du nom du groupe dans le champ Nom, puis cliquez sur Rechercher.

    3. Vous pouvez utiliser l'astérisque (*) comme caractère générique. (Les autres caractères génériques ne sont pas pris en charge.)

    Cliquez sur Tout visualiser pour enlever les filtres de recherche et afficher tous les groupes.

    Tableau 24-3 Informations sur les groupes 
    Propriété
    Description
    Nom du groupe
    Nom du groupe. Ce nom est un lien permettant d'accéder à la page Visualiser les détails de groupe. Reportez-vous à Modification de groupes.
    Appartenance aux groupes
    Groupe auquel ce groupe appartient. Ce nom est un lien permettant d'accéder à la page Visualiser les détails de groupe. Reportez-vous à Modification de groupes.
    Fournisseur d'authentification
    Fournisseur d'authentification contenant la définition du groupe.
    Supprimer
    Cliquez sur l'icône Supprimer pour supprimer un groupe spécifique. Reportez-vous à Suppression de groupes.
