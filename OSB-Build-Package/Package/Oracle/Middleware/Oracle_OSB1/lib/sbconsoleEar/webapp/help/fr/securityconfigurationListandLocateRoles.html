Localisation de rôles

  1. Sélectionnez Configuration de la sécurité > Rôles. La page Rôles globaux affiche les informations répertoriées dans le tableau 24-5. Pour obtenir une description plus détaillée des propriétés, reportez-vous à Modification de rôles.
  2. Pour limiter le nombre d'éléments dans la liste ou localiser un rôle spécifique, faites défiler les pages. Utilisez les contrôles de page au-dessus ou en dessous de la table.

    3. Tableau 24-5 Informations sur les rôles 
    Propriété
    Description
    Nom du rôle
    Nom du rôle. Ce nom est un lien permettant d'accéder à la page Visualiser les détails du rôle. Reportez-vous à Modification de rôles.
    Nom du fournisseur
    Fournisseur d'authentification du groupe.
