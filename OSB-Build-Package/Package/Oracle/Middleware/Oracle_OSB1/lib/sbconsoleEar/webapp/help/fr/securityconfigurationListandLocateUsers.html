Localisation d'utilisateurs

  1. Sélectionnez Configuration de la sécurité > Utilisateurs. La page Récapitulatif des utilisateurs affiche les informations répertoriées dans le tableau 24-1. Pour obtenir une description plus détaillée des propriétés, reportez-vous à Modification d'utilisateurs.
  2. Pour limiter le nombre d'éléments dans la liste ou localiser un utilisateur spécifique, vous pouvez filtrer par nom utilisateur. Entrez une partie ou la totalité du nom de l'utilisateur dans le champ Nom, puis cliquez sur Rechercher.

    3. Vous pouvez utiliser l'astérisque (*) comme caractère générique. (Les autres caractères génériques ne sont pas pris en charge.)

    Cliquez sur Tout visualiser pour enlever les filtres de recherche et afficher tous les utilisateurs.

    Tableau 24-1 Informations sur les utilisateurs 
    Propriété
    Description
    Nom utilisateur
    Nom affecté à l'utilisateur. Ce nom est un lien permettant d'accéder à la page Visualiser les détails d'utilisateur. Reportez-vous à Modification d'utilisateurs.
    Appartenance aux groupes
    Nom du groupe auquel l'utilisateur appartient. Ce nom est un lien permettant d'accéder à la page Visualiser les détails de groupe. Reportez-vous à Modification de groupes.
    Fournisseur d'authentification
    Fournisseur d'authentification de l'utilisateur.
    Options
    Cliquez sur l'icône Supprimer pour supprimer un utilisateur spécifique. Reportez-vous à Suppression d’utilisateurs.
