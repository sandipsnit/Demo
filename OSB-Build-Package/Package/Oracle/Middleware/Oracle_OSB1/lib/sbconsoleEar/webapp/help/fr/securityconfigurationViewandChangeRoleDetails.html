Modification de rôles

La page Visualiser les détails du rôle permet de visualiser et de modifier les détails d'un rôle spécifique.

  1. Connectez-vous à la console d'administration d'Oracle Service Bus en tant qu'utilisateur disposant des privilèges d'administration de WebLogic Server. Seuls les utilisateurs disposant du rôle d'administration peuvent modifier les données de configuration de sécurité. Reportez-vous à "Configuring Administrative Security" dans le manuel Oracle Fusion Middleware Developer's Guide for Oracle Service Bus à l'adresse suivante : http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539.
  2. Localisez le rôle. Reportez-vous à Localisation de rôles.
  3. Cliquez sur le nom du rôle. La page Visualiser les détails du rôle affiche les informations répertoriées dans le tableau 24-8.

    4. Tableau 24-8 Détails des rôles 
    Propriété
    Description
    Nom
    Nom du rôle.
    Conditions de rôle
    Conditions qui déterminent l'appartenance à ce rôle.

    Vous pouvez modifier les détails du rôle que vous soyez connecté ou non à une session.

  4. Effectuez l'une des opérations décrites dans le tableau 24-9.

    5. Tableau 24-9 Options des rôles 
    Opération
    Procédure
    Modifier le classement de l'expression sélectionnée.
    Cliquez sur Déplacer vers le haut ou Déplacer vers le bas.
    Fusionner ou annuler la fusion des conditions de rôle et permuter les éléments sélectionnés et/ou les instructions entre différentes expressions.
    Cliquez sur Combiner ou Dissocier.
    Rendre une condition négative. Par exemple, "NOT Groupe Opérateurs" exclut le groupe Opérateurs du rôle.
    Cliquez sur Réaliser inversion.
    Supprimer l'expression sélectionnée.
    Cliquez sur Enlever.

  5. Cliquez sur Enregistrer.

    6. La console d'administration d'Oracle Service Bus met à jour le rôle et la mise à jour est disponible immédiatement pour toutes les sessions. Si vous supprimez la session dans laquelle vous vous trouviez pendant la mise à jour du rôle, la console d'administration d'Oracle Service Bus ne supprime pas les mises à jour.
