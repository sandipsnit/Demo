Définition de conditions de rôle

  1. Sur la page Rôles globaux, cliquez sur le nom du nouveau rôle global à afficher dans la page Conditions de rôle global.
  2. Sous Conditions de rôle, cliquez sur Ajouter une condition.

    3. L'invite suivante apparaît :

    Choisissez le prédicat que vous souhaitez utiliser comme nouvelle condition

  3. Choisissez un prédicat dans la zone de liste. Généralement, vous choisissez Groupe. Lorsqu'un groupe est utilisé pour créer un rôle de sécurité, celui-ci peut être accordé à tous les membres du groupe (c'est-à-dire à plusieurs utilisateurs).
  4. Cliquez sur Suivant. En fonction du prédicat de condition choisi, suivez l'une des procédures décrites dans le tableau 24-6.

    5. Tableau 24-6 Options des prédicats de condition 
    Prédicat de condition...
    Exécutez les étapes suivantes...
    Si vous avez sélectionné Groupe, entrez des arguments qui définissent les groupes devant détenir ce rôle.
    1. Dans le champ Nom d'argument de groupe, entrez un argument qui définit le groupe.
    2. Cliquez sur Ajouter.
    3. Si nécessaire, répétez les étapes 1 et 2 afin d'ajouter tous les arguments souhaités. Pour enlever un argument de la liste, cliquez sur Enlever.
    4. Cliquez sur Fin.
    Si vous avez sélectionné Utilisateur, entrez des arguments qui définissent les utilisateurs devant détenir ce rôle.
    1. Dans le champ Nom d'argument utilisateur, entrez un argument qui définit l'utilisateur.
    2. Cliquez sur Ajouter.
    3. Si nécessaire, répétez les étapes 1 et 2 afin d'ajouter tous les arguments souhaités. Pour enlever un argument de la liste, cliquez sur Enlever.
    4. Cliquez sur Fin.
    Si vous avez sélectionné Serveur en mode développement, Accès autorisé à tous ou Accès refusé à tous
    Cliquez sur Fin.
    Si vous avez sélectionné un prédicat avec contrainte de temps, tel que Accès entre des heures précises, sélectionnez une heure de début et de fin ainsi qu'un décalage GMT.
    1. Dans le champ Heure de début, indiquez l'heure de début de la période autorisée au format hh:mm:ss AM|PM. Par exemple : 12:45:00 AM.
    2. Dans le champ Heure de fin, indiquez l'heure de fin de la période autorisée au format hh:mm:ss AM|PM. Par exemple : 12:45:00 AM.
    3. Dans le champ Décalage GMT, indiquez le nombre d'heures qui doivent être ajoutées au temps GMT au format GMT+hh:mm, ou le nombre d'heures qui doivent être soustraites au temps GMT au format GMT-hh:mm. Par exemple, aux Etats-Unis, l'heure de la côte Est est GMT-5:00.
    4. Cliquez sur Fin.
    Si vous avez sélectionné Défini par un élément de contexte, entrez le nom d'un élément de contexte.
    1. Dans le champ Nom de l'élément de contexte, entrez le nom d'un élément de contexte.
    2. Cliquez sur Fin.
    Si vous avez sélectionné Valeur de l'élément de contexte égale à une constante numérique, Valeur de l'élément de contexte supérieure à une constante numérique ou Valeur de l'élément de contexte inférieure à une constante numérique, entrez le nom d'un élément de contexte et une valeur numérique à des fins de comparaison.
    1. Dans le champ Nom de l'élément de contexte, entrez le nom de l'élément de contexte dont la valeur doit être évaluée.
    2. Dans le champ Valeur numérique, entrez une valeur numérique.
    3. Cliquez sur Fin.
    Si vous avez sélectionné Valeur de l'élément de contexte sous forme de chaîne, entrez le nom d'un élément de contexte et une valeur de chaîne à des fins de comparaison.
    1. Dans le champ Nom de l'élément de contexte, entrez le nom de l'élément de contexte dont la valeur doit être évaluée.
    2. Dans le champ Valeur de type String, indiquez la valeur de chaîne à comparer.
    3. Cliquez sur Fin.
    Si vous avez sélectionné un prédicat avec contrainte de temps, tel que Accès avant ou Accès après
    1. Dans le champ Date, indiquez une date au format mm/dd/yy. Par exemple : 1/1/04. Si vous le souhaitez, vous pouvez ajouter une heure au format hh:mm:ss AM|PM. Par exemple : 1/1/04 12:45:00 AM.
    2. Cliquez sur Fin.
    Si vous avez sélectionné le prédicat avec contrainte de temps Accès certains jours précis de la semaine, sélectionnez le jour de la semaine ainsi que le décalage GMT.
    1. Dans le champ Jour de la semaine, indiquez un jour de la semaine.
    2. Dans le champ Décalage GMT, indiquez le nombre d'heures qui doivent être ajoutées au temps GMT au format GMT+hh:mm, ou le nombre d'heures qui doivent être soustraites au temps GMT au format GMT-hh:mm. Par exemple, aux Etats-Unis, l'heure de la côte Est est GMT-5:00.
    3. Cliquez sur Fin.
    Si vous avez sélectionné un prédicat avec contrainte de temps, tel que Accès un jour précis du mois, Accès avant un jour précis du mois ou Accès après un jour précis du mois
    1. Dans le champ Jour du mois, indiquez le numéro ordinal du jour du mois en cours à l'aide d'une valeur comprise entre -31 et 31. Les valeurs négatives sont comptabilisées à partir de la fin du mois. Ainsi, -1 désigne le dernier jour du mois et 0 indique le jour précédant le premier jour du mois.
    2. Dans le champ Décalage GMT, indiquez le nombre d'heures qui doivent être ajoutées au temps GMT au format GMT+hh:mm, ou le nombre d'heures qui doivent être soustraites au temps GMT au format GMT-hh:mm. Par exemple, aux Etats-Unis, l'heure de la côte Est est GMT-5:00.
    3. Cliquez sur Fin.

  5. Si nécessaire, répétez les étapes qui permettent d'ajouter des expressions basées sur des conditions de rôle différentes. Dans la section Conditions de rôle vous pouvez suivre les procédures présentées dans le tableau 24-7, afin de modifier les expressions.

    6. Tableau 24-7 Options des conditions de rôle 
    Opération
    Exécutez les étapes suivantes...
    Modifier le classement de l'expression sélectionnée.
    Cliquez sur Déplacer vers le haut ou Déplacer vers le bas.
    Fusionner ou annuler la fusion des conditions de rôle et permuter les éléments sélectionnés et/ou les instructions entre différentes expressions.
    Cliquez sur Combiner ou Dissocier.
    Rendre une condition négative. Par exemple, "NOT Groupe Opérateurs" exclut le groupe Opérateurs du rôle.
    Cliquez sur Réaliser inversion.
    Supprimer l'expression sélectionnée.
    Cliquez sur Enlever.

  6. Une fois toutes les expressions de la section Conditions de rôle correctes, cliquez sur Enregistrer.
  7. Pour terminer la session et déployer la configuration lors de l'exécution, cliquez sur Activer sous Centre des modifications.
