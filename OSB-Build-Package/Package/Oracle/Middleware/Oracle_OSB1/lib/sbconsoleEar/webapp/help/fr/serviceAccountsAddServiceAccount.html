Ajout de comptes de service

  1. Si vous ne l'avez pas déjà fait, cliquez sur Créer afin de créer une session ou cliquez sur Modifier pour accéder à une session existante. Reportez-vous à Utilisation du centre des modifications.
  2. Sélectionnez Explorateur de projets, puis choisissez le projet ou le dossier auquel ajouter le compte de service. La page Vue du projet/Vue de dossier apparaît.
  3. Dans la liste déroulante Créer une ressource, sélectionnez Compte de service pour afficher la page Créer un compte de service.
  4. Dans le champ Nom de la ressource, entrez un nom unique pour ce compte de service.

    Pour obtenir des conseils sur la dénomination, suivez les restrictions concernant la dénomination des ressources.
  5. Dans le champ Description de la ressource, entrez la description du compte de service.
  6. Sous Type de ressource, effectuez l'une des opérations suivantes :
    • Pour créer un compte de service qui fournit les noms utilisateur et les mots de passe qu'il reçoit des demandes client entrantes, sélectionnez Pass-through.
    • Pour créer un compte de service qui fournit un nom utilisateur et un mot de passe que vous enregistrez avec la configuration du compte de service, sélectionnez Statique.
    • Pour créer un compte de service qui met en correspondance le nom utilisateur des clients authentifiés avec les noms utilisateurs et les mots de passe que vous spécifiez, sélectionnez Mise en correspondance.
  7. En fonction du type de ressource sélectionné à l'étape 6, suivez l'une des procédures décrites dans le tableau 16-2.

    8. Tableau 16-2 Options des types de ressource 
    Type de ressource sélectionné
    Procédure
    Pass-through
    Cliquez sur Dernier.
    Statique
    1. Cliquez sur Suivant.
    2. Entrez le nom utilisateur dans le champ Nom utilisateur, et le mot de passe dans les champs Mot de passe et Confirmer le mot de passe.
    3. Cliquez sur Dernier.
    Correspondance
    Pour créer un compte de service qui met en correspondance le nom utilisateur des clients avec les noms utilisateurs et les mots de passe que vous spécifiez, procédez comme suit :
    1. Cliquez sur Suivant.
    2. Dans la table Entrer un utilisateur distant autorisé, procédez comme suit :
      1. Dans les champs Nom utilisateur distant, Mot de passe et Confirmer le mot de passe, entrez le nom utilisateur et le mot de passe à envoyer dans les demandes sortantes.
      2. Cliquez sur Ajouter.

        3. La correspondance d'utilisateur est ajoutée à la table Utilisateurs distants.

      3. (Facultatif) Ajoutez des utilisateurs distants supplémentaires dans la table Entrer un utilisateur distant autorisé.
    3. Cliquez sur Suivant.
    4. Pour mettre en correspondance des clients autorisés avec des noms utilisateurs et mots de passe distants, procédez comme suit dans la table Entrer un utilisateur local autorisé :
      1. Dans le champ Nom utilisateur local, entrez le nom qui identifie un client authentifié sur sa demande entrante.

        2. Si vous n'avez pas encore ajouté cet utilisateur au module de configuration de sécurité de la console d'administration d'Oracle Service Bus, faites-le avant d'utiliser cette mise en correspondance dans un environnement d'exécution. Reportez-vous à Ajout d'utilisateurs. Oracle Service Bus vous autorise à créer une mise en correspondance pour un utilisateur local inexistant, mais celle-ci ne concordera jamais avec un utilisateur authentifié et ne sera jamais utilisée.

      2. Dans la liste Nom utilisateur distant, sélectionnez le nom utilisateur à envoyer dans les demandes sortantes pour l'utilisateur authentifié que vous avez spécifié dans le champ Nom utilisateur local.
      3. Cliquez sur Ajouter.
    5. Pour mettre en correspondance des clients anonymes avec des noms utilisateur distants, procédez comme suit :
      1. Cochez la case Mettre en correspondance les demandes anonymes.
      2. Dans la liste Sélectionnez un utilisateur distant, sélectionnez le nom utilisateur à envoyer dans les demandes sortantes pour tous les utilisateurs anonymes.
    6. Cliquez sur Dernier.

  8. Cliquez sur Enregistrer. Le compte de service est créé et enregistré dans la session en cours.
  9. Pour terminer la session et déployer la configuration lors de l'exécution, cliquez sur Activer sous Centre des modifications.
