Suppression de comptes de service

Lorsque vous supprimez un compte de service, le nom utilisateur, le mot de passe ou les données de mise en correspondance utilisateur local/utilisateur distant contenus dans le compte de service sont également supprimés.

  1. Si vous ne l'avez pas déjà fait, cliquez sur Créer afin de créer une session ou cliquez sur Modifier pour accéder à une session existante. Reportez-vous à Utilisation du centre des modifications.
  2. Si un service fonctionnel ou un service proxy est configuré de façon à utiliser le compte de service, vous devez enlever ce dernier du service.

    3. Reportez-vous à Modification des configurations des services fonctionnels ou à Modification des configurations des services proxy.

  3. Sélectionnez Navigateur Web de ressources > Comptes de service pour afficher la page Récapitulatif des comptes de service.
  4. Cliquez sur l'icône Supprimer dans le champ Options du compte de service à supprimer.

    5. Le compte de service est supprimé dans la session en cours. Si un service fonctionnel ou un service proxy a été configuré de façon à utiliser un compte de service, l'icône Avertissement relatif à la suppression indique que vous pouvez supprimer ce compte de service, à condition de confirmer l'opération lorsqu'un avertissement s'affiche. Toutefois, des conflits peuvent survenir, les références du service au compte de service supprimé n'étant pas résolues.

  5. Pour terminer la session et déployer la configuration lors de l'exécution, cliquez sur Activer sous Centre des modifications.
