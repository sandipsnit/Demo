Localisation de comptes de service

  1. Effectuez l'une des opérations suivantes :
    • Sélectionnez Explorateur de projets pour afficher la page Vue des projets ou Vue du projet/Vue de dossier. Ensuite, parcourez les projets et dossiers pour trouver le compte de service.
    • Sélectionnez Navigateur Web de ressources > Comptes de service. La page Récapitulatif des comptes de service affiche les informations présentées dans le tableau 16-1.
  2. Pour rechercher un compte de service, entrez tout ou partie du nom de compte dans le champ Nom. Vous pouvez également entrer tout ou partie du nom de projet du compte et de son·dossier dans les champs Chemin. Cliquez sur Rechercher.

    3. Cliquez sur Tout visualiser pour enlever les filtres de recherche et afficher tous les comptes de service.

    Tableau 16-1 Informations relatives au compte de service
    Propriété
    Description
    Nom du compte de service
    Nom unique du compte de service. Cliquez sur le nom pour visualiser la page Visualiser les détails du compte de service. Reportez-vous à Modification des comptes de service.
    Chemin
    Nom de projet et nom du dossier dans lequel réside le compte de service. Cliquez sur le nom pour visualiser le projet ou le dossier qui contient cette ressource. Reportez-vous à Qualification des noms de ressource à l'aide de projets et de dossiers.
    Options
    Contient une icône Supprimer. Si un service fonctionnel ou proxy a été configuré pour utiliser un compte de service, une icône Avertissement relatif à la suppression indique que vous pouvez supprimer le compte de service en confirmant l'opération. Toutefois, des conflits peuvent survenir, les références au compte de service supprimé n'étant pas résolues. Reportez-vous à Suppression de comptes de service.
