Utilisation des données et sessions des comptes de service

Les comptes de service et leurs données participent intégralement aux sessions Oracle Service Bus : vous devez ouvrir une session pour créer ou modifier un compte de service et, si vous annulez la session, le compte de service et ses données sont également annulés. Lorsque vous activez une session, Oracle Service Bus enregistre le nom utilisateur, le mot de passe et les autres données du compte de service dans le fournisseur de correspondance d'informations d'identification et de connexion des nom utilisateur/mot de passe, configuré pour le domaine.
