Opérations Fractionnement-Fusion

Les opérations Fractionnement-Fusion permettent d'envoyer des appels de message en parallèle (en sus d'un envoi séquentiel) et d'agréger les réponses. Pour en savoir plus sur les opérations Fractionnement-Fusion, reportez-vous à "Improving Service Performance with Split-Join" dans le manuel Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus à l'adresse http://www.oracle.com/pls/as111170/lookup?id=OSBAG291. Reportez-vous également à Working with Split-Join à l'adresse suivante : http://www.oracle.com/pls/as111170/lookup?id=OSBDV193 et à Split-Join User Interface Reference à l'adresse suivante : http://www.oracle.com/pls/as111170/lookup?id=OSBDV573 dans le manuel Oracle Fusion Middleware Developer's Guide for Oracle Service Bus.

Vous pouvez accéder aux opérations Fractionnement-Fusion via les services fonctionnels dotés d'un transport de type Flux. Pour plus d'informations, reportez-vous à la page Flux - Configuration du transport.
