Ajout de fournisseurs JNDI

Vous ajoutez et configurez une ressource de fournisseur JNDI et la mettez à disposition dans Oracle Service Bus en tant que ressource système. Vous devez vous trouver·dans une session active.

  1. Si vous ne l'avez pas déjà fait, cliquez sur Créer afin de créer une session ou cliquez sur Modifier pour accéder à une session existante. Reportez-vous à Utilisation du centre des modifications.
  2. Sélectionnez Administration du système > Fournisseurs JNDI.
  3. Cliquez sur Ajouter.
  4. Dans le champ Nom, entrez un nom unique pour la ressource de fournisseur JNDI. Ce champ est obligatoire.

    Pour obtenir des conseils sur la dénomination, suivez les restrictions concernant la dénomination des ressources.
  5. Dans le champ Description, entrez une description du fournisseur JNDI.
  6. Conservez le statut Activé (par défaut) de·l'option Cache JNDI ou sélectionnez Désactivé.

    7. Lorsque l'option est activée, le contexte JNDI et les objets JNDI sont mis en mémoire cache localement, ce qui améliore les performances lors d'une recherche d'objet. Oracle recommande de laisser le cache JNDI activé.

  7. Dans le champ URL du fournisseur, entrez l'URL du fournisseur JNDI au format suivant : protocol://host:port .

    8. Vous pouvez utiliser un protocole quelconque, par exemple, http, https, t3, t3s, iiop, iiops.

    Ce champ est obligatoire.

  8. Dans le cas des clusters, l'URL du fournisseur JNDI doit être configurée avec une liste de serveurs gérés séparés par une virgule, au format suivant :

    9. protocol://<hostname>:<ms1port>, <hostname>:<ms2port>

  9. Si l'accès à un fournisseur JNDI cible exige un nom utilisateur et un mot de passe, entrez un nom utilisateur dans le champ Nom utilisateur et le mot de passe associé dans les champs Mot de passe et Confirmer le mot de passe.

    10. Ces champs sont facultatifs et ne sont requis que lorsque l'arborescence JNDI est sécurisée.

  10. Cliquez sur Enregistrer pour créer et enregistrer la ressource du fournisseur JNDI dans la session en cours.
  11. Pour terminer la session et déployer la configuration lors de l'exécution, cliquez sur Activer sous Centre des modifications.
