Utilisation du statut d'import automatique

Vous pouvez utiliser la page Statut d'import automatique pour synchroniser les modifications apportées à un service avec celles présentes dans le registre. En cas de modification apportée à un service dans le registre, Oracle Service Bus fournit une notification sur la page Statut d'import automatique, qui énumère tous les services non synchronisés. Vous pouvez ensuite synchroniser le service de la console d'administration d'Oracle Service Bus avec le service correspondant dans le registre UDDI.

Lorsque vous configurez un registre, vous pouvez sélectionner l'option Activer l'import automatique qui synchronise automatiquement les services importés avec le registre UDDI. Si cette option est sélectionnée, tous les services importés seront maintenus en synchronisation avec le registre UDDI de façon automatique. Reportez-vous à Configuration des registres UDDI.

En cas d'échec lors de la synchronisation automatique, celui-ci sera indiqué sur la page Statut d'import automatique qui permet une mise à jour manuelle.

  1. Si vous ne l'avez pas déjà fait, cliquez sur Créer afin de créer une session ou cliquez sur Modifier pour accéder à une session existante. Reportez-vous à Utilisation du centre des modifications.
  2. Sélectionnez Administration du système > Statut d'import automatique.
  3. Sur la page Statut d'import automatique, sélectionnez le type de service à afficher dans la liste déroulante Visualiser.
  4. Cochez les cases situées en regard des services proxy à synchroniser avec les services correspondants dans le registre.
  5. Cliquez sur Synchroniser.
  6. Sur la page suivante, cliquez sur Terminer.
