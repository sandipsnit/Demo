Utilisation de la publication automatique

Vous pouvez utiliser la publication automatique pour publier les services proxy automatiquement sur un registre. Pour ce faire, vous devez activer l'option Publier vers le registre sur la page Créer/Modifier un service proxy - Configuration générale. Reportez-vous à Configuration générale, page. L'option Publier vers le registre n'apparaît que lorsqu'un registre par défaut existe. Reportez-vous à Définition d'une configuration UDDI par défaut.

  1. Si vous ne l'avez pas déjà fait, cliquez sur Créer afin de créer une session ou cliquez sur Modifier pour accéder à une session existante. Reportez-vous à Utilisation du centre des modifications.
  2. Sélectionnez l'explorateur de projets.
  3. Développez le dossier du projet et sélectionnez le service proxy à modifier.
  4. Au bas de la page Visualiser un service proxy, cliquez sur Modifier.
  5. Sélectionnez Publier vers le registre et cliquez sur Enregistrer pour valider les mises à jour dans la session en cours.
  6. Pour terminer la session et déployer la configuration lors de l'exécution, cliquez sur Activer sous Centre des modifications.

    7. Le service est configuré de façon à être publié sur le registre par défaut.

Remarque :
Remarque :
  • Lorsque vous avez publié un service sur un registre, l'option permettant de sélectionner le service en vue d'une publication automatique sur un registre n'est plus disponible.
Remarque :
  • Si la publication du service échoue, celui-ci peut être republié. Pour republier un service, sélectionnez-le sur la page Statut de publication automatique et cliquez sur Publier.
Remarque :
  • Si l'option Publier vers le registre est activée, les services proxy sont publiés dès leur création ou modification et la session est activée. Vous pouvez utiliser l'option Publier vers le registre avec tous les services proxy, à l'exception de ceux utilisant le transport local.
