Création de fichiers de personnalisation

La page Créer un fichier de personnalisation permet de générer de façon pratique un fichier de personnalisation pour un ensemble de ressources ou de projets que vous sélectionnez. Vous pouvez ensuite utiliser ce fichier comme point de départ pour apporter les modifications de votre choix en indiquant les valeurs réelles d'un environnement.

Vous pouvez utiliser les fichiers de personnalisation pour apporter des modifications aux valeurs d'environnement, ainsi qu'aux références dans les ressources. Les fichiers de personnalisation peuvent inclure toutes les valeurs d'environnement trouvées dans les ressources sélectionnées, y compris les types de valeur d'environnement complexe définis dans la classe EnvValueTypes. Ils incluent également un type de personnalisation de référence pour la modification des références de ressource dans les ressources dotées de dépendances.

Le schéma de personnalisation (Customization.xsd), qui décrit les types de personnalisation, est disponible à l'emplacement suivant dans votre installation Oracle Service Bus :

OSB_ORACLE_HOME\modules\com.bea.common.configfwk_version.jar.

  1. Sélectionnez Administration du système > Créer un fichier de personnalisation.

    2. La page Créer un fichier de personnalisation affiche la liste des objets compris dans votre configuration. Le nom et le type de chaque objet apparaissent.

  2. Sélectionnez les projets ou les ressources à inclure dans le fichier de personnalisation.
    1. Développez les dossiers de projet. Le nom et le type de chaque ressource contenue dans le projet apparaissent.
    2. Cochez les cases associées aux projets ou ressources à inclure dans le fichier de personnalisation.
    3. Désélectionnez les cases associées aux projets ou ressources à ne pas inclure dans le fichier de personnalisation.
  3. Cliquez sur Créer un fichier pour créer un fichier XML de personnalisation.
  4. Dans la boîte de dialogue Téléchargement de fichier, cliquez sur Ouvrir pour ouvrir le fichier de personnalisation ou cliquez sur Enregistrer pour enregistrer le fichier XML sur votre bureau.
