Définition d'une configuration UDDI par défaut

Vous pouvez désigner un des registres configurés comme registre UDDI par défaut pour le domaine. Reportez-vous à Configuration des registres UDDI.

Pour utiliser la fonction de publication automatique, vous devez au préalable définir un registre par défaut. Reportez-vous à Utilisation de la publication automatique.

  1. Si vous ne l'avez pas déjà fait, cliquez sur Créer afin de créer une session ou cliquez sur Modifier pour accéder à une session existante. Reportez-vous à Utilisation du centre des modifications.
  2. Sélectionnez Administration du système > Registres UDDI.
  3. Dans le panneau Configuration UDDI par défaut, cliquez sur Sélect. val./dft.
  4. Dans la liste Nom de registre par défaut, sélectionnez le nom du registre à définir par défaut.
  5. Pour définir un objet fonctionnel par défaut, choisissez-en un dans la liste Objet fonctionnel.
  6. Cliquez sur Définir comme valeur par défaut.
  7. Pour modifier le registre par défaut, cliquez sur Effacer la sélection et répétez les étapes précédentes.
