Modification de fournisseurs JNDI

Utilisez cette page pour visualiser les détails d'un fournisseur JNDI et en modifier la configuration, si nécessaire. Vous pouvez également réinitialiser le cache du fournisseur JNDI et tester la connexion au fournisseur JNDI.

  1. Si vous ne l'avez pas déjà fait, cliquez sur Créer afin de créer une session ou cliquez sur Modifier pour accéder à une session existante. Reportez-vous à Utilisation du centre des modifications.
  2. Localisez le fournisseur JNDI. Reportez-vous à Visualisation des fournisseurs JNDI.
  3. Cliquez sur le nom du fournisseur JNDI.

    4. La page Visualiser le fournisseur JNDI - Configuration affiche les informations relatives au fournisseur JNDI, présentées dans le tableau 30-2.

    Tableau 30-2 Informations relatives au fournisseur JNDI
    Propriété
    Description
    Dernière modification par
    Utilisateur ayant créé ou modifié la ressource de fournisseur JNDI, ou l'ayant importée dans la configuration.
    Dernière modification le
    Date et heure auxquelles l'utilisateur a créé ou modifié la ressource de fournisseur JNDI, ou l'a importée dans la configuration. Cliquez sur le lien de date et d'heure pour visualiser l'historique des modifications apportées à cette ressource. Reportez-vous à Page Visualiser l'historique des modifications.
    Références
    Nombre d'objets référencés par le fournisseur JNDI. Lorsque des références existent, cliquez sur le lien numérique pour visualiser la liste des références en question. Reportez-vous à Visualisation des références aux ressources.
    Référencé par
    Nombre d'objets référençant le fournisseur JNDI. Lorsque des références existent, cliquez sur le lien numérique pour visualiser la liste des objets en question. Reportez-vous à Visualisation des références aux ressources.
    Description
    Description du fournisseur JNDI.
    Cache JNDI
    Activé ou désactivé. Lorsque l'option est activée, le contexte JNDI et les objets JNDI sont mis en mémoire cache localement, ce qui améliore les performances lors d'une recherche d'objet. Oracle recommande de laisser le cache JNDI activé.
    URL du fournisseur
    URL du fournisseur JNDI.
    Temporisation de la demande JNDI
    Temporisation de la demande JNDI (en millisecondes). La valeur par défaut de zéro (0) signifie qu'aucune temporisation ne s'applique.
    Nom utilisateur et Mot de passe
    Nom utilisateur permettant d'accéder à un fournisseur JNDI sécurisé, avec la possibilité de modifier le mot de passe.

  4. Cliquez sur Modifier pour afficher la page Modifier un fournisseur JNDI - [provider-name].

    5. Vous ne pouvez pas modifier le contenu du champ Nom.

  5. Dans le champ Description, vous pouvez modifier la brève description du fournisseur JNDI.
  6. Dans le champ URL du fournisseur, vous pouvez modifier l'URL du fournisseur JNDI.
  7. Vous pouvez modifier les champs Nom utilisateur et Mot de passe.

    8. Si le fournisseur JNDI a été configuré avec un nom d'utilisateur et un mot de passe, il est impossible de modifier le nom d'utilisateur. Vous pouvez modifier les champs Nouveau mot de passe et Confirmer le mot de passe. Si le JNDI n'a pas été configuré avec un nom utilisateur et un mot de passe, vous pouvez entrer un nouveau nom dans le champ Nom utilisateur.

  8. Cliquez sur Enregistrer pour valider les mises à jour dans la session en cours.
  9. Pour terminer la session et déployer la configuration lors de l'exécution, cliquez sur Activer sous Centre des modifications.
