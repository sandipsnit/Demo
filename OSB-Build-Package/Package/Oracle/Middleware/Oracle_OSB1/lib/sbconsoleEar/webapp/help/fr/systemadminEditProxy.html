Modification des serveurs proxy

Utilisez cette page pour visualiser les détails d'un serveur proxy et en modifier la configuration, si nécessaire.

  1. Si vous ne l'avez pas déjà fait, cliquez sur Créer afin de créer une session ou cliquez sur Modifier pour accéder à une session existante. Reportez-vous à Utilisation du centre des modifications.
  2. Localisez le serveur proxy. Reportez-vous à Localisation des serveurs proxy.
  3. Cliquez sur le nom du serveur proxy.

    4. La page Visualiser les serveurs proxy - Configuration affiche les informations présentées dans le tableau 30-6.

    Tableau 30-6 Informations relatives au serveur proxy
    Propriété
    Description
    Dernière modification par
    Utilisateur ayant créé ou modifié la ressource de serveur proxy.
    Dernière modification le
    Date et heure auxquelles l'utilisateur a créé ou modifié la ressource de serveur proxy. Cliquez sur le lien de date et d'heure pour visualiser l'historique des modifications apportées à cette ressource. Reportez-vous à Page Visualiser l'historique des modifications.
    Références
    Nombre d'objets référencés par le serveur proxy. Lorsque des références existent, cliquez sur le lien numérique pour visualiser la liste des références en question. Reportez-vous à Visualisation des références aux ressources.
    Référencé par
    Nombre d'objets référençant le serveur proxy. Lorsque des références existent, cliquez sur le lien numérique pour visualiser la liste des objets en question. Reportez-vous à Visualisation des références aux ressources.
    Description
    Description du serveur proxy, le cas échéant.
    Paramètres d'hôte/de port
    Détails relatifs à chaque serveur proxy compris dans la ressource de serveur proxy, notamment les informations suivantes :
    • Hôte du serveur : nom d'hôte ou l'adresse IP du serveur proxy.

      • Le nom d'hôte du serveur proxy Oracle Service Bus doit être identique à celui du serveur proxy réel.

    • Port ClearText : numéro de port du serveur proxy en clair.
    • Port SSL : numéro de port SSL du serveur proxy.
    Nom utilisateur
    Nom utilisateur pour l'authentification proxy.

  4. Cliquez sur Modifier pour afficher la page Modifier un serveur proxy - [server-name].

    5. Vous ne pouvez pas modifier le contenu du champ Nom.

  5. Dans le champ Description, vous pouvez modifier la brève description du serveur proxy.
  6. Dans la section Paramètres d'hôte/de port, vous pouvez ajouter un nouveau serveur proxy en procédant comme suit :
    1. Dans le champ Hôte du serveur, entrez le nom d'hôte ou l'adresse IP du serveur proxy. Ce champ est obligatoire.

      2. Le nom d'hôte du serveur proxy Oracle Service Bus doit être identique à celui du serveur proxy réel.

    2. Dans le champ Port ClearText, entrez le numéro de port du serveur proxy en clair.
    3. Dans le champ Port SSL, entrez le numéro de port SSL du serveur proxy. Vous devez saisir un numéro de port SSL ou en clair.
    4. Cliquez sur Ajouter.

      5. Vous pouvez configurer plusieurs serveurs proxy pour chaque ressource de serveur proxy. Cela permet à Oracle Service Bus de procéder à l'équilibrage de la charge et d'offrir des fonctionnalités de tolérance de panne pour la ressource de serveur proxy.

      Vous pouvez également modifier ou supprimer une configuration de serveur proxy existante comme suit :

    • Cliquez sur l'icône Modifier dans le champ Options du serveur proxy à modifier. Changez la valeur des champs Hôte du serveur, Port ClearText et Port SSL, puis cliquez sur Mettre à jour.
    • Cliquez sur l'icône Supprimer dans le champ Options du serveur proxy à supprimer.
  7. Vous pouvez modifier les champs Nom utilisateur et Mot de passe, employés·pour l'authentification par proxy.
  8. Cliquez sur Enregistrer pour valider les mises à jour dans la session en cours.
  9. Pour terminer la session et déployer la configuration lors de l'exécution, cliquez sur Activer sous Centre des modifications.
