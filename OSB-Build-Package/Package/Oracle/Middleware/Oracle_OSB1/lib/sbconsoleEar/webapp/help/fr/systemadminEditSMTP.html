Modification des serveurs SMTP

Utilisez cette page pour visualiser les détails d'un serveur SMTP et en modifier la configuration, si nécessaire.

  1. Si vous ne l'avez pas déjà fait, cliquez sur Créer afin de créer une session ou cliquez sur Modifier pour accéder à une session existante. Reportez-vous à Utilisation du centre des modifications.
  2. Localisez le serveur SMTP. Reportez-vous à Localisation des serveurs SMTP.
  3. Cliquez sur le nom du serveur SMTP.

    4. La page Visualiser les serveurs SMTP - Configuration affiche les informations présentées dans le tableau 30-4.

    Tableau 30-4 Informations relatives au serveur SMTP
    Propriété
    Description
    Dernière modification par
    Utilisateur ayant créé ou modifié la ressource de serveur SMTP.
    Dernière modification le
    Date et heure auxquelles l'utilisateur a créé ou modifié la ressource de serveur SMTP. Cliquez sur le lien de date et d'heure pour visualiser l'historique des modifications apportées à cette ressource. Reportez-vous à Page Visualiser l'historique des modifications.
    Références
    Nombre d'objets référencés par le serveur SMTP. Lorsque des références existent, cliquez sur le lien numérique pour visualiser la liste des références en question. Reportez-vous à Visualisation des références aux ressources.
    Référencé par
    Nombre d'objets référençant le serveur SMTP. Lorsque des références existent, cliquez sur le lien numérique pour visualiser la liste des objets en question. Reportez-vous à Visualisation des références aux ressources.
    Description
    Description du serveur SMTP, le cas échéant.
    URL du serveur
    URL pointant vers l'emplacement du serveur SMTP.
    Numéro de port
    Numéro de port du serveur SMTP. Par défaut, il s'agit du port 25.
    Nom utilisateur
    Nom utilisateur permettant d'accéder à un serveur SMTP sécurisé.

  4. Cliquez sur Modifier pour afficher la page Modifier un serveur SMTP - [server-name].

    5. Vous ne pouvez pas modifier le contenu du champ Nom.

  5. Dans le champ Description, vous pouvez modifier la brève description du serveur SMTP.
  6. Dans le champ URL du serveur, vous pouvez modifier l'URL du serveur SMTP.
  7. Dans le champ Numéro du port, vous pouvez modifier le numéro de port.
  8. Vous pouvez modifier les champs Nom utilisateur et Mot de passe.
  9. Cliquez sur Enregistrer pour valider les mises à jour dans la session en cours.
  10. Pour terminer la session et déployer la configuration lors de l'exécution, cliquez sur Activer sous Centre des modifications.

Si vous renommez une ressource SMTP, le nouveau nom est automatiquement mis à jour dans les services fonctionnels de messagerie électronique qui référencent la ressource SMTP dans l'URI endpoint.
