Exécution de fichiers de personnalisation

Utilisez cette page pour exécuter un fichier XML de personnalisation préalablement enregistré sur le système. Les fichiers de personnalisation permettent de modifier votre environnement lors du déploiement, du développement à la préparation, de la préparation à la production ou pendant la phase·de conception.

Vous devez vous trouver dans une session pour exécuter un fichier de personnalisation.

  1. Si vous ne l'avez pas déjà fait, cliquez sur Créer afin de créer une session ou cliquez sur Modifier pour accéder à une session existante. Reportez-vous à Utilisation du centre des modifications.
  2. Sélectionnez Administration du système > Exécuter le fichier de personnalisation.
  3. Sélectionnez un fichier de personnalisation à exécuter.
    1. Dans le champ Nom de fichier, cliquez sur Parcourir pour localiser le répertoire dans lequel le fichier de personnalisation est stocké.
    2. Sélectionnez le fichier XML, puis cliquez sur Ouvrir.
  4. Cliquez sur Suivant. Les détails de personnalisation suivants, présentés dans le tableau 31-2, sont affichés.

    5. Tableau 31-2 Détails de personnalisation
    Propriété
    Description
    Opération
    • Rechercher et remplacer : permet de modifier les valeurs d'environnement par des substitutions de chaînes.
    • Mettre en correspondance les références : met en correspondance les références existantes trouvées dans les ressources avec d'autres références.
    • Définir des valeurs d'environnement : permet de modifier les valeurs d'environnement des ressources.
    Appliquer à
    Nom du service, du projet, du dossier ou de la ressource à personnaliser.
    Le nom constitue un lien vers la page Détails de configuration du service ou de la ressource, ou vers le projet ou le dossier dans l'explorateur de projets.
    Description
    Description de chaque personnalisation.

  5. Pour ne personnaliser que les ressources, services, projets ou dossiers modifiés dans la session en cours, cochez la case Uniquement les éléments modifiés dans la session en cours.

    6. Si vous limitez les personnalisations aux ressources modifiées dans la session en cours, la colonne Appliquer à est mise à jour avec les ressources modifiées pendant·la session en cours. Si vous limitez les personnalisations à un projet ou dossier, les ressources modifiées dans le projet ou dossier en question pendant·la session en cours apparaissent dans la colonne Appliquer à.

  6. Cliquez sur Exécuter pour valider les mises à jour dans la session en cours.

    7. Pour visualiser les détails de la personnalisation, dans la session, sélectionnez Visualiser les modifications sous le centre des modifications et cliquez sur la tâche Personnalisation. Reportez-vous à Visualisation des modifications de configuration.

  7. Pour terminer la session et déployer la configuration lors de l'exécution, cliquez sur Activer sous Centre des modifications.
