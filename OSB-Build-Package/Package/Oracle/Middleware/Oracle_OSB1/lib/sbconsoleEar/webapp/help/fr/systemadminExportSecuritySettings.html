Définition·des paramètres de sécurité de l'export

Si l'export inclut un compte de service ou un fournisseur de clés de service, la page Exporter les ressources - Paramètres de sécurité apparaît.

Pour chaque compte de service, Oracle Service Bus exporte les nom utilisateur et mot de passe ou la correspondance utilisateur local/utilisateur distant (en fonction des données stockées dans le compte de service). Pour chaque fournisseur de clés de service, Oracle Service Bus exporte l'alias vers la liaison de paire de clés à partir du fournisseur de correspondances d'informations d'identification et de connexion PKI ; il n'exporte pas les clés privées, les certificats ou les autres données des fichiers de clés. Ces dernières doivent être exportées à l'aide des outils mis à disposition par·le fournisseur de fichiers de clés.

Si vous souhaitez qu'Oracle Service Bus crypte les données de nom utilisateur et de mot de passe contenues dans le compte de service, le fournisseur de clés de service, le registre UDDI, le fournisseur JNDI ou les ressources du fournisseur SMTP, avant de les exporter dans le fichier JAR, procédez comme suit :

  1. Cochez la case Protéger les données sensibles.
  2. Entrez un mot de passe et confirmez-le.

    3. Lorsque vous ou quelqu'un d'autre tentez d'importer le fichier JAR, Oracle Service Bus n'importe pas les ressources dotées de nom utilisateur et de mot de passe cryptés, sauf si vous indiquez le mot de passe. Vous pouvez importer toutes les autres ressources non cryptées du fichier JAR sans indiquer de mot de passe.

  3. Cliquez sur le bouton Terminer l'export.
  4. Lorsque vous cliquez sur Exporter ou Terminer l'export, la fenêtre instantanée Traitement des données de configuration apparaît. Attendez que la fenêtre se ferme avant d'effectuer une autre opération sur la console d'administration d'Oracle Service Bus. La fenêtre se ferme automatiquement lorsque la boîte de dialogue Téléchargement de fichier apparaît.
  5. Dans la boîte de dialogue Téléchargement de fichier, cliquez sur Ouvrir pour ouvrir le fichier JAR exporté ou cliquez sur Enregistrer pour enregistrer le fichier JAR sur votre bureau.
