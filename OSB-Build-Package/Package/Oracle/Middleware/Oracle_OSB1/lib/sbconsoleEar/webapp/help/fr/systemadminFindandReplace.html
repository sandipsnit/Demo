Recherche et remplacement des valeurs d'environnement

Utilisez la page Rechercher et remplacer pour rechercher les valeurs d'environnement qui diffèrent entre les domaines. Les valeurs d'environnement correspondent à certains champs prédéfinis dans les données de configuration dont les valeurs sont très susceptibles de varier lorsque vous déplacez votre configuration d'un domaine à un autre (par exemple, du test à la production). Les valeurs d'environnement représentent des entités, telles que les URL, les URI, les noms de fichier et de répertoire, les noms de serveur, les adresses électroniques, etc. Par ailleurs, les valeurs d'environnement peuvent être trouvées dans les destinations d'alerte, les services proxy, les services fonctionnels, les ressources de serveur SMTP et de fournisseur JNDI, et les entrées du registre UDDI.

Cette page se comporte différemment selon que vous êtes dans une session ou non. Si vous êtes dans une session, vous pouvez rechercher et remplacer les valeurs d'environnement. Dans le cas inverse, en revanche, vous pouvez seulement rechercher les valeurs d'environnement ; l'option Tout remplacer est désactivée.

  1. Si vous ne l'avez pas déjà fait, cliquez sur Créer afin de créer une session ou cliquez sur Modifier pour accéder à une session existante. Reportez-vous à Utilisation du centre des modifications.
  2. Sélectionnez Administration du système > Rechercher et remplacer.
  3. Dans le champ Rechercher la valeur, entrez la valeur d'environnement à localiser.
  4. Pour afficher la·liste des valeurs d'environnement de·votre configuration qui contiennent la valeur saisie, cliquez sur Rechercher.

    5. Une liste de valeurs pertinentes apparaît avec les colonnes Propriétaire, Type et Valeur d'environnement.

  5. Pour ne localiser que les éléments modifiés dans la·session en cours, cochez la case Uniquement les éléments modifiés dans la session en cours.
  6. Sélectionnez le type de variable à rechercher, dans la liste déroulante Rechercher dans le type de variable, décrite dans le tableau 31-1.

  7. Pour localiser les valeurs d'environnement situées·dans un projet particulier, sélectionnez le nom du projet dans la liste déroulante Situé dans le projet.
  8. Dans le champ Remplacer par, entrez la nouvelle valeur d'environnement.
  9. Pour remplacer la valeur d'environnement d'origine par la nouvelle valeur, cliquez sur Tout remplacer.

    10. Toutes les occurrences de la valeur d'environnement que vous avez saisie dans le champ Rechercher la valeur sont remplacées par la valeur d'environnement indiquée·dans le champ Remplacer par (cela est valable dans la session en cours).

    Remarque : Certaines valeurs d'environnement sont des objets XML complexes, impossibles à rechercher et à remplacer via la console d'administration d'Oracle Service Bus. Toutefois, vous pouvez toujours définir ces valeurs d'environnement directement en utilisant le MBean de configuration ALSBConfigurationMBean à partir d'un script. Pour plus de détails sur ALSBConfigurationMBean, reportez-vous à la référence d'API d'Oracle Service Bus. Si vous ne souhaitez pas passer par l'API, vous pouvez définir les valeurs d'environnement complexes à l'aide de fichiers de personnalisation. Reportez-vous à Exécution de fichiers de personnalisation.
  10. Pour terminer la session et déployer la configuration lors de l'exécution, cliquez sur Activer sous Centre des modifications.
