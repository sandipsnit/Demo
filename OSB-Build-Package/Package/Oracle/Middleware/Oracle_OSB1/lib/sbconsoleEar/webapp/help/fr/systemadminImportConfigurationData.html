Import de ressources

Utilisez la page Importer les ressources pour importer un fichier JAR de projet complet ou un fichier JAR de ressource précédemment exporté d'un autre domaine Oracle Service Bus. Vous pouvez choisir de n'importer qu'un sous-ensemble des données exportées. Si la ressource existe déjà dans le système d'import, elle sera mise à jour. Dans le cas contraire, elle sera créée. Les ressources sont programmées pour la suppression uniquement lorsque le fichier JAR en cours d'import correspond à un fichier JAR de projet complet et que des ressources figurent dans le même projet au sein du système d'import mais·que celles-ci ne sont pas présentes dans le fichier JAR importé. Cette opération ne supprimera pas les ressources situées dans d'autres projets.

Vous pouvez importer des ressources au niveau de la ressource, même si elles ont été exportées au niveau du projet. Par exemple, même si des ressources système ont été exportées dans un fichier JAR de projet complet, vous pouvez les désélectionner lors de l'import.

Vous ne pouvez pas exporter d'utilisateurs, de groupes, de rôles ou de certificats lorsque vous exportez une configuration. Vous devez donc recréer ces objets lorsque vous importez une configuration exportée.

Remarque : Pour plus d'informations sur l'import de ressources qui ne font pas partie d'une configuration Oracle Service Bus, reportez-vous à Chargement de ressources à partir d'un fichier ZIP et à Chargement de ressources à partir d'une URL.
  1. Si vous ne l'avez pas déjà fait, cliquez sur Créer afin de créer une session ou cliquez sur Modifier pour accéder à une session existante. Reportez-vous à Utilisation du centre des modifications.
  2. Sélectionnez Administration du système > Importer les ressources.
  3. Sélectionnez un fichier JAR de projet ou de ressource exporté.
    1. Dans le champ Nom de fichier, cliquez sur Parcourir pour localiser le répertoire dans lequel est stocké le fichier JAR.
    2. Sélectionnez le fichier JAR, puis cliquez sur Ouvrir.
  4. Cliquez sur Suivant. Pour continuer, passez à Sélection de projets et de ressources à importer.
