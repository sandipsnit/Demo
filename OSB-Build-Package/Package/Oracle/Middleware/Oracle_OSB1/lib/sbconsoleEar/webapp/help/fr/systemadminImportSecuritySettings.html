Indication des paramètres de sécurité de l'import

Si le fichier JAR contient des ressources dont les données de nom utilisateur et de mot de passe ont été cryptées lors de l'export, la page Importer les ressources - Paramètres de sécurité apparaît.

  1. Saisissez le mot de passe utilisé pour crypter les données (reportez-vous à Export de ressources).

    2. Si vous ne connaissez pas le mot de passe utilisé pour crypter les données de ressource, cliquez sur le bouton Préc et désélectionnez la case située en regard de la ressource. Vous pouvez importer toutes les autres ressources non cryptées même si vous ne connaissez pas le mot de passe.

  2. Cliquez sur Terminer l'import.

    3. La fenêtre instantanée Traitement des données de configuration apparaît. Attendez que la fenêtre se ferme avant d'effectuer une autre opération sur la console d'administration d'Oracle Service Bus. La fenêtre se ferme automatiquement lorsque le système termine l'importation des objets. La page Importer les ressources apparaît. Elle inclut un message indiquant que vous avez déployé les données.

Après avoir fini

Pour continuer, passez à Consultation du récapitulatif d'import.
