Consultation du récapitulatif d'import

Les informations récapitulatives de l'import sont présentées dans le tableau 28-3.

Tableau 28-3 Informations récapitulatives de l'import
Propriété
Description
Statut
Indique si la ressource a été importée.
Un fichier WSDL peut temporairement être considéré comme non valide tant que toutes ses dépendances ne sont pas importées. Une fois l'import terminé, le service de configuration valide à nouveau les fichiers marqués comme non valides.
Nom
Nom de la ressource.
Chemin
Chemin de la ressource. Le chemin correspond au nom de projet et au nom du dossier dans lequel réside la ressource.
Type
Affiche les types de ressource d'Oracle Service Bus.
Message de diagnostic
Affiche les messages d'erreur ou d'informations de la ressource, le cas échéant.

  1. Cliquez sur Importer un autre élément.

    2. Vous pouvez importer un autre fichier JAR ou cliquer sur Visualiser les modifications dans·le centre des modifications, afin de visualiser les détails de l'import. Reportez-vous à Visualisation des modifications de configuration.

  2. Lorsque vous avez terminé, cliquez sur Activer dans·le centre des modifications.
