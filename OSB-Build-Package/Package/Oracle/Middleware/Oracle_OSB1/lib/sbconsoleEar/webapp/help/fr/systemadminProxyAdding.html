Ajout de serveurs proxy

Utilisez la page Récapitulatif des serveurs proxy pour ajouter et configurer des ressources de serveur proxy et les mettre à disposition dans Oracle Service Bus en tant que ressource système. Vous devez être dans une session active pour configurer ou reconfigurer les ressources de serveur proxy.

  1. Si vous ne l'avez pas déjà fait, cliquez sur Créer afin de créer une session ou cliquez sur Modifier pour accéder à une session existante. Reportez-vous à Utilisation du centre des modifications.
  2. Sélectionnez Administration du système > Serveurs proxy.
  3. Cliquez sur Ajouter.
  4. Dans le Nom, entrez un nom pour la ressource de serveur proxy. Ce champ est obligatoire.

    Pour obtenir des conseils sur la dénomination, suivez les restrictions concernant la dénomination des ressources.
  5. Dans le champ Description, entrez une brève description de la ressource de serveur proxy.
  6. Dans la section Paramètres d'hôte/de port, entrez les informations suivantes :
    1. Dans le champ Hôte du serveur, entrez le nom d'hôte ou l'adresse IP du serveur proxy. Ce champ est obligatoire.

      2. Le nom d'hôte du serveur proxy Oracle Service Bus doit être identique à celui du serveur proxy réel.

    2. Dans le champ Port ClearText, entrez le numéro de port du serveur proxy en clair.
    3. Dans le champ Port SSL, entrez le numéro de port SSL du serveur proxy. Vous devez saisir un numéro de port SSL ou en clair.
    4. Cliquez sur Ajouter.

      5. Vous pouvez configurer plusieurs serveurs proxy pour chaque ressource de serveur proxy. Cela permet à Oracle Service Bus de procéder à l'équilibrage de la charge et d'offrir des fonctionnalités de tolérance de panne pour la ressource de serveur proxy.

  7. Si le serveur proxy procède à l'authentification par proxy, entrez un nom utilisateur dans le champ Nom utilisateur, et le mot de passe associé dans les champs Mot de passe et Confirmer le mot de passe.

    8. Ces champs sont facultatifs et requis·uniquement·lorsque le serveur proxy est sécurisé.

  8. Cliquez sur Enregistrer pour créer et enregistrer la ressource de serveur proxy dans la session en cours.
  9. Pour terminer la session et déployer la configuration lors de l'exécution, cliquez sur Activer sous Centre des modifications.
