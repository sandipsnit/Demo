Publication des services proxy dans un registre UDDI

Vous pouvez publier votre service dans un registre et permettre à d'autres organisations de le repérer et de l'utiliser. Tous les services proxy décrits dans Oracle Service Bus peuvent être publiés dans un registre UDDI. Vous pouvez sélectionner l'objet fonctionnel sous lequel vous souhaitez publier votre service et publier plusieurs services à la fois.

Pour pouvoir effectuer une publication dans un registre, vous devez disposer d'un compte auprès du registre en question. Oracle Service Bus prend en charge l'interopérabilité avec les registres UDDI adaptés à la version 3.0. La configuration décrite dans cette section implique que vous utilisiez Oracle Service Registry.

Pour plus d'informations sur la configuration d'un compte utilisateur dans Oracle Service Registry, reportez-vous à la documentation du produit Oracle Service Registry, disponible à l'adresse suivante : http://www.oracle.com/technology/products/soa/registry/index.html.

L'annulation de la publication d'un service dans un registre s'effectue à partir de l'installation d'Oracle Service Registry.

  1. Sélectionnez Administration du système > Publier vers UDDI.

    2. Pour effectuer une publication, vous ne devez pas vous trouver dans une session. Quittez votre session pour accéder à la liste des registres. Tous les registres auxquels vous avez accès sont affichés dans la liste déroulante des registres sélectionnés.

  2. Dans la liste Nom du registre de publication, sélectionnez le nom du registre dans lequel effectuer la publication, puis cliquez sur Suivant.

    3. Lorsqu'un seul registre est configuré dans Oracle Service Bus, la page de sélection du registre initial n'est pas affichée, puisqu'il s'agit du registre par défaut. La page de sélection du registre n'apparaît que lorsque au moins deux registres sont définis.

    La page Publier vers UDDI : Sélectionner des services individuels et les publier apparaît. Elle indique une liste de services proxy Oracle Service Bus et leur emplacement dans le dossier du projet. Partant du dossier de projet du premier niveau, les détails de nom, de type et de description sont indiqués pour tous les services.

  3. Sélectionnez les services à publier.
    1. Développez le dossier Projet pour visualiser les services proxy définis. Par défaut, le dossier et son contenu sont marqués en vue de·la publication.
    2. Vous pouvez sélectionner les différents éléments à publier.
  4. Dans la liste déroulante Publier des services vers l'objet fonctionnel, sélectionnez l'objet fonctionnel auquel vous souhaitez ajouter le service dans le registre. Il s'agit de l'objet fonctionnel sous lequel le service sera classé dans le registre.
  5. Cliquez sur Publier pour publier les services.

    6. Un message de confirmation apparaît et indique que le service a été publié.

  6. Cliquez sur Publier un autre élément pour revenir à la page Publier vers UDDI - Sélectionner un registre.
