Visualisation des registres UDDI configurés

Utilisez cette page pour désigner un registre UDDI par défaut (reportez-vous à Définition d'une configuration UDDI par défaut) et pour visualiser les registres UDDI. Par ailleurs, vous pouvez rechercher les registres que vous avez configurés précédemment.

  1. Accédez au panneau Registres UDDI, entrez le nom du registre à rechercher dans le champ Nom ou entrez des caractères génériques. Utilisez * pour indiquer plusieurs caractères et ? pour un caractère unique. La recherche renvoie tous les enregistrements qui répondent aux critères de nom.
  2. Cliquez sur Rechercher.
  3. Cliquez sur Tout visualiser pour enlever les filtres de recherche et visualiser tous les registres configurés pour fonctionner avec Oracle Service Bus.

    4. Le tableau 29-1 affiche les paramètres de configuration du registre UDDI.

    Tableau 29-1 Paramètres de configuration du registre UDDI 
    Propriété
    Description
    Nom de registre
    Nom du registre. Nom affecté à un registre lors de sa création. Vous ne pouvez pas modifier le nom d'un registre une fois l'entrée enregistrée.
    URL d'interrogation
    URL de l'adresse de l'API d'interrogation utilisée pour localiser et importer les services.
    URL de publication
    URL de l'adresse de l'API de publication utilisée pour publier les services.
    URL de sécurité
    URL de l'adresse de l'API de sécurité utilisée pour l'obtention du jeton d'authentification permettant de publier dans le registre.
    URL d'abonnement
    URL de l'adresse de l'API d'abonnement utilisée pour l'abonnement aux modifications de registre, la création d'un abonnement au registre et l'écoute des modifications apportées aux services importés.
    Nom utilisateur
    Nom utilisateur obligatoire pour l'authentification de l'utilisateur.
    Mot de passe (Confirmer le mot de passe)
    Mot de passe obligatoire pour l'authentification de l'utilisateur.
    Charger les tModels dans le registre
    Charge les tModels dans le registre sélectionné. Cette option ne doit être sélectionnée qu'une fois par registre.
    Activer l'import automatique
    Synchronise automatiquement les services avec le registre UDDI.
