Ajout de serveurs SMTP

Utilisez la page Récapitulatif des serveurs SMTP pour ajouter et configurer des ressources de serveur SMTP et les mettre à disposition dans Oracle Service Bus en tant que ressource système. Vous devez être dans une session active pour configurer ou reconfigurer les ressources de serveur SMTP.

  1. Si vous ne l'avez pas déjà fait, cliquez sur Créer afin de créer une session ou cliquez sur Modifier pour accéder à une session existante. Reportez-vous à Utilisation du centre des modifications.
  2. Sélectionnez Administration du système > Serveurs SMTP.
  3. Cliquez sur Ajouter.
  4. Dans le champ Nom, entrez un nom pour la ressource de serveur SMTP. Ce champ est obligatoire.

    Pour obtenir des conseils sur la dénomination, suivez les restrictions concernant la dénomination des ressources.
  5. Dans le champ Description, entrez une brève description de la ressource de serveur SMTP.
  6. Dans le champ URL du serveur, entrez l'URL qui pointe vers le serveur SMTP. Ce champ est obligatoire.
  7. Dans le champ Numéro du port, entrez le numéro de port du serveur SMTP (le port par défaut est 25). Ce champ est obligatoire.
  8. Si l'accès au serveur SMTP cible exige un nom utilisateur et un mot de passe, entrez un nom utilisateur dans le champ Nom utilisateur et le mot de passe associé dans les champs Mot de passe et Confirmer le mot de passe.

    9. Ces champs sont facultatifs requis uniquement·lorsque le serveur SMTP est sécurisé.

  9. Cliquez sur Enregistrer pour créer et enregistrer la ressource de serveur SMTP dans la session en cours.
  10. Pour terminer la session et déployer la configuration lors de l'exécution, cliquez sur Activer sous Centre des modifications.
