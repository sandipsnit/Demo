Configuration d'un serveur SMTP par défaut

Vous pouvez désigner un des serveurs SMTP configurés comme serveur par défaut du domaine. Reportez-vous à Ajout de serveurs SMTP.

  1. Si vous ne l'avez pas déjà fait, cliquez sur Créer afin de créer une session ou cliquez sur Modifier pour accéder à une session existante. Reportez-vous à Utilisation du centre des modifications.
  2. Sélectionnez Administration du système > Serveurs SMTP.
  3. Dans le panneau Serveur SMTP par défaut, cliquez sur Sélect. val./dft.
  4. Dans la liste déroulante Serveur SMTP par défaut, sélectionnez un serveur SMTP.
  5. Cliquez sur Définir comme valeur par défaut.

    6. La page Récapitulatif des serveurs SMTP apparaît et présente le serveur SMTP sélectionné comme le serveur SMTP par défaut du domaine.

  6. Pour terminer la session et déployer la configuration lors de l'exécution, cliquez sur Activer sous Centre des modifications.
