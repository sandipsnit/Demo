Visualisation des fournisseurs JNDI

Cette page permet de visualiser la liste des fournisseurs JNDI et les informations présentées dans le tableau 30-1.

Vous pouvez également rechercher des fournisseurs JNDI précédemment configurés.

  1. Pour restreindre le nombre d'éléments répertoriés ou localiser des fournisseurs JNDI spécifiques, procédez comme suit :
    • Cliquez sur l'icône Open pour afficher les autres filtres de recherche. Si vous indiquez plusieurs critères de recherche, seuls les résultats correspondant à tous les critères seront renvoyés.
    • Entrez le nom du fournisseur JNDI à rechercher dans le champ Nom et cliquez sur Rechercher pour atteindre·une entrée spécifique.

      • Vous pouvez entrer des caractères génériques (? pour un caractère unique, * pour plusieurs caractères) afin d'effectuer une recherche plus générale.

    • Entrez l'URL du fournisseur JNDI à rechercher et cliquez sur Rechercher.
  2. Cliquez sur Tout visualiser pour enlever les filtres de recherche et afficher toutes les ressources de fournisseur JNDI configurées pour fonctionner avec Oracle Service Bus.

    3. Le tableau 30-1 affiche les paramètres de configuration du registre UDDI.

    Tableau 30-1 Informations relatives au fournisseur JNDI
    Propriété
    Description
    Nom
    Nom unique affecté au fournisseur JNDI. Ce nom constitue un lien vers la page Visualiser le fournisseur JNDI - Configuration. Reportez-vous à Modification de fournisseurs JNDI.
    URL du fournisseur
    URL du fournisseur JNDI. Reportez-vous à Ajout de fournisseurs JNDI.
    Cache JNDI
    Activé ou désactivé. Lorsque l'option est activée, le contexte JNDI et les objets JNDI sont mis en mémoire cache localement, ce qui améliore les performances lors d'une recherche d'objet. Oracle recommande de laisser le cache JNDI activé.
    Actions
    • Cliquez sur l'icône Réinitialiser pour réinitialiser le contexte JNDI afin de supprimer·la connexion JNDI et les objets mis en mémoire dans le cache local.
    • Cliquez sur l'icône Lancer la console de test pour tester la faculté du fournisseur JNDI à établir une connexion et à obtenir un contexte JNDI.
    Options
    • Cliquez sur l'icône Renommer pour renommer le fournisseur JNDI.
    • Cliquez sur l'icône Supprimer pour supprimer un fournisseur JNDI spécifique. Reportez-vous à Suppression de fournisseurs JNDI.
