Visualisation des serveurs proxy

Utilisez cette page pour visualiser la liste des serveurs proxy et les informations répertoriées dans le tableau 30-5.

Les ressources de serveur proxy sont utilisées lors de la configuration du transport HTTP pour les services fonctionnels. Pour plus d'informations, reportez-vous à Page HTTP - Configuration du transport.

Le tableau 30-5 présente les informations relatives aux serveurs proxy.

Tableau 30-5 Informations sur les serveurs proxy 
Propriété
Description
Nom
Nom unique affecté à la ressource de serveur proxy. Ce nom est un lien permettant d'accéder à la page Visualiser le serveur proxy - Configuration. Reportez-vous à Modification des serveurs proxy.
Options
  • Cliquez sur l'icône Renommer pour renommer la ressource de serveur proxy.
  • Cliquez sur l'icône Supprimer pour supprimer un serveur proxy spécifique. Reportez-vous à Suppression de serveurs proxy.
