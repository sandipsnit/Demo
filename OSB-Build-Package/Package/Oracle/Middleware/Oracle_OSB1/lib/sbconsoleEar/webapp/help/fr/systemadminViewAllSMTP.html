Visualisation des serveurs SMTP

Utilisez cette page pour désigner un serveur SMTP par défaut (reportez-vous à Configuration d'un serveur SMTP par défaut) et visualiser la liste des serveurs SMTP.

Les ressources de serveur SMTP sont utilisées lors de la configuration des ressources de destination d'alertes et des services fonctionnels basés sur le transport de messagerie électronique. Reportez-vous à Ajout de destinations d'alertes et à la page Créer/Modifier un service fonctionnel - Courrier électronique - Configuration du transport.

Le tableau 30-3 présente les informations relatives aux serveurs SMTP.

Tableau 30-3 Informations sur les serveurs SMTP
Propriété
Description
Nom
Nom unique affecté à la ressource de serveur SMTP. Ce nom est un lien permettant d'accéder à la page Visualiser le serveur SMTP - Configuration. Reportez-vous à Modification des serveurs SMTP.
URL du serveur
URL pointant sur le serveur SMTP. Reportez-vous à Ajout de serveurs SMTP.
Options
  • Cliquez sur l'icône Renommer pour renommer la ressource de serveur SMTP.
  • Cliquez sur l'icône Supprimer pour supprimer un serveur SMTP spécifique. Reportez-vous à Suppression de serveurs SMTP.
