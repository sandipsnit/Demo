Présentation de l'utilisation des paramètres de transport dans la console de test à l'exécution

Les sections Configuration des données de test des services proxy et Configuration des données de test des services fonctionnels expliquent comment configurer les valeurs des en-têtes de transport, les métadonnées de transport et les données de sécurité relatives au transport pour les demandes sortantes lorsque vous testez les services proxy ou fonctionnels dans la console de test. Cependant, il se peut que les spécifications que vous créez dans la console de test ne soient pas toutes respectées à l'exécution. Autrement dit, les valeurs de certains en-têtes ou de certaines métadonnées sont écrasées, ou Oracle Service Bus ne les prend pas en compte à l'exécution lors du test. Les en-têtes et les métadonnées pour lesquels il existe des restrictions sont décrits dans le tableau 32-10.
