Exécution du test XQuery

Vous pouvez modifier et tester une action dans le flux de messages à l'aide de l'éditeur d'expressions XQuery, de l'éditeur de conditions XQuery et de l'éditeur d'expressions XPath. Les tests réalisés au moyen des éditeurs d'expressions XQuery et de conditions XQuery sont similaires. Le scénario est toutefois différent pour l'éditeur d'expressions XPath car il n'utilise qu'une seule entrée.

Remarque : pour que le test XQuery fonctionne, vous devez désactiver les mécanismes de blocage des fenêtres instantanées dans votre navigateur Web. Si des barres d'outils apparaissent dans le navigateur Internet Explorer, vous devrez peut-être désactiver ces mécanismes de blocage des fenêtres instantanées à partir du menu Options, ainsi que toutes les barres d'outils configurées pour les bloquer.

  • Utilisation de l'éditeur d'expressions XPath
