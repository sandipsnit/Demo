Test des transformations

Vous pouvez tester les transformations après avoir activé une session ou au cours d'une session, afin de vous assurer que le comportement des ressources est celui attendu. Vous devez activer la session pour pouvoir tester l'exécution, sinon le test est effectué lors de la conception à l'aide de vos modifications locales.

Vous pouvez tester les transformations suivantes :

  • Test des transformations MFL
  • Test des transformations XSLT
  • Tests des transformations XQuery
