Cette section est initialement réduite si la console de test n'a pas modifié le message de demande. Elle est initialement développée pour les services SOAP configurés via l'onglet Panneau ou si la sécurité WS a été appliquée.

Document de réponse

Dans le cas d'un service SOAP avec une réponse de sécurité WS, cette section contient deux messages SOAP. Le premier correspond au message sécurisé tel qu'il est reçu par le client. Le second est le message en clair correspondant.
Métadonnées de la réponse
Métadonnées renvoyées avec la réponse au message.

Remarque : le message SOAP sécurisé est affiché avec mise en forme, par exemple, avec des caractères non imprimables supplémentaires. Etant donné que ces derniers peuvent avoir une incidence sur la sémantique du document, ce message SOAP ne peut pas toujours être utilisé en tant que données de littéral. Par exemple, les signatures numériques sont sensibles aux caractères non imprimables et peuvent de ce fait devenir non valides.
