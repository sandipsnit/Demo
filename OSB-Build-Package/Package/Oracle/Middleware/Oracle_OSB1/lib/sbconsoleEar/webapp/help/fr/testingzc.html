Test des transformations MFL

Un fichier MFL (Message Format Language) est un document XML spécialisé utilisé pour décrire la mise en page des données binaires. Les ressources MFL prennent en charge deux transformations. Chaque transformation n'accepte qu'une entrée et fournit une sortie unique : XML-binaire ou binaire-XML.

Le tableau 32-5 décrit les options de configuration MFL.

Tableau 32-5 Configuration des options MFL 
Section
Description
Nom
Nom de la ressource testée.
Transformations prises en charge
Pour choisir une transformation spécifique, sélectionnez son nom.
Actions de la console de test
Exécuter
Applique la transformation.
Réinitialiser
Réinitialise le champ d'entrée (dans le cas d'une transformation du format XML au format binaire, l'exemple de document XML est réinitialisé).
Fermer
Annule l'opération en cours.
Configuration des transformations MFL
Entrée XML
Obligatoire pour les transformations du format XML au format binaire :
Le schéma XML pour le document MFL peut être déduit. Un exemple de document XML est automatiquement saisi dans le champ de texte.
L'entrée XML peut être basée sur un fichier ou sur un texte. Le référencement d'un fichier pour entrée est prioritaire sur l'entrée textuelle. Sélectionnez le fichier à utiliser dans votre test.
Entrée binaire
Obligatoire pour les transformations du format binaire au format XML :
L'entrée binaire peut être basée sur un fichier ou sur un texte. Le référencement d'un fichier pour entrée est prioritaire sur l'entrée textuelle. Sélectionnez le fichier à utiliser dans votre test.

Vous pouvez tester la conception ou l'exécution.

  1. Pour tester l'exécution, cliquez sur Activer. N'activez pas la session pour tester la conception.
  2. Sélectionnez Navigateur Web de ressources > MFL pour afficher la page Récapitulatif des fichiers MFL.
  3. Sous Actions, cliquez sur l'icône Lancer la console de test correspondant à la ressource à tester. La page Test de ressource apparaît.
  4. Sélectionnez la transformation à tester.
  5. Configurez les données de test de la ressource MFL. Pour plus d'informations, reportez-vous au tableau 32-5.
  6. Cliquez sur Exécuter. La page Test de ressource affiche les résultats.
  7. Pour réexécuter le test, cliquez sur Précédent. Vous pouvez fermer la console de test, puis modifier et tester à nouveau la ressource.
