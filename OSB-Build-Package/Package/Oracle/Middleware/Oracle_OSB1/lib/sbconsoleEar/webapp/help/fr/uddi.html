UDDI

Les registres UDDI (Universal Description, Discovery and Integration) permettent aux entreprises de partager des services Web. Ils fournissent une structure dans laquelle vous pouvez classer votre business et ses services, ainsi que les détails techniques sur les services à exposer.

Pour pouvoir publier un service dans un registre, vous devez connaître le type du service et la structure de données le représentant dans le registre. Une entrée de registre contient certaines propriétés qui lui sont associées ; les types de ces propriétés sont définis lors de la création du registre. Vous pouvez publier votre service dans un registre et le rendre disponible à d'autres organisations. Les services proxy développés dans Oracle Service Bus peuvent être publiés dans un registre UDDI. Oracle Service Bus peut interagir avec tous les registres UDDI compatibles avec la version 3.0.

Reportez-vous également à "UDDI" dans le manuel Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus à l'adresse suivante : http://www.oracle.com/pls/as111170/lookup?id=OSBAG392.
