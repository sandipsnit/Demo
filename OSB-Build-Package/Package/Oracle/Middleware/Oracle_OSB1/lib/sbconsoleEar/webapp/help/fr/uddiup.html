Modification de registres UDDI

  1. Si vous ne l'avez pas déjà fait, cliquez sur Créer afin de créer une session ou cliquez sur Modifier pour accéder à une session existante. Reportez-vous à Utilisation du centre des modifications.
  2. Sélectionnez Administration du système > Registres UDDI.
  3. Dans la liste des registres disponibles du panneau Registres UDDI, cliquez sur le nom du registre à modifier.

    4. Vous pouvez également rechercher un registre spécifique à l'aide de l'option Rechercher. Reportez-vous à Visualisation des registres UDDI configurés.

    La page Configuration de registre UDDI affiche les propriétés de configuration qui définissent le registre, ainsi que les informations propres au registre répertoriées dans le tableau 29-2.

    Tableau 29-2 Informations sur le registre UDDI 
    Propriété
    Description
    Dernière modification par
    Utilisateur ayant apporté la dernière modification de configuration à cette ressource de registre UDDI.
    Dernière modification le
    Date et heure de la dernière modification de configuration. Cliquez sur le lien de date et d'heure pour visualiser l'historique des modifications apportées à cette ressource. Reportez-vous à Page Visualiser l'historique des modifications.
    Références
    Nombre d'objets référencés par le registre UDDI. Lorsque des références existent, cliquez sur le lien numérique pour visualiser la liste des références en question. Reportez-vous à Visualisation des références aux ressources.
    Référencé par
    Nombre d'objets référençant le registre UDDI. Lorsque des références existent, cliquez sur le lien numérique pour visualiser la liste des objets en question. Reportez-vous à Visualisation des références aux ressources.
    Description
    Description du registre.

  4. Au bas de la page, cliquez sur Modifier.
  5. Sur la page Configuration UDDI - Modifier un registre, modifiez les paramètres de configuration du registre.

    6. Ces propriétés sont décrites dans le tableau 29-1. Vous ne pouvez pas modifier le nom du registre.

  6. Cliquez sur Enregistrer pour valider les mises à jour dans la session en cours.
  7. Pour terminer la session et déployer la configuration lors de l'exécution, cliquez sur Activer sous Centre des modifications.
