WSDL

Un WSDL (Web Service Definition Language) représente la description formelle d'un service Web. Dans Oracle Service Bus, il décrit un service proxy ou un service fonctionnel. Un WSDL permet d'indiquer ce qu'un service Web peut faire, où il réside et comment l'appeler.

Vous pouvez baser les services SOAP et XML sur une ressource WSDL existante. Un document WSDL est disponible pour les services proxy et fonctionnels pour chaque transport. Reportez-vous à la page Créer/Modifier un service proxy - Configuration générale et à Création et configuration de services fonctionnels.

Pour plus d'informations, reportez-vous à "How WSDL is Used in Oracle Service Bus" dans le manuel Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus à l'adresse suivante : http://www.oracle.com/pls/as111170/lookup?id=OSBAG114.
