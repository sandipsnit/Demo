Ajout de ressources WSDL

  1. Si vous ne l'avez pas déjà fait, cliquez sur Créer afin de créer une session ou cliquez sur Modifier pour accéder à une session existante. Reportez-vous à Utilisation du centre des modifications.
  2. Sélectionnez Explorateur de projets, puis choisissez le projet ou le dossier auquel ajouter le WSDL. La page Vue du projet/Vue de dossier apparaît.
  3. Dans la liste déroulante Créer une ressource, sélectionnez WSDL sous Interface pour afficher la page Créer une ressource WSDL.
  4. Dans le champ Nom de la ressource, entrez ou modifiez un nom unique pour ce WSDL. Il s'agit d'un champ obligatoire.

    Pour obtenir des conseils sur la dénomination, suivez les restrictions concernant la dénomination des ressources.
  5. Dans le champ Description de la ressource, entrez la description du WSDL.
  6. Dans le champ WSDL, effectuez l'une des opérations suivantes :
    • Saisissez le texte du nouveau WSDL.
    • Cliquez sur Parcourir pour localiser et importer un WSDL existant.
    • Copiez et collez le texte d'un WSDL existant dans ce champ.

      • Ce champ est obligatoire.

  7. Cliquez sur Enregistrer. Le nouveau WSDL est enregistré dans la session en cours.

    8. Lorsque vous cliquez sur Enregistrer, le système affiche les références non résolues du nouveau WSDL, le cas échéant. Reportez-vous à Visualisation des références WSDL non résolues.

  8. Pour terminer la session et déployer la configuration lors de l'exécution, cliquez sur Activer sous Centre des modifications.
