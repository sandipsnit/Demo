Résolution des références WSDL non résolues

La page Modifier les références d'une ressource WSDL permet de résoudre les références WSDL non résolues via la configuration de la mise en correspondance des références WSDL, telles que les imports WSDL et les imports de schéma XML. Une ressource WSDL peut également présenter une référence non résolue à une stratégie WS.

Résolution d'une référence WSDL ou de schéma XML non résolue
  1. Si vous ne l'avez pas déjà fait, cliquez sur Créer afin de créer une session ou cliquez sur Modifier pour accéder à une session existante. Reportez-vous à Utilisation du centre des modifications.
  2. Sélectionnez Navigateur Web de ressources > WSDL.
  3. Sur la page Récapitulatif des ressources WSDL, dans la colonne Nom WSDL, cliquez sur le nom WSDL approprié afin de visualiser les détails de la référence non résolue. La page Visualiser les détails du WSDL apparaît.
  4. Cliquez sur Modifier les références pour afficher la page Modifier les références d'une ressource WSDL.
  5. Dans le champ Type de ressource, sélectionnez WSDL ou Schéma XML.
  6. Cliquez sur Parcourir. Selon le type de ressource, le navigateur Web WSDL ou le navigateur Web de schémas XML apparaît.
  7. Dans le navigateur Web WSDL ou le navigateur Web de schémas XML, sélectionnez une ressource WSDL ou un schéma XML, puis choisissez une définition dans le panneau Définitions.
  8. Cliquez sur Soumettre. La ressource WSDL ou le schéma XML que vous avez sélectionné apparaît dans le champ Nom de la ressource.
  9. Cliquez sur Enregistrer pour résoudre la référence.
  10. Sur la page Visualiser les détails du WSDL, cliquez sur OK. L'état de la ressource WSDL est défini sur Valide.
Résolution d'une référence de stratégie WS non résolue
  1. Si vous ne l'avez pas déjà fait, cliquez sur Créer afin de créer une session ou cliquez sur Modifier pour accéder à une session existante. Reportez-vous à Utilisation du centre des modifications.
  2. Sélectionnez Navigateur Web de ressources > WSDL.
  3. Sur la page Récapitulatif des ressources WSDL, dans la colonne Nom WSDL, cliquez sur le nom WSDL approprié afin de visualiser les détails de la référence non résolue. La page Visualiser les détails du WSDL apparaît.
  4. Cliquez sur Modifier les références pour afficher la page Modifier les références d'une ressource WSDL.
  5. Dans le champ Nom de la stratégie WS, cliquez sur Parcourir. Le navigateur Web de stratégies apparaît.
  6. Dans le navigateur Web de stratégies, sélectionnez une stratégie WS.
  7. Cliquez sur Soumettre. La stratégie WS que vous avez sélectionnée apparaît dans le champ Nom de la stratégie WS.
  8. Cliquez sur Enregistrer pour résoudre la référence.
  9. Sur la page Visualiser les détails du WSDL, cliquez sur OK. L'état de la ressource WSDL est défini sur Valide.
