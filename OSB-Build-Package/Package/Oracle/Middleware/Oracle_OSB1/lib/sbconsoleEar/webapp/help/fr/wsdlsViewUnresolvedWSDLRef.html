Visualisation des références WSDL non résolues

La page Visualiser les références WSDL non résolues affiche les informations répertoriées dans le tableau 6-5.

Tableau 6-5 Détails des références WSDL non résolues 
Propriété
Description
Nom WSDL
Nom affecté à la référence WSDL. Cliquez sur le nom d'une référence WSDL spécifique pour visualiser les détails la concernant.
Espace de noms
Adresse du WSDL. Elle est construite selon le format suivant :
http://example.com/example/example

Pour localiser une référence spécifique, procédez comme suit :

  • Re-triez la liste. Cliquez sur un nom de colonne souligné. Les flèches vers le haut et vers le bas indiquent l'ordre de tri. Cliquez sur le nom de la colonne pour modifier l'ordre de tri.
  • Faites défiler les pages. Utilisez les contrôles de page au-dessus ou au-dessous de la·table. Accédez à une page en sélectionnant son·numéro ou en utilisant les flèches pour accéder à la page suivante ou précédente, à la première ou à la dernière page.
