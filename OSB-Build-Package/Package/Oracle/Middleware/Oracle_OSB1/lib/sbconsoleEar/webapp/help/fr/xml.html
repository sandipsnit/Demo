Documents XML

Utilisez les ressources de document XML pour stocker les fichiers XML à utiliser dans les configurations de service proxy ou fonctionnel. Par exemple, créez des ressources de document XML pour les fichiers de mapping TopLink ou EclipseLink nécessaires dans les services fonctionnels ou proxy JCA qui communiquent avec les systèmes conformes JCA.

Pour plus d'informations sur les services JCA, reportez-vous au chapitre 12, Liaisons JCA.

Cette section contient les rubriques suivantes :

  • Localisation de documents XML
  • Ajout de documents XML
  • Modification de documents XML
  • Suppression de documents XML
