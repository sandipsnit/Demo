Ajout de documents XML

La méthode la plus simple de création de documents XML dans Oracle Service Bus consiste à les importer globalement. Par exemple, si vous importez globalement les ressources JCA que vous avez créées (fichier .jca, WSDL associé, et fichier de mapping TopLink ou EclipseLink), Oracle Service Bus génère automatiquement des ressources de document XML à partir des fichiers de mapping et gère les dépendances entre les fichiers de ressources.

Pour plus d'informations sur l'import, reportez-vous à Chargement de ressources à partir d'un fichier ZIP, Chargement de ressources à partir d'une URL et Import de ressources.

Si vous n'effectuez pas d'import global, vous pouvez créer manuellement les documents XML comme suit :

  1. Si ce n'est pas déjà fait, cliquez sur Créer pour créer une session ou sur Modifier pour indiquer une session existante. Reportez-vous à Utilisation du centre des modifications.
  2. Sélectionnez Explorateur de projets, puis choisissez le projet ou le dossier auquel ajouter le document XML. La page Vue du projet/Vue de dossier apparaît.
  3. Dans le champ Créer une ressource, sélectionnez Document XML pour afficher la page Créer un document XML.
  4. Dans le champ Fichier XML, cliquez surParcourir, puis sélectionnez le fichier XML à utiliser.

    5. Le champ Nom de la ressource est automatiquement rempli avec le nom du fichier (sans son extension).

    Au lieu de sélectionner un fichier, vous pouvez copier et coller le contenu d'un fichier XML dans la zone de texte du champ Fichier XML.

  5. Dans le champ Nom de la ressource, entrez ou modifiez le nom du document XML. Les noms de document XML doivent être uniques.

    Pour obtenir des conseils sur la dénomination, suivez les restrictions concernant la dénomination des ressources.
  6. Dans le champ Description de la ressource, entrez la description du document XML.
  7. Cliquez sur Enregistrer. Le nouveau document XML est enregistré dans la session en cours.
  8. Pour terminer la session et déployer la configuration lors de l'exécution, cliquez sur Activer sous Centre des modifications.
