Localisation de documents XML

La page Récapitulatif des documents XML répertorie les documents XML que vous avez ajoutés au domaine Oracle Service Bus.

  1. Effectuez l'une des opérations suivantes :
    • Sélectionnez Explorateur de projets pour afficher la page Vue des projets ou la page Vue du projet/Vue de dossier. Parcourez ensuite les projets et dossiers afin d'accéder au document XML.
    • Sélectionnez Navigateur Web de ressources > Documents XML. La page Récapitulatif des documents XML affiche les informations répertoriées dans le tableau 13-1. Pour obtenir une description plus détaillée des propriétés, reportez-vous à Modification de documents XML.
  2. Pour localiser un document XML spécifique, procédez comme suit :
    • Pour limiter le nombre d'éléments dans la liste, vous pouvez filtrer par nom et chemin avant de cliquer sur Rechercher.

      • Le chemin est constitué du nom de projet et du nom des dossiers dans lesquels la ressource réside.

    • Cliquez sur Tout visualiser pour enlever les filtres de recherche et afficher tous les documents XML.

      • Tableau 13-1 Détails des documents XML 
      Propriété
      Description
      Document XML
      Nom unique affecté au document XML. Ce nom est un lien permettant d'accéder à la page Visualiser le document XML. Reportez-vous à Modification de documents XML.
      Chemin
      Le chemin est constitué du nom de projet et du nom du dossier dans lequel le document XML réside. Il s'agit d'un lien permettant d'accéder au projet ou au dossier contenant cette ressource. Reportez-vous à Qualification des noms de ressource à l'aide de projets et de dossiers.
      Options
      Cliquez sur l'icône Supprimer pour supprimer un document XML spécifique. Reportez-vous à Suppression de documents XML.
