Modification de documents XML

La page Modifier un document XML permet de visualiser et de modifier les détails d'une liaison JCA.

  1. Si vous ne l'avez pas déjà fait, cliquez sur Créer afin de créer une session ou cliquez sur Modifier pour accéder à une session existante. Reportez-vous à Utilisation du centre des modifications.
  2. Localisez le document XML, comme décrit dans Localisation de documents XML.
  3. Cliquez sur le nom du document XML. La page Visualiser le document XML affiche les informations répertoriées dans le tableau 13-2.

    4. Tableau 13-2 Détails des documents XML 
    Propriété
    Description
    Dernière modification par
    Utilisateur ayant créé le document XML ou l'ayant importé dans la configuration.
    Dernière modification le
    Date et heure auxquelles l'utilisateur a créé le document XML ou l'a importé dans la configuration. Cliquez sur le lien de date et d'heure pour visualiser l'historique des modifications apportées à cette ressource. Reportez-vous à Page Visualiser l'historique des modifications.
    Références
    Nombre d'objets référencés par le document XML. Lorsque des références existent, cliquez sur le lien numérique pour visualiser la liste des objets en question. Reportez-vous à Visualisation des références aux ressources.
    Référencé par
    Nombre d'objets référençant le document XML. Lorsque des références existent, cliquez sur le lien numérique pour visualiser la liste des objets en question. Par exemple, si le fichier XML est un fichier TopLink référencé par la liaison JCA, cette dernière est répertoriée comme référence lorsque vous cliquez sur ce lien. Reportez-vous à Visualisation des références aux ressources.
    Description
    Description du document XML, le cas échéant.

    La vue en lecture seule du document XML est affichée par défaut.

  4. Pour modifier le document XML, cliquez sur Modifier. Pour obtenir la description de ces champs, reportez-vous à Ajout de documents XML.
  5. Apportez les modifications souhaitées et cliquez sur Enregistrer pour valider les mises à jour dans la session en cours.
  6. Pour terminer la session et déployer la configuration lors de l'exécution, cliquez sur Activer sous Centre des modifications.
