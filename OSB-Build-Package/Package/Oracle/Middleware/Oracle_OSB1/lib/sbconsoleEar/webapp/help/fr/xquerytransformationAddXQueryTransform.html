Ajout de transformations XQuery

  1. Si vous ne l'avez pas déjà fait, cliquez sur Créer afin de créer une session ou cliquez sur Modifier pour accéder à une session existante. Reportez-vous à Utilisation du centre des modifications.
  2. Sélectionnez Explorateur de projets pour afficher la page Vue des projets ou la page Vue du projet/Vue de dossier.
  3. Accédez au projet ou dossier auquel ajouter la transformation XQuery.
  4. Dans la liste Créer une ressource, sélectionnez XQuery. La page Créer une ressource XQuery apparaît.
  5. Dans le champ Nom de la ressource, entrez un nom unique pour cette transformation XQuery.

    Pour obtenir des conseils sur la dénomination, suivez les restrictions concernant la dénomination des ressources.
  6. Dans le champ Description de la ressource, entrez la description de la transformation XQuery.
  7. Dans le champ XQuery, effectuez l'une des opérations suivantes :
    • Saisissez le texte de la nouvelle transformation XQuery.
    • Cliquez sur Parcourir pour localiser et importer une transformation XQuery existante.
    • Copiez et collez le texte d'une transformation XQuery existante dans ce champ.
  8. Cliquez sur Enregistrer pour valider les mises à jour dans la session en cours.
  9. Pour terminer la session et déployer la configuration lors de l'exécution, sélectionnez Activer sous le centre des modifications.
