Localisation de transformations XQuery

  1. Effectuez l'une des opérations suivantes :
    • Sélectionnez Explorateur de projets pour afficher la page Vue des projets ou la page Vue du projet/Vue de dossier. Parcourez ensuite les projets et dossiers afin d'accéder à la ressource XQuery.
    • Sélectionnez Navigateur Web de ressources > Ressources XQuery. La page Récapitulatif des ressources XQuery affiche les informations répertoriées dans le tableau 10-1.
  2. Pour restreindre le nombre d'éléments dans la liste, vous pouvez filtrer par nom, par chemin ou les deux. Dans les champs Nom et Chemin, sous Rechercher, entrez le nom et/ou le chemin des cibles, puis cliquez sur le bouton Rechercher.

    3. Le chemin est constitué du nom de projet et du nom du dossier dans lequel la ressource XQuery réside.

    Les caractères génériques * et ? sont autorisés. La recherche distingue les majuscules des minuscules.

    Cliquez sur Tout visualiser pour afficher toutes les ressources XQuery du domaine. Cette opération permet d'effacer les paramètres de recherche de la recherche précédente.

    Tableau 10-1 Détails des transformations XQuery 
    Propriété
    Description
    Nom XQuery
    Nom affecté à la transformation XQuery. Cliquez sur ce nom pour que la page Visualiser les détails de la transformation XQuery apparaisse.
    Reportez-vous à Modification de transformations XQuery.
    Chemin
    Le chemin est constitué du nom de projet et du nom du dossier dans lequel la ressource XQuery réside. Par exemple, MyProject/MyFolder/MyResource.
    Cliquez sur le chemin d'une ressource XQuery pour afficher la page Vue du projet/Vue de dossier correspondante.
    Actions
    Cliquez sur l'icône Lancer la console de test pour appeler la console de test, qui permet de valider et tester la conception de vos services et transformations. Pour les transformations XQuery, vous pouvez utiliser cette console que vous soyez dans une session ou en dehors d'une session. Pour plus d'informations, reportez-vous à Test des transformations.
    Options
    Si vous ne l'avez pas déjà fait, cliquez sur Créer afin de créer une session ou cliquez sur Modifier pour accéder à une session existante. Reportez-vous à Utilisation du centre des modifications.
    Cliquez sur l'icône Supprimer pour supprimer la ressource XQuery. Si cette ressource est référencée par d'autres, l'icône Avertissement relatif à la suppression apparaît. Vous pouvez supprimer la ressource à condition de confirmer l'opération lorsqu'un avertissement s'affiche. Toutefois, des conflits peuvent survenir, les références à la ressource supprimée n'étant pas résolues. Pour plus d'informations, reportez-vous à Suppression d'une transformation XQuery.
