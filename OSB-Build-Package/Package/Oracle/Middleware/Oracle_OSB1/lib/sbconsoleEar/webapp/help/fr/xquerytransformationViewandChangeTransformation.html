Modification de transformations XQuery

  1. Si vous ne l'avez pas déjà fait, cliquez sur Créer afin de créer une session ou cliquez sur Modifier pour accéder à une session existante. Reportez-vous à Utilisation du centre des modifications.
  2. Localisez la transformation XQuery, comme décrit dans Localisation de transformations XQuery.
  3. Cliquez sur le nom de la transformation XQuery. La page Visualiser les détails de la transformation XQuery affiche les informations répertoriées dans le tableau 10-2.

    4. Tableau 10-2 Détails des ressources de transformation XQuery
    Propriété
    Description
    Dernière modification par
    Utilisateur ayant créé la transformation XQuery ou l'ayant importée dans la configuration.
    Dernière modification le
    Date et heure auxquelles l'utilisateur a créé la transformation XQuery ou l'a importée dans la configuration. Cliquez sur le lien de date et d'heure pour visualiser l'historique des modifications apportées à cette ressource. Reportez-vous à Page Visualiser l'historique des modifications.
    Références
    Nombre d'objets référencés par la transformation XQuery. Lorsque des références existent, cliquez sur le lien numérique pour visualiser la liste des objets en question. Pour plus d'informations, reportez-vous à Visualisation des références aux ressources.
    Référencé par
    Nombre d'objets référençant la transformation XQuery. Lorsque des références existent, cliquez sur le lien numérique pour visualiser la liste des objets en question. Par exemple, si vous affectez cette transformation XQuery à une variable dans le flux de messages d'un service proxy spécifique, ce dernier est répertorié comme référence lorsque vous cliquez sur ce lien. Pour plus d'informations, reportez-vous à Visualisation des références aux ressources.
    Description
    Description de la transformation XQuery, le cas échéant.
    XQuery
    Texte complet de la transformation XQuery.
    Variables XQuery
    Variables XQuery sélectionnées pour cette transformation XQuery.

  4. Pour modifier les champs, cliquez sur Modifier. Pour obtenir la description de ces champs, reportez-vous à Ajout de transformations XQuery.

    5. Vous ne pouvez pas modifier le champ Nom de la ressource.

  5. Cliquez sur Enregistrer pour valider les mises à jour dans la session en cours.
  6. Pour terminer la session et déployer la configuration lors de l'exécution, cliquez sur Activer sous Centre des modifications.
