Modification de transformations XSL

  1. Si vous ne l'avez pas déjà fait, cliquez sur Créer afin de créer une session ou cliquez sur Modifier pour accéder à une session existante. Reportez-vous à Utilisation du centre des modifications.
  2. Localisez la transformation XSL, comme décrit dans Localisation de transformations XSL.
  3. Cliquez sur le nom de la transformation XSL. La page Visualiser les détails de la transformation XSL affiche la vue structurelle de la ressource XSLT par défaut. Elle affiche les informations répertoriées dans le tableau 14-2.

    4. Tableau 14-2 Détails des ressources de transformation XSL 
    Propriété
    Description
    Dernière modification par
    Utilisateur ayant créé la transformation XSL ou l'ayant importée dans la configuration.
    Dernière modification le
    Date et heure auxquelles l'utilisateur a créé la transformation XSL ou l'a importée dans la configuration. Cliquez sur le lien de date et d'heure pour visualiser l'historique des modifications apportées à cette ressource. Reportez-vous à Page Visualiser l'historique des modifications.
    Références
    Nombre d'objets référencés par la transformation XSL. Lorsque des références existent, cliquez sur le lien numérique pour visualiser la liste des objets en question. Pour plus d'informations, reportez-vous à Visualisation des références aux ressources.
    Référencé par
    Nombre d'objets référençant la transformation XSL. Lorsque des références existent, cliquez sur le lien numérique pour visualiser la liste des objets en question. Par exemple, si vous affectez cette transformation XSL à une variable dans le flux de messages d'un service proxy spécifique, ce dernier est répertorié comme référence lorsque vous cliquez sur ce lien. Pour plus d'informations, reportez-vous à Visualisation des références aux ressources.
    Description
    Description de la transformation XSL.
    Espace de noms cible
    Espace de noms utilisé pour qualifier les définitions incluses dans la transformation XSL.
    Dépendances XSLT
    Les attributs et groupes associés à la ressource XSLT sont les suivants :
    • Emplacement de la ressource XSLT
    Etat
    Statut de la transformation XSL :
    • Valide

      • Tous les emplacements des transformations XSL inclus dans la transformation XSL en cours sont indiqués et valides. En outre, tous les emplacements des transformations XSL imbriquées (transformations XSL incluses par la suite dans les transformations XSL incluses) sont spécifiés et valides.

    • Non valide

      • Un des emplacements des transformations XSL incluses dans la transformation XSL en cours n'est pas indiqué ou n'est pas valide. En outre, il se peut que l'une des transformations XSL imbriquées (transformations XSL incluses par la suite dans les transformations XSL incluses) ne soit pas spécifiée ou valide. Pour résoudre une transformation XSL non valide, cliquez sur Modifier les références. Pour plus d'informations, reportez-vous à Résolution des références de transformation XSL non résolues.

  4. Cliquez sur Vue textuelle pour afficher une vue textuelle des détails de la transformation XSL. La vue textuelle comprend les informations répertoriées dans le tableau 14-3.

    5. Tableau 14-3 Détails textuels des transformations XSL
    Propriété
    Description
    Espace de noms cible
    Espace de noms utilisé pour qualifier les dépendances incluses dans la transformation XSL.
    XSLT
    Texte complet de la transformation XSL.

  5. Pour modifier les champs, cliquez sur Modifier. Pour obtenir la description de ces champs, reportez-vous à Ajout de transformations XSL.

    6. Vous ne pouvez pas modifier le champ Nom de la ressource.

  6. Cliquez sur Enregistrer pour valider les mises à jour dans la session en cours.
