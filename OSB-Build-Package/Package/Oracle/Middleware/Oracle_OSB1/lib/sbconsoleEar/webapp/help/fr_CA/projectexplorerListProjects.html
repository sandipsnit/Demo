Projects View page

The Projects View page lists the projects defined in the current domain. Table 3-1 describes the options and controls on the page.

Table 3-1 Projects View Page - Options and Controls 
Option or Control
Description
Enter New Project Name
To create a new project, enter a name in the text field and click Add Project. See Adding Projects.
See also Naming Projects and Folders.
Name
Click a project name to display the Project/Folder View page for that project.
Options
For any project listed, you can do any of the following:
  • Click the Rename Project icon to rename the project. See Renaming Projects.
  • Click the Move Project icon to move the project. See Moving Projects (Converting to a Folder).
  • Click the Clone Project icon to clone the project. See Cloning Projects.
  • Click the Delete Project icon to delete the project. A Deletion Warning icon is displayed when other resources reference this resource. You can delete this resource with a warning confirmation. This might result in conflicts due to unresolved references to the deleted resource. See Deleting Projects.
When you perform any of these actions, the result is saved in the current session. To end the session and deploy the configuration to the run time, click Activate under Change Center.
