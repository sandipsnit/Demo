Load Resources - Review Loaded Resources page

The Load Resources - Review Loaded Resources page displays details about resources being imported from a Zip or other file, as documented in Loading Resources from a Zip File or Loading Resources from a URL.

Table 3-6 describes the options and controls on the page.

Table 3-6 Load Resources: Review Loaded Resources page - Options and Controls 
Options and Controls
Description
File Name
The file name of the resource, including the file extension.
Name
The name of the resource.
Path
The path of the resource, which is the project and folder where the resource resides.
Resource Type
Lists the supported resource types.

A Warning icon next to file name indicates that the resource type is unknown. (A file without an extension or one that does not have an extension defined in the map is considered unknown.) Files of unknown file types cannot be imported.
