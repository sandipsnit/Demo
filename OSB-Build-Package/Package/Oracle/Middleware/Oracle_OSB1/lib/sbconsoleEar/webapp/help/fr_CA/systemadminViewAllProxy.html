Viewing Proxy Servers

Use this page to view a list of Proxy Servers and the information shown in Table 30-5.

Proxy Server resources are used while configuring the HTTP transport for business services. For more information, see HTTP Transport Configuration Page.

Table 30-5 displays Proxy Server information.

Table 30-5 Proxy Server Information 
Property
Description
Name
The unique name assigned to this Proxy Server resource. The name is a link to the View Proxy Server - Configuration page. See Editing Proxy Servers.
Options
  • Click the Rename icon to rename this Proxy Server resource.
  • Click the Delete icon to delete a specific Proxy Server. See Deleting Proxy Servers.
