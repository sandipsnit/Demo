Aggiunta di MFL

  1. Se questa operazione non è stata già eseguita, fare clic su Crea per creare una nuova sessione o su Modifica per accedere a una sessione esistente. Vedere Utilizzo del Centro modifiche.
  2. Selezionare Explorer progetto, quindi selezionare un progetto o una cartella in cui aggiungere il file MFL. Viene visualizzata la pagina Vista progetto/cartella.
  3. Nell'elenco a discesa Crea risorsa selezionare File MFL in Trasformazione per visualizzare la pagina Crea una nuova risorsa file MFL.
  4. Nel campo Nome risorsa immettere un nome univoco per questo MFL. Questo campo è obbligatorio.

    Vedere Limitazioni di denominazione delle risorse per le istruzioni di denominazione.
  5. Nel campo Descrizione risorsa immettere una descrizione per il file MFL.
  6. Nel campo Testo MFL effettuare una delle operazioni riportate di seguito.
    • Immettere il testo per il nuovo file MFL.
    • Fare clic su Sfoglia per individuare e importare un file MFL esistente.
    • Copiare e incollare in questo campo il testo di un file MFL esistente.

      • Questo campo è obbligatorio.

  7. Fare clic su Salva. La risorsa di file MFL viene creata e salvata nella sessione corrente.
  8. Per terminare la sessione e distribuire la configurazione in fase di esecuzione, fare clic su Attiva in Centro modifiche.
