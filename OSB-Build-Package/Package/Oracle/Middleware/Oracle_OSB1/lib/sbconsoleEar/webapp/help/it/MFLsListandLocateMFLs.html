Individuazione dei file MFL

  1. Effettuare una delle operazioni riportate di seguito.
    • Selezionare Explorer progetto per visualizzare la pagina Vista progetti o la pagina Vista progetto/cartella. Spostarsi tra i progetti e le cartelle per trovare il file MFL.
    • Selezionare Browser risorse > MFL. Nella pagina Riepilogo dei file MFL vengono visualizzate le informazioni mostrate nella Tabella 15-1. Per una descrizione più dettagliata delle proprietà, vedere Modifica di MFL.
  2. Per limitare il numero di elementi nella lista, è possibile filtrarli in base al nome, al percorso o a entrambi. Nei campi Nome e Percorso immettere il nome e il percorso della destinazione o delle destinazioni della ricerca, quindi fare clic su Cerca.

    3. Il percorso è costituito dal nome del progetto e dal nome della cartella in cui il file MFL risiede.

    Fare clic su Vis. tutto per rimuovere i filtri di ricerca e visualizzare tutti i file MFL.

    Tabella 15-1 Dettagli dei file MFL
    Proprietà
    Descrizione
    Nome file MFL
    Il nome univoco assegnato al file MFL. Il nome è un collegamento alla pagina Visualizza dettagli risorse MFL. Vedere Modifica di MFL.
    Percorso
    Il nome del progetto e il nome della cartella in cui il file MFL risiede. Si tratta di un collegamento al progetto o alla cartella che contiene questa risorsa. Vedere Qualificazione dei nomi delle risorse mediante progetti e cartelle.
    Azioni
    Fare clic sull'icona Avvia console di test per richiamare la console di test che è possibile utilizzare per convalidare e sottoporre a test la progettazione dei servizi e delle trasformazioni. Per i file MFL, è possibile utilizzare la console di test all'interno o al di fuori di una sessione. Vedere Test delle trasformazioni.
    Opzioni
    Fare clic sull'icona Elimina per eliminare un file MFL specifico. Vedere Eliminazione di MFL.
