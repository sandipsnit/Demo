Modifica di MFL

  1. Se questa operazione non è stata già eseguita, fare clic su Crea per creare una nuova sessione o su Modifica per accedere a una sessione esistente. Vedere Utilizzo del Centro modifiche.
  2. Individuare il file MFL, come descritto in Individuazione dei file MFL.
  3. Fare clic sul nome del file MFL. Nella pagina Visualizza dettagli risorse MFL vengono visualizzate le informazioni mostrate nella Tabella 15-2.

    4. Tabella 15-2 Dettagli delle risorse MFL 
    Proprietà
    Descrizione
    Autore ultima modifica
    L'utente che ha creato questo file MFL o che lo ha importato nella configurazione.
    Data ultima modifica
    La data e l'ora in cui l'utente ha creato questo file MFL o lo ha importato nella configurazione. Fare clic sul collegamento di data e ora per visualizzare la cronologia delle modifiche di questa risorsa. Vedere la pagina Visualizza cronologia modifiche.
    Riferimenti
    Il numero di oggetti a cui fa riferimento questo file MFL. Se tali riferimenti esistono, fare clic sul collegamento numerico per visualizzare una lista degli oggetti. Vedere Visualizzazione dei riferimenti alle risorse.
    Riferimento di
    Il numero di oggetti che fanno riferimento a questo file MFL. Se tali riferimenti esistono, fare clic sul collegamento numerico per visualizzare una lista degli oggetti. Ad esempio, se questo file MFL è stato selezionato come tipo di messaggio di richiesta o risposta per un servizio business o proxy di messaggistica, il servizio business o proxy viene elencato come riferimento quando si fa clic sul collegamento. Vedere Visualizzazione dei riferimenti alle risorse.
    Descrizione
    La descrizione di questo file MFL, se esistente.
    Testo MFL
    Il testo per questo file MFL.

  4. Per modificare il contenuto di questi campi, fare clic su Modifica. Per le descrizioni dei campi, vedere Aggiunta di MFL.

    5. Non è possibile modificare il campo Nome risorsa.

  5. Fare clic su Salva per eseguire il commit degli aggiornamenti nella sessione corrente.
  6. Per terminare la sessione e distribuire la configurazione in fase di esecuzione, fare clic su Attiva in Centro modifiche.
