Destinazioni alert

Una risorsa di destinazione degli alert recupera una lista di destinatari che possono ricevere le notifiche degli alert da parte di Oracle Service Bus. In contesti standard di monitoraggio del sistema, gli alert generati da Oracle Service Bus assumono rilevanza per un set limitato di utenti.

In Oracle Service Bus, è possibile configurare ogni risorsa di destinazione degli alert per includere un set di destinatari in base a un determinato contesto. Le destinazioni degli alert vengono utilizzate nelle azioni di alert configurate nel flusso dei messaggi e anche dalle regole di alert SLA. Una destinazione degli alert può includere uno o più dei tipi di destinazione seguenti: console (impostazione predefinita), Reporting Data Stream, interrupt SNMP, posta elettronica, coda JMS o argomento JMS.

Per la posta elettronica e le destinazioni JMS, una risorsa di destinazione può includere una lista di indirizzi di posta elettronica o di URI JMS, rispettivamente.

È possibile utilizzare di nuovo le destinazioni degli alert nelle configurazioni degli alert per i servizi. Vedere Aggiunta di azioni di alert.
