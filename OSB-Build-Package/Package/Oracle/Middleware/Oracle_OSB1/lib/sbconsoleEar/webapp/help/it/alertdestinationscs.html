Aggiunta di destinatari di posta elettronica

Prima di aggiungere una destinazione di posta elettronica è necessario configurare un server SMTP (vedere Aggiunta di server SMTP) o una sessione JavaMail in WebLogic Server (vedere "Configura accesso a JavaMail" nella Guida in linea Oracle Fusion Middleware Oracle WebLogic Server Administration Console all'indirizzo http://download.oracle.com/docs/cd/E15523_01/apirefs.1111/e13952/taskhelp/mail/CreateMailSessions.html). Se non vi sono risorse di server SMTP o sessioni di JavaMail disponibili configurate, non è possibile configurare un destinatario di posta elettronica.

  1. Selezionare Invia un alert via posta elettronica dall'elenco a discesa e fare clic su Aggiungi per visualizzare la pagina Aggiungi destinatario posta elettronica.
  2. Nel campo Destinatari posta immettere un destinatario di posta elettronica nel formato: mailto:nomeutente@nomehost.

    3. Questo campo è obbligatorio.

    È possibile specificare più destinatari di posta elettronica immettendo i nomi utente e i nomi host in una lista separati da virgola. Ad esempio, mailto:nomeutente@notehost [,nomeutente_1@nomehost_1]...[,nomeutente_n@nomehost_n].

    Solo al primo destinatario di posta elettronica è necessario anteporre il testo "mailto:".

  3. Nel campo Server SMTP selezionare il nome del server SMTP per la posta elettronica in uscita. Questo campo non è obbligatorio se nel passo successivo viene selezionata una sessione di posta.
  4. Nel campo Sessione posta elettronica selezionare una sessione di posta disponibile. Questo campo non è obbligatorio se nel passo precedente è stato selezionato un server SMTP.
  5. Nel campo Nome mittente specificare il nome di un mittente per la notifica dell'avviso. Questo campo è facoltativo.
  6. Nel campo Indirizzo del mittente specificare un indirizzo di posta elettronica valido. Questo campo è obbligatorio se è stato specificato un valore nel campo "Nome mittente".
  7. Nel campo Nome indirizzo di risposta specificare un nome a cui inviare una risposta. Questo campo è facoltativo.
  8. Nel campo Indirizzo di risposta specificare un indirizzo di posta elettronica a cui è possibile inviare una risposta. Questo campo è obbligatorio se è stato specificato un valore nel campo "Nome indirizzo di risposta".
  9. Nel campo Timeout connessione immettere il numero di secondi di attesa per la risposta di connessione da parte del server prima del timeout. Il valore predefinito è 0.
  10. Nel campo Codifica richiesta immettere un valore di codifica del set di caratteri. Il valore di codifica predefinito è iso-8859-1.
  11. Fare clic su Salva. Il destinatario di posta elettronica viene aggiunto alla lista dei destinatari di posta elettronica per questa destinazione degli alert e salvato nella sessione corrente.
