Aggiunta di destinazioni JMS

  1. Selezionare Invia un alert a una destinazione JMS dall'elenco a discesa e fare clic su Aggiungi per visualizzare la pagina Aggiungi destinazione JMS.
  2. Nel campo URI della destinazione immettere un URI per la destinazione JMS nel formato: jms://host:port/factoryJndiName/destJndiName.

    3. Questo campo è obbligatorio.

    Nota: anche se Oracle WebLogic Server consente l'utilizzo delle barre nei nomi JNDI, ad esempio "myqueues/myqueue", i nomi JNDI contenenti barre interferiscono con il formato URI richiesto da Oracle Service Bus e non sono pertanto utilizzabili. Una soluzione alternativa a questo problema consiste nel definire un server JMS esterno e farvi riferimento nell'URI. Per ulteriori informazioni, vedere "Configure foreign servers" in Oracle Fusion Middleware Oracle WebLogic Server Administration Console Online Help all'indirizzo http://www.oracle.com/pls/as111170/lookup?id=WLACH.

  3. Nel campo Tipo di destinazione selezionare Coda o Argomento.
  4. Nel campo Tipo di messaggio selezionare Byte o Testo.
  5. Nel campo Codifica richiesta immettere un valore di codifica del set di caratteri. Il valore di codifica predefinito è UTF-8.
  6. Fare clic su Salva. La destinazione JMS viene aggiunta alla lista delle destinazioni JMS da includere per questa destinazione degli alert e salvata nella sessione corrente.
