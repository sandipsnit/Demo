Pagina Configurazione trasporto FTP

La pagina Crea/Modifica un servizio business - Configurazione trasporto FTP fa parte di una serie di pagine per la creazione e la modifica di servizi business, come descritto in Creazione e configurazione dei servizi business e Modifica delle configurazioni dei servizi business.

Utilizzare questa pagina per configurare le impostazioni di trasporto per un servizio business che utilizza il protocollo di trasporto FTP. Nella Tabella 18-9 viene descritto come utilizzare questa pagina.

Tabella 18-9 Pagina Crea/Modifica un servizio business - Configurazione trasporto FTP
Opzione
Per creare o modificare...
Autenticazione utente
Selezionare anonimo se l'utente del server FTP è anonimo oppure selezionare utente esterno se l'utente del server FTP è un account configurato esternamente.
Identità (ID di posta elettronica)
Questo campo è disponibile solo se l'opzione Autenticazione utente è impostata su anonimo.
Immettere l'ID di posta per l'utente anonimo.
Account di servizio
Questo campo è disponibile solo se l'opzione Autenticazione utente è impostata su utente esterno.
Immettere l'account di servizio dell'utente anonimo.

Per ulteriori informazioni:

  • Account di servizio
  • Aggiunta di account di servizio
Timeout
Immettere l'intervallo di timeout del socket, in secondi, prima che la connessione venga interrotta. Il valore predefinito è 60 secondi.
Prefisso per il nome file di destinazione
Immettere un prefisso per il nome file con cui il file viene memorizzato sul server remoto.
Non immettere * in questo campo. Questo carattere provoca un'eccezione della fase di esecuzione.
Suffisso per il nome file di destinazione
Immettere un suffisso per il nome file con cui il file viene memorizzato sul server remoto.
Non immettere * in questo campo. Questo carattere provoca un'eccezione della fase di esecuzione.
Modalità di trasferimento
Selezionare ASCII o binario come modalità di trasferimento.
Codifica richiesta
Accettare l'impostazione predefinita UTF-8 come codifica del set di caratteri per le richieste nei trasporti FTP oppure immettere un'altra codifica del set di caratteri.

Al termine della procedura

Fare clic su Avanti per proseguire con la configurazione di questo servizio nella pagina successiva oppure fare clic su Ultimo per esaminare e salvare questa configurazione nella pagina Crea/Modifica un servizio business - Riepilogo.
