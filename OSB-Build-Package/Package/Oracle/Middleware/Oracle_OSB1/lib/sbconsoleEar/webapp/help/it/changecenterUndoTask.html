Annullamento di task

Utilizzare la pagina Visualizza modifiche alla configurazione per annullare i task effettuati nella configurazione di Oracle Service Bus durante la sessione corrente e per annullare le attivazioni delle sessioni al di fuori di una sessione. Oracle Service Bus permette di annullare più livelli di attivazione di sessione, limitati solo dalle risorse del proprio sistema. Per ulteriori informazioni, vedere Ordine di annullamento dei task.

Nota: se si effettua l'aggiornamento da Oracle Service Bus versione 2.1 alla versione 2.5, non sarà possibile annullare le sessioni attivate prima dell'aggiornamento. Le sessioni attivate dopo l'aggiornamento saranno disponibili per l'annullamento nella cronologia di attivazione delle sessioni.
Annullamento di un task in una sessione
  1. Accertarsi di trovarsi all'interno di una sessione.
  2. Fare clic su Visualizza modifiche in Centro modifiche.
  3. Nella colonna Opzioni di un task specifico fare clic sull'icona Annulla.

    4. Il task viene annullato.

    È possibile annullare qualsiasi modifica nella sessione corrente. Se tuttavia dopo questa operazione si cambia idea, è possibile annullare l'operazione di annullamento oppure se si cambia di nuova idea è possibile annullare l'operazione di annullamento che aveva annullato un precedente annullamento e così via.

Annullamento di una sessione attivata in precedenza
  1. Accertarsi di trovarsi al di fuori di una sessione.
  2. Fare clic su Visualizza modifiche in Centro modifiche.
  3. Effettuare una delle operazioni riportate di seguito.
    • Nella colonna Opzioni di un task specifico fare clic sull'icona Annulla. L'attivazione della sessione viene annullata.
    • Nella colonna Opzioni di un task specifico selezionare l'icona Annulla nella sessione. Viene creata una nuova sessione precompilata con i task necessari per annullare le modifiche attivate nella sessione. Per ulteriori informazioni, vedere Annullamento in una sessione.
