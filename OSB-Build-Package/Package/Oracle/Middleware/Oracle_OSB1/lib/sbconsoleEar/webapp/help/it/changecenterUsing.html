Utilizzo del Centro modifiche

Il modulo Centro modifiche è il punto di partenza per l'utilizzo della console di amministrazione di Oracle Service Bus per effettuare modifiche alla configurazione. Per apportare modifiche alla configurazione utilizzando la console di amministrazione di Oracle Service Bus, è necessario utilizzare il Centro modifiche per avviare una sessione.

  1. Fare clic su Modifica o Crea per iniziare una sessione. Il nome della sessione viene visualizzato in Centro modifiche.
  2. Effettuare le modifiche appropriate nella pagina pertinente della console di amministrazione di Oracle Service Bus.
  3. Fare clic su Salva in ciascuna pagina in cui sono state apportate delle modifiche. Tutte le modifiche effettuate nella sessione corrente vengono salvate.

    4. Fare clic su Elimina in Centro modifiche in qualsiasi momento durante la sessione per eliminare le modifiche finora apportate nella sessione corrente.

  4. Al termine delle modifiche da apportate, fare clic su Attiva in Centro modifiche. Continuare in Attivazione di sessioni.
