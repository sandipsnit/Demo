Visualizzazione di tutte le sessioni

Utilizzare questa pagina per visualizzare tutte le sessioni esistenti nella console di amministrazione di Oracle Service Bus. È possibile visualizzare queste sessioni quando l'utente si trova all'interno di una sessione o al di fuori di una sessione.

È possibile visualizzare tutte le sessioni solo se si sta utilizzando il ruolo Amministratore. Per ulteriori informazioni, vedere "Configuring Administrative Security" in Oracle Fusion Middleware Developer's Guide for Oracle Service Bus all'indirizzo http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539.

  1. Fare clic su Visualizza tutte le sessioni in Centro modifiche. Per ciascuna sessione esistente, nella pagina Visualizza tutte le sessioni vengono visualizzate le informazioni mostrate nella Tabella 2-5.

    2. Tabella 2-5 Dettagli della sessione
    Proprietà
    Descrizione
    Nome sessione
    Il nome della sessione. Il nome è un collegamento alla sessione.
    Utente
    Il nome dell'utente che ha creato la sessione.
    Ora creazione
    La data e l'ora in cui la sessione è stata creata.
    Ultima modifica
    La data e l'ora dell'ultima modifica effettuata durante la sessione.
    N. di oggetti modificati
    Il numero di oggetti che sono stati modificati durante la sessione.

  2. Per passare a un'altra sessione, fare clic sul collegamento nella colonna Nome sessione per quella sessione.
Nota: lo stesso utente collegato con più browser non è supportato. Ciò causa un funzionamento non previsto della console di amministrazione di Oracle Service Bus.
