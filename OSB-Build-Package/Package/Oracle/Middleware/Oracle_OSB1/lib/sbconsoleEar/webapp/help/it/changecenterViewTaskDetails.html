Visualizzazione dei dettagli del task

Utilizzare questa pagina per visualizzare i dettagli di una modifica specifica effettuata nella sessione corrente se ci si trova all'interno della sessione oppure per visualizzare i dettagli di modifiche specifiche effettuate nelle sessioni che sono state attivate se ci si trova al di fuori di una sessione.

  1. Selezionare Visualizza modifiche in Centro modifiche.
  2. Nella pagina Visualizza modifiche alla configurazione fare clic sul nome del task nella colonna Task. Nella pagina Dettagli task vengono visualizzate le informazioni mostrate nella Tabella 2-4.

    3. Tabella 2-4 Dettagli del task
    Proprietà
    Descrizione
    Task
    Il tipo di task eseguito.
    • Crea
    • Aggiorna
    • Elimina
    • Rinomina
    • Sposta
    Tipo di risorsa
    I tipi di risorsa Oracle Service Bus che facevano parte del task.
    Risorsa
    Il nome e il percorso della risorsa. Il percorso è il nome del progetto e della cartella in cui la risorsa risiede.

  3. Fare clic su Indietro per tornare alla pagina Visualizza modifiche alla configurazione.
