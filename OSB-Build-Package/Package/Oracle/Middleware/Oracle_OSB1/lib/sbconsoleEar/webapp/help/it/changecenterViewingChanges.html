Visualizzazione delle modifiche alla configurazione

In questa pagina vengono visualizzate informazioni diverse a seconda che l'utente si trovi all'interno di una sessione (ovvero abbia fatto clic su Modifica nel Centro modifiche) o al di fuori di una sessione.

  • Quando l'utente si trova all'interno di una sessione, nella pagina Visualizza modifiche alla configurazione viene visualizzata la lista delle modifiche alla configurazione effettuate durante la sessione corrente. Vedere Visualizzazione delle modifiche alla configurazione in una sessione.
  • Quando l'utente si trova al di fuori di una sessione, nella pagina Visualizza modifiche alla configurazione viene visualizzata la lista delle modifiche alla configurazione causate dalle attivazioni di sessione precedenti. Vedere Visualizzazione delle modifiche alla configurazione causate dalle attivazioni di sessione.
Visualizzazione delle modifiche alla configurazione in una sessione
  1. Se questa operazione non è stata già eseguita, fare clic su Crea per creare una nuova sessione e apportare modiche alla configurazione corrente. Vedere Utilizzo del Centro modifiche.
  2. Apportare almeno una modifica alla configurazione.
  3. Fare clic su Visualizza modifiche in Centro modifiche. Per ciascuna modifica apportata alla configurazione durante la sessione corrente, nella pagina Visualizza modifiche alla configurazione vengono visualizzate le informazioni mostrate nella Tabella 2-2.

    4. Tabella 2-2 Modifiche alla configurazione nella sessione
    Proprietà
    Descrizione
    Task
    Una descrizione del task che è stato implementato. Il task è un collegamento alla pagina Dettagli task. Vedere Visualizzazione dei dettagli del task.
    Ora di esecuzione
    La data e l'ora in cui il task è stato eseguito.
    Utente
    Il nome dell'utente che ha implementato la modifica.
    Stato task
    Lo stato del task:
    Completato: il task è stato completato.
    Annullato: il task è stato annullato.
    Annullamento completato: l'annullamento è stato completato.
    Annullato da
    Il nome dell'utente che ha annullato il task. In questo campo viene visualizzato Nessuno se il task non è stato annullato.
    Opzioni
    Fare clic sull'icona Annulla per annullare l'esecuzione del task.
    Quando si sta lavorando in una sessione, è possibile annullare i task in qualsiasi ordine.
    Annullando un task, l'oggetto del task ritorna alla stato in cui si trovava prima dell'esecuzione di quel task. Notare, dunque, che verranno annullati anche eventuali task eseguiti sullo stesso oggetto dopo il task annullato. Vedere Annullamento di task.

Visualizzazione delle modifiche alla configurazione causate dalle attivazioni di sessione
  1. Accertarsi di non trovarsi all'interno di una sessione. Vedere Uscita dalle sessioni.
  2. Fare clic su Visualizza modifiche in Centro modifiche. Per ciascuna sessione attivata in precedenza, nella pagina Visualizza modifiche alla configurazione vengono visualizzate le informazioni mostrate nella Tabella 2-3.

    3. Tabella 2-3 Modifiche alla configurazione dalle attivazioni
    Proprietà
    Descrizione
    Task
    Una descrizione della sessione che è stata attivata. Il task è un collegamento alla pagina Dettagli task in cui sono visualizzate le operazioni eseguite nella sessione. Vedere Visualizzazione dei dettagli del task.
    Descrizione
    Una descrizione aggiuntiva della sessione che è stata attivata, se una descrizione è stata immessa quando la sessione è stata attivata utilizzando la pagina Attiva sessione.
    Ora di esecuzione
    La data e l'ora in cui la sessione è stata attivata.
    Utente
    Il nome dell'utente che ha attivato la sessione.
    Stato task
    Lo stato della sessione:
    Attivato: la sessione è stata attivata.
    Parzialmente attivato: visualizzato durante l'attivazione della sessione se uno o più server in un cluster non sono accessibili, impedendo così l'attivazione della sessione su questi server. Quando i server diventano accessibili, saranno attivate su di essi tutte le modifiche non attivate.
    Annullato: la sessione attivata in precedenza è stata annullata e tutte le operazioni eseguite nella sessione sono state annullate.
    Annullamento attivato: l'annullamento è stato attivato.
    In corso: visualizzato quando è in corso l'attivazione di una sessione, poiché le attivazioni di sessione possono richiedere molto tempo.
    Non riuscito: visualizzato quando l'attivazione di una sessione non riesce. Oracle Service Bus registra gli errori di attivazione di sessione ma non gli errori dovuti a singoli aggiornamenti all'interno di una sessione.
    Annullato da
    Il nome dell'utente che ha annullato questo task. In questo campo viene visualizzato Nessuno se l'attivazione della sessione non è stata annullata.
    Opzioni
    Fare clic sull'icona Annulla per invertire l'attivazione della sessione e le operazioni eseguite in quella sessione. È possibile annullare le attivazioni di sessione in qualsiasi ordine.
    Fare clic sull'icona Annulla nella sessione per creare una nuova sessione precompilata con i task necessari per annullare le modifiche attivate nella sessione. Vedere Annullamento in una sessione.
