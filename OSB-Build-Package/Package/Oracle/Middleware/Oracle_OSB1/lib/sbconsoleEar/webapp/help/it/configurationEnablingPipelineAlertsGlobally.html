Abilitazione degli alert pipeline a livello globale

Per i servizi proxy, per attivare gli alert pipeline per tutti i servizi proxy a livello di dominio, effettuare le operazioni riportate di seguito.

  1. Se questa operazione non è stata già eseguita, fare clic su Crea per creare una nuova sessione o su Modifica per accedere a una sessione esistente. Vedere Utilizzo del Centro modifiche.
  2. Selezionare Operazioni > Impostazioni globali.
  3. Selezionare la casella di controllo Abilita alert pipeline per far sì che il sistema avvii l'esecuzione di tutte le azioni di alert delle pipeline per i servizi proxy.

    4. L'impostazione Alert pipeline non ha alcuna dipendenza sull'impostazione Monitoraggio a livello globale.

  4. Fare clic su Aggiorna per salvare tutte le impostazioni in questa pagina.
  5. In alternativa, fare clic su Reimposta in qualsiasi momento durante la sessione per annullare le modifiche e aggiornare la pagina con le impostazioni attualmente memorizzate.
  6. Per terminare la sessione e distribuire la configurazione in fase di esecuzione, fare clic su Attiva in Centro modifiche.
