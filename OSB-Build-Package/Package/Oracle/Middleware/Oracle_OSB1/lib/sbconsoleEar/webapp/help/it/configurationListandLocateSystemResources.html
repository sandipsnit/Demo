Ricerca e aggiornamento delle impostazioni operative

Utilizzare la pagina Ricerca intelligente per individuare facilmente servizi proxy, servizi business, destinazioni degli alert e regole di alert SLA. È inoltre possibile utilizzare la pagina Ricerca intelligente per specificare le impostazioni operative specifiche di servizio, con le eccezioni seguenti: non è possibile impostare un intervallo di aggregazione o specificare un livello di severità di alert o log. Per ulteriori informazioni, vedere Configurazione delle impostazioni operative per i servizi proxy e Configurazione delle impostazioni operative per i servizi business.

Gli effetti in fase di esecuzione delle impostazioni a livello di servizio dipendono dalle relative impostazioni globali corrispondenti. Per abilitare completamente un servizio in fase di esecuzione, è necessario abilitare entrambe le impostazioni globali e a livello di servizio. Inoltre anche Stato del servizio deve essere abilitato. Vedere Abilitazione delle impostazioni globali.

È possibile abilitare o disabilitare le impostazioni operative solo dall'interno di una sessione.
