Impostazione delle preferenze utente

Utilizzare questa pagina per personalizzare e rendere persistenti le opzioni di visualizzazione e le impostazioni della console di amministrazione di Oracle Service Bus. È possibile aggiornare (modificare) le preferenze utente senza attivare una sessione di modifica.

Nella Tabella 26-7 sono descritte le opzioni e i controlli di questa pagina.

Tabella 26-7 Preferenze utente - Opzioni e controlli
Operazione
Azione
Modificare la Home page, la pagina visualizzata quando si avvia la console di amministrazione di Oracle Service Bus
Dal primo elenco a discesa selezionare una delle scelte di menu principale: Operazioni, Browser risorse, Explorer progetto, Configurazione sicurezza o Amministrazione sistema.
Dal secondo elenco a discesa selezionare una delle scelte di sottomenu. Il contenuto dei sottomenu cambia in modo dinamico in base alla selezione del primo menu.
Visualizzare i filtri di ricerca
Selezionare Sì o No.
Per impostazione predefinita, i filtri di ricerca non sono visualizzati (chiusi). Se si seleziona Sì, tali filtri vengono visualizzati sempre (aperti).
Visualizzare le annotazioni di fase
Selezionare Sì o No.
Per impostazione predefinita, le annotazioni di fase della pipeline non sono visualizzate (chiuse). Se si seleziona Sì, tali annotazioni vengono visualizzate sempre (aperte).
Visualizzare i metadati della risorsa
Selezionare Sì o No.
Per impostazione predefinita, le sezioni dei metadati della risorsa sono visualizzate (aperte). Se si seleziona No, tali sezioni non vengono visualizzate (chiuse).
Modificare la frequenza con cui la console di amministrazione di Oracle Service Bus aggiorna la visualizzazione dei dati nel dashboard
Dall'elenco a discesa Velocità di aggiornamento dashboard selezionare Nessun aggiornamento, 1, 2, 3, 5, 10, 20, 30 o 60 minuti come frequenza di aggiornamento per il dashboard.
La frequenza predefinita è Nessun aggiornamento ma è possibile selezionare un altro intervallo di tempo predefinito. Ad esempio, se si seleziona 5 minuti, l'aggiornamento dei dati del dashboard viene effettuato ogni 5 minuti.
Attenzione: se si seleziona un intervallo di frequenza di aggiornamento del dashboard diverso da quello predefinito (Nessun aggiornamento), il browser aggiorna il dashboard a intervalli regolari. Ciò impedisce il timeout della sessione, anche in assenza di interazione con la console di amministrazione di Oracle Service Bus.
Attenzione: Oracle consiglia di utilizzare con cautela la funzione di frequenza dell'aggiornamento del dashboard. È opportuno non lasciare mai una sessione della console di amministrazione Oracle Service Bus senza supervisione perché mentre la sessione è attiva altre persone possono accedervi utilizzando il computer dell'utente senza effettuare di nuovo l'autenticazione.
Modificare l'intervallo temporale per la visualizzazione dei dati cronologici per gli alert sul dashboard.
Dall'elenco a discesa Durata cronologia alert selezionare 30 minuti, 1, 2, 3 o 6 ore come intervallo temporale per la visualizzazione dei dati cronologici degli alert. L'impostazione predefinita è 30 minuti (visualizza gli alert ricevuti negli ultimi 30 minuti) ma è possibile selezionare un'altra durata predefinita. Ad esempio, se si seleziona 1 ora, il dashboard visualizza gli alert ricevuti nell'ultima ora.
