Avvio della console di amministrazione di Oracle Service Bus

  1. Avviare Oracle Service Bus utilizzando uno dei metodi riportati di seguito.
    • Dal menu Start di Windows, selezionare:

      • Start > Tutti i programmi > Oracle WebLogic > User Projects > nome_dominio > Start Server for Oracle Service Bus Domain.

      Dove nome_dominio rappresenta il nome assegnato al dominio Oracle Service Bus quando è stato creato. User Projects è la posizione in cui il dominio viene creato per impostazione predefinita.

    • Richiamare il comando startWebLogic dalla radice del dominio Oracle Service Bus: startWebLogic.cmd o startWebLogic.sh

      • Viene visualizzata una finestra dei comandi contenente informazioni di stato su WebLogic Server.

      Se il server è stato avviato in modalità produzione, è necessario immettere un nome utente e una password validi per avviare il server.

  2. All'avvio del server immettere nel browser l'URL seguente:

    3. http://hostname:port/sbconsole

    dove host rappresenta il nome del computer su cui WebLogic Server è in esecuzione e port rappresenta il numero di porta.

    Ad esempio, se WebLogic Server è in esecuzione sul computer locale e utilizza la configurazione predefinita della porta, immettere nel browser l'URL seguente:

    http://localhost:7001/sbconsole

    Viene visualizzata la pagina di login alla console di amministrazione di Oracle Service Bus.

  3. Immettere il nome utente e la password specificati durante il processo di installazione.
    4. Nota: il nome utente e la password per il dominio Esempi di Oracle Service Bus sono weblogic / welcome1.
  4. Fare clic su Login.

Per eseguire il logout, fare clic su Logout sul banner accanto alla parte superiore del pannello destro della console di amministrazione di Oracle Service Bus.
