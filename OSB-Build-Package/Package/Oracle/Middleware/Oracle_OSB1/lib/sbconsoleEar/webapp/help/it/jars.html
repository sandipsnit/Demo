JAR

Un JAR (Java ARchive) è un file compresso che contiene un set di classi Java. È utilizzato per memorizzare classi Java compilate e metadati associati che possono costituire un programma. Un file JAR svolge le stesse funzioni di una libreria di programmi richiamabili per gli elementi in codice Java (in modo che un singolo collegamento di compilazione fornisca accesso a più elementi, piuttosto che richiedere associazioni per ogni singolo elemento). Per ulteriori informazioni sui file JAR, vedere http://java.sun.com/docs/books/tutorial/deployment/jar/

In Oracle Service Bus le risorse JAR vengono utilizzate:

  • nelle azioni di callout Java;
  • nei servizi business basati su EJB;
  • nei servizi proxy e business basati su Tuxedo.
