Modifica di associazioni JCA

Utilizzare la pagina Visualizza dettagli associazione JCA per visualizzare e modificare i dettagli di un'associazione JCA.

  1. Se questa operazione non è stata già eseguita, fare clic su Crea per creare una nuova sessione o su Modifica per accedere a una sessione esistente. Vedere Utilizzo del Centro modifiche.
  2. Individuare l'associazione JCA, come descritto in Individuazione di associazioni JCA.
  3. Fare clic sul nome dell'associazione JCA. Nella pagina Visualizza un'associazione JCA vengono visualizzate le informazioni mostrate nella Tabella 12-2.

    4. Tabella 12-2 Dettagli dell'associazione JCA 
    Proprietà
    Descrizione
    Autore ultima modifica
    L'utente che ha creato questa associazione JCA o che l'ha importata nella configurazione.
    Data ultima modifica
    La data e l'ora in cui l'utente ha creato questa associazione JCA o l'ha importata nella configurazione. Fare clic sul collegamento di data e ora per visualizzare la cronologia delle modifiche di questa risorsa. Vedere la pagina Visualizza cronologia modifiche.
    Riferimenti
    Fornisce un collegamento alla risorsa WSDL a cui fa riferimento l'associazione JCA. Vedere Visualizzazione dei riferimenti alle risorse.
    Riferimento di
    Il numero di oggetti che fanno riferimento a questa associazione JCA. Se tali riferimenti esistono, fare clic sul collegamento numerico per visualizzare una lista degli oggetti. Vedere Visualizzazione dei riferimenti alle risorse.
    Icona Azione
    Con il Centro modifiche in modalità di modifica, l'icona Azione permette di generare un servizio JCA e WSDL Oracle Service Bus dall'associazione JCA.
    File JCA
    Visualizza una vista di sola lettura dell'associazione JCA.
    WSDL astratto
    Visualizza il WSDL associato dell'associazione JCA. Se l'associazione JCA non fa riferimento a un WSDL, fare clic su Modifica riferimenti per fare riferimento al WSDL associato.
    XML TopLink
    Visualizza il file di mapping TopLink o EclipseLink richiesto dell'associazione JCA. Se la risorsa di associazione JCA non fa riferimento al file di mapping richiesto, fare clic su Modifica riferimenti per selezionare il file di mapping necessario. Questo campo viene visualizzato solo se il file JCA dichiara una risorsa XML.
    XSLT
    Visualizza il file XSLT richiesto dell'associazione JCA. Se la risorsa di associazione JCA non fa riferimento a una risorsa XSLT richiesta, fare clic su Modifica riferimenti per selezionare il file XSLT necessario. Questo campo viene visualizzato solo se il file JCA dichiara una risorsa XSLT, ad esempio per un adattatore socket.

  4. Per modificare il campo Descrizione o File JCA, fare clic su Modifica.

    5. Non è possibile modificare il campo Nome risorsa.

  5. Fare clic su Salva per eseguire il commit degli aggiornamenti nella sessione corrente.
  6. Per modificare i riferimenti alle risorse richieste dell'associazione JCA oppure per correggere riferimenti non risolti, fare clic su Modifica riferimenti e selezionare le risorse appropriate.
  7. Per terminare la sessione e distribuire la configurazione in fase di esecuzione, fare clic su Attiva in Centro modifiche.
