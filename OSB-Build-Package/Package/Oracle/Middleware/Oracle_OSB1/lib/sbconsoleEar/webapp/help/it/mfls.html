MFL

Un documento MFL (Message Format Language) è un documento XML specializzato utilizzato per descrivere il layout di dati binari. Si tratta di un linguaggio proprietario Oracle utilizzato per definire regole di trasformazione dei dati binari formattati in dati XML. Un documento MFL è conforme a mfl.dtd, che include elementi e attributi utilizzati per descrivere ciascun campo di dati, nonché raggruppamenti di campi (gruppi), ripetizione e aggregazione.

Quando si crea un servizio business o un servizio proxy di tipo Servizio messaggistica, è possibile selezionare i tipi MFL come il tipo di messaggio di richiesta o il tipo di messaggio di risposta del servizio.

Utilizzare Oracle Format Builder per creare i documenti MFL. Quando si definisce la gerarchia di un record binario, il layout dei campi e il raggruppamento di campi e gruppi, le informazioni vengono salvate sotto forma di documento MFL che è possibile utilizzare per eseguire conversioni in fase di esecuzione. È possibile utilizzare un documento MFL anche in Format Builder per generare il DTD corrispondente che descrive il relativo modello di contenuto. Per informazioni sull'utilizzo di Format Builder, vedere "Format Builder" in Oracle Fusion Middleware Developer's Guide for Oracle Service Bus all'indirizzo http://www.oracle.com/pls/as111170/lookup?id=OSBDV763.
