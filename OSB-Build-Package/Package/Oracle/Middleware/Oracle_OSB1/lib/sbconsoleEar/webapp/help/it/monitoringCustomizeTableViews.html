Personalizzazione delle viste delle tabelle

Utilizzare la funzione di personalizzazione della tabella per visualizzare le informazioni della tabella in base alle impostazioni specificate.

Notare che l'ordinamento predefinito per ciascuna tabella è determinato dalla prima colonna della tabella.

  1. Fare clic sull'icona Funzione di personalizzazione tabella. Vengono visualizzati ulteriori campi.

    2. È possibile fare clic su Annulla in qualsiasi momento per chiudere questa tabella e mantenere le impostazioni originali.

  2. Nel campo Colonne selezionare le colonne che si desidera visualizzare.
    1. Selezionare un nome di colonna nel campo Colonne disponibili.
    2. Fare clic sulla freccia per spostare questo nome di colonna nel campo Colonne selezionate.

      3. Un asterisco indica una colonna obbligatoria.

    3. Ripetere queste operazioni fino a quando tutti i nomi di colonna che si desidera visualizzare non sono elencati nel campo Colonne selezionate.

      4. Nel campo Colonne selezionate è possibile utilizzare le frecce su e giù per riordinare i nomi di colonna come richiesto.

  3. Per rimuovere le colonne dalla visualizzazione, spostare i nomi di colonna dal campo Colonne selezionate al campo Colonne disponibili.
  4. Dall'elenco a discesa Righe selezionare il numero di righe che si desidera visualizzare in una singola pagina.
  5. Effettuare una delle operazioni riportate di seguito.
    • Per salvare queste impostazioni e chiudere la funzione di personalizzazione della tabella, fare clic su Applica.
    • Per annullare le modifiche e mantenere le impostazioni originali, fare clic su Reimposta.
