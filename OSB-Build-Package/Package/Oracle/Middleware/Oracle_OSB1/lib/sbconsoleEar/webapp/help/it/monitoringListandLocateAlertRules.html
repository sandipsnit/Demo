Visualizzazione delle regole di alert

Utilizzare questa pagina per visualizzare le informazioni sulle regole di alert mostrate nella Tabella 25-19.

Tabella 25-19 Informazioni sulle regole di alert
Proprietà
Descrizione
Nome
Il nome assegnato a questa regola di alert. Il nome è un collegamento alla pagina Configurazione generale regola di alert. Vedere Visualizzazione delle configurazioni delle regole di alert.
Stato SLA
Lo stato della regola di alert: Abilitato o Disabilitato.
Descrizione
Questo campo è nascosto per impostazione predefinita.
Una descrizione della regola di alert.
Severità
La severità dell'alert che verrà attivato da questa regola: Normale, Avvertenza, Non grave, Grave, Critico o Irreversibile.
Intervallo aggregazione
La durata dell'intervallo di aggregazione in termini di ore e minuti.
Data di scadenza
La data in cui questa regola di alert non sarà più valida.
Interrompi elaborazione
Visualizza Sì o No.
Frequenza
La frequenza di questo alert:
  • Ogni volta
  • Notifica una volta
Opzioni
In questa colonna sono presenti le funzionalità riportate di seguito.
  • Fare clic sulle frecce per riordinare le regole di alert. È possibile spostare singole regole di alert verso l'alto o verso il basso nella lista. Tale operazione è possibile solo quando sono presenti più regole di alert configurate per questo servizio.
  • Fare clic sull'icona Rinomina per rinominare una regola di alert.
  • Fare clic sull'icona Elimina per eliminare una regola di alert specifica. Vedere Eliminazione di regole di alert.
