Individuazione delle regole di alert

  1. Selezionare Browser risorse > Servizi business o Servizi proxy.
  2. Fare clic sul nome del servizio.
  3. Selezionare la scheda Regole di alert SLA.
  4. Per individuare una regola di alert specifica per questo servizio, scorrere le pagine. Utilizzare i controlli di pagina presenti sopra e sotto la tabella. Per andare a una pagina, selezionare il numero di pagina oppure utilizzare i pulsanti freccia per andare alla pagina successiva, alla pagina precedente, alla prima pagina o all'ultima pagina.
