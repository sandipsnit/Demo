Rimozione di alert

Utilizzare questa pagina per eliminare in modo permanente gli alert SLA o pipeline specificati.

Nota: questa azione non può essere annullata.
  1. Selezionare Operazioni > Dashboard.
  2. Selezionare la scheda Alert SLA o Alert pipeline, quindi fare clic sul collegamento Cronologia alert estesi.
  3. Nella parte in alto a destra della pagina Cronologia alert estesi fare clic su Rimuovi cronologia alert SLA o Rimuovi cronologia alert pipeline.
  4. Effettuare uno dei passi riportati di seguito, descritti nella Tabella 25-3.

    5. Tabella 25-3 Opzioni di rimozione degli alert
    Operazione
    Procedura da eseguire
    • Rimuovere tutti gli alert SLA
    • Rimuovere tutti gli alert pipeline
    1. Selezionare Rimuovi tutti gli alert.
    2. Fare clic su Rimuovi.
    • Rimuovere gli alert SLA entro un intervallo di tempo specificato
    • Rimuovere gli alert pipeline entro un intervallo di tempo specificato
    1. Selezionare Rimuovi alert da... fino a...
    2. Nel campo Rimuovi da immettere mese, giorno, anno e ora.
    3. Nel campo Rimuovi fino a immettere mese, giorno, anno e ora.
    4. Fare clic su Rimuovi.

Nota: se il server dal quale gli alert vengono rimossi non è disponibile, la struttura di log rende persistente un record della rimozione ed esegue il polling del server ogni 30 minuti per verificarne la disponibilità. Quando la struttura di log rileva che il server è disponibile, gli alert vengono eliminati.
