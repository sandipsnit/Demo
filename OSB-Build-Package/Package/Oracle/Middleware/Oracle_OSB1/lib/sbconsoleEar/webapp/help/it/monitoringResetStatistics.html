Reimpostazione delle statistiche per i servizi

È possibile utilizzare la pagina Integrità servizio per reimpostare le statistiche di monitoraggio per tutti i servizi nella configurazione. È possibile reimpostare le statistiche sia all'interno sia all'esterno di una sessione.

  1. Selezionare Operazioni > Dashboard > Integrità servizio.
  2. Nel campo Visualizza statistiche selezionare A partire dalla reimpostazione più recente. Nella vista A partire dalla reimpostazione più recente viene visualizzata una vista di conteggio dinamico delle metriche.
  3. Per reimpostare le statistiche di monitoraggio per un servizio specifico, fare clic sull'icona Reimposta nella colonna Azione di tale servizio.

    4. Quando si conferma di voler eseguire questa operazione, il sistema elimina tutte le statistiche di monitoraggio raccolte per il servizio a partire dalla reimpostazione più recente delle statistiche. Il sistema, tuttavia, non elimina le statistiche raccolte durante il periodo di tempo di Intervallo di aggregazione corrente per il servizio. Inoltre, dopo aver fatto clic sull'icona Reimposta, il sistema avvia immediatamente la nuova raccolta delle statistiche di monitoraggio per il servizio.

  4. Per reimpostare le statistiche di monitoraggio per tutti i servizi, fare clic sul collegamento Reimposta tutte le statistiche nella parte superiore della pagina.
    5. Nota: facendo clic su Reimposta tutte le statistiche si reimpostano le statistiche di tutti i servizi monitorati indipendentemente dal fatto che siano visualizzati nella pagina o meno.

    Quando si conferma di voler reimpostare le statistiche, il sistema elimina tutte le statistiche di monitoraggio raccolte per tutti i servizi nella configurazione a partire dalla reimpostazione più recente delle statistiche. Il sistema non elimina le statistiche raccolte durante il periodo di tempo di Intervallo di aggregazione corrente per ciascuno dei servizi. Inoltre, dopo aver fatto clic su Reimposta tutte le statistiche, il sistema avvia immediatamente la nuova raccolta delle statistiche di monitoraggio per i servizi.
