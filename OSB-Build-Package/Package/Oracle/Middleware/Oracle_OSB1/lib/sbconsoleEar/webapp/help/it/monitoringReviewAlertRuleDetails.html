Revisione della configurazione della regola di alert

Utilizzare questa pagina per rivedere i dati di configurazione immessi per questa regola di alert. Se necessario, Ã¨ possibile fare clic su Modifica per apportare modifiche alla configurazione prima di salvare la regola di alert.

Effettuare una delle operazioni riportate di seguito.

  • Per apportare modifiche a una delle pagine di configurazione, fare clic sull'icona Modifica per la pagina appropriata.
  • Per tornare alla pagina precedente, fare clic su Prec..
  • Per creare l'alert, fare clic su Salva. La regola di alert viene creata e salvata nella sessione corrente.
  • Per annullare le modifiche, fare clic su Annulla.
  • Per terminare la sessione e distribuire la configurazione in fase di esecuzione, fare clic su Attiva in Centro modifiche.
