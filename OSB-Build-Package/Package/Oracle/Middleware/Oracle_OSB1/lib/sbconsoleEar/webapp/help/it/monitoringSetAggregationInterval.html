Impostazione dell'intervallo di aggregazione per un servizio

Utilizzare la pagina Impostazioni operative per impostare l'intervallo di aggregazione per un servizio specifico. L'intervallo di aggregazione è il periodo in cui le statistiche aggregate vengono elaborate per la visualizzazione nella console di amministrazione di Oracle Service Bus. L'impostazione di intervallo di aggregazione predefinita è 10 minuti.

  1. Se questa operazione non è stata già eseguita, fare clic su Crea per creare una nuova sessione o su Modifica per accedere a una sessione esistente. Vedere Utilizzo del Centro modifiche.
  2. Selezionare Browser risorse > Servizi business o Servizi proxy.
  3. Fare clic sul nome del servizio.
  4. Selezionare la scheda Impostazioni operative.
  5. Nel campo Intervallo di aggregazione selezionare la durata dell'intervallo di aggregazione in ore e minuti, quindi fare clic su Aggiorna.

    6. Se la selezione per le ore è maggiore di 1, la selezione predefinita per i minuti è sempre zero. Se invece la selezione per le ore è 0 o 1, è possibile configurare intervalli in termini di minuti.

  6. Fare clic su Aggiorna per eseguire il commit degli aggiornamenti nella sessione corrente.
  7. Per terminare la sessione e distribuire la configurazione in fase di esecuzione, fare clic su Attiva in Centro modifiche.
