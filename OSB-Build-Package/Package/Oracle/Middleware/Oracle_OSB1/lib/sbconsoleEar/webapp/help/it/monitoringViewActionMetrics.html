Visualizzazione delle metriche di azione

Utilizzare questa pagina per visualizzare le informazioni sul monitoraggio dei servizi, mostrate nella Tabella 25-10, e le metriche di azione per i servizi proxy con pipeline, mostrate nella Tabella 25-14.

Tabella 25-14 Metriche di azione
Proprietà
Descrizione
Nodo
Una visualizzazione gerarchica delle azioni nidificate.
Conteggio
Se nel campo Visualizza statistiche è stato selezionato Intervallo di aggregazione corrente, indica il numero di richiami entro il periodo dell'intervallo di aggregazione corrente.
Se nel campo Visualizza statistiche è stato selezionato A partire dalla reimpostazione più recente, indica il numero di richiami entro il periodo a partire dalla reimpostazione più recente delle statistiche.
Notare che questo valore differisce dal numero di messaggi perché un'azione può essere richiamata più volte in un messaggio.
Tempo medio
Se nel campo Visualizza statistiche è stato selezionato Intervallo di aggregazione corrente, indica il tempo medio di esecuzione per richiamo entro il periodo dell'intervallo di aggregazione corrente.
Se nel campo Visualizza statistiche è stato selezionato A partire dalla reimpostazione più recente, indica il tempo medio di esecuzione per richiamo entro il periodo a partire dalla reimpostazione più recente delle statistiche.
Tempo totale
Se nel campo Visualizza statistiche è stato selezionato Intervallo di aggregazione corrente, indica il tempo totale di esecuzione entro il periodo dell'intervallo di aggregazione corrente.
Se nel campo Visualizza statistiche è stato selezionato A partire dalla reimpostazione più recente, indica il tempo totale di esecuzione entro il periodo a partire dalla reimpostazione più recente delle statistiche.
