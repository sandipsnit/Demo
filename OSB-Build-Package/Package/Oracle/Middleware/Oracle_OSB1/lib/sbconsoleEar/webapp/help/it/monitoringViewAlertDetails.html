Visualizzazione dei dettagli degli alert

In questa pagina vengono visualizzati i dettagli di alert specifici che includono le informazioni mostrate nella Tabella 25-4.

Tabella 25-4 Dettagli degli alert
Proprietà
Descrizione
Nome alert
Solo per gli alert SLA.
Il nome assegnato all'alert. Il nome è un collegamento alla pagina Configurazione regola di alert. Vedere Visualizzazione delle configurazioni delle regole di alert.
Riepilogo alert
Solo per gli alert pipeline.
Una breve descrizione dell'azione di alert pipeline o Alert Oracle Service Bus (il valore predefinito fornito per le azioni di alert pipeline progettate senza testo di riepilogo dell'alert).
Descrizione
Una descrizione per la regola di alert.
Indicatore orario
  • Per gli alert pipeline: la data e l'ora in cui l'alert si è verificato.
  • Per gli alert SLA: la data e l'ora dell'ultima volta in cui la regola di alert ha restituito true.
Severità
La severità dell'alert definita dall'utente:
  • Irreversibile
  • Critico
  • Grave
  • Non grave
  • Avvertenza
  • Normale
Servizio
Il nome e il percorso del servizio associato a questo alert. Il campo è un collegamento alla pagina Dettagli monitoraggio servizio (solo quando il servizio viene monitorato attivamente). Vedere Visualizzazione delle metriche di servizio.
Tipo di servizio
Il tipo di servizio padre: servizio proxy o servizio business.
È possibile definire alert pipeline solo per i servizi proxy, dunque tutti i servizi padre degli alert pipeline sono servizi proxy.
Server
Il nome del server in cui questo alert è stato generato.
Solo per gli alert SLA: viene visualizzato N/A.
Annotazione
Una casella di testo in cui è possibile immettere note per questo alert. Un'icona Annotazione viene visualizzata nella cronologia degli alert per gli alert con annotazioni.
