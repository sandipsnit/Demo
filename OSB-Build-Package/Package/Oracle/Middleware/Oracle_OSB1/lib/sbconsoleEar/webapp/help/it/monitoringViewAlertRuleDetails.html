Visualizzazione delle configurazioni delle regole di alert

Utilizzare questa pagina per visualizzare i dettagli delle configurazioni delle regole di alert mostrati nella Tabella 25-22.

Tabella 25-22 Regola di alert - Configurazione generale
Proprietà
Descrizione
Nome regola
Il nome della regola di alert.
Descrizione regola
Una descrizione della regola di alert.
Riepilogo alert
Una breve descrizione della regola di alert. Questo testo viene utilizzato anche come riga dell'oggetto in un messaggio di posta elettronica se questa regola di alert è configurata con una notifica di alert di posta elettronica.
Destinazione alert
Il percorso e il nome della risorsa di destinazione degli alert che contiene l'indirizzo di destinazione per le notifiche di alert. Questo campo è un collegamento alla pagina Visualizza destinazione alert - Configurazione generale. Vedere Modifica delle destinazioni degli alert.
Ora di inizio (HH:MM)
L'ora di inizio per questa regola di alert.
Ora di fine (HH:MM)
L'ora di fine per questa regola di alert.
Data di scadenza della regola (/MM/GG/AAAA)
La data in cui questa regola di alert non sarà più valida.
Regola abilitata
Indica se questa regola di alert è abilitata o meno.
Severità alert
La severità dell'alert generata come risultato di questa regola di alert:
  • Normale
  • Avvertenza
  • Non grave
  • Grave
  • Critico
  • Irreversibile
Frequenza alert
La frequenza dell'alert:
  • Ogni volta
  • Notifica una volta
Arresta elaborazione di altre regole
Indica se l'alert arresta l'elaborazione di altre regole o meno.
