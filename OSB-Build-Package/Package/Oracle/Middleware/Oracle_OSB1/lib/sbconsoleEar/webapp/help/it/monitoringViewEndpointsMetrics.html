Tabella 25-15 Metriche degli URI endpoint

Descrizione
URI endpoint
L'URI endpoint associato a questo servizio business.
Conteggio messaggi
Se nel campo Visualizza statistiche è stato selezionato Intervallo di aggregazione corrente, indica il numero totale di messaggi eseguiti da questo URI endpoint entro il periodo dell'intervallo di aggregazione corrente.
Se nel campo Visualizza statistiche è stato selezionato A partire dalla reimpostazione più recente, indica il numero totale di messaggi eseguiti da questo URI endpoint entro il periodo a partire dalla reimpostazione più recente delle statistiche.

Stato
Lo stato dell'URI endpoint del servizio business.
Per i domini di server singolo o per un singolo server gestito in un cluster: In linea o Non in linea.
Per le statistiche a livello di cluster: In linea, Non in linea o Parziale, quando un determinato URI non è in linea su almeno uno dei server gestiti nel cluster.
Azione
Fare clic sull'icona Contrassegna endpoint In linea per contrassegnare come In linea un URI non in linea. Questa azione è abilitata solo se lo stato dell'URI è Non in linea o Parziale.
Per abilitare questa azione, è necessario che un servizio business sia abilitato a prendere gli URI non in linea e che le relative impostazioni operative di monitoraggio a livello globale e di servizio siano abilitate. Vedere Configurazione delle impostazioni operative per i servizi business.
Nota: se riuscita, l'azione è immediata. Tuttavia, dal momento che lo stato del relativo URI endpoint è ora In linea, viene eseguito un ciclo di aggregazione di monitoraggio che può durare un minuto e mezzo.
