Visualizzazione dei file di log del dominio

In questa pagina viene visualizzato un riepilogo delle voci dei file di log del dominio, incluse le informazioni mostrate nella Tabella 25-17.

  • Per personalizzare la visualizzazione delle informazioni sui file di log del dominio, vedere Personalizzazione della visualizzazione delle voci del file di log del dominio.
  • Per visualizzare i dettagli di un file di log specifico, selezionare il pulsante di scelta della voce appropriata, quindi fare clic su Visualizza. Vedere Visualizzazione dei dettagli dei file di log del dominio.
