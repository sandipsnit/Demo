WS-Policy personalizzati

Nota: nelle release future di Oracle Service Bus, i criteri Oracle Web Services Manager sostituiranno i criteri WLS 9.2. Sebbene questa versione di Oracle Service Bus continui a supportare i criteri WLS 9.2, è opportuno considerare l'utilizzo dei criteri Oracle Web Services Manager per la creazione di nuovi servizi in modo da essere pronti nell'eventuale mancanza di supporto dei criteri WLS 9.2. Per ulteriori informazioni, vedere "Securing Oracle Service Bus with Oracle Web Services Manager" in Oracle Fusion Middleware Developer's Guide for Oracle Service Bus all'indirizzo http://www.oracle.com/pls/as1111/lookup?id=OSBDV1681.

La struttura WS-Policy (Web Services Policy) è una struttura estensibile basata su XML che estende la configurazione di un servizio Web con asserzioni di sicurezza specifiche del dominio e specifica i requisiti di sicurezza, le aspettative e le capacità del servizio Web. In Oracle Service Bus, uno degli utilizzi principali di WS-Policy è la configurazione della sicurezza a livello di messaggio nei servizi proxy e nei servizi business.

Utilizzo dei WS-Policy di Oracle Service Bus

Oracle Service Bus include tre semplici file WS-Policy che è possibile utilizzare per richiedere ai client di autorizzare, cifrare in modo digitale o firmare in modo digitale i messaggi SOAP: Auth.xml, Encrypt.xml e Sign.xml. Oracle consiglia di utilizzare questi file inclusi nel package il più spesso possibile, a meno che l'utente non abbia esigenze di sicurezza specifiche.

Per ulteriori informazioni, vedere "Using WS-Policy in Oracle Service Bus Proxy and Business Services" in Oracle Fusion Middleware Developer's Guide for Oracle Service Bus all'indirizzo http://www.oracle.com/pls/as111170/lookup?id=OSBDV1599.

Utilizzo di WS-Policy personalizzati

Se le istruzioni dei WS-Policy di Oracle Service Bus non soddisfano le esigenze di sicurezza richieste, è possibile scrivere i propri WS-Policy (WS-Policy personalizzati), importarli in Oracle Service Bus e fare riferimento a essi dal WSDL. Le istruzioni dei WS-Policy di Oracle Service Bus sono di sola lettura.

Per informazioni sulla creazione dei WS-Policy personalizzati e su come fare riferimento a essi, vedere "Using WS-Policy in Oracle Service Bus Proxy and Business Services" in Oracle Fusion Middleware Developer's Guide for Oracle Service Bus all'indirizzo http://www.oracle.com/pls/as111170/lookup?id=OSBDV1599.
