Aggiunta di WS-Policy personalizzati

È possibile importare un file XML contenente il WS-Policy personalizzato oppure scrivere il WS-Policy direttamente nella console di amministrazione di Oracle Service Bus.

  1. Se questa operazione non è stata già eseguita, fare clic su Crea per creare una nuova sessione o su Modifica per accedere a una sessione esistente. Vedere Utilizzo del Centro modifiche.
  2. Selezionare Explorer progetto, quindi selezionare un progetto o una cartella in cui aggiungere il WS-Policy. Viene visualizzata la pagina Vista progetto/cartella.
  3. Nell'elenco a discesa Crea risorsa selezionare WS-Policy in Interfaccia per visualizzare la pagina Crea un nuovo WS-Policy.
  4. Nel campo Nome risorsa immettere un nome univoco per questo WS-Policy.

    Vedere Limitazioni di denominazione delle risorse per le istruzioni di denominazione.
  5. Nel campo Descrizione risorsa immettere una descrizione per questo WS-Policy.
  6. Nel campo Criterio effettuare una delle operazioni riportate di seguito.
    • Fare clic su Sfoglia per individuare e importare un file XML che contiene il WS-Policy personalizzato.
    • Immettere il testo per il nuovo WS-Policy.
    • Copiare e incollare in questo campo il testo di un WS-Policy esistente.
  7. Fare clic su Salva. Il WS-Policy viene salvato nella sessione corrente.
  8. Per terminare la sessione e distribuire la configurazione in fase di esecuzione, fare clic su Attiva in Centro modifiche.
