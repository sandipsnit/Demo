Eliminazione di progetti

  1. Se questa operazione non è stata già eseguita, fare clic su Crea per creare una nuova sessione o su Modifica per accedere a una sessione esistente. Vedere Utilizzo del Centro modifiche.
  2. Selezionare Explorer progetto > Progetti per visualizzare la pagina Vista progetti.
  3. Fare clic sull'icona Elimina progetto nella riga del progetto che si desidera eliminare. Il progetto viene eliminato nella sessione corrente.

    4. Quando si elimina un progetto, tutte le risorse che dipendono da tale progetto vengono eliminate. Se alle risorse di questo progetto fanno riferimento risorse di un altro progetto, è possibile eliminare il progetto confermando l'operazione in una finestra di avvertenza. Questa operazione potrebbe determinare dei conflitti a causa di riferimenti non risolti alla risorsa eliminata.

    Per queste risorse viene visualizzata l'icona Avvertenza di eliminazione anziché l'icona Elimina.

  4. Per terminare la sessione e distribuire la configurazione in fase di esecuzione, fare clic su Attiva in Centro modifiche.
