Eliminazione di cartelle

  1. Se questa operazione non è stata già eseguita, fare clic su Crea per creare una nuova sessione o su Modifica per accedere a una sessione esistente. Vedere Utilizzo del Centro modifiche.
  2. Selezionare Explorer progetto, quindi selezionare il progetto o la cartella contenente la cartella che si desidera eliminare.
  3. Nella pagina Vista progetto/cartella, fare clic sull'icona Elimina cartella nella riga della cartella che si desidera eliminare. La cartella viene eliminata nella sessione corrente.

    4. Quando si elimina una cartella, tutte le risorse che dipendono da tale cartella vengono eliminate. Se alle risorse di questa cartella fanno riferimento risorse di un'altra cartella, è possibile eliminare la cartella confermando l'operazione in una finestra di avvertenza. Questa operazione potrebbe determinare dei conflitti a causa di riferimenti non risolti alla risorsa eliminata.

    Per queste risorse viene visualizzata l'icona Avvertenza di eliminazione anziché l'icona Elimina.

  4. Per terminare la sessione e distribuire la configurazione in fase di esecuzione, fare clic su Attiva in Centro modifiche.
