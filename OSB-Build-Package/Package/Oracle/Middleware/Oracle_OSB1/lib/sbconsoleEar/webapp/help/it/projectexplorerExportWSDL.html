Esportazione di un WSDL

Utilizzare la pagina Vista progetto/cartella per esportare un WSDL associato a un servizio proxy o business. Il WSDL viene esportato come file JAR. È possibile esportare un WSDL solo quando si è all'esterno di una sessione. Vedere Utilizzo del Centro modifiche.

Utilizzare la funzione Esporta WSDL per rendere disponibile un WSDL per l'uso da parte di strumenti esterni, quali IDE. Tenere presente che questa funzione è diversa dalla funzione Esporta risorse disponibile nel modulo Amministrazione sistema, che consente di spostare e posizionare nell'area intermedia le risorse tra due domini.

  1. Nel riquadro di navigazione fare clic su un progetto o su una cartella in Explorer progetto. Viene visualizzata la pagina Vista progetto/cartella.
  2. Fare clic sull'icona Esporta WSDL nella riga del servizio proxy per il quale si desidera esportare il WSDL. Viene visualizzata la finestra di dialogo Scaricamento file.
  3. Nella finestra di dialogo Scaricamento file fare clic su Apri per aprire il file JAR oppure fare clic su Salva per salvare il file JAR sul desktop.
