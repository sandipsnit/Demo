Pagina Vista progetti

La pagina Vista progetti elenca i progetti definiti nel dominio corrente. Nella Tabella 3-1 vengono descritte le opzioni e i controlli disponibili nella pagina.

Tabella 3-1 Pagina Vista progetti - Opzioni e controlli 
Opzione o controllo
Descrizione
Immetti nome nuovo progetto
Per creare un nuovo progetto, immettere un nome nel campo di testo e fare clic su Aggiungi progetto. Vedere Aggiunta di progetti.
Vedere anche Denominazione dei progetti e delle cartelle.
Nome
Fare clic sul nome di un progetto per visualizzare la pagina Vista progetto/cartella per tale progetto.
Opzioni
Per qualsiasi progetto elencato è possibile effettuare una delle operazioni riportate di seguito.
  • Fare clic sull'icona Rinomina progetto per rinominare il progetto. Vedere Ridenominazione dei progetti.
  • Fare clic sull'icona Sposta progetto per spostare il progetto. Vedere Spostamento di progetti (Conversione in una cartella).
  • Fare clic sull'icona Duplica progetto per duplicare il progetto. Vedere Duplicazione dei progetti.
  • Fare clic sull'icona Elimina progetto per eliminare il progetto. Se altre risorse fanno riferimento a questa risorsa, viene visualizzata un'icona Avvertenza di eliminazione. È possibile eliminare questa risorsa associata a una conferma di avvertenza. Questa operazione potrebbe determinare dei conflitti a causa di riferimenti non risolti alla risorsa eliminata. Vedere Eliminazione di progetti.
Quando si esegue una qualsiasi di queste azioni, il risultato viene salvato nella sessione corrente. Per terminare la sessione e distribuire la configurazione in fase di esecuzione, fare clic su Attiva in Centro modifiche.
