Pagina Carica risorse - Carica risorse da URL

Utilizzare la pagina Carica risorse - Carica risorse da URL per importare le risorse da un file disponibile sul Web, come descritto in Caricamento delle risorse da un URL.

Nella Tabella 3-5 vengono descritte le opzioni e i controlli disponibili nella pagina.

Tabella 3-5 Pagina Carica risorse - Carica risorse da URL - Opzioni e controlli 
Opzioni e controlli
Descrizione
URL/percorso
Immettere l'URL in cui è posizionato il file.
Per specificare una risorsa locale, è possibile usare il protocollo file con un nome file completamente qualificato dal relativo percorso, ad esempio file:///c:/osbresources/ForeachAction.jar
Nome risorsa
Immettere un nome per la risorsa.
Tipo di risorsa
Selezionare il tipo di risorsa dalla lista.
