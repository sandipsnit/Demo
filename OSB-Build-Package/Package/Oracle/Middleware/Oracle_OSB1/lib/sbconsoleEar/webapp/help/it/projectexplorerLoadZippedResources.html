Tipo di risorsa ed estensione file
Ogni file contenuto nel file Zip è una possibile risorsa e l'estensione file costituisce il metodo comunemente utilizzato per identificare il tipo di file. Oracle Service Bus definisce un'estensione predefinita per ogni risorsa; tuttavia, se si dispone di estensioni aggiuntive per descrivere determinate risorse, questa mappa consente di dichiarare tali estensioni al loader.
Una volta definito il file Zip e la mappa di estensioni, il loader di Oracle Service Bus esegue la scansione del contenuto del file Zip e tenta di associare un tipo di risorsa a ciascun file. Un file senza estensione o un file per il quale non è definita un'estensione nella mappa viene considerato sconosciuto e viene automaticamente escluso dall'operazione di caricamento.
Per i file conosciuti, il nome della risorsa è il nome del file senza la relativa estensione. La struttura di cartelle del file Zip viene ricreata nella cartella della console di amministrazione di Oracle Service Bus di destinazione.
