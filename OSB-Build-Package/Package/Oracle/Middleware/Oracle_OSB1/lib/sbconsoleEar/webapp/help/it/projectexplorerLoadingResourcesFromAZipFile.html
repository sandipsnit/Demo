Caricamento delle risorse da un file Zip

È possibile selezionare un file Zip o JAR contenente le risorse che si desidera caricare. Oracle Service Bus individua e importa le risorse da questo file Zip.

Le risorse, quali WSDL o schemi XML, possono definire strutture complete di risorse dipendenti. La funzione bulk load consente di caricare un set di risorse contemporaneamente; non è necessario risolvere le dipendenze in modo manuale.

  1. Se questa operazione non è stata già eseguita, fare clic su Crea per creare una nuova sessione o su Modifica per accedere a una sessione esistente. Vedere Utilizzo del Centro modifiche.
  2. Selezionare Explorer progetto, quindi selezionare il progetto o la cartella a cui si desidera aggiungere le risorse. Viene visualizzata la pagina Vista progetto/cartella.
  3. Dall'elenco a discesa Crea risorsa selezionare Risorse compresse. Viene visualizzata la pagina Carica risorse.
  4. Nel campo Seleziona file Zip specificare un file Zip da importare. Immettere il nome del file (qualificato dal relativo percorso) oppure fare clic su Sfoglia, individuare il file, quindi fare clic su Apri.
  5. Rivedere il mapping dei tipi di risorsa alle estensioni file. Modificare o aggiungere estensioni in base alle esigenze. Utilizzare le virgole per separare più estensioni.

    6. Ogni file contenuto nel file Zip è una possibile risorsa e l'estensione file costituisce il metodo comunemente utilizzato per identificare il tipo di file. Oracle Service Bus definisce un'estensione predefinita per ogni risorsa; tuttavia, se si dispone di estensioni aggiuntive per descrivere determinate risorse, questa mappa consente di dichiarare tali estensioni al loader.

    Una volta definito il file Zip e la mappa di estensioni, il loader di Oracle Service Bus esegue la scansione del contenuto del file Zip e tenta di associare un tipo di risorsa a ciascun file. Un file senza estensione o un file per il quale non è definita un'estensione nella mappa viene considerato sconosciuto e viene automaticamente escluso dall'operazione di caricamento.

    Per i file conosciuti, il nome della risorsa è il nome del file senza la relativa estensione. La struttura di cartelle del file Zip viene ricreata nella cartella della console di amministrazione di Oracle Service Bus di destinazione.

    Un'estensione può essere associata a un solo tipo di file.

  6. Fare clic su Successivo.
  7. Nella pagina Carica risorse - Rivedi risorse caricate rivedere le risorse da importare. È possibile deselezionare le caselle di controllo accanto alle risorse che non si desidera includere nell'importazione corrente.
  8. Per importare le risorse selezionate, fare clic su Importa.
  9. Le risorse importate vengono visualizzate nella pagina Carica risorse - Importa risultato. Fare clic su Carica un altro per importare risorse da un altro file oppure procedere con altri task selezionando una pagina diversa nel riquadro di navigazione sinistro.
  10. Per terminare la sessione e distribuire la configurazione in fase di esecuzione, fare clic su Attiva in Centro modifiche.
