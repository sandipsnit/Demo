Caricamento delle risorse da un URL

È possibile eseguire un'importazione in bulk delle risorse disponibili in un URL o su un file system. L'importazione in bulk consente di importare una risorsa radice, ad esempio un WSDL, insieme alle risorse dipendenti, ad esempio altri WSDL e schemi. La mappa delle dipendenze viene risolta in modo automatico.

  1. Se questa operazione non è stata già eseguita, fare clic su Crea per creare una nuova sessione o su Modifica per accedere a una sessione esistente. Vedere Utilizzo del Centro modifiche.
  2. Selezionare Explorer progetto, quindi selezionare il progetto o la cartella a cui si desidera aggiungere le risorse. Viene visualizzata la pagina Vista progetto/cartella.
  3. Dall'elenco a discesa Crea risorsa selezionare Risorse da URL. Viene visualizzata la pagina Carica risorse - Carica risorse da URL.
  4. Nel campo URL/percorso immettere l'URL o il percorso della risorsa.

    5. Per specificare una risorsa locale, è possibile usare il protocollo file con un nome file completamente qualificato dal relativo percorso, ad esempio file:///c:/osbresources/ForeachAction.jar

  5. Nel campo Nome risorsa immettere il nome della risorsa.
  6. Dall'elenco a discesa Tipo di risorsa selezionare il tipo di risorsa.
  7. Fare clic su Successivo per visualizzare la pagina Carica risorse - Rivedi risorse caricate. Deselezionare le caselle di controllo accanto alle risorse che non si desidera includere nell'importazione corrente.
  8. Per importare le risorse selezionate, fare clic su Importa.
  9. Le risorse importate vengono visualizzate nella pagina Carica risorse - Importa risultato. Fare clic su Carica un altro per importare risorse da un altro file oppure procedere con altri task selezionando una pagina diversa nel riquadro di navigazione sinistro.
  10. Per terminare la sessione e distribuire la configurazione in fase di esecuzione, fare clic su Attiva in Centro modifiche.
