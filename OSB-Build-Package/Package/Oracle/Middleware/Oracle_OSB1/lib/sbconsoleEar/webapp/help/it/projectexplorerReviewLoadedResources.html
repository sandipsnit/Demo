Pagina Carica risorse - Rivedi risorse caricate

Nella pagina Carica risorse - Rivedi risorse caricate vengono visualizzati i dettagli sulle risorse da importare da un file Zip a un altro file, come descritto in Caricamento delle risorse da un file Zip o in Caricamento delle risorse da un URL.

Nella Tabella 3-6 vengono descritte le opzioni e i controlli disponibili nella pagina.

Tabella 3-6 Pagina Carica risorse - Rivedi risorse caricate - Opzioni e controlli 
Opzioni e controlli
Descrizione
Nome file
Il nome file della risorsa, inclusa l'estensione del file.
Nome
Il nome della risorsa.
Percorso
Il percorso della risorsa, ovvero il progetto e la cartella in cui risiede la risorsa.
Tipo di risorsa
Elenca i tipi di risorsa supportati.

Un'icona Avvertenza accanto al nome file indica che il tipo di risorsa è sconosciuto. Un file senza estensione o un file per il quale non è definita un'estensione nella mappa viene considerato sconosciuto. Non è possibile importare file di tipo sconosciuto.
