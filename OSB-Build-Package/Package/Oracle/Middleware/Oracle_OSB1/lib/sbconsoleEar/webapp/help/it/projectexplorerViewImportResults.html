Pagina Carica risorse - Importa risultato

Nella pagina Carica risorse - Importa risultato vengono visualizzati i risultati del caricamento delle risorse da un file Zip importato o da un URL.

A seconda che le risorse siano state caricate da un file Zip o da URL, assicurarsi di aver completato i passi descritti in Caricamento delle risorse da un file Zip o in Caricamento delle risorse da un URL e di aver completato i passi descritti nella pagina Carica risorse - Rivedi risorse caricate.

Viene visualizzata la pagina Importa risultato contenente le informazioni riportate di seguito.

Tabella 3-7 Pagina Vista progetto/cartella - Opzioni e controlli 
Opzione o controllo
Descrizione
Stato
Lo stato della risorsa; ad esempio se è stata importata correttamente o con errori.
È possibile che un WSDL sia contrassegnato temporaneamente come non valido finché non vengono importate tutte le relative dipendenze. Tuttavia, quando l'azione di importazione viene completata, il servizio di configurazione esegue di nuovo la convalida dei file contrassegnati come non validi.
Nome
Il nome della risorsa.
Percorso
Il percorso della risorsa, ovvero il progetto e la cartella in cui risiede la risorsa.
Tipo di risorsa
Visualizza il tipo di risorsa importata.
Messaggio di errore
Visualizza i dettagli del messaggio di errore se la risorsa non è stata importata correttamente.
