Pagina Vista progetto/cartella

Nella pagina Vista progetto/cartella vengono visualizzati i dettagli di un singolo progetto. Nella Tabella 3-2 vengono descritte le opzioni e i controlli disponibili nella pagina.

Tabella 3-2 Pagina Vista progetto/cartella - Opzioni e controlli 
Opzione o controllo
Descrizione
Nome progetto
Riferimenti
Fare clic sul numero in questo campo (se maggiore di 0) per visualizzare una lista delle risorse esterne a questo progetto da cui dipendono le risorse interne a questo progetto.
Riferimento di
Fare clic sul numero in questo campo (se maggiore di 0) per visualizzare una lista delle risorse esterne a questo progetto che dipendono dalle risorse interne a questo progetto.
Descrizione
Fare clic su Modifica descrizione per aggiungere o modificare una descrizione di questo progetto, quindi fare clic su Sottometti descrizione per aggiornarla.
Cartelle
Immettere un nuovo nome di cartella
Per creare una nuova cartella in questo progetto, immettere un nome nel campo di testo e fare clic su Aggiungi cartella. Vedere Aggiunta di cartelle.
Vedere anche Denominazione dei progetti e delle cartelle.
Nome
Fare clic sul nome di una cartella per visualizzare la pagina Vista progetto/cartella per tale cartella.
Opzioni
Per qualsiasi cartella elencata è possibile effettuare una delle operazioni riportate di seguito.
  • Fare clic sull'icona Rinomina cartella per rinominare la cartella. Vedere Ridenominazione delle cartelle.
  • Fare clic sull'icona Aggiorna a progetto o sposta cartella per spostare la cartella o aggiornarla in un progetto. Vedere Spostamento o aggiornamento delle cartelle.
  • Fare clic sull'icona Duplica cartella per duplicare la cartella. Vedere Duplicazione delle cartelle.
  • Fare clic sull'icona Elimina cartella per eliminare la cartella. Se altre risorse fanno riferimento a questa risorsa, viene visualizzata un'icona Avvertenza di eliminazione. È possibile eliminare la risorsa associata a una conferma di avvertenza. Questa operazione potrebbe determinare dei conflitti a causa di riferimenti non risolti alla risorsa eliminata. Vedere Eliminazione di cartelle.
Quando si esegue una qualsiasi di queste azioni, il risultato viene salvato nella sessione corrente. Per terminare la sessione e distribuire la configurazione in fase di esecuzione, fare clic su Attiva in Centro modifiche.
Elimina
Per eliminare le cartelle, selezionare le caselle di controllo accanto ai nomi delle cartelle da eliminare, quindi fare clic su Elimina.
Risorse
Crea risorsa
Per creare una nuova risorsa per questo progetto, selezionare un tipo di risorsa dalla lista Crea risorsa. Vedere Creazione delle risorse.
Nome
Fare clic sul nome di una risorsa per visualizzare i dettagli relativi alla risorsa.
Tipo di risorsa
Il tipo di questa risorsa.
Azioni
Le azioni disponibili per una risorsa dipendono dal tipo di risorsa.
  • Per i servizi proxy viene visualizzata l'icona Modifica flusso di messaggi. Fare clic su questa icona per modificare il messaggio di un servizio proxy. Vedere Creazione, modifica e visualizzazione dei flussi di messaggi.
  • Per i servizi business, i servizi proxy, le trasformazioni XQuery, le trasformazioni XSL e i file MFL viene visualizzata l'icona Avvia console di test. Fare clic su questa icona per richiamare la console di test, che consente di convalidare ed eseguire il test della progettazione dei servizi e delle trasformazioni. Per i servizi è possibile usare la console di test solo in fase di esecuzione, ovvero quando la sessione è attivata. Per le trasformazioni è possibile usare la console di test sia all'interno che all'esterno di una sessione. Vedere Test dei servizi e Test delle trasformazioni.
  • Per i servizi proxy basati su WSDL viene visualizzata l'icona Esporta WSDL. Fare clic su questa icona per esportare un WSDL, che è possibile visualizzare e modificare in un secondo momento in uno strumento esterno, ad esempio IDE.

    • Si noti che questa funzione è diversa dalla funzione Esporta risorse disponibile nel modulo Amministrazione sistema, che consente di spostare e posizionare nell'area intermedia le risorse tra due domini. Vedere Esportazione di un WSDL.

  • Per i servizi business con tipo di trasporto, ad esempio EJB e Flusso, viene visualizzata l'icona Genera WSDL. Fare clic su questa icona per generare un WSDL, che è possibile visualizzare o modificare in un secondo momento. Vedere Generazione di un WSDL.
Opzioni
Nella colonna Opzioni vengono visualizzate le icone riportate di seguito.
  • Fare clic sull'icona Rinomina risorsa per rinominare la risorsa. Vedere Ridenominazione delle risorse.
  • Fare clic sull'icona Sposta risorsa per spostare la risorsa. Vedere Spostamento delle risorse.
  • Fare clic sull'icona Duplica risorsa per duplicare la risorsa. Vedere Duplicazione delle risorse.
  • Fare clic sull'icona Elimina per eliminare la risorsa. Se altre risorse fanno riferimento a questa risorsa, viene visualizzata un'icona Avvertenza di eliminazione. È possibile eliminare questa risorsa associata a una conferma di avvertenza. Questa operazione potrebbe determinare conflitti a causa di riferimenti non risolti alla risorsa eliminata.
Quando si esegue una qualsiasi di queste azioni, il risultato viene salvato nella sessione corrente. Per terminare la sessione e distribuire la configurazione in fase di esecuzione, fare clic su Attiva in Centro modifiche.
