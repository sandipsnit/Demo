Visualizzazione delle risorse in un browser Web

Per visualizzare le risorse di Oracle Service Bus in un browser Web standard, usare gli URL riportati di seguito.

    • Percorso per le risorse WSDL:
      http://host:port/sbresource?WSDL/project_path/wsdlname
      o
      http://host:port/proxy_service_endpoint_URI?WSDL (per i servizi proxy HTTP basati su WSDL)
    • Percorso per le risorse del Servizio proxy:
      http://host:port/sbresource?PROXY/project_path/proxyname
      http://host:port/sbresource?ORAPROXY/project_path/proxyname (per i servizi proxy a cui sono collegati criteri di Oracle Web Services Manager)
    • Percorso per le risorse del Servizio business:
      http://host:port/sbresource?BIZ/project_path/bizname
      http://host:port/sbresource?ORABIZ/project_path/bizname (per i servizi business a cui sono collegati criteri di Oracle Web Services Manager)
    • Percorso per le risorse WSS:
      http://host:port/sbresource?POLICY/project_path/policyname
    • Percorso per le risorse MFL:
      http://host:port/sbresource?MFL/project_path/mflname
    • Percorso per le risorse SCHEMA:
      http://host:port/sbresource?SCHEMA/project_path/schemaname

    Nota: se si utilizzano caratteri speciali nei nomi di risorsa, gli URL utilizzati per esporre le risorse in Oracle Service Bus devono essere codificati in UTF-8 per applicare il carattere di escape ai caratteri speciali.
