Eliminazione dei provider di chiavi di servizio

Quando si elimina un provider di chiavi di servizio, Oracle Service Bus elimina anche le relative associazioni coppia di chiavi con alias dal provider di mapping delle credenziali PKI. Oracle Service Bus non elimina la coppia chiave-certificato dal keystore.

  1. Se questa operazione non è stata già eseguita, fare clic su Crea per creare una nuova sessione o su Modifica per accedere a una sessione esistente. Vedere Utilizzo del Centro modifiche.
  2. Se un servizio proxy è configurato per utilizzare il provider di chiavi di servizio, rimuovere quest'ultimo dal servizio proxy. Vedere Modifica delle configurazioni dei servizi proxy.
  3. Selezionare Browser risorse > Provider di chiavi servizio per visualizzare la pagina Riepilogo dei provider di chiavi servizio.
  4. Fare clic sull'icona Elimina nel campo Opzioni del provider di chiavi di servizio che si desidera eliminare. Il provider di chiavi di servizio viene eliminato nella sessione corrente. Se un servizio business o un servizio proxy è stato configurato per l'uso di un account di servizio, un'icona Avvertenza di eliminazione indica che è possibile eliminare il provider di chiavi di servizio con una conferma di avviso. Ciò può causare conflitti dovuti a riferimenti non risolti del servizio al provider di chiavi di servizio eliminato.
  5. Per terminare la sessione e distribuire la configurazione in fase di esecuzione, fare clic su Attiva in Centro modifiche.
