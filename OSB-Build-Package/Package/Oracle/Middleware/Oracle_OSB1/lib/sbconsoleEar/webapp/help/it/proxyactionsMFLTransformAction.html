Aggiunta di azioni di trasformazione MFL

Utilizzare l'azione di trasformazione MFL (Message Format Language) per convertire il contenuto del messaggio da XML a non XML o viceversa nella pipeline messaggi. Un MFL è un documento XML specializzato utilizzato per descrivere il layout dei dati binari. È un linguaggio proprietario Oracle utilizzato per definire regole di trasformazione dei dati binari formattati in dati XML o viceversa. Vedere MFL.

Prima di iniziare

Nelle istruzioni riportate di seguito si presume che sia già stato modificato un flusso di messaggi nella pagina Modifica flusso di messaggi, come descritto in Creazione, modifica e visualizzazione dei flussi di messaggi.

Per aggiungere un'azione di trasformazione MFL

  1. Spostarsi nella posizione in cui si desidera aggiungere l'azione, come descritto in Aggiunta e modifica di azioni nei flussi di messaggi.
  2. Fare clic sull'icona appropriata, quindi selezionare Aggiungi un'azione > Elaborazione messaggi > Trasformazione MFL.
  3. Dall'elenco a discesa Applica trasformazione MFL selezionare Da XML a non XML oppure Da non XML a XML in base alle esigenze.
  4. Fare clic su <Espressione>. Utilizzando l'Editor espressioni XQuery specificare la variabile sulla quale deve essere eseguita l'azione di trasformazione MFL. Questa variabile di input deve essere una variabile di testo o di tipo binario quando si esegue la trasformazione in XML e una variabile XML quando si esegue la trasformazione non XML. Il contenuto binario nel contesto del messaggio è rappresentato dall'elemento XML di contenuto binario. Questo XML deve essere il risultato dell'espressione XQuery quando il valore di input deve essere binario. Vedere Creazione e modifica delle espressioni XQuery e XPath in linea.
  5. Selezionare una delle opzioni riportate di seguito.
    • Risorsa MFL: fare clic sul collegamento <risorsa>. Viene visualizzata la pagina Seleziona MFL: selezionare la risorsa MFL statica che eseguirà l'azione di trasformazione MFL.
    • Risorsa MFL da: fare clic sul collegamento Espressione. Viene visualizzata la pagina Editor espressioni XQuery. Utilizzando l'Editor espressioni XQuery creare o modificare un'espressione XQuery per specificare in modo dinamico una risorsa MFL che eseguirà l'azione di trasformazione nel formato progetto/cartella/nomerisorsaMFL. Vedere Creazione e modifica delle espressioni XQuery e XPath in linea.
  6. Nel campo Assegna a variabile immettere il nome della variabile alla quale deve essere assegnato il risultato di questa azione di trasformazione. Il risultato sarà un elemento XML di contenuto binario.

Al termine della procedura

Una volta completata la configurazione di questa azione, proseguire configurando altre azioni o salvando la configurazione, come descritto in Aggiunta e modifica di azioni nei flussi di messaggi.
