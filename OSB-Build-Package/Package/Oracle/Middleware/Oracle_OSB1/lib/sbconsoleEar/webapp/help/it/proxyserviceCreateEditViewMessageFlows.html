Creazione, modifica e visualizzazione dei flussi di messaggi

  1. Se questa operazione non è stata già eseguita, fare clic su Crea per creare una nuova sessione o su Modifica per accedere a una sessione esistente. Vedere Utilizzo del Centro modifiche.
  2. Individuare il servizio proxy di cui si desidera visualizzare o modificare il flusso di messaggi. Effettuare una delle operazioni riportate di seguito.
    • Selezionare Explorer progetto per visualizzare la pagina Vista progetti oppure la pagina Vista progetto/cartella, quindi spostarsi tra i progetti e le cartelle per trovare il servizio proxy Pagina Vista progetto/cartella. quindi spostarsi tra i progetti e le cartelle per trovare il servizio proxy
    • Selezionare Browser risorse > Servizi proxy per visualizzare la pagina Riepilogo dei servizi proxy.
  3. Fare clic sull'icona Modifica flusso di messaggi nella riga del servizio proxy di cui si desidera visualizzare o modificare il flusso di messaggi.
    • Se non è stato ancora creato un flusso di messaggi per il servizio proxy selezionato, viene visualizzata la pagina Modifica flusso di messaggi con una singola icona sulla pagina, ovvero l'icona Servizio proxy. Si tratta del nodo di partenza per il servizio proxy. Fare clic su questa icona per iniziare a creare il flusso di messaggi.
    • Se il servizio proxy dispone già di un flusso di messaggi, la pagina contiene una rappresentazione grafica del flusso. Fare clic sulle icone per visualizzare o modificare le parti del flusso di messaggi.

      • Per informazioni sulle operazioni da eseguire per la creazione di un flusso di messaggi sulla pagina, vedere la pagina Modifica flusso di messaggi.

  4. Fare clic su Salva per eseguire il commit degli aggiornamenti nella sessione corrente.
  5. Per terminare la sessione e distribuire la configurazione in fase di esecuzione, fare clic su Attiva in Centro modifiche.
