Pagina Configurazione trasporto posta elettronica

La pagina Crea/Modifica un servizio proxy - Configurazione trasporto posta elettronica fa parte di una serie di pagine per la creazione e la modifica dei servizi proxy, come descritto in Creazione e configurazione dei servizi proxy e Modifica delle configurazioni dei servizi proxy.

Utilizzare questa pagina per configurare le impostazioni di trasporto per un servizio proxy mediante il protocollo di trasporto della posta elettronica. Nella Tabella 19-4 viene descritto come utilizzare la pagina.

Tabella 19-4 Pagina Crea/Modifica un servizio proxy - Configurazione trasporto posta elettronica 
Opzione
Per creare o modificare...
SSL obbligatorio
Selezionare questa opzione per comunicare mediante SSL (Secure Sockets Layer). I numeri di porta SSL predefiniti sono:
  • POP3(S): 995
  • IMAP: 993
Nota: affinché SSL possa funzionare, il certificato del server di posta elettronica deve essere presente nel truststore di Oracle Service Bus.
Account di servizio
Immettere un nome per l'account di servizio oppure fare clic su Sfoglia per selezionare gli account di servizio da un browser.
Server gestito
Questo campo è disponibile solo in un dominio in cluster.
Selezionare il server gestito che deve fungere da server di polling. Tutti i server gestiti possono elaborare il messaggio, ma solo uno può eseguirne il polling.
Intervallo di polling
Immettere un intervallo di polling, in secondi.
Protocollo posta elettronica
Selezionare POP3 o IMAP come tipo di server per l'account di posta elettronica.
Limite di lettura
Specificare il numero massimo di messaggi da leggere per cancellazione polling. Immettere 0 per specificare che non vi è alcun limite.
Passa in base a riferimento
Selezionare questa casella di controllo per inserire il file nella directory di archivio intermedia e passarlo come riferimento nelle intestazioni.
Per impostazione predefinita, quando si crea un nuovo servizio l'opzione Passa in base a riferimento è selezionata ed è necessario specificare la posizione della directory di archivio.
Passa allegati in base a riferimento
Selezionare questa casella di controllo per inserire gli allegati nella directory di archivio intermedia e passarli come riferimento nelle intestazioni.
Per impostazione predefinita, quando l'opzione Passa in base a riferimento è selezionata, l'opzione Passa allegati in base a riferimento viene impostata in modo implicito su true ed è necessario specificare la posizione della directory di archivio.
Azione successiva alla lettura
Selezionare l'azione che si verifica dopo la lettura di un messaggio.
  • Archivia: il messaggio viene archiviato.
  • Elimina: il messaggio viene eliminato.
  • Sposta: il messaggio viene spostato. Questa azione è disponibile solo con il protocollo IMAP.
Allegati
Selezionare il modo in cui vengono gestiti gli allegati.
  • Archivia: gli allegati vengono salvati nella directory di archivio.
  • Ignora: gli allegati vengono ignorati.
Cartella spostamento IMAP
Immettere la cartella in cui viene spostato il messaggio se il campo Azione successiva alla lettura è impostato su Sposta.
Directory di scaricamento
Immettere una posizione temporanea per lo scaricamento dei messaggi di posta elettronica.
Directory di archivio
Specifica il percorso della posizione di archivio se il campo Azione successiva alla lettura è impostato su Archivia.
Questo campo è obbligatorio se l'opzione Passa in base a riferimento o Passa allegati in base a riferimento è selezionata.
Directory degli errori
Immettere il percorso di directory del file system in cui scrivere il messaggio e qualsiasi allegato se si verifica un problema.
Codifica richiesta
Accettare l'impostazione predefinita ISO-8859-1 come codifica del set di caratteri per le richieste nei trasporti di tipo Posta elettronica oppure immettere un'altra codifica del set di caratteri.

Al termine della procedura

Fare clic su Successivo per proseguire con la configurazione di questo servizio nella pagina successiva oppure fare clic su Ultimo per esaminare e salvare questa configurazione nella pagina Crea/Modifica un servizio proxy - Riepilogo.
