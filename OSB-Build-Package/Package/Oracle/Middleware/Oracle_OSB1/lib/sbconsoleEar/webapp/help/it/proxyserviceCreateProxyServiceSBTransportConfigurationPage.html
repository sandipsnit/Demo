Pagina Configurazione trasporto SB

La pagina Crea/Modifica un servizio proxy - Configurazione trasporto SB fa parte di una serie di pagine per la creazione e la modifica dei servizi proxy, come descritto in Creazione e configurazione dei servizi proxy e Modifica delle configurazioni dei servizi proxy.

Per ulteriori informazioni, vedere "SB Transport" in Oracle Fusion Middleware Developer's Guide for Oracle Service Bus all'indirizzo http://www.oracle.com/pls/as111170/lookup?id=OSBDV997.

Utilizzare questa pagina per configurare le impostazioni di trasporto per un servizio proxy utilizzando il protocollo di trasporto SB (Service Bus). Nella tabella 19-12 viene descritto come utilizzare la pagina.

Tabella 19-12 Pagina Crea/Modifica un servizio proxy - Configurazione trasporto SB
Opzione
Per creare o modificare...
Criterio di spedizione
Selezionare un criterio di spedizione per l'endpoint o utilizzare il criterio di spedizione predefinito.
I criteri di spedizione fanno riferimento all'istanza di WLS Work Manager che si desidera utilizzare per l'endpoint del servizio per elaborare la richiesta. Per informazioni sui Work Manager, vedere "Using Work Managers to Optimize Scheduled Work" in Oracle Fusion Middleware Configuring Server Environments for Oracle WebLogic Server all'indirizzo http://www.oracle.com/pls/as111170/lookup?id=CNFGD112.
Usa SSL
Quando specificato, le richieste devono essere inviate tramite una connessione SSL.
Tuttavia, le connessioni non protette non sono vietate. L'amministratore deve chiudere tutti i protocolli non protetti sul server (ad esempio, t3 o http) per applicare rigidamente le connessioni client protette.

Al termine della procedura

Fare clic su Successivo per proseguire con la configurazione di questo servizio nella pagina successiva oppure fare clic su Ultimo per esaminare e salvare la configurazione nella pagina Crea/Modifica un servizio proxy - Riepilogo.
