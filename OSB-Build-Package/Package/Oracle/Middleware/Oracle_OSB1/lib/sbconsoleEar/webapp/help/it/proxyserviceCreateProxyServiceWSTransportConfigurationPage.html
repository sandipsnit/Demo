Pagina Configurazione trasporto WS

La pagina Crea/Modifica un servizio proxy - Configurazione trasporto WS fa parte di una serie di pagine per la creazione e la modifica dei servizi proxy, come descritto in Creazione e configurazione dei servizi proxy e Modifica delle configurazioni dei servizi proxy.

Per ulteriori informazioni, vedere "WS Transport" in Oracle Fusion Middleware Developer's Guide for Oracle Service Bus all'indirizzo http://www.oracle.com/pls/as111170/lookup?id=OSBDV1083.

Utilizzare questa pagina per configurare le impostazioni di trasporto per un servizio proxy utilizzando il protocollo di trasporto WS. Nella tabella 19-15 viene descritto come utilizzare la pagina.

Tabella 19-15 Pagina Crea/Modifica un servizio proxy - Configurazione trasporto WS
Opzione
Per creare o modificare...
Criterio di spedizione
Selezionare un criterio di spedizione per l'endpoint o utilizzare il criterio di spedizione predefinito.
Il criterio di spedizione fa riferimento all'istanza di Work Manager di WLS che si desidera utilizzare per l'endpoint del servizio. Per informazioni sui Work Manager, vedere "Using Work Managers to Optimize Scheduled Work" in Oracle Fusion Middleware Configuring Server Environments for Oracle WebLogic Server all'indirizzo http://www.oracle.com/pls/as111170/lookup?id=CNFGD112.
Conteggio nuovi tentativi
Il numero di nuovi tentativi di consegna di un messaggio alla pipeline.
Se si verifica un'eccezione non gestita nella pipeline della richiesta di un servizio proxy, il messaggio di trasporto WS in entrata verrà riconsegnato alla pipeline fino al numero di volte specificato per il conteggio dei tentativi. Questo valore è importante per l'elaborazione affidabile dei messaggi di trasporto WS.
Ritardo dei tentativi
Il numero di secondi di sospensione del sistema prima che venga ritentato l'invio di un messaggio alla pipeline dopo un errore.

Al termine della procedura

Fare clic su Successivo per proseguire con la configurazione di questo servizio nella pagina successiva oppure fare clic su Ultimo per esaminare e salvare la configurazione nella pagina Crea/Modifica un servizio proxy - Riepilogo.
