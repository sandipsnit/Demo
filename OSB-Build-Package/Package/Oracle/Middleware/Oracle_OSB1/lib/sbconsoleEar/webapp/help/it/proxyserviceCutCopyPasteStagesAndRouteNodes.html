Tagliare, copiare e incollare fasi e nodi di instradamento

Ãˆ possibile tagliare, copiare e incollare le fasi e i nodi di instradamento.

  • Per tagliare una fase o un nodo di instradamento, fare clic sulla relativa icona e selezionare Taglia o Copia.
  • Per incollare una fase tagliata o copiata da una coppia di pipeline differente nel flusso di messaggi del servizio proxy corrente o dal flusso di messaggi di un altro servizio proxy, effettuare una delle operazioni riportate di seguito.
    • Fare clic sull'icona Pipeline richieste o su Pipeline risposte, quindi fare clic su Incolla fase.
    • Fare clic sull'icona Fase in una pipeline, quindi fare clic su Incolla.
  • Per incollare un nodo di instradamento tagliato o copiato dal flusso di messaggi di un altro servizio proxy, fare clic sull'icona Nodo coppia pipeline per la coppia di pipeline, quindi fare clic su Incolla instradamento.
