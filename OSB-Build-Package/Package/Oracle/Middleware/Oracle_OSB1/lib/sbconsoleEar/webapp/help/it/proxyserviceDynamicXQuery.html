Associazione di espressioni XQuery dinamiche alle XQuery in linea

La pagina Editor espressioni XQuery consente di specificare un'espressione XQuery dinamica che restituisce in fase di esecuzione il nome di una risorsa XQuery preregistrata. Per ulteriori informazioni su questo editor, vedere Creazione e modifica delle espressioni XQuery e XPath in linea.

Per definire un'espressione XQuery dinamica, effettuare le operazioni riportate di seguito.

  1. Selezionare l'opzione XQuery dinamica.
  2. Nell'area Immetti espressione per risorsa XQuery immettere l'espressione XQuery che in fase di esecuzione restituirà il nome della risorsa XQuery preregistrata.

    3. Di seguito è riportata la sintassi per la risorsa XQuery (che rappresenta il nome completo della risorsa):

    Project/folder1/folder2/XQueryResourceName

  3. Nell'area Bind variable che usano un modello XQuery o le variabili personalizzate definire i parametri di input per la trasformazione.
    • Fare clic su Sfoglia nel campo Seleziona modello XQuery per selezionare una risorsa registrata esistente da utilizzare come modello per la forma della query (il numero e i nomi delle variabili). Dopo aver selezionato il modello, le variabili compaiono nell'area Bind variable. Il modello non viene reso persistente con la configurazione, ma può essere utilizzato come per semplificare l'indicazione delle variabili per la query.
    • Digitare un nome di variabile nel campo Aggiungi variabile personalizzata, quindi fare clic su Aggiungi. Per ciascuna variabile elencata in Nome variabile immettere un'espressione XQuery a cui mapparla. È necessario definire un mapping per ogni parametro. Ad esempio, se una trasformazione XQuery ha due parametri di input denominati uno e due, il campo Nome variabile avrà due etichette, uno e due. A ciascuna etichetta è associata una casella di testo in cui immettere l'espressione XQuery.

      • Le seguenti espressioni XQuery sono esempi di input validi per i campi delle variabili:

      $body/*[1]

      $body/po:PurchaseOrder

  4. Al termine, continuare con uno dei task riportati di seguito.
    • Fare clic su Convalida. Viene visualizzato un messaggio se l'espressione viene convalidata.
    • Fare clic su Test. Vedere Esecuzione del test XQuery.
    • Salvare o annullare le modifiche.
