Pagina Modifica handler errori

È possibile configurare la gestione degli errori a livello di flusso di messaggi, pipeline, nodo di instradamento e fase.

Configurare gli handler degli errori nella pagina Modifica handler errori. È necessario aggiungere sempre almeno una fase alla pagina per specificare il funzionamento dell'handler degli errori. Vedere i seguenti argomenti:

  • Aggiunta di handler degli errori dei servizi proxy
  • Aggiunta di handler degli errori delle pipeline
  • Aggiunta di handler degli errori della fase
  • Aggiunta di handler degli errori dei nodi di instradamento
  • Modifica degli handler degli errori
  • "Handling Errors in Message Flows" in Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus all'indirizzo http://www.oracle.com/pls/as111170/lookup?id=OSBAG232.
