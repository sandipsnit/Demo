Immissione di espressioni di confronto XQuery utilizzando l'opzione Creazione guidata

Prima di iniziare

Queste istruzioni presuppongono che si stia creando o modificando un'espressione condizionale XQuery nell'Editor condizione XQuery, come descritto in Creazione e modifica delle espressioni XQuery e XPath in linea.

Per immettere un'espressione di confronto XQuery, effettuare le operazioni riportate di seguito.

  1. In Editor condizione XQuery selezionare Costruzione guidata (sotto la barra dei pulsanti dell'area di lavoro), se questa opzione non è già selezionata.
  2. Nella casella Costruzione guidata espressione selezionare Espressione di confronto, se questa opzione non è già selezionata.
  3. Nel campo Operando immettere una variabile di contesto, una definizione di spazio di nomi o una funzione XQuery.

    4. Per creare l'operando, è possibile incollare le funzioni XQuery dalla tavolozza Funzioni XQuery e gli spazi di nomi dalla tavolozza Definizioni spazi di nomi. Vedere il passo 3 e il passo 4 in Creazione di espressioni nei campi di testo dell'area di lavoro dell'editor.

  4. Selezionare un operatore di confronto dalla lista Operatore.
  5. Nel campo Valore immettere il testo o una variabile di contesto.

    6. È necessario racchiudere il testo tra virgolette, ad esempio "true" è consentito, mentre true non lo è.

  6. Fare clic su Aggiungi. Il testo immesso viene visualizzato nel riquadro Espressioni.
  7. Ripetere i passi da 3 a 6 per creare ulteriori condizioni. Ogni condizione viene aggiunta alla fine della lista di condizioni.

    8. Quando si creano espressioni unarie, prendere in considerazione quanto riportato di seguito.

    • Quando si creano espressioni aggiuntive, assicurarsi di selezionare l'opzione And o Or nel campo Congiunzione.
    • È possibile selezionare una condizione e fare clic sulla freccia SU per spostarla in alto nella lista di condizioni o fare clic sulla freccia GIÙ per spostarla in basso nella lista di condizioni. È possibile fare clic sull'icona Modifica per aggiornare una condizione o sull'icona Elimina per eliminarla.
    • Le espressioni unarie possono essere alternate con le espressioni di confronto nella definizione complessiva di una condizione.
  8. Facoltativamente effettuare una delle operazioni riportate di seguito o tutte le operazioni elencate.
    • Fare clic su Convalida. Viene visualizzato un messaggio se l'espressione viene convalidata.
    • Fare clic su Esegui il test per eseguire il test dell'espressione. Vedere Esecuzione del test XQuery.
  9. Fare clic su Salva per chiudere l'editor e inserire l'espressione nell'azione.

Al termine della procedura

Continuare a configurare l'azione, come descritto in Aggiunta e modifica di azioni nei flussi di messaggi.
