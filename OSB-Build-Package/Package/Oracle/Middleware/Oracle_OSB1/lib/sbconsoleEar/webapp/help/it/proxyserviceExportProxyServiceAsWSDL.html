Esportazione di un WSDL associato a un servizio proxy

È possibile esportare il WSDL di un servizio proxy basato su WSDL, in modo da visualizzare o modificare il WSDL in uno strumento esterno, ad esempio IDE. Il WSDL viene esportato come file JAR.

Notare che questa operazione è diversa dalla funzionalità Esporta risorse del modulo Amministrazione sistema, che viene utilizzata per spostare e posizionare nell'area intermedia le risorse tra due domini. Vedere Esportazione di risorse.

Prima di iniziare

È possibile esportare un WSDL solo dall'esterno di una sessione. Vedere Utilizzo del Centro modifiche.

Per esportare un WSDL, effettuare le operazioni riportate di seguito.

  1. Effettuare una delle operazioni riportate di seguito.
    • Selezionare Browser risorse > Servizi proxy per visualizzare la pagina Riepilogo dei servizi proxy.
    • Selezionare Explorer progetto, quindi selezionare il progetto o la cartella contenente il servizio proxy che si desidera esportare come WSDL. Viene visualizzata la pagina Vista progetto/cartella.
  2. Fare clic sull'icona Esporta WSDL nella riga del servizio proxy di cui si desidera esportare il WSDL. Una finestra di dialogo richiede di aprire o salvare il file JAR esportato.
  3. Nella finestra di dialogo fare clic su Apri per aprire il file o su Salva per salvarlo.
