Associazione di risorse XSLT esterne alle XQuery in linea

La pagina Editor espressioni XQuery consente di selezionare una risorsa XSLT per l'esecuzione. Per ulteriori informazioni su questo editor, vedere Creazione e modifica delle espressioni XQuery e XPath in linea.

Per selezionare una risorsa XSLT per l'esecuzione, effettuare le operazioni riportate di seguito.

  1. Selezionare l'opzione Risorse XSLT.
  2. Nel campo Selezionare la risorsa XSLT da eseguire selezionare l'icona Browser XSLT.
  3. Nel Browser XSLT selezionare il pulsante di scelta associato all'XSLT da eseguire, quindi fare clic su Sottometti.
  4. Sotto il campo Bind variable vengono visualizzati un'etichetta e una casella di testo corrispondente per ogni parametro di input della trasformazione. Ogni etichetta corrisponde al nome di un parametro e ciascuna casella di testo consente di definire un'espressione XQuery da mappare al parametro. È necessario definire un mapping per ogni parametro. Ad esempio, se una trasformazione XSL ha due parametri di input denominati uno e due, il campo Mapping variabile avrà due etichette, uno e due, con una casella di testo associata in cui immettere l'espressione XQuery. Oltre al mapping per le variabili di input, è necessario specificare un'espressione XQuery per il documento di input. Il mapping viene specificato nella casella di testo con l'etichetta Documento di input.

    5. Le seguenti espressioni XQuery sono esempi di input validi per questo campo:

    $body/*[1]

    $body/po:PurchaseOrder

    Nota: il seguente nome di variabile non è una voce valida per questo campo e restituisce un'eccezione:
    body
  5. Continuare con uno dei seguenti task:
    • Creazione di spazi di nomi da utilizzare nelle espressioni in linea.
    • Creazione di strutture di variabili negli editor XQuery.
    • Creazione di espressioni nei campi di testo dell'area di lavoro dell'editor.
    • Associazione di risorse XQuery esterne alle XQuery in linea.
    • Fare clic su Convalida. Viene visualizzato un messaggio se l'espressione viene convalidata.
    • Fare clic su Test. Vedere Esecuzione del test XQuery.
