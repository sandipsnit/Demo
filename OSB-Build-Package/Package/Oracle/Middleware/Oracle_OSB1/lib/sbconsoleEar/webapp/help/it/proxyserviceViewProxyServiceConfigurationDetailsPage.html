Pagina Visualizza un servizio proxy - Dettagli configurazione

La pagina Visualizza un servizio proxy - Dettagli configurazione visualizza i dettagli di configurazione di un servizio proxy. Nella tabella 19-19 vengono descritte tutte le proprietà che possono comparire in questa pagina (le proprietà variano in base ai dettagli del servizio proxy).

Le categorie elencate in questa pagina corrispondono alle pagine Crea/Modifica un servizio proxy utilizzate per creare e modificare le configurazioni dei servizi proxy, come descritto in Crea/Modifica un servizio proxy - Riferimento di pagina.

Fare clic sul collegamento Modifica accanto al nome di una categoria per visualizzare la pagina di configurazione associata.
