Informazioni sui layout e sui task dell'editor XQuery

L'Editor espressioni XQuery, l'Editor condizione XQuery e l'Editor espressioni XPath sono costituiti dai seguenti componenti:

  • Tavolozze
  • Area di lavoro
  • Property Inspector

Tavolozze

Il pannello sinistro di ciascun editor contiene le tavolozze elencate di seguito. In uno degli editor, fare clic sul nome di una tavolozza per visualizzarla. Ogni tavolozza contiene entità che è possibile inserire nelle espressioni negli editor.

  • Nella tavolozza Definizioni spazi di nomi sono elencati gli spazi di nomi Oracle Service Bus predefiniti, variabili definiti dall'utente. È possibile definire nuovi spazi di nomi, che vengono quindi aggiunti alla lista di spazi di nomi definiti dall'utente.

    • Per ulteriori informazioni, vedere:

    • Creazione di espressioni nei campi di testo dell'area di lavoro dell'editor
    • Creazione di spazi di nomi da utilizzare nelle espressioni in linea
  • La tavolozza Funzioni XQuery elenca un set di funzioni XQuery standard. Quando si inserisce una funzione in un'espressione, vengono utilizzati dei segnaposto per i valori dei parametri da fornire.

    • Per ulteriori informazioni, vedere:

    • Creazione di espressioni nei campi di testo dell'area di lavoro dell'editor
    • Creazione di funzioni XPath personalizzate negli editor XQuery
    • Associazione di risorse XQuery esterne alle XQuery in linea
  • La tavolozza Strutture variabili fornisce un set di strumenti per l'inserimento di variabili e percorsi delle variabili utilizzando le espressioni XPath.

    • Le strutture delle variabili sono rappresentazioni grafiche delle variabili o dei percorsi delle variabili visualizzate nell'editor. È possibile utilizzarle per creare espressioni XQuery in linea facenti riferimento al contenuto della variabile.

    Nota: Le strutture delle variabili non creano le variabili. Le variabili vengono create in fase di esecuzione come destinazione dell'azione di assegnazione nella fase.

    Oracle Service Bus fornisce variabili di contesto dei messaggi predefinite (attachments, body, header, outbound e inbound), di cui è possibile visualizzare il contenuto come strutture di variabili. È anche possibile definire strutture di variabili personalizzate.

    Per maggiori informazioni, vedere

    • Creazione di espressioni nei campi di testo dell'area di lavoro dell'editor
    • Creazione di strutture di variabili negli editor XQuery

Area di lavoro

La parte destra della pagina fornisce un'area di lavoro per la creazione dell'espressione XQuery, della condizione XQuery o dell'XPath. L'area di lavoro è differente nei tre editor.

Property Inspector

In tutti e tre gli editor il Property Inspector viene visualizzato nella parte inferiore destra della pagina. Quando si seleziona un elemento da aggiungere all'espressione da una delle tavolozze, l'elemento selezionato compare nel Property Inspector. È possibile quindi incollarlo nell'area di lavoro. Vedere Creazione di espressioni nei campi di testo dell'area di lavoro dell'editor.
