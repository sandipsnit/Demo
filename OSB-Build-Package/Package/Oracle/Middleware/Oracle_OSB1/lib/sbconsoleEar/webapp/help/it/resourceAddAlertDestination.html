Aggiunta di destinazioni di alert

Una destinazione di alert è un indirizzo di destinazione per le notifiche di alert in Oracle Service Bus. È possibile creare le risorse di destinazione degli alert nella pagina Crea destinazione alert.

  1. Se questa operazione non è stata già eseguita, fare clic su Crea per creare una nuova sessione o su Modifica per accedere a una sessione esistente. Vedere Utilizzo del Centro modifiche.
  2. Selezionare Explorer progetto, quindi selezionare un progetto o una cartella in cui aggiungere la destinazione di alert. Viene visualizzata la pagina Vista progetto/cartella.
  3. Dall'elenco a discesa Crea risorsa selezionare Destinazione alert per visualizzare la pagina Crea destinazione alert.
  4. Nel campo Nome risorsa immettere un nome univoco per questa destinazione di alert. Questo campo è obbligatorio.

    Vedere Limitazioni di denominazione delle risorse per le istruzioni di denominazione.
  5. Nel campo Descrizione risorsa immettere una descrizione per la destinazione di alert.
  6. Selezionare una o tutte le destinazioni riportate di seguito per includerle nella risorsa della destinazione di alert.
    7. Nota: Gli alert vengono aggregati nella console di amministrazione di Oracle Service Bus in fase di esecuzione, dove possono essere visualizzati in Operazioni > Dashboard.
    • Interrupt SNMP: se si specifica Interrupt SNMP, gli alert vengono inviati come interrupt SNMP e possono essere elaborati da qualsiasi sistema di monitoraggio aziendale di terze parti.
    • Report: se si specifica Report, gli alert vengono inviati al modulo Report di Oracle Service Bus ed è possibile acquisirli utilizzando un provider di report che può essere sviluppato mediante il provider di API di report di Oracle Service Bus. Ciò consente alle terze parti di ricevere ed elaborare gli alert nel codice Java personalizzato.
    • Alert log: se si seleziona Sì, gli alert inviati alla destinazione di alert vengono registrati nell'alert log.
  7. Aggiungere un destinatario di posta elettronica alla definizione della risorsa della destinazione di alert. Vedere Aggiunta di destinatari di posta elettronica.
  8. Aggiungere una destinazione JMS alla definizione della risorsa della destinazione di alert. Vedere Aggiunta di destinazioni JMS.
  9. Fare clic su Salva. La nuova destinazione di alert viene salvata nella sessione corrente.
  10. Per terminare la sessione e distribuire la configurazione in fase di esecuzione, fare clic su Attiva in Centro modifiche.
