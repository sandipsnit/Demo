Uso del Browser risorse

Il Browser risorse fornisce gli strumenti per individuare le risorse ed eseguire azioni semplici sulle stesse, quali la visualizzazione, la modifica, l'esportazione e l'eliminazione.

  1. Per visualizzare il Browser risorse selezionare Browser risorse nel riquadro di navigazione sinistro.
