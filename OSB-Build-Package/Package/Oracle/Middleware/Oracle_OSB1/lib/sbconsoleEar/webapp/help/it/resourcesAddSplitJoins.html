Aggiunta di Split-Join

Prima di iniziare

Gli Split-Join richiedono un file .flow che viene creato quando si crea uno Split-Join utilizzando i plug-in di Oracle Service Bus per Eclipse o quando si importa un file JAR di configurazione dello Split-Join nel plug-in. Il file .flow potrebbe avere riferimenti ai servizi proxy, ai servizi business o alle risorse WSDL esterne. Le risorse dalle quali dipende lo Split-Join devono essere presenti sul server prima di attivare lo Split-Join.

  1. Se questa operazione non è stata già eseguita, fare clic su Crea per creare una nuova sessione o su Modifica per accedere a una sessione esistente. Vedere Utilizzo del Centro modifiche.
  2. Selezionare Explorer progetto, quindi selezionare un progetto o una cartella in cui aggiungere lo Split-Join. Viene visualizzata la pagina Vista progetto/cartella.
  3. Dall'elenco a discesa Crea risorsa selezionare Split-Join per visualizzare la pagina Crea una nuova risorsa Split-Join.

    4. Specificare i parametri mostrati nella tabella 9-2.

    Tabella 9-2 Parametri dello Split-Join
    Proprietà
    Per creare o modificare...
    Nome risorsa
    Immettere un nome univoco per questo Split-Join.

    Vedere Limitazioni di denominazione delle risorse per le istruzioni di denominazione.
    Descrizione risorsa
    Immettere una descrizione per lo Split-Join.
    Split-Join
    Fare clic su Sfoglia per individuare un file .flow sul proprio sistema. Generalmente è possibile trovarlo in MW_HOME\user_projects\workspaces\your_workspace.
    Criterio di spedizione
    Selezionare l'istanza del Work Manager di WebLogic Server utilizzata per creare i thread (rami paralleli) per l'esecuzione dello Split-Join. Se non esistono altri Work Manager, viene utilizzato quello predefinito.

  4. Fare clic su Salva. Lo Split-Join viene creato e salvato nella sessione corrente.
    5. Nota: le risorse dalle quali dipende lo Split-Join devono essere presenti prima di attivare lo Split-Join.
  5. Per terminare la sessione e distribuire la configurazione in fase di esecuzione, fare clic su Attiva in Centro modifiche.
