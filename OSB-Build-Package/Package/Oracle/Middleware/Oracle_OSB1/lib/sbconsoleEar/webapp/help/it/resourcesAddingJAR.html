Aggiunta di file JAR

  1. Se questa operazione non è stata già eseguita, fare clic su Crea per creare una nuova sessione o su Modifica per accedere una sessione esistente. Vedere Utilizzo del Centro modifiche.
  2. Selezionare Explorer progetto, quindi selezionare un progetto o una cartella in cui aggiungere il file JAR. Viene visualizzata la pagina Vista progetto/cartella.
  3. Dall'elenco a discesa Crea risorsa selezionare JAR da Utility per visualizzare la pagina Crea una nuova risorsa JAR.
  4. Nel campo Nome risorsa immettere un nome univoco per questo file JAR. Questo campo è obbligatorio.

    Vedere Limitazioni di denominazione delle risorse per le istruzioni di denominazione.
  5. Nel campo Descrizione risorsa immettere una descrizione per il file JAR.
  6. Nel campo File di origine JAR effettuare una delle operazioni riportate di seguito.
    • Immettere il percorso del file JAR.
    • Fare clic su Sfoglia per individuare e importare un file JAR esistente.

      • Questo campo è obbligatorio.

  7. Fare clic su Salva. La risorsa JAR viene creata e salvata nella sessione corrente.

    8. Se il file JAR viene caricato senza problemi, il contenuto del file JAR diventa disponibile per la visualizzazione. Per ulteriori informazioni su come visualizzare i dettagli dei file JAR, vedere Modifica dei file JAR.

  8. Per terminare la sessione e distribuire la configurazione in fase di esecuzione, fare clic su Attiva in Centro modifiche.
