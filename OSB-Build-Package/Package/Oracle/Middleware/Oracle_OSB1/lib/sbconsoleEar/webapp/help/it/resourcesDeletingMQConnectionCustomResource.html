Eliminazione di connessioni MQ

  1. Se questa operazione non è stata già eseguita, fare clic su Crea per creare una nuova sessione o su Modifica per accedere a una sessione esistente. Vedere Utilizzo del Centro modifiche.
  2. Selezionare Browser risorse > Connessioni MQ per visualizzare la pagina Riepilogo delle connessioni MQ.
  3. Fare clic sull'icona Elimina nel campo Opzioni della risorsa di connessione MQ che si desidera eliminare. La risorsa di connessione MQ viene eliminata nella sessione corrente. Quando altre risorse fanno riferimento a questa risorsa viene visualizzata un'icona Avvertenza di eliminazione. È possibile eliminare la risorsa con una conferma di avvertenza. Ciò può causare conflitti a causa di riferimenti non risolti alla risorsa eliminata.
  4. Per terminare la sessione e distribuire la configurazione in fase di esecuzione, fare clic su Attiva in Centro modifiche.
