Modifica delle dipendenze dei file JAR

In Oracle Service Bus, le risorse JAR possono contenere classi che dipendono da altre classi in file JAR differenti. Utilizzare la pagina Modifica dipendenze JAR per aggiungere o rimuovere le risorse JAR di riferimento.

  1. Se questa operazione non è stata già eseguita, fare clic su Crea per creare una nuova sessione o su Modifica per accedere a una sessione esistente. Vedere Utilizzo del Centro modifiche.
  2. Individuare il file JAR, come descritto in Individuazione dei file JAR.
  3. Fare clic sul nome del file JAR.

    4. Nella pagina Visualizza una risorsa JAR - Configurazione JAR un indicatore numerico nella scheda Dipendenze visualizza il numero di file JAR di riferimento configurati per la risorsa JAR oppure zero se non ve ne sono disponibili.

  4. Selezionare la scheda Dipendenze per visualizzare i file JAR di riferimento o aggiungerli.

    5. La pagina delle dipendenze JAR include le informazioni mostrate nella tabella 7-4.

    Tabella 7-4 Informazioni sulle dipendenze JAR
    Proprietà
    Descrizione
    Nome
    Il nome del file JAR di riferimento. Il nome è un collegamento alla pagina Visualizza una risorsa JAR - Configurazione JAR. Vedere Modifica dei file JAR.
    Percorso
    Il percorso è costituito dal nome del progetto e dal nome della cartella in cui risiede il file JAR.

  5. Per aggiungere o rimuovere i file JAR di riferimento, fare clic su Modifica.

    6. La pagina Modifica dipendenze JAR include le informazioni mostrate nella tabella 7-5.

    Tabella 7-5 Dettagli delle dipendenze JAR
    Proprietà
    Descrizione
    Classpath Manifest
    Voci Class-Path nel file manifest JAR.
    Nome
    Il nome e il percorso della risorsa JAR di riferimento.
    Opzioni
    • Utilizzare le frecce GIÙ e SU per modificare l'ordine di precedenza dei file JAR di riferimento.
    • Fare clic sull'icona Elimina per rimuovere un file JAR di riferimento.

  6. Fare clic su Aggiungi JAR.
  7. Nel Browser JAR, individuare o selezionare le risorse JAR di riferimento e fare clic su Sottometti.

    8. Ad esempio, se si modifica il file JAR A e dipende da entrambi i file JAR B e C, (sia per riferimento diretto che indiretto) è necessario aggiungere i file JAR B e JAR C alla tabella delle dipendenze.

    Utilizzare le frecce nella colonna Opzioni per riordinare la gerarchia di file JAR di riferimento o l'icona Elimina per rimuoverli.

  8. Fare clic su Salva per eseguire il commit degli aggiornamenti nella sessione corrente.
  9. Per terminare la sessione e distribuire la configurazione in fase di esecuzione, fare clic su Attiva in Centro modifiche.
