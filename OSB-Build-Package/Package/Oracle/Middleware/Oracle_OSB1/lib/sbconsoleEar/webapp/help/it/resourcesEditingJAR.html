Modifica dei file JAR

Utilizzare la pagina Visualizza una risorsa JAR - Configurazione per aggiornare i file JAR. L'aggiornamento di un file JAR essenzialmente fa in modo che la risorsa punti a una nuova versione del file JAR, in quanto non è possibile modificare un file JAR utilizzando Oracle Service Bus.

  1. Se questa operazione non è stata già eseguita, fare clic su Crea per creare una nuova sessione o su Modifica per accedere a una sessione esistente. Vedere Utilizzo del Centro modifiche.
  2. Individuare il file JAR, come descritto in Individuazione dei file JAR.
  3. Fare clic sul nome del file JAR. Nella pagina Visualizza una risorsa JAR - Configurazione JAR vengono visualizzate le informazioni mostrate nella tabella 7-2.

    4. Tabella 7-2 Informazioni sui file JAR 
    Proprietà
    Descrizione
    Autore ultima modifica
    L'utente che ha importato il file JAR nella configurazione.
    Data ultima modifica
    La data e l'ora in cui l'utente ha importato il file JAR nella configurazione. Fare clic sul collegamento relativo alla data e all'ora per visualizzare la cronologia delle modifiche della risorsa. Vedere la pagina Visualizza cronologia modifiche.
    Riferimenti
    Il numero di file JAR a cui fa riferimento il file JAR corrente. Se tali riferimenti esistono, fare clic sul collegamento numerico per visualizzare una lista di file JAR. Vedere Visualizzazione dei riferimenti alle risorse.
    Riferimento di
    Il numero di oggetti che fanno riferimento al file JAR. Se tali riferimenti esistono, fare clic sul collegamento numerico per visualizzare una lista degli oggetti. Vedere Visualizzazione dei riferimenti alle risorse.
    Descrizione
    Una descrizione del file JAR, se disponibile.

    La configurazione JAR viene visualizzata per impostazione predefinita e include le informazioni mostrate nella tabella 7-3.

    Tabella 7-3 Dettagli della configurazione JAR
    Proprietà
    Descrizione
    Configurazione JAR
    La lista di classi e interfacce nel file JAR.
    Fare clic sul segno più (+) accanto al nome di una classe Java per visualizzare una lista di metodi inclusi nella classe.

  4. Per apportare una modifica ai campi, fare clic su Aggiorna.

    5. L'unico campo modificabile è Descrizione. Non è possibile modificare un file JAR in Oracle Service Bus. Per aggiornare il file JAR, è necessario selezionare un nuovo file JAR da utilizzare. Per ulteriori informazioni, vedere Aggiunta di file JAR.

  5. Fare clic su Salva per eseguire il commit degli aggiornamenti nella sessione corrente.
  6. Per terminare la sessione e distribuire la configurazione in fase di esecuzione, fare clic su Attiva in Centro modifiche.
Nota: se si cambia il file JAR utilizzato da un servizio business EJB (puntare la risorsa JAR a una nuova versione del file JAR), è necessario ridistribuire l'EJB. È quindi necessario modificare gli eventuali servizi EJB che utilizzano il file JAR e riselezionare la risorsa JAR, salvarla e attivarla. In questo modo, viene creato un nuovo package del servizio business EJB in modo che utilizzi il nuovo file JAR.
Nota: le azioni di callout Java e i servizi basati su Tuxedo, tuttavia, selezioneranno automaticamente il nuovo file JAR.
