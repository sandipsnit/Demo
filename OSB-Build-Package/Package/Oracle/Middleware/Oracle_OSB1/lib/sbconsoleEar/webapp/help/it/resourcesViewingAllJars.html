Individuazione dei file JAR

  1. Effettuare una delle operazioni riportate di seguito.
    • Selezionare Explorer progetto per visualizzare la pagina Vista progetti o la pagina Vista progetto/cartella. Spostarsi quindi tra i progetti e le cartelle per trovare il file JAR.
    • Selezionare Browser risorse > JAR. Nella pagina Riepilogo dei file JAR vengono visualizzate le informazioni mostrate nella tabella 7-1. Per una descrizione più dettagliata delle proprietà, vedere Modifica dei file JAR.
  2. Per limitare il numero di elementi nella lista o individuare un file JAR specifico, cercare un file JAR utilizzando i filtri del nome e del percorso. Nei campi Nome e Percorso immettere il nome e il percorso della destinazione o delle destinazioni di ricerca, quindi fare clic su Cerca.

    3. Il percorso è costituito dal nome del progetto e dal nome della cartella in cui risiede il file JAR.

    La ricerca supporta i caratteri jolly. Ad esempio, è possibile immettere "j*" nel campo Nome per ottenere una lista dei file JAR che iniziano con "j".

    Fare clic su Vis. tutto per rimuovere i filtri di ricerca e visualizzare tutti i file JAR.

    Tabella 7-1 Dettagli del file JAR
    Proprietà
    Descrizione
    Nome JAR
    Il nome univoco assegnato al file JAR. Il nome è un collegamento alla pagina Visualizza una risorsa JAR -Configurazione JAR. Vedere Modifica dei file JAR.
    Percorso
    Il percorso è costituito dal nome del progetto e dal nome della cartella in cui risiede il file JAR. È un collegamento al progetto o alla cartella che contiene questa risorsa. Vedere Qualificazione dei nomi di risorsa utilizzando progetti e cartelle.
    N. di classi
    Il numero di classi contenute nel file JAR.
    Dimensione
    La dimensione del file JAR in byte.
    Opzioni
    Fare clic sull'icona Elimina per eliminare un file JAR specifico. Vedere Eliminazione di file JAR.
