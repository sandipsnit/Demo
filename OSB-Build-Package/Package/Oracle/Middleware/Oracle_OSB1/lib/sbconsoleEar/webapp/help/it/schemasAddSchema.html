Aggiunta di schemi XML

  1. Se questa operazione non è stata già eseguita, fare clic su Crea per creare una nuova sessione o su Modifica per accedere a una sessione esistente. Vedere Utilizzo del Centro modifiche.
  2. Selezionare Explorer progetto, quindi selezionare un progetto o una cartella in cui aggiungere il nuovo schema XML. Viene visualizzata la pagina Vista progetto/cartella.
  3. Dall'elenco a discesa Crea risorsa selezionare Schema XML per visualizzare la pagina Crea una nuova risorsa di schema XML.
  4. Nel campo Nome risorsa immettere un nome univoco per questo schema XML. Questo campo è obbligatorio.

    Vedere Limitazioni di denominazione delle risorse per le istruzioni di denominazione.
  5. Nel campo Descrizione risorsa immettere una descrizione per lo schema XML.
  6. Nel campo Schema XML effettuare una delle operazioni riportate di seguito.
    • Immettere il testo per il nuovo schema XML.
    • Fare clic su Sfoglia per individuare e importare uno schema XML esistente.
    • Copiare il testo da uno schema XML esistente e incollarlo in questo campo.

      • Questo campo è obbligatorio.

  7. Fare clic su Salva. Il nuovo schema XML viene salvato nella sessione corrente.

    8. Quando si fa clic su Salva il sistema visualizza gli eventuali riferimenti non risolti per il nuovo schema XML. Vedere Visualizzazione dei riferimenti agli schemi XML non risolti.

  8. Per terminare la sessione e distribuire la configurazione in fase di esecuzione, fare clic su Attiva in Centro modifiche.
