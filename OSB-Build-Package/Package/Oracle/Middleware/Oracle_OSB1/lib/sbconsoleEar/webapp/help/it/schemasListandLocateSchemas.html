Individuazione di schemi XML

  1. Effettuare una delle operazioni riportate di seguito.
    • Selezionare Explorer progetto per visualizzare la pagina Vista progetti o la pagina Vista progetto/cartella. Spostarsi quindi tra i progetti e le cartelle per trovare lo schema XML.
    • Selezionare Browser risorse > Schemi XML. Nella pagina Riepilogo degli schemi XML vengono visualizzate le informazioni mostrate nella tabella 4-1. Per una descrizione più dettagliata delle proprietà, vedere Modifica degli schemi XML.
  2. Per individuare uno schema XML specifico, effettuare le operazioni riportate di seguito.
    • Fare clic sull'icona Apri per visualizzare ulteriori filtri di ricerca. Se si specificano più criteri di ricerca, vengono restituiti solo i risultati che soddisfano tutti i criteri.
    • Per limitare il numero di elementi nella lista, è possibile filtrarli in base al nome, al percorso e allo spazio di nomi. Nei campi Nome, Percorso e Spazio di nomi immettere il nome, il percorso e lo spazio di nomi delle destinazioni di ricerca, quindi fare clic su Cerca.

      • Il percorso corrisponde al nome del progetto e al nome della cartella in cui risiede lo schema XML.

    • Fare clic su Vis. tutto per rimuovere i filtri di ricerca e visualizzare tutti gli schemi XML.

      • Tabella 4-1 Informazioni sugli schemi XML
      Proprietà
      Descrizione
      Nome schema XML
      Il nome univoco assegnato allo schema XML. Il nome è un collegamento alla pagina Dettagli schema XML. Vedere Modifica degli schemi XML.
      Percorso
      Il percorso è costituito dal nome del progetto e dal nome della cartella in cui risiede lo schema XML. È un collegamento al progetto o alla cartella che contiene la risorsa. Vedere Qualificazione dei nomi di risorsa utilizzando progetti e cartelle.
      Spazio di nomi schema XML
      Lo spazio di nomi utilizzato per qualificare le definizioni incluse nello schema XML.
      Opzioni
      Fare clic sull'icona Elimina per eliminare uno schema XML specifico. Vedere Eliminazione di schemi XML.
