Risoluzione dei riferimenti agli schemi XML non risolti

Utilizzare la pagina Modifica i riferimenti di una risorsa di schema XML per risolvere i riferimenti agli schemi XML non risolti configurando il mapping per i riferimenti agli schemi XML come le importazioni degli schemi XML.

  1. Se questa operazione non è stata già eseguita, fare clic su Crea per creare una nuova sessione o su Modifica per accedere a una sessione esistente. Vedere Utilizzo del Centro modifiche.
  2. Nella pagina Riepilogo degli schemi XML, nella colonna Nome schema XML, fare clic sul nome dello schema XML appropriato per visualizzare i dettagli del riferimento non risolto. Viene visualizzata la pagina Visualizza dettagli schema XML.
  3. Fare clic su Modifica riferimenti per visualizzare la pagina Modifica i riferimenti di una risorsa di schema XML.
  4. Nel campo Tipo di risorsa selezionare Schema XML.
  5. Fare clic su Sfoglia. Viene visualizzato il Browser schemi XML.
  6. Nel Browser schemi XML selezionare uno schema XML, quindi selezionare una definizione nel riquadro Definizioni.
  7. Fare clic su Sottometti. Lo schema XML selezionato viene visualizzato nel campo Nome risorsa.
  8. Fare clic su Salva per risolvere il riferimento.
  9. Nella pagina Dettagli schema XML fare clic su OK. Lo stato dello schema XML viene visualizzato come Valido.
