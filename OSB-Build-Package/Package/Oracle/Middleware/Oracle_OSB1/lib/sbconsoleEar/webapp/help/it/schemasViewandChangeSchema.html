Modifica di schemi XML

  1. Se questa operazione non è stata già eseguita, fare clic su Crea per creare una nuova sessione o su Modifica per accedere a una sessione esistente. Vedere Utilizzo del Centro modifiche.
  2. Individuare lo schema XML, come descritto in Individuazione degli schemi XML.
  3. Fare clic sul nome dello schema XML. Nella pagina Visualizza dettagli schema XML vengono visualizzate le informazioni mostrate nelle tabelle 4-2, 4-3 e 4-4.

    4. Tabella 4-2 Dettagli dello schema XML 
    Proprietà
    Descrizione
    Autore ultima modifica
    L'utente che ha creato lo schema XML o lo ha importato nella configurazione.
    Data ultima modifica
    La data e l'ora in cui l'utente ha creato lo schema XML o lo ha importato nella configurazione. Fare clic sul collegamento relativo alla data e all'ora per visualizzare la cronologia delle modifiche della risorsa. Vedere la pagina Visualizza cronologia modifiche.
    Riferimenti
    Il numero di oggetti a cui fa riferimento lo schema XML. Se tali riferimenti esistono, fare clic sul collegamento numerico per visualizzare una lista di oggetto. Vedere Visualizzazione dei riferimenti alle risorse.
    Riferimento di
    Il numero di oggetti che fanno riferimento allo schema XML. Se tali riferimenti esistono, fare clic sul collegamento numerico per visualizzare una degli oggetti. Ad esempio, se allo schema XML fa riferimento un WSDL specifico, il WSDL è elencato come riferimento quando si fa clic sul collegamento. Vedere Visualizzazione dei riferimenti alle risorse.
    Descrizione
    Una descrizione dello schema XML, se presente.

    La vista strutturale dello schema XML viene visualizzata per impostazione predefinita e include le informazioni mostrate nella tabella 4-3.

    Tabella 4-3 Dettagli della struttura XML
    Proprietà
    Descrizione
    Spazio di nomi di destinazione
    Lo spazio di nomi utilizzato per qualificare le definizioni incluse nello schema XML.
    Definizioni schema XML
    Gli attributi e i gruppi associati allo schema XML:
    • Tipi di schema XML
    • Elementi schema XML
    • Importazioni schema XML
    • Inclusioni/ridefinizioni schema XML
    Stato schema
    Stato dello schema XML:
    • Valido
      Uno schema XML viene mostrato come valido quando sono specificate tutte le posizioni per gli schemi XML inclusi dallo schema XML corrente. Inoltre, sono specificate tutte le posizioni per gli schemi XML nidificati (gli schemi XML inclusi successivamente dagli schemi XML inclusi).
    • Non valido
      Uno schema XML viene mostrato come non valido quando una o più posizioni per lo schema XML non sono specificate. Inoltre, la posizione per uno o più schemi XML nidificati (gli schemi XML inclusi successivamente dagli schemi XML inclusi) potrebbe non essere specificata. Per risolvere uno schema XML non valido, fare clic su Modifica riferimenti. Vedere Risoluzione dei riferimenti agli schemi XML non risolti.

  4. Fare clic su Vista testo per visualizzare una vista testo dei dettagli dello schema XML. La vista testo include le informazioni mostrate nella tabella 4-4.

    5. Tabella 4-4 Dettagli del testo dello schema XML
    Proprietà
    Descrizione
    Spazio di nomi di destinazione
    Lo spazio di nomi utilizzato per qualificare le definizioni incluse nello schema XML.
    Schema XML
    Il testo dello schema XML.

  5. Per apportare modifiche ai campi, fare clic su Modifica. Per le descrizioni dei campi, vedere Aggiunta di schemi XML.
  6. Apportare le modifiche appropriate.
  7. Fare clic su Salva per eseguire il commit degli aggiornamenti nella sessione corrente.
  8. Per terminare la sessione e distribuire la configurazione in fase di esecuzione, fare clic su Attiva in Centro modifiche.
