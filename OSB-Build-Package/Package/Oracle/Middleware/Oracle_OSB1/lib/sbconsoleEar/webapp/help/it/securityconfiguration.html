Configurazione sicurezza

Il modulo Configurazione sicurezza consente di creare e modificare i dati di sicurezza utilizzati nella sicurezza in entrata e nella sicurezza amministrativa di Oracle Service Bus.

La sicurezza a livello di messaggio e la sicurezza a livello di trasporto in entrata utilizzano i dati di utenti, gruppi e ruoli per autenticare le richieste client in entrata. Vengono applicati i criteri di controllo a livello di accesso per determinare quali utenti autenticati sono autorizzati a utilizzare i servizi proxy e i servizi business.

La sicurezza amministrativa utilizza i dati di utenti, gruppi e ruoli per determinare gli utenti autenticati autorizzati a creare o modificare i dati di configurazione Oracle Service Bus o a monitorare le prestazioni di Oracle Service Bus. Per ulteriori informazioni, vedere Informazioni su utenti, gruppi, ruoli di sicurezza e criteri. Vedere anche "Configuring Administrative Security" in Oracle Fusion Middleware Developer's Guide for Oracle Service Bus all'indirizzo http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539.

Nota: non è possibile esportare utenti, gruppi o ruoli quando si esporta una configurazione poiché questi oggetti si trovano nelle aree di memorizzazione dei provider di sicurezza. È necessario creare di nuovo questi oggetti quando si importa la configurazione esportata o utilizzare gli strumenti di WebLogic Server (se disponibili) per esportarli e importarli.
