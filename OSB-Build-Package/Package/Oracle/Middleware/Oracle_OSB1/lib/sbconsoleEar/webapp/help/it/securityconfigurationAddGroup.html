Aggiunta di gruppi

  1. Eseguire il login alla console di amministrazione di Oracle Service Bus come utente con privilegi di amministrazione di WebLogic Server. Solo gli utenti con il ruolo di amministratore possono modificare i dati di configurazione della sicurezza. Vedere "Configuring Administrative Security" in Oracle Fusion Middleware Developer's Guide for Oracle Service Bus all'indirizzo http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539.
  2. Selezionare Configurazione sicurezza > Gruppi per visualizzare la pagina Riepilogo dei gruppi.
  3. Fare clic su Aggiungi nuovo.

    4. È possibile aggiungere un gruppo dall'interno o dall'esterno di una sessione.

  4. Nel campo Nome gruppo immettere un nome univoco. Non è possibile immettere spazi o caratteri speciali. Questo campo è obbligatorio.
  5. Nel campo Provider di autenticazione selezionare il provider di autenticazione.
  6. Nel campo Appartenenza a gruppi selezionare un gruppo al quale appartiene il gruppo.
    1. Selezionare un gruppo dal campo Gruppi disponibili.
    2. Fare clic sulla freccia per spostare il gruppo nel campo Gruppi correnti.
  7. Fare clic su Salva per creare il gruppo.

    8. La console di amministrazione di Oracle Service Bus salva il gruppo, che diventa immediatamente disponibile per tutte le sessioni. Se ci si trova in una sessione quando si aggiunge il gruppo e quindi si elimina la sessione, la console di amministrazione di Oracle Service Bus non elimina il nuovo gruppo.
