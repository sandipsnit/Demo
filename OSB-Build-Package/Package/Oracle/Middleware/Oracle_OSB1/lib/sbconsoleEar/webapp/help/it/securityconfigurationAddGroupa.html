Modifica di gruppi

Utilizzare la pagina Visualizza dettagli gruppo per visualizzare e modificare i dettagli di un gruppo specifico.

  1. Eseguire il login alla console di amministrazione di Oracle Service Bus come utente con privilegi di amministrazione di WebLogic Server. Solo gli utenti con il ruolo di amministratore possono modificare i dati di configurazione della sicurezza. Vedere "Configuring Administrative Security" in Oracle Fusion Middleware Developer's Guide for Oracle Service Bus all'indirizzo http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539.
  2. Individuare il gruppo. Vedere Individuazione dei gruppi.
  3. Fare clic sul nome del gruppo. Nella pagina Visualizza dettagli gruppo vengono visualizzate le informazioni mostrate nella tabella 24-4.

    4. Tabella 24-4 Dettagli del gruppo
    Proprietà
    Descrizione
    Nome gruppo
    Il nome di questo gruppo.
    Provider di autenticazione
    Il provider di autenticazione che contiene la definizione del gruppo.
    Gruppi
    Il gruppo al quale appartiene il gruppo corrente.

  4. Per modificare i dettagli del gruppo, fare clic su Riconfigura per visualizzare la pagina Modifica dettagli gruppo.

    5. È possibile modificare i dettagli del gruppo dall'interno o dall'esterno di una sessione.

  5. Apportare le modifiche appropriate al campo Appartenenza gruppi. Per le descrizioni dei campi, vedere Aggiunta di gruppi.

    6. Non è possibile modificare il campo Nome gruppo.

  6. Fare clic su Salva modifiche per aggiornare il gruppo.

    7. La console di amministrazione di Oracle Service Bus aggiorna i dettagli del gruppo e l'aggiornamento diventa immediatamente disponibile per tutte le sessioni. Se ci si trova in una sessione quando si aggiorna il gruppo e quindi si elimina la sessione, la console di amministrazione di Oracle Service Bus non elimina gli aggiornamenti.
