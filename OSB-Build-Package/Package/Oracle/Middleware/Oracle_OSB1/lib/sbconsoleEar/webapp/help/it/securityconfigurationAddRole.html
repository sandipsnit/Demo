Aggiunta di ruoli

  1. Eseguire il login alla console di amministrazione di Oracle Service Bus come utente con privilegi di amministrazione di WebLogic Server. Solo gli utenti con il ruolo di amministratore possono modificare i dati di configurazione della sicurezza. Vedere "Configuring Administrative Security" in Oracle Fusion Middleware Developer's Guide for Oracle Service Bus all'indirizzo http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539.
  2. Selezionare Configurazione sicurezza > Ruoli per visualizzare la pagina Ruoli globali.
  3. Fare clic su Nuovo.

    4. È possibile aggiungere un ruolo dall'interno o dall'esterno di una sessione.

  4. Nel campo Nome ruolo immettere un nome univoco. Questo campo è obbligatorio.

    5. Assicurarsi che non ci siano spazi o caratteri < > nel nome del ruolo di sicurezza. I nomi dei ruoli di sicurezza distinguono tra maiuscole e minuscole. In base alla convenzione Oracle, tutti i nomi dei ruoli di sicurezza sono al singolare.

  5. Per creare il ruolo, fare clic su OK.

    6. La console di amministrazione di Oracle Service Bus salva il ruolo e quest'ultimo diventa immediatamente disponibile per tutte le sessioni. Se ci si trova in una sessione quando si aggiunge il ruolo e quindi si elimina la sessione, la console di amministrazione di Oracle Service Bus non elimina il nuovo ruolo.

    Quando si fa clic su OK per creare il ruolo, il passo successivo consiste nel definire le condizioni che devono essere soddisfatte affinché il ruolo venga applicato.

  6. Continuare con le operazioni descritte in Definizione delle condizioni del ruolo.
