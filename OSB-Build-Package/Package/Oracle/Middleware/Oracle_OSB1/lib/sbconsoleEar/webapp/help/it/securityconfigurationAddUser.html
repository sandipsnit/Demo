Aggiunta di utenti

  1. Eseguire il login alla console di amministrazione di Oracle Service Bus come utente con privilegi di amministrazione di WebLogic Server. Solo gli utenti con il ruolo di amministratore possono modificare i dati di configurazione della sicurezza. Vedere "Configuring Administrative Security" in Oracle Fusion Middleware Developer's Guide for Oracle Service Bus all'indirizzo http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539.
  2. Selezionare Configurazione sicurezza per visualizzare la pagina Riepilogo degli utenti.
  3. Fare clic su Aggiungi nuovo per visualizzare la pagina Crea un nuovo utente - Configurazione generale.

    4. È possibile aggiungere un utente dall'interno o dall'esterno di una sessione.

  4. Nome campo Nome utente immettere un nome univoco. Questo campo è obbligatorio.
  5. Nel campo Password immettere una password. Questo campo è obbligatorio.
    6. Nota: i provider di autenticazione possono imporre una lunghezza minima della password. Per un utente definito nel provider di autenticazione WebLogic, la lunghezza minima predefinita della password è 8 caratteri. È possibile personalizzare questa impostazione utilizzando la console di amministrazione di WebLogic Server. Il provider di autenticazione WebLogic è configurato nel realm di sicurezza predefinito con il nome DefaultAuthenticator.
  6. Nel campo Conferma password immettere la stessa password immessa per il campo Password. Questo campo è obbligatorio.
  7. Nel campo Provider di autenticazione selezionare il provider di autenticazione per l'utente.

    8. Se nel realm di sicurezza sono configurati più provider di autenticazione, verranno visualizzati nella lista. Selezionare il database del provider di autenticazione nel quale memorizzare le informazioni relative al nuovo utente.

  8. Nel campo Appartenenza a gruppi selezionare un gruppo per l'utente.
    1. Selezionare un gruppo dal campo Gruppi disponibili.
    2. Fare clic sulla freccia per spostare il gruppo nel campo Gruppi correnti.
  9. Fare clic su Salva per creare l'utente.

    10. La console di amministrazione di Oracle Service Bus salva l'utente e quest'ultimo diventa immediatamente disponibile per tutte le sessioni. Se ci si trova in una sessione quando si aggiunge l'utente e quindi si elimina la sessione, la console di amministrazione di Oracle Service Bus non elimina il nuovo utente.
