Eliminazione dei gruppi

Utilizzare la pagina Riepilogo dei gruppi per eliminare un gruppo selezionato o più gruppi.

  1. Eseguire il login alla console di amministrazione di Oracle Service Bus come utente con privilegi di amministrazione di WebLogic Server. Solo gli utenti con il ruolo di amministratore possono modificare i dati di configurazione della sicurezza. Vedere "Configuring Administrative Security" in Oracle Fusion Middleware Developer's Guide for Oracle Service Bus all'indirizzo http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539.
  2. Selezionare Configurazione sicurezza per visualizzare la pagina Riepilogo dei gruppi.
  3. Selezionare il gruppo da eliminare. Se necessario, è possibile selezionare più gruppi.

    4. È possibile eliminare i gruppi dall'interno o dall'esterno di una sessione.

  4. Fare clic su Elimina. Viene visualizzato un messaggio che richiede di confermare l'eliminazione del gruppo.
  5. Per eliminare un gruppo, fare clic su OK.

    6. La console di amministrazione di Oracle Service Bus elimina il gruppo. Se ci si trova in una sessione quando si elimina il gruppo e quindi si elimina la sessione, la console di amministrazione di Oracle Service Bus non annulla l'eliminazione del gruppo.

  6. In alternativa, è possibile fare clic sull'icona Elimina nella colonna Opzioni del gruppo da eliminare.
