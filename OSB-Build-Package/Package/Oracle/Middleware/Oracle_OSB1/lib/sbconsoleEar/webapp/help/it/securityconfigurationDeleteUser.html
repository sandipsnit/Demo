Eliminazione degli utenti

Utilizzare la pagina Riepilogo degli utenti per eliminare uno o più utenti selezionati.

  1. Eseguire il login alla console di amministrazione di Oracle Service Bus come utente con privilegi di amministrazione di WebLogic Server. Solo gli utenti con il ruolo di amministratore possono modificare i dati di configurazione della sicurezza. Vedere "Configuring Administrative Security" in Oracle Fusion Middleware Developer's Guide for Oracle Service Bus all'indirizzo http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539.
  2. Selezionare Configurazione sicurezza per visualizzare la pagina Riepilogo degli utenti.
  3. Selezionare l'utente da eliminare. Se necessario, è possibile selezionare più utenti.

    4. È possibile eliminare un utente dall'interno o dall'esterno di una sessione.

  4. Fare clic su Elimina. Viene visualizzato un messaggio che richiede di confermare l'eliminazione dell'utente.
  5. Per eliminare l'utente, fare clic su OK.

    6. La console di amministrazione di Oracle Service Bus elimina l'utente. Se ci si trova in una sessione quando si elimina l'utente e quindi si elimina la sessione, la console di amministrazione di Oracle Service Bus non elimina gli aggiornamenti.

  6. In alternativa, è possibile fare clic sull'icona Elimina nella colonna Opzioni dell'utente da eliminare.
