Modifica dei criteri di accesso a livello di trasporto

Utilizzare la pagina Visualizza dettagli criteri per modificare i criteri di controllo dell'accesso a livello di trasporto di un servizio proxy. Nella pagina vengono visualizzate le informazioni mostrate nella tabella 24-10.

Tabella 24-10 Dettagli dei criteri 
Proprietà
Descrizione
Nome servizio proxy
Visualizza il nome del servizio proxy per il quale è stato selezionato Controllo accesso trasporto nella pagina Sicurezza.
Provider
Visualizza i provider di autorizzazione configurati per il realm di sicurezza.
Condizioni criteri
Visualizza le condizioni che determinano gli utenti per i quali il servizio proxy elaborerà le richieste.

  1. Eseguire il login alla console di amministrazione di Oracle Service Bus come utente con privilegi di amministrazione di WebLogic Server. Solo gli utenti con il ruolo di amministratore possono modificare i dati di configurazione della sicurezza. Vedere "Configuring Administrative Security" in Oracle Fusion Middleware Developer's Guide for Oracle Service Bus all'indirizzo http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539.
  2. Individuare i criteri di accesso. Vedere Individuazione dei criteri di controllo dell'accesso.
  3. Nella pagina Sicurezza, nella colonna Controllo accesso trasporto, fare clic sul nome del servizio proxy. Viene visualizzato l'editor dei criteri.

    4. È possibile modificare i criteri di controllo dell'accesso da una sessione.

  4. Nel campo Provider di autorizzazioni selezionare un provider di autorizzazione. Oracle consiglia di selezionare XACMLAuthorizer.
    5. Nota: Oracle Service Bus non ha approvato il supporto del provider di autorizzazione predefinito WebLogic. Oracle consiglia di utilizzare il provider di autorizzazione WebLogic XACML.
  5. Aggiungere le condizioni dei criteri. Vedere Aggiunta di condizioni dei criteri.
  6. Dopo aver completato l'immissione delle condizioni nella sezione Condizioni criteri, fare clic su Salva.
