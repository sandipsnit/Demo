Individuazione dei criteri di controllo dell'accesso

La pagina Sicurezza fornisce un collegamento ai criteri di controllo dell'accesso per un servizio proxy nel dominio Oracle Service Bus corrente.

In questa pagina non sono elencati i servizi proxy creati nella sessione ma non ancora attivati. Se si desidera modificare i criteri di controllo dell'accesso per un nuovo servizio proxy, innanzi tutto attivare la sessione in cui è stato creato il servizio proxy.

  1. Se si desidera individuare i criteri di controllo dell'accesso per un nuovo servizio proxy, attivare la sessione in cui è stato creato il servizio proxy.
  2. Nella colonna Controllo accesso selezionare il nome del servizio proxy in Controllo accesso trasporto oppure il nome del servizio proxy service o di un'operazione specifica in Controllo accesso messaggio.
