Individuazione dei ruoli

  1. Selezionare Configurazione sicurezza > Ruoli. La pagina Ruoli globali visualizza le informazioni mostrate nella tabella 24-5. Per una descrizione più dettagliata delle proprietà, vedere Modifica dei ruoli.
  2. Per limitare il numero di elementi nella lista o individuare un ruolo specifico, scorrere le pagine. Utilizzare i controlli di pagina sopra o sotto la tabella.

    3. Tabella 24-5 Informazioni sul ruolo
    Proprietà
    Descrizione
    Nome ruolo
    Il nome del ruolo. Il nome è un collegamento alla pagina Visualizza dettagli ruolo. Vedere Modifica dei ruoli.
    Nome provider
    Il provider di autenticazione per il gruppo.
