Individuazione degli utenti

  1. Selezionare Configurazione sicurezza > Utenti. Nella pagina Riepilogo degli utenti vengono visualizzate le informazioni mostrate nella tabella 24-1. Per una descrizione più dettagliata delle proprietà, vedere Modifica degli utenti.
  2. Per limitare il numero di elementi nella lista o individuare un utente specifico, è possibile filtrare la lista in base al nome dell'utente. Immettere l'intero nome dell'utente o parte di esso nel campo Nome e fare clic su Cerca.

    3. È possibile utilizzare il carattere jolly asterisco (*). Gli altri caratteri jolly non sono supportati.

    Fare clic su Vis. tutto per rimuovere i filtri di ricerca e visualizzare tutti gli utenti.

    Tabella 24-1 Informazioni sull'utente
    Proprietà
    Descrizione
    Nome utente
    Il nome assegnato all'utente. Il nome è un collegamento alla pagina Visualizza dettagli utente. Vedere Modifica degli utenti.
    Appartenenza a gruppi
    Il nome del gruppo al quale appartiene l'utente. Il nome è un collegamento alla pagina Visualizza dettagli gruppo. Vedere Modifica dei gruppi.
    Provider di autenticazione
    Il provider di autenticazione per l'utente.
    Opzioni
    Fare clic sull'icona Elimina per eliminare un utente specifico. Vedere Eliminazione di utenti.
