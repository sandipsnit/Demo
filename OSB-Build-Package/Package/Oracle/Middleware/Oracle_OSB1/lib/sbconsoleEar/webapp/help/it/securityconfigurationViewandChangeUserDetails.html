Modifica degli utenti

Utilizzare la pagina Visualizza dettagli utente per visualizzare e modificare i dettagli di un utente specifico.

  1. Eseguire il login alla console di amministrazione di Oracle Service Bus come utente con privilegi di amministrazione di WebLogic Server. Solo gli utenti con il ruolo di amministratore possono modificare i dati di configurazione della sicurezza. Vedere "Configuring Administrative Security" in Oracle Fusion Middleware Developer's Guide for Oracle Service Bus all'indirizzo http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539.
  2. Individuare l'utente. Vedere Individuazione degli utenti.
  3. Fare clic sul nome utente. La pagina Visualizza dettagli utente visualizza le informazioni mostrate nella tabella 24-2.

    4. Tabella 24-2 Dettagli utente
    Proprietà
    Descrizione
    Nome utente
    Il nome dell'utente.
    Provider di autenticazione
    Il provider di autenticazione che contiene la definizione utente.
    Appartenenza a gruppi
    Il nome del gruppo al quale appartiene l'utente.

  4. Per modificare i dettagli dell'utente, fare clic su Riconfigura per visualizzare la pagina Modifica dettagli utente.

    5. È possibile modificare i dettagli utente dall'interno o dall'esterno di una sessione.

  5. Apportare le modifiche appropriate ai campi Nuova password, Conferma password e Appartenenza a gruppi. Per le descrizioni dei campi, vedere Aggiunta di utenti.

    6. Non è possibile modificare il campo Nome utente.

  6. Fare clic su Salva modifiche per aggiornare l'utente.

    7. La console di amministrazione di Oracle Service Bus aggiorna i dettagli dell'utente e l'aggiornamento diventa immediatamente disponibile per tutte le sessioni. Se ci si trova in una sessione quando si aggiorna l'utente e quindi si elimina la sessione, la console di amministrazione di Oracle Service Bus non elimina gli aggiornamenti.
