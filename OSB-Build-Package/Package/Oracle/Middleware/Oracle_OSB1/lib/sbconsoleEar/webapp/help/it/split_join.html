Split-Join

Gli Split-Join consentono di inviare i richiami dei messaggi in parallelo (oltre che in modo sequenziale) e di aggregare le risposte. Per ulteriori informazioni sugli Split-Join, vedere "Improving Service Performance with Split-Join" in Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus all'indirizzo http://www.oracle.com/pls/as111170/lookup?id=OSBAG291. Vedere anche "Working with Split-Join" all'indirizzo http://www.oracle.com/pls/as111170/lookup?id=OSBDV193 e "Split-Join User Interface Reference" all'indirizzo http://www.oracle.com/pls/as111170/lookup?id=OSBDV573 in Oracle Fusion Middleware Developer's Guide for Oracle Service Bus.

Si accede agli Split-Join tramite i servizi business del tipo di trasporto di flusso. Per ulteriori informazioni, vedere la pagina Configurazione trasporto di flusso.
