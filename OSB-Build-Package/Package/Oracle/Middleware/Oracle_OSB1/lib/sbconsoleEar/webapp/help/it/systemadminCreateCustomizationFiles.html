Creazione di file di personalizzazione

La pagina Crea file di personalizzazione fornisce un metodo semplice per generare un file di personalizzazione per un set di risorse o progetti selezionati. È possibile quindi utilizzare questo file come punto di partenza per apportare le modifiche desiderate specificando i valori effettivi per un ambiente.

È possibile utilizzare i file di personalizzazione per apportare modifiche ai valori di ambiente e per modificare i riferimenti nelle risorse. Tali file possono includere le personalizzazioni per tutti i valori di ambiente presenti nelle risorse selezionate, inclusi i tipi di valore di ambiente complessi definiti nella classe EnvValueTypes. Inoltre, includono un tipo di personalizzazione di riferimento per la modifica dei riferimenti alle risorse nelle risorse con dipendenze.

Lo schema di personalizzazione (Customization.xsd) che descrive i tipi di personalizzazione disponibili nella seguente posizione nell'installazione di Oracle Service Bus:

OSB_ORACLE_HOME\modules\com.bea.common.configfwk_version.jar.

  1. Selezionare Amministrazione sistema > Crea file di personalizzazione.

    2. La pagina Crea file di personalizzazione mostra la lista di oggetti nella propria configurazione. Vengono visualizzati il nome e il tipo di ciascun oggetto.

  2. Selezionare i progetti o le risorse che si desidera includere nel file di personalizzazione.
    1. Espandere le cartelle del progetto. Vengono visualizzati il nome e il tipo per ogni risorsa contenuta nel progetto.
    2. Selezionare le caselle di controllo associate ai progetti o alle risorse che si desidera includere nel file di personalizzazione.
    3. Deselezionare le caselle di controllo associate ai progetti o alle risorse che non si desidera includere nel file di personalizzazione.
  3. Fare clic su Crea file per creare un file XML di personalizzazione.
  4. Nella finestra di dialogo Scaricamento file fare clic su Apri per aprire il file di personalizzazione o fare clic su Salva per salvare il file XML sul desktop.
