Modifica dei provider JNDI

Utilizzare questa pagina per visualizzare i dettagli di un provider JNDI e modificare la configurazione, se necessario. Inoltre, è possibile reimpostare la cache provider JNDI ed eseguire il test della connessione al provider JNDI.

  1. Se questa operazione non è stata già eseguita, fare clic su Crea per creare una nuova sessione o su Modifica per accedere a una sessione esistente. Vedere Utilizzo del Centro modifiche.
  2. Individuare il provider JNDI. Vedere Visualizzazione dei provider JNDI.
  3. Fare clic sul nome del provider JNDI.

    4. Nella pagina Visualizza provider JNDI - Configurazione vengono visualizzate le informazioni sul provider JNDI mostrate nella tabella 30-2.

    Tabella 30-2 Informazioni sul provider JNDI
    Proprietà
    Descrizione
    Autore ultima modifica
    L'utente che ha creato o modificato la risorsa provider JNDI o l'ha importata nella configurazione.
    Data ultima modifica
    La data e l'ora in cui l'utente ha creato o modificato la risorsa del provider JNDI o l'ha importata nella configurazione. Fare clic sul collegamento relativo alla data e all'ora per visualizzare la cronologia delle modifiche della risorsa. Vedere la pagina Visualizza cronologia modifiche.
    Riferimenti
    Il numero di oggetti a cui fa riferimento il provider JNDI. Se tali riferimenti esistono, fare clic sul collegamento numerico per visualizzare una lista dei riferimenti. Vedere Visualizzazione dei riferimenti alle risorse.
    Riferimento di
    Il numero di oggetti che fanno riferimento al provider JNDI. Se tali riferimenti esistono, fare clic sul collegamento numerico per visualizzare una lista dei riferimenti. Vedere Visualizzazione dei riferimenti alle risorse.
    Descrizione
    Una descrizione del provider JNDI.
    Cache JNDI
    Abilitato o Disabilitato. Se questa opzione è abilitata, il contesto JNDI e gli oggetti JNDI vengono inseriti nella cache localmente. Ciò migliora le prestazioni durante la ricerca di un oggetto. Oracle consiglia di abilitare la cache JNDI.
    URL provider
    L'URL per il provider JNDI.
    Timeout richiesta JNDI
    Il timeout della richiesta JNDI in millisecondi. L'impostazione predefinita è zero (0) e indica nessun timeout.
    Nome utente e password
    Il nome utente utilizzato per accedere al provider JNDI protetto, con la possibilità di modificare la password.

  4. Fare clic su Modifica per visualizzare la pagina Modifica provider JNDI - [provider-name].

    5. Il campo Nome non è modificabile.

  5. Nel campo Descrizione è possibile modificare la descrizione breve del provider JNDI.
  6. Nel campo URL provider è possibile modificare l'URL del provider JNDI.
  7. È possibile modificare i campi Nome utente e Password.

    8. Se il provider JNDI è stato configurato con un nome utente e una password, il campo Nome utente non è modificabile. È possibile modificare i campi Nuova password e Conferma password. Se il provider JNDI non è stato configurato con un nome utente e una password, il campo Nome utente è modificabile ed è possibile immettere un nuovo nome utente.

  8. Fare clic su Salva per eseguire il commit degli aggiornamenti nella sessione corrente.
  9. Per terminare la sessione e distribuire la configurazione in fase di esecuzione, fare clic su Attiva in Centro modifiche.
