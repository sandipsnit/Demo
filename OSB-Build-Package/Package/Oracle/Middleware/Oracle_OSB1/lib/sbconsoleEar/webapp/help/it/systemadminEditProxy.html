Modifica dei server proxy

Utilizzare questa pagina per visualizzare i dettagli di un server proxy e modificare la configurazione, se richiesto.

  1. Se questa operazione non è stata già eseguita, fare clic su Crea per creare una nuova sessione o su Modifica per accedere a una sessione esistente. Vedere Utilizzo del Centro modifiche.
  2. Individuare il server proxy. Vedere Individuazione dei server proxy.
  3. Fare clic sul nome del server proxy.

    4. Nella pagina Visualizza server proxy - Configurazione vengono visualizzate le informazioni mostrate nella tabella 30-6.

    Tabella 30-6 Informazioni sul server proxy
    Proprietà
    Descrizione
    Autore ultima modifica
    L'utente che ha creato o modificato la risorsa server proxy.
    Data ultima modifica
    La data e l'ora in cui l'utente ha creato o modificato la risorsa del server proxy. Fare clic sul collegamento relativo alla data e all'ora per visualizzare la cronologia delle modifiche della risorsa. Vedere la pagina Visualizza cronologia modifiche.
    Riferimenti
    Il numero di oggetti a cui fa riferimento il server proxy. Se tali riferimenti esistono, fare clic sul collegamento numerico per visualizzare una lista dei riferimenti. Vedere Visualizzazione dei riferimenti alle risorse.
    Riferimento di
    Il numero di oggetti che fanno riferimento al server proxy. Se tali riferimenti esistono, fare clic sul collegamento numerico per visualizzare una lista dei riferimenti. Vedere Visualizzazione dei riferimenti alle risorse.
    Descrizione
    La descrizione del server proxy, se disponibile.
    Parametri host-porta
    I dettagli su ciascun server proxy nella risorsa del server proxy, incluse le informazioni riportate di seguito.
    • Host server: il nome host o l'indirizzo IP del server proxy.

      • Il nome host del server per il server proxy Oracle Service Bus deve essere identico al nome host del server proxy effettivo.

    • Porta testo in chiaro: il numero di porta con testo in chiaro del server proxy.
    • Porta SSL: il numero di porta SSL del server proxy.
    Nome utente
    Il nome utente utilizzato per l'autenticazione proxy.

  4. Fare clic su Modifica per visualizzare la pagina Modifica server proxy - [server-name].

    5. Il campo Nome non è modificabile.

  5. Nel campo Descrizione è possibile modificare la descrizione breve del server proxy.
  6. Nella sezione Parametri host-porta è possibile aggiungere un nuovo server proxy immettendo le informazioni riportate di seguito.
    1. Nel campo Host server immettere il nome host o l'indirizzo IP del server proxy. Questo campo è obbligatorio.

      2. Il nome host del server per il server proxy Oracle Service Bus deve essere identico al nome host del server proxy effettivo.

    2. Nel campo Porta testo in chiaro immettere il numero di porta con testo in chiaro del server proxy.
    3. Nel campo Porta SSL immettere il numero di porta SSL del server proxy. È necessario immetter un numero di porta con testo in chiaro o SSL.
    4. Fare clic su Aggiungi.

      5. È possibile configurare più server proxy per ogni risorsa server proxy. Ciò consente a Oracle Service Bus di eseguire il bilanciamento del carico e offrire funzioni di tolleranza degli errori per la risorsa server proxy.

      È anche possibile modificare o eliminare una configurazione del server proxy esistente effettuando le operazioni descritte di seguito.

    • Fare clic sull'icona Modifica nel campo Opzioni del server proxy da modificare. Modificare i campi Host server, Porta testo in chiaro e Porta SSL e fare clic su Aggiorna.
    • Fare clic sull'icona Elimina nel campo Opzioni del server proxy da eliminare.
  7. È possibile modificare i campi Nome utente e Password utilizzati per l'autenticazione proxy.
  8. Fare clic su Salva per eseguire il commit degli aggiornamenti nella sessione corrente.
  9. Per terminare la sessione e distribuire la configurazione in fase di esecuzione, fare clic su Attiva in Centro modifiche.
