Modifica dei server SMTP

Utilizzare questa pagina per visualizzare i dettagli di un server SMTP e modificare la configurazione, se richiesto.

  1. Se questa operazione non è stata già eseguita, fare clic su Crea per creare una nuova sessione o su Modifica per accedere a una sessione esistente. Vedere Utilizzo del Centro modifiche.
  2. Individuare il server SMTP. Vedere Individuazione dei server SMTP.
  3. Fare clic sul nome del server SMTP.

    4. Nella pagina Visualizza server SMNP - Configurazione vengono visualizzate le informazioni mostrate nella tabella 30-4.

    Tabella 30-4 Informazioni sul server SMTP 
    Proprietà
    Descrizione
    Autore ultima modifica
    L'utente che ha creato o modificato la risorsa server SMTP.
    Data ultima modifica
    La data e l'ora in cui l'utente ha creato o modificato la risorsa del server SMTP. Fare clic sul collegamento relativo alla data e all'ora per visualizzare la cronologia delle modifiche della risorsa. Vedere la pagina Visualizza cronologia modifiche.
    Riferimenti
    Il numero di oggetti a cui fa riferimento il server SMTP. Se tali riferimenti esistono, fare clic sul collegamento numerico per visualizzare una lista dei riferimenti. Vedere Visualizzazione dei riferimenti alle risorse.
    Riferimento di
    Il numero di oggetti che fanno riferimento al server SMTP. Se tali riferimenti esistono, fare clic sul collegamento numerico per visualizzare una lista dei riferimenti. Vedere Visualizzazione dei riferimenti alle risorse.
    Descrizione
    La descrizione del server SMTP, se disponibile.
    URL del server
    L'URL che punta alla posizione del server SMTP.
    Numero porta
    Il numero di porta per il server SMTP. Per impostazione predefinita, si tratta della porta 25.
    Nome utente
    Il nome utente utilizzato per accedere a un server SMTP protetto.

  4. Fare clic su Modifica per visualizzare la pagina Modifica server SMTP - [server-name].

    5. Il campo Nome non è modificabile.

  5. Nel campo Descrizione è possibile modificare la descrizione breve del server SMTP.
  6. Nel campo URL server è possibile modificare l'URL del server SMTP.
  7. Nel campo Numero porta è possibile modificare il numero della porta.
  8. È possibile modificare i campi Nome utente e Password.
  9. Fare clic su Salva per eseguire il commit degli aggiornamenti nella sessione corrente.
  10. Per terminare la sessione e distribuire la configurazione in fase di esecuzione, fare clic su Attiva in Centro modifiche.

Se si rinomina una risorsa SMTP, il nuovo nome viene aggiornato automaticamente nei servizi business di posta elettronica che fanno riferimento alla risorsa SMTP nell'URI endpoint.
