Esecuzione dei file di personalizzazione

Utilizzare questa pagina per eseguire un file XML di personalizzazione precedentemente salvato sul sistema. I file di personalizzazione forniscono un metodo pratico per modificare l'ambiente durante la distribuzione, dallo sviluppo al posizionamento nell'area intermedia, dal posizionamento nell'area intermedia alla produzione, o durante la fase di progettazione.

È necessario essere in una sessione per eseguire un file di personalizzazione.

  1. Se questa operazione non è stata già eseguita, fare clic su Crea per creare una nuova sessione o su Modifica per accedere a una sessione esistente. Vedere Utilizzo del Centro modifiche.
  2. Selezionare Amministrazione sistema > Esegui file di personalizzazione.
  3. Selezionare un file di personalizzazione da eseguire.
    1. Nel campo Nome file fare clic su Sfoglia per individuare la directory in cui è memorizzato il file di personalizzazione.
    2. Selezionare il file XML, quindi fare clic su Apri.
  4. Fare clic su Successivo. Vengono visualizzati i dettagli di personalizzazione mostrati nella tabella 31-2.

    5. Tabella 31-2 Dettagli della personalizzazione
    Proprietà
    Descrizione
    Operazione
    • Trova e sostituisci: consente di modificare i valori di ambiente effettuando sostituzioni delle stringhe.
    • Mappa riferimenti: consente di mappare i riferimenti esistenti trovati nelle risorse ad altri riferimenti.
    • Imposta valori di ambiente: consente di modificare i valori di ambiente per le risorse.
    Applica a
    Nome del servizio, del progetto, della cartella o della risorsa da personalizzare.
    Il nome è un collegamento alla pagina Dettagli configurazione del servizio o della risorsa o al progetto o alla cartella in Explorer progetto.
    Descrizione
    Una descrizione di ciascuna personalizzazione.

  5. Per personalizzare solo le risorse, i servizi, i progetti o le cartelle modificati nella sessione corrente, selezionare la casella di controllo Solo elementi modificati nella sessione corrente.

    6. Se si limitano le personalizzazioni soltanto alle risorse modificate nella sessione corrente, la colonna Applica a viene aggiornata con le risorse modificate nella sessione corrente. Se si limitano le personalizzazioni di un progetto o una cartella, le risorse modificate nella sessione corrente nel progetto o nella cartella specifica vengono visualizzate nella colonna Applica a.

  6. Fare clic su Esegui per eseguire il commit degli aggiornamenti nella sessione corrente.

    7. Per visualizzare i dettagli delle personalizzazioni, all'interno della sessione selezionare Visualizza modifiche nel Centro modifiche e fare clic sul task Personalizzazione. Vedere Visualizzazione delle modifiche alla configurazione.

  7. Per terminare la sessione e distribuire la configurazione in fase di esecuzione, fare clic su Attiva in Centro modifiche.
