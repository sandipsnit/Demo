Esportazione di risorse

Utilizzare la pagina Esporta risorse per esportare i progetti o le risorse tramite il comando Esporta progetti o Esporta risorse. Se si sceglie di esportare un progetto completo, è possibile selezionare uno o più progetti per l'esportazione. È possibile espandere i progetti per vedere le cartelle e le risorse, ma la selezione è disponibile solo a livello di progetto. Inoltre, non è possibile selezionare Includi dipendenze.

È possibile esportare i progetti e le risorse sia all'interno che all'esterno di una sessione. Se si esegue l'esportazione all'interno di una sessione, le risorse sono risorse di sessione e la configurazione può essere incompleta o presentare conflitti. Se si esegue l'esportazione all'esterno di una sessione, le risorse sono risorse di base attivate.

Oracle Service Bus non può esportare gli utenti, i gruppi o i ruoli creati nel modulo Configurazione sicurezza della console di amministrazione di Oracle Service Bus. Non può inoltre esportare le mappe delle credenziali o altri dati dei provider di sicurezza creati nella console di amministrazione di WebLogic Server. Per esportare questi dati, utilizzare la console di amministrazione di WebLogic Server. Vedere "Migrating Security Data" in Oracle Fusion Middleware Securing Oracle WebLogic Server all'indirizzo http://www.oracle.com/pls/as111170/lookup?id=SECMG319.

  1. Selezionare Amministrazione sistema > Esporta risorse.

    2. La pagina Esporta risorse visualizza la lista degli oggetti nella propria configurazione. Vengono visualizzati il nome, il tipo, l'ora dell'ultima modifica e i riferimenti per ciascun oggetto.

    Per i servizi, l'indicatore orario dell'ultima modifica riflette le modifiche apportate alla definizione del servizio, nonché la configurazione del trasporto o della pipeline del servizio. Non riflette, tuttavia, le modifiche apportate agli alert definiti per il servizio.

  2. Selezionare Esporta progetti o Esporta risorse.
    3. Nota: l'esportazione dei progetti può causare l'eliminazione delle risorse quando si importa il file JAR del progetto completo. Per ulteriori informazioni, vedere Importazione delle risorse.
  3. Selezionare i progetti o le risorse da esportare. Se si seleziona Esporta progetti, non è possibile deselezionare le caselle di controllo per le singole risorse.
    1. Espandere le cartelle del progetto. Vengono visualizzati il nome, il tipo, l'ora dell'ultima modifica e i riferimenti per ciascuna risorsa contenuta nel progetto.
    2. Selezionare le caselle di controllo associate ai progetti o alle risorse da esportare.
    3. Deselezionare le caselle di controllo associate ai progetti o alle risorse che non si desidera esportare.
  4. Se si seleziona Esporta risorse, selezionare o deselezionare la casella di controllo Includi dipendenze. Utilizzare questa opzione per esportare le eventuali altre risorse a cui fa riferimento la risorsa corrente.
  5. Fare clic su Esporta per creare un file JAR ed esportarlo.

Al termine della procedura

Se applicabile, continuare con le operazioni descritte in Specifica delle impostazioni di sicurezza dell'esportazione.
