Specifica delle impostazioni di sicurezza dell'esportazione

Se l'esportazione include un account di servizio o un provider di chiavi di servizio, viene visualizzata la pagina Esporta risorse - Impostazioni di sicurezza.

Per ogni account di servizio, Oracle Service Bus esporta il nome utente e la password o la mappa di utente locale e utente remoto (a seconda dei dati memorizzati nell'account di servizio). Per ogni provider di chiavi di servizio, Oracle Service Bus esporta l'associazione coppia di chiavi con alias dal provider di mapping delle credenziali PKI; non esporta le chiavi private, i certificati o altri dati dai keystore. I dati del keystore devono essere esportati utilizzando gli strumenti offerti dal fornitore del keystore.

Se si desidera che Oracle Service Bus cifri i dati di nome utente e password che si trovano nell'account di servizio, nel provider di chiavi di servizio, nel registro UDDI, nel provider JNDI o nelle risorse del provider SMTP prima di esportarli nel file JAR, effettuare le operazioni riportate di seguito.

  1. Selezionare la casella di controllo Proteggi dati sensibili.
  2. Immettere una password e confermarla.
