Importazione di risorse

Utilizzare la pagina Importa risorse per importare un file JAR di progetto e di risorsa completo precedentemente esportato da un altro dominio Oracle Service Bus. Si può scegliere di importare solo un subset dei dati esportati. Se la risorsa esiste già nel sistema di importazione, verrà aggiornata. Altrimenti, verrà creata. Le risorse vengono pianificate per l'eliminazione solo se il file JAR da importare è un file JAR di progetto completo e nello stesso progetto nel sistema di importazione sono presenti risorse che non si trovano nel file JAR importato. Non vengono eliminate le risorse presenti in altri progetti.

È possibile importare le risorse a livello di risorsa anche se sono state esportate a livello di progetto. Ad esempio, anche se le risorse di sistema sono state esportate in un file JAR di progetto completo, è possibile deselezionarle durante l'importazione.

Non è possibile esportare utenti, gruppi, ruoli o certificati durante l'esportazione di una configurazione. Pertanto, è necessario creare questi oggetti nuovamente quando si importa una configurazione esportata.

Nota: per informazioni sull'importazione delle risorse che non fanno parte di una configurazione Oracle Service Bus, vedere Caricamento delle risorse da un file Zip e Caricamento delle risorse da un URL.
  1. Se questa operazione non è stata già eseguita, fare clic su Crea per creare una nuova sessione o su Modifica per accedere a una sessione esistente. Vedere Utilizzo del Centro modifiche.
  2. Selezionare Amministrazione sistema > Importa risorse.
  3. Selezionare il file JAR di un progetto o una risorsa esportata.
    1. Nel campo Nome file fare clic su Sfoglia per individuare la directory in cui è memorizzato il file JAR.
    2. Selezionare il file JAR, quindi fare clic su Apri.
  4. Fare clic su Successivo. Continuare con le operazioni descritte in Selezione di progetti e risorse da importare.
