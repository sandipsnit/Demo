Specifica delle impostazioni di sicurezza dell'importazione

Se il file JAR contiene risorse con dati di nome utente e password cifrati al momento dell'esportazione, viene visualizzata la pagina Importa risorse - Impostazioni di sicurezza.

  1. Immettere la password utilizzata per cifrare i dati (vedere Esportazione di risorse).

    2. Se non si conosce la password utilizzata per cifrare i dati della risorsa, fare clic sul pulsante Prec. e deselezionare la casella di controllo accanto alla risorsa. È possibile importare tutte le altre risorse non cifrate senza conoscere la password.

  2. Fare clic su Fine importazione.

    3. Viene visualizzata la finestra popup Elaborazione dei dati di configurazione in corso. Prima di eseguire altre operazioni nella console di amministrazione di Oracle Service Bus, è necessario attendere la chiusura di questa finestra. La finestra viene chiusa automaticamente quando il sistema completa l'importazione degli oggetti. Viene visualizzata la pagina Importa risorse con un messaggio indicante il completamento della distribuzione dei dati.

Al termine della procedura

Continuare con le operazioni descritte in Controllo del riepilogo dell'importazione.
