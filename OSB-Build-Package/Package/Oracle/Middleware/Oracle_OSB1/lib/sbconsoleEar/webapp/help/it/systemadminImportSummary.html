Controllo del riepilogo dell'importazione

Le informazioni sul riepilogo dell'importazione sono mostrate nella tabella 28-3.

Tabella 28-3 Informazioni sul riepilogo dell'importazione
Proprietà
Descrizione
Stato
Indica se l'importazione della risorsa è stata completata.
È possibile che un WSDL sia contrassegnato temporaneamente come non valido finché non vengono importate tutte le relative dipendenze. Tuttavia, quando l'azione di importazione viene completata, il servizio di configurazione esegue di nuovo la convalida dei file contrassegnati come non validi.
Nome
Il nome della risorsa.
Percorso
Il percorso della risorsa, costituito dal nome del progetto e dal nome della cartella in cui risiede la risorsa.
Tipo
Visualizza i tipi di risorsa Oracle Service Bus.
Messaggio di diagnostica
Visualizza i messaggi di errore o informativi se presenti per la risorsa.

  1. Fare clic su Importa un altro.

    2. È possibile importare un altro file JAR o fare clic su Visualizza modifiche nel Centro modifiche per visualizzare i dettagli dell'importazione. Vedere Visualizzazione delle modifiche alla configurazione.

  2. Al termine, fare clic su Attiva nel Centro modifiche.
