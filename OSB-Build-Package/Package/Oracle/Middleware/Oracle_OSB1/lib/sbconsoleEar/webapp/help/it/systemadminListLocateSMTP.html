Individuazione dei server SMTP

  1. Selezionare Amministrazione sistema > Server SMTP.
  2. Per individuare server SMTP specifici, effettuare le operazioni riportate di seguito.
    • Riordinare la lista. Fare clic su un nome di colonna sottolineato. La freccia crescente e la freccia decrescente indicano il criterio di ordinamento. Fare clic sul nome della colonna per modificare il criterio di ordinamento.
    • Scorrere le pagine. Utilizzare i controlli della pagina sopra o sotto la tabella. Andare alla pagina selezionando il numero della pagina o utilizzando i pulsanti freccia per passare alla pagina successiva, alla pagina precedente, alla prima pagina o all'ultima pagina.
