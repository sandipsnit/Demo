Aggiunta di server proxy

Utilizzare la pagina Riepilogo dei server proxy per aggiungere e configurare le risorse server proxy e renderle disponibili in Oracle Service Bus come risorsa di sistema. È necessario trovarsi in una sessione attiva per configurare o riconfigurare le risorse server proxy.

  1. Se questa operazione non è stata già eseguita, fare clic su Crea per creare una nuova sessione o su Modifica per accedere a una sessione esistente. Vedere Utilizzo del Centro modifiche.
  2. Selezionare Amministrazione sistema > Server proxy.
  3. Fare clic su Aggiungi.
  4. Nel campo Nome immettere un nome per la risorsa server proxy. Questo campo è obbligatorio.

    Vedere Limitazioni di denominazione delle risorse per le istruzioni di denominazione.
  5. Nel campo Descrizione immettere una breve descrizione per la risorsa del server proxy.
  6. Nella sezione Parametri host-porta immettere le informazioni riportate di seguito.
    1. Nel campo Host server immettere il nome host o l'indirizzo IP del server proxy. Questo campo è obbligatorio.

      2. Il nome host del server per il server proxy Oracle Service Bus deve essere identico al nome host del server proxy effettivo.

    2. Nel campo Porta testo in chiaro immettere il numero di porta con testo in chiaro del server proxy.
    3. Nel campo Porta SSL immettere il numero di porta SSL del server proxy. È necessario immetter un numero di porta con testo in chiaro o SSL.
    4. Fare clic su Aggiungi.

      5. È possibile configurare più server proxy per ogni risorsa server proxy. Ciò consente a Oracle Service Bus di eseguire il bilanciamento del carico e offrire funzioni di tolleranza degli errori per la risorsa server proxy.

  7. Se il servizio proxy esegue l'autenticazione proxy immettere un nome utente nel campo Nome utente e la password associata nei campi Password e Conferma password.

    8. Questi campi sono facoltativi; sono obbligatori solo se il server proxy è protetto.

  8. Fare clic su Salva per creare e salvare la risorsa del server proxy nella sessione corrente.
  9. Per terminare la sessione e distribuire la configurazione in fase di esecuzione, fare clic su Attiva in Centro modifiche.
