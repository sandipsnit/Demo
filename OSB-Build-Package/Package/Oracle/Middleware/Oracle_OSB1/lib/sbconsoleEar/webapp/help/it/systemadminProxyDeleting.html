Eliminazione di server proxy

  1. Se questa operazione non è stata già eseguita, fare clic su Crea per creare una nuova sessione o su Modifica per accedere a una sessione esistente. Vedere Utilizzo del Centro modifiche.
  2. Selezionare Amministrazione sistema > Server proxy.
  3. Nella pagina Riepilogo dei server proxy selezionare un server proxy. Vedere Individuazione dei server SMTP.
  4. Fare clic sull'icona Elimina nel campo Opzioni della risorsa del server proxy che si desidera eliminare. La risorsa viene eliminata nella sessione corrente. Se altre risorse vi fanno riferimento, viene visualizzata un'icona Avvertenza di eliminazione. È possibile eliminare la risorsa con una conferma di avvertenza. Ciò può causare conflitti dovuti a riferimenti non risolti alla risorsa eliminata.
  5. Per terminare la sessione e distribuire la configurazione in fase di esecuzione, fare clic su Attiva in Centro modifiche.
