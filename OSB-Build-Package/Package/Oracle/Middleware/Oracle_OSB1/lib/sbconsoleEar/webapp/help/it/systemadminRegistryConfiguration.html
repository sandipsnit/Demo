Visualizzazione dei registri UDDI configurati

Utilizzare questa pagina per designare un registro UDDI predefinito (vedere Impostazione di una configurazione UDDI predefinita) e visualizzare i registri UDDI. Inoltre, è possibile cercare i registri configurati in precedenza.

  1. Nel pannello Registri UDDI immettere il nome del registro che si desidera trovare nel campo Nome o immettere caratteri jolly; utilizzare * per più caratteri e ? per singoli caratteri. Vengono restituiti tutti i record che soddisfano i criteri del nome.
  2. Fare clic su Cerca.
  3. Fare clic su Vis. tutto per rimuovere i filtri di ricerca e visualizzare tutti i registri configurati per il funzionamento con Oracle Service Bus.

    4. Nella tabella 29-1 vengono visualizzate le impostazioni di configurazione del registro UDDI.

    Tabella 29-1 Impostazioni di configurazione del registro UDDI 
    Proprietà
    Descrizione
    Nome del registro
    Il nome del registro. Il nome assegnato a un registro nel momento in cui viene creato. Non è possibile modificare il nome di un registro dopo aver salvato la voce.
    URL interrogazione
    L'URL dell'endpoint dell'API di interrogazione per individuare e importare i servizi.
    URL pubblicazione
    L'URL dell'endpoint dell'API di pubblicazione utilizzato per i servizi di pubblicazione.
    URL sicurezza
    L'URL dell'endpoint dell'API di sicurezza utilizzato per ottenere un token di autenticazione allo scopo di eseguire la pubblicazione nel registro.
    URL sottoscrizione
    L'URL dell'endpoint dell'API di sottoscrizione utilizzato per sottoscriversi alle modifiche del registro, creare una sottoscrizione al registro e ascoltare le modifiche ai servizi importati.
    Nome utente
    Il nome utente richiesto per l'autenticazione utente.
    Password (Conferma password)
    La password richiesta per l'autenticazione.
    Carica tModel nel registro
    Carica i tModel nel registro selezionato. Questa opzione deve essere selezionata solo una volta per ogni registro.
    Abilita importazione automatica
    Sincronizza automaticamente i servizi con il registro UDDI.
