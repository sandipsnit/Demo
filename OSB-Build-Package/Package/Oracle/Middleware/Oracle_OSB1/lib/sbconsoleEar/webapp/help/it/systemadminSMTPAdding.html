Aggiunta di server SMTP

Utilizzare la pagina Riepilogo dei server SMTP per aggiungere e configurare le risorse server SMTP e renderle disponibili in Oracle Service Bus come risorsa di sistema. È necessario trovarsi in una sessione attiva per configurare o riconfigurare le risorse server SMTP.

  1. Se questa operazione non è stata già eseguita, fare clic su Crea per creare una nuova sessione o su Modifica per accedere a una sessione esistente. Vedere Utilizzo del Centro modifiche.
  2. Selezionare Amministrazione sistema > Server SMTP.
  3. Fare clic su Aggiungi.
  4. Nel campo Nome immettere un nome per la risorsa server SMTP. Questo campo è obbligatorio.

    Vedere Limitazioni di denominazione delle risorse per le istruzioni di denominazione.
  5. Nel campo Descrizione immettere una breve descrizione per la risorsa del server SMTP.
  6. Nel campo URL server immettere l'URL che punta al server SMTP. Questo campo è obbligatorio.
  7. Nel campo Numero porta immettere un numero di porta per il server SMTP (la porta predefinita è 25). Questo campo è obbligatorio.
  8. Se l'accesso la server SMTP di destinazione richiede un nome utente e una password, immettere un nome utente nel campo Nome utente e la password associata nei campi Password e Conferma password.

    9. Questi campi sono facoltativi; sono obbligatori solo se il server SMTP è protetto.

  9. Fare clic su Salva per creare e salvare la risorsa server SMTP nella sessione corrente.
  10. Per terminare la sessione e distribuire la configurazione in fase di esecuzione, fare clic su Attiva in Centro modifiche.
