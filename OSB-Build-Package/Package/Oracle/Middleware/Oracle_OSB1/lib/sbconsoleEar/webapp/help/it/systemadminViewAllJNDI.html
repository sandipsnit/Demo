Visualizzazione dei provider JNDI

Utilizzare questa pagina per visualizzare una lista di provider JNDI e le informazioni mostrate nella tabella 30-1.

Inoltre, è possibile cercare i provider JNDI configurati in precedenza.

  1. Per limitare il numero di elementi nella lista o individuare provider JNDI specifici, effettuare le operazioni riportate di seguito.
    • Fare clic sull'icona Apri per visualizzare ulteriori filtri di ricerca. Se si specificano più criteri di ricerca, vengono restituiti solo i risultati che soddisfano tutti i criteri.
    • Immettere il nome del provider JNDI che si desidera trovare nel campo Nome e fare clic su Cerca per cercare una voce specifica.

      • È possibile immettere i caratteri jolly (? per un singolo carattere e * per più caratteri) per eseguire una ricerca più generica.

    • In URL provider immettere l'URL del provider JNDI che si desidera cercare, quindi fare clic su Cerca.
  2. Fare clic su Vis. tutto per rimuovere i filtri di ricerca e visualizzare tutte le risorse provider JNDI configurate per il funzionamento con Oracle Service Bus.

    3. Nella tabella 30-1 vengono visualizzate le impostazioni di configurazione del registro UDDI.

    Tabella 30-1 Informazioni sul provider JNDI 
    Proprietà
    Descrizione
    Nome
    Il nome univoco assegnato al provider JNDI. Il nome è un collegamento alla pagina Visualizza provider JNDI - Configurazione. Vedere Modifica di provider JNDI.
    URL provider
    L'URL del provider JNDI. Vedere Aggiunta di provider JNDI.
    Cache JNDI
    Abilitato o Disabilitato. Se questa opzione è abilitata, il contesto JNDI e gli oggetti JNDI vengono inseriti nella cache localmente. Ciò migliora le prestazioni durante la ricerca di un oggetto. Oracle consiglia di abilitare la cache JNDI.
    Azioni
    • Fare clic sull'icona Reimposta per reimpostare il contesto JNDI per eliminare la connessione JNDI e gli oggetti inseriti localmente nella cache.
    • Fare Avvia console di test per verificare se il provider JNDI è in grado di stabilire una connessione e ottenere un contesto JNDI.
    Opzioni
    • Fare clic sull'icona Rinomina per rinominare il provider JNDI.
    • Fare clic sul'icona Elimina per eliminare un provider JNDI specifico. Vedere Eliminazione di provider JNDI
