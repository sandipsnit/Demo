Visualizzazione dei server proxy

Utilizzare questa pagina per visualizzare una lista di server proxy e le informazioni mostrate nella tabella 30-5.

Le risorse server proxy vengono utilizzate durante la configurazione del trasporto HTTP per i servizi business. Per ulteriori informazioni, vedere la pagina Configurazione trasporto HTTP.

Nella tabella 30-5 sono visualizzate le informazioni sul server proxy.

Tabella 30-5 Informazioni sul server proxy 
Proprietà
Descrizione
Nome
Il nome univoco assegnato alla risorsa server proxy. Il nome è un collegamento alla pagina Visualizza server proxy - Configurazione. Vedere Modifica dei server proxy.
Opzioni
  • Fare clic sull'icona Rinomina per rinominare la risorsa server proxy.
  • Fare clic sull'icona Elimina per eliminare un server proxy specifico. Vedere Eliminazione di server proxy.
