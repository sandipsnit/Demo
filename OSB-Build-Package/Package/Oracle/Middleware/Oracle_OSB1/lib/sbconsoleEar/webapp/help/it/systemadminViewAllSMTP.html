Visualizzazione dei server SMTP

Utilizzare questa pagina per designare un server SMTP predefinito (vedere Configurazione di un server SMTP predefinito) e visualizzare una lista di server SMTP.

Le risorse server SMTP vengono utilizzate durante la configurazione delle risorse delle destinazioni di e dei servizi business basati sul trasporto di posta elettronica. Vedere Aggiunta di destinazioni di alert e la pagina Crea/modifica un servizio business - Configurazione trasporto posta elettronica.

Nella tabella 30-3 vengono visualizzate le informazioni sul server SMTP.

Tabella 30-3 Informazioni sul server SMTP
Proprietà
Descrizione
Nome
Il nome univoco assegnato alla risorsa server SMTP. Il nome è un collegamento alla pagina Visualizza server SMTP - Configurazione. Vedere Modifica dei server SMTP.
URL del server
L'URL che punta al server SMTP. Vedere Aggiunta di server SMTP.
Opzioni
  • Fare clic sull'icona Rinomina per rinominare la risorsa server SMTP.
  • Fare clic sull'icona Elimina per eliminare un server SMTP specifico. Vedere Eliminazione di server SMTP.
