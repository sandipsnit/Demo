Console di test

La console di test di Oracle Service Bus è un ambiente di test basato su browser utilizzato per convalidare la progettazione del sistema ed eseguirne il test. È possibile configurare l'oggetto del test (servizio proxy, servizio business o risorsa), eseguire il test e visualizzare i risultati nella console di test. In alcuni casi, è possibile esaminare lo stato del messaggio in punti di trace specifici. Per ulteriori informazioni sulla console di test, vedere "Using the Test Console" in Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus all'indirizzo http://www.oracle.com/pls/as111170/lookup?id=OSBAG357.

Solo gli utenti nei ruoli IntegrationAdmin e IntegrationDeployer possono utilizzare la console di test. Per ulteriori informazioni sui ruoli, vedere "Roles in Oracle Service Bus" in Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus all'indirizzo http://www.oracle.com/pls/as111170/lookup?id=OSBAG463.

Nota: se viene visualizzato un messaggio di errore che comunica che il servizio Console di test non è in esecuzione, provare a impostare l'indirizzo di ascolto del server di amministrazione su un indirizzo valido specifico, ad esempio localhost. Nella console di Oracle WebLogic Server accedere a Ambiente > Server > admin_server_name > Configurazione > Generale per impostare l'indirizzo di ascolto.
