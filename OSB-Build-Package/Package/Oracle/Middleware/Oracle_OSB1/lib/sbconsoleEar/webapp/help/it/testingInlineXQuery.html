Esecuzione del test XQuery

È possibile modificare un'azione ed eseguirne il test nel flusso di messaggi utilizzando i seguenti editor: Editor espressioni XQuery, Editor condizione XQuery ed Editor espressioni XPath. L'esecuzione del test avviene nello stesso modo per l'Editor espressioni XQuery e l'Editor condizione XQuery. Tuttavia, lo scenario è differente per l'Editor espressioni XPath poiché accetta un solo input.

Nota: è necessario disabilitare i popup blocker nel browser affinché il test XQuery funzioni. Se sono presenti barre degli strumenti nel browser Internet Explorer, potrebbe essere necessario disabilitare i popup blocker dal menu Opzioni e per tutte le barre degli strumenti configurate per il blocco dei popup.

In questa sezione sono inclusi gli argomenti riportati di seguito.

  • Utilizzo dell'Editor espressioni XQuery e dell'Editor condizione XQuery
  • Utilizzo dell'Editor espressioni XPath
