Test delle trasformazioni

È possibile eseguire il test delle trasformazioni dopo avere attivato una sessione o durante una sessione per assicurarsi che il funzionamento delle risorse sia quello previsto. È necessario attivare la sessione per eseguire il test della fase di esecuzione, altrimenti il test viene eseguito nella fase di progettazione utilizzando le modifiche locali.

È possibile eseguire il test delle seguenti trasformazioni:

  • Test delle trasformazioni MFL
  • Test delle trasformazioni XSLT
  • Test delle trasformazioni XQuery
