Test delle trasformazioni MFL

Un documento MFL (Message Format Language) è un documento XML specializzato utilizzato per descrivere il layout dei dati binari. Le risorse MFL supportano due trasformazioni. Ogni trasformazione accetta solo un input e fornisce un singolo output: da XML a binario o da binario a XML.

Nella tabella 32-5 sono descritte le opzioni di configurazione MFL.

Tabella 32-5 Configurazione delle opzioni MFL
Sezione
Descrizione
Nome
Il nome della risorsa di cui si esegue il test.
Trasformazioni supportate
Per selezionare una trasformazione specifica, selezionare il nome della trasformazione.
Azioni della console di test
Esegui
Applica la trasformazione.
Reimposta
Reimposta il campo di input (per le trasformazioni da XML a binario, il documento XML di esempio viene reimpostato).

Annulla l'operazione corrente.
Configurazione trasformazione MFL
Input XML
Richiesto per le trasformazioni da XML a binario:
Lo schema XML per il documento MFL può essere dedotto. Un documento XML di esempio viene immesso automaticamente nel campo di testo.
L'input XML può essere basato su file o su testo. Il riferimento a un file per l'input ha precedenza sull'input testuale. Cercare e selezionare il file da utilizzare nel test.
Input binario
Richiesto per le trasformazioni da binario a XML:
L'input binario può essere basato su file o su testo. Il riferimento a un file per l'input ha precedenza sull'input testuale. Cercare e selezionare il file da utilizzare nel test.

È possibile eseguire il test della fase di progettazione o della fase di esecuzione.

  1. Fare clic su Attiva per eseguire il test della fase di esecuzione. Non attivare la sessione per eseguire il test della fase di progettazione.
  2. Selezionare Browser risorse > MFL per visualizzare la pagina Riepilogo dei file MFL.
  3. Sotto Azioni fare clic sull'icona Avvia console di test associata alla risorsa di cui si desidera eseguire il test. La console di test apre la pagina Test della risorsa.
  4. Selezionare la trasformazione di cui si desidera eseguire il test.
  5. Configurare i dati di test per la risorsa MFL. Per ulteriori informazioni, vedere la tabella 32-5.
  6. Fare clic su Esegui. La pagina Test della risorsa visualizza i risultati.
  7. Per eseguire di nuovo il test, fare clic su Indietro. È possibile chiudere la console di test, modificare la risorsa e sottoporla nuovamente a test.
