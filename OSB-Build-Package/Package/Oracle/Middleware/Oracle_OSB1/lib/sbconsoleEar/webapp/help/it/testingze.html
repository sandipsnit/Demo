Test delle trasformazioni XQuery

Le mappe XQuery possono descrivere i mapping da XML a XML, da XML a non XML e da non XML a XML. Una trasformazione XQuery può accettare più input e restituire un unico output. Ogni input corrisponde a una variabile esterna XQuery dichiarata nella risorsa XQuery. Il valore di una variabile di input XQuery può essere un valore primitivo (String, Integer, Date e così via), un documento XML o una sequenza di questi tipi. Il valore di output può essere un valore primitivo (String, Integer, Date e così via), un documento XML o una sequenza di questi tipi.

Nota: la console di test non supporta le sequenze per l'input.

Variabili
Questa sezione contiene un campo di input per ognuna delle variabili esterne XQuery.
<param_name> (come XML)
param_name è un nome di variabile XQuery nella risorsa XQuery.
Nella console di test viene visualizzata una casella di modifica con una singola riga se il tipo è un tipo semplice. Se i dati sono XML viene visualizzata una casella di modifica con più righe.
Se non si definisce un tipo per la variabile, viene utilizzato un input combinato (<param_name> ([] come XML)). È necessario dichiarare il tipo di variabile. Selezionare la casella di controllo per identificare un parametro di tipo XML.
L'input XML può essere basato su file o su testo. Il riferimento a un file per l'input ha precedenza sull'input testuale. Cercare e selezionare il file da utilizzare nel test.
L'input nella console di test viene visualizzato in base al tipo per rendere più semplice l'individuazione del tipo di dati da immettere. Se il tipo non è definito, il tipo predefinito è XML.

È possibile eseguire il test della fase di progettazione o della fase di esecuzione.

  1. Fare clic su Attiva per eseguire il test della fase di esecuzione. Non attivare la sessione per eseguire il test della fase di progettazione.
  2. Selezionare Explorer progetto > XQueries per visualizzare la pagina Riepilogo delle XQueries.
  3. Sotto Azioni fare clic sull'icona Avvia console di test associata alla risorsa di cui si desidera eseguire il test. La console di test apre la pagina Test della risorsa.
  4. Configurare i dati di test per la risorsa nella sezione Variabili della pagina. Per ulteriori informazioni, vedere la tabella 32-7.
  5. Fare clic su Esegui. La pagina Test della risorsa visualizza i risultati.
  6. Per eseguire di nuovo il test, fare clic su Indietro. È possibile chiudere la console di test, modificare la risorsa e sottoporla nuovamente a test.
