Aggiunta di registri UDDI

Utilizzare questa pagina per aggiungere e configurare i registri UDDI. Dopo aver configurato i registri, è possibile pubblicarvi i servizi proxy Oracle Service Bus o importare da tali registri i servizi business da utilizzare in un servizio proxy Oracle Service Bus.

È necessario essere in una sessione attiva per aggiungere un registro.

  1. Se questa operazione non è stata già eseguita, fare clic su Crea per creare una nuova sessione o su Modifica per accedere a una sessione esistente. Vedere Utilizzo del Centro modifiche.
  2. Selezionare Amministrazione sistema > Registri UDDI.
  3. Nel pannello Registri UDDI fare clic su Aggiungi registro.
  4. Per configurare il registro, completare i passi descritti di seguito. Gli asterischi indicano i campi obbligatori. Per le descrizioni delle proprietà da impostare quando si aggiunge un registro, vedere la tabella 29-1.
    1. Nel campo Nome immettere un nome per il registro.

      Vedere Limitazioni di denominazione delle risorse per le istruzioni di denominazione.
    2. Nel campo URL interrogazione immettere un URL di interrogazione nel formato http://host:port/APPLICATION_SERVER_CONTEXT/uddi/inquiry
    3. Nel campo URL pubblicazione immettere un URL di pubblicazione nel formato http://host:port/APPLICATION_SERVER_CONTEXT/uddi/publishing
    4. Nel campo Security URL immettere un URL di sicurezza nel formato http://host:port/APPLICATION_SERVER_CONTEXT/uddi/security
    5. Nel campo URL sottoscrizione immettere un URL di sottoscrizione nel formato
      http://host:port/APPLICATION_SERVER_CONTEXT/uddi/subscription

      6. Per impostazione predefinita, il valore per APPLICATION_SERVER_CONTEXT nell'Installer è registry.

    6. Nel campo Nome utente immettere il nome utente per collegarsi alla console del registro.
    7. Nel campo Password /(Conferma password) immettere la password per collegarsi alla console del registro.
    8. Selezionare la casella di controllo Carica tModel nel registro per pubblicare i tModel Oracle Service Bus nel registro.

      9. Questo campo è obbligatorio solo quando si pubblicano i servizi proxy nel registro.

    9. Selezionare la casella di controllo Abilita importazione automatica per sincronizzare automaticamente i servizi importati con il registro UDDI. I servizi importati con questa opzione selezionata sono sempre sincronizzati con il registro UDDI.
      10. Nota: la sincronizzazione automatica è un processo in background; non è possibile invertirlo utilizzando la funzione Annulla della sessione. L'annullamento di una modifica della sincronizzazione automatica non è permanente, in quanto il servizio viene sincronizzato nuovamente nel ciclo di sincronizzazione successivo. Se si desidera che un servizio importato non venga sincronizzato con il registro UDDI, è necessario scollegarlo per evitare ulteriori aggiornamenti. Vedere Scollegamento di servizi.
  5. Fare clic su Convalida per convalidare gli URL forniti.
  6. Fare clic su Salva per configurare il registro con le impostazioni fornite e salvarlo nella sessione corrente.
  7. Per terminare la sessione e distribuire la configurazione in fase di esecuzione, fare clic su Attiva in Centro modifiche.
