WSDL

Un WSDL (Web Service Definition Language) è la descrizione formale di un servizio Web; in Oracle Service Bus descrive un servizio proxy o business. Viene utilizzato per descrivere le operazioni che un servizio Web può eseguire, dove risiede e come richiamarlo.

È possibile basare i servizi SOAP e XML su una risorsa WSDL esistente. È disponibile un documento WSDL per i servizi proxy e business per qualsiasi trasporto. Vedere la pagina Crea/Modifica un servizio proxy - Configurazione generale e Creazione e configurazione dei servizi business.

Per ulteriori informazioni, vedere "How WSDL is Used in Oracle Service Bus" in Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus all'indirizzo http://www.oracle.com/pls/as111170/lookup?id=OSBAG114.
