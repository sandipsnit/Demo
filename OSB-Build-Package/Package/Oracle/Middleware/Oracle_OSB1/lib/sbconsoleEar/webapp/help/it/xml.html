Documenti XML

Utilizzare le risorse dei documenti XML per memorizzare i file XML per l'uso nelle configurazioni dei servizi proxy o business. Ad esempio, creare le risorse dei documenti XML per i file di mapping TopLink o EclipseLink necessari nei servizi business o proxy JCA che comunicano con i sistemi compatibili con JCA.

Per ulteriori informazioni sui servizi JCA, vedere il capitolo 12 sulle associazioni JCA.

In questa sezione sono inclusi gli argomenti riportati di seguito.

  • Individuazione di documenti XML
  • Aggiunta di documenti XML
  • Modifica di documenti XML
  • Eliminazione di documenti XML
