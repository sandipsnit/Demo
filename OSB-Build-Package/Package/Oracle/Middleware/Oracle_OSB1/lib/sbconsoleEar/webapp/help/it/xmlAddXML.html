Aggiunta di documenti XML

Il modo più semplice per creare i documenti XML in Oracle Service Bus è tramite l'importazione in bulk degli stessi. Ad esempio, se si esegue l'importazione in bulk delle risorse JCA create (file .jca, WSDL associato e file di mapping TopLink o EclipseLink), Oracle Service Bus genera automaticamente le risorse dei documenti XML dai file di mapping e gestisce le dipendenze tra i file delle risorse.

Per ulteriori informazioni sull'importazione, vedere Caricamento delle risorse da un file Zip, Caricamento delle risorse da un URL e Importazione di risorse.

Se non si esegue l'importazione in bulk, è possibile creare manualmente i documenti XML, come descritto di seguito.

  1. Se questa operazione non è stata già eseguita, fare clic su Crea per creare una nuova sessione o su Modifica per accedere a una sessione esistente. Vedere Utilizzo del Centro modifiche.
  2. Selezionare Explorer progetto, quindi selezionare un progetto o una cartella in cui aggiungere il documento XML. Viene visualizzata la pagina Vista progetto/cartella.
  3. Nel campo Crea risorsa selezionare Documento XML per visualizzare la pagina Crea un nuovo documento XML.
  4. Nel campo File XML fare clic su Sfoglia e selezionare il file XML che si desidera utilizzare.

    5. Nel campo Nome risorsa viene inserito automaticamente il nome del file (senza l'estensione del file).

    Invece di selezionare un file, è possibile copiare e incollare il contenuto di un file XML nell'area di testo del campo File XML.

  5. Nel campo Nome risorsa immettere o modificare il nome del documento XML. I nomi dei documenti XML devono essere univoci.

    Vedere Limitazioni di denominazione delle risorse per le istruzioni di denominazione.
  6. Nel campo Descrizione risorsa immettere una descrizione per il documento XML.
  7. Fare clic su Salva. Il nuovo documento XML viene salvato nella sessione corrente.
  8. Per terminare la sessione e distribuire la configurazione in fase di esecuzione, fare clic su Attiva in Centro modifiche.
