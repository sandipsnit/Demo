Individuazione di documenti XML

Nella pagina Riepilogo dei documenti XML sono elencati i documenti XML aggiunti al dominio Oracle Service Bus.

  1. Effettuare una delle operazioni riportate di seguito.
    • Selezionare Explorer progetto per visualizzare la pagina Vista progetti o la pagina Vista progetto/cartella. Spostarsi quindi tra i progetti e le cartelle per trovare il documento XML.
    • Selezionare Browser risorse > Documenti XML. Nella pagina Riepilogo dei documenti XML vengono visualizzate le informazioni mostrate nella tabella 13-1. Per una descrizione più dettagliata delle proprietà, vedere Modifica dei documenti XML.
  2. Per individuare un documento XML specifico, effettuare le operazioni riportate di seguito.
    • Per limitare il numero di elementi nella lista, è possibile filtrarli in base al nome e al percorso prima di fare clic su Cerca.

      • Il percorso è costituito dal nome del progetto e dal nome della cartella o delle cartelle in cui la risorsa risiede.

    • Fare clic su Vis. tutto per rimuovere i filtri di ricerca e visualizzare tutti i documenti XML.

      • Tabella 13-1 Dettagli dei documenti XML
      Proprietà
      Descrizione
      Documento XML
      Il nome univoco assegnato al documento XML. Si tratta di un collegamento alla pagina Visualizza il documento XML. Vedere Modifica dei documenti XML.
      Percorso
      Il percorso è costituito dal nome del progetto e dal nome della cartella in cui risiede il documento XML. Si tratta di un collegamento al progetto o alla cartella che contiene la risorsa. Vedere Qualificazione dei nomi di risorsa utilizzando progetti e cartelle.
      Opzioni
      Fare clic sull'icona Elimina per eliminare un documento XML specifico. Vedere Eliminazione di documenti XML.
