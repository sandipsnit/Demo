Modifica di documenti XML

Utilizzare la pagina Modifica un documento XML per visualizzare e modificare i dettagli di un'associazione JCA.

  1. Se questa operazione non è stata già eseguita, fare clic su Crea per creare una nuova sessione o su Modifica per accedere a una sessione esistente. Vedere Utilizzo del Centro modifiche.
  2. Individuare il documento XML, come descritto in Individuazione dei documenti XML.
  3. Fare clic sul nome del documento XML. Nella pagina Visualizza il documento XML vengono visualizzate le informazioni mostrate nella tabella 13-2.

    4. Tabella 13-2 Dettagli del documento XML
    Proprietà
    Descrizione
    Autore ultima modifica
    L'utente che ha creato il documento XML o lo ha importato nella configurazione.
    Data ultima modifica
    La data e l'ora in cui l'utente ha creato il documento XML o lo ha importato nella configurazione. Fare clic sul collegamento relativo alla data e all'ora per visualizzare la cronologia delle modifiche della risorsa. Vedere la pagina Visualizza cronologia modifiche.
    Riferimenti
    Il numero di oggetti a cui fa riferimento il documento XML. Se tali riferimenti esistono, fare clic sul collegamento numerico per visualizzare una lista degli oggetti. Vedere Visualizzazione dei riferimenti alle risorse.
    Riferimento di
    Il numero di oggetti che fa riferimento al documento XML. Se tali riferimenti esistono, fare clic sul collegamento numerico per visualizzare una lista degli oggetti. Ad esempio, se il file XML è un file TopLink a cui fa riferimento un'associazione JCA, quest'ultima è elencata come riferimento quando si fa clic sul collegamento. Vedere Visualizzazione dei riferimenti alle risorse.
    Descrizione
    Una descrizione del documento XML, se disponibile.

    Viene visualizzata per impostazione predefinita una vista di sola lettura del documento XML.

  4. Per modificare il documento XML, fare clic su Modifica. Per le descrizioni dei campi, vedere Aggiunta di documenti XML.
  5. Apportare le modifiche appropriate e fare clic su Salva per eseguire il commit degli aggiornamenti nella sessione corrente.
  6. Per terminare la sessione e distribuire la configurazione in fase di esecuzione, fare clic su Attiva in Centro modifiche.
