ビジネス・サービスの作成/編集 - ページ・リファレンス

ビジネス・サービスを選択するには、プロジェクト/フォルダ・ビュー・ページの「リソースの作成」ドロップダウン・リストで、「ビジネス・サービス」を選択します。このオプションを選択すると、ビジネス・サービスを構成および追加する一連のページにこれが表示されます。表示されるページは、それまでに選択したオプションによって異なります。次のページがあります。

  • 全般的な構成ページ
  • メッセージ・タイプの構成ページ
  • トランスポート構成ページ
  • (プロトコル)トランスポート構成ページ
  • SOAPバインド構成ページ
  • メッセージ処理構成ページ
  • サマリー・ページ
  • ビジネス・サービスのポリシー・ページ
  • ビジネス・サービスのセキュリティ・ページ
