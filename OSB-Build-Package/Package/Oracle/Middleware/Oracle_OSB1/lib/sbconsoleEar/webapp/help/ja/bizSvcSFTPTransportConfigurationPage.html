SFTPトランスポート構成ページ

ビジネス・サービスの作成/編集 - SFTPトランスポート構成ページは、ビジネス・サービスの作成および編集(ビジネス・サービスの作成と構成およびビジネス・サービス構成の編集を参照)に関する一連のページの1つです。

詳細は、Oracle Fusion Middleware Oracle Service Busの開発者ガイドのHTTPトランスポートおよびポーラー・トランスポート(http://www.oracle.com/pls/as111170/lookup?id=OSBDV947)を参照してください。

このページでは、SFTPプロトコルを使用するビジネス・サービスのトランスポート設定を構成します。表18-16に、このページの使用方法を示します。

表18-16 ビジネス・サービスの作成/編集 - SFTPトランスポート構成ページ
オプション
作成または編集の方法
ユーザー認証
次のいずれかを選択します:
  • 「ユーザー名/パスワード認証」 - 静的なサービス・アカウントがこの認証メソッドに関連付けられていること、およびクライアントが提供された資格証明を使用して認証されることを指定します。
  • 「ホスト・ベース認証」 - この認証メソッドを使用するには、ユーザー名およびサービス・キー・プロバイダが必要であることを指定します。既知のホストから接続しているユーザーは、ホストのプライベート・キーを使用して認証されます。
  • 「公開鍵認証」 - この認証メソッドを使用するには、ユーザー名およびサービス・キー・プロバイダが必要であることを指定します。すべてのユーザーには独自のプライベート・キーがあります。
サービス・アカウント
ユーザーのサービス・アカウントを入力するか、「参照」をクリックして、サービス・アカウントをブラウザから選択します。
サービス・キー・プロバイダ
このオプションは、「ホスト・ベース認証」または「公開鍵認証」が選択されている場合のみ有効です。
「サービス・キー・プロバイダ」フィールドに、サービス・キー・プロバイダを入力します。「参照」をクリックして、ブラウザからサービス・キー・プロバイダを選択できます。このフィールドは必須です。
ユーザー名
このオプションは、「ホスト・ベース認証」または「公開鍵認証」が選択されている場合のみ有効です。
ユーザー名を入力します。
タイムアウト
接続が切断されるまでのソケット・タイムアウト間隔を秒単位で入力します。0と入力した場合、タイムアウトは発生しません。デフォルト値は60です。
宛先ファイル名の接頭辞
リモート・サーバーに格納されるファイルのファイル名に使用する接頭辞を入力します。
宛先ファイル名の接尾辞
リモート・サーバーに格納されるファイルのファイル名に使用する接尾辞を入力します。
要求エンコーディング
SFTPトランスポートにおける要求の文字セット・エンコーディングとして、デフォルトのUTF-8を受け入れます。
優先暗号スイート
サーバーとの通信時に使用する暗号スイートを選択します。
デフォルト値「ランタイム・デフォルトの使用」を使用する場合は、サポートされている暗号スイートのリストがサーバーに送信され、一致が見つかるまで各暗号スイートが試行されます。
優先データ整合性アルゴリズム
データ整合性チェックに使用するバルク・ハッシュ・アルゴリズムを選択します。
デフォルト値のランタイム・デフォルトの使用を使用する場合、Oracle Service Busでは優先的にhmac-sha1アルゴリズムがサーバーに送信されます。サーバーがサポートしていない場合は、サポートされているアルゴリズムのリストが送信され、一致が見つかるまで各アルゴリズムが試行されます。
優先公開鍵アルゴリズム
公開鍵の暗号化の非対称鍵アルゴリズムを選択します。
デフォルト値ランタイム・デフォルトの使用を使用する場合、Oracle Service Busではssh-dssアルゴリズムが優先的にサーバーに送信されます。サーバーがサポートしていない場合は、サポートされているアルゴリズムのリストがサーバーに送信され、一致が見つかるまで各アルゴリズムが試行されます。
優先鍵交換アルゴリズム
メッセージ暗号化のセッション鍵のネゴシエーション用に、デフォルトの鍵交換プロトコルを選択します。
デフォルト値、ランタイム・デフォルトの使用を使用する場合、Oracle Service Busでは、diffie-hellman-group1-sha1アルゴリズムが優先的にサーバーに送信されます。サーバーがサポートしていない場合は、サポートされているアルゴリズムのリストが送信され、一致が見つかるまで各アルゴリズムが試行されます。
優先圧縮アルゴリズム
zlibを使用して、処理中のデータを圧縮するかどうか指定します。zlibを選択してデータを圧縮するか、「なし」を選択します。デフォルトは「なし」です。

終了後

「次」をクリックして次のページでこのサービスの構成を続行するか、「最後」をクリックしてビジネス・サービスの作成/編集 - サマリー・ページでこの構成を確認して保存します。
