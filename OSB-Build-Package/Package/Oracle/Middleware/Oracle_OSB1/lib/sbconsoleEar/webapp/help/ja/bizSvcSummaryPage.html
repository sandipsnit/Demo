サマリー・ページ

ビジネス・サービスの作成/編集 - サマリーページは、ビジネス・サービスの作成および編集(ビジネス・サービスの作成と構成およびビジネス・サービス構成の編集を参照)に関する一連のページの1つです。

このページで、ビジネス・サービスの構成の設定の表示および変更ができます。

設定を確認または変更するには、該当するカテゴリ(「全般的な構成」、「トランスポート構成」など)の行で「編集」をクリックします。編集できるページは、ビジネス・サービスを作成したときに構成したページによって異なります。すべてのページを次のリストに示します:

  • 全般的な構成ページ
  • メッセージ・タイプの構成ページ
  • トランスポート構成ページ
  • (プロトコル)トランスポート構成ページ
  • ビジネス・サービスのポリシー・ページ
  • ビジネス・サービスのセキュリティ・ページ
