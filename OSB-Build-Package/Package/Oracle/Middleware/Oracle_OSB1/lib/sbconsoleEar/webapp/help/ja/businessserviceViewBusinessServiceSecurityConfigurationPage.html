ビジネス・サービスのセキュリティ・ページ

ビジネス・サービスの表示 - セキュリティ構成ページは、ビジネス・サービスの編集(ビジネス・サービスの作成と構成およびビジネス・サービスの表示 - 構成の詳細ページを参照)に関する一連のページの1つです。

このページに表示されるフィールドは、ビジネス・サービスで使用するポリシーによって異なります。たとえば、ビジネス・サービスでOWSMポリシーを使用している場合(推奨)、セキュリティ・ページには「ポリシーのオーバーライド」が表示されます。詳細は、Oracle Fusion Middleware Oracle Service Busの開発者ガイドのOracle Web Services ManagerによるOracle Service Busの保護(http://www.oracle.com/pls/as111170/lookup?id=OSBDV1681)を参照してください。

サービスにバインドされているWLS 9.2のポリシーは、有効なWSDL内に置かれます(インライン化)。抽象ポリシーは、インライン化される前に前処理されます。OWSMポリシーは参照によってバインドされ、有効なWSDL内にインライン化されません。

このページでは、WSDLベースのポリシーを持つ、またはカスタム・ポリシー・バインドを使用するビジネス・サービスのセキュリティ設定を構成します。表18-24に、このページの使用方法を示します。

表18-24 ビジネス・サービスの表示 - セキュリティ・ページ
オプション
編集の方法
サービス・アカウント
「参照」をクリックし、表示されるリストでサービス・アカウントを選択します。サービス・アカウントを指定しない場合は、匿名サブジェクトが使用されます。
注意: このオプションは、Oracle Web Services Managerポリシーを使用するサービスでは使用できません。ユーザーの資格証明はオーバーライド・キーによって提供されるためです。

詳細は次を参照してください:

  • サービス・アカウントの指定
  • サービス・アカウントの追加
ポリシーのオーバーライド
OWSMポリシーに対し、許可されているオーバーライドを必要に応じて指定します。
詳細は、表18-23およびOracle Fusion Middleware Oracle Service Busの開発者ガイドのOracle Web Services ManagerによるOracle Service Busの保護(http://www.oracle.com/pls/as111170/lookup?id=OSBDV1681)を参照してください。
