セッションの終了

「チェンジ・センター」の下にある「終了」をクリックすると、任意の時点でセッションを終了できます。ただし、セッションが破棄されるわけではありません。

「>チェンジ・センター」の「編集」をクリックすると、セッションに戻り変更を続行できます。この動作は、「ログアウト」をクリックしてOracle Service Bus管理コンソールからログアウトするか、ブラウザを閉じる場合にも適用されます。セッションおよびセッション内で行った変更はすべて、Oracle Service Bus管理コンソールからログアウトするか、サーバーを再起動した場合でも存続します。

セッションを終了できるのは、セッションがアクティブにされている場合のみです。セッションのアクティブ化を参照してください。
