結果キャッシュのグローバルな有効化

結果がほとんど変わらないビジネス・サービスを呼び出す場合は、結果キャッシュを使用すると、サービスの呼出しの結果を待機するかわりにキャッシュされた結果をクライアントに戻すことにより、ビジネス・サービスのパフォーマンスが向上します。

個別のビジネス・サービスの結果キャッシュを構成する前に、次の手順に従って結果キャッシュをグローバルに有効化しておく必要があります。結果キャッシュのデフォルト設定は「有効」です。

  1. セッションの作成または編集をまだ行っていない場合は、「作成」をクリックして新しいセッションを作成するか、「編集」をクリックして既存のセッションに入ります。チェンジ・センターの使用を参照してください。
  2. 「操作」→「グローバル設定」を選択します。
  3. 「結果キャッシュの有効化」チェック・ボックスを選択します。
  4. 「更新」をクリックして、このページのすべての設定を保存します。
  5. または、セッション中の任意の時点で「リセット」をクリックして、変更を破棄し、現在保持されている設定を使用してページをリフレッシュします。
  6. セッションを終了して構成をランタイムにデプロイするには、「チェンジ・センター」の「アクティブ化」をクリックします。

ビジネス・サービスの結果キャッシュの構成方法は、メッセージ処理構成ページを参照してください。
