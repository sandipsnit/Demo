Oracle Service Bus管理コンソールの起動

  1. 次のいずれかの方法に従って、Oracle Service Busを起動します。
    • Windowsの「スタート」メニューから、次のように選択します。

      • 「スタート」→「すべてのプログラム」→「Oracle WebLogic」→「User Projects」→「domain_name」→「Start Server for Oracle Service Bus Domain」。

      ここで、domain_nameは、Oracle Service Busドメインを作成したときに指定した名前を表します。「User Projects」は、ユーザーのドメインがデフォルトで作成されるデフォルトの場所です。

    • Oracle Service Busドメインのルートから、startWebLogicコマンド(startWebLogic.cmdまたはstartWebLogic.sh)を実行します。

      • コマンド・ウィンドウが表示され、WebLogic Serverに関するステータス情報が示されます。

      本番モードでサーバーを起動する場合は、有効なユーザー名とパスワードを入力する必要があります。

  2. サーバーが起動した後、ブラウザに次のURLを入力します。

    3. http://hostname:port/sbconsole

    ここで、hostはWebLogic Serverが実行されているマシンの名前を表し、portはポート番号を表します。

    たとえば、デフォルトのポート構成を使用してWebLogic Serverがローカル・マシンで実行されている場合は、ブラウザで次のURLを入力します。

    http://localhost:7001/sbconsole

    Oracle Service Bus管理コンソールのログイン・ページが表示されます。

  3. インストール・プロセス時に指定したユーザー名とパスワードを入力します。
    4. 注意: Oracle Service Busサンプル・ドメインのユーザー名とパスワードは、weblogic / welcome1です。
  4. 「ログイン」をクリックします。

ログアウトするには、Oracle Service Bus管理コンソールの右側のパネルの上部付近にある「ログアウト」をクリックします。
