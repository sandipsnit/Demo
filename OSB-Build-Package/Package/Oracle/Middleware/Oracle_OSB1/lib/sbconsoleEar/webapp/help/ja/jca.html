JCAバインド

Oracle SOA Suiteでは、J2EE Connector Architecture (JCA)アダプタを使用してバックエンドの企業情報システム(EIS)と統合できます。Oracle JDeveloperおよびサポートされているその他のJCA開発環境を使用して、バックエンド統合に使用するアダプタを構成し、JCAアダプタ・リソース(.jcaファイルおよび抽象WSDL)を作成します。

Oracle Service Bus (OSB)を通じてJCA対応のEISコンポーネントと通信できるため、JCAリソースをJCAバインド・リソースの形式で消費できます。JCAバインドを使用して、Oracle Service BusのJCAトランスポートを使用するプロキシ・サービスおよびビジネス・サービスを生成し、JCAアダプタを通じてEISアプリケーションと通信できます。

この章では、Oracle Service BusでJCAバインドを追加、編集、指定および削除する方法について説明します。この章には、次のトピックが含まれます。

  • JCAバインドの検索
  • JCAバインドの追加
  • JCAバインドの編集
  • JCAバインドの削除

JCAトランスポートの詳細は、Oracle Fusion Middleware Oracle Service Busの開発者ガイドのJCAトランスポート(http://www.oracle.com/pls/as111170/lookup?id=OSBDV910)を参照してください。

Oracle SOA Suite JCAアダプタの詳細は、Oracle Fusion Middleware Technology Adaptersのユーザーズ・ガイド(http://www.oracle.com/pls/as111170/lookup?id=TKADP)を参照してください。
