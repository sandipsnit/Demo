JCAバインドの検索

JCAバインドのサマリーページには、Oracle Service Busドメインに追加されたJCAバインドが表示されます。

  1. 次のいずれかを実行します:
    • 「プロジェクト・エクスプローラ」を選択して、プロジェクト・ビュー・ページまたはプロジェクト/フォルダ・ビュー・ページを表示します。次に、プロジェクトおよびフォルダに移動して、JCAバインドを検索します。
    • 「リソース・ブラウザ」→「JCAバインド」を選択します。JCAバインドのサマリー・ページに、表12-1に示す情報が表示されます。各プロパティの詳細は、JCAバインドの検索を参照してください。
  2. 特定のJCAバインドを検索するには:
    • リスト内の項目数を制限するには、「検索」をクリックする前に名前とパスでフィルタします。

      • パスは、プロジェクト名、およびリソースが格納されるフォルダの名前になります。

    • 「すべて表示」をクリックして、検索フィルタを解除し、JCAバインドをすべて表示します。

      • 表12-1 JCAバインド情報
      プロパティ
      説明
      JCAバインド名
      JCAバインドに割り当てられる一意の名前。名前をクリックすると、JCAバインドの詳細を表示ページが表示されます。JCAバインドの編集を参照してください。
      パス
      プロジェクト名と、JCAバインドが格納されているフォルダの名前。名前をクリックすると、このリソースを含むプロジェクトまたはフォルダが表示されます。プロジェクトとフォルダを使用したリソース名の修飾を参照してください。
      アクション
      JCAバインドからOracle Service Bus WSDLおよびJCAサービスを生成する「生成」アイコンを含みます。
      オプション
      「削除」アイコンが含まれています。ビジネス・サービスまたはプロキシ・サービスがJCAバインドを使用する構成の場合、削除の警告アイコンは、確認の警告が表示されてもそのJCAバインドを削除できることを示します。これにより、削除したJCAバインドに対してサービスからの参照が解決されないために、競合が起きることがあります。JCAバインドの削除を参照してください。
