ドメイン・ログ・ファイル・エントリの表示のカスタマイズ

  1. この表のカスタマイズをクリックします。追加のフィールドが表示されます。

    2. この表のカスタマイズ・リンクをクリックすると、いつでもこの表を閉じて元の設定に戻すことができます。

  2. 「フィルタ」フィールドで、次の操作を行います。
    1. 「時間」フィールドで、ログ・エントリを表示する間隔を選択します。
    2. 開始時刻フィールドに、MM/DD/YYYY HH:MM:SSという書式で開始時刻を入力します。たとえば、10/25/06 08:39:48と入力します。開始時刻フィールドと終了時刻フィールドを使用して、ログ・エントリを表示する時間帯を指定します。
    3. 終了時刻フィールドに、MM/DD/YYYY HH:MM:SSという書式で終了時刻を入力します。たとえば、10/25/06 13:20:51と入力します。開始時刻フィールドと終了時刻フィールドを使用して、ログ・エントリを表示する時間帯を指定します。
    4. 条件テキスト・ボックスで、フィルタ条件(テキスト文字列)を指定します。フィルタ処理の条件は、WLDF問合せ言語で、文字列として指定されます。問合せ言語は、ブール演算子(AND、OR、NOT)および比較演算子をサポートしています。問合せ構文の詳細は、Oracle Fusion Middleware Oracle WebLogic Serverの診断フレームワークの構成と使用方法のWLDF問合せ言語(http://www.oracle.com/pls/as111170/lookup?id=WLDFC316)を参照してください。
  3. 「列」フィールドで、表示する列を選択します。
    1. 使用可能フィールドから列名を選択します。
    2. 矢印をクリックして、この列名を選択済みフィールドに移動します。
    3. 表示するすべての列名が選択済みフィールドに表示されるまで繰り返します。

      4. 選択済みフィールドで、上向き矢印と下向き矢印を使用して列名の順序を変更することもできます。

  4. 1ページごとの表示行数フィールドで、1つのページに表示するログ・エントリの数を選択します。
  5. 「戻される結果の最大数」フィールドで、表示するログ・エントリの最大数を選択するか、「すべて表示」を選択してすべてを表示します。
  6. 次のいずれか1つを実行します。
    • 「適用」をクリックして、新しい設定を保存します。
    • 「リセット」をクリックして変更を破棄し、元の設定を保持します。
