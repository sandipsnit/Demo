表の表示のカスタマイズ

この表カスタマイザで、指定した設定に基づいた表情報を表示できます。

表のデフォルトのソート順は、表の最初の列で決まることに注意してください。

  1. 「表・カスタマイザ」アイコンをクリックします。追加のフィールドが表示されます。

    2. 「取消し」をクリックすると、いつでもこの表を閉じて元の設定に戻すことができます。

  2. 「列」フィールドで、表示する列を選択します。
    1. 「使用可能な列」フィールドから列名を選択します。
    2. 矢印をクリックして、この列名を「選択された列」フィールドに移動します。

      3. アスタリスク(*)は必須列を示しています。

    3. 表示するすべての列名が「選択された列」フィールドに表示されるまで繰り返します。

      4. 「選択された列」フィールドで、上向き矢印と下向き矢印を使用して列名の順序を変更することもできます。

  3. 列を表示しないようにするには、「選択された列」フィールドにある列名を「使用可能な列」フィールドに移動します。
  4. 「行」ドロップダウン・リストで、1ページに表示する行数を選択します。
  5. 次のいずれか1つを実行します。
    • 「適用」をクリックしてこれらの設定を保存し、表カスタマイザを閉じます。
    • 「リセット」をクリックして変更を破棄し、元の設定を保持します。
