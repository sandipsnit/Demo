アラート・ルールの検索

  1. 「リソース・ブラウザ」→「ビジネス・サービス」または「プロキシ・サービス」を選択します。
  2. サービス名をクリックします。
  3. 「SLAアラート・ルール」タブを選択します。
  4. このサービスの特定のアラート・ルールを見つけるために、ページをスクロールします。表の上下にあるページ・コントロールを使用します。ページを移動するには、ページ番号を選択するか、次のページ、前のページ、最初のページまたは最後のページに移動する矢印ボタンを使用します。

このページでは、次の操作も可能です。

  • 「操作設定」タブを選択して、このサービスのモニターを構成します。プロキシ・サービスの操作設定の構成およびビジネス・サービスの操作設定の構成を参照してください。
  • 「新しく追加」をクリックして、新しいアラート・ルールを追加します。アラート・ルールの作成および編集を参照してください。
