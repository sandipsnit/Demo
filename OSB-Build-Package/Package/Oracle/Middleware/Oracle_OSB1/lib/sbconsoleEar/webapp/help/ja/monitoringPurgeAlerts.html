アラートのパージ

このページでは、指定したSLAアラートまたはパイプライン・アラートを永続的に削除できます。

注意: このアクションは取り消すことができません。
  1. 「操作」→「ダッシュボード」を選択します。
  2. 「SLAアラート」タブまたは「パイプライン・アラート」タブで、「拡張アラート履歴」リンクをクリックします。
  3. 拡張アラート履歴ページの右上にある「SLAのアラート履歴のパージ」または「パイプラインのアラート履歴のパージ」をクリックします。
  4. 表25-3のいずれかの手順を実行します。

    5. 表25-3 アラートのパージのオプション
    操作
    これらの手順を完了するには...
    • すべてのSLAアラートをパージ
    • すべてのパイプライン・アラートをパージ
    1. すべてのアラートのパージを選択します。
    2. 「パージ」をクリックします。
    • 特定期間に該当するSLAアラートをパージ
    • 特定期間に該当するパイプライン・アラートをパージ
    1. パージするアラートの開始範囲...「まで」を選択します。
    2. 「パージ開始日時」フィールドで、月、日、年および時間を入力します。
    3. 「パージ終了日時」フィールドで、月、日、年および時間を入力します。
    4. 「パージ」をクリックします。

注意: パージ中のアラートがあるサーバーが使用不可の場合、ロギング・フレームワークによってパージの記録が保持され、30分おきにサーバーの使用可能性がポーリングされます。ロギング・フレームワークでサーバーが使用可能であることが検出されると、アラートがパージされます。
