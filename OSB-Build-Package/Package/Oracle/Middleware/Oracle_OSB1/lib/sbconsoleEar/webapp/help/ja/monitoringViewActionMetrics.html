アクション・メトリックの表示

このページでは、サービス・モニター情報(表25-10)およびパイプラインを持つプロキシ・サービスのアクション・メトリック(表25-14)を表示します。

表25-14 アクション・メトリック
プロパティ
説明
ノード
ネストされたアクションの階層表示。
カウント
「統計の表示」フィールドで「現在の集約間隔」を選択した場合は、現在の集約間隔での呼出し数。
「統計の表示」フィールドで「最後のリセット以降」を選択した場合は、最後の統計リセット以降の呼出し数。
メッセージにつき何回もアクションを呼び出せるため、これはメッセージの数とは異なります。
平均時間
「統計の表示」フィールドで「現在の集約間隔」を選択した場合は、現在の集約間隔で呼出しにかかる平均実行時間。
「統計の表示」フィールドで「最後のリセット以降」を選択した場合は、最後の統計リセット以降で呼出しにかかる平均実行時間。
合計時間
「統計の表示」フィールドで「現在の集約間隔」を選択した場合は、現在の集約間隔で呼出しにかかる合計実行時間。
「統計の表示」フィールドで「最後のリセット以降」を選択した場合は、最後の統計リセット以降での合計実行時間。
