MQ接続

注意: Oracle Service BusでサポートされているWebSphere MQのバージョンについては、http://www.oracle.com/technology/software/products/ias/files/fusion_certification.htmlのOracle Fusion Middlewareでサポートされているシステム構成を参照してください。WebSphere MQクライアント・ライブラリはOracle Service Busインストーラにバンドルされていません。WebSphere MQのインストールからWebSphere MQクライアント・ライブラリを取得してください。

MQ接続は、複数のMQプロキシおよびビジネス・サービスにわたって再利用できる共有リソースです。MQプロキシとビジネス・サービスは、MQキューにアクセスする前に、MQキュー・マネージャに接続する必要があります。MQ接続リソースでは、MQキュー・マネージャに接続するために必要な接続パラメータを提供しています。

各MQ接続リソースには接続プールがあります。指定されたMQ接続リソースを使用して特定のキュー・マネージャに接続するすべてのビジネス・サービスまたはプロキシ・サービスでは、そのリソース用に作成された同じ接続プールを使用します。そのため、同じキュー・マネージャを使用する複数のビジネス・サービスとプロキシ・サービスは接続プールを共有します。

Oracle Service Bus MQ接続リソースとネイティブMQトランスポートの詳細は、Oracle Fusion Middleware Oracle Service Busのための開発者ガイドのMQトランスポート(http://www.oracle.com/pls/as111170/lookup?id=OSBDV1117)を参照してください。

WebSphere MQの基礎概念の詳細は、 http://www.redbooks.ibm.com/redbooks/SG247128/wwhelp/wwhimpl/java/html/wwhelp.htmを参照してください。
