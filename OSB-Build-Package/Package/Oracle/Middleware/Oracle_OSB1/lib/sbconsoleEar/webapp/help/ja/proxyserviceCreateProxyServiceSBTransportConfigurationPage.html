SBトランスポート構成ページ

プロキシ・サービスの作成/編集 - SBトランスポート構成ページは、プロキシ・サービスを作成および編集(プロキシ・サービスの作成と構成およびプロキシ・サービス構成の編集を参照)に使用する一連のページのうちの1つです。

詳細は、Oracle Fusion Middleware Oracle Service Busのための開発者ガイドのSBトランスポート(http://www.oracle.com/pls/as111170/lookup?id=OSBDV997)を参照してください。

このページでは、SB (サービス・バス)トランスポート・プロトコルを使用してプロキシ・サービスのトランスポート設定を構成します。表19-12に、このページの使用方法を示します。

表19-12 プロキシ・サービスの作成/編集 - SBトランスポート構成ページ
オプション
作成または編集の方法
ディスパッチ・ポリシー
このエンドポイントのディスパッチ・ポリシーを選択するか、またはデフォルトのディスパッチ・ポリシーを使用します。
ディスパッチ・ポリシーは、リクエストを処理するサービスのエンドポイントに使用するWLSワーク・マネージャのインスタンスを参照します。ワーク・マネージャの詳細は、Oracle Fusion Middleware Oracle WebLogic Serverのサーバー環境の構成のワーク・マネージャを使用したスケジューリング済作業の最適化(http://www.oracle.com/pls/as111170/lookup?id=CNFGD112)を参照してください。
SSLを使用
指定されている場合、要求をSSL接続を使用して送信する必要があります。
ただし、非セキュアな接続が禁止されるわけではありません。管理者は、サーバーのすべての非セキュアなプロトコル(T3またはHTTPなど)を閉じて、セキュアなクライアント接続を厳格に実施する必要があります。

終了後

「次」をクリックして次のページでこのサービスの構成を続行するか、「最後」をクリックしてプロキシ・サービスの作成/編集 - サマリー・ページでこの構成を確認および保存します。
