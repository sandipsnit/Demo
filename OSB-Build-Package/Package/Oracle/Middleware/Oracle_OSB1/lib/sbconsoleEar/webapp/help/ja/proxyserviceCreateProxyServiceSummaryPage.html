サマリー・ページ

プロキシ・サービスの作成と構成およびプロキシ・サービス構成の編集で説明されているように、プロキシ・サービスの作成/編集 - サマリー・ページは、プロキシ・サービスを作成および編集するための一連のページのうちの1つです。

このページでは、プロキシ・サービスの構成設定を保存する前に、表示または変更します。

設定を表示または変更するには、該当するカテゴリ(たとえば、「全般的な構成」、「トランスポート構成」など)の行の「編集」をクリックします。編集できるページは、プロキシ・サービスの作成時に構成したページによって異なります。すべてのページを次のリストに示します:

  • 全般的な構成ページ
  • メッセージ・タイプの構成ページ
  • トランスポート構成ページ
  • (プロトコル)トランスポート構成ページ
  • 電子メール・トランスポート構成ページ
  • ファイル・トランスポート構成ページ
  • FTPトランスポート構成ページ
  • HTTPトランスポート構成ページ
  • JCAトランスポート構成ページ
  • JEJBトランスポート構成ページ
  • JMSトランスポート構成ページ
  • MQトランスポート構成ページ
  • SBトランスポート構成ページ
  • SFTPトランスポート構成ページ
  • Tuxedoトランスポート構成ページ
  • WSトランスポート構成ページ
  • 操作選択構成ページ
  • メッセージ処理ページ
