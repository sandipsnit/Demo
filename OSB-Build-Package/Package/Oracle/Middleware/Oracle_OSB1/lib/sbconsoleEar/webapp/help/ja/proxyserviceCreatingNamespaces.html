インライン式で使用するためのネームスペースの作成

「ネームスペース定義」パレットには、デフォルトのネームスペースのリストが含まれています。新しいネームスペースを定義することもできます。

開始前

この説明では、インラインXQueryおよびXPath式の作成と編集で説明したように、XQuery式エディタ、XQuery条件エディタまたはXPath式エディタで式を作成または編集しているものと仮定します。

ネームスペースを作成してインライン式で使用するには

  1. 「ネームスペース定義」を選択します。「ネームスペース定義」パレットには、デフォルトのネームスペースのリストと、存在する場合は、変数ネームスペースおよびユーザー定義済のネームスペースのリストが含まれています。
  2. ユーザーのネームスペースを定義および追加するには
    1. 「ネームスペースの追加」をクリックします。
    2. 「接頭辞」フィールドに、ネームスペースの一意の識別子を入力します。同じ接頭辞を複数回使用することはできません。
    3. 「URI」フィールドに、このネームスペースのURLをhttp://url/.../の書式で入力するか、またはURNをuddi:server:の書式で入力します。
    4. 「追加」をクリックして、「ユーザー定義済みネームスペース」にネームスペースを追加します。
    5. インラインXQueryおよびXPath式の作成と編集の説明に従って、ユーザー定義済ネームスペースをコピーして、XQuery式、XQuery条件またはXPathに貼り付けます。

終了後

インラインXQueryおよびXPath式の作成と編集の説明に従って、続行します。
