動的XQuery式からインラインXQueryへのバインド

XQuery式エディタ・ページでは、実行時に事前登録されたXQueryリソース名を評価する動的XQuery式を指定できます。このエディタの詳細は、インラインXQueryおよびXPath式の作成と編集を参照してください。

動的XQuery式を定義するには

  1. 「動的XQuery」オプションを選択します。
  2. 「XQueryリソースの式を入力」領域で、実行時に事前登録されたXQueryリソース名を評価するXQuery式を入力します。

    3. 次にXQueryリソースの構文(リソースのフルネーム表示)を示します。

    Project/folder1/folder2/XQueryResourceName

  3. カスタム変数のXQueryテンプレートを使用して変数をバインド領域で、変換の入力パラメータを定義します。
    • 「XQueryテンプレートの選択」フィールドで「参照」をクリックして、問合せのテンプレート(変数の番号と名前)として提供する既存の登録されたリソースを選択します。テンプレートを選択後、「変数のバインド」領域に変数が表示されます。テンプレートが構成に存続していないことに留意してください。そのかわりに、テンプレートは問合せの変数を指定するクイック・スタートとして役立ちます。
    • 「カスタム変数の追加」フィールドに変数名を入力して、「追加」をクリックします。「変数名」の下にリストされた各変数に、マップされるXQuery式を入力します。マッピングはパラメータごとに定義する必要があります。たとえば、XQuery変換にoneおよびtwoという入力パラメータがある場合、「変数名」フィールドには、oneおよびtwoという2つのラベルが表示されます。XQuery式が入力されるテキスト・ボックスは、各ラベルに関連付けられます。

      • 次のXQuery式は、変数フィールドの有効な入力例です。

      $body/*[1]

      $body/po:PurchaseOrder

  4. 終了した後は、次のいずれかの操作に進みます。
    • 「検証」をクリックします。式が正しく検証された場合は、メッセージが表示されます。
    • 「テスト」をクリックします。XQueryテストの実行を参照してください。
    • 変更を保存または破棄します。
