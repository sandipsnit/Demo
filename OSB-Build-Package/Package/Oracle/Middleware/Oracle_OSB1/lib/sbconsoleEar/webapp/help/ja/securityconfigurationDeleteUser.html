ユーザーの削除

ユーザーのサマリー・ページを使用すると、選択したユーザーまたは複数のユーザーを削除できます。

  1. WebLogic ServerのAdmin特権を持つユーザーとしてOracle Service Bus管理コンソールにログインします。Adminロールのユーザーのみがセキュリティ構成データを変更できます。Oracle Fusion Middleware Oracle Service Busのための開発者ガイドの管理セキュリティの構成(http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539)を参照してください。
  2. 「セキュリティ構成」を選択して、ユーザーのサマリー・ページを表示します。
  3. 削除するユーザーを選択します。必要であれば、複数のユーザーを選択することもできます。

    4. セッション内で作業しているときもセッション外で作業しているときも、ユーザーを削除できます。

  4. 「削除」をクリックします。ユーザーの削除の確認を要求するメッセージが表示されます。
  5. 「OK」をクリックして、ユーザーを削除します。

    6. Oracle Service Bus管理コンソールでは、ユーザーが削除されます。ユーザーを削除しているときにセッション内で作業していて、その後にセッションを破棄しても、Oracle Service Bus管理コンソールでは更新内容は削除されません。

  6. 削除するユーザーの「オプション」列にある「削除」アイコンをクリックする方法もあります。
