分割-結合

分割-結合を使用すると、メッセージ呼出しを(順次送信するのみでなく)並行して送信し、応答を集約できます。分割-結合の詳細は、Oracle Fusion Middleware Oracle Service Busのための管理者ガイドの分割-結合によるサービスのパフォーマンスの向上(http://www.oracle.com/pls/as111170/lookup?id=OSBAG291)を参照してください。また、Oracle Fusion Middleware Oracle Service Bus開発者ガイドの分割-結合の操作(http://www.oracle.com/pls/as111170/lookup?id=OSBDV193)および分割-結合ユーザー・インタフェース・リファレンス(http://www.oracle.com/pls/as111170/lookup?id=OSBDV573)も参照してください。

フロー・トランスポート型のビジネス・サービスを介して分割-結合にアクセスできます。詳細は、フロー・トランスポート構成ページを参照してください。
