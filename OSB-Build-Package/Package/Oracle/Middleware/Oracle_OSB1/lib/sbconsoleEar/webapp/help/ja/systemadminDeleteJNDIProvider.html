JNDIプロバイダの削除

  1. セッションの作成または編集をまだ行っていない場合は、「作成」をクリックして新しいセッションを作成するか、「編集」をクリックして既存のセッションに入ります。チェンジ・センターの使用を参照してください。
  2. 「システム管理」→「JNDIプロバイダ」を選択します。
  3. JNDIプロバイダのサマリー・ページで、JNDIプロバイダを選択します。「検索」オプションを使用して、特定のJNDIプロバイダを検索することもできます。JNDIプロバイダの表示を参照してください。
  4. 削除するJNDIプロバイダの「オプション」フィールドで「削除」アイコンをクリックします。このリソースが他のリソースから参照されている場合、削除の警告アイコンが表示されます。確認の警告が表示されますが、リソースは削除できます。これにより、削除されたリソースへの参照が解決されないため、競合が起きる場合があります。
  5. セッションを終了して構成をランタイムにデプロイするには、「チェンジ・センター」の「アクティブ化」をクリックします。
