プロキシ・サーバーの表示

このページには、プロキシ・サーバーのリストと、表30-5に示す情報を表示できます。

プロキシ・サーバー・リソースは、ビジネス・サービスのHTTPトランスポートを構成する場合に使用されます。詳細は、HTTPトランスポート構成ページを参照してください。

表30-5にプロキシ・サーバー情報を示します。

表30-5 プロキシ・サーバー情報 
プロパティ
説明
名前
プロキシ・サーバー・リソースに割り当てられた一意の名前。この名前はプロキシ・サーバーの表示 - 構成ページにリンクされています。プロキシ・サーバーの編集を参照してください。
オプション
  • このプロキシ・サーバー・リソースの名前を変更するには、「名前変更」アイコンをクリックします。
  • 特定のプロキシ・サーバーを削除「削除」アイコンをクリックして、特定のプロキシ・サーバーを削除します。プロキシ・サーバーの削除を参照してください。
