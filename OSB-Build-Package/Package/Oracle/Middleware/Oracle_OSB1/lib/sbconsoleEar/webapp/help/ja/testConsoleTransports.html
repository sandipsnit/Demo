テスト・コンソールでのランタイムによるトランスポート設定の使用方法

プロキシ・サービスのテスト・データの構成およびビジネス・サービスのテスト・データの構成では、テスト・コンソールを起動してプロキシ・サービスまたはビジネス・サービスをテストするときに、アウトバウンド・リクエストのトランスポート・ヘッダー、トランスポート・メタデータおよびトランスポート関連のセキュリティ・データの値を構成する方法を説明しました。ただし、テスト・コンソールで作成できる仕様の一部は、実行時に無視されます。つまり、特定のヘッダーやメタデータの値は、テストの実行時にOracle Service Busランタイムで上書きされるか無視されます。制限のあるヘッダーとメタデータは、表32-10を参照してください。
