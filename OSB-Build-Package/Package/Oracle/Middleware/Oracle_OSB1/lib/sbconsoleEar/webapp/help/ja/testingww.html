プロキシ・サービスのテスト結果の表示

表32-2でプロキシ・サービスのテスト結果について説明します。トレースは、「直接呼出し」オプションと「トレースを含む」オプションが両方とも選択されている場合にのみ有効になります。

表32-2 プロキシ・サービスのテスト結果
セクション
説明
プロキシ・サービス名
テスト対象のプロキシ・サービスの名前です。
テスト・コンソールのアクション
「戻る」で前のブラウザ・ページが表示されます。
「閉じる」でテスト・コンソールが閉じます。
要求ドキュメント
テスト・コンソールによってプロキシ・サービスに送信された要求メッセージです。
要求メッセージがテスト・コンソールによって変更されなかった場合は、このセクションは最初は折りたたまれています。「フォーム」タブでSOAPサービスを構成した場合、またはWS-Securityが適用されている場合は、このセクションは最初は展開されています。
WS-Securityが適用されている場合、このセクションには2つのSOAPメッセージが表示されます。1つ目はクリア・テキストのメッセージ、2つ目は保護されたSOAPメッセージです。
応答ドキュメント
応答メッセージです。
応答WS-Securityが適用されたSOAPサービスの場合、このセクションには2つのSOAPメッセージが表示されます。1つ目のSOAPメッセージは、クライアントが受信する保護されたメッセージです。2つ目のSOAPメッセージは、対応するクリア・テキストのメッセージです。
応答メタデータ
メッセージの応答と共に戻されたメタデータです。
プロキシ・サービスのトレース
システムでパススルーされるメッセージの状態をトレースに表示します。「直接呼出し」オプションが選択されていない場合、トレースは実行されません。トレースの詳細については、「プロキシ・サービスのトレース」を参照してください。
