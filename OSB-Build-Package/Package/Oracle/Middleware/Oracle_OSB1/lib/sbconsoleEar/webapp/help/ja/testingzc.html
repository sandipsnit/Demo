MFL変換のテスト

MFL (メッセージ・フォーマット言語)ドキュメントは、バイナリ・データのレイアウトを記述するために使用する特別なXMLドキュメントです。MFLリソースは、2つの変換をサポートします。どちらの変換も、XMLからバイナリ、またはバイナリからXMLという、1つの入力だけを受け入れ、1つの出力を返します。

表32-5にMFL構成オプションを示します。

MFL変換構成
XML入力
XMLからバイナリへの変換では必須です。
MFLドキュメントのXMLスキーマは推測できます。テキスト・フィールドにサンプルXMLドキュメントが自動的に入力されます。
XML入力はファイルベースまたはテキストベースで入力できます。テキスト入力よりファイルの参照による入力が優先されます。テストに使用するファイルを参照して選択します。
バイナリ入力
バイナリからXMLへの変換では必須です。
バイナリ入力はファイルベースまたはテキストベースで入力できます。テキスト入力よりファイルの参照による入力が優先されます。テストに使用するファイルを参照して選択します。

設計時テストまたは実行時テストを行うことができます。

  1. 実行時テストを行うには、「アクティブ化」をクリックします。設計時テストを行う場合は、セッションをアクティブ化しません。
  2. 「リソース・ブラウザ > MFL」を選択して。「MFLファイルの要約」ページを表示します。
  3. 「アクション」の下で、テストするリソースに対応する「テスト・コンソールの起動」アイコンをクリックします。テスト・コンソールによって、リソース・テスト・ページが開かれます。
  4. テストする変換を選択します。
  5. MFLリソースのテスト・データを構成します。詳細は、表32-5を参照してください。
  6. 「実行」をクリックします。「リソース・テスト」ページに結果が表示されます。
  7. もう一度テストする場合は、「戻る」をクリックします。テスト・コンソールを閉じたら、リソースを変更し、再度テストできます。
