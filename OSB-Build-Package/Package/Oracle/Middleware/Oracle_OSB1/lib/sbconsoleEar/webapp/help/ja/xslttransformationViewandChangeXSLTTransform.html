XSL変換の編集

  1. セッションの作成または編集をまだ行っていない場合は、「作成」をクリックして新しいセッションを作成するか、「編集」をクリックして既存のセッションに入ります。チェンジ・センターの使用を参照してください。
  2. XSL変換の検索の説明に従って、XSL変換を検索します。
  3. XSL変換名をクリックします。XSL変換の詳細を表示ページに、デフォルトでXSLTの構造ビューが表示されます。これには表14-2に示す情報が含まれています。

    4. 表14-2 XSL変換リソースの詳細 
    プロパティ
    説明
    最終更新者
    このXSL変換を作成したか、構成にインポートしたユーザー。
    最終更新日
    ユーザーがこのXSL変換を作成または構成にインポートした日時。このリソースの変更履歴を表示するには、日時のリンクをクリックします。変更履歴の表示ページを参照してください。
    参照
    このXSL変換が参照するオブジェクトの数。該当する参照がある場合は、数字のリンクをクリックするとオブジェクトのリストが表示されます。詳細は、リソースへの参照の表示を参照してください。
    参照元
    このXSL変換を参照するオブジェクトの数。該当する参照がある場合は、数字のリンクをクリックするとオブジェクトのリストが表示されます。たとえば、このXSL変換を特定のプロキシ・サービスのメッセージ・フロー内の変数に割り当てた場合は、このリンクをクリックすると、そのプロキシ・サービスが参照としてリストに表示されます。詳細は、リソースへの参照の表示を参照してください。
    説明
    このXSL変換の説明。
    対象ネームスペース
    XSL変換に含まれる定義を修飾するために使用されるネームスペース。
    XSLTの依存関係
    XSLTに関連付けられている次の属性とグループ。
    • XSLTの場所
    状態
    次のXSL変換のステータス。
    • 有効

      • 現在のXSL変換に含まれているすべてのXSL変換の場所が指定され、有効です。ネストされたXSL変換(含まれているXSL変換がさらに内包するXSL変換)についてもすべての場所が指定され、有効な状態です。

    • 無効

      • 現在のXSL変換に含まれているXSL変換の1つまたは複数の場所が指定されていないか、これらが有効ではありません。また、1つまたは複数のネストされたXSL変換(含まれているXSL変換がさらに内包するXSL変換)が指定されていない場合や、有効でない場合もあります。無効なXSL変換を解決するには、「参照の編集」をクリックします。詳細は、未解決のXSL変換の参照の解決を参照してください。

  4. 「テキスト・ビュー」をクリックして、XSL変換の詳細をテキストで表示します。テキスト・ビューには、表14-3に示す情報が含まれています。

    5. 表14-3 XSL変換テキストの詳細
    プロパティ
    説明
    対象ネームスペース
    XSL変換に含まれる依存関係を修飾するために使用されるネームスペース。
    XSLT
    XSL変換のフル・テキスト。

  5. フィールドを変更するには、「編集」をクリックします。各フィールドの説明については、XSL変換の追加を参照してください。

    6. 「リソース名」フィールドは変更できません。

  6. 「保存」をクリックして、現在のセッションで更新をコミットします。
