비즈니스 서비스 구성 편집

  1. 아직 이 작업을 수행하지 않은 경우 생성을 눌러 새 세션을 생성하거나 편집을 눌러 기존 세션을 입력합니다. 변경 센터 사용을 참조하십시오.

    2. 세부 정보를 보기만 하려면 세션을 입력하지 않아도 됩니다.

  2. 비즈니스 서비스 찾기에 설명된 대로 보거나 편집할 비즈니스 서비스를 찾습니다.
  3. 비즈니스 서비스 이름을 누릅니다. 비즈니스 서비스 보기 - 구성 세부 정보 페이지에는 선택한 비즈니스 서비스에 대한 구성 정보가 표시됩니다.
  4. 설정을 보거나 수정하려면 다음 중 하나를 수행합니다.
    • 속성을 보거나 편집할 범주 이름 옆에 있는 편집 아이콘을 누릅니다(예: 일반 구성, 전송 구성 등). 편집할 수 있는 페이지는 비즈니스 서비스를 생성할 때 구성한 페이지에 따라 다릅니다.

      • 이러한 모든 페이지 목록은 비즈니스 서비스 보기 - 구성 세부 정보 페이지를 참조하십시오.

    • 페이지 하단에서 편집을 눌러 이 비즈니스 서비스를 구성하기 위한 일련의 페이지 중 첫번째 페이지인 비즈니스 서비스 생성/편집 - 일반 구성 페이지를 표시합니다.
  5. 비즈니스 서비스 생성 및 구성에 설명된 대로 보기 또는 편집을 계속합니다.
  6. 비즈니스 서비스 편집 - 요약 페이지에서 저장을 눌러 현재 세션에서 갱신을 커밋합니다.
  7. 세션을 종료하고 구성을 런타임에 배치하려면 변경 센터에서 활성화를 누릅니다.
