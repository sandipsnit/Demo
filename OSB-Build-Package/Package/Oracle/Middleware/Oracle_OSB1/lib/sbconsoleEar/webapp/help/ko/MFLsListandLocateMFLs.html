MFL 찾기

  1. 다음 작업을 수행합니다.
    • 프로젝트 탐색기를 선택하여 프로젝트 뷰 페이지 또는 프로젝트/폴더 뷰 페이지를 표시합니다. 그런 다음 프로젝트와 폴더를 탐색하여 MFL을 찾습니다.
    • 리소스 브라우저 > MFL을 선택합니다. 그러면 MFL 파일 요약 페이지에 표 15-1과 같은 정보가 표시됩니다. 속성에 대한 자세한 설명은 MFL 편집을 참조하십시오.
  2. 목록에서 항목 수를 제한하기 위해 이름, 경로 또는 이 모두를 사용하여 필터링할 수 있습니다. 이름 및 경로 필드에 검색 대상의 이름과 경로를 입력한 다음 검색을 누릅니다.

    3. MFL 파일이 있는 폴더 이름과 프로젝트 이름입니다.

    모두 보기를 눌러 검색 필터를 제거하고 모든 MFL 파일을 표시합니다.

    표 15-1 MFL 세부 정보
    속성
    설명
    MFL 파일 이름
    MFL에 할당된 고유 이름입니다. 이 이름은 MFL 리소스 세부 정보 보기 페이지에 대한 링크입니다. MFL 편집을 참조하십시오.
    경로
    MFL 파일이 있는 폴더 이름과 프로젝트 이름입니다. 경로는 이 리소스가 포함된 프로젝트 또는 폴더에 대한 링크입니다. 프로젝트 및 폴더를 사용하여 리소스 이름 식별을 참조하십시오.
    작업
    서비스 및 변환 설계 검증 및 테스트에 사용할 테스트 콘솔을 호출하려면 테스트 콘솔 시작 아이콘을 누릅니다. MFL에 대해 세션의 사용 여부와 관계없이 테스트 콘솔을 사용할 수 있습니다. 변환 테스트를 참조하십시오.
    옵션
    삭제 아이콘을 눌러 특정 MFL을 삭제할 수 있습니다. MFL 삭제를 참조하십시오.
