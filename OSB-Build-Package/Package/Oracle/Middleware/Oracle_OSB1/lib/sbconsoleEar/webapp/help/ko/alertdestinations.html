경보 대상

경보 대상 리소스는 Oracle Service Bus에서 경보 통지를 수신할 수 있는 받는 사람 목록을 캡처합니다. 일반적인 시스템 모니터링 컨텍스트에서 Oracle Service Bus에서 생성한 경보는 한정된 사용자 집합에 대해서만 중요성을 가집니다.

Oracle Service Bus에서 제공된 컨텍스트에 따라 받는 사람 집합을 포함하도록 각 경보 대상 리소스를 구성할 수 있습니다. 경보 대상은 메시지 흐름에서 구성된 경보 작업과 SLA 경보 규칙에 의해 사용됩니다. 경보 대상에는 콘솔(기본값), 보고 데이터 스트림, SNMP 트랩, 전자 메일, JMS 대기열 또는 JMS 항목 등의 유형이 하나 이상 포함될 수 있습니다.

전자 메일과 JMS 대상의 경우 대상 리소스에 전자 메일 주소 목록 또는 JMS URI 목록이 각각 포함될 수 있습니다.

서비스의 경보 구성에서 경보 대상을 재사용할 수 있습니다. 경보 작업 추가를 참조하십시오.
