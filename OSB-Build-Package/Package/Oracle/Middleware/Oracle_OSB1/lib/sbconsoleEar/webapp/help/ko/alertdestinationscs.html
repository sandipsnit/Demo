전자 메일 받는 사람 추가

전자 메일 대상을 추가하기 전에 SMTP 서버를 구성(SMTP 서버 추가 참조)하거나 WebLogic Server에서 JavaMail 세션을 구성(http://download.oracle.com/docs/cd/E15523_01/apirefs.1111/e13952/taskhelp/mail/CreateMailSessions.html에서 Oracle Fusion Middleware Oracle WebLogic Server 관리 콘솔 온라인 도움말의 "Configure access to JavaMail" 참조)해야 합니다. 사용 가능한 구성된 SMTP 서버 리소스 또는 JavaMail 세션이 없을 경우 전자 메일 받는 사람을 구성할 수 없습니다.

  1. 드롭다운 목록에서 전자 메일을 통해 경보 보내기를 선택하고 추가를 눌러 전자 메일 받는 사람 추가 페이지를 표시합니다.
  2. 메일 받는 사람 필드에 다음 형식으로 전자 메일 받는 사람을 입력합니다. mailto:username@hostname

    3. 이 필드는 필수 필드입니다.

    다중 전자 메일 받는 사람을 지정하려면 다음과 같이 사용자 이름과 호스트 이름을 콤마로 구분하여 입력합니다. 예: mailto:username@hostname [,username_1@hostname_1]...[,username_n@hostname_n]

    메일 받는 사람 중 첫 번째 주소에만 접두어 "mailto:"가 추가됩니다.

  3. SMTP 서버 필드에서 보내는 전자 메일의 SMTP 서버 이름을 선택합니다. 다음 단계에서 메일 세션을 선택할 경우 이 필드는 필수가 아닙니다.
  4. 메일 세션 필드에서 사용 가능한 메일 세션을 선택합니다. 이전 단계에서 SMTP 서버를 선택한 경우 이 필드는 필수 필드가 아닙니다.
  5. 보낸 사람 이름 필드에 경보 통지를 보내는 사람 이름을 입력합니다. 이 필드는 선택 사항입니다.
  6. 보낸 사람 주소 필드에 적합한 전자 메일 주소를 입력합니다. "보낸 사람 이름" 필드에 값을 지정한 경우 이 필드는 필수 필드입니다.
  7. 회신 이름 필드에 회신 대상의 이름을 입력합니다. 이 필드는 선택 사항입니다.
  8. 회신 주소 필드에 회신받을 수 있는 전자 메일 주소를 입력합니다. "회신 이름" 필드에 값을 지정한 경우 이 필드는 필수 필드입니다.
  9. 연결 제한 시간 필드에 연결 제한 시간에 도달하기 전에 서버 응답을 기다릴 시간(초)을 입력합니다. 기본값은 0입니다.
  10. 요청 인코딩 필드에 문자 집합 인코딩 값을 입력합니다. 기본 인코딩 값은 iso-8859-1입니다.
  11. 저장을 누릅니다. 전자 메일 받는 사람이 이 경보 대상의 전자 메일 받는 사람 목록에 추가되고 현재 세션에 저장됩니다.
