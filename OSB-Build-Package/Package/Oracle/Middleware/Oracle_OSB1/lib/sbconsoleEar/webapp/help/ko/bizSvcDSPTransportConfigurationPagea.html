DSP 전송 구성 페이지

비즈니스 서비스 생성/편집 - DSP 전송 구성 페이지는 비즈니스 서비스 생성 및 구성과 비즈니스 서비스 구성 편집에 설명된 것처럼 비즈니스 서비스를 생성하고 편집하기 위한 일련의 페이지 중 하나입니다.

자세한 내용은 Oracle Service Bus를 통해 데이터 서비스 액세스를 참조하십시오.

이 페이지에서는 DSP(Oracle Data Services Integrator) 전송 프로토콜을 사용하여 비즈니스 서비스의 전송 설정을 구성할 수 있습니다. 표 17-4에서는 이 페이지를 사용하는 방법에 대해 설명합니다.

표 17-4 비즈니스 서비스 생성/편집 - DSP 전송 구성 페이지 
옵션
생성 또는 편집
디버그 수준
다음 중 하나를 지정합니다.
  • 0 - 디버그 정보가 없는 경우
  • 1 - 요청 메시지에 대한 정보를 출력하려는 경우
  • 3 - 요청 및 응답 메시지에 대한 정보를 출력하려는 경우
서비스 계정
찾아보기를 누르고 표시된 목록에서 서비스 계정을 선택합니다. 서비스 계정을 지정하지 않을 경우 익명 제목이 사용됩니다.

자세한 내용은 다음을 참조하십시오.

  • 서비스 계정
  • 서비스 계정 추가
작업 할당 정책
이 끝점에 대해 작업 할당 정책에 사용할 WebLogic Server 작업 관리자의 인스턴스를 선택합니다. 다른 작업 관리자가 존재하지 않으면 기본 작업 관리자가 사용됩니다.
작업 관리자는 응답 처리에 대한 회신 메시지를 게시하는 데에 사용됩니다. 작업 관리자에 대한 자세한 내용은 작업 관리자를 사용하여 예약된 작업 최적화 및 WebLogic Server Administration Console 온라인 도움말의 전역 작업 관리자 생성를 참조하십시오.

Oracle Service Bus에서 Oracle Data Services Integrator에 액세스하는 방법에 대한 자세한 내용은 Oracle Service Bus를 통해 데이터 서비스 액세스를 참조하십시오.

마친 후에
