FTP 전송 구성 페이지

비즈니스 서비스 생성/편집 - FTP 전송 구성 페이지는 비즈니스 서비스 생성 및 구성과 비즈니스 서비스 구성 편집에 설명된 것처럼 비즈니스 서비스를 생성하고 편집하기 위한 일련의 페이지 중 하나입니다.

이 페이지에서는 FTP 전송 프로토콜을 사용하여 비즈니스 서비스의 전송 설정을 구성할 수 있습니다. 표 18-9에서는 이 페이지를 사용하는 방법에 대해 설명합니다.

표 18-9 비즈니스 서비스 생성/편집 - FTP 전송 구성 페이지
옵션
생성 또는 편집
사용자 인증
FTP 서버 사용자가 익명인 경우 익명을 선택하고 FTP 서버 사용자가 외부적으로 구성된 계정인 경우 외부 사용자를 선택합니다.
ID(전자 메일 ID)
이 필드는 사용자 인증 옵션을 익명으로 설정한 경우에만 사용할 수 있습니다.
익명 사용자에 대한 메일 ID를 입력합니다.
서비스 계정
이 필드는 사용자 인증 옵션을 외부 사용자로 설정한 경우에만 사용할 수 있습니다.
외부 사용자에 대한 서비스 계정을 입력합니다.

자세한 내용은 다음을 참조하십시오.

  • 서비스 계정
  • 서비스 계정 추가
시간 초과
접속이 끊기기 전의 소켓 시간 초과 간격(초)을 입력합니다. 기본값은 60초입니다.
대상 파일 이름의 접두어
파일이 원격 서버에 저장될 때 사용되는 파일 이름의 접두어를 입력합니다.
이 필드에는 *를 입력하지 마십시오. 이 문자를 입력하면 런타임 예외 사항이 발생합니다.
대상 파일 이름의 접미어
파일이 원격 서버에 저장될 때 사용되는 파일 이름의 접미어를 입력합니다.
이 필드에는 *를 입력하지 마십시오. 이 문자를 입력하면 런타임 예외 사항이 발생합니다.
전송 모드
전송 모드로 ASCII 또는 이진을 선택합니다.
요청 인코딩
기본값 UTF-8을 ftp 전송 요청을 위한 문자 집합 인코딩으로 적용하거나 다른 문자 집합 인코딩을 입력합니다.

마친 후에

다음을 눌러 다음 페이지에서 이 서비스를 계속 구성하거나 마지막을 눌러 비즈니스 서비스 생성/편집 - 요약 페이지에서 이 구성을 검토 및 저장합니다.
