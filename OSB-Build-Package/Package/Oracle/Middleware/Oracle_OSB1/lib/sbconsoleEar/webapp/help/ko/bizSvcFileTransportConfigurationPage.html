파일 전송 구성 페이지

비즈니스 서비스 생성/편집 - 파일 전송 구성 페이지는 비즈니스 서비스 생성 및 구성과 비즈니스 서비스 구성 편집에 설명된 것처럼 비즈니스 서비스를 생성하고 편집하기 위한 일련의 페이지 중 하나입니다.

이 페이지에서는 파일 전송 프로토콜을 사용하여 비즈니스 서비스의 전송 설정을 구성할 수 있습니다. 표 18-7에서는 이 페이지를 사용하는 방법에 대해 설명합니다.

표 18-7 비즈니스 서비스 생성/편집 - 파일 전송 구성 페이지
옵션
생성 또는 편집
접두어
파일 이름 앞에 추가할 접두어를 입력합니다.
이 필드에는 *를 입력하지 마십시오. 이 문자를 입력하면 런타임 예외 사항이 발생합니다.
접미어
파일 이름 뒤에 추가할 접미어를 입력합니다. 이 필드는 필수 필드입니다.
이 필드에는 *를 입력하지 마십시오. 이 문자를 입력하면 런타임 예외 사항이 발생합니다.
요청 인코딩
기본값 utf-8을 파일 전송 요청을 위한 문자 집합 인코딩으로 적용하거나 다른 문자 집합 인코딩을 입력합니다.

마친 후에

다음을 눌러 다음 페이지에서 이 서비스를 계속 구성하거나 마지막을 눌러 비즈니스 서비스 생성/편집 - 요약 페이지에서 이 구성을 검토 및 저장합니다.
