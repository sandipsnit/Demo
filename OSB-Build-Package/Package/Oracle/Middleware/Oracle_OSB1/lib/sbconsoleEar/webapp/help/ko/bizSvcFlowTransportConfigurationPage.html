흐름 전송 구성 페이지

비즈니스 서비스 생성/편집 - 흐름 전송 구성 페이지는 비즈니스 서비스 생성 및 구성과 비즈니스 서비스 구성 편집에 설명된 것처럼 비즈니스 서비스를 생성하고 편집하기 위한 일련의 페이지 중 하나입니다.

분할-조인을 액세스하려면 흐름 전송 유형 비즈니스 서비스를 사용합니다. 분할-조인에 대한 자세한 내용은 http://www.oracle.com/pls/as111170/lookup?id=OSBAG291에서 Oracle Fusion Middleware Developer's Guide for Oracle Service Bus의 "Improving Service Performance with Split-Join"을 참조하십시오.

이 페이지에서 흐름 전송 프로토콜을 사용하여 비즈니스 서비스의 전송 설정을 구성할 수 있습니다. 표 18-8에서는 이 페이지를 사용하는 방법에 대해 설명합니다.

표 18-8 비즈니스 서비스 생성/편집 - 흐름 전송 구성 페이지
옵션
생성 또는 편집
시간 초과
응답을 기다릴 시간(초)을 입력합니다. 이 값은 요청 전용(단방향) 작업에는 무시됩니다.

마친 후에

다음을 눌러 다음 페이지에서 이 서비스를 계속 구성하거나 마지막을 눌러 비즈니스 서비스 생성/편집 - 요약 페이지에서 이 구성을 검토 및 저장합니다.
