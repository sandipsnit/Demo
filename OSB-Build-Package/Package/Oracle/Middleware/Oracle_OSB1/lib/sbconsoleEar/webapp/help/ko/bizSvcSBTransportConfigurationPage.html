SB 전송 구성 페이지

비즈니스 서비스 생성/편집 - SB 전송 구성 페이지는 비즈니스 서비스 생성 및 구성과 비즈니스 서비스 구성 편집에 설명된 것처럼 비즈니스 서비스를 생성하고 편집하기 위한 일련의 페이지 중 하나입니다.

자세한 내용은 http://www.oracle.com/pls/as111170/lookup?id=OSBDV997에서 Oracle Fusion Middleware Developer's Guide for Oracle Service Bus의 "SB Transport"를 참조하십시오.

이 페이지에서는 SB(Service Bus) 전송 프로토콜을 사용하여 비즈니스 서비스의 전송 설정을 구성할 수 있습니다. 표 18-15에서는 이 페이지를 사용하는 방법에 대해 설명합니다.

표 18-15 비즈니스 서비스 생성/편집 - SB 구성 페이지
옵션
생성 또는 편집
작업 할당 정책
이 끝점에 대해 작업 할당 정책에 사용할 WebLogic Server 작업 관리자의 인스턴스를 선택합니다. 다른 작업 관리자가 존재하지 않을 경우 기본 작업 관리자가 사용됩니다. 작업 관리자에 대한 자세한 내용은 http://www.oracle.com/pls/as111170/lookup?id=CNFGD112에서 Oracle Fusion Middleware Configuring Server Environments for Oracle WebLogic Server의 "Using Work Managers to Optimize Scheduled Work"를 참조하십시오.
시간 초과
서비스가 시간 초과되기까지의 시간(초)입니다.
주: 서비스가 일회성인 경우 시간 초과가 무시됩니다.
서비스 계정
찾아보기를 누르고 표시된 목록에서 서비스 계정을 선택합니다. 서비스 계정을 지정하지 않을 경우 익명 제목이 사용됩니다.

자세한 내용은 다음을 참조하십시오.

  • 서비스 계정
  • 서비스 계정 추가
