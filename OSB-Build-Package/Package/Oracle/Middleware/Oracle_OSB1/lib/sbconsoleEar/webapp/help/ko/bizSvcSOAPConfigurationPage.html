SOAP 바인딩 구성 페이지

비즈니스 서비스 생성/편집 - SOAP 바인딩 구성 페이지는 비즈니스 서비스 생성 및 구성과 비즈니스 서비스 구성 편집에 설명된 것처럼 비즈니스 서비스를 생성하고 편집하기 위한 일련의 페이지 중 하나입니다.

이 페이지에서는 WSDL에 기반한 비즈니스 서비스의 SOAP 바인딩을 구성할 수 있습니다.

서비스가 Web Services Interoperability Organization에서 정의한 기본 프로파일을 따를지 여부를 지정하려면 WS-I 규격 강제 적용을 선택합니다. 이 옵션은 SOAP 1.1 서비스에서만 사용할 수 있습니다.

서비스가 WS-I 규격인 경우 해당 서비스에서 보내고 받는 메시지에 대해 검사가 수행됩니다.
