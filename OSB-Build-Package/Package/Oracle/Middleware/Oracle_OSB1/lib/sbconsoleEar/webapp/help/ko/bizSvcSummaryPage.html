요약 페이지

비즈니스 서비스 생성/편집 - 요약 페이지는 비즈니스 서비스 생성 및 구성과 비즈니스 서비스 구성 편집에 설명된 것처럼 비즈니스 서비스를 생성하고 편집하기 위한 일련의 페이지 중 하나입니다.

이 페이지에서는 비즈니스 서비스의 구성 설정을 보거나 수정한 후 저장할 수 있습니다.

설정을 보거나 수정하려면 해당 범주(예: 일반 구성, 전송 구성 등)의 행에서 편집을 누릅니다. 편집할 수 있는 페이지는 비즈니스 서비스를 생성할 때 구성한 페이지에 따라 다릅니다. 다음은 모든 페이지 목록입니다.

  • 일반 구성 페이지
  • 메시지 유형 구성 페이지
  • 전송 구성 페이지
  • 프로토콜별 전송 구성 페이지
  • 비즈니스 서비스 정책 페이지
  • 비즈니스 서비스 보안 정책 페이지
