비즈니스 서비스: 생성 및 관리

비즈니스 서비스는 메시지를 교환할 엔터프라이즈 서비스의 Oracle Service Bus 정의입니다. 프록시 서비스를 정의하는 것과 같은 방식으로 WSDL(Web Services Definition Language)을 사용하여 비즈니스 서비스를 정의합니다. 그러나 비즈니스 서비스에는 파이프 라인이 없다는 점에서 비즈니스 서비스의 구성은 프록시 서비스의 구성과 다릅니다. 따라서 비즈니스 서비스는 Oracle Service Bus 파이프 라인에 의해 구현되지 않는 모든 서비스를 말합니다.

자세한 내용은 http://www.oracle.com/pls/as111170/lookup?id=OSBAG111에서 Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus의 "Configuring Proxy Services and Business Services"를 참조하십시오.
