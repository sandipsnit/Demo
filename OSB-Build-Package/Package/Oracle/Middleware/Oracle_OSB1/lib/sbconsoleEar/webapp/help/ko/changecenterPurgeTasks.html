세션 활성화 내역 비우기

시작 및 종료 날짜별로 구분된 특정 기간 동안 활성화된 세션을 비울 수 있습니다. 이 작업은 세션 외부에서만 수행할 수 있습니다.

주의: 세션 활성화 내역을 비우면 다중 레벨의 실행 취소가 가능한 데이터가 삭제됩니다. 즉, 특정 기간 동안의 세션 활성화 내역을 비우면 해당 기간 동안 활성화된 세션을 실행 취소할 수 없게 됩니다.
  1. 사용 중인 세션이 활성 세션인지 확인한 다음 변경 내용 보기를 누릅니다.
  2. 구성 변경 내용 보기 페이지에서 날짜별 비우기를 누릅니다. 날짜별로 세션 활성화 내역 비우기 페이지가 표시됩니다.
  3. 각 필드에 MM/DD/YYYY 형식으로 시작 및 종료 날짜를 입력합니다.

    4. 지정된 날짜 간격에 해당되는 모든 세션의 세션 활성화 내역이 Oracle Service Bus에서 삭제됩니다.
