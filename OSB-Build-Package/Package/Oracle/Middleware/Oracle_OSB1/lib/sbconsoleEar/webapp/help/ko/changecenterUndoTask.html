작업 실행 취소

구성 변경 내용 보기 페이지를 사용하여 현재 세션 중에 Oracle Service Bus 구성에서 수행한 작업을 실행 취소하고 세션 외부에서 세션 활성화를 실행 취소할 수 있습니다. Oracle Service Bus를 사용하여 사용자의 시스템 리소스에 한정하여 다중 레벨의 세션 활성화를 실행 취소할 수 있습니다. 자세한 내용은 작업 실행 취소 순서를 참조하십시오.

주: Oracle Service Bus 버전 2.1에서 2.5로 업그레이드한 경우에는 업그레이드 전에 활성화된 세션을 실행 취소할 수 없습니다. 업그레이드 후에 활성화된 세션은 세션 활성화 내역에서 실행 취소가 가능합니다.
세션에서 작업 실행 취소
  1. 세션을 사용 중인지 확인합니다.
  2. 변경 센터에서 변경 내용 보기를 누릅니다.
  3. 특정 작업에 대한 옵션 열에서 실행 취소 아이콘을 누릅니다.

    4. 해당 작업이 실행 취소됩니다.

    현재 세션의 모든 변경 내용을 실행 취소할 수 있습니다. 그러나 그 이후에도 원하는 경우에는 실행 취소한 작업을 다시 실행 취소할 수 있으며 그 전에 실행 취소한 내용도 마찬가지로 다시 실행 취소할 수 있습니다.

이전에 활성화된 세션 실행 취소
  1. 세션 외부에 있는지 확인합니다.
  2. 변경 센터에서 변경 내용 보기를 누릅니다.
  3. 다음 작업 중 하나를 수행합니다.
    • 특정 작업에 대한 옵션 열에서 실행 취소 아이콘을 누릅니다. 세션 활성화가 실행 취소됩니다.
    • 특정 작업에 대한 옵션 열에서 세션에서 실행 취소 아이콘을 선택합니다. 새로운 세션이 생성되어 세션에서 활성화된 변경 내용을 실행 취소하는 데 필요한 작업으로 미리 채웁니다. 자세한 내용은 세션에서 실행 취소를 참조하십시오.
