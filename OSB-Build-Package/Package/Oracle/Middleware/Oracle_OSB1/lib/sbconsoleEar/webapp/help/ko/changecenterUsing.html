변경 센터 사용

변경 센터 모듈은 Oracle Service Bus 관리 콘솔을 사용하여 구성을 변경하기 위한 시작점입니다. Oracle Service Bus 관리 콘솔을 사용하여 구성을 변경하려면 변경 센터에서 세션을 시작해야 합니다.

  1. 편집 또는 생성을 눌러 세션을 시작합니다. 세션 이름이 변경 센터 아래 표시됩니다.
  2. Oracle Service Bus 관리 콘솔의 해당 페이지에서 적절하게 변경합니다.
  3. 변경한 각 페이지에서 저장을 누릅니다. 현재 세션에서의 변경 내용이 모두 저장됩니다.

    4. 세션 중에 언제든지 변경 센터에서 삭제를 눌러 현재 세션에서 지금까지 변경된 내용을 폐기할 수 있습니다.

  4. 변경을 완료하면 변경 센터에서 활성화를 누릅니다. 세션 활성화에서 계속합니다.
