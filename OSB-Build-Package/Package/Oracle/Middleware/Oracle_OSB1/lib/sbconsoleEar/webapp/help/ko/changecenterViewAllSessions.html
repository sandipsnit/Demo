모든 세션 보기

이 페이지에서는 Oracle Service Bus 관리 콘솔 내의 모든 기존 세션을 볼 수 있습니다. 현재 세션의 사용 여부와 관계없이 이러한 세션을 볼 수 있습니다.

관리자 롤을 사용 중인 경우에만 모든 세션을 볼 수 있습니다. 자세한 내용은 http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539에서 Oracle Fusion Middleware Developer's Guide for Oracle Service Bus의 "Configuring Administrative Security"를 참조하십시오.

  1. 변경 센터에서 모든 세션 보기를 누릅니다. 각 기존 세션에 대해 모든 세션 보기 페이지에 표 2-5에 나열된 정보가 표시됩니다.

    2. 표 2-5 세션 세부 정보
    속성
    설명
    세션 이름
    세션의 이름입니다. 이름은 세션에 대한 링크입니다.
    사용자
    세션을 생성한 사용자의 이름입니다.
    생성 시간
    세션이 생성된 날짜와 시간입니다.
    마지막 수정 시간
    세션 중에 마지막으로 변경된 날짜와 시간입니다.
    수정된 객체 수
    세션 중에 변경된 객체의 수입니다.

  2. 다른 세션으로 전환하려면 해당 세션의 세션 이름 열에서 링크를 누릅니다.
주: 동일한 사용자가 다중 브라우저에서 로그인할 수는 없습니다. Oracle Service Bus 관리 콘솔에서 예기치 않은 동작이 발생합니다.
