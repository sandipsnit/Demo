작업 세부 정보 보기

이 페이지에서는 세션을 사용 중인 경우, 현재 세션에서의 특정 변경 세부 정보를 볼 수 있으며 세션 외부에 존재하는 경우에는 세션에서 활성화된 특정 변경 세부 정보를 볼 수 있습니다.

  1. 변경 센터에서 변경 내용 보기를 선택합니다.
  2. 구성 변경 내용 보기 페이지의 작업 열에서 작업 이름을 누릅니다. 작업 세부 정보 페이지에는 표 2-4에 나열된 정보가 표시됩니다.

    3. 표 2-4 작업 세부 정보
    속성
    설명
    작업
    수행한 작업의 유형입니다.
    • 생성
    • 갱신
    • 삭제
    • 이름 바꾸기
    • 이동
    리소스 유형
    작업의 일부인 Oracle Service Bus 리소스 유형입니다.
    리소스
    리소스의 이름과 경로입니다. 경로는 리소스가 있는 폴더 이름과 프로젝트의 이름입니다.

  3. 뒤로를 누르면 구성 변경 내용 보기 페이지로 돌아갑니다.
