구성 변경 내용 보기

이 페이지는 세션을 사용 중인지(즉, 변경 센터에서 편집을 누른 경우) 또는 세션 외부에 있는지에 따라 다른 정보를 표시합니다.

  • 세션을 사용 중이면 구성 변경 내용 보기 페이지에는 현재 세션에서의 구성 변경 내용 목록이 표시됩니다. 세션에서 구성 변경 내용 보기를 참조하십시오.
  • 세션 외부에 존재하는 경우에는 구성 변경 내용 보기 페이지에서 이전 세션 활성화로 인한 구성 변경 내용 목록이 표시됩니다. 세션 활성화로 인한 구성 변경 내용 보기를 참조하십시오.
세션에서 구성 변경 내용 보기
  1. 아직 수행하지 않은 경우 생성을 눌러 현재 구성으로 변경할 새 세션을 생성합니다. 변경 센터 사용을 참조하십시오.
  2. 구성을 한 개 이상 변경합니다.
  3. 변경 센터에서 변경 내용 보기를 누릅니다. 현재 세션 동안 구성에 적용된 각 변경 내용에 대해 구성 변경 내용 보기 페이지에 표 2-2에 나열된 정보가 표시됩니다.

    4. 표 2-2 세션에서의 구성 변경 내용
    속성
    설명
    작업
    구현된 작업에 대한 설명입니다. 작업은 작업 세부 정보 페이지에 대한 링크입니다. 작업 세부 정보 보기를 참조하십시오.
    실행 시간
    작업이 실행된 날짜와 시간입니다.
    사용자
    변경을 구현한 사용자의 이름입니다.
    작업 상태
    작업의 상태입니다.
    완료 - 작업이 완료되었습니다.
    실행 취소 - 작업이 실행 취소되었습니다.
    실행 취소 완료됨 - 실행 취소가 완료되었습니다.
    실행 취소인
    작업을 실행 취소한 사용자의 이름입니다. 작업이 실행 취소되지 않았으면 이 필드에 없음이 표시됩니다.
    옵션
    실행 취소 아이콘을 누르면 작업 실행이 취소됩니다.
    세션에서 작업 중인 경우 순서에 관계없이 작업 실행을 취소할 수 있습니다.
    작업을 실행 취소하면 작업의 객체는 해당 작업이 수행되기 전의 상태로 돌아갑니다. 그러나 실행 취소한 작업 후에 동일한 객체에 대해 수행한 모든 작업도 실행 취소됩니다. 작업 실행 취소를 참조하십시오.

세션 활성화로 인한 구성 변경 내용 보기
  1. 세션이 사용 중이 아닌지 확인합니다. 기존 세션을 참조하십시오.
  2. 변경 센터에서 변경 내용 보기를 누릅니다. 이전에 활성화된 각 세션에 대해 구성 변경 내용 보기 페이지에 표 2-3에 나열된 정보가 표시됩니다.

    3. 표 2-3 활성화 구성 변경 내용
    속성
    설명
    작업
    활성화된 세션에 대한 설명입니다. 작업은 작업 세부 정보 페이지에 대한 링크로, 세션에서 수행된 작업을 표시합니다. 작업 세부 정보 보기를 참조하십시오.
    설명
    세션 활성화 페이지를 사용하여 세션이 활성화될 때 설명을 입력하는 경우 활성화된 세션의 추가 설명입니다.
    실행 시간
    세션이 활성화된 날짜와 시간입니다.
    사용자
    세션을 활성화한 사용자의 이름입니다.
    작업 상태
    세션의 상태입니다.
    활성화됨 - 세션이 활성화되었습니다.
    부분적으로 활성화 - 클러스터에 있는 하나 이상의 서버에 액세스할 수 없는 경우 세션 활성화 동안 표시되며, 해당 서버의 세션 활성화를 방지합니다. 서버에 액세스할 수 있게 된 경우, 모든 활성화되지 않은 변경 사항이 서버에서 활성화됩니다.
    실행 취소됨 - 이전에 활성화되었던 세션이 실행 취소되었으며 세션에서 수행한 모든 작업이 삭제되었습니다.
    실행 취소 활성화됨 - 실행 취소가 활성화되었습니다.
    진행 중 - 세션을 활성화하는 데 시간이 걸릴 수 있으므로 세션 활성화가 진행 중일 때 표시됩니다.
    실패 - 세션 활성화에 실패했을 때 표시됩니다. Oracle Service Bus는 세션 활성화 실패를 추적하지만 세션 내 개별 갱신으로 인한 실패는 추적하지 않습니다.
    실행 취소인
    이 작업을 실행 취소한 사용자의 이름입니다. 세션 활성화가 실행 취소되지 않았으면 이 필드에 없음이 표시됩니다.
    옵션
    실행 취소 아이콘을 누르면 해당 세션에서 수행한 작업과 세션 활성화가 취소됩니다. 순서에 관계없이 세션 활성화를 실행 취소할 수 있습니다.
    세션에서 활성화된 변경 내용을 실행 취소하기 위해 필요한 작업으로 미리 채워진 새로운 세션을 생성하려면 세션에서 실행 취소 아이콘을 누릅니다. 자세한 내용은 세션에서 실행 취소를 참조하십시오.
