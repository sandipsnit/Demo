서비스의 전역 모니터링을 사용 안함으로 설정

도메인 레벨에서 모든 서비스에 대한 모니터링을 해제하려면 다음을 수행합니다.

주: 모든 서비스에 대한 모니터링을 사용 안함으로 설정하려면 해당 서비스에 대해 수집된 모든 통계도 함께 삭제됩니다. 삭제된 이 통계는 세션 실행 취소 기능을 사용하여 복구할 수 없습니다. 실행 취소를 사용으로 설정하면 해당 서비스에 대한 모니터링을 다시 사용으로 설정할 수 있지만 삭제된 통계는 복원할 수 없습니다.
  1. 아직 이 작업을 수행하지 않은 경우 생성을 눌러 새 세션을 생성하거나 편집을 눌러 기존 세션을 입력합니다. 변경 센터 사용을 참조하십시오.
  2. 작업 > 전역 설정을 선택합니다.
  3. 모니터링 사용 체크 박스를 선택 취소하면 현재 구성의 모든 서비스에 대한 모니터링 통계 수집이 정지됩니다.
    4. 주: 이 옵션은 특정 비즈니스 서비스와 프록시 서비스에 대해 선택할 수 있는 서비스 모니터링 사용 옵션을 무효화합니다. 비즈니스 서비스의 작업 설정 구성 및 프록시 서비스의 작업 설정 구성을 참조하십시오.
    주: 또한 이 옵션은 SLA 경보 사용 옵션도 무효화합니다. 전역 레벨에서 모니터링을 사용 안함으로 설정하면 해당 SLA 경보 사용 체크 박스를 선택하더라도 SLA 경보가 런타임 시에도 사용 안함으로 설정됩니다. SLA 경보를 전체적으로 사용으로 설정을 참조하십시오.
  4. 갱신을 누릅니다. 현재 세션에서 모니터링 구성이 저장됩니다.
  5. 또한 세션 도중 언제든지 재설정을 눌러 변경 사항을 무시하고 현재 저장된 설정으로 페이지를 새로 고칠 수 있습니다.
  6. 세션을 종료하고 구성을 런타임에 배치하려면 변경 센터에서 활성화를 누릅니다.
