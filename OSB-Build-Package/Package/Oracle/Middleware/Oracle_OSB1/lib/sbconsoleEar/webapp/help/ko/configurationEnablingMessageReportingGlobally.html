메시지 보고를 전체적으로 사용으로 설정

이 옵션은 메시지 컨텍스트에서만 프록시 서비스 파이프 라인 보고서 작업을 제어합니다. 보고 프레임워크를 대상으로 하는 SLA 경보 또는 파이프 라인 경보에는 영향을 주지 않습니다.

도메인 레벨에서 모든 프록시 서비스에 대한 메시지 보고를 설정하려면 다음을 수행합니다.

  1. 아직 이 작업을 수행하지 않은 경우 생성을 눌러 새 세션을 생성하거나 편집을 눌러 기존 세션을 입력합니다. 변경 센터 사용을 참조하십시오.
  2. 작업 > 전역 설정을 선택합니다.
  3. 모든 프록시 서비스에 대한 파이프 라인 보고서 작업을 시작하려면 보고 사용 체크 박스를 선택합니다.
  4. 이 페이지의 모든 설정을 저장하려면 갱신을 누릅니다.
  5. 또한 세션 도중 언제든지 재설정을 눌러 변경 사항을 무시하고 현재 저장된 설정으로 페이지를 새로 고칠 수 있습니다.
  6. 세션을 종료하고 구성을 런타임에 배치하려면 변경 센터에서 활성화를 누릅니다.
