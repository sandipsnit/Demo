SLA 경보를 전체적으로 사용으로 설정

모니터링에서 SLA 경보를 개별적으로 구성할 수 있지만 런타임 시 두 설정은 상호 작용합니다. 전역 모니터링을 사용으로 설정하면 SLA 경보를 사용 또는 사용 안함으로 설정할 수 있습니다. 그러나 전역 모니터링을 사용 안함으로 설정하면 SLA 경보 규칙 조건이 평가할 모니터링 통계에 따라 다르기 때문에 SLA 경보가 효율적으로 사용 안함으로 설정됩니다.

도메인 레벨에서 모든 서비스에 대한 SLA 경보를 설정하려면 다음을 수행합니다.

  1. 아직 이 작업을 수행하지 않은 경우 생성을 눌러 새 세션을 생성하거나 편집을 눌러 기존 세션을 입력합니다. 변경 센터 사용을 참조하십시오.
  2. 작업 > 전역 설정을 선택합니다.
  3. SAL 경보 사용 체크 박스를 선택하면 현재 구성의 모든 서비스에 대한 경보 규칙 평가가 시작됩니다.
  4. 이 페이지의 모든 설정을 저장하려면 갱신을 누릅니다.
  5. 또한 세션 도중 언제든지 재설정을 눌러 변경 사항을 무시하고 현재 저장된 설정으로 페이지를 새로 고칠 수 있습니다.
  6. 세션을 종료하고 구성을 런타임에 배치하려면 변경 센터에서 활성화를 누릅니다.
