경보 대상 찾기

  1. 작업 > 스마트 검색을 선택합니다.
  2. 유형 드롭다운 목록에서 경보 대상을 선택한 다음 검색을 누릅니다. 그러면 경보 대상 요약 페이지에 표 26-5와 같은 정보가 표시됩니다.
  3. 목록의 항목 수를 제한하거나 특정 경보 대상을 찾으려면 다음을 수행합니다.
    • 열기 아이콘을 눌러 검색 필터를 추가로 표시합니다. 다중 검색 조건을 지정하는 경우에는 모든 조건과 일치하는 결과만 반환됩니다.
    • 경보 대상 이름 및 경로로 필터링합니다. 이름 및 경로 필드에 검색 대상의 이름과 경로를 입력한 다음 검색을 누릅니다. 경로는 경보 대상이 있는 폴더 이름과 프로젝트 이름입니다. * 및 ?를 와일드 카드 문자로 사용하여 보다 일반적인 검색을 수행할 수 있습니다.
    • 경보 대상 브리지 대상을 조건으로 필터링합니다. 대상 다중 선택 상자에서 SNMP 트랩, 보고, 경보 로깅, 전자 메일 또는 JMS 중에서 경보 대상 브리지 대상을 하나 이상 선택한 다음 검색을 누릅니다. 선택된 대상 중 하나 이상을 가진 경보 대상만 표시됩니다. 경보 대상 필터 조건을 무시하려면 [경보 대상 브리지 대상 없음]을 선택합니다.
    • 검색 패턴으로 필터링합니다. 검색 패턴(모든 문자열) 텍스트 상자에서 문자열을 지정한 다음 검색을 누릅니다. 시스템은 문자열을 사용하여 전자 메일 및 JMS 대상의 특정 세부 필드뿐만 아니라 경보 대상의 모든 설명 필드를 검색합니다. 문자열이 경보 대상 필드에 나타나는 경우 검색 조건과 일치하는 경보 대상이 표시됩니다.

      • 표 26-5 경보 대상 요약 정보
      속성
      설명
      이름
      경보 대상 리소스의 이름입니다. 이 필드는 경보 대상 보기 - 구성 페이지에 대한 링크입니다. 경보 대상 찾기를 참조하십시오.
      경로
      경보 대상과 연관된 프로젝트입니다. 경보 대상이 프로젝트 폴더에 있으면 이 폴더도 나열됩니다. 경로는 다음 형식으로 표시됩니다.
      프로젝트 이름/루트 폴더/ . . ./상위 폴더
      프로젝트 탐색기의 해당 경로로 링크된 경로입니다.
      옵션
      삭제 아이콘을 눌러 특정 경보 대상을 삭제할 수 있습니다. 다른 리소스에서 이 리소스를 참조하는 경우 삭제 경고 아이콘이 표시됩니다. 경고 확인을 통해 리소스를 삭제할 수 있습니다. 이 경우 삭제된 리소스에 대한 해결되지 않은 참조로 인해 충돌이 발생할 수 있습니다. 경보 대상 삭제를 참조하십시오.

이 페이지에서는 다음을 수행할 수 있습니다.

  • 검색 필터를 제거하고 모든 경보 대상을 표시하려면 모두 보기를 누릅니다.
  • 다른 리소스에서 참조되지 않은 경우 특정 경보 대상을 삭제하려면 해당 경보 대상 옆에 있는 체크 박스를 선택하고 삭제를 누릅니다.
  • 작업 설정 정보의 보기를 사용자 정의하려면 테이블 사용자 정의기 아이콘을 누릅니다. 테이블 뷰 사용자 정의를 참조하십시오.
