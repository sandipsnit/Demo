비즈니스 서비스 찾기

  1. 작업 > 스마트 검색을 선택합니다.
  2. 유형 드롭다운 목록에서 비즈니스 서비스를 선택한 다음 검색을 누릅니다. 그러면 비즈니스 서비스 요약 페이지에 표 26-3과 같은 정보가 표시됩니다.
  3. 목록의 항목 수를 제한하거나 특정 서비스를 찾으려면 다음을 수행합니다.
    • 열기 아이콘을 눌러 검색 필터를 추가로 표시합니다. 다중 검색 조건을 지정하는 경우에는 모든 조건과 일치하는 결과만 반환됩니다.
    • 비즈니스 서비스 이름과 경로로 필터링합니다. 이름 및 경로 필드에 검색 대상의 이름과 경로를 입력한 다음 검색을 누릅니다. 경로는 비즈니스 서비스가 있는 폴더 이름과 프로젝트 이름입니다. * 및 ?를 와일드 카드 문자로 사용하여 보다 일반적인 검색을 수행할 수 있습니다.
    • 서비스 상태로 필터링합니다. 상태 드롭다운 목록에서 사용 또는 사용 안함으로 설정됨과 같은 비즈니스 서비스 상태 중 하나를 선택한 다음 검색을 누릅니다. 서비스 상태 필터 조건을 무시하려면 [모두]를 선택합니다.
    • 서비스 모니터링으로 필터링합니다. 모니터링 드롭다운 목록에서 사용 또는 사용 안함으로 설정됨과 같은 비즈니스 서비스 모니터링 상태 중 하나를 선택한 다음 검색을 누릅니다. 서비스 모니터링 필터 조건을 무시하려면 [모두]를 선택합니다.
    • 비즈니스 서비스의 SLA 경보 설정으로 필터링합니다. SLA 경보 드롭다운 목록에서 사용 또는 사용 안함으로 설정됨과 같은 비즈니스 서비스의 SLA 경보 설정 중 하나를 선택한 다음 검색을 누릅니다. 서비스 SLA 경보 필터 조건을 무시하려면 [모두]를 선택합니다.
    • 메시지 추적으로 필터링합니다. 메시지 추적이 사용 또는 사용 안함으로 설정된 비즈니스 서비스에서 필터링을 수행할지 여부를 선택한 다음 검색을 누릅니다. 메시지 추적 필터 조건을 무시하려면 [모두]를 선택합니다.
    • 캐싱으로 필터링합니다. 결과 캐싱이 사용 또는 사용 안함으로 설정된 비즈니스 서비스에서 필터링을 수행할지 여부를 선택한 다음 검색을 누릅니다. 결과 캐싱 필터 조건을 무시하려면 [모두]를 선택합니다.

      • 표 26-3 비즈니스 서비스 요약 정보
      속성
      설명
      이름
      비즈니스 서비스에 할당된 이름입니다. 이름은 작업 설정 페이지에 대한 링크입니다. 비즈니스 서비스의 작업 설정 구성을 참조하십시오.
      경로
      비즈니스 서비스와 연관된 프로젝트입니다. 서비스가 프로젝트 폴더에 있으면 이 폴더도 나열됩니다. 경로는 다음 형식으로 표시됩니다.
      프로젝트 이름/루트 폴더/ . . ./상위 폴더
      프로젝트 탐색기의 해당 경로로 링크된 경로입니다.
      유형
      상위 서비스 유형: 비즈니스 서비스입니다.
      상태
      비즈니스 서비스 상태(사용 또는 사용 안함으로 설정됨)입니다.
      모니터링
      비즈니스 서비스 모니터링 상태(사용 또는 사용 안함으로 설정됨)입니다.
      SLA 경보
      SLA 경보 상태(사용 또는 사용 안함으로 설정됨)이고 다음 레벨 이상에서 사용합니다. 일반(N), 경고(W), 사소(Mn), 중요(Mj), 심각(C) 또는 치명적(F)
      메시지 추적
      비즈니스 서비스에 대해 메시지 추적이 사용 또는 사용 안함으로 설정되었는지 여부입니다.
      캐싱
      비즈니스 서비스에 대해 결과 캐싱이 사용 또는 사용 안함으로 설정되었는지 여부입니다.

이 페이지에서는 다음을 수행할 수 있습니다.

  • 검색 필터를 제거하고 모든 비즈니스 서비스를 표시하려면 모두 보기를 누릅니다.
  • 변경 사항을 무시하고 동일한 검색 조건을 사용하여 현재 저장된 설정으로 페이지를 새로 고친 후 재설정을 누릅니다.
  • 세션 내에서 서비스 레벨 설정을 사용 또는 사용 안함으로 설정하려면 해당 서비스 레벨 옆에 있는 체크 박스를 선택하거나 선택을 취소하고 갱신을 누릅니다.
    • 주: 서비스 레벨 설정의 런타임 효과는 해당 전역 설정에 따라 다릅니다. 서비스를 런타임 시 완벽하게 사용으로 설정하려면 전역 및 서비스 레벨 설정 모두를 사용으로 설정해야 합니다. 또한 SLA 경보는 전역 및 서비스 레벨 모니터링 상태에 따라 다르며 SLA 경보를 런타임 시 사용으로 설정하려면 두 가지 상태 모두 사용으로 설정해야 합니다. 전역 설정 사용을 참조하십시오.
    주: 또한 서비스 상태도 사용으로 설정해야 합니다.
  • 작업 설정 정보의 보기를 사용자 정의하려면 테이블 사용자 정의기 아이콘을 누릅니다. 테이블 뷰 사용자 정의를 참조하십시오.
