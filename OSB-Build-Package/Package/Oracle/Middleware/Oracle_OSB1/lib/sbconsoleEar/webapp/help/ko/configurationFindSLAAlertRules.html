SLA 경보 규칙 찾기

  1. 작업 > 스마트 검색을 선택합니다.
  2. 유형 드롭다운 목록에서 SLA 경보를 선택한 다음 검색을 누릅니다. 그러면 SLA 경보 규칙 요약 페이지에 표 26-6과 같은 정보가 표시됩니다.
  3. 목록의 항목 수를 제한하거나 특정 SLA 경보 규칙을 찾으려면 다음을 수행합니다.
    • 열기 아이콘을 눌러 검색 필터를 추가로 표시합니다. 다중 검색 조건을 지정하는 경우에는 모든 조건과 일치하는 결과만 반환됩니다.
    • 경보 규칙 이름으로 필터링합니다. 이름 필드에 검색 대상 이름을 입력하거나 와일드 카드 문자(* 및 ?를 와일드 카드 문자로 사용하여 보다 일반적인 검색 수행)를 입력한 후 검색을 누릅니다.

      • 이 검색 방법은 경보 규칙 이름이 모든 프로젝트와 경로에서 고유할 경우 사용하는 것이 좋습니다.

    • 상위 서비스 경로로 필터링합니다. 경로 필드에서 상위 서비스 경로를 입력합니다. 이 경로는 상위 서비스가 있는 프로젝트 이름 및 폴더 이름입니다. * 및 ?를 와일드 카드 문자로 사용하여 보다 일반적인 검색을 수행할 수 있습니다. 해당 경로에 상위 서비스가 있는 모든 경보 규칙이 표시됩니다.

      • 경로 필드의 형식은 다음과 같습니다.
      프로젝트 이름/루트 폴더/ . . ./상위 폴더

      서비스가 프로젝트 바로 아래에 존재하는 경우 형식은 다음과 같습니다. 프로젝트 이름

    • 상위 서비스로 필터링합니다. 상위 서비스 필드에서 찾아보기를 눌러 서비스 선택 페이지를 표시합니다. 목록에서 상위 서비스를 선택하고 제출을 누릅니다. 상위 서비스 필터 조건을 제거하려면 지우기를 누르거나 검색을 누릅니다.
      • 주: 상위 서비스 필드의 선택 항목으로 서비스 유형 필드의 선택 항목이 제한됩니다.
    • 상위 서비스 유형으로 필터링합니다. 서비스 유형 드롭다운 목록에서 프록시 서비스 또는 비즈니스 서비스를 선택한 다음 검색을 누릅니다. 상위 서비스 유형 조건별 필터링을 무시하려면 [모든 서비스]를 선택합니다.
      • 주: 서비스 유형 필드의 선택 항목으로 상위 서비스 필드의 선택 항목이 제한됩니다.
    • 규칙 상태로 필터링합니다. 규칙 상태 드롭다운 목록에서 사용 또는 사용 안함으로 설정됨과 같은 경보 규칙 상태 중 하나를 선택한 다음 검색을 누릅니다. 규칙 상태 필터 조건을 무시하려면 [모두]를 선택합니다.
    • 심각도로 필터링합니다. 심각도 드롭다운 목록에서 SLA 경보 규칙을 특정 레벨로 제한하도록 선택한 다음 검색을 누릅니다.

      • 검색을 특정 심각도 레벨 또는 그 이상(가장 포괄적인 레벨에서 가장 제한적인 레벨까지 나열됨: 일반, 경고, 사소, 심각 또는 치명적)으로 제한하려면 이상 체크 박스를 선택합니다.

      표 26-6 SLA 경보 규칙 요약 정보
      속성
      설명
      이름
      이 경보 규칙에 할당된 이름입니다. 이름은 경보 규칙 구성 페이지에 대한 링크입니다. 경보 규칙 구성 보기를 참조하십시오.
      SLA 상태
      경보 규칙의 상태(사용 또는 사용 안함으로 설정됨)입니다.
      설명
      주: 이 필드는 기본적으로 숨겨져 있습니다.
      경보 규칙에 대한 설명입니다.
      서비스 이름
      상위 서비스의 이름입니다. 이름은 SLA 경보 규칙 페이지에 대한 링크입니다. 경보 규칙 보기를 참조하십시오.
      경로
      경보 규칙의 상위 서비스와 연관된 프로젝트입니다. 경보 규칙의 상위 서비스가 프로젝트 폴더에 있으면 이 폴더도 나열됩니다. 경로는 다음 형식으로 표시됩니다.
      프로젝트 이름/루트 폴더/ . . ./상위 폴더
      프로젝트 탐색기의 해당 경로로 링크된 경로입니다.
      심각도
      이 규칙에 의해 트리거되는 경보 심각도(일반, 경고, 사소, 중요, 심각 또는 치명적)입니다.
      집계 수집 간격
      수집 간격의 길이는 시간 및 분 단위입니다.
      만료 날짜
      이 경보 규칙이 더 이상 적용되지 않는 날짜입니다.
      처리 중지
      주: 이 필드는 기본적으로 숨겨져 있습니다.
      예 또는 아니오를 표시합니다.
      빈도
      이 경보의 빈도입니다.
      • 매번
      • 한 번 통지

이 페이지에서는 다음을 수행할 수 있습니다.

  • 검색 필터를 제거하고 모든 SLA 경보 규칙을 표시하려면 모두 보기를 누릅니다.
  • 변경 사항을 무시하고 동일한 검색 조건을 사용하여 현재 저장된 설정으로 페이지를 새로 고친 후 재설정을 누릅니다.
  • 세션 내에서 SLA 경보 규칙을 사용 또는 사용 안함으로 설정하려면 해당 서비스 레벨 옆에 있는 체크 박스를 선택하거나 선택을 취소하고 갱신을 누릅니다.
  • 작업 설정 정보의 보기를 사용자 정의하려면 테이블 사용자 정의기 아이콘을 누릅니다. 테이블 뷰 사용자 정의를 참조하십시오.
