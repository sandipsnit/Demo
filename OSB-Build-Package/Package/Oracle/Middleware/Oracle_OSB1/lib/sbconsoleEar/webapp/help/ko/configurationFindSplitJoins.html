분할-조인 찾기

  1. 작업 > 스마트 검색을 선택합니다.
  2. 유형 드롭다운 목록에서 분할-조인을 선택한 다음 검색을 누릅니다. 그러면 분할-조인 요약 페이지에 표 26-4와 같은 정보가 표시됩니다.
  3. 목록의 항목 수를 제한하거나 특정 서비스를 찾으려면 다음을 수행합니다.
    • 열기 아이콘을 눌러 검색 필터를 추가로 표시합니다. 다중 검색 조건을 지정하는 경우에는 모든 조건과 일치하는 결과만 반환됩니다.
    • 분할-조인 이름과 경로로 필터링합니다. 이름 및 경로 필드에 검색 대상의 이름과 경로를 입력한 다음 검색을 누릅니다. 경로는 분할-조인이 있는 폴더 이름과 프로젝트 이름입니다. * 및 ?를 와일드 카드 문자로 사용하여 보다 일반적인 검색을 수행할 수 있습니다.
    • 로깅이 있는 분할-조인으로 필터링합니다. 로그 드롭다운 목록에서 사용 또는 사용 안함으로 설정됨과 같은 분할-조인의 로깅 상태 중 하나를 선택한 다음 검색을 누릅니다. 로깅 필터 조건을 무시하려면 [모두]를 선택합니다.
    • 실행 추적으로 필터링합니다. 로그 드롭다운 목록에서 사용 또는 사용 안함으로 설정됨과 같은 분할-조인의 실행 추적 상태 중 하나를 선택한 다음 검색을 누릅니다. 서비스 로깅 필터 조건을 무시하려면 [모두]를 선택합니다.

      • 표 26-4 분할-조인 요약 정보
      속성
      설명
      이름
      분할-조인에 할당된 이름입니다. 이름은 작업 설정 페이지에 대한 링크입니다. 분할-조인 편집을 참조하십시오.
      경로
      분할-조인과 연관된 프로젝트입니다. 분할-조인이 프로젝트 폴더에 있으면 이 폴더도 나열됩니다. 경로는 다음 형식으로 표시됩니다.
      프로젝트 이름/루트 폴더/ . . ./상위 폴더
      프로젝트 탐색기의 해당 경로로 링크된 경로입니다.
      유형
      상위 서비스 유형: 분할-조인입니다.
      로그
      로깅 상태(사용 또는 사용 안함으로 설정됨)이고 다음 레벨 이상에서 사용합니다. 디버그(D), 정보(I), 경고(W) 또는 오류(E)
      추적 실행
      분할-조인의 런타임 추적 상태(사용 또는 사용 안함으로 설정됨)입니다.

이 페이지에서는 다음을 수행할 수 있습니다.

  • 검색 필터를 제거하고 모든 분할-조인을 표시하려면 모두 보기를 누릅니다.
  • 변경 사항을 무시하고 동일한 검색 조건을 사용하여 현재 저장된 설정으로 페이지를 새로 고친 후 재설정을 누릅니다.
  • 세션 내에서 서비스 레벨 설정을 사용 또는 사용 안함으로 설정하려면 해당 서비스 레벨 옆에 있는 체크 박스를 선택하거나 선택을 취소하고 갱신을 누릅니다.
    • 주: 분할-조인 레벨 설정에 대한 런타임 효과는 해당 전역 설정에 따라 달라집니다. 서비스에 대한 전역 및 서비스 레벨 설정이 모두 런타임에 완전히 사용으로 설정되도록 해야 합니다.
  • 작업 설정 정보의 보기를 사용자 정의하려면 테이블 사용자 정의기 아이콘을 누릅니다. 테이블 뷰 사용자 정의를 참조하십시오.
