작업 설정 찾기 및 갱신

스마트 검색 페이지를 통해 프록시 서비스, 비즈니스 서비스, 경보 대상 및 SLA 경보 규칙을 쉽게 찾을 수 있습니다. 또한 스마트 검색 페이지에서는 서비스별 작업 설정을 지정할 수는 있지만 집계 간격을 설정하거나 경보 또는 로깅 심각도 레벨을 지정할 수는 없습니다. 자세한 내용은 프록시 서비스의 작업 설정 구성 및 비즈니스 서비스의 작업 설정 구성을 참조하십시오.

서비스 레벨 설정의 런타임 효과는 해당 전역 설정에 따라 다릅니다. 서비스를 런타임 시 완벽하게 사용으로 설정하려면 전역 및 서비스 레벨 설정 모두를 사용으로 설정해야 합니다. 또한 서비스 상태도 사용으로 설정해야 합니다. 전역 설정 사용을 참조하십시오.

세션 내에서만 작업 설정을 사용 또는 사용 안함으로 설정할 수 있습니다.
