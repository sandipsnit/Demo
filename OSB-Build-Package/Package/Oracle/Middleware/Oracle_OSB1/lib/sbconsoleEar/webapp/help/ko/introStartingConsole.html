Oracle Service Bus 관리 콘솔 시작

  1. 다음 방법 중 하나를 사용하여 Oracle Service Bus를 시작합니다.
    • Windows 시작 메뉴에서 다음을 선택합니다.

      • 시작 > 모든 프로그램 > Oracle WebLogic > 사용자 프로젝트 > domain_name > Oracle Service Bus 도메인용 서버 시작

      여기서 domain_name은 Oracle Service Bus 도메인을 생성할 때 할당한 이름을 나타냅니다. User Projects는 도메인이 기본적으로 생성된 기본 위치입니다.

    • Oracle Service Bus 도메인 startWebLogic.cmd 또는 startWebLogic.sh의 루트에서 startWebLogic 명령을 호출합니다.

      • 명령 창이 표시되고 WebLogic Server에 대한 상태 정보를 보여줍니다.

      서버를 프로덕션 모드로 시작한 경우 적합한 사용자 이름과 비밀번호를 입력해야 서버를 시작할 수 있습니다.

  2. 서버가 시작되면 브라우저에 다음 URL을 입력합니다.

    3. http://hostname:port/sbconsole

    여기서 host는 WebLogic Server를 실행 중인 시스템의 이름을 나타내고 port는 포트 번호를 나타냅니다.

    예를 들어, 기본 포트 구성을 사용하여 로컬 시스템에서 WebLogic Server를 실행 중이면 브라우저에 다음 URL을 입력합니다.

    http://localhost:7001/sbconsole

    Oracle Service Bus 관리 콘솔 로그인 페이지가 표시됩니다.

  3. 설치 중에 지정한 사용자 이름과 비밀번호를 입력합니다.
    4. 주: Oracle Service Bus Examples 도메인의 사용자 이름과 비밀번호는 weblogic/welcome1입니다.
  4. 로그인을 누릅니다.

로그아웃하려면 Oracle Service Bus 관리 콘솔의 오른쪽 패널 위쪽에 있는 배너에서 로그아웃을 누릅니다.
