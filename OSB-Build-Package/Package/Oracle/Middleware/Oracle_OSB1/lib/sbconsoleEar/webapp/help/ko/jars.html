JAR

JAR(Java ARchive)은 Java 클래스 집합을 압축한 파일입니다. 이 파일은 프로그램을 구성할 수 있는 컴파일된 Java 클래스와 관련 메타 데이터를 저장하는 데 사용됩니다. JAR은 Java 코드 요소에 대해 호출 가능 프로그램 라이브러리처럼 작동합니다. 따라서 각 개별 요소를 바인딩하지 않아도 하나의 컴파일 링크를 통해 다중 요소에 액세스할 수 있습니다. JAR에 대한 자세한 내용은 http://java.sun.com/docs/books/tutorial/deployment/jar/을 참조하십시오.

Oracle Service Bus의 JAR 리소스는 다음에 사용됩니다.

  • Java 콜아웃 작업
  • EJB-기반 비즈니스 서비스
  • Tuxedo 기반 프록시 및 비즈니스 서비스
