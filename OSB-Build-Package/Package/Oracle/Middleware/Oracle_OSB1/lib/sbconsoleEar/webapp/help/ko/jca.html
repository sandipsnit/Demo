JCA 바인딩

Oracle SOA Suite에서는 JCA(J2EE 커넥터 아키텍처) 어댑터를 사용하여 백엔드 EIS(엔터프라이즈 정보 시스템)를 통합할 수 있습니다. Oracle JDeveloper 및 기타 지원되는 JCA 개발 환경을 사용하여 어댑터를 구성하고 백엔드 통합에 사용되는 JCA 어댑터 리소스(.jca 파일 및 추상적인 WSDL)를 생성하면 됩니다.

OSB(Oracle Service Bus)를 통해 JCA 바인딩 리소스 형식의 JCA 리소스를 사용하여 JCA 호환 EIS 구성 요소와 통신할 수 있습니다. JCA 바인딩을 사용할 경우 Oracle Service Bus JCA 전송을 사용하여 JCA 어댑터를 통해 EIS 응용 프로그램과 통신하는 프록시 및 비즈니스 서비스를 생성할 수 있습니다.

이 장에서는 Oracle Service Bus에서 JCA 바인딩을 추가, 편집, 검색 및 삭제하는 방법에 대해 설명하며 다음 항목을 다룹니다.

  • JCA 바인딩 찾기
  • JCA 바인딩 추가
  • JCA 바인딩 편집
  • JCA 바인딩 삭제

JCA 전송에 대한 자세한 내용은 http://www.oracle.com/pls/as111170/lookup?id=OSBDV910에서 Oracle Fusion Middleware Developer's Guide for Oracle Service Bus의 "JCA Transport"를 참조하십시오.

Oracle SOA Suite JCA 어댑터에 대한 자세한 내용은 http://www.oracle.com/pls/as111170/lookup?id=TKADP에서 Oracle Fusion Middleware User's Guide for Technology Adapters를 참조하십시오.
