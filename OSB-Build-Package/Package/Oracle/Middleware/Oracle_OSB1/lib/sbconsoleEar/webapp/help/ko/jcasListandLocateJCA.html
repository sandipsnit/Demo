JCA 바인딩 찾기

JCA 바인딩 요약 페이지에는 Oracle Service Bus 도메인에 추가한 JCA 바인딩이 나열됩니다.

  1. 다음 작업을 수행합니다.
    • 프로젝트 탐색기를 선택하여 프로젝트 뷰 페이지 또는 프로젝트/폴더 뷰 페이지를 표시합니다. 그런 다음 프로젝트와 폴더를 탐색하여 JCA 바인딩을 찾습니다.
    • 리소스 브라우저 > JCA 바인딩을 선택합니다. 그러면 JCA 바인딩 요약 페이지에 표 12-1과 같은 정보가 표시됩니다. 속성에 대한 자세한 설명은 JCA 바인딩 찾기를 참조하십시오.
  2. 특정 JCA 바인딩을 찾으려면 다음과 같이 합니다.
    • 목록의 항목 수를 제한하려는 경우 검색을 누르기 전에 이름 및 경로를 조건으로 필터링할 수 있습니다.

      • 경로는 리소스가 상주하는 프로젝트 이름과 폴더 이름입니다.

    • 모두 보기를 눌러 검색 필터를 제거하고 모든 JCA 바인딩을 표시합니다.

      • 표 12-1 JCA 바인딩 정보
      속성
      설명
      JCA 바인딩 이름
      JCA 바인딩에 지정된 고유한 이름입니다. JCA 바인딩 세부 정보 보기 페이지를 보려면 이름을 누릅니다. JCA 바인딩 편집을 참조하십시오.
      경로
      JCA 바인딩이 상주하는 프로젝트 이름과 폴더 이름입니다. 이 리소스가 포함된 프로젝트 또는 폴더를 보려면 이름을 누릅니다. 프로젝트 및 폴더를 사용하여 리소스 이름 식별을 참조하십시오.
      작업
      JCA 바인딩에서 Oracle Service Bus WSDL 및 JCA 서비스를 생성할 수 있는 생성 아이콘이 포함되어 있습니다.
      옵션
      삭제 아이콘이 포함되어 있습니다. 비즈니스 서비스 또는 프록시 서비스가 JCA 바인딩을 사용하도록 구성된 경우 삭제 경고 아이콘은 경고 확인을 통해 JCA 바인딩을 삭제할 수 있음을 나타냅니다. 이 경우 삭제된 JCA 바인딩을 참조하는 서비스의 참조가 확인되지 않아 충돌이 발생할 수 있습니다. JCA 바인딩 삭제를 참조하십시오.
