MFL

MFL(Message Format Language) 문서는 이진 데이터의 레이아웃을 설명하는 데 사용되는 특수 XML 문서로서, 형식이 있는 이진 데이터를 XML 데이터로 변환하는 규칙을 정의하기 위한 Oracle 전용 언어입니다. MFL 문서는 각 데이터 필드와 필드(그룹) 그룹화, 반복 및 집계를 설명하는 데 사용되는 요소와 속성이 포함된 mfl.dtd에 따릅니다.

메시징 서비스 유형의 비즈니스 서비스 또는 프록시 서비스를 생성하는 경우 서비스의 요청 메시지 유형이나 응답 메시지 유형으로 MFL 유형을 선택할 수 있습니다.

Oracle Format Builder를 사용하여 MFL을 생성합니다. 이진 레코드 층, 필드 레이아웃, 필드 및 그룹의 그룹화를 정의하면 해당 정보가 MFL 문서로 저장되므로 나중에 이 문서를 사용하여 런타임 변환을 수행할 수 있습니다. Format Builder에서 MFL 문서를 사용하여 콘텐츠 모델을 기술하는 해당 DTD를 생성할 수도 있습니다. Format Builder 사용 방법에 대한 자세한 내용은 http://www.oracle.com/pls/as111170/lookup?id=OSBDV763에서 Oracle Fusion Middleware Developer's Guide for Oracle Service Bus의 "Format Builder"를 참조하십시오.
