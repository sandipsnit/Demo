모니터링

비즈니스 서비스나 프록시 서비스를 생성할 때 해당 서비스에 대한 모니터링은 기본적으로 사용 안함으로 설정됩니다. 프록시 서비스 모니터링을 사용으로 설정하는 방법을 알아보려면 프록시 서비스의 작업 설정 구성을 참조하고 비즈니스 서비스의 경우 비즈니스 서비스의 작업 설정 구성을 참조하십시오.

모니터링에 대한 설정을 개별적으로 사용 또는 사용 안함으로 설정한 모든 서비스의 모니터링을 사용 또는 사용 안함으로 설정하려면 작업 > 전역 설정 페이지에서 모니터링 사용 옵션을 사용합니다. 전역 설정 사용을 참조하십시오.

런타임에 모니터링 수행에 대한 자세한 내용은 http://www.oracle.com/pls/as111170/lookup?id=OSBAG472에서 Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus의 "Monitoring Oracle Service Bus at Run Time"을 참조하십시오.
