도메인 로그 파일 항목의 사용자 보기 사용자 정의

  1. 이 테이블 사용자 정의를 누릅니다. 추가 필드가 표시됩니다.

    2. 언제든지 이 테이블 사용자 정의를 눌러 이 테이블을 닫고 원래 설정을 유지할 수 있습니다.

  2. 필터 필드에서 다음 작업을 수행합니다.
    1. 시간 필드에서 로그 항목을 표시할 시간 간격을 선택합니다.
    2. 시작 시간 필드에 MM/DD/YYYY HH:MM:SS 형식으로 시작 시간을 입력합니다. 예를 들어, 10/25/06 08:39:48을 입력합니다. 시작 시간 및 종료 시간 필드를 사용하여 로그 항목을 표시할 시간 창을 지정합니다.
    3. 종료 시간 필드에 MM/DD/YYYY HH:MM:SS 형식으로 종료 시간을 입력합니다. 예를 들어, 10/25/06 13:20:51을 입력합니다. 시작 시간 및 종료 시간 필드를 사용하여 로그 항목을 표시할 시간 창을 지정합니다.
    4. 조건 텍스트 상자에 필터링 박스에 필터링 조건(텍스트 문자열)을 지정합니다. 필터링 조건은 WLDF 질의 언어로 된 문자열로 지정됩니다. 이 질의 언어에는 부울 연산자 AND, OR 및 NOT와 관계형 연산자가 지원됩니다. 질의 구문에 대한 자세한 내용은 http://www.oracle.com/pls/as111170/lookup?id=WLDFC316에서 Oracle Fusion Middleware Configuring and Using the Diagnostics Framework for Oracle WebLogic Server의 "WLDF Query Language"를 참조하십시오.
  3. 열 필드에서 표시할 열을 선택합니다.
    1. 사용 가능 필드에서 열 이름을 선택합니다.
    2. 화살표를 눌러 이 열 이름을 선택됨 필드로 이동합니다.
    3. 선택됨 필드에 표시할 열 이름이 모두 나열될 때까지 반복합니다.

      4. 위쪽 및 아래쪽 화살표를 사용하여 필요에 따라 선택됨 필드의 열 이름 순서를 변경할 수도 있습니다.

  4. 페이지당 표시된 행 수 필드에서 한 페이지에 표시할 로그 항목 수를 선택합니다.
  5. 반환된 최대 결과 필드에서 표시할 최대 로그 항목 총 수를 선택하거나 모두 표시를 선택하여 모두 표시합니다.
  6. 다음 작업 중 하나를 수행합니다.
    • 새 설정을 저장하려면 적용을 누릅니다.
    • 변경 내용을 무시하고 원래 설정을 유지하려면 재설정을 누릅니다.
