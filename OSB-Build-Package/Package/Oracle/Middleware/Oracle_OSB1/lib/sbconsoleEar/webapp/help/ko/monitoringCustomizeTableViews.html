테이블 뷰 사용자 정의

테이블 사용자 정의기를 사용하여 지정된 설정에 따라 테이블 정보를 표시합니다.

테이블의 기본 정렬 순서는 테이블의 첫번째 열에서 결정됩니다.

  1. 테이블 사용자 정의기 아이콘을 누릅니다. 추가 필드가 표시됩니다.

    2. 언제든지 취소를 눌러 이 테이블을 닫고 원래 설정을 유지할 수 있습니다.

  2. 열 필드에서 표시할 열을 선택합니다.
    1. 사용 가능한 열 필드에서 열 이름을 선택합니다.
    2. 화살표를 눌러 이 열 이름을 선택된 열 필드로 이동합니다.

      3. 별표는 필수 열을 나타냅니다.

    3. 선택된 열 필드에 표시할 열 이름이 모두 나열될 때까지 반복합니다.

      4. 위쪽 및 아래쪽 화살표를 사용하여 필요에 따라 선택된 열 필드의 열 이름 순서를 변경할 수도 있습니다.

  3. 표시된 열 이름을 제거하려면 선택된 열 필드에서 사용 가능한 열 필드로 열 이름을 이동합니다.
  4. 행 드롭다운 목록에서 한 페이지에 표시할 행 수를 선택합니다.
  5. 다음 작업 중 하나를 수행합니다.
    • 이 설정을 저장하려면 테이블 사용자 정의기를 닫고 적용을 누릅니다.
    • 변경 내용을 무시하고 원래 설정을 유지하려면 재설정을 누릅니다.
