경보 규칙 편집

  1. 아직 이 작업을 수행하지 않은 경우 생성을 눌러 새 세션을 생성하거나 편집을 눌러 기존 세션을 입력합니다. 변경 센터 사용을 참조하십시오.
  2. 경보 규칙을 찾습니다. 경보 규칙 찾기를 참조하십시오.
  3. 경보 규칙의 이름을 누릅니다.
  4. 구성 페이지의 필드를 변경하려면 해당 페이지의 편집을 누르거나 페이지 하단의 편집 단추를 누릅니다. 페이지 및 필드에 대한 설명은 경보 규칙 생성 및 편집을 참조하십시오.
  5. 다음 작업 중 하나를 수행합니다.
    • 다음 페이지로 이동하려면 다음을 누릅니다.
    • 이전 페이지로 돌아가려면 이전을 누릅니다.
    • 요약 페이지로 이동하려면 마지막을 누릅니다.
    • 현재 세션에서 갱신을 커밋하려면 저장을 누릅니다.
    • 변경 내용을 무시하고 SLA 경보 규칙 페이지로 돌아가려면 취소를 누릅니다.
  6. 세션을 종료하고 구성을 런타임에 배치하려면 변경 센터에서 활성화를 누릅니다.
