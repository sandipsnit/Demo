경보 규칙 찾기

  1. 리소스 브라우저 > 비즈니스 서비스 또는 프록시 서비스를 선택합니다.
  2. 서비스 이름을 누릅니다.
  3. SLA 경보 규칙 탭을 선택합니다.
  4. 이 서비스에 대한 특정 경보 규칙을 찾으려면 페이지를 스크롤합니다. 테이블의 위 또는 아래에 있는 페이지 콘트롤을 사용합니다. 페이지 번호를 선택하여 특정 페이지로 이동하거나 화살표 단추를 사용하여 다음, 이전, 첫 페이지 또는 마지막 페이지로 이동할 수 있습니다.

이 페이지에서는 다음을 수행할 수 있습니다.

  • 이 서비스의 모니터링을 구성하려면 작업 설정 탭을 선택합니다. 프록시 서비스의 작업 설정 구성 및 비즈니스 서비스의 작업 설정 구성을 참조하십시오.
  • 새 경보 규칙을 추가하려면 새 항목 추가를 누릅니다. 경보 규칙 생성 및 편집을 참조하십시오.
