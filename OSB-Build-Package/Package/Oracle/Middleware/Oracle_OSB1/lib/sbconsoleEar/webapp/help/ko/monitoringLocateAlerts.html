경보 찾기

  1. 작업 > 대시보드를 선택합니다.
  2. SLA 경보 또는 파이프 라인 경보 탭을 선택하여 SLA 또는 파이프 라인 경보를 표시합니다.
  3. 확장된 경보 내역 링크를 누릅니다. 확장된 SLA 경보 내역 및 확장된 파이프 라인 경보 내역 페이지에는 자세한 경보 내역 정보가 표시됩니다.
  4. 목록의 항목 수를 제한하거나 특정 경보를 찾으려면 다음을 수행합니다.
    • 열기 아이콘을 눌러 검색 필터를 추가로 표시합니다. 다중 검색 조건을 지정하는 경우에는 모든 조건과 일치하는 결과만 반환됩니다.
    • 날짜 범위로 필터링합니다. 다음 작업 중 하나를 수행합니다.
      • 날짜 범위 및 시간 간격 필터 조건을 무시하고 모든 경보를 표시하려면 모두를 선택합니다.
      • 시작 및 끝 필드에 MM/DD/YY HH:MM AM|PM 형식으로 날짜 및 시간을 입력한 다음 검색을 누릅니다. 지정한 날짜 및 시간 범위 동안 발생한 경보가 표시됩니다.
      • 마지막의 경우 필드에 일 수를 입력하고 시간 및 분을 지정한 다음 검색을 누릅니다. 지정한 시간 범위 동안 발생한 경보가 표시됩니다.
    • 경보 심각도로 필터링합니다. 경보 심각도 드롭다운 목록에서 경보를 특정 심각도 레벨로 제한하도록 선택한 다음 검색을 누릅니다.

      • 검색을 특정 심각도 레벨 또는 그 이상(가장 포괄적인 레벨에서 가장 제한적인 레벨까지 나열됨: 일반, 경고, 사소, 중요, 심각 또는 치명적)으로 제한하려면 그 이상 체크 박스를 선택합니다.

    • 상위 서비스로 필터링합니다. 찾아보기를 눌러 서비스 선택 페이지를 표시합니다. 목록에서 상위 서비스를 선택하고 제출을 누릅니다. 상위 서비스 필터 조건을 제거하려면 지우기를 누르거나 검색을 누릅니다.
    • SLA 경보만 찾을 경우 경보 이름(모든 문자열)으로 필터링합니다. 와일드 카드 문자를 포함하여 모든 문자열을 입력한 다음 검색을 누릅니다. 자세한 내용은 WebLogic 진단 프레임워크 질의 언어를 참조하십시오.
    • 파이프 라인 경보만 찾을 경우 경보 요약(모든 문자열)으로 필터링합니다. 와일드 카드 문자를 포함하여 모든 문자열을 입력한 다음 검색을 누릅니다. 자세한 내용은 WebLogic 진단 프레임워크 질의 언어를 참조하십시오. 경보 요약 텍스트가 없이 설계된 파이프 라인 경보 작업을 찾으려면 [Oracle Service Bus 경보]를 입력합니다.
    • 모두 보기를 눌러 검색 필터를 제거하고 모든 경보를 표시합니다.

또한 확장된 SLA 경보 내역 및 확장된 파이프 라인 경보 내역 페이지를 사용하여 다음을 수행합니다.

  • 파이 차트에서 경보 데이터를 표시하려면 파이 차트 뷰를 선택합니다.
  • 막대형 차트에서 동일한 데이터를 표시하려면 막대형 차트 보기를 선택합니다.
  • 데이터를 표 형식으로 표시하려면 테이블 뷰를 선택합니다.
  • 갱신된 버전의 데이터를 표시하려면 새로 고침을 누릅니다.
  • 개별 경보를 삭제하려면 해당 체크 박스를 선택하고 페이지 하단에서 삭제를 누릅니다.
  • 모든 경보 또는 지정된 날짜 및 시간 범위 내에서 발생한 경보를 삭제하려면 페이지 맨 위에 있는 경보 내역 비우기 링크를 누릅니다. 경보 비우기를 참조하십시오.

WebLogic 진단 프레임워크 질의 언어

경보는 WebLogic 진단 프레임워크를 사용하여 저장됩니다. 이 프레임워크는 와일드 카드 문자를 포함하여 고유한 질의 언어를 제공합니다. 확장된 경보 내역에서 경보를 필터링하려면 http://www.oracle.com/pls/as111170/lookup?id=WLDFC316에서 Oracle Fusion Middleware Configuring and Using the Diagnostics Framework for Oracle WebLogic Server의 "WLDF Query Language"에 설명된 구문을 사용하십시오.
