경보 비우기

이 페이지에서는 지정된 SLA 또는 파이프 라인 경보를 영구적으로 제거할 수 있습니다.

주: 이 작업은 실행 취소할 수 없습니다.
  1. 작업 > 대시보드를 선택합니다.
  2. SLA 경보 또는 파이프 라인 경보 탭을 선택한 다음 확장된 경보 내역 링크를 누릅니다.
  3. 확장된 경보 내역 페이지의 오른쪽 상단에서 SLA 경보 내역 비우기 또는 파이프 라인 경보 내역 비우기를 누릅니다.
  4. 표 25-3에 표시된 다음 단계 중 하나를 수행합니다.

    5. 표 25-3 경보 비우기 옵션
    수행할 작업
    수행할 단계
    • 모든 SLA 경보 비우기
    • 모든 파이프 라인 경보 비우기
    1. 모든 경보 비우기를 선택합니다.
    2. 비우기를 누릅니다.
    • 지정한 시간 프레임 내의 SLA 경보 비우기
    • 지정한 시간 프레임 내의 파이프 라인 경보 비우기
    1. 경보 비우기 시작... 종료...를 선택합니다.
    2. 비우기 시작 필드에서 월, 일, 연도 및 시간을 입력합니다.
    3. 비우기 종료 필드에서 월, 일, 연도 및 시간을 입력합니다.
    4. 비우기를 누릅니다.

주: 경보를 비우려는 서버를 사용할 수 없을 경우 로깅 프레임워크가 비울 레코드를 지속하고 서버 가용성을 확인하기 위해 30분마다 한 번씩 서버를 풀링합니다. 서버를 사용할 수 있게 되면 경보를 비웁니다.
