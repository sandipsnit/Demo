서비스의 통계 재설정

서비스 건전성 페이지에서 구성에 있는 모든 서비스의 모니터링 통계를 재설정할 수 있습니다. 세션의 사용 여부와 관계없이 통계를 재설정할 수 있습니다.

  1. 작업 > 대시보드 > 서비스 건전성을 선택합니다.
  2. 통계 표시 필드에서 마지막 재설정 이후를 선택합니다. 마지막 재설정 이후 보기는 실행 중인 측정 단위 수 보기를 표시합니다.
  3. 특정 서비스에 대한 모니터링 통계를 재설정하려면 해당 서비스의 작업 열에 있는 재설정 아이콘을 누릅니다.

    4. 이 작업을 수행할 것인지 확인하는 경우, 시스템이 통계를 마지막으로 재설정한 이후 서비스에 대해 수집된 모든 모니터링 통계를 삭제합니다. 그러나 해당 서비스의 현재 집계 간격 중에 수집된 통계는 삭제하지 않습니다. 또한 재설정 아이콘을 누르면 시스템은 해당 서비스의 모니터링 통계를 즉시 다시 수집하기 시작합니다.

  4. 모든 서비스에 대해 모니터링 통계를 재설정하려면 페이지 상단의 모든 통계 재설정 링크를 누릅니다.
    5. 주: 모든 통계 재설정을 누르면 모니터링된 모든 서비스의 통계가 페이지에 표시되는지 여부에 관계없이 해당 통계가 재설정됩니다.

    통계를 재설정하려고 확인하면 시스템은 마지막으로 통계 재설정을 한 이후 구성에서 수집된 모든 서비스에 대한 모든 모니터링 통계를 삭제합니다. 어떤 서비스에 대해서도 현재 집계 간격 사이에 수집되는 통계는 삭제되지 않습니다. 또한 모든 통계 재설정을 누르면 즉시 해당 서비스에 대한 모니터링 통계가 다시 수집되기 시작합니다.
