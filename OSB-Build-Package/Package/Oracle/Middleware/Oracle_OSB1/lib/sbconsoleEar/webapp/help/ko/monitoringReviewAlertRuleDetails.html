경보 규칙 구성 검토

이 페이지에서는 이 경보 규칙에 대해 입력한 구성 데이터를 검토할 수 있습니다. 필요한 경우 경보 규칙을 저장하기 전에 편집을 눌러 구성을 변경할 수 있습니다.

다음 작업 중 하나를 수행합니다.

  • 구성 페이지 중 하나를 변경하려면 해당 페이지의 편집 아이콘을 누릅니다.
  • 이전 페이지로 돌아가려면 이전을 누릅니다.
  • 경보를 생성하려면 저장을 누릅니다. 현재 세션에서 경보 규칙이 생성되고 저장됩니다.
  • 변경 내용을 무시하려면 취소를 누릅니다.
  • 세션을 종료하고 구성을 런타임에 배치하려면 변경 센터에서 활성화를 누릅니다.
