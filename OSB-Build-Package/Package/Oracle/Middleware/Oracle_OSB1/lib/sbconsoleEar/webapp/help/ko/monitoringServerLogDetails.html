심각도
메시지로 보고된 이벤트의 영향이나 심각도를 나타냅니다.
  • 경보 - 시스템의 다른 부분은 계속 작동하지만 특정 서비스의 상태가 불안정합니다. 자동으로 복구할 수 없으며, 문제를 해결하기 위해 관리자의 즉각적인 주의가 필요합니다.
  • 심각 - 시스템 또는 서비스 오류가 발생했습니다. 시스템을 복구할 수 있지만, 서비스가 일시적으로 손실되거나 성능이 영구적으로 저하될 수 있습니다.
  • 응급 - 서버 상태가 불안정합니다. 이 심각도는 심각한 시스템 오류나 장애를 나타냅니다.
  • 오류 - 사용자 오류가 발생했습니다. 시스템 또는 응용 프로그램에서 서비스를 중단하지 않고 성능 저하를 방지하면서 오류를 처리할 수 있습니다.
  • 정보 - 정상적인 작업을 보고하는 데 사용되며 낮은 레벨의 정보 메시지입니다.
  • 주의 - 중요도가 높은 정보 메시지입니다.
  • 경고 - 의심스러운 작업이나 구성이 수행되었지만 정상적인 작업에 영향을 주지 않습니다.
