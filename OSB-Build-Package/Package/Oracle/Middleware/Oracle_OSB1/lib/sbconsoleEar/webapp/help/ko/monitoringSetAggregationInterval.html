서비스의 집계 간격 설정

작업 설정 페이지를 사용하여 특정 서비스의 집계 간격을 설정할 수 있습니다. 집계 간격은 Oracle Service Bus 관리 콘솔에 표시하기 위해 집계된 통계가 계산되는 기간입니다. 기본 집계 간격 설정은 10분입니다.

  1. 아직 이 작업을 수행하지 않은 경우 생성을 눌러 새 세션을 생성하거나 편집을 눌러 기존 세션을 입력합니다. 변경 센터 사용을 참조하십시오.
  2. 리소스 브라우저 > 비즈니스 서비스 또는 프록시 서비스를 선택합니다.
  3. 서비스 이름을 누릅니다.
  4. 작업 설정 탭을 선택합니다.
  5. 집계 간격 필드에서 시간과 분 단위로 집계 간격을 선택한 다음 갱신을 누릅니다.

    6. 시간 단위를 1보다 크게 선택하면 분 단위는 항상 기본적으로 0으로 선택됩니다. 그러나, 시간 단위를 0 또는 1로 선택하면 분 단위로 간격을 구성할 수 있습니다.

  6. 갱신을 눌러 현재 세션에서 갱신을 커밋합니다.
  7. 세션을 종료하고 구성을 런타임에 배치하려면 변경 센터에서 활성화를 누릅니다.
