경보 세부 정보 보기

이 페이지에는 표 25-4와 같은 정보를 포함하여 특정 경보의 세부 정보가 표시됩니다.

표 25-4 경보 세부 정보
속성
설명
경보 이름
SLA 경보에만 해당:
경보에 할당된 이름입니다. 이름은 경보 규칙 구성 페이지에 대한 링크입니다. 경보 규칙 구성 보기를 참조하십시오.
경보 요약
파이프 라인 경보에만 해당:
파이프 라인 경보 작업에 대한 간략한 설명 또는 Oracle Service Bus Alert(경보 요약 텍스트가 없이 설계된 파이프 라인 경보 작업에 제공되는 기본값)입니다.
설명
경보 규칙에 대한 설명입니다.
시간 기록
  • 파이프 라인 경보: 경보가 발생한 날짜와 시간입니다.
  • SLA 경보: 경보 규칙이 마지막으로 true로 평가된 날짜와 시간입니다.
심각도
경보의 사용자 정의 심각도입니다.
  • 치명적
  • 심각
  • 중요
  • 사소
  • 경고
  • 일반
서비스
이 경보와 연관된 서비스의 이름 및 경로입니다. 이 필드는 서비스 모니터링 세부 정보 페이지로 링크됩니다(서비스가 활발하게 모니터링되는 경우에만). 서비스 측정 단위 보기를 참조하십시오.
서비스 유형
상위 서비스 유형: 프록시 서비스 또는 비즈니스 서비스입니다.
프록시 서비스만 파이프 라인 경보를 정의할 수 있기 때문에 파이프 라인 경보의 경우 모든 상위 서비스가 프록시 서비스입니다.
서버
이 경보가 생성된 서버의 이름입니다.
SLA 경보에만 해당: N/A가 표시됩니다.
주석
경보에 대한 메모를 입력할 수 있는 텍스트 상자입니다. 주석이 있는 경보의 경우 주석 아이콘이 경보 내역에 표시됩니다.
