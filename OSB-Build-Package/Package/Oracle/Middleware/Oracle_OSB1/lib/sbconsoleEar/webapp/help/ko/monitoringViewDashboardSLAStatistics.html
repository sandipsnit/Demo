SLA 경보 보기

SLA 경보 페이지에서는 표 25-1과 같은 SLA 경보 요약을 보고 자세한 경보 내역 정보에 액세스할 수 있습니다.

SLA 경보 페이지는 다음을 포함합니다.

  • 파이프 라인 경보, 서비스 건전성 및 서버 건전성 정보를 표시하는 탭입니다.

    • SLA 경보 탭 헤더에서 숫자 표시기는 경보를 마지막으로 본 이후 또는 마지막 대시보드 새로 고침 기간 동안 발생한 새 경보의 수를 표시합니다. 기본적으로 대시보드 새로 고침 빈도는 새로 고침 없음입니다. 대시보드 새로 고침 빈도를 변경하려면 사용자 환경설정 설정을 참조하십시오.

주의: 기본값(새로 고침 없음) 이외의 다른 대시보드 새로 고침 빈도를 선택하면 브라우저가 정기적으로 대시보드를 새로 고칩니다. 이렇게 하면 Oracle Service Bus 관리 콘솔과의 상호 작용이 없는 경우에도 세션 시간 초과가 발생하지 않습니다.
주의: 대시보드 새로 고침 빈도 기능은 주의하여 사용하는 것이 좋습니다. 세션을 무인 상태로 두지 마십시오. 세션이 활성 상태인 동안 다른 사용자가 재인증 없이 시스템에서 액세스 권한을 얻을 수 있습니다.
  • 기본 경보 내역 지속 기간 간격인, 이전 30분 동안 SLA 경보가 발생한 서비스의 파이 차트입니다. 경보 데이터 내역을 표시할 시간 간격을 변경하려면 사용자 환경설정 설정을 참조하십시오.

    • 동일한 시간 간격 동안 심각도 레벨별 경보 세부 정보를 표시한 파이 차트와 페이지의 SLA 경보 수가 일치합니다. 차트에서 경보 심각도에 해당하는 영역을 눌러 세부 정보를 표시할 수 있습니다.

  • SLA 경보가 있는 상위 10개 서비스가 해당 서비스에서 발행한 경보의 내림차순으로 나열됩니다. 각 서비스 이름은 서비스 모니터링 세부 정보에 대한 링크입니다. 서비스 측정 단위 보기를 참조하십시오.
  • 경보 테이블에 대한 사용자 정의 설정에 따라 경보 정보를 표시하는 필드입니다. 예를 들어, 시간 기록, 경보 이름, 경보 심각도, 작업 및 경보와 연관된 서비스 및 서비스 유형 이름이 표시됩니다. 이러한 필드에 대한 자세한 내용은 표 25-1을 참조하십시오.

    • 경보 이름 옆의 주석 아이콘은 주석을 나타냅니다. 아이콘 위로 마우스 포인터를 임포트하면 주석 텍스트를 미리볼 수 있습니다. 아이콘을 누르면 전체 주석 텍스트를 볼 수 있습니다.

  • 테이블 사용자 정의기 아이콘입니다. 테이블 뷰 사용자 정의를 참조하십시오.
  • 사용자 정의된 시스템 경보 내역 페이지에 대한 링크입니다. 경보 찾기를 참조하십시오.

    • 표 25-1 SLA 경보 내역 정보
    속성
    설명
    시간 기록
    경보 규칙이 마지막으로 true로 평가된 날짜와 시간입니다.
    기본적으로 모든 경보는 시간 기록에 대해 오름차순으로 정렬됩니다.
    경보 이름
    경보에 할당된 이름입니다. 이 이름은 경보 세부 정보 페이지에 대한 링크입니다. 경보 세부 정보 보기를 참조하십시오.
    경보 이름 옆의 주석 아이콘은 주석을 나타냅니다. 아이콘 위로 마우스 포인터를 임포트하면 주석 텍스트를 미리볼 수 있습니다.
    경보 심각도
    경보의 사용자 정의 심각도입니다.
    • 치명적
    • 심각
    • 중요
    • 사소
    • 경고
    • 일반
    서비스
    이 경보와 연관된 서비스 이름입니다. 이 이름은 서비스 모니터링 세부 정보 페이지로 링크됩니다(서비스가 활발하게 모니터링되는 경우에만). 서비스 측정 단위 보기를 참조하십시오.
    서비스 유형
    상위 서비스 유형: 프록시 서비스 또는 비즈니스 서비스입니다.
    작업
    경보 규칙 세부 정보 보기 아이콘은 경보 규칙 일반 구성 페이지에 대한 링크입니다. 경보 규칙 구성 보기를 참조하십시오.
