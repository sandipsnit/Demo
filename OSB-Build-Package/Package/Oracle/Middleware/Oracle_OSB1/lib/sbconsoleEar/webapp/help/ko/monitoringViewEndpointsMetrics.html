작업
오프라인 URI를 온라인으로 표시하려면 끝점 온라인 표시 아이콘을 누릅니다. 이 작업은 URI가 오프라인 또는 부분 상태일 경우에만 사용으로 설정됩니다.
이 작업을 사용으로 설정하려면 URI를 오프라인 상태로 만들기 위해 비즈니스 서비스를 사용으로 설정해야 하며 서비스 및 전역 레벨의 모니터링 작업 설정을 사용으로 설정해야 합니다. 비즈니스 서비스의 작업 설정 구성을 참조하십시오.
주: 성공할 경우 작업이 즉시 수행됩니다. 그러나 끝점 URI 상태가 현재 온라인 상태인 점을 반영하면 하나의 모니터링 집계 주기만 실행되며, 이 경우 1분 30초 정도 걸릴 수 있습니다.
