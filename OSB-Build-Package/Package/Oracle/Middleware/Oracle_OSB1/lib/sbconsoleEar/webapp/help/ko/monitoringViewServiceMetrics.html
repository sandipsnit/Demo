서비스 측정 단위 보기

이 페이지에서는 표 25-10과 같은 특정 서비스의 모니터링 세부 정보 및 표 25-11과 같은 서비스 측정 단위를 볼 수 있습니다.

표 25-10 서비스 모니터링 정보
속성
설명
서비스 이름
서비스의 이름과 경로입니다.
서비스 유형
상위 서비스 유형: 프록시 서비스 또는 비즈니스 서비스입니다.
통계 표시
현재 집계 간격의 기간에 대해 서비스의 모니터링 통계를 표시하려면 현재 집계 간격을 선택합니다. 현재 집계 간격에는 서비스 측정 단위의 이동 통계 보기가 표시됩니다.
통계를 마지막으로 재설정한 기간 이후의 기간에 대해 이 서비스의 모니터링 통계를 표시하려면 마지막 재설정 이후를 선택합니다. 마지막 재설정 이후 보기는 실행 중인 측정 단위 수 보기를 표시합니다.
서버
드롭다운 목록에서 서버를 선택하여 해당 서버에 대한 측정 단위를 표시할 수 있습니다.
클러스터가 존재하는 경우, 클러스터 범위 측정 단위가 기본값으로 표시됩니다. 개별 관리 서버를 선택하여 해당 서버에 대한 측정 단위를 표시합니다.
클러스터의 개별 관리 서버에 대한 측정 단위를 표시하는 경우, SLA 경보 수가 N/A입니다(클러스터 범위 SLA 경보 수만 표시됨).
집계 간격
통계 표시 필드에서 현재 집계 간격을 선택한 경우 이 서비스 모니터링에 대해 설정된 시간과 분 단위의 현재 집계 간격입니다. 이 간격은 작업 설정 페이지에서 설정합니다. 서비스의 집계 간격 설정을 참조하십시오.
통계 재설정
통계 표시 필드에서 마지막 재설정 이후를 선택한 경우 이 서비스에 대한 모니터링 통계를 재설정할 수 있습니다. 페이지 하단에서 통계 재설정을 누릅니다.

표 25-11 서비스 측정 단위
속성
설명
SLA 경보 수
  • 경보 수는 현재 집계 간격 또는 마지막 재설정 이후 기간에 대해 통계 표시 뷰를 기반으로 합니다.
  • 경보 심각도 및 시간 기록은 현재 집계 간격 뷰에만 표시됩니다. 마지막 재설정 이후 뷰의 경우 횟수 값만 표시됩니다.
  • 현재 집계 간격 내의 가장 심각한 경보에 대한 경보 심각도입니다. 다중 경보가 존재하는 경우 현재 집계 간격 내의 최고 경보 심각도가 표시됩니다. 현재 집계 간격에 경보가 없는 경우 정상이 표시됩니다.
  • 시간 기록은 서버에서 경보 정보가 마지막으로 폴링된 시간을 표시하고 서버가 마지막으로 폴링되었을 때의 경보 상태를 표시합니다. 서버가 마지막으로 폴링되었을 때 경보가 없는 경우 정상이 표시되고 폴링 시간이 표시됩니다.
