MQ 접속

주: Oracle Service Bus에서 지원되는 WebSphere MQ 버전은 http://www.oracle.com/technology/software/products/ias/files/fusion_certification.html의 Oracle Fusion Middleware Supported System Configurations를 참조하십시오. WebSphere MQ 클라이언트 라이브러리는 Oracle Service Bus 설치 프로그램과 함께 제공되지 않습니다. WebSphere MQ를 설치하여 WebSphere MQ 클라이언트 라이브러리를 얻으십시오.

MQ 접속은 다중 MQ 프록시와 비즈니스 서비스에서 재사용할 수 있는 공유 가능한 리소스입니다. MQ 프록시와 비즈니스 서비스는 MQ 대기열 관리자에 접속된 후 MQ 대기열에 액세스할 수 있습니다. MQ 접속 리소스는 MQ 대기열 관리자에 접속하는 데 필요한 접속 매개변수를 제공합니다.

각 MQ 접속 리소스에는 접속 풀이 있습니다. 지정된 MQ 접속 리소스를 사용하여 지정된 대기열 관리자에 액세스하는 모든 비즈니스 또는 프록시 서비스는 해당 리소스에 대해 생성된 것과 동일한 접속 풀을 사용합니다. 따라서 동일한 대기열 관리자를 사용하는 다중 비즈니스 서비스와 프록시 서비스는 접속 풀을 공유합니다.

Oracle Service Bus MQ 접속 리소스 및 고유 MQ 전송에 대한 자세한 내용은 http://www.oracle.com/pls/as111170/lookup?id=OSBDV1117에서 Oracle Fusion Middleware Developer's Guide for Oracle Service Bus의 "MQ Transport"를 참조하십시오.

WebSphere MQ 기본 개념에 대한 자세한 내용은 http://www.redbooks.ibm.com/redbooks/SG247128/wwhelp/wwhimpl/java/html/wwhelp.htm을 참조하십시오.
