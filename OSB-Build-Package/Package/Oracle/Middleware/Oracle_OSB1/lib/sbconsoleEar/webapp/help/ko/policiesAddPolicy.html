사용자 정의 WS 정책 추가

WS 정책이 포함된 XML 파일을 임포트하거나 Oracle Service Bus 관리 콘솔에서 WS 정책을 직접 쓸 수 있습니다.

  1. 아직 이 작업을 수행하지 않은 경우 생성을 눌러 새 세션을 생성하거나 편집을 눌러 기존 세션을 입력합니다. 변경 센터 사용을 참조하십시오.
  2. 프로젝트 탐색기를 선택한 다음 WS 정책을 추가할 프로젝트나 폴더를 선택합니다. 프로젝트/폴더 뷰 페이지가 표시됩니다.
  3. 리소스 생성 드롭다운 목록에서 인터페이스의 WS 정책을 선택하여 새 WS 정책 생성 페이지를 표시합니다.
  4. 리소스 이름 필드에 이 WS 정책에 대한 고유 이름을 입력합니다.

    리소스 이름 지정 제한 사항의 이름 지정 지침을 따르십시오.
  5. 리소스 설명 필드에 이 WS 정책에 대한 설명을 입력합니다.
  6. 정책 필드에서 다음 작업 중 하나를 수행합니다.
    • 찾아보기 를 눌러 사용자 정의 WS 정책이 포함된 XML 파일을 찾아서 임포트합니다.
    • 새 WS 정책에 사용할 텍스트를 입력합니다.
    • 기존 WS 정책의 데이터를 복사하여 이 필드에 붙여 넣습니다.
  7. 저장을 누릅니다. 현재 세션에 해당 WS 정책이 저장됩니다.
  8. 세션을 종료하고 구성을 런타임에 배치하려면 변경 센터에서 활성화를 누릅니다.
