사용자 정의 WS 정책 삭제

비즈니스 서비스나 프록시 서비스가 WS 정책을 사용하도록 구성된 경우, 비즈니스 서비스 또는 프록시 서비스에서 해당 WS 정책을 제거합니다.

  1. 아직 이 작업을 수행하지 않은 경우 생성을 눌러 새 세션을 생성하거나 편집을 눌러 기존 세션을 입력합니다. 변경 센터 사용을 참조하십시오.
  2. 리소스 브라우저 > WS 정책을 선택합니다.
  3. 삭제할 WS 정책의 옵션 필드에서 삭제 아이콘을 누릅니다.

    4. 현재 세션에서 해당 WS 정책이 삭제됩니다. 비즈니스 서비스 또는 프록시 서비스가 WS 정책을 사용하도록 구성된 경우 삭제 경고 아이콘은 경고 확인을 통해 WS 정책을 삭제할 수 있음을 나타냅니다. 이 경우 삭제된 WS 정책에 대한 서비스의 해결되지 않은 참조로 인해 충돌이 발생할 수 있습니다.

  4. 세션을 종료하고 구성을 런타임에 배치하려면 변경 센터에서 활성화를 누릅니다.
