사용자 정의 WS 정책 찾기

WS 정책 요약 페이지에서는 현재 Oracle Service Bus 도메인에 추가된 사용자 정의 웹 서비스 정책(WS 정책)을 나열합니다. Oracle Service Bus 관리 콘솔은 미리 패키지되어 모든 Oracle Service Bus 도메인에 사용할 수 있는 Oracle Service Bus WS 정책을 표시하지 않습니다.

  1. 다음 작업을 수행합니다.
    • 프로젝트 탐색기를 선택하여 프로젝트 뷰 페이지 또는 프로젝트/폴더 뷰 페이지를 표시합니다. 그런 다음 프로젝트와 폴더를 탐색하여 WS 정책을 찾습니다.
    • 리소스 브라우저 > WS 정책을 선택합니다. WS 정책 요약 페이지에는 표 11-1에 나열된 정보가 표시됩니다. 속성에 대한 자세한 설명은 사용자 정의 WS 정책 찾기를 참조하십시오.
  2. 사용자 정의 WS 정책을 검색하려면 WS 정책 이름의 일부 또는 전부를 이름 필드에 입력합니다. WS 정책의 프로젝트 이름 및 폴더의 일부 또는 전부를 경로 필드에 입력해도 됩니다. 검색을 누릅니다.

    3. 모두 보기를 눌러 검색 필터를 제거하고 모든 WS 정책을 표시합니다.

    표 11-1 WS 정책 정보
    속성
    설명
    WS 정책 이름
    WS 정책에 할당된 고유 이름입니다. 정책 이름을 누르면 WS 정책 세부 정보 보기 페이지가 표시됩니다. 사용자 정의 WS 정책 편집을 참조하십시오.
    경로
    WS 정책이 저장된 폴더 이름과 프로젝트 이름입니다. 이 리소스가 포함된 프로젝트 또는 폴더를 보려면 이름을 누릅니다. 프로젝트 및 폴더를 사용하여 리소스 이름 식별을 참조하십시오.
    옵션
    삭제 아이콘이 포함되어 있습니다. 비즈니스 서비스 또는 프록시 서비스가 WS 정책을 사용하도록 구성된 경우 삭제 경고 아이콘은 경고 확인을 통해 WS 정책을 삭제할 수 있음을 나타냅니다. 이 경우 삭제된 WS 정책에 대한 서비스의 해결되지 않은 참조로 인해 충돌이 발생할 수 있습니다. 사용자 정의 WS 정책 삭제를 참조하십시오.
