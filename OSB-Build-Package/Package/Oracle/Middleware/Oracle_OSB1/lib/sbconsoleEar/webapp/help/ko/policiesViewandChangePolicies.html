사용자 정의 WS 정책 편집

WS 정책 세부 정보 보기 페이지에서 특정 사용자 정의 웹 서비스 정책(WS 정책)의 세부 정보를 보고 변경할 수 있습니다. Oracle Service Bus 관리 콘솔은 Oracle Service Bus WS 정책(읽기 전용)을 표시하지 않습니다.

  1. 아직 이 작업을 수행하지 않은 경우 생성을 눌러 새 세션을 생성하거나 편집을 눌러 기존 세션을 입력합니다. 변경 센터 사용을 참조하십시오.
  2. 사용자 정의 WS 정책 찾기에 표시된 대로 WS 정책을 찾습니다.
  3. WS 정책 이름을 누릅니다. WS 정책 세부 정보 보기 페이지에는 표 11-2에 나열된 정보가 표시됩니다.

    4. 표 11-2 WS 정책 세부 정보 
    속성
    설명
    마지막으로 수정한 사람
    이 WS 정책을 생성했거나 구성으로 임포트한 사용자입니다.
    마지막 수정 시간
    사용자가 이 WS 정책을 생성했거나 구성으로 임포트한 날짜와 시간입니다. 날짜 및 시간 링크를 눌러 이 리소스의 변경 내역을 봅니다. 변경 내역 보기 페이지를 참조하십시오.
    참조
    이 WS 정책에서 참조하는 객체의 수입니다. 해당 참조가 존재하는 경우 숫자 링크를 누르면 객체 목록이 표시됩니다. 리소스에 대한 참조 보기를 참조하십시오.
    참조자
    이 WS 정책을 참조하는 객체의 수입니다. 해당 참조가 존재하는 경우 숫자 링크를 누르면 객체 목록이 표시됩니다. 리소스에 대한 참조 보기를 참조하십시오.
    설명
    이 WS 정책(존재하는 경우)에 대한 설명입니다.
    정책
    이 WS 정책에 사용할 텍스트입니다.

  4. 필드를 변경하려면 편집을 누릅니다. 필드에 대한 설명은 사용자 정의 WS 정책 추가를 참조하십시오.

    5. 리소스 이름 필드는 변경할 수 없습니다.

  5. 저장을 눌러 현재 세션에서 갱신을 커밋합니다.
  6. 세션을 종료하고 구성을 런타임에 배치하려면 변경 센터에서 활성화를 누릅니다.
