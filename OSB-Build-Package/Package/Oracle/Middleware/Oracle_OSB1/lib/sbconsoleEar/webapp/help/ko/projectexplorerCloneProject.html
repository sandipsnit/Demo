프로젝트 복제

다음과 같은 세 가지 방법으로 프로젝트를 복제(복사)할 수 있습니다.

  • 루트 레벨에서 새 이름으로 프로젝트를 복제합니다. 이 경우, 원본 프로젝트와 그 복제본이 피어 프로젝트로 존재합니다. 복제된 프로젝트에는 원본과 동일한 폴더와 리소스가 포함되어 있습니다(소스 또는 복제본을 수정하기 전까지). 자세한 내용은 루트 레벨에서 프로젝트 복제를 참조하십시오.
  • 기존 프로젝트(소스 프로젝트 제외)의 이름을 사용하여 프로젝트를 복제합니다. 이 경우, 소스 프로젝트의 폴더와 리소스는 해당 이름이 복제본에 할당된 프로젝트의 폴더와 리소스로 병합됩니다. 원본 프로젝트는 변경되지 않은 채로 남아 있습니다. 자세한 내용은 루트 레벨에서 프로젝트 복제를 참조하십시오.
  • 프로젝트를 기존 프로젝트 또는 프로젝트 폴더의 새 위치에 복제합니다. 이 경우, 복제된 프로젝트는 새 위치에서 한 폴더 아래로 이동합니다. 여기에는 원본 프로젝트와 동일한 폴더와 리소스가 포함되어 있습니다.

    • 동일한 위치에서 기존 폴더로 동일한 이름의 프로젝트를 복제하면 프로젝트의 내용(하위 폴더 및 리소스)이 대상 폴더의 내용으로 병합됩니다.

    이 경우, 원본 프로젝트는 변경되지 않은 채로 남아 있습니다.

    자세한 내용은 프로젝트 복제 및 해당 복제본을 한 폴더 아래로 내리기를 참조하십시오.

루트 레벨에서 프로젝트 복제
  1. 아직 이 작업을 수행하지 않은 경우 생성을 눌러 새 세션을 생성하거나 편집을 눌러 기존 세션을 입력합니다. 변경 센터 사용을 참조하십시오.
  2. 프로젝트 탐색기 > 프로젝트를 선택하여 프로젝트 뷰 페이지를 표시합니다.
  3. 복제할 프로젝트 행에서 프로젝트 복제 아이콘을 누릅니다. 프로젝트 복제 페이지가 현재 프로젝트 위치가 강조 표시된 상태로 표시됩니다.
  4. 새 이름 필드에 복제본의 이름을 입력합니다. 프로젝트 및 폴더 이름 지정을 참조하십시오.

    5. 복제본을 새 프로젝트로 생성하려면 고유한 이름을 사용하고, 복제본을 해당 프로젝트와 병합하려면 기존 프로젝트의 이름을 사용합니다.

  5. 위치 패널에서 프로젝트를 루트 레벨에서 복제하는 경우 기본 루트 위치를 유지합니다.
  6. 프로젝트 복제를 누릅니다. 현재 세션에서 해당 프로젝트가 복제됩니다.
  7. 세션을 종료하고 구성을 런타임에 배치하려면 변경 센터에서 활성화를 누릅니다.
프로젝트 복제 및 해당 복제본을 한 폴더 아래로 내리기
  1. 아직 이 작업을 수행하지 않은 경우 생성을 눌러 새 세션을 생성하거나 편집을 눌러 기존 세션을 입력합니다. 변경 센터 사용을 참조하십시오.
  2. 프로젝트 탐색기 > 프로젝트를 선택하여 프로젝트 뷰 페이지를 표시합니다.
  3. 복제할 프로젝트 행에서 프로젝트 복제 아이콘을 누릅니다. 프로젝트 복제 페이지가 현재 프로젝트 위치가 강조 표시된 상태로 표시됩니다.
  4. 새 이름 필드에 폴더의 새 이름을 입력합니다(선택 사항). 프로젝트 및 폴더 이름 지정을 참조하십시오.

    5. 새 폴더를 생성하려면 고유한 이름을 사용하고, 복제본을 기존 폴더와 병합하려면 기존 폴더의 이름을 사용합니다(다음 단계에서 선택한 위치에서).

  5. 위치 패널에서 프로젝트 또는 폴더를 선택한 다음 프로젝트 복제를 누릅니다. 복제된 프로젝트가 새 위치의 폴더로 변환되고 현재 세션에 저장됩니다.
  6. 세션을 종료하고 구성을 런타임에 배치하려면 변경 센터에서 활성화를 누릅니다.
