폴더 복제

  1. 아직 이 작업을 수행하지 않은 경우 생성을 눌러 새 세션을 생성하거나 편집을 눌러 기존 세션을 입력합니다. 변경 센터 사용을 참조하십시오.
  2. 프로젝트 탐색기를 선택하고 복제할 폴더가 있는 상위 프로젝트 또는 폴더로 이동합니다. 프로젝트/폴더 뷰 페이지가 표시됩니다.
  3. 복제할 폴더 행에서 폴더 복제 아이콘을 누릅니다. 폴더 복제 페이지가 현재 폴더 위치가 강조 표시된 상태로 표시됩니다.
  4. 동일한 위치 또는 다른 위치에서 폴더를 복제하거나 다음과 같이 폴더를 프로젝트로 업그레이드합니다.
    • 동일한 위치에서 폴더를 복제하려면, 새 이름 필드에 새 이름을 입력합니다. 프로젝트 및 폴더 이름 지정을 참조하십시오.

      • 동일한 위치에 폴더를 복제하려면 위치 패널(선택한 현재 위치와 동일)의 모든 항목에서 어떠한 위치도 선택하지 않거나 현재 프로젝트 및 폴더를 명시적으로 선택합니다.

    • 다른 위치에 폴더를 복제하려면 새 이름 필드에 새 이름을 입력하고(선택 사항) 위치 영역에서 프로젝트 및/또는 폴더를 선택합니다. 프로젝트 및 폴더 이름 지정을 참조하십시오.

      • 기존 폴더와 동일한 이름을 사용한 경우, 원래 폴더의 콘텐츠가 대상 폴더의 콘텐츠에 병합됩니다.

    • 폴더를 프로젝트에 업그레이드하려면 루트 위치를 선택합니다.
  5. 폴더 복제를 누릅니다. 현재 세션에서 해당 폴더가 복제됩니다.

    6. 폴더를 복제하는 경우 폴더 아래의 모든 폴더 및 리소스가 반복해서 복제됩니다.

  6. 세션을 종료하고 구성을 런타임에 배치하려면 변경 센터에서 활성화를 누릅니다.
