리소스 생성

  1. 아직 이 작업을 수행하지 않은 경우 생성을 눌러 새 세션을 생성하거나 편집을 눌러 기존 세션을 입력합니다. 변경 센터 사용을 참조하십시오.
  2. 프로젝트 탐색기를 선택한 다음 리소스를 생성할 프로젝트 또는 폴더를 선택합니다.
  3. 프로젝트/폴더 뷰 페이지의 리소스 생성 드롭다운 목록에서 생성할 리소스 유형을 선택합니다. 선택한 리소스 유형에 따라 다른 페이지가 표시됩니다.
  4. 표시되는 메시지에 따라 정보를 각 페이지에 추가하여 리소스를 생성합니다. 각 유형의 리소스 생성에 대한 자세한 내용은 표 3-3에 나열된 대로 해당 항목을 참조하십시오.
