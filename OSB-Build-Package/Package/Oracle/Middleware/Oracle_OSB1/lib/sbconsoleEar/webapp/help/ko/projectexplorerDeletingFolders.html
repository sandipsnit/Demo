폴더 삭제

  1. 아직 이 작업을 수행하지 않은 경우 생성을 눌러 새 세션을 생성하거나 편집을 눌러 기존 세션을 입력합니다. 변경 센터 사용을 참조하십시오.
  2. 프로젝트 탐색기를 선택하고 삭제할 폴더가 있는 프로젝트 또는 폴더를 선택합니다.
  3. 프로젝트/폴더 뷰 페이지의 삭제할 폴더 행에서 폴더 삭제 아이콘을 누릅니다. 현재 세션에서 해당 폴더가 삭제됩니다.

    4. 폴더를 삭제하면 해당 폴더 아래의 모든 리소스가 삭제됩니다. 이 폴더 아래의 모든 리소스를 다른 폴더의 리소스에서 참조하는 경우 폴더를 삭제할 수는 있지만 경고를 확인해야 합니다. 이 경우 삭제된 리소스에 대한 해결되지 않은 참조로 인해 충돌이 발생할 수 있습니다.

    이러한 리소스에 대해 삭제 아이콘 대신 삭제 경고 아이콘이 표시됩니다.

  4. 세션을 종료하고 구성을 런타임에 배치하려면 변경 센터에서 활성화를 누릅니다.
