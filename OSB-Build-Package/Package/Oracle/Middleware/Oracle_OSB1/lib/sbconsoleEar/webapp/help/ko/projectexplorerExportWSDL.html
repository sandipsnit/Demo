WSDL 익스포트

프로젝트/폴더 뷰 페이지를 사용하여 프록시 서비스 또는 비즈니스와 연관된 WSDL을 익스포트합니다. WSDL을 JAR 파일로 익스포트합니다. 사용자가 세션 외부에 존재하는 경우에만 WSDL을 익스포트할 수 있습니다. 변경 센터 사용을 참조하십시오.

WSDL 익스포트 기능을 사용하면 IDE와 같은 외부 툴로 WSDL을 사용할 수 있습니다. 이 기능은 두 개의 도메인 간 리소스 이동 및 준비에 사용하는 시스템 관리 모듈의 리소스 익스포트 기능과는 다릅니다.

  1. 왼쪽 탐색 창의 프로젝트 탐색기에서 프로젝트 또는 폴더를 누릅니다. 프로젝트/폴더 뷰 페이지가 표시됩니다.
  2. WSDL을 익스포트할 프록시 서비스의 행에서 WSDL 익스포트 아이콘을 누릅니다. 파일 다운로드 대화상자가 표시됩니다.
  3. 파일 다운로드 대화상자에서 열기를 눌러 JAR 파일을 열거나 저장을 눌러 JAR 파일을 데스크톱에 저장합니다.
