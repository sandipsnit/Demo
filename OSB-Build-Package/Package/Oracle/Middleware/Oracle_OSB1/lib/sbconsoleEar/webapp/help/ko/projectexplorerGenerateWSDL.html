WSDL 생성

생성된 WSDL은 WSDL 리소스로 시작할 수 없지만 WSDL을 사용하여 설명할 수는 있는 서비스에 대해 Oracle Service Bus가 생성하는 WSDL 리소스입니다. 예를 들어, EJB 기반 서비스로부터 생성된 WSDL이 생성된 WSDL입니다. 자세한 내용은 http://www.oracle.com/pls/as111170/lookup?id=OSBAG114에서 Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus의 "How WSDL is Used in Oracle Service Bus"를 참조하십시오.

프로젝트/폴더 뷰 페이지를 사용하여 EJB 또는 흐름 전송 유형 비즈니스 서비스와 연관된 WSDL을 생성합니다.

  1. 왼쪽 탐색 창의 프로젝트 탐색기에서 프로젝트 또는 폴더를 누릅니다. 프로젝트/폴더 뷰 페이지가 표시됩니다.
  2. WSDL을 생성할 비즈니스 서비스 행에서 WSDL 생성 아이콘을 누릅니다. 현재 위치(비즈니스 서비스가 있는 프로젝트 이름 및 폴더 이름)가 강조 표시된 상태로 WSDL 리소스 생성 대화상자가 표시됩니다.
  3. WSDL 리소스 생성 대화상자에서 새 WSDL 리소스 이름과 WSDL의 위치를 지정하고 WSDL 생성을 누릅니다.

    리소스 이름 지정 제한 사항의 이름 지정 지침을 따르십시오.

마친 후에

계속해서 WSDL 편집에서 생성된 WSDL을 보거나 편집할 수 있습니다.
