프로젝트 뷰 페이지

프로젝트 뷰 페이지에는 현재 도메인에서 정의된 프로젝트가 나열됩니다. 표 3-1에서는 페이지의 옵션 및 콘트롤을 설명합니다.

표 3-1 프로젝트 뷰 페이지 - 옵션 및 콘트롤
옵션 또는 콘트롤
설명
새 프로젝트 이름 입력
새 프로젝트를 생성하려면 텍스트 필드에 이름을 입력하고 프로젝트 추가를 누릅니다. 프로젝트 추가를 참조하십시오.
또한 프로젝트 및 폴더 이름 지정을 참조하십시오.
이름
프로젝트 이름을 눌러 해당 프로젝트의 프로젝트/폴더 뷰 페이지를 표시합니다.
옵션
나열된 프로젝트에 대해 다음을 수행할 수 있습니다.
  • 프로젝트 이름 바꾸기 아이콘을 눌러 프로젝트 이름을 바꿀 수 있습니다. 프로젝트 이름 바꾸기를 참조하십시오.
  • 프로젝트를 이동하려면 프로젝트 이동 아이콘을 누릅니다. 프로젝트 이동(폴더로 변환)을 참조하십시오.
  • 프로젝트 복제 아이콘을 눌러 프로젝트를 복제합니다. 프로젝트 복제를 참조하십시오.
  • 프로젝트를 삭제하려면 프로젝트 삭제 아이콘을 누릅니다. 다른 리소스에서 이 리소스를 참조하는 경우 삭제 경고 아이콘이 표시됩니다. 경고 확인을 통해 이 리소스를 삭제할 수 있습니다. 이 경우 삭제된 리소스에 대한 해결되지 않은 참조로 인해 충돌이 발생할 수 있습니다. 프로젝트 삭제를 참조하십시오.
이 작업을 수행하면 결과가 현재 세션에 저장됩니다. 세션을 종료하고 구성을 런타임에 배치하려면 변경 센터에서 활성화를 누릅니다.
