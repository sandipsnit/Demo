리소스 로드 - URL에서 리소스 로드 페이지

리소스 로드 - URL에서 리소스 로드 페이지를 사용하여 URL에서 리소스 로드에 설명된 대로 웹에서 사용 가능한 파일에서 리소스를 임포트할 수 있습니다.

표 3-5에서는 페이지의 옵션 및 콘트롤을 설명합니다.

표 3-5 리소스 로드: URL에서 리소스 로드 페이지 - 옵션 및 콘트롤
옵션 및 콘트롤
설명
URL/경로
파일이 위치한 URL을 입력합니다.
로컬 리소스를 지정하려면 경로로 정규화된 파일 이름이 지정된 파일 프로토콜을 사용할 수 있습니다(예: file:///c:/osbresources/ForeachAction.jar).
리소스 이름
리소스 이름을 입력합니다.
리소스 유형
목록에서 리소스 유형을 선택합니다.
