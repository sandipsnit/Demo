URL에서 리소스 로드

URL 또는 파일 시스템에서 사용 가능한 리소스에 대해 대량 임포트를 수행할 수 있습니다. 대량 임포트를 수행하면 WSDL과 같은 루트 리소스를 다른 WSDL 및 스키마와 같은 해당 종속 항목과 함께 임포트할 수 있습니다. 종속성 맵은 자동으로 확인됩니다.

  1. 아직 이 작업을 수행하지 않은 경우 생성을 눌러 새 세션을 생성하거나 편집을 눌러 기존 세션을 입력합니다. 변경 센터 사용을 참조하십시오.
  2. 프로젝트 탐색기를 선택한 다음 리소스를 추가할 프로젝트 또는 폴더를 선택합니다. 프로젝트/폴더 뷰 페이지가 표시됩니다.
  3. 리소스 생성 드롭다운 목록에서 URL 리소스를 선택합니다. 리소스 로드 - URL에서 리소스 로드 페이지가 표시됩니다.
  4. URL/경로 필드에 리소스의 URL 또는 경로를 입력합니다.

    5. 로컬 리소스를 지정하려면 경로로 정규화된 파일 이름이 지정된 파일 프로토콜을 사용할 수 있습니다(예: file:///c:/osbresources/ForeachAction.jar).

  5. 리소스 이름 필드에 리소스 이름을 입력합니다.
  6. 리소스 유형 드롭다운 목록에서 리소스 유형을 선택합니다.
  7. 다음을 눌러 리소스 로드 - 로드된 리소스 검토 페이지를 표시합니다. 이 임포트에 포함하지 않을 리소스는 옆에 있는 체크 박스의 선택을 취소합니다.
  8. 선택한 리소스를 임포트하려면 임포트를 누릅니다.
  9. 리소스 로드 - 임포트 결과 페이지에는 임포트한 리소스가 표시됩니다. 다른 항목 로드를 눌러 다른 파일에서 리소스를 임포트하거나 왼쪽 탐색 창에서 다른 페이지를 선택하여 다른 작업을 계속합니다.
  10. 세션을 종료하고 구성을 런타임에 배치하려면 변경 센터에서 활성화를 누릅니다.
