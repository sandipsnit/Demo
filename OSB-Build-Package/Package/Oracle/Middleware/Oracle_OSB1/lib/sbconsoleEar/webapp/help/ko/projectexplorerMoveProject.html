프로젝트 이동(폴더로 변환)

프로젝트를 "이동"하면 프로젝트가 새 위치의 폴더로 변환됩니다.

  1. 아직 이 작업을 수행하지 않은 경우 생성을 눌러 새 세션을 생성하거나 편집을 눌러 기존 세션을 입력합니다. 변경 센터 사용을 참조하십시오.
  2. 프로젝트 탐색기 > 프로젝트를 선택하여 프로젝트 뷰 페이지를 표시합니다.
  3. 이동할 프로젝트 행에서 프로젝트 이동 아이콘을 누릅니다. 프로젝트 이동 페이지가 현재 프로젝트 위치가 강조 표시된 상태로 표시됩니다.
  4. 위치에서 프로젝트를 "이동"할 프로젝트 또는 폴더를 선택합니다. 프로젝트가 새 위치의 폴더로 변환되고 현재 세션에 저장됩니다.

    5. 프로젝트를 새 위치로 이동하고 동일한 이름을 가진 폴더가 해당 위치에 이미 존재하는 경우 프로젝트 콘텐츠가 기존 폴더의 콘텐츠와 병합됩니다.

  5. 세션을 종료하고 구성을 런타임에 배치하려면 변경 센터에서 활성화를 누릅니다.
