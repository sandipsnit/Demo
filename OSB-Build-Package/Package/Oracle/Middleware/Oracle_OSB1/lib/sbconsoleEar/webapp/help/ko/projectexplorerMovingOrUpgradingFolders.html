폴더 이동 또는 업그레이드

  1. 아직 이 작업을 수행하지 않은 경우 생성을 눌러 새 세션을 생성하거나 편집을 눌러 기존 세션을 입력합니다. 변경 센터 사용을 참조하십시오.
  2. 프로젝트 탐색기를 선택하고 이동 또는 업그레이드할 폴더가 있는 프로젝트 또는 폴더를 선택합니다. 프로젝트/폴더 뷰 페이지가 표시됩니다.
  3. 이동 또는 업그레이드할 폴더 행에서 폴더 이동 아이콘을 누릅니다. 폴더 이동 페이지가 현재 폴더 위치가 강조 표시된 상태로 표시됩니다.
  4. 폴더를 다른 프로젝트의 새 위치로 이동하거나 폴더를 프로젝트에 업그레이드할지 여부를 결정합니다.
    • 폴더를 새 위치로 이동하려면 위치 패널에서 폴더를 이동할 프로젝트 또는 폴더를 선택합니다.
    • 폴더를 프로젝트에 업그레이드하려면 루트 위치를 선택합니다.
  5. 제출을 누릅니다. 폴더(또는 프로젝트, 폴더에서 업그레이드된 경우) 현재 세션의 새 위치에 저장됩니다.

    6. 폴더를 이동하거나 업그레이드할 때, 동일한 이름의 다른 폴더가 해당 위치에 이미 존재하는 경우, 콘텐츠가 업그레이드된 폴더로 이동되고 기존 폴더 또는 프로젝트의 콘텐츠와 병합됩니다.

  6. 세션을 종료하고 구성을 런타임에 배치하려면 변경 센터에서 활성화를 누릅니다.
