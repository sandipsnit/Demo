리소스 이름 바꾸기

  1. 아직 이 작업을 수행하지 않은 경우 생성을 눌러 새 세션을 생성하거나 편집을 눌러 기존 세션을 입력합니다. 변경 센터 사용을 참조하십시오.
  2. 프로젝트 탐색기를 선택한 다음 이름을 변경할 리소스가 있는 프로젝트 또는 폴더를 선택합니다. 프로젝트/폴더 뷰 페이지가 표시됩니다.
  3. 이름을 변경할 리소스 행에서 이름 바꾸기 아이콘을 누릅니다. 이름 바꾸기 페이지가 표시됩니다.
  4. 새 리소스 이름 필드에 리소스에 대한 새 이름을 입력합니다. 리소스 이름 지정 제한 사항을 참조하십시오.
  5. 이름 바꾸기를 누릅니다.
    6. 주: UDDI 레지스트리에서 비즈니스 임포트한 비즈니스 서비스의 이름을 변경하는 경우, 서비스는 레지스트리에서 분리됩니다.
  6. 세션을 종료하고 구성을 런타임에 배치하려면 변경 센터에서 활성화를 누릅니다.
