리소스 로드 - 로드된 리소스 검토 페이지

리소스 로드 - 로드된 리소스 검토 페이지에는 Zip 파일에서 리소스 로드 또는 URL에서 리소스 로드에 설명된 대로 Zip 또는 기타 파일에서 임포트할 리소스에 대한 세부 정보가 표시됩니다.

표 3-6에서는 페이지의 옵션 및 콘트롤을 설명합니다.

설명

지원되는 리소스 유형을 나열합니다.

파일 이름 옆에 있는 경고 아이콘은 리소스 유형을 알 수 없음을 나타냅니다. 확장이 없는 파일이거나 맵에 정의된 확장이 없는 파일은 알려지지 않음으로 간주됩니다. 알 수 없는 유형의 파일은 임포트할 수 없습니다.
