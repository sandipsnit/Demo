리소스 로드 - 결과 임포트 페이지

리소스 로드 - 결과 임포트 페이지는 임포트한 Zip 파일 또는 URL에서 리소스를 로드한 결과를 표시합니다.

Zip 파일 또는 URL에서 리소스를 로드했는지 여부에 따라 Zip 파일에서 리소스 로드 또는 URL에서 리소스 로드 중 한 단계를 완료하고 리소스 로드 - 로드된 리소스 검토 페이지의 단계를 완료합니다.

임포트 결과 페이지에는 다음과 같은 정보가 표시됩니다.

표 3-7 프로젝트/폴더 뷰 페이지 - 옵션 및 콘트롤
옵션 또는 콘트롤
설명
상태
리소스의 상태(예: 리소스를 성공적으로 임포트했는지 여부)입니다.
WSDL은 모든 종속성을 임포트할 때까지 부적합으로 잠시 표시될 수 있습니다. 그러나 임포트 작업이 완료되면 구성 서비스는 부적합으로 표시된 파일을 다시 검증합니다.
이름
리소스 이름입니다.
경로
리소스가 위치해 있는 프로젝트 및 폴더인 리소스 경로입니다.
리소스 유형
임포트한 리소스 유형을 표시합니다.
오류 메시지
리소스를 성공적으로 임포트하지 않은 경우 오류 메시지의 세부 정보를 표시합니다.
