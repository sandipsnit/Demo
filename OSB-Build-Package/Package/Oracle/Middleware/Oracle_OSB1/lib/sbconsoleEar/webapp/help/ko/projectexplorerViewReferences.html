참조 보기 페이지

프로젝트/폴더 뷰 페이지에서 참조 또는 참조자를 눌렀는지 여부에 따라 참조 보기 페이지에 다음 목록이 표시됩니다.

  • 프로젝트 또는 폴더 내의 리소스에서 참조하는 현재 프로젝트 또는 폴더 외부의 리소스
  • 프로젝트 또는 폴더 내의 리소스를 참조하는 현재 프로젝트 또는 폴더 외부의 리소스.

표 3-8에서는 페이지의 옵션 및 콘트롤을 설명합니다.

표 3-8 프로젝트 참조 페이지 - 옵션 및 콘트롤
옵션 또는 콘트롤
설명
이름
리소스 이름을 눌러 리소스의 세부 정보를 표시합니다.
리소스 유형
프로젝트, 폴더 또는 지원되는 모든 Oracle Service Bus 리소스 유형을 표시합니다.
