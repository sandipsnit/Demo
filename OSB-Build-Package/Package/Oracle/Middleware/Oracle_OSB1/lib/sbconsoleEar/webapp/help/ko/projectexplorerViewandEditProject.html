프로젝트/폴더 뷰 페이지

프로젝트/폴더 뷰 페이지에서는 단일 프로젝트의 세부 정보를 표시합니다. 표 3-2에서는 페이지의 옵션 및 콘트롤을 설명합니다.

표 3-2 프로젝트/폴더 뷰 페이지 - 옵션 및 콘트롤
옵션 또는 콘트롤
설명
프로젝트 이름
참조
이 필드에서 숫자를 누르면(0보다 클 경우) 이 프로젝트 내의 리소스가 종속된 이 프로젝트 외부의 리소스 목록이 표시됩니다.
참조자
이 필드에서 숫자를 누르면( 0보다 클 경우) 이 프로젝트 내의 리소스에 종속된 이 프로젝트 외부의 리소스 목록이 표시됩니다.
설명
이 프로젝트의 설명을 추가하거나 편집하려면 설명 편집을 누르고 설명을 갱신하려면 설명 제출을 누릅니다.
폴더
새 폴더 이름 입력
이 프로젝트에서 새 폴더를 생성하려면 텍스트 필드에 이름을 입력하고 폴더 추가를 누릅니다. 폴더 추가를 참조하십시오.
또한 프로젝트 및 폴더 이름 지정을 참조하십시오.
이름
폴더 이름을 눌러 해당 폴더의 프로젝트/폴더 뷰 페이지를 표시합니다.
옵션
나열된 폴더에 대해 다음을 수행할 수 있습니다.
  • 폴더 이름 바꾸기 아이콘을 눌러 폴더 이름을 바꿀 수 있습니다. 폴더 이름 바꾸기를 참조하십시오.
  • 프로젝트로 업그레이드 또는 폴더 이동 아이콘을 눌러 폴더를 이동하거나 프로젝트로 업그레이드할 수 있습니다. 폴더 이동 또는 업그레이드를 참조하십시오.
  • 폴더 복제 아이콘을 눌러 폴더를 복제할 수 있습니다. 폴더 복제를 참조하십시오.
  • 폴더 삭제 아이콘을 눌러 폴더를 삭제할 수 있습니다. 다른 리소스에서 이 리소스를 참조하는 경우 삭제 경고 아이콘이 표시됩니다. 경고 확인을 통해 리소스를 삭제할 수 있습니다. 이 경우 삭제된 리소스에 대한 해결되지 않은 참조로 인해 충돌이 발생할 수 있습니다. 폴더 삭제를 참조하십시오.
이 작업을 수행하면 결과가 현재 세션에 저장됩니다. 세션을 종료하고 구성을 런타임에 배치하려면 변경 센터에서 활성화를 누릅니다.
삭제
폴더를 삭제하려면 삭제할 폴더 이름 옆에 있는 체크 박스를 선택한 다음 삭제를 누릅니다.
리소스
리소스 생성
이 프로젝트의 새 리소스를 생성하려면 리소스 생성 목록에서 리소스 유형을 선택합니다. 리소스 생성를 참조하십시오.
이름
리소스 이름을 눌러 리소스에 대한 세부 정보를 표시합니다.
리소스 유형
이 리소스의 유형입니다.
작업
리소스에 대해 사용할 수 있는 작업은 리소스 유형에 따라 다릅니다.
  • 메시지 흐름 편집 아이콘이 프록시 서비스에 대해 표시됩니다. 이 아이콘을 눌러 프록시 서비스의 메시지를 편집할 수 있습니다. 메시지 흐름 생성, 편집 및 보기를 참조하십시오.
  • 테스트 콘솔 시작 아이콘이 비즈니스 서비스, 프록시 서비스, XQuery 변환, XSL 변환 및 MFL 파일에 대해 표시됩니다. 서비스 및 변환 설계 검증 및 테스트에 사용할 수 있는 테스트 콘솔을 호출하려면 이 아이콘을 누릅니다. 서비스의 경우 런타임 시, 즉 세션이 활성화될 때 테스트 콘솔만 사용할 수 있습니다. 변환에 대해 세션의 사용 여부와 관계없이 테스트 콘솔을 사용할 수 있습니다. 자세한 내용은 서비스 테스트 및 변환 테스트를 참조하십시오.
  • WSDL 기반 프록시 서비스에 대해 WSDL 익스포트 아이콘이 표시됩니다. 이 아이콘을 눌러 WSDL을 익스포트하면 IDE와 같은 외부 툴에서 WSDL을 보거나 수정할 수 있습니다.

    • 이 기능은 두 개의 도메인 간 리소스 이동 및 준비에 사용하는 시스템 관리 모듈의 리소스 익스포트 기능과는 다릅니다. WSDL 익스포트를 참조하십시오.

  • WSDL 생성 아이콘은 전송 유형 비즈니스 서비스(예: EJB 및 흐름)에 대해 표시됩니다. 이 아이콘을 눌러 WSDL을 생성합니다. 이렇게 하면 WSDL을 보거나 수정할 수 있습니다. WSDL 생성을 참조하십시오.
옵션
옵션 열에는 다음 아이콘이 표시됩니다.
  • 리소스 이름 바꾸기 아이콘을 눌러 리소스 이름을 바꿀 수 있습니다. 리소스 이름 바꾸기를 참조하십시오.
  • 리소스 이동 아이콘을 눌러 리소스를 이동할 수 있습니다. 리소스 이동을 참조하십시오.
  • 리소스 복제 아이콘을 눌러 리소스를 복제할 수 있습니다. 리소스 복제를 참조하십시오.
  • 삭제 아이콘을 눌러 리소스를 삭제할 수 있습니다. 다른 리소스에서 이 리소스를 참조하는 경우 삭제 경고 아이콘이 표시됩니다. 경고 확인을 통해 이 리소스를 삭제할 수 있습니다. 이 경우 삭제된 리소스에 대한 해결되지 않은 참조로 인해 충돌이 발생할 수 있습니다.
이 작업을 수행하면 결과가 현재 세션에 저장됩니다. 세션을 종료하고 구성을 런타임에 배치하려면 변경 센터에서 활성화를 누릅니다.
