웹 브라우저에서 리소스 보기

다음 URL을 사용하여 표준 웹 브라우저에서 Oracle Service Bus 리소스 보기

    • WSDL 리소스의 경로:
      http://host:port/sbresource?WSDL/project_path/wsdlname
      또는
      http://host:port/proxy_service_endpoint_URI?WSDL (WSDL 기반 HTTP 프록시 서비스)
    • 프록시 서비스 리소스의 경로:
      http://host:port/sbresource?PROXY/project_path/proxyname
      http://host:port/sbresource?ORAPROXY/project_path/proxyname (Oracle Web Services Manager 정책이 첨부된 프록시 서비스)
    • 비즈니스 서비스 리소스의 경로:
      http://host:port/sbresource?BIZ/project_path/bizname
      http://host:port/sbresource?ORABIZ/project_path/bizname (Oracle Web Services Manager 정책이 첨부된 비즈니스 서비스)
    • WSS 리소스의 경로:
      http://host:port/sbresource?POLICY/project_path/policyname
    • MFL 리소스의 경로:
      http://host:port/sbresource?MFL/project_path/mflname
    • SCHEMA 리소스의 경로:
      http://host:port/sbresource?SCHEMA/project_path/schemaname

    주: 리소스 이름에 특수 문자를 사용하는 경우 특수 문자를 이스케이프 처리하려면 Oracle Service Bus의 리소스 표시에 사용되는 URL을 UTF-8로 인코딩해야 합니다.

    또한 Oracle Web Services Manager 정책을 포함하는 WSDL을 검색하여 해당 정책이 지원되는 WS 정책과 WS 보안 정책 표준을 준수하는지 확인할 수 있습니다. 자세한 내용은 http://www.oracle.com/pls/as111170/lookup?id=OSBDV1877에서 Oracle Fusion Middleware Developer's Guide for Oracle Service Bus의 "Advertising WSDLs to Support WS Standards"를 참조하십시오.
