서비스 키 제공자 삭제

서비스 키 제공자를 삭제할 경우 Oracle Service Bus는 PKI 인증서 매핑 제공자에서 키 쌍 바인딩에 대한 연관된 별칭도 삭제합니다. 키 저장소에서 연관된 키와 인증서 쌍이 삭제되지는 않습니다.

  1. 아직 이 작업을 수행하지 않은 경우 생성을 눌러 새 세션을 생성하거나 편집을 눌러 기존 세션을 입력합니다. 변경 센터 사용을 참조하십시오.
  2. 프록시 서비스가 서비스 키 제공자를 사용하도록 구성된 경우 프록시 서비스에서 서비스 키 제공자를 제거합니다. 프록시 서비스 구성 편집을 참조하십시오.
  3. 리소스 브라우저 > 서비스 키 제공자를 선택하여 서비스 키 제공자 요약 페이지를 표시합니다.
  4. 삭제할 서비스 키 제공자의 옵션 필드에서 삭제 아이콘을 누릅니다. 현재 세션에서 서비스 키 제공자가 삭제됩니다. 서비스 계정을 사용하도록 비즈니스 서비스 또는 프록시 서비스가 구성된 경우 삭제 경고 아이콘은 경고를 확인하면 서비스 키 제공자를 삭제할 수 있음을 나타냅니다. 이 경우 삭제된 서비스 키 제공자를 참조하는 서비스의 해결되지 않은 참조로 인해 충돌이 발생할 수 있습니다.
  5. 세션을 종료하고 구성을 런타임에 배치하려면 변경 센터에서 활성화를 누릅니다.
