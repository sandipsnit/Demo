서비스 키 제공자 찾기

  1. 다음 작업을 수행합니다.
    • 프로젝트 탐색기를 선택하여 프로젝트 뷰 페이지 또는 프로젝트/폴더 뷰 페이지를 표시합니다. 그런 다음 프로젝트와 폴더를 탐색하여 서비스 키 제공자를 찾습니다.
    • 리소스 브라우저 > 서비스 키 제공자를 선택합니다. 그러면 서비스 키 제공자 요약 페이지에 표 17-1과 같은 정보가 표시됩니다.
  2. 서비스 키 제공자를 검색하려면 제공자 이름의 일부 또는 전부를 이름 필드에 입력합니다. 또는 제공자의 프로젝트 이름 및 폴더의 일부 또는 전부를 경로 필드에 입력합니다. 검색을 누릅니다.

    3. 모두 보기를 눌러 검색 필터를 제거하고 모든 서비스 키 제공자를 표시합니다.

    표 17-1 서비스 키 제공자 정보 
    속성
    설명
    이름
    서비스 키 제공자의 고유 이름입니다. 서비스 키 제공자 세부 정보 보기 페이지를 보려면 이름을 누릅니다. 서비스 키 제공자 편집을 참조하십시오.
    경로
    서비스 키 제공자가 있는 폴더 이름과 프로젝트 이름입니다. 이 리소스를 포함하는 프로젝트나 폴더를 보려면 이름을 누릅니다. 프로젝트 및 폴더를 사용하여 리소스 이름 식별을 참조하십시오.
