할당 작업 추가

할당 작업을 사용하여 컨텍스트 변수에 XQuery 표현식 결과를 할당할 수 있습니다.

시작하기 전에

이 지침에서는 메시지 흐름 생성, 편집 및 보기에 설명된 것처럼 메시지 흐름 편집 페이지에서 메시지 흐름을 편집하고 있다고 가정합니다.

할당 작업을 추가하려면 다음을 수행합니다.

  1. 메시지 흐름에서 작업 추가 및 편집에 설명된 대로 작업을 추가할 위치로 이동합니다.
  2. 해당 아이콘을 누른 다음 작업 추가 > 메시지 처리 > 할당을 선택합니다.
  3. <표현식>을 누릅니다. XQuery 표현식 편집기 페이지가 표시됩니다. XQuery 표현식은 이름이 지정된 변수에 할당할 데이터를 생성하는 데 사용됩니다. 인라인 XQuery 및 XPath 표현식 생성 및 편집을 참조하십시오.
  4. 표현식 편집을 마쳤으면 변수 필드에 컨텍스트 변수를 입력합니다. 컨텍스트 변수에 대한 자세한 내용은 http://www.oracle.com/pls/as111170/lookup?id=OSBAG325에서 "Inbound and Outbound Variables" 및 http://www.oracle.com/pls/as111170/lookup?id=OSBAG348에서 Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus의 "Constructing Messages to Dispatch"를 참조하십시오.

마친 후에

이 작업의 구성을 완료하면 메시지 흐름에서 작업 추가 및 편집에 설명된 대로 계속해서 다른 작업을 구성하거나 구성을 저장합니다.
