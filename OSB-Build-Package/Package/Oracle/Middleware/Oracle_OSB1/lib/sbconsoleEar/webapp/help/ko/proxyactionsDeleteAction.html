삭제 작업 추가

삭제 작업을 사용하여 컨텍스트 변수 또는 XPath 표현식에 지정된 노드 세트를 삭제할 수 있습니다. 삭제 작업은 갱신 작업 세트 중 하나입니다.

시작하기 전에

이 지침에서는 메시지 흐름 생성, 편집 및 보기에 설명된 것처럼 메시지 흐름 편집 페이지에서 메시지 흐름을 편집하고 있다고 가정합니다.

삭제 작업을 추가하려면 다음을 수행합니다.

  1. 메시지 흐름에서 작업 추가 및 편집에 설명된 대로 작업을 추가할 위치로 이동합니다.
  2. 컨텍스트 변수를 삭제하려면 변수 라디오 단추를 선택한 다음 변수 필드에 컨텍스트 변수의 이름을 입력합니다. 컨텍스트 변수에 대한 보다 자세한 내용은 부록 A, 메시지 컨텍스트를 참조하십시오.

    3. 또는 XPath 표현식에서 선택한 모든 노드를 삭제하려면 XPath 라디오 단추를 선택한 다음 <XPath>를 누릅니다. XPath 표현식 편집기 페이지가 표시됩니다. 인라인 XQuery 및 XPath 표현식 생성 및 편집을 참조하십시오. 표현식을 저장한 후 변수 필드에 컨텍스트 변수를 입력합니다. 컨텍스트 변수에 대한 보다 자세한 내용은 부록 A, 메시지 컨텍스트를 참조하십시오.

마친 후에

이 작업의 구성을 완료하면 메시지 흐름에서 작업 추가 및 편집에 설명된 대로 계속해서 다른 작업을 구성하거나 구성을 저장합니다.
