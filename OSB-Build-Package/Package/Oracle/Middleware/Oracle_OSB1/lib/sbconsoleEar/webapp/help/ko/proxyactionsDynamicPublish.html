동적 게시 작업 추가

동적 게시 작업을 사용하여 메시지를 XQuery 표현식으로 지정된 서비스에 게시합니다.

게시 동작에 대한 자세한 내용은 Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus(http://www.oracle.com/pls/as111170/lookup?id=OSBAG205)에서 "Performing Transformations in Message Flows"를 참조하십시오.

  1. 메시지 흐름에서 작업 추가 및 편집에 설명된 대로 작업을 추가할 위치로 이동합니다.
  2. 해당 아이콘을 누른 다음 작업 추가 > 통신 > 동적 게시를 선택합니다.
  3. <표현식>을 누릅니다.
  4. XQuery 표현식 편집기에서 Xquery 표현식을 입력하거나 다음과 비슷한 결과를 생성하는 XQuery 리소스를 선택합니다. 
    5. <ctx:route>

    <ctx:service 
isProxy="false">project/folder/businessservicename</ctx:service>

    <ctx:operation>foo</ctx:operation>

    </ctx:route>
    주: 프록시 서비스가 호출될 경우 isProxy 속성을 true로 설정해야 합니다. 비즈니스 서비스를 호출 중인 경우 isProxy를 false로 설정합니다.
    작업 요소는 선택 사항입니다.
  5. 저장을 누릅니다.
  6. 요청 작업 필드에서 작업 추가를 눌러 작업을 추가한 다음 서비스와 연관시킬 작업을 선택합니다. 둘 이상의 작업을 추가할 수 있습니다. 추가할 작업 유형에 대한 자세한 내용은 메시지 흐름에서 작업 추가 및 편집의 작업 테이블을 참조하십시오.

마친 후에

이 작업의 구성을 완료하면 메시지 흐름에서 작업 추가 및 편집에 설명된 대로 계속해서 다른 작업을 구성하거나 구성을 저장합니다.
