For Each 작업 추가

For Each 작업을 사용하여 값 시퀀스를 반복하고 작업을 차단할 수 있습니다.

시작하기 전에

이 지침에서는 메시지 흐름 생성, 편집 및 보기에 설명된 것처럼 메시지 흐름 편집 페이지에서 메시지 흐름을 편집하고 있다고 가정합니다.

For Each 작업을 추가하려면 다음을 수행합니다.

  1. 메시지 흐름에서 작업 추가 및 편집에 설명된 대로 작업을 추가할 위치로 이동합니다.
  2. 해당 아이콘을 누른 다음 작업 추가 > 흐름 제어 > For Each를 선택합니다.
  3. 변수 필드에 변수 이름을 입력한 다음 <XPath>를 눌러 XPath 표현식을 생성할 XPath 편집기를 열고 Do () 루프에서 작업을 구성합니다.

마친 후에

이 작업의 구성을 완료하면 메시지 흐름에서 작업 추가 및 편집에 설명된 대로 계속해서 다른 작업을 구성하거나 구성을 저장합니다.
