If... Then... 작업 추가

if... then... 작업을 사용하여 XQuery 표현식의 부울 결과에 따라 작업 또는 작업 세트를 조건부로 수행할 수 있습니다.

시작하기 전에

이 지침에서는 메시지 흐름 생성, 편집 및 보기에 설명된 것처럼 메시지 흐름 편집 페이지에서 메시지 흐름을 편집하고 있다고 가정합니다.

If...Then 작업을 추가하려면 다음을 수행합니다.

  1. 메시지 흐름에서 작업 추가 및 편집에 설명된 대로 작업을 추가할 위치로 이동합니다.
  2. 해당 아이콘을 누른 다음 작업 추가 > 흐름 제어 > If...Then을 선택합니다.
  3. <조건>을 눌러 XQuery 조건 편집기 페이지를 표시합니다.

    4. 작성한 조건은 then() 절이 입력되기 전에 표준 if...then 로직 단위로 실행되는 테스트로 사용됩니다. 인라인 XQuery 및 XPath 표현식 생성 및 편집을 참조하십시오.

  4. XQuery 조건 편집이 완료되면 작업 추가를 누른 다음 조건과 연관시킬 작업을 선택합니다. 추가할 작업 유형에 대한 자세한 내용은 메시지 흐름에서 작업 추가 및 편집을 참조하십시오.

    5. 경로 지정 노드에서는 라우팅, 동적 라우팅 또는 라우팅 테이블 작업만 선택할 수 있습니다. 그러나 이러한 작업 내에 요청 작업과 응답 작업이 포함될 수 있습니다.

  5. 로직이 필요하면 If...Then... 아이콘을 누른 다음 else-if 조건 추가 또는 else 조건 추가를 눌러 else-if 조건 또는 else 조건을 추가합니다. 작업 추가를 눌러 작업을 이러한 조건과 연관시킵니다.

    6. 조건 작업을 중첩할 수 있습니다. 그러나 스테이지 편집기에서는 중첩 누적 레벨이 4개까지로 제한됩니다. 5번째 레벨을 추가하면 이 중첩 작업이 표시되지 않습니다. 누적 레벨은 분기 작업인 If... Then... 조건, 게시 테이블 및 경로 테이블을 모두 포함합니다. 예를 들어, 2개 레벨의 조건부가 있고 내부에 경로 테이블이 포함된 게시 테이블 1개가 있으면 총 4개 레벨이 됩니다. 마지막 게시 테이블에 다른 조건부 작업을 추가하면 해당 조건부 작업은 표시되지 않습니다.

마친 후에

이 작업의 구성을 완료하면 메시지 흐름에서 작업 추가 및 편집에 설명된 대로 계속해서 다른 작업을 구성하거나 구성을 저장합니다.
