MFL 변환 작업 추가

MFL(Message Format Language) 변환 작업을 사용하여 메시지 파이프 라인에서, XML에서 비XML로 또는 그 반대로 메시지 내용을 변환할 수 있습니다. MFL은 이진 데이터의 레이아웃을 설명하는 데 사용되는 특수 XML 문서입니다. 형식이 있는 이진 데이터를 XML 데이터로 또는 그 반대로 변환하는 규칙을 정의하기 위한 Oracle 전용 언어입니다. MFL을 참조하십시오.

시작하기 전에

이 지침에서는 메시지 흐름 생성, 편집 및 보기에 설명된 것처럼 메시지 흐름 편집 페이지에서 메시지 흐름을 편집하고 있다고 가정합니다.

MFL 변환 작업을 추가하려면 다음을 수행합니다.

  1. 메시지 흐름에서 작업 추가 및 편집에 설명된 대로 작업을 추가할 위치로 이동합니다.
  2. 해당 아이콘을 누른 다음 작업 추가 > 메시지 처리 > MFL 변환을 선택합니다.
  3. MFL 변환 적용 드롭다운 목록에서 요구 사항에 맞게 XML에서 비XML로 변환 또는 비XML에서 XML로 변환을 선택합니다.
  4. <표현식>을 누릅니다. XQuery 표현식 편집기를 사용하여 MFL 변환 작업이 수행될 변수를 지정합니다. 이 입력은 XML로 변환할 때 텍스트 또는 이진이어야 하며 비 XML로 변환할 때 XML이어야 합니다. 메시지 컨텍스트의 이진 내용은 바이너리 내용 XML 요소로 표시됩니다. 이 XML은 입력이 이진을 필요로 할 때 Xquery 표현식의 결과이어야 합니다. 인라인 XQuery 및 XPath 표현식 생성 및 편집을 참조하십시오.
  5. 다음 옵션 중 하나를 선택합니다.
    • MFL 리소스: <리소스> 링크를 누릅니다. MFL 선택 페이지가 표시됩니다. MFL 변환 작업을 수행할 정적 MFL 리소스를 선택합니다.
    • MFL 리소스: 표현식 링크를 누릅니다. XQuery 표현식 편집기 페이지가 표시됩니다. XQuery 표현식 편집기를 사용하여 변환 작업을 수행할 MFL 리소스를 동적으로 지정하는 XQuery 표현식을 project/folder/MFLresourcename 형식으로 작성하거나 편집합니다. 인라인 XQuery 및 XPath 표현식 생성 및 편집을 참조하십시오.
  6. 할당 변수 필드에 이 변환 작업의 결과가 할당될 변수의 이름을 입력합니다. 결과는 binary-content XML 요소가 됩니다.

마친 후에

이 작업의 구성을 완료하면 메시지 흐름에서 작업 추가 및 편집에 설명된 대로 계속해서 다른 작업을 구성하거나 구성을 저장합니다.
