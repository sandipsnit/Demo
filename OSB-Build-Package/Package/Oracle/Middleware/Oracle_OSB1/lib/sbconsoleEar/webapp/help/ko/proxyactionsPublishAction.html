게시 작업 추가

게시 작업을 사용하여 메시지에 대해 정적으로 지정된 대상 서비스를 식별하고 메시지가 패키징되어 서비스로 전송되는 방법을 구성합니다.

게시 동작에 대한 자세한 내용은 Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus(http://www.oracle.com/pls/as111170/lookup?id=OSBAG205)에서 "Performing Transformations in Message Flows"를 참조하십시오.

  1. 메시지 흐름에서 작업 추가 및 편집에 설명된 대로 작업을 추가할 위치로 이동합니다.
  2. 작업 추가 > 통신 > 게시를 선택합니다.
  3. <서비스>를 누릅니다. 서비스 선택 페이지가 표시됩니다.
  4. 목록에서 서비스를 선택하고 제출을 누릅니다. 이는 메시지의 대상 서비스입니다.
  5. 서비스에 정의된 작업이 존재하는 경우 작업 목록에서 선택하여 호출할 작업을 지정할 수 있습니다.
  6. 아웃바운드 작업을 인바운드 작업과 같게 하려면 아웃바운드에 인바운드 작업 사용 체크 박스를 선택합니다.
  7. 메시지가 패키징되어 서비스로 전송되는 방법을 구성하려면 요청 작업 필드에서 작업 추가를 누릅니다. 그런 다음 서비스에 연관시킬 작업을 선택합니다. 둘 이상의 작업을 추가할 수 있습니다. 메시지 흐름에서 작업 추가 및 편집을 참조하십시오.

마친 후에

이 작업의 구성을 완료하면 메시지 흐름에서 작업 추가 및 편집에 설명된 대로 계속해서 다른 작업을 구성하거나 구성을 저장합니다.
