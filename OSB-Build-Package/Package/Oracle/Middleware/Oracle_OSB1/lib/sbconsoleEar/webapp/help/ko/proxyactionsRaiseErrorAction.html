오류 생성 작업 추가

오류 생성 작업을 사용하여 특정 오류 코드(문자열) 및 설명이 지정된 예외 사항을 생성할 수 있습니다.

시작하기 전에

이 지침에서는 메시지 흐름 생성, 편집 및 보기에 설명된 것처럼 메시지 흐름 편집 페이지에서 메시지 흐름을 편집하고 있다고 가정합니다.

오류 생성 작업을 추가하려면 다음을 수행합니다.

  1. 메시지 흐름에서 작업 추가 및 편집에 설명된 대로 작업을 추가할 위치로 이동합니다.
  2. 해당 아이콘을 누른 다음 작업 추가 > 흐름 제어 > 오류 생성을 선택합니다.
  3. 오류 코드 필드에 생성할 오류 코드를 입력합니다.
  4. 오류 메시지 필드에 오류 코드에 대한 설명을 입력합니다.

마친 후에

이 작업의 구성을 완료하면 메시지 흐름에서 작업 추가 및 편집에 설명된 대로 계속해서 다른 작업을 구성하거나 구성을 저장합니다.

트랜잭션

서비스가 트랜잭션인 경우 트리거된 오류 생성 작업으로 인해 요청(비동기) 또는 요청 또는 응답(동기)의 트랜잭션이 중지됩니다. 예를 들어 메시지를 검사하여 SOAP 오류가 발생하지 않는 경우에도 오류 생성 작업이 발생하거나 오류 발생이 트랜잭션을 중지시키는 상황을 판단할 수 있습니다.
