건너뛰기 작업 추가

건너뛰기 작업을 사용하여 런타임 시 이 스테이지의 실행을 건너뛰고 메시지 흐름에서 다음 스테이지로 처리를 진행하도록 지정할 수 있습니다. 이 작업은 매개변수가 없으며 요청, 응답 또는 오류 파이프 라인에서 사용할 수 있습니다.

시작하기 전에

이 지침에서는 메시지 흐름 생성, 편집 및 보기에 설명된 것처럼 메시지 흐름 편집 페이지에서 메시지 흐름을 편집하고 있다고 가정합니다.

건너뛰기 작업을 추가하려면 다음을 수행합니다.

  1. 메시지 흐름에서 작업 추가 및 편집에 설명된 대로 작업을 추가할 위치로 이동합니다.
  2. 해당 아이콘을 누른 다음 작업 추가 > 흐름 제어 > 건너뛰기를 선택합니다.

마친 후에

이 작업의 구성을 완료하면 메시지 흐름에서 작업 추가 및 편집에 설명된 대로 계속해서 다른 작업을 구성하거나 구성을 저장합니다.
