프록시 서비스: XQuery 및 XPath 편집기

프록시 서비스의 메시지 흐름에서 XQuery 표현식을 메시지 컨텍스트 변수에 할당하고, XQuery 표현식의 부울 결과를 기반으로 If...Then... 작업을 할당하고, XPath 표현식에 상대적으로 식별된 위치에 XQuery 표현식의 결과를 삽입하고, 컨텍스트 변수에 대한 XQuery 표현식을 통해 기록할 메시지 내용을 지정할 수 있습니다.

XQuery 표현식 편집기, XQuery 조건 편집기 및 XPath 표현식 편집기는 컨텍스트에서 호출되는 표현식 유형을 생성하기 위해 메시지 흐름의 해당 컨텍스트에서 사용할 수 있습니다.
