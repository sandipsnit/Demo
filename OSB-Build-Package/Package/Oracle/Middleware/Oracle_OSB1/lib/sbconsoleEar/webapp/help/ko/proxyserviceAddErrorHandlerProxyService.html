프록시 서비스 오류 처리기 추가

시작하기 전에

이 지침에서는 메시지 흐름 생성, 편집 및 보기에 설명된 것처럼 메시지 흐름 편집 페이지에서 메시지 흐름을 편집하고 있다고 가정합니다.

프록시 서비스 오류 처리기를 추가하려면 다음을 수행합니다.

  1. 프록시 서비스 아이콘을 누른 다음 서비스 오류 처리기 추가를 누릅니다. 오류 처리기 편집 페이지가 표시됩니다.
  2. 오류 처리기 아이콘을 누른 다음 스테이지 추가를 누릅니다.
  3. 스테이지 아이콘을 누른 다음 스테이지 편집을 누릅니다. 스테이지 구성 편집 페이지가 표시됩니다.
  4. 작업 추가를 누른 다음 추가할 작업을 선택합니다.

    5. 오류 처리기는 파이프 라인이기 때문에 다른 파이프 라인처럼 구성됩니다. 예를 들어, 게시 작업을 사용하여 다른 서비스에 오류 통지를 보내거나 할당 작업을 사용하여 컨텍스트 변수를 수정할 수 있습니다. 메시지 흐름에서 작업 추가 및 편집을 참조하십시오. 연결할 수 있는 작업에는 제한이 없습니다.

    자주 사용되는 세 가지 오류 작업은 오류 생성, 회신 및 재개입니다.

  5. 원하는 경우 스테이지 구성 편집 페이지에서 다른 작업을 추가하고 기타 편집을 수행합니다.
  6. 스테이지 구성 편집 페이지에서 저장을 눌러 현재 세션에서 갱신을 커밋합니다.
  7. 오류 처리기 편집 페이지에서 저장을 눌러 현재 세션에서 갱신을 커밋합니다.

마친 후에

메시지 흐름 편집 페이지에서 메시지 흐름 생성, 편집 및 보기에 설명된 것처럼 메시지 흐름을 계속해서 생성합니다.
