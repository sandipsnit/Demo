편집기 작업 영역 텍스트 필드에서 표현식 작성

XQuery 표현식 편집기, XQuery 조건 편집기 및 XPath 표현식 편집기는 모두 텍스트 필드를 제공하므로 여기에 직접 입력하거나 팔레트에서 항목을 붙여 넣어 표현식을 작성할 수 있습니다.

시작하기 전에

이 지침에서는 인라인 XQuery 및 XPath 표현식 생성 및 편집에 설명된 것처럼 XQuery 표현식 편집기, XQuery 조건 편집기 또는 XPath 표현식 편집기에서 표현식을 생성하거나 편집하고 있다고 가정합니다.

텍스트 필드에서 표현식을 작성하려면:

  1. 텍스트 필드가 포함된 패널을 표시합니다. 편집기에 따라 다음 중 하나를 수행합니다.
    • 아직 선택하지 않은 경우, XQuery 표현식 편집기에서 XQuery 텍스트(작업 영역 단추 모음 아래에 있음)를 선택합니다.
    • 아직 선택하지 않은 경우, XQuery 조건 편집기에서 텍스트(작업 영역 단추 모음 아래에 있음)를 선택합니다.
    • XPath 표현식 편집기에서는 다른 툴을 선택할 수 있는 옵션이 없기 때문에 다른 툴을 선택할 필요가 없습니다.
      • 주: 위의 링크를 선택하면 컨텍스트에 적합한 전체 표현식을 생성할 수 있는 텍스트 필드가 표시됩니다. 그러나 이 항목에서 설명하는 툴 및 방법은 편집기에 텍스트 필드가 제공될 때마다 사용할 수 있습니다. 예를 들면, 외부 XQuery 리소스를 인라인 XQueries에 바인딩 및 외부 XSLT 리소스를 인라인 XQueries에 바인딩에 설명된 것처럼 임포트한 리소스의 변수를 인라인 표현식에 바인딩하는 경우입니다.
  2. 원하는 경우 표현식 또는 표현식 부분을 필드에 입력하거나 붙여 넣습니다. 이러한 방법으로 전체 표현식을 생성하는 경우 아래의 단계 7로 건너뜁니다. 그렇지 않으면 다음 단계를 계속합니다.
  3. 아래의 표 22-1에 설명된 대로 표현식에 추가할 항목이 포함된 팔레트를 선택하고 추가할 항목을 찾습니다.

    4. 표 22-1 팔레트 
    팔레트
    설명 및 사용
    네임스페이스 정의
    기본 Oracle Service Bus 네임스페이스, 변수 네임스페이스 및 사용자 정의 네임스페이스가 나열됩니다. 정의할 때는 네임스페이스 약어가 나열됩니다.
    목록을 스크롤하여 원하는 네임스페이스를 찾습니다.
    네임스페이스도 정의할 수 있습니다. 인라인 표현식에 사용할 네임스페이스 생성를 참조하십시오.
    XQuery 함수
    표준 XQuery 함수 세트와 더불어 사용자가 개발한 함수가 포함되며 알파벳순 및 유형별로 구성됩니다. 더하기 기호(+) 및 빼기 기호(-)를 눌러 트리에서 노드를 확장 또는 축소합니다.
    참고 항목:
    • XQuery 변환
    • http://www.oracle.com/pls/as111170/lookup?id=OSBAG444에서 Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus의 "XQuery Implementation"
    변수 구조
    변수 및 내용이 트리로 표시되므로 시각화에 도움이 됩니다.
    변수 구조 및 내용을 표시하려면 다음을 수행합니다.
    • 팔레트의 맨 위에 있는 드롭다운 목록에서 구조의 이름을 선택합니다. 목록에는 내장 메시지 컨텍스트 변수(attachments, body, header, outbound 및 inbound)와 함께 유형(XML 유형, 서비스 인터페이스 및 단순 유형)별로 구성된 사용자 정의 구조가 표시됩니다.
    • 더하기 기호(+) 및 빼기 기호(-)를 눌러 트리에서 노드를 확장 또는 축소합니다.
    변수 구조를 정의할 수도 있습니다. XQuery 편집기에서 변수 구조 생성를 참조하십시오.
    변수 구조에서 변수를 생성하지 않습니다. 변수는 런타임 시 스테이지의 할당 작업 대상으로 생성됩니다.
    변수 구조 트리의 항목을 텍스트 필드에 삽입하는 경우 경로를 설명하는 XPath 표현식으로 삽입됩니다.
    XQuery 변환을 참조하십시오.

  4. 아래의 표 22-2와 같은 방법을 사용하여 원하는 항목을 텍스트 필드에 붙여 넣습니다.

    5. 표 22-2 항목을 편집기 텍스트 필드에 붙여 넣는 방법
    대상 팔레트
    작업 단계
    네임스페이스 정의 팔레트
    표준 마우스 또는 키보드를 사용하여 네임스페이스를 선택, 복사 및 붙여 넣습니다. 예를 들면 다음과 같습니다.
    1. 마우스 포인터를 문자열 위로 끌어 전체 네임스페이스 문자열(또는 존재하는 경우 약어)을 선택합니다.
    2. 문자열을 복사하려면 Ctrl-C를 누릅니다.
    3. 텍스트 필드에서 네임스페이스를 삽입할 위치를 누릅니다.
    4. 문자열을 복사하려면 Ctrl-V를 누릅니다.
    XQuery 함수 팔레트 또는 변수 구조 팔레트
    팔레트의 항목을 텍스트 필드로 끕니다.
    주: 팔레트에서 작업 영역으로 끌기는 Internet Explorer에서만 지원됩니다.
    XQuery 함수 팔레트 또는 변수 구조 팔레트
    1. 팔레트에서 항목을 누릅니다. 항목이 속성 검사기 창에 표시됩니다.
      • 함수는 제공해야 하는 값에 대한 위치 표시자로 표시됩니다.
      • 변수 및 그 속성이 XPath 표현식으로 표시됩니다.
    2. 텍스트 필드에서 항목을 삽입할 위치를 누릅니다.
    3. 항목을 텍스트 필드에서 선택한 위치에 붙여 넣으려면 속성 복사를 누릅니다.
    XQuery 함수 팔레트 또는 변수 구조 팔레트
    1. 팔레트에서 항목을 누릅니다. 항목이 속성 검사기 창에 표시됩니다.
    2. 표준 키보드 또는 마우스 작업을 사용하여 속성 검사기에서 항목을 선택하고 복사합니다.
    3. 텍스트 필드에서 위치를 선택하고 표준 키보드 또는 마우스 작업을 사용하여 항목을 텍스트 필드에 붙여 넣습니다.

  5. 계속해서 함수를 끌어 놓아 원하는 표현식을 작성합니다.
  6. 필요에 따라 텍스트 필드에서 표현식을 편집합니다.
  7. 선택 사항으로 다음 작업 중 하나 또는 모두를 수행합니다.
    • 검증을 누릅니다. 표현식이 성공적으로 검증되면 메시지가 표시됩니다.
    • 테스트를 눌러 표현식을 테스트합니다. XQuery 테스트 수행을 참조하십시오.
  8. 저장을 눌러 편집기를 닫고 작업의 표현식을 삽입합니다.

마친 후에

메시지 흐름에서 작업 추가 및 편집에 설명된 대로 계속해서 작업을 구성합니다.
