메시지 흐름 생성, 편집 및 보기

  1. 아직 이 작업을 수행하지 않은 경우 생성을 눌러 새 세션을 생성하거나 편집을 눌러 기존 세션을 입력합니다. 변경 센터 사용을 참조하십시오.
  2. 메시지 흐름을 보거나 편집할 프록시 서비스를 찾습니다. 다음 작업을 수행합니다.
    • 프로젝트 탐색기를 선택하여 프로젝트 뷰 페이지 또는 프로젝트/폴더 뷰 페이지를 표시합니다. 그런 다음 프로젝트와 폴더를 탐색하여 프록시 서비스를 찾습니다.
    • 리소스 브라우저 > 프록시 서비스를 선택하여 프록시 서비스 요약 페이지를 표시합니다.
  3. 메시지 흐름을 보거나 편집할 프록시 서비스의 행에서 메시지 흐름 편집 아이콘을 누릅니다.
    • 선택한 프록시 서비스에 대해 메시지 흐름을 생성하지 않은 경우 메시지 흐름 편집 페이지가 페이지의 유일한 아이콘인 프록시 서비스 아이콘과 함께 표시됩니다. 이 노드는 프록시 서비스의 시작 노드입니다. 이 아이콘을 눌러 메시지 흐름 생성을 시작합니다.
    • 프록시 서비스에 이미 메시지 흐름이 존재하는 경우 페이지에 흐름에 대한 그래픽이 표시됩니다. 아이콘을 눌러 메시지 흐름의 일부를 보거나 편집합니다.

      • 해당 페이지의 메시지 흐름을 생성하기 위해 수행할 수 있는 작업에 대한 자세한 내용은 메시지 흐름 편집 페이지를 참조하십시오.

  4. 저장을 눌러 현재 세션에서 갱신을 커밋합니다.
  5. 세션을 종료하고 구성을 런타임에 배치하려면 변경 센터에서 활성화를 누릅니다.
