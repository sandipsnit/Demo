메시지 유형 구성 페이지

프록시 서비스 생성/편집 - 메시지 유형 구성 페이지는 프록시 서비스 생성 및 구성과 프록시 서비스 구성 편집에 설명된 것처럼 프록시 서비스를 생성하고 편집하기 위한 일련의 페이지 중 하나입니다.

이 페이지에서는 유형이 메시징 서비스인 프록시 서비스의 메시지 유형을 구성할 수 있습니다.

메시징 서비스의 바인딩 정의는 교환되는 메시지의 콘텐츠 유형 구성으로 이루어집니다. 응답의 콘텐츠 유형이 요청과 같을 필요는 없으므로 응답은 별도로 구성됩니다(예를 들어, 서비스는 MFL 메시지를 받아서 XML 인증 수신을 반환할 수 있음).

주: 메시징 서비스 유형에 속하는 전자 메일, 파일, FTP 또는 SFTP 전송 프록시 서비스는 단방향 메시징만 지원합니다. 응답 메시지 유형은 없음이어야 합니다. 없음 이외의 옵션을 선택하면 전송 구성 페이지에서 전자 메일, 파일, FTP 또는 SFTP 프로토콜을 사용할 수 없습니다.

표 19-2에서는 프록시 서비스 생성/편집 - 메시지 유형 구성 페이지를 사용하는 방법에 대해 설명합니다.

표 19-2 프록시 서비스 생성/편집 - 메시지 유형 구성 페이지
옵션
생성 또는 편집
요청 메시지 유형
요청 메시지의 메시지 유형을 선택합니다.
  • 없음 - 요청 메시지가 없는 경우 이 옵션을 선택합니다.
  • 이진 - 메시지의 콘텐츠 유형을 알 수 없거나 중요하지 않은 경우 이 옵션을 선택합니다.
  • 텍스트 - 메시지를 텍스트로 제한할 수 존재하는 경우 이 옵션을 선택합니다.
  • MFL - 메시지가 MFL 정의를 준수하는 이진 문서인 경우 이 옵션을 선택합니다. MFL 파일 이름(경로로 정규화됨)을 입력하거나찾아보기를 눌러 파일을 선택합니다.

    • 한 개의 MFL 파일만 구성할 수 있습니다.

    주: 다중 MFL 파일을 지원하려면 이진 또는 텍스트로 내용을 정의하고 메시지 흐름의 MFL 작업을 사용하여 XML로 변환합니다.

  • XML - 메시지가 XML 문서인 경우 이 옵션을 선택합니다. XML 파일 이름(경로로 정규화됨)을 입력하거나찾아보기를 눌러 파일을 선택합니다.

    • 요소 또는 유형 필드에서 변경된 XML 문서의 XML 스키마 유형을 선언하여 일부 유형 정보를 선택적으로 제공합니다.

  • Java - Java 객체가 요청에 전송되는 경우 이 옵션을 선택합니다. JMS 전송이 Java 객체 메시지에 사용됩니다. 자세한 내용은 http://www.oracle.com/pls/as111170/lookup?id=OSBDV1874에서 Oracle Fusion Middleware Developer's Guide for Oracle Service Bus의 "Sending and Receiving Java Objects in Messages"를 참조하십시오.
응답 메시지 유형
응답 메시지의 메시지 유형을 선택합니다.
  • 없음 - 응답 메시지가 없는 경우 이 옵션을 선택합니다.
  • 이진 - 메시지의 콘텐츠 유형을 알 수 없거나 중요하지 않은 경우 이 옵션을 선택합니다.
  • 텍스트 - 메시지를 텍스트로 제한할 수 존재하는 경우 이 옵션을 선택합니다.
  • MFL - 메시지가 MFL 정의를 준수하는 이진 문서인 경우 이 옵션을 선택합니다. MFL 파일 이름(경로로 정규화됨)을 입력하거나찾아보기를 눌러 파일을 선택합니다.

    • 한 개의 MFL 파일만 구성할 수 있습니다.

    주: 다중 MFL 파일을 지원하려면 이진 또는 텍스트로 내용을 정의하고 메시지 흐름의 MFL 작업을 사용하여 XML로 변환합니다.

  • XML - 메시지가 XML 문서인 경우 이 옵션을 선택합니다. XML 파일 이름(경로로 정규화됨)을 입력하거나찾아보기를 눌러 파일을 선택합니다.

    • 요소 또는 유형 필드에서 변경된 XML 문서의 XML 스키마 유형을 선언하여 일부 유형 정보를 선택적으로 제공합니다.

  • Java - Java 객체가 응답에 수신되는 경우 이 옵션을 선택합니다. JMS 전송이 Java 객체 메시지에 사용됩니다. 자세한 내용은 http://www.oracle.com/pls/as111170/lookup?id=OSBDV1874에서 Oracle Fusion Middleware Developer's Guide for Oracle Service Bus의 "Sending and Receiving Java Objects in Messages"를 참조하십시오.
