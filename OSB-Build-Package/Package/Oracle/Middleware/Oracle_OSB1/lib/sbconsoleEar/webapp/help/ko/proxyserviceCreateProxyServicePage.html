프록시 서비스 생성/편집 - 페이지 참조

프로젝트/폴더 뷰 페이지의 리소스 생성 드롭다운 목록에서 프록시 서비스를 선택하여 프록시 서비스를 생성합니다. 이 옵션을 선택하면 프록시 서비스를 구성하고 추가하기 위한 일련의 페이지에 이 옵션이 가장 먼저 표시됩니다. 선택한 옵션에 따라 페이지가 다르게 표시됩니다. 표시되는 페이지는 다음과 같습니다.

  • 일반 구성 페이지
  • 메시지 유형 구성 페이지
  • 전송 구성 페이지
  • 프로토콜별 전송 구성 페이지
  • 작업 선택 구성 페이지
  • 메시지 처리 페이지
  • 요약 페이지
  • 프록시 서비스 정책 페이지
  • 프록시 서비스 보안 페이지
