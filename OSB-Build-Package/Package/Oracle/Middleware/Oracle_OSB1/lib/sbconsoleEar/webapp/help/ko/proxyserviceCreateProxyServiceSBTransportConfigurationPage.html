SB 전송 구성 페이지

프록시 서비스 생성/편집 - SB 전송 구성 페이지는 프록시 서비스 생성 및 구성과 프록시 서비스 구성 편집에 설명된 것처럼 프록시 서비스를 생성하고 편집하기 위한 일련의 페이지 중 하나입니다.

자세한 내용은 http://www.oracle.com/pls/as111170/lookup?id=OSBDV997에서 Oracle Fusion Middleware Developer's Guide for Oracle Service Bus의 "SB Transport"를 참조하십시오.

이 페이지에서는 SB(Service Bus) 전송 프로토콜을 사용하여 프록시 서비스의 전송 설정을 구성할 수 있습니다. 표 19-12에서는 이 페이지를 사용하는 방법에 대해 설명합니다.

표 19-12 프록시 서비스 생성/편집 - SB 전송 구성 페이지
옵션
생성 또는 편집
작업 할당 정책
이 끝점의 작업 할당 정책을 선택하거나 기본 작업 할당 정책을 사용합니다.
작업 할당 정책은 요청을 처리하기 위해 서비스 끝점에 사용할 WLS 작업 관리자의 인스턴스를 참조합니다. 작업 관리자에 대한 자세한 내용은 http://www.oracle.com/pls/as111170/lookup?id=CNFGD112에서 Oracle Fusion Middleware Configuring Server Environments for Oracle WebLogic Server의 "Using Work Managers to Optimize Scheduled Work"를 참조하십시오.
SSL 사용
이 옵션을 지정하면 요청이 SSL 접속을 통해 전송되어야 합니다.
단, 안전하지 않은 경우 접속이 되지 않습니다. 관리자는 안전한 클라이언트만 접속하도록 서버에서 안전하지 않는 모든 프로토콜(예: t3 또는 http)을 차단해야 합니다.

마친 후에

다음을 눌러 다음 페이지에서 이 서비스를 계속 구성하거나 마지막을 눌러 프록시 서비스 생성/편집 - 요약 페이지에서 이 구성을 검토 및 저장합니다.
