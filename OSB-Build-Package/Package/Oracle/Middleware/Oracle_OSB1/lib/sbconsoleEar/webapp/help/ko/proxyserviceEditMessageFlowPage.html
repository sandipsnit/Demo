메시지 흐름 편집 페이지

메시지 흐름 편집 페이지를 사용하여 프록시 서버의 메시지 흐름을 생성할 수 있습니다.

메시지 흐름 편집 페이지의 왼쪽 탐색 창에 메시지 흐름에 있는 노드 및 객체의 트리 뷰가 표시됩니다. 개별 페이지에서 객체의 세부 정보를 정의하면 객체 이름을 눌러 관련 페이지를 표시할 수 있습니다.

오른쪽 창에는 메시지 흐름을 생성하는 필드가 있습니다. 메시지 흐름이 아직 정의되지 않은 경우 창에는 서비스의 시작 노드를 나타내는 프록시 서비스 아이콘이 하나 있습니다. 이 아이콘을 누르면 파이프 라인 쌍 노드, 경로 지정 노드, 조건 분기, 작업 분기 및 서비스의 오류 처리를 추가할 수 있습니다.

페이지에 객체를 추가하면 객체를 나타내기 위해 페이지에 아이콘이 표시됩니다. 객체 간의 관계는 선 및 경계 상자로 표시됩니다. 메시지 흐름 편집 페이지의 아이콘을 누르면 해당 객체에 대해 수행할 수 있는 작업 메뉴가 표시됩니다. 메뉴에서 사용 가능한 옵션은 컨텍스트에 따라 다를 수 있습니다. 전체 아이콘 및 옵션 목록은 표 20-1을 참조하십시오.
