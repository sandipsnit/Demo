스테이지 구성 편집 페이지

스테이지 구성 편집 페이지를 사용하여 메시지 흐름의 파이프 라인 스테이지, 오류 처리기 스테이지 및 경로 지정 노드에 작업을 추가할 수 있습니다.

  • 스테이지 구성 편집 페이지에서 정의된 내용이 없는 경우 표시되는 객체는 작업 추가 아이콘 뿐입니다. 시작하려면 아이콘을 누릅니다.
  • 스테이지 또는 경로 지정 노드가 이미 구성된 경우 해당 스테이지 또는 경로 지정 노드에 대해 정의된 작업 또는 객체가 페이지에 표시됩니다. 스테이지에 작업을 추가하려면 기존 작업을 적절하게 편집하거나 작업을 나타내는 아이콘을 누릅니다.

스테이지에 추가할 수 있는 모든 종류의 작업을 사용하는 작업에 대한 지침은 메시지 흐름에서 작업 추가 및 편집을 참조하십시오.
