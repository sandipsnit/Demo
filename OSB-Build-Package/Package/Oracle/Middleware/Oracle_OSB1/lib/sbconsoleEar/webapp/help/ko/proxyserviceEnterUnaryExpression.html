빌더 옵션을 사용하여 단항 표현식 입력

시작하기 전에

이러한 지침은 인라인 XQuery 및 XPath 표현식 생성 및 편집에서 설명하는 것처럼 사용자가 XQuery 조건 편집기에서 XQuery 조건식을 생성하거나 편집한다고 가정합니다.

XQuery 비교식을 입력하려면 다음을 수행합니다.

  1. 선택한 옵션이 없는 경우 XQuery 조건 편집기에서 작업 영역 단추 모음 아래에 있는 빌더를 선택합니다.
  2. 선택하지 않은 경우 표현식 빌더 상자에서 단항 표현식을 선택합니다.
  3. Not 체크 박스를 선택하여 단항 표현식을 음수로 나타내거나 체크 박스를 선택하지 않습니다.
  4. 표현식 필드에 컨텍스트 변수, 네임스페이스 정의 또는 XQuery 함수를 입력합니다.

    5. 표현식을 작성하려면 XQuery 함수 팔레트에서 XQuery 함수 및 네임스페이스 정의 팔레트에서 네임스페이스를 붙여 넣을 수 있습니다. 편집기 작업 영역 텍스트 필드에서 표현식 작성의 단계 3 및 단계 4를 참조하십시오.

  5. 추가를 눌러 표현식 창에 텍스트를 추가합니다.
  6. 단계 3-5를 반복하여 추가 조건을 작성합니다. 각 조건은 조건 목록 끝에 추가됩니다.

    7. 단항 표현식을 작성하는 경우 다음 사항을 고려합니다.

    • 추가 표현식을 작성할 경우 연결 필드에서 And 또는 Or 옵션을 선택해야 합니다.
    • 조건을 선택하고 위로 화살표를 눌러 조건 목록에서 해당 조건을 위로 이동하거나 아래로 화살표를 눌러 아래로 이동할 수 있습니다. 편집 아이콘을 눌러 갱신하거나 삭제 아이콘을 눌러 삭제할 수 있습니다.
    • 조건의 전체 정의에서 비교식을 사용하여 단항 표현식을 결합할 수 있습니다.
  7. 선택 사항으로 다음 작업 중 하나 또는 모두를 수행합니다.
    • 검증을 누릅니다. 표현식이 성공적으로 검증되면 메시지가 표시됩니다.
    • 테스트를 눌러 표현식을 테스트합니다. XQuery 테스트 수행을 참조하십시오.
  8. 저장을 눌러 편집기를 닫고 작업의 표현식을 삽입합니다.

마친 후에

메시지 흐름에서 작업 추가 및 편집에 설명된 대로 계속해서 작업을 구성합니다.
