프록시 서비스와 관련된 WSDL 익스포트

WSDL 기반 프록시 서비스의 WSDL을 익스포트할 수 있으므로 IDE와 같은 외부 툴에서 WSDL을 보거나 수정할 수 있습니다. WSDL을 JAR 파일로 익스포트합니다.

이 기능은 두 개의 도메인 간 리소스 이동 및 준비에 사용하는 시스템 관리 모듈의 리소스 익스포트 기능과는 다릅니다. 리소스 익스포트를 참조하십시오.

시작하기 전에

사용자가 세션 외부에 존재하는 경우에만 WSDL을 익스포트할 수 있습니다. 변경 센터 사용을 참조하십시오.

WSDL을 익스포트하려면

  1. 다음 작업을 수행합니다.
    • 리소스 브라우저 > 프록시 서비스를 선택하여 프록시 서비스 요약 페이지를 표시합니다.
    • 프로젝트 탐색기를 선택한 다음 WSDL로 익스포트하려는 프록시 서비스를 포함한 프로젝트 또는 폴더를 선택합니다. 프로젝트/폴더 뷰 페이지가 표시됩니다.
  2. WSDL을 익스포트할 프록시 서비스의 행에서 WSDL 익스포트 아이콘을 누릅니다. 익스포트한 JAR 파일을 열거나 저장할지 묻는 대화상자가 표시됩니다.
  3. 대화상자에서 열기를 눌러 파일을 열거나 저장을 눌러 파일을 저장합니다.
