외부 XQuery 리소스를 인라인 XQuery에 바인딩

XQuery 리소스를 인라인 XQuery 표현식에 바인드할 수 있으므로 해당 XQuery 리소스는 작업의 일부로 실행되는 인라인이 됩니다.

시작하기 전에

이 지침에서는 인라인 XQuery 및 XPath 표현식 생성 및 편집에 설명된 것처럼 XQuery 표현식 편집기, XQuery 조건 편집기 또는 XPath 표현식 편집기에서 표현식을 생성하거나 편집하고 있다고 가정합니다.

XQuery 리소스를 인라인 표현식에 바인드하려면:

  1. 변수 구조를 누릅니다.
  2. 단추 모음 아래에 있는 작업 영역에서 XQuery 리소스를 선택합니다.
  3. 1. 실행할 XQuery 리소스를 선택하십시오 상자에서 찾아보기를 누릅니다.
  4. Xquery 브라우저에서 사용할 XQuery와 연관된 라디오 단추를 선택한 다음 제출을 누릅니다.
  5. 2. 바인드 변수 상자에서 변환에 사용할 입력 매개변수를 정의합니다. 변수 이름 아래에 나열된 각 변수에 대해 해당 변수로 매핑할 XQuery 표현식을 입력합니다. 각 매개변수에 대해 매핑을 정의해야 합니다. 예를 들어, XQuery 변환에 one과 two라는 두 개의 입력 매개변수가 있으면 변수 이름 필드에 one과 two라는 두 개의 레이블이 생성됩니다. XQuery 표현식이 입력되는 텍스트 상자는 각 레이블과 연관됩니다.

    6. 다음 XQuery 표현식은 이 필드에 대한 적합한 입력의 예입니다.

    $body/*[1]

    $body/po:PurchaseOrder

    주: 다음 변수 이름은 이 필드에 적합한 입력이 아니며 예외 사항이 발생합니다.
    body
  6. 마친 후에 다음 작업 중 하나를 계속 진행합니다.
    • 검증을 누릅니다. 표현식이 성공적으로 검증되면 메시지가 표시됩니다.
    • 테스트를 누릅니다. XQuery 테스트 수행을 참조하십시오.
    • 변경 사항을 저장하거나 무시합니다.
