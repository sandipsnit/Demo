외부 XSLT 리소스를 인라인 XQuery에 바인딩

XQuery 표현식 편집기 페이지에서 실행할 XSLT 리소스를 선택할 수 있습니다. 이 편집기에 대한 보다 자세한 내용은 인라인 XQuery 및 XPath 표현식 생성 및 편집을 참조하십시오.

실행할 XSLT 리소스를 선택하려면 다음을 수행합니다.

  1. XSLT 리소스 옵션을 선택합니다.
  2. 실행할 XSLT 리소스 선택 필드에서 XSLT 브라우저 아이콘을 선택합니다.
  3. XSLT 브라우저에서 실행할 XSLT와 연관된 라디오 단추를 선택하고 제출을 누릅니다.
  4. 변환의 각 입력 매개변수에 대한 레이블과 해당 텍스트 상자가 바인드 변수 필드 아래에 표시됩니다. 각 레이블은 매개변수의 이름에 해당하고 각 텍스트 상자는 매개변수에 매핑할 XQuery 표현식을 정의하는 데 사용됩니다. 각 매개변수에 대해 매핑을 정의해야 합니다. 예를 들어, XQuery 변환에 one과 two라는 두 개의 입력 매개변수가 있으면 변수 매핑 필드에는 one과 two라는 두 개의 레이블과 함께 XQuery 표현식을 입력할 텍스트 상자가 각각 생성됩니다. 입력 변수에 대한 매핑 외에 변환 입력 문서에 대한 XQuery 표현식도 지정해야 합니다. 매핑은 입력 문서라는 레이블이 있는 텍스트 상자에 지정됩니다.

    5. 다음 XQuery 표현식은 이 필드에 대한 적합한 입력의 예입니다.

    $body/*[1]

    $body/po:PurchaseOrder

    주: 다음 변수 이름은 이 필드에 적합한 입력이 아니며 예외 사항이 발생합니다.
    body
  5. 다음 작업 중 하나를 계속해서 수행합니다.
    • 인라인 표현식에서 사용할 네임스페이스 생성
    • XQuery 편집기에서 변수 구조 생성
    • 편집기 작업 영역 텍스트 필드에서 표현식 생성
    • 외부 Xquery 리소스를 인라인 Xquery에 바인딩
    • 검증을 누릅니다. 표현식이 성공적으로 검증되면 메시지가 표시됩니다.
    • 테스트를 누릅니다. XQuery 테스트 수행을 참조하십시오.
