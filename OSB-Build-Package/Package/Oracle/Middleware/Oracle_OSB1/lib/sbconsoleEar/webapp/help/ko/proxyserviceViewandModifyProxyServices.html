프록시 서비스 구성 편집

이 섹션에서는 프록시 서비스 구성을 편집하는 방법을 설명합니다.

  1. 아직 이 작업을 수행하지 않은 경우 생성을 눌러 새 세션을 생성하거나 편집을 눌러 기존 세션을 입력합니다. 변경 센터 사용을 참조하십시오.
  2. 보거나 편집할 프록시 서비스를 찾습니다. 프록시 서비스 찾기를 참조하십시오.
  3. 프록시 서비스 이름을 누릅니다. 프록시 서비스 보기 - 구성 세부 정보 페이지에는 선택한 프록시 서비스의 구성 정보가 표시됩니다.
  4. 설정을 보거나 수정하려면 다음 중 하나를 수행합니다.
    • 속성을 보거나 편집할 범주 이름 옆의 편집 아이콘을 누릅니다(예: 일반 구성, 전송 구성 등). 편집할 수 있는 페이지는 프록시 서비스를 생성할 때 구성한 페이지에 따라 다릅니다.

      • 이러한 모든 페이지 목록을 보려면 프록시 서비스 보기 - 구성 세부 정보 페이지를 참조하십시오.

    • 페이지 아래쪽에 있는 편집을 눌러 프록시 서비스 생성/편집 - 일반 구성 페이지를 표시합니다. 이 페이지는 이 프록시 서비스를 구성하기 위한 일련의 페이지 중 첫번째 페이지입니다.
  5. 프록시 서비스 생성 및 구성에 설명된 것처럼 프록시 서비스 보기나 편집을 계속합니다.
  6. 프록시 서비스 편집 - 요약 페이지에서 저장을 눌러 현재 세션에서 갱신을 커밋합니다.
  7. 세션을 종료하고 구성을 런타임에 배치하려면 변경 센터에서 활성화를 누릅니다.
