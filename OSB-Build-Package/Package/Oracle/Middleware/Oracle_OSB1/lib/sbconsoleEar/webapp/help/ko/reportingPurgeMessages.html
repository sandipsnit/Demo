메시지 비우기

이 페이지에서는 보고 데이터 저장소에서 선택한 메시지를 비율 수 있습니다.

메시지 비우기는 백그라운드에서 수행되므로 비우기 중에는 메시지 보고서 요약 페이지를 잠글 수 없습니다. 따라서 오랜 시간이 걸리는 비우기 작업을 수행할 수 있으며 비워야 하는 메시지 수에 따라 소요 시간이 달라집니다. 그러나 보고 메시지를 필터링하면 비우기 속도가 느려집니다. 일부 데이터가 아직 비우지 않은 경우 메시지 보고서 요약 페이지에 잘못된 데이터가 표시될 수도 있습니다.

메시지 비우기 기능은 프로젝트의 디자인과 테스트 단계 동안 유용할 수 있습니다. 그러나 운영 환경에서 데이터베이스의 데이터의 관리(비우기 포함)는 데이터베이스 관리 툴을 사용하는 데이터의 관리자가 처리하는 것이 좋습니다.

  1. 작업 > 메시지 비우기를 선택합니다.
  2. 표 27-9에 표시된 다음 단계 중 하나를 수행합니다.

    3. 표 27-9 메시지 비우기 옵션
    수행할 작업
    수행할 단계
    모든 메시지 비우기
    1. 모든 메시지 비우기 라디오 단추를 선택합니다.
    2. 비우기를 누릅니다.
    지정한 시간 프레임 내의 메시지 비우기
    1. 메시지 비우기 시작 날짜... 끝 날짜... 라디오 단추를 선택합니다.
    2. 다음에서 메시지 비우기 필드에서 MM/DD/YY HH:MM:SS 형식으로 월, 일, 연도 및 시간을 입력합니다. 예를 들어 10/10/07 12:45:00 AM을 입력할 수 있습니다.
    3. 메시지 비우기 끝 필드에서 MM/DD/YY HH:MM:SS 형식으로 월, 일, 연도 및 시간을 입력합니다.
    4. 비우기를 누릅니다.

관련 주제

메시지 비우기에 대한 자세한 내용은 http://www.oracle.com/pls/as111170/lookup?id=OSBAG564에서 Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus의 "Reporting"을 참조하십시오.
