리소스 브라우저 사용

리소스 브라우저에서는 리소스를 찾고 해당 리소스에 대해 표시, 편집, 익스포트 및 삭제와 같은 단순한 작업을 수행하는 데 필요한 툴을 제공합니다.

  1. 리소스 브라우저를 표시하려면 왼쪽 탐색 창에서 리소스 브라우저를 선택합니다.
  2. 페이지의 왼쪽에서 찾으려는 리소스 유형의 이름을 누릅니다. 표 3-10에는 선택할 수 있는 항목과 각 항목에 대해 표시된 페이지와 관련된 작업이 표시됩니다.
