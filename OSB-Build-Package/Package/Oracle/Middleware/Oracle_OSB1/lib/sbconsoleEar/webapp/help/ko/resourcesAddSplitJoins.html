분할-조인 추가

시작하기 전에

분할-조인에는 Eclipse용 Oracle Service Bus 플러그인을 사용하여 분할-조인을 생성하거나 분할-조인 구성 JAR을 플러그인에 임포트할 때 생성된 .flow 파일이 필요합니다. .flow 파일에 프록시 서비스, 비즈니스 서비스 또는 외부 WSDL 리소스에 대한 참조가 있을 수 있습니다. 분할-조인을 활성화하기 전에 분할-조인이 사용하는 리소스가 서버에 있어야 합니다.

  1. 아직 이 작업을 수행하지 않은 경우 생성을 눌러 새 세션을 생성하거나 편집을 눌러 기존 세션을 입력합니다. 변경 센터 사용을 참조하십시오.
  2. 프로젝트 탐색기를 선택한 다음 분할-조인을 추가할 프로젝트나 폴더를 선택합니다. 프로젝트/폴더 뷰 페이지가 표시됩니다.
  3. 리소스 생성 드롭다운 목록에서 분할-조인을 선택하여 새 분할-조인 리소스 생성 페이지를 표시합니다.

    4. 표 9-2에 표시된 매개변수를 지정합니다.

    표 9-2 분할-조인 매개변수
    속성
    생성 또는 편집
    리소스 이름
    이 분할-조인에 대한 고유 이름을 입력합니다.

    리소스 이름 지정 제한 사항의 이름 지정 지침을 따르십시오.
    리소스 설명
    분할-조인의 대상을 입력합니다.
    분할-조인
    찾아보기를 눌러 시스템에서 .flow 파일(일반적으로 MW_HOME\user_projects\workspaces\your_workspace에 있음)을 찾습니다.
    작업 할당 정책
    스레드(parallel 분기)를 생성하는 데 사용된 WebLogic Server 작업 관리자 인스턴스를 선택하여 분할-조인을 실행합니다. 다른 작업 관리자가 존재하지 않으면 기본 작업 관리자가 사용됩니다.

  4. 저장을 누릅니다. 분할-조인이 생성되고 현재 세션에 저장됩니다.
    5. 주: 분할-조인을 활성화하기 전에 분할-조인이 사용하는 리소스가 있어야 합니다.
  5. 세션을 종료하고 구성을 런타임에 배치하려면 변경 센터에서 활성화를 누릅니다.
