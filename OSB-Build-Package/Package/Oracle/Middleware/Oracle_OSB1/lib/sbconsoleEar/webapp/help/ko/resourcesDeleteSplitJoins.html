분할-조인 삭제

  1. 아직 이 작업을 수행하지 않은 경우 생성을 눌러 새 세션을 생성하거나 편집을 눌러 기존 세션을 입력합니다. 변경 센터 사용을 참조하십시오.
  2. 리소스 브라우저 > 분할-조인을 선택하여 분할-조인 요약 페이지를 표시합니다.
  3. 삭제할 분할-조인의 옵션 필드에서 삭제 아이콘을 선택합니다. 현재 세션에서 분할-조인이 삭제됩니다. 다른 리소스에서 이 리소스를 참조하는 경우 삭제 경고 아이콘이 표시됩니다. 경고 확인을 통해 리소스를 삭제할 수 있습니다. 이 경우 삭제된 리소스에 대해 해결되지 않은 참조로 인해 충돌이 발생할 수 있습니다.
  4. 세션을 종료하고 구성을 런타임에 배치하려면 변경 센터에서 활성화를 누릅니다.
