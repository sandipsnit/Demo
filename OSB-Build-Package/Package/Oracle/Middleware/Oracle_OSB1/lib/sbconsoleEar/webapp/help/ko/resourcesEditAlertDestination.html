경보 대상 편집

  1. 아직 이 작업을 수행하지 않은 경우 생성을 눌러 새 세션을 생성하거나 편집을 눌러 기존 세션을 입력합니다. 변경 센터 사용을 참조하십시오.
  2. 경보 대상 찾기 또는 경보 대상 검색에서 설명된 대로 경보 대상을 찾습니다.
  3. 경보 대상의 이름을 누릅니다. 경보 대상 보기 - 구성 페이지에는 표 5-2에 나열된 정보가 표시됩니다.

    4. 표 5-2 경보 대상 구성 세부 정보 
    속성
    설명
    마지막으로 수정한 사람
    이 경보 대상을 생성하거나 편집한 사용자입니다.
    마지막 수정 시간
    사용자가 이 경보 대상을 생성하거나 편집한 날짜와 시간입니다. 날짜 및 시간 링크를 눌러 이 리소스의 변경 내역을 봅니다. 변경 내역 보기 페이지를 참조하십시오.
    참조
    이 경보 대상에서 참조하는 객체의 수입니다. 해당 참조가 존재하는 경우 숫자 링크를 누르면 객체 목록이 표시됩니다. 리소스에 대한 참조 보기를 참조하십시오.
    참조자
    이 경보 대상을 참조하는 객체의 수입니다. 해당 참조가 존재하는 경우 숫자 링크를 누르면 객체 목록이 표시됩니다. 리소스에 대한 참조 보기를 참조하십시오.
    설명
    이 경보 대상에 대한 설명입니다(존재하는 경우).
    SNMP 트랩
    SNMP 트랩 옵션을 구성할 때 이 경보 대상으로 SNMP 트랩 옵션을 지정했는지 여부를 [예] 또는 [아니오]로 나타냅니다.
    보고
    보고 옵션을 구성할 때 이 경보 대상으로 보고 옵션을 지정했는지 여부를 [예] 또는 [아니오]로 나타냅니다.
    경보 로깅
    경보 로깅이 사용으로 설정되어 있는지 여부를 [예] 또는 [아니오]로 나타냅니다.
    전자 메일 및 JMS
    전자 메일 받는 사람 목록이나 JMS 대상 목록을 표시하고 둘 다 구성된 경우 모두 표시합니다.

  4. 선택한 [SNMP 트랩], [보고] 또는 [경보 로깅] 설정을 변경할 수 있습니다.
  5. 전자 메일 받는 사람 또는 JMS 대상을 편집하거나 삭제하려면 페이지 하단의 편집을 누릅니다.
    1. 전자 메일 받는 사람을 편집하려면 전자 메일의 옵션에서 편집할 전자 메일 받는 사람의 편집 아이콘을 누릅니다. 전자 메일 받는 사람 편집 페이지에서는 전자 메일 받는 사람 정보를 변경할 수 있습니다. 전자 메일 받는 사람 추가를 참조하십시오.
    2. JMS 대상을 편집하려면 JMS의 옵션에서 편집할 JMS 대상의편집 아이콘을 누릅니다. JMS 대상 편집 페이지에서는 JMS 대상 정보를 변경할 수 있습니다. JMS 대상 추가를 참조하십시오.
    3. 전자 메일 받는 사람 또는 JMS 대상을 하나 또는 모두 삭제하려면 옵션에서 삭제할 전자 메일 받는 사람 또는 JMS 대상의 삭제 아이콘을 누릅니다.
  6. 저장을 눌러 현재 세션에서 갱신을 커밋합니다.
  7. 세션을 종료하고 구성을 런타임에 배치하려면 변경 센터에서 활성화를 누릅니다.
