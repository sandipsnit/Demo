JAR 종속성 편집

Oracle Service Bus에서 JAR 리소스에는 여러 JAR의 다른 클래스에 종속된 클래스가 포함될 수 있습니다. JAR 종속성 편집 페이지를 사용하여 참조된 JAR 리소스를 추가 또는 제거합니다.

  1. 아직 이 작업을 수행하지 않은 경우 생성을 눌러 새 세션을 생성하거나 편집을 눌러 기존 세션을 입력합니다. 변경 센터 사용을 참조하십시오.
  2. JAR 찾기에 설명된 대로 JAR을 찾습니다.
  3. JAR 이름을 누릅니다.

    4. JAR 리소스 보기 - JAR 구성 페이지에 있는 종속성 탭의 숫자 표시기에는 이 JAR 리소스에 대해 구성된 참조 JAR 수가 표시되거나 참조 JAR이 없는 경우 0이 표시됩니다.

  4. 종속성 탭을 선택하여 참조된 JAR을 표시하거나 추가합니다.

    5. JAR 종속성 페이지에는 표 7-4에 표시된 정보가 있습니다.

    표 7-4 JAR 종속성 정보
    속성
    설명
    이름
    참조된 JAR의 이름입니다. 이름은 JAR 리소스 보기 - JAR 구성 페이지에 대한 링크입니다. JAR 편집을 참조하십시오.
    경로
    경로는 JAR이 저장된 폴더 이름과 프로젝트 이름입니다.

  5. 참조된 JAR을 추가하거나 제거하려면 편집을 누릅니다.

    6. JAR 종속성 편집 페이지에는 표 7-5에 표시된 정보가 있습니다.

    표 7-5 JAR 종속성 세부 정보
    속성
    설명
    Manifest 클래스 경로
    JAR Manifest 파일의 클래스 경로 항목입니다.
    이름
    참조된 JAR 리소스의 이름 및 경로입니다.
    옵션
    • 위로 및 아래로 화살표를 사용하여 참조된 JAR의 우선순위를 변경합니다.
    • 삭제 아이콘을 눌러 참조된 JAR을 제거합니다.

  6. JAR 추가를 누릅니다.
  7. JAR 브라우저에서 참조된 JAR 리소스를 찾거나 선택하고 제출을 누릅니다.

    8. 예를 들어, JAR A를 편집하고 JAR B 및 C를 추가하면 JAR A 참조는 JAR B 및 JAR C를 의존합니다.

    옵션 열에서 화살표를 사용하여 참조된 JAR 순서를 변경하거나 삭제 아이콘을 사용하여 참조된 JAR을 제거합니다.

  8. 저장을 눌러 현재 세션에서 갱신을 커밋합니다.
  9. 세션을 종료하고 구성을 런타임에 배치하려면 변경 센터에서 활성화를 누릅니다.
