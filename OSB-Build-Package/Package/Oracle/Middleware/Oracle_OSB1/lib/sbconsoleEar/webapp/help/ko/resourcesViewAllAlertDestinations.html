경보 대상 찾기

경보 대상 요약 페이지에서는 경보 대상 목록을 볼 수 있습니다. 아직 경보 대상을 생성하지 않은 상태에서 경보 대상을 생성하려면 프로젝트 탐색기에서 새 경보 대상 리소스를 생성합니다. 생성하는 방법은 경보 대상 추가를 참조하십시오.

  1. 다음 작업을 수행합니다.
    • 프로젝트 탐색기를 선택하여 프로젝트 뷰 페이지 또는 프로젝트/폴더 뷰 페이지를 표시합니다. 그런 다음 프로젝트와 폴더를 탐색하여 경보 대상을 찾습니다.
    • 리소스 브라우저 > 경보 대상을 선택합니다. 경보 대상 요약 페이지에는 표 5-1에 나열된 정보가 표시됩니다.
  2. 목록에서 항목 수를 제한하기 위해 이름, 경로 또는 이 모두를 사용하여 필터링할 수 있습니다. 이름 및 경로 필드에 검색 대상의 이름과 경로를 입력한 다음 검색을 누릅니다.

    3. 경보 대상이 있는 폴더 이름과 프로젝트 이름입니다.

    모두 보기를 눌러 검색 필터를 제거하고 모든 경보 대상을 표시합니다.

    표 5-1 경보 대상 세부 정보
    속성
    설명
    이름
    경보 대상 리소스의 이름입니다. 이름은 경보 대상 보기 - 구성 페이지에 대한 링크입니다. 경보 대상 편집을 참조하십시오.
    경로
    경보 대상이 있는 폴더 이름과 프로젝트 이름입니다. 경로는 이 리소스가 포함된 프로젝트 또는 폴더에 대한 링크입니다. 프로젝트 및 폴더를 사용하여 리소스 이름 식별을 참조하십시오.
    옵션
    삭제 아이콘을 눌러 특정 경보 대상을 삭제합니다. 경보 대상 삭제를 참조하십시오.
