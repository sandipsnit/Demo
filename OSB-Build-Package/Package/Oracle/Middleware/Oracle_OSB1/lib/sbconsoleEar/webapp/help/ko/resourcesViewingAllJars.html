JAR 찾기

  1. 다음 작업을 수행합니다.
    • 프로젝트 탐색기를 선택하여 프로젝트 뷰 페이지 또는 프로젝트/폴더 뷰 페이지를 표시합니다. 그런 다음 프로젝트와 폴더를 탐색하여 JAR을 찾습니다.
    • 리소스 브라우저 > JAR을 선택합니다. JAR 요약 페이지에는 표 7-1에 나열된 정보가 표시됩니다. 속성에 대한 자세한 내용은 JAR 편집을 참조하십시오.
  2. 목록에 표시되는 항목 수를 제한하거나 특정 JAR을 찾으려면 이름 및 경로 필터를 사용하여 JAR을 검색합니다. 이름 및 경로 필드에 검색 대상의 이름과 경로를 입력한 다음 검색을 누릅니다.

    3. 경로는 JAR이 저장된 폴더 이름과 프로젝트 이름입니다.

    이 검색은 와일드 카드 문자를 지원합니다. 예를 들어 이름 필드에 “j*”를 입력하면 "j"로 시작하는 JAR 목록을 모두 임포트할 수 있습니다.

    모두 보기를 눌러 검색 필터를 제거하고 모든 JAR을 표시합니다.

    표 7-1 JAR 세부 정보
    속성
    설명
    JAR 이름
    JAR에 할당된 고유 이름입니다. 이름은 JAR 리소스 보기 - JAR 구성 페이지에 대한 링크입니다. JAR 편집을 참조하십시오.
    경로
    경로는 JAR이 저장된 폴더 이름과 프로젝트 이름입니다. 경로는 이 리소스가 포함된 프로젝트 또는 폴더에 대한 링크입니다. 프로젝트 및 폴더를 사용하여 리소스 이름 식별을 참조하십시오.
    클래스 수
    JAR에 포함된 클래스의 수입니다.
    크기
    바이트로 표시된 JAR의 크기입니다.
    옵션
    삭제 아이콘을 눌러 특정 JAR을 삭제합니다. JAR 삭제를 참조하십시오.
