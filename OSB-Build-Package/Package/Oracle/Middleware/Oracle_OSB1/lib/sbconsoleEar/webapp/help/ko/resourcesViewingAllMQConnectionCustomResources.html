MQ 접속 찾기

  1. 다음 작업을 수행합니다.
    • 프로젝트 탐색기를 선택하여 프로젝트 뷰 페이지 또는 프로젝트/폴더 뷰 페이지를 표시합니다. 그런 다음 프로젝트와 폴더를 탐색하여 MQ 접속 리소스를 찾습니다.
    • 리소스 브라우저 > MQ Connections를 선택합니다. MQ 접속 요약 페이지에는 표 8-1에 나열된 정보가 표시됩니다. 속성에 대한 자세한 내용은 MQ 접속 편집을 참조하십시오.
  2. 목록에서 항목 수를 제한하기 위해 이름, 경로 또는 이 모두를 사용하여 필터링할 수 있습니다. 이름 및 경로 필드에 검색 대상의 이름과 경로를 입력한 다음 검색을 누릅니다.
  3. 열기 아이콘을 눌러 검색 필터를 추가로 표시합니다. 다중 검색 조건을 지정하는 경우에는 모든 조건과 일치하는 결과만 반환됩니다.
    • 대기열 관리자 이름을 기준으로 필터링합니다. 와일드 카드 문자를 포함한 임의의 문자열을 입력한 다음 검색을 누릅니다.
    • Queue Manager Channel Name을 기준으로 필터링합니다. 와일드 카드 문자를 포함한 임의의 문자열을 입력한 다음 검색을 누릅니다.
    • 모두 보기를 눌러 검색 필터를 제거하고 모든 MQ 접속 리소스를 표시합니다.

      • 표 8-1 MQ 접속 세부 정보
      속성
      설명
      MQ 접속 이름
      MQ 접속 리소스에 할당된 이름입니다. 이 이름은 MQ 접속 구성 페이지에 대한 링크입니다. MQ 접속 편집을 참조하십시오.
      경로
      MQ 접속 리소스가 있는 폴더 이름과 프로젝트 이름입니다. 경로는 이 리소스가 포함된 프로젝트 또는 폴더에 대한 링크입니다. 프로젝트 및 폴더를 사용하여 리소스 이름 식별을 참조하십시오.
      옵션
      삭제 아이콘을 눌러 특정 MQ 접속 리소스를 삭제합니다. MQ 접속 삭제를 참조하십시오.
