보안 구성

보안 구성 모듈을 사용하여 Oracle Service Bus 인바운드 보안 및 관리 보안에서 사용되는 보안 데이터를 생성하고 수정합니다.

인바운드 전송 레벨 보안 및 메시지 레벨 보안은 사용자, 그룹 및 롤 데이터를 사용하여 인바운드 클라이언트 요청을 인증합니다. 액세스 제어 정책을 적용하여 프록시 서비스 및 비즈니스 서비스를 사용하도록 인증받은 사용자를 결정합니다.

관리 보안은 사용자, 그룹 및 롤 데이터를 사용하여 Oracle Service Bus 구성 데이터를 생성 또는 수정하거나 Oracle Service Bus 성능을 모니터하도록 인증 받은 사용자를 결정합니다. 자세한 내용은 사용자, 그룹, 보안 롤 및 정책 이해를 참조하십시오. 또한 http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539에서 Oracle Fusion Middleware Developer's Guide for Oracle Service Bus의 "Configuring Administrative Security"를 참조하십시오.

주: 사용자, 그룹 또는 롤은 보안 제공자 저장소에 있으므로 구성을 익스포트할 때 이러한 객체는 익스포트할 수 없습니다. 익스포트한 구성을 임포트할 때 해당 객체를 다시 생성하거나, 사용 가능한 경우 특정 WebLogic Server 툴을 사용하여 이러한 객체를 익스포트하고 임포트해야 합니다.
