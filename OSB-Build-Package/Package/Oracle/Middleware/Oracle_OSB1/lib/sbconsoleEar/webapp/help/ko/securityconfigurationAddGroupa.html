그룹 편집

그룹 세부 정보 보기 페이지를 사용하여 특정 그룹의 세부 정보를 보고 변경할 수 있습니다.

  1. WebLogic Server 관리 권한을 가진 사용자로 Oracle Service Bus 관리 콘솔에 로그인합니다. 관리 롤을 가진 사용자만 보안 구성 데이터를 수정할 수 있습니다. http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539에서 Oracle Fusion Middleware Developer's Guide for Oracle Service Bus의 "Configuring Administrative Security"를 참조하십시오.
  2. 그룹을 찾습니다. 그룹 찾기를 참조하십시오.
  3. 그룹 이름을 누릅니다. 그러면 그룹 세부 정보 보기 페이지에 표 24-4와 같은 정보가 표시됩니다.

    4. 표 24-4 그룹 세부 정보
    속성
    설명
    그룹 이름
    이 그룹의 이름입니다.
    인증 제공자
    이 그룹 정의를 포함하는 인증 제공자입니다.
    그룹
    이 그룹이 속한 그룹입니다.

  4. 그룹 세부 정보를 편집하려면 재구성을 눌러 그룹 세부 정보 편집 페이지를 표시합니다.

    5. 세션 내부 또는 외부에서 그룹 세부 정보를 편집할 수 있습니다.

  5. 그룹 멤버쉽 필드를 적절하게 변경합니다. 필드에 대한 설명은 그룹 추가를 참조하십시오.

    6. 그룹 이름 필드는 변경할 수 없습니다.

  6. 변경 내용 저장을 눌러 그룹을 갱신합니다.

    7. Oracle Service Bus 관리 콘솔은 그룹 세부 정보를 갱신하며, 갱신된 내용은 모든 세션에서 즉시 사용할 수 있게 됩니다. 그룹을 갱신할 때 한 세션을 사용한 다음 해당 세션을 폐기하면 Oracle Service Bus 관리 콘솔은 갱신 내용을 삭제하지 않습니다.
