롤 추가

  1. WebLogic Server 관리 권한을 가진 사용자로 Oracle Service Bus 관리 콘솔에 로그인합니다. 관리 롤을 가진 사용자만 보안 구성 데이터를 수정할 수 있습니다. http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539에서 Oracle Fusion Middleware Developer's Guide for Oracle Service Bus의 "Configuring Administrative Security"를 참조하십시오.
  2. 보안 구성 > 롤을 선택하여 전역 롤 페이지를 표시합니다.
  3. 새로 만들기를 누릅니다.

    4. 세션 내부나 외부에서 롤을 추가할 수 있습니다.

  4. 롤 이름 필드에 고유 이름을 입력합니다. 이 필드는 필수 필드입니다.

    5. 보안 롤 이름에 공백이나 < > 문자를 사용하지 마십시오. 보안 롤 이름은 대소문자를 구분합니다. Oracle 규칙에 따르면 보안 롤 이름은 모두 단수입니다.

  5. 롤을 생성하려면 확인을 누릅니다.

    6. Oracle Service Bus 관리 콘솔이 롤을 저장하며, 저장된 롤은 모든 세션에서 즉시 사용할 수 있게 됩니다. 롤을 선택할 때 한 세션을 사용한 다음 해당 세션을 폐기하면 Oracle Service Bus 관리 콘솔은 새 롤을 삭제하지 않습니다.

    확인을 눌러 롤을 생성하는 경우 다음 단계는 롤이 적용되는 조건을 정의하는 것입니다.

  6. 롤 조건 정의에서 계속 진행합니다.
