그룹 삭제

그룹 요약 페이지에서 선택한 그룹을 한 개 이상 삭제할 수 있습니다.

  1. WebLogic Server 관리 권한을 가진 사용자로 Oracle Service Bus 관리 콘솔에 로그인합니다. 관리 롤을 가진 사용자만 보안 구성 데이터를 수정할 수 있습니다. http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539에서 Oracle Fusion Middleware Developer's Guide for Oracle Service Bus의 "Configuring Administrative Security"를 참조하십시오.
  2. 보안 구성을 선택하여 그룹 요약 페이지를 표시합니다.
  3. 삭제할 그룹을 선택합니다. 필요한 경우 다중 그룹을 선택할 수 있습니다.

    4. 세션 내부 또는 외부에서 그룹을 삭제할 수 있습니다.

  4. 삭제를 누릅니다. 그룹을 삭제할 것인지 확인하는 메시지가 표시됩니다.
  5. 그룹을 삭제하려면 확인을 누릅니다.

    6. Oracle Service Bus 관리 콘솔이 그룹을 삭제합니다. 그룹을 삭제할 때 한 세션을 사용한 다음 해당 세션을 폐기하면 Oracle Service Bus 관리 콘솔은 그룹을 삭제 취소하지 않습니다.

  6. 또는 삭제할 그룹의 옵션 열에 있는 삭제 아이콘을 누를 수도 있습니다.
