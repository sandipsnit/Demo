액세스 제어 정책 찾기

보안 페이지는 현재 Oracle Service Bus 도메인에서 프록시 서비스의 액세스 제어 정책에 대한 링크를 제공합니다.

세션에서 생성했지만 아직 활성화되지 않은 프록시 서비스는 이 페이지 목록에 나열되지 않습니다. 새 프록시 서비스에 대한 액세스 제어 정책을 편집하려면 먼저 프록시 서비스를 생성한 세션을 활성화합니다.

  1. 새 프록시 서비스에 대한 액세스 제어 정책을 찾으려면 프록시 서비스를 생성한 세션을 활성화합니다.
  2. 액세스 제어 열의 전송 액세스 제어에서 프록시 서비스 이름을 선택하거나 메시지 액세스 제어에서 프록시 서비스 이름 또는 특정 작업을 선택합니다.
