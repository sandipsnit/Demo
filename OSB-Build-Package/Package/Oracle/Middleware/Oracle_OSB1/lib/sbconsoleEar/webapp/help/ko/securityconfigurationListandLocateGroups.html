그룹 찾기

  1. 보안 구성 > 그룹을 선택합니다. 그러면 그룹 요약 페이지에 표 24-3과 같은 정보가 표시됩니다. 속성에 대한 자세한 설명은 그룹 편집을 참조하십시오.
  2. 목록에 표시되는 항목 수를 제한하거나 특정 그룹을 찾으려면 그룹 이름으로 필터링할 수 있습니다. 이름 필드에 그룹 이름의 일부 또는 전체를 입력한 다음 검색을 누릅니다.

    3. 별표(*) 와일드 카드 문자를 사용할 수 있습니다. (다른 와일드 카드 문자는 지원되지 않습니다.)

    모두 보기를 눌러 검색 필터를 제거하고 모든 그룹을 표시합니다.

    표 24-3 그룹 정보 
    속성
    설명
    그룹 이름
    그룹의 이름입니다. 이 이름은 그룹 세부 정보 보기 페이지에 대한 링크입니다. 그룹 편집을 참조하십시오.
    그룹 멤버쉽
    이 그룹이 속한 그룹입니다. 이 이름은 그룹 세부 정보 보기 페이지에 대한 링크입니다. 그룹 편집을 참조하십시오.
    인증 제공자
    이 그룹 정의를 포함하는 인증 제공자입니다.
    삭제
    삭제 아이콘을 눌러 특정 그룹을 삭제합니다. 그룹 삭제를 참조하십시오.
