롤 찾기

  1. 보안 구성 > 롤을 선택합니다. 그러면 전역 롤 페이지에 표 24-5와 같은 정보가 표시됩니다. 속성에 대한 자세한 설명은 롤 편집을 참조하십시오.
  2. 목록에 표시되는 항목 수를 제한하거나 특정 롤을 찾으려면 페이지를 스크롤합니다. 테이블의 위 또는 아래에 있는 페이지 콘트롤을 사용합니다.

    3. 표 24-5 롤 정보
    속성
    설명
    롤 이름
    롤의 이름입니다. 이 이름은 롤 세부 정보 보기 페이지에 대한 링크입니다. 롤 편집을 참조하십시오.
    제공자 이름
    이 그룹의 인증 제공자입니다.
