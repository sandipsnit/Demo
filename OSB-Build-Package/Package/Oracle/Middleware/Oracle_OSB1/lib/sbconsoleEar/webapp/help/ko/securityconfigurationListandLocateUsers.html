사용자 찾기

  1. 보안 구성 > 사용자를 선택합니다. 그러면 사용자 요약 페이지에 표 24-1과 같은 정보가 표시됩니다. 속성에 대한 자세한 설명은 사용자 편집을 참조하십시오.
  2. 목록에 표시되는 항목 수를 제한하거나 특정 사용자를 찾으려면 사용자 이름으로 필터링할 수 있습니다. 이름 필드에 사용자 이름의 일부 또는 전체를 입력한 다음 검색을 누릅니다.

    3. 별표(*) 와일드 카드 문자를 사용할 수 있습니다. (다른 와일드 카드 문자는 지원되지 않습니다.)

    모두 보기를 눌러 검색 필터를 제거하고 모든 사용자를 표시합니다.

    표 24-1 사용자 정보
    속성
    설명
    사용자 이름
    사용자에 할당된 이름입니다. 이름은 사용자 세부 정보 보기 페이지에 대한 링크입니다. 사용자 편집을 참조하십시오.
    그룹 멤버쉽
    이 사용자가 속한 그룹의 이름입니다. 이 이름은 그룹 세부 정보 보기 페이지에 대한 링크입니다. 그룹 편집을 참조하십시오.
    인증 제공자
    이 사용자의 인증 제공자입니다.
    옵션
    삭제 아이콘을 눌러 특정 사용자를 삭제합니다. 사용자 삭제를 참조하십시오.
