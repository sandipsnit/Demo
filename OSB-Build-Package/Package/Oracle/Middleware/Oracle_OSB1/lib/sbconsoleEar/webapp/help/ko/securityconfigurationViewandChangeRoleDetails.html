롤 편집

롤 세부 정보 보기 페이지를 사용하여 특정 롤의 세부 정보를 보고 변경할 수 있습니다.

  1. WebLogic Server 관리 권한을 가진 사용자로 Oracle Service Bus 관리 콘솔에 로그인합니다. 관리 롤을 가진 사용자만 보안 구성 데이터를 수정할 수 있습니다. http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539에서 Oracle Fusion Middleware Developer's Guide for Oracle Service Bus의 "Configuring Administrative Security"를 참조하십시오.
  2. 롤을 찾습니다. 롤 찾기를 참조하십시오.
  3. 롤 이름을 누릅니다. 롤 세부 정보 보기 페이지에는 표 24-8에 나열된 정보가 표시됩니다.

    4. 표 24-8 롤 세부 정보
    속성
    설명
    이름
    롤의 이름입니다.
    롤 조건
    이 롤의 멤버쉽을 결정하는 조건입니다.

    세션 내부 또는 외부에서 롤 세부 정보를 변경할 수 있습니다.

  4. 표 24-9와 같은 단계 중 하나를 수행합니다.

    5. 표 24-9 롤 옵션
    수행할 작업
    작업 단계
    선택한 표현식의 순서를 변경합니다.
    위로 이동 및 아래로 이동을 누릅니다.
    롤 조건을 병합하거나 병합을 취소하고 강조 표시된 and 및 or 문을 표현식 간에 전환합니다.
    결합 및 결합 취소를 누릅니다.
    조건을 부정으로 생성합니다. 예를 들어, 연산자 그룹 아님은 롤에서 연산자 그룹을 제외합니다.
    무효화를 누릅니다.
    선택한 표현식을 삭제합니다.
    제거를 누릅니다.

  5. 저장을 누릅니다.

    6. Oracle Service Bus 관리 콘솔은 롤을 갱신하며, 갱신된 내용은 모든 세션에서 즉시 사용할 수 있게 됩니다. 롤을 갱신할 때 한 세션을 사용한 다음 해당 세션을 폐기하면 Oracle Service Bus 관리 콘솔은 갱신 내용을 삭제하지 않습니다.
