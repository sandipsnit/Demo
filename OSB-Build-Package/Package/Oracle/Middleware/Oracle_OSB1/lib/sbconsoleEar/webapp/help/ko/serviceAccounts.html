서비스 계정

서비스 계정은 프록시 서비스와 비즈니스 서비스가 아웃바운드 인증 또는 FTP 서버나 JMS 서버와 같은 로컬이나 원격 리소스의 인증에 사용하는 사용자 이름과 비밀번호를 제공합니다. 예를 들어, 비즈니스 서비스가 웹 서비스를 사용한 전송 레벨 인증을 위한 사용자 이름과 비밀번호를 제공해야 한다면 사용자 이름과 비밀번호를 지정하는 서비스 계정을 생성한 다음 아웃바운드 요청에 서비스-계정 인증서를 포함하도록 비즈니스 서비스를 구성합니다.

서비스 계정에 입력한 사용자 이름과 비밀번호는 아웃바운드 인증에 사용되거나 로컬이나 원격 리소스에 인증서를 제공하는 데 사용됩니다. Oracle Service Bus 관리 콘솔의 보안 구성 모듈에 입력한 사용자 이름과 비밀번호는 인바운드 인증과 관리 요청 인증에 사용됩니다. 서비스 계정 지정을 참조하십시오.
