서비스 계정 삭제

서비스 계정을 삭제하면 해당 서비스 계정에 포함된 사용자 이름, 비밀번호 및 로컬 사용자와 원격 사용자 간의 매핑 데이터도 함께 삭제됩니다.

  1. 아직 이 작업을 수행하지 않은 경우 생성을 눌러 새 세션을 생성하거나 편집을 눌러 기존 세션을 입력합니다. 변경 센터 사용을 참조하십시오.
  2. 비즈니스 서비스나 프록시 서비스가 서비스 계정을 사용하도록 구성되었다면 비즈니스 서비스 또는 프록시 서비스에서 해당 서비스 계정을 제거합니다.

    3. 비즈니스 서비스 구성 편집 또는 프록시 서비스 구성 편집을 참조하십시오.

  3. 리소스 브라우저 > 서비스 계정을 선택하여 서비스 계정 요약 페이지를 표시합니다.
  4. 삭제할 서비스 계정의 옵션 필드에서 삭제 아이콘을 누릅니다.

    5. 현재 세션에서 서비스 계정이 삭제됩니다. 서비스 계정을 사용하도록 비즈니스 서비스 또는 프록시 서비스가 구성된 경우 삭제 경고 아이콘은 경고 확인을 통해 서비스 계정을 삭제할 수 있음을 나타냅니다. 이 경우 삭제된 서비스 계정을 참조하는 서비스의 해결되지 않은 참조로 인해 충돌이 발생할 수 있습니다.

  5. 세션을 종료하고 구성을 런타임에 배치하려면 변경 센터에서 활성화를 누릅니다.
