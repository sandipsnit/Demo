서비스 계정 찾기

  1. 다음 작업을 수행합니다.
    • 프로젝트 탐색기를 선택하여 프로젝트 뷰 페이지 또는 프로젝트/폴더 뷰 페이지를 표시합니다. 그런 다음 프로젝트와 폴더를 탐색하여 서비스 계정을 찾습니다.
    • 리소스 브라우저 > 서비스 계정을 선택합니다. 그러면 서비스 계정 요약에 표 16-1과 같은 정보가 표시됩니다.
  2. 서비스 계정을 찾으려면 이름 필드에 계정 이름의 일부나 모두를 입력합니다. 또는 계정 프로젝트 이름 및 폴더의 일부 또는 전부를 경로 필드에 입력할 수 있습니다. 검색을 누릅니다.

    3. 모두 보기를 눌러 검색 필터를 제거하고 모든 서비스 계정을 표시합니다.

    표 16-1 서비스 계정 정보
    속성
    설명
    서비스 계정 이름
    서비스 계정의 고유 이름입니다. 이름을 눌러 서비스 계정 세부 정보 보기 페이지를 봅니다. 서비스 계정 편집을 참조하십시오.
    경로
    서비스 계정이 있는 폴더 이름과 프로젝트 이름입니다. 이 리소스를 포함하는 프로젝트나 폴더를 보려면 이름을 누릅니다. 프로젝트 및 폴더를 사용하여 리소스 이름 식별을 참조하십시오.
    옵션
    삭제 아이콘이 포함되어 있습니다. 비즈니스 서비스 또는 프록시 서비스가 서비스 계정을 사용하도록 구성된 경우 삭제 경고 아이콘은 경고 확인을 통해 서비스 계정을 삭제할 수 있음을 나타냅니다. 이 경우 삭제된 서비스 계정을 참조하는 서비스의 해결되지 않은 참조로 인해 충돌이 발생할 수 있습니다. 서비스 계정 삭제를 참조하십시오.
