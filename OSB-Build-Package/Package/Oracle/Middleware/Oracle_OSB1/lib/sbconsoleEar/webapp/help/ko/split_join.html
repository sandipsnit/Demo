분할-조인

분할-조인을 통해 메시지 호출을 병렬로(직렬도 가능) 전송하고 응답을 집계할 수 있습니다. 분할-조인에 대한 자세한 내용은 http://www.oracle.com/pls/as111170/lookup?id=OSBAG291에서 Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus의 "Improving Service Performance with Split-Join"을 참조하십시오. 또한 http://www.oracle.com/pls/as111170/lookup?id=OSBDV193에서 "Working with Split-Join" 및 http://www.oracle.com/pls/as111170/lookup?id=OSBDV573에서 Oracle Fusion Middleware Developer's Guide for Oracle Service Bus의 "Split-Join User Interface Reference"도 참조하십시오.

흐름 전송 유형의 비즈니스 서비스를 통해 분할-조인에 액세스합니다. 자세한 내용은 흐름 전송 구성 페이지를 참조하십시오.
