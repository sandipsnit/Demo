자동 임포트 상태 사용

자동 임포트 상태 페이지를 사용하여 서비스의 변경 내용을 레지스트리에 있는 변경 내용과 동기화할 수 있습니다. 레지스트리에 있는 서비스를 변경하면 Oracle Service Bus는 동기화되지 않은 모든 서비스를 나열하는 자동 임포트 상태 페이지에서 변경 통지를 제공합니다. 그러면 Oracle Service Bus 관리 콘솔의 서비스를 UDDI 레지스트리의 해당 서비스와 동기화할 수 있습니다.

레지스트리를 구성하는 경우 임포트한 서비스를 UDDI 레지스트리와 자동으로 동기화하는 자동 임포트를 사용으로 설정 옵션을 선택할 수 있습니다. 선택한 이 옵션과 함께 임포트한 모든 서비스가 UDDI 레지스트리와 동기화되어 자동으로 보관됩니다. UDDI 레지스트리 구성을 참조하십시오.

자동 동기화 작업 중 오류가 발생하면 자동 임포트 상태 페이지에 보고되고 여기서 수동으로 갱신할 수 있습니다.

  1. 아직 이 작업을 수행하지 않은 경우 생성을 눌러 새 세션을 생성하거나 편집을 눌러 기존 세션을 입력합니다. 변경 센터 사용을 참조하십시오.
  2. 시스템 관리 > 자동 임포트 상태를 선택합니다.
  3. 자동 임포트 상태 페이지에서 보기 드롭다운 목록에 표시할 서비스 유형을 선택합니다.
  4. 레지스트리에서 해당 서비스와 동기화할 프록시 서비스 옆에 있는 체크 박스를 선택합니다.
  5. 동기화를 누릅니다.
  6. 다음 페이지에서 마침을 누릅니다.
