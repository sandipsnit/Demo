사용자 정의 파일 생성

사용자 정의 파일 생성 페이지에서는 선택한 리소스 또는 프로젝트 세트의 사용자 정의 파일을 손쉽게 생성할 수 있습니다. 그런 다음 이 파일을 시작점으로 사용하여 환경의 실제 값을 지정함으로써 원하는 내용을 수정할 수 있습니다.

사용자 정의 파일을 사용하여 리소스 내에서 참조를 변경할 수 있을 뿐만 아니라 환경 값을 변경할 수도 있습니다. 사용자 정의 파일에는 EnvValueTypes 클래스 내에 정의된 복잡한 환경 값 유형을 포함하여 선택한 리소스에 있는 모든 환경 값에 대한 사용자 정의가 포함되어 있습니다. 또한 종속이 있는 리소스 내의 리소스 참조 변경에 대한 참조 사용자 정의 유형을 포함합니다.

사용자 정의 유형을 설명하는 사용자 정의 스키마(Customization.xsd)는 Oracle Service Bus 설치 시 다음 위치에서 사용할 수 있습니다.

OSB_ORACLE_HOME\modules\com.bea.common.configfwk_version.jar

  1. 시스템 관리 > 사용자 정의 파일 생성를 선택합니다.

    2. 사용자 정의 파일 생성 페이지에 구성의 객체 목록이 표시됩니다. 각 객체의 이름과 유형이 표시됩니다.

  2. 사용자 정의 파일에 포함할 프로젝트 또는 리소스를 선택합니다.
    1. 프로젝트 폴더를 확장합니다. 프로젝트에 포함된 각 리소스의 이름과 유형이 표시됩니다.
    2. 사용자 정의 파일에 포함할 프로젝트 또는 리소스와 연관된 체크 박스를 선택합니다.
    3. 사용자 정의 파일에 포함하지 않으려는 프로젝트 또는 리소스와 연관된 체크 박스의 선택을 취소합니다.
  3. 사용자 정의 XML 파일을 생성하려면 파일 생성를 누릅니다.
  4. 파일 다운로드 대화상자에서 열기를 눌러 사용자 정의 파일을 열거나 저장을 눌러 XML 파일을 데스크톱에 저장합니다.
