기본 UDDI 구성 설정

구성한 레지스트리 중 하나를 도메인의 기본 UDDI 레지스트리로 지정할 수 있습니다. UDDI 레지스트리 구성을 참조하십시오.

자동 게시 기능을 사용하려면 먼저 기본 레지스트리를 설정해야 합니다. 자동 게시 사용을 참조하십시오.

  1. 아직 이 작업을 수행하지 않은 경우 생성을 눌러 새 세션을 생성하거나 편집을 눌러 기존 세션을 입력합니다. 변경 센터 사용을 참조하십시오.
  2. 시스템 관리 > UDDI 레지스트리를 선택합니다.
  3. UDDI 기본 구성 패널에서 기본값 선택을 누릅니다.
  4. 기본 레지스트리 이름 목록에서 기본 레지스트리로 설정할 레지스트리 이름을 선택합니다.
  5. 기본 비즈니스 엔티티를 설정하려면 비즈니스 엔티티 목록에서 엔티티를 선택합니다.
  6. 기본값으로를 누릅니다.
  7. 기본 레지스트리를 변경하려면 지우기 선택을 누르고 위의 단계를 반복합니다.
