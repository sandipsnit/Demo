서비스 분리

Oracle Service Bus 관리 콘솔의 서비스를 레지스트리의 해당 서비스와 동기화하지 않으려면 다음 방법 중 하나를 사용하여 레지스트리에서 서비스를 분리하여 동기화를 방지할 수 있습니다.

  • 구성을 편집하여 서비스 분리
  • 자동 임포트 상태 페이지에서 서비스 분리
구성을 편집하여 서비스 분리
  1. 아직 이 작업을 수행하지 않은 경우 생성을 눌러 새 세션을 생성하거나 편집을 눌러 기존 세션을 입력합니다. 변경 센터 사용을 참조하십시오.
  2. 프로젝트 탐색기를 선택합니다.
  3. 프로젝트 폴더를 확장하고 레지스트리의 해당 서비스에서 분리할 비즈니스 서비스를 선택합니다.
  4. 비즈니스 서비스 보기 페이지의 아래쪽에서 편집을 누릅니다.
  5. 레지스트리에서 분리 옵션을 사용으로 설정합니다. 비즈니스 서비스가 UDDI 레지스트리의 해당 서비스에서 분리됩니다.
자동 임포트 상태 페이지에서 서비스 분리

자동 임포트 상태 페이지에서 변경 사항에 대한 통지를 임포트한 경우 해당 분리 옵션을 사용할 수 있습니다.

  1. 시스템 관리 > 자동 임포트 상태 페이지에서 UDDI 레지스트리의 해당 서비스에서 분리하려는 비즈니스 서비스 옆의 체크 박스를 선택합니다.

    2. 레지스트리에 있는 원래 서비스에서 변경 사항이 존재하는 경우에만 이 페이지에 서비스가 표시됩니다. 이 페이지에 모든 서비스가 표시되지는 않습니다.

  2. 분리를 누릅니다.
