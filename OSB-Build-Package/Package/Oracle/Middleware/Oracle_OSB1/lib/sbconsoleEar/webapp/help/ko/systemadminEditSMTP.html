SMTP 서버 편집

이 페이지를 사용하여 필요한 경우 SMTP 서버의 세부 정보를 보고 구성을 편집할 수 있습니다.

  1. 아직 이 작업을 수행하지 않은 경우 생성을 눌러 새 세션을 생성하거나 편집을 눌러 기존 세션을 입력합니다. 변경 센터 사용을 참조하십시오.
  2. SMTP 서버를 찾습니다. SMTP 서버 찾기를 참조하십시오.
  3. SMTP 서버 이름을 누릅니다.

    4. MNP 서버 보기 - 구성 페이지에 표 30-4와 같은 정보가 표시됩니다.

    표 30-4 SMTP 서버 정보
    속성
    설명
    마지막으로 수정한 사람
    SMTP 서버 리소스를 생성하거나 수정한 사용자입니다.
    마지막 수정 시간
    사용자가 SMTP 서버 리소스를 생성하거나 수정한 날짜 및 시간입니다. 날짜 및 시간 링크를 눌러 이 리소스의 변경 내역을 봅니다. 변경 내역 보기 페이지를 참조하십시오.
    참조
    이 SMTP 서버가 참조하는 객체의 수입니다. 해당 참조가 존재하는 경우 숫자 링크를 누르면 참조 목록이 표시됩니다. 리소스에 대한 참조 보기를 참조하십시오.
    참조자
    이 SMTP 서버를 참조하는 객체의 수입니다. 해당 참조가 존재하는 경우 숫자 링크를 누르면 객체 목록이 표시됩니다. 리소스에 대한 참조 보기를 참조하십시오.
    설명
    이 SMTP 서버에 대한 설명(제공된 경우)입니다.
    서버 URL
    SMTP 서버의 위치를 가리키는 URL입니다.
    포트 번호
    이 SMTP 서버의 포트 번호입니다. 포트 25가 기본값입니다.
    사용자 이름
    보안 SMTP 서버에 액세스하는 데 사용되는 사용자 이름입니다.

  4. 편집을 눌러 SMTP 서버 편집- [server-name] 페이지를 표시합니다.

    5. 이름 필드는 편집할 수 없습니다.

  5. 설명 필드에서 SMTP 서버에 대한 간략한 설명을 편집할 수 있습니다.
  6. 서버 URL 필드에서 SMTP 서버 URL을 편집할 수 있습니다.
  7. 포트 번호 필드에서 포트 번호를 편집할 수 있습니다.
  8. 사용자 이름 및 비밀번호 필드를 편집할 수 있습니다.
  9. 저장을 눌러 현재 세션에서 갱신을 커밋합니다.
  10. 세션을 종료하고 구성을 런타임에 배치하려면 변경 센터에서 활성화를 누릅니다.

SMTP 리소스의 이름을 바꿀 경우 끝점 URI의 SMTP 리소스를 참조하는 전자 메일 비즈니스 서비스에서 자동으로 새 이름이 갱신됩니다.
