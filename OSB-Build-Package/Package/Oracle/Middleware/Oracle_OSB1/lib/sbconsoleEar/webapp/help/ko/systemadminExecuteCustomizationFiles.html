사용자 정의 파일 실행

이 페이지를 사용하여 시스템에서 이전에 저장한 사용자 정의 XML 파일을 실행할 수 있습니다. 사용자 정의 파일에서는 개발에서 스테이징으로, 스테이징에서 운영 단계로 배치하는 동안 또는 디자인 시간 동안 사용자 환경을 간편하게 수정할 수 있는 방법을 제공합니다.

사용자 정의 파일을 실행하려면 세션에 있어야 합니다.

  1. 아직 이 작업을 수행하지 않은 경우 생성을 눌러 새 세션을 생성하거나 편집을 눌러 기존 세션을 입력합니다. 변경 센터 사용을 참조하십시오.
  2. 시스템 관리 > 사용자 정의 파일 실행을 선택합니다.
  3. 실행한 사용자 정의 파일을 선택합니다.
    1. 파일 이름 필드에서 찾아보기를 눌러 사용자 정의 파일이 저장된 디렉토리를 찾습니다.
    2. XML 파일을 선택하고 열기를 누릅니다.
  4. 다음을 누릅니다. 그러면 표 31-2과 같은 사용자 정의 세부 정보가 표시됩니다.

    5. 표 31-2 사용자 정의 세부 정보
    속성
    설명
    작업
    • 찾기 및 바꾸기 - 문자열 치환을 통해 환경 값을 수정합니다.
    • 참조 매핑 - 리소스에 있는 기존의 참조를 다른 참조에 매핑합니다.
    • 환경 값 설정 - 리소스에 대한 환경 값 수정
    적용 대상
    사용자 정의할 서비스, 프로젝트, 폴더 또는 리소스의 이름입니다.
    이름은 서비스 또는 리소스의 구성 세부 정보 페이지 또는 프로젝트 탐색기의 프로젝트 또는 폴더로 링크됩니다.
    설명
    각 사용자 정의에 대한 설명입니다.

  5. 현재 세션에서 변경된 리소스, 서비스, 프로젝트 또는 폴더만 사용자 정의하려면 현재 세션에서 변경된 항목만 체크 박스를 선택합니다.

    6. 현재 세션에서 수정된 리소스로만 사용자 정의를 제한하는 경우, 적용 대상 열이 현재 세션에서 수정된 리소스로 갱신됩니다. 프로젝트 또는 폴더의 사용자 정의를 제한하는 경우, 프로젝트 또는 폴더내의 현재 세션에서 수정된 리소스가 적용 대상 열에 표시됩니다.

  6. 실행을 눌러 현재 세션에서 갱신을 커밋합니다.

    7. 사용자 정의 세부 정보를 표시하려면, 세션 내의 변경 센터에서 변경 사항 보기를 선택하고 사용자 정의 작업을 누릅니다. 구성 변경 내용 보기를 참조하십시오.

  7. 세션을 종료하고 구성을 런타임에 배치하려면 변경 센터에서 활성화를 누릅니다.
