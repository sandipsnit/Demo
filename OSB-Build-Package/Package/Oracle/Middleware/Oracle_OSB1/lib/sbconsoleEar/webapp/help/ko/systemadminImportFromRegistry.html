UDDI 레지스트리에서 비즈니스 서비스 익스포트

UDDI 레지스트리에서 Oracle Service Bus로 다음 비즈니스 서비스 유형을 임포트할 수 있습니다.

  • HTTP 전송을 통한 WSDL 서비스
  • UDDI 레지스트리에 게시되는 Oracle Service Bus 프록시 서비스. 기본적으로 이 기능은 한 도메인의 프록시 서비스가 다른 도메인의 프록시 서비스를 찾아 경로 지정해야 하는 다중 도메인 Oracle Service Bus 배치에서 사용됩니다.

사용자가 한 개의 레지스트리만 구성한 경우 Oracle Service Bus에서 자동으로 해당 레지스트리에 접속하여 모든 비즈니스 엔티티를 검색하고 검색 양식을 작성합니다. 사용자가 두 개 이상의 레지스트리를 구성한 경우 임포트 마법사에서 서비스를 임포트하려고 하는 레지스트리를 먼저 선택할 수 있습니다.

레지스트리에는 몇 가지 유형의 서비스가 있기 때문에 특정 서비스를 찾으려면 임포트 시 문의 URL이 사용됩니다.

  1. 아직 이 작업을 수행하지 않은 경우 생성을 눌러 새 세션을 생성하거나 편집을 눌러 기존 세션을 입력합니다. 변경 센터 사용을 참조하십시오.
  2. 시스템 관리 > UDDI에서 임포트를 선택합니다.
  3. 레지스트리 이름 임포트 드롭다운 목록에서 서비스를 임포트할 레지스트리 이름을 선택하고 다음을 누릅니다.

    4. 단일 레지스트리가 Oracle Service Bus에서 구성되면 이 레지스트리가 기본 레지스트리가 되므로 초기 레지스트리 선택 페이지가 표시되지 않습니다. 두 개 이상의 레지스트리가 정의된 경우에만 레지스트리 선택 페이지가 표시됩니다.

    UDDI에서 임포트 페이지에서는 특정 비즈니스 서비스를 검색하여 임포트할 수 있습니다. 비즈니스 엔티티나 패턴별 또는 이 두 가지의 조합을 사용하여 서비스를 찾을 수 있습니다.

  4. 엔티티 이름 드롭다운 목록에서 비즈니스 엔티티 이름(예: 문서 서비스)을 선택하여 비즈니스 엔티티별로 검색합니다.
  5. 서비스 이름 필드에 검색할 패턴(예: a%)을 입력합니다. 입력한 검색 조건과 일치하는 비즈니스 서비스의 목록이 표시됩니다.
  6. 임포트하려는 서비스를 선택하고 다음을 누릅니다.

    7. 원하는 서비스를 찾을 수 없다면 해당 레코드를 볼 권한을 가진 보안 그룹에 속해 있지 않아서 그런 것일 수 있습니다.

  7. 비즈니스 서비스를 생성하려는 바인딩 템플리트를 선택하고 다음을 누릅니다.

    8. 선택한 서비스에 다수의 바인딩 템플리트가 있으면 각 바인딩 템플리트가 비즈니스 서비스로 됩니다. 이런 경우 임포트하려는 바인딩 템플리트 중에서 선택을 좀 더 좁히라는 메시지가 표시됩니다.

  8. UDDI 임포트: 임포트 위치 선택 페이지의 프로젝트 드롭다운 목록에서 프로젝트를 선택하고, 서비스와 관련 리소스를 임포트할 폴더를 선택한 후 다음을 누릅니다.
  9. UDDI 임포트: 서비스 검토 및 임포트 페이지에 시스템에 생성될 모든 리소스(비즈니스 서비스, MFL, 스키마, WSDL 등)의 목록이 표시됩니다. 임포트할 수 없는 리소스에 대해 경고 메시지가 표시됩니다.
    기본적으로 목록의 모든 항목이 임포트용으로 표시됩니다.
    1. 시스템 내에 특정 리소스를 이미 가지고 있어서 종속성을 수동으로 해결하려면 해당 리소스 옆에 있는 체크 박스의 선택을 취소합니다.
    2. 임포트 프로세스가 완료된 후 변경 센터의 충돌 보기 페이지에서 종속성을 해결할 수 있습니다. 충돌 보기 및 해결에 대한 자세한 내용은 충돌 보기 및 해결을 참조하십시오.
  10. 임포트 프로세스를 시작하려면 임포트를 누릅니다.

    11. 임포트 진행 상태를 보여주는 팝업 창이 표시됩니다. 임포트가 완료되면 임포트 요약 페이지에 임포트 결과가 표시됩니다. 임포트에 성공하면 페이지 맨 위에 성공했다는 메시지가 표시됩니다. 한 개 이상의 리소스를 임포트할 수 없는 경우 오류 메시지가 표시됩니다.

    • 해당 리소스의 상태 열로 이동하여 오류 원인을 확인합니다.
    • 다른 충돌을 보고 해결하려면 충돌 보기를 누릅니다.
  11. 더 많은 서비스를 임포트하려면 다른 항목 임포트를 누릅니다.
  12. 선택한 항목을 변경하려면 뒤로를 누릅니다.
