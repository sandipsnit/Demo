보안 설정 임포트 지정

JAR 파일에 익스포트할 때 암호화된 사용자 이름 및 비밀번호 데이터가 포함된 리소스가 있으면 리소스 임포트 - 보안 설정 페이지가 표시됩니다.

  1. 데이터를 암호화하는 데 사용된 비밀번호를 입력합니다(리소스 익스포트 참조).

    2. 리소스 데이터를 암호화하는 데 사용된 비밀번호를 모르는 경우 이전 단추를 누르고 리소스 옆에 있는 체크 박스의 선택을 취소합니다. 비밀번호를 모르는 경우에도 암호화되지 않은 다른 모든 리소스를 임포트할 수 있습니다.

  2. 임포트 완료를 누릅니다.

    3. 구성 데이터 처리 팝업 창이 표시됩니다. Oracle Service Bus 관리 콘솔에서 다른 작업을 수행하기 전에 이 창이 닫힐 때까지 기다려야 합니다. 객체 임포트가 완료되면 창이 자동으로 닫힙니다. 리소스 임포트 페이지가 표시되고 데이터를 성공적으로 배치했다는 메시지를 보여줍니다.

마친 후에

임포트 요약 검토에서 계속 진행합니다.
