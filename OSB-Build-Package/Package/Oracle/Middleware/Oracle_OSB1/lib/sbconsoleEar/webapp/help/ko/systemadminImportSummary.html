임포트 요약 검토

임포트 요약 정보는 표 28-3에 표시되어 있습니다.

표 28-3 임포트 요약 정보
속성
설명
상태
리소스를 성공적으로 임포트했는지 여부를 나타냅니다.
WSDL은 모든 종속성을 임포트할 때까지 부적합으로 잠시 표시될 수 있습니다. 그러나 임포트 작업이 완료되면 구성 서비스는 부적합으로 표시된 파일을 다시 검증합니다.
이름
리소스 이름입니다.
경로
리소스 경로입니다. 리소스가 있는 폴더 이름과 프로젝트 이름입니다.
유형
Oracle Service Bus 리소스 유형을 표시합니다.
진단 메시지
메시지가 이 리소스에 존재하는 경우 오류 또는 정보 메시지를 표시합니다.

  1. 다른 항목 임포트를 누릅니다.

    2. 다른 JAR 파일을 임포트하거나 변경 센터에서 변경 내용 보기를 눌러 임포트 세부 정보를 볼 수 있습니다. 구성 변경 내용 보기를 참조하십시오.

  2. 작업을 마쳤으면 변경 센터에서 활성화를 누릅니다.
