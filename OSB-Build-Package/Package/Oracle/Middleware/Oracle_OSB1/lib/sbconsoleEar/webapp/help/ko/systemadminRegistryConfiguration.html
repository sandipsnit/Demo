구성된 UDDI 레지스트리 보기

이 페이지를 사용하여 기본 UDDI 레지스트리(기본 UDDI 구성 설정 참조)를 지정하고 UDDI 레지스트리를 볼 수 있습니다. 또한 이전에 구성한 레지스트리도 검색할 수 있습니다.

  1. UDDI 레지스트리 패널에서 이름 필드에 찾을 레지스트리 이름을 입력하거나 와일드 카드 문자를 입력합니다. 다중 문자일 경우 *를 사용하고 단일 문자일 경우 ?를 사용합니다. 검색 결과에는 이름 기준을 충족하는 모든 레코드가 반환됩니다.
  2. 검색을 누릅니다.
  3. 검색 필터를 제거하고 Oracle Service Bus와 작동하도록 구성된 모든 레지스트리를 보려면 모두 보기를 누릅니다.

    4. 표 29-1에는 UDDI 레지스트리 구성 설정이 표시됩니다.

    표 29-1 UDDI 레지스트리 구성 설정 
    속성
    설명
    레지스트리 이름
    레지스트리 이름으로, 레지스트리를 처음 생성할 때 레지스트리에 할당된 이름입니다. 항목을 저장한 후에는 레지스트리 이름을 편집할 수 없습니다.
    조회 URL
    서비스를 검색하고 임포트하는 데 사용되는 조회 API 끝점에 대한 URL입니다.
    게시 URL
    서비스를 게시하는 데 사용되는 게시 API 끝점에 대한 URL입니다.
    보안 URL
    레지스트리에 게시할 수 있도록 인증 토큰을 임포트하는 데 사용되는 보안 API 끝점에 대한 URL입니다.
    가입 URL
    레지스트리의 변경 내용을 등록하고 레지스트리 가입을 생성하며 임포트한 서비스에 대한 변경 내용을 수신하는 데 사용되는 가입 API 끝점에 대한 URL입니다.
    사용자 이름
    사용자 인증에 필요한 사용자 이름입니다.
    비밀번호(비밀번호 확인)
    사용자 인증에 필요한 비밀번호입니다.
    레지스트리로 tModel 로드
    선택한 레지스트리로 tModel을 로드합니다. 이 옵션은 레지스트리당 한 번만 선택해야 합니다.
    자동 임포트를 사용으로 설정
    UDDI 레지스트리와 서비스를 자동으로 동기화합니다.
