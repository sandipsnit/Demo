SMTP 서버 추가

SMTP 서버 요약 페이지를 사용하여 SMTP 서버 리소스를 추가 및 구성하고 Oracle Service Bus에서 시스템 리소스로 사용할 수 있습니다. SMTP 서버 리소스를 구성하거나 재구성하려면 활성 세션에 있어야 합니다.

  1. 아직 이 작업을 수행하지 않은 경우 생성을 눌러 새 세션을 생성하거나 편집을 눌러 기존 세션을 입력합니다. 변경 센터 사용을 참조하십시오.
  2. 시스템 관리 > SMTP 서버를 선택합니다.
  3. 추가를 누릅니다.
  4. 이름 필드에 SMTP 서버 리소스에 대한 이름을 입력합니다. 이 필드는 필수입니다.

    리소스 이름 지정 제한 사항의 이름 지정 지침을 따르십시오.
  5. 설명 필드에 SMTP 서버 리소스에 대한 간략한 설명을 입력합니다.
  6. 서버 URL 필드에 SMTP 서버를 가리키는 URL을 입력합니다. 이 필드는 필수 필드입니다.
  7. 포트 번호 필드에 SMTP 서버의 포트 번호를 입력합니다(기본 포트는 25임). 이 필드는 필수 필드입니다.
  8. 사용자 이름과 비밀번호를 입력해야 대상 SMTP 서버에 액세스할 수 존재하는 경우 사용자 이름 필드에 사용자 이름을 입력하고 비밀번호와 비밀번호 확인 필드에 해당 비밀번호를 입력합니다.

    9. 이 필드는 선택 사항으로, SMTP 서버에 보안이 유지되는 경우에만 필요합니다.

  9. 저장을 눌러 현재 세션에서 SMTP 서버 리소스를 생성하고 저장합니다.
  10. 세션을 종료하고 구성을 런타임에 배치하려면 변경 센터에서 활성화를 누릅니다.
