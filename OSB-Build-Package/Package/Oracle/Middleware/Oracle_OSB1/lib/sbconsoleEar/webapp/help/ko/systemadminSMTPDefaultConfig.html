기본 SMTP 서버 구성

구성된 SMTP 서버 중 하나를 도메인의 기본 서버로 지정할 수 있습니다. SMTP 서버 추가를 참조하십시오.

  1. 아직 이 작업을 수행하지 않은 경우 생성을 눌러 새 세션을 생성하거나 편집을 눌러 기존 세션을 입력합니다. 변경 센터 사용을 참조하십시오.
  2. 시스템 관리 > SMTP 서버를 선택합니다.
  3. 기본 SMTP 서버 패널에서 기본값 선택을 누릅니다.
  4. 기본 SMTP 서버 드롭다운 목록에서 SMTP 서버를 선택합니다.
  5. 기본값으로를 누릅니다.

    6. SMTP 서버 요약 페이지에 선택한 SMTP 서버가 해당 도메인의 기본 SMTP 서버로 표시됩니다.

  6. 세션을 종료하고 구성을 런타임에 배치하려면 변경 센터에서 활성화를 누릅니다.
