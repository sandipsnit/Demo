JNDI 제공자 보기

이 페이지에서는 JNDI 제공자 목록 및 표 30-1과 같은 정보를 볼 수 있습니다.

또한 이전에 구성한 JNDI 제공자를 검색할 수 있습니다.

  1. 목록에 표시되는 항목 수를 제한하거나 특정 JNDI 제공자를 찾으려면:
    • 열기 아이콘을 눌러 검색 필터를 추가로 표시합니다. 다중 검색 조건을 지정하는 경우에는 모든 조건과 일치하는 결과만 반환됩니다.
    • 검색할 JNDI 제공자의 이름을 이름 필드에 입력하고 검색을 눌러 특정 항목을 검색합니다.

      • 와일드 카드 문자(단일 문자용은 ?, 다중 문자용은 *)를 입력하여 보다 일반적인 검색을 수행할 수 있습니다.

    • 찾을 JNDI 제공자의 제공자 URL을 입력한 다음 검색을 누릅니다.
  2. 모두 보기를 눌러 검색 필터를 제거하고 Oracle Service Bus와 함께 작동하도록 구성된 JNDI 제공자 리소스를 모두 표시합니다.

    3. 표 30-1에는 UDDI 레지스트리 구성 설정이 표시됩니다.

    표 30-1 JNDI 제공자 정보
    속성
    설명
    이름
    이 JNDI 제공자에 할당된 고유 이름입니다. 이름은 JNDI 제공자 보기 - 구성 페이지에 대한 링크입니다. JNDI 제공자 편집을 참조하십시오.
    제공자 URL
    JNDI 제공자의 URL입니다. JNDI 제공자 추가를 참조하십시오.
    JNDI 캐시
    사용 또는 사용 안함으로 설정 상태입니다. 사용으로 설정된 경우, JNDI 컨텍스트 및 JNDI 객체가 로컬로 캐시되고 객체 조회를 수행할 때 성능이 향상됩니다. JNDI 캐시를 사용으로 설정으로 유지하는 것이 좋습니다.
    작업
    • 재설정 아이콘을 눌러 JNDI 접속 및 로컬로 캐시된 객체를 폐기하도록 JNDI 컨텍스트를 재설정합니다.
    • 테스트 콘솔 시작 아이콘을 눌러 JNDI 제공자가 접속을 설정하고 JNDI 컨텍스트를 임포트할 수 있는지 여부를 테스트합니다.
    옵션
    • JNDI 제공자 이름을 변경하려면 이름 바꾸기 아이콘을 누릅니다.
    • 특정 JNDI 제공자를 삭제하려면 삭제 아이콘을 누릅니다. JNDI 제공자 삭제를 참조하십시오.
