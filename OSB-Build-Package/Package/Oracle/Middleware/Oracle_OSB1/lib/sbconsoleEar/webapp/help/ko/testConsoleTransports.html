테스트 콘솔에서 런타임이 전송 설정을 사용하는 방법 이해

프록시 서비스 테스트 데이터 구성 및 비즈니스 서비스 테스트 데이터 구성에서는 테스트 콘솔에서 프록시 서비스 또는 비즈니스 서비스를 테스트할 때 아웃바운드 요청에 대한 전송 관련 보안 데이터, 전송 메타 데이터 및 전송 헤더의 값을 구성하는 방법에 대해 설명합니다. 단, 테스트 콘솔에서 지정할 수 있는 일부 내용은 런타임 시 적용되지 않습니다. 즉, 테스트를 실행할 때 런타임 시 Oracle Service Bus가 특정 헤더 또는 메타 데이터 값을 겹쳐 쓰거나 무시할 수 있습니다. 제한을 받는 헤더 및 메타 데이터는 표 32-10에서 설명됩니다.
