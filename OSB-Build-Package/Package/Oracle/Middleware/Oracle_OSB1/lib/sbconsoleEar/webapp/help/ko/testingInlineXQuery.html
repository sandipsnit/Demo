XQuery 테스트 수행

XQuery 표현식 편집기, XQuery 조건 편집기 및 XPath 표현식 편집기를 사용하여 메시지 흐름에서 작업을 편집하고 테스트할 수 있습니다. 테스트는 XQuery 표현식 편집기와 조건 편집기 모두에 대해 동일한 형식을 사용합니다. 하지만 XPath 표현식 편집기에서는 하나의 입력만 사용하기 때문에 XPath 표현식 편집기에 대한 시나리오는 다릅니다.

주: XQuery 테스트를 작동하게 하려면 브라우저의 팝업 차단기를 사용 안함으로 설정해야 합니다. IE 브라우저에서 툴 모음을 사용하는 경우 옵션 메뉴에서 툴 모음을 차단하도록 구성된 모든 툴 모음과 팝업 차단기를 사용 안함으로 설정해야 할 수도 있습니다.

이 섹션에는 다음과 같은 항목이 포함되어 있습니다.

  • XQuery 표현식 편집기 및 XQuery 조건 편집기 사용
  • XPath 표현식 편집기 사용
