프록시 서비스 테스트 결과 보기

표 32-2에서는 프록시 서비스 테스트 결과에 대해 설명합니다. 추적은 직접 호출 및 추적 포함 옵션을 모두 선택한 경우에만 사용으로 설정됩니다.

표 32-2 프록시 서비스의 테스트 결과
섹션
설명
프록시 서비스 이름
테스트할 프록시 서비스의 이름입니다.
테스트 콘솔 작업
뒤로를 눌러 이전 브라우저 페이지를 표시합니다.
닫기를 눌러 테스트 콘솔을 닫습니다.
요청 문서
테스트 콘솔에서 프록시 서비스로 전송된 요청 메시지입니다.
테스트 콘솔이 요청 메시지를 수정하지 않은 경우 이 섹션은 처음에 축소되어 있습니다. 폼 탭을 사용하여 SOAP 서비스가 구성되었거나 WS 보안이 적용된 경우 이 섹션은 처음에 확장되어 있습니다.
WS 보안이 적용된 경우, 이 섹션에는 두 개의 SOAP 메시지가 포함되는데 첫 번째 메시지는 일반 텍스트 메시지이고 두 번째 메시지는 보안 SOAP 메시지입니다.
응답 문서
메시지 응답입니다.
WS 보안 응답이 있는 SOAP 서비스에 대해 이 섹션에는 두 개의 SOAP 메시지가 포함되어 있는데 첫 번째 SOAP 메시지는 클라이언트에서 수신된 것처럼 보안 메시지입니다. 두 번째 SOAP 메시지는 이에 해당하는 일반 텍스트 메시지입니다.
응답 메타 데이터
메시지 응답으로 반환된 메타 데이터입니다.
프록시 서비스 추적
추적은 시스템을 통해 전달하는 메시지의 상태를 보여줍니다. 직접 호출 옵션이 선택되어 있지 않은 경우 추적을 수행할 수 없습니다. 자세한 내용은 프록시 서비스 추적을 참조하십시오.
