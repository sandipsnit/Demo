표 32-4 비즈니스 서비스 테스트 결과
섹션
설명
비즈니스 서비스 이름
비즈니스 서비스의 이름입니다.
테스트 콘솔 작업
이전 브라우저 페이지를 표시하려면 뒤로를 누릅니다.
테스트 콘솔 창을 닫으려면 닫기를 누릅니다.
요청 문서
테스트 콘솔에서 비즈니스 서비스로 전송된 요청 메시지입니다.
테스트 콘솔이 요청 메시지를 수정하지 않은 경우 이 섹션은 처음에 축소되어 있습니다. 폼 탭을 사용하여 SOAP 서비스가 구성되었거나 WS 보안이 적용된 경우 이 섹션은 처음에 확장되어 있습니다.
WS 보안이 적용되면 이 섹션에는 두 개의 SOAP 메시지가 표시됩니다. 첫 번째 메시지는 일반 텍스트 메시지입니다. 두 번째 메시지는 보안 SOAP 메시지입니다.
응답 문서
메시지 응답입니다.
WS 보안 응답이 있는 SOAP 서비스에 대해 이 섹션에는 두 개의 SOAP 메시지가 포함되어 있는데 첫 번째 SOAP 메시지는 클라이언트에서 수신된 것처럼 보안 메시지입니다. 두 번째 SOAP 메시지는 이에 해당하는 일반 텍스트 메시지입니다.
응답 메타 데이터
메시지 응답으로 반환된 메타 데이터입니다.

주: 예를 들어, 보안 SOAP 메시지는 별도의 공백을 사용하여 인쇄되어 표시됩니다. 공백은 문서 구문에 영향을 줄 수 있으므로 이 SOAP 메시지를 리터럴 데이터로 항상 사용할 수 있는 것은 아닙니다. 예를 들어, 디지털 서명은 공백을 구분하며 부적합하게 될 수 있습니다.
