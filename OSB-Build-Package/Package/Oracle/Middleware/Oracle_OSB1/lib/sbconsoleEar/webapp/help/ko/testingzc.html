MFL 변환 테스트

MFL(Message Format Language) 문서는 이진 데이터의 레이아웃을 설명하는 데 사용되는 특수 XML 문서로서, MFL 리소스는 다음 두 변환을 지원합니다. 각 변환에는 하나의 입력만 적용되고 XML에서 이진으로 또는 이진에서 XML로 등과 같이 단일 출력을 제공합니다.

표 32-5에서는 MFL 구성 옵션에 대해 설명합니다.

표 32-5 MFL 구성 옵션
섹션
설명
이름
테스트되는 리소스의 이름입니다.
지원되는 변환
특정 변환을 선택하려면 변환 이름을 선택하십시오.
테스트 콘솔 작업
실행
변환을 적용합니다.
재설정
입력 필드를 재설정합니다. XML에서 이진으로의 경우 샘플 XML 문서가 재설정됩니다.
닫기
현재 작업을 취소합니다.
MFL 변환 구성
XML 입력
XML에서 이진으로의 변환에 필요:
MFL 문서에 대한 XML 스키마는 추측이 가능합니다. 샘플 XML 문서는 자동으로 텍스트 필드에 입력됩니다.
XML 입력은 파일 기반 또는 텍스트 기반일 수 있습니다. 입력에 대한 파일 참조는 텍스트 입력보다 우선합니다. 테스트에 사용할 파일을 찾아서 선택합니다.
이진 입력
이진에서 XML로의 변환에 필요:
이진 입력은 파일 기반 또는 텍스트 기반일 수 있습니다. 입력에 대한 파일 참조는 텍스트 입력보다 우선합니다. 테스트에 사용할 파일을 찾아서 선택합니다.

디자인 시간 또는 런타임을 테스트할 수 있습니다.

  1. 런타임을 테스트하려면 활성화를 누릅니다. 디자인 시간을 테스트하기 위해 세션을 활성화하지 마십시오.
  2. 리소스 브라우저 > MFL을 선택하여 MFL 파일 요약 페이지를 표시합니다.
  3. 작업에서 테스트할 리소스와 연관된 테스트 콘솔 시작 아이콘을 누릅니다. 그러면 테스트 콘솔에서 리소스 테스트 페이지가 열립니다.
  4. 테스트할 변환을 선택합니다.
  5. MFL 리소스에 대한 테스트 데이터를 구성합니다. 자세한 내용은 표 32-5를 참조하십시오.
  6. 실행을 누릅니다. 리소스 테스트 페이지에 결과가 표시됩니다.
  7. 다시 테스트하려면 뒤로를 누릅니다. 테스트 콘솔을 닫고 리소스를 수정하여 다시 테스트할 수 있습니다.
