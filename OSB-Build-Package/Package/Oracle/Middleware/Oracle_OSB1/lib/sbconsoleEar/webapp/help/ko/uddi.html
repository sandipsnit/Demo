UDDI

UDDI(Universal Description, Discovery and Integration) 레지스트리는 엔터프라이즈에서 웹 서비스를 공유하기 위해 사용됩니다. UDDI는 비즈니스, 해당 비즈니스의 서비스 및 표시하려는 서비스에 대한 기술 세부 정보를 분류하는 프레임워크를 제공합니다.

레지스트리에 서비스를 게시하려면 레지스트리에서 해당 서비스를 나타내는 데이터 구조 및 서비스 유형에 대한 지식이 있어야 합니다. 레지스트리 항목에는 해당 항목과 연관된 특정 속성이 포함되어 있고 이러한 속성 유형은 레지스트리 작성 시 정의됩니다. 서비스를 레지스트리에 게시하여 다른 조직에서 이 서비스를 발견하고 사용 가능하게 할 수 있습니다. Oracle Service Bus에서 개발한 프록시 서비스는 UDDI 레지스트리에 게시할 수 있습니다. Oracle Service Bus는 모든 UDDI 버전 3.0 호환 레지스트리와 상호 작용할 수 있습니다.
