UDDI 레지스트리 편집

  1. 아직 이 작업을 수행하지 않은 경우 생성을 눌러 새 세션을 생성하거나 편집을 눌러 기존 세션을 입력합니다. 변경 센터 사용을 참조하십시오.
  2. 시스템 관리 > UDDI 레지스트리를 선택합니다.
  3. UDDI 레지스트리 패널의 사용 가능한 레지스트리 목록에서 편집할 레지스트리 이름을 누릅니다.

    4. 또는 검색 옵션을 사용하여 특정 레지스트리를 검색할 수 있습니다. 구성된 UDDI 레지스트리 보기를 참조하십시오.

    UDDI 레지스트리 구성 페이지에 표 29-2와 같은 레지스트리별 정보와 레지스트리를 정의하는 구성 속성이 표시됩니다.

    표 29-2 UDDI 레지스트리 정보
    속성
    설명
    마지막으로 수정한 사람
    UDDI 레지스트리 리소스의 구성을 마지막으로 변경한 사용자입니다.
    마지막 수정 시간
    마지막으로 구성을 변경한 날짜와 시간입니다. 이 리소스의 변경 내역을 보려면 날짜 및 시간 링크를 누릅니다. 변경 내역 보기 페이지를 참조하십시오.
    참조
    이 UDDI 레지스트리가 참조하는 객체의 수입니다. 해당 참조가 존재할 경우 참조 목록을 보려면 숫자 링크를 누릅니다. 리소스에 대한 참조 보기를 참조하십시오.
    참조자
    이 UDDI 레지스트리를 참조하는 객체의 수입니다. 해당 참조가 존재할 경우 객체 목록을 보려면 숫자 링크를 누릅니다. 리소스에 대한 참조 보기를 참조하십시오.
    설명
    레지스트리에 대한 설명입니다.

  4. 페이지 맨 아래에서 편집을 누릅니다.
  5. UDDI 구성-레지스트리 편집 페이지에서 레지스트리 구성 매개변수를 편집합니다.

    6. 이러한 속성은 표 29-1에서 설명됩니다. 레지스트리 이름은 편집할 수 없습니다.

  6. 저장을 눌러 현재 세션에서 갱신을 커밋합니다.
  7. 세션을 종료하고 구성을 런타임에 배치하려면 변경 센터에서 활성화를 누릅니다.
