WSDL

WSDL(Web Service Definition Language)은 웹 서비스에 대한 공식적인 설명입니다. Oracle Service Bus에서 WSDL은 프록시 서비스나 비즈니스 서비스를 설명합니다. WSDL(Web Service Definition Language)은 웹 서비스의 기능, 위치 및 호출 방법을 설명하는 데 사용됩니다.

SOAP 및 XML 서비스는 기존 WSDL 리소스를 기반으로 할 수 있습니다. 모든 전송에 대한 프록시 서비스와 비즈니스 서비스에서 WSDL 문서를 사용할 수 있습니다. 프록시 서비스 생성/편집 - 일반 구성 페이지 및 비즈니스 서비스 생성 및 구성을 참조하십시오.

자세한 내용은 http://www.oracle.com/pls/as111170/lookup?id=OSBAG114에서 Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus의 "How WSDL is Used in Oracle Service Bus"를 참조하십시오.
