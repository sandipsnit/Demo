WSDL 추가

  1. 아직 이 작업을 수행하지 않은 경우 생성을 눌러 새 세션을 생성하거나 편집을 눌러 기존 세션을 입력합니다. 변경 센터 사용을 참조하십시오.
  2. 프로젝트 탐색기를 선택한 다음 WSDL을 추가할 프로젝트나 폴더를 선택합니다. 프로젝트/폴더 뷰 페이지가 표시됩니다.
  3. 리소스 생성 드롭다운 목록의 인터페이스에서 WSDL을 선택하여 새 WSDL 리소스 생성 페이지를 표시합니다.
  4. 리소스 이름 필드에 이 WSDL에 대한 고유 이름을 입력합니다. 이 필드는 필수입니다.

    리소스 이름 지정 제한 사항의 이름 지정 지침을 따르십시오.
  5. 리소스 설명 필드에 WSDL에 대한 설명을 입력합니다.
  6. WSDL 필드에서 다음 작업 중 하나를 수행합니다.
    • 새 WSDL에 사용할 텍스트를 입력합니다.
    • 찾아보기 를 눌러 기존 WSDL을 찾아서 임포트합니다.
    • 기존 WSDL의 데이터를 복사하여 이 필드에 붙여 넣습니다.

      • 이 필드는 필수 필드입니다.

  7. 저장을 누릅니다. 현재 세션에 새 WSDL이 저장됩니다.

    8. 저장을 누르면 새 WSDL에 대한 해결되지 않은 참조가 표시됩니다(있을 경우). 해결되지 않은 WSDL 참조 보기를 참조하십시오.

  8. 세션을 종료하고 구성을 런타임에 배치하려면 변경 센터에서 활성화를 누릅니다.
