해결되지 않은 WSDL 참조 보기 페이지에는 표 6-5에 나열된 정보가 표시됩니다.

표 6-5 해결되지 않은 WSDL 참조 세부 정보
속성
설명
WSDL 이름
WSDL에 할당된 이름입니다. WSDL의 이름을 누르면 해당 WSDL에 대한 세부 정보가 표시됩니다.
네임스페이스
WSDL의 주소입니다. 주소는 다음 형식으로 표시됩니다.
http://example.com/example/example

특정 참조를 찾으려면

  • 목록을 다시 정렬합니다. 밑줄로 표시된 열 이름을 누릅니다. 오름차순 및 내림차순 화살표는 정렬 순서를 나타냅니다. 열 이름을 누르면 정렬 순서가 변경됩니다.
  • 페이지를 스크롤합니다. 테이블의 위 또는 아래에 있는 페이지 콘트롤을 사용합니다. 페이지 번호를 선택하여 특정 페이지로 이동하거나 화살표 단추를 사용하여 다음, 이전, 첫 페이지 또는 마지막 페이지로 이동할 수 있습니다.
