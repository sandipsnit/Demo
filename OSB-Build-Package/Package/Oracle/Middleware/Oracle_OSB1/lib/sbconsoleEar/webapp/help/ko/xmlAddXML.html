XML 문서 추가

Oracle Service Bus에서 XML 문서를 생성하는 가장 간편한 방법은 XML 문서를 대량 임포트하는 것입니다. 예를 들어, 이전에 생성한 JCA 리소스(.jca 파일, 연관된 WSDL 및 TopLink 또는 EclipseLink 매핑 파일)를 대량 임포트하면 Oracle Service Bus는 자동으로 매핑 파일에서 XML 문서 리소스를 생성하고 리소스 파일 간의 종속성을 유지합니다.

임포트에 대한 자세한 내용은 Zip 파일에서 리소스 로드, URL에서 리소스 로드 및 리소스 임포트를 참조하십시오.

대량 임포트하지 않을 경우 다음 단계에 설명된 대로 수동으로 XML 문서를 생성할 수 있습니다.

  1. 아직 이 작업을 수행하지 않은 경우 생성을 눌러 새 세션을 생성하거나 편집을 눌러 기존 세션을 입력합니다. 변경 센터 사용을 참조하십시오.
  2. 프로젝트 탐색기를 선택한 다음 XML 문서를 추가할 프로젝트나 폴더를 선택합니다. 그러면 프로젝트/폴더 뷰 페이지가 표시됩니다.
  3. 리소스 생성 필드에서 XML 문서를 선택하여 새 XML 문서 생성 페이지를 표시합니다.
  4. XML 파일 필드에서 찾아보기를 누르고 사용할 XML 파일을 선택합니다.

    5. [리소스 이름] 필드에 자동으로 파일 이름(파일 확장자 제외)이 채워집니다.

    파일을 선택하는 대신 [XML 파일] 필드 텍스트 영역에서 XML 파일 내용을 복사하여 붙여 넣을 수도 있습니다.

  5. 리소스 이름 필드에서 XML 문서의 이름을 입력하거나 변경합니다. XML 문서 이름은 고유해야 합니다.

    리소스 이름 지정 제한 사항의 이름 지정 지침을 따르십시오.
  6. 리소스 설명 필드에 XML 문서에 대한 설명을 입력합니다.
  7. 저장을 누릅니다. 그러면 현재 세션에 새 XML 문서가 저장됩니다.
  8. 세션을 종료하고 구성을 런타임에 배치하려면 변경 센터에서 활성화를 누릅니다.
