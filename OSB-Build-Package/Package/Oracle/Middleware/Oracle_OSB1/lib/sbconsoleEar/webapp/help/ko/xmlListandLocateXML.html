XML 문서 찾기

XML 문서 요약 페이지에는 Oracle Service Bus 도메인에 추가한 XML 문서가 나열됩니다.

  1. 다음 작업을 수행합니다.
    • 프로젝트 탐색기를 선택하여 프로젝트 뷰 페이지 또는 프로젝트/폴더 뷰 페이지를 표시합니다. 그런 다음 프로젝트와 폴더를 탐색하여 XML 문서를 찾습니다.
    • 리소스 브라우저 > XML 문서를 선택합니다. 그러면 XML 문서 요약 페이지에 표 13-1과 같은 정보가 표시됩니다. 속성에 대한 자세한 설명은 XML 문서 편집을 참조하십시오.
  2. 특정 XML 문서를 찾으려면 다음과 같이 합니다.
    • 목록의 항목 수를 제한하려는 경우 검색을 누르기 전에 이름 및 경로를 조건으로 필터링할 수 있습니다.

      • 경로는 리소스가 상주하는 프로젝트 이름과 폴더 이름입니다.

    • 모두 보기를 눌러 검색 필터를 제거하고 모든 XML 문서를 표시합니다.

      • 표 13-1 XML 문서 세부 정보
      속성
      설명
      XML 문서
      XML 문서에 지정된 고유한 이름입니다. 이름은 XML 문서 보기 페이지에 대한 링크입니다. XML 문서 편집을 참조하십시오.
      경로
      경로는 XML 문서가 상주하는 프로젝트 이름과 폴더 이름입니다. 경로는 이 리소스가 포함된 프로젝트 또는 폴더에 대한 링크입니다. 프로젝트 및 폴더를 사용하여 리소스 이름 식별을 참조하십시오.
      옵션
      특정 XML 문서를 삭제하려면 삭제 아이콘을 누릅니다. XML 스키마 삭제를 참조하십시오.
