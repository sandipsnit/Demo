XML 문서 편집

XML 문서 편집 페이지에서는 JCA 바인딩의 세부 정보를 확인하고 변경할 수 있습니다.

  1. 아직 이 작업을 수행하지 않은 경우 생성을 눌러 새 세션을 생성하거나 편집을 눌러 기존 세션을 입력합니다. 변경 센터 사용을 참조하십시오.
  2. XML 문서 로드에 설명된 대로 XML 문서를 찾습니다.
  3. XML 문서 이름을 누릅니다. 그러면 XML 문서 보기 페이지에 표 13-2와 같은 정보가 표시됩니다.

    4. 표 13-2 XML 문서 세부 정보
    속성
    설명
    마지막으로 수정한 사람
    이 XML 문서를 생성하거나 구성으로 임포트한 사용자입니다.
    마지막 수정 시간
    사용자가 이 XML 문서를 생성하거나 구성으로 임포트한 날짜와 시간입니다. 이 리소스의 변경 내역을 보려면 날짜 및 시간 링크를 누릅니다. 변경 내역 보기 페이지를 참조하십시오.
    참조
    이 XML 문서가 참조하는 객체의 수입니다. 해당 참조가 존재할 경우 객체 목록을 보려면 숫자 링크를 누릅니다. 리소스에 대한 참조 보기를 참조하십시오.
    참조자
    이 XML 문서를 참조하는 객체의 수입니다. 해당 참조가 존재할 경우 객체 목록을 보려면 숫자 링크를 누릅니다. 예를 들어, XML 파일이 JCA 바인딩에서 참조하는 TopLink 파일이면 이 링크를 누를 때 JCA 바인딩이 참조로 나열됩니다. 리소스에 대한 참조 보기를 참조하십시오.
    설명
    이 XML 문서에 대한 설명입니다(존재하는 경우).

    기본적으로 XML 문서가 읽기 전용 뷰로 표시됩니다.

  4. XML 문서를 편집하려면 편집을 누릅니다. 필드 설명은 XML 문서 추가를 참조하십시오.
  5. 적절히 편집하고 저장을 눌러 현재 세션에 갱신 사항을 커밋합니다.
  6. 세션을 종료하고 구성을 런타임에 배치하려면 변경 센터에서 활성화를 누릅니다.
