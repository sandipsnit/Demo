XQuery 변환 편집

  1. 아직 이 작업을 수행하지 않은 경우 생성을 눌러 새 세션을 생성하거나 편집을 눌러 기존 세션을 입력합니다. 변경 센터 사용을 참조하십시오.
  2. XQuery 변환 찾기에 설명된 대로 XQuery 변환을 찾습니다.
  3. XQuery 변환 이름을 누릅니다. XQuery 변환 세부 정보 보기 페이지에는 표 10-2에 나열된 정보가 표시됩니다.

    4. 표 10-2 XQuery 변환 리소스 세부 정보
    속성
    설명
    마지막으로 수정한 사람
    이 XQuery 변환을 생성하거나 XQuery 변환을 구성으로 임포트한 사용자입니다.
    마지막 수정 시간
    사용자가 이 XQuery 변환을 생성하거나 XQuery 변환을 구성으로 임포트한 날짜와 시간입니다. 날짜 및 시간 링크를 눌러 이 리소스의 변경 내역을 봅니다. 변경 내역 보기 페이지를 참조하십시오.
    참조
    이 XQuery 변환에서 참조하는 객체의 수입니다. 해당 참조가 존재하는 경우 숫자 링크를 누르면 객체 목록이 표시됩니다. 자세한 내용은 리소스에 대한 참조 보기를 참조하십시오.
    참조자
    이 XQuery 변환을 참조하는 객체의 수입니다. 해당 참조가 존재하는 경우 숫자 링크를 누르면 객체 목록이 표시됩니다. 예를 들어, 특정 프록시 서비스의 메시지 흐름에서 XQuery 변환을 변수에 할당한 경우 링크를 누르면 해당 프록시 서비스가 참조로 나열됩니다. 자세한 내용은 리소스에 대한 참조 보기를 참조하십시오.
    설명
    이 XQuery 변환에 대한 설명입니다(존재하는 경우).
    XQuery
    XQuery 변환의 전체 텍스트입니다.
    XQuery 변수
    이 XQuery 변환에 대해 선택한 XQuery 변수입니다.

  4. 필드를 변경하려면 편집을 누릅니다. 필드에 대한 설명은 Xquery 변환 추가를 참조하십시오.

    5. 리소스 이름 필드는 변경할 수 없습니다.

  5. 저장을 눌러 현재 세션에서 갱신을 커밋합니다.
  6. 세션을 종료하고 구성을 런타임에 배치하려면 변경 센터에서 활성화를 누릅니다.
