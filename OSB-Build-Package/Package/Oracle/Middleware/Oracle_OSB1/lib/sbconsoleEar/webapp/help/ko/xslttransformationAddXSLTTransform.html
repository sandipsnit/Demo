XSL 변환 추가

  1. 아직 이 작업을 수행하지 않은 경우 생성을 눌러 새 세션을 생성하거나 편집을 눌러 기존 세션을 입력합니다. 변경 센터 사용을 참조하십시오.
  2. 프로젝트 탐색기를 선택하여 프로젝트 뷰 페이지 또는 프로젝트/폴더 뷰 페이지를 표시합니다.
  3. XSL 변환을 추가할 프로젝트 또는 폴더를 탐색합니다.
  4. 리소스 생성 목록에서 XSLT를 선택하면 새 XSLT 생성 페이지가 표시됩니다.
  5. 리소스 이름 필드에 이 XSL 변환에 대한 고유 이름을 입력합니다.

    리소스 이름 지정 제한 사항의 이름 지정 지침을 따르십시오.
  6. 리소스 설명 필드에 XSL 변환에 대한 설명을 입력합니다.
  7. XSLT 필드에서 다음 작업 중 하나를 수행합니다.
    • 새 XSL 변환에 사용할 텍스트를 입력합니다.
    • 찾아보기를 눌러 기존 XSL 변환을 찾아서 임포트합니다.
    • 기존 XSL 변환의 데이터를 복사하여 이 필드에 붙여 넣습니다.
  8. 저장을 눌러 현재 세션에서 갱신을 커밋합니다.
  9. 세션을 종료하고 구성을 런타임에 배치하려면 변경 센터에서 활성화를 누릅니다.
