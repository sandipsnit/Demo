XSL 변환 찾기

  1. 다음 작업을 수행합니다.
    • 프로젝트 탐색기를 선택하여 프로젝트 뷰 페이지 또는 프로젝트/폴더 뷰 페이지를 표시합니다. 그런 다음 프로젝트와 폴더를 탐색하여 XSL 변환을 찾습니다.
    • 리소스 브라우저 > XSLT를 선택합니다. 그러면 XSLT 요약 페이지에 표 14-1과 같은 정보가 표시됩니다.
  2. 목록에서 항목 수를 제한하기 위해 이름, 경로 또는 이 모두를 사용하여 필터링할 수 있습니다. 이름 및 경로 필드의 검색 아래에서 대상의 이름 및/또는 경로를 입력한 다음 검색 단추를 누릅니다.

    3. 경로는 XSLT가 있는 폴더 이름과 프로젝트 이름입니다.

    와일드 카드 문자 * 및 ?를 사용할 수 있습니다. 검색 결과는 대소문자를 구분합니다.

    모두 보기를 눌러 도메인의 모든 XSLT를 표시합니다. 이렇게 하면 이전 검색에서 검색 매개변수가 지워집니다.

    표 14-1 XSL 변환 세부 정보
    속성
    설명
    XSLT 이름
    XSL 변환에 할당된 이름입니다. 이름을 누르면 XSLT 변환 세부 정보 보기 페이지가 표시됩니다.
    XSL 변환 편집을 참조하십시오.
    경로
    경로는 XSL 변환이 있는 폴더 이름과 프로젝트 이름입니다(예: MyProject/MyFolder/MyResource).
    XSLT 경로를 눌러 XSLT를 포함하는 프로젝트/폴더 뷰 페이지를 표시합니다.
    작업
    서비스 및 변환 설계 검증 및 테스트에 사용할 수 있는 테스트 콘솔을 호출하려면 테스트 콘솔 시작 아이콘을 누릅니다. XSL 변환에 대해 세션의 사용 여부와 관계없이 테스트 콘솔을 사용할 수 있습니다. 자세한 내용은 변환 테스트를 참조하십시오.
    옵션
    아직 이 작업을 수행하지 않은 경우 생성을 눌러 새 세션을 생성하거나 편집을 눌러 기존 세션을 입력합니다. 변경 센터 사용을 참조하십시오.
    삭제 아이콘을 눌러 XSLT를 삭제할 수 있습니다. 다른 리소스에서 이 리소스를 참조하는 경우 삭제 경고 아이콘이 표시됩니다. 경고 확인을 통해 리소스를 삭제할 수 있습니다. 이 경우 삭제된 리소스에 대한 해결되지 않은 참조로 인해 충돌이 발생할 수 있습니다. 자세한 내용은 XSL 변환 삭제를 참조하십시오.
