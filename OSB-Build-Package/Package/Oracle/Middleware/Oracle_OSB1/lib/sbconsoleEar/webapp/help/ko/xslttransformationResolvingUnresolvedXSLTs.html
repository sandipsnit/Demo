해결되지 않은 XSL 변환 참조 해결

  1. 아직 이 작업을 수행하지 않은 경우 생성을 눌러 새 세션을 생성하거나 편집을 눌러 기존 세션을 입력합니다. 변경 센터 사용을 참조하십시오.
  2. XSLT 요약 페이지를 표시하려면 리소스 브라우저 > XSLT를 선택합니다.
  3. XSLT 이름 열에서 해당 XSLT 이름을 누르면 해결되지 않은 참조의 세부 정보를 볼 수 있습니다. XSL 변환 세부 정보 보기 페이지가 표시됩니다.
  4. XSL 변환 리소스의 참조 편집 페이지를 표시하려면 참조 편집을 누릅니다.
  5. 리소스 유형 필드에서 XSLT를 선택합니다.
  6. 찾아보기를 누릅니다. XSLT 브라우저가 표시됩니다.
  7. XSLT 브라우저에서 XSLT를 선택한 다음 종속성 창에서 종속성을 선택합니다.
  8. 제출을 누릅니다. 선택한 XSLT가 리소스 이름 필드에 표시됩니다.
  9. 저장을 누릅니다. XSLT 세부 정보 보기 페이지가 표시됩니다.
  10. 확인을 누릅니다. 해당 XSL 변환이 적합 상태로 표시됩니다.
  11. 세션을 종료하고 구성을 런타임에 배치하려면 변경 센터에서 활성화를 누릅니다.
