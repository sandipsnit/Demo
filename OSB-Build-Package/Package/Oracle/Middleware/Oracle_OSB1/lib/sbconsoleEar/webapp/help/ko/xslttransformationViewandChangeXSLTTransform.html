XSL 변환 편집

  1. 아직 이 작업을 수행하지 않은 경우 생성을 눌러 새 세션을 생성하거나 편집을 눌러 기존 세션을 입력합니다. 변경 센터 사용을 참조하십시오.
  2. XSL 변환 찾기에 설명된 대로 XSL 변환을 찾습니다.
  3. XSL 변환 이름을 누릅니다. 그러면 기본적으로 XSL 변환 세부 정보 보기 페이지에 표 14-2와 같은 정보가 포함된 XSLT의 구조적 뷰가 표시됩니다.

    4. 표 14-2 XSL 변환 리소스 세부 정보 
    속성
    설명
    마지막으로 수정한 사람
    이 XSL 변환을 생성하거나 XSL 변환을 구성으로 임포트한 사용자입니다.
    마지막 수정 시간
    사용자가 이 XSL 변환을 생성하거나 XSL 변환을 구성으로 임포트한 날짜와 시간입니다. 날짜 및 시간 링크를 눌러 이 리소스의 변경 내역을 봅니다. 변경 내역 보기 페이지를 참조하십시오.
    참조
    이 XSL 변환에서 참조하는 객체의 수입니다. 해당 참조가 존재하는 경우 숫자 링크를 누르면 객체 목록이 표시됩니다. 자세한 내용은 리소스에 대한 참조 보기를 참조하십시오.
    참조자
    이 XSL 변환을 참조하는 객체의 수입니다. 해당 참조가 존재하는 경우 숫자 링크를 누르면 객체 목록이 표시됩니다. 예를 들어, 특정 프록시 서비스의 메시지 흐름에서 이 XSL 변환을 변수에 할당한 경우 링크를 누르면 해당 프록시 서비스가 참조로 나열됩니다. 자세한 내용은 리소스에 대한 참조 보기를 참조하십시오.
    설명
    이 XSL 변환에 대한 설명입니다.
    대상 네임스페이스
    XSL 변환에 포함된 정의를 식별하는 데 사용되는 네임스페이스입니다.
    XSLT 종속성
    XSLT와 관련된 속성 및 그룹입니다.
    • XSLT 위치
    상태
    XSL 변환의 상태입니다.
    • 적합

      • 현재 XSL 변환에 포함된 모든 XSL 변환 위치가 지정되고 적합합니다. 또한 중첩된 XSL 변환(포함된 XSL 변환에 포함된 XSL 변환)의 모든 위치가 지정되고 적합합니다.

    • 부적합

      • 현재 XSL 변환에 포함된 XSL 변환의 위치가 한 개 이상 지정되지 않았거나 적합하지 않습니다. 또한 중첩된 XSL 변환(포함된 XSL 변환에 포함된 XSL 변환)이 한 개 이상 지정되지 않았거나 적합하지 않을 수 있습니다. 부적합한 XSL 변환을 해결하려면 참조 편집을 누릅니다. 자세한 내용은 해결되지 않은 XSL 변환 참조 해결을 참조하십시오.

  4. 텍스트 뷰를 눌러 XSL 변환 세부 정보의 텍스트 뷰를 표시합니다. 텍스트 뷰에는 표 14-3과 같은 정보가 포함됩니다.

    5. 표 14-3 XSL 변환 텍스트 세부 정보
    속성
    설명
    대상 네임스페이스
    XSL 변환에 포함된 종속성을 식별하는 데 사용되는 네임스페이스입니다.
    XSLT
    XSL 변환의 전체 텍스트입니다.

  5. 필드를 변경하려면 편집을 누릅니다. 필드에 대한 설명은 XSL 변환 추가를 참조하십시오.

    6. 리소스 이름 필드는 변경할 수 없습니다.

  6. 저장을 눌러 현재 세션에서 갱신을 커밋합니다.
