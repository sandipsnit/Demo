Editando Configurações do Serviço de Negócios

  1. Se ainda não tiver feito isso, clique em Criar para criar uma nova sessão ou clique em Editar para informar uma sessão existente. Consulte Usando o Centro de Alterações.

    2. Você não precisa entrar em uma sessão se só quiser exibir os detalhes.

  2. Localize o serviço de negócios que você deseja exibir ou editar, conforme explicado em Localizando Serviços de Negócios.
  3. Clique no nome do serviço de negócios. A página Exibir um Serviço de Negócios - Detalhes da Configuração exibe informações de configuração do serviço de negócios selecionado.
  4. Para exibir ou modificar as definições, execute um destes procedimentos:
    • Clique no ícone Editar ao lado do nome da categoria cujas propriedades você deseja exibir ou editar (por exemplo, Configuração Geral, Configuração do Transporte, etc.). As páginas que você pode editar dependem das páginas que configurou durante a criação do serviço de negócios.

      • Para obter uma lista de todas essas páginas, consulte a página Exibir um Serviço de Negócios - Detalhes da Configuração.

    • Clique em Editar na parte inferior da página para exibir a página Criar/Editar um Serviço de Negócios - Configuração Geral, que é a primeira página na sequência de páginas para configuração deste serviço de negócios.
  5. Continue para exibir ou editar, conforme descrito em Criando e Configurando Serviços de Negócios.
  6. Na página Editar um Serviço de Negócios- página Resumo, clique em Salvar para confirmar as atualizações na sessão atual.
  7. Para encerrar a sessão e implantar a configuração no run-time, clique em Ativar em Centro de Alterações.
