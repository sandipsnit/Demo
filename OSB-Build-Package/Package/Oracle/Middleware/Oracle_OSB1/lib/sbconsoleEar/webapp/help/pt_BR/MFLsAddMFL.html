Adicionando MFLs

  1. Se ainda não tiver feito isso, clique em Criar para criar uma nova sessão ou clique em Editar para informar uma sessão existente. Consulte Usando o Centro de Alterações.
  2. Selecione o Project Explorer, depois selecione um projeto ou pasta no qual adicionar o arquivo MFL. A página View de Projeto/Pasta é mostrada.
  3. Na lista drop-down Criar Recurso, selecione Arquivo MFL em Transformação para exibir a página Criar um Novo Recurso de Arquivo MFL.
  4. No campo Nome do Recurso, informe um nome exclusivo para esse MFL. Esse campo é obrigatório.

    Siga as Restrições para Nomeação de Recursos para obter orientações em relação a nomes.
  5. No campo Descrição do Recurso, informe uma descrição para o MFL.
  6. No campo Texto MFL, execute um dos seguintes procedimentos:
    • Informe texto para o novo MFL.
    • Clique em Procurar para localizar e importar um MFL existente.
    • Copie e cole texto a partir de um MFL existente neste campo.

      • Esse campo é obrigatório.

  7. Clique em Salvar. O recurso do arquivo MFL é criado e salvo na sessão atual.
  8. Para encerrar a sessão e implantar a configuração no run-time, clique em Ativar em Centro de Alterações.
