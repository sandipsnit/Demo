Localizando MFLs

  1. Execute um dos seguintes procedimentos:
    • Selecione o Project Explorer para exibir a página View de Projetos ou a página View de Projeto/Pasta. Em seguida, navegue pelos projetos e pastas para localizar o MFL.
    • Selecione Resource Browser > MFLs. A página Resumo de Arquivos MFL exibe as informações mostradas na Tabela 15-1. Para obter uma descrição mais detalhada das propriedades, consulte Editando MFLs.
  2. Para restringir o número de itens na lista, você pode filtrar por nome, caminho ou ambos. Nos campos Nome e Caminho, informe o nome e o caminho do(s) destino(s) de pesquisa e, em seguida, clique em Pesquisar.

    3. O caminho é o nome do projeto e o nome da pasta em que reside o arquivo MFL.

    Clique em Exibir Tudo para remover os filtros de pesquisa e exibir todos os arquivos MFL.

    Tabela 15-1 - Detalhes do MFL
    Propriedade
    Descrição
    Nome do Arquivo MFL
    O nome exclusivo designado ao MFL. O nome é um link para a página Exibir Detalhes do Recurso MFL. Consulte Editando MFLs.
    Caminho
    O caminho é o nome do projeto e o nome da pasta em que reside o arquivo MFL. É um link para o projeto ou pasta que contém este recurso. Consulte Qualificando Nomes de Recursos Usando Projetos e Pastas.
    Ações
    Clique no ícone Acionar Console de Teste para chamar a Console de Teste, que você usa para validar e testar o design de seus serviços e transformações. Para MFLs, você pode usar a Console de Teste estando dentro ou fora de uma sessão. Consulte Testando Transformações.
    Opções
    Clique no ícone Deletar para deletar um MFL específico. Consulte Deletando MFLs.
