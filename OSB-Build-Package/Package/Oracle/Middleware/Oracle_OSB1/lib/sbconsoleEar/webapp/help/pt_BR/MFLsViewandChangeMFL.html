Editando MFLs

  1. Se ainda não tiver feito isso, clique em Criar para criar uma nova sessão ou clique em Editar para informar uma sessão existente. Consulte Usando o Centro de Alterações.
  2. Localize o arquivo MFL, conforme descrito em Localizando MFLs.
  3. Clique no nome do arquivo MFL. A página Exibir Detalhes do Recurso MFL exibe as informações mostradas na Tabela 15-2.

    4. Tabela 15-2 Detalhes do Recurso MFL 
    Propriedade
    Descrição
    Última Modificação
    O usuário que criou este arquivo XML ou o importou para a configuração.
    Última Modificação em
    A data e hora em que o usuário criou este arquivo MFL ou o importou para a configuração. Clique no link da data e hora para exibir o histórico de alterações deste recurso. Consulte a página Exibir Histórico de Alteração.
    Referências
    O número de objetos referenciados por este arquivo MFL. Se tais referências existirem, clique no link numérico para exibir uma lista dos objetos. Consulte Exibindo Referências a Recursos.
    Referenciado por
    O número de objetos que referenciam esse arquivo MFL. Se tais referências existirem, clique no link numérico para exibir uma lista dos objetos. Por exemplo, se você selecionou esta porta ou binding de WSDL como o tipo de mensagem de solicitação ou resposta para um serviço de negócios de mensagens ou de proxy, esse serviço de negócios ou de proxy estará listado como referência quando você clicar no link. Consulte Exibindo Referências para Recursos.
    Descrição
    Uma descrição deste arquivo MFL, se existir.
    Texto MFL
    O texto deste MFL.

  4. Para fazer uma alteração nos campos, clique em Editar. Para obter descrições dos campos, consulte Adicionando MFLs.

    5. Não é possível alterar o campo Nome do Recurso.

  5. Clique em Salvar para confirmar as atualizações na sessão atual.
  6. Para encerrar a sessão e implantar a configuração no run-time, clique em Ativar em Centro de Alterações.
