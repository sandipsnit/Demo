Deletando Destinos de Alerta

Para deletar um destino de alerta:

  1. Se ainda não tiver feito isso, clique em Criar para criar uma nova sessão ou clique em Editar para informar uma sessão existente. Consulte Usando o Centro de Alterações.
  2. Selecione Resource Browser > Destinos de Alerta para exibir a página Resumo de Destinos de Alerta.
  3. Clique no ícone Deletar no campo Opções do destino de alerta que você deseja deletar. O destino de alerta é deletado na sessão atual. Um ícone Advertência de Deleção será exibido quando outros recursos referenciarem este recurso. É possível deletar o recurso com uma confirmação de advertência. Isso pode resultar em conflitos devido a referências não resolvidas referentes ao recurso deletado.

    4. Para deletar destinatários de e-mail ou destinos JMS específicos configurados em um destino de alerta, consulte Editando Destinos de Alerta.

  4. Para encerrar a sessão e implantar a configuração no run-time, clique em Ativar em Centro de Alterações.
