Adicionando Destinos JMS

  1. Selecione Enviar um alerta para um Destino JMS na lista drop-down e clique em Adicionar para exibir a página Adicionar Destino JMS.
  2. No campo do URI de destino, informe um URI de destino JMS no formato: jms://host:port/factoryJndiName/destJndiName

    3. Esse campo é obrigatório.

    Observação: Embora o Oracle WebLogic Server permita o uso de barras em nomes JNDI, como "myqueues/myqueue", os nomes JNDI com barras interferem no formato de URI exigido pelo Oracle Service Bus e você não pode usá-los. Para resolver esse problema, defina e referencie um servidor externo JMS no URI. Para obter mais informações, consulte "Configure foreign servers" em Oracle Fusion Middleware Oracle WebLogic Server Administration Console Online Help em http://www.oracle.com/pls/as111170/lookup?id=WLACH.

  3. No campo Tipo de Destino, selecione Fila ou Tópico.
  4. No campo Tipo de Mensagem, selecione Bytes ou Texto.
  5. No campo Codificação da Solicitação, informe um valor de codificação de conjunto de caracteres. O valor de codificação default é UTF-8.
  6. Clique em Salvar. O destino JMS será adicionado à lista de destinos JMS a serem incluídos para este destino de alerta e salvo na sessão atual.
