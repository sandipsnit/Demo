Criar/Editar um Serviço de Negócios - Referência de Página

Crie um serviço de negócios selecionando Serviço de Negócios na lista drop-down Criar Recurso na página View de Projeto/Pasta. Quando você seleciona essa opção, ela exibe a primeira de uma série de páginas para configuração e adição de serviços de negócios. As páginas exibidas variam dependendo das opções escolhidas. As páginas são:

  • Página Configuração Geral
  • Página Configuração do Tipo de Mensagem
  • Página Configuração do Transporte
  • Página Configuração do Transporte Específico ao Protocolo
  • Página Configuração de Binding de SOAP
  • Página Configuração do Tratamento de Mensagens
  • Página Resumo
  • Página Políticas do Serviço de Negócios
  • Página Segurança do Serviço de Negócios
