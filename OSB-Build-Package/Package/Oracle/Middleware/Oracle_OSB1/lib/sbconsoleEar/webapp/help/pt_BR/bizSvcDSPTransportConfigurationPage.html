Página Configuração do Transporte DSP

A página Criar/Editar um Serviço de Negócios - Configuração do Transporte DSP pertence a uma série de páginas para criação e edição de serviços de negócios, conforme descrito em Criando e Configurando Serviços de Negócios e Editando Configurações do Serviço de Negócios.

Para obter mais informações, consulte "DSP and Oracle Data Service Integrator Transport" no Oracle Fusion Middleware Developer's Guide for Oracle Service Bus em http://www.oracle.com/pls/as111170/lookup?id=OSBDV1229.

Use esta página para configurar as definições de transporte para um serviço de negócios usando o protocolo de transporte DSP (Oracle Data Service Integrator). Tabela 18-4 descreve como usar esta página.

Tabela 18-4 - Página Criar/Editar um Serviço de Negócios - Configuração do Transporte DSP
Opção
Para criar ou editar...
Nível de Depuração
Especifique um dos seguintes
  • 0 - para nenhuma informação de depuração
  • 1 - gerar informações na mensagem de solicitação
  • 3 - gerar informações na mensagem de solicitação e na mensagem de resposta
Conta do Serviço
Clique em Procurar e selecione uma conta de serviço na lista exibida. Se nenhuma conta de serviço for especificada, será usado um assunto anônimo.

Para obter mais informações:

  • Contas de Serviço
  • Adicionando Contas de Serviço
Política de Distribuição
Selecione a instância do Gerenciador de Trabalho do WebLogic Server que você deseja usar para a política de despacho deste ponto final. O Gerenciador de Trabalho default será usado se nenhum outro existir.
O Gerenciador de Trabalho é usado para postar a mensagem de resposta para processamento da resposta. Para obter informações sobre Gerenciadores de Trabalho, consulte "Using Work Managers to Optimize Scheduled Work" no Oracle Fusion Middleware Configuring Server Environments for Oracle WebLogic Server em http://www.oracle.com/pls/as111170/lookup?id=CNFGD112.

Depois de finalizar

Clique em Próximo para continuar configurando este serviço na próxima página; ou clique em Último para rever e salvar essa configuração na página Criar/Editar um Serviço de Negócios - Resumo.
