Página Configuração do Transporte FTP

A página Criar/Editar um Serviço de Negócios - Configuração do Transporte FTP pertence a uma série de páginas para criação e edição de serviços de negócios, conforme descrito em Criando e Configurando Serviços de Negócios e Editando Configurações do Serviço de Negócios.

Use esta página para configurar as definições de transporte para um serviço de negócios usando o protocolo de transporte FTP. Tabela 18-9 descreve como usar esta página.

Tabela 18-9 - Página Criar/Editar um Serviço de Negócios - Configuração do Transporte FTP
Opção
Para criar ou editar...
Autenticação do Usuário
Selecione anônimo se o usuário do servidor FTP for anônimo ou selecione usuário externo se o usuário do servidor FTP for uma conta configurada externamente.
Identidade (Id do e-mail)
Este campo só está disponível se a opção Autenticação do Usuário estiver definida como anônimo.
Informe o ID de e-mail do usuário anônimo.
Conta do Serviço
Este campo só está disponível se a opção Autenticação do Usuário estiver definida como usuário externo.
Informe a conta de serviço para o usuário externo.

Para obter mais informações:

  • Contas de Serviço
  • Adicionando Contas de Serviço.
Timeout
Informe o intervalo de timeout do soquete, em segundos, antes que a conexão seja eliminada. O default são 60 segundos.
Prefixo para Nome do Arquivo de destino
Informe um prefixo para o nome do arquivo com o qual o arquivo será armazenado no servidor remoto.
Não informe * neste campo. Esse caractere causa uma exceção de run-time.
Sufixo para Nome do Arquivo de destino
Informe um sufixo para o nome do arquivo com o qual o arquivo será armazenado no servidor remoto.
Não informe * neste campo. Esse caractere causa uma exceção de run-time.
Modo de Transferência
Selecione ASCII ou binário como o modo de transferência.
Codificação da Solicitação
Aceite o default UTF-8 como a codificação de conjunto de caracteres para solicitações em transportes ftp ou informe outra codificação de conjunto de caracteres.

Depois de finalizar

Clique em Próximo para continuar configurando este serviço na próxima página; ou clique em Último para rever e salvar essa configuração na página Criar/Editar um Serviço de Negócios - Resumo.
