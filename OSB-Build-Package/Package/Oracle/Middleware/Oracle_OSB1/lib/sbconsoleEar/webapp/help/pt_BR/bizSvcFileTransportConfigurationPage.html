Página Configuração do Transporte de Arquivos

A página Criar/Editar um Serviço de Negócios - Configuração do Transporte de Arquivos pertence a uma série de páginas para criação e edição de serviços de negócios, conforme descrito em Criando e Configurando Serviços de Negócios e Editando Configurações do Serviço de Negócios.

Use esta página para configurar as definições de transporte para um serviço de negócios usando o protocolo de transporte de arquivos. Tabela 18-7 descreve como usar esta página.

Tabela 18-7 - Página Criar/Editar um Serviço de Negócios - Configuração do Transporte de Arquivos
Opção
Para criar ou editar...
Prefixo
Informe um prefixo a ser pré-anexado ao nome do arquivo.
Não informe * neste campo. Esse caractere causa uma exceção de run-time.
Sufixo
Informe um sufixo a ser anexado ao nome do arquivo. Esse campo é obrigatório.
Não informe * neste campo. Esse caractere causa uma exceção de run-time.
Codificação da Solicitação
Aceite o default utf-8 como a codificação de conjunto de caracteres para solicitações em transportes de arquivo ou informe outra codificação de conjunto de caracteres.

Depois de finalizar

Clique em Próximo para continuar configurando este serviço na próxima página; ou clique em Último para rever e salvar essa configuração na página Criar/Editar um Serviço de Negócios - Resumo.
