Página Configuração do Transporte de Fluxo

A página Criar/Editar um Serviço de Negócios - Configuração do Transporte de Fluxo pertence a uma série de páginas para criação e edição de serviços de negócios, conforme descrito em Criando e Configurando Serviços de Negócios e Editando Configurações do Serviço de Negócios.

Use serviços de negócios do tipo de transporte de Fluxo para acessar Split-Joins. Para saber mais sobre Split-Joins, consulte "Improving Service Performance with Split-Join" no Oracle Fusion Middleware Developer's Guide for Oracle Service Bus em http://www.oracle.com/pls/as111170/lookup?id=OSBAG291.

Use esta página para configurar as definições de transporte para um serviço de negócios usando o protocolo de transporte de Fluxo. Tabela 18-8 descreve como usar esta página.

Tabela 18-8 - Página Criar/Editar um Serviço de Negócios - Configuração do Transporte de Fluxo
Opção
Para criar ou editar...
Timeout
Informe o número de segundos para aguardar uma resposta. Esse valor é ignorado para uma operação apenas de solicitação (unilateral).

Depois de finalizar

Clique em Próximo para continuar configurando este serviço na próxima página; ou clique em Último para rever e salvar essa configuração na página Criar/Editar um Serviço de Negócios - Resumo.
