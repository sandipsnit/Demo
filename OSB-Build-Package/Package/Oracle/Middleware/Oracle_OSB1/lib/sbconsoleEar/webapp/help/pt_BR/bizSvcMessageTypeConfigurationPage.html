Página Configuração do Tipo de Mensagem

A página Criar/Editar um Serviço de Negócios - Configuração do Tipo de Mensagem pertence a uma série de páginas para criação e edição de serviços de negócios, conforme descrito em Criando e Configurando Serviços de Negócios e Editando Configurações do Serviço de Negócios.

Use esta página para configurar os tipos de mensagens para um serviço de negócios cujo tipo é Serviço de Mensagens.

A definição de binding para serviços de mensagens consiste em configurar o tipo de conteúdo das mensagens que são trocadas. O tipo de conteúdo para a resposta não precisa ser o mesmo para a solicitação; portanto, a resposta é configurada separadamente (por exemplo, o serviço pode aceitar uma mensagem MFL e retornar um recebimento de confirmação XML).

Observação: Serviços de negócios de transporte de E-mail, Arquivo, FTP ou SFTP cujo tipo seja Serviço de Mensagens suportam mensagens unilaterais apenas; o Tipo de Mensagem da Resposta deve ser nenhum. Se você selecionar uma opção diferente de nenhum, o protocolo de arquivo, ftp ou sftp não estará disponível na página Configuração do Transporte.

Tabela 18-2 descreve como usar a página Criar/Editar um Serviço de Negócios - Configuração do Tipo de Mensagem.

Tabela 18-2 - Página Criar/Editar um Serviço de Negócios - Configuração do Tipo de Mensagem
Opção
Para criar ou editar...
Tipo de Mensagem de Solicitação
Selecione um tipo de mensagem para a mensagem de solicitação:
  • Nenhum - Selecione esta opção se não houver mensagem de solicitação (exemplo HTTP GET)
  • Binário - Selecione esta opção se o tipo de conteúdo da mensagem for desconhecido ou não for importante.
  • Texto - Selecione esta opção se a mensagem puder ser restrita a texto.
  • MFL - Selecione esta opção se a mensagem for um documento binário de acordo com uma definição MFL. É possível configurar apenas um arquivo MFL.

    • Para MFLs, você pode clicar em Procurar para selecionar um MFL no Browser MFL e, em seguida, clicar em Submeter.

  • XML - Selecione esta opção se a mensagem for um documento XML. Para fornecer informações sobre o tipo, é possível optar por declarar o tipo de esquema XML do documento XML trocado.
  • Java - Selecione esta opção se o Objeto Java estiver sendo enviado na solicitação. O transporte de JMS é usado para mensagens do Objeto Java. Para obter mais informações, consulte "Sending and Receiving Java Objects in Messages" no Oracle Fusion Middleware Developer's Guide for Oracle Service Bus em http://www.oracle.com/pls/as111170/lookup?id=OSBDV1874.
Tipo de Mensagem de Resposta
Selecione um tipo de mensagem para a mensagem de resposta:
  • Nenhum - Selecione esta opção se não houver mensagem de resposta.
  • Binário - Selecione esta opção se o tipo de conteúdo da mensagem for desconhecido ou não for importante.
  • Texto - Selecione esta opção se a mensagem puder ser restrita a texto.
  • MFL - Selecione esta opção se a mensagem for um documento binário de acordo com uma definição MFL. É possível configurar apenas um arquivo MFL.

    • Para MFLs, você pode clicar em Procurar para selecionar um MFL no Browser MFL e, em seguida, clicar em Submeter.

  • XML - Selecione esta opção se a mensagem for um documento XML. Para fornecer informações sobre o tipo, é possível optar por declarar o tipo de esquema XML do documento XML trocado.
  • Java - Selecione esta opção se um Objeto Java estiver sendo recebido na resposta. O transporte de JMS é usado para mensagens do Objeto Java. Para obter mais informações, consulte "Sending and Receiving Java Objects in Messages" no Oracle Fusion Middleware Developer's Guide for Oracle Service Bus em http://www.oracle.com/pls/as111170/lookup?id=OSBDV1874.

Depois de finalizar

Clique em Próximo para continuar configurando este serviço na próxima página; ou clique em Último para rever e salvar essa configuração na página Criar/Editar um Serviço de Negócios - Resumo.
