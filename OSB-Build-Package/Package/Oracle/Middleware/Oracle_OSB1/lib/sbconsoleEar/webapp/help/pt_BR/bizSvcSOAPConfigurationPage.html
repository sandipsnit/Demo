Página Configuração de Binding de SOAP

A página Criar/Editar um Serviço de Negócios - Configuração de Binding de SOAP pertence a uma série de páginas para criação e edição de serviços de negócios, conforme descrito em Criando e Configurando Serviços de Negócios e Editando Configurações do Serviço de Negócios.

Use esta página para configurar o Binding SOAP para um serviço de negócios baseado em um WSDL.

Selecione Impor Conformidade WS-I para especificar se ou não o serviço está de acordo com o Perfil Básico definido pela Organização de Interoperabilidade de Web Services. Esta opção só está disponível para os serviços SOAP 1.1

Quando um serviço é marcado como compatível com WS-I, verificações serão executadas nas mensagens enviadas para e desse serviço.

Depois de finalizar

Clique em Próximo para continuar configurando este serviço na próxima página; ou clique em Último para rever e salvar essa configuração na página Criar/Editar um Serviço de Negócios - Resumo.
