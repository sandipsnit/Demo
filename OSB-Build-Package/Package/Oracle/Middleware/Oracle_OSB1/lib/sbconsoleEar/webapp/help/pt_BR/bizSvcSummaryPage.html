Página Resumo

A página Criar/Editar um Serviço de Negócios - Resumo pertence a uma série de páginas para criação e edição de serviços de negócios, conforme descrito em Criando e Configurando Serviços de Negócios e Editando Configurações do Serviço de Negócios.

Use esta página para exibir ou modificar as definições de configuração para um serviço de negócios antes de salvá-lo.

Para exibir ou modificar definições, clique em Editar na linha da categoria apropriada (por exemplo, Configuração Geral, Configuração de Transporte, etc.). As páginas que você pode editar dependem das páginas que configurou durante a criação do serviço de negócios. A lista a seguir mostra todas as páginas:

  • Página Configuração Geral
  • Página Configuração do Tipo de Mensagem
  • Página Configuração do Transporte
  • Página Configuração do Transporte Específico ao Protocolo
  • Página Políticas do Serviço de Negócios
  • Página Segurança do Serviço de Negócios
