Página Configuração do Transporte WS

A página Criar/Editar um Serviço de Negócios - Configuração do Transporte WS pertence a uma série de páginas para criação e edição de serviços de negócios, conforme descrito em Criando e Configurando Serviços de Negócios e Editando Configurações do Serviço de Negócios.

Para obter mais informações, consulte "WS Transport" no Oracle Fusion Middleware Developer's Guide for Oracle Service Bus em http://www.oracle.com/pls/as111170/lookup?id=OSBDV1083.

Use esta página para configurar as definições de transporte para um serviço de negócios usando o protocolo de transporte WS. Tabela 18-19 descreve como usar a página.

Tabela 18-19 - Página Criar/Editar um Serviço de Negócios - Configuração do Transporte WS
Opção
Para criar ou editar...
Timeout da Resposta
Informe o número de segundos para aguardar uma resposta.
Deixar este campo em branco indica que não há timeout de resposta. O serviço de negócios aguardará a duração do timeout de sequência configurado na política RM.
Se você informar um valor zero (0), não haverá timeout; portanto, nunca atingirá o timeout.
Conta do Serviço
Informe uma conta de serviço ou clique em Procurar para selecionar uma na lista exibida.
A conta de serviço especifica as credenciais a serem usadas quando há uma autenticação básica HTTP neste serviço.
Definições Avançadas
Mensagens de Erro da Fila
Marque a caixa de seleção para ativar o envio de mensagens de erro para a fila de erros configurada.
URI da Fila de Erros
Esta opção está disponível apenas se a caixa de seleção Mensagens de Erro da Fila estiver marcada.
Informe o URI da fila JMS para armazenar mensagens de erro, no seguinte formato:

jms://host:port/connFactoryJndiName/queueJndiName

Observação: Embora o Oracle WebLogic Server permita o uso de barras em nomes JNDI, como "myqueues/myqueue", os nomes JNDI com barras interferem no formato de URI exigido pelo Oracle Service Bus e você não pode usá-los. Para resolver esse problema, defina e referencie um servidor externo JMS no URI. Para obter mais informações, consulte "Configure foreign servers" em Oracle Fusion Middleware Oracle WebLogic Server Administration Console Online Help em http://www.oracle.com/pls/as111170/lookup?id=WLACH.

Conta de Serviço da Fila de Erros JMS
Esta opção está disponível apenas se a caixa de seleção Mensagens de Erro da Fila estiver marcada.
Informe uma conta de serviço ou clique em Procurar para selecionar uma na lista exibida.
A conta de serviço especifica as credenciais a serem usadas para pesquisas de JNDI e para o envio de mensagens de erro para a fila de erros.
Usar SSL para Fila de Erros
Esta opção está disponível apenas se a caixa de seleção Mensagens de Erro da Fila estiver marcada.
Marque a caixa de seleção para usar o SSL para conexão com a fila de erros.

Depois de finalizar

Clique em Próximo para continuar configurando este serviço na próxima página; ou clique em Último para rever e salvar essa configuração na página Criar/Editar um Serviço de Negócios - Resumo.
