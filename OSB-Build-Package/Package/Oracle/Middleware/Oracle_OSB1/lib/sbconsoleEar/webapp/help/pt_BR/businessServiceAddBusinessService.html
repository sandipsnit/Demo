Criando e Configurando Serviços de Negócios

Esta seção descreve como criar e configurar serviços de negócios.

Para serviços JCA, você pode gerar serviços de negócios a partir de Bindings de JCA, conforme descrito em Gerando um Serviço de Negócios a partir de um Recurso de Binding de JCA, ou no Oracle Enterprise Repository, conforme descrito em Gerando um Serviço de Negócios a partir do Oracle Enterprise Repository.

  1. Se ainda não tiver feito isso, clique em Criar para criar uma nova sessão ou clique em Editar para informar uma sessão existente. Consulte Usando o Centro de Alterações.
  2. Selecione o Project Explorer, depois selecione o projeto ou pasta em que você deseja adicionar o serviço de negócios.
  3. Na página View de Projeto/Pasta, selecione Serviço de Negócios na lista drop-down Criar Recurso.
  4. Em Criar/Editar um Serviço de Negócios - página Configuração Geral, informe um nome para o serviço e selecione o tipo de serviço a ser criado.

    Siga as Restrições para Nomeação de Recursos para obter orientações em relação a nomes.
  5. Clique em Próximo. As páginas seguintes dependem das escolhas feitas na primeira página. Insira as informações apropriadas em cada uma das páginas subsequentes até chegar à página de resumo e, em seguida, clique em Salvar para salvar o serviço na sessão atual.

    6. Para obter instruções detalhadas sobre o preenchimento de cada página, consulte o seguinte:

    • Página Configuração Geral
    • Página Configuração do Tipo de Mensagem
    • Página Configuração do Transporte
    • Página Configuração do Transporte Específico ao Protocolo
    • Página Configuração de Binding de SOAP
    • Página Configuração do Tratamento de Mensagens
    • Página Resumo
    • Página Políticas do Serviço de Negócios
    • Página Segurança do Serviço de Negócios

Se um serviço de negócios for criado com base em um WSDL que inclua anexos de Política de Web Services, as políticas (somente para leitura) serão exibidas na Protocol-Specific Transport Configuration page. Se qualquer uma das Políticas de Web Services do serviço especificar autenticação, então você deverá selecionar uma conta de serviço. Um serviço de proxy que encaminha para este serviço de negócios usará essa conta de serviço para autenticação no serviço de negócios.

Gerando um Serviço de Negócios a partir de um Recurso de Binding de JCA

É possível gerar um serviço de negócios de JCA a partir de um recurso de Binding de JCA de saída no Oracle Service Bus. Para obter mais informações sobre os recursos de Binding de JCA, consulte o Capítulo 12, Bindings de JCA.

Para gerar um serviço de negócios JCA a partir de um Binding de JCA:

  1. Na Console do Oracle Service Bus Administration, clique em Criar ou em Editar no Centro de Alterações, caso ainda não esteja no modo Criar ou Editar.
  2. No Browser do Recurso, clique em Bindings de JCA.
  3. Localize o Binding de JCA a partir do qual você deseja gerar um serviço e clique no ícone Ação.
  4. Na janela que aparece, confirme o nome do WSDL e do serviço que você deseja gerar, selecione o local para esses novos recursos e clique em Gerar.

    5. O Oracle Service Bus gera o serviço e seu WSDL correspondente.

  5. Modifique outros detalhes de configuração no serviço gerado, conforme apropriado, como o URI do Ponto Final.

Gerando um Serviço de Negócios a partir do Oracle Enterprise Repository

Você pode gerar um serviço de negócios com base no Oracle Enterprise Repository usando o ambiente de desenvolvimento do Oracle Service Bus. Para obter mais informações, consulte "Generating a Business Service from Oracle Enterprise Repository" no Oracle Fusion Middleware Developer's Guide for Oracle Service Bus em http://www.oracle.com/pls/as111170/lookup?id=OSBDV1797.
