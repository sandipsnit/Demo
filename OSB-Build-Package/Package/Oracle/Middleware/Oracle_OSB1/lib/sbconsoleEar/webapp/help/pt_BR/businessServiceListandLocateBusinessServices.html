Localizando Serviços de Negócios

  1. Execute um dos seguintes procedimentos:
    • Selecione o Project Explorer para exibir a página View de Projetos ou a página View de Projeto/Pasta. Em seguida, navegue pelos projetos e pastas para localizar o serviço de negócios.
    • Selecione Resource Browser > Serviços de Negócios. A página Resumo de Serviços de Negócios exibe as informações mostradas na Tabela 18-21. Para obter uma descrição mais detalhada das propriedades, consulte Editando Configurações do Serviço de Negócios.
  2. Para restringir o número de itens na lista, você pode filtrar por nome, caminho ou ambos. Nos campos Nome e Caminho, em Pesquisar, informe o nome e/ou o caminho do(s) destino(s) e, em seguida, clique no botão Pesquisar.

    3. O caminho é o nome do projeto e o nome da pasta em que reside o serviço de negócios.

    Caracteres curingas * e ? são permitidos. A pesquisa faz distinção entre maiúsculas e minúsculas.

    Clique em Exibir Tudo para exibir todos os serviços de negócios no domínio. Isso limpa os parâmetros de pesquisa da pesquisa anterior.

    Tabela 18-21 - Página Resumo de Serviços de Negócios
    Propriedade
    Descrição
    Nome
    Um nome exclusivo para o serviço de negócios. Clique no nome para exibir a página Exibir um Serviço de Negócios - Detalhes da Configuração.
    Consulte Editando Configurações do Serviço de Negócios.
    Caminho
    O caminho é o nome do projeto e o nome da pasta em que reside o serviço de negócios, por exemplo, UDDI/BusServices/OSB_services.
    Clique no caminho de um serviço de negócios para exibir a página View de Projeto/Pasta que o contém.
    Ações
    Execute uma das etapas a seguir:
    • Clique no ícone Acionar Console de Teste para chamar a Console de Teste, que você usa para validar e testar o design de seus serviços e transformações. Para serviços de negócios, você só pode usar a Console de Teste no run-time, ou seja, quando a sessão for ativada. Para transformações, é possível usar a Console de Teste dentro ou fora de uma sessão. Consulte Testando Serviços e Testando Transformações.
    • O ícone Exportar WSDL é exibido para serviços de negócios baseados no WSDL. Clique nesse ícone para exportar um WSDL, que você pode exibir ou modificar em uma ferramenta externa, como um IDE.

      • Observe que isso é diferente da funcionalidade Exportar Recursos no módulo Administração do Sistema, que você usa para mover e separar recursos entre dois domínios. Consulte Exportando um WSDL.

    • O ícone Gerar WSDL é exibido para serviços de negócios do tipo de transporte, como EJB e JEJB (mas não para serviços de fluxo Split-Join). Clique neste ícone para gerar um WSDL, que você pode depois exibir ou modificar. Consulte Gerando um WSDL.
    Opções
    • Clique no ícone Deletar para deletar o serviço. Um ícone Advertência de Deleção é exibido quando outros recursos referenciam esse recurso. É possível deletar o recurso com uma confirmação de advertência. Isso pode resultar em conflitos devido a referências não resolvidas referentes ao recurso deletado. Para obter mais informações, consulte Deletando Serviços de Negócios.
