Deletando Serviços de Negócios

  1. Se ainda não tiver feito isso, clique em Criar para criar uma nova sessão ou clique em Editar para informar uma sessão existente. Consulte Usando o Centro de Alterações.
  2. Selecione Resource Browser > Serviços de Negócios para exibir a página Resumo de Serviços de Negócios.
  3. Clique no ícone Deletar na linha do serviço de negócios que você deseja deletar. O serviço de negócios é deletado na sessão atual. Um ícone Advertência de Deleção é exibido quando outros recursos referenciam esse recurso. É possível deletar o recurso com uma confirmação de advertência. Isso pode resultar em conflitos devido a referências não resolvidas referentes ao recurso deletado.
  4. Para encerrar a sessão e implantar a configuração no run-time, clique em Ativar em Centro de Alterações.
