Exportando um WSDL Associado a um Serviço de Negócios

É possível exportar o WSDL de um serviço de negócios baseado no WSDL, para que você possa exibir ou modificar o WSDL em uma ferramenta externa como um IDE. O WSDL será exportado como um arquivo JAR.

Observe que isso é diferente da funcionalidade Exportar Recursos no módulo Administração do Sistema, que você usa para mover e separar recursos entre dois domínios. Consulte Exportando Recursos.

Antes de começar

É possível exportar apenas um WSDL quando estiver fora de uma sessão. Consulte Usando o Centro de Alterações.

Para exportar um WSDL

  1. Execute um dos seguintes procedimentos:
    • Selecione Resource Browser > Serviços de Negócios para exibir a página Resumo de Serviços de Negócios.
    • Selecione o Project Explorer, depois selecione o projeto ou pasta que contém o serviço de negócios que você deseja exportar como um WSDL. A página View de Projeto/Pasta é mostrada.
  2. Clique no ícone Exportar WSDL na linha do serviço de negócios cujo WSDL você deseja exportar. Uma caixa de diálogo solicita que você abra ou salve o arquivo JAR exportado.
  3. Na caixa de diálogo, clique em Abrir para abrir o arquivo ou clique em Salvar para salvá-lo.
