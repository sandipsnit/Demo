Página Exibir um Serviço de Negócios - Detalhes da Configuração

A página Exibir Serviço de Negócios - Detalhes da Configuração exibe os detalhes da configuração de um serviço de negócios. A Tabela 18-22 descreve todas as propriedades que podem aparecer nesta página. (As propriedades variam de acordo com os detalhes do serviço de negócios.)

As categorias listadas nesta página correspondem às páginas Criar/Editar um Serviço de Negócios usadas para criação e edição de configurações do serviço de negócios, conforme descrito em Criar/Editar um Serviço de Negócios - Referência de Página.

Clique no link Editar ao lado do nome da categoria para exibir a página de configuração associada.

Tabela 18-22 - Página Exibir um Serviço de Negócios: Detalhes da Configuração
Propriedades
Descrição
Última Modificação
O usuário que criou este serviço de negócios ou o importou para a configuração.
Última Modificação em
A data e hora em que o usuário criou este serviço de negócios ou o importou para a configuração. Clique no link da data e hora para exibir o histórico de alterações deste recurso. Consulte a página Exibir Histórico de Alteração.
Referências
O número de objetos referenciados por este serviço de negócios. Se tais referências existirem, clique no link numérico para exibir uma lista dos objetos. Consulte Exibindo Referências a Recursos.
Referenciado por
O número de objetos que referenciam esse serviço de negócios. Se tais referências existirem, clique no link numérico para exibir uma lista dos objetos. Consulte Exibindo Referências a Recursos.
Descrição
