Página Segurança do Serviço de Negócios

A página Exibir um Serviço de Negócios - Configuração de Segurança pertence a uma série de páginas para edição de serviços de negócios, conforme descrito em Criando e Configurando Serviços de Negócios e na página Exibir um Serviço de Negócios - Detalhes da Configuração.

Os campos disponíveis nesta página dependem do uso de políticas no serviço de negócios. Por exemplo, se o serviço usar políticas do OWSM (recomendado), Substituições de Política aparecerão na página Segurança. Para obter mais informações, consulte "Securing Oracle Service Bus with Oracle Web Services Manager" no Oracle Fusion Middleware Developer's Guide for Oracle Service Bus em http://www.oracle.com/pls/as111170/lookup?id=OSBDV1681.

Políticas do WLS 9.2 vinculadas ao serviço são exportas (alinhadas) no WSDL efetivo. Políticas abstratas são pré-processadas antes de serem alinhadas. As políticas do OWSM são vinculadas por referência, e não alinhadas no WSDL efetivo.

Use esta página para configurar as definições de segurança para um serviço de negócios que possua uma política baseada no WSDL ou que use bindings de política personalizada. A Tabela 18-24 descreve como usar a página.

Tabela 18-24 - Página Exibir um Serviço de Negócios - Segurança

Para editar...
Conta do Serviço
Clique em Procurar e selecione uma conta de serviço na lista exibida. Se nenhuma conta de serviço for especificada, será usado um assunto anônimo.
Observação: Esta opção não está disponível para os serviços que usam Políticas do Oracle Web Services Manager, porque as credenciais do usuário são fornecidas por chaves sobrepostas.

Para obter mais informações:

  • Especificando Contas de Serviço
  • Adicionando Contas de Serviço
Substituições de Política
Para políticas do OWSM, forneça as substituições desejadas e permitidas.
Para obter mais informações, consulte a Tabela 18-23 e "Securing Oracle Service Bus with Oracle Web Services Manager" no Oracle Fusion Middleware Developer's Guide for Oracle Service Bus em http://www.oracle.com/pls/as111170/lookup?id=OSBDV1681.

Depois de finalizar

Clique em Atualizar para salvar esta configuração, ou clique em Redefinir para desfazer suas alterações.
