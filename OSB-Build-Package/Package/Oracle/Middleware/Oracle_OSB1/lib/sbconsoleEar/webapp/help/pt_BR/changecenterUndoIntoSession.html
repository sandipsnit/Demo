Desfazendo em uma Sessão

Se erros semânticos resultarem da reversão de uma ativação de sessão, será impedido que você faça desfaça. No entanto, há uma alternativa neste caso. É possível desfazer a ativação da sessão e fazer com que as alterações sejam colocadas em uma nova sessão. Então, é possível corrigir os erros de semântica e ativar a sessão. Também é possível usar este recurso de reversão em uma sessão para explorar as ramificações de uma reversão de ativação de sessão. É possível examinar todas as alterações resultantes e decidir se realmente deseja executar a reversão. Se decidir que realmente deseja fazer isso, você pode ativar a sessão.
