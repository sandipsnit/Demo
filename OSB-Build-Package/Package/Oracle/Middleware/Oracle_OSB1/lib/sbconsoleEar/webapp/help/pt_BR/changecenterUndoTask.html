Desfazendo Tarefas

Use a página Exibir Alterações de Configuração para desfazer as tarefas executadas na configuração do Oracle Service Bus durante sua sessão atual e para desfazer ativações de sessão fora de uma sessão. O Oracle Service Bus permite desfazer vários níveis de ativação de sessão, com restrição apenas aos recursos do sistema. Para obter mais informações, consulte Ordem de Tarefas Desfazer.

Observação: Se você fizer upgrade do Oracle Service Bus versão 2.1 para a versão 2.5, não poderá desfazer as sessões ativadas antes do upgrade. As sessões ativadas após o upgrade estarão disponíveis no histórico de ativação da sessão, para reversão.
Desfazendo uma Tarefa em uma Sessão
  1. Certifique-se de que você esteja em uma sessão.
  2. Clique em Exibir Alterações no Centro de Alterações.
  3. Na coluna Opções de uma tarefa específica, clique no ícone Desfazer.

    4. A tarefa foi desfeita.

    É possível desfazer qualquer alteração na sessão atual. No entanto, depois disso, se você mudar de ideia, po' a reversão que desfez uma reversão anterior, e assim por diante.

Desfazendo uma Sessão que Foi Ativada Anteriormente
  1. Certifique-se de que você esteja fora de uma sessão.
  2. Clique em Exibir Alterações no Centro de Alterações.
  3. Execute um dos seguintes procedimentos:
    • Na coluna Opções de uma tarefa específica, clique no ícone Desfazer. A ativação da sessão será desfeita.
    • Na coluna Opções de uma tarefa específica, selecione o ícone Desfazer na Sessão. Uma nova sessão será criada pré-preenchida com as tarefas necessárias para desfazer as alterações ativadas na sessão. Para obter mais informações, consulte Desfazendo em uma Sessão.
