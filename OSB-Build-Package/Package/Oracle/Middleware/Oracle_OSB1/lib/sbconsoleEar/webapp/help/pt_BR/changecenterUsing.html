Usando o Centro de Alterações

O módulo Centro de Alterações é o ponto inicial do uso da Console do Oracle Service Bus Administration para alterar a configuração. Para alterar a configuração usando a Console do Oracle Service Bus Administration, você deve usar o Centro de Alterações para iniciar uma sessão.

  1. Clique em Editar ou em Criar para iniciar uma sessão. O nome da sessão será exibido no Centro de Alterações.
  2. Faça as alterações apropriadas na página relevante da Console do Oracle Service Bus Administration.
  3. Clique em Salvar em cada página em que você fizer uma alteração. Todas as alterações feitas na sessão atual serão salvas.

    4. Clique em Descartar no Centro de Alterações, a qualquer momento durante a sessão, para descartar as alterações feitas até o momento na sessão atual.

  4. Quando tiver terminado de fazer alterações, clique em Ativar no Centro de Alterações. Continue em Ativando Sessões.
