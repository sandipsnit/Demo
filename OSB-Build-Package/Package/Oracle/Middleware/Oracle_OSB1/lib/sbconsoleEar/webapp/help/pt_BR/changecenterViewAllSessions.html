Exibindo Todas as Sessões

Use esta página para exibir todas as sessões existentes na Console do Oracle Service Bus Administration. Você pode exibir estas sessões se estiver atualmente dentro ou fora de uma sessão.

Você pode exibir todas as sessões somente se estiver usando a atribuição de Administrador. Para obter mais informações, consulte "Configuring Administrative Security" no Oracle Fusion Middleware Developer's Guide for Oracle Service Bus em http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539.

  1. Clique em Exibir Todas as Sessões no Centro de Alterações. Para cada sessão existente, a página Exibir Todas as Sessões exibe as informações mostradas na Tabela 2-5.

    2. Tabela 2-5 - Detalhes da Sessão
    Propriedade
    Descrição
    Nome da Sessão
    O nome da sessão. O nome é um link para a sessão.
    Usuário
    O nome do usuário que criou a sessão.
    Horário de Criação
    A data e hora em que a sessão foi criada.
    Última Modificação
    A data e hora em que uma alteração foi feita pela última vez durante a sessão.
    No. de Objetos Modificados
    O número de objetos que foram alterados durante a sessão.

  2. Para alternar para outra sessão, clique no link na coluna Nome da Sessão para essa sessão.
Observação: O mesmo usuário com log-in efetuado em vários browsers não é suportado. Isso causa um comportamento imprevisível na Console do Oracle Service Bus Administration.
