Configuração

Use o módulo Configuração para localizar e gerenciar serviços de proxy, serviços de negócios, destinos de alerta, regras de alerta de SLA e o rastreamento do run-time para serviços de proxy.

É possível especificar definições operacionais para todos os serviços, no nível de serviço e global, e usar as definições globais para ativar e desativar o monitoramento, alertas de SLA e para serviços de proxy apenas, relatórios de mensagens do pipeline e log de mensagens do pipeline.
