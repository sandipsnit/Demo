Desativando o Monitoramento Global de Serviços

Para desativar o monitoramento para todos os serviços de domínio:

Observação: Se você desativar o monitoramento de todos os serviços, todas as estatísticas coletadas até o momento para esses serviços também serão deletadas. Essas estatísticas não podem ser restauradas usando a função Desfazer da sessão. É possível usar a opção Desfazer para ativar o monitoramento novamente para os serviços, mas a deleção das estatísticas é irreversível.
  1. Se ainda não tiver feito isso, clique em Criar para criar uma nova sessão ou clique em Editar para informar uma sessão existente. Consulte Usando o Centro de Alterações.
  2. Selecione Operações > Definições Globais.
  3. Desmarque a caixa de seleção Ativar Monitoramento do sistema para parar a coleta de estatísticas de monitoramento de todos os serviços na configuração.
    4. Observação: Esta opção substitui a opção Ativar Monitoramento do Serviço que você pode selecionar para serviços de negócios e de proxy específicos. Consulte Configurando Definições Operacionais para Serviços de Negócios e Configurando Definições Operacionais para Serviços de Proxy.
    Observação: Esta opção também substitui a opção Ativar Alertas de SLA. Se você desativar o monitoramento global, os alertas de SLA também serão desativados no run-time, mesmo que a caixa de seleção Ativar Alertas de SLA correspondente pode estar marcada. Consulte Ativando Alertas de SLA Globalmente.
  4. Clique em Atualizar. A configuração de monitoramento será salva na sessão atual.
  5. Como alternativa, clique em Redefinir a qualquer momento durante a sessão para descartar as alterações e atualizar a página com as definições armazenadas no momento.
  6. Para encerrar a sessão e implantar a configuração no run-time, clique em Ativar em Centro de Alterações.
