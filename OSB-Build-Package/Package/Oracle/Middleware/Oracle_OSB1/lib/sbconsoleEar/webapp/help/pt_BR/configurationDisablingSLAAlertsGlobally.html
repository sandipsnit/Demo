Desativando Alertas de SLA Globalmente

Para desativar alertas de SLA para todos os serviços de domínio:

  1. Se ainda não tiver feito isso, clique em Criar para criar uma nova sessão ou clique em Editar para informar uma sessão existente. Consulte Usando o Centro de Alterações.
  2. Selecione Operações > Definições Globais.
  3. Desmarque a caixa de seleção Ativar Alertas de SLA para que o sistema pare de avaliar regras de alerta para todos os serviços na configuração.
    4. Observação: Esta opção substitui qualquer configuração de Alerta de SLA que você possa selecionar para serviços de negócios e de proxy específicos. Consulte Configurando Definições Operacionais para Serviços de Negócios e Configurando Definições Operacionais para Serviços de Proxy.
  4. Clique em Atualizar. A configuração de alertas de SLA será salva na sessão atual.
  5. Como alternativa, clique em Redefinir a qualquer momento durante a sessão para descartar as alterações e atualizar a página com as definições armazenadas no momento.
  6. Para encerrar a sessão e implantar a configuração no run-time, clique em Ativar em Centro de Alterações.
