Ativando Armazenamento no Cache do Resultado Globalmente

Se você chamar serviços de negócios cujos resultados raramente mudam, o armazenamento no cache do resultado melhorará o desempenho do serviço de negócios retornando os resultados armazenados no cache para o cliente, em vez de aguardar os resultados de uma chamada de serviço.

Para configurar o armazenamento no cache do resultado de serviços de negócios individuais, é necessário primeiro ativar o armazenamento no cache dos resultados globalmente executando as etapas a seguir. A definição default para armazenamento no cache do resultado é ativada.

  1. Se ainda não tiver feito isso, clique em Criar para criar uma nova sessão ou clique em Editar para informar uma sessão existente. Consulte Usando o Centro de Alterações.
  2. Selecione Operações > Definições Globais.
  3. Marque a caixa de seleção Ativar Caches de Resultados.
  4. Clique em Atualizar para salvar todas as definições nesta página.
  5. Como alternativa, clique em Redefinir a qualquer momento durante a sessão para descartar as alterações e atualizar a página com as definições armazenadas no momento.
  6. Para encerrar a sessão e implantar a configuração no run-time, clique em Ativar em Centro de Alterações.

Para obter informações sobre a configuração do armazenamento no cache dos resultados de serviços de negócios, consulte a Página Configuração do Tratamento de Mensagens.
