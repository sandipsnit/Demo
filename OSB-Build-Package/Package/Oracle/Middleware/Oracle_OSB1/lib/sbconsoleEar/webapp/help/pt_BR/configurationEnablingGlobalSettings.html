  • Ativando Alertas de SLA Globalmente

  • Ativando Alertas do Pipeline Globalmente
  • Desativando Alertas do Pipeline Globalmente
  • Ativando Relatórios de Mensagens Globalmente
  • Desativando Relatórios de Mensagens Globalmente
  • Ativando Logs Globalmente
  • Desativando Logs Globalmente
  • Ativando Armazenamento no Cache do Resultado Globalmente
  • Desativando Armazenamento no Cache do Resultado Globalmente
