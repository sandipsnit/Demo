Ativando Logs Globalmente

Se você selecionar Ativa Log, as mensagens de ação de Log do pipeline serão enviadas para o serviço de log do WebLogic Server. Para conseguir exibi-las, você deve configurar o WebLogic Server para encaminhar essas mensagens para o log de domínio.

Para ativar a saída de log para todos os serviços de proxy de domínio:

  1. Se ainda não tiver feito isso, clique em Criar para criar uma nova sessão ou clique em Editar para informar uma sessão existente. Consulte Usando o Centro de Alterações.
  2. Selecione Operações > Definições Globais.
  3. Marque a caixa de seleção Ativar Log para iniciar qualquer ação de Log do pipeline para todos os serviços de proxy.
  4. Clique em Atualizar para salvar todas as definições nesta página.
  5. Como alternativa, clique em Redefinir a qualquer momento durante a sessão para descartar as alterações e atualizar a página com as definições armazenadas no momento.
  6. Para encerrar a sessão e implantar a configuração no run-time, clique em Ativar em Centro de Alterações.
