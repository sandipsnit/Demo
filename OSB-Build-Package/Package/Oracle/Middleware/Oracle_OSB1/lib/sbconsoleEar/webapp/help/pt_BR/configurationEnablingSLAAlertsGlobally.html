Ativando Alertas de SLA Globalmente

Embora seja possível configurar Alertas de SLA independentemente do Monitoramento, há uma interação entre as duas definições no run-time. Se o monitoramento global estiver ativado, alertas de SLA poderão ser ativados ou desativados. No entanto, se o monitoramento global estiver desativado, então os alertas de SLA serão desativados com eficácia, porque as condições da regra de alerta de SLA dependem das estatísticas de monitoramento sendo avaliadas.

Para ativar alertas de SLA para todos os serviços de domínio:

  1. Se ainda não tiver feito isso, clique em Criar para criar uma nova sessão ou clique em Editar para informar uma sessão existente. Consulte Usando o Centro de Alterações.
  2. Selecione Operações > Definições Globais.
  3. Marque a caixa de seleção Ativar Alertas de SLA para que o sistema comece a avaliar regras de alerta para todos os serviços na configuração.
  4. Clique em Atualizar para salvar todas as definições nesta página.
  5. Como alternativa, clique em Redefinir a qualquer momento durante a sessão para descartar as alterações e atualizar a página com as definições armazenadas no momento.
  6. Para encerrar a sessão e implantar a configuração no run-time, clique em Ativar em Centro de Alterações.
