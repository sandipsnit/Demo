Encontrando Destinos de Alerta

  1. Selecione Operações > Pesquisa Inteligente.
  2. Na lista drop-down Tipo, selecione Destinos de Alerta, depois clique em Pesquisar. A página Resumo de Destinos de Alerta exibirá as informações mostradas na Tabela 26-5.
  3. Para restringir o número de itens na lista ou para localizar destinos de alerta específicos:
    • Clique no ícone Abrir para exibir filtros de pesquisa adicionais. Se você especificar vários critérios de pesquisa, apenas os resultados que correspondem a todos os critérios serão retornados.
    • Filtrar por nome e caminho do destino de alerta. Nos campos Nome e Caminho, informe o nome e o caminho do destino de pesquisa, depois clique em Pesquisar. O caminho é o nome do projeto e o nome da pasta em que reside o destino de alerta. É possível usar * e ? como caracteres curingas para executar uma pesquisa mais generalizada.
    • Filtrar por alvo do destino de alerta. Na caixa de seleção múltipla Alvo, selecione um ou mais alvos do destino de alerta: Trap do SNMP, Relatórios, Log de Alerta, e-mail ou JMS, depois clique em Pesquisar. Apenas os destinos do alerta com pelo menos um dos alvos selecionados serão exibidos. Selecione nenhum alvo do destino de alerta para ignorar os critérios do filtro do destino de alerta.
    • Filtrar por padrão de pesquisa. Na caixa de texto Padrão de Pesquisa (Qualquer String), especifique qualquer string, depois clique em Pesquisar. O sistema usará a string para pesquisar todos os campos Descrição dos destinos de alerta, além dos campos detalhados específicos dos destinos de e-mail e JMS. Se a string aparecer em qualquer um dos campos do destino de alerta, os destinos de alerta correspondentes aos critérios de pesquisa serão exibidos.

      • Tabela 26-5 - Informações Resumidas de Destinos de Alerta
      Propriedade
      Descrição
      Nome
      O nome do recurso de destino de alerta. Este campo é um link para a página Exibir Destino de Alerta - Configuração. Consulte Localizando Destinos de Alerta.
      Caminho
      O projeto associado ao destino de alerta. Se o destino de alerta residir em uma pasta do projeto, essa pasta também será listada. O caminho é exibido no formato:
      project-name/root-folder/ . . ./parent-folder
      O caminho é um link para o caminho correspondente no Project Explorer.
      Opções
      Clique no ícone Deletar para deletar um destino de alerta específico. Um ícone Advertência de Deleção é exibido quando outros recursos referenciam esse recurso. É possível deletar o recurso com uma confirmação de advertência. Isso pode resultar em conflitos devido a referências não resolvidas referentes ao recurso deletado. Consulte Deletando Destinos de Alerta.

Também é possível utilizar esta página para fazer o seguinte:

  • Para remover o filtro de pesquisa e exibir todos os destinos de alerta, clique em Exibir Tudo.
  • Para deletar um destino de alerta específico se não referenciado por outros recursos, marque a caixa de seleção ao lado dele e clique em Deletar.
  • Para personalizar sua exibição das informações de definições operacionais, clique no ícone Personalizador da Tabela. Consulte Personalizando Views da Tabela.
