Definindo Preferências do Usuário

Use esta página para personalizar e persistir as opções de exibição e as definições da Console do Oracle Service Bus Administration. É possível atualizar (alterar) as Preferências do Usuário sem ativar uma sessão de alteração.

A Tabela 26-7 descreve as opções e controles na página.

Tabela 26-7 - Preferências do Usuário - Opções e Controles
Para...
Faça Isto...
Alterar sua Home Page, aberta quando você inicia a Console do Oracle Service Bus Administration
Na primeira lista drop-down, selecione qualquer opção do menu principal: Operações, Browser do Recurso, Project Explorer, Segurança Configuração ou Administração do Sistema.
Na segunda lista drop-down, selecione qualquer uma das opções do submenu. (O conteúdo dos submenus muda dinamicamente de acordo com a primeira seleção do menu.)
Exibir Filtros de Pesquisa
Selecione Sim ou Não.
Por default, os filtros de pesquisa não são exibidos (fechados). Selecionando Sim, os filtros de pesquisa são sempre exibidos (abertos).
Exibir Anotações do Estágio
Selecione Sim ou Não.
Por default, as anotações de estágio do pipeline não são exibidas (fechadas). Selecionando Sim, as anotações de estágio são sempre exibidas (abertas).
Exibir Metadados do Recurso
Selecione Sim ou Não.
Por default, as seções de metadados do recurso são exibidas (abertas). Selecionando Não, as seções de metadados do recurso are não são exibidas (fechadas).
Alterar a frequência com a qual a Console do Oracle Service Bus Administration atualiza a exibição dos dados no Painel de Controle
Na lista drop-down Taxa de Atualização do Painel de Controle, selecione Nenhuma Atualização, 1, 2, 3, 5, 10, 20, 30 ou 60 minutos como a taxa de atualização para o Painel de Controle.
A taxa default é Sem Atualização, mas é possível selecionar outro tempo predefinido. Por exemplo, se você selecionar 5 minutos, o Painel de Controle será atualizado com dados a cada 5 minutos.
Cuidado: Selecionando-se um intervalo de taxa de atualização do painel de controle diferente do default (Sem Atualização), o browser atualiza o Painel de Controle em intervalos regulares. Isso impede que a sessão atinja o timeout, mesmo sem que você interaja com a Console do Oracle Service Bus Administration.
Cuidado: A Oracle recomenda que você use o recurso de taxa de atualização do Painel de Controle com cuidado. Nunca deixe uma sessão da Console do Oracle Service Bus Administration sem assistência. Enquanto a sessão está ativa, outras pessoas podem obter acesso a ela em sua máquina sem nova autenticação.
Altere o intervalo de tempo para exibição de dados históricos para alertas no Painel de Controle
Na lista drop-down Duração do Histórico de Alertas, selecione 30 minutos, 1, 2, 3 ou 6 horas como o intervalo de tempo para exibição dos dados de alerta histórico. O default são 30 minutos (exibe os alertas recebidos dentro dos últimos 30 minutos), mas é possível selecionar outra duração predefinida. Por exemplo, se você selecionar 1 hora, o Painel de Controle exibirá os alertas recebidos na última hora.
