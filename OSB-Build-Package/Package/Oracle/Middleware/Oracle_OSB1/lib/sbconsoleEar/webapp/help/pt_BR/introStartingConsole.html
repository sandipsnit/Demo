Iniciando a Console do Oracle Service Bus Administration

  1. Inicie o Oracle Service Bus usando um dos métodos a seguir:
    • No menu Iniciar do Windows, selecione:

      • Inicie > Todos os Programas > Oracle WebLogic > Projetos do Usuário > domain_name > Iniciar Servidor do Domínio do Oracle Service Bus.

      Em que domain_name representa o nome designado ao domínio do Oracle Service Bus quando foi criado. Projetos do Usuário é o local default em que o domínio é criado por default.

    • Chame o comando startWebLogic da raiz do domínio do Oracle Service Bus: startWebLogic.cmd ou startWebLogic.sh

      • Uma janela de comandos é exibida mostrando informações de status sobre o WebLogic Server.

      Se você iniciou o servidor no modo de produção, deverá informar um nome de usuário e senha válidos para iniciar o servidor.

  2. Quando o servidor for iniciado, informe o seguinte URL no browser:

    3. http://hostname:port/sbconsole

    em que host representa o nome da máquina em que o WebLogic Server está executando e port representa o número da porta.

    Por exemplo, se o WebLogic Server estiver executando na máquina local usando a configuração de porta default, informe o seguinte URL no browser:

    http://localhost:7001/sbconsole

    A página de log-in da Console do Oracle Service Bus Administration é exibida.

  3. Informe o nome de usuário e a senha especificados durante o processo de instalação.
    4. Observação: O nome de usuário e a senha para o domínio de Exemplos do Oracle Service Bus é weblogic / welcome1.
  4. Clique em Log-in.

Para efetuar log-out, clique em Efetuar Log-out no banner próximo à parte superior do painel direito da Console do Oracle Service Bus Administration.
