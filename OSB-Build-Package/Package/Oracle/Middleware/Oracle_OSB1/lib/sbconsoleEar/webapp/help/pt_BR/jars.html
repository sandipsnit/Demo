JARs

Um JAR (Java ARchive) é um arquivo compactado que contém um conjunto de classes Java. É usado para armazenar classes Java compiladas e metadados associados que possam constituir um programa. Um JAR atua como uma biblioteca de programas chamáveis para elementos de código Java (para que um único link de compilação ofereça acesso a vários elementos, em vez de exigir bindings para cada elemento individualmente). Para saber mais sobre JARs, consulte http://java.sun.com/docs/books/tutorial/deployment/jar/

Os recursos JAR no Oracle Service Bus são usados em:

  • Ações de chamada Java
  • Serviços de negócios baseado no EJB
  • Serviços de proxy e de negócios baseados no Tuxedo
