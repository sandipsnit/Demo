Bindings de JCA

O Oracle SOA Suite permite a integração com o EIS (Enterprise Information Systems) usando adaptadores de JCA (J2EE Connector Architecture). Usando o Oracle JDeveloper e outros ambientes de desenvolvimento de JCA suportados, configure os adaptadores e crie os recursos do adaptador de JCA (arquivos .jca e WSDLs abstratos) usados para integração de back-end.

Você pode se comunicar com os componentes EIS compatíveis com JCA por meio do Oracle Service Bus (OSB), o que permite consumir os recursos de JCA na forma de recursos de Binding de JCA. Com Bindings de JCA, é possível gerar serviços de proxy e de negócios que usam o transporte JCA do Oracle Service Bus para comunicação com as aplicações EIS por meio dos adaptadores de JCA.

Este capítulo, que descreve como adicionar, editar, localizar e deletar Bindings de JCA no Oracle Service Bus, contém os seguintes tópicos:

  • Localizando Bindings de JCA
  • Adicionando Bindings de JCA
  • Editando Bindings de JCA
  • Deletando Bindings de JCA

Para obter mais informações sobre o transporte JCA, consulte "JCA Transport" no Oracle Fusion Middleware Developer's Guide for Oracle Service Bus em http://www.oracle.com/pls/as111170/lookup?id=OSBDV910.

Para obter mais informações sobre os adaptadores JCA do Oracle SOA Suite, consulte o Oracle Fusion Middleware User's Guide for Technology Adapters em http://www.oracle.com/pls/as111170/lookup?id=TKADP.
