Deletando Bindings de JCA

Se qualquer serviço de negócios ou de proxy for baseado no Binding de JCA, remova o Binding de JCA do serviço de negócios ou de proxy antes de deletá-lo.

  1. Se ainda não tiver feito isso, clique em Criar para criar uma nova sessão ou clique em Editar para informar uma sessão existente. Consulte Usando o Centro de Alterações.
  2. Selecione Resource Browser > Bindings de JCA.
  3. Clique no ícone Deletar no campo Opções do Binding de JCA que você deseja deletar.

    4. O Binding de JCA será deletado na sessão atual. Se um serviço de negócios ou um serviço de proxy tiver sido configurado para usar um Binding de JCA, um ícone Advertência de Deleção indicará que é possível deletar o Binding de JCA com uma confirmação de advertência. Isso pode resultar em conflitos devido a referências não resolvidas do serviço ao Binding de JCA deletado.

  4. Para encerrar a sessão e implantar a configuração no run-time, clique em Ativar em Centro de Alterações.
