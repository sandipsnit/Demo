Localizando Bindings de JCA

A página Resumo de Bindings de JCA lista os Bindings de JCA que foram adicionados ao domínio do Oracle Service Bus.

  1. Execute um dos seguintes procedimentos:
    • Selecione o Project Explorer para exibir a página View de Projetos ou a página View de Projeto/Pasta. Em seguida, navegue pelos projetos e pastas para localizar o Binding de JCA.
    • Selecione Resource Browser > Bindings de JCA. A página Resumo de Bindings de JCA exibe as informações mostradas na Tabela 12-1. Para obter uma descrição mais detalhada das propriedades, consulte Localizando Bindings de JCA.
  2. Para localizar um Binding de JCA específico:
    • Para restringir o número de itens na lista, é possível filtrar pelo nome e caminho antes de clicar em Pesquisar.

      • Caminho é o nome do produto e o nome da(s) pasta(s) em que reside o recurso.

    • Clique em Exibir Tudo para remover os filtros de pesquisa e exibir todos os Bindings de JCA.

      • Tabela 12-1 - Informações sobre Binding de JCA
      Propriedade
      Descrição
      Nome do Binding de JCA
      O nome exclusivo designado ao Binding de JCA. Clique no nome para ver a página Exibir Detalhes de Binding de JCA. Consulte Editando Bindings de JCA.
      Caminho
      O nome do projeto e o nome da pasta em que reside o Binding de JCA. Clique no nome para ver o projeto ou pasta que contém este recurso. Consulte Qualificando Nomes de Recursos Usando Projetos e Pastas.
      Ação
      Contém um ícone Gerar que permite gerar um serviço WSDL e JCA do Oracle Service Bus a partir do Binding de JCA.
      Opções
      Contém um ícone Deletar. Se um serviço de negócios ou um serviço de proxy tiver sido configurado para usar um Binding de JCA, um ícone Advertência de Deleção indicará que é possível deletar o Binding de JCA com uma confirmação de advertência. Isso pode resultar em conflitos devido a referências não resolvidas do serviço ao Binding de JCA deletado. Consulte Deletando Bindings de JCA.
