Editando Bindings de JCA

Use a página Exibir Detalhe do Binding de JCA para exibir e alterar os detalhes de um Binding de JCA.

  1. Se ainda não tiver feito isso, clique em Criar para criar uma nova sessão ou clique em Editar para informar uma sessão existente. Consulte Usando o Centro de Alterações.
  2. Localize o Binding de JCA, conforme descrito em Localizando Bindings de JCA.
  3. Clique no nome do Binding de JCA. A página Exibir um Binding de JCA exibe as informações mostradas na Tabela 12-2.

    4. Tabela 12-2 Detalhes do Binding de JCA 
    Propriedade
    Descrição
    Última Modificação
    O usuário que criou este Binding de JCA ou o importou para a configuração.
    Última Modificação em
    A data e hora em que o usuário criou este Binding de JCA ou o importou para a configuração. Clique no link da data e hora para exibir o histórico de alterações deste recurso. Consulte a página Exibir Histórico de Alteração.
    Referências
    Oferece um link para o recurso WSDL referenciado pelo Binding de JCA. Consulte Exibindo Referências a Recursos.
    Referenciado por
    O número de objetos que referenciam esse Binding de JCA. Se tais referências existirem, clique no link numérico para exibir uma lista dos objetos. Consulte Exibindo Referências a Recursos.
    Ícone Ação
    Com o Centro de Alterações no modo de edição, a Ação permite gerar um serviço WSDL e JCA do Oracle Service Bus a partir do Binding de JCA.
    Arquivo JCA
    Exibe uma view somente para leitura do Binding de JCA.
    WSDL Abstrato
    Exibe o WSDL associado do Binding de JCA. Se o Binding de JCA não referenciar um WSDL, clique em Editar Referências para referenciar o WSDL associado.
    XML TopLink
    Exibe o arquivo de mapeamento TopLink ou EclipseLink necessário do Binding de JCA. Se o recurso de Binding de JCA não referenciar um arquivo de mapeamento necessário, clique em Editar Referências selecionar o arquivo de mapeamento necessário. Este campo só aparece se o arquivo JCA declarar um recurso XML.
    XSLT
    Exibe o arquivo XSLT necessário do Binding de JCA. Se o recurso de Binding de JCA não referenciar um recurso XSLT necessário, clique em Editar Referências para selecionar o arquivo XSLT necessário. Este campo só aparece se o arquivo JCA declarar um recurso XSLT, por exemplo, para um adaptador de soquete.

  4. Para modificar a Descrição ou o Arquivo de JCA, clique em Editar.

    5. Não é possível alterar o Nome do Recurso.

  5. Clique em Salvar para confirmar as atualizações na sessão atual.
  6. Para alterar as referências aos recursos necessários do Binding de JCA ou corrigir referências não resolvidas, clique em Editar Referências e selecione os recursos apropriados.
  7. Para encerrar a sessão e implantar a configuração no run-time, clique em Ativar em Centro de Alterações.
