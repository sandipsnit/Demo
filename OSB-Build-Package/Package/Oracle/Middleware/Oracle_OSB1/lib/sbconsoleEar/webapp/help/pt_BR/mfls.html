MFLs

Um documento MFL (Message Format Language) é um documento XML especializado usado para descrever o layout de dados binários. É uma linguagem proprietária da Oracle usada para definir regras que transformem dados binários formatados em dados XML. Um documento MFL segue o mfl.dtd, que inclui elementos e atributos usados para descrever cada campo de dados, além de agrupamentos de campos (grupos), repetição e agregação.

Ao criar serviços de negócios ou serviços de proxy do tipo Serviço de Mensagens, você pode selecionar tipos MFL como o tipo de mensagem de solicitação ou o tipo de mensagem de resposta do serviço.

Use o Oracle Format Builder para criar MFLs. Quando você define a hierarquia de um registro binário, o layout dos campos e o agrupamento de campos e grupos, as informações são salvas como um documento MFL que pode ser usado para executar conversões de run-time. Um documento MFL também pode ser usado no Format Builder para gerar o DTD correspondente que descreva seu modelo de conteúdo. Para saber como usar o Format Builder, consulte "Format Builder" no Oracle Fusion Middleware Developer's Guide for Oracle Service Bus em http://www.oracle.com/pls/as111170/lookup?id=OSBDV763.
