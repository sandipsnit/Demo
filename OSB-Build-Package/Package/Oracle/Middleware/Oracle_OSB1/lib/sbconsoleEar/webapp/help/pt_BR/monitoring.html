Monitoramento

Quando você cria um serviço de negócios ou de proxy, o monitoramento é desativado por default para esse serviço. Para saber como ativar o monitoramento para serviços de proxy, consulte Configurando Definições Operacionais para Serviços de Proxy; para serviços de negócios, consulte Configurando Definições Operacionais para Serviços de Negócios.

Para ativar ou desativar o monitoramento de todos os serviços que foram ativados ou desativados individualmente para monitoramento, use a opção Ativar Monitoramento na página Operações > Definições Globais. Consulte Ativando Definições Globais.

Para saber mais sobre o monitoramento no run-time, consulte "Monitoring Oracle Service Bus at Run Time" no Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus em http://www.oracle.com/pls/as111170/lookup?id=OSBAG472.
