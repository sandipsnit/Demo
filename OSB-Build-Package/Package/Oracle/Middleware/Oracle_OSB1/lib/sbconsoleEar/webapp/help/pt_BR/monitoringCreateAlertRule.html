Criando e Editando Regras de Alerta

As tarefas de criação e edição de regras de alerta incluem:

  • Configurando Informações Gerais para Regras de Alerta
  • Definindo Condições da Regra de Alerta
  • Revendo a Configuração da Regra de Alerta
