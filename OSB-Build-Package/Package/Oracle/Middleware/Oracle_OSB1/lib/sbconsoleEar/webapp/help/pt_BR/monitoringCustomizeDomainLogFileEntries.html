Personalizando a Exibição de Entradas de Arquivo de Log do Domínio

  1. Clique em Personalizar esta tabela. Campos adicionais são exibidos.

    2. Você pode clicar no link Personalizar esta tabela a qualquer momento para fechar esta tabela e manter as definições originais.

  2. Nos campos Filtrar, faça o seguinte:
    1. No campo Tempo, selecione o intervalo de tempo para o qual você deseja exibir entradas de log.
    2. No campo Hora de Início, informe uma hora de início no formato MM/DD/AAAA HH:MM:SS. Por exemplo, informe 10/25/06 08:39:48. Use os campos Hora de Início e Hora de Término para especificar um intervalo de tempo durante o qual você deseja exibir entradas de log.
    3. No campo Hora de Término, informe uma hora de término no formato MM/DD/AAAA HH:MM:SS. Por exemplo, informe 10/25/06 13:20:51. Use os campos Hora de Início e Hora de Término para especificar um intervalo de tempo durante o qual você deseja exibir entradas de log.
    4. Na caixa de texto Critérios, especifique um critério de filtragem (uma string de texto). Os critérios de filtragem são especificados como uma string na Linguagem da Consulta WLDF. A linguagem de consulta suporta operadores Booleanos: E, OU e NÃO, além de operadores relacionais. Para obter mais informações sobre a sintaxe de consulta, veja "WLDF Query Language" no Oracle Fusion Middleware Configuring and Using the Diagnostics Framework for Oracle WebLogic Server em http://www.oracle.com/pls/as111170/lookup?id=WLDFC316.
  3. No campo Colunas, selecione as colunas que você deseja exibir:
    1. Selecione um nome de coluna no campo Disponíveis.
    2. Clique na seta para mover este nome de coluna para o campo Escolhido.
    3. Repetir até que você tenha listado todos os nomes de coluna que deseja exibir no campo Escolhido.

      4. No campo Escolhido, você pode usar as setas Para Cima e Para Baixo para reordenar os nomes de coluna, conforme necessário.

  4. No campo Número de linhas exibidas por página, selecione o número de entradas de log que você deseja exibir em uma única página.
  5. No campo Máximo de Resultados Retornados, selecione o número máximo de entradas de log que você deseja exibir ao total ou selecione Mostrar Tudo para exibir todas elas.
  6. Execute um dos seguintes procedimentos:
    • Para salvar as novas definições, clique em Aplicar.
    • Para descartar suas alterações e manter as definições originais, clique em Redefinir.
