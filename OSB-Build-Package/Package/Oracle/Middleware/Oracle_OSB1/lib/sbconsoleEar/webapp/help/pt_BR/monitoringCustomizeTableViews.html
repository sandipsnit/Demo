Personalizando Exibições de Tabela

Use o personalizador de tabela para exibir informações da tabela de acordo com suas definições especificadas.

Observe que a ordem de classificação default para qualquer tabela é determinada pela primeira coluna na tabela.

  1. Clique no ícone Personalizador de Tabela. Campos adicionais são exibidos.

    2. Você pode clicar em Cancelar a qualquer momento para fechar esta tabela e manter as definições originais.

  2. No campo Colunas, selecione as colunas que você deseja exibir:
    1. Selecione um nome de coluna no campo Colunas Disponíveis.
    2. Clique na seta para mover este nome de coluna para o campo Colunas Selecionadas.

      3. Um asterisco denota uma coluna obrigatória.

    3. Repetir até que você tenha listado todos os nomes de coluna que deseja exibir no campo Colunas Selecionadas.

      4. No campo Colunas Selecionadas, é possível usar as setas Para Cima e Para Baixo para reordenar os nomes das colunas conforme necessário.

  3. Para remover as colunas da exibição, mova os nomes de colunas do campo Colunas Selecionadas para o campo Colunas Disponíveis.
  4. Na lista drop-down Linhas, selecione o número de linhas que você deseja exibir em uma única página.
  5. Execute um dos seguintes procedimentos:
    • Para salvar essas definições e fechar o personalizador de tabela, clique em Aplicar.
    • Para descartar suas alterações e manter as definições originais, clique em Redefinir.
