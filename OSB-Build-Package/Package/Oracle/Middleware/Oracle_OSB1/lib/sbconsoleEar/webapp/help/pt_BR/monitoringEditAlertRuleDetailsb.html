Editando Regras de Alerta

  1. Se ainda não tiver feito isso, clique em Criar para criar uma nova sessão ou clique em Editar para informar uma sessão existente. Consulte Usando o Centro de Alterações.
  2. Localizar a regra de alerta. Consulte Localizando Regras de Alerta.
  3. Clique no nome da regra de alerta.
  4. Para fazer uma alteração nos campos das páginas de configuração, clique no ícone Editar da página apropriada ou no botão Editar na parte inferior da página. Consulte Criando e Editando Regras de Alerta para obter as descrições das páginas e campos.
  5. Execute uma das etapas a seguir:
    • Para ir para a próxima página, clique em Próximo.
    • Para retornar à página anterior, clique em Anterior.
    • Para ir para a página de resumo, clique em Último.
    • Para confirmar as atualizações na sessão atual, clique em Salvar.
    • Para descartar as alterações e retornar para a página Regras de Alerta de SLA, clique em Cancelar.
  6. Para encerrar a sessão e implantar a configuração no run-time, clique em Ativar em Centro de Alterações.
