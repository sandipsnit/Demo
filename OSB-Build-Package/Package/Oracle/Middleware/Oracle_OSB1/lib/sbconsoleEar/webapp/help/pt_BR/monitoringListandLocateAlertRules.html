Exibindo Regras de Alerta

Use esta página para exibir as informações de regras de alerta mostradas na Tabela 25-19.

Tabela 25-19 - Informações de Regras de Alerta
Propriedade
Descrição
Nome
O nome designado a esta regra de alerta. O nome é um link para a página Configuração Geral da Regra de Alerta. Consulte Exibindo Configurações da Regra de Alerta.
Estado de SLA
O status da regra de alerta: Ativada ou Desativada.
Descrição
Este campo está oculto por default.
Uma descrição da regra de alerta.
Gravidade
A severidade do alerta que será acionado por esta regra: Normal, Advertência, Menor, Maior, Crítico ou Fatal.
Agregação Intervalo
A duração do intervalo de agregação em termos de horas e minutos.
Data de Expiração
A data em que esta regra de alerta não está mais em vigor.
Parar Processamento
Exibe Sim ou Não.
Frequência
A frequência deste alerta:
  • Sempre
  • Notificar Uma Vez
Opções
Esta coluna inclui a seguinte funcionalidade:
  • Clique nas setas para reordenar as regras de alerta. É possível mover regras de alerta individuais para cima ou para baixo na lista. Você só pode fazer isso quando mais de uma regra de alerta está configurada para este serviço.
  • Clique no ícone Renomear para renomear uma regra de alerta.
  • Clique no ícone Deletar para deletar uma regra de alerta específica. Consulte Deletando Regras de Alerta.
