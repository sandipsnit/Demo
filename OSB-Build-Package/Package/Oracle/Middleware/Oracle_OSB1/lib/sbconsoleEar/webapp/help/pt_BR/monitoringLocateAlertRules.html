Localizando Regras de Alerta

  1. Selecione Browser do Recurso > Serviços de Negócios ou Serviços de Proxy.
  2. Clique no nome do serviço.
  3. Selecione a guia Regras de Alerta de SLA.
  4. Para localizar uma regra de alerta específica para este serviço, role pelas páginas. Use os controles de página acima ou abaixo da tabela. Para ir para uma página, selecione o número da página ou use os botões de seta para ir para a próxima página, página anterior, primeira ou última página.

Também é possível utilizar esta página para fazer o seguinte:

  • Para configurar o monitoramento para este serviço, selecione a guia Definições Operacionais. Consulte Configurando Definições Operacionais para Serviços de Proxy e Configurando Definições Operacionais para Serviços de Negócios.
  • Para adicionar uma nova regra de alerta, clique em Adicionar Novo. Consulte Criando e Editando Regras de Alerta.
