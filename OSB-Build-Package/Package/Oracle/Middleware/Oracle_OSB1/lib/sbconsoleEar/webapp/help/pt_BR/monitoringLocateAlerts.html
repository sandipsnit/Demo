Localizando Alertas

  1. Selecione Operações > Painel de Controle.
  2. Selecione a guia Alertas de SLA ou Alertas do Pipeline para exibir alertas de SLA ou do pipeline.
  3. Clique no link Histórico de Alertas Estendidos. As páginasHistórico de Alertas de SLA Estendidos e Histórico de Alertas do Pipeline Estendidos exibem informações detalhadas do histórico de alertas.
  4. Para restringir o número de itens na lista ou para localizar alertas específicos:
    • Clique no ícone Abrir para exibir filtros de pesquisa adicionais. Se você especificar vários critérios de pesquisa, apenas os resultados que correspondem a todos os critérios serão retornados.
    • Filtrar por intervalo de datas. Execute um dos seguintes procedimentos:
      • Selecione Tudo para ignorar o intervalo de datas e os critérios do filtro do intervalo de tempo e exibir todos os alertas.
      • Nos campos De e Até, informe uma data e hora no formato MM/DD/AA HH:MM AM|PM, depois clique em Pesquisar. Os alertas que ocorrem no intervalo especificado de data e hora são exibidos.
      • No campo Para o Último, informe o número de dias, depois especifique as horas e os minutos, depois clique em Pesquisar. Os alertas ocorridos em um intervalo de tempo especificado serão exibidos.
    • Filtrar por severidade do alerta. Na lista drop-down Severidade do Alerta, selecione para restringir os alertas ao nível de severidade especificado, depois clique em Pesquisar.

      • Marque a caixa de seleção ou acima para restringir sua pesquisa ao nível de severidade especificado ou acima (listado do nível mais inclusivo para o nível mais restritivo): Normal, Advertência, Menor, Maior, Crítico e Fatal.

    • Filtrar por Serviço pai. Clique em Procurar para exibir a página Selecionar Serviço. Selecione o serviço pai na lista e clique em Submeter. Clique em Limpar para remover os critérios do filtro de serviço pai ou clique em Pesquisar.
    • Apenas para alertas de SLA, filtre pelo Nome do Alerta (Qualquer String). Informe qualquer string, incluindo caracteres curingas, depois clique em Pesquisar. Para obter mais detalhes, consulte Linguagem de Consulta do WebLogic Diagnostics Framework.
    • Apenas para alertas de pipeline, filtre pelo Resumo do Alerta (Qualquer String). Informe qualquer string, incluindo caracteres curingas, depois clique em Pesquisar. Para obter mais detalhes, consulte Linguagem de Consulta do WebLogic Diagnostics Framework. Para localizar ações do alerta de pipeline que foram desenvolvidas sem texto do resumo de alerta, informe Alerta do Oracle Service Bus.
    • Clique em Exibir Tudo para remover os filtros de pesquisa e exibir todos os alertas.

Use as páginas Histórico de Alertas de SLA Estendidos e Histórico de Alertas do Pipeline Estendidos e também faça o seguinte:

  • Para exibir os dados de alerta em um gráfico de pizza, selecione View de Gráfico de Pizza.
  • Para exibir os mesmos dados em um gráfico de barras, selecione View de Gráfico de Barras.
  • Para exibir os dados em formato tabular, selecione View de Tabela.
  • Para exibir uma versão atualizada dos dados, clique em Atualizar.
  • Para deletar alertas individuais, marque as caixas de seleção correspondentes e clique em Deletar na parte inferior da página.
  • Para deletar todos os alertas ou aqueles que foram emitidos em um intervalo de data e hora especificado, clique no link Expurgar Histórico de Alertas na parte superior da página. Consulte Expurgando Alertas.

Linguagem de Consulta do WebLogic Diagnostics Framework

Alertas são armazenados usando o WebLogic Diagnostics Framework, que fornece sua própria linguagem de consulta, incluindo curingas. Para filtrar alertas no histórico de alertas estendidos, use a sintaxe descrita em "WLDF Query Language" no Oracle Fusion Middleware Configuring and Using the Diagnostics Framework for Oracle WebLogic Server em http://www.oracle.com/pls/as111170/lookup?id=WLDFC316.
