Redefinindo Estatísticas para Serviços

É possível usar a página Condição do Serviço para redefinir as estatísticas de monitoramento de todos os serviços na configuração. Você pode redefinir as estatísticas independente de estar ou não em uma sessão.

  1. Selecione Operações > Painel de Controle > Condição do Serviço.
  2. No campo Exibir Estatísticas, selecione Desde a Última Redefinição. A view Desde a Última Redefinição exibe uma view de contagem atual das métricas.
  3. Para redefinir as estatísticas de monitoramento para um serviço específico, clique no ícone Redefinir na coluna Ação desse serviço.

    4. Quando você confirma que deseja fazer isso, o sistema deleta todas as estatísticas de monitoramento que foram coletadas para o serviço desde que você redefiniu as estatísticas pela última vez. No entanto, o sistema não deleta as estatísticas sendo coletadas durante o Intervalo de Agregação Atual para o serviço. Além disso, depois que você clica no ícone Redefinir, o sistema começa imediatamente a coletar estatísticas de monitoramento para o serviço de novo.

  4. Para redefinir as estatísticas de monitoramento para todos os serviços, clique no link Redefinir Todas as Estatísticas na parte superior da página.
    5. Observação: Clicar em Redefinir Todas as Estatísticas redefine as estatísticas para todos os serviços monitorados, independente se são exibidos na página ou não.

    Quando você confirma que deseja redefinir as estatísticas, o sistema deleta todas as estatísticas de monitoramento que foram coletadas para todos os serviços na configuração desde que você redefiniu as estatísticas pela última vez. O sistema não deleta as estatísticas sendo coletadas durante o Intervalo de Agregação Atual para nenhum dos serviços. Além disso, depois que você clica em Redefinir Todas as Estatísticas, o sistema começa imediatamente a coletar estatísticas de monitoramento para os serviços de novo.
