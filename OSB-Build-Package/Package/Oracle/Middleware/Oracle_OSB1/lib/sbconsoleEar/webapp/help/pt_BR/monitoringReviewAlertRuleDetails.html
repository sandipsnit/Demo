Revendo a Configuração da Regra de Alerta

Use esta página para rever os dados de configuração informados para esta regra de alerta. Se necessário, você pode clicar em Editar para fazer alterações na configuração antes de salvar a regra de alerta.

Execute uma das etapas a seguir:

  • Para fazer uma alteração em uma das páginas de configuração, clique no ícone Editar da página apropriada.
  • Para retornar à página anterior, clique em Anterior.
  • Para criar o alerta, clique em Salvar. A regra de alerta será criada e salva na sessão atual.
  • Para desconsiderar as alterações, clique em Cancelar.
  • Para encerrar a sessão e implantar a configuração no run-time, clique em Ativar em Centro de Alterações.
