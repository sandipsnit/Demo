Definindo o Intervalo de Agregação para um Serviço

Use a página Definições Operacionais para definir o intervalo de agregação para um serviço específico. O intervalo de agregação é o período em que as estatísticas agregadas são calculadas para exibição na Console do Oracle Service Bus Administration. A definição do intervalo de agregação default é de 10 minutos.

  1. Se ainda não tiver feito isso, clique em Criar para criar uma nova sessão ou clique em Editar para informar uma sessão existente. Consulte Usando o Centro de Alterações.
  2. Selecione Browser do Recurso > Serviços de Negócios ou Serviços de Proxy.
  3. Clique no nome do serviço.
  4. Selecione a guia Definições Operacionais.
  5. No campo Intervalo de Agregação, selecione a duração do intervalo de agregação em horas e minutos, depois clique em Atualizar.

    6. Se sua seleção para horas exceder 1, então a seleção default para minutos será sempre zero. No entanto, se sua seleção para horas for 0 ou 1, então você poderá configurar intervalos em termos de minutos.

  6. Clique em Atualizar para confirmar as atualizações na sessão atual.
  7. Para encerrar a sessão e implantar a configuração no run-time, clique em Ativar em Centro de Alterações.
