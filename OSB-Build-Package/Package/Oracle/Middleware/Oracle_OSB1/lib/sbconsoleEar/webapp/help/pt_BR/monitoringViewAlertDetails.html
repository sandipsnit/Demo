Exibindo Detalhes de Alerta

Esta página exibe detalhes de alertas específicos, incluindo as informações mostradas na Tabela 25-4.

Tabela 25-4 - Detalhes do Alerta
Propriedade
Descrição
Nome de Alerta
Apenas para alertas de SLA:
O nome designado ao alerta. O nome é um link para a página Configuração da Regra de Alerta. Consulte Exibindo Configurações da Regra de Alerta.
Resumo de Alertas
Apenas para alertas do pipeline:
Uma breve descrição da ação de alerta do pipeline ou do Alerta do Oracle Service Bus (o valor default fornecido para ações de alerta do pipeline que foram desenvolvidas sem texto de resumo do alerta).
Descrição
Uma descrição da regra de alerta.
Timestamp
  • Para alertas do pipeline: A data e hora em que ocorreu o alerta.
  • Para alertas de SLA: A data e hora em que a regra de alerta foi avaliada pela última vez como verdadeira.
Gravidade
A severidade definida pelo usuário do alerta.
  • Fatal
  • Crítico
  • Maior
  • Menor
  • Advertência
  • Normal
Serviço
O nome e o caminho do serviço associado a este alerta. O campo é um link para a página Detalhes de Monitoramento do Serviço (apenas quando o serviço está sendo monitorado ativamente). Consulte Exibindo Métricas de Serviço.
Tipo de Serviço
O tipo de serviço pai: serviço de proxy ou serviço de negócios.
Apenas serviços de proxy podem definir alertas de pipeline, portanto, para alertas de pipeline, todos os serviços-pai são serviços de proxy.
Servidor
O nome do servidor em que este alerta foi gerado.
Apenas para alertas de SLA: exibe N/D.
Anotação
Uma caixa de texto em que é possível informar observações para este alerta. Um ícone Anotação é exibido no Histórico de Alertas para alertas com anotações.
