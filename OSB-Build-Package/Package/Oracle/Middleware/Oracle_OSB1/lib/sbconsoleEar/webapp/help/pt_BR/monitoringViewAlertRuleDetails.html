Exibindo Configurações da Regra de Alerta

Use esta página para exibir detalhes da configuração da regra de alerta, mostrados na Tabela 25-22.

Tabela 25-22 - Regra de Alerta - Configuração Geral
Propriedade
Descrição
Nome da Regra
O nome da regra de alerta.
Descrição da Regra
Uma descrição da regra de alerta.
Resumo de Alertas
Uma breve descrição da regra de alerta. Esse texto também é usado como a linha de assunto de uma mensagem de e-mail se esta regra de alerta for configurada com uma notificação de alerta de e-mail.
Destino de Alerta
O caminho e o nome do recurso do destino de alerta que contém o endereço de destino para notificações de alerta. Este campo é um link para a página Exibir Destino de Alerta - Configuração Geral. Consulte Editando Destinos de Alerta.
Horário de Início (HH:MM)
O horário de início desta regra de alerta.
Horário de Término (HH:MM)
O horário de término desta regra de alerta.
Data de Expiração da Regra (MM/DD/AAAA)
A data em que esta regra de alerta não se aplica mais.
Regra Ativada
Independente de esta regra de alerta estar ativada.
Severidade do Alerta
A severidade do alerta emitido como um resultado desta regra de alerta:
  • Normal
  • Advertência
  • Menor
  • Maior
  • Crítico
  • Fatal
Frequência de Alerta
A frequência do alerta:
  • Sempre
  • Notificar Uma Vez
Parar de Processar Mais Regras
Se ou não o alerta para de processar mais regras.
