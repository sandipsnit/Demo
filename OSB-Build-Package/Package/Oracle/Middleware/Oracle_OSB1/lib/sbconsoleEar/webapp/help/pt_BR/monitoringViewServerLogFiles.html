Exibindo Arquivos de Log do Domínio

Esta página exibe um resumo das entradas do arquivo de log do domínio, incluindo as informações mostradas na Tabela 25-17.

  • Para personalizar a exibição de informações do arquivo de log do domínio, consulte Personalizando a Exibição de Entradas de Arquivo de Log do Domínio.
  • Para exibir detalhes de um arquivo de log específico, selecione o botão de rádio da entrada apropriada, depois clique em Exibir. Consulte Exibindo Detalhes de Arquivos de Log do Domínio.

    • Tabela 25-17 - Informações Resumidas de Log do Domínio
    Propriedade
    Descrição
    Data
    A data em que a entrada foi registrada.
    Subsistema
    O subsistema associado à entrada.
    Gravidade
    A severidade da mensagem.
    ID da Mensagem
    A identificação exclusiva para a mensagem.
    Mensagem
    A descrição da mensagem.
