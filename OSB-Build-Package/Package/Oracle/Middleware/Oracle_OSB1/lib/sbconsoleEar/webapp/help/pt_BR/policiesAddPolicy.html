Adicionando Políticas de Web Services Personalizadas

É possível importar um arquivo XML que contenha a Política de Web Services ou gravar a Política de Web Services diretamente na Console do Oracle Service Bus Administration.

  1. Se ainda não tiver feito isso, clique em Criar para criar uma nova sessão ou clique em Editar para informar uma sessão existente. Consulte Usando o Centro de Alterações.
  2. Selecione o Project Explorer, depois selecione um projeto ou pasta no qual adicionar a Política de Web Services. A página View de Projeto/Pasta é mostrada.
  3. Na lista drop-down Criar Recurso, selecione Política de Web Services na Interface para exibir a página Criar uma Nova Política de Web Services.
  4. No campo Nome do Recurso, informe um nome exclusivo para essa Política WS.

    Siga as Restrições para Nomeação de Recursos para obter orientações em relação a nomes.
  5. No campo Descrição do Recurso, informe uma descrição desta Política de Web Services.
  6. No campo Política, execute um dos seguintes procedimentos:
    • Clique em Procurar para localizar e importar um arquivo XML que contenha sua Política de Web Services personalizada.
    • Informe texto para a nova Política de Web Services.
    • Copie e cole texto a partir de uma Política de Web Services existente neste campo.
  7. Clique em Salvar. A Política de Web Services será salva na sessão atual.
  8. Para encerrar a sessão e implantar a configuração no run-time, clique em Ativar em Centro de Alterações.
