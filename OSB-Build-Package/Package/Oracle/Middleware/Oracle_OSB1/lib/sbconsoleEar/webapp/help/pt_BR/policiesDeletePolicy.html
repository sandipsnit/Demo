Deletando Políticas de Web Services Personalizadas

Se qualquer serviço de negócios ou de proxy for configurado para usar a Política de Web Services, remova-a do serviço de negócios ou de proxy.

  1. Se ainda não tiver feito isso, clique em Criar para criar uma nova sessão ou clique em Editar para informar uma sessão existente. Consulte Usando o Centro de Alterações.
  2. Selecione Resource Browser > Políticas de Web Services.
  3. Clique no ícone Deletar no campo Opções da Política de Web Services que você deseja deletar.

    4. A Política de Web Services será deletada na sessão atual. Se um serviço de negócios ou um serviço de proxy tiver sido configurado para usar uma Política de Web Services, um ícone Advertência de Deleção indicará que é possível deletar uma Política de Web Services com uma confirmação de advertência. Isso pode resultar em conflitos devido a referências não resolvidas do serviço à Política de Web Services deletada.

  4. Para encerrar a sessão e implantar a configuração no run-time, clique em Ativar em Centro de Alterações.
