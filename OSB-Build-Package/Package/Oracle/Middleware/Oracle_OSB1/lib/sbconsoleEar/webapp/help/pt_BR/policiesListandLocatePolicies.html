Localizando Políticas de Web Services Personalizadas

A página Resumo de Políticas de Web Services lista as Políticas de Web Services (WS-Policies) personalizadas que você adicionou ao domínio atual do Oracle Service Bus. A Console do Oracle Service Bus Administration não exibe as Políticas de Web Services do Oracle Service Bus, que são pré-compactadas e disponibilizadas a todos os domínios do Oracle Service Bus.

  1. Execute um dos seguintes procedimentos:
    • Selecione o Project Explorer para exibir a página View de Projetos ou a página View de Projeto/Pasta. Em seguida, navegue pelos projetos e pastas para localizar a Política de Web Services.
    • Selecione Resource Browser > Políticas de Web Services. A página Resumo de Políticas de Web Services exibe as informações mostradas na Tabela 11-1. Para obter uma descrição mais detalhada das propriedades, consulte Localizando Políticas de Web Services Personalizadas.
  2. Para procurar uma Política de Web Services personalizada, informe parte ou todo o nome da Política de Web Services no campo Nome. Também é possível informar parte ou todo o nome e pasta do projeto de Política de Web Services nos campos Caminho. Clique em Pesquisar.

    3. Clique em Exibir Tudo para remover os filtros de pesquisa e exibir todas as políticas de web services.

    Tabela 11-1 - Informações sobre Política de Web Services
    Propriedade
    Descrição
    Nome da Política de Web Services
    O nome exclusivo designado à Política de Web Services. Clique no nome para ver a página Exibir Detalhes da Política de Web Services. Consulte Editando Políticas de Web Services Personalizadas.
    Caminho
    O nome do projeto e o nome da pasta em que reside a Política de Web Services. Clique no nome para ver o projeto ou pasta que contém este recurso. Consulte Qualificando Nomes de Recursos Usando Projetos e Pastas.
    Opções
    Contém um ícone Deletar. Se um serviço de negócios ou um serviço de proxy tiver sido configurado para usar uma Política de Web Services, um ícone Advertência de Deleção indicará que é possível deletar uma Política de Web Services com uma confirmação de advertência. Isso pode resultar em conflitos devido a referências não resolvidas do serviço à Política de Web Services deletada. Consulte Deletando Políticas de Web Services Personalizadas
