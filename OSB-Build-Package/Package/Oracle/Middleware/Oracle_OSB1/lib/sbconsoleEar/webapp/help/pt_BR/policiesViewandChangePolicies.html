Editando Políticas de Web Services Personalizadas

Use a página Exibir Política de Web Services para exibir e alterar detalhes de uma Política de Web Services (WS-Policy) personalizada específica. A Console do Oracle Service Bus Administration não exibe as Políticas de Web Services do Oracle Service Bus, que são somente para leitura.

  1. Se ainda não tiver feito isso, clique em Criar para criar uma nova sessão ou clique em Editar para informar uma sessão existente. Consulte Usando o Centro de Alterações.
  2. Localize a Política de Web Services, conforme descrito em Localizando Políticas de Web Services Personalizadas.
  3. Clique no nome da Política de Web Services. A página Exibir Detalhes da Política de Web Services exibe as informações mostradas na Tabela 11-2.

    4. Tabela 11-2 Detalhes da Política de Web Services 
    Propriedade
    Descrição
    Última Modificação
    O usuário que criou esta Política de Web Services ou a importou para a configuração.
    Última Modificação em
    A data e hora em que o usuário criou esta Política de Web Services ou a importou para a configuração. Clique no link da data e hora para exibir o histórico de alterações deste recurso. Consulte a página Exibir Histórico de Alteração.
    Referências
    O número de objetos referenciados por esta Política de Web Services. Se tais referências existirem, clique no link numérico para exibir uma lista dos objetos. Consulte Exibindo Referências a Recursos.
    Referenciado por
    O número de objetos que referenciam essa Política de Web Services. Se tais referências existirem, clique no link numérico para exibir uma lista dos objetos. Consulte Exibindo Referências a Recursos.
    Descrição
    Uma descrição desta Política de Web Services, se existir.
    Política
    O texto desta Política de Web Services.

  4. Para fazer uma alteração nos campos, clique em Editar. Consulte Adicionando Políticas de Web Services Personalizadas para obter descrições dos campos.

    5. Não é possível alterar o campo Nome do Recurso.

  5. Clique em Salvar para confirmar as atualizações na sessão atual.
  6. Para encerrar a sessão e implantar a configuração no run-time, clique em Ativar em Centro de Alterações.
