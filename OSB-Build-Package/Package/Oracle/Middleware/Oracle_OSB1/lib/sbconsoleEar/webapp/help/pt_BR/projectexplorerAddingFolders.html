Adicionando Pastas

  1. Se ainda não tiver feito isso, clique em Criar para criar uma nova sessão ou clique em Editar para informar uma sessão existente. Consulte Usando o Centro de Alterações.
  2. Selecione o Project Explorer, depois selecione um projeto ou pasta no qual adicionar a nova pasta.
  3. No campo Informar Novo Nome da Pasta, da página View de Projeto/Pasta, informe um nome exclusivo para a pasta. Consulte Nomeando Projetos e Pastas.
  4. Clique em Adicionar Pasta. A nova pasta será salva na sessão atual.
  5. Para encerrar a sessão e implantar a configuração no run-time, clique em Ativar em Centro de Alterações.
