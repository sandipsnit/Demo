Clonando Projetos

Há três maneiras de clonar (copiar) um projeto:

  • Clone o projeto com um novo nome no nível-raiz. Nesse caso, o projeto original e seu clone existirão como projetos semelhantes. O projeto clonado conterá as mesmas pastas e recursos do original (pelo menos até que você modifique a origem ou o clone). Para obter instruções, consulte Clonando um Projeto no Nível-raiz.
  • Clone o projeto usando o nome de um projeto existente (diferente do projeto de origem). Nesse caso, as pastas e recursos do projeto de origem serão mesclados com as pastas e recursos do projeto cujo nome foi designado para o clone. O projeto original permanece inalterado. Para obter instruções, consulte Clonando um Projeto no Nível-raiz.
  • Clone o projeto em um novo local, em um projeto existente ou na pasta do projeto. Nesse caso, o projeto clonado será rebaixado para uma pasta no novo local. Ele conterá as mesmas pastas e recursos do projeto original.

    • Se você clonar um projeto com o mesmo nome e no mesmo local de uma pasta existente, o conteúdo (subpastas e recursos) do projeto será mesclado com o conteúdo da pasta de destino.

    Em qualquer um dos casos, o projeto original permanecerá inalterado.

    Para obter instruções, consulte Clonando um Projeto e Rebaixando o Clone para uma Pasta.

Clonando um Projeto no Nível-raiz
  1. Se ainda não tiver feito isso, clique em Criar para criar uma nova sessão ou clique em Editar para informar uma sessão existente. Consulte Usando o Centro de Alterações.
  2. Selecione o Project Explorer > Projetos para exibir a página View de Projetos.
  3. Clique no ícone Clonar Projeto na linha do projeto que você deseja clonar. A página Clonar Projeto será exibida com o local atual do projeto destacado.
  4. No campo Novo Nome, informe um nome para o clone. Consulte Nomeando Projetos e Pastas.

    5. Use um nome exclusivo para criar o clone como um novo projeto ou use o nome de um projeto existente para mesclar o clone com esse projeto.

  5. No painel Local, mantenha o local Raiz default ao clonar um projeto no nível raiz.
  6. Clique em Clonar Projeto. O projeto será clonado na sessão atual.
  7. Para encerrar a sessão e implantar a configuração no run-time, clique em Ativar em Centro de Alterações.
Clonando um Projeto e Rebaixando o Clone para uma Pasta
  1. Se ainda não tiver feito isso, clique em Criar para criar uma nova sessão ou clique em Editar para informar uma sessão existente. Consulte Usando o Centro de Alterações.
  2. Selecione o Project Explorer > Projetos para exibir a página View de Projetos.
  3. Clique no ícone Clonar Projeto na linha do projeto que você deseja clonar. A página Clonar Projeto será exibida com o local atual do projeto destacado.
  4. No campo Novo Nome, como opção, informe um novo nome para a pasta. Consulte Nomeando Projetos e Pastas.

    5. Use um nome exclusivo para criar uma pasta nova ou use o nome de uma pasta existente para mesclar o clone com a pasta existente (no local que você selecionará na próxima etapa).

  5. No painel Local, selecione um projeto ou pasta, depois clique em Clonar Projeto. O projeto clonado será convertido em uma pasta no novo local e salvo na sessão atual.
  6. Para encerrar a sessão e implantar a configuração no run-time, clique em Ativar em Centro de Alterações.
