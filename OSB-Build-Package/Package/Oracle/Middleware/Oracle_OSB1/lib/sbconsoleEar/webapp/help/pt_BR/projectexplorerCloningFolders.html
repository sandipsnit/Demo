Clonando Pastas

  1. Se ainda não tiver feito isso, clique em Criar para criar uma nova sessão ou clique em Editar para informar uma sessão existente. Consulte Usando o Centro de Alterações.
  2. Selecione o Project Explorer, navegue até o projeto ou pasta pai que contém a pasta que você deseja clonar. A página View de Projeto/Pasta é mostrada.
  3. Clique no ícone Clonar Pasta na linha da pasta que você deseja clonar. A página Clonar Pasta será exibida com o local atual da pasta destacada.
  4. Clone a pasta no mesmo local ou em um local diferente ou faça upgrade da pasta para um projeto, conforme a seguir:
    • Para clonar a pasta no mesmo local, informe um novo nome no campo Novo Nome. Consulte Nomeando Projetos e Pastas.

      • Ao clonar no mesmo local, não selecione absolutamente nenhum local no painel Local (que é equivalente a selecionar o local atual) ou selecione explicitamente o projeto e pasta atuais.

    • Para clonar a pasta em um local diferente, como opção informe um nome novo no campo Novo Nome e selecione um projeto e/ou pasta na área Local. Consulte Nomeando Projetos e Pastas.

      • Se você usar o mesmo nome de uma pasta existente, o conteúdo da pasta original será mesclado com o conteúdo da pasta de destino.

    • Para fazer upgrade da pasta para um projeto, selecione o local Raiz.
  5. Clique em Clonar Pasta. A pasta será clonada na sessão atual.

    6. Quando você clona uma pasta, todas as pastas e recursos abaixo da pasta são clonados recursivamente

  6. Para encerrar a sessão e implantar a configuração no run-time, clique em Ativar em Centro de Alterações.
