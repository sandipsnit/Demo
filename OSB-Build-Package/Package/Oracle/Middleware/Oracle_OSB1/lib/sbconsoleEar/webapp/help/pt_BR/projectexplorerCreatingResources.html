Criando Recursos

  1. Se ainda não tiver feito isso, clique em Criar para criar uma nova sessão ou clique em Editar para informar uma sessão existente. Consulte Usando o Centro de Alterações.
  2. Selecione o Project Explorer, depois selecione o projeto ou pasta em que você deseja criar o recurso.
  3. Na lista drop-down Criar Recurso na página View de Projeto/Pasta, selecione o tipo de recurso que você deseja criar. Páginas diferentes são exibidas, dependendo do tipo de recurso selecionado.
  4. Siga os prompts para adicionar informações a cada página e criar o recurso. Para obter mais informações sobre a criação de cada tipo de recurso, consulte o tópico apropriado, conforme listado na Tabela 3-3.

  6. Para encerrar a sessão e implantar a configuração no run-time, clique em Ativar em Centro de Alterações.
