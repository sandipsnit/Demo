Deletando Projetos

  1. Se ainda não tiver feito isso, clique em Criar para criar uma nova sessão ou clique em Editar para informar uma sessão existente. Consulte Usando o Centro de Alterações.
  2. Selecione o Project Explorer > Projetos para exibir a página View de Projetos.
  3. Clique no ícone Deletar Projeto na linha do projeto que você deseja deletar. O projeto será deletado na sessão atual.

    4. Quando você deleta um projeto, todos os recursos abaixo do projeto são deletados. Se qualquer recurso abaixo desse projeto for referenciado por recursos abaixo de um projeto diferente, você poderá deletar o projeto mas com uma confirmação de aviso. Isso pode resultar em conflitos devido a referências não resolvidas referentes ao recurso deletado.

    Em vez do ícone Deletar, um ícone Advertência de Deleção é exibido para estes recursos.

  4. Para encerrar a sessão e implantar a configuração no run-time, clique em Ativar em Centro de Alterações.
