Exportando um WSDL

Use a página View de Projeto/Pasta para exportar um WSDL associado a um serviço de proxy ou a um negócio. O WSDL será exportado como um arquivo JAR. Só é possível exportar um WSDL quando você está fora de uma sessão. Consulte Usando o Centro de Alterações.

Use o recurso Exportar WSDL para disponibilizar um WSDL para consumo por ferramentas externas, como IDEs. Observe que isso é diferente da funcionalidade Exportar Recursos no módulo Administração do Sistema, que você usa para mover e separar recursos entre dois domínios.

  1. No painel de navegação à esquerda, clique em um projeto ou pasta no Project Explorer. A página View de Projeto/Pasta é mostrada.
  2. Clique no ícone Exportar WSDL na linha do serviço de proxy cujo WSDL você deseja exportar. A caixa de diálogo Download de Arquivo é exibida.
  3. Na caixa de diálogo Download de Arquivo, clique em Abrir para abrir o arquivo JAR ou clique em Salvar para salvar o arquivo JAR na área de trabalho.
