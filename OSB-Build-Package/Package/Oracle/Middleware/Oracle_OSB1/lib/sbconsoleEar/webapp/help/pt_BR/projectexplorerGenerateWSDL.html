Gerando um WSDL

Um WSDL gerado é um recurso WSDL que o Oracle Service Bus gera para um serviço que não iniciou com um recurso WSDL, mas que pode ser descrito usando um WSDL. Por exemplo, um WSDL criado com base em um serviço baseado no EJB é um WSDL gerado. Para obter mais informações, consulte "How WSDL is Used in Oracle Service Bus" no Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus em http://www.oracle.com/pls/as111170/lookup?id=OSBAG114.

Use a página View de Projeto/Pasta para gerar um WSDL associado a um serviço de negócios do tipo de transporte EJB ou Fluxo.

  1. No painel de navegação à esquerda, clique em um projeto ou pasta no Project Explorer. A página View de Projeto/Pasta é mostrada.
  2. Clique no ícone Gerar WSDL na linha do serviço de negócios cujo WSDL você deseja gerar. A caixa de diálogo Gerar Recurso WSDL é exibida com o local atual (nome do projeto e nome da pasta em que reside o serviço de negócios) destacado.
  3. Na caixa de diálogo Gerar Recurso WSDL, especifique um Novo Nome do Recurso WSDL e o local para o WSDL e clique em Gerar WSDL.

    Siga as Restrições para Nomeação de Recursos para obter orientações em relação a nomes.

Depois de finalizar

Continue em Editando WSDLs para exibir ou modificar o WSDL gerado.
