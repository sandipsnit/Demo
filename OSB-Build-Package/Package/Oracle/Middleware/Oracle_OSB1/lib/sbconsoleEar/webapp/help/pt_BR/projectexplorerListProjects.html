Página View de Projetos

A página View de Projetos lista os projetos definidos no domínio atual. A Tabela 3-1 descreve as opções e controles na página.

Tabela 3-1 - Página View de Projetos - Opções e Controles
Opção ou Controle
Descrição
Informar Novo Nome do Projeto
Para criar um novo projeto, informe um nome no campo de texto e clique em Adicionar Projeto. Consulte Adicionando Projetos.
Consulte também Nomeando Projetos e Pastas.
Nome
Clique no nome de um projeto para exibir a página View de Projeto/Pasta desse projeto.
Opções
Para qualquer projeto listado, você pode executar um dos seguintes procedimentos:
  • Clique no ícone Renomear Projeto para renomear o projeto. Consulte Renomeando Projetos.
  • Clique no ícone Mover Projeto para mover o projeto. Consulte Movendo Projetos (Convertendo em uma Pasta).
  • Clique no ícone Clonar Projeto para clonar o projeto. Consulte Clonando Projetos.
  • Clique no ícone Deletar Projeto para deletar o projeto. Um ícone Advertência de Deleção é exibido quando outros recursos referenciam esse recurso. É possível deletar este recurso com uma confirmação de advertência. Isso pode resultar em conflitos devido a referências não resolvidas referentes ao recurso deletado. Consulte Deletando Projetos.
Quando você executa qualquer uma destas ações, o resultado é salvo na sessão atual. Para encerrar a sessão e implantar a configuração no run-time, clique em Ativar em Centro de Alterações.
