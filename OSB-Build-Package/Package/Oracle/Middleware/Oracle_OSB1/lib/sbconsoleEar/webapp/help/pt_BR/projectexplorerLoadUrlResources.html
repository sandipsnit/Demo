Página Carregar Recursos - Carregar Recursos do URL

Use a página Carregar Recursos - Carregar Recursos do URL para importar recursos de um arquivo disponível na Web, conforme documentado em Carregando Recursos de um URL.

Tabela 3-5 descreve as opções e controles na página.

Tabela 3-5 - Página Carregar Recursos: Carregar Recursos do URL - Opções e Controles
Opções e Controles
Descrição
URL/Caminho
Informe o URL onde o arquivo está localizado.
Para especificar um recurso local, você pode usar o protocolo file com um nome de arquivo totalmente qualificado por seu caminho, por exemplo, file:///c:/osbresources/ForeachAction.jar
Nome do Recurso
Informe um nome para o recurso.
Tipo de Recurso
Selecione o tipo de recurso na lista.
