Tipo de Recurso e Extensão do Arquivo
Cada arquivo no arquivo Zip é um possível recurso, e a extensão do arquivo é a forma geralmente usada para identificar o tipo de arquivo. O Oracle Service Bus define uma extensão default para cada recurso, mas, se você tiver extensões adicionais para descrever esses recursos, esse mapa permite que você declare essas extensões para o carregador.
Depois de definir o arquivo Zip e o mapa de extensão, o carregador do Oracle Service Bus verifica o conteúdo do arquivo e tenta associar um tipo de recurso a cada arquivo. Um arquivo sem uma extensão ou um que não possua uma extensão definida no mapa é considerado desconhecido e é excluído automaticamente do carregamento.
Em arquivos conhecidos, o nome do recurso é o nome do arquivo sem extensão. A estrutura de pastas do arquivo Zip é recriada na pasta da Console do Oracle Service Bus Administration de destino.
