Carregando Recursos de um Arquivo Zip

É possível selecionar um arquivo zip ou JAR que contenha os recursos que deseja carregar; o Oracle Service Bus localiza e importa os recursos deste arquivo zip.

Recursos, como WSDLs ou Esquemas XML, podem definir árvores completas de recursos dependentes. O recurso de carga em alto volume permite fazer upload de um conjunto de recursos por vez; não é necessário resolver as dependências manualmente.

  1. Se ainda não tiver feito isso, clique em Criar para criar uma nova sessão ou clique em Editar para informar uma sessão existente. Consulte Usando o Centro de Alterações.
  2. Selecione o Project Explorer, depois selecione o projeto ou pasta em que você deseja adicionar recursos. A página View de Projeto/Pasta é mostrada.
  3. Na lista drop-down Criar Recurso, selecione Recursos Compactados. A página Carregar Recursos será exibida.
  4. No campo Selecionar Arquivo Zip, especifique um arquivo zip para importação. Informe o nome do arquivo (qualificado por seu caminho) ou clique em Procurar, localize o arquivo, depois clique em Abrir.
  5. Revise o mapeamento de tipos de recurso para extensões de arquivos. Altere ou adicione extensões, de acordo com suas necessidades. Separe várias extensões com vírgulas.

    6. Cada arquivo no arquivo Zip é um possível recurso, e a extensão do arquivo é a forma geralmente usada para identificar o tipo de arquivo. O Oracle Service Bus define uma extensão default para cada recurso, mas, se você tiver extensões adicionais para descrever esses recursos, esse mapa permite que você declare essas extensões para o carregador.

    Depois de definir o arquivo Zip e o mapa de extensão, o carregador do Oracle Service Bus verifica o conteúdo do arquivo e tenta associar um tipo de recurso a cada arquivo. Um arquivo sem uma extensão ou um que não possua uma extensão definida no mapa é considerado desconhecido e é excluído automaticamente do carregamento.

    Em arquivos conhecidos, o nome do recurso é o nome do arquivo sem extensão. A estrutura de pastas do arquivo Zip é recriada na pasta da Console do Oracle Service Bus Administration de destino.

    Uma extensão só pode ser associada a um tipo de arquivo.

  6. Clique em Próximo.
  7. Na página Carregar Recursos - Revisar Recursos Carregados, revise os recursos que você está importando. É possível desmarcar as caixas de seleção ao lado dos recursos que você não deseja incluir nesta importação.
  8. Para importar os recursos selecionados, clique em Importar.
  9. A página Carregar Recursos - Resultado da Importação exibe os recursos importados. Clique em Carregar Outro para importar recursos de outro arquivo ou prossiga com outras tarefas selecionando uma página diferente do painel de navegação à esquerda.
  10. Para encerrar a sessão e implantar a configuração no run-time, clique em Ativar em Centro de Alterações.
