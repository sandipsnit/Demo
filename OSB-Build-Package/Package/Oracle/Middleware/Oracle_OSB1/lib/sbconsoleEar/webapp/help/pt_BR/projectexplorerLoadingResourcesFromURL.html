Carregando Recursos de um URL

É possível executar uma importação em alto volume dos recursos disponíveis em um URL ou no sistema de arquivos. A importação em alto volume permite importar um recurso-raiz, como um WSDL, junto com seus dependentes, como outros WSDLs e esquemas. O mapa de dependência é resolvido automaticamente.

  1. Se ainda não tiver feito isso, clique em Criar para criar uma nova sessão ou clique em Editar para informar uma sessão existente. Consulte Usando o Centro de Alterações.
  2. Selecione o Project Explorer, depois selecione o projeto ou pasta em que você deseja adicionar recursos. A página View de Projeto/Pasta é mostrada.
  3. Na lista drop-down Criar Recurso, selecione Recursos do URL. A página Carregar Recursos - Carregar Recursos do URL será exibida.
  4. No campo URL/Caminho, informe o URL ou caminho do recurso.

    5. Para especificar um recurso local, você pode usar o protocolo file com um nome de arquivo totalmente qualificado por seu caminho, por exemplo, file:///c:/osbresources/ForeachAction.jar

  5. No campo Nome, informe o nome do recurso.
  6. Na lista drop-down Tipo de Recurso, selecione o tipo do recurso.
  7. Clique em Próximo para exibir a página Carregar Recursos - Revisar Recursos Carregados. Desmarque as caixas de seleção ao lado dos recursos que você não deseja incluir nesta importação.
  8. Para importar os recursos selecionados, clique em Importar.
  9. A página Carregar Recursos - Resultado da Importação exibe os recursos importados. Clique em Carregar Outro para importar recursos de outro arquivo ou prossiga com outras tarefas selecionando uma página diferente do painel de navegação à esquerda.
  10. Para encerrar a sessão e implantar a configuração no run-time, clique em Ativar em Centro de Alterações.
