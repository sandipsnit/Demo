Movendo Projetos (Convertendo em uma Pasta)

Quando você "move" um projeto, ele é convertido em uma pasta no novo local.

  1. Se ainda não tiver feito isso, clique em Criar para criar uma nova sessão ou clique em Editar para informar uma sessão existente. Consulte Usando o Centro de Alterações.
  2. Selecione o Project Explorer > Projetos para exibir a página View de Projetos.
  3. Clique no ícone Mover Projeto na linha do projeto que você deseja mover. A página Mover Projeto será exibida com o local atual do projeto destacado.
  4. Em Local, selecione um projeto ou pasta para onde "mover" o projeto. O projeto será convertido em uma pasta no novo local e salvo na sessão atual.

    5. Se você mover um projeto para um novo local e uma pasta com o mesmo nome já existir nesse local, o conteúdo do projeto será mesclado com o conteúdo da pasta existente.

  5. Para encerrar a sessão e implantar a configuração no run-time, clique em Ativar em Centro de Alterações.
