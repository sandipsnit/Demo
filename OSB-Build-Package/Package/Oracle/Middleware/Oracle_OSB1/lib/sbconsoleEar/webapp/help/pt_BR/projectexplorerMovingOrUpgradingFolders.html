Movendo ou Fazendo Upgrade de Pastas

  1. Se ainda não tiver feito isso, clique em Criar para criar uma nova sessão ou clique em Editar para informar uma sessão existente. Consulte Usando o Centro de Alterações.
  2. Selecione o Project Explorer, depois selecione o projeto ou pasta que contém a pasta que você deseja mover ou fazer upgrade. A página View de Projeto/Pasta é mostrada.
  3. Clique no ícone Mover Pasta na linha da pasta que você deseja mover ou atualizar. A página Mover Pasta será exibida com o local atual da pasta destacada.
  4. Determine se você moverá a pasta para um novo local em um projeto diferente ou se fará upgrade da pasta para um projeto:
    • Para movê-lo para um novo local, no painel Local, selecione um projeto ou pasta para o qual mover a pasta.
    • Para fazer upgrade da pasta para um projeto, selecione o local Raiz.
  5. Clique em Submeter. A pasta (ou projeto, se tiver sido atualizado de uma pasta) será salva em seu novo local na sessão atual.

    6. Quando você move ou faz upgrade de uma pasta, se outra pasta já existir nesse local com o mesmo nome, o conteúdo da pasta movida ou atualizada será mesclado com o conteúdo da pasta ou projeto existente.

  6. Para encerrar a sessão e implantar a configuração no run-time, clique em Ativar em Centro de Alterações.
