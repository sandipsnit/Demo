Movendo Recursos

  1. Se ainda não tiver feito isso, clique em Criar para criar uma nova sessão ou clique em Editar para informar uma sessão existente. Consulte Usando o Centro de Alterações.
  2. Selecione o Project Explorer, depois selecione o projeto ou pasta que contém o recurso que você deseja mover. A página View de Projeto/Pasta é mostrada.
  3. Clique no ícone Mover na linha do recurso que você deseja mover. A página Mover será exibida com o local atual do recurso destacado.
  4. No campo Local, selecione o projeto e a pasta para o qual você deseja mover o arquivo.
  5. Clique em Mover Arquivo. O recurso será movido para o novo local na sessão atual.
  6. Para encerrar a sessão e implantar a configuração no run-time, clique em Ativar em Centro de Alterações.
