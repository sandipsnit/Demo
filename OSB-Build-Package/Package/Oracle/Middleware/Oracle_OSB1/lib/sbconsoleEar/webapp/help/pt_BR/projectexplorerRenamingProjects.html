Renomeando Projetos

  1. Se ainda não tiver feito isso, clique em Criar para criar uma nova sessão ou clique em Editar para informar uma sessão existente. Consulte Usando o Centro de Alterações.
  2. Selecione o Project Explorer > Projetos para exibir a página View de Projetos.
  3. Clique no ícone Renomear Projeto na linha do projeto que você deseja renomear. A página Renomear Projeto será exibida.
  4. No campo Novo Nome do Projeto, informe o novo nome para o projeto. Consulte Nomeando Projetos e Pastas.

    5. Se você usar o nome de um projeto existente, os recursos de ambos os projetos serão mesclados com esse nome. Por exemplo, considere o projeto MyProject com os recursos A e B, e o projeto ThisProject com os recursos C e D. Se você renomear MyProject como ThisProject, o novo ThisProject terá os recursos A, B, C e D.

  5. Clique em Renomear. O projeto será renomeado na sessão atual.
  6. Para encerrar a sessão e implantar a configuração no run-time, clique em Ativar em Centro de Alterações.
