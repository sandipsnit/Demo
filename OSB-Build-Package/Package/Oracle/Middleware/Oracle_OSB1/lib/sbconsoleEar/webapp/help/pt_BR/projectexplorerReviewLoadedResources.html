Página Carregar Recursos - Revisar Recursos Carregados

A página Carregar Recursos - Revisar Recursos Carregados exibe detalhes sobre recursos sendo importados de um arquivo Zip ou de outro arquivo, conforme documentado em Carregando Recursos de um Arquivo Zip ou em Carregando Recursos de um URL.

Tabela 3-6 descreve as opções e controles na página.

Tabela 3-6 - Página Carregar Recursos: Revisar Recursos Carregados - Opções e Controles
Opções e Controles
Descrição
Nome do Arquivo
O nome do arquivo do recurso, incluindo a extensão do arquivo.
Nome
O nome do recurso.
Caminho
O caminho do recurso, que é o projeto e a pasta em que o recurso reside.
Tipo de Recurso
Lista os tipos de recurso suportados.

Um ícone de Advertência ao lado do nome do arquivo indica que o tipo do recurso é desconhecido. (Um arquivo sem uma extensão ou um que não possua uma extensão definida no mapa é considerado desconhecido.) Arquivos de tipos de arquivo desconhecidos não podem ser importados.
