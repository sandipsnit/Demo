Página Exibir Histórico de Alterações

A página Exibir Histórico de Alterações exibe as dez alterações mais recentes feitas em um recurso, entre todas as ativações de sessão.

Tabela 3-9 descreve as opções e controles na página.

Tabela 3-9 - Página Histórico de Alterações - Opções e Controles
Opção ou Controle
Descrição
Tarefa
O tipo de tarefa executada:
  • Criar
  • Atualizar
  • Clonagem
  • Renomear
  • Mover
Usuário
O usuário que criou ou editou este recurso.
Tempo de Execução
A data e hora em que o usuário criou ou editou este recurso.
