Página Exibir Referências

Dependendo se você clicou em Referências ou Referenciado por em uma página View de Projeto/Pasta, a página Exibir Referências exibe uma lista de:

  • Recursos fora do projeto ou pasta atual que sejam referenciados por recursos dentro do projeto ou pasta.
  • Recursos fora do projeto ou pasta atual que referenciem recursos dentro do projeto ou pasta.

Tabela 3-8 descreve as opções e controles na página.

Tabela 3-8 - Página Referências do Projeto - Opções e Controles 
Opção ou Controle
Descrição
Nome
Clique no nome do recurso para exibir detalhes sobre ele.
Tipo de Recurso
Exibe Projeto, Pasta ou qualquer um dos tipos de recurso suportados pelo Oracle Service Bus.
