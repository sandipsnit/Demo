Página View de Projeto/Pasta

A página View de Projeto/Pasta exibe detalhes de um único projeto. A Tabela 3-2 descreve as opções e controles na página.

Tabela 3-2 - Página View de Projeto/Pasta - Opções e Controles
Opção ou Controle
Descrição
Nome do Projeto
Referências
Clique no número neste campo (se for maior que 0) para exibir uma lista dos recursos fora deste projeto dos quais dependem os recursos deste projeto.
Referenciado por
Clique no número neste campo (se for maior que 0) para exibir uma lista dos recursos fora deste projeto que são dependentes dos recursos deste projeto.
Descrição
Clique em Editar Descrição para adicionar ou editar uma descrição deste projeto, depois clique em Submeter Descrição para atualizá-la.
Pastas
Informar um Novo Nome de Pasta
Para criar uma nova pasta neste projeto, informe um nome no campo de texto e clique em Adicionar Pasta. Consulte Adicionando Pastas.
Consulte também Nomeando Projetos e Pastas.
Nome
Clique no nome de uma pasta para exibir a página View de Projeto/Pasta dessa pasta.
Opções
Para qualquer pasta listada, você pode executar um dos seguintes procedimentos:
  • Clique no ícone Renomear Pasta para renomear a pasta. Consulte Renomeando Pastas
  • Clique no ícone Fazer Upgrade para o Projeto ou Mover Pasta para mover a pasta ou para fazer upgrade dela para um projeto. Consulte Movendo ou Fazendo Upgrade de Pastas.
  • Clique no ícone Clonar Pasta para clonar a pasta. Consulte Clonando Pastas
  • Clique no ícone Deletar Pasta para deletar a pasta. Um ícone Advertência de Deleção é exibido quando outros recursos referenciam esse recurso. É possível deletar o recurso com uma confirmação de advertência. Isso pode resultar em conflitos devido a referências não resolvidas referentes ao recurso deletado. Consulte Deletando Pastas.
Quando você executa qualquer uma destas ações, o resultado é salvo na sessão atual. Para encerrar a sessão e implantar a configuração no run-time, clique em Ativar em Centro de Alterações.
Deletar
Para deletar pastas, marque as caixas de seleção ao lado dos nomes das pastas para deletar, em seguida clique em Deletar.
Recursos
Criar Recurso
Para criar um novo recurso para este projeto, selecione um tipo de recurso na lista Criar Recurso. Consulte Criando Recursos.
Nome
Clique no nome de um recurso para exibir detalhes sobre ele.
Tipo de Recurso
O tipo deste recurso.
Ações
As ações disponíveis para um recurso dependem do tipo de recurso:
  • O ícone Editar Fluxo de Mensagens é exibido para serviços de proxy. Clique nesse ícone para editar a mensagem de um serviço de proxy. Consulte Criando, Editando e Exibindo Fluxos de Mensagens.
  • O ícone Acionar Console de Teste é exibido para serviços de negócios, serviços de proxy, transformações XQuery, transformações XSL e arquivos MFL. Clique nesse ícone para chamar a Console de Teste, que você usa para validar e testar o design de serviços e transformações. Para serviços, você só pode usar a Console de Teste no run-time; ou seja, quando a sessão for ativada. Para transformações, é possível usar a Console de Teste dentro ou fora de uma sessão. Consulte Testando Serviços e Testando Transformações.
  • O ícone Exportar WSDL é exibido para serviços de proxy baseados no WSDL. Clique nesse ícone para exportar um WSDL, que você pode exibir ou modificar em uma ferramenta externa, como um IDE.

    • Observe que isso é diferente da funcionalidade Exportar Recursos no módulo Administração do Sistema, que você usa para mover e separar recursos entre dois domínios. Consulte Exportando um WSDL.

  • O ícone Gerar WSDL é exibido para serviços de negócios do tipo de transporte, como EJB e Fluxo. Clique neste ícone para gerar um WSDL, que você pode depois exibir ou modificar. Consulte Gerando um WSDL.
Opções
A coluna Opções exibe os seguintes ícones:
  • Clique no ícone Renomear Recurso para renomear o recurso. Consulte Renomeando Recursos
  • Clique no ícone Mover Recurso para mover o recurso. Consulte Movendo Recursos.
  • Clique no ícone Clonar Recurso para clonar o recurso. Consulte Clonando Recursos
  • Clique no ícone Deletar para deletar o recurso. Um ícone Advertência de Deleção é exibido quando outros recursos referenciam esse recurso. É possível deletar este recurso com uma confirmação de advertência. Isso pode resultar em conflitos devido a referências não resolvidas referentes ao recurso deletado.
Quando você executa qualquer uma destas ações, o resultado é salvo na sessão atual. Para encerrar a sessão e implantar a configuração no run-time, clique em Ativar em Centro de Alterações.
