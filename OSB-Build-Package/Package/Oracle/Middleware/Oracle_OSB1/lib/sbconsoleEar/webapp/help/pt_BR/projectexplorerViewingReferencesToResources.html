Exibindo Referências a Recursos

Você pode exibir referências a recursos independente de estar em uma sessão. Consulte Usando o Centro de Alterações.

  1. Clique em Referências ou em Referenciado por em uma página de detalhes do projeto, pasta ou recurso nos módulos Project Explorer ou Resource Browser.
    • Clique em Referencia para exibir recursos fora de um projeto ou pasta que sejam referenciados por recursos dentro do projeto ou pasta.
    • Clique em Referenciado por para exibir recursos fora de um projeto ou pasta que referenciem recursos dentro do projeto ou pasta.

      • A página Exibir Referências é exibida.

  2. Clique no nome de um recurso para exibir detalhes sobre ele ou clique em Voltar para retornar para a página anterior.
