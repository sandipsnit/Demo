Deletando Provedores de Chaves de Serviço

Quando você deleta um provedor de chaves de serviço, o Oracle Service Bus também deleta os bindings de alias ao par de chaves associados do provedor de mapeamento de credenciais de PKI. O Oracle Service Bus não deleta o par de certificados de chave associados da área de armazenamento de chaves.

  1. Se ainda não tiver feito isso, clique em Criar para criar uma nova sessão ou clique em Editar para informar uma sessão existente. Consulte Usando o Centro de Alterações.
  2. Se qualquer serviço de proxy for configurado para usar o provedor de chaves de serviço, remova o provedor de chaves de serviço do serviço de proxy. Consulte Editando Configurações de Serviço de Proxy.
  3. Selecione Resource Browser > Provedores de Chaves de Serviço para exibir a página Resumo de Provedores de Chaves de Serviço.
  4. Clique no ícone Deletar no campo Opções do provedor de chaves de serviço que você deseja deletar. O provedor de chaves de serviço será deletado na sessão atual. Se um serviço de negócios ou um serviço de proxy tiver sido configurado para usar uma conta de serviço, um ícone de Advertência de Deleção indicará que é possível deletar o provedor de chaves de serviço com uma confirmação de advertência. Isso pode resultar em conflitos devido a referências não resolvidas do serviço ao provedor de chaves de serviço deletado.
  5. Para encerrar a sessão e implantar a configuração no run-time, clique em Ativar em Centro de Alterações.
