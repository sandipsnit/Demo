Localizando Provedores de Chaves de Serviço

  1. Execute um dos seguintes procedimentos:
    • Selecione o Project Explorer para exibir a página View de Projetos ou a página View de Projeto/Pasta. Em seguida, navegue pelos projetos e pastas para localizar o provedor de chaves de serviço.
    • Selecione Resource Browser > Provedores de Chaves de Serviço. O Resumo de Provedores de Chaves de Serviço exibe as informações mostradas na Tabela 17-1.
  2. Para procurar um provedor de chaves de serviço, informe parte ou todo o nome do provedor no campo Nome. Também é possível informar parte ou todo o nome e pasta do projeto de provedor nos campos Caminho. Clique em Pesquisar.

    3. Clique em Exibir Tudo para remover os filtros de pesquisa e exibir todos os provedores de chaves de serviço.

    Tabela 17-1 Informações sobre o Provedor de Chaves de Serviço 
    Propriedade
    Descrição
    Nome
    Um nome exclusivo para o provedor de chaves de serviço. Clique no nome para ver a página Exibir Detalhes do Provedor de Chaves de Serviço. Consulte Editando Provedores de Chaves de Serviço.
    Caminho
    O nome do projeto e o nome da pasta em que reside o provedor de chaves de serviço. Clique no nome para ver o projeto ou pasta que contém este recurso. Consulte Qualificando Nomes de Recursos Usando Projetos e Pastas.
