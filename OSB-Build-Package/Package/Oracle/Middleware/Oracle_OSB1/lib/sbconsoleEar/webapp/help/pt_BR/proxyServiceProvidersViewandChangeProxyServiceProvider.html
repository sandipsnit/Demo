Editando Provedores de Chaves de Serviço

Use a página Exibir Detalhes do Provedor de Chaves de Serviço para exibir e alterar os detalhes de um provedor de chaves de serviço específico.

  1. Se ainda não tiver feito isso, clique em Criar para criar uma nova sessão ou clique em Editar para informar uma sessão existente. Consulte Usando o Centro de Alterações.
  2. Localize o provedor de chaves de serviço, conforme descrito em Localizando Provedores de Chaves de Serviço.
  3. Clique no nome do provedor de chaves de serviço. A página Exibir Detalhes do Provedor de Chaves de Serviço exibe as informações mostradas na Tabela 17-3.

    4. Tabela 17-3 - Detalhes do Provedor de Chaves de Serviço
    Propriedade
    Descrição
    Nome do Provedor de Chaves de Serviço
    O nome deste provedor de chaves de serviço.
    Última Modificação
    O usuário que criou este provedor de chaves de serviço ou o importou para a configuração.
    Última Modificação em
    A data e hora em que o usuário criou este provedor de chaves de serviço ou o importou para a configuração. Clique no link da data e hora para exibir o histórico de alterações deste recurso. Consulte a página Exibir Histórico de Alteração.
    Referências
    O número de objetos referenciados por este provedor de chaves de serviço. Se tais referências existirem, clique no link numérico para exibir uma lista dos objetos. Consulte Exibindo Referências a Recursos.
    Referenciado por
    O número de objetos que referenciam este provedor de chaves de serviço. Se tais referências existirem, clique no link numérico para exibir uma lista dos objetos. Por exemplo, se você selecionou este provedor de chaves de serviço como o provedor de serviço para um serviço de proxy específico, o serviço de proxy será listado como uma referência quando você clicar no link. Consulte Exibindo Referências a Recursos.
    Descrição
    Uma descrição deste provedor de chaves de serviço, se existir.

  4. Para fazer uma alteração nos campos, clique em Editar. Consulte Adicionando Provedores de Chaves de Serviço para obter descrições dos campos.
  5. Clique em Salvar para confirmar as atualizações na sessão atual.
  6. Para encerrar a sessão e implantar a configuração no run-time, clique em Ativar em Centro de Alterações.
