Adicionando Ações de Transformação MFL

Use a ação de transformação do MFL (Message Format Language) para converter o conteúdo da mensagem de XML para não-XML, ou vice-versa, no pipeline da mensagem. Um MFL é um documento XML especializado usado para descrever o layout de dados binários. Trata-se de uma linguagem proprietária da Oracle usada para definir regras para transformação de dados binários formatados em dados XML, ou vice-versa. Consulte MFLs.

Antes de começar

Estas instruções consideram que você já esteja editando um fluxo de mensagens na página Editar Fluxo de Mensagens, conforme explicado em Criando, Editando e Exibindo Fluxos de Mensagens.

Para adicionar uma ação de Transformação MFL

  1. Navegue até o local onde você deseja adicionar a ação, conforme descrito em Adicionando e Editando Ações em Fluxos de Mensagens.
  2. Clique no ícone apropriado, depois selecione Adicionar uma Ação > Processamento da Mensagem > Transformação MFL.
  3. Na lista drop-down Aplicar Transformação MFL, selecione XML para Não-XML ou Não-XML para XML, de acordo com a necessidade.
  4. Clique em <Expression>. Usando o Editor de Expressão XQuery, especifique a variável em que a ação de transformação MFL deve ser executada. Essa entrada deve ser texto ou entrada na transformação para XML e deve ser XML na transformação para não-XML. O conteúdo binário no contexto da mensagem é representado pelo elemento XML do conteúdo binário. Esse XML deve ser o resultado da expressão Xquery quando a entrada precisar ser binária. Consulte Criando e Editando Expressões XQuery e XPath em Linha.
  5. Selecione uma das seguintes opções:
    • Recurso MFL: clique no link <resource>. A página Selecionar MFL será exibida. Selecione o recurso MFL estático que executará a ação de transformação MFL.
    • Recurso MFL de: clique no link Expressão. A página Editor de Expressão XQuery será exibida. Usando o Editor de Expressão XQuery, crie ou edite uma expressão XQuery para especificar dinamicamente um recurso MFL que executará a ação de transformação, no formato projeto/pasta/nomedorecursoMFL. Consulte Criando e Editando Expressões XQuery e XPath em Linha.
  6. No campo Designar à Variável, informe o nome da variável à qual o resultado dessa ação de transformação deve ser designado. O resultado será um elemento XML de conteúdo binário.

Depois de finalizar

Ao concluir a configuração desta ação, continue configurando outras ações ou salvando sua configuração, conforme descrito em Adicionando e Editando Ações em Fluxos de Mensagens.
