Adicionando Ações de Publicação

Use uma ação de publicação para identificar um serviço de destino especificado estaticamente para uma mensagem e configurar como a mensagem é compactada e enviada para esse serviço.

Para obter mais informações sobre comportamento de publicação, consulte "Performing Transformations in Message Flows" em Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus em http://www.oracle.com/pls/as111170/lookup?id=OSBAG205.

  1. Navegue até o local onde você deseja adicionar a ação, conforme descrito em Adicionando e Editando Ações em Fluxos de Mensagens.
  2. Selecione Adicionar uma Ação > Comunicação > Publicar.
  3. Clique <Service>. A página Selecionar Serviço será exibida.
  4. Selecione um serviço na lista e, em seguida, clique em Submeter. Este é o serviço de destino para a mensagem.
  5. Se o serviço tiver operações definidas, você poderá especificar uma operação a ser chamada selecionando-a na lista Operação.
  6. Para que a operação de saída seja igual à operação de entrada, marque a caixa de seleção Usar operação de entrada para saída.
  7. Para configurar como a mensagem é compactada e enviada para o serviço, no campo Ações de Solicitação, clique em Adicionar uma Ação. Em seguida, selecione uma ação a ser associada ao serviço. Você pode adicionar mais de uma ação. Consulte Adicionando e Editando Ações em Fluxos de Mensagens.

Depois de finalizar

Ao concluir a configuração desta ação, continue configurando outras ações ou salvando sua configuração, conforme descrito em Adicionando e Editando Ações em Fluxos de Mensagens.
