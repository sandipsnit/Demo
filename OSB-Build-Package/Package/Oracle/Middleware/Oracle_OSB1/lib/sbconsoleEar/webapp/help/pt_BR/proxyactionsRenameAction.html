Adicionando Ações de Renomeação

Use a ação de renomeação para renomear os elementos selecionados por uma expressão XPath sem modificar o conteúdo do elemento. A ação de renomeação pertence a um conjunto de ações de atualização.

Antes de começar

Estas instruções consideram que você já esteja editando um fluxo de mensagens na página Editar Fluxo de Mensagens, conforme explicado em Criando, Editando e Exibindo Fluxos de Mensagens.

Para adicionar uma ação para Renomear

  1. Navegue até o local onde você deseja adicionar a ação, conforme descrito em Adicionando e Editando Ações em Fluxos de Mensagens.
  2. Clique no ícone apropriado, depois selecione Adicionar uma Ação > Processamento da Mensagem > Renomear.
  3. Clique em <XPath>. A página Editor de Expressão XPath será exibida. A expressão XPath é usada para especificar os dados (na variável nomeada) que serão renomeados. Consulte Criando e Editando Expressões XQuery e XPath em Linha.
  4. No campo variável, informe a variável de contexto que conterá o elemento que você deseja renomear. Para saber mais sobre variáveis de contexto, consulte o Apêndice A, Contexto da Mensagem.
  5. Execute um dos seguintes procedimentos:
    • Para renomear os elementos selecionados usando um nome local, selecione o primeiro botão de rádio nome local, depois informe um nome local no campo nome local.
    • Para renomear os elementos selecionados usando um namespace, selecione o primeiro botão de rádio namespace, depois informe um namespace no campo namespace.
    • Para renomear os elementos selecionados usando um nome local e um namespace, selecione o botão de rádio nome local e namespace, depois informe um nome e um namespace locais nos campos nome local e namespace.

Depois de finalizar

Ao concluir a configuração desta ação, continue configurando outras ações ou salvando sua configuração, conforme descrito em Adicionando e Editando Ações em Fluxos de Mensagens.
