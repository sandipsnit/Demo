Adicionando Ações de Resposta

Use a ação de resposta para especifica uma resposta imediata a ser enviada para o chamador.

A ação de resposta pode ser usada na solicitação, na resposta ou no pipeline do erro. É possível configurá-la para resultar em uma resposta com sucesso ou falha. No caso de resposta com falha em que o transporte de entrada seja HTTP, a ação de resposta especificará que uma resposta imediata seja enviada para o chamador.

Antes de começar

Estas instruções consideram que você já esteja editando um fluxo de mensagens na página Editar Fluxo de Mensagens, conforme explicado em Criando, Editando e Exibindo Fluxos de Mensagens.

Para adicionar uma ação de Resposta

  1. Navegue até o local onde você deseja adicionar a ação, conforme descrito em Adicionando e Editando Ações em Fluxos de Mensagens.
  2. Clique no ícone apropriado, depois selecione Adicionar uma Ação > Controle de Fluxo > Responder.
  3. Selecione Com Sucesso para responder que a mensagem foi bem-sucedida ou selecione Com Falha para responder que a mensagem apresenta uma falha.

    4. Resposta com Falha fará com que uma transação, se iniciada pelo Oracle Service Bus, seja abortada.

Depois de finalizar

Ao concluir a configuração desta ação, continue configurando outras ações ou salvando sua configuração, conforme descrito em Adicionando e Editando Ações em Fluxos de Mensagens.
