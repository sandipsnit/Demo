Adicionando Ações de Opções de Roteamento

Use a opção Opções de Roteamento para modificar qualquer ou todas as propriedades a seguir para a solicitação de saída em $outbound: parâmetros URI, Qualidade de Serviço, Modo, Nova Tentativa. Embora essas propriedades possam ser modificadas usando as ações Designar, Inserir, Substituir ou Deletar em $outbound, o uso de opções de Roteamento oferece uma forma mais simples para executar esta tarefa, sem requerer o conhecimento de XPath, XQuery ou da estrutura da variável de contexto $outbound.

A ação Opções de Roteamento só pode ser usada onde a variável de contexto $outbound for válida. Pode ser adicionada às seguintes ações:

  • Publicar
  • Publicação Dinâmica
  • Tabela de Publicação
  • Chamada de Serviço
  • Roteamento
  • Roteamento Dinâmico
  • Tabela de Roteamento

Para obter mais informações sobre roteamento, consulte "Modeling Message Flow in Oracle Service Bus" em Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus em http://www.oracle.com/pls/as111170/lookup?id=OSBAG181.

 

Para configurar uma ação de opções de roteamento:

  1. Navegue até o local onde você deseja adicionar a ação, conforme descrito em Adicionando e Editando Ações em Fluxos de Mensagens.
  2. Clique no ícone apropriado, depois selecione Adicionar uma Ação > Comunicação > Opções de Roteamento.
  3. Execute qualquer ou todas as etapas a seguir:
    • Para definir o URI para a mensagem de saída: Selecione URI e clique no Editor de Expressão XQuery. Informe uma expressão que retorne um URI. Isso substitui o URI do serviço chamado.
    • Para definir o elemento de Qualidade de Serviço: Selecione Qualidade de Serviço e selecione a opção Qualidade de Serviço na lista drop-down. Isso substitui o default que é calculado automaticamente.
    • Para definir o Modo: Selecione Modo e selecione solicitação ou resposta de solicitação na lista drop-down.
      • Observação: Isso normalmente já está definido de forma automática, com base na interface do serviço chamado. No entanto, em alguns casos como os serviços Qualquer Soap ou Qualquer XML, isso não é verdadeiro.
    • Para definir o Intervalo de Repetições: Selecione Intervalo de Repetições e especifique o número de segundos entre as repetições. Isso substitui o default configurado pelo serviço chamado.
    • Para definir a Contagem de Repetições: Selecione Contagem de Repetições e especifique o número de repetições que o sistema deve tentar antes de interromper a ação. Isso substitui o default configurado pelo serviço chamado.
    • Para definir a Prioridade da Mensagem: Selecione Prioridade e clique no Editor de Expressão XQuery. Informe uma expressão que retorne um número inteiro positivo.

Depois de finalizar

Ao concluir a configuração desta ação, continue configurando outras ações ou salvando sua configuração, conforme descrito em Adicionando e Editando Ações em Fluxos de Mensagens.
