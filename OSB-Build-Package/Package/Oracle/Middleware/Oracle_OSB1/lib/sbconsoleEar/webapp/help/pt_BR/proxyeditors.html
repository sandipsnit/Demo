Serviços de Proxy: Editores XQuery e XPath

No fluxo de mensagens de um serviço de proxy, é possível designar expressões XQuery para variáveis de contexto da mensagem, designar ações If...Then... com base no resultado Booleano de uma expressão XQuery, inserir o resultado de uma expressão XQuery em um local identificado em relação a uma expressão XPath, especificar o contexto da mensagem que você deseja registrar por meio de expressões XQuery em variáveis de contexto, etc.

O Editor de Expressão XQuery, o Editor de Condição XQuery e o Editor de Expressão XPath estão disponíveis no contexto apropriado em fluxos de mensagens para construir o tipo de expressão chamado no contexto.
