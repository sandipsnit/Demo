Adicionando Handlers de Erro do Serviço de Proxy

Antes de começar

Estas instruções consideram que você já esteja editando um fluxo de mensagens na página Editar Fluxo de Mensagens, conforme explicado em Criando, Editando e Exibindo Fluxos de Mensagens.

Para adicionar um Handler de Erro do serviço de proxy

  1. Clique no ícone Serviço de Proxy e, em seguida, clique em Adicionar Handler de Erro do Serviço. A página Editar Handler de Erro será exibida.
  2. Clique no ícone Handler de Erro, depois clique em Adicionar Estágio.
  3. Clique no ícone Estágio, depois clique em Editar Estágio. A página Editar Configuração do Estágio será exibida.
  4. Clique em Adicionar uma Ação depois selecione a ação que você deseja adicionar.

    5. Um handler de erro é um pipeline, configurado tal como qualquer outro. Por exemplo, se você usar a ação Publicar para enviar notificações de erro a outros serviços, use a ação Designar para modificar as variáveis de contexto e assim por diante. Consulte Adicionando e Editando Ações em Fluxos de Mensagens. Não há restrições sobre quais ações podem ser encadeadas em um fluxo de mensagens.

    Três ações de erro geralmente usadas são Emissão de Erro, Responder e Retomar.

  5. Adicione outras ações e faça outras edições na página Editar Configuração do Estágio, conforme desejado.
  6. Na página Editar Configuração do Estágio, clique em Salvar para confirmar as atualizações na sessão atual.
  7. Na página Editar Handler de Erro, clique em Salvar para confirmar as atualizações na sessão atual.

Depois de finalizar

Na página Editar Fluxo de Mensagens, continue para construir o fluxo de mensagens, conforme descrito em Criando, Editando e Exibindo Fluxos de Mensagens.
