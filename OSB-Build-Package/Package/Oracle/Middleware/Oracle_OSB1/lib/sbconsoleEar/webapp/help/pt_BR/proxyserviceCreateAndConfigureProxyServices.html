Criando e Configurando Serviços de Proxy

Esta seção descreve como criar e configurar serviços de proxy.

Para serviços JCA, você pode gerar serviços de proxy a partir de Bindings de JCA, conforme descrito em Gerando um Serviço de Proxy a partir de um Recurso de Binding de JCA.

  1. Se ainda não tiver feito isso, clique em Criar para criar uma nova sessão ou clique em Editar para informar uma sessão existente. Consulte Usando o Centro de Alterações.
  2. Selecione o Project Explorer, depois selecione o projeto ou pasta em que você deseja adicionar o serviço de proxy.
  3. Na página View de Projeto/Pasta, selecione Serviço de Proxy na lista drop-down Criar Recurso.
  4. Na página Configuração Geral, informe um nome para o serviço e selecione o tipo de serviço a ser criado.

    Siga as Restrições para Nomeação de Recursos para obter orientações em relação a nomes.
  5. Clique em Próximo. As páginas seguintes dependem das escolhas feitas na primeira página. Insira as informações apropriadas em cada uma das páginas subsequentes até chegar à página de resumo e, em seguida, clique em Salvar para salvar o serviço na sessão atual.

    6. Para obter instruções detalhadas sobre o preenchimento de cada página, consulte o seguinte:

    • Página Configuração Geral
    • Página Configuração do Tipo de Mensagem
    • Página Configuração do Transporte
    • Páginas Configuração do Transporte Específico ao Protocolo
    • Página Configuração da Seleção de Operação
    • Página Tratamento de Mensagens
    • Página Resumo
    • Página Política do Serviço de Proxy
    • Página Segurança do Serviço de Proxy
  6. Para encerrar a sessão e implantar a configuração no run-time, clique em Ativar em Centro de Alterações.
  7. Após a criação de um serviço de proxy, a próxima etapa é configurar seu fluxo de mensagens. O fluxo de mensagens define a implementação de um serviço de proxy. Para saber mais, consulte Criando, Editando e Exibindo Fluxos de Mensagens.

Gerando um Serviço de Proxy a partir de um Recurso de Binding de JCA

É possível gerar um serviço de proxy de JCA a partir de um recurso de Binding de JCA de entrada no Oracle Service Bus. Para obter mais informações sobre os recursos de Binding de JCA, consulte o Capítulo 12, Bindings de JCA.

Para gerar um serviço de proxy JCA a partir de um Binding de JCA:

  1. Na Console do Oracle Service Bus Administration, clique em Criar ou em Editar no Centro de Alterações, caso ainda não esteja no modo Criar ou Editar.
  2. No Browser do Recurso, clique em Bindings de JCA.
  3. Localize o Binding de JCA a partir do qual você deseja gerar um serviço e clique no ícone Ação.
  4. Na janela que aparece, confirme o nome do WSDL e do serviço que você deseja gerar, selecione o local para esses novos recursos e clique em Gerar.

    5. O Oracle Service Bus gera o serviço e seu WSDL correspondente.

  5. Modifique outros detalhes de configuração no serviço gerado, conforme apropriado, como o URI do Ponto Final.
