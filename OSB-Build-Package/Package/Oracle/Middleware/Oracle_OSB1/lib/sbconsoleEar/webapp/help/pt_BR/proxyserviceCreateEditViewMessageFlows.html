Criando, Editando e Exibindo Fluxos de Mensagens

  1. Se ainda não tiver feito isso, clique em Criar para criar uma nova sessão ou clique em Editar para informar uma sessão existente. Consulte Usando o Centro de Alterações.
  2. Localize o serviço de proxy cujo fluxo de mensagens você deseja exibir ou editar. Execute um dos seguintes procedimentos:
    • Selecione o Project Explorer para exibir a página View de Projetos ou a página View de Projeto/Pasta. Em seguida, navegue pelos projetos e pastas para localizar o serviço de proxy.
    • Selecione Browser do Recurso > Serviços de Proxy para exibir a página Resumo de Serviços de Proxy.
  3. Clique no ícone Editar Fluxo de Mensagens na linha do serviço de proxy cujo fluxo de mensagens você deseja exibir ou editar.
    • Se nenhum fluxo de mensagens tiver sido criado ainda para o serviço de proxy selecionado, a página Editar Fluxo de Mensagens será exibida com um único ícone na página: o ícone Serviço de Proxy. Esse é o nó inicial para o serviço de proxy. Clique nesse ícone para começar a construir o fluxo de mensagens.
    • Se o serviço de proxy já tiver um fluxo de mensagens, a página conterá uma representação gráfica do fluxo. Clique nos ícones para exibir ou editar as partes do fluxo de mensagens.

      • Consulte a página Editar Fluxo de Mensagens para obter informações sobre o que você pode fazer para construir um fluxo de mensagens nessa página.

  4. Clique em Salvar para confirmar as atualizações na sessão atual.
  5. Para encerrar a sessão e implantar a configuração no run-time, clique em Ativar em Centro de Alterações.
