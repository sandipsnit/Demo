Página Configuração do Transporte FTP

A página Criar/Editar um Serviço de Proxy - Configuração do Transporte FTP pertence a uma série de páginas para criação e edição de serviços de negócios, conforme descrito em Criando e Configurando Serviços de Negócios e Editando Configurações do Serviço de Negócios.

Use esta página para configurar as definições de transporte para um serviço de proxy usando o protocolo de transporte FTP. Tabela 19-6 descreve como usar esta página.

Tabela 19-6 - Página Criar/Editar um Serviço de Proxy - Configuração do Transporte FTP
Opção
Para criar ou editar...
Autenticação do Usuário
Selecione anônimo se o usuário do servidor FTP for anônimo ou selecione usuário externo se o usuário do servidor FTP for uma conta configurada externamente.
Identidade (ID do E-mail)
Este campo só está disponível se a opção Autenticação do Usuário estiver definida como anônimo.
Informe o ID de e-mail do usuário anônimo.
Conta do Serviço
Este campo só está disponível se a opção Autenticação do Usuário estiver definida como usuário externo.
Informe a conta de serviço do usuário. Este é um campo obrigatório quando a opção Autenticação do Usuário é definida como usuário externo.
Passar por Referência
Marque esta caixa de seleção para separar o arquivo no diretório de archive e transmiti-lo como uma referência nos cabeçalhos.
Fluxo Remoto
Marque esta caixa de seleção para fazer stream dos arquivos FTP diretamente do servidor remoto no momento do processamento. Quando você seleciona esta opção, o diretório de archive será o diretório remoto na máquina remota do servidor FTP. Portanto, você deve especificar o diretório de archive em relação ao diretório de usuário FTP.
Máscara do Arquivo
Informe a expressão regular para os arquivos a serem selecionados. O default é *.*.
Servidor Gerenciado
Este campo só é exibido em um domínio em cluster.
Selecione o Servidor Gerenciado para atuar como o servidor de sondagem. Todos os Servidores Gerenciados podem processar a mensagem, mas apenas um pode sondar a mensagem.
Intervalo de Sondagem
Informe um intervalo de sondagem, em segundos. O default é 60.
Limite de Leitura
Especifique o número máximo de mensagens a serem lidas por sweep de polling. Informe 0 para especificar nenhum limite. O default é 10.
Ação Pós-Leitura
Selecione o que acontece a uma mensagem depois que é lida.
  • Arquivar - A mensagem é arquivada.
  • Deletar - A mensagem é deletada.
Modo de Transferência
Selecione ASCII ou binário como o modo de transferência.
Diretório de Archive
Especifique o caminho para o local de archive se a opção Ação Pós-Leitura for definida como Archive. Esse campo é obrigatório se a opção Passar por Referência for selecionada.
Observação: Os diretórios de Archive, Download e Erro são caminhos absolutos e são criados automaticamente. Se você especificar um caminho relativo, os arquivos serão criados em relação ao processo Java que inicia o WebLogic Server.
Diretório de Download
Informe o diretório na máquina local para o qual os arquivos são submetidos a download durante a transferência de arquivos.
Observação: Os diretórios de Archive, Download e Erro são caminhos absolutos e são criados automaticamente. Se você especificar um caminho relativo, os arquivos serão criados em relação ao processo Java que inicia o WebLogic Server.
Diretório de Erros
Informe o local em que as mensagens são postadas se houver um problema.
Observação: Os diretórios de Archive, Download e Erro são caminhos absolutos e são criados automaticamente. Se você especificar um caminho relativo, os arquivos serão criados em relação ao processo Java que inicia o WebLogic Server.
Codificação da Solicitação
Aceite o default UTF-8 como a codificação de conjunto de caracteres para solicitações em transportes FTP.
Verificar Subdiretórios
Marque esta caixa de seleção para verificar recursivamente todos os diretórios
Classificar por Chegada
Marque esta caixa de seleção para enviar eventos na ordem de chegada.
Timeout
Informe o intervalo de timeout do soquete, em segundos, antes que a conexão seja eliminada. Se você informar 0, não haverá timeout.
Contagem de Repetições
Especifique o número de repetições para falhas de conexão FTP.

Depois de finalizar

Clique em Próximo para continuar configurando este serviço na próxima página; ou clique em Último para rever e salvar essa configuração na página Criar/Editar um Serviço de Proxy - Resumo.
