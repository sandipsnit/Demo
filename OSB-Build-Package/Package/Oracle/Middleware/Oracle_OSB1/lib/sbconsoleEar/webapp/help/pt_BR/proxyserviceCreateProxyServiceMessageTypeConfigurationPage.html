Página Configuração do Tipo de Mensagem

A página Criar/Editar um Serviço de Proxy - Configuração do Tipo de Mensagem pertence a uma série de páginas para criação e edição de serviços de proxy, conforme descrito em Criando e Configurando Serviços de Proxy e Editando Configurações do Serviço de Proxy.

Use esta página para configurar os tipos de mensagens para um serviço de proxy cujo tipo é Serviço de Mensagens.

A definição de binding para serviços de mensagens consiste em configurar o tipo de conteúdo das mensagens que são trocadas. O tipo de conteúdo para a resposta não precisa ser o mesmo para a solicitação; portanto, a resposta é configurada separadamente (por exemplo, o serviço pode aceitar uma mensagem MFL e retornar um recebimento de confirmação XML).

Observação: Serviços de proxy de transporte de E-mail, Arquivo, FTP ou SFTP cujo tipo seja Serviço de Mensagens suportam mensagens unilaterais apenas; o Tipo de Mensagem da Resposta deve ser nenhum. Se você selecionar uma opção diferente de nenhum, os protocolos de E-mail, Arquivo, FTP ou SFTP não estarão disponíveis na página Configuração do Transporte.

Tabela 19-2 descreve como usar a página Criar/Editar um Serviço de Proxy - Configuração do Tipo de Mensagem.

Tabela 19-2 - Página Criar/Editar um Serviço de Proxy - Configuração do Tipo de Mensagem
Opção
Para criar ou editar...
Tipo de Mensagem de Solicitação
Selecione um tipo de mensagem para a mensagem de solicitação:
  • Nenhum - Selecione esta opção se não houver mensagem de solicitação.
  • Binário - Selecione esta opção se o tipo de conteúdo da mensagem for desconhecido ou não for importante.
  • Texto - Selecione esta opção se a mensagem puder ser restrita a texto.
  • MFL - Selecione esta opção se a mensagem for um documento binário de acordo com uma definição MFL. Informe o nome do arquivo MFL (qualificado por seu caminho) ou clique em Procurar para selecionar um arquivo.

    • É possível configurar apenas um arquivo MFL.

    Observação: para suportar vários arquivos MFL, defina o conteúdo como binário ou texto e use a ação MFL no fluxo de mensagens para converter em XML.

  • XML - Selecione esta opção se a mensagem for um documento XML. Informe o nome do arquivo XML (qualificado por seu caminho) ou clique em Procurar para selecionar um arquivo.

    • Como opção, forneça algumas informações do tipo declarando (no campo elemento ou tipo) o tipo de esquema XML do documento XML trocado.

  • Java - Selecione esta opção se o Objeto Java estiver sendo enviado na solicitação. O transporte de JMS é usado para mensagens do Objeto Java. Para obter mais informações, consulte "Sending and Receiving Java Objects in Messages" no Oracle Fusion Middleware Developer's Guide for Oracle Service Bus em http://www.oracle.com/pls/as111170/lookup?id=OSBDV1874.
Tipo de Mensagem de Resposta
Selecione um tipo de mensagem para a mensagem de resposta:
  • Nenhum - Selecione esta opção se não houver mensagem de resposta.
  • Binário - Selecione esta opção se o tipo de conteúdo da mensagem for desconhecido ou não for importante.
  • Texto - Selecione esta opção se a mensagem puder ser restrita a texto.
  • MFL - Selecione esta opção se a mensagem for um documento binário de acordo com uma definição MFL. Informe o nome do arquivo MFL (qualificado por seu caminho) ou clique em Procurar para selecionar um arquivo.

    • É possível configurar apenas um arquivo MFL.

    Observação: para suportar vários arquivos MFL, defina o conteúdo como binário ou texto e use a ação MFL no fluxo de mensagens para converter em XML.

  • XML - Selecione esta opção se a mensagem for um documento XML. Informe o nome do arquivo XML (qualificado por seu caminho) ou clique em Procurar para selecionar um arquivo.

    • Como opção, forneça algumas informações do tipo declarando (no campo elemento ou tipo) o tipo de esquema XML do documento XML trocado.

  • Java - Selecione esta opção se um Objeto Java estiver sendo recebido na resposta. O transporte de JMS é usado para mensagens do Objeto Java. Para obter mais informações, consulte "Sending and Receiving Java Objects in Messages" no Oracle Fusion Middleware Developer's Guide for Oracle Service Bus em http://www.oracle.com/pls/as111170/lookup?id=OSBDV1874.
