Página Configuração do Transporte SB

A página Criar/Editar um Serviço de Proxy - Configuração do Transporte SB pertence a uma série de páginas para criação e edição de serviços de proxy, conforme descrito em Criando e Configurando Serviços de Proxy e Editando Configurações do Serviço de Proxy.

Para obter mais informações, consulte "SB Transport" no Oracle Fusion Middleware Developer's Guide for Oracle Service Bus em http://www.oracle.com/pls/as111170/lookup?id=OSBDV997.

Use esta página para configurar as definições de proxy para um serviço de negócios usando o protocolo de transporte do SB (Service Bus). Tabela 19-12 descreve como usar a página.

Tabela 19-12 - Página Criar/Editar um Serviço de Proxy - Configuração do Transporte SB
Opção
Para criar ou editar...
Política de Distribuição
Selecione uma política de despacho para esse ponto final ou use a política de despacho default.
Política de despacho refere-se à instância do Gerenciador de Trabalho do WLS que você deseja usar para o ponto final do serviço processar a solicitação. Para obter informações sobre Gerenciadores de Trabalho, consulte "Using Work Managers to Optimize Scheduled Work" no Oracle Fusion Middleware Configuring Server Environments for Oracle WebLogic Server em http://www.oracle.com/pls/as111170/lookup?id=CNFGD112.
Usar SSL
Quando especificado, as solicitações devem ser enviadas sobre uma conexão SSL.
No entanto, conexões não seguras não são proibidas. O administrador deve fechar todos os protocolos não seguros no servidor (por exemplo, t3 ou http) para impor estritamente conexões de cliente seguras.

Depois de finalizar

Clique em Próximo para continuar configurando este serviço na próxima página; ou clique em Último para rever e salvar essa configuração na página Criar/Editar um Serviço de Proxy - Resumo.
