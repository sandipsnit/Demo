Página Resumo

A página Criar/Editar um Serviço de Proxy - Resumo pertence a uma série de páginas para criação e edição de serviços de proxy, conforme descrito em Criando e Configurando Serviços de Proxy e Editando Configurações do Serviço de Proxy.

Use esta página para exibir ou modificar as definições de configuração para um serviço de proxy antes de salvá-lo.

Para exibir ou modificar definições, clique em Editar na linha da categoria apropriada (por exemplo, Configuração Geral, Configuração de Transporte, etc.). As páginas que podem ser editadas dependem das páginas configuradas durante a criação do serviço de proxy. A lista a seguir mostra todas as páginas:

  • Página Configuração Geral
  • Página Configuração do Tipo de Mensagem
  • Página Configuração do Transporte
  • Páginas Configuração do Transporte Específico ao Protocolo
  • Página Configuração do Transporte de E-mail
  • Página Configuração do Transporte de Arquivos
  • Página Configuração do Transporte FTP
  • Página Configuração do Transporte HTTP
  • Página Configuração do Transporte JCA
  • Página Configuração do Transporte JEJB
  • Página Configuração do Transporte JMS
  • Página Configuração do Transporte MQ
  • Página Configuração do Transporte SB
  • Página Configuração do Transporte SFTP
  • Página Configuração do Transporte de Tuxedo
  • Página Configuração do Transporte WS
  • Página Configuração da Seleção de Operação
  • Página Tratamento de Mensagens
