Página Configuração do Transporte WS

A página Criar/Editar um Serviço de Proxy - Configuração do Transporte WS pertence a uma série de páginas para criação e edição de serviços de proxy, conforme descrito em Criando e Configurando Serviços de Proxy e Editando Configurações do Serviço de Proxy.

Para obter mais informações, consulte "WS Transport" no Oracle Fusion Middleware Developer's Guide for Oracle Service Bus em http://www.oracle.com/pls/as111170/lookup?id=OSBDV1083.

Use esta página para configurar as definições de transporte para um serviço de proxy usando o protocolo de transporte WS. Tabela 19-15 descreve como usar a página.

Tabela 19-15 - Página Criar/Editar um Serviço de Proxy - Configuração do Transporte WS
Opção
Para criar ou editar...
Política de Distribuição
Selecione uma política de despacho para esse ponto final ou use a política de despacho default.
Política de despacho refere-se à instância do Gerenciador de Trabalho do WLS que você deseja usar para o ponto final do serviço. Para obter informações sobre Gerenciadores de Trabalho, consulte "Using Work Managers to Optimize Scheduled Work" no Oracle Fusion Middleware Configuring Server Environments for Oracle WebLogic Server em http://www.oracle.com/pls/as111170/lookup?id=CNFGD112.
Contagem de Repetições
O número de vezes para repetir a entrega de uma mensagem para o pipeline.
Se uma exceção não-tratada ocorrer no pipeline de solicitação de um serviço de proxy, a mensagem do transporte WS de entrada será reentregue ao pipeline, limitado ao número de vezes especificado pela contagem de repetições. Esse valor é importante para o processamento confiável de mensagens de transporte WS.
Retardo de Repetição
O número de segundos que o sistema pausa antes de tentar novamente enviar uma mensagem para o pipeline após um erro.

Depois de finalizar

Clique em Próximo para continuar configurando este serviço na próxima página; ou clique em Último para rever e salvar essa configuração na página Criar/Editar um Serviço de Proxy - Resumo.
