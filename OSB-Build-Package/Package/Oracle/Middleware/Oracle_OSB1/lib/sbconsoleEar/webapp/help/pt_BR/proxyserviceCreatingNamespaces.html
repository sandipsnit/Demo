Criando Namespaces a Serem Usados em Expressões em Linha

A paleta Definições de Namespace inclui uma lista de namespaces default, mas você também pode definir novos.

Antes de começar

Estas instruções consideram que você está criando ou editando uma expressão no Editor de Expressão XQuery, Editor de Condição XQuery ou no Editor de Expressão XPath, conforme descrito em Criando e Editando Expressões XQuery e XPath em Linha.

Para criar e usar um namespace em uma expressão em linha

  1. Selecione Definições de Namespace. A paleta Definições de Namespace inclui uma lista de namespaces default, além de listas de namespaces variáveis e namespaces definidos pelo usuário, se existir.
  2. Para definir e adicionar um namespace do usuário,
    1. Clique em Adicionar Namespace.
    2. No campo Prefixo, informe um identificador exclusivo para o namespace. Não é possível usar o mesmo prefixo mais de uma vez.
    3. No campo URI, informe um URL para esse namespace no formato http://url/.../ ou informe um URN no formato uddi:server:.
    4. Clique em Adicionar para adicionar o namespace à lista Namespaces Definidos pelo Usuário.
    5. Copie e cole o namespace definido pelo usuário na expressão XQuery, condição XQuery ou XPath, conforme descrito em Criando e Editando Expressões XQuery e XPath em Linha.

Depois de finalizar

Continue conforme descrito em Criando e Editando Expressões XQuery e XPath em Linha.
