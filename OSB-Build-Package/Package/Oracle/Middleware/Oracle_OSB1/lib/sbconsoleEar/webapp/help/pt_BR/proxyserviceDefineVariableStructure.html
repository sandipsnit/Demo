Criando Estruturas Variáveis nos Editores XQuery

A paleta Estruturas Variáveis nos editores XQuery e XPath exibe representações gráficas do conteúdo de variáveis. Por default, inclui as variáveis de contexto da mensagem integradas anexos, corpo, cabeçalho, saída e entrada.

Cada entrada do mapeamento de estrutura variável possui um label e mapeia uma variável ou um caminho de variável para uma ou mais estruturas. O escopo desses mapeamentos é um estágio ou um nó de rota.

Também é possível declarar suas próprias estruturas variáveis, com base em:

  • Tipos XML, incluindo:
    • Elementos do esquema
    • Elementos de WSDL
    • Tipos de esquema
    • Tipos de WSDL
    • MFLs
  • Interfaces de serviço
  • Tipos simples (string ou qualquer XML)

É possível usar este recurso diretamente para todas as variáveis definidas pelo usuário, assim como $inbound, $outbound e $fault. No entanto, não é possível usá-lo diretamente para acessar anexos XML em $attachments, cabeçalhos em $header ou documentos e parâmetros RPC em $body, com uma exceção: você pode usá-lo diretamente para acessar documentos e parâmetros em $body para mensagens de solicitação recebidas por um serviço de proxy WSDL.

Antes de começar

Estas instruções consideram que você está criando ou editando uma expressão no Editor de Expressão XQuery, Editor de Condição XQuery ou no Editor de Expressão XPath, conforme descrito em Criando e Editando Expressões XQuery e XPath em Linha.

Para criar uma estrutura variável:

  1. Selecione Estruturas Variáveis.
  2. Na paleta Estruturas Variáveis, clique em Adicionar Nova Estrutura.
  3. Continue com qualquer uma das tarefas listadas na Tabela 22-3 a seguir.

    4. Tabela 22-3 - Criar uma Nova Estrutura Variável
    Para...
    Execute estas etapas...
    Criar uma estrutura variável que mapeia uma variável para um tipo de Esquema XML
    1. Selecione Tipo XML na parte superior da paleta Estruturas Variáveis, caso ainda não esteja selecionado.
    2. No campo Label da Estrutura, informe um nome de exibição para a variável que você deseja criar. Esse nome de exibição permite que você dê um nome significativo à estrutura para poder reconhecê-la no tempo de design, mas que não tenha impacto no run-time.
    3. No campo Caminho da Estrutura, informe o caminho da estrutura variável no run-time. O caminho deve começar com $.
    4. No campo Tipo, selecione o botão de rádio apropriado e, em seguida, selecione Tipo de Esquema.
    5. Clique em Procurar. O Browser de Esquema XML será exibido. Selecione um Esquema XML na lista, selecione um tipo de Esquema XML no painel Definições e, em seguida, clique em Submeter.
    6. Clique em Adicionar para criar a estrutura variável.
    Criar uma estrutura variável que mapeia uma variável para um tipo WSDL
    1. Selecione Tipo XML na parte superior da paleta Estruturas Variáveis, caso ainda não esteja selecionado.
    2. No campo Label da Estrutura, informe um nome de exibição para a variável que você deseja criar. Esse nome de exibição permite que você dê um nome significativo à estrutura para poder reconhecê-la no tempo de design, mas que não tenha impacto no run-time.
    3. No campo Caminho da Estrutura, informe o caminho da estrutura variável no run-time.

    5. Clique em Procurar. O Browser do WSDL é exibido. Selecione um WSDL na lista de WSDLs, selecione um tipo de WSDL no painel Definições e, em seguida, clique em Submeter.
    6. Clique em Adicionar para criar a estrutura variável.
    Criar uma estrutura variável que mapeia uma variável para um elemento de Esquema XML
    1. Na parte superior da paleta Estruturas Variáveis, selecione Tipo XML, se ainda não estiver selecionada.
    2. No campo Label da Estrutura, informe um nome de exibição para a variável que você deseja criar. Esse nome de exibição permite que você dê um nome significativo à estrutura para poder reconhecê-la no tempo de design, mas que não tenha impacto no run-time.
    3. No campo Caminho da Estrutura, informe o caminho da estrutura variável no run-time.
    4. No campo Tipo, selecione o botão de rádio apropriado e, em seguida, selecione Elemento do Esquema.
    5. Clique em Procurar. O Browser de Esquema XML será exibido. Selecione um Esquema XML na lista, selecione um tipo de Esquema XML no painel Definições e, em seguida, clique em Submeter.
    6. Clique em Adicionar para criar a estrutura variável.
    Criar uma estrutura variável que mapeia uma variável para um elemento WSDL
    1. Na parte superior da paleta Estruturas Variáveis, selecione Tipo XML, se ainda não estiver selecionada.
    2. No campo Label da Estrutura, informe um nome de exibição para a variável que você deseja criar. Esse nome de exibição permite que você dê um nome significativo à estrutura para poder reconhecê-la no tempo de design, mas que não tenha impacto no run-time.
    3. No campo Caminho da Estrutura, informe o caminho da estrutura variável no run-time.

    5. Clique em Procurar. O Browser do WSDL é exibido. Selecione um WSDL na lista de WSDLs, selecione um elemento de WSDL no painel Definições e, em seguida, clique em Submeter.
    6. Clique em Adicionar para criar a estrutura variável.
    Criar uma estrutura variável que mapeia uma variável para um elemento-filho
    1. Na parte superior da paleta Estruturas Variáveis, selecione Tipo XML, se ainda não estiver selecionada.
    2. No campo Label da Estrutura, informe um nome de exibição para a variável que você deseja criar. Esse nome de exibição permite que você dê um nome significativo à estrutura para poder reconhecê-la no tempo de design, mas que não tenha impacto no run-time.
    3. No campo Caminho da Estrutura, informe o caminho da estrutura variável no run-time.
    4. No campo Tipo, selecione o tipo de variável que você deseja criar:

      5. Para criar uma variável de Elemento do Esquema XML ou de Elemento WSDL, selecione o botão de rádio associado a esta opção, depois selecione Elemento do Esquema ou Elemento do WSDL.

      Para criar uma variável MFL, selecione o botão de rádio associado a esta opção, depois selecione MFL.

    5. Para o Esquema XML, WSDL ou MFL, clique em Procurar para selecionar um objeto na lista exibida pelo browser, depois clique em Submeter. Por exemplo, selecione um MFL na lista de MFLs, depois clique em Submeter.
    6. Marque a caixa de seleção Definir como filho para definir o elemento como um filho da estrutura sendo criada.
    7. Clique em Adicionar para criar a estrutura variável.
    Criar uma estrutura variável que use um recurso MFL
    1. Na parte superior da paleta Estruturas Variáveis, selecione Tipo XML, se ainda não estiver selecionada.
    2. No campo Label da Estrutura, informe um nome de exibição para a variável que você deseja criar. Esse nome de exibição permite que você dê um nome significativo à estrutura para poder reconhecê-la no tempo de design, mas que não tenha impacto no run-time.
    3. No campo Caminho da Estrutura, informe o caminho da estrutura variável no run-time.
    4. No campo Tipo, selecione o botão de rádio apropriado e, em seguida, clique em Procurar. O Browser MFL será exibido.
    5. Selecione um MFL na lista de MFLs e, em seguida, clique em Submeter.
    6. Clique em Adicionar para criar a estrutura variável.
    Criar uma estrutura variável de Interface de Serviço
    1. Na parte superior da paleta Estruturas Variáveis, selecione Interface de Serviço.
    2. No campo Label da Estrutura, informe um nome de exibição para a variável que você deseja criar. Esse nome de exibição permite que você dê um nome significativo à estrutura para poder reconhecê-la no tempo de design, mas que não tenha impacto no run-time.
    3. No campo Caminho da Estrutura, o default já estará definido como $body. Não é possível alterar este campo.
    4. No campo Serviço Baseado no WSDL, selecione o ícone Browser do Serviço, selecione um serviço na lista de serviços exibidos pelo Browser do Serviço, depois clique em Submeter.

      5. O serviço selecionado será exibido no campo Serviço Baseado no WSDL.

    5. No campo Operação, selecione uma operação ou selecione Nenhum para não incluir uma operação.
    6. Clique em Adicionar para criar a variável.
    Criar uma estrutura variável Simples
    1. Na parte superior da paleta Estruturas Variáveis, selecione Tipo Simples.
    2. No campo Label da Estrutura, informe um nome de exibição para a variável que você deseja criar. Esse nome de exibição permite que você dê um nome significativo à estrutura para poder reconhecê-la no tempo de design, mas que não tenha impacto no run-time.
    3. No campo Nome da Estrutura, informe um nome para a estrutura variável que deseja criar.
    4. No campo Tipo, selecione String ou Qualquer XML.
    5. Clique em Adicionar para criar a variável.
