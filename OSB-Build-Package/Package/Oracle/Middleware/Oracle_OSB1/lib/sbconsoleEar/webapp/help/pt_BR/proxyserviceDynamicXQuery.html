Vinculando Expressões XQuery Dinâmicas a XQueries em Linha

A página Editor de Expressão XQuery permite especificar uma expressão XQuery dinâmica que seja avaliada no run-time para o nome de um recurso XQuery pré-registrado. Para saber mais sobre este editor, consulte Criando e Editando Expressões XQuery e XPath em Linha.

Para definir uma expressão XQuery dinâmico

  1. Selecione a opção XQuery Dinâmico.
  2. Na área Informar Expressão para Recurso XQuery, informe a expressão que será avaliada no run-time com o nome de um recurso XQuery pré-registrado.

    3. A seguir, está a sintaxe do recurso XQuery (representando o nome completo do recurso):

    Project/folder1/folder2/XQueryResourceName

  3. Na área Variáveis de Bind usando Modelo XQuery de Variáveis Personalizadas, defina os parâmetros de entrada para a transformação.
    • Clique em Procurar no campo Selecionar Modelo XQuery para selecionar um recurso registrado existente que sirva como modelo para o formato da consulta (o número e nomes das variáveis). Após a seleção de um modelo, as variáveis aparecem na área Variáveis de Bind. Observe que o modelo não é persistido na configuração. Em vez disso, o modelo serve como um início rápido para ajudá-lo a especificar as variáveis para a consulta.
    • Digite um nome de variável no campo Adicionar Variável Personalizada, depois clique em Adicionar. Para cada variável listada em Nome da Variável, informe uma expressão XQuery a ser mapeada para ele. Você deve definir um mapeamento para cada parâmetro. Por exemplo, se uma transformação XQuery tiver dois parâmetros de entrada nomeados um e dois, o campo Nome da Variável terá dois labels: um e dois. Uma caixa de texto, em que a expressão XQuery é informada, é associada a cada label.

      • As seguintes expressões XQuery são exemplos de entrada válida para os campos variáveis:

      $body/*[1]

      $body/po:PurchaseOrder

  4. Depois de finalizar, continue com qualquer uma das tarefas a seguir.
    • Clique em Validar. Uma mensagem será exibida se a expressão for validada com sucesso.
    • Clique em Teste. Consulte Executando Teste XQuery.
    • Salve ou descarte suas alterações.
