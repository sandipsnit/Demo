Página Editar Fluxo de Mensagens

Use a página Editar Fluxo de Mensagens para construir um fluxo de mensagens para um serviço de proxy.

O painel de navegação esquerdo da página Editar Fluxo de Mensagens mostra uma exibição em árvore dos nós e objetos no fluxo de mensagens. Quando os detalhes de um objeto são definidos em uma página separada, é possível clicar no nome do objeto para exibir a página associada.

O painel direito apresenta um campo em que é possível construir o fluxo de mensagens. Quando um fluxo de mensagens ainda não está definido, o painel inclui um único ícone Serviço de Proxy que significa o nó inicial do serviço. Clique no ícone para adicionar nós do par de pipelines, nós de rota, ramificações condicionais, ramificações operacionais e tratamento de erro para o serviço.

Quando você adiciona objetos à página, ícones são exibidos na página para representar os objetos. Os relacionamentos entre os objetos são mostrados com linhas e caixas delimitadoras. Clique em um ícone na página Editar Fluxo de Mensagens para exibir um menu das ações que podem ser executadas nesse objeto. As opções disponíveis no menu variam de acordo com o contexto.
