Página Editar Configuração do Estágio

Use a página Editar Configuração do Estágio para adicionar ações aos estágios do pipeline, estágios do handler de erro e nós de rota em um fluxo de mensagens.

  • Quando nada ainda tiver sido definido na página Editar Configuração do Estágio, o único objeto exibido é o ícone Adicionar uma Ação. Clique nesse ícone para iniciar.
  • Quando um estágio ou nó de rota já tiver sido configurado, as ações e objetos definidos para esse estágio ou nó de rota aparecerão na página. Edite as ações existentes, conforme apropriado, ou clique em qualquer um dos ícones que representa ações, para adicionar mais ações ao estágio.

Consulte Adicionando e Editando Ações em Fluxos de Mensagens para obter instruções sobre como trabalhar com todos os tipos de ações que você pode adicionar a um estágio.
