Informando Expressões de Comparação XQuery Usando a Opção do Builder

Antes de começar

Estas instruções consideram que você está criando ou editando uma expressão condicional XQuery no Editor de Condição XQuery, conforme descrito em Criando e Editando Expressões XQuery e XPath em Linha.

Para informar uma expressão de comparação XQuery:

  1. No Editor de Condição XQuery, selecione Builder (localizado abaixo da barra de botões do espaço de trabalho), caso ainda não tenha sido selecionado.
  2. Na caixa Builder de Expressões, selecione Expressão de Comparação caso ainda não esteja selecionado.
  3. No campo Operando, informe uma variável de contexto, uma definição do namespace ou uma função XQuery.

    4. Para construir o operando, é possível colar funções XQuery da paleta Funções XQuery e namespaces da paleta Definições de Namespace. Consulte a etapa 3 e a etapa 4 em Expressões de Construção nos Campos de Texto do Espaço de Trabalho do Editor.

  4. Na lista Operador, selecione um operador de comparação.
  5. No campo Valor, informe texto ou uma variável de contexto.

    6. Você deve inserir texto entre aspas, por exemplo, "true" é válido; true não é.

  6. Clique em Adicionar. O texto informado é exibido no painel Expressões.
  7. Repita as etapas 3-6 para construir condições adicionais. Cada condição é adicionada ao final da lista de condições.

    8. Considere o seguinte ao usar várias condições:

    • Ao construir expressões adicionais, certifique-se de selecionar as opções E ou Ou no campo Conjunção.
    • É possível selecionar uma condição e clicar na seta Para Cima para movê-la para cima na lista de condições ou clicar na seta Para Baixo para movê-la para baixo na lista de condições. Também é possível clicar no ícone Editar para atualizar uma condição ou clicar no ícone Deletar para deletá-la.
    • Expressões unárias podem ser mescladas com expressões de Comparação na definição geral de uma condição.
  8. Como opção, execute um ou ambos os procedimentos a seguir:
    • Clique em Validar. Uma mensagem será exibida se a expressão for validada com sucesso.
    • Clique em Teste para testar a expressão. Consulte Executando Teste XQuery.
  9. Clique em Salvar para fechar o editor e insira a expressão na ação.

Depois de finalizar

Continue configurando a ação, conforme descrito em Adicionando e Editando Ações em Fluxos de Mensagens.
