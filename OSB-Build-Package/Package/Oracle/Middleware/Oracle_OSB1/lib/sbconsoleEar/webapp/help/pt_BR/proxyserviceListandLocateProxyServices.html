Localizando Serviços de Proxy

  1. Execute um dos seguintes procedimentos:
    • Selecione o Project Explorer para exibir a página View de Projetos ou a página View de Projeto/Pasta. Em seguida, navegue pelos projetos e pastas para localizar o serviço de proxy.
    • Selecione Resource Browser > Serviços de Proxy. A página Resumo de Serviços de Proxy exibe as informações mostradas na Tabela 19-18. Para obter uma descrição mais detalhada das propriedades, consulte Editando Configurações de Serviço de Proxy.
  2. Para restringir o número de itens na lista, você pode filtrar por nome, caminho ou ambos. Nos campos Nome e Caminho, em Pesquisar, informe o nome e/ou o caminho do(s) destino(s) e, em seguida, clique no botão Pesquisar.

    3. O caminho é o nome do projeto e o nome da pasta em que o serviço de proxy reside.

    Caracteres curingas * e ? são permitidos. A pesquisa faz distinção entre maiúsculas e minúsculas.

    Clique em Exibir Tudo para exibir todos os serviços de proxy no domínio. Isso limpa os parâmetros de pesquisa da pesquisa anterior.

    Tabela 19-18 Página Resumo de Serviços de Proxy 
    Propriedade
    Descrição
    Nome
    Um nome exclusivo para o serviço de proxy. Clique no nome para exibir a página Exibir um Serviço de Proxy - Detalhes da Configuração.
    Consulte Editando Configurações de Serviço de Proxy.
    Caminho
    O caminho é o nome do projeto e o nome da pasta em que reside o serviço de proxy, por exemplo, UDDI/Proxies/OSB_services.
    Clique no caminho de um serviço de proxy para exibir a página View de Projeto/Pasta que o contém.
    Ações
    Execute uma das etapas a seguir:
    • Clique no ícone Editar Fluxo de Mensagens para editar o fluxo de mensagens do serviço de proxy. Consulte Criando, Editando e Exibindo Fluxos de Mensagens.
    • Clique no ícone Acionar Console de Teste para chamar a Console de Teste, que você usa para validar e testar o design de seus serviços e transformações. Para serviços de negócios, você só pode usar a Console de Teste no run-time, ou seja, quando a sessão for ativada. Para transformações, é possível usar a Console de Teste dentro ou fora de uma sessão. Consulte Testando Serviços e Testando Transformações.
    • O ícone Exportar WSDL é exibido para serviços de negócios baseados no WSDL. Clique nesse ícone para exportar um WSDL, que você pode exibir ou modificar em uma ferramenta externa, como um IDE.

      • Observe que isso é diferente da funcionalidade Exportar Recursos no módulo Administração do Sistema, que você usa para mover e separar recursos entre dois domínios. Consulte Exportando um WSDL.

    Opções
    Clique no ícone Deletar para deletar o serviço. Um ícone Advertência de Deleção é exibido quando outros recursos referenciam esse recurso. É possível deletar o recurso com uma confirmação de advertência. Isso pode resultar em conflitos devido a referências não resolvidas referentes ao recurso deletado. Para obter mais informações, consulte Deletando Serviços de Proxy.
