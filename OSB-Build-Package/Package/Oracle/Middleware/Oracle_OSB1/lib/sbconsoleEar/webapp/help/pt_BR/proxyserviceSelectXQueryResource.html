Vinculando Recursos XQuery Externos a XQueries em Linha

É possível efetuar binding de recursos XQuery para expressões XQuery em linha, para que sejam executados em linha como parte de uma ação.

Antes de começar

Estas instruções consideram que você está criando ou editando uma expressão no Editor de Expressão XQuery, Editor de Condição XQuery ou no Editor de Expressão XPath, conforme descrito em Criando e Editando Expressões XQuery e XPath em Linha.

Para efetuar binding de um Recurso XQuery a uma expressão em linha:

  1. Clique em Estruturas Variáveis.
  2. No espaço de trabalho (abaixo da barra de botões), selecione Recursos XQuery.
  3. Na caixa 1. Selecione um recurso XQuery para executar, clique em Procurar.
  4. No Browser XQuery, selecione o botão de rádio associado ao XQuery que você deseja usar, depois clique em Submeter.
  5. Na caixa 2. Variáveis de Bind, defina os parâmetros de entrada para a transformação. Para cada variável listada em Nome da Variável, informe uma expressão XQuery a ser mapeada para ele. Você deve definir um mapeamento para cada parâmetro. Por exemplo, se uma transformação XQuery tiver dois parâmetros de entrada nomeados um e dois, o campo Nome da Variável terá dois labels: um e dois. Uma caixa de texto, em que a expressão XQuery é informada, é associada a cada label.

    6. As seguintes expressões XQuery são exemplos de entrada válida para este campo:

    $body/*[1]

    $body/po:PurchaseOrder

    Observação: O seguinte nome variável não é uma entrada válida para este campo e resulta em uma exceção:
    body
  6. Depois de finalizar, continue com qualquer uma das tarefas a seguir.
    • Clique em Validar. Uma mensagem será exibida se a expressão for validada com sucesso.
    • Clique em Teste. Consulte Executando Teste XQuery.
    • Salve ou descarte suas alterações.
