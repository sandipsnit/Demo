Vinculando Recursos XSLT Externos a XQueries em Linha

A página Editor de Expressão XQuery permite selecionar um recurso XSLT para execução. Para saber mais sobre este editor, consulte Criando e Editando Expressões XQuery e XPath em Linha.

Para Selecionar um Recurso XSLT para Execução

  1. Selecione a opção Recursos XSLT.
  2. No campo Selecionar o recurso XSLT para execução, selecione o ícone Browser XSLT.
  3. No Browser XSLT, selecione o botão de rádio associado ao XSLT que você deseja executar, depois clique em Submeter.
  4. No campo Variáveis de Bind, um label e uma caixa de texto correspondente são exibidos para cada parâmetro de entrada da transformação. Cada label corresponde ao nome de um parâmetro e cada caixa de texto serve para definir uma expressão XQuery a ser mapeada para o parâmetro. Você deve definir um mapeamento para cada parâmetro. Por exemplo, se uma transformação XSL tiver dois parâmetros de entrada nomeados um e dois, o campo Mapeamento da Variável terá dois labels - um e dois - com uma caixa de texto associada a cada qual em que a expressão XQuery é informada. Além do mapeamento para qualquer variável de entrada, você também deve especificar uma expressão XQuery para o Documento de Entrada para a transformação. O mapeamento é especificado na caixa de texto com o label Documento de Entrada.

    5. As seguintes expressões XQuery são exemplos de entrada válida para este campo:

    $body/*[1]

    $body/po:PurchaseOrder

    Observação: O seguinte nome variável não é uma entrada válida para este campo e resulta em uma exceção:
    body
  5. Continue com qualquer uma das seguintes tarefas:
    • Criando Namespaces a Serem Usados em Expressões em Linha.
    • Criando Estruturas Variáveis nos Editores XQuery.
    • Expressões de Construção nos Campos de Texto do Espaço de Trabalho do Editor.
    • Vinculando Recursos XQuery Externos a XQueries em Linha.
    • Clique em Validar. Uma mensagem será exibida se a expressão for validada com sucesso.
    • Clique em Teste. Consulte Executando Teste XQuery.
