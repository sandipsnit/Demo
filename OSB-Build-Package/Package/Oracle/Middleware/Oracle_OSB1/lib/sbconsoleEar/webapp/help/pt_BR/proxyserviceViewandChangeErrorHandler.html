Editando Handlers de Erro

Antes de começar

Estas instruções consideram que você já esteja editando um fluxo de mensagens na página Editar Fluxo de Mensagens, conforme explicado em Criando, Editando e Exibindo Fluxos de Mensagens.

Para exibir e alterar um Handler de Erro

  1. Execute um dos seguintes procedimentos.

    2. Tabela 23-1 Exibindo e Alterando o Handler de Erro 
    Para...
    Execute Esta Etapa...
    Exibir e alterar o handler de erro do fluxo de mensagens do serviço de proxy
    Clique no ícone Serviço de Proxy e, em seguida, em Adicionar Handler de Erro do Serviço. A página Editar Handler de Erro será exibida. Consulte Adicionando Handlers de Erro do Serviço de Proxy.
    Exibir e alterar o handler de erro do pipeline
    Clique no ícone Pipeline de Solicitação apropriado ou no ícone Pipeline de Resposta, depois clique em Editar Handler de Erro do Pipeline. A página Editar Handler de Erro será exibida. Consulte Adicionando Handlers de Erro do Pipeline.
    Exibir e alterar o handler de erro do nó de rota
    Clique no ícone Nó de Rota apropriado, depois clique em Editar Handler de Erro da Rota. A página Editar Handler de Erro será exibida. Consulte Adicionando Handlers de Erro do Nó de Rota.
    Exibir e alterar o handler de erro do estágio
    Clique no ícone Estágio apropriado e, em seguida, clique em Editar Handler de Erro do Estágio. A página Editar Handler de Erro será exibida. Consulte Adicionando Handlers de Erro do Estágio.

Depois de finalizar

Na página Editar Fluxo de Mensagens, continue para construir o fluxo de mensagens, conforme descrito em Criando, Editando e Exibindo Fluxos de Mensagens.
