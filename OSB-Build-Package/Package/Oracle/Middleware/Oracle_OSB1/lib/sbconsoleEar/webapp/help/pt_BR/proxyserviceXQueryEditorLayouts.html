Noções Básicas de Layouts e Tarefas do Editor de XQuery

O Editor de Expressão XQuery, o Editor de Expressão XQuery e o Editor de Expressão XPath são compostos pelos seguintes componentes:

  • Paletas
  • Espaço de Trabalho
  • Inspetor de Propriedades

Paletas

O painel esquerdo de cada editor contém as paletas listadas a seguir. Em qualquer dos editores, clique no nome de uma paleta para exibi-la. Cada paleta contém entidades que você pode inserir em expressões nos editores.

  • A paleta Definições de Namespace lista namespaces default do Oracle Service Bus, namespaces variáveis e namespaces definidos pelo usuário. É possível definir novos namespaces, que são adicionados à lista de namespaces definidos pelo usuário.

    • Para obter mais informações, consulte:

    • Expressões de Construção nos Campos de Texto do Espaço de Trabalho do Editor
    • Criando Namespaces a Serem Usados em Expressões em Linha
  • A paleta Funções XQuery lista um conjunto de funções XQuery padrão. Quando você insere uma função em uma expressão, marcadores são usados para os valores de parâmetro que você deve fornecer.

    • Para obter mais informações, consulte:

    • Expressões de Construção nos Campos de Texto do Espaço de Trabalho do Editor
    • Criando Funções XPath Personalizadas nos Editores XQuery
    • Vinculando Recursos XQuery Externos a XQueries em Linha
  • A paleta Estruturas Variáveis oferece um conjunto de ferramentas para inserir variáveis e caminhos para as variáveis, usando expressões XPath.

    • Estruturas variáveis são representações gráficas de variáveis ou caminhos variáveis exibidos no editor. Podem ajudar a visualizar a estrutura variável e podem ser usadas para construir expressões XQuery em linha que referenciam o conteúdo da variável.

    Observação: Estruturas variáveis não criam variáveis. As variáveis são criadas no run-time como destino da ação Atribuir no estágio.

    O Oracle Service Bus oferece diversas variáveis de contexto da mensagem predefinidas (anexos, corpo, cabeçalho, saída e entrada), cujo conteúdo você pode exibir como estruturas variáveis. Também é possível definir suas próprias estruturas variáveis.

    Para obter mais informações, consulte

    • Expressões de Construção nos Campos de Texto do Espaço de Trabalho do Editor
    • Criando Estruturas Variáveis nos Editores XQuery

Espaço de Trabalho

O lado direito da página fornece um espaço de trabalho para construir a expressão XQuery, a condição XQuery ou o XPath. O espaço de trabalho é diferente nos três editores.

Inspetor de Propriedades

Em todos os três editores, o Inspetor de Propriedades é exibido no lado inferior direito da página. Quando você seleciona um item em uma das paletas a serem adicionadas à expressão, esse item aparece no Inspetor de Propriedades. Você pode, então, colar o item no espaço de trabalho. Consulte Expressões de Construção nos Campos de Texto do Espaço de Trabalho do Editor.
