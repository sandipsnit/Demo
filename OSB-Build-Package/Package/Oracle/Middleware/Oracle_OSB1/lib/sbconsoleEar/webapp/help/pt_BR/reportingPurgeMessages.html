Expurgando Mensagens

Use esta página para expurgar mensagens selecionadas do armazenamento de dados de relatórios.

A expurgação da mensagem ocorre em segundo plano, portanto a página Resumo do Relatório de Mensagens não é bloqueada até que a expurgação ocorra. Portanto, você pode executar uma expurgação que demora um tempo para ser concluída; a duração depende de quantas mensagens precisam ser expurgadas. No entanto, se você filtrar mensagens de relatórios, o processo de expurgação será atrasado; a página Resumo do Relatório de Mensagens pode exibir dados incorretos, pois alguns dados ainda não foram expurgados.

O recurso Expurgar Mensagens pode ser útil durante as fases de design e teste do projeto. No entanto, em um ambiente de produção, a Oracle recomenda que o gerenciamento dos dados no banco de dados (incluindo expurgação) seja controlada pelo Administrador do Banco de Dados usando ferramentas de gerenciamento do banco de dados.

  1. Selecione Operações > Expurgar Mensagens.
  2. Execute uma das etapas a seguir, mostradas na Tabela 27-9.

    3. Tabela 27-9 - Opções de Expurgação da Mensagem
    Para...
    Execute Estas Etapas...
    Expurgar todas as mensagens
    1. Selecione o botão de rádio Expurgar Todas as Mensagens.
    2. Clique em Expurgar.
    Expurgar mensagens dentro de um período especificado
    1. Selecione o botão de rádio Expurgar Mensagens de... para... .
    2. No campo Expurgar Mensagens de, informe um mês, um dia, um ano e uma hora no formato MM/DD/AA HH:MM:SS. Por exemplo, é possível informar 10/10/07 12:45:00 AM.
    3. No campo Expurgar Mensagens para, informe um mês, um dia, um ano e uma hora no formato MM/DD/AA HH:MM:SS.
    4. Clique em Expurgar.
