Exibindo Detalhes da Mensagem

Use esta página para exibir detalhes de uma mensagem específica.

  1. Localize a mensagem. Consulte Exibindo Mensagens de Relatórios.
  2. Clique no índice do relatório. A página Exibir Detalhes da Mensagem exibe as informações de Configuração Geral mostradas na Tabela 27-3.

    3. Tabela 27-3 - Parâmetros de Configuração da Mensagem
    Propriedade
    Descrição
    ID da Mensagem
    A identificação exclusiva para esta mensagem.
    Timestamp do Banco de Dados
    A data e hora em que a mensagem foi registrada no banco de dados.
    Horário no ponto de Log
    A data e hora em que a mensagem foi registrada.
    Nome do servidor
    O nome do servidor em que esta mensagem foi gerada.
    Estado
    Estado do pipeline em que esta mensagem foi gerada:
    SOLICITAÇÃO - indica que a ação de geração de relatórios é executada em um pipeline de solicitação.
    RESPOSTA - indica que a ação de geração de relatórios é executada em um pipeline de resposta.
    ERRO - a ação está em execução no handler de erro de serviço.
    Nome do Nó
    O nome do nó do pipeline em que esta mensagem foi gerada.
    Nome do Pipeline
    O nome do pipeline em que esta mensagem foi gerada.
    Nome do Estágio
    O nome do estágio em que esta mensagem foi gerada.

    A página exibe as informações do Serviço de Entrada mostradas na Tabela 27-4.

    Tabela 27-4 - Detalhes do Serviço de Entrada
    Propriedade
    Descrição
    Nome
    O serviço de proxy de entrada associado a esta mensagem. Este é um link para a página Exibir um Serviço de Proxy - Detalhes da Configuração. Consulte Editando Configurações de Serviço de Proxy.
    URI
    O URI associado ao serviço de proxy.
    Operação
    A operação de entrada associada a esta mensagem.

    A página exibe as informações do Serviço de Saída mostradas na Tabela 27-5.

    Tabela 27-5 - Detalhes do Serviço de Saída
    Propriedade
    Descrição
    Nome
    O serviço de negócios de saída associado a esta mensagem. Este é um link para a página Exibir um Serviço de Negócios - Detalhes da Configuração. Consulte Editando Configurações de Serviço de Negócios.
    URI
    O URI para o ponto final do serviço de negócios de saída.
    Operação
    Nome da operação chamada no serviço de saída.

    A página exibe as informações do Índice do Relatório mostradas na Tabela 27-6.

    Tabela 27-6 - Detalhes de Índice do Relatório
    Propriedade
    Descrição
    Texto do Índice do Relatório
    Índices de relatório são pares de valor de chave usados para extrair identificadores-chave de uma variável de contexto da mensagem. Chaves são uma forma conveniente de filtrar mensagens. Use a ação do relatório para adicionar pares de valor de chave. Consulte Adicionando e Editando Ações em Fluxos de Mensagens.

    A página exibe as informações de Falha mostradas na Tabela 27-7.

    Tabela 27-7 - Detalhes de Informações sobre Falhas
    Propriedade
    Descrição
    Código do Erro
    O código do erro associado à mensagem, se houver.
    Especifique códigos de erro ao configurar a ação de emissão de erro. Consulte "Handling Errors in Message Flows" no Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus em http://www.oracle.com/pls/as111170/lookup?id=OSBAG232 e em Adicionando Handlers de Erro do Serviço de Proxy.
    Motivo
    O motivo do código do erro.
    Detalhe
    Os detalhes da falha associados ao código do erro. Esses detalhes, se presentes, geralmente são um rastreamento de pilha de onde ocorreu uma falha específica e podem ser truncados devido a uma limitação de tamanho no banco de dados. O limite é de 2048 caracteres.

    A página exibe as informações do Corpo do Relatório mostradas na Tabela 27-8.

    Tabela 27-8 - Detalhes do Corpo do Relatório
    Propriedade
    Descrição
    Texto do Corpo do Relatório
    O link abre outro browser que exibe os detalhes do corpo do relatório. Use uma expressão na ação de relatório para capturar o texto do corpo do relatório. Consulte o procedimento para a ação de relatório em Adicionando e Editando Ações em Fluxos de Mensagens.

  3. Clique em OK quando terminar de exibir os detalhes de uma mensagem específica.
