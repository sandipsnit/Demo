Usando o Browser do Recurso

O Resource Browser fornece ferramentas para localizar recursos e para executar ações simples neles, como Exibir, Editar, Exportar e Deletar.

  1. Para exibir o Resource Browser, selecione Resource Browser no painel de navegação esquerdo.
  2. No lado esquerdo da página, clique no nome do tipo de recurso que você deseja localizar. A Tabela 3-10 mostra os itens que você pode selecionar e as tarefas associadas à página exibida para cada um.
