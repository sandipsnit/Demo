Adicionando JARs

  1. Se ainda não tiver feito isso, clique em Criar para criar uma nova sessão ou clique em Editar para informar uma sessão existente. Consulte Usando o Centro de Alterações.
  2. Selecione o Project Explorer, depois selecione um projeto ou pasta no qual adicionar o JAR. A página View de Projeto/Pasta é mostrada.
  3. Na lista drop-down Criar Recurso, selecione JAR em Utilitário para exibir a página Criar um Novo Recurso JAR.
  4. No campo Nome do Recurso, informe um nome exclusivo para esse JAR. Esse campo é obrigatório.

    Siga as Restrições para Nomeação de Recursos para obter orientações em relação a nomes.
  5. No campo Descrição do Recurso, informe uma descrição para o JAR.
  6. No campo Arquivo de Origem JAR, execute um destes procedimentos:
    • Informe o caminho para o local do JAR.
    • Clique em Procurar para localizar e importar um JAR existente.

      • Esse campo é obrigatório.

  7. Clique em Salvar. O recurso JAR será criado e salvo na sessão atual.

    8. Se o JAR for carregado sem problemas, o conteúdo do JAR será disponibilizado para exibição. Para saber como exibir detalhes do JAR, consulte Editando JARs.

  8. Para encerrar a sessão e implantar a configuração no run-time, clique em Ativar em Centro de Alterações.
