Editando Destinos de Alerta

  1. Se ainda não tiver feito isso, clique em Criar para criar uma nova sessão ou clique em Editar para informar uma sessão existente. Consulte Usando o Centro de Alterações.
  2. Localize o destino de alerta, conforme descrito em Localizando Destinos de Alerta ou em Encontrando Destinos de Alerta.
  3. Clique no nome do destino de alerta. A página Exibir Destino de Alerta - Configuração exibe as informações mostradas na Tabela 5-2.

    4. Tabela 5-2 Detalhes de Configuração do Destino de Alerta 
    Propriedade
    Descrição
    Última Modificação
    O usuário que criou ou editou este destino de alerta.
    Última Modificação em
    A data e hora em que o usuário criou ou editou este serviço destino de alerta. Clique no link da data e hora para exibir o histórico de alterações deste recurso. Consulte a página Exibir Histórico de Alteração.
    Referências
    O número de objetos referenciados por este destino de alerta. Se tais referências existirem, clique no link numérico para exibir uma lista dos objetos. Consulte Exibindo Referências a Recursos.
    Referenciado por
    O número de objetos que referenciam esse destino de alerta. Se tais referências existirem, clique no link numérico para exibir uma lista dos objetos. Consulte Exibindo Referências a Recursos.
    Descrição
    Uma descrição deste destino de alerta, se existir.
    Trap do SNMP
    Sim ou Não, para indicar se a opção Trap do SNMP foi especificada como um destino para este alerta quando foi configurada.
    Relatórios
    Sim ou Não, para indicar se a opção Relatórios foi especificada como um destino para este alerta quando foi configurada.
    Log de Alerta
    Sim ou Não, para indicar se o Log de Alerta está ativado.
    E-mail e JMS
    Exibe uma lista de destinatários de e-mail ou destinos JMS, ou ambos, se foram configurados.

  4. É possível alterar sua seleção da definição de Trap do SNMP, Relatórios ou Log de Alerta.
  5. Para editar ou deletar um destinatário de e-mail ou destino JMS, clique em Editar na parte inferior da página.
    1. Para editar o destinatário de e-mail, em E-mail clique no ícone Editar em Opções do destinatário de e-mail que você deseja editar. Use a página Editar Destinatário de E-mail para fazer alterações nas informações do destinatário do e-mail. Consulte Adicionando Destinatários de E-mail.
    2. Para editar o destino JMS, em JMS clique no ícone Editar em Opções do destino JMS que você deseja editar. Use a página Editar Destino JMS para fazer alterações nas informações do destino JMS. Consulte Adicionando Destinos JMS.
    3. Para deletar um ou todos os destinatários de e-mail ou destinos JMS, clique no ícone Deletar em Opções dos destinatários de e-mail ou destinos JMS que você deseja deletar.
  6. Clique em Salvar para confirmar as atualizações na sessão atual.
  7. Para encerrar a sessão e implantar a configuração no run-time, clique em Ativar em Centro de Alterações.
