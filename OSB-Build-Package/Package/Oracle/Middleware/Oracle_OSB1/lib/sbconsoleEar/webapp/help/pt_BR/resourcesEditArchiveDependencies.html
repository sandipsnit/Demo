Editando Dependências do JAR

No Oracle Service Bus, os recursos JAR podem conter classes que dependem de outras classes em JARs diferentes. Use a página Editar Dependências do JAR para adicionar ou remover recursos do JAR referenciados.

  1. Se ainda não tiver feito isso, clique em Criar para criar uma nova sessão ou clique em Editar para informar uma sessão existente. Consulte Usando o Centro de Alterações.
  2. Localize o JAR, conforme descrito em Localizando JARs.
  3. Clique no nome do JAR.

    4. Na página Exibir um Recurso JAR - Configuração do JAR, um indicador numérico na guia Dependências exibe o número de JARs referenciados configurados para este recurso JAR, ou um zero se não houver nenhum.

  4. Selecione a guia Dependências para exibir os JARs referenciados ou para adicioná-los.

    5. A página de dependências do JAR inclui as informações mostradas na Tabela 7-4.

    Tabela 7-4 - Informações de Dependências do JAR
    Propriedade
    Descrição
    Nome
    O nome do JAR referenciado. O nome é um link para a página Exibir um Recurso JAR - Configuração do JAR. Consulte Editando JARs.
    Caminho
    O caminho é o nome do projeto e o nome da pasta em que reside o JAR.

  5. Para adicionar ou remover JARs referenciados, clique em Editar.

    6. A página Editar dependências do JAR inclui as informações mostradas na Tabela 7-5.

    Tabela 7-5 - Detalhes de Dependências do JAR
    Propriedade
    Descrição
    Caminho de Classe do Manifesto
    Entradas do Caminho de Classe no arquivo de manifesto do JAR.
    Nome
    O nome e o caminho do recurso JAR referenciado.
    Opções
    • Use as setas para Cima e para Baixo para alterar a ordem de precedência dos JARs referenciados.
    • Clique no ícone Deletar para remover um JAR referenciado.

  6. Clique em Adicionar JARS.
  7. No Browser de JARs, localize ou selecione recursos JAR referenciados e clique em Submeter.

    8. Por exemplo, se estiver editando o JAR A e adicionando os JARs B e C, o JAR A referenciará (depende) o JAR B e o JAR A referenciará (depende) o JAR C.

    Use as setas na coluna Opções para reordenar a hierarquia de JARs referenciados ou o ícone Deletar para removê-los.

  8. Clique em Salvar para confirmar as atualizações na sessão atual.
  9. Para encerrar a sessão e implantar a configuração no run-time, clique em Ativar em Centro de Alterações.
