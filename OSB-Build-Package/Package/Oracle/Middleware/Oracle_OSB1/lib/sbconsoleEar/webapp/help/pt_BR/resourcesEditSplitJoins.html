Editando Split-Joins

  1. Se ainda não tiver feito isso, clique em Criar para criar uma nova sessão ou clique em Editar para informar uma sessão existente. Consulte Usando o Centro de Alterações.
  2. Localize o Split-Join, conforme descrito em Localizando Split-Joins.
  3. Clique no nome do Split-Join.

    4. A página Exibir um Split-Join - Configuração exibe as informações mostradas na Tabela 9-3.

    Tabela 9-3 Detalhes do Split-Join 
    Propriedade
    Descrição
    Última Modificação
    O usuário que criou este Split-Join ou o importou para a configuração.
    Última Modificação em
    A data e hora em que o usuário criou este Split-Join ou o importou para a configuração. Clique no link da data e hora para exibir o histórico de alterações deste recurso. Consulte a página Exibir Histórico de Alteração.
    Referências
    O número de objetos referenciados por este Split-Join. Se tais referências existirem, clique no link numérico para exibir uma lista dos objetos. Consulte Exibindo Referências a Recursos.
    Referenciado por
    O número de objetos que referenciam esse Split-Join. Se tais referências existirem, clique no link numérico para exibir uma lista dos objetos. Consulte Exibindo Referências a Recursos.
    Descrição
    Uma descrição deste Split-Join, se existir.

  4. É possível fazer as seguintes definições operacionais em um Split-Join:

    5. Tabela 9-4 - Definições Operacionais do Split-Join
    Propriedade
    Descrição
    Logs
    Se deseja capturar informações de log em um Split-Join, selecione Ativar Log e selecione o nível de log que deseja capturar.
    Rastreamento da Execução
    Se deseja ativar o rastreamento de run-time, selecione Ativado.

  5. Clique em Atualizar para confirmar as atualizações na sessão atual.
  6. Para encerrar a sessão e implantar a configuração no run-time, clique em Ativar em Centro de Alterações.
