Editando JARs

Use a página Exibir um Recurso JAR - Configuração para atualizar os arquivos JAR. A atualização de um arquivo JAR envolve essencialmente apontar o recurso para uma nova versão do JAR, pois não é possível editar um JAR usando o Oracle Service Bus.

  1. Se ainda não tiver feito isso, clique em Criar para criar uma nova sessão ou clique em Editar para informar uma sessão existente. Consulte Usando o Centro de Alterações.
  2. Localize o JAR, conforme descrito em Localizando JARs.
  3. Clique no nome do JAR. A página Exibir um Recurso JAR - Configuração do JAR exibe as informações mostradas na Tabela 7-2.

    4. Tabela 7-2 Informações do JAR 
    Propriedade
    Descrição
    Última Modificação
    O usuário que importou este JAR para a configuração.
    Última Modificação em
    A data e hora em que o usuário importou este JAR para a configuração. Clique no link da data e hora para exibir o histórico de alterações deste recurso. Consulte a página Exibir Histórico de Alteração.
    Referências
    O número de JARs referenciados por este JAR. Se tais referências existirem, clique no link numérico para exibir uma lista dos JARs. Consulte Exibindo Referências a Recursos.
    Referenciado por
    O número de objetos que referenciam esse JAR. Se tais referências existirem, clique no link numérico para exibir uma lista dos objetos. Consulte Exibindo Referências a Recursos.
    Descrição
    Uma descrição deste JAR, se existir.

    A configuração do JAR é exibida por default, que inclui as informações mostradas na Tabela 7-3.

    Tabela 7-3 - Detalhes de Configuração do JAR
    Propriedade
    Descrição
    Configuração JAR
    A lista de classes e interfaces neste JAR.
    Clique em (+) ao lado do nome de uma classe Java para exibir uma lista de métodos incluídos na classe.

  4. Para fazer uma alteração nos campos, clique em Atualizar.

    5. O único campo que você pode editar é Descrição. Não é possível editar um JAR no Oracle Service Bus. Para atualizar o JAR, você deve selecionar um novo JAR para usar. Para saber como, consulte Adicionando JARs.

  5. Clique em Salvar para confirmar as atualizações na sessão atual.
  6. Para encerrar a sessão e implantar a configuração no run-time, clique em Ativar em Centro de Alterações.
Observação: Se você alterar o JAR usado por um Serviço de Negócios EJB (aponte o recurso JAR para uma nova versão do JAR), será necessário reimplantar o EJB. Depois será necessário editar qualquer serviço EJB que use esse JAR e selecionar o recurso JAR novamente, salvar e ativar. Isso compactará novamente o serviço de negócios EJB para usar o novo JAR.
Observação: Ações de chamada Java e serviços baseados no Tuxedo, no entanto, selecionará o novo JAR automaticamente.
