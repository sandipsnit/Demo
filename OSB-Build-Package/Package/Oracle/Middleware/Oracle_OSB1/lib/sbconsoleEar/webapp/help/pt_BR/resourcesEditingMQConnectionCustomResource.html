Editando Conexões MQ

Esta seção descreve como editar recursos existentes de Conexão MQ.

  1. Se ainda não tiver feito isso, clique em Criar para criar uma nova sessão ou clique em Editar para informar uma sessão existente. Consulte Usando o Centro de Alterações.
  2. Localize o recurso de Conexão MQ, conforme descrito em Localizando Conexões MQ.
  3. Clique no nome do recurso de Conexão MQ.

    4. A página Exibir Conexão MQ - Configuração exibe as informações mostradas na Tabela 8-3.

    Tabela 8-3 Detalhes da Conexão MQ 
    Propriedade
    Descrição
    Última Modificação
    O usuário que criou este recurso de Conexão MQ ou o importou para a configuração.
    Última Modificação em
    A data e hora em que o usuário criou este recurso de Conexão MQ ou o importou para a configuração. Clique no link da data e hora para exibir o histórico de alterações deste recurso. Consulte a página Exibir Histórico de Alteração.
    Referências
    O número de objetos referenciados por este recurso de Conexão MQ. Se tais referências existirem, clique no link numérico para exibir uma lista dos objetos. Consulte Exibindo Referências a Recursos.
    Referenciado por
    O número de objetos que referenciam esse recurso de Conexão MQ. Se tais referências existirem, clique no link numérico para exibir uma lista dos objetos. Consulte Exibindo Referências a Recursos.
    Descrição
    Uma descrição deste recurso de Conexão MQ, se existir.

  4. Para fazer uma alteração nos campos, clique em Editar. Para obter descrições dos campos, consulte Adicionando Conexões MQ.

    5. Não é possível alterar o campo Nome do Recurso.

  5. Clique em Salvar para confirmar as atualizações na sessão atual.
  6. Para encerrar a sessão e implantar a configuração no run-time, clique em Ativar em Centro de Alterações.
