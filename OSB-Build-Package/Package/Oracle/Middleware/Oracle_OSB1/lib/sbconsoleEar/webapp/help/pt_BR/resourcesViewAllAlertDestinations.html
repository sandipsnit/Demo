Localizando Destinos de Alerta

É possível exibir uma lista de destinos de alerta na página Resumo de Destinos de Alerta . Se você ainda não tiver criado um destino de alerta e quiser fazê-lo, crie um novo recurso do Destino de alerta no Project Explorer. Para saber como, consulte Adicionando Destinos de Alerta.

  1. Execute um dos seguintes procedimentos:
    • Selecione o Project Explorer para exibir a página View de Projetos ou a página View de Projeto/Pasta. Em seguida, navegue pelos projetos e pastas para localizar o destino de alerta.
    • Selecione Resource Browser > Destinos de Alerta. A página Resumo de Destinos de Alerta exibe as informações mostradas na Tabela 5-1.
  2. Para restringir o número de itens na lista, você pode filtrar por nome, caminho ou ambos. Nos campos Nome e Caminho, informe o nome e o caminho do(s) destino(s) de pesquisa e, em seguida, clique em Pesquisar.

    3. O caminho é o nome do projeto e o nome da pasta em que reside o destino de alerta.

    Clique em Exibir Tudo para remover os filtros de pesquisa e exibir todos os destinos de alerta.

    Tabela 5-1 - Detalhes do Destino de Alerta
    Propriedade
    Descrição
    Nome
    O nome do recurso de destino de alerta. O nome é um link para a página Exibir um Destino de Alerta - Configuração. Consulte Editando Destinos de Alerta.
    Caminho
    O caminho é o nome do projeto e o nome da pasta em que reside o destino de alerta. É um link para o projeto ou pasta que contém este recurso. Consulte Qualificando Nomes de Recursos Usando Projetos e Pastas.
    Opções
    Clique no ícone Deletar para deletar um Destino de Alerta específico. Consulte Deletando Destinos de Alerta.
