Localizando JARs

  1. Execute um dos seguintes procedimentos:
    • Selecione o Project Explorer para exibir a página View de Projetos ou a página View de Projeto/Pasta. Em seguida, navegue pelos projetos e pastas para localizar o JAR.
    • Selecione Resource Browser > JARs. A página Resumo de JARs exibe as informações mostradas na Tabela 7-1. Para obter uma descrição mais detalhada das propriedades, consulte Editando JARs.
  2. Para restringir o número de itens na lista ou para localizar um JAR específico, procure um JAR usando filtros de nome e de caminho. Nos campos Nome e Caminho, informe o nome e o caminho do(s) destino(s) de pesquisa e, em seguida, clique em Pesquisar.

    3. O caminho é o nome do projeto e o nome da pasta em que reside o JAR.

    Esta pesquisa suporta caracteres curingas. Por exemplo, você pode informar "j*" no campo Nome para obter uma lista de JARs que começam com "j".

    Clique em Exibir Tudo para remover os filtros de pesquisa e exibir todos os JARs.

    Tabela 7-1 - Detalhes do JAR
    Propriedade
    Descrição
    Nome JAR
    O nome exclusivo designado ao JAR. O nome é um link para a página Exibir um Recurso JAR - Configuração do JAR. Consulte Editando JARs.
    Caminho
    O caminho é o nome do projeto e o nome da pasta em que reside o JAR. É um link para o projeto ou pasta que contém este recurso. Consulte Qualificando Nomes de Recursos Usando Projetos e Pastas.
    No. de Classes
    O número de classes contidas pelo JAR.
    Tamanho
    O tamanho do JAR em bytes.
    Opções
    Clique no ícone Deletar para deletar um JAR específico. Consulte Deletando JARs.
