Localizando Conexões MQ

  1. Execute um dos seguintes procedimentos:
    • Selecione o Project Explorer para exibir a página View de Projetos ou a página View de Projeto/Pasta. Em seguida, navegue pelos projetos e pastas para localizar o recurso de Conexão MQ.
    • Selecione Resource Browser > Conexões MQ. A página Resumo de Conexões MQ exibe as informações mostradas na Tabela 8-1. Para obter uma descrição mais detalhada das propriedades, consulte Editando Conexões MQ.
  2. Para restringir o número de itens na lista, você pode filtrar por nome, caminho ou ambos. Nos campos Nome e Caminho, informe o nome e o caminho do(s) destino(s) de pesquisa e, em seguida, clique em Pesquisar.
  3. Clique no ícone Abrir para exibir filtros de pesquisa adicionais. Se você especificar vários critérios de pesquisa, apenas os resultados que correspondem a todos os critérios serão retornados.
    • Filtrar por Nome do Gerenciador de Filas. Informe qualquer string, incluindo caracteres curingas, depois clique em Pesquisar.
    • Filtrar por Nome do Canal do Gerenciador de Filas. Informe qualquer string, incluindo caracteres curingas, depois clique em Pesquisar.
    • Clique em Exibir Tudo para remover os filtros de pesquisa e exibir todos os recursos de Conexão MQ.

      • Tabela 8-1 - Detalhes da Conexão MQ
      Propriedade
      Descrição
      Nome da Conexão MQ
      O nome designado ao recurso de Conexão MQ. O nome é um link para a página Configuração da Conexão MQ. Consulte Editando Conexões MQ.
      Caminho
      O caminho é o nome do projeto e o nome da pasta em que reside o recurso de Conexão MQ. É um link para o projeto ou pasta que contém este recurso. Consulte Qualificando Nomes de Recursos Usando Projetos e Pastas.
      Opções
      Clique no ícone Deletar para deletar um recurso específico da Conexão MQ. Consulte Deletando Conexões MQ.
