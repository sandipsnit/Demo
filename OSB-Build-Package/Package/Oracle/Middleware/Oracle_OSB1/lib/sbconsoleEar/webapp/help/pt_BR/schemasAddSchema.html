Adicionando Esquemas XML

  1. Se ainda não tiver feito isso, clique em Criar para criar uma nova sessão ou clique em Editar para informar uma sessão existente. Consulte Usando o Centro de Alterações.
  2. Selecione o Project Explorer, depois selecione um projeto ou pasta no qual adicionar o novo esquema XML. A página View de Projeto/Pasta é mostrada.
  3. Na lista drop-down Criar Recurso, selecione Esquema XML para exibir a página Criar um Novo Recurso de Esquema XML.
  4. No campo Nome do Recurso, informe um nome exclusivo para esse esquema XML. Esse campo é obrigatório.

    Siga as Restrições para Nomeação de Recursos para obter orientações em relação a nomes.
  5. No campo Descrição do Recurso, informe uma descrição para o esquema XML.
  6. No campo Esquema XML, execute um dos seguintes procedimentos:
    • Informe texto para o novo esquema XML.
    • Clique em Procurar para localizar e importar um esquema XML existente.
    • Copie e cole texto a partir de um esquema XML existente neste campo.

      • Esse campo é obrigatório.

  7. Clique em Salvar. O novo esquema XML é salvo na sessão atual.

    8. Quando você clicar em Salvar, se houver referências não resolvidas para o novo esquema XML, o sistema as exibirá. Consulte Exibindo Referências XML Não Resolvidas.

  8. Para encerrar a sessão e implantar a configuração no run-time, clique em Ativar em Centro de Alterações.
