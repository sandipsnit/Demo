Localizando Esquemas XML

  1. Execute um dos seguintes procedimentos:
    • Selecione o Project Explorer para exibir a página View de Projetos ou a página View de Projeto/Pasta. Em seguida, navegue pelos projetos e pastas para localizar o esquema XML.
    • Selecione Resource Browser > Esquemas XML. A página Resumo de Esquemas XML exibe as informações mostradas na Tabela 4-1. Para obter uma descrição mais detalhada das propriedades, consulte Editando Esquemas XML.
  2. Para localizar um esquema XML específico:
    • Clique no ícone Abrir para exibir filtros de pesquisa adicionais. Se você especificar vários critérios de pesquisa, apenas os resultados que correspondem a todos os critérios serão retornados.
    • Para restringir o número de itens na lista, você pode filtrar por nome, caminho e namespace. Nos campos Nome, Caminho e Namespace, informe o nome, o caminho e o namespace do(s) destino(s) de pesquisa e, em seguida, clique em Pesquisar.

      • O caminho é o nome do projeto e o nome da pasta em que reside o esquema XML.

    • Clique em Exibir Tudo para remover os filtros de pesquisa e exibir todos os esquemas XML.

      • Tabela 4-1 - Informações do Esquema XML
      Propriedade
      Descrição
      Nome do Esquema XML
      O nome exclusivo designado ao esquema XML. O nome é um link para a página Detalhes do Esquema XML. Consulte Editando Esquemas XML.
      Caminho
      O caminho é o nome do projeto e o nome da pasta em que reside o esquema XML. É um link para o projeto ou pasta que contém este recurso. Consulte Qualificando Nomes de Recursos Usando Projetos e Pastas.
      Namespace do Esquema XML
      O namespace usado para qualificar qualquer uma das definições incluídas no esquema XML.
      Opções
      Clique no ícone Deletar para deletar um esquema XML específicos. Consulte Deletando Esquemas XML.
