Resolvendo Referências a Esquema XML Não Resolvidas

Use a página Editar as Referências de um Recurso de Esquema XML para resolver referências do esquema XML não resolvidas configurando o mapeamento para referências do esquema XML, como importações do esquema XML.

  1. Se ainda não tiver feito isso, clique em Criar para criar uma nova sessão ou clique em Editar para informar uma sessão existente. Consulte Usando o Centro de Alterações.
  2. Na página Resumo de Esquemas XML, na coluna Nome do Esquema XML, clique no nome do esquema XML apropriado para exibir detalhes da referência não resolvida. A página Exibir Detalhes do Esquema XML é exibida.
  3. Clique em Editar Referências para exibir a página Editar as Referências de um Recurso de Esquema XML.
  4. No campo Tipo de Recurso, selecione Esquema XML.
  5. Clique em Procurar. O Browser de Esquema XML será exibido.
  6. No Browser do Esquema XML, selecione um esquema XML, depois selecione uma definição no painel Definições.
  7. Clique em Submeter. O esquema XML selecionado será exibido no campo Nome do Recurso.
  8. Clique em Salvar para resolver a referência.
  9. Na página Detalhes do Esquema XML, clique em OK. O estado do esquema XML será exibido como Válido.
