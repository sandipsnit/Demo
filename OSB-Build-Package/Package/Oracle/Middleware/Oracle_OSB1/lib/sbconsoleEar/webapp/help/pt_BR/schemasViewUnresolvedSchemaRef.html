Exibindo Referências a Esquemas XML Não Resolvidas

Para cada referência do esquema XML não resolvida, esta página exibe as informações mostradas na Tabela 4-5.

Tabela 4-5 - Detalhes do Esquema XML Não Resolvido
Propriedade
Descrição
Nome do Esquema
O nome exclusivo designado ao esquema XML. Clique no nome de um esquema XML específico para exibir detalhes desse esquema XML. Consulte Editando Esquemas XML.
Namespace
O namespace usado para qualificar qualquer uma das definições incluídas no esquema XML.

Para localizar uma referência específica:

  • Classificar a lista novamente. Clique em um nome de coluna sublinhado. Setas crescentes e decrescentes indicam a ordem de classificação. Clique no nome da coluna para alterar a ordem de classificação.
  • Role pelas páginas. Use os controles de página acima ou abaixo da tabela. Para ir para uma página, selecione o número da página ou use os botões de seta para ir para a próxima página, página anterior, primeira ou última página.
