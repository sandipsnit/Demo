Editando Esquemas XML

  1. Se ainda não tiver feito isso, clique em Criar para criar uma nova sessão ou clique em Editar para informar uma sessão existente. Consulte Usando o Centro de Alterações.
  2. Localize o esquema XML, conforme descrito em Localizando Esquemas XML.
  3. Clique no nome do esquema XML. A página Exibir Detalhes do Esquema XML exibe as informações mostradas na Tabela 4-2, Tabela 4-3 e Tabela 4-4.

    4. Tabela 4-2 Detalhes do Esquema XML 
    Propriedade
    Descrição
    Última Modificação
    O usuário que criou este esquema XML ou o importou para a configuração.
    Última Modificação em
    A data e hora em que o usuário criou este esquema XML ou o importou para a configuração. Clique no link da data e hora para exibir o histórico de alterações deste recurso. Consulte a página Exibir Histórico de Alteração.
    Referências
    O número de objetos referenciados por este esquema XML. Se tais referências existirem, clique no link numérico para exibir uma lista dos objetos. Consulte Exibindo Referências a Recursos.
    Referenciado por
    O número de objetos que referenciam este esquema XML. Se tais referências existirem, clique no link numérico para exibir uma lista dos objetos. Por exemplo, se este esquema XML for referenciado por um WSDL específico, o WSDL será listado como uma referência quando você clicar no link. Consulte Exibindo Referências a Recursos.
    Descrição
    Uma descrição deste esquema XML, se existir.

    A view estrutural do esquema XML é exibida por default, que inclui as informações mostradas na Tabela 4-3.

    Tabela 4-3 - Detalhes da Estrutura XML
    Propriedade
    Descrição
    Namespace de Destino
    O namespace usado para qualificar qualquer uma das definições incluídas no esquema XML.
    Definições de Esquema XML
    Os atributos e grupos associados ao esquema XML:
    • Tipos de Esquema XML
    • Elementos do Esquema XML
    • Importações do Esquema XML
    • Inclusões/Redefinições do Esquema XML
    Estado do Esquema
    Status do esquema XML:
    • Válido
      Um esquema XML é mostrado como válido quando todos os locais de esquemas XML incluídos pelo esquema XML atual são especificados. Além disso, todos os locais de esquemas XML aninhados (esquemas XML incluídos subsequentemente pelos esquemas XML incluídos) são especificados.
    • Inválido
      Um esquema XML é mostrado como inválido quando um ou mais dos locais do esquema XML não foi especificado. Além disso, o local de um ou mais esquemas XML aninhados (esquemas XML incluídos subsequentemente pelos esquemas XML incluídos) pode não estar especificado. Para resolver um esquema XML inválido, clique em Editar Referências. Consulte Resolvendo Referências a Esquema XML Não Resolvidas.

  4. Clique em View de Texto para exibir uma view de texto do esquema XML. A view de texto inclui as informações mostradas na Tabela 4-4.

    5. Tabela 4-4 - Detalhes do Texto do Esquema
    Propriedade
    Descrição
    Namespace de Destino
    O namespace usado para qualificar qualquer uma das definições incluídas no esquema XML.
    Esquema XML
    O texto deste esquema XML.

  5. Para fazer uma alteração nos campos, clique em Editar. Consulte Adicionando Esquemas XML para obter descrições dos campos.
  6. Faça as edições apropriadas.
  7. Clique em Salvar para confirmar as atualizações na sessão atual.
  8. Para encerrar a sessão e implantar a configuração no run-time, clique em Ativar em Centro de Alterações.
