Configuração da Segurança

Use o módulo Configuração de Segurança para criar e modificar dados de segurança usados na segurança de entrada e na segurança administrativa do Oracle Service Bus.

A segurança de transporte de entrada e a segurança de mensagem usam os dados do usuário, grupo e atribuição para autenticar solicitações do cliente de entrada. Aplica políticas de controle de acesso para determinar quais usuários autenticados têm autorização para usar serviços de proxy e serviços de negócios.

A segurança administrativa usa os dados do usuário, grupo e atribuição para determinar quais usuários autenticados têm autorização para criar ou modificar os dados de configuração do Oracle Service Bus ou para monitorar o desempenho do Oracle Service Bus. Para obter mais informações, consulte Understanding Users, Groups, Security Roles and Policies. Consulte também "Configuring Administrative Security" no Oracle Fusion Middleware Developer's Guide for Oracle Service Bus em http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539.

Observação: Não é possível exportar usuários, grupos ou atribuições ao exportar uma configuração, porque esses objetos estão localizados em armazenamentos do provedor de segurança. Você deve criar esses objetos novamente ao importar a configuração exportada ou usar as ferramentas do WebLogic Server (se disponíveis) para exportar e importá-los.
