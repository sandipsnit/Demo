Adicionando Grupos

  1. Efetue log-in na Console do Oracle Service Bus Administration como um usuário com privilégios Admin do WebLogic Server. Apenas usuários com a atribuição Admin podem modificar dados da configuração de segurança. Consulte "Configuring Administrative Security" in the Oracle Fusion Middleware Developer's Guide for Oracle Service Bus em http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539.
  2. Selecione Configuração de Segurança > Grupos para exibir a página Resumo de Grupos.
  3. Clique em Adicionar Novo.

    4. Você pode adicionar um grupo estando dentro ou fora de uma sessão.

  4. No campo Nome do Grupo, informe um nome exclusivo. Observe que não é possível informar espaços nem caracteres especiais. Esse campo é obrigatório.
  5. No campo Provedor de Autenticação, selecione o provedor de autenticação.
  6. No campo Associação ao Grupo, selecione um grupo ao qual este usuário pertence.
    1. Selecione um grupo no campo Grupos Disponíveis.
    2. Clique na seta para mover o grupo para o campo Grupos Atuais.
  7. Clique em Salvar para criar o grupo.

    8. A Console do Oracle Service Bus Administration salva o grupo, que é disponibilizado imediatamente para todas as sessões. Se você estiver em uma sessão ao adicionar o grupo e descartar a sessão, a Console do Oracle Service Bus Administration não deletará o novo grupo.
