Editando Grupos

Use a página Exibir Detalhes do Grupo para exibir e alterar os detalhes de um grupo específico.

  1. Efetue log-in na Console do Oracle Service Bus Administration como um usuário com privilégios Admin do WebLogic Server. Apenas usuários com a atribuição Admin podem modificar dados da configuração de segurança. Consulte "Configuring Administrative Security" in the Oracle Fusion Middleware Developer's Guide for Oracle Service Bus em http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539.
  2. Localize o grupo. Consulte Localizando Grupos.
  3. Clique no nome do grupo. A página Exibir Detalhes do Grupo exibe as informações mostradas na Tabela 24-4.

    4. Tabela 24-4 - Detalhes do Grupo
    Propriedade
    Descrição
    Nome do Grupo
    O nome desse grupo.
    Provedor de Autenticação
    O provedor de autenticação que contém esta definição do grupo.
    Grupos
    O grupo ao qual este grupo pertence.

  4. Para editar os detalhes do grupo, clique em Reconfigurar para exibir a página Editar Detalhes do Grupo.

    5. Você pode editar detalhes do grupo estando dentro ou fora de uma sessão.

  5. Faça as alterações apropriadas no campo Associação do Grupo. Consulte Adicionando Grupos para obter descrições do campo.

    6. Não é possível alterar o campo Nome do Grupo.

  6. Clique em Salvar Alterações para atualizar o grupo.

    7. A Console do Oracle Service Bus Administration atualiza os detalhes do grupo e a atualização é disponibilizada imediatamente para todas as sessões. Se você estiver em uma sessão ao atualizar o grupo e descartar a sessão, a Console do Oracle Service Bus Administration não deletará as atualizações.
