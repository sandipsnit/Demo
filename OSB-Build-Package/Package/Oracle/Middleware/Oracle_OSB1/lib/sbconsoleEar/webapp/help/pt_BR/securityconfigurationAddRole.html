Adicionando Atribuições

  1. Efetue log-in na Console do Oracle Service Bus Administration como um usuário com privilégios Admin do WebLogic Server. Apenas usuários com a atribuição Admin podem modificar dados da configuração de segurança. Consulte "Configuring Administrative Security" in the Oracle Fusion Middleware Developer's Guide for Oracle Service Bus em http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539.
  2. Selecione Configuração de Segurança > Atribuições para exibir a página Atribuições Globais.
  3. Clique em Novo.

    4. Você pode adicionar uma atribuição estando dentro ou fora de uma sessão.

  4. No campo Nome da Atribuição, informe um nome exclusivo. Esse campo é obrigatório.

    5. Certifique-se de que não haja espaços nem caracteres < > no nome da atribuição de segurança. Nomes de atribuição de segurança fazem distinção entre maiúsculas e minúsculas. A convenção Oracle é que todos os nomes de atribuição de segurança sejam singulares.

  5. Para criar a atribuição, clique em OK.

    6. A Console do Oracle Service Bus Administration salva a atribuição, que é disponibilizada imediatamente para todas as sessões. Se você estiver em uma sessão ao adicionar a atribuição e descartar a sessão, a Console do Oracle Service Bus Administration não deletará a nova atribuição.

    Quando você clica em OK para criar a atribuição, a próxima etapa é definir as condições em que a atribuição se aplica.

  6. Continue em Definindo Condições da Atribuição.
