Adicionando Usuários

  1. Efetue log-in na Console do Oracle Service Bus Administration como um usuário com privilégios Admin do WebLogic Server. Apenas usuários com a atribuição Admin podem modificar dados da configuração de segurança. Consulte "Configuring Administrative Security" in the Oracle Fusion Middleware Developer's Guide for Oracle Service Bus em http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539.
  2. Selecione Configuração de Segurança para exibir a página Resumo de Usuários.
  3. Clique em Adicionar Novo para exibir a página Criar um Novo Usuário - Configuração Geral.

    4. Você pode adicionar um usuário estando dentro ou fora de uma sessão.

  4. No campo Nome do Usuário, informe um nome exclusivo. Esse campo é obrigatório.
  5. No campo Senha, informe uma senha. Esse campo é obrigatório.
    6. Observação: Provedores de autenticação podem impor um comprimento mínimo de senha. Para um usuário definido no provedor de Autenticação do WebLogic, o comprimento mínimo da senha default é de 8 caracteres. É possível personalizar essa definição usando a Console de Administração do WebLogic Server. (O provedor de Autenticação do WebLogic é configurado no realm de segurança default com o nome DefaultAuthenticator.)
  6. No campo Confirmar Senha, informe a mesma senha informada para o campo Senha. Esse campo é obrigatório.
  7. Na lista drop-down Provedor de Autenticação, selecione o provedor de autenticação para este usuário.

    8. Se vários provedores de autenticação forem configurados no realm de segurança, eles aparecerão na lista. Selecione o banco de dados do provedor de autenticação que deve armazenar informações para o novo usuário.

  8. No campo Associação ao Grupo, selecione um grupo para este usuário.
    1. Selecione um grupo no campo Grupos Disponíveis.
    2. Clique na seta para mover o grupo para o campo Grupos Atuais.
  9. Clique em Salvar para criar o usuário.

    10. A Console do Oracle Service Bus Administration salva o usuário, que é disponibilizado imediatamente para todas as sessões. Se você estiver em uma sessão ao adicionar o usuário e descartar a sessão, a Console do Oracle Service Bus Administration não deletará o novo usuário.
