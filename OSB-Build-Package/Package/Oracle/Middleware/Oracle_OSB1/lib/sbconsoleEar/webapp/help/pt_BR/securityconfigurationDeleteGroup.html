Deletando Grupos

Use a página Resumo de Grupos para deletar um grupo selecionado ou vários grupos.

  1. Efetue log-in na Console do Oracle Service Bus Administration como um usuário com privilégios Admin do WebLogic Server. Apenas usuários com a atribuição Admin podem modificar dados da configuração de segurança. Consulte "Configuring Administrative Security" in the Oracle Fusion Middleware Developer's Guide for Oracle Service Bus em http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539.
  2. Selecione Configuração de Segurança para exibir a página Resumo de Grupos.
  3. Selecione o grupo que você deseja deletar. É possível selecionar vários grupos, se necessário.

    4. Você pode deletar grupos estando dentro ou fora de uma sessão.

  4. Clique em Deletar. Uma mensagem solicitando que você confirme se deseja deletar o grupo será exibida.
  5. Para deletar o grupo, clique em OK.

    6. A Console do Oracle Service Bus Administration deleta o grupo. Se você estiver em uma sessão ao deletar o grupo e descartar a sessão, a Console do Oracle Service Bus Administration não cancelará a deleção do grupo.

  6. Como alternativa, você pode clicar no ícone Deletar na coluna Opções do grupo que deseja deletar.
