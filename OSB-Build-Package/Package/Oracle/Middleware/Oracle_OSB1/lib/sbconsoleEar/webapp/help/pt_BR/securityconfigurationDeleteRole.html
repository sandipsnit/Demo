Deletando Atribuições

  1. Efetue log-in na Console do Oracle Service Bus Administration como um usuário com privilégios Admin do WebLogic Server. Apenas usuários com a atribuição Admin podem modificar dados da configuração de segurança. Consulte "Configuring Administrative Security" in the Oracle Fusion Middleware Developer's Guide for Oracle Service Bus em http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539.
  2. Selecione Configuração de Segurança > Atribuições para exibir a página Atribuições Globais.
  3. Selecione a atribuição que você deseja deletar. É possível selecionar várias atribuições, se necessário.

    4. Você pode deletar atribuições estando dentro ou fora de uma sessão.

  4. Clique em Deletar. Uma mensagem solicitando que você confirme se deseja deletar a atribuição será exibida.
  5. Para deletar a atribuição, clique em OK.

    6. A Console do Oracle Service Bus Administration deleta a atribuição. Se você estiver em uma sessão ao deletar o grupo e descartar a sessão, a Console do Oracle Service Bus Administration não cancelará a deleção da atribuição.
