Deletando Usuários

Use a página Resumo de Usuários para deletar um usuário selecionado ou vários usuários.

  1. Efetue log-in na Console do Oracle Service Bus Administration como um usuário com privilégios Admin do WebLogic Server. Apenas usuários com a atribuição Admin podem modificar dados da configuração de segurança. Consulte "Configuring Administrative Security" in the Oracle Fusion Middleware Developer's Guide for Oracle Service Bus em http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539.
  2. Selecione Configuração de Segurança para exibir a página Resumo de Usuários.
  3. Selecione o usuário que você deseja deletar. É possível selecionar vários usuários, se necessário.

    4. Você pode deletar um usuário estando dentro ou fora de uma sessão.

  4. Clique em Deletar. Uma mensagem solicitando que você confirme se deseja deletar o usuário será exibida.
  5. Para deletar o usuário, clique em OK.

    6. A Console do Oracle Service Bus Administration deleta o usuário. Se você estiver em uma sessão ao deletar o usuário e descartar a sessão, a Console do Oracle Service Bus Administration não deletará as atualizações.

  6. Como alternativa, você pode clicar no ícone Deletar na coluna Opções do usuário que deseja deletar.
