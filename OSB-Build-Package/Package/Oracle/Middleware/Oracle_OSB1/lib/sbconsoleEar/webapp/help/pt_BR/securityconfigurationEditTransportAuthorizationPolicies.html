Editando Políticas de Acesso de Transporte

Use a página Exibir Detalhes da Política para editar a política de controle de acesso de transporte de um serviço de proxy. A página exibe as informações mostradas na Tabela 24-10.

Tabela 24-10 - Detalhes da Política
Propriedade
Descrição
Nome do Serviço de Proxy
Exibe o nome do serviço de proxy para o qual você selecionou Controle de Acesso do Transporte na página Segurança.
Provedores
Exibe os provedores de autorização configurados para o realm de segurança.
Condições de Política
Determina as condições que estabelecem para quais usuários o serviço de proxy processará solicitações.

  1. Efetue log-in na Console do Oracle Service Bus Administration como um usuário com privilégios Admin do WebLogic Server. Apenas usuários com a atribuição Admin podem modificar dados da configuração de segurança. Consulte "Configuring Administrative Security" in the Oracle Fusion Middleware Developer's Guide for Oracle Service Bus em http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539.
  2. Localizar a política de acesso. Consulte Localizando Políticas de Controle de Acesso.
  3. Na página Segurança, na coluna Controle de Acesso do Transporte, clique no nome do serviço de proxy. O editor de política será exibido.

    4. É possível editar uma política de controle de acesso dentro de uma sessão.

  4. No campo Provedores de Autorização, selecione um provedor de autorização. A Oracle recomenda que você selecione o XACMLAuthorizer.
    5. Observação: O Oracle Service Bus reprovou suporte para o provedor de Autorização Default do WebLogic. Em vez disso, a Oracle recomenda que você use o provedor de Autorização XACML do WebLogic.
  5. Adicionar condições da política. Consulte Adicionando Condições da Política.
  6. Ao terminar de informar condições na seção Condições da Política, clique em Salvar.
