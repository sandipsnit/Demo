Localizando Políticas de Controle de Acesso

A página Segurança oferece um link para as política de controle de acesso de um serviço de proxy no domínio atual do Oracle Service Bus.

Esta página lista os serviços de proxy que você criou na sessão, mas que ainda não foram ativados. Se você quiser editar políticas de controle de acesso para um novo serviço de proxy, primeiro ative a sessão em que você criou o serviço de proxy.

  1. Se deseja localizar as políticas de controle de acesso para um novo serviço de proxy, ative a sessão em que você criou o serviço de proxy.
  2. Na coluna Controle de Acesso , selecione o nome do serviço de proxy no Controle de Acesso do Transporte ou no nome do serviço de proxy ou em uma operação específica em Gerenciar Controle de Acesso.
