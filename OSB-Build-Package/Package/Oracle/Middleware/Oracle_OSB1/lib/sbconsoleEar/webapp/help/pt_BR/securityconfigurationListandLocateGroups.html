Localizando Grupos

  1. Selecione Configuração de Segurança > Grupos. A página Resumo de Grupos exibe as informações mostradas na Tabela 24-3. Para obter uma descrição mais detalhada das propriedades, consulte Editando Grupos.
  2. Para restringir o número de itens na lista ou localizar um grupo específico, é possível filtrar pelo nome do grupo. Informe parte ou todo o nome do grupo no campo Nome e clique em Pesquisar.

    3. É possível usar o asterisco (*) como caractere curinga. (Outros caracteres curingas não são suportados.)

    Clique em Exibir Tudo para remover os filtros de pesquisa e exibir todos os grupos.

    Tabela 24-3 Informações do Grupo 
    Propriedade
    Descrição
    Nome do Grupo
    O nome do grupo. O nome é um link para a página Exibir Detalhes do Grupo. Consulte Editando Grupos.
    Associação ao Grupo
    O grupo ao qual este grupo pertence. O nome é um link para a página Exibir Detalhes do Grupo. Consulte Editando Grupos.
    Provedor de Autenticação
    O provedor de autenticação que contém esta definição do grupo.
    Deletar
    Clique no ícone Deletar para deletar um grupo específico. Consulte Deletando Grupos.
