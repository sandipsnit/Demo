Localizando Atribuições

  1. Selecione Configuração de Segurança > Atribuições. A página Atribuições Globais exibe as informações mostradas na Tabela 24-5. Para obter uma descrição mais detalhada das propriedades, consulte Editando Atribuições.
  2. Para restringir o número de itens na lista ou localizar uma atribuição específica, role pelas páginas. Use os controles de página acima ou abaixo da tabela.

    3. Tabela 24-5 - Informações sobre Atribuição
    Propriedade
    Descrição
    Nome da Atribuição
    O nome da atribuição. O nome é um link para a página Exibir Detalhes da Atribuição. Consulte Editando Atribuições.
    Nome do Provedor
    O provedor de autenticação para este grupo.
