Localizando Usuários

  1. Selecione Configuração de Segurança > Usuários. A página Resumo de Usuários exibe as informações mostradas na Tabela 24-1. Para obter uma descrição mais detalhada das propriedades, consulte Editando Usuários.
  2. Para restringir o número de itens na lista ou localizar um usuário específico, é possível filtrar pelo nome do usuário. Informe parte ou todo o nome do usuário no campo Nome e clique em Pesquisar.

    3. É possível usar o asterisco (*) como caractere curinga. (Outros caracteres curingas não são suportados.)

    Clique em Exibir Tudo para remover os filtros de pesquisa e exibir todos os usuários.

    Tabela 24-1 - Informações de Usuários
    Propriedade
    Descrição
    Nome do Usuário
    O nome designado ao usuário. O nome é um link para a página Exibir Detalhes do Usuário. Consulte Editando Usuários.
    Associação ao Grupo
    O nome do grupo ao qual este usuário pertence. O nome é um link para a página Exibir Detalhes do Grupo. Consulte Editando Grupos.
    Provedor de Autenticação
    O provedor de autenticação para este usuário.
    Opções
    Clique no ícone Deletar para deletar um usuário específico. Consulte Deletando Usuários.
