Editando Usuários

Use a página Exibir Detalhes do Usuário para exibir e alterar os detalhes de um usuário específico.

  1. Efetue log-in na Console do Oracle Service Bus Administration como um usuário com privilégios Admin do WebLogic Server. Apenas usuários com a atribuição Admin podem modificar dados da configuração de segurança. Consulte "Configuring Administrative Security" in the Oracle Fusion Middleware Developer's Guide for Oracle Service Bus em http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539.
  2. Localize o usuário. Consulte Localizando Usuários.
  3. Clique no nome do usuário. A página Exibir Detalhes do Usuário exibe as informações mostradas na Tabela 24-2.

    4. Tabela 24-2 - Detalhes do Usuário
    Propriedade
    Descrição
    Nome do Usuário
    O nome deste usuário
    Provedor de Autenticação
    O provedor de autenticação que contém esta definição do usuário.
    Associação ao Grupo
    O nome do grupo ao qual este usuário pertence.

  4. Para editar os detalhes do usuário, clique em Reconfigurar para exibir a página Editar Detalhes do Usuário.

    5. Você pode editar detalhes do usuário estando dentro ou fora de uma sessão.

  5. Faça as alterações apropriadas nos campos Nova Senha, Confirmar Senha e Associação do Grupo. Consulte Adicionando Usuários para obter descrições dos campos.

    6. Não é possível alterar o campo Nome de Usuário.

  6. Clique em Salvar Alterações para atualizar o usuário.

    7. A Console do Oracle Service Bus Administration atualiza os detalhes do usuário e a atualização é disponibilizada imediatamente para todas as sessões. Se você estiver em uma sessão ao atualizar o usuário e descartar a sessão, a Console do Oracle Service Bus Administration não deletará as atualizações.
