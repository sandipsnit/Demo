Contas de Serviço

Uma conta de serviço oferece um nome de usuário e senha que os serviços de proxy e os serviços de negócios usam para autenticação de saída ou autenticação em um recurso local ou remoto, como um servidor FTP ou um servidor JMS. Por exemplo, se um serviço de negócios precisar fornecer um nome de usuário e senha para autenticação de transporte em um Web Service, crie uma conta de serviço que especifica o nome de usuário e senha, depois configure o serviço de negócios para incluir as credenciais de conta do serviço em suas solicitações de saída.

Os nomes de usuário e senhas informados em contas de serviço são usados para autenticação de saída ou para fornecer credenciais para recursos locais ou remotos. Os nomes de usuário e senhas informados no módulo Configuração da Segurança da Console do Oracle Service Bus Administration são usados para autenticação de entrada e para autenticação de solicitações administrativas. Consulte Especificando Contas de Serviço.
