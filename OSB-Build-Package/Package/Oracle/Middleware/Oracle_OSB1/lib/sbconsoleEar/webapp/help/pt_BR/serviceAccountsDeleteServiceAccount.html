Deletando Contas de Serviço

Quando você deleta uma conta de serviço, o nome de usuário, a senha ou os dados do mapeamento do usuário local para o usuário remoto que a conta de serviço contém também são deletados.

  1. Se ainda não tiver feito isso, clique em Criar para criar uma nova sessão ou clique em Editar para informar uma sessão existente. Consulte Usando o Centro de Alterações.
  2. Se qualquer serviço de negócios ou de proxy for configurado para usar a conta de serviço, remova-a do serviço de negócios ou de proxy.

    3. Consulte Editando Configurações do Serviço de Negócios ou Editando Configurações de Serviço de Proxy.

  3. Selecione Resource Browser > Contas de Serviço para exibir a página Resumo de Contas de Serviço.
  4. Clique no ícone Deletar no campo Opções da conta de serviço que você deseja deletar.

    5. A conta de serviço será deletada na sessão atual. Se um serviço de negócios ou um serviço de proxy tiver sido configurado para usar uma conta de serviço, um ícone Advertência de Deleção indicará que é possível deletar a conta de serviço com uma confirmação de advertência. Isso pode resultar em conflitos devido a referências não resolvidas do serviço à conta de serviço deletada.

  5. Para encerrar a sessão e implantar a configuração no run-time, clique em Ativar em Centro de Alterações.
