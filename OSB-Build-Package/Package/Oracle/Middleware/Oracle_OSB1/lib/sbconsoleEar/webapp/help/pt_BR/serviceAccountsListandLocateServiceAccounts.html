Localizando Contas de Serviço

  1. Execute um dos seguintes procedimentos:
    • Selecione o Project Explorer para exibir a página View de Projetos ou a página View de Projeto/Pasta. Em seguida, navegue pelos projetos e pastas para localizar a conta de serviço.
    • Selecione Resource Browser > Contas de Serviço. O Resumo de Contas de Serviço exibe as informações mostradas na Tabela 16-1.
  2. Para procurar uma conta de serviço, informe parte ou todo o nome da conta no campo Nome. Também é possível informar parte ou todo o nome e pasta do projeto de conta nos campos Caminho. Clique em Pesquisar.

    3. Clique em Exibir Tudo para remover os filtros de pesquisa e exibir todas as contas de serviço.

    Tabela 16-1 - Informações da Conta de Serviço
    Propriedade
    Descrição
    Nome da Conta do Serviço
    Um nome exclusivo para a conta de serviço. Clique no nome para ver a página Exibir Detalhes da Conta de Serviço. Consulte Editando Contas de Serviço.
    Caminho
    O nome do projeto e o nome da pasta em que reside a conta de serviço. Clique no nome para ver o projeto ou pasta que contém este recurso. Consulte Qualificando Nomes de Recursos Usando Projetos e Pastas.
    Opções
    Contém um ícone Deletar. Se um serviço de negócios ou um serviço de proxy tiver sido configurado para usar uma conta de serviço, um ícone Advertência de Deleção indicará que é possível deletar a conta de serviço com uma confirmação de advertência. Isso pode resultar em conflitos devido a referências não resolvidas do serviço à conta de serviço deletada. Consulte Deletando Contas de Serviço.
