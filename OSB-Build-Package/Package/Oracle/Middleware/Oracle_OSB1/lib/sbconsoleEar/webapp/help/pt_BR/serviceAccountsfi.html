Usando Dados e Sessões de Contas de Serviço

As contas de serviço e seus dados participam totalmente das sessões do Oracle Service Bus: você deve estar em uma sessão para criar ou modificar uma conta de serviço; se quiser descartar a sessão, a conta de serviço e seus dados também serão descartados. Quando você ativa uma sessão, o Oracle Service Bus salva o nome de usuário, a senha e outros dados da conta de serviço no provedor do mapeamento de credencial de nome de usuário/senha configurado para o domínio.
