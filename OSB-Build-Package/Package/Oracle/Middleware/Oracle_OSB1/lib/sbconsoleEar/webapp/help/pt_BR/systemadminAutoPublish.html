Usando Publicação Automática

Você pode usar publicação automática para publicar automaticamente serviços de proxy em um registro. Para fazer isso, você deve ativar a opção Publicar no Registro na página Criar/Editar um Serviço Proxy - Configuração Geral. Consulte a página Configuração Geral. Observe que a opção Publicar no Registro é exibida quando um registro default existe. Consulte Definindo uma Configuração UDDI Default.

  1. Se ainda não tiver feito isso, clique em Criar para criar uma nova sessão ou clique em Editar para informar uma sessão existente. Consulte Usando o Centro de Alterações.
  2. Selecione o Project Explorer.
  3. Expanda a pasta do projeto e selecione o serviço de proxy que você deseja editar.
  4. Na parte inferior da página Exibir um Serviço de Proxy, clique em Editar.
  5. Selecione Publicar no Registro e clique em Salvar para confirmar as atualizações na sessão atual.
  6. Para encerrar a sessão e implantar a configuração no run-time, clique em Ativar em Centro de Alterações.

    7. O serviço é configurado para ser publicado no registro default.

Observações:
Observações:
  • Quando você tiver publicado com sucesso um serviço em um registro, a opção de selecionar o serviço para publicação automática em um registro não estará mais disponível.
Observações:
  • Se o serviço não tiver sido publicado com sucesso, ele poderá ser publicado novamente. Para publicar um serviço novamente, selecione o serviço na página Status da Publicação Automática e clique em Publicar.
Observações:
  • Se a opção Publicar no Registro for ativada, os serviços de proxy serão publicados assim que forem criados ou editados, e a sessão será ativada. É possível usar a opção Publicar no Registro com todos os serviços de proxy, exceto com aqueles que usam Transporte Local.
