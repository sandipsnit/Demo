Criando Arquivos de Personalização

A página Criar Arquivo de Personalização oferece uma forma conveniente para gerar um arquivo de personalização para um conjunto de recursos ou projetos selecionados. Você pode usar esse arquivo como um ponto inicial para fazer as modificações desejadas, especificando os valores reais para um ambiente.

É possível usar arquivos de personalização para fazer alterações em valores de ambiente e também para alterar referências em recursos. Arquivos de personalização podem incluir personalizações para todos os valores de ambiente encontrados nos recursos selecionados, incluindo tipos de valores de ambiente complexos definidos na classe EnvValueTypes. Além disso, inclui um tipo de personalização de referência para alteração das referências a recursos dentro de recursos com dependências.

O esquema de personalização (Customization.xsd) que descreve os tipos de personalização está disponível no seguinte local na instalação do Oracle Service Bus:

OSB_ORACLE_HOME\modules\com.bea.common.configfwk_version.jar.

  1. Selecione Administração do Sistema > Criar Arquivo de Personalização.

    2. A página Criar Arquivo de Personalização exibe a lista de objetos na configuração. O nome e o tipo de cada objeto é exibido.

  2. Selecione os projetos ou recursos que você deseja incluir no arquivo de personalização.
    1. Expanda as pastas do projeto. O nome e o tipo de cada recurso contido no projeto será exibido.
    2. Marque as caixas de seleção associadas aos projetos ou recursos que você deseja incluir no arquivo de personalização.
    3. Desmarque as caixas de seleção associadas aos projetos ou recursos que você não deseja incluir no arquivo de personalização.
  3. Clique em Criar Arquivo para criar um arquivo XML de personalização.
  4. Na caixa de diálogo, clique em Download de Arquivo, clique em Abrir para abrir o arquivo de personalização ou clique em Salvar para salvar o arquivo XML na área de trabalho.
