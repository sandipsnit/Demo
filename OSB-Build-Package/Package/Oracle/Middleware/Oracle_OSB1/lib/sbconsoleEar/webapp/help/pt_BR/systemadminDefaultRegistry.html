Definindo uma Configuração UDDI Default

É possível designar um dos registros configurados como o registro UDDI default para o domínio. Consulte Configurando Registros UDDI.

Para usar a funcionalidade de publicação automática, primeiro defina um registro default. Consulte Usando Publicação Automática.

  1. Se ainda não tiver feito isso, clique em Criar para criar uma nova sessão ou clique em Editar para informar uma sessão existente. Consulte Usando o Centro de Alterações.
  2. Selecione Administração do Sistema > Registros UDDI.
  3. No painel Configuração Default do UDDI, clique em Selecionar Default.
  4. Na lista Nome do Registro Default, selecione o nome do registro que você deseja definir como o default.
  5. Para definir uma entidade de negócios default, escolha uma entidade na lista Entidade de Negócios.
  6. Clique em Tornar Default.
  7. Para alterar o registro default, clique em Limpar Seleção e repita as etapas acima.
