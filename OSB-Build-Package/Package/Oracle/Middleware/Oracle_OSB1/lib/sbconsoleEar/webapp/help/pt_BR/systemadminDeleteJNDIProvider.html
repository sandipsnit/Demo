Deletando Provedores de JNDI

  1. Se ainda não tiver feito isso, clique em Criar para criar uma nova sessão ou clique em Editar para informar uma sessão existente. Consulte Usando o Centro de Alterações.
  2. Selecione Administração do Sistema > Provedores de JNDI.
  3. Na página Resumo de Provedores de JNDI, selecione um Provedor de JNDI. Também é possível procurar um Provedor de JNDI específico usando a opção Pesquisar. Consulte Exibindo Provedores de JNDI.
  4. Clique no ícone Deletar no campo Opções do Provedor de JNDI que você deseja deletar. Um ícone Advertência de Deleção será exibido quando outros recursos referenciarem este recurso. É possível deletar o recurso com uma confirmação de advertência. Isso pode resultar em conflitos devido a referências não resolvidas referentes ao recurso deletado.
  5. Para encerrar a sessão e implantar a configuração no run-time, clique em Ativar em Centro de Alterações.
