Editando Provedores de JNDI

Use esta página para exibir os detalhes de um Provedor de JNDI e editar a configuração, se necessário. Além disso, é possível redefinir o cache do Provedor de JNDI e testar a conexão com o Provedor de JNDI.

  1. Se ainda não tiver feito isso, clique em Criar para criar uma nova sessão ou clique em Editar para informar uma sessão existente. Consulte Usando o Centro de Alterações.
  2. Localize o Provedor de JNDI. Consulte Exibindo Provedores de JNDI.
  3. Clique no nome do Provedor de JNDI.

    4. A página Exibir Provedor de JNDI - Configuração exibe as informações do Provedor de JNDI mostradas na Tabela 30-2.

    Tabela 30-2 - Informações do Provedor de JNDI
    Propriedade
    Descrição
    Última Modificação
    O usuário que criou ou modificou este recurso do Provedor de JNDI ou o importou para a configuração.
    Última Modificação em
    A data e hora em que o usuário criou ou modificou este recurso de Provedor de JNDI ou o importou para a configuração. Clique no link da data e hora para exibir o histórico de alterações deste recurso. Consulte a página Exibir Histórico de Alteração.
    Referências
    O número de objetos referenciados por este Provedor de JNDI. Se tais referências existirem, clique no link numérico para exibir uma lista das referências. Consulte Exibindo Referências a Recursos.
    Referenciado por
    O número de objetos que referenciam esse Provedor de JNDI. Se tais referências existirem, clique no link numérico para exibir uma lista dos objetos. Consulte Exibindo Referências a Recursos.
    Descrição
    Uma descrição deste Provedor de JNDI.
    Cache de JNDI
    Ativado ou Desativado. Quando ativado, o contexto e os objetos de JDNI são armazenados no cache localmente, o que melhora o desempenho durante uma pesquisa de objeto. A Oracle recomenda que você mantenha o cache de JNDI ativado.
    URL do Provedor
    O URL para o Provedor de JNDI.
    Timeout da Solicitação de JNDI
    O timeout da solicitação de JNDI em milissegundos. O default zero (0) significa que não há timeout.
    Nome do Usuário e senha
    O nome de usuário usado para acessar um Provedor de JNDI seguro, com a capacidade de alterar a senha.

  4. Clique em Editar para exibir a página Editar Provedor de JNDI - [provider-name] .

    5. O campo Nome não é editável.

  5. No campo Descrição, você pode editar a breve descrição do Provedor de JNDI.
  6. No campo URL do Provedor, você pode editar o URL do Provedor de JNDI.
  7. É possível editar os campos Nome de Usuário e Senha.

    8. Se o Provedor de JNDI foi configurado com um nome de usuário e senha, então o Nome de Usuário não será editável. É possível editar os campos Nova Senha e Confirmar Senha. Se a JNDI não foi configurada com um nome de usuário e senha, então o campo Nome de Usuário será editável e será possível informar um novo nome de usuário.

  8. Clique em Salvar para confirmar as atualizações na sessão atual.
  9. Para encerrar a sessão e implantar a configuração no run-time, clique em Ativar em Centro de Alterações.
