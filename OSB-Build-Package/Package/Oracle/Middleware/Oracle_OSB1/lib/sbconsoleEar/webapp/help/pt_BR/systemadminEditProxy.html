Editando Servidores Proxy

Use esta página para exibir os detalhes de um Servidor Proxy e editar a configuração, se necessário.

  1. Se ainda não tiver feito isso, clique em Criar para criar uma nova sessão ou clique em Editar para informar uma sessão existente. Consulte Usando o Centro de Alterações.
  2. Localize o Servidor Proxy. Consulte Localizando Servidores Proxy.
  3. Clique no nome do Servidor Proxy.

    4. A página Exibir Servidores Proxy - Configuração exibe as informações mostradas na Tabela 30-6.

    Tabela 30-6 - Informações do Servidor Proxy
    Propriedade
    Descrição
    Última Modificação
    O usuário que criou ou modificou este recurso do Servidor Proxy.
    Última Modificação em
    A data e hora em que o usuário criou ou modificou este recurso do Servidor Proxy. Clique no link da data e hora para exibir o histórico de alterações deste recurso. Consulte a página Exibir Histórico de Alteração.
    Referências
    O número de objetos que este Servidor Proxy referencia. Se tais referências existirem, clique no link numérico para exibir uma lista das referências. Consulte Exibindo Referências a Recursos.
    Referenciado por
    O número de objetos que referenciam esse Servidor Proxy. Se tais referências existirem, clique no link numérico para exibir uma lista dos objetos. Consulte Exibindo Referências a Recursos.
    Descrição
    A descrição deste Servidor Proxy, se fornecida.
    Parâmetros do Host-Porta
    Detalhes sobre cada Servidor Proxy no recurso do Servidor Proxy, incluindo as seguintes informações:
    • Host do Servidor: Informe o nome do host ou endereço IP do Servidor Proxy.

      • O nome do Host do Servidor para o servidor proxy do Oracle Service Bus deve ser idêntico ao nome do host do servidor proxy real.

    • Porta de Texto sem Formatação: o número da porta de texto sem formatação do Servidor Proxy.
    • Porta SSL: o número da porta SSL do Servidor Proxy.
    Nome do Usuário
    O nome do usuário usado para autenticação de proxy.

  4. Clique em Editar para exibir a página Editar Servidor Proxy - [server-name] .

    5. O campo Nome não é editável.

  5. No campo Descrição, você pode editar a breve descrição do Servidor Proxy.
  6. Na seção Parâmetros do Host-Porta, é possível adicionar um novo Servidor Proxy especificando as seguintes informações:
    1. No campo Host do Servidor, informe o nome do host ou endereço IP do Servidor Proxy. Esse campo é obrigatório.

      2. O nome do Host do Servidor para o servidor proxy do Oracle Service Bus deve ser idêntico ao nome do host do servidor proxy real.

    2. No campo Porta de Texto sem Formatação, informe o número da porta de texto sem formatação do Servidor Proxy.
    3. No campo Porta SSL, informe o número da porta SSL do Servidor Proxy. Você deve informar um texto sem formatação ou um número de porta SSL.
    4. Clique em Adicionar.

      5. É possível configurar vários Servidores Proxy para cada recurso do Servidor Proxy. Isso permite que o Oracle Service Bus execute balanceamento de carga e ofereça recursos de tolerância a falhas para o recurso do Servidor Proxy.

      Também é possível editar ou deletar uma configuração existente do Servidor Proxy fazendo o seguinte:

    • Clique no ícone Editar no campo Opções do Servidor Proxy que deseja editar. Edite o campo Host do Servidor, o campo Porta de Texto sem Formatação, e o campo Porta SSL e clique em Atualizar.
    • Clique no ícone Deletar no campo Opções do Servidor Proxy que você deseja deletar.
  7. É possível editar os campos Nome de Usuário e Senha, usados para autenticação de proxy.
  8. Clique em Salvar para confirmar as atualizações na sessão atual.
  9. Para encerrar a sessão e implantar a configuração no run-time, clique em Ativar em Centro de Alterações.
