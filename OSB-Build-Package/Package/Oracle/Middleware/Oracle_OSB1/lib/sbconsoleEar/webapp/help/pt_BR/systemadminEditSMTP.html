Editando Servidores SMTP

Use esta página para exibir os detalhes de um Servidor SMTP e editar a configuração, se necessário.

  1. Se ainda não tiver feito isso, clique em Criar para criar uma nova sessão ou clique em Editar para informar uma sessão existente. Consulte Usando o Centro de Alterações.
  2. Localize o Servidor SMTP. Consulte Localizando Servidores SMTP.
  3. Clique no nome do Servidor SMTP.

    4. A página Exibir Servidores SMNP - Configuração exibe as informações mostradas na Tabela 30-4.

    Tabela 30-4 - Informações do Servidor SMTP
    Propriedade
    Descrição
    Última Modificação
    O usuário que criou ou modificou este recurso do Servidor SMTP.
    Última Modificação em
    A data e hora em que o usuário criou ou modificou este recurso do Servidor SMTP. Clique no link da data e hora para exibir o histórico de alterações deste recurso. Consulte a página Exibir Histórico de Alteração.
    Referências
    O número de objetos que este Servidor SMTP referencia. Se tais referências existirem, clique no link numérico para exibir uma lista das referências. Consulte Exibindo Referências a Recursos.
    Referenciado por
    O número de objetos que referenciam esse Servidor SMTP. Se tais referências existirem, clique no link numérico para exibir uma lista dos objetos. Consulte Exibindo Referências a Recursos.
    Descrição
    A descrição deste Servidor SMTP, se fornecida.
    URL do Servidor
    O URL que aponta para o local do Servidor SMTP.
    Número da Porta
    O número da porta para este Servidor SMTP. É a porta 25 por default.
    Nome do Usuário
    O nome de usuário usado para acessar um Servidor SMTP seguro.

  4. Clique em Editar para exibir a página Editar Servidor SMTP - [server-name] .

    5. O campo Nome não é editável.

  5. No campo Descrição, você pode editar a breve descrição do Servidor SMTP.
  6. No campo URL do Servidor, é possível editar o URL do Servidor SMTP.
  7. No campo Número da Porta, é possível editar o número da porta.
  8. É possível editar os campos Nome de Usuário e Senha.
  9. Clique em Salvar para confirmar as atualizações na sessão atual.
  10. Para encerrar a sessão e implantar a configuração no run-time, clique em Ativar em Centro de Alterações.

Se você renomear um recurso SMTP, o novo nome será atualizado automaticamente nos serviços de negócios de e-mail que referenciam o recurso SMTP no URI do ponto final.
