Executando Arquivos de Personalização

Use esta página para executar um arquivo XML de personalização que tenha sido salvo anteriormente no sistema. Arquivos de personalização são uma forma conveniente de modificar seu ambiente durante a implantação, desde o desenvolvimento até a preparação, desde a preparação até a produção ou durante o tempo de design.

Você deve estar em uma sessão para executar um arquivo de personalização.

  1. Se ainda não tiver feito isso, clique em Criar para criar uma nova sessão ou clique em Editar para informar uma sessão existente. Consulte Usando o Centro de Alterações.
  2. Selecione Administração do Sistema > Executar Arquivo de Personalização.
  3. Selecione um arquivo de personalização para executar.
    1. No campo Nome do Arquivo, clique em Procurar para localizar o diretório em que o arquivo de personalização está armazenado.
    2. Selecione o arquivo XML, depois clique em Abrir.
  4. Clique em Próximo. Os detalhes de personalização a seguir são exibidos, mostrados na Tabela 31-2.

    5. Tabela 31-2 - Detalhes da Personalização
    Propriedade
    Descrição
    Operação
    • Localizar e Substituir - modifique os valores de ambiente executando substituições de string.
    • Mapear Referências - mapeia as referências existentes encontradas nos recursos para outras referências.
    • Definir Valores de Ambiente - modifique os valores de ambiente para os recursos.
    Aplicar a
    Nome do serviço, projeto, pasta ou recurso a ser personalizado.
    O nome é um link para a página Detalhes da Configuração do serviço ou recurso, ou do projeto ou pasta no Project Explorer.
    Descrição
    Uma descrição de cada personalização.

  5. Para personalizar apenas os recursos, serviços, projetos ou pastas alterados na sessão atual, marque a caixa de seleção Apenas Itens Alterados na Sessão Atual.

    6. Se você limitar as personalizações apenas aos recursos modificados na sessão atual, a coluna Aplicar a será atualizada com os recursos modificados na sessão atual. Se você limitar as personalizações de um projeto ou pasta, então os recursos modificados na sessão atual dentro desse projeto ou pasta serão exibidos na coluna Aplicar a.

  6. Clique em Executar para confirmar as atualizações na sessão atual.

    7. Para exibir detalhes de personalização, na sessão, selecione Exibir Alterações no Centro de Alterações e clique na tarefa Personalização. Consulte Exibindo Alterações de Configuração.

  7. Para encerrar a sessão e implantar a configuração no run-time, clique em Ativar em Centro de Alterações.
