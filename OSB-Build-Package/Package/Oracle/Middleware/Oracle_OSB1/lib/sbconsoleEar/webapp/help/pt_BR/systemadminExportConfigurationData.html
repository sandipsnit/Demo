Exportando Recursos

Use a página Exportar Recursos para Exportar Projetos ou Exportar Recursos. Se você optar por exportar um projeto inteiro, um ou mais projetos poderão ser selecionados para exportação. Você ainda pode expandir os projetos para ver as pastas e os recursos, mas só é permitida seleção no projeto. Além disso, Incluir Dependências não pode ser selecionado.

Não é possível exportar projetos e recursos se estiver trabalhando dentro ou fora de uma sessão. Se você exportar em uma sessão, os recursos serão recursos da sessão e a configuração poderá ficar incompleta ou apresentar conflitos. Se você exportar fora de uma sessão, os recursos serão recursos principais ativados.

O Oracle Service Bus não pode exportar os usuários, grupos ou atribuições que você cria no módulo Configuração da Segurança da Console do Oracle Service Bus Administration. Também não pode exportar mapas de credencial ou outros dados do provedor de segurança que você cria na Console de Administração do WebLogic Server. Em vez disso, use a Console de Administração do WebLogic Server para exportar esses dados. Consulte "Migrating Security Data" em Oracle Fusion Middleware Securing Oracle WebLogic Server em http://www.oracle.com/pls/as111170/lookup?id=SECMG319.

  1. Selecione Administração do Sistema > Exportar Recursos.

    2. A página Exportar Recursos exibe a lista de objetos na configuração. O nome, o tipo, a data da última modificação e as referências de cada objeto são exibidos.

    Para serviços, o último timestamp modificado reflete as modificações feitas na definição do serviço, além da configuração do transporte ou do pipeline do serviço. No entanto, isso não reflete as alterações feitas nos alertas definidos para esse serviço.

  2. Selecione se Exportar Projetos ou Exportar Recursos.
    3. Observação: A exportação de projetos pode causar deleção do recurso quando você importa o arquivo JAR do projeto inteiro. Para obter mais informações, consulte Importando Recursos.
  3. Selecione os projetos ou recursos que você deseja exportar. Se você selecionou Exportar Projetos, não poderá desmarcar as caixas de seleção de recursos individuais.
    1. Expanda as pastas do projeto. O nome, o tipo, a hora da última modificação e as referências a cada recurso contido no projeto serão exibidas.
    2. Marque as caixas de seleção associadas aos projetos ou recursos que você deseja exportar.
    3. Desmarque as caixas de seleção associadas aos projetos ou recursos que você não deseja exportar.
  4. Se você selecionou Exportar Recursos, marque ou desmarque a caixa de seleção Incluir Dependências. Use esta opção para exportar qualquer outro recurso referenciado por este recurso.
  5. Clique em Exportar para criar um arquivo JAR e exportá-lo.

Depois de finalizar

Se aplicável, continue em Especificando as Definições de Segurança de Exportação.
