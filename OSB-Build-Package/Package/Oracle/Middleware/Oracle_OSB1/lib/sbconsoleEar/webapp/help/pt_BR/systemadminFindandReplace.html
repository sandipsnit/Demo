Localizando e Substituindo Valores de Ambiente

Use a página Localizar e Substituir para procurar valores de ambiente que apresentem diferença entre os domínios. Valores de ambiente são determinados campos predefinidos nos dados de configuração cujos valores muito provavelmente mudarão quando você mover sua configuração de um domínio para outro (por exemplo, de teste para produção). Os valores de ambiente representam entidades tais como URLs, URIs, nomes de arquivos, diretório, servidor, e-mails e assim por diante. Além disso, eles podem ser encontrados em destinos de alerta, serviços de proxy, serviços de negócios, recursos de Servidor SMTP e de Provedor de JNDI e entradas de Registro UDDI .

O comportamento desta página varia dependendo se você está ou não em uma sessão. Se você estiver em uma sessão, poderá localizar e substituir valores de ambiente. No entanto, caso esteja fora de uma sessão. você só poderá localizar valores de ambiente; Substituir Tudo está desativado.

  1. Se ainda não tiver feito isso, clique em Criar para criar uma nova sessão ou clique em Editar para informar uma sessão existente. Consulte Usando o Centro de Alterações.
  2. Selecione Administração do Sistema > Localizar e Substituir.
  3. No campo Localizar Valor, informe o valor de ambiente que você deseja localizar.
  4. Para exibir uma lista de valores de ambiente na configuração que contêm o valor informado, clique em Localizar.

    5. Uma lista de valores relevantes é exibida, incluindo as colunas Proprietário, Tipo e Valor de Ambiente.

  5. Para localizar apenas os itens alterados na sessão atual, marque a caixa de seleção Apenas Itens Alterados na Sessão Atual.
  6. Selecione o tipo de variável pela qual procurar, na lista drop-down Localizar no Tipo de Variável, descrita na Tabela 31-1.

  7. Para localizar variáveis de ambiente localizadas em um projeto específico, selecione o nome do projeto na lista drop-down Localizado no Projeto.
  8. No campo Substituir por, informe o novo valor de ambiente.
  9. Para substituir o valor de ambiente original pelo novo valor, clique em Substituir Tudo.

    10. Todas as ocorrências do valor de ambiente informadas no campo Localizar Valor serão substituídas pelo valor de ambiente informado no campo Substituir por na sessão atual.

    Observação: Determinados valores de ambiente são objetos XML complexos que não podem ser localizados e substituídos usando a Console do Oracle Service Bus Administration. No entanto, ainda é possível definir esses valores de ambiente diretamente usando o ALSBConfigurationMBean de um script. Para obter informações detalhadas sobre o ALSBConfigurationMBean, consulte a Referência da API do Oracle Service Bus. Além de defini-los por meio da API, você pode definir valores de ambiente de tipo complexo usando arquivos de personalização. Consulte Executando Arquivos de Personalização.
  10. Para encerrar a sessão e implantar a configuração no run-time, clique em Ativar em Centro de Alterações.
