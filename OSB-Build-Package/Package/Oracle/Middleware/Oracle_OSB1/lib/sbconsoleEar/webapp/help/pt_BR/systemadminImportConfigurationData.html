Importando Recursos

Use a página Importar Recursos para importar um arquivo JAR de projeto inteiro ou um arquivo JAR de recurso que tenha sido exportado anteriormente de outro domínio do Oracle Service Bus. Você pode optar por importar apenas um subconjunto dos dados exportados. Se o recurso já existir no sistema de importação, ele será atualizado. Caso contrário, ele será criado. Recursos só são programados para deleção quando o JAR sendo importado é um JAR de projeto inteiro e há recursos localizados no mesmo projeto no sistema de importação que não estão presentes no arquivo JAR importado. Ele não deletará recursos que estejam localizados em outros projetos.

É possível importar recursos no projeto mesmo quando foram exportados no projeto. Por exemplo, mesmo que recursos do sistema tenham sido exportados para um arquivo JAR do projeto inteiro, é possível desmarcá-los durante a importação.

Não é possível exportar usuários, grupos, atribuições ou certificados ao exportar uma configuração. Portanto, você deve criar esses objetos novamente ao importar uma configuração exportada.

Observação: Para obter informações sobre a importação de recursos que não fazem parte de uma configuração do Oracle Service Bus, consulte Carregando Recursos de um Arquivo Zip e Carregando Recursos de um URL.
  1. Se ainda não tiver feito isso, clique em Criar para criar uma nova sessão ou clique em Editar para informar uma sessão existente. Consulte Usando o Centro de Alterações.
  2. Selecione Administração do Sistema > Importar Recursos.
  3. Selecione um arquivo JAR do projeto ou recurso exportado.
    1. No campo Nome do Arquivo, clique em Procurar para localizar o diretório em que o arquivo JAR está armazenado.
    2. Selecione o arquivo JAR, depois clique em Abrir.
  4. Clique em Próximo. Continue em Selecionando Projetos e Recursos para Importar.
