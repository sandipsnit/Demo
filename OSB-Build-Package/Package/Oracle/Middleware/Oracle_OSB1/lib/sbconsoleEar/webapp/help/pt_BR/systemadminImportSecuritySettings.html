Especificando as Definições de Segurança de Importação

Se o arquivo JAR contive recursos com dados do nome de usuário e senha que foram criptografados durante a exportação, a página Importar Recursos - Definições de Segurança será exibida.

  1. Informe a senha que foi usada para criptografar os dados (consulte Exportando Recursos).

    2. Se você não souber a senha que foi usada para criptografar os dados do recurso, clique no botão Anterior e desmarque a caixa de seleção que está ao lado do recurso. É impossível importar todos os recursos não-criptografados sem saber a senha.

  2. Clique em Concluir Importação.

    3. A janela pop-up Processando Dados da Configuração é exibida. Você deve aguardar o fechamento dessa janela antes de fazer qualquer coisa na Console do Oracle Service Bus Administration. A janela será fechada automaticamente quando o sistema terminar de importar os objetos. A página Importar Recursos é exibida, incluindo uma mensagem de que você implantou os dados com sucesso.
