Publicando Serviços de Proxy em um Registro UDDI

É possível publicar o serviço em um registro e disponibilizá-lo para descoberta e uso por outras organizações. Todos os serviços de proxy desenvolvidos no Oracle Service Bus podem ser publicados em um registro UDDI. É possível selecionar a entidade de negócios em que você deseja publicar o serviço, e você pode publicar diversos serviços por vez.

Para poder publicar em um registro, você deve ter uma conta com esse registro. O Oracle Service Bus suporta interoperabilidade com registros UDDI compatíveis com a versão 3.0. A configuração descrita nesta seção considera que você esteja usando o Oracle Service Registry.

Para obter informações sobre a configuração de uma conta do usuário no Oracle Service Registry, consulte a documentação do produto Oracle Service Registry, acessível em http://www.oracle.com/technology/products/soa/registry/index.html.

O cancelamento da publicação de um registro é feito na instalação do Oracle Service Registry.

  1. Selecione Administração do Sistema > Publicar no UDDI.

    2. Só é possível publicar quando você não está em uma sessão. Saia da sessão para acessar a lista de registros. Todos os registros aos quais você tem acesso são exibidos na lista drop-down de registros selecionados.

  2. Na lista drop-down Nome do Registro para Publicação, selecione o nome do registro para o qual você deseja publicar, depois clique em Próximo.

    3. Quando um único registro é configurado no Oracle Service Bus, a página de seleção de registro inicial não é exibida pois esse é o registro default. A página de seleção de registro só é exibida quando dois ou mais registros são definidos.

    A página Publicar no UDDI: Selecionar Serviços individuais e Publicar é exibida, mostrando uma lista de serviços de proxy do Oracle Service Bus e seus locais na pasta do projeto. Começando com a pasta de nível do projeto mais superior, os detalhes do nome, tipo e descrição são mostrados para todos os serviços.

  3. Selecione o(s) serviço(s) que você deseja publicar.
    1. Expanda a pasta Projeto para ver os serviços de proxy definidos. Por default, a pasta e seu conteúdo são marcados para publicação.
    2. É possível selecionar itens individuais para publicar.
  4. Na lista drop-down Publicar Serviços na Entidade de Negócios, selecione a entidade de negócios em que você deseja adicionar o serviço no registro. Essa é a entidade de negócios em que o serviço será classificado no registro.
  5. Clique em Publicar para publicar os serviços.

    6. Uma mensagem de confirmação é exibida, indicando que o serviço foi publicado com sucesso.

  6. Clique em Publicar Outro para retornar para a página Publicar no UDDI: selecionar registro.
