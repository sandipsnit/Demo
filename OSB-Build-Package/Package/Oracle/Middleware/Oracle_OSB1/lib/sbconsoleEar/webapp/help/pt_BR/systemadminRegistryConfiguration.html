Exibindo Registros UDDI Configurados

Use esta página para designar um registro UDDI default (consulte Definindo uma Configuração UDDI Default) e para exibir registros UDDI. Além disso, você pode procurar registros que foram configurados anteriormente.

  1. O painel Registros UDDI, informe o nome do registro que você deseja localizar no campo Nome ou informe caracteres curingas; use * para vários caracteres e ? para caracteres simples. A pesquisa retorna todos os registros que satisfazem os critérios do nome.
  2. Clique em Pesquisar.
  3. Clique em Exibir Tudo para remover os filtros de pesquisa e exibir todos os registros configurados para funcionar com o Oracle Service Bus.

    4. Tabela 29-1 exibe definições de configuração do registro UDDI.

    Tabela 29-1 Definições de Configuração do Registro UDDI 
    Propriedade
    Descrição
    Nome do Registro
    O nome do registro. O nome designado a um registro quando é criado pela primeira vez. Não é possível editar o nome de um registro depois que a entrada é salva.
    URL de Consulta
    O URL do ponto final da API de Consulta usado para localizar e importar serviços.
    URL de Publicação
    O URL do ponto final da API de Publicação usado para publicar serviços.
    URL de Segurança
    O URL do ponto final da API de Segurança usado para obter um token de autenticação para que você possa publicar no registro.
    URL de Assinatura
    O URL do ponto final da API de Assinatura usado para se inscrever em alterações de registro, criando uma assinatura de registro e atendendo a alterações em serviços importados.
    Nome do Usuário
    O nome de usuário necessário para autenticação do usuário.
    Senha (Confirmar Senha)
    A senha necessária para autenticação do usuário.
    Carregar tModels no registro
    Carrega os tModels no registro selecionado. Esta opção só precisa ser selecionada uma vez por registro.
    Ativar Importação Automática
    Sincroniza serviços automaticamente com o registro UDDI.
