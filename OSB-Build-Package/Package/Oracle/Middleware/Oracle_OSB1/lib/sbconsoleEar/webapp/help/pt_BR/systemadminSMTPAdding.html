Adicionando Servidores SMTP

Use a página Resumo de Servidores SMTP para adicionar e configurar recursos do Servidor SMTP e disponibilizá-los no Oracle Service Bus como um recurso do sistema. Você deve estar em uma sessão ativa para configurar ou reconfigurar recursos do Servidor SMTP.

  1. Se ainda não tiver feito isso, clique em Criar para criar uma nova sessão ou clique em Editar para informar uma sessão existente. Consulte Usando o Centro de Alterações.
  2. Selecione Administração do Sistema > Servidores SMTP.
  3. Clique em Adicionar.
  4. No campo Nome, informe um nome para o recurso Servidor SMTP. Esse campo é obrigatório.

    Siga as Restrições para Nomeação de Recursos para obter orientações em relação a nomes.
  5. No campo Descrição, informe uma breve descrição para o recurso do Servidor SMTP.
  6. No campo URL do Servidor, informe o URL que aponta para o Servidor SMTP. Esse campo é obrigatório.
  7. No campo Número da Porta, informe um número de porta para o Servidor SMTP (a porta default é 25). Esse campo é obrigatório.
  8. Se o acesso ao Servidor SMTP de destino exigir um nome de usuário e senha, informe um nome de usuário no campo Nome do Usuário e a senha associada nos campos Senha e Confirmar Senha.

    9. Estes campos são opcionais e obrigatórios apenas se o Servidor SMTP estiver protegido.

  9. Clique em Salvar para criar e salvar o recurso do Servidor SMTP na sessão atual.
  10. Para encerrar a sessão e implantar a configuração no run-time, clique em Ativar em Centro de Alterações.
