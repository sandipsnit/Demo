Configurando um Servidor SMTP Default

É possível designar um dos Servidores SMTP configurados como o servidor default para o domínio. Consulte Adicionando Servidores SMTP.

  1. Se ainda não tiver feito isso, clique em Criar para criar uma nova sessão ou clique em Editar para informar uma sessão existente. Consulte Usando o Centro de Alterações.
  2. Selecione Administração do Sistema > Servidores SMTP.
  3. No painel Servidor SMTP Default, clique em Selecionar Default.
  4. Na lista drop-down Servidor SMTP Default, selecione um Servidor SMTP.
  5. Clique em Tornar Default.

    6. A página Resumo de Servidores SMTP é exibida com o Servidor SMTP selecionado mostrado como o Servidor SMTP default para o domínio.

  6. Para encerrar a sessão e implantar a configuração no run-time, clique em Ativar em Centro de Alterações.
