Exibindo Provedores de JNDI

Use esta página para exibir uma lista de Provedores de JNDI e as informações mostradas na Tabela 30-1.

Além disso, é possível procurar Provedores de JNDI que tenha configurado anteriormente.

  1. Para restringir o número de itens na lista ou para localizar Provedores de JNDI específicos:
    • Clique no ícone Abrir para exibir filtros de pesquisa adicionais. Se você especificar vários critérios de pesquisa, apenas os resultados que correspondem a todos os critérios serão retornados.
    • Informe o nome do Provedor de JNDI que você deseja localizar no campo Nome e clique em Pesquisar para procurar uma entrada específica.

      • Você pode informar caracteres curingas (? para um único caractere; * para vários caracteres) para executar uma pesquisa mais geral.

    • Informe o URL do Provedor do Provedor de JNDI que você deseja localizar e clique em Pesquisar.
  2. Clique em Exibir Tudo para remover os filtros de pesquisa e exibir todos os recursos do Provedor de JNDI configurados para funcionar com o Oracle Service Bus.

    3. Tabela 30-1 exibe definições de configuração do registro UDDI.

    Tabela 30-1 - Informações do Provedor de JNDI
    Propriedade
    Descrição
    Nome
    O nome exclusivo designado ao Provedor de JNDI. O nome é um link para a página Exibir Provedor de JNDI - Configuração. Consulte Editando Provedores de JNDI.
    URL do Provedor
    O URL para o Provedor de JNDI. Consulte Adicionando Provedores de JNDI.
    Cache de JNDI
    Ativado ou Desativado. Quando ativado, o contexto e os objetos de JDNI são armazenados no cache localmente, o que melhora o desempenho durante uma pesquisa de objeto. A Oracle recomenda que você mantenha o cache de JNDI ativado.
    Ações
    • Clique no ícone Redefinir para redefinir o contexto de JNDI para descartar a conexão de JNDI e objetos armazenados no cache localmente.
    • Clique no ícone Iniciar Console de Teste para testar se o Provedor de JNDI pode estabelecer uma conexão com sucesso e obter um contexto de JNDI.
    Opções
    • Clique no ícone Renomear para renomear o Provedor de JNDI.
    • Clique no ícone Deletar para deletar um Provedor de JNDI específico. Consulte Deletando Provedores de JNDI
