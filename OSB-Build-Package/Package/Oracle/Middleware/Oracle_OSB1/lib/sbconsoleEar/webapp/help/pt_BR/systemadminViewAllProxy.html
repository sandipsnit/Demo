Exibindo Servidores Proxy

Use esta página para exibir uma lista de Servidores Proxy e as informações mostradas na Tabela 30-5.

Os recursos do Servidor Proxy são usados durante a configuração do transporte HTTP para serviços de negócios. Para obter mais informações, consulte a Página Configuração do Transporte HTTP.

Tabela 30-5 exibe informações do Servidor Proxy.

Tabela 30-5 Informações do Servidor Proxy 
Propriedade
Descrição
Nome
O nome exclusivo designado a este recurso do Servidor Proxy. O nome é um link para a página Exibir Servidor Proxy - Configuração. Consulte Editando Servidores Proxy.
Opções
  • Clique no ícone Renomear para renomear este recurso do Servidor Proxy.
  • Clique no ícone Deletar para deletar um Servidor Proxy específico. Consulte Deletando Servidores Proxy.
