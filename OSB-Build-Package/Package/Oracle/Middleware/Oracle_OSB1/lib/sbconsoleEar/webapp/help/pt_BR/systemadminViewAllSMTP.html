Exibindo Servidores SMTP

Use esta página para designar um Servidor SMTP default (consulte Configurando um Servidor SMTP Default) e para exibir uma lista de Servidores SMTP.

Os recursos do Servidor SMTP são usados durante a configuração de recursos do destino de alerta e serviços de negócios baseados no transporte de E-mail. Consulte Adicionando Destinos de Alerta e Página Criar/Editar um Serviço de Negócios - Configuração do Transporte de E-mail.

Tabela 30-3 exibe informações do Servidor SMTP.

Tabela 30-3 - Informações do Servidor SMTP
Propriedade
Descrição
Nome
O nome exclusivo designado a este recurso do Servidor SMTP. O nome é um link para a página Exibir Servidor SMTP - Configuração. Consulte Editando Servidores SMTP.
URL do Servidor
O URL que aponta para o servidor SMTP. Consulte Adicionando Servidores SMTP.
Opções
  • Clique no ícone Renomear para renomear este recurso do Servidor SMTP.
  • Clique no ícone Deletar para deletar um Servidor SMTP específico. Consulte Deletando Servidores SMTP.
