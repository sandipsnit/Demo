Entendendo Como o Run-time Usa as Definições de Transporte na Console de Teste

Configurando Dados do Teste de Serviços Proxy e Configurando Dados do Teste de Serviços de Negócios descrevem como você configura os valores dos cabeçalhos de transporte, metadados de transporte e dados de segurança relacionados ao transporte para solicitações de saída ao testar serviços de proxy ou de negócios na Console de Teste. No entanto, algumas especificações que você possa fazer na Console de Teste não são respeitadas no run-time. Ou seja, os valores de determinados cabeçalhos ou metadados são substituídos ou ignorados pelo Oracle Service Bus no runtime quando o teste é executado. Os cabeçalhos e metadados para os quais existem limitações estão descritos na Tabela 32-10.
