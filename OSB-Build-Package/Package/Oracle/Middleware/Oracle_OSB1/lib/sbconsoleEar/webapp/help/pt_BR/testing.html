Console de Teste

A Console de Teste do Oracle Service Bus é um ambiente de teste baseado em browser usado para validar e testar o design do sistema. Você pode configurar o objeto do teste (serviço de proxy, serviço de negócios ou recurso), executar o teste e exibir os resultados na console de teste. Em algumas instâncias, é possível rastrear pelo código e examinar o estado da mensagem em pontos de rastreamento específicos. Para saber mais sobre a Console de Teste, consulte "Using the Test Console" em Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus em http://www.oracle.com/pls/as111170/lookup?id=OSBAG357.

Apenas usuários com as atribuições IntegrationAdmin e IntegrationDeployer podem usar a Console de Teste. Para obter mais informações sobre atribuições, consulte "Roles in Oracle Service Bus" em Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus em http://www.oracle.com/pls/as111170/lookup?id=OSBAG463.

Observação: Se receber uma mensagem de erro informando que o serviço de Console de Teste não está em execução, experimente definir o endereço de listen do servidor Admin com um endereço válido específico, como localhost. No Oracle WebLogic Server Console, vá até Ambiente > Servidores > admin_server_name > Configuração > Geral para definir o Endereço de Listen.
