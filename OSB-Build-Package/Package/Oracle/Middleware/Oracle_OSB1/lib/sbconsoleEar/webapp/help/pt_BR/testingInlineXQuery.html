Executando Testes XQuery

É possível editar e testar uma ação no fluxo de mensagens usando os seguintes Editores: Editor de Expressão XQuery, Editor de Condição XQuery e Editor de Expressão XPath. O teste demora o mesmo tempo para os Editores de Expressão e de Condição XQuery. No entanto, o cenário é diferente para o Editor de Expressão XPath, porque utiliza apenas uma entrada.

Observação: Você deve desativar os bloqueadores de pop-up no browser para que o teste XQuery funcione. Se você tiver barras de ferramentas no browser do IE, pode ser necessário desativar os bloqueadores de pop-up no menu Opções e para todas as barras de ferramentas configuradas para bloqueá-los.
