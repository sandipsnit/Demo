Exibindo Resultados do Teste de Serviços de Proxy

A Tabela 32-2 descreve os resultados do teste do serviço de proxy. O rastreamento só será ativado se você tiver selecionado as opções Chamada Direta e Incluir Rastreamento.

Tabela 32-2 - Resultados do Teste de Serviços de Proxy
Seção
Descrição
Nome do Serviço de Proxy
O nome do serviço de proxy que está sendo testado.
Ações da Console de Teste
Voltar exibe a página do browser anterior.
Fechar fecha a Console de Teste.
Documento de Solicitação
A mensagem de solicitação enviada para o serviço de proxy pela Console de Teste.
Esta seção estará inicialmente recolhida se a Console de Teste não tiver modificado a mensagem da solicitação. Ela estará inicialmente expandida para serviços SOAP configurados usando a guia Form ou se a Segurança de Web Services tiver sido aplicada.
Se a segurança de Web Service tiver sido aplicada, esta seção conterá duas mensagens SOAP: a primeira mensagem é a mensagem sem formatação; a segunda é a mensagem SOAP protegida.
Documento de Resposta
A resposta da mensagem.
Para um serviço SOAP com uma resposta de Segurança de Web Services, esta seção contém duas mensagens SOAP. A primeira mensagem SOAP é a mensagem protegida conforme recebida pelo cliente. A segunda mensagem SOAP é a mensagem de texto sem formatação correspondente.
Metadados de Resposta
Os metadados retornaram com a resposta da mensagem.
Rastreando Serviços de Proxy
O rastreamento mostra o estado da mensagem à medida que passa pelo sistema. Quando a opção Chamada Direta não é selecionada, o rastreamento não é executado. Para obter informações sobre rastreamento, consulte Rastreando Serviços de Proxy.
