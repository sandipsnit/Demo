Rastreando Serviços de Proxy

O rastreamento é ativado quando você testa serviços de proxy usando uma chamada direta. A caixa de seleção Incluir Rastreamento é selecionada por default com a opção Chamada Direta. Se você não deseja rastreamento, desmarque a caixa de seleção Incluir Rastreamento. Quando você ativa o rastreamento, os resultados dos teste incluem os detalhes do mesmo. Use o rastreamento para rastrear problemas no sistema e para isolá-los para correção. As informações sobre rastreamento mostram o caminho seguido pela solicitação pelos caminhos de solicitação e de resposta no código.

  1. Clique em Ativar em Centro de Alterações para ativar o recurso de teste na Console do Oracle Service Bus Administration.
  2. Selecione Browser do Recurso > Serviços de Proxy para exibir a página Resumo de Serviços de Proxy.
  3. Em Ações, clique no ícone Acionar Console de Teste associado ao serviço que você deseja testar. A Console de Teste abre a página Teste do Serviço de Proxy.
  4. Configure os dados de teste para o serviço de proxy. Você deve ter as opções Chamada Direta e Incluir Rastreamento selecionadas para ativar o rastreamento. Consulte Configurando Dados do Teste de Serviços de Proxy.
  5. Clique em Executar. A página Teste do Serviço de Proxy exibirá os resultados do teste do serviço e as informações de rastreamento.
  6. Role para a seção Rastreamento da Chamada.

    7. Esta seção exibe uma representação gráfica do fluxo de mensagens. É possível rastrear a mensagem por meio do serviço e exibir o estado da mensagem nos pontos pré-selecionados no rastreio. Os pontos de rastreio são definidos automaticamente.

  7. Clique no sinal de mais + para expandir o fluxo de mensagens e exibir mais detalhes.

    8. Ao exibir o rastreamento, também é possível exibir o fluxo de mensagens na Console do Oracle Service Bus Administration. Isso ajuda a relacionar o rastreamento aos estágios e ações no fluxo de mensagens. É possível modificar o fluxo de mensagens e executar o rastreamento novamente para exibir a saída.
