Exibindo Resultados do Teste de Serviços de Negócios

A Tabela 32-4 descreve os resultados do teste do serviço de negócios.

Tabela 32-4 - Resultados do Teste de Serviços de Negócios
Seção
Descrição
Nome do Serviço de Negócios
O nome do serviço de negócios.
Ações da Console de Teste
Clique em Voltar para exibir a página do browser anterior.
Clique em Fechar para fechar a janela da Console de Teste.
Documento de Solicitação
A mensagem de solicitação enviada para o serviço de negócios pela Console de Teste.
Esta seção estará inicialmente recolhida se a Console de Teste não tiver modificado a mensagem da solicitação. Ela estará inicialmente expandida para serviços SOAP configurados usando a guia Form ou se a Segurança de Web Services tiver sido aplicada.
Se a segurança de Web Service tiver sido aplicada, esta seção conterá duas mensagens SOAP. A primeira mensagem é a mensagem sem formatação. A segunda é a mensagem SOAP protegida.
Documento de Resposta
A resposta da mensagem.
Para um serviço SOAP com uma resposta de Segurança de Web Services, esta seção conterá duas mensagens SOAP. A primeira mensagem SOAP é a mensagem protegida conforme recebida pelo cliente. A segunda mensagem SOAP é a mensagem de texto sem formatação correspondente.
Metadados de Resposta
Os metadados retornaram com a resposta da mensagem.

Observação: A mensagem SOAP protegida é exibida em pretty print, por exemplo, com espaços em branco extras. Essa mensagem SOAP nem sempre pode ser usada como os dados literais, pois espaços em branco podem afetar a semântica do documento. Por exemplo, assinaturas digitais são sensíveis a espaços em branco e podem ser invalidadas.
