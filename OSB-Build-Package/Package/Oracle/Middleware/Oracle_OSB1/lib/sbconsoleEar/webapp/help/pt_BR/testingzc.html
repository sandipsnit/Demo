Testando Transformações MFL

Um documento MFL (Message Format Language) é um documento XML especializado usado para descrever o layout de dados binários. Os recursos MFL suportam duas transformações. Cada transformação aceita apenas uma entrada e oferece uma única saída: XML para binário ou binário para XML.

Tabela 32-5 descreve as opções de configuração MFL.

Tabela 32-5 - Configurando Opções MFL

O nome do recurso que está sendo testado.
Transformações suportadas
Para selecionar uma transformação específica, selecione o nome da transformação.

Configuração da Transformação MFL
Entrada XML
Obrigatório para transformações de XML em binário:
O esquema XML para o documento MFL pode ser inferido. Um documento XML de amostra é informado automaticamente no campo de texto.
A entrada XML pode ser baseada em arquivo ou em texto. A referência de um arquivo para entrada tem precedência sobre entrada textual. Pesquise e selecione o arquivo que você deseja usar no teste.
Entrada Binária
Obrigatório para transformações de binário em XML:
A entrada binária pode ser baseada em arquivo ou em texto. A referência de um arquivo para entrada tem precedência sobre entrada textual. Pesquise e selecione o arquivo que você deseja usar no teste.

É possível testar o tempo de design ou o run-time.

  1. Clique em Ativar se quiser testar o run-time. Não ative a sessão para testar o tempo de design.
  2. Selecione Browser do Recurso > MFLs para exibir a página Resumo de Arquivos MFL.
  3. Em Ações, clique no ícone Acionar Console de Teste associado ao recurso que você deseja testar. A Console de Teste abre a página Teste do Recurso.
  4. Selecione a transformação que você deseja testar.
  5. Configure os dados de teste para o recurso MFL. Para obter mais informações, consulte a Tabela 32-5.
  6. Clique em Executar. A página Teste do Recurso exibe os resultados.
  7. Para testar novamente, clique em Voltar. É possível fechar a Console de Teste, modificar e testar o recurso novamente.
