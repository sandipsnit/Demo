Usando o Editor de Expressão XPath

Use expressões XPath para selecionar um subconjunto de uma variável de contexto XML. É possível usar a Console de Teste no Editor de Expressão XPath para testar a definição da expressão XPath. Uma expressão XPath utiliza um único documento XML como entrada e gera uma sequência de documentos XML, tipos primitivos, ou ambos, como saída.

Tabela 32-9 descreve as opções de configuração de expressão XPath.

Tabela 32-9 - Configurando Opções de XPath

O tipo de expressão sendo testado.

Variáveis
Esta seção contém um único campo de entrada correspondente ao documento XML com o qual esta expressão XPath está sendo testada.
 
A entrada XML pode ser baseada em arquivo ou em texto. A referência de um arquivo para entrada tem precedência sobre entrada textual. Pesquise e selecione o arquivo que você deseja usar no teste.

  1. Acesse a Console de Teste ao editar uma ação no fluxo de mensagens de um pipeline. Para acessar o Editor de Expressão XPath, consulte Criando e Editando Expressões XQuery e XPath em Linha.
  2. Configure os dados de teste para a expressão XPath na seção Variáveis da página. Para obter mais informações, consulte a Tabela 32-9.
  3. Clique em Executar. A página de teste exibe os resultados.
  4. Ao concluir um teste, você pode usar o botão Voltar para executar um teste novo. Para executar um teste novo depois de fazer alterações na expressão XPath, você deve fechar e reabrir a Console de Teste para que as alterações tenham efeito.
