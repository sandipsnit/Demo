Editando Registros UDDI

  1. Se ainda não tiver feito isso, clique em Criar para criar uma nova sessão ou clique em Editar para informar uma sessão existente. Consulte Usando o Centro de Alterações.
  2. Selecione Administração do Sistema > Registros UDDI.
  3. No painel Registros UDDI, na lista de registros disponíveis, clique no nome do registro que você deseja editar.

    4. Também é possível procurar um registro específico usando a opção Pesquisar. Consulte Exibindo Registros UDDI Configurados.

    A página Configuração do Registro UDDI exibe as propriedades de configuração que definem o registro e as seguintes informações específicas ao registro, mostradas na Tabela 29-2.

    Tabela 29-2 - Informações do Registro UDDI
    Propriedade
    Descrição
    Última Modificação
    O usuário que fez a última alteração de configuração neste recurso de registro UDDI.
    Última Modificação em
    A data e hora em que a última alteração de configuração foi feita. Clique no link da data e hora para exibir o histórico de alterações deste recurso. Consulte a página Exibir Histórico de Alteração.
    Referências
    O número de objetos referenciados por este registro UDDI. Se tais referências existirem, clique no link numérico para exibir uma lista das referências. Consulte Exibindo Referências a Recursos.
    Referenciado por
    O número de objetos que referenciam esse registro UDDI. Se tais referências existirem, clique no link numérico para exibir uma lista dos objetos. Consulte Exibindo Referências a Recursos.
    Descrição
    Uma descrição do registro.

  4. Na parte inferior da página, clique em Editar.
  5. Na página Configuração UDDI - Editar Registro, edite os parâmetros de configuração do registro.

    6. Estas propriedades estão descritas na Tabela 29-1. Não é possível editar o nome do registro.

  6. Clique em Salvar para confirmar as atualizações na sessão atual.
  7. Para encerrar a sessão e implantar a configuração no run-time, clique em Ativar em Centro de Alterações.
