WSDLs

Um WSDL (Web Service Definition Language) é a descrição formal de um Web Service; no Oracle Service Bus, ele descreve um serviço de proxy ou um serviço de negócios. Um WSDL é usado para descrever o que um Web Service pode fazer, onde reside e como chamá-lo.

Você pode basear os serviços SOAP e XML em um recurso WSDL existente. Um documento WSDL está disponível para serviços de proxy e de negócios de qualquer transporte. Consulte a página Criar/Editar um Serviço de Proxy - Configuração Geral e Criando e Configurando Serviços de Negócios.

Para obter mais informações, consulte "How WSDL is Used in Oracle Service Bus" no Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus em http://www.oracle.com/pls/as111170/lookup?id=OSBAG114.
