Adicionando WSDLs

  1. Se ainda não tiver feito isso, clique em Criar para criar uma nova sessão ou clique em Editar para informar uma sessão existente. Consulte Usando o Centro de Alterações.
  2. Selecione o Project Explorer, depois selecione um projeto ou pasta no qual adicionar o WSDL. A página View de Projeto/Pasta é mostrada.
  3. Na lista drop-down Criar Recurso, selecione WSDL em Interface para exibir a página Criar um Novo Recurso de WSDL.
  4. No campo Nome do Recurso, informe um nome exclusivo para esse WSDL. Esse campo é obrigatório.

    Siga as Restrições para Nomeação de Recursos para obter orientações em relação a nomes.
  5. No campo Descrição do Recurso, informe uma descrição para o WSDL.
  6. No campo WSDL, execute um dos seguintes procedimentos:
    • Informe texto para o novo WSDL.
    • Clique em Procurar para localizar e importar um WSDL existente.
    • Copie e cole texto a partir de um WSDL existente neste campo.

      • Esse campo é obrigatório.

  7. Clique em Salvar. O novo WSDL é salvo na sessão atual.

    8. Quando você clicar em Salvar, se houver referências não resolvidas para o novo WSDL, o sistema as exibirá. Consulte Exibindo Referências WSDL Não Resolvidas.

  8. Para encerrar a sessão e implantar a configuração no run-time, clique em Ativar em Centro de Alterações.
