Localizando WSDLs

  1. Execute um dos seguintes procedimentos:
    • Selecione o Project Explorer para exibir a página View de Projetos ou a página View de Projeto/Pasta. Em seguida, navegue pelos projetos e pastas para localizar o WSDL.
    • Selecione Resource Browser > WSDLs. A página Resumo de WSDLs exibe as informações mostradas na Tabela 6-1. Para obter uma descrição mais detalhada das propriedades, consulte Editando WSDLs.
  2. Para localizar um WSDL específico:
    • Clique no ícone Abrir para exibir filtros de pesquisa adicionais. Se você especificar vários critérios de pesquisa, apenas os resultados que correspondem a todos os critérios serão retornados.
    • Para restringir o número de itens na lista, você pode filtrar por nome, caminho e namespace. Nos campos Nome, Caminho e Namespace, informe o nome, o caminho e o namespace do(s) destino(s) de pesquisa e, em seguida, clique em Pesquisar.

      • O caminho é o nome do projeto e o nome da pasta em que reside o WSDL.

    • Clique em Exibir Tudo para remover os filtros de pesquisa e exibir todos os WSDLs.

      • Tabela 6-1 - Detalhes do WSDL
      Propriedade
      Descrição
      Nome do WSDL
      O nome exclusivo designado ao WSDL. O nome é um link para a página Exibir Detalhes do WSDL. Consulte Editando WSDLs.
      Caminho
      O caminho é o nome do projeto e o nome da pasta em que reside o WSDL. É um link para o projeto ou pasta que contém este recurso. Consulte Qualificando Nomes de Recursos Usando Projetos e Pastas.
      Namespace de WSDL
      O endereço do WSDL. É exibido no formato:
      http://example.com/example/example
      Opções
      Clique no ícone Deletar para deletar um WSDL específico. Consulte Deletando WSDLs.
