Resolvendo Referências WSDL Não Resolvidas

Use a página Editar as Referências de um Recurso WSDL para resolver referências do WSDL não resolvidas configurando o mapeamento para referências do WSDL, como importações do WSDL e importações do esquema XML. Um WSDL também pode ter uma referência não resolvida para uma Política de Web Services.

Resolvendo uma Referência de WSDL ou de Esquema XML Não Resolvida
  1. Se ainda não tiver feito isso, clique em Criar para criar uma nova sessão ou clique em Editar para informar uma sessão existente. Consulte Usando o Centro de Alterações.
  2. Selecione Resource Browser > WSDLs.
  3. Na página Resumo de WSDLs, na coluna Nome do WSDL, clique no nome do WSDL apropriado para exibir detalhes da referência não resolvida. A página Exibir Detalhes do WSDL é exibida.
  4. Clique em Editar Referências para exibir a página Editar as Referências de um Recurso WSDL.
  5. No campo Tipo de Recurso, selecione WSDL ou Esquema XML.
  6. Clique em Procurar. Dependendo do tipo de recurso, o Browser do WSDL ou o Browser do Esquema XML será exibido.
  7. No Browser do WSDL ou no Browser do Esquema XML, selecione um WSDL ou um Esquema XML, depois selecione uma definição no painel Definições.
  8. Clique em Submeter. O WSDL ou esquema XML selecionado será exibido no campo Nome do Recurso.
  9. Clique em Salvar para resolver a referência.
  10. Na página Exibir Detalhes do WSDL, clique em OK. O estado do WSDL é exibido como Válido.
Resolvendo uma Referência de Política de Web Services Não Resolvida
  1. Se ainda não tiver feito isso, clique em Criar para criar uma nova sessão ou clique em Editar para informar uma sessão existente. Consulte Usando o Centro de Alterações.
  2. Selecione Resource Browser > WSDLs.
  3. Na página Resumo de WSDLs, na coluna Nome do WSDL, clique no nome do WSDL apropriado para exibir detalhes da referência não resolvida. A página Exibir Detalhes do WSDL é exibida.
  4. Clique em Editar Referências para exibir a página Editar as Referências de um Recurso WSDL.
  5. No campo Nome da Política de Web Services, clique em Procurar. OBrowser da Política será exibido.
  6. No Browser da Política, selecione uma Política de Web Services.
  7. Clique em Submeter. A Política de Web Services selecionada será exibida no campo Nome da Política de Web Services.
  8. Clique em Salvar para resolver a referência.
  9. Na página Exibir Detalhes do WSDL, clique em OK. O estado do WSDL é exibido como Válido.
