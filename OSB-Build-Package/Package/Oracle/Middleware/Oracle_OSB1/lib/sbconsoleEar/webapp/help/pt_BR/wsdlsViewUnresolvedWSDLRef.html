Exibindo Referências WSDL Não Resolvidas

A página Exibir Referências WSDL Não Resolvidas exibe as informações mostradas na Tabela 6-5.

Tabela 6-5 - Detalhes da Referência WSDL Não Resolvida
Propriedade
Descrição
Nome do WSDL
O nome designado ao WSDL. Clique no nome de um WSDL específico para exibir detalhes sobre esse WSDL.
Namespace
O endereço do WSDL. É exibido no formato:
http://example.com/example/example

Para localizar uma referência específica:

  • Classificar a lista novamente. Clique em um nome de coluna sublinhado. Setas crescentes e decrescentes indicam a ordem de classificação. Clique no nome da coluna para alterar a ordem de classificação.
  • Role pelas páginas. Use os controles de página acima ou abaixo da tabela. Para ir para uma página, selecione o número da página ou use os botões de seta para ir para a próxima página, página anterior, primeira ou última página.
