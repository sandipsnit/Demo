Localizando Documentos XML

A página Resumo de Documentos XML lista os Documentos XML que você adicionou ao domínio do Oracle Service Bus.

  1. Execute um dos seguintes procedimentos:
    • Selecione o Project Explorer para exibir a página View de Projetos ou a página View de Projeto/Pasta. Em seguida, navegue pelos projetos e pastas para localizar o Documento XML.
    • Selecione Resource Browser > Documentos XML. A página Resumo de Documentos XML exibe as informações mostradas na Tabela 13-1. Para obter uma descrição mais detalhada das propriedades, consulte Editando Documentos XML.
  2. Para localizar um Documento XML específico:
    • Para restringir o número de itens na lista, é possível filtrar pelo nome e caminho antes de clicar em Pesquisar.

      • Caminho é o nome do produto e o nome da(s) pasta(s) em que reside o recurso.

    • Clique em Exibir Tudo para remover os filtros de pesquisa e exibir todos os Documentos XML.

      • Tabela 13-1 - Detalhes de Documentos XML
      Propriedade
      Descrição
      Documento XML
      O nome exclusivo designado ao Documento XML. O nome é um link para a página ExibirDocumento XML. Consulte Editando Documentos XML.
      Caminho
      O caminho é o nome do projeto e o nome da pasta em que reside o Documento XML. É um link para o projeto ou pasta que contém este recurso. Consulte Qualificando Nomes de Recursos Usando Projetos e Pastas.
      Opções
      Clique no ícone Deletar para deletar Documentos XML específicos. Consulte Deletando Documentos XML.
