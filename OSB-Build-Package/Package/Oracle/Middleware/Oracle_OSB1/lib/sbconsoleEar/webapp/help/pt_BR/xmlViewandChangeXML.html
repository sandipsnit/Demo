Editando Documentos XML

Use a página Editar um Documento XML para exibir e alterar detalhes de um Binding de JCA.

  1. Se ainda não tiver feito isso, clique em Criar para criar uma nova sessão ou clique em Editar para informar uma sessão existente. Consulte Usando o Centro de Alterações.
  2. Localize o Documento XML, conforme descrito em Localizando Documentos XML.
  3. Clique no nome do Documento XML. A página Exibir Documento XML exibe as informações mostradas na Tabela 13-2.

    4. Tabela 13-2 - Detalhes do Documento XML
    Propriedade
    Descrição
    Última Modificação
    O usuário que criou este Documento XML ou o importou para a configuração.
    Última Modificação em
    A data e hora em que o usuário criou este Documento XML ou o importou para a configuração. Clique no link da data e hora para exibir o histórico de alterações deste recurso. Consulte a página Exibir Histórico de Alteração.
    Referências
    O número de objetos referenciados por este Documento XML. Se tais referências existirem, clique no link numérico para exibir uma lista dos objetos. Consulte Exibindo Referências a Recursos.
    Referenciado por
    O número de objetos que referenciam este Documento XML. Se tais referências existirem, clique no link numérico para exibir uma lista dos objetos. Por exemplo, se o arquivo XML for um arquivo TopLink referenciado por um Binding de JCA, o Binding de JCA será listado como uma referência quando você clicar no link. Consulte Exibindo Referências a Recursos.
    Descrição
    Uma descrição deste Documento XML, se existir.

    A view somente para leitura do Documento XML é exibida por default.

  4. Para editar o Documento XML, clique em Editar. Consulte Adicionando Documentos XML para obter descrições dos campos.
  5. Faça as edições apropriadas e clique em Salvar para confirmar as atualizações na sessão atual.
  6. Para encerrar a sessão e implantar a configuração no run-time, clique em Ativar em Centro de Alterações.
