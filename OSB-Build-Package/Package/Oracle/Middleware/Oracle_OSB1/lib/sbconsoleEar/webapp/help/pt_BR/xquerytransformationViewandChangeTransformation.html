Editando Transformações XQuery

  1. Se ainda não tiver feito isso, clique em Criar para criar uma nova sessão ou clique em Editar para informar uma sessão existente. Consulte Usando o Centro de Alterações.
  2. Localize a transformação XQuery, conforme descrito em Localizando Transformações XQuery.
  3. Clique no nome da transformação XQuery. A página Exibir Detalhes da Transformação XQuery exibe as informações mostradas na Tabela 10-2.

    4. Tabela 10-2 - Detalhes do Recurso de Transformação XQuery 
    Propriedade
    Descrição
    Última Modificação
    O usuário que criou esta transformação XQuery ou a importou para a configuração.
    Última Modificação em
    A data e hora em que o usuário criou esta transformação XQuery ou a importou para a configuração. Clique no link da data e hora para exibir o histórico de alterações deste recurso. Consulte a página Exibir Histórico de Alteração.
    Referências
    O número de objetos referenciados por esta transformação XQuery. Se tais referências existirem, clique no link numérico para exibir uma lista dos objetos. Consulte Exibindo Referências a Recursos.
    Referenciado por
    O número de objetos que referenciam esta transformação XQuery. Se tais referências existirem, clique no link numérico para exibir uma lista dos objetos. Por exemplo, se você tiver designado esta transformação XQuery para uma variável no Fluxo de Mensagens de um serviço de proxy específico, esse serviço será listado como referência quando você clicar no link. Para saber mais, consulte Exibindo Referências para Recursos.
    Descrição
    Uma descrição desta transformação XQuery, se existir.
    XQuery
    O texto completo da transformação XQuery.
    Variáveis XQuery
    As variáveis XQuery selecionadas para esta transformação XQuery.

  4. Para fazer uma alteração nos campos, clique em Editar. Consulte Adicionando Transformações XQuery para obter descrições dos campos.

    5. Não é possível alterar o campo Nome do Recurso.

  5. Clique em Salvar para confirmar as atualizações na sessão atual.
  6. Para encerrar a sessão e implantar a configuração no run-time, clique em Ativar em Centro de Alterações.
