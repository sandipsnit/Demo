Adicionando uma Transformação XSL

  1. Se ainda não tiver feito isso, clique em Criar para criar uma nova sessão ou clique em Editar para informar uma sessão existente. Consulte Usando o Centro de Alterações.
  2. Selecione o Project Explorer para exibir a página View de Projetos ou a página View de Projeto/Pasta.
  3. Navegue para o projeto ou pasta em que você deseja adicionar a transformação XSL.
  4. Na lista Criar Recurso, selecione XSLT para exibir a página Criar um Novo XSLT .
  5. No campo Nome do Recurso, informe um nome exclusivo para essa transformação XSL.

    Siga as Restrições para Nomeação de Recursos para obter orientações em relação a nomes.
  6. No campo Descrição do Recurso, informe uma descrição para a transformação XSL.
  7. No campo XSLT, execute um dos seguintes procedimentos:
    • Informe texto para a nova transformação XSL.
    • Clique em Procurar para localizar e importar uma transformação XSL existente.
    • Copie e cole texto a partir de uma transformação XSL existente neste campo.
  8. Clique em Salvar para confirmar as atualizações na sessão atual.
  9. Para encerrar a sessão e implantar a configuração no run-time, clique em Ativar em Centro de Alterações.
