Editando Transformações XSL

  1. Se ainda não tiver feito isso, clique em Criar para criar uma nova sessão ou clique em Editar para informar uma sessão existente. Consulte Usando o Centro de Alterações.
  2. Localize a transformação XSL, conforme descrito em Localizando Transformações XSL.
  3. Clique no nome da transformação XSL. A página Exibir Detalhes da Transformação XSL exibe a view estrutural do XSLT por default, que inclui as informações mostradas na Tabela 14-2.

    4. Tabela 14-2 Detalhes do Recurso de Transformação XQuery 
    Propriedade
    Descrição
    Última Modificação
    O usuário que criou esta transformação XSL ou a importou para a configuração.
    Última Modificação em
    A data e hora em que o usuário criou esta transformação XSL ou a importou para a configuração. Clique no link da data e hora para exibir o histórico de alterações deste recurso. Consulte a página Exibir Histórico de Alteração.
    Referências
    O número de objetos referenciados por esta transformação XSL. Se tais referências existirem, clique no link numérico para exibir uma lista dos objetos. Consulte Exibindo Referências a Recursos.
    Referenciado por
    O número de objetos que referenciam esta transformação XSL. Se tais referências existirem, clique no link numérico para exibir uma lista dos objetos. Por exemplo, se você designou esta transformação XSL para uma variável no Fluxo de Mensagens de um serviço de proxy específico, o serviço de proxy será listado como uma referência quando você clicar no link. Consulte Exibindo Referências a Recursos.
    Descrição
    Uma descrição desta transformação XSL.
    Namespace de Destino
    O namespace usado para qualificar qualquer uma das definições incluídas na transformação XSL.
    Dependências XSLT
    Os atributos e grupos associados ao XSLT:
    • O local XSLT
    Estado
    O status da transformação XSL:
    • Válido

      • Todos os locais da transformação XSL incluídos pela transformação XSL atual são especificados e válidos. Além disso, todos os locais para transformações XSL aninhadas (transformações XSL incluídas subsequentemente pelas transformações XSL incluídas) são especificados e válidos.

    • Inválida

      • Um ou mais locais para transformações XML incluídos pela transformação XLS atual não foram especificados ou são inválidos. Além disso, uma ou mais transformações XSL aninhadas (transformações XSL incluídas subsequentemente pelas transformações XSL incluídas) talvez não estejam especificadas ou sejam inválidas. Para resolver uma transformação XSL inválida, clique em Editar Referências. Para saber mais, consulte Resolvendo Referências a Transformação XSL Não Resolvidas.

  4. Clique em View de texto para exibir uma view de texto da transformação XSL. A view de texto inclui as informações mostradas na Tabela 14-3.

    5. Tabela 14-3 - Detalhes do Texto de Transformação XSL
    Propriedade
    Descrição
    Namespace de Destino
    O namespace usado para qualificar qualquer uma das dependências incluídas na transformação XSL.
    XSLT
    O texto completo da transformação XSL.

  5. Para fazer uma alteração nos campos, clique em Editar. Consulte Adicionando uma Transformação XSL para obter uma descrição dos campos.

    6. Não é possível alterar o campo Nome do Recurso.

  6. Clique em Salvar para confirmar as atualizações na sessão atual.
