编辑业务服务配置

  1. 如果尚未执行此操作，请单击创建新建一个会话，或单击编辑进入现有会话。请参阅使用更改中心。

    2. 如果仅希望查看详细资料，则无须进入会话。

  2. 查找要查看或编辑的业务服务，如查找业务服务中所述。
  3. 单击业务服务名。“查看业务服务 - 配置详细资料”页显示所选业务服务的配置信息。
  4. 要查看或修改设置，请执行以下操作之一：
    • 单击您要查看或编辑其属性的类别的名称 (例如一般配置、传输配置等) 旁边的编辑图标。可编辑的页取决于创建业务服务时配置的页。

      • 有关所有这些页的列表，请参阅“查看业务服务 - 配置详细资料”页。

    • 单击该页底部的编辑以便显示“创建/编辑业务服务 - 一般配置”页，即用于配置此业务服务的一系列页中的第一页。
  5. 继续进行查看或编辑，如创建和配置业务服务中所述。
  6. 在“编辑业务服务 - 概要”页上，单击保存可提交当前会话中的更新。
  7. 要结束该会话并将配置部署到运行时，请单击更改中心下的激活。
