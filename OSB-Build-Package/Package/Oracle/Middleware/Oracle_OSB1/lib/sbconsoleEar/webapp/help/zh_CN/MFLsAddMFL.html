添加 MFL

  1. 如果尚未执行此操作，请单击创建新建一个会话，或单击编辑进入现有会话。请参阅使用更改中心。
  2. 选择项目浏览器，然后选择要在其中添加 MFL 文件的项目或文件夹。此时将显示项目/文件夹视图页。
  3. 从创建资源下拉列表中，选择转换下的 MFL 文件以便显示创建新的 MFL 文件资源页。
  4. 在资源名字段中，输入此 MFL 的唯一名称。这是必填字段。

    如需命名指导，请参阅资源命名限制。
  5. 在资源说明字段中，输入此 MFL 的说明。
  6. 在 MFL 文本字段中，执行以下操作之一：
    • 输入新 MFL 的文本。
    • 单击浏览以定位和导入现有 MFL。
    • 将文本从现有 MFL 复制并粘贴到此字段中。

      • 这是必填字段。

  7. 单击保存。此时将创建 MFL 文件资源并将其保存在当前会话中。
  8. 要结束该会话并将配置部署到运行时，请单击更改中心下的激活。
