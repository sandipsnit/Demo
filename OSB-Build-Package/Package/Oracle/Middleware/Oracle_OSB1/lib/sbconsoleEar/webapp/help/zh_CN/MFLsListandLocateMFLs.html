查找 MFL

  1. 请执行以下两个操作之一：
    • 选择项目浏览器以便显示“项目视图”页或“项目/文件夹视图”页。然后在项目和文件夹中导航以查找 MFL。
    • 选择资源浏览器 > MFL。MFL 文件概要页显示表 15-1 中所示的信息。有关属性的更详细说明，请参阅编辑 MFL。
  2. 要限制列表中项的数量，可按名称、路径或同时按这两个条件进行筛选。在名称和路径字段中，输入搜索目标的名称和路径，然后单击搜索。

    3. 该路径由项目名和 MFL 文件所在的文件夹名称组成。

    单击查看全部可删除搜索筛选器，并显示所有 MFL 文件。

    表 15-1 MFL 详细资料
    属性
    说明
    MFL 文件名
    指定给 MFL 的唯一名称。此名称是指向查看 MFL 资源详细资料页的链接。请参阅编辑 MFL。
    路径
    该路径由项目名和 MFL 文件所在的文件夹名称组成。它是一个指向包含此资源的项目或文件夹的链接。请参阅使用项目和文件夹限定资源名。
    操作
    单击启动测试控制台图标可调用测试控制台，该控制台可用于验证和测试服务与转换的设计。对于 MFL，无论在会话内还是会话外都可以使用测试控制台。请参阅测试转换。
    选项
    单击删除图标可删除特定的 MFL。请参阅删除 MFL。
