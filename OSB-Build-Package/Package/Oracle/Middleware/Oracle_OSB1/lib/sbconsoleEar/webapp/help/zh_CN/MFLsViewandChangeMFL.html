编辑 MFL

  1. 如果尚未执行此操作，请单击“创建”新建一个会话，或单击编辑进入现有会话。请参阅使用更改中心。
  2. 查找 MFL 文件，如查找 MFL 中所述。
  3. 单击 MFL 文件名。查看 MFL 资源详细资料页显示表 15-2 中所示的信息。

    4. 表 15-2 MFL 资源详细资料 
    属性
    说明
    上次修改者
    创建此 MFL 文件或将其导入到配置中的用户。
    上次修改时间
    用户创建此 MFL 文件或将其导入到配置中的日期和时间。单击日期和时间链接可查看此资源的更改历史记录。请参阅“查看更改历史记录”页。
    引用
    此 MFL 文件引用的对象数。如果存在此类引用，可单击数字链接以查看对象列表。请参阅查看对资源的引用。
    引用项
    引用此 MFL 文件的对象数。如果存在此类引用，可单击数字链接以查看对象列表。例如，如果您选择了此 MFL 文件作为某个消息传送业务服务或代理服务的请求消息类型或响应消息类型，则当您单击此链接时，该业务服务或代理服务会作为引用列出。请参阅查看对资源的引用。
    说明
    此 MFL 文件的说明 (如果存在)。
    MFL 文本
    此 MFL 的文本。

  4. 要对这些字段进行更改，请单击编辑。有关这些字段的说明，请参阅添加 MFL。

    5. 您不能更改资源名字段。

  5. 单击保存可提交当前会话中的更新。
  6. 要结束该会话并将配置部署到运行时，请单击更改中心下的激活。
