预警目标

预警目标资源会捕获可从 Oracle Service Bus 接收预警通知的收件人列表。在典型的系统监视上下文中，Oracle Service Bus 生成的预警对于限定范围内的用户来说具有重要意义。

在 Oracle Service Bus 中，可根据给定的上下文将各个预警目标资源都配置为包括一组收件人。预警目标可供消息流中配置的预警操作使用，也可供 SLA 预警规则使用。预警目标可以包括一个或多个以下类型的目标：控制台 (默认值)、报告数据流、SNMP 陷阱、电子邮件、JMS 队列或 JMS 主题。

如果是电子邮件和 JMS 目标，则目标资源可以分别包括一组电子邮件地址或一组 JMS URI。

可在服务的预警配置之间重用预警目标。请参阅添加预警操作。
