DSP 传输配置页

“创建/编辑业务服务 - DSP 传输配置”页是一系列用于配置和编辑业务服务的页面中的一个页面，如创建和配置业务服务和编辑业务服务配置所述。

有关详细信息，请参阅通过 Oracle Service Bus 访问数据服务。

使用此页可为使用 DSP (Oracle Data Service Integrator) 传输协议的业务服务配置传输设置。表 17-4 介绍了如何使用此页。

表 17-4“创建/编辑业务服务 - DSP 传输配置”页
选项
创建或编辑...
调试级别
指定下列级别之一
  • 0 - 不提供调试信息
  • 1 - 输出有关请求消息的信息
  • 3 - 输出有关请求和响应消息的信息
服务帐户
单击“浏览”，然后从显示的列表中选择服务帐户。如果未指定任何服务帐户，则将使用匿名主题。

有关详细信息，请参阅下列页面：

  • 服务帐户
  • 添加服务帐户
分派策略
选择要对此端点的分派策略使用的 WebLogic Server 工作管理器实例。如果不存在其他工作管理器，则使用默认工作管理器。
此工作管理器用于发布响应处理的回复消息。有关工作管理器的信息，请参阅“WebLogic Server 管理控制台联机帮助”中的使用工作管理器优化调度的工作和创建全局工作管理器。

有关从 Oracle Service Bus 访问 Oracle Data Service Integrator 的详细信息，请参阅通过 Oracle Service Bus 访问数据服务。

完成之后

单击“下一步”继续在下一页上配置此服务；或单击“最后一步”在创建/编辑业务服务 - 概要页上查看并保存此配置。
