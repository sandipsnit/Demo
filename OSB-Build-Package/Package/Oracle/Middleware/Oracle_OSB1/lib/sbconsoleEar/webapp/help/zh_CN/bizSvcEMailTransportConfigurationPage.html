“电子邮件传输配置”页

创建/编辑业务服务 - 电子邮件传输配置页是一系列用于创建和编辑业务服务的页中的一个页面，如创建和配置业务服务和编辑业务服务配置中所述。

使用此页可为使用电子邮件传输协议的业务服务配置传输设置。表 18-6 介绍了如何使用此页。

表 18-6“创建/编辑业务服务 - 电子邮件传输配置”页
选项
创建或编辑...
需要使用 SSL
选择此选项可使用安全套接字层进行通信。SMTP 的默认 SSL 端口号是 465。
注释：
  • 要使 SSL 工作，Oracle Service Bus truststore 中必须要有电子邮件服务器证书。
  • 如果使用的是 TLS，则必须将信任证书放在默认信任存储 ($JAVA_HOME/jre/lib/security/cacerts) 中，或者放在任何信任存储中，并在 $DOMAIN_HOME/bin/setDomainEnv.* 文件中添加路径，如下所示：

    set EXTRA_JAVA_PROPERTIES=%EXTRA_JAVA_PROPERTIES%
    -Djavax.net.ssl.trustStore=<信任存储的路径>

SMTP 服务器
选择默认 SMTP 服务器用于 name@domain_name.com 的端点 URI 条目。如果在端点 URI 中提供 SMTP 服务器参数 (如 表 18-3 中所述)，将使用这些服务器资源，而不是使用此 SMTP 服务器设置。
如果使用“邮件会话”选项，则不要选择 SMTP 服务器。
有关创建 SMTP 资源的详细信息，请参阅添加 SMTP 服务器。
邮件会话
输入已配置邮件会话的 JNDI 名称，用于 name@domain_name.com 的端点 URI 条目。如果在端点 URI 中提供 JNDI 邮件会话参数 (如表 18-3 中所述)，将使用这些邮件会话，而不是使用此邮件会话设置。
如果使用“SMTP 服务器”选项，则不要输入邮件会话。
发件人名称
输入此服务的来源电子邮件帐户显示名称。
发件人地址
输入此服务的来源电子邮件帐户。
回复名称
输入回复电子邮件帐户的显示名称。
回复地址
输入要回复的电子邮件地址。
连接超时
输入在删除连接之前所需经过的超时间隔 (秒)。如果输入 0，则没有超时。
请求编码
接受默认值 ISO-8859-1 作为电子邮件传输中请求的字符集编码，或者输入其他字符集编码。

完成之后

单击下一步继续在下一页上配置此服务；或单击最后一步在“创建/编辑业务服务 - 概要”页上查看并保存此配置。
