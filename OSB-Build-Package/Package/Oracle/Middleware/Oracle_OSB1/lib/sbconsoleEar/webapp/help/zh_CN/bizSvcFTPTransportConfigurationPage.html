“FTP 传输配置”页

创建/编辑业务服务 - FTP 传输配置页是一系列用于创建和编辑业务服务的页中的一个页面，如创建和配置业务服务和编辑业务服务配置中所述。

使用此页可为使用 FTP 传输协议的业务服务配置传输设置。表 18-9 介绍了如何使用此页。

表 18-9“创建/编辑业务服务 - FTP 传输配置”页
选项
创建或编辑...
用户验证
如果 FTP 服务器的用户是匿名用户，则选择匿名；如果 FTP 服务器的用户是外部配置的帐户，则选择外部用户。
标识 (电子邮件 ID)
仅当用户验证选项设置为匿名时，此字段才可用。
输入匿名用户的邮件 ID。
服务帐户
仅当用户验证选项设置为外部用户时，此字段才可用。
输入外部用户的服务帐户。

详细信息：

  • 服务帐户
  • 添加服务帐户。
超时
输入在删除连接之前所需经过的套接字超时间隔 (秒)。默认值为 60 秒。
目标文件名的前缀
输入将文件存储在远程服务器上时所用的文件名的前缀。
不要在此字段中输入 *。该字符会导致运行时异常错误。
目标文件名的后缀
输入将文件存储在远程服务器上时所用的文件名的后缀。
不要在此字段中输入 *。该字符会导致运行时异常错误。
传输模式
选择 ASCII 或二进制作为传输模式。
请求编码
接受默认值 UTF-8 作为 FTP 传输中请求的字符集编码，或者输入其他字符集编码。

完成之后

单击下一步继续在下一页上配置此服务；或单击最后一步在“创建/编辑业务服务 - 概要”页上查看并保存此配置。
