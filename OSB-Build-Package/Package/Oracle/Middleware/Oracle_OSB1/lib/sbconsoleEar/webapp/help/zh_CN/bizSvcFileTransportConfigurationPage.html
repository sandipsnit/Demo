“文件传输配置”页

创建/编辑业务服务 - 文件传输配置页是一系列用于配置和编辑业务服务的页中的一个页面，如创建和配置业务服务和编辑业务服务配置中所述。

使用此页可为使用文件传输协议的业务服务配置传输设置。表 18-7 介绍了如何使用此页。

表 18-7“创建/编辑业务服务 - 文件传输配置”页
选项
创建或编辑...
前缀
输入要追加到文件名上的前缀。
不要在此字段中输入 *。该字符会导致运行时异常错误。
后缀
输入要追加到文件名上的后缀。这是必填字段。
不要在此字段中输入 *。该字符会导致运行时异常错误。
请求编码
接受默认值 utf-8 作为文件传输中请求的字符集编码，或者输入其他字符集编码。

完成之后

单击下一步继续在下一页上配置此服务；或单击最后一步在“创建/编辑业务服务 - 概要”页上查看并保存此配置。
