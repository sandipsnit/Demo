“SB 传输配置”页

创建/编辑业务服务 - SB 传输配置页是一系列用于创建和编辑业务服务的页中的一个页面，如创建和配置业务服务和编辑业务服务配置中所述。

有关详细信息，请参阅 Oracle Fusion Middleware Developer's Guide for Oracle Service Bus 中的“SB Transport”，网址为：http://www.oracle.com/pls/as111170/lookup?id=OSBDV997。

使用此页可为使用 SB (Service Bus) 传输协议的业务服务配置传输设置。表 18-15 介绍了如何使用此页。

表 18-15“创建/编辑业务服务 - SB 配置”页
选项
创建或编辑...
分派策略
选择要对此端点的分派策略使用的 WebLogic Server 工作管理器实例。如果不存在其他工作管理器，则使用默认工作管理器。有关工作管理器的信息，请参阅 Oracle Fusion Middleware Configuring Server Environments for Oracle WebLogic Server 中的“Using Work Managers to Optimize Scheduled Work”，网址为：http://www.oracle.com/pls/as111170/lookup?id=CNFGD112。
超时
服务超时所需的时间 (秒)。
注： 服务的质量为“恰好一次”时，将忽略超时。
服务帐户
单击浏览，然后从显示的列表中选择服务帐户。如果未指定任何服务帐户，则将使用匿名主题。
