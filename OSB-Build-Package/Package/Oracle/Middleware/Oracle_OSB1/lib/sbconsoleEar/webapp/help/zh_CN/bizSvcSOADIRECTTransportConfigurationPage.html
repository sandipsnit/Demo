“SOA-DIRECT 传输配置”页

表 18-17 介绍了 SOA-DIRECT 传输的传输特定配置选项。

有关 SOA-DIRECT 传输的详细信息，请参阅 Oracle Fusion Middleware Developer's Guide for Oracle Service Bus 中的“SOA-DIRECT Transport”，网址为： http://www.oracle.com/pls/as111170/lookup?id=OSBDV909。

表 18-17 创建/编辑业务服务 - SOA-DIRECT 传输配置
属性
说明
JNDI 服务帐户
可选。指定目标 SOA 服务的 JNDI 查找的安全身份证明。服务帐户必须是静态的。单击浏览并选择服务帐户。如果您未指定服务帐户，则将使用匿名主题。
角色
必需。标识服务使用的通信模式。选择以下选项之一：
  • 同步客户机 (默认值) - 在此角色中，由于没有异步回调，会禁用“回调代理”选项。同时也会禁用“WS-Addressing 版本”选项。
  • 异步客户机 - 在此角色中，由于通常需要异步回调，因此您可以标识“回调代理”，且必须选择“WS-Addressing 版本”。仅当 WSDL 服务为 SOAP 类型时，才会启用异步选项。
  • 服务回调 - 此角色用于在外部服务调用之后将异步回调返回 SOA 服务。

    • 负载平衡或故障转移不适用于服务回调角色中的服务。

回调代理
可选。仅针对“异步客户机”角色启用。
此选项用于指定接收回调的代理服务。当选择回调代理时，如果请求或代理服务管道未提供 WS-Addressing，Oracle Service Bus 会自动填充 SOAP 标头中的 ReplyTo 属性。您必须选择使用 SB 传输 (对于 RMI) 的 WSDL 代理服务，且回调代理服务必须理解 WS-Addressing。
将使用请求中发送的或代理服务管道中设置的 WS-Addressing 属性，而不是您在此选项中设置的回调代理。
如果未指定回调代理，且请求中不包含 ReplyTo 属性，则必须通过代理服务管道在 SOAP 标头中提供 ReplyTo 属性。
故障代理
当前不支持此选项。您必须配置回调服务，以处理异步模式中的故障。
WS-Addressing 版本
必需。仅针对“异步客户机”角色启用。
指定请求或代理服务管道中未提供 WS-Addressing 时要使用的默认 WS-Addressing 版本。将使用请求中发送的或代理服务管道中设置的 WS-Addressing 属性，而不是您在此选项中设置的 WS-Addressing 版本。
对于各环境之间 WS-Addressing 版本不匹配的情况，请在代理服务管道中执行任何必要的转换。有关详细信息，请参阅 Oracle Fusion Middleware Developer's Guide for Oracle Service Bus 中的“Transformation Examples”，网址为：http://www.oracle.com/pls/as111170/lookup?id=OSBDV1827。
分派策略
选择要对此端点的分派策略使用的 WebLogic Server 工作管理器实例。如果不存在其他工作管理器，则使用默认工作管理器。
有关工作管理器的信息，请参阅 Oracle Fusion Middleware Configuring Server Environments for Oracle WebLogic Server 中的“Using Work Managers to Optimize Scheduled Work”，网址为：http://www.oracle.com/pls/as111170/lookup?id=CNFGD112。
传递调用者的主题
可选。选择此选项，以使 Oracle Service Bus 在调用 SOA 服务时传递代理服务的已通过验证的主题。当选择此选项时，会禁用“调用服务帐户”选项 (“传递调用者的主题”的替代选项)。
注：请确保客户机和目标服务器之间已启用域信任 (如果它们处于不同域中)。有关详细信息，请参阅 Oracle Fusion Middleware Securing Oracle WebLogic Server 中的“Important Information Regarding Cross-Domain Security Support”，网址为： http://www.oracle.com/pls/as111170/lookup?id=SECMG403。
调用服务帐户
可选。“传递调用者的主题”的替代选项，用于通过选择 RMI 调用的服务帐户来指定定制安全身份证明。您可以指定任何类型的服务帐户 (“通过”、“静态”和“映射”)。
单击浏览并选择服务帐户。如果未指定服务帐户，则将使用匿名主题。
