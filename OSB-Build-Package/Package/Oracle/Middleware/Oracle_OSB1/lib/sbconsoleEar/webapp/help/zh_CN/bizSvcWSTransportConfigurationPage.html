“WS 传输配置”页

创建/编辑业务服务 - WS 传输配置页是一系列用于创建和编辑业务服务的页中的一个页面，如创建和配置业务服务和编辑业务服务配置中所述。

有关详细信息，请参阅 Oracle Fusion Middleware Developer's Guide for Oracle Service Bus 中的“WS Transport”，网址为：http://www.oracle.com/pls/as111170/lookup?id=OSBDV1083。

使用此页可为使用 WS 传输协议的业务服务配置传输设置。表 18-19 介绍了如何使用此页。

表 18-19“创建/编辑业务服务 - WS 传输配置”页
选项
创建或编辑...
响应超时
输入等待响应的秒数。
将此字段留空表示不存在响应超时。业务服务将等待一段时间，其长度为 RM 策略中配置的序列超时时段。
如果输入零 (0) 值，则没有超时；在这种情况下，它将永不超时。
服务帐户
输入服务帐户，或单击浏览从显示的列表中选择一个服务帐户。
服务帐户指定此服务上有 HTTP 基本验证策略时使用的身份证明。
高级设置
队列错误消息
选中该复选框可向配置的错误队列发送错误消息。
错误队列 URI
只有在选中队列错误消息复选框后，此选项才可用。
按以下格式输入存储错误消息的 JMS 队列的 URI：

jms://host:port/connFactoryJndiName/queueJndiName

注：虽然 Oracle WebLogic Server 允许在 JNDI 名称中使用正斜杠 (例如“myqueues/myqueue”)，但是带正斜杠的 JNDI 名称会对 Oracle Service Bus 所需的 URI 格式产生干扰，因此您不能使用这些名称。要解决此问题，请定义一个 JMS 外部服务器，然后在 URI 中引用该外部服务器。有关详细信息，请参阅 Oracle Fusion Middleware Oracle WebLogic Server Administration Console Online Help 中的“Configure foreign servers”，网址为：http://www.oracle.com/pls/as111170/lookup?id=WLACH。

JMS 错误队列服务帐户
只有在选中队列错误消息复选框后，此选项才可用。
输入服务帐户，或单击浏览从显示的列表中选择一个服务帐户。
服务帐户指定要用于 JNDI 查找和向错误队列发送错误消息的身份证明。
为错误队列使用 SSL
只有在选中队列错误消息复选框后，此选项才可用。
选中该复选框可将 SSL 用于连接到错误队列。

完成之后

单击下一步继续在下一页上配置此服务；或单击最后一步在“创建/编辑业务服务 - 概要”页上查看并保存此配置。
