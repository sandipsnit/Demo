查找业务服务

  1. 请执行以下两个操作之一：
    • 选择项目浏览器以便显示“项目视图”页或“项目/文件夹视图”页。然后，在项目和文件夹中导航以查找业务服务。
    • 选择资源浏览器 > 业务服务。业务服务概要页显示表 18-21 中所示的信息。有关属性的更详细说明，请参阅编辑业务服务配置。
  2. 要限制列表中项的数量，可按名称、路径或同时按这两个条件进行筛选。在搜索下的名称和路径字段中，分别输入目标的名称和/或路径，然后单击搜索按钮。

    3. 该路径由项目名和业务服务所在的文件夹名称组成。

    允许使用通配符 * 和 ?。搜索区分大小写。

    单击查看全部可显示域中的所有业务服务。这会清除以前的搜索中使用的搜索参数。

    表 18-21“业务服务概要”页
    属性
    说明
    名称
    业务服务的唯一名称。单击该名称可显示“查看业务服务 - 配置详细资料”页。
    请参阅编辑业务服务配置。
    路径
    该路径由项目名和业务服务所在的文件夹名称组成，例如 UDDI/BusServices/OSB_services。
    单击业务服务的路径可显示包含该业务服务的“项目/文件夹视图”页。
    操作
    请执行以下操作之一：
    • 单击启动测试控制台图标可调用测试控制台，可使用其验证并测试服务与转换的设计。对于业务服务，只能在运行时 (即在激活会话后) 使用测试控制台。对于转换，无论在会话内还是会话外，均可以使用测试控制台。请参阅测试服务和测试转换。
    • 对于基于 WSDL 的业务服务，将显示导出 WSDL 图标。单击此图标可导出 WSDL，随后可以使用外部工具 (如 IDE) 对其进行查看或修改。

      • 请注意，此功能与“系统管理”模块中的“导出资源”功能不同，后者用于在两个域之间移动和暂存资源。请参阅导出 WSDL。

    • 对于传输类型的业务服务 (例如 EJB 和 JEJB )，而不是拆分-联接流业务，将显示生成 WSDL 图标。单击此图标可生成 WSDL，随后可对其进行查看或修改。请参阅生成 WSDL。
    选项
    • 单击删除图标可删除服务。如果有其他资源引用此资源，将会显示删除警告图标。确认警告后即可删除该资源。这可能会由于存在对已删除资源的未解析引用而导致冲突。有关详细信息，请参阅删除业务服务。
