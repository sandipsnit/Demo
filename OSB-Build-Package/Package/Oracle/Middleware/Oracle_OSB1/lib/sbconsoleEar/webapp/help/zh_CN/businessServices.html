业务服务：创建和管理

业务服务是您要与之交换消息的企业服务的 Oracle Service Bus 定义。使用 WSDL (Web 服务定义语言) 定义业务服务的方式与定义代理服务的方式相同。但是，业务服务的配置与代理服务的配置不同，因为业务服务没有管道。因此，不是由 Oracle Service Bus 管道实施的服务即为业务服务。

有关详细信息，请参阅 Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus 中的“Configuring Proxy Services and Business Services”，网址为：http://www.oracle.com/pls/as111170/lookup?id=OSBAG111。
