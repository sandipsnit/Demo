“查看业务服务 - 配置详细资料”页

查看业务服务 - 配置详细资料页显示业务服务的配置详细资料。表 18-22 介绍了此页上可显示的所有属性。(属性因业务服务的详细资料而异。)

此页上列出的类别与用于创建和编辑业务服务配置的创建/编辑业务服务页相对应，如创建/编辑业务服务 - 页引用中所述。

单击任何类别名称旁边的编辑链接可显示关联的配置页。

表 18-22“查看业务服务：配置详细资料”页
属性
说明
上次修改者
创建该业务服务或将其导入到配置中的用户。
上次修改时间
用户创建该业务服务或将其导入到配置中的日期和时间。单击日期和时间链接可查看此资源的更改历史记录。请参阅“查看更改历史记录”页。
引用
此业务服务所引用对象的数量。如果存在此类引用，可单击数字链接以查看对象列表。要了解详细信息，请参阅查看对资源的引用。
引用项
