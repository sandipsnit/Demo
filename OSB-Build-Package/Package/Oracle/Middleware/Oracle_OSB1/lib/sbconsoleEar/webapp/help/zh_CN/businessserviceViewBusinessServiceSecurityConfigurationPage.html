“业务服务安全”页

查看业务服务 - 安全配置页是一系列用于编辑业务服务的页中的一个页面，如创建和配置业务服务和查看业务服务 - 配置详细资料页中所述。

此页上可用的字段取决于业务服务中使用的策略。例如，如果服务使用 OWSM 策略 (推荐)，“策略覆盖”会显示在“安全”页上。有关详细信息，请参阅 Oracle Fusion Middleware Developer's Guide for Oracle Service Bus 中的“Securing Oracle Service Bus with Oracle Web Services Manager”，网址为： http://www.oracle.com/pls/as111170/lookup?id=OSBDV1681。

绑定到服务的 WLS 9.2 策略是在有效 WSDL 中公开 (内嵌) 的。抽象策略在内嵌之前会进行预处理。OWSM 策略是通过引用绑定的，而不是内嵌在有效 WSDL 中。

使用此页可为具有基于 WSDL 的策略或使用定制策略绑定的业务服务配置安全设置。表 18-24 介绍了如何使用此页。
