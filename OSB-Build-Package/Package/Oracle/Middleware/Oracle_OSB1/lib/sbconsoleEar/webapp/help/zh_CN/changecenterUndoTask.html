撤消任务

使用查看配置更改页可以撤消当前会话期间在 Oracle Service Bus 配置中所执行的任务，并可以在会话之外撤消会话激活。通过 Oracle Service Bus 可以撤消多个级别的会话激活，这仅受系统资源的限制。有关详细信息，请参阅撤消任务的顺序。

注： 如果从 Oracle Service Bus 版本 2.1 升级到版本 2.5，则无法撤消在升级前激活的会话。在会话激活历史记录中可以撤消在升级后激活的会话。
在会话中撤消任务
  1. 确保您在会话中。
  2. 单击更改中心下的查看更改。
  3. 在特定任务的选项列中，单击撤消图标。

    4. 此任务会撤消。

    您可以撤消当前会话中的任何更改。但是进行撤消操作之后，如果您改变了主意，您可以撤消该撤消操作，或者如果您再次改变主意，您可以对撤消上一次撤消的撤消操作进行撤消，依此类推。

撤消较早激活的会话
  1. 确保您在会话之外。
  2. 单击更改中心下的查看更改。
  3. 请执行以下操作之一：
    • 在特定任务的选项列中，单击撤消图标。此时将撤消会话激活。
    • 在特定任务的选项列中，选择在会话中完成撤消图标。此时将新建一个会话，其中预先填充了在会话中撤消已激活更改所需的任务。有关详细信息，请参阅在会话中完成撤消。
