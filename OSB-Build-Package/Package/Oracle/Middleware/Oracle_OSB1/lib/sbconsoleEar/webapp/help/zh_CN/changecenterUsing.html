使用更改中心

更改中心模块是使用 Oracle Service Bus 管理控制台对配置进行更改的起始点。要使用 Oracle Service Bus 管理控制台进行配置更改，必须使用更改中心来开始会话。

  1. 单击编辑或创建可开始会话。更改中心下将显示会话名称。
  2. 在 Oracle Service Bus 管理控制台的相关页中进行相应的更改。
  3. 在您进行更改的每个页上单击保存。此时将保存在当前会话中所做的所有更改。

    4. 可以在会话期间随时单击更改中心下的放弃，以放弃到目前为止已在当前会话中所做的更改。

  4. 完成更改后，单击更改中心下的激活。继续激活会话。
