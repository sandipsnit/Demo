查看所有会话

使用此页可查看 Oracle Service Bus 管理控制台中的所有现有会话。无论您当前在会话中还是在会话外，都可以查看这些会话。

只有在使用管理员角色时才能查看所有会话。有关详细信息，请参阅 Oracle Fusion Middleware Developer's Guide for Oracle Service Bus 中的“Configuring Administrative Security”，网址为：http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539。

  1. 单击更改中心下的查看所有会话。对于每个现有会话，查看所有会话页都会显示表 2-5 中所示的信息。

    2. 表 2-5 会话详细资料
    属性
    说明
    会话名称
    会话的名称。此名称是指向会话的链接。
    用户
    创建会话的用户名。
    创建时间
    会话的创建日期和时间。
    上次修改时间
    会话期间上次进行更改的日期和时间。
    已修改的对象数
    在会话期间进行更改的对象数量。

  2. 要切换到另一个会话，请单击会话名称列中该会话的链接。
注： 不支持同一用户使用多个浏览器登录。这样会导致 Oracle Service Bus 管理控制台中出现不可预测的行为。
