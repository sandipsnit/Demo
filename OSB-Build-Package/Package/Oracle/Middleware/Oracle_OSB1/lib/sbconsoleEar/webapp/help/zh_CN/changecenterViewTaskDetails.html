查看任务详细资料

如果您在会话中，使用该页可以查看在当前会话中所做的特定更改的详细资料；如果您在会话外，使用该页可以查看在已激活的会话中所做的特定更改的详细资料。

  1. 选择更改中心下的查看更改。
  2. 在查看配置更改页上，单击任务列中的任务名称。任务详细资料页将显示表 2-4 中所示的信息。

    3. 表 2-4 任务详细资料
    属性
    说明
    任务
    执行的任务的类型。
    • 创建
    • 更新
    • 删除
    • 重命名
    • 移动
    资源类型
    任务中的 Oracle Service Bus 资源类型。
    资源
    资源的名称和路径。此路径为该资源所在的项目名和文件夹名称。

  3. 单击上一步可返回到查看配置更改页。
