查看配置更改

此页会根据您是否在会话中 (即是否在更改中心中单击了编辑) 而显示不同的信息。

  • 当您在会话中时，查看配置更改页将显示一个在当前会话期间所做的配置更改的列表。请参阅在会话中查看配置更改。
  • 当您在会话之外时，查看配置更改页将显示一个由于以前的会话激活而导致的配置更改的列表。请参阅查看由会话激活导致的配置更改。
在会话中查看配置更改
  1. 如果尚未执行此操作，请单击创建以新建用于更改当前配置的会话。请参阅使用更改中心。
  2. 对此配置至少进行一项更改。
  3. 单击更改中心下的查看更改。对于您在当前会话期间对配置所做的每项更改，查看配置更改页将显示表 2-2 中所示的信息。

    4. 表 2-2 会话中配置更改
    属性
    说明
    任务
    已执行的任务的说明。此任务是指向任务详细资料页的链接。请参阅查看任务详细资料。
    执行时间
    任务的执行日期和时间。
    用户
    执行更改的用户的名称。
    任务状态
    任务的状态：
    已完成 - 任务已完成。
    已撤消 - 任务已撤消。
    已完成撤消 - 已完成撤消。
    撤消者
    撤消此任务的用户的名称。如果尚未撤消此任务，则此字段中将包含 None。
    选项
    单击撤消图标可取消任务的执行。
    当您在会话中工作时，可以按任意顺序撤消任务。
    作为撤消任务的结果，任务的对象将返回到执行相应任务之前的状态。但请注意，在您撤消该任务之后对同一个对象执行的任何任务也会撤消。请参阅撤消任务。

查看会话激活导致的配置更改
  1. 确保您不在会话中。请参阅退出会话。
  2. 单击更改中心下的查看更改。对于您以前已激活的每个会话，查看配置更改页将显示表 2-3 中所示的信息。

    3. 表 2-3 激活配置更改
    属性
    说明
    任务
    已激活的会话的说明。此任务是指向任务详细资料页的链接，该页显示了在该会话中执行的操作。请参阅查看任务详细资料。
    说明
    已激活会话的其他说明，前提是在使用激活会话页激活会话时输入了说明。
    执行时间
    会话的激活日期和时间。
    用户
    激活会话的用户的名称。
    任务状态
    会话的状态：
    已激活 - 会话已激活。
    已部分激活 - 如果集群中的一个或多个服务器不可访问，则会在会话激活期间显示此状态，从而避免在这些服务器上激活会话。当服务器变得可访问时，便可以在这些服务器上激活任何未激活的更改。
    已撤消 - 以前激活的会话已撤消，在该会话中执行的所有操作均已放弃。
    已激活撤消 - 已激活撤消。
    进行中 - 如果会话激活正在进行中，则会显示此状态，因为会话激活可能需要较长时间。
    失败 - 如果会话激活失败，则显示此状态。Oracle Service Bus 会跟踪会话激活失败，但是不会跟踪会话中由于单个更新而导致的失败。
    撤消者
    撤消此任务的用户的名称。如果尚未撤消此会话激活，则此字段中将包含无。
    选项
    单击撤消图标可取消会话激活及在该会话中执行的操作。您可以按任意顺序撤消会话激活。
    单击在会话中完成撤消图标可新建一个会话，其中预先填充了在会话中撤消已激活更改所需的任务。请参阅在会话中完成撤消。
