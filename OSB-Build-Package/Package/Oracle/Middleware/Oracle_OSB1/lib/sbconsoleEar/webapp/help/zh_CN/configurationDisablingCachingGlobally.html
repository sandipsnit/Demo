全局禁用结果高速缓存

全局禁用结果高速缓存时，Oracle Service Bus 会刷新整个高速缓存，这将删除启用了结果高速缓存的所有业务服务的已高速缓存结果。结果高速缓存的默认设置是启用。

要在域级别禁用结果高速缓存，请执行以下操作：

  1. 如果尚未执行此操作，请单击“创建”新建一个会话，或单击“编辑”进入现有会话。请参阅使用更改中心。
  2. 选择操作 > 全局设置。
  3. 清除启用结果高速缓存复选框。
    4. 注： 此设置将覆盖单个业务服务上配置的任何结果高速缓存。
  4. 单击更新。此配置会保存到当前会话中。
  5. 另外，可以在会话期间随时单击重置以放弃更改并使用当前存储的设置刷新页。
  6. 要结束该会话并将配置部署到运行时，请单击更改中心下的激活。
