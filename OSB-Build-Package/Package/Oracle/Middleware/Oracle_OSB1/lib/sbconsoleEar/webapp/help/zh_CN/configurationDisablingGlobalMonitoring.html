禁用服务的全局监视

要在域级别为所有服务禁用监视，请执行以下操作：

注： 如果禁用所有服务的监视，则到目前为止为这些服务收集的所有统计信息也将被删除。无法使用会话撤消功能恢复这些统计信息。可以使用撤消再次启用对服务的监视，但是删除统计信息是不可逆的操作。
  1. 如果尚未执行此操作，请单击创建新建一个会话，或单击编辑进入现有会话。请参阅使用更改中心。
  2. 选择操作 > 全局设置。
  3. 清除启用监视复选框，使系统停止为配置中的所有服务收集监视统计信息。
    4. 注： 此选项可覆盖为特定业务服务和代理服务选择的启用服务监视选项。请参阅为业务服务配置操作设置和为代理服务配置操作设置。
    注： 此选项也将覆盖启用 SLA 预警选项。如果禁用全局级别的监视，则运行时的 SLA 预警也将被禁用，即使已选中相应的启用 SLA 预警复选框也是如此。请参阅全局启用 SLA 预警。
  4. 单击更新。此监视配置会保存到当前会话中。
  5. 另外，可以在会话期间随时单击重置以放弃更改并使用当前存储的设置刷新页。
  6. 要结束该会话并将配置部署到运行时，请单击更改中心下的激活。
