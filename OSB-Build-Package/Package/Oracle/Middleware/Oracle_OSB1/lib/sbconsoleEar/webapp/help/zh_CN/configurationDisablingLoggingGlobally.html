全局禁用日志记录

对于代理服务，要在域级别为所有代理服务禁用日志记录输出，请执行以下操作：

  1. 如果尚未执行此操作，请单击创建新建一个会话，或单击编辑进入现有会话。请参阅使用更改中心。
  2. 选择操作 > 全局设置。
  3. 清除启用日志记录复选框以停止所有代理服务的所有管道记录操作。
    4. 注： 此选项可覆盖为特定代理服务选择的任何日志记录设置。请参阅为代理服务配置操作设置。
  4. 单击更新。此日志记录配置会保存到当前会话中。
  5. 另外，可以在会话期间随时单击重置以放弃更改并使用当前存储的设置刷新页。
  6. 要结束该会话并将配置部署到运行时，请单击更改中心下的激活。
