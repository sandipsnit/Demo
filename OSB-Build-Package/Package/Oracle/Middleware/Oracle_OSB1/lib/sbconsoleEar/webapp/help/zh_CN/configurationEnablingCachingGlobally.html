全局启用结果高速缓存

如果调用其结果很少更改的业务服务，则结果高速缓存可以将高速缓存的结果返回给客户机而不是等待服务调用的结果，从而提高业务服务的性能。

在配置单个业务服务的结果高速缓存之前，首先必须使用以下步骤全局启用结果高速缓存。结果高速缓存的默认设置是启用。

  1. 如果尚未执行此操作，请单击“创建”新建一个会话，或单击“编辑”进入现有会话。请参阅使用更改中心。
  2. 选择操作 > 全局设置。
  3. 选中启用结果高速缓存复选框。
  4. 单击更新以保存此页上的所有设置。
  5. 另外，可以在会话期间随时单击重置以放弃更改并使用当前存储的设置刷新页。
  6. 要结束该会话并将配置部署到运行时，请单击更改中心下的激活。

有关为业务服务配置结果高速缓存的信息，请参阅“消息处理配置”页。
