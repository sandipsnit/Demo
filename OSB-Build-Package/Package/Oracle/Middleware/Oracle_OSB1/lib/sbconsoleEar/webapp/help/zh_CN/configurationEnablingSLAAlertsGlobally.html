全局启用 SLA 预警

尽管可以将 SLA 预警独立于监视进行配置，但是在运行时，这两个设置之间存在交互。如果启用全局监视，则可以启用或禁用 SLA 预警。但是，如果禁用了全局监视，则也将有效地禁用 SLA 预警，因为 SLA 预警规则条件依赖于所评估的监视统计信息。

要在域级别为所有服务启用 SLA 预警，请执行以下操作：

  1. 如果尚未执行此操作，请单击创建新建一个会话，或单击编辑进入现有会话。请参阅使用更改中心。
  2. 选择操作 > 全局设置。
  3. 选中启用 SLA 预警复选框，使系统开始为配置中的所有服务评估预警规则。
  4. 单击更新以保存此页上的所有设置。
  5. 另外，可以在会话期间随时单击重置以放弃更改并使用当前存储的设置刷新页。
  6. 要结束该会话并将配置部署到运行时，请单击更改中心下的激活。
