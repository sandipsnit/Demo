查找业务服务

  1. 选择操作 > 智能搜索。
  2. 从类型下拉列表中选择业务服务，然后单击搜索。业务服务概要页显示表 26-3 中所示的信息。
  3. 要限制列表中的项目个数或查找特定业务服务，请执行以下操作：
    • 单击打开图标可显示其他搜索筛选器。如果指定多个搜索条件，则仅返回与所有条件均匹配的结果。
    • 按业务服务名称和路径进行筛选。在名称和路径字段中，分别输入搜索目标的名称和路径，然后单击搜索。该路径由项目名和业务服务所在的文件夹名称组成。可将 * 和 ? 用作通配符以执行更全面的搜索。
    • 按服务状态进行筛选。从状态下拉列表中，选择业务服务状态：“已启用”或“已禁用”，然后单击搜索。选择“所有”可忽略服务状态筛选条件。
    • 按服务监视进行筛选。从监视下拉列表中，选择业务服务的监视状态：“已启用”或“已禁用”，然后单击搜索。选择“所有”可忽略服务监视筛选条件。
    • 按业务服务的 SLA 预警设置进行筛选。从 SLA 预警下拉列表中，选择业务服务的 SLA 预警设置：“已启用”或“已禁用”，然后单击搜索。选择“所有”可忽略服务的 SLA 预警筛选条件。
    • 按消息跟踪进行筛选。选择要筛选启用消息跟踪还是禁用消息跟踪的业务服务，然后单击搜索。选择“所有”可忽略消息跟踪筛选条件。
    • 按高速缓存进行筛选。选择要筛选启用结果高速缓存还是禁用结果高速缓存的业务服务，然后单击搜索。选择“所有”可忽略结果高速缓存筛选条件。

      • 表 26-3 业务服务概要信息
      属性
      说明
      名称
      分配给业务服务的名称。此名称是指向操作设置页的链接。请参阅为业务服务配置操作设置。
      路径
      与业务服务相关联的项目。如果服务位于项目文件夹中，则也会列出此文件夹。此路径按以下格式显示：
      project-name/root-folder/ . . ./parent-folder
      此路径是指向项目浏览器中相应路径的链接。
      类型
      父服务类型：业务服务。
      状态
      此业务服务的状态：“已启用”或者“已禁用”。
      监视
      此业务服务的监视状态：“已启用”或“已禁用”。
      SLA 预警
      SLA 预警状态：“已启用”或“已禁用”，以及启用级别和更高级别：普通 (N)、警告 (W)、次要 (Mn)、主要 (Mj)、严重 (C) 或致命 (F)。
      消息跟踪
      此业务服务已启用还是已禁用消息跟踪。
      高速缓存
      此业务服务已启用还是已禁用结果高速缓存。

您还可以使用此页执行以下操作：

  • 要删除搜索筛选器并显示所有业务服务，请单击查看全部。
  • 要放弃更改，并通过使用相同搜索条件的当前已存储设置来刷新页，请单击重置。
  • 要在会话中启用或禁用任何服务级设置，请选中或清除它旁边的复选框并单击更新。
    • 注： 服务级设置的运行时效果取决于相应的全局设置。必须同时启用全局和服务级设置，才能在运行时完全启用服务。此外，SLA 预警依赖于全局和服务级监视状态，必须启用这两者才能在运行时启用 SLA 预警。请参阅启用全局设置。
    注： 此外，还必须启用服务的状态。
  • 要定制操作设置信息的视图，请单击表定制程序图标。请参阅定制表视图。
