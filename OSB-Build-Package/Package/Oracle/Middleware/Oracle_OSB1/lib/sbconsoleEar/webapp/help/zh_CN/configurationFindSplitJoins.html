查找拆分-联接

  1. 选择操作 > 智能搜索。
  2. 从类型下拉列表中选择拆分-联接，然后单击搜索。拆分-联接概要页显示表 26-4 中所示的信息。
  3. 要限制列表中的项目个数或查找特定业务服务，请执行以下操作：
    • 单击打开图标可显示其他搜索筛选器。如果指定多个搜索条件，则仅返回与所有条件均匹配的结果。
    • 按拆分-联接的名称和路径进行筛选。在名称和路径字段中，分别输入搜索目标的名称和路径，然后单击搜索。该路径由项目名和拆分-联接所在的文件夹名称组成。可将 * 和 ? 用作通配符以执行更全面的搜索。
    • 按带有日志记录的拆分-联接进行筛选。从日志下拉列表中，选择拆分-联接的日志记录状态：“已启用”或“已禁用”，然后单击搜索。选择“所有”可忽略日志记录筛选条件。
    • 按执行跟踪进行筛选。从执行跟踪下拉列表中，选择拆分-联接的执行跟踪状态：“已启用”或“已禁用”，然后单击搜索。选择“所有”可忽略服务日志记录筛选条件。

      • 表 26-4 拆分-联接概要信息
      属性
      说明
      名称
      分配给拆分-联接的名称。此名称是指向操作设置页的链接。请参阅编辑拆分-联接。
      路径
      与拆分-联接关联的项目。如果拆分-联接位于项目文件夹中，则还会列出此文件夹。此路径按以下格式显示：
      project-name/root-folder/ . . ./parent-folder
      此路径是指向项目浏览器中相应路径的链接。
      类型
      父服务类型：拆分-联接。
      日志
      日志记录状态：已启用或已禁用，在以下级别以及更高级别启用：调试 (D)、信息 (I)、警告 (W) 或错误 (E)。
      执行跟踪
      此拆分-联接的运行时跟踪状态：“已启用”或“已禁用”。

您还可以使用此页执行以下操作：

  • 要删除搜索筛选器并显示所有拆分-联接，请单击查看全部。
  • 要放弃更改，并通过使用相同搜索条件的当前已存储设置来刷新页，请单击重置。
  • 要在会话中启用或禁用任何服务级设置，请选中或清除它旁边的复选框并单击更新。
    • 注： 拆分-联接级设置的运行时效果取决于相应的全局设置。必须同时启用全局和服务级设置，才能在运行时完全启用服务。
  • 要定制操作设置信息的视图，请单击表定制程序图标。请参阅定制表视图。
