启动 Oracle Service Bus 管理控制台

  1. 使用以下方法之一启动 Oracle Service Bus：
    • 从 Windows 开始菜单中选择：

      • 开始 > 程序 > Oracle WebLogic > User Projects > domain_name > Start Server for Oracle Service Bus Domain。

      其中 domain_name 表示您创建 Oracle Service Bus 域时为其指定的名称。用户项目是默认情况下将域创建到的默认位置。

    • 从 Oracle Service Bus 域的根中调用 startWebLogic 命令：startWebLogic.cmd 或 startWebLogic.sh

      • 此时将显示命令窗口，其中显示有关 WebLogic 服务器的状态信息。

      如果您以生产模式启动服务器，则需要输入有效的用户名和口令才能启动该服务器。

  2. 服务器启动后，请在浏览器中输入下面的 URL：

    3. http://hostname:port/sbconsole

    其中 host 表示在其上正运行 WebLogic 服务器的计算机的名称，port 表示端口号。

    例如，如果 WebLogic 服务器正在使用默认端口配置在本地计算机上运行，则请在浏览器中输入下面的 URL：

    http://localhost:7001/sbconsole

    此时会显示 Oracle Service Bus 管理控制台登录页。

  3. 输入在安装过程中指定的用户名和口令。
    4. 注： Oracle Service Bus 示例域的用户名和口令为 weblogic/welcome1。
  4. 单击登录。

若要注销，请在 Oracle Service Bus 管理控制台的右侧面板顶部附近的标帜上，单击注销。
