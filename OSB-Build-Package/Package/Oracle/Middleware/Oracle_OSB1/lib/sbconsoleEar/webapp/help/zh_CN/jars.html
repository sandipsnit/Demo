JAR

JAR (Java 归档文件) 是包含一组 Java 类的压缩文件。它用于存储可组成程序的已编译的 Java 类和相关联的元数据。JAR 的作用就像 Java 代码元素的可调用程序库一样 (这样，单个编译链接就可提供对多个元素的访问，而无须对每个元素单独绑定)。要了解有关 JAR 的详细信息，请参阅 http://java.sun.com/docs/books/tutorial/deployment/jar/。

Oracle Service Bus 中的 JAR 资源可用于：

  • Java 标注操作
  • 基于 EJB 的业务服务
  • 基于 Tuxedo 的代理服务和业务服务
