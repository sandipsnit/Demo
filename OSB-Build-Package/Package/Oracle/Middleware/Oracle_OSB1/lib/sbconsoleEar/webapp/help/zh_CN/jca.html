JCA 绑定

通过 Oracle SOA Suite，您可以使用 J2EE Connector Architecture (JCA) 适配器与后端 Enterprise Information Systems (EIS) 集成。使用 Oracle JDeveloper 和其他受支持的 JCA 开发环境时，您可以配置适配器并创建用于后端集成的 JCA 适配器资源 (.jca 文件和抽象 WSDL)。

通过 Oracle Service Bus (OSB) 可以与符合 JCA 的 EIS 组件通信，这样您就可以使用 JCA 绑定资源形式的 JCA 资源。借助 JCA 绑定，您可以生成使用 Oracle Service Bus JCA 传输并通过 JCA 适配器与 EIS 应用程序通信的代理和业务服务。

本章介绍如何在 Oracle Service Bus 中添加、编辑、查找和删除 JCA 绑定，其中包含以下主题：

  • 查找 JCA 绑定
  • 添加 JCA 绑定
  • 编辑 JCA 绑定
  • 删除 JCA 绑定

有关 JCA 传输的详细信息，请参阅 Oracle Fusion Middleware Developer's Guide for Oracle Service Bus 中的“JCA Transport”，网址为：http://www.oracle.com/pls/as111170/lookup?id=OSBDV910。

有关 Oracle SOA Suite JCA 适配器的详细信息，请参阅 Oracle Fusion Middleware User's Guide for Technology Adapters，网址为：http://www.oracle.com/pls/as111170/lookup?id=TKADP。
