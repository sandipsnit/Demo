删除 JCA 绑定

如果任何业务服务或代理服务基于 JCA 绑定，则在删除 JCA 绑定之前，请先从该业务服务或代理服务中删除 JCA 绑定。

  1. 如果尚未执行此操作，请单击创建新建一个会话，或单击编辑进入现有会话。请参阅使用更改中心。
  2. 选择资源浏览器 > JCA 绑定。
  3. 在要删除的 JCA 绑定的选项字段中，单击删除图标。

    4. 此 JCA 绑定将会从当前会话中删除。如果业务服务或代理服务已配置为使用 JCA 绑定，则删除警告图标表明您可在确认警告后删除 JCA 绑定。这可能会由于服务中存在对已删除 JCA 绑定的未解析引用而导致冲突。

  4. 要结束该会话并将配置部署到运行时，请单击更改中心下的激活。
