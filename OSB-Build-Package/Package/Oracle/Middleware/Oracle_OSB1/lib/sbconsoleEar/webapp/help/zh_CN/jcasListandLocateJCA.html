查找 JCA 绑定

JCA 绑定概要页列出了已添加到 Oracle Service Bus 域的 JCA 绑定。

  1. 请执行以下两个操作之一：
    • 选择项目浏览器，以显示“项目视图”页或“项目/文件夹视图”页。然后，在项目和文件夹中导航以查找 JCA 绑定。
    • 选择资源浏览器 > JCA 绑定。JCA 绑定概要页显示表 12-1 中所示的信息。有关属性的更详细说明，请参阅查找 JCA 绑定。
  2. 要查找特定 JCA 绑定，请执行以下操作：
    • 要限制列表中的项数，您可以先按名称和路径进行筛选，然后单击搜索。

      • 路径由项目名和资源所在的文件夹名称组成。

    • 单击查看全部可删除搜索筛选器，并显示所有 JCA 绑定。

      • 表 12-1 JCA 绑定信息
      属性
      说明
      JCA 绑定名称
      分配给 JCA 绑定的唯一名称。单击该名称可显示查看 JCA 绑定详细资料页。请参阅编辑 JCA 绑定。
      路径
      JCA 绑定所在的项目名称和文件夹名称。单击该名称可查看包含此资源的项目或文件夹。请参阅使用项目和文件夹限定资源名。
      操作
      包含的“生成”图标可用于从 JCA 绑定生成 Oracle Service Bus WSDL 和 JCA 服务。
      选项
      包含删除图标。如果业务服务或代理服务已配置为使用 JCA 绑定，则删除警告图标将表明您可在确认警告后删除 JCA 绑定。这可能会由于存在服务对已删除 JCA 绑定的未解析引用而导致冲突。请参阅删除 JCA 绑定。
