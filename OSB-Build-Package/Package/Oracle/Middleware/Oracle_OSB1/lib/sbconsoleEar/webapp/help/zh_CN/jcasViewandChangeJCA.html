编辑 JCA 绑定

使用查看 JCA 绑定详细资料页可查看和更改 JCA 绑定的详细资料。

  1. 如果尚未执行此操作，请单击创建新建一个会话，或单击编辑进入现有会话。请参阅使用更改中心。
  2. 查找 JCA 绑定，如查找 JCA 绑定中所述。
  3. 单击 JCA 绑定名称。查看 JCA 绑定页显示表 12-2 中所示的信息。

    4. 表 12-2 JCA 绑定详细资料 
    属性
    说明
    上次修改者
    创建此 JCA 绑定或将其导入到配置中的用户。
    上次修改时间
    用户创建此 JCA 绑定或将其导入到配置中的日期和时间。单击日期和时间链接可查看此资源的更改历史记录。请参阅“查看更改历史记录”页。
    引用
    提供 JCA 绑定所引用的 WSDL 资源的链接。请参阅查看对资源的引用。
    引用项
    引用此 JCA 绑定的对象数。如果存在此类引用，可单击数字链接以查看对象列表。请参阅查看对资源的引用。
    “操作”图标
    当更改中心处于编辑模式时，使用“操作”可以从 JCA 绑定生成 Oracle Service Bus WSDL 和 JCA 服务。
    JCA 文件
    显示 JCA 绑定的只读视图。
    抽象 WSDL
    显示与 JCA 绑定相关联的 WSDL。如果 JCA 绑定未引用 WSDL，请单击编辑引用以引用相关联的 WSDL。
    TopLink XML
    显示 JCA 绑定所需的 TopLink 或 EclipseLink 映射文件。如果 JCA 绑定资源未引用所需的映射文件，请单击编辑引用以选择所需的映射文件。仅当 JCA 文件声明 XML 资源时，才会显示此字段。
    XSLT
    显示 JCA 绑定所需的 XSLT 文件。如果 JCA 绑定资源未引用所需的 XSLT 资源，请单击编辑引用以选择所需的 XSLT 文件。仅当 JCA 文件声明 XSLT 资源 (例如，为套接字适配器声明 XSLT 资源) 时，才会显示此字段。

  4. 要修改“说明”或“JCA 文件”，请单击编辑。

    5. 您不能更改资源名。

  5. 单击保存可提交当前会话中的更新。
  6. 要更改对 JCA 绑定所需资源的引用，或要修复未解析的引用，请单击编辑引用并选择相应的资源。
  7. 要结束该会话并将配置部署到运行时，请单击更改中心下的激活。
