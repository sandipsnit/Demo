MFL

消息格式语言 (MFL) 文档是用于描述二进制数据布局的专用 XML 文档。它是一种 Oracle 专用语言，用于定义将格式化的二进制数据转换为 XML 数据的规则。MFL 文档符合 mfl.dtd，其中包括用于描述每个数据字段的元素和属性，以及字段 (组) 的分组、重复和聚集。

创建“消息传送服务”类型的业务服务和代理服务时，可以选择 MFL 类型作为服务的请求消息类型或响应消息类型。

使用 Oracle Format Builder 可创建 MFL。定义二进制记录的层次、字段的布局以及字段和组的分组时，该信息将被保存为 MFL 文档，之后可以使用该文档执行运行时转换。还可以在 Format Builder 中使用 MFL 文档来生成描述其内容模型的相应 DTD。要了解如何使用 Format Builder，请参阅 Oracle Fusion Middleware Developer's Guide for Oracle Service Bus 中的“Format Builder”，网址为：http://www.oracle.com/pls/as111170/lookup?id=OSBDV763。
