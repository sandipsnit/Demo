监视

当创建业务服务或代理服务时，在默认情况下会禁用对该服务的监视。要了解如何启用对代理服务的监视，请参阅为代理服务配置操作设置；对于业务服务，请参阅为业务服务配置操作设置。

要启用或禁用已单独启用或禁用了监视的所有服务的监视，请使用操作 > 全局设置页上的启用监视选项。请参阅启用全局设置。

要了解有关在运行时进行监视的详细信息，请参阅 Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus 中的“Monitoring Oracle Service Bus at Run Time”，网址为：http://www.oracle.com/pls/as111170/lookup?id=OSBAG472。
