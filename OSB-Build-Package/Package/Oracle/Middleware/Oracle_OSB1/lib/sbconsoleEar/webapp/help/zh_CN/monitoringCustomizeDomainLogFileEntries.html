定制域日志文件项视图

  1. 单击定制该表。此时将显示附加字段。

    2. 可以随时单击定制该表链接来关闭此表并保留原始设置。

  2. 在筛选器字段中，请执行以下操作：
    1. 在时间字段中，选择查看日志条目的时间间隔。
    2. 在开始时间字段中，以 YYYY/MM/DD HH:MM:SS 格式输入开始时间。例如，输入 10/25/06 08:39:48。使用开始时间和结束时间字段可指定要查看日志条目的时段。
    3. 在结束时间字段中，以 YYYY/MM/DD HH:MM:SS 格式输入结束时间。例如，输入 10/25/06 13:20:51。使用开始时间和结束时间字段可指定要查看日志条目的时段。
    4. 在条件文本框中，指定筛选条件 (文本字符串)。在 WLDF 查询语言中，将筛选条件指定为字符串。此查询语言支持布尔运算符：AND、OR、NOT 和关系运算符。有关查询语法的详细信息，请参阅 Oracle Fusion Middleware Configuring and Using the Diagnostics Framework for Oracle WebLogic Server 中的“WLDF Query Language”，网址为：http://www.oracle.com/pls/as111170/lookup?id=WLDFC316。
  3. 在列字段中，选择要显示的列：
    1. 从可用字段中选择一个列名。
    2. 单击箭头可将此列名移动到所选字段中。
    3. 重复此操作，直到列出所有要在所选字段中显示的列名。

      4. 在所选字段中，可以根据需要，使用上箭头和下箭头对列名进行重新排序。

  4. 在每页上显示的行数字段中，选择要在每页显示的日志条目数。
  5. 在返回的最大结果数字段中，选择总共要显示的最大日志条目数；或者选择显示全部可显示所有内容。
  6. 请执行以下操作之一：
    • 要保存新设置，请单击应用。
    • 要放弃更改并保留原始设置，请单击重置。
