定制表视图

使用该表定制程序可根据指定的设置显示表信息。

请注意，任何表的默认排序顺序都是由该表的第一列确定的。

  1. 单击表定制程序图标。此时将显示附加字段。

    2. 可以随时单击取消来关闭此表并保留原始设置。

  2. 在列字段中，选择要显示的列：
    1. 从可用列字段中选择列名。
    2. 单击箭头可将此列名移动到所选列字段中。

      3. 星号表示必需的列。

    3. 重复此操作，直到列出所有要在所选列字段中显示的列名。

      4. 在所选列字段中，可以根据需要，使用上箭头和下箭头对列名进行重新排序。

  3. 要删除正在显示的列，请将相应列名从所选列字段移动到可用列字段中。
  4. 从行下拉列表中，选择要在单个页上显示的行数。
  5. 请执行以下操作之一：
    • 要保存这些设置并关闭表定制程序，请单击应用。
    • 要放弃更改并保留原始设置，请单击重置。
