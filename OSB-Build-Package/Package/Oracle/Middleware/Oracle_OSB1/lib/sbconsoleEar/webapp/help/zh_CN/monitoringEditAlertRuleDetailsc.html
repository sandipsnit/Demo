编辑预警规则

  1. 如果尚未执行此操作，请单击创建新建一个会话，或单击编辑进入现有会话。请参阅使用更改中心。
  2. 查找预警规则。请参阅查找预警规则。
  3. 单击该预警规则名称。
  4. 要更改配置页上的字段，请单击相应页的编辑图标或单击页底部的编辑按钮。有关这些页和字段的说明，请参阅创建和编辑预警规则。
  5. 请执行下列操作之一：
    • 要转到下一页，请单击“下一页”。
    • 要返回上一页，请单击“上一页”。
    • 要转到概要页，请单击“最后一页”。
    • 要提交当前会话中的更新，请单击“保存”。
    • 要放弃更改并返回“SLA 预警规则”页，请单击“取消”。
  6. 要结束该会话并将配置部署到运行时，请单击“更改中心”下的“激活”。
