查看预警规则

使用此页可查看表 25-19 中所示的预警规则信息。

表 25-19 预警规则信息
属性
说明
名称
分配给此预警规则的名称。该名称是一个指向预警规则一般配置页的链接。请参阅查看预警规则配置。
SLA 状态
此预警规则的状态：“已启用”或者“已禁用”。
说明
默认情况下，将隐藏此字段。
预警规则的说明。
严重性
此规则触发的预警严重性：“普通”、“警告”、“次要”、“主要”、“严重”或“致命”。
聚集间隔
聚集间隔的长度 (小时和分钟)。
失效日期
此预警规则不再有效的日期。
停止处理
显示“是”或“否”。
频率
此预警的频率：
  • 每次
  • 通知一次
选项
此列包括以下功能：
  • 单击这些箭头可对预警规则重新排序。可以将各个预警规则在列表中上下移动。只有在为此服务配置了多个预警规则的情况下，才能执行此操作。
  • 单击重命名图标可重命名预警规则。
  • 单击删除图标可删除特定的预警规则。请参阅删除预警规则。
