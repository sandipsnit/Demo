查找预警

  1. 选择操作 > 面板。
  2. 选择 SLA 预警或管道预警选项卡以便显示 SLA 预警或管道预警。
  3. 单击扩展预警历史记录链接。扩展 SLA 预警历史记录页和扩展管道预警历史记录页显示预警历史记录详细资料。
  4. 要限制列表中的项目个数或查找特定预警，请执行以下操作：
    • 单击打开图标可显示其他搜索筛选器。如果指定多个搜索条件，则仅返回与所有条件均匹配的结果。
    • 按日期范围进行筛选。执行以下操作之一：
      • 选择所有可忽略日期范围和时间间隔筛选条件，并显示所有预警。
      • 在从和到字段中，以 MM/DD/YY HH:MM AM|PM 格式输入日期和时间，然后单击搜索。此时将显示在指定日期和时间范围内发生的预警。
      • 在最后一次字段中，输入天数，然后指定小时数和分钟数，然后单击搜索。此时将显示在指定时间范围内发生的预警。
    • 按预警严重性进行筛选。从预警严重性下拉列表中选择条目，将预警限制到指定严重性级别，然后单击搜索。

      • 选中或更高级别复选框可将搜索限制到指定严重性级别或更高级别 (按限制性最低级别到最高级别列出)：“普通”、“警告”、“次要”、“主要”、“严重”和“致命”。

    • 按父服务进行筛选。单击浏览以便显示选择服务页。从列表中选择所需的父服务，并单击提交。单击清除可删除父服务筛选器条件，或单击搜索。
    • 仅适用于 SLA 预警：按预警名 (任意字符串) 进行筛选。输入包括通配符在内的任意字符串，然后单击搜索。有关详细信息，请参阅 WebLogic 诊断框架查询语言。
    • 仅适用于管道预警：按预警概要 (任意字符串) 进行筛选。输入包括通配符在内的任意字符串，然后单击搜索。有关详细信息，请参阅 WebLogic 诊断框架查询语言。要查找已设计为不带预警概要文本的管道预警操作，请输入“Oracle Service Bus 预警”。
    • 单击查看全部可删除搜索筛选器，并显示所有预警。

使用扩展 SLA 预警历史记录页和扩展管道预警历史记录页还可以执行以下操作：

  • 要以饼图形式显示预警数据，请选择饼图视图。
  • 要以条形图形式显示相同的数据，请选择条形图视图。
  • 要以表格形式显示相同的数据，请选择表视图。
  • 要显示数据的更新版本，请单击刷新。
  • 要删除单个预警，请选中相应的复选框，然后单击该页底部的删除。
  • 要删除所有预警或在指定日期和时间范围内发出的预警，请单击该页顶部的清除预警历史记录链接。请参阅清除预警。

WebLogic 诊断框架查询语言

预警是使用 WebLogic 诊断框架进行存储的，该框架提供自身的查询语言，可使用通配符。要在扩展预警历史记录中筛选预警，请使用 Oracle Fusion Middleware Configuring and Using the Diagnostics Framework for Oracle WebLogic Server 的“WLDF Query Language”中所述的语法，网址为：http://www.oracle.com/pls/as111170/lookup?id=WLDFC316。
