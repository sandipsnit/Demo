清除预警

使用此页可永久删除指定的 SLA 预警或管道预警。

注： 此操作不能撤消。
  1. 选择操作 > 面板。
  2. 选择 SLA 预警或管道预警选项卡，然后单击扩展预警历史记录链接。
  3. 在扩展预警历史记录页的右上侧，单击清除 SLA 预警历史记录或清除管道预警历史记录。
  4. 执行表 25-3 中所示的以下步骤之一。

    5. 表 25-3 预警清除选项
    操作
    步骤
    • 清除所有 SLA 预警
    • 清除所有管道预警
    1. 选择清除所有预警。
    2. 单击清除。
    • 清除指定时段内的 SLA 预警
    • 清除指定时段内的管道预警
    1. 选择清除预警自和到。
    2. 在清除开始字段中，输入月、日、年和时间。
    3. 在清除结束字段中，输入月、日、年和时间。
    4. 单击清除。

注： 如果要从中清除预警的服务器不可用，日志记录框架会保留清除记录，并每 30 分钟轮询一次服务器以检查其可用性。当日志记录框架发现服务器可用时，它会清除预警。
