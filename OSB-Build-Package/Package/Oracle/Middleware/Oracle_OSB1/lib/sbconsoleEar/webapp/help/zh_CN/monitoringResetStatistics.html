重置服务的统计信息

可使用服务健康状况页重置配置中所有服务的监视统计信息。无论是否在会话中，您都可以重置统计信息。

  1. 选择操作 > 面板 > 服务健康状况。
  2. 在显示统计信息字段中，选择自从上次重置。“自从上次重置”视图显示度量的累计动态计数视图。
  3. 要重置特定服务的监视统计信息，请在该服务的操作列中单击重置图标。

    4. 如果确认要重置统计信息，则系统会删除从上次重置统计信息以来为该服务收集的所有监视统计信息。但系统不删除当前聚集间隔内为该服务收集的统计信息。此外，单击重置图标后，系统将立即再次开始为服务重新收集监视统计信息。

  4. 要重置所有服务的监视统计信息，请单击页顶部的重置所有统计信息链接。
    5. 注： 单击重置所有统计信息将重置所有被监视服务的统计信息，而无论其是否显示在页上。

    如果确认要重置统计信息，则系统会删除从上次重置统计信息以来，为配置中的所有服务收集的所有监视统计信息。系统不删除在当前聚集间隔期间为任何服务收集的统计信息。此外，单击重置所有统计信息后，系统将立即再次开始为服务重新收集监视统计信息。
