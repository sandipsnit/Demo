设置服务的聚集间隔

使用操作设置页可以为特定服务设置聚集间隔。聚集间隔指的是一个时段，在此时段内会计算要在 Oracle Service Bus 管理控制台中显示的聚集统计信息。默认的聚集间隔设置为 10 分钟。

  1. 如果尚未执行此操作，请单击创建新建一个会话，或单击编辑进入现有会话。请参阅使用更改中心。
  2. 选择资源浏览器 > 业务服务或代理服务。
  3. 单击服务名。
  4. 选择操作设置选项卡。
  5. 在聚集间隔字段中，以小时和分钟数选择聚集间隔的长度，然后单击更新。

    6. 如果选择的小时数超过 1，则默认选择的分钟数始终为零。但如果选择的小时数为 0 或 1，则可以根据分钟数配置间隔。

  6. 单击更新可提交当前会话中的更新。
  7. 要结束该会话并将配置部署到运行时，请单击更改中心下的激活。
