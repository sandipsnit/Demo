查看操作度量

使用此页可查看表 25-10 中所示的服务监视信息，以及表 25-14 中所示的、具有管道的代理服务的操作度量。

表 25-14 操作度量
属性
说明
节点
嵌套操作的分层显示。
计数
如果选择了显示统计信息字段中的当前聚集间隔，则显示当前聚集间隔的时段内的调用次数。
如果选择了显示统计信息字段中的自从上次重置，则显示自上次重置统计信息以来的时段内的调用次数。
请注意，它不同于消息数，因为可以对每个消息调用一个操作多次。
平均时间
如果选择了显示统计信息字段中的当前聚集间隔，则显示当前聚集间隔的时段内每次调用的平均执行时间。
如果选择了显示统计信息字段中的自从上次重置，则显示自上次重置统计信息以来的时段内每次调用的平均执行时间。
总时间
如果选择了显示统计信息字段中的当前聚集间隔，则显示当前聚集间隔的时段内的总执行时间。
如果选择了显示统计信息字段中的自从上次重置，则显示自上次重置统计信息以来的时段内的总执行时间。
