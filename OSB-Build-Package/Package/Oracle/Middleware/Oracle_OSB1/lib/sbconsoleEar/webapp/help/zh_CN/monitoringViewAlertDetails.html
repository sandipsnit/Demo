查看预警详细资料

此页面显示特定预警的详细资料，其中包括表 25-4 中所示的信息。

表 25-4 预警详细资料
属性
说明
预警名
仅适用于 SLA 预警：
指定给此预警的名称。该名称是一个指向预警规则配置页的链接。请参阅查看预警规则配置。
预警概要
仅适用于管道预警：
管道预警操作的简要说明或 Oracle Service Bus 预警 (为已设计为不带预警概要文本的管道预警操作提供的默认值)。
说明
预警规则的说明。
时间戳
  • 对于管道预警：预警发生的日期和时间。
  • 对于 SLA 预警：预警规则上次求值结果为“真”的日期和时间。
严重性
用户定义的预警严重性。
  • 致命
  • 严重
  • 主要
  • 次要
  • 警告
  • 普通
服务
与此预警相关联的服务的名称和路径。此字段是一个指向服务监视详细资料页的链接 (仅当主动监视服务时)。请参阅查看服务度量。
服务类型
父服务类型：代理服务或业务服务。
只有代理服务才能定义管道预警，因此，对于管道预警，所有父服务都是代理服务。
服务器
生成此预警的服务器的名称。
仅适用于 SLA 预警：显示 N/A。
批注
可以在其中输入此预警的注释的文本框。对于具有批注的预警，会在“预警历史记录”中显示批注图标。
