查看管道预警

使用管道预警页可查看表 25-2 中所示的管道预警概要，并可访问该表中所示的预警历史记录详细信息。

管道预警页中包括以下内容：

  • 用于显示 SLA 预警、服务健康状况和服务器健康状况信息的选项卡。

    • 在“管道预警”选项卡标题上，一个数字指示符会显示自上次查看预警以来或上一个面板刷新时段内 (两者之间取较近的时间) 发生的新预警的数目。默认情况下，面板刷新率为不刷新。要更改面板刷新率间隔，请参阅设置用户首选项。

注意： 可通过选择默认值 (不刷新) 之外的面板刷新率间隔，使浏览器定期刷新面板。这可防止会话超时，即使在不与 Oracle Service Bus 管理控制台交互的情况下，也是如此。
注意： Oracle 建议您谨慎使用面板刷新率功能。绝不应将会话置于无人值守模式。当会话处于活动状态时，其他人可从您的计算机访问该会话而无须重新进行验证。
  • 过去 30 分钟 (默认预警历史记录持续间隔) 内发生管道预警的服务的饼图。要更改显示预警历史数据的时间间隔，请参阅设置用户首选项。

    • 对于同一时间间隔，该页上的管道预警数与按严重性级别显示预警细分数据的饼图相匹配。可以单击该饼图中与预警严重性相对应的区域以便显示详细信息。

  • 按服务发出的管道预警数降序列出的前 10 个具有管道预警的服务。每个服务名都是一个指向服务监视详细资料的链接。请参阅查看服务度量。
  • 根据预警表的定制设置显示预警信息的字段。例如，时间戳、预警概要、预警严重性、操作以及与此预警关联的服务和服务类型的名称。要了解有关这些字段的详细信息，请参阅表 25-2。

    • 预警概要旁边的批注图标表示批注。将鼠标指针放在此图标上可以预览批注文本。单击此图标可查看整个批注文本。

  • 表定制程序图标。请参阅定制表视图。
  • 指向扩展预警历史记录页的链接。

    • 表 25-2 管道预警历史记录信息
    属性
    说明
    时间戳
    预警发生的日期和时间。
    默认情况下，所有预警都按照时间戳排序。
    预警概要
    管道预警操作的简要说明或 Oracle Service Bus 预警 (为已设计为不带预警概要文本的管道预警操作提供的默认值)。预警概要是一个指向预警详细资料页的链接。请参阅查看预警详细资料。
    预警概要旁边的批注图标表示批注。将鼠标指针放在此图标上可以预览批注文本。
    预警严重性
    用户定义的预警严重性。
    • 致命
    • 严重
    • 主要
    • 次要
    • 警告
    • 普通
    服务
    与预警关联的服务的名称。此名称是一个指向服务监视详细资料页的链接 (仅当主动监视服务时)。请参阅查看服务度量。
    服务类型
    只有代理服务才能定义管道预警，因此，对于管道预警，所有父服务都是代理服务。
    操作
    编辑消息流图标是指向该代理服务的“编辑消息流”页的链接。
