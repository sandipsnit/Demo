查看业务服务端点 URI 度量

使用此页可查看表 25-10 中所示的服务监视信息，以及表 25-15 中所示的业务服务的端点 URI 度量。有关详细信息，请参阅 Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus 中的“Managing Endpoint URIs for Business Services”，网址为：http://www.oracle.com/pls/as111170/lookup?id=OSBAG595。

表 25-15 端点 URI 度量
属性
说明
端点 URI
与此业务服务相关联的端点 URL。
消息计数
如果选择了显示统计信息字段中的当前聚集间隔，则显示当前聚集间隔的时段内此端点 URI 所执行的消息总数。
如果选择了显示统计信息字段中的自从上次重置，则显示自上次重置统计信息以来的时段内此端点 URI 所执行的消息总数。
错误计数
如果选择了显示统计信息字段中的当前聚集间隔，则显示当前聚集间隔的时段内此端点 URI 所执行的包含错误的消息数。
如果选择了显示统计信息字段中的自从上次重置，则显示自上次重置统计信息以来的时段内此端点 URI 所执行的包含错误的消息数。
最短响应时间
如果选择了显示统计信息字段中的当前聚集间隔，则显示当前聚集间隔的时段内此端点 URI 执行消息所需的最短响应时间。
如果选择了显示统计信息字段中的自从上次重置，则显示自上次重置统计信息以来的时段内此端点 URI 执行消息所需的最短响应时间。
最长响应时间
如果选择了显示统计信息字段中的当前聚集间隔，则显示当前聚集间隔的时段内此端点 URI 执行消息所需的最长响应时间。
如果选择了显示统计信息字段中的自从上次重置，则显示自上次重置统计信息以来的时段内此端点 URI 执行消息所需的最长响应时间。
平均响应时间
如果选择了显示统计信息字段中的当前聚集间隔，则显示当前聚集间隔的时段内此端点 URI 执行消息所需的总平均时间。
如果选择了显示统计信息字段中的自从上次重置，则显示自上次重置统计信息以来的时段内此端点 URI 执行消息所需的总平均时间。
状态
业务服务端点 URI 的状态。
对于单个服务器域或集群中的单独托管服务器：“联机”或“脱机”。
对于集群级统计信息：“联机”、“脱机”或“部分”(当特定 URI 在集群中的至少一个托管服务器上处于脱机状态时)。
操作
单击标记联机端点图标可将脱机 URI 标记为“联机”。仅当 URI 的状态为“脱机”或“部分”时，才启用此操作。
要启用此操作，必须启用业务服务以便使 URI 处于脱机状态，并且必须启用其服务和全局级监视操作设置。请参阅为业务服务配置操作设置。
注： 如果成功，则操作是即时的。但是，反映其端点 URI 现处于联机状态则需要经过一个监视聚集循环，这可能需要 1.5 分钟。
