查看管道度量

使用此页可查看表 25-10 中所示的服务监视信息，以及表 25-13 中所示的、具有管道的代理服务的管道度量。

表 25-13 管道度量
属性
说明
组件名
组件的名称。
消息计数
如果选择了显示统计信息字段中的当前聚集间隔，则显示当前聚集间隔的时段内该组件所执行的消息总数。
如果选择了显示统计信息字段中的自从上次重置，则显示自上次重置统计信息以来的时段内该组件所执行的消息总数。
错误计数
如果选择了显示统计信息字段中的当前聚集间隔，则显示当前聚集间隔的时段内该组件所执行的包含错误的消息数。
如果选择了显示统计信息字段中的自从上次重置，则显示自上次重置统计信息以来的时段内该组件所执行的包含错误的消息数。
最短响应时间
如果选择了显示统计信息字段中的当前聚集间隔，则显示当前聚集间隔的时段内此组件执行消息所用的最短响应时间。
如果选择了显示统计信息字段中的自从上次重置，则显示自上次重置统计信息以来的时段内此组件执行消息所用的最短响应时间。
最长响应时间
如果选择了显示统计信息字段中的当前聚集间隔，则显示当前聚集间隔的时段内此组件执行消息所用的最长响应时间。
如果选择了显示统计信息字段中的自从上次重置，则显示自上次重置统计信息以来的时段内此组件执行消息所用的最长响应时间。
平均响应时间 (毫秒)
如果选择了显示统计信息字段中的当前聚集间隔，则显示当前聚集间隔的时段内该组件执行消息所用的总平均时间。
如果选择了显示统计信息字段中的自从上次重置，则显示自上次重置统计信息以来的时段内该组件执行消息所用的总平均时间。
