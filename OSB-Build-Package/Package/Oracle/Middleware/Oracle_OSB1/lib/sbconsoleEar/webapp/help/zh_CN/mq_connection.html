MQ 连接

注： 有关 Oracle Service Bus 上支持的 WebSphere MQ 版本，请参阅 Oracle Fusion Middleware Supported System Configurations，网址为：http://www.oracle.com/technology/software/products/ias/files/fusion_certification.html。WebSphere MQ 客户机库不随 Oracle Service Bus 安装程序一起提供。请从 WebSphere MQ 安装中获取 WebSphere MQ 客户机库。

MQ 连接是可在多个 MQ 代理服务和业务服务上重用的可共享资源。在访问 MQ 队列之前，MQ 代理服务和业务服务必须连接到 MQ 队列管理器。MQ 连接资源提供连接到 MQ 队列管理器所需的连接参数。

每个 MQ 连接资源都具有一个连接池。使用给定 MQ 连接资源获取到给定队列管理器的连接的每个业务或代理服务，均使用为该资源创建的相同连接池。因此，使用相同队列管理器的多个业务服务和代理服务共享一个连接池。

要了解有关 Oracle Service Bus MQ 连接资源和本机 MQ 传输的详细信息，请参阅 Oracle Fusion Middleware Developer's Guide for Oracle Service Bus 中的“MQ Transport”，网址为：http://www.oracle.com/pls/as111170/lookup?id=OSBDV1117。

要了解有关 WebSphere MQ 基础的详细信息，请参阅 http://www.redbooks.ibm.com/redbooks/SG247128/wwhelp/wwhimpl/java/html/wwhelp.htm。
