添加定制 WS-Policy

可以导入包含您的 WS-Policy 的 XML 文件，也可以在 Oracle Service Bus 管理控制台中直接编写 WS-Policy。

  1. 如果尚未执行此操作，请单击创建新建一个会话，或单击编辑进入现有会话。请参阅使用更改中心。
  2. 选择项目浏览器，然后选择要在其中添加 WS-Policy 的项目或文件夹。此时将显示项目/文件夹视图页。
  3. 从创建资源下拉列表中，选择接口下的 WS-Policy 以便显示创建新的 WS-Policy 页。
  4. 在资源名字段中，输入此 WS-Policy 的唯一名称。

    如需命名指导，请参阅资源命名限制。
  5. 在资源说明字段中，输入此 WS-Policy 的说明。
  6. 在策略字段中，执行以下操作之一：
    • 单击浏览，定位并导入包含您的定制 WS-Policy 的 XML 文件。
    • 输入新 WS-Policy 的文本。
    • 将文本从现有 WS-Policy 复制并粘贴到此字段中。
  7. 单击保存。此时将在当前会话中保存该 WS-Policy。
  8. 要结束该会话并将配置部署到运行时，请单击更改中心下的激活。
