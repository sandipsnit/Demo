查找定制 WS-Policy

WS-Policy 概要页列出了已添加到当前 Oracle Service Bus 域的定制 Web 服务策略 (WS-Policy)。Oracle Service Bus 管理控制台不会显示已预打包并可用于所有 Oracle Service Bus 域的 Oracle Service Bus WS-Policy。

  1. 请执行以下两个操作之一：
    • 选择项目浏览器，以便显示“项目视图”页或“项目/文件夹视图”页。然后，在项目和文件夹中导航以查找 WS-Policy。
    • 选择资源浏览器 > WS-Policy。WS-Policy 概要页显示表 11-1 中所示的信息。有关属性的更详细说明，请参阅查找定制 WS-Policy。
  2. 要搜索定制 WS-Policy，请在名称字段中输入部分或完整的 WS-Policy 名称。还可以在路径字段中输入部分或完整的 WS-Policy 项目名和文件夹名称。单击搜索。

    3. 单击查看全部可删除搜索筛选器，并显示所有 WS-Policy。

    表 11-1 WS-Policy 信息
    属性
    说明
    WS-Policy 名称
    指定给 WS-Policy 的唯一名称。单击该名称可查看查看 WS-Policy 详细资料页。请参阅编辑定制 WS-Policy。
    路径
    WS-Policy 所在的项目名和文件夹名称。单击该名称可查看包含此资源的项目或文件夹。请参阅使用项目和文件夹限定资源名。
    选项
    包含删除图标。如果业务服务或代理服务已配置为使用 WS-Policy，则删除警告图标表明您可在确认警告后删除 WS-Policy。这可能会由于服务中存在对已删除 WS-Policy 的未解析引用而导致冲突。请参阅删除定制 WS-Policy。
