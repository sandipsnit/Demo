编辑定制 WS-Policy

使用查看 WS-Policy 详细资料页可查看和更改特定定制 Web 服务策略 (WS-Policy) 的详细信息。Oracle Service Bus 管理控制台不显示只读的 Oracle Service Bus WS-Policy。

  1. 如果尚未执行此操作，请单击创建新建一个会话，或单击编辑进入现有会话。请参阅使用更改中心。
  2. 查找 WS-Policy，如查找定制 WS-Policy 中所述。
  3. 单击该 WS-Policy 名称。查看 WS-Policy 详细资料页将显示表 11-2 中所示的信息。

    4. 表 11-2 WS-Policy 详细资料 
    属性
    说明
    上次修改者
    创建此 WS-Policy 或将其导入到配置中的用户。
    上次修改时间
    用户创建此 WS-Policy 或将其导入到配置中的日期和时间。单击日期和时间链接可查看此资源的更改历史记录。请参阅“查看更改历史记录”页。
    引用
    此 WS-Policy 引用的对象数。如果存在此类引用，可单击数字链接查看对象列表。请参阅查看对资源的引用。
    引用项
    引用此 WS-Policy 的对象数。如果存在此类引用，可单击数字链接查看对象列表。请参阅查看对资源的引用。
    说明
    此 WS-Policy 的说明 (如果存在)。
    策略
    此 WS-Policy 的文本。

  4. 要对这些字段进行更改，请单击编辑。有关这些字段的说明，请参阅添加定制 WS-Policy。

    5. 您不能更改资源名字段。

  5. 单击保存可提交当前会话中的更新。
  6. 要结束该会话并将配置部署到运行时，请单击更改中心下的激活。
