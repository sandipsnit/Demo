克隆项目

克隆 (复制) 项目的方法有下面三种：

  • 在根级别使用新名称克隆项目。在这种情况下，原始项目及其克隆项目作为对等项目存在。克隆的项目与原始项目包含相同的文件夹和资源 (至少在修改源项目或克隆项目之前如此)。有关说明，请参阅在根级别克隆项目。
  • 使用现有项目 (而不是源项目) 的名称克隆项目。在这种情况下，源项目中的文件夹和资源与具有克隆项目名称的项目的文件夹和资源进行合并。原始项目保持不变。有关说明，请参阅在根级别克隆项目。
  • 在新位置、现有项目或项目文件夹中克隆项目。在这种情况下，克隆的项目降级到新位置的文件夹中。它与原始项目包含相同的文件夹和资源。

    • 如果在现有文件夹所在的位置使用相同的名称克隆项目，则项目的内容 (子文件夹和资源) 与目标文件夹的内容进行合并。

    无论何种情况，原始项目都保持不变。

    有关说明，请参阅克隆项目并将克隆的项目降级到文件夹。

在根级别克隆项目
  1. 如果尚未执行此操作，请单击创建新建一个会话，或单击编辑进入现有会话。请参阅使用更改中心。
  2. 选择项目浏览器 > 项目，以便显示“项目视图”页。
  3. 在要克隆的项目所在的行中，单击克隆项目图标。此时将显示克隆项目页，其中突出显示了项目的当前位置。
  4. 在新名称字段中，输入克隆项目的名称。请参阅命名项目和文件夹。

    5. 使用唯一的名称将克隆项目创建为新项目，或者使用现有项目的名称将克隆项目与该项目进行合并。

  5. 在根级别克隆项目时，保留位置面板中的默认根位置。
  6. 单击克隆项目。此时将在当前会话中克隆项目。
  7. 要结束该会话并将配置部署到运行时，请单击更改中心下的激活。
克隆项目并将克隆的项目降级到文件夹
  1. 如果尚未执行此操作，请单击创建新建一个会话，或单击编辑进入现有会话。请参阅使用更改中心。
  2. 选择项目浏览器 > 项目，以便显示“项目视图”页。
  3. 在要克隆的项目所在的行中，单击克隆项目图标。此时将显示克隆项目页，其中突出显示了项目的当前位置。
  4. 也可以在新名称字段中为文件夹选择输入一个新名称。请参阅命名项目和文件夹。

    5. 使用唯一的名称新建文件夹，或者使用现有文件夹的名称将克隆的项目与现有文件夹进行合并 (在下一步中选择的位置)。

  5. 在位置面板中，选择一个项目或文件夹，然后单击克隆项目。克隆项目将转换为新位置中的文件夹，并保存在当前会话中。
  6. 要结束该会话并将配置部署到运行时，请单击更改中心下的激活。
