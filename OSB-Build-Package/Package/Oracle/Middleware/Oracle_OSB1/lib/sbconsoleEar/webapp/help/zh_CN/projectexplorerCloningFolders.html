克隆文件夹

  1. 如果尚未执行此操作，请单击创建新建一个会话，或单击编辑进入现有会话。请参阅使用更改中心。
  2. 选择项目浏览器，然后导航至包含要克隆的文件夹的父项目或文件夹。此时将显示“项目/文件夹视图”页。
  3. 在要克隆的文件夹所在的行中，单击克隆文件夹图标。此时将显示克隆文件夹页，其中突出显示了文件夹的当前位置。
  4. 在同一个位置或另一个位置上克隆文件夹，或将文件夹升级到项目，如下所示：
    • 要在同一个位置克隆文件夹，请在新名称字段中输入新名称。请参阅命名项目和文件夹。

      • 在同一个位置克隆时，要么在位置面板中根本不选择任何位置 (相当于选择当前位置)，要么明确选择当前项目和文件夹。

    • 要在另一个位置克隆文件夹，也可以在新名称字段中输入新名称，然后从位置区域中选择项目和/或文件夹。请参阅命名项目和文件夹。

      • 如果使用与现有文件夹相同的名称，则原始文件夹的内容将与目标文件夹的内容进行合并。

    • 要将文件夹升级到项目，请选择根位置。
  5. 单击克隆文件夹。此时将在当前会话中克隆此文件夹。

    6. 克隆文件夹时，将以递归方式克隆该文件夹下的所有文件夹和资源

  6. 要结束该会话并将配置部署到运行时，请单击更改中心下的激活。
