克隆资源

  1. 如果尚未执行此操作，请单击创建新建一个会话，或单击编辑进入现有会话。请参阅使用更改中心。
  2. 选择项目浏览器，然后选择包含要克隆的资源的项目或文件夹。此时将显示“项目/文件夹视图”页。
  3. 在要克隆的资源所在的行中，单击克隆资源图标。此时将显示克隆资源页，其中突出显示了资源的当前位置。
  4. 可以在同一个位置或另一个位置克隆资源：
    • 要在同一个位置克隆资源，请在新资源名字段中输入新名称。请参阅资源命名限制。

      • 要在同一个位置克隆，要么在位置面板中根本不选择任何位置 (相当于选择当前位置)，要么明确选择当前项目和文件夹。

    • 要在另一个位置克隆资源，也可以在新资源名字段中输入新名称，然后从位置区域中选择项目和/或文件夹。
  5. 单击克隆资源。此时将在当前会话中克隆资源。

    6. 如果克隆从 UDDI 注册表导入的业务服务，将从该注册表分离克隆的服务

  6. 要结束该会话并将配置部署到运行时，请单击更改中心下的激活。
