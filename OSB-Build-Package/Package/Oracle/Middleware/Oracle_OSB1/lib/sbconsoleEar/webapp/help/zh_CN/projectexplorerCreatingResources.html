创建资源

  1. 如果尚未执行此操作，请单击创建新建一个会话，或单击编辑进入现有会话。请参阅使用更改中心。
  2. 选择项目浏览器，然后选择要在其中创建资源的项目或文件夹。
  3. 从“项目/文件夹视图”页上的创建资源下拉列表中，选择要创建的资源类型。根据所选的资源类型，会显示不同的页。
  4. 按照提示将信息添加至每页以创建资源。有关创建每种资源的详细信息，请参阅相应的主题，如表 3-3 所列。

  6. 要结束该会话并将配置部署到运行时，请单击更改中心下的激活。
