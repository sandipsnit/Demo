导出 WSDL

使用“项目/文件夹视图”页可导出与代理服务或业务关联的 WSDL。WSDL 是以 JAR 文件的形式导出的。仅当您位于会话之外时，才能导出 WSDL。请参阅使用更改中心。

使用“导出 WSDL”功能，使 WSDL 可供外部工具 (如 IDE) 使用。请注意，此功能与“系统管理”模块中的“导出资源”功能不同，后者用于在两个域之间移动和临时保存资源。

  1. 在左侧导航窗格中，单击项目浏览器下的项目或文件夹。此时将显示“项目/文件夹视图”页。
  2. 在要导出其 WSDL 的代理服务所在的行中，单击导出 WSDL 图标。此时将显示文件下载对话框。
  3. 在文件下载对话框中，单击打开以打开 JAR 文件，或单击保存将此 JAR 文件保存到桌面。
