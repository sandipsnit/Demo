生成 WSDL

已生成的 WSDL 是一种 WSDL 资源，此资源由 Oracle Service Bus 针对一种未使用 WSDL 资源启动但可使用 WSDL 描述的服务生成的。例如，通过基于 EJB 的服务创建的 WSDL 就是已生成的 WSDL。有关详细信息，请参阅 Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus 中的“How WSDL is Used in Oracle Service Bus”，网址为：http://www.oracle.com/pls/as111170/lookup?id=OSBAG114。

使用“项目/文件夹视图”页可生成与 EJB 或流传输类型业务服务关联的 WSDL。

  1. 在左侧导航窗格中，单击项目浏览器下的项目或文件夹。此时将显示“项目/文件夹视图”页。
  2. 在要生成 WSDL 的业务服务所在的行中，单击生成 WSDL 图标。此时将显示生成 WSDL 资源对话框，其中突出显示当前位置 (业务服务所在的项目名称和文件夹名称)。
  3. 在生成 WSDL 资源对话框中，指定新的 WSDL 资源名以及 WSDL 的位置，然后单击生成 WSDL。

    如需命名指导，请参阅资源命名限制。

完成之后

继续编辑 WSDL 以查看或修改已生成的 WSDL。
