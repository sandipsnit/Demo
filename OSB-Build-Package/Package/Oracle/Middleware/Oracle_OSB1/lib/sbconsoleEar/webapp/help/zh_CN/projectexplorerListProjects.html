“项目视图”页

项目视图页列出了在当前域中定义的项目。表 3-1 介绍了该页上的选项和控件。

表 3-1“项目视图”页 - 选项和控件
选项或控件
说明
输入新项目名称
要新建项目，请在文本字段中输入一个名称，然后单击添加项目。请参阅添加项目。
另请参阅命名项目和文件夹。
名称
单击项目名，以便为该项目显示“项目/文件夹视图”页。
选项
对于列出的任何项目，您都可以执行以下操作之一：
  • 单击重命名项目图标可重命名项目。请参阅重命名项目。
  • 单击移动项目图标可移动项目。请参阅移动项目 (转换为文件夹)。
  • 单击克隆项目图标可克隆项目。请参阅克隆项目。
  • 单击删除项目图标可删除项目。如果有其他资源引用此资源，将会显示删除警告图标。确认警告后即可删除该资源。这可能会由于存在对已删除资源的未解析引用而导致冲突。请参阅删除项目。
执行上述任何操作之后，结果将保存在当前会话中。要结束该会话并将配置部署到运行时，请单击更改中心下的激活。
