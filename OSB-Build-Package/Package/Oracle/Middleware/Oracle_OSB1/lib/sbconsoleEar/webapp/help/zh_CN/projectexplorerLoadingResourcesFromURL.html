从 URL 加载资源

您可以批量导入文件系统或某个 URL 上的可用资源。使用批量导入功能可以导入根资源 (如 WSDL) 及其从属项 (如其他 WSDL 和方案)。此时将自动解析相关性映射。

  1. 如果尚未执行此操作，请单击创建新建一个会话，或单击编辑进入现有会话。请参阅使用更改中心。
  2. 选择项目浏览器，然后选择要向其中添加资源的项目或文件夹。此时将显示“项目/文件夹视图”页。
  3. 在创建资源下拉列表中，选择 URL 上的资源。此时将显示“加载资源 - 从 URL 加载资源”页。
  4. 在 URL/路径字段中，输入资源的 URL 或路径。

    5. 要指定本地资源，可以将 file 协议与由路径完全限定的文件名结合使用，例如 file:///c:/osbresources/ForeachAction.jar。

  5. 在资源名字段中，输入资源的名称。
  6. 从资源类型下拉列表中，选择资源的类型。
  7. 单击下一步以便显示“加载资源 - 复查已加载的资源”页。如果您不希望在此导入操作中包含某些资源，则可清除这些资源旁的复选框。
  8. 要导入所选资源，请单击导入。
  9. “加载资源 - 导入结果”页显示了导入的资源。单击加载另一个从其他文件导入资源，或者从左侧导航窗格中选择另一页以继续执行其他任务。
  10. 要结束该会话并将配置部署到运行时，请单击更改中心下的激活。
