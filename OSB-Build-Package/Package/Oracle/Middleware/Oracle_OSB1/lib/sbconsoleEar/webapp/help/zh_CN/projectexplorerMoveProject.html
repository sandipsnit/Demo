移动项目 (转换为文件夹)

“移动”项目时，会将该项目转换为新位置中的一个文件夹。

  1. 如果尚未执行此操作，请单击创建新建一个会话，或单击编辑进入现有会话。请参阅使用更改中心。
  2. 选择项目浏览器 > 项目，以便显示“项目视图”页。
  3. 在要移动的项目所在的行中，单击移动项目图标。此时将显示移动项目页，其中突出显示了项目的当前位置。
  4. 在位置下，选择要将项目“移动”到其中的项目或文件夹。该项目将转换为新位置中的文件夹，并保存在当前会话中。

    5. 如果将项目移至新位置，且该位置已存在同名文件夹，则该项目的内容将与现有文件夹的内容进行合并。

  5. 要结束该会话并将配置部署到运行时，请单击更改中心下的激活。
