移动或升级文件夹

  1. 如果尚未执行此操作，请单击创建新建一个会话，或单击编辑进入现有会话。请参阅使用更改中心。
  2. 选择项目浏览器，然后选择包含要移动或升级的文件夹的项目或文件夹。此时将显示“项目/文件夹视图”页。
  3. 在要移动或升级的文件夹所在的行中，单击移动文件夹图标。此时将显示移动文件夹页，其中突出显示了文件夹的当前位置。
  4. 确定是要将文件夹移至其他项目下的新位置，还是将文件夹升级到某个项目：
    • 要将文件夹移至新位置，请在位置面板中，选择要移入该文件夹的项目或文件夹。
    • 要将文件夹升级到项目，请选择根位置。
  5. 单击提交。文件夹 (或项目，如果是从文件夹升级的) 保存在当前会话中的新位置。

    6. 当移动或升级文件夹时，如果该位置已存在同名的其他文件夹，则移动或升级的文件夹中的内容将与现有文件夹或项目中的内容合并。

  6. 要结束该会话并将配置部署到运行时，请单击更改中心下的激活。
