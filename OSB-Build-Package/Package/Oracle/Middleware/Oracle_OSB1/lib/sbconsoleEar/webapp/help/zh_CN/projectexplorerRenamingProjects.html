重命名项目

  1. 如果尚未执行此操作，请单击创建新建一个会话，或单击编辑进入现有会话。请参阅使用更改中心。
  2. 选择项目浏览器 > 项目，以便显示“项目视图”页。
  3. 在要重命名的项目所在的行中，单击重命名项目图标。此时将显示重命名项目页。
  4. 在新建项目名字段中，输入项目的新名称。请参阅命名项目和文件夹。

    5. 如果使用现有项目的名称，则两个项目的资源将以该名称进行合并。例如，考虑以下情况：项目 MyProject 包含资源 A 和资源 B，而项目 ThisProject 包含资源 C 和资源 D。如果将 MyProject 重命名为 ThisProject，则新的 ThisProject 将包含 A、B、C 和 D 四种资源。

  5. 单击重命名。此时将在当前会话中重命名该项目。
  6. 要结束该会话并将配置部署到运行时，请单击更改中心下的激活。
