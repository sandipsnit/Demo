“加载资源 - 导入结果”页

加载资源 - 导入结果页显示了从导入的 Zip 文件或从 URL 加载资源的结果。

根据是从 Zip 文件还是 URL 加载了资源，确保完成从 Zip 文件加载资源或从 URL 加载资源中的步骤，以及“加载资源 - 复查已加载的资源”页中的步骤。

此时将显示导入结果页，其中显示了以下信息。

表 3-7“项目/文件夹视图”页 - 选项和控件
选项或控件
说明
状态
资源的状态；例如，是否已成功导入资源。
在某个 WSDL 的所有相关性均被导入之前，该 WSDL 可能会临时标记为无效。但是，完成导入操作后，配置服务会重新验证标记为无效的文件。
名称
资源的名称。
路径
资源的路径，即资源所在的项目和文件夹。
资源类型
显示导入的资源类型。
错误消息
显示未成功导入资源时的错误消息的详细资料。
