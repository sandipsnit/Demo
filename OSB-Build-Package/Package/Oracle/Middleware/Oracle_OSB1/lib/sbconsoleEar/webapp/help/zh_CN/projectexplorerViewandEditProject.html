“项目/文件夹视图”页

项目/文件夹视图页显示了单个项目的详细资料。表 3-2 介绍了该页上的选项和控件。

表 3-2“项目/文件夹视图”页 - 选项和控件
选项或控件
说明
项目名
引用
单击此字段中的数字 (如果大于 0) 可显示一组此项目之外的资源，此项目内的资源依赖于这些资源。
引用项
单击此字段中的数字 (如果大于 0) 可显示一组此项目之外的资源，这些资源依赖于此项目内的资源。
说明
单击编辑说明可添加或编辑该项目的说明，然后单击提交说明对其进行更新。
文件夹
输入新文件夹名称
要在此项目中新建文件夹，请在文本字段中输入一个名称，然后单击添加文件夹。请参阅添加文件夹。
另请参阅命名项目和文件夹。
名称
单击文件夹名称，以便为该文件夹显示“项目/文件夹视图”页。
选项
对于列出的任何文件夹，您都可以执行以下操作之一：
  • 单击重命名文件夹图标可重命名文件夹。请参阅重命名文件夹。
  • 单击升级到项目或移动文件夹图标可移动文件夹或将其升级到项目。请参阅移动或升级文件夹。
  • 单击克隆文件夹图标可克隆文件夹。请参阅克隆文件夹。
  • 单击删除文件夹图标可删除文件夹。如果有其他资源引用此资源，将会显示删除警告图标。确认警告后即可删除该资源。这可能会由于存在对已删除资源的未解析引用而导致冲突。请参阅删除文件夹。
执行上述任何操作之后，结果将保存在当前会话中。要结束该会话并将配置部署到运行时，请单击更改中心下的激活。
删除
要删除文件夹，请选中要删除的文件夹名称旁的复选框，然后单击删除。
资源
创建资源
要为此项目新建资源，请从创建资源列表中选择资源类型。请参阅创建资源。
名称
单击资源名，可显示与该资源有关的详细资料。
资源类型
此资源的类型。
操作
可对资源执行的操作取决于资源的类型：
  • 对于代理服务，将显示编辑消息流图标。单击此图标可编辑代理服务的消息。请参阅创建、编辑和查看消息流。
  • 对于业务服务、代理服务、XQuery 转换、XSL 转换和 MFL 文件，将显示启动测试控制台图标。单击此图标可调用测试控制台，该控制台用来验证和测试服务和转换的设计。对于服务，只能在运行期间 (即在激活会话后) 使用测试控制台。对于转换，无论是在会话内还是会话外，均可使用测试控制台。请参阅测试服务和测试转换。
  • 对于基于 WSDL 的代理服务，将显示导出 WSDL 图标。单击此图标可导出 WSDL，随后可以使用外部工具 (如 IDE) 对其进行查看或修改。

    • 请注意，此功能与“系统管理”模块中的“导出资源”功能不同，后者用于在两个域之间移动和临时保存资源。请参阅导出 WSDL。

  • 对于传输类型业务服务 (例如 EJB 和流)，将显示生成 WSDL 图标。单击此图标可生成 WSDL，您随后可以对其进行查看或修改。请参阅生成 WSDL。
选项
选项列显示了以下图标：
  • 单击重命名资源图标可重命名资源。请参阅重命名资源。
  • 单击移动资源图标可移动资源。请参阅移动资源。
  • 单击克隆资源图标可克隆资源。请参阅克隆资源。
  • 单击删除图标可删除资源。如果有其他资源引用此资源，将会显示删除警告图标。确认警告后即可删除该资源。这可能会由于存在对已删除资源的未解析引用而导致冲突。
执行上述任何操作之后，结果将保存在当前会话中。要结束该会话并将配置部署到运行时，请单击更改中心下的激活。
