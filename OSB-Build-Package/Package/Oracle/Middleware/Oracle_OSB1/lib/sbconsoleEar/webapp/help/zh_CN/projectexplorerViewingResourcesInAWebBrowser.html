在 Web 浏览器中查看资源

使用以下 URL 在标准 Web 浏览器中查看 Oracle Service Bus 资源

    • WSDL 资源的路径：
      http://host:port/sbresource?WSDL/project_path/wsdlname
      或
      http://host:port/proxy_service_endpoint_URI?WSDL (适用于基于 WSDL 的 HTTP 代理服务)
    • 代理服务资源的路径：
      http://host:port/sbresource?PROXY/project_path/proxyname
      http://host:port/sbresource?ORAPROXY/project_path/proxyname (适用于附加了 Oracle Web Services Manager 策略的代理服务)
    • 业务服务资源的路径：
      http://host:port/sbresource?BIZ/project_path/bizname
      http://host:port/sbresource?ORABIZ/project_path/bizname (适用于附加了 Oracle Web Services Manager 策略的业务服务)
    • WSS 资源的路径：
      http://host:port/sbresource?POLICY/project_path/policyname
    • MFL 资源的路径：
      http://host:port/sbresource?MFL/project_path/mflname
    • SCHEMA 资源的路径：
      http://host:port/sbresource?SCHEMA/project_path/schemaname

    注：如果您在资源名中使用特殊字符，则用于公开 Oracle Service Bus 中资源的 URL 必须以 UTF-8 进行编码以对特殊字符进行转义。

    您也可以检索包含 Oracle Web Services Manager 策略的 WSDL，以使这些策略符合支持的 WS-Policy 和 WS-Security 策略标准。有关详细信息，请参阅 Oracle Fusion Middleware Developer's Guide for Oracle Service Bus 中的“Advertising WSDLs to Support WS Standards”，网址为：http://www.oracle.com/pls/as111170/lookup?id=OSBDV1877。
