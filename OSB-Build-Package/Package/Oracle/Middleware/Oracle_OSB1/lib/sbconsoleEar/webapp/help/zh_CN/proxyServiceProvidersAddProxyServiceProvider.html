添加服务密钥提供程序

  1. 如果尚未执行此操作，请单击创建新建一个会话，或单击编辑进入现有会话。请参阅使用更改中心。
  2. 选择项目浏览器，然后选择要在其中添加服务密钥提供程序的项目或文件夹。此时会显示项目/文件夹视图页。
  3. 在创建资源下拉列表中，选择服务密钥提供程序，以显示创建新服务密钥提供程序页。
  4. 在服务密钥提供程序名称字段中，输入此服务密钥提供程序的唯一名称。

    如需命名指导，请参阅资源命名限制。
  5. 在说明字段中，输入此服务密钥提供程序的说明。
  6. 执行下面的任一步骤，如表 17-2 中所示。

    7. 表 17-2 验证选项
    添加密钥对...
    步骤
    数字加密
    1. 单击加密密钥旁边的浏览。

      2. 为加密密钥选择别名窗口显示了您的领域中 PKI 身份证明映射器所使用的密钥库的密钥别名。

    2. 在为加密密钥选择别名窗口中，输入用于确保密钥库安全访问的口令。(创建密钥库时设置的口令。)
    3. 选择映射到 X.509 证书并支持加密的密钥别名。
    4. 单击提交。
    当将此服务密钥提供程序与某个代理服务相关联时，Oracle Service Bus 会将 X.509 证书嵌入到该代理服务的 WSDL 中。然后，此代理服务就可以使用该证书来加密它发送到其端点的消息。该代理服务使用 PKI 身份证明中的私有密钥加密端点返回的消息。
    数字签名
    1. 单击数字签名密钥旁边的浏览。

      2. 为数字签名密钥选择别名窗口显示了您的领域中 PKI 身份证明映射器所使用的密钥库的密钥别名。

    2. 在为数字签名密钥选择别名窗口中，输入用于确保密钥库安全访问的口令。(创建密钥库时设置的口令。)
    3. 选择密钥别名。
    4. 单击提交。
    SSL 客户机验证 (双向 SSL)
    1. 单击 SSL 客户机验证密钥旁边的浏览。

      2. 为 SSL 客户机验证密钥选择别名窗口显示了您的领域中 PKI 身份证明映射器所使用的密钥库的密钥别名。

    2. 在为 SSL 客户机验证密钥选择别名窗口中，输入用于确保密钥库安全访问的口令。(创建密钥库时设置的口令。)
    3. 选择密钥别名。
    4. 单击提交。

  7. 单击保存。该服务密钥提供程序会保存在当前会话中。
  8. 要结束该会话并将配置部署到运行时，请单击更改中心下的激活。
