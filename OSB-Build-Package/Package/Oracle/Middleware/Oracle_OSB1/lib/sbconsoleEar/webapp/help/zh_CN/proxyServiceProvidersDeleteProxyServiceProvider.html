删除服务密钥提供程序

当删除某个服务密钥提供程序时，Oracle Service Bus 也会从 PKI 身份证明映射提供程序中删除与密钥对绑定相关联的别名。Oracle Service Bus 未从密钥库中删除相关联的密钥证书对。

  1. 如果尚未执行此操作，请单击创建新建一个会话，或单击编辑进入现有会话。请参阅使用更改中心。
  2. 如果将任何代理服务配置为使用服务密钥提供程序，请从该代理服务中删除此服务密钥提供程序。请参阅编辑代理服务配置。
  3. 选择资源浏览器 > 服务密钥提供程序，以便显示服务密钥提供程序概要页。
  4. 在要删除的服务密钥提供程序的选项字段中，单击删除图标。此服务密钥提供程序将从当前会话中删除。如果已将业务服务或代理服务配置为使用服务帐户，则删除警告图标将表明您可在确认警告后删除服务密钥提供程序。这可能会由于存在对已删除服务密钥提供程序的未解析引用而导致冲突。
  5. 要结束该会话并将配置部署到运行时，请单击更改中心下的激活。
