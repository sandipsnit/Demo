查找服务密钥提供程序

  1. 请执行以下两个操作之一：
    • 选择项目浏览器以便显示“项目视图”页或“项目/文件夹视图”页。然后，在项目和文件夹中导航以查找服务密钥提供程序。
    • 选择资源浏览器 > 服务密钥提供程序。服务密钥提供程序概要页显示表 17-1 中所示的信息。
  2. 要搜索服务密钥提供程序，请在名称字段中输入部分或完整的提供程序名称。还可以在路径字段中输入部分或完整的提供程序项目名和文件夹。单击搜索。

    3. 单击查看全部可删除搜索筛选器，并显示所有服务密钥提供程序。

    表 17-1 服务密钥提供程序信息 
    属性
    说明
    名称
    服务密钥提供程序的唯一名称。单击此名称可参阅查看服务密钥提供程序详细资料页。请参阅编辑服务密钥提供程序。
    路径
    服务密钥提供程序所在的项目名和文件夹名称。单击该名称可查看包含此资源的项目或文件夹。请参阅使用项目和文件夹限定资源名。
