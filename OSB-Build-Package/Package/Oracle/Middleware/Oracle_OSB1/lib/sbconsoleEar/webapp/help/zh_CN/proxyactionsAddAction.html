向消息流添加操作和编辑消息流中的操作

操作是管道阶段、错误处理程序阶段、路由节点和分支节点的元素，它们定义了如何对流过代理服务的消息进行定义。

开始之前

这些说明假定您已在编辑消息流页中编辑消息流，如创建、编辑和查看消息流中所述。

它们还假定，您已添加管道阶段、路由节点和/或错误处理程序阶段。请参阅以下页：

  • 向消息流添加管道对节点
  • 向管道添加阶段
  • 添加代理服务错误处理程序

向消息流添加操作

  1. 选择要将操作添加到的组件。例如，依次单击阶段图标、编辑阶段、路由节点图标、编辑路由。
  2. 根据操作是已添加至阶段还是已添加至路由节点，执行以下操作之一：
    • 如果尚未添加任何操作，则编辑阶段配置页只显示添加操作图标。单击该图标，然后选择操作类型。
    • 如果已添加一个或多个操作，则编辑阶段配置页将显示一个或多个代表这些操作的图标，例如发布图标和或路由图标等。单击相应的图标，再单击添加操作，然后选择操作类型。
    • 某些操作 (例如发布操作中的请求和响应操作) 包括适用于操作的添加操作链接。单击该图标，然后选择操作类型。

      • 对于可在消息流中链接在一起的操作，没有任何限制。

  3. 完成添加操作后，可以进一步配置阶段或路由节点中的操作，如表 21-2 中所述。

    4. 表 21-2“编辑阶段配置”任务 
    操作
    步骤
    删除操作
    单击相应的图标，然后单击删除此操作。
    下移操作 (降级)
    单击相应的图标，然后单击下移操作。操作会移至此阶段中包含的下一个操作之下。
    只有一个阶段包含两个或更多操作时才显示此选项。
    上移操作 (升级)
    单击相应的图标，然后单击上移操作。操作会移至此阶段中包含的上一个操作之上。

    只有该阶段包含两个或更多操作时才显示此选项。

    剪切操作
    单击相应的图标，然后单击剪切。
    复制操作
    单击相应的图标，然后单击复制。
    粘贴已剪切或复制的操作
    单击相应的图标，然后单击粘贴操作。
    可以跨阶段复制和粘贴操作。但是，对于分配、替换或插入操作，请注意以下事项：
    • 对于所有来自源 (复制) 阶段的变量相关的名称空间和用户定义的名称空间，都作为目标 (粘贴) 阶段中用户定义的名称空间进行添加。
    • 不会复制重复的名称空间 (源阶段和目标阶段中的相同名称空间)。
    • 会复制冲突的名称空间 (使用的前缀相同但使用的 URI 不同的名称空间声明)。用户将可保存配置，但仅在删除阶段 B 中冲突的名称空间声明后，才能激活该配置。
    验证阶段
    在编辑阶段配置页中，单击验证，以验证该阶段中配置的全部操作。

  4. 单击保存可提交当前会话中的更新。
  5. 在编辑消息流页上，继续构造消息流，如创建、编辑和查看消息流中所述。
  6. 单击保存可提交当前会话中的更新。
  7. 要结束该会话并将配置部署到运行时，请单击更改中心下的激活。
