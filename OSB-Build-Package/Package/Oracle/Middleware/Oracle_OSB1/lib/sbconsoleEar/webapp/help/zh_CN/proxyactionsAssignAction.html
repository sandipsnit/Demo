添加分配操作

使用分配操作可将 XQuery 表达式的结果分配给上下文变量。

开始之前

这些说明假定您已在编辑消息流页中编辑消息流，如创建、编辑和查看消息流中所述。

添加分配操作

  1. 导航至要添加操作的位置，如向消息流添加操作和编辑消息流中的操作中所述。
  2. 单击相应的图标，然后选择添加操作 > 消息处理 > 分配。
  3. 单击 <表达式>。此时将显示 XQuery 表达式编辑器页。XQuery 表达式用来创建将分配给命名变量的数据。请参阅创建和编辑内嵌 XQuery 和 XPath 表达式。
  4. 编辑完表达式后，在变量字段中输入一个上下文变量。要了解有关上下文变量的详细信息，请参阅 Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus 中的“Inbound and Outbound Variables”(网址为：http://www.oracle.com/pls/as111170/lookup?id=OSBAG325) 和“Constructing Messages to Dispatch”(网址为：http://www.oracle.com/pls/as111170/lookup?id=OSBAG348)。

完成之后

配置完此操作后，继续配置其他操作或保存您的配置，如向消息流添加操作和编辑消息流中的操作中所述。
