添加删除操作

使用删除操作可删除上下文变量或 XPath 表达式指定的一组节点。删除操作是一组更新操作中的一个操作。

开始之前

这些说明假定您已在编辑消息流页中编辑消息流，如创建、编辑和查看消息流中所述。

添加删除操作

  1. 导航至要添加操作的位置，如向消息流添加操作和编辑消息流中的操作中所述。
  2. 要删除上下文变量，请选择变量单选按钮，然后在变量字段中输入上下文变量的名称。要了解上下文变量的详细信息，请参阅附录 A“消息上下文”。

    3. 或者，要删除 XPath 表达式所选的全部节点，请选择 XPath 单选按钮，然后单击 <XPath>。此时将显示 XPath 表达式编辑器页。请参阅创建和编辑内嵌 XQuery 和 XPath 表达式。保存表达式后，在变量字段中输入一个上下文变量。要了解有关上下文变量的详细信息，请参阅附录 A“消息上下文”。

完成之后

配置完此操作后，继续配置其他操作或保存您的配置，如向消息流添加操作和编辑消息流中的操作中所述。
