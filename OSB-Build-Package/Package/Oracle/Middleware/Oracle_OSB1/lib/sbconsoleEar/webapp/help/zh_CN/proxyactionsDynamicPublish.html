添加动态发布操作

使用动态发布操作可将消息发布到由 XQuery 表达式指定的服务。

有关发布行为的详细信息，请参阅 Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus 中的“Performing Transformations in Message Flows”，网址为：http://www.oracle.com/pls/as111170/lookup?id=OSBAG205。

  1. 导航至要添加操作的位置，如向消息流添加操作和编辑消息流中的操作中所述。
  2. 单击相应的图标，然后选择添加操作 > 通信 > 动态发布。
  3. 单击 <表达式>。
  4. 在 XQuery 表达式编辑器中，输入 XQuery 表达式或选择提供类似以下结果的 XQuery 资源： 
    5. <ctx:route>

    <ctx:service 
isProxy="false">project/folder/businessservicename</ctx:service>

    <ctx:operation>foo</ctx:operation>

    </ctx:route>
    注： 调用代理服务时，应将 isProxy 属性设置为“真”。调用业务服务时，请将 isProxy 设置为“假”。
    元素 operation 是可选的。
  5. 单击保存。
  6. 在请求操作字段中，单击添加操作以添加操作，然后选择要与此服务相关联的操作。可以添加多个操作。要了解有关可以添加的操作类型的详细信息，请参阅向消息流添加操作和编辑消息流中的操作中的操作表。

完成之后

配置完此操作后，继续配置其他操作或保存您的配置，如向消息流添加操作和编辑消息流中的操作中所述。
