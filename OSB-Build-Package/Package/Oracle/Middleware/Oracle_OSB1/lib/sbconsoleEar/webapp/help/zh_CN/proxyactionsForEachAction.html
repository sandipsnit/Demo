添加 For Each 操作

使用 For Each 操作可对一系列值进行迭代，并执行一组操作。

开始之前

这些说明假定您已在编辑消息流页中编辑消息流，如创建、编辑和查看消息流中所述。

添加 For Each 操作

  1. 导航至要添加操作的位置，如向消息流添加操作和编辑消息流中的操作中所述。
  2. 单击相应的图标，然后选择添加操作 > 流控制 > For Each。
  3. 在变量字段中输入变量名，单击 <XPath> 打开 XPath 编辑器以创建 XPath 表达式，并在 Do () 循环中配置操作。

完成之后

配置完此操作后，继续配置其他操作或保存您的配置，如向消息流添加操作和编辑消息流中的操作中所述。
