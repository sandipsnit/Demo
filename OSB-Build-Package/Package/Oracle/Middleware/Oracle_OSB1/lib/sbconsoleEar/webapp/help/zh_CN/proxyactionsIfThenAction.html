添加 If...Then... 操作

使用 if...then... 操作，可根据 XQuery 表达式的布尔结果，有条件地执行某一个操作或一组操作。

开始之前

这些说明假定您已在编辑消息流页中编辑消息流，如创建、编辑和查看消息流中所述。

添加 If...Then 操作

  1. 导航至要添加操作的位置，如向消息流添加操作和编辑消息流中的操作中所述。
  2. 单击相应的图标，然后选择添加操作 > 流控制 > If...Then。
  3. 单击 <条件>，以便显示“XQuery 条件编辑器”页。

    4. 创建的条件可用作在进入 then() 子句前执行的测试，适用于每个标准 if...then 逻辑。请参阅创建和编辑内嵌 XQuery 和 XPath 表达式。

  4. 完成 XQuery 条件的编辑后，单击添加操作，然后选择要与此条件相关联的操作。要了解有关希望添加的操作类型的详细信息，请参阅向消息流添加操作和编辑消息流中的操作。

    5. 在路由节点中，只能选择路由、动态路由或路由表操作。但是，这些操作内可以包含请求和响应操作。

  5. 根据逻辑需要，单击 If...Then... 图标，然后单击添加 else-if 条件或添加 else 条件，以添加 else-if 条件或 else 条件。单击添加操作，使操作与这些条件关联。

    6. 条件操作可进行嵌套。但是，在阶段编辑器中，存在累积级别最多为 4 的嵌套限制。如果试图添加第 5 层，将不显示此嵌套操作。累积级别包括所有分支操作：if...then... 条件、发布表和路由表。例如，如果您有 2 层的条件操作，还有一个其中包含路由表的发布表，则累积级别总计为 4 层。如果尝试添加其他条件操作 (到最后一个发布表)，将不显示该条件。

完成之后

配置完此操作后，继续配置其他操作或保存您的配置，如向消息流添加操作和编辑消息流中的操作中所述。
