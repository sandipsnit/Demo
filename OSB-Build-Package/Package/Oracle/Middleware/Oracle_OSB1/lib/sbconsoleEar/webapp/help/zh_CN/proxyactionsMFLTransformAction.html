添加 MFL 转换操作

使用 MFL (消息格式语言) 转换操作可在消息管道中将消息内容从 XML 格式转换为非 XML 格式，反之亦然。MFL 是用于描述二进制数据布局的专用 XML 文档。它是一种 Oracle 专用语言，用于定义规则，以便将格式化的二进制数据转换为 XML 数据，或者将 XML 数据转换为格式化的二进制数据。请参阅 MFL。

开始之前

这些说明假定您已在编辑消息流页中编辑消息流，如创建、编辑和查看消息流中所述。

添加 MFL 转换操作

  1. 导航至要添加操作的位置，如向消息流添加操作和编辑消息流中的操作中所述。
  2. 单击相应的图标，然后选择添加操作 > 消息处理 > MFL 转换。
  3. 在应用 MFL 转换下拉列表中，根据需要选择 XML 到非 XML 或非 XML 到 XML。
  4. 单击 <表达式>。使用 XQuery 表达式编辑器指定将要对其执行 MFL 转换操作的变量。转换为 XML 格式时，此输入必须为文本或二进制数据；转换为非 XML 格式时，此输入必须为 XML。消息上下文中的二进制内容由 binary-content XML 元素表示。输入内容需要为二进制数据时，此 XML 应该是 XQuery 表达式的结果。请参阅创建和编辑内嵌 XQuery 和 XPath 表达式。
  5. 选择以下选项之一：
    • MFL 资源：单击 <资源> 链接。此时将显示选择 MFL 页。选择将执行 MFL 转换操作的静态 MFL 资源。
    • MFL 资源来自：单击表达式链接。此时将显示 XQuery 表达式编辑器页。使用 XQuery 表达式编辑器创建或编辑 XQuery 表达式，以采用 project/folder/MFLresourcename 格式动态指定将执行该转换操作的 MFL 资源。请参阅创建和编辑内嵌 XQuery 和 XPath 表达式。
  6. 在分配给变量字段中，输入要向其分配此转换操作结果的变量的名称。该结果将是 binary-content XML 元素。

完成之后

配置完此操作后，继续配置其他操作或保存您的配置，如向消息流添加操作和编辑消息流中的操作中所述。
