添加发布操作

使用发布操作可为消息识别静态指定的目标服务，并配置如何包装消息并将其发送到该服务。

有关发布行为的详细信息，请参阅 Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus 中的“Performing Transformations in Message Flows”，网址为：http://www.oracle.com/pls/as111170/lookup?id=OSBAG205。

  1. 导航至要添加操作的位置，如向消息流添加操作和编辑消息流中的操作中所述。
  2. 选择添加操作 > 通信 > 发布。
  3. 单击 <服务>。此时将显示选择服务页。
  4. 从列表中选择一个服务，然后单击提交。这是此消息的目标服务。
  5. 如果该服务已定义操作，可以通过从操作列表中选择操作来指定要调用的操作。
  6. 要让出站操作与入站操作相同，请选中为出站使用入站操作复选框。
  7. 要配置如何包装消息并将其发送到服务，请在请求操作字段中，单击添加操作。然后，选择要与该服务关联的操作。可以添加多个操作。请参阅向消息流添加操作和编辑消息流中的操作。

完成之后

配置完此操作后，继续配置其他操作或保存您的配置，如向消息流添加操作和编辑消息流中的操作中所述。
