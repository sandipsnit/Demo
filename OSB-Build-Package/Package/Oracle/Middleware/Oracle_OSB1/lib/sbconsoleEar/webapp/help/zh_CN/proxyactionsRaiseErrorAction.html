添加引发错误操作

使用引发错误操作可引发具有指定错误代码 (字符串) 和说明的异常错误。

开始之前

这些说明假定您已在编辑消息流页中编辑消息流，如创建、编辑和查看消息流中所述。

添加引发错误操作

  1. 导航至要添加操作的位置，如向消息流添加操作和编辑消息流中的操作中所述。
  2. 单击相应的图标，然后选择添加操作 > 流控制 > 引发错误。
  3. 在错误代码字段中，输入要引发的错误代码。
  4. 在错误消息字段中，输入错误代码的说明。

完成之后

配置完此操作后，继续配置其他操作或保存您的配置，如向消息流添加操作和编辑消息流中的操作中所述。

事务处理

如果服务为事务处理，则触发的“引发错误”操作会中止请求 (异步) 或者请求或响应 (同步) 中的事务处理。例如，可以自检消息和确定应发生“引发错误”操作的条件 (即使未发生 SOAP 故障)，“引发错误”将导致事务处理中止。
