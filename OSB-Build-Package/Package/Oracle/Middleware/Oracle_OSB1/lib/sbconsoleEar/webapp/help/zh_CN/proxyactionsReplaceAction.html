添加替换操作

使用替换操作可替换由 XPath 表达式指定的节点或节点内容。节点或节点内容由 XQuery 表达式返回的值替换。

替换操作可用于替换简单值、元素甚至属性。如果 XQuery 表达式不返回任何结果，就相当于删除已识别的节点或使其为空，具体取决于该操作替换的是整个节点还只是节点内容。

替换操作是一组更新操作中的一个操作。

开始之前

这些说明假定您已在编辑消息流页中编辑消息流，如创建、编辑和查看消息流中所述。

添加替换操作

  1. 导航至要添加操作的位置，如向消息流添加操作和编辑消息流中的操作中所述。
  2. 单击相应的图标，然后选择添加操作 > 消息处理 > 替换。
  3. 单击 <XPath>。此时将显示 XPath 表达式编辑器页。XPath 表达式用于指定命名变量中要被替换的数据。请参阅创建和编辑内嵌 XQuery 和 XPath 表达式。
  4. 完成 XPath 表达式的编辑后，在变量字段中输入一个上下文变量。请参阅附录 A“消息上下文”。
  5. 单击 <表达式>。此时将显示 XQuery 表达式编辑器页。XQuery 表达式用于创建命名变量中的数据 (用来替换 XPath 指定的数据)。请参阅创建和编辑内嵌 XQuery 和 XPath 表达式。
  6. 编辑完 XQuery 表达式之后，请选择以下选项之一：
    • 替换整个节点 - 指定替换由所定义的 XPath 表达式选择的节点及其全部内容。
    • 替换节点内容 - 指定只替换节点内容而不替换节点。
      • 注： 选择替换节点内容选项并将 XPath 字段留空，比选择替换整个节点选项并将 XPath 设置为 ./* 更为高效。

完成之后

配置完此操作后，继续配置其他操作或保存您的配置，如向消息流添加操作和编辑消息流中的操作中所述。
