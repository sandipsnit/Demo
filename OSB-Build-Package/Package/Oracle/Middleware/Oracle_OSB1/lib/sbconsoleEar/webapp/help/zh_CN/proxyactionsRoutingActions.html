向路由节点添加路由操作

识别消息的目标服务并配置如何将消息路由到此服务。

这是一项最终操作，表示在此操作后不能添加其他操作。但是，此操作可以包含请求和响应操作。

有关路由的详细信息，请参阅 Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus 中的“Modeling Message Flow in Oracle Service Bus”，网址为：http://www.oracle.com/pls/as111170/lookup?id=OSBAG181。

开始之前

这些说明假定您已在编辑消息流页中编辑消息流，如创建、编辑和查看消息流中所述。

向路由节点添加路由操作

  1. 导航至要添加操作的位置，如向消息流添加操作和编辑消息流中的操作中所述。
  2. 单击路由节点图标，然后单击编辑路由。此时将显示编辑阶段配置页。
  3. 单击添加操作图标，然后选择通信 > 路由。
  4. 单击服务。此时将显示服务浏览器。
  5. 从列表中选择一个服务，然后单击提交。此时将显示该服务，而不是默认链接。
  6. 如果希望使出站操作与入站操作相同，请选中为出站使用入站操作复选框。
  7. 在请求操作字段中，单击添加操作以添加操作，然后选择要与此服务相关联的操作。可以添加多个操作。要了解有关可以添加的操作类型的详细信息，请参阅向消息流添加操作和编辑消息流中的操作中的操作表。
  8. 在响应操作字段中，单击添加操作以添加操作，然后选择要与此服务相关联的操作。可以添加多个操作。要了解有关可以添加的操作类型的详细信息，请参阅向消息流添加操作和编辑消息流中的操作中的操作表。
  9. 单击保存。
  10. 在编辑消息流页上，继续构造消息流，如创建、编辑和查看消息流中所述。
  11. 单击保存可提交当前会话中的更新。
  12. 要结束该会话并将配置部署到运行时，请单击更改中心下的激活。
