添加验证操作

使用验证操作可根据 XML 方案元素或 WSDL 资源验证 XPath 表达式所选的元素。只能验证全局元素；Oracle Service Bus 不支持对本地元素进行验证。

开始之前

这些说明假定您已在编辑消息流页中编辑消息流，如创建、编辑和查看消息流中所述。

添加验证操作

  1. 导航至要添加操作的位置，如向消息流添加操作和编辑消息流中的操作中所述。
  2. 单击相应的图标，然后选择添加操作 > 消息处理 > 验证。
  3. 单击 XPath，以构造指定要验证的元素的 XPath 表达式。请参阅创建和编辑内嵌 XQuery 和 XPath 表达式。在 XPath 表达式编辑器中构造完表达式后，单击保存，将该表达式插入编辑阶段配置页中。
  4. 在变量字段中，输入变量的名称，以存放要验证的元素。
  5. 单击资源，然后选择 WSDL 或方案。
  6. 在 WSDL 浏览器或 XML 方案浏览器中，执行以下操作：
    1. 选择 WSDL 或 XML 方案
    2. 选择 WSDL 或 XML 方案类型或元素
    3. 单击提交。
  7. 要保存该验证的结果 (布尔结果)，请选择在变量中保存验证结果，并输入要在其中保存结果的变量的名称。

    8. 另外，要在根据 WSDL 或 XML 方案元素验证元素失败时引发错误，请在验证失败时选择引发错误。

完成之后

配置完此操作后，继续配置其他操作或保存您的配置，如向消息流添加操作和编辑消息流中的操作中所述。
