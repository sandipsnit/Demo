添加路由节点错误处理程序

开始之前

这些说明假定您已在“编辑消息流”页中编辑了消息流，如创建、编辑和查看消息流中所述。

它们还假定，您已创建路由节点，如向消息流添加路由节点中所述。

要添加路由节点处理程序，请执行以下操作：

  1. 单击路由节点图标，然后单击添加错误处理程序。此时将显示编辑错误处理程序页。
  2. 单击错误处理程序图标，然后单击添加阶段。
  3. 单击阶段图标，然后单击编辑阶段。此时将显示编辑阶段配置页。
  4. 单击添加操作，然后选择要添加的操作。

    5. 因为错误处理程序也是管道，所以其配置方式类似于其他任何管道。例如，可以使用发布操作向其他服务发送错误通知，还可以使用分配操作修改上下文变量，等等。要了解有关想添加的操作类型的详细信息，请参阅向消息流添加操作和编辑消息流中的操作中的相应过程。对于可在消息流中链接在一起的操作，没有任何限制。

    三个常用的错误操作是引发错误、回复和恢复。

  5. 根据需要，添加其他操作，并在编辑阶段配置页上进行其他编辑。
  6. 在编辑阶段配置页上，单击保存可提交当前会话中的更新。
  7. 在编辑错误处理程序页上，单击保存可提交当前会话中的更新。

完成之后

在“编辑消息流”页上，继续构造消息流，如创建、编辑和查看消息流中所述。
