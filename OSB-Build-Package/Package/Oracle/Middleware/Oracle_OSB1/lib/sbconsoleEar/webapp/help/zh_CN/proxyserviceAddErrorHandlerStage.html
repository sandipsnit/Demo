添加阶段错误处理程序

开始之前

这些说明假定您已在“编辑消息流”页中编辑了消息流，如创建、编辑和查看消息流中所述。

它们还假定，您已将阶段添加至管道，如向管道添加阶段中所述。

要添加阶段错误处理程序，请执行以下操作：

  1. 导航至要将错误处理添加到的阶段。
  2. 单击阶段图标，然后单击添加阶段错误处理程序。此时将显示编辑错误处理程序页。
  3. 单击错误处理程序图标，然后单击添加阶段。
  4. 单击阶段图标，然后单击编辑阶段。此时将显示编辑阶段配置页。
  5. 单击添加操作，然后选择要添加的操作。

    6. 错误处理程序是管道，因此，其配置方式与其他任何管道类似。例如，可以使用发布操作向其他服务发送错误通知，还可以使用分配操作修改上下文变量，等等。要了解有关希望添加的操作类型的详细信息，请参阅向消息流添加操作和编辑消息流中的操作中的相应过程。对于可在消息流中链接在一起的操作，没有任何限制。

    三个常用的错误操作是引发错误、回复和恢复。

  6. 根据需要，添加其他操作，并在编辑阶段配置页上进行其他编辑。
  7. 单击保存可提交当前会话中的更新。

完成之后

在“编辑消息流”页上，继续构造消息流，如创建、编辑和查看消息流中所述。
