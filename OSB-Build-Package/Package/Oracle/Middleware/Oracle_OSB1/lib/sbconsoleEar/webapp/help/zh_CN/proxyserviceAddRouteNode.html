向消息流添加路由节点

开始之前

这些说明假定您已在编辑消息流页中编辑了消息流，如创建、编辑和查看消息流中所述。

要向消息流添加路由节点，请执行以下操作：

  1. 请单击管道对的管道对节点图标，然后单击添加路由。
  2. 要更改路由节点的默认名称并为路由节点添加说明，请单击路由节点图标，然后单击编辑名称和批注。根据需要更改名称和说明，然后单击保存。
    3. 注： 重命名管道或路由节点时，由于管道计数器重置为零，监视模块中面板页上显示的消息数可能与其他组件的消息数不关联。这是因为 Oracle Service Bus 将重命名视为删除和重新创建操作。经过与服务的监视间隔一样长的时间后，这些数目会再次关联。
  3. 要向路由节点添加操作，请单击路由节点图标，然后单击编辑路由。此时将显示编辑消息流页。有关可添加到路由节点的操作的详细信息，请参阅以下部分：
    • 添加 If...Then... 操作
    • 向路由节点添加动态路由
    • 向路由节点添加路由操作
    • 向路由节点添加路由表
    • 添加路由节点错误处理程序
  4. 在编辑消息流页上，继续构造消息流，如创建、编辑和查看消息流中所述。
  5. 单击保存可提交当前会话中的更新。
  6. 要结束该会话并将配置部署到运行时，请单击更改中心下的激活。
