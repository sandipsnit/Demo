向管道添加阶段

开始之前

这些说明假定您已在编辑消息流页中编辑了消息流，如创建、编辑和查看消息流中所述。

要向管道添加阶段，请执行以下操作：

  1. 如有必要，请单击管道对节点图标左侧的加号将其展开。一个管道对包含一个请求管道和一个响应管道。
  2. 单击要向其添加阶段的管道，然后单击添加阶段。
  3. 要更改阶段的默认名称并为阶段添加说明，请执行以下操作：
    1. 单击阶段图标，然后单击编辑名称和批注。
    2. 根据需要更改名称和说明。
    3. 单击保存。
  4. 要向阶段添加操作，请单击阶段图标，然后单击编辑阶段。请参阅向消息流添加操作和编辑消息流中的操作。
  5. 要向阶段添加错误处理程序，请单击阶段图标，然后单击添加阶段错误处理程序。请参阅添加阶段错误处理程序。此时将显示编辑消息流页。
  6. 继续构造消息流，如创建、编辑和查看消息流中所述。
  7. 单击保存可提交当前会话中的更新。
  8. 要结束该会话并将配置部署到运行时，请单击更改中心下的激活。
