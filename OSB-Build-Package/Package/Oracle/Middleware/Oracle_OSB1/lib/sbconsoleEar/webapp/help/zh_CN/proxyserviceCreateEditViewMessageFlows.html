创建、编辑和查看消息流

  1. 如果尚未执行此操作，请单击创建新建一个会话，或单击编辑进入现有会话。请参阅使用更改中心。
  2. 查找要查看或编辑的消息流的代理服务。请执行以下两个操作之一：
    • 请选择项目浏览器以便显示“项目视图”页或“项目/文件夹视图”页。然后，在项目和文件夹中导航以查找代理服务。
    • 选择资源浏览器 > 代理服务以便显示代理服务概要页。
  3. 在要查看或编辑其消息流的代理服务所在的行中，单击编辑消息流图标。
    • 如果尚未为选定的代理服务创建消息流，则将显示编辑消息流页，该页上带有一个代理服务图标。这是代理服务的启动节点。单击此图标可开始构造消息流。
    • 如果代理服务已经有一个消息流，则该页将包含该消息流的图形表示形式。单击这些图标可查看或编辑消息流的各个部分。

      • 有关在该页上构造消息流所需执行的操作的信息，请参阅“编辑消息流”页。

  4. 单击保存可提交当前会话中的更新。
  5. 要结束该会话并将配置部署到运行时，请单击更改中心下的激活。
