“电子邮件传输配置”页

创建/编辑代理服务 - 电子邮件传输配置页是一系列用于创建和编辑代理服务的页面中的一个页面，如创建和配置代理服务和编辑代理服务配置中所述。

使用此页可为使用电子邮件传输协议的代理服务配置传输设置。表 19-4 介绍了如何使用此页。

表 19-4“创建/编辑代理服务 - 电子邮件传输配置”页
选项
创建或编辑...
需要使用 SSL
选择此选项可使用安全套接字层进行通信。默认 SSL 端口号为：
  • POP3(S)：995
  • IMAP：993
注：要使 SSL 工作，Oracle Service Bus truststore 中必须要有电子邮件服务器证书。
服务帐户
输入服务帐户名称，或者单击浏览从浏览器中选择服务帐户。
托管服务器
此字段仅在集群域中可用。
选择托管服务器作为轮询服务器。所有托管服务器均可处理消息，但是只有一个托管服务器可以轮询消息。
轮询间隔
输入轮询间隔 (秒)。
电子邮件协议
选择 POP3 或 IMAP 作为电子邮件帐户的服务器类型。
读取限制
指定每次轮询扫描读取的最大消息数。输入 0 可指定没有限制。
引用传递
选中此复选框可将文件临时保存在归档目录中，并将其作为标头中的引用进行传递。
默认情况下，将在新建服务时选中引用传递选项，并且必须指定归档目录位置。
通过引用传递附件
选中此复选框可将附件临时保存在归档目录中，并将其作为标头中的引用进行传递。
默认情况下，选中引用传递选项时，便会隐式选中通过引用传递附件选项，并且必须指定归档目录位置。
读后操作
选择如何处理阅读后的消息：
  • 归档 - 将消息归档。
  • 删除 - 将消息删除。
  • 移动 - 将消息移动。移动只适用于 IMAP 协议。
附件
选择附件的处理方式：
  • 归档 - 将附件保存到归档目录中。
  • 忽略 - 将附件忽略。
IMAP 移动文件夹
如果读后操作字段设置为移动，则输入要将消息移至的文件夹。
下载目录
输入可供下载电子邮件的临时位置。
归档目录
如果读后操作字段设置为归档，请指定归档位置的路径。
如果选中了引用传递或通过引用传递附件选项，则需要填写此字段。
错误目录
请输入在出现问题的情况下用于写入消息和任何附件的文件系统目录路径。
请求编码
接受默认值 ISO-8859-1 作为电子邮件传输中请求的字符集编码，或者输入其他字符集编码。

完成之后

单击下一步继续在下一页上配置此服务；或单击最后一步在“创建/编辑代理服务 - 概要”页上查看并保存此配置。
