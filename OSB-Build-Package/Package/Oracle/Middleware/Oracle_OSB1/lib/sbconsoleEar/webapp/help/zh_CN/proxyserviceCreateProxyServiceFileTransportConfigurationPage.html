“文件传输配置”页

创建/编辑代理服务 - 文件传输配置页是一系列用于创建和编辑业务服务的页中的一个页面，如创建和配置代理服务和编辑代理服务配置中所述。

使用此页可为使用文件传输协议的代理服务配置传输设置。表 19-5 介绍了如何使用此页。

表 19-5“创建/编辑代理服务 - 文件传输配置”页
选项
创建或编辑...
文件掩码
指定应由代理服务轮询的文件。如果 URI 是目录并且指定了 *.*，则该服务将轮询目录中的所有文件。文件掩码中只允许通配符 * 和 ?。不支持正则表达式。
托管服务器
此字段仅在集群域中可用。
选择托管服务器作为轮询服务器。所有托管服务器均可处理消息，但是只有一个托管服务器可以轮询消息。
轮询间隔
输入轮询间隔 (秒)。默认值为 60。
读取限制
指定每次轮询扫描读取的最大消息数。输入 0 可指定没有限制。默认值为 10。
按到达时间排序
选中此复选框可指定按照到达时间顺序传送事件。
请注意，如果为集群环境中执行的代理服务选择了该选项，则始终会将消息发送到同一服务器。换言之，选择该选项后会忽略服务器间的负载平衡。
扫描子目录
选中此复选框可递归扫描所有目录。
引用传递
选中此复选框可将文件临时保存在归档目录中，并将其作为标头中的引用进行传递。
读后操作
选择如何处理阅读后的消息：
  • 归档 - 将消息归档。
  • 删除 - 将消息删除。
临时目录
输入处理文件时临时保存文件的中间目录。
请勿将临时目录放在轮询目录 (在文件传输代理服务的 URL 中标识的目录，例如 file:///c:/dir1/dir2) 的内部。
归档目录
如果读后操作选项设置为归档，请指定归档位置的路径。如果选中了“引用传递”字段，则“归档目录”字段也是必填字段。
请勿将归档目录放在轮询目录的内部。
错误目录
输入在出现问题的情况下要将消息和附件发布到的位置。
请勿将错误目录放在轮询目录的内部。
请求编码
接受默认值 UTF-8 作为文件传输中请求的字符集编码，或者输入其他字符集编码。

完成之后

单击下一步继续在下一页上配置此服务；或单击最后一步在“创建/编辑代理服务 - 概要”页上查看并保存此配置。
