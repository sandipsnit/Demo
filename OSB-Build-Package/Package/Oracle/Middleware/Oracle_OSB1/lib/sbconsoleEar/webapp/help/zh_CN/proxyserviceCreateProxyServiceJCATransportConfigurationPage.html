“JCA 传输配置”页

使用此页可使用 JCA 传输协议配置传输设置。有关使用 JCA 传输的详细信息，请参阅 Oracle Fusion Middleware Developer's Guide for Oracle Service Bus 中的“JCA Transport”(网址为： http://www.oracle.com/pls/as111170/lookup?id=OSBDV910) 和从 JCA 绑定资源生成代理服务。

表 19-8 JCA 传输配置
选项
说明
JCA 文件
单击浏览选择 JCA 绑定。JCA 绑定定义服务的不同方面，例如，有关使用的适配器的详细资料、到 WSDL 和 TopLink 或 EclipseLink 映射文件的绑定，以及服务所需的激活/交互规范属性。
选择有效的 JCA 绑定后，其余传输配置字段就变得可用。
有关 JCA 绑定的详细信息，请参阅第 12 章“JCA 绑定”。
适配器名称
一个只读值，说明 JCA 服务将使用的适配器的名称。
适配器类型
一个只读值，说明适配器类型。
分派策略
选择要对此端点的分派策略使用的 WebLogic Server 工作管理器实例。如果不存在其他工作管理器，则使用默认工作管理器。
有关工作管理器的信息，请参阅 Oracle Fusion Middleware Configuring Server Environments for Oracle WebLogic Server 中的“Using Work Managers to Optimize Scheduled Work”，网址为：http://www.oracle.com/pls/as111170/lookup?id=CNFGD112。
JNDI 服务帐户
JNDI 服务帐户是为 JNDI 上下文安全而设的，用于访问 EIS 适配器管理的连接工厂。单击浏览并选择服务帐户。如果未指定任何服务帐户，则将使用匿名主题。
对于 JCA 业务服务，对可配置的 JNDI 服务帐户的类型 (如“静态”或“通过”) 没有限制，但是运行时必须能够访问用户名和口令。JCA 代理服务只能使用静态 JNDI 服务帐户。
有关 JNDI 服务帐户的详细信息，请参阅 Oracle Fusion Middleware Developer's Guide for Oracle Service Bus 的“JCA Transport”一章中的“Security”，网址为：http://www.oracle.com/pls/as111170/lookup?id=OSBDV916。
端点属性
使用此字段可将值分配到服务所使用的适配器类型的端点属性，例如“重试次数”。
有关受支持的 JCA 端点属性的列表，请参阅 Oracle Fusion Middleware Developer's Guide for Oracle Service Bus 中的“Endpoint Properties”，网址为：http://www.oracle.com/pls/as111170/lookup?id=OSBDV1841。
动态端点属性
使用此选项可将请求参数传递到符合 JCA 的服务。例如，可以使用动态端点属性将数据库查询参数传递到 Oracle JCA 数据库适配器。
有关通过参数查询的详细信息，请参阅 Oracle Fusion Middleware User's Guide for Technology Adapters 中的“Oracle JCA Adapter for Database”，网址为：http://www.oracle.com/pls/as111170/lookup?id=TKADP2117。
输入要提供的每个动态端点属性的名称/值对。端点属性密钥与查询参数名匹配。
始终使用来自 JCA 文件的配置
此选项确定是否始终使用来自 JCA 文件的激活规范属性 (代理服务) 和交互规范属性 (业务服务)。
如果选择此选项 (默认值)，则 JCA 传输将使用 JCA 文件中的激活/交互规范属性与 JCA 框架进行交互。
如果取消选择此选项，则可以覆盖激活/交互规范属性。
有关使用此选项对重新部署的影响，请参阅 Oracle Fusion Middleware Developer's Guide for Oracle Service Bus 的“JCA Transport”一章中的“Endpoint Redeployment”，网址为：http://www.oracle.com/pls/as111170/lookup?id=OSBDV940。
操作名
显示所选 WSDL 操作的只读名称。一个操作可以有其自己的激活/交互规范属性，如“激活规范属性/交互规范属性”字段中所示。
激活规范属性/交互规范属性
激活规范属性是代理服务的字段名称；交互规范属性是业务服务的字段名称。
如果此服务是由 EIS 应用程序调用的入站服务，则此字段将显示“操作名”字段中所示的 JCA 入站操作的激活规范属性。
如果取消选择始终使用来自 JCA 文件的配置，则可以覆盖激活/交互规范属性。
注： 对于 Oracle Adapter Suite 适配器，激活/交互规范属性显示为只读状态。Oracle Adapter Suite 适配器存储其自己的配置，您必须在 Oracle Adapter Suite 管理工具中更改这些配置。
连接属性 (旧)
有关使用非托管模式连接属性的旧 JCA 服务 (在此发行版中已废弃)，请参阅 http://download.oracle.com/docs/cd/E13159_01/osb/docs10gr3/jcatransport/transport.html#wp1105451 中的连接配置选项。

有关端点和激活/交互规范属性的详细信息，请参阅 Oracle Fusion Middleware User's Guide for Technology Adapters，网址为：http://www.oracle.com/pls/as111170/lookup?id=TKADP。
