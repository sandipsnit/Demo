“JMS 传输配置”页

创建/编辑代理服务 - JMS 传输配置页是一系列用于创建和编辑代理服务的页面中的一个页面，如创建和配置代理服务和编辑代理服务配置中所述。

有关详细信息，请参阅 Oracle Fusion Middleware Developer's Guide for Oracle Service Bus 中的“JMS Transport”，网址为：http://www.oracle.com/pls/as111170/lookup?id=OSBDV1037。

使用此页可为使用 JMS 传输协议的代理服务配置传输设置。表 19-10 介绍了如何使用此页。

表 19-10“创建/编辑代理服务 - JMS 传输配置”页
选项
创建或编辑...
目标类型
选择以下选项之一：
  • 队列 (用于点对点目标类型)
  • 主题 (用于发布/订阅目标)
是否需要响应
只有在对目标类型选择队列后，此选项才可用。
选择此选项可指定希望在发送出站消息之后获得响应。
响应模式
只有在选中“是否需要响应”复选框后，该选项才可用。
选择以下选项之一：
  • 为运行在 Oracle WebLogic Server 上的 JAX-RPC 服务选择 JMSMessageID。
  • 为所有其他服务选择 JMSCorrelationID。选择此选项时，还必须输入响应 URI。
响应消息类型
只有在选中是否需要响应复选框后，该选项才可用。
选择以下选项之一：
  • 字节 (用于未解析的字节流)
  • 文本 (用于文本消息)
如果您为响应选择 Java 消息类型，则此选项将禁用。
客户机 Jar
当服务为具有 Java 请求类型的消息传送服务时，此选项可用。请选择要用于将包含 Java 对象的消息出队的客户机 JAR。选择客户机 JAR 时请确保它位于类路径中。

有关详细信息，请参阅 Oracle Fusion Middleware Developer's Guide for Oracle Service Bus 中的“Sending and Receiving Java Objects in Messages”，网址为：http://www.oracle.com/pls/as111170/lookup?id=OSBDV1874。
分派策略
选择要对此端点的分派策略使用的 Oracle WebLogic Server 工作管理器实例。如果不存在其他工作管理器，则使用默认工作管理器。
例如，如果业务服务具有 JMS 传输协议，则该业务服务端点是一个 MDB (消息驱动 Bean) JAR 文件，您可以将其与特定的分派策略进行关联。
有关工作管理器的信息，请参阅 Oracle Fusion Middleware Configuring Server Environments for Oracle WebLogic Server 中的“Using Work Managers to Optimize Scheduled Work”，网址为：http://www.oracle.com/pls/as111170/lookup?id=CNFGD112。
请求编码
输入编码请求的字符集。默认值为 UTF-8。
响应编码
只有在选中“是否需要响应”复选框后，该选项才可用。
输入编码响应的字符集。默认值为 UTF-8。
客户机响应超时
只有在选中“是否需要响应”复选框后，该选项才可用。
输入在删除连接之前等待服务器响应的秒数。仅当客户机是同一域中的另一个代理服务时，此选项才适用。
响应 URI
只有在对“响应相关性模式”选择了 JMSCorrelationID 后，该选项才可用。
按照以下格式输入响应 URI：
jms://host:port/connection_factory/jndi_destination
要定位多个服务器，请使用以下格式：
jms://host1:port,host2:port/connection_factory/jndi_destination
您也可以省略响应 URI 中的主机和端口。例如：
jms:///connection_factory/jndi_destination

省略主机和端口时，会在本地服务器上进行连接工厂/目标查找。在以下情况下，这很有用：例如，如果请求 URI 转至外部连接工厂/目标，但您希望响应发送到本地服务器。

注：虽然 Oracle WebLogic Server 允许在 JNDI 名称中使用正斜杠 (例如“myqueues/myqueue”)，但是带正斜杠的 JNDI 名称会对 Oracle Service Bus 所需的 URI 格式产生干扰，因此您不能使用这些名称。要解决此问题，请定义一个 JMS 外部服务器，然后在 URI 中引用该外部服务器。有关详细信息，请参阅 Oracle Fusion Middleware Oracle WebLogic Server Administration Console Online Help 中的“Configure foreign servers”，网址为：http://www.oracle.com/pls/as111170/lookup?id=WLACH。

响应连接工厂
只有在对“响应相关性模式”选择了 JMSMessageID 后，该选项才可用。
输入响应连接工厂 URI。
如果未指定连接工厂，则对响应使用用于请求的连接工厂。
JMS 服务帐户
选择要用于 JMS 服务器管理的 JMS 资源的服务帐户。服务帐户是用于请求和响应的用户 ID 和口令的别名资源。对于 JMS 和 JNDI 使用相同的服务帐户。
有关详细信息，请参阅服务帐户。
使用 SSL
只有通过 TLS/SSL 连接创建请求后，才能选择此选项。
TLS/SSL (安全套接字层) 通过允许网络上连接的两个应用程序对对方的标识进行验证，并通过对应用程序之间交换的数据进行加密，来提供安全连接。验证操作使服务器 (还可以使客户机) 能够验证网络连接另一端应用程序的标识。另外，如果管理员通过对目标的 JNDI 条目设置访问控制，限制了对各个 JMS 目标 (队列或主题) 的访问权限，则当此服务在 JNDI 树中查找此条目时，必须进行验证。
消息选择器
输入消息选择器表达式。
仅处理其属性与表达式相匹配的消息。
JMS 客户机 ID

仅为主题输入要用于订户的客户机 ID。如果未输入值，则将自动生成客户机 ID。分配定制 ID 有助于在监视服务时标识此组件。请注意，这仅对持久订户生效。

JMS 客户机 ID 是多个 MDB 描述符之一，用于为主题的订户生成订户名称和 ClientID 值。定义 JMS 客户机 ID 可以通过查看订户名称或客户机 ID 来更方便地标识和查看特定主题的订户。

有关详细信息，请参阅 Oracle Fusion Middleware Developer's Guide for Oracle Service Bus 中的“JMS Transport”，网址为：http://www.oracle.com/pls/as111170/lookup?id=OSBDV1037。
持久订阅
只有在对目标类型选择主题后，此选项才可用。
如果订阅是持久订阅，请选中此复选框；如果订阅不是持久订阅，请将其留空。
重试计数
输入将消息移动到错误目标前，消息可具有的传送重试的次数。该字段仅适用于 Oracle WebLogic Server JMS 目标。
重试间隔
输入重新传送回滚的消息或恢复的消息之前所需经过的时间 (毫秒)。该字段仅适用于 Oracle WebLogic Server JMS 目标。
错误目标
输入已达到重新传送限制的消息的目标名称。该字段仅适用于 Oracle WebLogic Server JMS 目标。
失效策略
选择在某个目标上遇到失效消息时要使用的失效策略。该字段仅适用于 Oracle WebLogic Server JMS 目标。
XA 是否是必需的
如果连接工厂为 XA，请选中此复选框。
当远程连接工厂不可用时，请考虑此值。如果连接工厂可用且此值为“真”，请确保连接工厂被定义为事务处理连接工厂。
事务处理超时
输入 JMS 代理服务等待处理事务处理的时间 (秒)。此选项仅适用于使用 XA 连接工厂的服务。默认值是 600 秒。
主题消息分发
为目标类型选择主题时可使用此选项。主题消息分发提供了不同选项以用于处理入站 JMS 消息以及提供高可用性和故障转移。

选择以下属性之一可确定消息驱动 Bean 处理传入 JMS 消息的方式：
  • 每个应用程序一个副本 (默认值) – 如果将 JMS 代理服务部署到集群，则此选项将通过确保仅在该集群中的一个可用服务器上处理入站 JMS 消息来提供高可用性和可扩展性。
  • 每个服务器一个副本 – 如果您希望集群的每个成员上的代理服务接收发布到主题的入站 JMS 消息，请选择此选项。
  • 兼容性 – 如果您希望在集群中的特定托管服务器或所有托管服务器上处理入站 JMS 消息，请选择此选项。如果选择此属性，请选择目标服务器。
注：对于部署在早于版本 10.3.4 的 Oracle WebLogic Server 上的主题目标，唯一有效的选项是“兼容性”。
您为“主题消息分发”选择的“一个副本”选项将覆盖“JMS 连接工厂”上配置的订阅共享策略和/或客户机 ID 策略。
目标
仅当为“主题消息分发”选项选择“兼容性”时，此选项才在 Oracle Service Bus 集群中可用。请选择将处理入站 JMS 消息的目标服务器。如果您为“主题消息分发”选择了“一个副本”选项之一，则此字段会显示集群的名称。

如果您未设置目标，则集群中的每个托管服务器上的 JMS 代理服务实例 (从主题读出消息) 均将获得消息的副本。
JNDI 超时
在 JNDI 树中查找目标或连接工厂时使用的 JNDI 连接超时 (秒)。
