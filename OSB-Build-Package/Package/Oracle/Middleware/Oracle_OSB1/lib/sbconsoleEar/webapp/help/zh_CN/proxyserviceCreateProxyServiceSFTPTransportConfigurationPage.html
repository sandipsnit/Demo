“SFTP 传输配置”页

创建/编辑代理服务 - SFTP 传输配置页是一系列用于创建和编辑代理服务的页面中的一个页面，如创建和配置代理服务和编辑代理服务配置中所述。

有关详细信息，请参阅 Oracle Fusion Middleware Developer's Guide for Oracle Service Bus 中的“HTTP and Poller Transports”，网址为：http://www.oracle.com/pls/as111170/lookup?id=OSBDV947。

使用此页可为使用 SFTP 传输协议的代理服务配置传输设置。表 19-13 介绍了如何使用此页。

表 19-13“创建/编辑代理服务 - SFTP 传输配置”页
选项
创建或编辑...
用户验证
选择以下选项之一：
  • 用户名和口令验证 - 指定静态服务帐户与此验证方法相关联，并使用所提供的身份证明对客户机进行验证。
  • 基于主机的验证 - 指定要使用此验证方法，用户名和服务密钥提供程序是必需的。从已知的主机进行连接的任何用户都使用主机的私有密钥进行验证。
  • 公共密钥验证 - 指定要使用此验证方法，用户名和服务密钥提供程序是必需的。每个用户都拥有自己的私有密钥。
服务帐户
输入用户的服务帐户，或者单击浏览从浏览器中选择服务帐户。
服务密钥提供程序
只有在选择基于主机或公共密钥验证后，此选项才可用。
在服务密钥提供程序字段中输入服务密钥提供程序。可单击浏览从浏览器中选择服务密钥提供程序。这是必填字段。
用户名
只有在选择基于主机或公共密钥验证后，此选项才可用。
输入用户名。
引用传递
选中此复选框可将文件临时保存在归档目录中，并将其作为标头中的引用进行传递。
远程传输
选中该复选框可在进行处理时直接从远程服务器传输 SFTP 文件。如果选择此选项，则归档目录即是远程 SFTP 服务器上的远程目录。因此，应以 SFTP 用户目录的相对路径的形式指定归档目录。
文件掩码
输入要选取的文件的正则表达式。默认值为 *.*。
托管服务器
此字段仅在集群域中可用。
选择托管服务器作为轮询服务器。所有托管服务器均可处理消息，但是只有一个托管服务器可以轮询消息。
轮询间隔
输入从指定位置轮询文件的间隔 (秒)。默认值为 60。
读取限制
指定每次轮询扫描读取的最大消息数。输入 0 可指定没有限制。默认值为 10。
读后操作
选择如何处理阅读后的消息。
  • 归档 - 将消息归档。
  • 删除 - 将消息删除。
归档目录
如果读后操作字段设置为归档，请指定归档位置的路径。如果选择了引用传递选项，则此字段是必填字段。
注： 归档、下载和错误目录都是绝对路径，并且是自动创建的。如果指定了相对路径，则将相对于启动 WebLogic 服务器的 Java 进程创建文件。
下载目录
输入在文件传输期间将文件下载到的本地计算机上的目录。
注： 归档、下载和错误目录都是绝对路径，并且是自动创建的。如果指定了相对路径，则将相对于启动 WebLogic 服务器的 Java 进程创建文件。
错误目录
输入在出现问题的情况下要将消息发布到的位置。
注： 归档、下载和错误目录都是绝对路径，并且是自动创建的。如果指定了相对路径，则将相对于启动 WebLogic 服务器的 Java 进程创建文件。
请求编码
接受默认值 UTF-8 作为 SFTP 传输中请求的字符集编码。
首选密码套件
选择与服务器通信时使用的密码套件。
如果使用默认值使用运行时默认值，则会将支持的密码套件列表发送到服务器，并且将尝试每个密码套件，直到发现匹配项。
首选数据完整性算法
为数据完整性检查选择批量散列算法。
如果使用默认值使用运行时默认值，则 Oracle Service Bus 会将首选算法 hmac-sha1 发送到服务器。如果服务器不支持该算法，则会将支持的算法列表发送到服务器并尝试每个算法，直到发现匹配项。
首选公共密钥算法
为公共密钥加密选择非对称密钥算法。
如果使用默认值使用运行时默认值，则 Oracle Service Bus 会将首选算法 ssh-dss 发送到服务器。如果服务器不支持该算法，则会将支持的算法列表发送到服务器并尝试每个算法，直到发现匹配项。
首选密钥交换算法
为协商会话密钥选择加密消息所用的默认密钥交换协议。
如果使用默认值使用运行时默认值，则 Oracle Service Bus 会将首选算法 diffie-hellman-group1-sha1 发送到服务器。如果服务器不支持该算法，则会发送支持的算法列表并尝试每个算法，直到发现匹配项。
首选压缩算法
选择是否使用 zlib 压缩正在使用的数据。选择 zlib 可压缩数据；否则选择无。默认值为无。
扫描子目录
选中该复选框可递归扫描所有目录
按到达时间排序
选中该复选框可按到达时间顺序传送事件。
超时
输入在删除连接之前所需经过的套接字超时间隔 (秒)。如果输入 0，则没有超时。默认值为 60。
重试计数
指定 SFTP 连接失败时重试的次数。

完成之后

单击下一步继续在下一页上配置此服务；或单击最后一步在“创建/编辑代理服务 - 概要”页上查看并保存此配置。
