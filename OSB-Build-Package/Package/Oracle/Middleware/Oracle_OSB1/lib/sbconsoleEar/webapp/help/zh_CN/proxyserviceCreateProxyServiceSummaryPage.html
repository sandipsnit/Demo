“概要”页

创建/编辑代理服务 - 概要页是一系列用于创建和编辑代理服务的页面中的一个页面，如创建和配置代理服务和编辑代理服务配置中所述。

使用此页可在保存代理服务的配置设置之前对其进行查看或修改。

要查看或修改设置，请在相应类别 (如一般配置、传输配置等) 的行中单击编辑。可编辑的页取决于创建代理服务时配置的页。下面的列表显示了所有页：

  • “一般配置”页
  • “消息类型配置”页
  • “传输配置”页
  • “协议特定的传输配置”页
  • “电子邮件传输配置”页
  • “文件传输配置”页
  • “FTP 传输配置”页
  • “HTTP 传输配置”页
  • “JCA 传输配置”页
  • “JEJB 传输配置”页
  • “JMS 传输配置”页
  • “MQ 传输配置”页
  • “SB 传输配置”页
  • “SFTP 传输配置”页
  • “Tuxedo 传输配置”页
  • “WS 传输配置”页
  • “操作选择配置”页
  • “消息处理”页
