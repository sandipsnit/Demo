“Tuxedo 传输配置”页

创建/编辑代理服务 - Tuxedo 传输配置页是一系列用于创建和编辑代理服务的页面中的一个页面，如创建和配置代理服务和编辑代理服务配置中所述。

有关详细信息，请参阅 Oracle Fusion Middleware Developer's Guide for Oracle Service Bus 中的“Tuxedo Transport”，网址为：http://www.oracle.com/pls/as111170/lookup?id=OSBDV1200。

使用此页可为使用 Tuxedo 传输协议的代理服务配置传输设置。表 19-14 介绍了如何使用此页。

表 19-14“创建/编辑代理服务 - Tuxedo 传输配置”页
选项
创建或编辑...
字段表类
输入描述收到的 FML/FML32 缓冲区的类的名称。FML/FML32-to-XML 转换例行程序使用它们将字段名映射到元素名。这是一个由空格分隔的全限定类名列表。
视图类
输入描述收到或发送的 VIEW/VIEW32 缓冲区的类的名称。VIEW-to-XML 或 VIEW32-to-XML 转换例行程序使用它们将字段名映射到元素名。这是一个由空格分隔的全限定类名列表。
X_C_TYPE 和 X_COMMON Tuxedo 缓冲区类型的处理方式与 VIEW/VIEW32 缓冲区相同。
如果传入请求包含 VIEW，则应在 Oracle Service Bus CLASSPATH 中指定相应的 VIEW 类。
类 Jar
选择一个 JAR 资源，该资源应包含一个 JAR 文件，以及此端点操作所需的 FML/FML32 或 VIEW/VIEW32 类。
本地访问点
从下拉列表中选择一个与导出相关联的本地访问点。该下拉列表包含 WTC 中配置的本地访问点。如果没有相关联的本地访问点，则无法创建代理服务。
如果没有任何本地访问点存在，或者要新建本地访问点，请选择新建。在相邻字段中输入相应的本地访问点名称和本地网络地址。在验证端点时，会将该访问点添加到每个 WTC 服务器的 WTC 配置中。如果不存在任何 WTC 服务器，则会创建一个 WTC 服务器。
可在选择新建选项之后输入现有访问点名称。这将导致使用新参数更新现有信息。您只能更改主机名和端口号。
远程访问点
只有在选择本地访问点字段中的新建后，才会显示此字段。
在下拉列表中，选择要与新建的本地访问点关联的远程访问点。如果不存在任何本地访问点，或者要新建本地访问点，请选择新建。在相邻字段中输入相应的访问点名称和网络地址。
可在选择新建选项之后输入现有访问点名称。这将导致使用新参数更新现有信息。您只能更改主机名和端口号。
远程访问点也将是用于入站请求的 WTC 连接的验证主用户。还可以使用默认安全领域中的同一访问点 ID 创建用户，以允许传入调用。为此，请从“是否创建用户？”下拉列表中选择“是”。将使用临时变量随机生成口令以避免安全问题。
回复缓冲区类型
只有在选择是否需要响应？字段的情况下，此选项才可用。
选择远程 Tuxedo 客户机将接收的缓冲区的类型。
回复缓冲区子类型
只有在选择是否需要响应？字段且回复缓冲区类型值为 VIEW 或 VIEW32 的情况下，此选项才可用。
输入与回复缓冲区关联的缓冲区子类型。
是否需要响应？
如果此服务应发送响应，则选中此复选框。
默认状态是此选项处于选定状态。
如果服务类型为消息传送服务且响应消息类型为无，则该选项会被清除且不可用。
请求编码
指定 Tuxedo 传输中请求的字符集编码。
响应编码
指定 Tuxedo 传输中响应的字符集编码。
转换样式
选择以下选项之一：
  • 无 - (默认值) 可以不考虑字段顺序。
  • 有序 - 将按照字段出现的正确顺序显示它们。
  • 有序和分组 - 如果字段作为记录进行逻辑组织，则这些字段将按出现的先后进行排序，并且按记录进行分组。

完成之后

单击下一步继续在下一页上配置此服务；或单击最后一步在“创建/编辑代理服务 - 概要”页上查看并保存此配置。
