“WS 传输配置”页

创建/编辑代理服务 - WS 传输配置页是一系列用于创建和编辑代理服务的页面中的一个页面，如创建和配置代理服务和编辑代理服务配置中所述。

有关详细信息，请参阅 Oracle Fusion Middleware Developer's Guide for Oracle Service Bus 中的“WS Transport”，网址为：http://www.oracle.com/pls/as111170/lookup?id=OSBDV1083。

使用此页可为使用 WS 传输协议的代理服务配置传输设置。表 19-15 介绍了如何使用此页。

表 19-15“创建/编辑代理服务 - WS 传输配置”页
选项
创建或编辑...
分派策略
为此端点选择一个分派策略，或使用默认分派策略。
分派策略是指您希望用于此服务端点的 WLS 工作管理器实例。有关工作管理器的信息，请参阅 Oracle Fusion Middleware Configuring Server Environments for Oracle WebLogic Server 中的“Using Work Managers to Optimize Scheduled Work”，网址为：http://www.oracle.com/pls/as111170/lookup?id=CNFGD112。
重试计数
反复尝试将消息传送到管道的次数。
如果代理服务的请求管道中发生未处理的异常错误，则传入 WS 传输消息将重新传送到该管道，直到达到了由该重试计数指定的次数。此值对于可靠处理 WS 传输消息非常重要。
重试延迟
出错之后，系统在重新尝试将消息发送到管道之前暂停的秒数。

完成之后

单击下一步继续在下一页上配置此服务；或单击最后一步在“创建/编辑代理服务 - 概要”页上查看并保存此配置。
