创建要在内嵌表达式中使用的名称空间

名称空间定义选项板包括默认名称空间的列表，但您也可以定义新的名称空间。

开始之前

这些说明假定您将在 XQuery 表达式编辑器、XQuery 条件编辑器或 XPath 表达式编辑器中创建或编辑表达式，如创建和编辑内嵌 XQuery 和 XPath 表达式中所述。

要在内嵌表达式中创建并使用名称空间，请执行以下操作：

  1. 选择名称空间定义。名称空间定义选项板包括默认名称空间的列表，以及变量名称空间列表和用户定义的名称空间列表 (如果存在)。
  2. 要定义并添加用户名称空间，请执行以下操作：
    1. 单击添加名称空间。
    2. 在前缀字段中，输入名称空间的唯一标识符。不能多次使用同一个前缀。
    3. 在 URI 字段中，以 http://url/.../ 格式输入此名称空间的 URL，或者以 uddi:server: 格式输入 URN。
    4. 单击添加将名称空间添加到用户定义的名称空间列表中。
    5. 复制用户定义的名称空间，然后将其粘贴到 XQuery 表达式、XQuery 条件或 XPath 中，如创建和编辑内嵌 XQuery 和 XPath 表达式中所述。

完成之后

继续执行操作，如创建和编辑内嵌 XQuery 和 XPath 表达式中所述。
