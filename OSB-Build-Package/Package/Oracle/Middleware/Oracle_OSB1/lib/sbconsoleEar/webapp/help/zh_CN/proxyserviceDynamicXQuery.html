将动态 XQuery 表达式绑定到内嵌 XQuery

通过 XQuery 表达式编辑器页，可以指定在运行时求值结果为预注册 XQuery 资源名称的动态 XQuery 表达式。要了解此编辑器的详细信息，请参阅创建和编辑内嵌 XQuery 和 XPath 表达式。

要定义动态 XQuery 表达式，请执行以下操作：

  1. 选择动态 XQuery 选项。
  2. 在输入 XQuery 资源的表达式区域中，输入在运行时求值结果为预注册 XQuery 资源名称的 XQuery 表达式。

    3. 下面说明了 XQuery 资源的语法 (代表资源的全称)：

    Project/folder1/folder2/XQueryResourceName

  3. 在使用 XQuery 模板绑定变量，或绑定定制变量区域中，定义用于转换的输入参数。
    • 单击选择 XQuery 模板字段中的“浏览”，选择一个现有的注册资源以用作查询的形式（变量的数量和名称）的模板。选择模板后，变量将显示在绑定变量区域中。请注意，模板不会随配置一起保留下去。而是用作一种快速启动方法，帮助您指定查询的变量。
    • 在添加定制变量字段中键入变量名称，然后单击“添加”。为变量名下列出的每个变量输入要映射到该变量的 XQuery 表达式。必须为每个参数都定义一个映射。例如，如果一个 XQuery 转换有两个输入参数，名为 one 和 two，则变量名字段会有两个标签 - one 和 two。用于输入 XQuery 表达式的文本框与每一个标签相关联。

      • 以下 XQuery 表达式是变量字段有效输入值的示例：

      $body/*[1]

      $body/po:PurchaseOrder

  4. 完成后，请继续执行以下任何任务。
    • 单击验证。如果表达式验证成功，则会显示消息。
    • 单击测试。请参阅执行 XQuery 测试。
    • 保存或放弃更改。
