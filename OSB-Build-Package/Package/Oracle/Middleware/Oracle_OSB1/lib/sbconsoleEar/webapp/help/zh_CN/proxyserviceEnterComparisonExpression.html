使用构建器选项输入 XQuery 比较表达式

开始之前

这些说明假定您将在 XQuery 条件编辑器中创建或编辑 XQuery 条件表达式，如创建和编辑内嵌 XQuery 和 XPath 表达式中所述。

要输入 XQuery 比较表达式，请执行以下操作：

  1. 在 XQuery 条件编辑器中，如果构建器 (位于工作区按钮栏下) 尚未处于选定状态，请选择该选项。
  2. 在表达式构建器框中，选择比较表达式 (如果该选项尚未处于选定状态)。
  3. 在操作数字段中，输入上下文变量、名称空间定义或 XQuery 函数。

    4. 要构建操作数，可从 XQuery 函数选项板和名称空间定义选项板中分别粘贴 XQuery 函数和名称空间。请参阅在编辑器工作区的文本字段中构建表达式中的步骤 3 和步骤 4。

  4. 在运算符列表中，选择比较运算符。
  5. 在值字段中，输入文本或上下文变量。

    6. 必须使用引号将输入文本括起来，例如，"true" 是有效的，而 true 是无效的。

  6. 单击添加。输入的文本会显示在表达式窗格中。
  7. 重复步骤 3 至步骤 6，以构建其他条件。每个条件会添加到条件列表的末尾。

    8. 使用多个条件时，请考虑以下事项：

    • 构建其他表达式时，请确保在关联字段中选择了与或或选项。
    • 可以选择某个条件，然后单击上箭头在条件列表中上移该条件，或者单击下箭头在条件列表中下移该条件。此外，可以单击编辑图标更新条件，或者单击删除图标将其删除。
    • 在某一条件的总体定义中，一元表达式可以与比较表达式混合使用。
  8. 还可以执行下面的任一或全部操作：
    • 单击验证。如果表达式验证成功，则会显示消息。
    • 单击测试可测试表达式。请参阅执行 XQuery 测试。
  9. 单击保存关闭编辑器并将表达式插入操作中。

完成之后

继续配置该操作，如向消息流添加操作和编辑消息流中的操作中所述。
