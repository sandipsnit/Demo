使用构建器选项输入一元表达式

开始之前

这些说明假定您将在 XQuery 条件编辑器中创建或编辑 XQuery 条件表达式，如创建和编辑内嵌 XQuery 和 XPath 表达式中所述。

要输入 XQuery 比较表达式，请执行以下操作：

  1. 在 XQuery 条件编辑器中，如果构建器 (位于工作区按钮栏下) 尚未处于选定状态，请选择该选项。
  2. 在表达式构建器框中，选择一元表达式 (如果该选项尚未处于选定状态)。
  3. 选中非复选框使该表达式成为负表达式，或将该复选框留空。
  4. 在表达式字段中输入上下文变量、名称空间定义或 XQuery 函数。

    5. 要构建表达式，可从 XQuery 函数选项板和名称空间定义选项板中分别粘贴 XQuery 函数和名称空间。请参阅在编辑器工作区的文本字段中构建表达式中的步骤 3 和步骤 4。

  5. 单击添加将文本添加至表达式窗格。
  6. 重复步骤 3 至步骤 5，以构建其他条件。每个条件会添加到条件列表的末尾。

    7. 构建一元表达式时，请考虑以下事项。

    • 构建其他表达式时，请确保在关联字段中选择了与或或选项。
    • 可以选择一个条件，然后，单击上箭头在条件列表中将其上移，单击下箭头在条件列表中将其下移，单击编辑图标将其更新，或者单击删除图标将其删除。
    • 在某一条件的总体定义中，一元表达式可以与比较表达式混合使用。
  7. 还可以执行下面的任一或全部操作：
    • 单击验证。如果表达式验证成功，则会显示消息。
    • 单击测试可测试表达式。请参阅执行 XQuery 测试。
  8. 单击保存关闭编辑器并将表达式插入操作中。

完成之后

继续配置该操作，如向消息流添加操作和编辑消息流中的操作中所述。
