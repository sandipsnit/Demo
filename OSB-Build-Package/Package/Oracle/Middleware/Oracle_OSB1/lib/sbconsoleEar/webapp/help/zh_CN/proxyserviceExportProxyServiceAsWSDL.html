导出与代理服务关联的 WSDL

您可以导出基于 WSDL 的代理服务的 WSDL，这样，您就可以在外部工具 (如 IDE) 中查看或修改该 WSDL。WSDL 是以 JAR 文件的形式导出的。

请注意，此功能与“系统管理”模块中的“导出资源”功能不同，后者用于在两个域之间移动和暂存资源。请参阅导出资源。

开始之前

只有在会话之外时，才能导出 WSDL。请参阅使用更改中心。

导出 WSDL

  1. 请执行以下两个操作之一：
    • 选择资源浏览器 > 代理服务以便显示代理服务概要页。
    • 选择项目浏览器，然后选择包含要作为 WSDL 导出的代理服务的项目或文件夹。此时将显示“项目/文件夹视图”页。
  2. 在要导出其 WSDL 的代理服务所在的行中，单击导出 WSDL 图标。对话框将提示您打开或保存导出的 JAR 文件。
  3. 在该对话框中，单击打开以打开文件，或者单击保存以保存该文件。
