将外部 XQuery 资源绑定到内嵌 XQuery

可将 XQuery 资源绑定到内嵌 XQuery 表达式，以便将其作为操作的一部分在内嵌模式下来执行。

开始之前

这些说明假定您将在 XQuery 表达式编辑器、XQuery 条件编辑器或 XPath 表达式编辑器中创建或编辑表达式，如创建和编辑内嵌 XQuery 和 XPath 表达式中所述。

要将 XQuery 资源绑定到内嵌表达式，请执行以下操作：

  1. 单击变量结构。
  2. 在工作区 (位于按钮栏下) 中，选择 XQuery 资源。
  3. 在 1.选择要执行的 XQuery 资源框中，单击浏览。
  4. 在 XQuery 浏览器中，选择与要使用的 XQuery 关联的单选按钮，然后单击提交。
  5. 在 2.绑定变量框中，定义用于转换的输入参数。为变量名之下列出的每个变量输入要映射到该变量的 XQuery 表达式。必须为每个参数都定义一个映射。例如，如果一个 XQuery 转换有两个输入参数，名为 one 和 two，则变量名字段会有两个标签 - one 和 two。XQuery 表达式输入到的文本框与每一个标签相关联。

    6. 以下 XQuery 表达式是此字段有效输入值的示例：

    $body/*[1]

    $body/po:PurchaseOrder

    注： 以下变量名不是该字段的有效输入，将会导致异常错误：
    body
  6. 完成后，请继续执行以下任何任务。
    • 单击验证。如果表达式验证成功，则会显示消息。
    • 单击测试。请参阅执行 XQuery 测试。
    • 保存或放弃更改。
