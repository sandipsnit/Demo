将外部 XSLT 资源绑定到内嵌 XQuery

通过 XQuery 表达式编辑器页，可以选择要执行的 XSLT 资源。要了解此编辑器的详细信息，请参阅创建和编辑内嵌 XQuery 和 XPath 表达式。

要选择要执行的 XSLT 资源，请执行以下操作：

  1. 选择 XSLT 资源选项。
  2. 在选择要执行的 XSLT 资源字段中，选择 XSLT 浏览器图标。
  3. 在 XSLT 浏览器中，选择与要执行的 XSLT 相关联的单选按钮，然后单击提交。
  4. 在绑定变量字段下，会为该转换的每一个输入参数显示一个标签和相应的文本框。每一标签对应于一个参数的名称，每一文本框用于定义要映射到该参数的 XQuery 表达式。必须为每个参数都定义一个映射。例如，如果一个 XSL 转换有两个输入参数，名为 one 和 two，则变量映射字段会有两个标签：one 和 two，以及与每一个参数相关联的文本框 (XQuery 表达式在此文本框中输入)。除了任何输入变量的映射，还必须为该转换的输入文档指定 XQuery 表达式。映射在带有输入文档标签的文本框中指定。

    5. 以下 XQuery 表达式是此字段有效输入值的示例：

    $body/*[1]

    $body/po:PurchaseOrder

    注： 以下变量名不是该字段的有效输入，将会导致异常错误：
    body
  5. 继续执行以下任务之一：
    • 创建要在内嵌表达式中使用的名称空间。
    • 在 XQuery 编辑器中创建变量结构。
    • 在编辑器工作区的文本字段中构建表达式。
    • 将外部 XQuery 资源绑定到内嵌 XQuery。
    • 单击验证。如果表达式验证成功，则会显示消息。
    • 单击测试。请参阅执行 XQuery 测试。
