“查看代理服务 - 配置详细资料”页

查看代理服务 - 配置详细资料页显示代理服务的配置详细资料。表 19-19 介绍了此页上可显示的所有属性。(属性因代理服务的详细资料而异。)

此页上列出的类别与用来创建和编辑代理服务配置的创建/编辑代理服务页相对应，如创建/编辑代理服务 - 页引用中所述。

单击任何类别名称旁边的编辑链接可显示关联的配置页。

表 19-19“查看代理服务：配置详细资料”页
属性
说明
上次修改者
创建该代理服务或将其导入配置的用户。
上次修改时间
用户创建该代理服务或将其导入到配置中的日期和时间。单击日期和时间链接可查看此资源的更改历史记录。请参阅“查看更改历史记录”页。
引用
此代理服务引用的对象数。如果存在此类引用，可单击数字链接以查看对象列表。
引用项

说明
