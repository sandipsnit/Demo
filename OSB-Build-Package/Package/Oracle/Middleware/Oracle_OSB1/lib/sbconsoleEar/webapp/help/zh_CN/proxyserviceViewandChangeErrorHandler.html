编辑错误处理程序

开始之前

这些说明假定您已在“编辑消息流”页中编辑了消息流，如创建、编辑和查看消息流中所述。

要查看和更改错误处理程序，请执行以下操作：

  1. 请执行以下操作之一。

    2. 表 23-1 查看和更改错误处理程序 
    操作
    步骤
    查看和更改代理服务消息流错误处理程序
    单击代理服务图标，然后单击编辑服务错误处理程序。此时将显示编辑错误处理程序页。请参阅添加代理服务错误处理程序。
    查看和更改管道错误处理程序
    单击相应的请求管道图标或响应管道图标，然后单击编辑管道错误处理程序。此时将显示编辑错误处理程序页。请参阅添加管道错误处理程序。
    查看和更改路由节点错误处理程序
    单击相应的路由节点图标，然后单击编辑路由错误处理程序。此时将显示编辑错误处理程序页。请参阅添加路由节点错误处理程序。
    查看和更改阶段错误处理程序
    单击相应的阶段图标，然后单击编辑阶段错误处理程序。此时将显示编辑错误处理程序页。请参阅添加阶段错误处理程序。

完成之后

在“编辑消息流”页上，继续构造消息流，如创建、编辑和查看消息流中所述。
