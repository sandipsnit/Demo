清除消息

使用此页可从报告数据存储中清除选定消息。

消息清除在后台执行，因此，执行清除时，不会锁定消息报告概要页。因此，可以执行需要一段时间才能完成的清除操作；时间的长度取决于需要清除的消息数。但是，如果筛选报告消息，则清除过程会变慢；消息报告概要页可能会因为有些数据尚未被清除而显示不正确的数据。

清除消息功能可能在项目的设计和测试阶段非常有用。但是，在生产环境中，Oracle 建议数据库中数据的管理 (包括清除) 由数据库管理员使用数据库管理工具来处理。

  1. 选择操作 > 清除消息。
  2. 执行以下步骤之一，如表 27-9 中所示。

    3. 表 27-9 消息清除选项
    操作
    步骤
    清除所有消息
    1. 选择清除所有消息单选按钮。
    2. 单击清除。
    清除指定时段内的消息
    1. 选择清除消息自... 到... 单选按钮。
    2. 在清除消息自字段中，以 MM/DD/YY HH:MM:SS 格式输入月日年。例如，可以输入 10/10/07 12:45:00 AM。
    3. 在到字段中，以 MM/DD/YY HH:MM:SS 格式输入月日年。
    4. 单击清除。
