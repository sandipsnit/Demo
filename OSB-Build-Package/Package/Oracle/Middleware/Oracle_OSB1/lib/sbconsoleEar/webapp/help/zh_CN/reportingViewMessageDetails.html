查看消息详细资料

使用此页可查看特定消息的详细资料。

  1. 查找消息。请参阅查看报告消息。
  2. 单击报告索引。查看消息详细资料页将显示表 27-3 中所示的一般配置信息。

    3. 表 27-3 消息配置参数
    属性
    说明
    消息 ID
    此消息的唯一标识。
    数据库时间戳
    消息在数据库中注册的日期和时间。
    日志记录的时间
    消息的记录日期和时间。
    服务器名
    生成此消息的服务器的名称。
    状态
    生成此消息的管道的状态：
    请求 - 表示报告操作在请求管道中执行。
    响应 - 表示报告操作在响应管道中执行。
    错误 - 操作在服务级错误处理程序中运行。
    节点名
    生成此消息时所在管道的节点名。
    管道名
    生成此消息的管道的名称。
    阶段名
    生成此消息的阶段的名称。

    此页显示表 27-4 中所示的入站服务信息。

    表 27-4 入站服务详细资料
    属性
    说明
    名称
    与此消息关联的入站代理服务。这是指向查看代理服务 - 配置详细资料页的链接。请参阅编辑代理服务配置。
    URI
    与代理服务关联的 URI。
    操作
    与此消息关联的入站操作。

    此页显示表 27-5 中所示的出站服务信息。

    表 27-5 出站服务详细资料
    属性
    说明
    名称
    与此消息关联的出站业务服务。这是指向查看业务服务 - 配置详细资料页的链接。请参阅编辑业务服务配置。
    URI
    指向出站业务服务端点的 URI。
    操作
    对出站服务调用的操作的名称。

    此页显示表 27-6 中所示的报告索引信息。

    表 27-6 报告索引详细资料
    属性
    说明
    报告索引文本
    报告索引是用于从消息上下文变量中提取键标识符的键值对。键是筛选消息的便利方法。可以使用报告操作添加键值对。请参阅向消息流添加操作和编辑消息流中的操作。

    此页显示表 27-7 中所示的故障信息。

    表 27-7 故障信息详细资料
    属性
    说明
    错误代码
    与消息关联的错误代码 (如果有)。
    配置引发错误操作时可以指定错误代码。请参阅 Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus 中的“Handling Errors in Message Flows”(网址为：http://www.oracle.com/pls/as111170/lookup?id=OSBAG232) 和添加代理服务错误处理程序。
    原因
    错误代码的原因。
    详细资料
    与错误代码关联的故障详细资料。这些详细资料 (如果存在) 通常是对特定故障发生位置的堆栈跟踪，并且可能会由于数据库中的大小限制而被截断。限制为 2048 个字符。

    此页显示表 27-8 中所示的报告正文信息。

    表 27-8 报告正文详细资料
    属性
    说明
    报告正文文本
    该链接会打开另一个浏览器，其中显示报告正文的详细资料。可以在报告操作中使用表达式来捕获报告正文文本。有关报告操作的过程，请参阅向消息流添加操作和编辑消息流中的操作。

  3. 查看完特定消息的详细资料后，单击确定。
