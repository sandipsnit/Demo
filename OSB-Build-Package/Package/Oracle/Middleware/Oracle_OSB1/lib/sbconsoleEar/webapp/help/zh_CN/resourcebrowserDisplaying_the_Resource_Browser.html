使用资源浏览器

资源浏览器提供了用于查找资源的工具，以及用于对这些资源执行简单操作 (如“显示”、“编辑”、“导出”和“删除”) 的工具。

  1. 要显示资源浏览器，可选择左侧导航窗格中的资源浏览器。
  2. 在该页的左侧，单击要查找的资源类型的名称。表 3-10 显示了可以选择的项目以及与为每个项目显示的页相关联的任务。
