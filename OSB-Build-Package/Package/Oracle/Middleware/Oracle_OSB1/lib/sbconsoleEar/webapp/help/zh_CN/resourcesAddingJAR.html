添加 JAR

  1. 如果尚未执行此操作，请单击创建新建一个会话，或单击编辑进入现有会话。请参阅使用更改中心。
  2. 选择项目浏览器，然后选择要在其中添加 JAR 的项目或文件夹。此时会显示项目视图/文件夹视图页。
  3. 从创建资源下拉列表中选择实用程序下的 JAR，以便显示创建新的 JAR 资源页。
  4. 在资源名字段中，输入此 JAR 的唯一名称。这是必填字段。

    如需命名指导，请参阅资源命名限制。
  5. 在资源说明字段中，输入此 JAR 的说明。
  6. 在 JAR 资源文件字段中，执行以下操作之一：
    • 输入 JAR 位置的路径。
    • 单击浏览以查找和导入现有 JAR。

      • 这是必填字段。

  7. 单击保存。将创建 JAR 资源并将其保存在当前会话中。

    8. 如果 JAR 加载没有问题，则 JAR 的内容将可供查看。要了解如何查看 JAR 详细资料，请参阅编辑 JAR。

  8. 要结束该会话并将配置部署到运行时，请单击更改中心下的激活。
