编辑预警目标

  1. 如果尚未执行此操作，请单击创建新建一个会话，或单击编辑进入现有会话。请参阅使用更改中心。
  2. 查找预警目标，如查找预警目标或搜索预警目标中所述。
  3. 单击预警目标名称。查看预警目标 - 配置页显示了表 5-2 中所示的信息。

    4. 表 5-2 预警目标配置详细资料 
    属性
    说明
    上次修改者
    创建或编辑此预警目标的用户。
    上次修改时间
    用户创建或编辑此预警目标的日期和时间。单击日期和时间链接可查看此资源的更改历史记录。请参阅“查看更改历史记录”页。
    引用
    此预警目标引用的对象数。如果存在此类引用，可单击数字链接以查看对象列表。请参阅查看对资源的引用。
    引用项
    引用此预警目标的对象数。如果存在此类引用，可单击数字链接以查看对象列表。请参阅查看对资源的引用。
    说明
    此预警目标的说明 (如果存在)。
    SNMP 陷阱
    “是”或“否”，表示进行配置时是否将“SNMP 陷阱”选项指定为此预警的目标。
    报告
    “是”或“否”，表示进行配置时是否将“报告”选项指定为此预警的目标。
    预警日志记录
    “是”或“否”，表示是否启用“预警日志记录”。
    电子邮件和 JMS
    显示电子邮件收件人和/或 JMS 目标的列表 (如果已配置)。

  4. 可以更改您所选的“SNMP 陷阱”、“报告”或“预警日志记录”设置。
  5. 要编辑或删除电子邮件收件人或 JMS 目标，请单击该页底部的编辑。
    1. 要编辑电子邮件收件人，请在电子邮件中，针对要编辑的电子邮件收件人单击选项下的编辑图标。使用编辑电子邮件收件人页可对电子邮件收件人信息进行更改。请参阅添加电子邮件收件人。
    2. 要编辑 JMS 目标，请在 JMS 中，针对要编辑的 JMS 目标单击选项下的编辑图标。使用编辑 JMS 目标页可对 JMS 目标信息进行更改。请参阅添加 JMS 目标。
    3. 要删除一个或所有电子邮件收件人或 JMS 目标，请针对要删除的电子邮件收件人或 JMS 目标单击选项下的删除图标。
  6. 单击保存可提交当前会话中的更新。
  7. 要结束该会话并将配置部署到运行时，请单击更改中心下的激活。
