编辑 JAR 相关性

在 Oracle Service Bus 中，JAR 资源可包含依赖于其他 JAR 中其他类的类。使用编辑 JAR 相关性页可添加或删除引用的 JAR 资源。

  1. 如果尚未执行此操作，请单击创建新建一个会话，或者单击编辑进入现有会话。请参阅使用更改中心。
  2. 查找 JAR，如查找 JAR 中所述。
  3. 单击 JAR 名称。

    4. 在查看 JAR 资源 - JAR 配置页上，相关性选项卡中的数字指示符将显示为此 JAR 资源配置的引用 JAR 的数量，如果没有，则显示零。

  4. 选择相关性选项卡可显示引用的 JAR，也可以添加引用的 JAR。

    5. “JAR 相关性”页包括表 7-4 中所示的信息。

    表 7-4 JAR 相关性信息
    属性
    说明
    名称
    引用的 JAR 的名称。该名称是一个指向查看 JAR 资源 - JAR 配置页的链接。请参阅编辑 JAR。
    路径
    该路径由项目名和 JAR 所在的文件夹名称组成。

  5. 要添加或删除引用的 JAR，请单击编辑。

    6. 编辑 JAR 相关性页包括表 7-5 中所示的信息。

    表 7-5 JAR 相关性详细资料
    属性
    说明
    清单类路径
    JAR 清单文件中的类路径条目。
    名称
    引用的 JAR 资源的名称和路径。
    选项
    • 使用上箭头和下箭头可更改引用的 JAR 的优先顺序。
    • 单击删除图标可删除某个引用的 JAR。

  6. 单击添加 JAR。
  7. 在 JAR 浏览器中，查找或选择引用的 JAR 资源，然后单击提交。

    8. 例如，如果正在编辑 JAR A，并添加了 JAR B 和 C，JAR A 引用 (依赖于) JAR B，并且 JAR A 引用 (依赖于) JAR C。

    使用选项列中的箭头重新排序引用的 JAR 的层次，或者使用删除图标删除它们。

  8. 单击保存可提交当前会话中的更新。
  9. 要结束该会话并将配置部署到运行时，请单击更改中心下的激活。
