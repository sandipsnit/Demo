编辑 MQ 连接

此部分介绍了如何编辑现有的 MQ 连接资源。

  1. 如果尚未执行此操作，请单击“创建”新建一个会话，或单击编辑进入现有会话。请参阅使用更改中心。
  2. 查找 MQ 连接资源，如查找 MQ 连接中所述。
  3. 单击 MQ 连接资源名称。

    4. 查看 MQ 连接 - 配置页显示表 8-3 中所示的信息。

    表 8-3 MQ 连接详细资料 
    属性
    说明
    上次修改者
    创建此 MQ 连接资源或将其导入到配置中的用户。
    上次修改时间
    用户创建此 MQ 连接资源或将其导入到配置中的日期和时间。单击日期和时间链接可查看此资源的更改历史记录。请参阅“查看更改历史记录”页。
    引用
    此 MQ 连接资源引用的对象数。如果存在此类引用，可单击数字链接以查看对象列表。请参阅查看对资源的引用。
    引用项
    引用此 MQ 连接资源的对象数。如果存在此类引用，可单击数字链接以查看对象列表。请参阅查看对资源的引用。
    说明
    此 MQ 连接资源的说明 (如果存在)。

  4. 要对这些字段进行更改，请单击编辑。有关这些字段的说明，请参阅添加 MQ 连接。

    5. 您不能更改资源名字段。

  5. 单击保存可提交当前会话中的更新。
  6. 要结束该会话并将配置部署到运行时，请单击更改中心下的激活。
