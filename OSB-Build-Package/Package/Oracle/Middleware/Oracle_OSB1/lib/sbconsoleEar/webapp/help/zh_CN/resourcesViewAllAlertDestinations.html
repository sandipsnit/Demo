查找预警目标

在预警目标概要页中，可以查看预警目标列表。如果尚未创建预警目标，但希望执行此操作，请在项目浏览器中新建预警目标资源。要了解如何执行此操作，请参阅添加预警目标。

  1. 请执行以下两个操作之一：
    • 选择项目浏览器以便显示“项目视图”页或“项目/文件夹视图”页。然后，在项目和文件夹中导航以查找预警目标。
    • 选择资源浏览器 > 预警目标。预警目标概要页显示了表 5-1 中所示的信息。
  2. 要限制列表中项的数量，可按名称、路径或同时按这两个条件进行筛选。在名称和路径字段中，输入搜索目标的名称和路径，然后单击搜索。

    3. 该路径由项目名和预警目标所在的文件夹名称组成。

    单击查看全部可删除搜索筛选器，并显示所有预警目标。

    表 5-1 预警目标详细资料
    属性
    说明
    名称
    预警目标资源的名称。该名称是一个指向查看预警目标 - 配置页的链接。请参阅编辑预警目标。
    路径
    该路径由项目名和预警目标所在的文件夹名称组成。它是一个指向包含此资源的项目或文件夹的链接。请参阅使用项目和文件夹限定资源名。
    选项
    单击删除图标可删除特定的预警目标。请参阅删除预警目标。
