查找 JAR

  1. 请执行以下两个操作之一：
    • 选择项目浏览器，以便显示“项目视图”页或“项目/文件夹视图”页。然后，在项目和文件夹中导航以查找 JAR。
    • 选择资源浏览器 > JAR。JAR 概要页显示了表 7-1 中所示的信息。有关属性的更详细说明，请参阅编辑 JAR。
  2. 要限制该列表中项的数量或查找特定 JAR，可使用名称和路径筛选器搜索 JAR。在名称和路径字段中，输入搜索目标的名称和路径，然后单击搜索。

    3. 该路径由项目名和 JAR 所在的文件夹名称组成。

    此搜索支持通配符。例如，可以在名称字段中输入“j*”获得以“j”开头的 JAR 列表。

    单击查看全部可删除搜索筛选器，并显示所有 JAR。

    表 7-1 JAR 详细资料
    属性
    说明
    JAR 名称
    分配给此 JAR 的唯一名称。该名称是一个指向查看 JAR 资源 - JAR 配置页的链接。请参阅编辑 JAR。
    路径
    该路径由项目名和 JAR 所在的文件夹名称组成。它是一个指向包含此资源的项目或文件夹的链接。请参阅使用项目和文件夹限定资源名。
    类的数量
    JAR 包含的类的数量。
    大小
    JAR 的大小 (字节)。
    选项
    单击删除图标可删除特定的 JAR。请参阅删除 JAR。
