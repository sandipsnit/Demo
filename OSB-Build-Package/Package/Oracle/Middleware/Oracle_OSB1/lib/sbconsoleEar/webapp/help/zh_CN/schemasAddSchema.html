添加 XML 方案

  1. 如果尚未执行此操作，请单击创建新建一个会话，或单击编辑进入现有会话。请参阅使用更改中心。
  2. 选择项目浏览器，然后选择要在其中添加新的 XML 方案的项目或文件夹。此时会显示项目视图/文件夹视图页。
  3. 从创建资源下拉列表中选择 XML 方案，以便显示创建新的 XML 方案资源页。
  4. 在资源名字段中，输入此 XML 方案的唯一名称。这是必填字段。

    如需命名指导，请参阅资源命名限制。
  5. 在资源说明字段中，输入此 XML 方案的说明。
  6. 在 XML 方案字段中，执行以下操作之一：
    • 输入新 XML 方案的文本。
    • 单击浏览以定位和导入现有 XML 方案。
    • 将文本从现有的 XML 方案复制并粘贴到此字段中。

      • 这是必填字段。

  7. 单击保存。新的 XML 方案会保存在当前会话中。

    8. 单击保存后，如果对于新的 XML 方案，存在任何未解析的引用，则系统会显示这些引用。请参阅查看未解析的 XML 方案引用。

  8. 要结束该会话并将配置部署到运行时，请单击更改中心下的激活。
