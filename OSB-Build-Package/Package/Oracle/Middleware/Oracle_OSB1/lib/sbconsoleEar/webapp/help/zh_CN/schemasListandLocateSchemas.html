查找 XML 方案

  1. 请执行以下两个操作之一：
    • 选择项目浏览器，以便显示“项目视图”页或“项目/文件夹视图”页。然后，在项目和文件夹中导航以查找 XML 方案。
    • 选择资源浏览器 > XML 方案。XML 方案概要页显示了表 4-1 中所示的信息。有关属性的更详细说明，请参阅编辑 XML 方案。
  2. 要查找特定 XML 方案，请执行以下操作：
    • 单击打开图标可显示其他搜索筛选器。如果指定多个搜索条件，则仅返回与所有条件均匹配的结果。
    • 要限制列表中项的数量，可按名称、路径和名称空间进行筛选。在名称、路径和名称空间字段中，输入搜索目标的名称、路径和名称空间，然后单击搜索。

      • 该路径由项目名和 XML 方案所在的文件夹名称组成。

    • 单击查看全部可删除搜索筛选器，并显示所有 XML 方案。

      • 表 4-1 XML 方案信息
      属性
      说明
      XML 方案名称
      分配给此 XML 方案的唯一名称。此名称是一个指向 XML 方案详细资料页的链接。请参阅编辑 XML 方案。
      路径
      该路径由项目名和 XML 方案所在的文件夹名称组成。它是一个指向包含此资源的项目或文件夹的链接。请参阅使用项目和文件夹限定资源名。
      XML 方案名称空间
      用来限定 XML 方案中所包括的任何定义的名称空间。
      选项
      单击删除图标可删除特定的 XML 方案。请参阅删除 XML 方案。
