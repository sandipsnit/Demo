解析未解析的 XML 方案引用

在编辑 XML 方案资源的引用页中，可通过配置 XML 方案引用的映射 (如 XML 方案导入) 来解析未解析的 XML 方案引用。

  1. 如果尚未执行此操作，请单击创建新建一个会话，或单击编辑进入现有会话。请参阅使用更改中心。
  2. 在 XML 方案概要页的 XML 方案名称列中，单击相应的 XML 方案名称，以查看未解析的引用的详细资料。此时会显示查看 XML 方案详细资料页。
  3. 单击编辑引用，以便显示编辑 XML 方案资源的引用页。
  4. 在资源类型字段中，选择 XML 方案。
  5. 单击浏览。此时会显示 XML 方案浏览器。
  6. 在 XML 方案浏览器中，选择一个 XML 方案，然后从定义窗格中选择一个定义。
  7. 单击提交。您选择的 XML 方案将显示在资源名字段中。
  8. 单击保存以解析引用。
  9. 在 XML 方案详细资料页中，单击确定。XML 方案的状态显示为有效。
