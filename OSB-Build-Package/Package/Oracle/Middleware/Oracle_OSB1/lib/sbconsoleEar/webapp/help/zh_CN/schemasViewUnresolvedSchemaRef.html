查看未解析的 XML 方案引用

对于每个未解析的 XML 方案引用，此页显示了表 4-5 中所示的信息。

表 4-5 未解析的 XML 方案详细资料
属性
说明
方案名称
分配给此 XML 方案唯一名称。单击特定 XML 方案的名称可查看该 XML 方案的详细资料。请参阅编辑 XML 方案。
名称空间
用来限定 XML 方案中所包括的任何定义的名称空间。

要查找特定引用，请执行以下操作：

  • 对列表进行重新排序。单击带下划线的列名。升序箭头和降序箭头指示排序顺序。单击列名可更改排序顺序。
  • 滚动查看各页。使用表上面的页控件，或表下面的页控件。通过选择页码转到某一页，或通过使用箭头按钮转到下一页、上一页、第一页或最后一页。
