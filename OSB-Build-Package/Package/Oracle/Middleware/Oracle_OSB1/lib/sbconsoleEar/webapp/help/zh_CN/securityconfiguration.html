安全配置

使用安全配置模块可创建和修改在 Oracle Service Bus 入站安全和管理安全中使用的安全数据。

入站传输级安全和消息级安全使用用户、组和角色数据来对入站客户机请求进行验证。它应用访问控制策略以确定哪些经过验证的用户已被授权使用代理服务和业务服务。

管理安全使用用户、组和角色数据来确定哪些经过验证的用户已被授权创建或修改 Oracle Service Bus 配置数据或监视 Oracle Service Bus 性能。有关详细信息，请参阅了解用户、组、安全角色和策略。另请参阅 Oracle Fusion Middleware Developer's Guide for Oracle Service Bus 中的“Configuring Administrative Security”，网址为：http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539。

注： 在导出配置时，不能导出用户、组或角色，因为这些对象位于安全提供程序存储中。必须在导入已导出的配置或使用 WebLogic 服务器工具 (如果可用) 导出和导入这些对象时再次创建这些对象。
