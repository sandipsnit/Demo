添加组

  1. 以具有 WebLogic Server 管理权限的用户身份登录到 Oracle Service Bus 管理控制台。只有拥有管理员角色的用户才能修改安全配置数据。请参阅 Oracle Fusion Middleware Developer's Guide for Oracle Service Bus 中的“Configuring Administrative Security”，网址为：http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539。
  2. 选择安全配置 > 组，以便显示组概要页。
  3. 单击添加新的。

    4. 无论在会话内还是会话外，都可添加组。

  4. 在组名字段中，输入唯一的名称。请注意，不能输入空格或特殊字符。这是必填字段。
  5. 在验证提供程序字段中，选择验证提供程序。
  6. 在组成员资格字段中，选择此组所属的组。
    1. 从可用组字段中选择一个组。
    2. 单击箭头将组移入到当前组字段中。
  7. 单击保存创建组。

    8. Oracle Service Bus 管理控制台会保存该组，该组会立即可用于所有会话。如果在某个会话中时添加了组，然后放弃了该会话，则 Oracle Service Bus 管理控制台不会删除该新组。
