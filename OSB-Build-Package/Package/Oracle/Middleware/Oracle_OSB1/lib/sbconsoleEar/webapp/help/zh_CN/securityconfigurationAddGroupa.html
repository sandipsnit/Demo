编辑组

使用“查看组详细信息”页可查看和更改特定组的详细信息。

  1. 以具有 WebLogic Server 管理权限的用户身份登录到 Oracle Service Bus 管理控制台。只有拥有管理员角色的用户才能修改安全配置数据。请参阅 Oracle Fusion Middleware Developer's Guide for Oracle Service Bus 中的“Configuring Administrative Security”，网址为：http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539。
  2. 查找组。请参阅查找组。
  3. 单击组名。查看组详细资料页显示表 24-4 中所示的信息。

    4. 表 24-4 组详细资料
    属性
    说明
    组名
    此组的名称。
    验证提供程序
    包含此组定义的验证提供程序。
    组
    此组所属的组。

  4. 要编辑组详细资料，请单击重新配置以便显示编辑组详细资料页。

    5. 无论在会话内还是会话外，都可编辑组详细信息。

  5. 对组成员资格字段进行适当的更改。有关该字段的说明，请参阅添加组。

    6. 不能更改组名字段。

  6. 单击保存更改对组进行更新。

    7. Oracle Service Bus 管理控制台会更新该组的详细信息，此更新会立即可用于所有会话。如果在某个会话中时更新了组，然后放弃了该会话，则 Oracle Service Bus 管理控制台不会删除这些更新。
