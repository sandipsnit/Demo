编辑组

使用“查看组详细信息”页可查看和更改特定组的详细信息。

  1. 以具有 WebLogic Server 管理权限的用户身份登录到 Oracle Service Bus 控制台。只有 Admin 角色中的用户才能修改安全配置数据。请参阅“Oracle Service Bus 安全指南”中的配置管理安全。
  2. 查找组。请参阅查找组。
  3. 单击组名。查看组详细资料页显示表 23-4 中所示的信息。

    4. 表 23-4 组详细资料
    属性
    描述
    组名
    此组的名称。
    验证提供程序
    包含此组定义的验证提供程序。
    组
    此组所属的组。

  4. 要编辑组详细信息，请单击“重新配置”以便显示“编辑组详细信息”页。

    5. 无论在会话内还是会话外，都可编辑组详细信息。

  5. 对“组成员资格”字段进行适当的更改。有关该字段的描述，请参阅添加组。

    6. 不能更改“组名”字段。

  6. 单击“保存更改”对组进行更新。

    7. Oracle Service Bus 控制台会更新该组的详细信息，更新会立即可用于所有会话。如果在某个会话中时更新了该组，然后放弃了该会话，则 Oracle Service Bus 控制台不会删除这些更新。
