添加角色

  1. 以具有 WebLogic Server 管理权限的用户身份登录到 Oracle Service Bus 管理控制台。只有拥有管理员角色的用户才能修改安全配置数据。请参阅 Oracle Fusion Middleware Developer's Guide for Oracle Service Bus 中的“Configuring Administrative Security”，网址为：http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539。
  2. 选择安全配置 > 角色，以便显示全局角色页。
  3. 单击新建。

    4. 无论在会话内还是会话外，都可添加角色。

  4. 在角色名字段中，输入唯一的名称。这是必填字段。

    5. 请确保安全角色名中没有空格或“<”、“>”字符。安全角色名区分大小写。Oracle 的约定是，所有安全角色名都是独一无二的。

  5. 要创建角色，请单击确定。

    6. Oracle Service Bus 管理控制台会保存该角色，该角色会立即可用于所有会话。如果在某个会话中时添加了角色，然后放弃了该会话，则 Oracle Service Bus 管理控制台不会删除该新角色。

    如果单击确定创建角色，则下一步是定义应用角色所依据的条件。

  6. 继续定义角色条件。
