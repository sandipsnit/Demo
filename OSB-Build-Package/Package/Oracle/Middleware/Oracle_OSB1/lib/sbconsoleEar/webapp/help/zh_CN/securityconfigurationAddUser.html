添加用户

  1. 以具有 WebLogic Server 管理权限的用户身份登录到 Oracle Service Bus 管理控制台。只有拥有管理员角色的用户才能修改安全配置数据。请参阅 Oracle Fusion Middleware Developer's Guide for Oracle Service Bus 中的“Configuring Administrative Security”，网址为：http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539。
  2. 选择安全配置以便显示用户概要页。
  3. 单击添加新的，以便显示创建新用户 - 一般配置页。

    4. 无论在会话内还是会话外，都可添加用户。

  4. 在用户名字段中，输入唯一的名称。这是必填字段。
  5. 在口令字段中，输入口令。这是必填字段。
    6. 注： 验证提供程序可以强制使用最小口令长度。对于在 WebLogic 验证提供程序中定义的用户，最小默认口令长度为 8 个字符。可使用 WebLogic 服务器管理控制台定制此设置。(WebLogic 验证提供程序是在默认安全领域中使用名称 DefaultAuthenticator 进行配置的。)
  6. 在确认口令字段中，输入您为口令字段输入的同一个口令。这是必填字段。
  7. 在验证提供程序字段中，选择此用户的验证提供程序。

    8. 如果在安全领域中配置了多个验证提供程序，则这些提供程序也将出现在列表中。选择用来存储新用户的信息的验证提供程序数据库。

  8. 在组成员资格字段中，为此用户选择一个组。
    1. 从可用组字段中选择一个组。
    2. 单击箭头将组移入到当前组字段中。
  9. 单击保存创建用户。

    10. Oracle Service Bus 管理控制台会保存该用户，该用户会立即可用于所有会话。如果在某个会话中时添加了用户，然后放弃了该会话，则 Oracle Service Bus 管理控制台不会删除该新用户。
