添加策略条件

  1. 通过访问策略编辑器可访问一个访问控制策略。请参阅编辑传输级访问策略或编辑消息级访问策略。
  2. 在策略编辑器的策略条件下，单击添加条件。

    3. 此时会显示以下提示：

    选择您希望用作新条件的谓词

  3. 从列表中选择谓词。
  4. 单击下一步。根据您为条件谓词所做的选择，执行以下步骤之一，如表 24-12 中所示。

      2. 如果尚未创建在此字段中输入的角色，则可以在完成创建访问控制策略后创建该角色。请参阅添加角色。如果未创建此角色，则不会向任何角色授予访问权。
