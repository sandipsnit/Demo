编辑传输级访问策略

使用查看策略详细资料页可编辑代理服务的传输级访问控制策略。此页显示表 24-10 中所示的信息。

表 24-10 策略详细资料
属性
说明
代理服务名称
显示您在安全页上为其选择传输访问控制的代理服务名称。
提供程序
显示为该安全领域配置的授权提供程序。
策略条件
显示决定代理服务将为哪些用户处理请求的条件。

  1. 以具有 WebLogic Server 管理权限的用户身份登录到 Oracle Service Bus 管理控制台。只有拥有管理员角色的用户才能修改安全配置数据。请参阅 Oracle Fusion Middleware Developer's Guide for Oracle Service Bus 中的“Configuring Administrative Security”，网址为：http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539。
  2. 查找访问策略。请参阅查找访问控制策略。
  3. 在安全页的传输访问控制列中，单击代理服务的名称。此时会显示策略编辑器。

    4. 可以在会话内编辑访问控制策略。

  4. 在授权提供程序字段中，选择一个授权提供程序。Oracle 建议您选择 XACMLAuthorizer。
    5. 注： Oracle Service Bus 已放弃对 WebLogic 默认授权提供程序的支持。Oracle 建议您改为使用 WebLogic XACML 授权提供程序。
  5. 添加策略条件。请参阅添加策略条件。
  6. 在策略条件部分中输入完条件后，请单击保存。
