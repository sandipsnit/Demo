查找组

  1. 选择安全配置 > 组。组概要页显示表 24-3 中所示的信息。有关属性的更详细说明，请参阅编辑组。
  2. 要限制列表中项的数量或查找特定组，可按组名进行筛选。在名称字段中输入该组名的一部分或全名，然后单击搜索。

    3. 可以使用星号 (*) 通配符。(不支持其他通配符。)

    单击查看全部可删除搜索筛选器，并显示所有组。

    表 24-3 组信息 
    属性
    说明
    组名
    组的名称。此名称是一个指向查看组详细资料页的链接。请参阅编辑组。
    组成员资格
    此组所属的组。此名称是一个指向查看组详细资料页的链接。请参阅编辑组。
    验证提供程序
    包含此组定义的验证提供程序。
    删除
    单击删除图标可删除特定组。请参阅删除组。
