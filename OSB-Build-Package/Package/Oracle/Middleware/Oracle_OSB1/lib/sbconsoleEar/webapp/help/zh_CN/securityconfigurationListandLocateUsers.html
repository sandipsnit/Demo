查找用户

  1. 选择安全配置 > 用户。用户概要页显示表 24-1 中所示的信息。有关属性的更详细说明，请参阅编辑用户。
  2. 要限制列表中项的数量或查找特定用户，可按用户名进行筛选。在名称字段中输入该用户名的一部分或全名，然后单击搜索。

    3. 可以使用星号 (*) 通配符。(不支持其他通配符。)

    单击查看全部可删除搜索筛选器，并显示所有用户。

    表 24-1 用户信息
    属性
    说明
    用户名
    分配给用户的名称。此名称是一个指向查看用户详细资料页的链接。请参阅编辑用户。
    组成员资格
    此用户所属的组的名称。此名称是一个指向查看组详细资料页的链接。请参阅编辑组。
    验证提供程序
    此用户的验证提供程序。
    选项
    单击删除图标可删除特定用户。请参阅删除用户。
