编辑角色

使用查看角色详细资料页可查看和更改特定角色的详细资料。

  1. 以具有 WebLogic Server 管理权限的用户身份登录到 Oracle Service Bus 管理控制台。只有拥有管理员角色的用户才能修改安全配置数据。请参阅 Oracle Fusion Middleware Developer's Guide for Oracle Service Bus 中的“Configuring Administrative Security”，网址为：http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539。
  2. 查找角色。请参阅查找角色。
  3. 单击角色名。查看角色详细资料页显示表 24-8 中所示的信息。

    4. 表 24-8 角色详细资料
    属性
    说明
    名称
    角色的名称。
    角色条件
    可确定此角色中成员资格的条件。

    无论在会话内还是会话外，都可更改角色详细资料。

  4. 执行下列步骤之一，如表 24-9 中所述。

    5. 表 24-9 角色选项
    操作
    步骤
    更改选定表达式的顺序。
    单击上移和下移。
    合并或取消合并角色条件，并切换表达式之间突出显示的 and 和 or 语句。
    单击组合和取消组合。
    构造否定条件；例如“NOT Group Operators”会从角色中排除 Operator 组。
    单击排除。
    删除选定表达式。
    单击删除。

  5. 单击保存。

    6. Oracle Service Bus 管理控制台会更新该角色，此更新会立即可用于所有会话。如果在某个会话中时更新了角色，然后放弃了该会话，则 Oracle Service Bus 管理控制台不会删除这些更新。
