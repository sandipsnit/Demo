编辑用户

使用查看用户详细资料页可查看和更改特定用户的详细资料。

  1. 以具有 WebLogic Server 管理权限的用户身份登录到 Oracle Service Bus 管理控制台。只有拥有管理员角色的用户才能修改安全配置数据。请参阅 Oracle Fusion Middleware Developer's Guide for Oracle Service Bus 中的“Configuring Administrative Security”，网址为：http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539。
  2. 查找用户。请参阅查找用户。
  3. 单击用户名。查看用户详细资料页显示表 24-2 中所示的信息。

    4. 表 24-2 用户详细资料
    属性
    说明
    用户名
    此用户的名称
    验证提供程序
    包含此用户定义的验证提供程序。
    组成员资格
    此用户所属的组的名称。

  4. 要编辑用户详细资料，请单击重新配置以便显示编辑用户详细资料页。

    5. 无论在会话内还是会话外，都可编辑用户详细资料。

  5. 对新口令、确认口令和组成员资格字段进行适当的更改。有关这些字段的说明，请参阅添加用户。

    6. 不能更改用户名字段。

  6. 单击保存更改对用户进行更新。

    7. Oracle Service Bus 管理控制台会更新该用户的详细信息，此更新会立即可用于所有会话。如果在某个会话中时更新了用户，然后放弃了该会话，则 Oracle Service Bus 管理控制台不会删除这些更新。
