删除服务帐户

删除服务帐户时，也会删除服务帐户包含的用户名、口令或本地用户到远程用户的映射数据。

  1. 如果尚未执行此操作，请单击创建新建一个会话，或单击编辑进入现有会话。请参阅使用更改中心。
  2. 如果任何业务服务或代理服务已配置为使用该服务帐户，请从该业务服务或代理服务中删除该服务帐户。

    3. 请参阅编辑业务服务配置或编辑代理服务配置。

  3. 选择资源浏览器 > 服务帐户以便显示服务帐户概要页。
  4. 在要删除的服务帐户的选项字段中单击删除图标。

    5. 此服务帐户将会从当前会话中删除。如果业务服务或代理服务已配置为使用服务帐户，则删除警告图标将表明您可在确认警告后删除服务帐户。这可能会由于存在服务对已删除服务帐户的未解析引用而导致冲突。

  5. 要结束该会话并将配置部署到运行时，请单击更改中心下的激活。
