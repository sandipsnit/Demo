查找服务帐户

  1. 请执行以下两个操作之一：
    • 选择项目浏览器以便显示“项目视图”页或“项目/文件夹视图”页。然后，在项目和文件夹中导航以查找服务帐户。
    • 选择资源浏览器 > 服务帐户。服务帐户概要显示表 16-1 中所示的信息。
  2. 要搜索服务帐户，请在名称字段中输入部分或完整的帐户名。您还可以在路径字段中输入部分或完整的帐户项目名和文件夹。单击搜索。

    3. 单击查看全部可删除搜索筛选器，并显示所有服务帐户。

    表 16-1 服务帐户信息
    属性
    说明
    服务帐户名
    此服务帐户的唯一名称。单击该名称可查看查看服务帐户详细资料页。请参阅编辑服务帐户。
    路径
    服务帐户所在的项目名称和文件夹名称。单击该名称可查看包含此资源的项目或文件夹。请参阅使用项目和文件夹限定资源名。
    选项
    包含删除图标。如果业务服务或代理服务已配置为使用服务帐户，则删除警告图标将表明您可在确认警告后删除服务帐户。这可能会由于存在服务对已删除服务帐户的未解析引用而导致冲突。请参阅删除服务帐户。
