拆分-联接

通过拆分-联接，您可以并行 (还可以按顺序) 发送消息调用，并且可以聚集响应。要了解有关拆分-联接的详细信息，请参阅 Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus 中的“Improving Service Performance with Split-Join”，网址为：http://www.oracle.com/pls/as111170/lookup?id=OSBAG291。另请参阅 Oracle Fusion Middleware Developer's Guide for Oracle Service Bus 中的“Working with Split-Join”(网址为：http://www.oracle.com/pls/as111170/lookup?id=OSBDV193) 和“Split-Join User Interface Reference”(网址为：http://www.oracle.com/pls/as111170/lookup?id=OSBDV573)。

可通过流传输类型业务服务访问拆分-联接。有关详细信息，请参阅“流传输配置”页。
