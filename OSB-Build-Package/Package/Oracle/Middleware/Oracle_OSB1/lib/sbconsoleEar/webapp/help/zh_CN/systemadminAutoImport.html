使用自动导入状态

可以使用自动导入状态页将服务更改与注册表中的服务进行同步。在对注册表中的服务进行任何更改时，Oracle Service Bus 都会在列出所有非同步服务的自动导入状态页上提供更改通知。然后您可以将 Oracle Service Bus 管理控制台中的服务与 UDDI 注册表中相应的服务进行同步。

配置注册表时，可以选择启用自动导入选项，这会使导入的服务自动与 UDDI 注册表进行同步。在选定此选项的情况下导入的任何服务都将自动与 UDDI 注册表保持同步。请参阅配置 UDDI 注册表。

如果在自动同步过程中遇到任何故障，则会在自动导入状态页上报告此故障，您可在该页上手动对其进行更新。

  1. 如果尚未执行此操作，请单击创建新建一个会话，或单击编辑进入现有会话。请参阅使用更改中心。
  2. 选择系统管理 > 自动导入状态。
  3. 从自动导入状态页的查看下拉列表中选择要显示的服务类型。
  4. 选中您想要将其与注册表中相应的服务同步的代理服务旁边的复选框。
  5. 单击同步。
  6. 在下一页上，单击完成。
