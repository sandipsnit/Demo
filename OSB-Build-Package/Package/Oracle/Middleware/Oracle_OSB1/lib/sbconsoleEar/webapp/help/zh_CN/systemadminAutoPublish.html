使用自动发布

您可以使用自动发布将代理服务自动发布到注册表。要完成此操作，必须在创建/编辑代理服务 - 一般配置页上启用发布到注册表选项。请参阅“一般配置”页。请注意，仅当默认注册表存在时，才显示发布到注册表选项。请参阅设置默认 UDDI 配置。

  1. 如果尚未执行此操作，请单击创建新建一个会话，或单击编辑进入现有会话。请参阅使用更改中心。
  2. 选择项目浏览器。
  3. 展开项目文件夹并选择要编辑的代理服务。
  4. 在查看代理服务页的底部，单击编辑。
  5. 选择发布到注册表，然后单击保存可提交当前会话中的更新。
  6. 要结束该会话并将配置部署到运行时，请单击更改中心下的激活。

    7. 将服务配置为发布到默认注册表。

注释：
注释：
  • 当成功地将服务发布到注册表后，用于选择要自动发布到注册表的服务的选项将不再可用。
注释：
  • 如果未成功发布服务，则可以进行重新发布。要重新发布服务，请在自动发布状态页上选择此服务并单击发布。
注释：
  • 如果已启用发布到注册表选项，则在创建或编辑代理服务后会立即发布这些代理服务，并会激活会话。可将发布到注册表选项用于所有代理服务，那些使用本地传输的代理服务除外。
