创建定制设置文件

创建定制设置文件页为生成您选择的一组资源或项目的定制设置文件提供了一种便捷的方式。之后可通过指定针对某个环境的实际值，从此文件开始进行所需更改。

可以使用定制设置文件更改环境变量以及资源中的引用。定制设置文件可以包括选定资源中找到的所有环境值 (包括 EnvValueTypes 类中定义的复杂环境值类型) 的定制设置。此外，它还包括一个引用定制设置类型，用于更改带有相关性的资源内的资源引用。

描述定制设置类型的定制设置方案 (Customization.xsd) 位于 Oracle Service Bus 安装的以下位置：

OSB_ORACLE_HOME\modules\com.bea.common.configfwk_version.jar。

  1. 选择系统管理 > 创建定制设置文件。

    2. 创建定制设置文件页显示了配置中对象的列表。会显示每个对象的名称和类型。

  2. 选择要包含在定制设置文件中的项目或资源。
    1. 展开项目文件夹。将会显示该项目中包含的每个资源的名称和类型。
    2. 选中要包含在定制设置文件中的项目或资源所对应的复选框。
    3. 清除您不想添加到定制设置文件中的项目或资源所对应的复选框。
  3. 单击创建文件以创建定制设置 XML 文件。
  4. 在文件下载对话框中，单击打开打开该定制设置文件，或者单击保存将此 XML 文件保存到桌面。
