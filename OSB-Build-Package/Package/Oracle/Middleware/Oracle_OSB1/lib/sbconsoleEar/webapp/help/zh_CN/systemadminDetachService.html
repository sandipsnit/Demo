分离服务

如果您不想使 Oracle Service Bus 管理控制台中的服务与注册表中相应的服务同步，您可以通过以下方式之一将其从注册表分离，从而可避免进行同步：

  • 通过编辑服务的配置分离服务
  • 从“自动导入状态”页分离服务
通过编辑服务的配置分离服务
  1. 如果尚未执行此操作，请单击创建新建一个会话，或单击编辑进入现有会话。请参阅使用更改中心。
  2. 选择项目浏览器。
  3. 展开项目文件夹，选择您要将其与注册表中相应的服务分离的业务服务。
  4. 在查看业务服务页的底部，单击编辑。
  5. 启用从注册表分离选项。该业务服务将从 UDDI 注册表中的相应服务分离。
从“自动导入状态”页分离服务

只有在获得有关自动导入状态页上更改的通知后，才可以使用此“分离”选项。

  1. 在系统管理 > 自动导入状态页上，选中希望从 UDDI 注册表中的相应服务分离的业务服务旁边的复选框。

    2. 只有当注册表中的原始服务发生更改时才会在该页上显示服务。并非所有服务都会显示在该页上。

  2. 单击分离。
