编辑 JNDI 提供程序

使用此页可查看 JNDI 提供程序的详细资料，并可编辑配置 (如果需要)。此外，还可以重置 JNDI 提供程序高速缓存并测试与 JNDI 提供程序的连接。

  1. 如果尚未执行此操作，请单击创建新建一个会话，或单击编辑进入现有会话。请参阅使用更改中心。
  2. 查找 JNDI 提供程序。请参阅查看 JNDI 提供程序。
  3. 单击 JNDI 提供程序名。

    4. 查看 JNDI 提供程序 - 配置页显示表 30-2 中所示的 JNDI 提供程序信息。

    表 30-2 JNDI 提供程序信息
    属性
    说明
    上次修改者
    创建或修改此 JNDI 提供程序资源，或将其导入到配置中的用户。
    上次修改时间
    用户创建或修改 JNDI 提供程序资源，或将其导入到配置中的日期和时间。单击日期和时间链接可查看此资源的更改历史记录。请参阅“查看更改历史记录”页。
    引用
    此 JNDI 提供程序引用的对象数。如果存在此类引用，可单击数字链接以查看引用列表。请参阅查看对资源的引用。
    引用项
    引用此 JNDI 提供程序的对象数。如果存在此类引用，可单击数字链接以查看对象列表。请参阅查看对资源的引用。
    说明
    此 JNDI 提供程序的说明。
    JNDI 高速缓存
    “已启用”或“已禁用”。启用后，会在本地高速缓存 JNDI 上下文和 JNDI 对象，这样可在执行对象查找时提升性能。Oracle 建议保持 JNDI 高速缓存处于启用状态。
    提供程序 URL
    JNDI 提供程序的 URL。
    JNDI 请求超时
    JNDI 请求超时 (毫秒)。默认值零 (0) 表示没有超时。
    用户名和口令
    用于访问安全的 JNDI 提供程序并且能够更改口令的用户名。

  4. 单击编辑以便显示编辑 JNDI 提供程序 - [provider-name] 页。

    5. 名称字段不可编辑。

  5. 在说明字段中，可以编辑此 JNDI 提供程序的简要说明。
  6. 在提供程序 URL 字段中，可以编辑此 JNDI 提供程序的 URL。
  7. 可以编辑用户名和口令字段。

    8. 如果 JNDI 提供程序是使用用户名和口令配置的，则用户名将不可编辑。您可以编辑新口令和确认口令字段。如果 JNDI 不是使用用户名和口令配置的，则可以编辑用户名字段，并且您可以输入新的用户名。

  8. 单击保存可提交当前会话中的更新。
  9. 要结束该会话并将配置部署到运行时，请单击更改中心下的激活。
