编辑代理服务器

使用此页可查看代理服务器的详细资料，并可编辑配置 (如果需要)。

  1. 如果尚未执行此操作，请单击创建新建一个会话，或单击编辑进入现有会话。请参阅使用更改中心。
  2. 查找代理服务器。请参阅查找代理服务器。
  3. 单击代理服务器名称。

    4. 查看代理服务器 - 配置页显示表 30-6 中所示的信息。

    表 30-6 代理服务器信息
    属性
    说明
    上次修改者
    创建或修改此代理服务器资源的用户。
    上次修改时间
    用户创建或修改代理服务器资源的日期和时间。单击日期和时间链接可查看此资源的更改历史记录。请参阅“查看更改历史记录”页。
    引用
    此代理服务器引用的对象数。如果存在此类引用，可单击数字链接以查看引用列表。请参阅查看对资源的引用。
    引用项
    引用此代理服务器的对象数。如果存在此类引用，可单击数字链接以查看对象列表。请参阅查看对资源的引用。
    说明
    此代理服务器的说明 (如果已提供)。
    主机端口参数
    代理服务器资源中每个代理服务器的相关详细资料，包括以下信息：
    • 服务器主机：代理服务器的主机名或 IP 地址。

      • Oracle Service Bus 代理服务器的服务器主机名必须与实际代理服务器的服务器主机名相同。

    • 明文端口：代理服务器明文端口号。
    • SSL 端口：代理服务器 SSL 端口号。
    用户名
    用于代理验证的用户名。

  4. 单击编辑以便显示编辑代理服务器 - [server-name] 页。

    5. 名称字段不可编辑。

  5. 在说明字段中，可以编辑代理服务器的简要说明。
  6. 在主机端口参数部分中，可以通过输入以下信息添加新代理服务器：
    1. 在服务器主机字段中，输入代理服务器的主机名或 IP 地址。这是必填字段。

      2. Oracle Service Bus 代理服务器的服务器主机名必须与实际代理服务器的服务器主机名相同。

    2. 在明文端口字段中，输入代理服务器明文端口号。
    3. 在 SSL 端口字段中，输入代理服务器 SSL 端口号。必须输入明文或 SSL 端口号。
    4. 单击添加。

      5. 可以为每个代理服务器资源配置多个代理服务器。这可使 Oracle Service Bus 执行负载平衡并为代理服务器资源提供容错功能。

      此外，还可以通过执行以下操作编辑或删除现有代理服务器配置：

    • 在要编辑的代理服务器的选项字段中单击编辑图标。编辑服务器主机字段、明文端口字段和 SSL 端口字段，然后单击更新。
    • 单击要删除的代理服务器的选项字段中的删除图标。
  7. 可以编辑用于代理验证的用户名和口令字段。
  8. 单击保存可提交当前会话中的更新。
  9. 要结束该会话并将配置部署到运行时，请单击更改中心下的激活。
