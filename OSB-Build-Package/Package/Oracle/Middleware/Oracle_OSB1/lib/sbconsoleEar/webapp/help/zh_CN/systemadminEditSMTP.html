编辑 SMTP 服务器

使用此页可查看 SMTP 服务器的详细资料，并可编辑配置 (如果需要)。

  1. 如果尚未执行此操作，请单击创建新建一个会话，或单击编辑进入现有会话。请参阅使用更改中心。
  2. 查找 SMTP 服务器。请参阅查找 SMTP 服务器。
  3. 单击 SMTP 服务器名称。

    4. 查看 SMNP 服务器 - 配置页显示表 30-4 中所示的信息。

    表 30-4 SMTP 服务器信息
    属性
    说明
    上次修改者
    创建或修改此 SMTP 服务器资源的用户。
    上次修改时间
    用户创建或修改 SMTP 服务器资源的日期和时间。单击日期和时间链接可查看此资源的更改历史记录。请参阅“查看更改历史记录”页。
    引用
    此 SMTP 服务器引用的对象数。如果存在此类引用，可单击数字链接以查看引用列表。请参阅查看对资源的引用。
    引用项
    引用此 SMTP 服务器的对象数。如果存在此类引用，可单击数字链接以查看对象列表。请参阅查看对资源的引用。
    说明
    此 SMTP 服务器的说明 (如果已提供)。
    服务器 URL
    指向 SMTP 服务器位置的 URL。
    端口号
    此 SMTP 服务器的端口号。默认情况下是端口 25。
    用户名
    用于访问安全的 SMTP 服务器的用户名。

  4. 单击编辑以便显示编辑 SMTP 服务器 - [server-name] 页。

    5. 名称字段不可编辑。

  5. 在说明字段中，可以编辑 SMTP 服务器的简要说明。
  6. 在服务器 URL 字段中，您可以编辑此 SMTP 服务器的 URL。
  7. 在端口号字段中，可以编辑端口号。
  8. 可以编辑用户名和口令字段。
  9. 单击保存可提交当前会话中的更新。
  10. 要结束该会话并将配置部署到运行时，请单击更改中心下的激活。

如果重命名 SMTP 资源，则新名称将自动在引用端点 URI 中的 SMTP 资源的所有电子邮件业务服务中更新。
