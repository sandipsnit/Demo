导出资源

使用导出资源页可导出项目或导出资源。如果选择导出完整的项目，则可以选择一个或多个项目进行导出。您还可以展开项目来查看文件夹和资源，但是只能在项目级别上进行选择。此外，不能选择包括相关性。

无论您在会话之中或会话之外工作，都可以导出项目和资源。如果您在会话之中执行导出，则资源是会话资源，而且配置可能是不完整的或具有冲突。如果在会话之外执行导出，则资源是激活的核心资源。

Oracle Service Bus 不能导出在 Oracle Service Bus 管理控制台的“安全配置”模块中创建的用户、组或角色。它也不能导出您在 WebLogic Server 管理控制台中创建的身份证明映射或其他安全提供程序数据。但是，可以使用 WebLogic Server 管理控制台来导出这些数据。请参阅 Oracle Fusion Middleware Securing Oracle WebLogic Server 中的“Migrating Security Data”，网址为：http://www.oracle.com/pls/as111170/lookup?id=SECMG319。

  1. 选择系统管理 > 导出资源。

    2. 导出资源页可显示配置中对象的列表。同时还可显示每个对象的名称、类型、最后修改时间和引用。

    对于服务而言，最后修改的时间戳反映了对服务定义的修改，以及对服务传输或管道配置的修改。但是，它不反映对针对此服务定义的预警做出的更改。

  2. 选择导出项目或导出资源。
    3. 注： 当导入完整的项目 JAR 文件时，导出项目可能会删除资源。有关详细信息，请参阅导入资源。
  3. 选择要导出的项目或资源。如果选择导出项目，则不能针对单独的资源取消选中复选框。
    1. 展开项目文件夹。将会显示该项目中包含的每个资源的名称、类型、最后修改时间以及引用。
    2. 选中要导出的项目或资源所对应的复选框。
    3. 清除不想导出的项目或资源所对应的复选框。
  4. 如果选择了导出资源，请选中或清除包括相关性复选框。使用此选项可导出此资源引用的任何其他资源。
  5. 单击导出以创建一个 JAR 文件并将其导出。

完成之后

如果适用，请继续指定导出安全设置。
