指定导出安全设置

如果您的导出内容包括服务帐户或服务密钥提供程序，则此时会显示导出资源 - 安全设置页。

对于每个服务帐户，Oracle Service Bus 会导出用户名和口令或本地用户到远程用户的映射 (具体取决于服务帐户中存储的数据)。对于每个服务密钥提供程序，Oracle Service Bus 会从 PKI 身份证明映射提供程序中导出别名到密钥对的绑定；它不会从密钥库中导出私有密钥、证书或其他数据。必须使用密钥库供应商提供的工具导出密钥库数据。

如果希望 Oracle Service Bus 在将服务帐户、服务密钥提供程序、UDDI 注册表、JNDI 提供程序或 SMTP 提供程序资源中的用户名和口令数据导出到 JAR 文件中之前对这些数据进行加密，请执行以下操作：

  1. 选中保护敏感数据复选框。
  2. 输入并确认口令。

    3. 当您或其他人试图导入此 JAR 时，除非指定此口令，否则 Oracle Service Bus 将不会导入具有已加密的用户名和口令数据的资源。可以导入 JAR 中的所有其他非加密资源而无须指定口令。

  3. 单击完成导出按钮。
  4. 单击“导出”或“完成导出”后，会显示正在处理配置数据弹出式窗口。请等到此窗口关闭后，再在 Oracle Service Bus 管理控制台上执行任何其他操作。在显示文件下载对话框后，此窗口会自动关闭。
  5. 在文件下载对话框中，单击打开打开导出的 JAR 文件，或者单击保存将此 JAR 文件保存到桌面。
