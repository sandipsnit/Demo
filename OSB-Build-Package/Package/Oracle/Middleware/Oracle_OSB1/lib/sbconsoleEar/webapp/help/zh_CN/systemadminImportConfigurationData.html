导入资源

使用导入资源页可导入以前从其他 Oracle Service Bus 域中导出的完整项目 JAR 文件或资源 JAR 文件。也可以选择仅导入已导出数据的一部分。如果导入系统中已存在资源，则会更新此资源。否则，会创建一个资源。只有在将导入的 JAR 是完整的项目 JAR，并且导入系统中存在导入的 JAR 文件在同一项目中没有提供的资源时，才会安排资源的删除。它将不会删除位于其他项目的资源。

您可以在资源级别导入资源，即使这些资源是在项目级别导出的，也是如此。例如，即使系统资源已导出到完整的项目 JAR 文件中，也可以在导入时取消选择它们。

导出配置时，无法导出用户、组、角色或证书。因此，必须在导入已导出的配置后再次创建这些对象。

注： 有关导入不属于 Oracle Service Bus 配置的资源的信息，请参阅从 Zip 文件加载资源和从 URL 加载资源。
  1. 如果尚未执行此操作，请单击创建新建一个会话，或单击编辑进入现有会话。请参阅使用更改中心。
  2. 选择系统管理 > 导入资源。
  3. 选择一个已导出的项目或资源 JAR 文件。
    1. 在文件名字段中，单击浏览以查找存储此 JAR 文件的目录。
    2. 选择此 JAR 文件，然后单击打开。
  4. 单击下一步。继续选择要导入的项目和资源。
