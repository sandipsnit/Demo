指定导入安全设置

如果 JAR 文件所包含的资源的用户名和口令数据在导出时即进行了加密，则会显示导入资源 - 安全设置页。

  1. 输入用于对数据进行加密的口令 (请参阅导出资源)。

    2. 如果不知道用于加密资源数据的口令，请单击上一步按钮，并清除资源旁边的复选框。可以导入所有其他非加密资源而无须知道口令。

  2. 单击完成导入。

    3. 此时会显示正在处理配置数据弹出式窗口。应等到此窗口关闭后，再在 Oracle Service Bus 管理控制台上执行任何其他操作。当系统完成导入对象时，此窗口会自动关闭。此时会显示导入资源页，该页包括了关于您已成功部署数据的消息。

完成之后

继续复查导入概要。
