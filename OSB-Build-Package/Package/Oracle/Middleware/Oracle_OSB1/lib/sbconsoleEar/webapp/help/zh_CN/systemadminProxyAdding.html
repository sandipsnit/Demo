添加代理服务器

使用代理服务器概要页可以添加和配置代理服务器资源，并使这些资源可在 Oracle Service Bus 中作为系统资源使用。必须处于活动会话中才可以配置或重新配置代理服务器资源。

  1. 如果尚未执行此操作，请单击创建新建一个会话，或单击编辑进入现有会话。请参阅使用更改中心。
  2. 选择系统管理 > 代理服务器。
  3. 单击添加。
  4. 在名称字段中，输入代理服务器资源的名称。这是必填字段。

    如需命名指导，请参阅资源命名限制。
  5. 在说明字段中，输入此代理服务器资源的简要说明。
  6. 在主机端口参数部分中，输入以下信息：
    1. 在服务器主机字段中，输入代理服务器的主机名或 IP 地址。这是必填字段。

      2. Oracle Service Bus 代理服务器的服务器主机名必须与实际代理服务器的服务器主机名相同。

    2. 在明文端口字段中，输入代理服务器明文端口号。
    3. 在 SSL 端口字段中，输入代理服务器 SSL 端口号。必须输入明文或 SSL 端口号。
    4. 单击添加。

      5. 可以为每个代理服务器资源配置多个代理服务器。这可使 Oracle Service Bus 执行负载平衡并为代理服务器资源提供容错功能。

  7. 如果代理服务器执行代理验证，则在用户名字段中输入用户名，在口令和确认口令字段中输入关联的口令。

    8. 这些字段是可选的，仅当代理服务器受保护时，这些字段才是必填的。

  8. 单击保存可在当前会话中创建并保存代理服务器资源。
  9. 要结束该会话并将配置部署到运行时，请单击更改中心下的激活。
