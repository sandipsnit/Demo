查看已配置的 UDDI 注册表

使用此页可指定默认的 UDDI 注册表 (请参阅设置默认 UDDI 配置) 并查看 UDDI 注册表。此外，还可以搜索您以前配置的注册表。

  1. 在 UDDI 注册表面板的名称字段中，输入您要查找的注册表的名称，或输入通配符；* 用于多个字符，? 用于单个字符。搜索返回满足名称条件的全部记录。
  2. 单击搜索。
  3. 单击查看全部可删除搜索筛选器，并可查看所有配置为与 Oracle Service Bus 一起使用的注册表。

    4. 表 29-1 显示了 UDDI 注册表配置设置。

    表 29-1 UDDI 注册表配置设置 
    属性
    说明
    注册表名称
    注册表的名称。首次创建注册表时分配给注册表的名称。保存注册表项之后，不能编辑该注册表的名称。
    查询 URL
    用于定位和导入服务的查询 API 端点的 URL。
    发布 URL
    用于发布服务的发布 API 端点的 URL。
    安全 URL
    用于获取验证标记，以便可以向注册表发布内容的安全 API 端点的 URL。
    订阅 URL
    用于订阅注册表更改、创建注册表订阅以及监听对导入的服务所做的更改的订阅 API 端点的 URL。
    用户名
    进行用户验证所必需的用户名。
    口令 (确认口令)
    进行用户验证所必需的口令。
    将 tModels 加载到注册表
    将 tModels 加载到选定的注册表中。对于每个注册表，此选项都只能选择一次。
    启用自动导入
    自动将服务与 UDDI 注册表同步。
