添加 SMTP 服务器

使用 SMTP 服务器概要页可以添加和配置 SMTP 服务器资源，并使这些资源可在 Oracle Service Bus 中作为系统资源使用。必须处于活动会话中才可以配置或重新配置 SMTP 服务器资源。

  1. 如果尚未执行此操作，请单击创建新建一个会话，或单击编辑进入现有会话。请参阅使用更改中心。
  2. 选择系统管理 > SMTP 服务器。
  3. 单击添加。
  4. 在名称字段中，输入 SMTP 服务器资源的名称。这是必填字段。

    如需命名指导，请参阅资源命名限制。
  5. 在说明字段中，输入此 SMTP 服务器资源的简要说明。
  6. 在服务器 URL 字段中，输入指向此 SMTP 服务器的 URL。这是必填字段。
  7. 在端口号字段中，输入 SMTP 服务器的端口号 (默认端口为 25)。这是必填字段。
  8. 如果需要提供用户名和口令才能访问目标 SMTP 服务器，请在用户名字段中输入用户名，在口令和确认口令字段中输入关联的口令。

    9. 这些字段是可选的，仅当 SMTP 服务器受保护时，这些字段才是必填的。

  9. 单击保存可在当前会话中创建并保存 SMTP 服务器资源。
  10. 要结束该会话并将配置部署到运行时，请单击更改中心下的激活。
