查看 JNDI 提供程序

使用此页可查看 JNDI 提供程序列表和表 30-1 中所示的信息。

此外，还可以搜索您以前配置的 JNDI 提供程序。

  1. 要限制列表中项的数量或查找特定 JNDI 提供程序，请执行以下操作：
    • 单击打开图标可显示其他搜索筛选器。如果指定多个搜索条件，则仅返回与所有条件均匹配的结果。
    • 在名称字段中输入要查找的 JNDI 提供程序的名称，然后单击搜索以搜索特定条目。

      • 可输入通配符 (对于单个字符，使用 ?；对于多个字符，使用 *) 来执行更全面的搜索。

    • 输入要查找的 JNDI 提供程序的提供程序 URL，然后单击搜索。
  2. 单击查看全部可删除搜索筛选器，并显示所有配置为与 Oracle Service Bus 一起使用的 JNDI 提供程序资源。

    3. 表 30-1 显示 UDDI 注册表配置设置。

    表 30-1 JNDI 提供程序信息
    属性
    说明
    名称
    分配给此 JNDI 提供程序的唯一名称。此名称是指向查看 JNDI 提供程序 - 配置页的链接。请参阅编辑 JNDI 提供程序。
    提供程序 URL
    JNDI 提供程序的 URL。请参阅添加 JNDI 提供程序。
    JNDI 高速缓存
    “已启用”或“已禁用”。启用后，会在本地高速缓存 JNDI 上下文和 JNDI 对象，这样可在执行对象查找时提升性能。Oracle 建议保持 JNDI 高速缓存处于启用状态。
    操作
    • 单击重置图标可重置 JNDI 上下文以放弃 JNDI 连接和本地高速缓存的对象。
    • 单击启动测试控制台图标可测试 JNDI 提供程序是否能够成功建立连接并获取 JNDI 上下文。
    选项
    • 单击重命名图标可重命名 JNDI 提供程序。
    • 单击删除图标可删除特定的 JNDI 提供程序。请参阅删除 JNDI 提供程序。
