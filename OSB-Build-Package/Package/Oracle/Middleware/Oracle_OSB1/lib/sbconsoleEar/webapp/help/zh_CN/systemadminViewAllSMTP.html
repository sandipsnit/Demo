查看 SMTP 服务器

使用此页可指定默认 SMTP 服务器 (请参阅配置默认 SMTP 服务器)，并可查看 SMTP 服务器列表。

配置预警目标资源和基于电子邮件传输的业务服务时，可使用 SMTP 服务器资源。请参阅添加预警目标和“创建/编辑业务服务 - 电子邮件传输配置”页。

表 30-3 显示 SMTP 服务器信息。

表 30-3 SMTP 服务器信息
属性
说明
名称
分配给此 SMTP 服务器资源的唯一名称。此名称是一个指向查看 SMTP 服务器 - 配置页的链接。请参阅编辑 SMTP 服务器。
服务器 URL
指向此 SMTP 服务器的 URL。请参阅添加 SMTP 服务器。
选项
  • 单击重命名图标可重命名此 SMTP 服务器资源。
  • 单击删除图标可删除特定 SMTP 服务器。请参阅删除 SMTP 服务器。
