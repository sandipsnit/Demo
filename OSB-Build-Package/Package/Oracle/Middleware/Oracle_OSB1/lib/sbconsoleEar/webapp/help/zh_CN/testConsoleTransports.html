了解运行时如何在测试控制台中使用传输设置

配置代理服务测试数据和配置业务服务测试数据描述当您在测试控制台中测试代理服务或业务服务时，应如何配置用于出站请求的传输标头、传输元数据以及与传输相关的安全数据的值。但是，某些可以在测试控制台中制定的规范在运行时不生效。也就是说，在执行测试过程中，Oracle Service Bus 在运行时会覆盖或忽略某些标头值或元数据值。表 32-10 中描述了存在某些限制的标头和元数据。
