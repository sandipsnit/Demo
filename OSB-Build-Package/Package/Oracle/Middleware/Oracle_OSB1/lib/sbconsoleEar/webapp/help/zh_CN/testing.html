测试控制台

Oracle Service Bus 测试控制台是基于浏览器的测试环境，用于验证和测试系统的设计。可以在测试控制台中配置测试的对象 (代理服务、业务服务或资源)、执行测试以及查看结果。在某些情况下，可以通过代码进行跟踪并检查特定跟踪点上的消息状态。要了解有关测试控制台的详细信息，请参阅 Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus 中的“Using the Test Console”，网址为：http://www.oracle.com/pls/as111170/lookup?id=OSBAG357。

只有具有 IntegrationAdmin 和 IntegrationDeployer 角色的用户才能使用测试控制台。有关角色的详细信息，请参阅 Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus 中的“Roles in Oracle Service Bus”，网址为：http://www.oracle.com/pls/as111170/lookup?id=OSBAG463。

注：如果您收到一个说明测试控制台服务未在运行的错误，请尝试将管理服务器监听地址设置为特定的有效地址，例如 localhost。在 Oracle WebLogic Server 控制台中，转至环境 > 服务器 > admin_server_name > 配置 > 一般信息可设置监听地址。
