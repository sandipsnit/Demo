执行 XQuery 测试

可以使用以下编辑器编辑并测试消息流中的操作：XQuery 表达式编辑器、XQuery 条件编辑器和 XPath 表达式编辑器。对于 XQuery 表达式编辑器和条件编辑器，测试采用相同的形式。但是，XPath 表达式编辑器的情况有所不同，因为它只接受一个输入。

注： 必须禁用浏览器中的弹出式窗口阻止程序，才能进行 XQuery 测试。如果 IE 浏览器上有工具栏，则可能需要从选项菜单下禁用弹出式窗口阻止程序，并将所有工具栏配置为阻止弹出式窗口。

此部分包含以下主题：

  • 使用 XQuery 表达式编辑器和 XQuery 条件编辑器
  • 使用 XPath 表达式编辑器
