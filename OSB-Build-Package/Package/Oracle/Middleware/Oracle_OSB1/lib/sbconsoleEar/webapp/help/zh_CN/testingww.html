查看代理服务测试结果

表 32-2 描述代理服务测试结果。只有在选择了直接调用和包括跟踪选项之后，才会启用跟踪。

表 32-2 代理服务的测试结果
部分
说明
代理服务名称
所测试的代理服务的名称。
测试控制台操作
“返回”用于显示上一个浏览页。
关闭用于关闭测试控制台。
请求文档
通过测试控制台发送到代理服务的请求消息。
如果测试控制台没有修改此请求消息，则此部分最初会处于折叠状态。对于使用表单选项卡配置的 SOAP 服务，或者在应用了 WS-Security 的情况下，则此部分最初会处于展开状态。
如果应用了 WS-Security，则此部分将包含两个 SOAP 消息 - 第一个消息是明文消息；第二个消息是受保护的 SOAP 消息。
响应文档
消息响应。
对于带有 WS-Security 响应的 SOAP 服务，此部分包含两个 SOAP 消息。第一个 SOAP 消息是客户机接收的受保护的消息。第二个 SOAP 消息是相应的明文消息。
响应元数据
随消息响应返回的元数据。
跟踪代理服务
跟踪会显示消息在系统中传递时的状态。如果未选择“直接调用”选项，则不执行跟踪。For more information on tracing, see 跟踪代理服务。
