测试 MFL 转换

消息格式语言（Message Format Language，简称 MFL）文档是用于描述二进制数据布局的专用 XML 文档。MFL 资源支持两种转换。每种转换只接受一种输入并提供单一输出：XML 到二进制的转换或二进制到 XML 的转换。

表 32-5 描述 MFL 配置选项。

MFL 转换配置
XML 输入
对 XML 到二进制的转换的要求：
可以推算出 MFL 文档的 XML 方案。示例 XML 文档会自动输入到文本字段中。
XML 输入可以是基于文件的，也可以是基于文本的。通过引用某个文件来提供输入优先于文本输入。浏览并选择要在测试中使用的文件。
二进制输入
对二进制到 XML 的转换的要求：
二进制输入可以是基于文件的，也可以是基于文本的。通过引用某个文件来提供输入优先于文本输入。浏览并选择要在测试中使用的文件。

可以针对设计时或运行时进行测试。

  1. 如果要针对运行时进行测试，请单击“激活”。如果要针对设计时进行测试，请不要激活会话。
  2. 选择“资源浏览器”>“MFL”以便显示“MFL 文件概要”页。
  3. 在操作下，单击与要测试的资源关联的启动测试控制台图标。此时，测试控制台将打开资源测试页。
  4. 选择要测试的转换。
  5. 为 MFL 资源配置测试数据。有关详细信息，请参阅表 32-5。
  6. 单击“执行”。“资源测试”页会显示结果。
  7. 要重新测试，请单击返回。您可以关闭测试控制台，然后修改并重新测试资源。
