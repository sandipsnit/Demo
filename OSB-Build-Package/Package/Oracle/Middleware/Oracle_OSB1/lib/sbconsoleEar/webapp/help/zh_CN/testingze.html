测试 XQuery 转换

XQuery 映射可以描述 XML 到 XML、XML 到非 XML、和非 XML 到 XML 的映射。XQuery 转换可接受多个输入并返回一个输出结果。每个输入都分别与 XQuery 资源中声明的一个 XQuery 外部变量相对应。XQuery 输入变量的值可以是基元值（字符串、整数、日期等等）、XML 文档或这些类型的序列。输出值可以是基元值（字符串、整数、日期等等）、XML 文档或这些类型的序列。

注： 测试控制台不支持输入序列。

变量
对于每个 XQuery 外部变量，该部分都包含一个与之对应的输入字段。
<param_name> ([] as XML)
param_name 是 XQuery 资源中的 XQuery 变量名。
在测试控制台中，如果类型为简单类型，将显示单行编辑框。如果数据为 XML，则将显示多行编辑框。
如果变量为非类型的，则将使用组合输入 (<param_name> ([] as XML))。必须声明变量类型。选中该复选框可标识类型为 XML 的参数。
XML 输入可以是基于文件的，也可以是基于文本的。通过引用某个文件来提供输入优先于文本输入。浏览并选择要在测试中使用的文件。
测试控制台中的输入基于类型来呈现，以便于了解必须输入的数据类型。如果是无类型的，则默认类型为 XML。

可以针对设计时或运行时进行测试。

  1. 如果要针对运行时进行测试，请单击“激活”。如果要针对设计时进行测试，请不要激活会话。
  2. 选择“项目浏览器”>“XQuery”以便显示“XQuery 概要”页。
  3. 在操作下，单击与要测试的资源关联的启动测试控制台图标。此时，测试控制台将打开资源测试页。
  4. 在此页面的变量部分中，为资源配置测试数据。有关详细信息，请参阅表 32-7。
  5. 单击“执行”。“资源测试”页会显示结果。
  6. 要重新测试，请单击返回。您可以关闭测试控制台，然后修改并重新测试资源。
