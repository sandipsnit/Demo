使用 XPath 表达式编辑器

使用 XPath 表达式可选择 XML 消息上下文变量的子集。可以在 XPath 表达式编辑器中使用测试控制台测试 XPath 表达式的定义。XPath 表达式接受单个 XML 文档作为输入并生成 XML 文档的序列和/或基元类型作为输出。

表 32-9 描述 XPath 表达式配置选项。

此部分包含一个与作为 XPath 表达式测试依据的 XML 文档相对应的输入字段。
 
XML 输入可以是基于文件的，也可以是基于文本的。通过引用某个文件来提供输入优先于文本输入。浏览并选择要在测试中使用的文件。

  1. 在编辑管道消息流中的操作时访问测试控制台。要访问 XPath 表达式编辑器，请参阅创建和编辑内嵌 XQuery 和 XPath 表达式。
  2. 在此页的变量部分中，为 XPath 表达式配置测试数据。有关详细信息，请参阅表 32-9。
  3. 单击“执行”。测试页会显示结果。
  4. 完成测试后，可单击返回按钮以执行新的测试。要在更改 XPath 表达式后执行新的测试，必须先关闭测试控制台，然后再将其打开，以使更改生效。
