UDDI

通用描述、发现和集成 (UDDI) 注册表在企业中用于共享 Web 服务。UDDI 提供了一个框架，在该框架中，对您的业务、其服务以及有关您要公开的服务的详细技术信息进行了分类。

将服务发布到注册表需要了解有关该服务类型和注册表中代表该服务的数据结构的知识。注册表项具有与之相关联的某些属性，并且这些属性类型是在创建注册表时定义的。可以将您的服务发布到注册表供其他组织“发现”和使用。在 Oracle Service Bus 中开发的代理服务能够发布到 UDDI 注册表。Oracle Service Bus 可与任何与 UDDI 3.0 版本兼容的注册表交互。

另请参阅 Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus 中的“UDDI”，网址为：http://www.oracle.com/pls/as111170/lookup?id=OSBAG392。
