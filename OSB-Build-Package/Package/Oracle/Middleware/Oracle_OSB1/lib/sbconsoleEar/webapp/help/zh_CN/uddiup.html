编辑 UDDI 注册表

  1. 如果尚未执行此操作，请单击创建新建一个会话，或单击编辑进入现有会话。请参阅使用更改中心。
  2. 选择系统管理 > UDDI 注册表。
  3. 在“UDDI 注册表”面板中，从可用注册表的列表中单击要编辑的注册表的名称。

    4. 还可以使用搜索选项搜索特定注册表。请参阅查看已配置的 UDDI 注册表。

    UDDI 注册表配置页显示定义注册表的配置属性和下列特定于注册表的信息，如表 29-2 中所示。

    表 29-2 UDDI 注册表信息
    属性
    说明
    上次修改者
    对该 UDDI 注册表资源进行最后一次配置更改的用户。
    上次修改时间
    进行最后一次配置更改的日期和时间。单击日期和时间链接可查看此资源的更改历史记录。请参阅“查看更改历史记录”页。
    引用
    此 UDDI 注册表引用的对象数。如果存在此类引用，可单击数字链接以查看引用列表。请参阅查看对资源的引用。
    引用项
    引用此 UDDI 注册表的对象数。如果存在此类引用，可单击数字链接以查看对象列表。请参阅查看对资源的引用。
    说明
    注册表的描述。

  4. 在页面底部，单击“编辑”。
  5. 在“UDDI 配置 - 编辑注册表”页上，编辑注册表配置参数。

    6. 表 29-1 中描述了这些属性。您不能编辑注册表的名称。

  6. 单击保存可提交当前会话中的更新。
  7. 要结束该会话并将配置部署到运行时，请单击更改中心下的激活。
