WSDL

WSDL (Web 服务定义语言) 是 Web 服务的正式描述；在 Oracle Service Bus 中，它描述代理服务或业务服务。WSDL 用来描述 Web 服务可执行的任务、所在位置以及其调用方式。

可以使 SOAP 和 XML 服务以现有的 WSDL 资源为基础。代理服务和业务服务可以使用 WSDL 文档进行任何传输。请参阅“创建/编辑代理服务 - 一般配置”页和创建和配置业务服务。

有关详细信息，请参阅 Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus 中的“How WSDL is Used in Oracle Service Bus”，网址为：http://www.oracle.com/pls/as111170/lookup?id=OSBAG114。
