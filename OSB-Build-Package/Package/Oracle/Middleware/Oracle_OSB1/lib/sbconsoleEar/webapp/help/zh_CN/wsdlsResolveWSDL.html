解析未解析的 WSDL 引用

使用编辑 WSDL 资源的引用页可通过为 WSDL 引用 (例如 WSDL 导入和 XML 方案导入) 配置映射来解析未解析的 WSDL 引用。WSDL 还可以具有对 WS-Policy 的未解析引用。

解析未解析的 WSDL 或 XML 方案引用
  1. 如果尚未执行此操作，请单击创建新建一个会话，或单击编辑进入现有会话。请参阅使用更改中心。
  2. 选择资源浏览器 > WSDL。
  3. 在 WSDL 概要页的 WSDL 名称列中，单击相应的 WSDL 名称可以查看未解析引用的详细资料。此时将显示查看 WSDL 详细资料页。
  4. 单击编辑引用以便显示编辑 WSDL 资源的引用页。
  5. 在资源类型字段中，选择 WSDL 或 XML 方案。
  6. 单击浏览。根据资源类型，会显示 WSDL 浏览器或 XML 方案浏览器。
  7. 在 WSDL 浏览器或 XML 方案浏览器中，选择一个 WSDL 或 XML 方案，然后从定义窗格中选择定义。
  8. 单击提交。您选择的 WSDL 或 XML 方案将显示在资源名字段中。
  9. 单击保存以解析引用。
  10. 在查看 WSDL 详细资料页上，单击确定。WSDL 的状态显示为有效。
解析未解析的 WS-Policy 引用
  1. 如果尚未执行此操作，请单击创建新建一个会话，或单击编辑进入现有会话。请参阅使用更改中心。
  2. 选择资源浏览器 > WSDL。
  3. 在 WSDL 概要页的 WSDL 名称列中，单击相应的 WSDL 名称可以查看未解析引用的详细资料。此时将显示查看 WSDL 详细资料页。
  4. 单击编辑引用以便显示编辑 WSDL 资源的引用页。
  5. 在 WS-Policy 名称字段中，单击浏览。此时会显示策略浏览器。
  6. 在策略浏览器中，选择一个 WS-Policy。
  7. 单击提交。您选择的 WS-Policy 将显示在 WS-Policy 名称字段中。
  8. 单击保存以解析引用。
  9. 在查看 WSDL 详细资料页上，单击确定。WSDL 的状态显示为有效。
