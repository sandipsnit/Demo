要查找特定引用，请执行以下操作：

  • 对列表进行重新排序。单击带下划线的列名。升序箭头和降序箭头指示排序顺序。单击列名可更改排序顺序。
  • 滚动查看各页。使用表上面的页控件，或表下面的页控件。通过选择页码转到某一页，或通过使用箭头按钮转到下一页、上一页、第一页或最后一页。
