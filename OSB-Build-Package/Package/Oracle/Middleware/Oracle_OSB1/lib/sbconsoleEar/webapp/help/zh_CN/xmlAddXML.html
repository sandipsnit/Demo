添加 XML 文档

在 Oracle Service Bus 中创建 XML 文档的最简单方法是批量导入 XML 文档。例如，如果您批量导入已创建的 JCA 资源 (.jca 文件、相关联的 WSDL 和 TopLink 或 EclipseLink 映射文件)，Oracle Service Bus 会自动根据映射文件生成 XML 文档资源，并维护资源文件之间的相关性。

有关导入的详细信息，请参阅从 Zip 文件加载资源、从 URL 加载资源和导入资源。

如果未批量导入，则可以手动创建 XML 文档，如以下步骤所述：

  1. 如果尚未执行此操作，请单击创建新建一个会话，或者单击编辑进入现有会话。请参阅使用更改中心。
  2. 选择项目浏览器，然后选择要在其中添加 XML 文档的项目或文件夹。此时会显示项目/文件夹视图页。
  3. 从创建资源字段中选择 XML，以显示创建新的 XML 文档页。
  4. 在 XML 文件字段中，单击浏览并选择要使用的 XML 文件。

    5. “资源名”字段会自动填充文件名 (无文件扩展名)。

    如果不选择文件，请复制 XML 文件的内容并粘贴在“XML 文件”字段文本区域。

  5. 在资源名字段中，输入或更改 XML 文档的名称。XML 文档名必须是唯一的。

    如需命名指导，请参阅资源命名限制。
  6. 在资源说明字段中，输入此 XML 文档的说明。
  7. 单击保存。新的 XML 文档会保存在当前会话中。
  8. 要结束该会话并将配置部署到运行时，请单击更改中心下的激活。
