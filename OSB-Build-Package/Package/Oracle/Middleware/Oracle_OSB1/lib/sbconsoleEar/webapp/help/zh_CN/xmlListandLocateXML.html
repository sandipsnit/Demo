查找 XML 文档

XML 文档概要页列出了已添加到 Oracle Service Bus 域的 XML 文档。

  1. 请执行以下两个操作之一：
    • 选择项目浏览器，以显示“项目视图”页或“项目/文件夹视图”页。然后，在项目和文件夹中导航以查找 XML 文档。
    • 选择资源浏览器 > XML 文档。XML 文档概要页显示表 13-1 中所示的信息。有关属性的更详细说明，请参阅编辑 XML 文档。
  2. 要查找特定 XML 文档，请执行以下操作：
    • 要限制列表中的项数，您可以先按名称和路径进行筛选，然后单击搜索。

      • 路径由项目名和资源所在的文件夹名称组成。

    • 单击查看全部可删除搜索筛选器，并显示所有 XML 文档。

      • 表 13-1 XML 文档详细资料
      属性
      说明
      XML 文档
      分配给此 XML 文档的唯一名称。此名称是一个指向查看 XML 文档页的链接。请参阅编辑 XML 文档。
      路径
      该路径由项目名和 XML 文档所在的文件夹名称组成。它是一个指向包含此资源的项目或文件夹的链接。请参阅使用项目和文件夹限定资源名。
      选项
      单击删除图标可删除特定的 XML 文档。请参阅删除 XML 文档。
