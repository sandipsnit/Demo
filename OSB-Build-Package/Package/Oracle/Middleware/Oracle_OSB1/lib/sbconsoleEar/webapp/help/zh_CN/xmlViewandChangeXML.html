编辑 XML 文档

使用编辑 XML 文档页可查看和更改 JCA 绑定的详细资料。

  1. 如果尚未执行此操作，请单击创建新建一个会话，或单击编辑进入现有会话。请参阅使用更改中心。
  2. 查找 XML 文档，如查找 XML 文档中所述。
  3. 单击 XML 文档名。查看 XML 文档页显示表 13-2 中所示的信息。

    4. 表 13-2 XML 文档详细资料
    属性
    说明
    上次修改者
    创建此 XML 文档或将其导入到配置中的用户。
    上次修改时间
    用户创建此 XML 文档或将其导入到配置中的日期和时间。单击日期和时间链接可查看此资源的更改历史记录。请参阅“查看更改历史记录”页。
    引用
    此 XML 文档引用的对象数。如果存在此类引用，可单击数字链接以查看对象列表。请参阅查看对资源的引用。
    引用项
    引用此 XML 文档的对象数。如果存在此类引用，可单击数字链接以查看对象列表。例如，如果 XML 文件是 JCA 绑定所引用的 TopLink 文件，则单击此链接后，此 JCA 绑定将作为引用列出。请参阅查看对资源的引用。
    说明
    此 XML 文档的说明 (如果存在)。

    默认情况下，将显示 XML 文档的只读视图。

  4. 要编辑 XML 文档，请单击编辑。有关字段说明，请参阅添加 XML 文档。
  5. 进行相应的编辑，然后单击保存提交当前会话中的更新。
  6. 要结束该会话并将配置部署到运行时，请单击更改中心下的激活。
