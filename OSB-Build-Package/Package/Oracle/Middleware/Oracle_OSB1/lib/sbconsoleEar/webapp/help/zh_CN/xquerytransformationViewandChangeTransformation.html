编辑 XQuery 转换

  1. 如果尚未执行此操作，请单击创建新建一个会话，或单击编辑进入现有会话。请参阅使用更改中心。
  2. 查找 XQuery 转换，如查找 XQuery 转换中所述。
  3. 单击 XQuery 转换名。查看 XQuery 转换详细资料页显示表 10-2 中所示的信息。

    4. 表 10-2 XQuery 转换资源详细资料
    属性
    说明
    上次修改者
    创建此 XQuery 转换或将其导入到配置中的用户。
    上次修改时间
    用户创建此 XQuery 转换或将其导入到配置中的日期和时间。单击日期和时间链接可查看此资源的更改历史记录。请参阅“查看更改历史记录”页。
    引用
    此 XQuery 转换引用的对象数。如果存在此类引用，可单击数字链接以查看对象列表。要了解详细信息，请参阅查看对资源的引用。
    引用项
    引用此 XQuery 转换的对象数。如果存在此类引用，可单击数字链接以查看对象列表。例如，如果将该 XQuery 转换分配给特定代理服务的消息流中的变量，则在单击该链接后，代理服务将作为引用列出。要了解详细信息，请参阅查看对资源的引用。
    说明
    此 XQuery 转换的说明 (如果存在)。
    XQuery
    此 XQuery 转换的完整文本。
    XQuery 变量
    为此 XQuery 转换选定的 XQuery 变量。

  4. 要对这些字段进行更改，请单击编辑。有关这些字段的说明，请参阅添加 XQuery 转换。

    5. 您不能更改资源名字段。

  5. 单击保存可提交当前会话中的更新。
  6. 要结束该会话并将配置部署到运行时，请单击更改中心下的激活。
