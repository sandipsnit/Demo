添加 XSL 转换

  1. 如果尚未执行此操作，请单击创建新建一个会话，或单击编辑进入现有会话。请参阅使用更改中心。
  2. 选择项目浏览器将显示“项目视图”页或“项目/文件夹视图”页。
  3. 导航至要向其中添加 XSL 转换的项目或文件夹。
  4. 从创建资源列表中选择 XSLT，以便显示创建新的 XSLT 页。
  5. 在资源名字段中，输入此 XSL 转换的唯一名称。

    如需命名指导，请参阅资源命名限制。
  6. 在资源说明字段中，输入此 XSL 转换的说明。
  7. 在 XSLT 字段中，执行以下操作之一：
    • 输入新 XSL 转换的文本。
    • 单击浏览以定位和导入现有 XSL 转换。
    • 将文本从现有的 XSL 转换复制并粘贴到此字段中。
  8. 单击保存可提交当前会话中的更新。
  9. 要结束该会话并将配置部署到运行时，请单击更改中心下的激活。
