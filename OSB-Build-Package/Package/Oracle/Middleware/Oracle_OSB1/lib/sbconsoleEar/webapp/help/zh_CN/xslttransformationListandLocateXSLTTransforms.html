查找 XSL 转换

  1. 请执行以下两个操作之一：
    • 选择项目浏览器以便显示“项目视图”页或“项目/文件夹视图”页。然后，在项目和文件夹中导航以查找 XSL 转换。
    • 选择资源浏览器 > XSLT。XSLT 概要页显示表 14-1 中所示的信息。
  2. 要限制列表中项的数量，可按名称、路径或同时按这两个条件进行筛选。在搜索下的名称和路径字段中，分别输入目标的名称和/或路径，然后单击搜索按钮。

    3. 该路径由项目名和 XSLT 所在的文件夹名称组成。

    允许使用通配符 * 和 ?。搜索区分大小写。

    单击查看全部可显示域中的所有 XSLT。这会清除以前的搜索中使用的搜索参数。

    表 14-1 XSL 转换详细资料
    属性
    说明
    XSLT 名称
    分配给 XSL 转换的名称。单击该名称可显示查看 XSL 转换详细资料页。
    请参阅编辑 XSL 转换。
    路径
    该路径由项目名和 XSL 转换所在的文件夹名称组成，例如，MyProject/MyFolder/MyResource。
    单击 XSLT 的路径可显示包含它的“项目/文件夹视图”页。
    操作
    单击启动测试控制台图标可调用测试控制台，测试控制台可用于验证和测试服务与转换的设计。对于 XSL 转换，无论在会话内还是会话外都可以使用测试控制台。要了解详细信息，请参阅测试转换。
    选项
    如果尚未执行此操作，请单击创建新建一个会话，或单击编辑进入现有会话。请参阅使用更改中心。
    单击删除图标可删除 XSLT。如果其他资源引用此资源，将会显示删除警告图标。确认警告后即可删除该资源。这可能会由于存在对已删除资源的未解析引用而导致冲突。有关详细信息，请参阅删除 XSL 转换。
