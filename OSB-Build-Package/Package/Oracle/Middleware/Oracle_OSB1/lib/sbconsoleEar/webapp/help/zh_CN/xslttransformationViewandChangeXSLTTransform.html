编辑 XSL 转换

  1. 如果尚未执行此操作，请单击创建新建一个会话，或单击编辑进入现有会话。请参阅使用更改中心。
  2. 查找 XSL 转换，如查找 XSL 转换中所述。
  3. 单击 XSL 转换名称。默认情况下，查看 XSL 转换详细资料页显示 XSLT 的结构视图，其中包括表 14-2 中所示的信息。

    4. 表 14-2 XSL 转换资源详细资料 
    属性
    说明
    上次修改者
    创建此 XSL 转换或将其导入到配置中的用户。
    上次修改时间
    用户创建此 XSL 转换或其导入到配置中的日期和时间。单击日期和时间链接可查看此资源的更改历史记录。请参阅“查看更改历史记录”页。
    引用
    此 XSL 转换引用的对象数。如果存在此类引用，可单击数字链接以查看对象列表。要了解详细信息，请参阅查看对资源的引用。
    引用项
    引用此 XSL 转换的对象数。如果存在此类引用，可单击数字链接以查看对象列表。例如，如果将此 XSL 转换分配给特定代理服务的消息流中的某个变量，则在单击该链接后，此代理服务将作为引用列出。要了解详细信息，请参阅查看对资源的引用。
    说明
    此 XSL 转换的说明。
    目标名称空间
    用来限定 XSL 转换中所包括的任何定义的名称空间。
    XSLT 相关性
    与此 XSLT 关联的属性和组：
    • XSLT 位置
    状态
    XSL 转换的状态：
    • 有效

      • 当前 XSL 转换包括的所有 XSL 转换的位置都已指定并且有效。而且，所有嵌套 XSL 转换 (由包括的 XSL 转换随后包括的 XSL 转换) 的位置都已指定并且有效。

    • 无效

      • 当前 XSL 转换所包括的一个或多个 XSL 转换的位置没有指定或者无效。此外，一个或多个嵌套 XSL 转换（由包括的 XSL 转换接着包括的 XSL 转换）的位置可能没有指定或者无效。要解析无效的 XSL 转换，请单击编辑引用。要了解详细信息，请参阅解析未解析的 XSL 转换引用。

  4. 单击文本视图可显示 XSL 转换详细资料的文本视图。文本视图包括表 14-3 中所示的信息。

    5. 表 14-3 XSL 转换文本详细资料
    属性
    说明
    目标名称空间
    用来限定 XSL 转换中所包括的任何相关性的名称空间。
    XSLT
    此 XSL 转换的完整文本。

  5. 要对这些字段进行更改，请单击编辑。请参阅添加 XSL 转换以获得字段的说明。

    6. 您不能更改资源名字段。

  6. 单击保存可提交当前会话中的更新。
