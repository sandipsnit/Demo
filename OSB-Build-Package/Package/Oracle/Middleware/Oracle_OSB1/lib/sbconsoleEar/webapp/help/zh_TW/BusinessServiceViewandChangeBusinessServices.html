編輯業務服務組態

  1. 如果尚未進行這樣的處理，請按一下建立以建立新的階段作業，或按一下編輯以輸入現有的階段作業。請參閱使用變更中心。

    2. 如果您只想檢視詳細資訊，則不需進入階段作業。

  2. 找出您要檢視或編輯的業務服務，如尋找業務服務中所說明。
  3. 按一下業務服務名稱。檢視業務服務 - 組態詳細資訊頁面會顯示選定業務服務的組態資訊。
  4. 若要檢視或修改設定，請執行下列任一項操作：
    • 按一下想要檢視或編輯特性之類別名稱 (例如一般組態、傳輸組態等) 旁的編輯圖示。您所能編輯的頁面，視您在建立業務服務時所設定的頁面而定。

      • 有關所有這些頁面的完整清單，請參閱檢視業務服務 - 組態詳細資訊頁面。

    • 按一下頁面底部的「編輯」，以顯示建立/編輯業務服務 - 一般組態頁面，這是一系列用來設定此業務服務的頁面中所顯示的第一頁。
  5. 繼續檢視或編輯，如建立和設定業務服務中所說明。
  6. 在編輯業務服務 - 摘要頁面上，按一下「儲存」，確認目前階段作業中的更新。
  7. 若要結束階段作業並將組態建置至執行時期，請按一下「變更中心」下的「啟動」。
