新增 MFL

  1. 如果尚未進行這樣的處理，請按一下建立以建立新的階段作業，或按一下編輯以輸入現有的階段作業。請參閱使用變更中心。
  2. 選取專案瀏覽器，然後選取要新增 MFL 檔案的專案或資料夾。隨即顯示專案/資料夾檢視畫面頁面。
  3. 在「建立資源」下拉清單中，選取「轉換」下方的「MFL 檔案」，以顯示「建立新的 MFL 檔案資源」頁面。
  4. 在資源名稱欄位中，輸入此 MFL 的唯一名稱。這是必要欄位。

    請遵循資源命名限制的命名指南。
  5. 在「資源描述」欄位中，輸入 MFL 的描述。
  6. 在「MFL 文字」欄位中，執行下列其中一項操作：
    • 輸入新 MFL 的文字。
    • 按一下「瀏覽」以尋找並匯入現有的 MFL。
    • 從現有的 MFL 中複製文字並貼在此欄位。

      • 這是必要欄位。

  7. 按一下「儲存」。會建立 MFL 檔案資源，並儲存在目前的階段作業中。
  8. 若要結束階段作業並將組態建置至執行時期，請按一下「變更中心」下的「啟動」。
