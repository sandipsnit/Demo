尋找 MFL

  1. 請執行下列任一項操作：
    • 選取專案瀏覽器以顯示專案檢視畫面頁面或專案/資料夾檢視畫面頁面。然後瀏覽專案和資料夾，以尋找 MFL。
    • 選取資源瀏覽器 > MFL。MFL 檔案的摘要頁面會顯示表 15-1 中所示的資訊。有關特性的更詳細描述，請參閱編輯 MFL。
  2. 若要限制清單中的項目數，則可以依據名稱和 (或) 路徑來進行篩選。在「名稱」和「路徑」欄位中，輸入搜尋目標的名稱和路徑，然後按一下「搜尋」。

    3. 路徑是專案名稱和 MFL 檔案所在資料夾的名稱。

    按一下「檢視全部」移除搜尋篩選，並顯示所有 MFL 檔案。

    表 15-1 MFL 詳細資訊
    特性
    描述
    MFL 檔案名稱
    指派給 MFL 的唯一名稱。此名稱是連到檢視 MFL 資源詳細資訊頁面的連結。請參閱編輯 MFL。
    路徑
    路徑是專案名稱和 MFL 檔案所在資料夾的名稱。它是連到包含此資源之專案或資料夾的連結。請參閱使用專案與資料夾限定資源名稱。
    動作
    按一下啟動測試主控台圖示來叫用「測試主控台」，您可以用此來驗證和測試服務及轉換的設計。對於 MFL，無論在階段作業內或外，您都可以使用「測試主控台」。請參閱測試轉換。
    選項
    按一下刪除圖示，以刪除特定的 MFL。請參閱刪除 MFL。
