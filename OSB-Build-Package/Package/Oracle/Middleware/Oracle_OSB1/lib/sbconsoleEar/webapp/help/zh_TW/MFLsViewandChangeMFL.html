編輯 MFL

  1. 如果尚未進行這樣的處理，請按一下「建立」以建立新的階段作業，或按一下編輯以輸入現有的階段作業。請參閱使用變更中心。
  2. 找出 MFL 檔案 (如尋找 MFL 中所述)。
  3. 按一下 MFL 檔案名稱。檢視 MFL 資源詳細資訊頁面會顯示表 15-2 中所示的資訊。

    4. 表 15-2 MFL 資源詳細資訊 
    特性
    描述
    上次修改者
    建立此 MFL 檔案或將檔案匯入組態中的使用者。
    上次修改時間
    使用者建立此 MFL 檔案或將其匯入組態中的日期和時間。按一下日期和時間連結，以檢視此資源的變更歷史記錄。請參閱檢視變更歷史記錄頁面。
    參照
    此 MFL 檔案參照的物件數。如果有這類參照，請按一下數字連結以檢視物件清單。請參閱檢視資源參照。
    參照者
    參照此 MFL 檔案的物件數。如果有這類參照，請按一下數字連結以檢視物件清單。例如，若您選取此 MFL 檔案做為訊息業務服務或代理主機服務的要求或回應訊息類型，則當您按一下連結時，即會列出此業務服務或代理主機服務做為參照。請參閱檢視資源參照。
    描述
    此 MFL 檔案的描述 (如果有的話)。
    MFL 文字
    此 MFL 的文字。

  4. 若要變更欄位，請按一下編輯。有關欄位的描述，請參閱新增 MFL。

    5. 您不可以變更「資源名稱」欄位。

  5. 按一下「儲存」，確認目前階段作業中的更新。
  6. 若要結束階段作業並將組態建置至執行時期，請按一下「變更中心」下的「啟動」。
