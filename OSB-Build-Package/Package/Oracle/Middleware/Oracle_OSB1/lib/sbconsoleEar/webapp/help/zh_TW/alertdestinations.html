警示目標

警示目標資源會擷取可以接收來自 Oracle Service Bus 之警示通知的收件者清單。在一般系統監督相關資訊環境中，Oracle Service Bus 產生的警示可以承載有限的一組使用者的變更。

在 Oracle Service Bus 中，您可以根據指定的相關資訊環境來設定警示目標資源，以納入一組收件者。在訊息流程中設定的「警示」動作及 SLA 警示規則，都會使用「警示目標」。警示目標可包含下列一或多種目標類型：主控台 (預設值)、報告資料串流、SNMP 陷阱、電子郵件、JMS 佇列或 JMS 主題。

如果是電子郵件和 JMS 目標，則目標資源可以分別包含電子郵件地址清單或 JMS URI 清單。

您可以針對服務的警示組態，重複使用警示目標。請參閱新增警示動作。
