DSP 傳輸設定頁面

「建立/編輯商業服務 - DSP 傳輸設定」頁面是一系列頁面中的其中一頁，供您建立和編輯商業服務 (如建立和設定商業服務和編輯商業服務設定中所述)。

有關詳細資訊，請參閱 透過 Oracle Service Bus 存取資料服務。

使用此頁面，可讓您透過 DSP (Oracle Data Service Integrator) 傳輸通訊協定來設定商業服務的傳輸設定。表 17-4 說明如何使用此頁面。

表 17-4 建立/編輯商業服務 - DSP 傳輸組態頁面 
選項
建立或編輯...
除錯層級
指定下列其中一項
  • 0 - 無除錯資訊
  • 1 - 輸出要求訊息的相關資訊
  • 3 - 輸出要求與回應訊息的相關資訊
服務帳戶
按一下「瀏覽」，然後從顯示的清單中選擇服務帳戶。若未指定服務帳戶，則會使用匿名主題。

有關詳細資訊：

  • 服務帳戶
  • 新增服務帳戶
分派原則
選擇要針對此端點的分派原則而使用的 WebLogic Server 工作管理器實例。如果不存在其他工作管理器，則會使用預設的工作管理器。
工作管理器用於發佈回應處理的回覆訊息。有關工作管理器的資訊，請參閱「WebLogic Server 管理主控台線上說明」中的 使用工作管理器以最佳化排定的工作與 建立全域工作管理器。

有關從 Oracle Service Bus 存取 Oracle Data Service Integrator 的詳細資訊，請參閱 透過 Oracle Service Bus 存取資料服務。

完成作業後

按「下一頁」繼續在下一個頁面上設定此服務，或按一下「最後一頁」，在建立/編輯商業服務 - 摘要頁面上檢查並儲存這項設定。
