FTP 傳輸組態頁面

「建立/編輯業務服務 - FTP 傳輸組態」頁面是一系列頁面中的其中一頁，供您建立和編輯業務服務 (如建立和設定業務服務和編輯業務服務組態中所述)。

使用此頁面，可讓您透過 FTP 傳輸通訊協定來設定業務服務的傳輸設定。表 18-9 說明如何使用此頁面。

表 18-9 建立/編輯業務服務 - FTP 傳輸組態頁面 
選項
建立或編輯...
使用者認證
如果 FTP 伺服器的使用者是匿名使用者，請選取「匿名」，如果 FTP 伺服器的使用者是外部設定的帳號，請選取「外部使用者」。
識別 (電子郵件 ID)
只有在「使用者認證」選項設為「匿名」時，才可以使用此欄位。
輸入匿名使用者的郵件 ID。
服務帳號
只有在「使用者認證」選項設為「外部使用者」時，才可以使用此欄位。
輸入外部使用者的服務帳號。

有關詳細資訊：

  • 服務帳號
  • 新增服務帳號
逾時
輸入在捨棄連線之前的通訊端逾時間隔 (秒)。預設值為 60 秒。
目標檔案名稱的字首
輸入檔案名稱的字首，檔案以此名稱儲存在遠端伺服器上。
請勿在此欄位中輸入 *。此字元會造成執行時期的異常狀況。
目標檔案名稱的字尾
輸入檔案名稱的字尾，檔案以此名稱儲存在遠端伺服器上。
請勿在此欄位中輸入 *。此字元會造成執行時期的異常狀況。
傳輸模式
選取「ASCII」或「二進位」做為傳輸模式。
要求編碼
接受預設值 UTF-8 做為 FTP 傳輸中要求的字元集編碼，或輸入其他字元集編碼。

完成作業後

按「下一頁」繼續在下一個頁面上設定此服務，或按一下「最後一頁」，在建立/編輯業務服務 - 摘要頁面上檢查並儲存這項組態。
