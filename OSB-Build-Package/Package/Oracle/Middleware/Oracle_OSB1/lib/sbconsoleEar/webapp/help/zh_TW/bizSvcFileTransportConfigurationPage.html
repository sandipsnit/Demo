檔案傳輸組態頁面

「建立/編輯業務服務 - 檔案傳輸組態」頁面是一系列頁面中的其中一頁，供您建立和編輯業務服務 (如建立和設定業務服務及編輯業務服務組態中所述)。

使用此頁面，可讓您透過檔案傳輸通訊協定來設定業務服務的傳輸設定。表 18-7 描述如何使用此頁面。

表 18-7 建立/編輯業務服務 - 檔案傳輸組態頁面
選項
建立或編輯...
字首
輸入將加至檔案名稱前面的字首。
請勿在此欄位中輸入 *。此字元會造成執行時期的異常狀況。
字尾
輸入將加至檔案名稱後面的字尾。這是必要欄位。
請勿在此欄位中輸入 *。此字元會造成執行時期的異常狀況。
要求編碼
接受預設值 utf-8 做為檔案傳輸中要求的字元集編碼，或輸入其他字元集編碼。

完成作業後

按「下一頁」繼續在下一個頁面上設定此服務，或按一下「最後一頁」，在建立/編輯業務服務 - 摘要頁面上檢查並儲存這項組態。
