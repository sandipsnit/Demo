流程傳輸組態頁面

「建立/編輯業務服務 - 流程傳輸組態」頁面是一系列頁面中的其中一頁，供您建立和編輯業務服務 (如建立和設定業務服務和編輯業務服務組態中所述)。

您可使用流程傳輸類型業務服務來存取分割連接。若要進一步瞭解分割連接，請參閱「Oracle Fusion Middleware Developer's Guide for Oracle Service Bus」中的「Improving Service Performance with Split-Join」，網址為： http://www.oracle.com/pls/as111170/lookup?id=OSBAG291。

使用此頁面，可讓您透過流程傳輸通訊協定來設定業務服務的傳輸設定。表 18-8 說明如何使用此頁面。

表 18-8 建立/編輯業務服務 - 流量傳輸組態頁面

輸入等待回應的秒數。在僅要求 (單向) 作業中會忽略此值。

完成作業後

按「下一頁」繼續在下一個頁面上設定此服務，或按一下「最後一頁」，在建立/編輯業務服務 - 摘要頁面上檢查並儲存這項組態。
