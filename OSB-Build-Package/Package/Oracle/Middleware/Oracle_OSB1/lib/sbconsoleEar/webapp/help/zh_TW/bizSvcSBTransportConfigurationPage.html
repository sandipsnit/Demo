SB 傳輸組態頁面

「建立/編輯業務服務 - SB 傳輸組態」頁面是一系列頁面中的其中一頁，供您建立和編輯業務服務 (如建立和設定業務服務和編輯業務服務組態中所述)。

如需詳細資訊，請參閱「Oracle Fusion Middleware Developer's Guide for Oracle Service Bus」中的「SB Transport」，網址為：http://www.oracle.com/pls/as111170/lookup?id=OSBDV997。

使用此頁面，可讓您透過 SB (Service Bus) 傳輸通訊協定來設定業務服務的傳輸設定。表 18-15 說明如何使用此頁面。

表 18-15 建立/編輯業務服務 - SB 組態頁面
選項
建立或編輯...
分配原則
選取要針對此端點的分配原則所使用的 WebLogic Server 工作管理程式執行處理。如果沒有其他的工作管理程式，就會使用預設的工作管理程式。如需工作管理程式的相關資訊，請參閱 Oracle Fusion Middleware Configuring Server Environments for Oracle WebLogic Server 中的 "Using Work Managers to Optimize Scheduled Work"，網址為：http://www.oracle.com/pls/as111170/lookup?id=CNFGD112。
逾時
服務逾時的時間量 (秒)。
注意： 如果服務品質為「正好一次」，則會忽略逾時。
服務帳號
按一下瀏覽，然後從顯示的清單中選取服務帳號。若未指定服務帳號，則會使用匿名主題。

有關詳細資訊：

  • 服務帳號
  • 新增服務帳號

完成作業後

按「下一頁」繼續在下一個頁面上設定此服務，或按一下「最後一頁」，在建立/編輯業務服務 - 摘要頁面上檢查並儲存這項組態。
