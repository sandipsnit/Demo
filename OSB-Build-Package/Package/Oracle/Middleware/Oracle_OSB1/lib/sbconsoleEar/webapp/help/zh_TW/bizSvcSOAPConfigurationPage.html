SOAP 連結組態頁面

「建立/編輯業務服務 - SOAP 連結組態」頁面是一系列頁面中的其中一頁，供您建立和編輯業務服務 (如建立和設定業務服務和編輯業務服務組態中所述)。

使用此頁面，可讓您根據 WSDL，來設定業務服務的 SOAP 連結。

選取強制 WS-I 相容，指定服務是否要符合 Web Services Interoperability Organization 所定義的「基本設定檔」。此選項僅適用於 SOAP 1.1 服務

服務標示為 WS-I 相容時，會檢查從該服務傳送及接收的訊息。

完成作業後

按「下一頁」繼續在下一個頁面上設定此服務，或按一下「最後一頁」，在建立/編輯業務服務 - 摘要頁面上檢查並儲存這項組態。
