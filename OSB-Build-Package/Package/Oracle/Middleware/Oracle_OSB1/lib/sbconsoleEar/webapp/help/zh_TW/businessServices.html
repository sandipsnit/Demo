業務服務：建立和管理

業務服務是您想要與其交換訊息的企業服務之 Oracle Service Bus 定義。如同定義代理主機服務一樣，您可以使用 WSDL (Web Services Definition Language) 來定義業務服務。但是，業務服務組態和代理主機服務組態的不同之處在於業務服務沒有管線。因此，業務服務是未以 Oracle Service Bus 管線實行的任何服務。

如需詳細資訊，請參閱 Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus 中的 "Configuring Proxy Services and Business Services"，網址為： http://www.oracle.com/pls/as111170/lookup?id=OSBAG111。
