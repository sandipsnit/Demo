匯出與業務服務關聯的 WSDL

您可以匯出 WSDL 型業務服務的 WSDL，以便在 IDE 之類的外部工具中檢視或修改 WSDL。WSDL 會匯出為 JAR 檔案。

請注意，這和「系統管理」模組中的「匯出資源」功能不同，該功能用來在兩個網域間移動及計劃資源。請參閱匯出資源。

開始之前

只有不在階段作業內時，才能匯出 WSDL。請參閱使用變更中心。

若要匯出 WSDL

  1. 請執行下列任一項操作：
    • 選取「資源瀏覽器」>「業務服務」，以顯示「業務服務的摘要」頁面。
    • 選取專案瀏覽器，然後選取要匯出為 WSDL 的業務服務所在的專案或資料夾。隨即顯示專案/資料夾檢視畫面頁面。
  2. 在要匯出 WSDL 的業務服務列上按一下「匯出 WSDL」圖示，即會顯示對話方塊提示您開啟或儲存匯出的 JAR 檔案。
  3. 在此對話方塊中，按一下「開啟」以開啟檔案，或按一下「儲存」以儲存檔案。
