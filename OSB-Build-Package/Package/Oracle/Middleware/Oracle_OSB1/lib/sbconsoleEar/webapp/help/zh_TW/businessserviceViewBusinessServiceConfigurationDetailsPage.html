檢視業務服務 - 組態詳細資訊頁面

檢視業務服務 - 組態詳細資訊頁面會顯示業務服務的組態詳細資訊。 表 18-22說明此頁面上顯示的所有特性(特性會隨業務服務的詳細資訊而有所不同)。

此頁面所列的類別對應到用來建立和編輯業務服務組態的「建立/編輯業務服務」頁面，如建立/編輯業務服務 - 頁面參照中所說明。

按一下任一類別名稱旁的「編輯」連結，以顯示關聯的組態頁面。
