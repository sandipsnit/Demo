業務服務安全頁面

「檢視業務服務 - 安全組態」頁面是一系列頁面中的其中一頁，供您編輯業務服務 (如建立和設定業務服務和檢視業務服務 - 組態詳細資訊頁面中所述)。

在這個頁面中的欄位，取決於業務服務中所使用的原則。例如，如果服務使用 OWSM 原則 (建議)，「原則覆寫」會出現在「安全」頁面上。如需更多資訊，請參閱 Oracle Fusion Middleware Developer's Guide for Oracle Service Bus 中的 "Securing Oracle Service Bus with Oracle Web Services Manager"，網址為： http://www.oracle.com/pls/as111170/lookup?id=OSBDV1681。

連結至服務的 WLS 9.2 原則會都會公開 (內置) 於有效的 WSDL 中。抽象原則在內置之前會先經過預先處理。OWSM 原則由參照連結，不會內置至有效的 WSDL 中。

使用此頁面可以針對具有 WSDL 型原則或使用自訂原則連結的業務服務設定安全設定。表 18-24 會描述如何使用此頁面。
