清除階段作業啟動歷史記錄

您可以清除在特定期間啟動的階段作業 (以開始和結束日期分隔)。這個動作只能在階段作業之外執行。

注意： 清除階段作業啟動歷史記錄牽涉到刪除可啟用多層級還原的資料。換句話說，如果您清除特定期間的階段作業啟動歷史記錄，則將無法還原該期間內啟動的階段作業。
  1. 確定您不是在作用中的階段作業，然後按一下「檢視變更」。
  2. 在檢視組態變更頁面中，按一下依日期清除。依日期清除階段作業啟動歷史記錄頁面便會顯示。
  3. 在相對的欄位中，以 MM/DD/YYYY 格式輸入「開始」和「結束」日期。

    4. 會從 Oracle Service Bus 刪除指定日期之間的所有階段作業之階段作業啟動歷史記錄。
