還原工作

使用檢視組態變更頁面，可讓您在目前階段作業期間還原在 Oracle Service Bus 組態中所執行的作業，並在階段作業之外還原階段作業啟動。Oracle Service Bus 可讓您復原多層次的階段作業啟動，僅受限於您的系統資源。有關詳細資訊，請參閱還原工作的順序。

注意： 如果您從 Oracle Service Bus 2.1 版升級到 2.5 版，將無法還原升級之前啟動的階段作業。升級之後啟動的階段作業將會在階段作業啟動歷史記錄中，可進行還原。
還原階段作業中的工作
  1. 確定您在階段作業中。
  2. 按一下「變更中心」下方的「檢視變更」。
  3. 在特定工作的「選項」欄中，按一下「還原」圖示。

    4. 工作便會還原。

    您可以在目前的階段作業還原任何變更。但是，在變更之後，如果您改變心意，可以還原所做的還原，或者當您再度改變心意時，可以還原您還原前一個還原的還原，以此類推。

還原稍早啟動的階段作業
  1. 確定您在階段作業之外。
  2. 按一下「變更中心」下方的「檢視變更」。
  3. 請執行下列其中一項操作：
    • 在特定工作的「選項」欄中，按一下「還原」圖示。階段作業啟動便會還原。
    • 在特定工作的選項欄中，選取還原到階段作業中圖示。便會建立新的階段作業，並預先填入所需的工作以便還原在階段作業中啟動的變更。有關詳細資訊，請參閱還原到階段作業中。
