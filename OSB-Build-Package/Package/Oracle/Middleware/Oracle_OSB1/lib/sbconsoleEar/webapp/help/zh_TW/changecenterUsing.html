使用變更中心

變更中心模組是使用「Oracle Service Bus 管理主控台」變更組態的起始點。如需使用「Oracle Service Bus 管理主控台」進行組態變更，必須使用變更中心來啟動階段作業。

  1. 按一下編輯或建立以開始階段作業。階段作業的名稱會顯示在變更中心底下。
  2. 在「Oracle Service Bus 管理主控台」的相關頁面上做適當變更。
  3. 在您做變更的各個頁面上按一下「儲存」。您在目前階段作業中所做的所有變更均已儲存。

    4. 在階段作業執行期間隨時按一下「變更中心」下方的「捨棄」，以捨棄到目前為止在目前階段作業中所進行的變更。

  4. 完成變更後，請按一下變更中心下的啟動。繼續進行啟動階段作業。
