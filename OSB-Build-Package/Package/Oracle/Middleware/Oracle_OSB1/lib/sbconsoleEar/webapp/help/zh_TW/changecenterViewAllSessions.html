檢視所有階段作業

使用此頁面可檢視「Oracle Service Bus 管理主控台」內所有現有的階段作業。如果您目前在階段作業中或在階段作業之外，就可以檢視這些階段作業。

只有使用管理員角色時，才可以檢視所有執行階段。如需詳細資訊，請參閱 Oracle Fusion Middleware Developer's Guide for Oracle Service Bus 中的 "Configuring Administrative Security"，網址為： http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539。

  1. 按一下變更中心下方的檢視所有階段作業。針對每個現有的階段作業，檢視所有階段作業頁面會顯示表 2-5 所示的資訊。

    2. 表 2-5 階段作業詳細資訊
    特性
    描述
    階段作業名稱
    階段作業的名稱。此名稱是連到階段作業的連結。
    使用者
    建立階段作業的使用者名稱。
    建立時間
    建立階段作業的日期和時間。
    上次修改時間
    階段作業期間最後一次做變更的日期和時間。
    已修改的物件數
    已經在階段作業期間變更的物件數。

  2. 若要切換到其他階段作業，請按一下該階段作業在「階段作業名稱」欄中的連結。
注意： 不支援相同使用者以多個瀏覽器登入。它會造成「Oracle Service Bus 管理主控台」無法預期的行為。
