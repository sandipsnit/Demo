檢視作業詳細資訊

如果您在階段作業中，使用此頁面可讓您檢視在目前階段作業中所做的特定變更之詳細資訊，如果您在階段作業之外，則可以檢視在已經啟動的階段作業中所做的特定變更之詳細資訊。

  1. 按一下「變更中心」下方的「檢視變更」。
  2. 在檢視組態變更頁面上，按一下工作欄中的工作名稱。作業詳細資訊頁面會顯示表 2-4 所示的資訊。

    3. 表 2-4 作業詳細資訊
    特性
    描述
    工作
    執行的工作類型。
    • 建立
    • 更新
    • 刪除
    • 重新命名
    • 移動
    資源類型
    屬於工作一部分的 Oracle Service Bus 資源。
    資源
    資源的名稱和路徑。路徑是資源所在的專案和資料夾名稱。

  3. 按一下「上一頁」以返回「檢視組態變更」頁面。
