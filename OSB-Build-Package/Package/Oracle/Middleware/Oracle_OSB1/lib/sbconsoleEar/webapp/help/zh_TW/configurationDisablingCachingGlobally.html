停用全域結果快取

當您全域停用結果快取時，Oracle Service Bus 會清理整個快取，並針對所有啟用結果快取的業務服務移除快取的結果。預設設定會啟用結果快取。

在網域層級停用結果快取：

  1. 如果尚未進行這樣的處理，請按一下建立以建立新的階段作業，或按一下編輯以輸入現有的階段作業。請參閱使用變更中心。
  2. 選取「作業」>「全域設定」。
  3. 清除「啟用結果快取」核取方塊。
    4. 注意： 這個設定會覆寫任何在個別業務服務上設定的結果快取。
  4. 按一下更新。組態會在目前的階段作業儲存。
  5. 或者，在階段作業執行期間隨時按一下「重設」，以捨棄變更並以目前儲存的設定重新整理頁面。
  6. 若要結束階段作業並將組態建置至執行時期，請按一下「變更中心」下的「啟動」。
