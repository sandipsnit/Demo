停用全域服務監督

在網域層級關閉所有服務的監督：

注意： 如果您停用所有服務的監督，也會刪除到當時為止為那些服務收集的所有統計資訊。這些統計資訊無法使用階段作業還原功能來回復。您可以使用還原來重新啟用服務的監控，但是刪除的統計資訊則無法回復。
  1. 如果尚未進行這樣的處理，請按一下建立以建立新的階段作業，或按一下編輯以輸入現有的階段作業。請參閱使用變更中心。
  2. 選取「作業」>「全域設定」。
  3. 為系統清除「啟用監督」核取方塊以停止收集組態中所有服務的監控統計資訊。
    4. 注意： 此選項會覆寫您對特定商業和代理主機服務所選取的啟用服務監控選項。請參閱設定業務服務的作業設定與設定代理主機服務的作業設定。
    注意： 此選項也會覆寫啟用 SLA 警示選項。如果您在全域層級停用監控，則即使已選取對應的啟用 SLA 警示核取方塊，也會在執行時期停用 SLA 警示。請參閱啟用全域 SLA 警示。
  4. 按一下更新。監控組態會在目前的階段作業儲存。
  5. 或者，在階段作業執行期間隨時按一下「重設」，以捨棄變更並以目前儲存的設定重新整理頁面。
  6. 若要結束階段作業並將組態建置至執行時期，請按一下「變更中心」下的「啟動」。
