停用全域管線警示

在代理主機服務中，若要在網域層級關閉所有代理主機服務的管線警示：

  1. 如果尚未進行這樣的處理，請按一下建立以建立新的階段作業，或按一下編輯以輸入現有的階段作業。請參閱使用變更中心。
  2. 選取「作業」>「全域設定」。
  3. 清除「啟用管線警示」核取方塊，讓系統停止執行代理主機服務的任何管線警示動作。
    4. 注意： 此選項會覆寫您對特定代理主機服務所選取的管線警示設定。請參閱設定代理主機服務的作業設定。
  4. 按一下更新。管線警示組態會在目前的階段作業儲存。
  5. 或者，在階段作業執行期間隨時按一下「重設」，以捨棄變更並以目前儲存的設定重新整理頁面。
  6. 若要結束階段作業並將組態建置至執行時期，請按一下「變更中心」下的「啟動」。
