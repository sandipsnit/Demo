啟用全域結果快取

如果您叫用結果不常變更的業務服務，則結果快取會將快取的結果傳回至從屬端，而不用等待服務叫用的結果，如此便可改善業務服務效能。

在設定個別業務服務的結果快取之前，首先您必須利用下列步驟啟用全域的結果快取。預設設定會啟用結果快取。

  1. 如果尚未進行這樣的處理，請按一下建立以建立新的階段作業，或按一下編輯以輸入現有的階段作業。請參閱使用變更中心。
  2. 選取「作業」>「全域設定」。
  3. 選擇「啟用結果快取」核取方塊。
  4. 按一下「更新」儲存此頁上的所有設定。
  5. 或者，在階段作業執行期間隨時按一下「重設」，以捨棄變更並以目前儲存的設定重新整理頁面。
  6. 若要結束階段作業並將組態建置至執行時期，請按一下「變更中心」下的「啟動」。

如需設定業務服務的結果快取資訊，請參閱訊息處理組態頁面。
