啟用全域 SLA 警示

雖然可以個別設定 SLA 警示及監控，但這兩種設定會在執行時期互動。如果已啟用全域監督，就可以啟用或停用 SLA 警示。但如果停用全域監督，則也會強制停用 SLA 警示，因為 SLA 警示規則條件依存於正在評估的監控統計資訊。

在網域層級開啟所有服務的 SLA 警示：

  1. 如果尚未進行這樣的處理，請按一下建立以建立新的階段作業，或按一下編輯以輸入現有的階段作業。請參閱使用變更中心。
  2. 選取「作業」>「全域設定」。
  3. 選取「啟用 SLA 警示」核取方塊，讓系統開始評估組態中所有服務的警示規則。
  4. 按一下「更新」儲存此頁上的所有設定。
  5. 或者，在階段作業執行期間隨時按一下「重設」，以捨棄變更並以目前儲存的設定重新整理頁面。
  6. 若要結束階段作業並將組態建置至執行時期，請按一下「變更中心」下的「啟動」。
