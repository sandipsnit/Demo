啟動 Oracle Service Bus 管理主控台

  1. 使用下列其中一種方法啟動 Oracle Service Bus:
    • 從 Windows「開始」功能表，選取：

      • 開始 > 所有程式 > Oracle WebLogic > 使用者專案 > domain_name > 啟動 Oracle Service Bus 網域的伺服器。

      其中 domain_name 代表您建立 Oracle Service Bus 網域時所指派的名稱。「使用者專案」是建立網域的預設位置。

    • 從 Oracle Service Bus 根網域叫用 startWebLogic 命令：startWebLogic.cmd 或 startWebLogic.sh

      • 顯示的命令視窗含有 WebLogic Server 的狀態資訊。

      如果您在生產模式下啟動伺服器，必須輸入有效的使用者名稱和密碼才能啟動伺服器。

  2. 當伺服器啟動時，請在瀏覽器中輸入下列 URL：

    3. http://hostname:port/sbconsole

    其中 host 代表執行 WebLogic Server 的機器，port 則代表連接埠號碼。

    例如，如果 WebLogic Server 使用預設連接埠組態在您的本機機器上執行，請在瀏覽器上輸入下列 URL：

    http://localhost:7001/sbconsole

    顯示「Oracle Service Bus 管理主控台」登入頁面。

  3. 輸入您在安裝過程中指定的使用者名稱和密碼。
    4. 注意： Oracle Service Bus 範例網域的使用者名稱和密碼是 weblogic / welcome1。
  4. 按一下「登入」。

若要登出，請按一下靠近「Oracle Service Bus 管理主控台」右邊面板頂端之橫幅上的登出。
