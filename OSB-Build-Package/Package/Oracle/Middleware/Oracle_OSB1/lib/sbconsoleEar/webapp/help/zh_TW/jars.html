JAR

JAR (Java 封存) 是含有一組 Java 類型的壓縮檔案。它可用來儲存編譯後的 Java 類別和相關聯的描述資料，這些 Java 類別和描述資料可以組成程式。JAR 就像是一個可呼叫的 Java 程式碼元素程式庫 (這樣只需有單一編譯連結，就能提供多個元素的存取，而不需要個別建立每個元素的連結)。若要進一步瞭解 JAR 的相關資訊，請參閱 http://java.sun.com/docs/books/tutorial/deployment/jar/。

在 Oracle Service Bus 中，JAR 資源會用於：

  • Java 外部呼叫動作
  • EJB 型業務服務
  • Tuxedo 型代理和業務服務
