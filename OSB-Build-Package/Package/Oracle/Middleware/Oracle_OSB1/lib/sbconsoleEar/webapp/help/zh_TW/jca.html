JCA 連結

Oracle SOA Suite 讓您透過 J2EE 連接器架構 (JCA) 轉接器整合後端 Enterprise Information Systems (EIS)。利用 Oracle JDeveloper 及其它支援的 JCA 開發環境，您可設定轉接器及建立後端整合使用的 JCA 轉接器資源 (.jca 檔案及抽象 WSDL)。

您可以透過 Oracle Service Bus (OSB) 與 JCA 相容的 EIS 元件通訊，讓您能以 JCA 連結資源的形式使用 JCA 資源。您可以使用 JCA 連結，產生透過 JCA 轉接器利用 Oracle Service Bus JCA 傳輸與 EIS 應用程式溝通的代理主機及業務服務。

這個章節描述如何新增、編輯、尋找，及刪除在 Oracle Service Bus 中的 JCA 連結，包括下列主題：

  • 尋找 JCA 連結
  • 新增 JCA 連結
  • 編輯 JCA 連結
  • 刪除 JCA 連結

如需 JCA 傳輸的詳細資訊，請參閱 Oracle Fusion Middleware Developer's Guide for Oracle Service Bus 中的 "JCA Transport"，網址為： http://www.oracle.com/pls/as111170/lookup?id=OSBDV910。

如需更多在 Oracle SOA Suite JCA 連接器的資訊，請參閱 Oracle Fusion Middleware User's Guide for Technology Adapters，網址為：http://www.oracle.com/pls/as111170/lookup?id=TKADP。
