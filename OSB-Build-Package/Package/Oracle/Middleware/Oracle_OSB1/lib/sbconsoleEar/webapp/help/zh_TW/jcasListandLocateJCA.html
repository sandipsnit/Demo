尋找 JCA 連結

JCA 連結的摘要頁面列出您已經新增到 Oracle Service Bus 網域的 JCA 連結。

  1. 請執行下列任一項操作：
    • 選取專案瀏覽器以顯示專案檢視畫面頁面或專案/資料夾檢視畫面頁面。然後瀏覽專案和資料夾，以尋找 JCA 連結。
    • 選取資源瀏覽器 > JCA 連結。JCA 連結的摘要頁面顯示在表 12-1 中所示的資訊。有關特性的更詳細描述，請參閱尋找 JCA 連結。
  2. 尋找特定的 JCA 連結：
    • 若要限制在清單中項目的數量，您可以在按下「搜尋」之前以名稱及路徑篩選。

      • 路徑是專案名稱和資源所在資料夾的名稱。

    • 按一下「檢視全部」移除搜尋篩選，並顯示所有 JCA 連結。

      • 表 12-1 JCA 連結資訊
      特性
      描述
      JCA 連結名稱
      指派到 JCA 連結的唯一名稱。按一下名稱以查看檢視 JCA 連結詳細資訊頁面。請參閱編輯 JCA 連結。
      路徑
      專案名稱和 JCA 連結所在資料夾的名稱。按一下名稱以查看包含此資源的專案或資料夾。請參閱「使用專案和資料夾限定資源名稱」。
      動作
      包含讓您從 JCA 連結產生 Oracle Service Bus WSDL 及 JCA 服務的「產生」圖示。
      選項
      包含刪除圖示。如果業務服務或代理主機服務已經設定為使用 JCA 連結，刪除警告圖示指出您可以在確認警告後刪除 JCA 連結。因為仍有從服務至已刪除 JCA 連結的未解決參照，所以可能會導致衝突。請參閱刪除 JCA 連結。
