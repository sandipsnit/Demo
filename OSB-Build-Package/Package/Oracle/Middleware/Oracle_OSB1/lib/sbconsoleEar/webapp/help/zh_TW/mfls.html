MFL

「訊息格式語言」(MFL) 文件是用來說明二進位資料配置的特殊 XML 文件。它是 Oracle 專用語言，用來定義將格式化二進位資料轉換為 XML 資料的規則。MFL 文件符合 mfl.dtd，此檔案包含用於說明資料的每個欄位及欄位分組 (群組)、重複及彙總的元素和屬性。

在建立「訊息傳遞服務」類型的業務服務或代理主機服務時，您可以選取 MFL 類型做為服務的要求訊息類型或回應訊息類型。

您可以使用 Oracle Format Builder 來建立 MFL。當您定義二進位記錄的階層、欄位配置及欄位和群組的分組時，資訊會儲存為可用來進行執行時期轉譯的 MFL 文件。Format Builder 也可以使用 MFL 文件來產生能描述內容模型的對應 DTD。若想瞭解如何使用 Format Builder，請參閱 Oracle Fusion Middleware Developer's Guide for Oracle Service Bus 中的 "Format Builder"，網址為： http://www.oracle.com/pls/as111170/lookup?id=OSBDV763。
