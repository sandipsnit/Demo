監督

當建立商業或代理主機服務時，依預設會停用該服務的監督。若要瞭解如何啟用代理主機服務的監督，請參閱設定代理主機服務的作業設定；若為業務服務，請參閱設定業務服務的作業設定。

若要啟用或停用所有已個別啟用或停用監控之服務的監控功能，請使用作業 > 全域設定頁面上的啟用監控選項。請參閱啟用全域設定。

若要進一步瞭解如何在執行時期進行監督，請參閱 Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus」中的「Monitoring Oracle Service Bus at Run Time」，網址為： http://www.oracle.com/pls/as111170/lookup?id=OSBAG472。
