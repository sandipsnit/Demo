自訂表格檢視畫面

使用表格自訂器可以根據所指定的設定來顯示表格資訊。

請注意，任何表格的預設排序順序都是由表格的第一欄所決定。

  1. 按一下「表格自訂器」圖示，即會顯示其他欄位。

    2. 您可以隨時按一下「取消」，關閉此表格並保留原始設定。

  2. 在「欄」欄位中，選取您要顯示的欄：
    1. 從「可用的欄」欄位中選取欄名稱。
    2. 按一下箭頭，將此欄名稱移至「選取的欄」欄位。

      3. 星號表示必要欄。

    3. 一直重複此動作，直到「選取的欄」欄位中列出您要顯示的所有欄名稱為止。

      4. 在「選取的欄」欄位中，您可以依需要利用「向上鍵」和「向下鍵」來重新排序欄名稱。

  3. 若不要顯示某幾欄，請將「選取的欄」欄位中的欄名稱移至「可用欄」欄位。
  4. 從「列數」下拉清單中，選取要在單一頁面顯示的列數。
  5. 請執行下列其中一項操作：
    • 若要儲存這些設定並關閉表格自訂器，請按一下「套用」。
    • 若要捨棄變更並保留原始設定，請按一下「重設」。
