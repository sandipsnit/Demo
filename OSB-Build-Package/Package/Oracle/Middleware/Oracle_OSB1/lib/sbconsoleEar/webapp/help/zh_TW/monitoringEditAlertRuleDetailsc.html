編輯警示規則

  1. 如果尚未進行這樣的處理，請按一下「建立」以建立新的階段作業，或按一下「編輯」以輸入現有的階段作業。請參閱使用變更中心。
  2. 尋找警示規則。請參閱尋找警示規則。
  3. 按一下警示規則名稱。
  4. 若要變更組態頁面上的欄位，請按一下適當頁面的「編輯」圖示，或頁面底端的「編輯」按鈕。有關頁面和欄位的描述，請參閱建立和編輯警示規則。
  5. 請執行下列任何操作：
    • 若要前往下一頁，請按「下一頁」。
    • 若要返回上一頁，請按一下「上一頁」。
    • 若要前往摘要頁，請按一下「最後一頁」。
    • 若要提交目前階段作業中的更新，請按一下「儲存」。
    • 若要捨棄變更並返回「SLA 警示規則」頁面，請按一下「取消」。
  6. 若要結束階段作業並將設定部署至執行時期，請按一下「變更中心」下的「啟動」。
