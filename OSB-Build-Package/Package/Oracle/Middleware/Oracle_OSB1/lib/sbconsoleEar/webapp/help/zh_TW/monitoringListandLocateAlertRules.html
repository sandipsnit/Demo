檢視警示規則

使用此頁面可以檢視表 25-19 中所示的警示規則資訊。

表 25-19 警示規則資訊
特性
描述
名稱
指派給此警示規則的名稱。此名稱是連到警示規則一般組態頁面的連結。請參閱檢視警示規則組態。
SLA 狀態
警示規則狀態：「已啟用」或「已停用」。
描述
此欄位預設為隱藏。
警示規則的描述。
嚴重程度
此規則將觸發的警示的嚴重程度：正常、警告、次要、主要、臨界或嚴重。
彙總間隔
以小時及分鐘表示的彙總間隔長度。
到期日
此警示規則不再有效的日期。
停止處理
顯示「是」或「否」。
頻率
此警示的頻率：
  • 每次
  • 通知一次
選項
此欄包括下列功能：
  • 按一下箭頭，以重新排序警示規則。您可以在清單中上下移動個別警示規則。只有在對此服務設定多個警示規則時，才能這樣做。
  • 按一下「重新命名」圖示，以重新命名警示規則。
  • 按一下刪除圖示，以刪除特定警示規則。請參閱刪除警示規則。
