尋找警示

  1. 選取「作業」>「儀表板」。
  2. 選取「SLA 警示」或「管線警示」頁籤，以顯示 SLA 或管線警示。
  3. 按一下延伸警示歷史記錄連結。延伸 SLA 警示歷史記錄和延伸管線警示歷史記錄頁面會顯示詳細的警示歷史記錄資訊。
  4. 若要限制清單中的項目數或尋找特定警示：
    • 按一下開啟圖示，以顯示其他搜尋篩選。如果指定了多個搜尋條件，則只會傳回符合全部條件的結果。
    • 依日期範圍篩選。請執行下列其中一項操作：
      • 選取「全部」以忽略日期範圍和時間間隔篩選條件，並顯示所有警示。
      • 在開始時間和結束時間欄位中，以 MM/DD/YY HH:MM AM|PM 格式輸入日期與時間，然後按一下搜尋。將顯示在指定日期及時間範圍發出的警示。
      • 在針對最近欄位中輸入天數，然後指定小時和分鐘，再按一下搜尋。將顯示在指定時間範圍發出的警示。
    • 依警示嚴重程度篩選。從「警示嚴重程度」下拉清單中，選取將警示限制在指定的嚴重度層級，然後按一下「搜尋」。

      • 選取「或以上」核取方塊，可將搜尋限制在指定的嚴重度層級或更高層級 (從最寬鬆到最受限層級依序排列)：正常、警告、次要、臨界及嚴重。

    • 依父項服務篩選。按一下瀏覽顯示選取服務頁面。從清單中選取父項服務，然後按一下提交。按一下清除移除父項服務篩選條件，或按一下搜尋。
    • 僅限 SLA 警示，依警示名稱 (任何字串)篩選。輸入任何字串，並在其中包括萬用字元，然後按一下搜尋。如需詳細資訊，請參閱 WebLogic 診斷架構查詢語言。
    • 僅限管線警示，依警示摘要 (任何字串) 篩選。輸入任何字串，並在其中包括萬用字元，然後按一下搜尋。如需詳細資訊，請參閱 WebLogic 診斷架構查詢語言。若要尋找不包含警示摘要文字的管線警示動作，請輸入 Oracle Service Bus Alert。
    • 按一下「檢視全部」移除搜尋篩選，並顯示所有警示。

使用「延伸 SLA 警示歷史記錄」和「延伸管線警示歷史記錄」頁面也可以執行下列操作：

  • 若要以圓餅圖顯示警示資料，請選取「圓餅圖檢視」。
  • 若要以直條圖顯示相同的資料，請選取「直條圖檢視」。
  • 若要以表格格式顯示資料，請選取「表格檢視」。
  • 若要顯示已更新的資料版本，請按一下「重新整理」。
  • 若要刪除個別警示，請選取對應的核取方塊，並按一下頁面底端的「刪除」。
  • 若要刪除所有警示或在所指定日期和時間範圍內發生的警示，請按一下頁面頂端的清除警示歷史記錄連結。請參閱清除警示。

WebLogic 診斷架構查詢語言

警示使用 WebLogic 診斷架構進行儲存，它會提供自己的查詢語言 (包括萬用字元在內)。若要篩選延伸警示歷史記錄中的警示，請使用 Oracle Fusion Middleware Configuring and Using the Diagnostics Framework for Oracle WebLogic Server 中的 "WLDF Query Language"，網址為： http://www.oracle.com/pls/as111170/lookup?id=WLDFC316。
