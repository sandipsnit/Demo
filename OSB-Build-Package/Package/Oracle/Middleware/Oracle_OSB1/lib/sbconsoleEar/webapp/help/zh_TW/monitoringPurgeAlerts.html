清除警示

使用此頁面可以永久刪除指定的 SLA 或管線警示。

注意： 此動作無法還原。
  1. 選取「作業」>「儀表板」。
  2. 選取「SLA 警示」或「管線警示」頁籤，然後按一下「延伸警示歷史記錄」連結。
  3. 在「延伸警示歷史記錄」頁面的左上方，按一下「清除 SLA 警示歷史記錄」或「清除 Pipeline 警示歷史記錄」。
  4. 執行下列其中一個步驟，如表 25-3 中所示。

    5. 表 25-3 警示清除選項 
    若要...
    請完成這些步驟...
    • 清除所有 SLA 警示
    • 清除所有管線警示
    1. 選取「清除所有警示」。
    2. 按一下「清除」。
    • 清除所指定時段內的 SLA 警示
    • 清除所指定時段內的管線警示
    1. 選取「清除 xxx 警示來源... 目標 ...」
    2. 在「清除自」欄位中，輸入月、日、年和時間。
    3. 在「清除到」欄位中，輸入月、日、年和時間。
    4. 按一下「清除」。

注意： 如果要整個清除警示的伺服器無法使用，記錄日誌架構會保存清除的記錄，且每 30 分鐘輪詢伺服器以檢查使用狀態。當記錄日誌架構發現伺服器可以使用，則會清除警示。
