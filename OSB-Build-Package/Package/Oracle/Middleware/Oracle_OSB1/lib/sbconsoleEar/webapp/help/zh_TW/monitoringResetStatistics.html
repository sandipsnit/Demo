重設服務的統計資訊

您可以使用服務狀況頁面，重設您組態中所有服務的監控統計資訊。不管您是否在階段作業中，都可以重設統計資訊。

  1. 選取「作業」>「儀表板」>「服務狀況」。
  2. 在顯示統計資訊欄位中，選取上次重設之後。「上次重設之後」檢視畫面顯示測量結果的執行中計數檢視畫面。
  3. 若要重設特定服務的監控統計資訊，請按一下該服務「動作」欄中的「重設」圖示。

    4. 當您確認要這樣做時，系統會刪除上次重設統計資訊之後，為服務收集的所有監控統計資訊。不過，系統並不會刪除在服務的目前的彙總間隔期間所收集的統計資訊。此外，在您按一下重設圖示之後，系統會立即為服務重新開始收集監控統計資訊。

  4. 若要重設所有服務的監控統計資訊，請按一下頁面頂端的「重設所有統計資訊」連結。
    5. 注意： 按一下「重設所有統計資訊」，會重設所有受監督服務的統計資訊，而不論其是否顯示在頁面上。

    當您確認要重設統計資訊時，系統會在您的設定中刪除上次重設統計資訊之後，為所有服務收集的所有監控統計資訊。系統不會刪除在任一服務的目前的彙總間隔期間所收集的統計資訊。此外，在您按一下重設所有統計資訊之後，系統會立即為服務重新開始收集監控統計資訊。
