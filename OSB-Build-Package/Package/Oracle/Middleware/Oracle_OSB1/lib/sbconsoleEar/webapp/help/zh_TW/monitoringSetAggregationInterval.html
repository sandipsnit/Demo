設定服務的彙總間隔

使用作業設定頁面可以設定特定服務的彙總間隔。彙總間隔是指計算並在「Oracle Service Bus 管理主控台」顯示彙總統計資訊的期間。預設彙總間隔設定為 10 分鐘。

  1. 如果尚未進行這樣的處理，請按一下建立以建立新的階段作業，或按一下編輯以輸入現有的階段作業。請參閱使用變更中心。
  2. 選取「資源瀏覽器」>「業務服務」或「代理主機服務」。
  3. 按一下服務名稱。
  4. 選取「作業設定」頁籤。
  5. 在「彙總間隔」欄位中，以小時和分鐘來選取彙總間隔的長度，然後按一下「更新」。

    6. 如果選取的時數超過 1，則預設的分鐘選取一律為零。但如果選取的時數為 0 或 1，則可設定以分鐘表示的間隔。

  6. 按一下「更新」，確認目前階段作業中的更新。
  7. 若要結束階段作業並將組態建置至執行時期，請按一下「變更中心」下的「啟動」。
