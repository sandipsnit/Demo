檢視動作量測

使用此頁面可以檢視服務監控資訊 (如表 25-10 中所示)，以及具有管線之代理主機服務的動作測量結果 (如表 25-14 中所示)。

表 25-14 動作測量結果
特性
描述
節點
巢狀動作的階層顯示。
計數
如果已在「顯示統計資訊」欄位中選取「目前的彙總間隔」，則這是在目前彙總間隔期間內的叫用數目。
如果已在「顯示統計資訊」欄位中選取「上次重設之後」，則這是您自上次重設統計資訊至今的期間內的叫用數目。
請注意，這與自每個訊息可以叫用多次動作後的訊息數目不同。
平均時間
如果已在「顯示統計資訊」欄位中選取「目前的彙總間隔」，則為目前彙總間隔期間內每個叫用的平均執行時間。
如果已在「顯示統計資訊」欄位中選取「上次重設之後」，則為自上次重設統計資訊至今之期間內每個叫用的平均執行時間。
總時間
如果已在「顯示統計資訊」欄位中選取「目前的彙總間隔」，則為目前彙總間隔期間內的總執行時間。
如果已在「顯示統計資訊」欄位中選取「上次重設之後」，則為自上次重設統計資訊至今之期間內的總執行時間。
