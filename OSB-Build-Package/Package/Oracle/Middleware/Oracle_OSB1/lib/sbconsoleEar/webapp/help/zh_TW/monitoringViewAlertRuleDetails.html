檢視警示規則組態

使用此頁面可以檢視警示規則組態詳細資訊 (如表 25-22 中所示)。

表 25-22 警示規則 - 一般組態
特性
描述
規則名稱
警示規則的名稱。
規則描述
警示規則的描述。
警示摘要
警示規則的簡短描述。如果此警示規則設有電子郵件警示通知，則也會在電子郵件訊息的主旨行使用此文字。
警示目標
警示目標資源 (包含警示通知的目標位址) 的路徑和名稱。此欄位是連到檢視警示目標 - 一般組態頁面的連結。請參閱編輯警示目標。
開始時間 (HH:MM)
此警示規則的開始時間。
結束時間 (HH:MM)
此警示規則的結束時間。
規則到期日 (MM/DD/YYYY)
不再套用此警示規則的日期。
已啟用規則
是否啟用此警示規則。
警示嚴重程度
由於此警示規則而產生的警示嚴重程度：
  • 正常
  • 警告
  • 次要
  • 主要
  • 臨界
  • 嚴重
警示頻率
警示的頻率：
  • 每次
  • 通知一次
停止處理其他規則
警示是否停止處理其他規則。
