檢視 SLA 警示

使用 SLA 警示頁面可檢視 SLA 警示的摘要，並存取詳細的警示歷史記錄資訊 (如表 25-1 中所示)。

「SLA 警示」頁面包含下列項目：

  • 顯示「管線警示」、「服務狀況」和「伺服器狀況」資訊的頁籤。

    • 在 SLA 警示頁籤標頭上，數值指示器會顯示自上次檢視之後或自上次「儀表板」重新整理期間 (視何者較新) 發生的新警示數目。「儀表板重新整理速率」預設為不重新整理。若要變更「儀表板重新整理速率」間隔，請參閱設定使用者偏好設定。

注意： 選取預設值 (不重新整理) 之外的儀表板重新整理速率間隔，瀏覽器就會定期重新整理「儀表板」。這可防止階段作業逾時，即使您未與「Oracle Service Bus 管理主控台」互動也一樣。
注意： Oracle 建議您小心使用「儀表板重新整理速率」功能。請不要將階段作業保留為自動。當您的階段作業在作用中時，其他人就可以從您的機器存取該階段作業，而不需要重新進行認證。
  • 過去 30 分鐘 (預設警示歷史記錄期間間隔) 內具有 SLA 警示之服務的圓餅圖。若要變更用來顯示歷史警示資料的時間間隔，請參閱設定使用者偏好設定。

    • 頁面上的 SLA 警示數目會符合圓餅圖，而此圓餅圖是依嚴重度層級顯示相同時間間隔的警示細分。您可以按一下圖表中對應於警示嚴重程度的區域，以顯示其他詳細資訊。

  • 具有管線警示的前 10 名服務會依這些服務所發出的警示數，以遞減次序列出。每一個服務名稱都是連到服務監督詳細資訊的連結。請參閱檢視服務測量結果。
  • 顯示警示資訊的欄位，是根據警示表格的自訂設定而定。例如，時間戳記、警示名稱、警示嚴重程度、動作，以及與警示相關的服務和服務類型名稱。若要進一步瞭解這些欄位，請參閱表 25-1。

    • 警示名稱旁邊的註解圖示表示註解。將滑鼠指標移至圖示上即可預覽註解文字。按一下此圖示，可檢視整段註解文字。

  • 表格自訂器圖示。請參閱自訂表格檢視畫面。
  • 「延伸警示歷史記錄」頁面的連結。請參閱尋找警示。

    • 表 25-1 SLA 警示歷史記錄資訊
    特性
    描述
    時間戳記
    警示規則上次評估為 true 的日期和時間。
    預設會根據時間戳記來排序所有警示。
    警示名稱
    指定給警示的名稱。此名稱是連到警示詳細資訊頁面的連結。請參閱檢視警示詳細資訊。
    警示名稱旁邊的註解圖示表示註解。將滑鼠指標移至圖示上即可預覽註解文字。
    警示嚴重程度
    使用者定義的警示嚴重程度。
    • 嚴重
    • 臨界
    • 主要
    • 次要
    • 警告
    • 正常
    服務
    警示相關的服務名稱。此名稱是連到服務監控詳細資訊頁面的連結 (僅限正在監控服務時)。請參閱檢視服務測量結果。
    服務類型
    父項服務類型：代理主機服務或業務服務。
    動作
    檢視警示規則詳細資訊圖示是警示規則一般組態頁面的連結。請參閱檢視警示規則組態。
