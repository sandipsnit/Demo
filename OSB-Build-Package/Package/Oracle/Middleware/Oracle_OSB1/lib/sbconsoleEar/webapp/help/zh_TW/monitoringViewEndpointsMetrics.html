狀態
業務服務端點 URI 的狀態。
適用於單一伺服器網域或是叢集中的個別受管理伺服器：「線上」或「離線」。
適用於叢集層級統計資訊：「線上」、「離線」或「部分」(代表特定 URI 在叢集中至少一台受管理伺服器上是離線狀態)。
動作
按一下將端點標示為線上圖示，將離線 URI 標示為「線上」。只有當 URI 的狀態為「離線」或「部分」時，才會啟用此動作。
若要啟用此動作，則業務服務必須啟用以讓 URI 離線，而且必須啟用它的服務和全域層級監督作業設定。請參閱設定業務服務的作業設定。
注意： 如果成功，則會立即進行動作。然而，反映其端點 URI 狀態為現在上線，需要一個監督彙總循環，這可能只要 1 分半的時間。
