檢視服務測量結果

使用此頁面可以檢視特定服務的監控詳細資訊 (如表 25-10 中所示)，以及服務測量結果 (如表 25-11 中所示)。

表 25-10 服務監控資訊
特性
描述
服務名稱
服務的名稱與路徑。
服務類型
父項服務類型：代理主機服務或業務服務。
顯示統計資訊
若要顯示服務在目前彙總間隔期間的監控統計資訊，請選取目前的彙總間隔。「目前彙總間隔」檢視畫面會顯示服務測量結果的動態統計檢視畫面。
若要顯示此服務自您上次重設統計資訊至今期間內的監控統計資訊，請選取上次重設之後。「上次重設之後」檢視畫面顯示測量結果的執行中計數檢視畫面。
伺服器
從下拉清單中選取伺服器，以顯示該伺服器的量測。
如果叢集存在，預設會顯示叢集範圍的量測。選取個別的「受管理伺服器」以顯示該伺服器的量測。
如果顯示叢集中個別「受管理伺服器」的量測，則「SLA 警示計數」會是 N/A (僅顯示叢集範圍的 SLA 警示計數)。
彙總間隔
如果在顯示統計資訊欄位中選取目前的彙總間隔，則此為監控此服務的目前彙總間隔，以時和分為單位。您可以在作業設定頁面中設定此間隔。請參閱設定服務的彙總間隔。
重設統計資訊
如果在顯示統計資訊欄位中選取上次重設之後，則可以重設此服務的監控統計資訊。按一下頁面底端的重設統計資訊。

SLA 警示計數
  • 警示計數取決於「目前的彙總間隔」或「上次重設之後」期間的「顯示統計資訊」檢視畫面。

管線警示計數

  • 管線警示計數取決於「目前的彙總間隔」或「上次重設之後」期間的「顯示統計資訊」檢視畫面。

  • 目前的彙總間隔中，最嚴重警示的警示嚴重程度。如果有多個警示，會顯示目前彙總間隔中最高的警示嚴重程度。如果目前彙總間隔內沒有管線警示，會顯示 NORMAL。
  • 時間戳記會顯示上次輪詢伺服器是否有管線警示資訊的時間，並顯示上次輪詢伺服器當時的警示狀態。如果上次輪詢伺服器時並沒有警示，會顯示 NORMAL 和輪詢時間。
最短回應時間
如果在「顯示統計資訊」欄位中選取「目前的彙總間隔」，這是此服務在目前彙總間隔期間內執行訊息所花的時間。
如果在「顯示統計資訊」欄位中選取「上次重設之後」，這是此服務在您上次重設統計資訊至今的期間內執行訊息所花的最短回應時間。
最長回應時間
如果在「顯示統計資訊」欄位中選取「目前的彙總間隔」，這是此服務在目前彙總間隔期間內執行訊息所花的最長回應時間。
如果在「顯示統計資訊」欄位中選取「上次重設之後」，這是此服務在您上次重設統計資訊至今的期間內執行訊息所花的最長回應時間。
整體平均回應時間
如果在「顯示統計資訊」欄位中選取了「目前的彙總間隔」，這是此服務在目前彙總間隔期間內執行訊息所花的整體平均時間 (毫秒)。
如果在「顯示統計資訊」欄位中選取「上次重設之後」，這是此服務在您上次重設統計資訊至今的期間內執行訊息所花的整體平均時間。
訊息計數
如果在「顯示統計資訊」欄位中選取「目前的彙總間隔」，這是此服務在目前彙總間隔期間內執行的訊息總數。
如果在「顯示統計資訊」欄位中選取「上次重設之後」，這是此服務在您上次重設統計資訊至今的期間內執行的訊息總數。
錯誤計數
如果在「顯示統計資訊」欄位中選取「目前的彙總間隔」，這是此服務在目前彙總間隔期間內已執行的錯誤訊息數。
如果在「顯示統計資訊」欄位中選取「上次重設之後」，這是此服務在您上次重設統計資訊至今的期間內執行的錯誤訊息數。
容錯移轉計數
僅適用於業務服務：
如果在「顯示統計資訊」欄位中選取「目前的彙總間隔」，這是此服務在目前彙總間隔期間內已嘗試的容錯移轉次數。
如果在「顯示統計資訊」欄位中選取「上次重設之後」，這是此服務在您上次重設統計資訊至今的這段期間內已嘗試的容錯移轉次數。
成功率
如果在「顯示統計資訊」欄位中選取「目前的彙總間隔」，這是此服務在目前彙總間隔期間內的成功率。
如果在「顯示統計資訊」欄位中選取「上次重設之後」，這是此服務在您上次重設統計資訊至今的期間內的成功率。
例如，服務執行的 10 個訊息中若有 9 個成功，成功率就是 90%。
失敗率
如果在「顯示統計資訊」欄位中選取「目前的彙總間隔」，這是此服務在目前彙總間隔期間內的失敗率。

如果在「顯示統計資訊」欄位中選取「目前的彙總間隔」，這是此服務在目前彙總間隔期間內執行的 WS 安全錯誤訊息數。
如果在「顯示統計資訊」欄位中選取「上次重設之後」，這是此服務在您上次重設統計資訊至今的期間內執行的 WS 安全錯誤訊息數。
