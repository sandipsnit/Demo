MQ 連線

注意： 請參閱 Oracle Fusion Middleware Supported System Configurations，網址為： http://www.oracle.com/technology/software/products/ias/files/fusion_certification.html，瞭解 Oracle Service Bus 支援的 WebSphere MQ 版本。WebSphere MQ 從屬端程式庫未與 Oracle Service Bus 安裝程式組合在一起。請從您的 WebSphere MQ 安裝取得 WebSphere MQ 從屬端程式庫。

MQ 連線是可共用的資源，可在多個 MQ 代理主機和業務服務中重複使用。MQ 代理主機和業務服務必須先連線至 MQ 佇列管理程式，再存取 MQ 佇列。MQ 連線資源提供連線至 MQ 佇列管理程式所需的連線參數。

每個 MQ 連線資源都有一個連線集區。每個使用指定 MQ 連線資源連線指定佇列管理程式的商業或代理主機服務，都會使用針對該資源建立的相同連線集區。因此，多個使用相同佇列管理程式的業務服務和代理主機服務會共用一個連線集區。

若要進一步瞭解 Oracle Service Bus MQ 連線資源和原生 MQ 傳輸，請參閱「Oracle Fusion Middleware Developer's Guide for Oracle Service Bus」，網址為： http://www.oracle.com/pls/as111170/lookup?id=OSBDV1117。

若要進一步瞭解 WebSphere MQ 基礎架構的相關資訊，請參閱 http://www.redbooks.ibm.com/redbooks/SG247128/wwhelp/wwhimpl/java/html/wwhelp.htm。
