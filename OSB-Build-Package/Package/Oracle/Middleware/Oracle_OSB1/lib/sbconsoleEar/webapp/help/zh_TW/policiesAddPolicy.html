新增自訂 WS 原則

您可以匯入包含「WS 原則」的 XML 檔案，或直接在「Oracle Service Bus 管理主控台」寫入「WS 原則」。

  1. 如果尚未進行這樣的處理，請按一下建立以建立新的階段作業，或按一下編輯以輸入現有的階段作業。請參閱使用變更中心。
  2. 選取專案瀏覽器，然後選取要在其中新增「WS 原則」的專案或資料夾。隨即顯示專案/資料夾檢視畫面頁面。
  3. 從「建立資源」下拉清單的「介面」中選取「WS 原則」，以顯示「建立新的 WS 原則」頁面。
  4. 在資源名稱欄位中，輸入此「WS 原則」的唯一名稱。

    請遵循資源命名限制的命名指南。
  5. 在「資源描述」欄位中，輸入此「WS 原則」的描述。
  6. 在「原則」欄位中，請執行下列其中一項操作：
    • 按一下「瀏覽」以尋找並匯入包含自訂 WS 原則的 XML 檔案。
    • 輸入新 WS 原則的文字。
    • 從現有的 WS 原則中複製文字，並貼入此欄位。
  7. 按一下「儲存」。「WS 原則」會儲存在目前階段作業中。
  8. 若要結束階段作業並將組態建置至執行時期，請按一下「變更中心」下的「啟動」。
