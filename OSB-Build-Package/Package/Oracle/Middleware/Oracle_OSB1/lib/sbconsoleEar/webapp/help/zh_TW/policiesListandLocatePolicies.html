尋找自訂 WS 原則

WS-Policy 的摘要頁面會列出您已經新增至目前 Oracle Service Bus 網域的自訂「Web 服務原則」(WS-Policy)。「Oracle Service Bus 管理主控台」不會顯示 Oracle Service Bus WS-Policy，因為它們已預先包裝，可供所有 Oracle Service Bus 網域使用。

  1. 請執行下列任一項操作：
    • 選取專案瀏覽器以顯示專案檢視畫面頁面或專案/資料夾檢視畫面頁面。然後瀏覽專案和資料夾，以尋找 WS 原則。
    • 選擇資源瀏覽器 > WS 原則。WS 原則的摘要頁面會顯示表 11-1 中所示的資訊。有關特性的更詳細描述，請參閱尋找自訂 WS 原則。
  2. 若要搜尋自訂「WS 原則」，請在名稱欄位中輸入部分或全部的「WS 原則」名稱。您也可以在路徑欄位中輸入部分或全部的「WS 原則」專案名稱和資料夾。按一下搜尋。

    3. 按一下「檢視全部」移除搜尋篩選，並顯示所有「WS 原則」。

    表 11-1 WS 原則資訊 
    特性
    描述
    WS 原則名稱
    指定給「WS 原則」的唯一名稱。按一下名稱即可查看檢視 WS 原則詳細資訊頁面。請參閱編輯自訂 WS 原則。
    路徑
    WS 原則是專案名稱和 WS 原則所在資料夾的名稱。按一下名稱，以查看包含此資源的專案或資料夾。請參閱使用專案和資料夾限定資源名稱。
    選項
    包含刪除圖示。如果業務服務或代理主機服務已設定成使用 WS 原則，則刪除警告圖示指出您可以確認警告來刪除 WS 原則。因為仍有服務參照已刪除 WS 原則的未解析參照，所以可能會導致衝突。請參閱刪除自訂 WS 原則。
