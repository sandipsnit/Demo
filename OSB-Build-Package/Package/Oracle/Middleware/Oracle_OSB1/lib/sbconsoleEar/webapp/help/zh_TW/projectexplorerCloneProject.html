複製專案

有三種方式可以複製專案：

  • 在根層級使用新名稱來複製專案。在此情況下，原始專案和其複製是以對等專案形式存在。複製的專案會含有與原始專案相同的資料夾和資源 (至少在修改來源或複製之前)。有關指示，請參閱在根層級複製專案。
  • 使用現有專案的名稱 (非來源專案) 來複製專案。在此情況下，來源專案的資料夾和資源會與現有專案的資料夾和資源合併，而現有專案的名稱已指派給複製。原始專案會保持原狀。有關指示，請參閱在根層級複製專案。
  • 在現有專案或專案資料夾的新位置中複製專案。在此情況下，會將複製的專案下移至新位置的資料夾中。而此專案含有與原始專案相同的資料夾和資源。

    • 如果複製的專案與現有資料夾同名且位在相同位置中，則專案的內容 (子資料夾和資源) 會與目標資料夾的內容合併。

    在任一種情況下，原始專案都會保持原狀。

    有關指示，請參閱複製專案並將複製下移至資料夾。

在根層級複製專案
  1. 如果尚未進行這樣的處理，請按一下建立以建立新的階段作業，或按一下編輯以輸入現有的階段作業。請參閱使用變更中心。
  2. 選取「專案瀏覽器」>「專案」，以顯示專案檢視畫面。
  3. 按一下想要複製之專案列的「複製專案」圖示，隨即顯示「複製專案」頁面，並反白專案的目前位置。
  4. 在新名稱欄位中，輸入複製的名稱。請參閱命名專案和資料夾。

    5. 請使用唯一名稱將複製建立為新專案，或使用現有專案的名稱來合併此複製與該專案。

  5. 在「位置」面板中，保留在根層級複製專案時的預設「根」位置。
  6. 按一下複製專案。會在目前階段作業中複製專案。
  7. 若要結束階段作業並將組態建置至執行時期，請按一下「變更中心」下的「啟動」。
複製專案並將複製下移至資料夾
  1. 如果尚未進行這樣的處理，請按一下建立以建立新的階段作業，或按一下編輯以輸入現有的階段作業。請參閱使用變更中心。
  2. 選取「專案瀏覽器」>「專案」，以顯示專案檢視畫面。
  3. 按一下想要複製之專案列的「複製專案」圖示，隨即顯示「複製專案」頁面，並反白專案的目前位置。
  4. 在新名稱欄位中，選擇性地輸入資料夾的新名稱。請參閱命名專案和資料夾。

    5. 請使用唯一名稱來建立新資料夾，或使用現有資料夾的名稱來合併此複製與現有資料夾 (在下一個步驟選取的位置中)。

  5. 在位置面板中，選取專案或資料夾，然後按一下複製專案。複製的專案會在新位置中轉換為資料夾，而且會儲存在目前階段作業中。
  6. 若要結束階段作業並將組態建置至執行時期，請按一下「變更中心」下的「啟動」。
