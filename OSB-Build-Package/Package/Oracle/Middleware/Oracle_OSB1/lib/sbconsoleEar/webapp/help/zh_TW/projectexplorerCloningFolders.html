複製資料夾

  1. 如果尚未進行這樣的處理，請按一下建立以建立新的階段作業，或按一下編輯以輸入現有的階段作業。請參閱使用變更中心。
  2. 選取專案瀏覽器，瀏覽至含有想要複製之資料夾的父項專案或資料夾。隨即顯示專案/資料夾檢視畫面頁面。
  3. 按一下想要複製之資料夾列的「複製資料夾」圖示，隨即顯示「複製資料夾」頁面，並反白資料夾的目前位置。
  4. 複製相同位置或不同位置中的資料夾，或是將資料夾升級至專案，如下所示：
    • 若要複製相同位置中的資料夾，請在新名稱欄位中輸入新名稱。請參閱命名專案和資料夾。

      • 在相同位置進行複製時，請不要在「位置」面板中選取任何位置 (這等同於選取目前位置)，或者明確地選取目前專案和資料夾。

    • 若要複製不同位置中的資料夾，請選擇性地在新名稱欄位中輸入新名稱，然後從位置區域中選取專案和 (或) 資料夾。請參閱命名專案和資料夾。

      • 如果使用與現有資料夾相同的名稱，則原始資料夾的內容會合併目標資料夾的內容。

    • 若要將資料夾升級為專案，請選取「根」位置。
  5. 按一下複製資料夾。會在目前階段作業中複製資料夾。

    6. 複製資料夾時，會以遞迴方式複製資料夾下的所有資料夾和資源

  6. 若要結束階段作業並將組態建置至執行時期，請按一下「變更中心」下的「啟動」。
