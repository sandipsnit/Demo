複製資源

  1. 如果尚未進行這樣的處理，請按一下建立以建立新的階段作業，或按一下編輯以輸入現有的階段作業。請參閱使用變更中心。
  2. 選取專案瀏覽器，然後選取含有想要複製之資源的專案或資料夾。隨即顯示專案/資料夾檢視畫面頁面。
  3. 按一下想要複製之資源列的複製資源圖示，隨即顯示複製資源頁面，並反白資源的目前位置。
  4. 您可以複製相同位置或不同位置中的資源：
    • 若要複製相同位置中的資源，請在新資源名稱欄位中輸入新名稱。請參閱資源命名限制。

      • 若要在相同位置進行複製，請不要在「位置」面板中選取任何位置 (這等同於選取目前位置)，或者明確地選取目前專案和資料夾。

    • 若要複製不同位置中的資源，請選擇性地在「新資源名稱」欄位中輸入新名稱，以及從「位置」區域中選取專案和 (或) 資料夾。
  5. 按一下複製資源。會在目前階段作業中複製資源。

    6. 如果對從 UDDI 登錄檔中輸出的業務服務進行複製，則會中斷該複製服務與登錄檔的連接。

  6. 若要結束階段作業並將組態建置至執行時期，請按一下「變更中心」下的「啟動」。
