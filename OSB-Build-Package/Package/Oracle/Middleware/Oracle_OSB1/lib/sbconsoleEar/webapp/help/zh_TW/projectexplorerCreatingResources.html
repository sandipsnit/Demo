建立資源

  1. 如果尚未進行這樣的處理，請按一下建立以建立新的階段作業，或按一下編輯以輸入現有的階段作業。請參閱使用變更中心。
  2. 選取「專案瀏覽器」，然後選取要在其中建立資源的專案或資料夾。
  3. 從專案/資料夾檢視畫面頁面的「建立資源」下拉清單中，選取想要建立的資源類型。會根據選取的資源類型，顯示不同的頁面。
  4. 請遵循提示新增每個頁面的資訊，以建立資源。有關建立每種資源的詳細資訊，請參閱適當主題 (如表 3-3 所列)。

  5. 在您建立資源時，會在目前階段作業中建立該資源。
  6. 若要結束階段作業並將組態建置至執行時期，請按一下「變更中心」下的「啟動」。
