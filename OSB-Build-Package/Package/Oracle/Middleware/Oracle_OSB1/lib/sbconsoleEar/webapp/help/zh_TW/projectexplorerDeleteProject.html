刪除專案

  1. 如果尚未進行這樣的處理，請按一下建立以建立新的階段作業，或按一下編輯以輸入現有的階段作業。請參閱使用變更中心。
  2. 選取「專案瀏覽器」>「專案」，以顯示專案檢視畫面。
  3. 按一下想要刪除之專案列的刪除專案圖示。即會在目前的階段作業中刪除專案。

    4. 刪除專案時，也會刪除專案下的所有資源。如果不同專案下的資源參照此專案下的任何資源，還是可以刪除專案，只是會出現警告確認。因為仍有已刪除資源的未解析參照，所以可能會導致衝突。

    這些資源會顯示「刪除警告」圖示，而不是「刪除」圖示。

  4. 若要結束階段作業並將組態建置至執行時期，請按一下「變更中心」下的「啟動」。
