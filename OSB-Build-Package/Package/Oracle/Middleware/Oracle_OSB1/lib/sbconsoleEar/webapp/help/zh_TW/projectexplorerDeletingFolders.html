刪除資料夾

  1. 如果尚未進行這樣的處理，請按一下建立以建立新的階段作業，或按一下編輯以輸入現有的階段作業。請參閱使用變更中心。
  2. 選取「專案瀏覽器」，然後選取含有想要刪除之資料夾的專案或資料夾。
  3. 在專案/資料夾檢視畫面頁面中，按一下想要刪除之資料夾列的刪除資料夾圖示。將在目前的階段作業中刪除資料夾。

    4. 刪除資料夾時，也會刪除資料夾下的所有資源。如果不同資料夾下的資源參照此資料夾下的任何資源，還是可以刪除資料夾，只是會出現警告確認。因為會出現已刪除資源的未解析參照，所以這可能會導致衝突。

    這些資源會顯示「刪除警告」圖示，而不是「刪除」圖示。

  4. 若要結束階段作業並將組態建置至執行時期，請按一下「變更中心」下的「啟動」。
