匯出 WSDL

使用專案/資料夾檢視畫面頁面，可以匯出與代理主機服務或商業相關的 WSDL。WSDL 會匯出為 JAR 檔案。只有不在階段作業內時，才能匯出 WSDL。請參閱使用變更中心。

使用「匯出 WSDL」功能，可以讓外部工具 (例如 IDE) 可以使用 WSDL。請注意，這和「系統管理」模組中的「匯出資源」功能不同，該功能用於在兩個網域間移動及計劃資源。

  1. 從左側瀏覽窗格中，按一下專案瀏覽器下的專案或資料夾。隨即顯示專案/資料夾檢視畫面頁面。
  2. 按一下想要匯出 WSDL 之代理主機服務列的「匯出 WSDL」圖示，隨即顯示「檔案下載」對話方塊。
  3. 在「檔案下載」對話方塊中，按一下「開啟」開啟 JAR 檔案，或按一下「儲存」，將 JAR 檔案儲存至您的桌面。
