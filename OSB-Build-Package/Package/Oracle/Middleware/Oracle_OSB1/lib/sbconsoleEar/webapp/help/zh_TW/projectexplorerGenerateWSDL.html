產生 WSDL

產生的 WSDL 是 Oracle Service Bus 針對並非以 WSDL 資源啟動，但是可用 WSDL 描述的服務，所產生的 WSDL 資源。例如，從 EJB 型服務建立的 WSDL 就是產生的 WSDL。有關詳細資訊，請參閱「Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus」中的「How WSDL is Used in Oracle Service Bus」，網址為： http://www.oracle.com/pls/as111170/lookup?id=OSBAG114。

使用專案/資料夾檢視畫面頁面，可以產生與 EJB 或流程傳輸型業務服務相關聯的 WSDL。

  1. 從左側瀏覽窗格中，按一下專案瀏覽器下的專案或資料夾。隨即顯示專案/資料夾檢視畫面頁面。
  2. 在要產生 WSDL 的業務服務所在的列中，按一下「產生 WSDL」圖示，隨即顯示「產生 WSDL 資源」對話方塊，其中反白了目前的位置 (專案名稱和業務服務所在的資料夾名稱)。
  3. 在產生 WSDL 資源對話方塊中，指定新 WSDL 資源名稱以及 WSDL 的位置，然後按一下產生 WSDL。

    請遵循資源命名限制的命名指南。

完成作業後

繼續進行編輯 WSDL，以檢視或修改產生的 WSDL。
