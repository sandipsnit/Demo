描述
輸入新的專案名稱
若要建立新專案，請在文字欄位中輸入名稱，然後按一下新增專案。請參閱新增專案。
請參閱命名專案和資料夾。
名稱
按一下專案名稱，以顯示該專案的專案/資料夾檢視畫面頁面。
選項
針對任何列出的專案，您都可以執行下列任一操作：
  • 按一下重新命名專案圖示，以重新命名專案。請參閱重新命名專案。
  • 按一下移動專案圖示來移動專案。請參閱移動專案 (轉換為資料夾)。
  • 按一下複製專案圖示來複製專案。請參閱複製專案。
  • 按一下刪除專案圖示來刪除專案。當其他資源參照此資源時，會顯示刪除警告圖示。確認警告後即可刪除此資源。因為仍有已刪除資源的未解析參照，所以可能會導致衝突。請參閱刪除專案。
執行上述任一動作時，結果都會儲存在目前階段作業中。若要結束階段作業並將組態建置至執行時期，請按一下「變更中心」下的「啟動」。
