從 URL 載入資源

您可以在 URL 可用的資源上或檔案系統上進行批次匯入。批次匯入可讓您匯入根資源 (例如 WSDL) 以及其相依項 (例如其他 WSDL和網要)。會自動解析相依性對應。

  1. 如果尚未進行這樣的處理，請按一下建立以建立新的階段作業，或按一下編輯以輸入現有的階段作業。請參閱使用變更中心。
  2. 選取專案瀏覽器，然後選取要在其中新增資源的專案或資料夾。隨即顯示專案/資料夾檢視畫面頁面。
  3. 從建立資源下拉清單中，選取 URL 的資源。隨即顯示載入資源 - 從 URL 載入資源頁面。
  4. 在「URL/路徑」欄位中，輸入資源的 URL 或路徑。

    5. 若要指定本機資源，您可以使用 file 通訊協定加上含路徑的完整檔案名稱，例如 file:///c:/osbresources/ForeachAction.jar。

  5. 在「資源名稱」欄位中，輸入資源的名稱。
  6. 從「資源類型」下拉清單中，選取資源的類型。
  7. 按「下一頁」顯示載入資源 - 檢查已載入的資源頁面。清除任何不想要併入此匯入之資源旁邊的核取方塊。
  8. 若要匯入選取的資源，請按一下「匯入」。
  9. 載入資源 - 匯入結果頁面會顯示匯入的資源。按一下「載入另一個」，從另一個檔案匯入資源，或從左瀏覽窗格選取不同頁面，繼續進行其他工作。
  10. 若要結束階段作業並將組態建置至執行時期，請按一下「變更中心」下的「啟動」。
