移動專案 (轉換為資料夾)

「移動」專案時，專案會在新位置中轉換為資料夾。

  1. 如果尚未進行這樣的處理，請按一下建立以建立新的階段作業，或按一下編輯以輸入現有的階段作業。請參閱使用變更中心。
  2. 選取「專案瀏覽器」>「專案」，以顯示專案檢視畫面。
  3. 按一下想要移動之專案列的「移動專案」圖示，隨即顯示「移動專案」頁面，並反白專案的目前位置。
  4. 在位置下，選取要移入專案的專案或資料夾。專案會在新位置中轉換為資料夾，而且會儲存在目前階段作業中。

    5. 如果將專案移入新位置，而該位置已有同名的資料夾，則專案的內容會與現有資料夾的內容合併。

  5. 若要結束階段作業並將組態建置至執行時期，請按一下「變更中心」下的「啟動」。
