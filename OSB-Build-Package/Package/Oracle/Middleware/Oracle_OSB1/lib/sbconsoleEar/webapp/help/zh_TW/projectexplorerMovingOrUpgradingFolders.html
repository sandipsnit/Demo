移動或升級資料夾

  1. 如果尚未進行這樣的處理，請按一下建立以建立新的階段作業，或按一下編輯以輸入現有的階段作業。請參閱使用變更中心。
  2. 選取專案瀏覽器，然後選取含有想要移動或升級之資料夾的專案或資料夾。隨即顯示專案/資料夾檢視畫面頁面。
  3. 按一下想要移動或升級之資料夾列的「移動資料夾」圖示，隨即顯示「移動資料夾」頁面，並反白資料夾的目前位置。
  4. 判斷是否要將資料夾移入不同專案下的新位置，或將資料夾升級為專案：
    • 若要將它移入新位置，請在「位置」面板中選取要移入資料夾的專案或資料夾。
    • 若要將資料夾升級為專案，請選取「根」位置。
  5. 按一下「提交」。資料夾 (或專案，如果其自資料夾升級而來) 將儲存在目前階段作業中其新位置下。

    6. 移動或升級資料夾時，如果該位置已存在同名的另一個資料夾，則移動或升級之資料夾的內容將與現有資料夾或專案的內容合併。

  6. 若要結束階段作業並將組態建置至執行時期，請按一下「變更中心」下的「啟動」。
