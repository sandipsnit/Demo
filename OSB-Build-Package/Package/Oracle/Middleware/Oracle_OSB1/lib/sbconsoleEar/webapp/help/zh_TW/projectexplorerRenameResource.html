重新命名資源

  1. 如果尚未進行這樣的處理，請按一下建立以建立新的階段作業，或按一下編輯以輸入現有的階段作業。請參閱使用變更中心。
  2. 選取專案瀏覽器，然後選取含有想要重新命名之資源的專案或資料夾。隨即顯示專案/資料夾檢視畫面頁面。
  3. 按一下想要重新命名之資源列的「重新命名」圖示，隨即顯示「重新命名」頁面。
  4. 在新資源名稱欄位中，輸入資源的新名稱。請參閱資源命名限制。
  5. 按一下「重新命名」。
    6. 注意： 如果將從 UDDI 登錄檔輸出的業務服務重新命名，則會中斷該服務與登錄檔的連接。
  6. 若要結束階段作業並將組態建置至執行時期，請按一下「變更中心」下的「啟動」。
