重新命名專案

  1. 如果尚未進行這樣的處理，請按一下建立以建立新的階段作業，或按一下編輯以輸入現有的階段作業。請參閱使用變更中心。
  2. 選取「專案瀏覽器」>「專案」，以顯示專案檢視畫面。
  3. 按一下想要重新命名之專案列的「重新命名專案」圖示，隨即顯示「重新命名專案」頁面。
  4. 在新專案名稱欄位中，輸入專案的新名稱。請參閱命名專案和資料夾。

    5. 如果使用現有專案的名稱，則這兩個專案的資源都會合併至該名稱下。例如，假設專案 MyProject 含有資源 A 和 B，而專案 ThisProject 含有資源 C 和 D。如果將 MyProject 重新命名為 ThisProject，則新的 ThisProject 會有資源 A、B、C 和 D。

  5. 按一下「重新命名」。會在目前階段作業中重新命名專案。
  6. 若要結束階段作業並將組態建置至執行時期，請按一下「變更中心」下的「啟動」。
