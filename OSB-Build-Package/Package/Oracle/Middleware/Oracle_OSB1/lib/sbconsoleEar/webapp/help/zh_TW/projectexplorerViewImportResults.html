載入資源 - 匯入結果頁面

「載入資源 - 匯入結果」頁面會顯示從已匯入 Zip 檔案或 URL 載入資源的結果。

根據是從 Zip 檔案或 URL 載入資源，確定您已完成從 Zip 檔案載入資源或從 URL 載入資源中的步驟，並確定您已完成載入資源 - 檢查已載入的資源頁面中的步驟。

隨即顯示「匯入結果」頁面，其中會顯示下列資訊。

表 3-7 專案/資料夾檢視畫面頁面 - 選項和控制項
選項或控制項
描述
狀態
資源的狀態；例如，是否已順利匯入。
在完成匯入 WSDL 所有的相依性之前，WSDL 可能會暫時標示為無效。但在匯入動作完成後，組態服務即會恢復標示為無效檔案的有效性。
名稱
資源的名稱。
路徑
資源的路徑，它是資源所在的專案及資料夾。
資源類型
顯示匯入的資源類型。
錯誤訊息
如果未順利匯入資源，將顯示錯誤訊息的詳細資訊。
