專案/資料夾檢視畫面頁面會顯示單一專案的詳細資訊。 表 3-2 說明頁面上的選項和控制項。

表 3-2 專案/資料夾檢視畫面頁面 - 選項和控制項

描述

參照
按一下此欄位中的數字 (如果大於 0)，以顯示此專案內部資源相依之此專案外部資源的清單。
參照者
按一下此欄位中的數字 (如果大於 0)，以顯示此專案內部資源相依之此專案外部資源的清單。
描述
按一下「編輯描述」可以新增或編輯此專案的描述，然後按一下「提交描述」進行更新。
資料夾
輸入新資料夾名稱
若要在此專案中建立新資料夾，請在文字欄位中輸入名稱，然後按一下新增資料夾。請參閱新增資料夾。
請參閱命名專案和資料夾。
名稱
按一下資料夾名稱，以顯示該資料夾的專案/資料夾檢視畫面頁面。
選項
針對任何列出的資料夾，您都可以執行下列任一操作：
  • 按一下重新命名資料夾圖示來重新命名資料夾。請參閱重新命名資料夾。
  • 按一下升級至專案或移動資料夾圖示，以移動資料夾或將它升級為專案。請參閱移動或升級資料夾。
  • 按一下複製資料夾圖示來複製資料夾。請參閱複製資料夾。
  • 按一下刪除資料夾圖示來刪除資料夾。當其他資源參照此資源時，會顯示刪除警告圖示。確認警告後即可刪除資源。因為仍有已刪除資源的未解析參照，所以可能會導致衝突。請參閱刪除資料夾。
執行上述任一動作時，結果都會儲存在目前階段作業中。若要結束階段作業並將組態建置至執行時期，請按一下「變更中心」下的「啟動」。
刪除
若要刪除資料夾，請選取要刪除之資料夾名稱旁邊的核取方塊，然後按一下「刪除」。
資源
建立資源
若要建立此專案的新資源，請從建立資源清單中選取資源類型。請參閱建立資源。
名稱
按一下資源名稱以顯示資源的詳細資訊。
資源類型
此資源的類型。
動作
資源適用的動作是取決於資源類型：
  • 若為代理主機服務，會顯示編輯訊息流程圖示。按一下此圖示即可編輯代理主機服務的訊息。請參閱建立、編輯和檢視訊息流程。
  • 會為業務服務、代理主機服務、XQuery 轉換、XSL 轉換以及 MFL 檔案顯示啟動測試主控台圖示。按一下此圖示會叫用「測試主控台」，您可以用此來驗證並測試服務和轉換的設計。針對服務，您只能在執行時期使用「測試主控台」；也就是當階段作業啟動時。而針對轉換，無論您在階段作業內或外，您都可以使用「測試主控台」。如需詳細資訊，請參閱測試服務和測試轉換。
  • 將針對 WSDL 型代理主機服務顯示匯出 WSDL 圖示。按一下此圖示以匯出 WSDL，您接著可以使用外部工具 (例如 IDE) 來檢視或修改該 WSDL。

    • 請注意，這和「系統管理」模組中的「匯出資源」功能不同，該功能用於在兩個網域間移動及計劃資源。請參閱匯出 WSDL。

  • 將針對傳輸型業務服務 (例如 EJB 和流程) 顯示產生 WSDL圖示。按一下此圖示以產生 WSDL，您接著可以檢視或修改該 WSDL。請參閱產生 WSDL。
選項
「選項」欄會顯示下列圖示：
  • 按一下重新命名資源圖示，以重新命名資源。請參閱重新命名資源。
  • 按一下移動資源圖示移動資源。請參閱移動資源。
  • 按一下複製資源圖示來複製資源。請參閱複製資源。
  • 按一下刪除圖示來刪除資源。當其他資源參照此資源時，會顯示刪除警告圖示。確認警告後即可刪除此資源。因為仍有已刪除資源的未解析參照，所以可能會導致衝突。
執行上述任一動作時，結果都會儲存在目前階段作業中。若要結束階段作業並將組態建置至執行時期，請按一下「變更中心」下的「啟動」。
