在 Web 瀏覽器中檢視資源

在標準 Web 瀏覽器中使用下列 URL 來檢視 Oracle Service Bus 資源

    • WSDL 資源路徑：
      http://host:port/sbresource?WSDL/project_path/wsdlname
      或
      http://host:port/proxy_service_endpoint_URI?WSDL (適用於 WSDL 型 HTTP 代理主機服務)
    • 代理主機服務資源路徑：
      http://host:port/sbresource?PROXY/project_path/proxyname
      http://host:port/sbresource?ORAPROXY/project_path/proxyname (適用於含 Oracle Web Services Manager 原則的代理主機服務)
    • 業務服務資源路徑：
      http://host:port/sbresource?BIZ/project_path/bizname
      http://host:port/sbresource?ORABIZ/project_path/bizname (適用於含 Oracle Web Services Manager 原則的業務服務)
    • WSS 資源路徑：
      http://host:port/sbresource?POLICY/project_path/policyname
    • MFL 資源路徑：
      http://host:port/sbresource?MFL/project_path/mflname
    • SCHEMA 資源路徑：
      http://host:port/sbresource?SCHEMA/project_path/schemaname

    注意：如果在資源名稱中使用特殊字元，則用來公開 Oracle Service Bus 中資源的 URL 必須使用 UTF-8 編碼才能遁離特殊字元。

    您也可以擷取含 Oracle Web Services Manager 原則的 WSDL，讓原則可符合支援之 WS-Policy 和 WS 安全原則標準的規範。有關詳細資訊，請參閱 Oracle Fusion Middleware Developer's Guide for Oracle Service Bus 中的 "Advertising WSDLs to Support WS Standards"，網址為：http://www.oracle.com/pls/as111170/lookup?id=OSBDV1877。
