刪除服務金鑰提供者

刪除服務金鑰提供者時，Oracle Service Bus 也會從 PKI 證明資料對應提供者刪除關聯的金鑰組連結別名。Oracle Service Bus 不會從金鑰儲存庫刪除關聯的金鑰憑證組。

  1. 如果尚未進行這樣的處理，請按一下建立以建立新的階段作業，或按一下編輯以輸入現有的階段作業。請參閱使用變更中心。
  2. 如有任何代理主機服務被設定為使用服務金鑰提供者，請從代理主機服務中移除服務金鑰提供者。請參閱編輯代理主機服務組態。
  3. 選取「資源瀏覽器」>「服務金鑰提供者」，以顯示「服務金鑰提供者摘要」頁面。
  4. 在想要刪除之服務金鑰提供者的選項欄位中按一下刪除圖示。服務金鑰提供者會在目前的階段作業中刪除。如果業務服務或代理主機服務已設定成使用服務帳號，則刪除警告圖示表示確認警告後即可刪除服務金鑰提供者。因為仍有服務參照已刪除 WS 原則的未解析參照，所以可能會導致衝突。
  5. 若要結束階段作業並將組態建置至執行時期，請按一下「變更中心」下的「啟動」。
