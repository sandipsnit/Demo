尋找服務金鑰提供者

  1. 請執行下列任一項操作：
    • 選取專案瀏覽器以顯示專案檢視畫面頁面或專案/資料夾檢視畫面頁面。然後瀏覽專案和資料夾，以尋找服務金鑰提供者。
    • 選取資源瀏覽器 > 服務金鑰提供者。服務金鑰提供者摘要頁面會顯示表 17-1 中所顯示的資訊。
  2. 若要搜尋服務金鑰提供者，請在名稱欄位中輸入部分或全部的提供者名稱。您也可以在路徑欄位中輸入部分或整個提供者專案名稱和資料夾。按一下搜尋。

    3. 按一下「檢視全部」移除搜尋篩選，並顯示所有服務金鑰提供者。

    表 17-1 服務金鑰提供者資訊 
    特性
    描述
    名稱
    服務金鑰提供者的唯一名稱。按一下名稱即可查看檢視服務金鑰提供者詳細資訊頁面。請參閱編輯服務金鑰提供者。
    路徑
    服務金鑰提供者所在的專案名稱和資料夾名稱。按一下名稱，以查看包含此資源的專案或資料夾。請參閱使用專案和資料夾限定資源名稱。
