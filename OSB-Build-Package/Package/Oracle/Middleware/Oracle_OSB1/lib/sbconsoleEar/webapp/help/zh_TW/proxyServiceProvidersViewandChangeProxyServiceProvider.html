編輯服務金鑰提供者

使用「檢視服務金鑰提供者詳細資訊」頁面，可以檢視和變更特定服務金鑰提供者的詳細資訊。

  1. 如果尚未進行這樣的處理，請按一下建立以建立新的階段作業，或按一下編輯以輸入現有的階段作業。請參閱使用變更中心。
  2. 尋找服務金鑰提供者 (如尋找服務金鑰提供者中所述)。
  3. 按一下服務金鑰提供者名稱。「檢視服務金鑰提供者詳細資訊」頁面會顯示表 17-3 中所示的資訊。

    4. 表 17-3 服務金鑰提供者詳細資訊
    特性
    描述
    服務金鑰提供者名稱
    此服務金鑰提供者的名稱。
    上次修改者
    建立此服務金鑰提供者或將其匯入組態的使用者。
    上次修改時間
    使用者建立此服務金鑰提供者或將其匯入組態的日期和時間。按一下日期和時間連結，以檢視此資源的變更歷史記錄。請參閱檢視變更歷史記錄頁面。
    參照
    此服務金鑰提供者所參照的物件數。如果有這類參照，請按一下數字連結以檢視物件清單。請參閱檢視資源參照。
    參照者
    參照此服務金鑰提供者的物件數。如果有這類參照，請按一下數字連結以檢視物件清單。例如，如果選取此服務金鑰提供者作為特定代理主機服務的服務提供者，則在您按一下連結時，會以參照形式列出該代理主機服務。請參閱檢視資源參照。
    描述
    此服務金鑰提供者的描述 (如果有的話)。

  4. 若要變更欄位，請按一下編輯。有關欄位的描述，請參閱新增服務金鑰提供者。
  5. 按一下「儲存」，確認目前階段作業中的更新。
  6. 若要結束階段作業並將組態建置至執行時期，請按一下「變更中心」下的「啟動」。
