新增指派動作

使用指派動作，將 XQuery 表示式的結果指派至相關資訊環境變數。

開始之前

這些指示假設您已在「編輯訊息流程」頁面中編輯訊息流程 (如建立、編輯和檢視訊息流程中所述)。

若要新增指派動作

  1. 瀏覽至想要新增動作的位置 (如新增和編輯訊息流程的動作中所述)。
  2. 按一下適當的圖示，然後選取「新增動作」>「訊息處理」>「指派」。
  3. 按一下「<表示式>」。隨即顯示 XQuery 表示式編輯器頁面。XQuery 表示式可用來建立將指派至具名變數的資料。請參閱建立和編輯內置 XQuery 和 XPath 表示式。
  4. 當您完成編輯表示式之後，請在變數欄位中輸入相關資訊環境變數。若要進一步瞭解相關資訊環境變數，請參閱 Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus 中的 "Inbound and Outbound Variables"，網址為： http://www.oracle.com/pls/as111170/lookup?id=OSBAG325 及 "Constructing Messages to Dispatch"，網址為： http://www.oracle.com/pls/as111170/lookup?id=OSBAG348。

完成作業後

完成此動作的組態時，請繼續設定其他動作，或儲存組態 (如新增和編輯訊息流程的動作中所述)。
