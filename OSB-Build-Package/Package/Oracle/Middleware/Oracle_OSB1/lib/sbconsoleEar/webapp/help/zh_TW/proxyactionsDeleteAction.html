新增刪除動作

使用刪除動作來刪除相關資訊環境變數或 XPath 表示式指定的一組節點。刪除動作是更新動作集的其中一項動作。

開始之前

這些指示假設您已在「編輯訊息流程」頁面中編輯訊息流程 (如建立、編輯和檢視訊息流程中所述)。

若要新增刪除動作

  1. 瀏覽至想要新增動作的位置 (如新增和編輯訊息流程的動作中所述)。
  2. 若要刪除相關資訊環境變數，請選取變數單選按鈕，然後在變數欄位中輸入相關資訊環境變數的名稱。若要進一步瞭解相關資訊環境變數的相關資訊，請參閱附錄 A，訊息相關資訊環境。

    3. 或刪除 XPath 表示式選取的所有節點，並選取 XPath 圓鈕，然後按一下 <XPath>。隨即顯示 XPath 表示式編輯器頁面。請參閱建立和編輯內置 XQuery 和 XPath 表示式。儲存表示式之後，請在變數欄位中輸入相關資訊環境變數。若要進一步瞭解相關資訊環境變數的相關資訊，請參閱附錄 A，訊息相關資訊環境。

完成作業後

完成此動作的組態時，請繼續設定其他動作，或儲存組態 (如新增和編輯訊息流程的動作中所述)。
