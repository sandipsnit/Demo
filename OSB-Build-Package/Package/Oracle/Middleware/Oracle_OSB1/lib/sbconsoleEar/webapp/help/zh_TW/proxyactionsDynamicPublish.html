新增動態發佈動作

使用動態發佈動作可以將訊息發佈至 XQuery 表示式指定的服務。

如需發佈行為的詳細資訊，請參閱 Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus 中的 Performing Transformations in Message Flows，網址為 http://www.oracle.com/pls/as111170/lookup?id=OSBAG205。

  1. 瀏覽至想要新增動作的位置 (如新增和編輯訊息流程的動作中所述)。
  2. 按一下適當的圖示，然後選取「新增動作」>「通訊」>「動態發佈」。
  3. 按一下「<表示式>」。
  4. 在「XQuery 表示式編輯器」中，輸入 Xquery 表示式或選取可提供類似下列結果的 XQuery 資源： 
    5. <ctx:route>

    <ctx:service 
isProxy="false">project/folder/businessservicename</ctx:service>

    <ctx:operation>foo</ctx:operation>

    </ctx:route>
    注意： 如果正在呼叫代理主機服務，應將 isProxy 屬性設定為 true。如果正在呼叫業務服務，請將 isProxy 設定為 false。
    operation 元素為選擇性元素。
  5. 按一下「儲存」。
  6. 在要求動作欄位中，按一下新增動作以新增動作，然後選取想要與服務相關聯的動作。您可以新增多個動作。若要進一步瞭解可以新增之動作類型的相關資訊，請參閱新增和編輯訊息流程的動作中的動作表格。

完成作業後

完成此動作的組態時，請繼續設定其他動作，或儲存組態 (如新增和編輯訊息流程的動作中所述)。
