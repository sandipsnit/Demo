新增 For Each 動作

使用 for each 動作可以逐一執行一連串的值，並執行一段動作。

開始之前

這些指示假設您已在「編輯訊息流程」頁面中編輯訊息流程 (如建立、編輯和檢視訊息流程中所述)。

若要新增 For Each 動作

  1. 瀏覽至想要新增動作的位置 (如新增和編輯訊息流程的動作中所述)。
  2. 按一下適當的圖示，然後選取「新增動作」>「流程控制」>「For Each」。
  3. 在「...變數」欄位中輸入變數名稱，並按一下「<XPath>」開啟 XPath 編輯器以建立 XPath 表示式，然後在「Do ()」迴圈中設定動作。

完成作業後

完成此動作的組態時，請繼續設定其他動作，或儲存組態 (如新增和編輯訊息流程的動作中所述)。
