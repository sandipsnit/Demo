新增 If... Then... 動作

使用 if...then... 動作，可以根據 XQuery 表示式的布林結果，有條件地執行一個動作或一組動作。

開始之前

這些指示假設您已在「編輯訊息流程」頁面中編輯訊息流程 (如建立、編輯和檢視訊息流程中所述)。

若要新增 If...Then 動作

  1. 瀏覽至想要新增動作的位置 (如新增和編輯訊息流程的動作中所述)。
  2. 按一下適當的圖示，然後選取「新增動作」>「流程控制」>「If...Then」。
  3. 按一下「<條件>」顯示「XQuery 條件編輯器」頁面。

    4. 您建立的條件會依據標準的 if...then 邏輯，在進入 then() 子句之前執行的測試。請參閱建立和編輯內置 XQuery 和 XPath 表示式。

  4. 當您完成編輯 XQuery 條件時，請按一下新增動作，然後選取想要與該條件相關聯的動作。若要進一步瞭解想要新增之動作類型的相關資訊，請參閱新增和編輯訊息流程的動作。

    5. 在路由節點中，您只可以選取路由、動態路由或路由表格動作。不過，這些動作中可以包含要求和回應動作。

  5. 基於邏輯需要，請按一下 If...Then... 圖示，然後按一下 新增 else-if 條件 或 新增 else 條件 來新增 else-if 條件或 else 條件。按一下新增動作，將各個動作與這些條件產生關聯。

    6. 條件動作可以是巢狀的。然而，階段編輯器中有四個累計層級的巢狀限制。如果嘗試新增第五層，則不會顯示此巢狀動作。累計層級包含所有分支動作：if...then... 條件、發佈表格和路由表格。例如，您可以建立含有兩個層級的條件，然後再建立一個含有路由表格的發佈表格，這樣總共就有四個層級。但是如果您嘗試新增另一個條件動作 (到最後一個發佈表格)，則不會顯示它。

完成作業後

完成此動作的組態時，請繼續設定其他動作，或儲存組態 (如新增和編輯訊息流程的動作中所述)。
