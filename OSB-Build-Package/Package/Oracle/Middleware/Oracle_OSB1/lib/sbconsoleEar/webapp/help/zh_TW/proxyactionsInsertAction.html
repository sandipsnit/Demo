新增插入動作

使用插入動作，可以將 XQuery 表示式的結果插入至與 XPath 表示式所選節點相對的確定位置。插入動作是一種更新動作。

開始之前

這些指示假設您已在「編輯訊息流程」頁面中編輯訊息流程 (如建立、編輯和檢視訊息流程中所述)。

若要新增插入動作

  1. 瀏覽至想要新增動作的位置 (如新增和編輯訊息流程的動作中所述)。
  2. 按一下適當的圖示，然後選取「新增動作」>「訊息處理」>「插入」。
  3. 按一下 <表示式> 編輯 XQuery 表示式。XQuery 表示式用來建立將在具名變數之指定位置插入的資料。隨即顯示 XQuery 表示式編輯器頁面。請參閱建立和編輯內置 XQuery 和 XPath 表示式。
  4. 當您完成表示式的編輯時，請從下拉清單中選取相對的位置。相對位置是用來控制執行插入的位置，它相對於 XPath 表示式的結果：
    • 「之前」-作為 XPath 表示式選取之每一個元素或屬性之前的同層級
    • 「之後」-作為 XPath 表示式選取之每一個元素或屬性之後的同層級
    • 作為第一個子項-作為 XPath 表示式識別之每一個元素的第一個子項。如果 XPath 的結果傳回屬性，則會發生錯誤。
    • 作為最後一個子項-作為 XPath 表示式識別之每一個元素的最後一個子項。如果 XPath 傳回屬性，則會發生錯誤。
  5. 按一下 XPath。隨即顯示 XPath 表示式編輯器頁面。請參閱建立和編輯內置 XQuery 和 XPath 表示式。

    6. 有效組態如下：

    • XQuery 和 XPath 表示式都會傳回元素。
    • XQuery 和 XPath 表示式都可能會傳回屬性，在此情況下，XQuery 表示式必須傳回屬性。
  6. 當您完成 XPath 表示式的編輯時，請在變數欄位中輸入相關資訊環境變數。XPath 會評估此變數的內容。若要進一步瞭解相關資訊環境變數的相關資訊，請參閱附錄 A，訊息相關資訊環境。

完成作業後

完成此動作的組態時，請繼續設定其他動作，或儲存組態 (如新增和編輯訊息流程的動作中所述)。
