新增 MFL 轉換動作

使用 MFL (訊息格式語言) 轉換動作，可以將訊息管線中的訊息內容從 XML 轉換為非 XML，反之亦然。MFL 是說明二進位資料配置的專用 XML 文件。它是 Oracle 專用語言，用來定義將格式化二進位資料轉換為 XML 資料的規則，反之亦然。請參閱 MFL。

開始之前

這些指示假設您已在「編輯訊息流程」頁面中編輯訊息流程 (如建立、編輯和檢視訊息流程中所述)。

若要新增 MFL 轉換動作

  1. 瀏覽至想要新增動作的位置 (如新增和編輯訊息流程的動作中所述)。
  2. 按一下適當的圖示，然後選取「新增動作」>「訊息處理」>「MFL 轉換」。
  3. 根據您的需求，從「套用 MFL 轉換」下拉清單選取「XML 至非 XML」或「非 XML 至 XML」。
  4. 按一下「<表示式>」。使用 XQuery 表示式編輯器，指定要執行 MFL 轉換動作的變數。轉換為 XML 時，此輸入必須是文字或二進位，而轉換為非 XML 時，此輸入必須是 XML。訊息相關資訊環境的二進位內容由二進位內容 XML 元素表示。當輸入必須為二進位時，此 XML 應該是 Xquery 表示式的結果。請參閱建立和編輯內置 XQuery 和 XPath 表示式。
  5. 選取下列其中一個選項：
    • MFL 資源：按一下 <資源> 連結。隨即顯示選取 MFL 頁面。選擇要執行 MFL 轉換動作的靜態 MFL 資源。
    • 來自...的 MFL 資源：按一下表示式連結。隨即顯示 XQuery 表示式編輯器頁面。使用「XQuery 表示式編輯器」建立或編輯 XQuery 表示式，動態指定要執行該轉換動作的 MFL 資源，格式為 project/folder/MFLresourcename。請參閱建立和編輯內置 XQuery 和 XPath 表示式。
  6. 在指派給變數欄位中，輸入此轉換動作的結果要指派給哪一個變數名稱。其結果將是二進位內容 XML 元素。

完成作業後

完成此動作的組態時，請繼續設定其他動作，或儲存組態 (如新增和編輯訊息流程的動作中所述)。
