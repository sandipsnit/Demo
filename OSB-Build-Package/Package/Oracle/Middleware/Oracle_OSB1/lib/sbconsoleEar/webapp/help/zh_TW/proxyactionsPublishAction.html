新增發佈動作

使用發佈動作可以識別針對訊息靜態指定的目標服務，以及設定如何封裝訊息並將訊息傳送至該服務。

如需發佈行為的詳細資訊，請參閱 Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus 中的 Performing Transformations in Message Flows，網址為 http://www.oracle.com/pls/as111170/lookup?id=OSBAG205。

  1. 瀏覽至想要新增動作的位置 (如新增和編輯訊息流程的動作中所述)。
  2. 選取「新增動作」>「通訊」>「發佈」。
  3. 按一下 <服務>。隨即顯示選擇服務頁面。
  4. 從清單中選擇一個服務，然後按一下「提交」。這是訊息的目標服務。
  5. 如果服務已定義作業，您可以從「作業」清單中選取作業來指定要叫用的作業。
  6. 若要讓外送作業和內送作業相同，請選取「使用內送作業進行外送」核取方塊。
  7. 若要設定如何封裝訊息並將訊息傳送給服務，請按一下要求動作欄位中的新增動作。然後選擇動作以與服務產生關聯。您可以新增多個動作。請參閱新增和編輯訊息流程的動作。

完成作業後

完成此動作的組態時，請繼續設定其他動作，或儲存組態 (如新增和編輯訊息流程的動作中所述)。
