新增重新命名動作

使用重新命名動作，可以重新命名 XPath 表示式所選取的元素，而不會修改元素的內容。重新命名動作是更新動作集的其中一種動作。

開始之前

這些指示假設您已在「編輯訊息流程」頁面中編輯訊息流程 (如建立、編輯和檢視訊息流程中所述)。

若要新增重新命名動作

  1. 瀏覽至想要新增動作的位置 (如新增和編輯訊息流程的動作中所述)。
  2. 按一下適當的圖示，然後選取「新增動作」>「訊息處理」>「重新命名」。
  3. 按一下 <XPath>。隨即顯示 XPath 表示式編輯器頁面。XPath 表示式可用來指定將被重新命名的資料 (於具名變數中)。請參閱建立和編輯內置 XQuery 和 XPath 表示式。
  4. 在變數欄位中，輸入保留想要重新命名之元素的相關資訊環境變數。若要進一步瞭解相關資訊環境變數的相關資訊，請參閱附錄 A，訊息相關資訊環境。
  5. 請執行下列其中一項操作：
    • 若要使用本機名稱重新命名選定元素，請選取第一個「本機名稱」單選按鈕，然後在「本機名稱」欄位中輸入本機名稱。
    • 若要使用命名空間重新命名選定元素，請選取第一個「命名空間」單選按鈕，然後在「命名空間」欄位中輸入命名空間。
    • 若要使用本機名稱和命名空間重新命名選定元素，請選取「本機名稱和命名空間」單選按鈕，然後在「本機名稱和命名空間」欄位中輸入本機名稱和命名空間。

完成作業後

完成此動作的組態時，請繼續設定其他動作，或儲存組態 (如新增和編輯訊息流程的動作中所述)。
