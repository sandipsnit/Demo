新增取代動作

使用取代動作，可以取代節點，或 XPath 表示式所指定的節點內容。節點或其內容會以 XQuery 表示式所傳回的值加以取代。

取代動作可以用來取代簡單的值、元素，甚至也可以取代屬性。未傳回任何內容的 XQuery 表示式等同於刪除已識別節點或讓節點變成空的，將視動作取代整個節點還是只取代節點內容而定。

取代動作是更新動作集的其中一項動作。

開始之前

這些指示假設您已在「編輯訊息流程」頁面中編輯訊息流程 (如建立、編輯和檢視訊息流程中所述)。

若要新增取代動作

  1. 瀏覽至想要新增動作的位置 (如新增和編輯訊息流程的動作中所述)。
  2. 按一下適當的圖示，然後選取「新增動作」>「訊息處理」>「取代」。
  3. 按一下 <XPath>。隨即顯示「XPath 表示式編輯器」XPath 表示式可用來指定要取代的資料 (於具名變數中)。請參閱建立和編輯內置 XQuery 和 XPath 表示式。
  4. 當您完成編輯 XPath 表示式時，請在變數欄位中輸入相關資訊環境變數。請參閱附錄 A 訊息相關資訊環境。
  5. 按一下「<表示式>」。隨即顯示「XQuery 表示式編輯器」頁面。XQuery 表示式可用來建立資料，以取代 XPath 在具名變數中指定的資料。請參閱建立和編輯內置 XQuery 和 XPath 表示式。
  6. 當編輯完 XQuery 表示式時，請選取其中一個選項：
    • 「取代整個節點」- 指定是否取代您定義的 XPath 表示式所選取的節點及其所有內容。
    • 「取代節點內容」- 指定不取代節點；只取代內容。
      • 注意： 選取「取代節點內容」選項並將 XPath 欄位保留為空白，比選取「取代整個節點」選項並將 XPath 設定為 ./* 更有效。

完成作業後

完成此動作的組態時，請繼續設定其他動作，或儲存組態 (如新增和編輯訊息流程的動作中所述)。
