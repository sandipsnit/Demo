新增驗證動作

使用驗證動作，可以根據 XML 綱要元素或 WSDL 資源來驗證 XPath 表示式所選取的元素。您只可以驗證全域元素；Oracle Service Bus 不支援針對本機元素進行驗證。

開始之前

這些指示假設您已在「編輯訊息流程」頁面中編輯訊息流程 (如建立、編輯和檢視訊息流程中所述)。

若要新增驗證動作

  1. 瀏覽至想要新增動作的位置 (如新增和編輯訊息流程的動作中所述)。
  2. 按一下適當的圖示，然後選取「新增動作」>「訊息處理」>「驗證」。
  3. 按一下 XPath，建構指定要驗證之元素的 XPath 表示式。請參閱建立和編輯內置 XQuery 和 XPath 表示式。完成「XPath 表示式編輯器」中的表示式建構時，請按一下儲存，在編輯階段組態頁面中插入表示式。
  4. 在「在變數中」欄位中，輸入保留要驗證之元素的變數名稱。
  5. 按一下「資源」，然後選取 WSDL 或綱要。
  6. 從「WSDL 瀏覽器」或「XML 綱要瀏覽器」執行下列操作：
    1. 選取 WSDL 或 XML 綱要
    2. 選取 WSDL 或 XML 綱要類型或元素
    3. 按一下「提交」。
  7. 若要儲存此驗證的結果 (布林結果)，請選取「將驗證結果儲存在變數中」，然後輸入要儲存結果的變數名稱。

    8. 或者，若要在元素無法對照 WSDL 或 XML 綱要元素進行驗證時發出錯誤，請選取「驗證失敗時發出錯誤」。

完成作業後

完成此動作的組態時，請繼續設定其他動作，或儲存組態 (如新增和編輯訊息流程的動作中所述)。
