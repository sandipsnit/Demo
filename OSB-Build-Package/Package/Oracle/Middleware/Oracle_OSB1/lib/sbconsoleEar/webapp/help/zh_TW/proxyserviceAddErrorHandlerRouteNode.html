新增路由節點錯誤處理程式

開始之前

這些指示假設您已在編輯訊息流程頁面中編輯訊息流程 (如建立、編輯和檢視訊息流程中所述)。

這些指示也假設您已建立路由節點 (如將路由節點新增至訊息流程中所述)。

若要新增路由節點處理程式

  1. 按一下路由節點圖示，然後按一下新增錯誤處理程式。隨即顯示編輯錯誤處理程式頁面。
  2. 按一下「錯誤處理程式」圖示，然後按一下「新增階段」。
  3. 按一下階段圖示，然後按一下編輯階段。隨即顯示「編輯階段組態」頁面。
  4. 按一下「新增動作」，然後選取想要新增的動作。

    5. 因為錯誤處理程式不過是另一個管線，所以其設定方式也與任何其他管線一樣。例如，您可以使用發佈動作傳送錯誤通知給其他服務，並使用指派動作修改相關資訊環境變數等等。若要進一步瞭解想要新增之動作類型的相關資訊，請參閱新增和編輯訊息流程的動作中的適當程序。動作的鏈結沒有限制。

    三個常用的錯誤動作是「發出錯誤」、「回覆」和「恢復」。

  5. 新增其他動作，並依需要在「編輯階段組態」頁面上進行其他編輯。
  6. 在「編輯階段組態」頁面上，按一下「儲存」確認目前階段作業中的更新。
  7. 在「編輯錯誤處理程式」頁面上，按一下「儲存」確認目前階段作業中的更新。

完成作業後

在編輯訊息流程頁面上，繼續建構訊息流程 (如建立、編輯和檢視訊息流程中所述)。
