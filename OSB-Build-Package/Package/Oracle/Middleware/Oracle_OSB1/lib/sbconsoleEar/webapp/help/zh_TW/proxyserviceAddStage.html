將階段新增至管線

開始之前

這些指示假設您已在「編輯訊息流程」頁面中編輯過訊息流程 (如建立、編輯和檢視訊息流程中所述)。

若要將階段新增至管線

  1. 必要時，請按一下管線組節點圖示左邊的加號，以展開管線組節點。管線組包含要求管線和回應管線。
  2. 按一下要新增階段的目標管線，然後按一下「新增階段」。
  3. 若要變更階段的預設名稱並新增描述，請執行下列操作：
    1. 按一下「階段」圖示，然後按一下「編輯名稱和註解」。
    2. 依需要變更名稱和描述。
    3. 按一下「儲存」。
  4. 若要將動作新增至階段，請按一下階段圖示，然後按一下編輯階段。請參閱新增和編輯訊息流程的動作。
  5. 若要將錯誤處理新增至階段，請按一下階段圖示，然後按一下新增階段錯誤處理程式。請參閱新增階段錯誤處理程式。隨即顯示編輯訊息流程頁面。
  6. 繼續建構訊息流程 (如建立、編輯和檢視訊息流程中所述)。
  7. 按一下「儲存」，確認目前階段作業中的更新。
  8. 若要結束階段作業並將組態建置至執行時期，請按一下「變更中心」下的「啟動」。
