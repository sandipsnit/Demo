在編輯器工作區文字欄位中建置表示式

「XQuery 表示式編輯器」、「XQuery 條件編輯器」和「XPath 表示式編輯器」都提供文字欄位，您可以在其中直接進行輸入或從面板貼入項目，以建置表示式。

開始之前

這些指示假設您在「XQuery 表示式編輯器」、「XQuery 條件編輯器」或「XPath 表示式編輯器」中建立或編輯表示式 (如建立和編輯內置 XQuery 和 XPath 表示式中所述)。

若要在文字欄位中建置表示式：

  1. 顯示含有文字欄位的面板。根據編輯器，執行下列其中一項操作：
    • 在「XQuery 表示式編輯器」中，選取「XQuery 文字」(位在工作區按鈕列下方) (如果尚未選取)。
    • 在「XQuery 條件編輯器」中，選取「文字」(位在工作區按鈕列下方) (如果尚未選取)。
    • 在「XPath 表示式編輯器」中，您不需要選取任何項目，因為沒有選項可用來選取其他工具。
      • 注意： 選取上述任一連結，會顯示可以在其中建立相關資訊環境適用之完整表示式的文字欄位。然而，只要編輯器中提供文字欄位，就可以使用本主題說明的工具和技術，例如，將變數從匯入的資源連結至內置表示式時 (如將外部 XQuery 資源連結至內置 XQuery 和將外部 XSLT 資源連結至內置 XQuery中所述)。
  2. 需要時，請在欄位中輸入或貼上表示式或表示式片段。如果利用此方式建立完整表示式，請跳至下面的步驟 7。否則，請繼續進行下一個步驟。
  3. 選擇含有想要新增至表示式之項目的面板，並找到想要新增的項目 (如下面的表 22-1 中所述)。

    4. 表格 22-1 面板  
    面板
    描述和使用
    命名空間定義
    列出預設 Oracle Service Bus 命名空間、變數命名空間和使用者定義的命名空間。會列出定義的命名空間縮寫。

    另請參閱：
    • XQuery 轉換
    • 「Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus」中的「XQuery Implementation」，網址為： http://www.oracle.com/pls/as111170/lookup?id=OSBAG444。
    變數結構
    以樹狀圖顯示變數和其內容，協助您進行檢視。
    若要顯示變數結構和其內容
    • 從面板頂端的下拉清單中，選取結構名稱。此清單會顯示「內建」訊息相關資訊環境變數 (attachments、body、header、outbound 和 inbound)，以及任何使用者定義的結構 (依類型組織) (「XML 類型」、「服務介面」和「簡單類型」)。
    • 按一下加號 (+) 或減號 (-)，展開或隱藏樹狀圖中的節點。
    您也可以定義自己的變數結構。請參閱在 XQuery 編輯器中建立變數結構。
    變數結構並不會建立變數。變數是在執行時期建立，做為階段中的「指派」動作目標。
    將項目從變數結構樹狀圖插入至文字欄位時，是以用來說明路徑的 XPath 表示式形式插入。
    另請參閱 XQuery 轉換。

  4. 使用下方表 22-2 中的任一方法，將想要的項目貼入文字欄位：

    5. 表 22-2 將項目貼入編輯器文字欄位的方式 
    從此面板...
    請...
    「命名空間定義」面板
    使用標準滑鼠或鍵盤選取、複製和貼上命名空間，例如：
    1. 將滑鼠指標拖到命名空間字串上方，以選取整個命名空間字串 (或其現有縮寫)。
    2. 按 Ctrl-C 複製字串。
    3. 按一下想要在其中插入命名空間之文字欄位中的位置。
    4. 按 Ctrl-V 貼上字串。
    「XQuery 函數」面板或「變數結構」面板
    將項目從面板拖曳至文字欄位。
    注意： 只有 Internet Explorer 才支援從面板拖曳至工作區。
    「XQuery 函數」面板或「變數結構」面板
    1. 按一下面板中的項目。項目會顯示在「特性檢查器」窗格中：
      • 函數會與需要提供任何值的預留位置一起顯示。
      • 變數和其屬性會顯示為 XPath 表示式。
    2. 按一下想要在其中插入項目的文字欄位
    3. 按一下「複製特性」，將項目貼入文字欄位中選取的位置。
    「XQuery 函數」面板或「變數結構」面板
    1. 按一下面板中的項目。項目會顯示在「特性檢查器」窗格中。
    2. 使用標準鍵盤或滑鼠動作，選取並複製「特性檢查器」中的項目。
    3. 選取文字欄位中的位置，並使用標準鍵盤或滑鼠動作將項目貼入文字欄位。

  5. 繼續拖放函數以建置所要的表示式。
  6. 依需要在文字欄位中編輯表示式。
  7. 選擇性地執行下列任一項或兩項操作：
    • 按一下驗證。如果成功驗證表示式，則會顯示一則訊息。
    • 按一下測試來測試表示式。請參閱執行 XQuery 測試。
  8. 按一下「儲存」關閉編輯器，並在動作中插入表示式。

完成作業後

繼續設定動作 (如新增和編輯訊息流程的動作中所述)。
