建立、編輯和檢視訊息流程

  1. 如果尚未進行這樣的處理，請按一下建立以建立新的階段作業，或按一下編輯以輸入現有的階段作業。請參閱使用變更中心。
  2. 尋找想要檢視或編輯其訊息流程的代理主機服務。請執行下列任一項操作：
    • 選取「專案瀏覽器」顯示專案檢視畫面頁面或專案/資料夾檢視畫面頁面。然後瀏覽專案和資料夾，以尋找代理主機服務。
    • 選取「資源瀏覽器」>「代理主機服務」，以顯示「代理主機服務的摘要」頁面。
  3. 在想要檢視或編輯訊息流程的代理主機服務列，按一下「編輯訊息流程」圖示。
    • 如果尚未針對選取的代理主機服務建立任何訊息流程，顯示的編輯訊息流程頁面中只會有一個代理主機服務圖示。這是代理主機服務的開始節點。按一下此圖示即可開始建構訊息流程。
    • 如果代理主機服務已有訊息流程，則頁面會以圖形來表示流程。按一下圖示，可以檢視或編輯訊息流程的部分。

      • 有關如何在該頁面上建構訊息流程的詳細資訊，請參閱編輯訊息流程頁面。

  4. 按一下「儲存」，確認目前階段作業中的更新。
  5. 若要結束階段作業並將組態建置至執行時期，請按一下「變更中心」下的「啟動」。
