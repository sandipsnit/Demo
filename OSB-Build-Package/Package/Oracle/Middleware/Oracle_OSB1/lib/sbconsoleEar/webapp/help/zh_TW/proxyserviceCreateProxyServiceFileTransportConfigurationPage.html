檔案傳輸組態頁面

「建立/編輯代理主機服務 - 檔案傳輸組態」頁面是一系列頁面中的其中一頁，供您建立和編輯代理主機服務 (如建立和設定代理主機服務和編輯代理主機服務組態中所述)。

使用此頁面，可以設定利用檔案傳輸通訊協定之代理主機服務的傳輸設定。表 19-5 說明如何使用此頁面。

表 19-5 建立/編輯代理主機服務 - 檔案傳輸組態頁面 
選項
建立或編輯...
檔案遮罩
指定代理主機服務必須輪詢的欄位。如果 URI 是目錄且指定了 *.*，則服務將輪詢目錄中的所有檔案。「檔案遮罩」中只允許使用萬用字元 * 和 ?。不支援正規表示式。
受管理伺服器
此欄位只能用於叢集網域。
選取要作為輪詢伺服器的受管理伺服器。所有受管理伺服器都可以處理訊息，但只有一個受管理伺服器可以輪詢訊息。
輪詢間隔
輸入輪詢間隔 (秒)。預設值是 60。
讀取限制
指定每次輪詢清除所讀取的訊息數目上限。請輸入 0，以指定沒有限制。預設值為 10。
依到達時間排序
選取此核取方塊，以指定依到達的順序傳遞事件。
請注意，如果針對在叢集環境中執行的代理主機服務選取此選項，則訊息一律會傳送至相同伺服器。換言之，選取此選項時，會忽略跨伺服器的負載平衡。
掃描子目錄
選取此核取方塊，以遞迴方式掃描所有目錄。
透過參照來傳遞
選取此核取方塊，將檔案臨時放置於封存檔目錄中，並以參照形式在標頭中傳遞該檔案。
讀取後動作
選取訊息讀取後的動作：
  • 封存 - 封存訊息。
  • 刪除 - 刪除訊息。
暫存目錄
輸入在處理檔案時用來臨時放置檔案的中繼目錄。
請勿將暫存目錄放在輪詢目錄內 (在檔案傳輸代理主機服務的 URL 中識別的目錄，例如 file:///c:/dir1/dir2)。
封存檔目錄
如果讀取後動作選項設為封存檔，請指定封存檔位置的路徑。如果已選擇「透過參照來傳遞」欄位，則「封存檔目錄」欄位也是必要欄位。
請勿將封存檔目錄放在輪詢目錄內。
錯誤目錄
輸入在發生問題時用來張貼訊息和附件的位置。
請勿將錯誤目錄放在輪詢目錄內。
要求編碼
接受預設值 UTF-8 做為檔案傳輸中要求的字元集編碼，或輸入其他字元集編碼。

完成作業後

按「下一頁」繼續在下一個頁面上設定此服務，或按一下「最後一頁」在建立/編輯代理主機服務 - 摘要頁面上檢查並儲存這項組態。
