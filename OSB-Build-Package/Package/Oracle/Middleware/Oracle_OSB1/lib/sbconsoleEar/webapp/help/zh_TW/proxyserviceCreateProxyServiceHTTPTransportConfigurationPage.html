HTTP 傳輸組態頁面

HTTP 傳輸現在同時支援 HTTP 和 HTTPS 端點。

「建立/編輯代理主機服務 - HTTP 傳輸組態」頁面是一系列頁面中的其中一頁，供您建立和編輯代理主機服務 (如建立和設定代理主機服務和編輯代理主機服務組態中所述)。

使用此頁面，可以使用 HTTP 傳輸通訊協定來設定代理主機服務的傳輸設定。表 19-7 說明如何使用此頁面。

表 19-7 建立/編輯代理主機服務 - HTTP 傳輸組態頁面 
選項
建立或編輯...
需要 HTTPS
如果是內送 HTTPS 端點，請選取此核取方塊。
若要進一步瞭解，請參閱「Oracle Fusion Middleware Developer's Guide for Oracle Service Bus」中的「Configuring Transport-Level Security」，網址為： http://www.oracle.com/pls/as111170/lookup?id=OSBDV1557。
認證
選取下列其中一項：
  • 無 - 指定不需要進行認證。
  • 基本 - 指定需要進行基本認證，才能存取此服務。

    • 基本認證會指示 WebLogic Server 使用使用者名稱和密碼，向安全範圍中所設定的認證提供者 (如輕量型目錄存取通訊協定 (LDAP) 目錄服務和 Windows Active Directory) 進行從屬端的認證。從屬端必須在 HTTP 要求標頭上傳送它的使用者名稱和密碼。

    因為密碼會以純文字格式傳送，所以強烈建議不要透過 HTTP 進行基本認證。不過，透過 HTTPS 傳送密碼是安全的，因為 HTTPS 可提供加密通道。

    警告：所有使用者 (授權和匿名) 預設都可以存取代理主機服務。若要限制可以存取代理主機服務的使用者，請建立傳輸層級授權原則。請參閱編輯傳輸層級存取原則。

  • 從屬端憑證 - 指定加密通訊和嚴密的從屬端認證 (雙向 SSL)。若要進一步瞭解，請參閱「Oracle Fusion Middleware Developer's Guide for Oracle Service Bus」中的「Configuring Transport-Level Security」，網址為： http://www.oracle.com/pls/as111170/lookup?id=OSBDV1557。
  • 自訂認證 - 指定認證記號會包含在 HTTP 標頭中。從屬端的識別是使用此從屬端提供的記號所建立。您必須設定用來對應記號與 Oracle Service Bus 使用者的識別宣告提供者。

    • 自訂認證記號可以是所設定 WebLogic Server 識別宣告提供者支援的任意作用中記號類型。

分配原則
選取此端點的分配原則。保留空白，則會使用預設分配原則。
分配原則指的是您想用於服務端點的 WLS 工作管理程式執行處理。如需工作管理程式的相關資訊，請參閱 Oracle Fusion Middleware Configuring Server Environments for Oracle WebLogic Server 中的 "Using Work Managers to Optimize Scheduled Work"，網址為： http://www.oracle.com/pls/as111170/lookup?id=CNFGD112。
要求編碼
  • HTTP 內送傳輸：

    • 如果未在「從屬端要求」中指定 Content-Type 標頭的字元集編碼參數，請輸入字元集編碼參數。如果未輸入值，則欄位會預設為 ISO-8859-1。

  • HTTP 外送傳輸：

    • 如果尚未設定要求編碼，Oracle Service Bus 執行時期會在向業務服務提出要求時決定最適當的編碼。如果是非傳遞通道案例，則執行時期的預設字元編碼是 UTF-8。然而，如果是傳遞通道案例，則執行時期會對外送回應完整使用所接收到的編碼。

回應編碼
接受預設值 ISO-8859-1 做為 HTTP 傳輸中回應的字元集編碼，或輸入其他字元集編碼。
認證標頭
輸入 Oracle Service Bus 要從中解壓縮記號的 HTTP 標頭 (Authorization 以外的任何項目)。只有在選取「自訂認證」核取方塊時，才可以使用此欄位。
例如，client-xyz-token。
認證記號類型
選擇認證記號類型。只可以使用設定給標識聲明提供者的作用中記號類型(有關「自訂記號」的詳細資訊，請參閱設定識別宣告提供者)。只有在選取「自訂認證」核取方塊時，才可以使用此欄位。

完成作業後

按「下一頁」繼續在下一個頁面上設定此服務，或按一下「最後一頁」在建立/編輯代理主機服務 - 摘要頁面上檢查並儲存這項組態。
