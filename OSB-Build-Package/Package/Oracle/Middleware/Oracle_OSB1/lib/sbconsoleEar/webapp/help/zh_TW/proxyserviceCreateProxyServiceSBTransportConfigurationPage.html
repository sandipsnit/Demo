SB 傳輸組態頁面

「建立/編輯代理主機服務 - SB 傳輸組態」頁面是一系列頁面中的其中一頁，供您建立和編輯代理主機服務 (如建立和設定代理主機服務和編輯代理主機服務組態中所述)。

如需詳細資訊，請參閱「Oracle Fusion Middleware Developer's Guide for Oracle Service Bus」中的「SB Transport」，網址為：http://www.oracle.com/pls/as111170/lookup?id=OSBDV997。

使用此頁面，可使用 SB (Service Bus) 傳輸通訊協定來設定代理主機服務的傳輸設定。表 19-12 說明如何使用此頁面。

表 19-12 建立/編輯代理主機服務 - SB 傳輸組態頁面 
選項
建立或編輯...
分配原則
選取此端點的分配原則，或使用預設分配原則。
分配原則指的是您希望服務端點在處理要求時使用的 WLS 工作管理程式執行處理。如需工作管理程式的相關資訊，請參閱 Oracle Fusion Middleware Configuring Server Environments for Oracle WebLogic Server 中的 "Using Work Managers to Optimize Scheduled Work"，網址為：http://www.oracle.com/pls/as111170/lookup?id=CNFGD112。
使用 SSL
指定時，必須透過 SSL 連線傳送要求。
不過，不會禁止不安全的連線。管理員必須關閉伺服器上的所有不安全通訊協定 (例如，t3 或 http)，以限制執行安全的從屬端連線。

完成作業後

按「下一頁」繼續在下一個頁面上設定此服務，或按一下「最後一頁」在建立/編輯代理主機服務 - 摘要頁面上檢查並儲存這項組態。
