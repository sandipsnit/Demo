SFTP 傳輸組態頁面

「建立/編輯代理主機服務 - SFTP 傳輸組態」頁面是一系列頁面中的其中一頁，供您建立和編輯代理主機服務 (如建立和設定代理主機服務和編輯代理主機服務組態中所述)。

如需詳細資訊，請參閱 Oracle Fusion Middleware Developer's Guide for Oracle Service Bus 中的 "HTTP and Poller Transports"，網址為： http://www.oracle.com/pls/as111170/lookup?id=OSBDV947。

使用此頁面，可以使用 SFTP 傳輸通訊協定來設定代理主機服務的傳輸設定。表 19-13 說明如何使用此頁面。

表 19-13 建立/編輯代理主機服務 - SFTP 傳輸組態頁面 
選項
建立或編輯...
使用者認證
選取下列其中一項：
  • 使用者名稱密碼認證 - 指定靜態服務帳號與此認證方法關聯，而且使用提供的證明資料對從屬端進行認證。
  • 主機型認證 - 指定需要有使用者名稱和服務金鑰提供者，才能使用此認證方法。所有從已知主機連線的使用者都是使用主機的私密金鑰來認證。
  • 公用金鑰認證 - 指定需要有使用者名稱和服務金鑰提供者，才能使用此認證方法。每位使用者都有他們自己的私密金鑰。
服務帳號
輸入使用者的服務帳號，或按一下「瀏覽」，從瀏覽器中選取服務帳號。
服務金鑰提供者
只有在選取「主機型」或「公用金鑰認證」時，才能使用此選項。
在服務金鑰提供者欄位中輸入服務金鑰提供者。您可以按一下瀏覽，從瀏覽器中選取服務金鑰提供者。這是必要欄位。
使用者名稱
只有在選取「主機型」或「公用金鑰認證」時，才能使用此選項。
輸入使用者名稱。
透過參照來傳遞
選取此核取方塊，將檔案臨時放置於封存檔目錄中，並以參照形式在標頭中傳遞該檔案。
遠端串流
選取此核取方塊，以在處理時直接從遠端伺服器用串流的方式傳送 SFTP 檔案。選取此選項時，封存檔目錄就是遠端 SFTP 伺服器機器上的遠端目錄。因此，應該指定與 SFTP 使用者目錄相對的封存檔目錄。
檔案遮罩
輸入要選擇之檔案的正規表示式。預設值為 *.*。
受管理伺服器
此欄位只能用於叢集網域。
選取要作為輪詢伺服器的受管理伺服器。所有受管理伺服器都可以處理訊息，但只有一個受管理伺服器可以輪詢訊息。
輪詢間隔
輸入從指定的位置輪詢檔案的間隔 (秒)。預設值是 60。
讀取限制
指定每次輪詢清除所讀取的訊息數目上限。請輸入 0，以指定沒有限制。預設值為 10。
讀取後動作
選取訊息讀取後的動作。
  • 封存 - 封存訊息。
  • 刪除 - 刪除訊息。
封存檔目錄
如果讀取後動作選項設為封存檔，請指定封存檔位置的路徑。如果選取透過參照來傳遞選項，則此為必要欄位。
注意： 封存檔、下載和錯誤目錄都是絕對路徑，而且會自動予以建立。如果指定相對路徑，則會在與啟動 WebLogic Server 之 Java 流程相對的位置建立檔案。
下載目錄
輸入本機機器上的目錄，這是檔案傳輸期間下載檔案的位置。
注意： 封存檔、下載和錯誤目錄都是絕對路徑，而且會自動予以建立。如果指定相對路徑，則會在與啟動 WebLogic Server 之 Java 流程相對的位置建立檔案。
錯誤目錄
輸入在發生問題時用來張貼訊息的位置。
注意： 封存檔、下載和錯誤目錄都是絕對路徑，而且會自動予以建立。如果指定相對路徑，則會在與啟動 WebLogic Server 之 Java 流程相對的位置建立檔案。
要求編碼
接受預設值 UTF-8 做為 SFTP 傳輸中要求的字元集編碼。
偏好的密碼組
選取與伺服器通訊時所要用的密碼組。
如果使用預設值 (使用執行時期預設值)，系統會將支援的密碼組清單傳送給伺服器，然後嘗試每個密碼，直到找到相符的密碼為止。
偏好的資料完整性演算法
選取用於資料完整性檢查的批次雜湊演算法。
如果您使用預設值 (使用執行時期預設值)，Oracle Service Bus 會將偏好的 hmac-sha1 演算法傳送給伺服器。如果伺服器不支援該演算法，系統便會傳送支援的演算法清單給伺服器並逐一嘗試，直到找到適用的演算法為止。
偏好的公用金鑰演算法
選取用於公用金鑰加密的非對稱金鑰演算法。
如果您使用預設值 (使用執行時期預設值)，Oracle Service Bus 會將偏好的 ssh-dss 演算法傳送給伺服器。如果伺服器不支援該演算法，系統便會傳送支援的演算法清單給伺服器並逐一嘗試，直到找到適用的演算法為止。
偏好的金鑰交換演算法
選取用來交涉加密訊息之階段作業金鑰的預設金鑰交換協定。
如果您使用預設值 (使用執行時期預設值)，Oracle Service Bus 會將偏好的 diffie-hellman-group1-sha1 演算法傳送給伺服器。如果伺服器不支援該演算法，系統便會傳送支援的演算法清單並逐一嘗試，直到找到適用的演算法為止。
偏好的壓縮演算法
選取是否要使用 zlib 壓縮進行中的資料。請選取 zlib 以壓縮資料，否則請選取無。預設為 None。
掃描子目錄
選取此核取方塊，以遞迴方式掃描所有目錄。
依到達時間排序
選取此核取方塊，依到達的順序傳遞事件。
逾時
輸入在捨棄連線之前的通訊端逾時間隔 (秒)。如果輸入 0，則沒有逾時。預設值為 60。
重試計數
指定 SFTP 連線失敗的重試次數。

完成作業後

按「下一頁」繼續在下一個頁面上設定此服務，或按一下「最後一頁」在建立/編輯代理主機服務 - 摘要頁面上檢查並儲存這項組態。
