摘要頁面

「建立/編輯代理主機服務 - 摘要」頁面是一系列頁面中的其中一頁，供您建立和編輯代理主機服務 (如建立和設定代理主機服務和編輯代理主機服務組態中所述)。

使用此頁面，可以在儲存之前檢視或修改代理主機服務的組態。

若要檢視或修改設定，請按一下適當類別列 (例如一般組態、傳輸組態等) 的編輯。您所能編輯的頁面，視您在建立代理主機服務時所設定的頁面而定。下列清單會顯示所有頁面：

  • 一般組態頁面
  • 訊息類型組態頁面
  • 傳輸組態頁面
  • 通訊協定特定傳輸組態頁面
  • 電子郵件傳輸組態頁面
  • 檔案傳輸組態頁面
  • FTP 傳輸組態頁面
  • HTTP 傳輸組態頁面
  • JCA 傳輸組態頁面
  • JEJB 傳輸組態頁面
  • JMS 傳輸組態頁面
  • MQ 傳輸組態頁面
  • SB 傳輸組態頁面
  • SFTP 傳輸組態頁面
  • Tuxedo 傳輸組態頁面
  • WS 傳輸組態頁面
  • 作業選取組態頁面
  • 訊息處理頁面
