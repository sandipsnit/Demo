建立用於內置表示式的命名空間

「命名空間定義」面板會包含預設命名空間的清單，而您也可以定義新的命名空間。

開始之前

這些指示假設您在「XQuery 表示式編輯器」、「XQuery 條件編輯器」或「XPath 表示式編輯器」中建立或編輯表示式 (如建立和編輯內置 XQuery 和 XPath 表示式中所述)。

若要在內置表示式中建立和使用命名空間

  1. 選取命名空間定義。命名空間定義面板包含預設命名空間清單，以及變數命名空間和使用者定義之命名空間 (如果有的話) 的清單。
  2. 若要定義和新增使用者命名空間：
    1. 按一下「新增命名空間」。
    2. 在字首欄位中，輸入命名空間的唯一識別碼。您不可以重複使用相同的字首。
    3. 在「URI」欄位中，以 http://url/.../ 格式輸入此命名空間的 URL，或以 uddi:server: 格式輸入 URN。
    4. 按一下「新增」，將命名空間新增至「使用者定義的命名空間」清單。
    5. 複製使用者定義的命名空間，並將其貼入 XQuery 表示式、XQuery 條件或 XPath 中 (如建立和編輯內置 XQuery 和 XPath 表示式中所述)。

完成作業後

請如建立和編輯內置 XQuery 和 XPath 表示式所述繼續進行。
