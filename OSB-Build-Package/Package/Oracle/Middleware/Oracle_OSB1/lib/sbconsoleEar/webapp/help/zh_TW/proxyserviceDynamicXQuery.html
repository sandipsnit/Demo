將動態 XQuery 表示式連結至內置 XQuery

您可以在 XQuery 表示式編輯器頁面指定動態 XQuery 表示式，以在執行時期評估預先登錄之 XQuery 資源的名稱。若要進一步瞭解此編輯器的相關資訊，請參閱建立和編輯內置 XQuery 和 XPath 表示式。

定義動態 XQuery 表示式

  1. 選取「動態 XQuery」選項。
  2. 在「輸入 XQuery 資源的表示式」區域中輸入 XQuery 表示式，以在執行時期評估預先登錄之 XQuery 資源的名稱。

    3. 下列顯示 XQuery 資源的語法 (代表資源的完整名稱)：

    Project/folder1/folder2/XQueryResourceName

  3. 在「使用 XQuery 樣板連結變數，或連結自訂變數」區塊中，定義進行轉換的輸入參數。
    • 按一下選取 XQuery 樣板欄位中的「瀏覽」，以選取現有的已登錄資源作為查詢形式的樣板 (變數號碼和名稱)。選取樣板之後，變數就會出現在連結變數區域中。請注意，此組態不會持續保有樣板。而是將樣板作為協助您指定查詢之變數的快速起點。
    • 在新增自訂變數欄位中輸入變數名稱，然後按一下「新增」。針對變數名稱下所列的每個變數，輸入要與其對應的 XQuery 表示式。您必須定義每個參數的對應。例如，如果 XQuery 轉換具有兩個輸入參數 (名稱分別為 one 和 two)，變數名稱欄位就會有兩個標籤：one 和 two。每個標籤都會有一個可以輸入 XQuery 表示式的相關聯文字方塊。

      • 下列 XQuery 表示式是變數欄位之有效輸入的範例：

      $body/*[1]

      $body/po:PurchaseOrder

  4. 完成之後，請繼續下列任一工作。
    • 按一下驗證。如果成功驗證表示式，則會顯示一則訊息。
    • 按一下測試。請參閱執行 XQuery 測試。
    • 儲存或捨棄變更。
