編輯錯誤處理程式頁面

您可以設定訊息流程、管線、路由節點和階段層級的錯誤處理。

在編輯錯誤處理程式頁面中設定錯誤處理程式。您至少必須要在此頁面上新增一個階段，才能指定錯誤處理程式的運作方式。請參閱下列主題：

  • 新增代理主機服務錯誤處理程式
  • 新增管線錯誤處理程式
  • 新增階段錯誤處理程式
  • 新增路由節點錯誤處理程式
  • 編輯錯誤處理程式
  • Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus 中的 "Handling Errors in Message Flows"，網址為：http://www.oracle.com/pls/as111170/lookup?id=OSBAG232。
