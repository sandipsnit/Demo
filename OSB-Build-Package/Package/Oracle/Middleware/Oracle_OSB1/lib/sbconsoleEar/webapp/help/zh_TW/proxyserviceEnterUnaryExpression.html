使用製作器選項輸入一元表示式

開始之前

這些指示假設您在「XQuery 條件編輯器」中建立或編輯 XQuery 條件表示式 (如建立和編輯內置 XQuery 和 XPath 表示式中所述)。

若要輸入 XQuery 比較表示式：

  1. 在「XQuery 條件編輯器」中，選取「製作器」選項 (位在工作區按鈕列下方) (如果尚未選取)。 選項。
  2. 在「表示式製作器」方塊中，選取「一元表示式」(如果尚未選取)。
  3. 選取「Not」核取方塊讓此表示式變成負表示式，或保留空白。
  4. 在「表示式」欄位中，輸入相關資訊環境變數、命名空間定義或 XQuery 函數。

    5. 若要建置表示式，可以從 XQuery 函數面板貼上 XQuery 函數，以及從命名空間定義面板貼上命名空間。請參閱在編輯器工作區文字欄位中建置表示式中的步驟 3 和步驟 4。

  5. 按一下「新增」，將文字新增至「表示式」窗格。
  6. 重複步驟 3-5，以建置其他條件。每個條件都會新增至條件清單的結尾。

    7. 建置一元表示式時，請考慮下列項目。

    • 建置其他表示式時，請務必選取「結合」欄位中的「And」或「Or」選項。
    • 您可以選取條件，並按一下「向上」箭頭將該條件在條件清單中上移一個位置、按一下「向下」箭頭將該條件在條件清單中下移一個位置、按一下「編輯」圖示來更新該條件，或按一下「刪除」圖示來刪除該條件。
    • 在整個條件定義中，可以混合使用一元表示式和比較表示式。
  7. 選擇性地執行下列任一項或兩項操作：
    • 按一下驗證。如果成功驗證表示式，則會顯示一則訊息。
    • 按一下測試來測試表示式。請參閱執行 XQuery 測試。
  8. 按一下「儲存」關閉編輯器，並在動作中插入表示式。

完成作業後

繼續設定動作 (如新增和編輯訊息流程的動作中所述)。
