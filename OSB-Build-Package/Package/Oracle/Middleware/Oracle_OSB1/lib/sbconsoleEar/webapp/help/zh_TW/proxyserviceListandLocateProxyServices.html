尋找代理主機服務

  1. 請執行下列任一項操作：
    • 選取專案瀏覽器以顯示專案檢視畫面頁面或專案/資料夾檢視畫面頁面。然後瀏覽專案和資料夾，以尋找代理主機服務。
    • 選取資源瀏覽器 > 代理主機服務。代理主機服務的摘要頁面會顯示表 19-18 中所顯示的資訊。有關特性的詳細描述，請參閱編輯代理主機服務組態。
  2. 若要限制清單中的項目數，則可以依據名稱和 (或) 路徑來進行篩選。在「名稱」和「路徑」欄位的「搜尋」下，輸入目標的名稱和 (或) 路徑，然後按一下「搜尋」按鈕。

    3. 此路徑是專案名稱和代理主機服務所在資料夾的名稱。

    萬用字元 * 和 ? 皆可使用。搜尋會區分大小寫。

    按一下檢視全部可顯示網域中的所有代理主機服務。這樣會清除前次搜尋的搜尋參數。

    表 19-18 代理主機服務的摘要頁面 
    特性
    描述
    名稱
    代理主機服務的唯一名稱。按一下名稱，以顯示檢視代理主機服務 - 組態詳細資訊頁面。
    請參閱編輯代理主機服務組態。
    路徑
    此路徑是專案名稱和代理主機服務所在資料夾的名稱，例如，UDDI/Proxies/OSB_services。
    按一下代理主機服務的路徑，以顯示包含它的專案/資料夾檢視畫面頁面。
    動作
    請執行下列任何操作：
    • 按一下編輯訊息流程圖示，以編輯代理主機服務的訊息流程。請參閱建立、編輯和檢視訊息流程。
    • 按一下啟動測試主控台圖示來叫用「測試主控台」，您可以用此來驗證和測試服務及轉換的設計。在業務服務中，只能在執行期間 (也就是說，在階段作業啟動時) 使用「測試主控台」。至於轉換，則不論在階段作業內或外，都可以使用「測試主控台」。請參閱測試服務和測試轉換。
    • 將針對 WSDL 型業務服務顯示匯出 WSDL 圖示。按一下此圖示以匯出 WSDL，您接著可以使用外部工具 (例如 IDE) 來檢視或修改該 WSDL。

      • 請注意，這和「系統管理」模組中的「匯出資源」功能不同，該功能用來在兩個網域間移動及計劃資源。請參閱匯出 WSDL。

    選項
    按一下刪除圖示刪除服務。當其他資源參照此資源時，會顯示刪除警告圖示。確認警告後即可刪除資源。因為仍有已刪除資源的未解析參照，所以可能會導致衝突。有關詳細資訊，請參閱刪除代理主機服務。
