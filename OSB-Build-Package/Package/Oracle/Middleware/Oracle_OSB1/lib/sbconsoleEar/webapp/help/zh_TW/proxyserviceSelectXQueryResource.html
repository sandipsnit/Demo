將外部 XQuery 資源連結至內置 XQuery

您可以將 XQuery 資源連結至內置 XQuery 表示式，這樣它們就會以內置方式做為動作的一部分來執行。

開始之前

這些指示假設您在「XQuery 表示式編輯器」、「XQuery 條件編輯器」或「XPath 表示式編輯器」中建立或編輯表示式 (如建立和編輯內置 XQuery 和 XPath 表示式中所述)。

若要將 XQuery 資源連結至內置表示式：

  1. 按一下「變數結構」。
  2. 在按鈕列下的工作區中，選取「XQuery 資源」。
  3. 在 1.選取要執行的 XQuery 資源方塊中，按一下瀏覽。
  4. 在「XQuery 瀏覽器」中，選取與想要使用之 XQuery 相關的單選按鈕，然後按一下「提交」。
  5. 在 2.連結變數方塊中，定義轉換的輸入參數。針對變數名稱下所列的每個變數，輸入要與其對應的 XQuery 表示式。您必須定義每個參數的對應。例如，如果 XQuery 轉換具有兩個輸入參數 (名稱分別為 one 和 two)，變數名稱欄位就會有兩個標籤：one 和 two。每個標籤都會有一個可以輸入 XQuery 表示式的相關聯文字方塊。

    6. 下列 XQuery 表示式是此欄位之有效輸入的範例：

    $body/*[1]

    $body/po:PurchaseOrder

    注意： 下列變數名稱不是此欄位的有效輸入，而且會導致異常狀況：
    body
  6. 完成之後，請繼續下列任一工作。
    • 按一下驗證。如果成功驗證表示式，則會顯示一則訊息。
    • 按一下測試。請參閱執行 XQuery 測試。
    • 儲存或捨棄變更。
