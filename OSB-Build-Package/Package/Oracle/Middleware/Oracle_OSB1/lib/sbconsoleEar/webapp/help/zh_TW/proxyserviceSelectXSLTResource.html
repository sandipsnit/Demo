將外部 XSLT 資源連結至內置 XQuery

您可以在 XQuery 表示式編輯器頁面選取要執行的 XSLT 資源。若要進一步瞭解此編輯器的相關資訊，請參閱建立和編輯內置 XQuery 和 XPath 表示式。

若要選取要執行的 XSLT 資源

  1. 選取「XSLT 資源」選項。
  2. 在「選取要執行的 XSLT 資源」欄位下選取「XSLT 瀏覽器」圖示。
  3. 在「XSLT 瀏覽器」中，選取與想要執行之 XSLT 相關聯的單選按鈕，然後按一下「提交」。
  4. 在連結變數欄位下，會針對轉換作業的每個輸入參數顯示標籤和對應的文字方塊。每個標籤都會對應到一個參數的名稱，而每個文字方塊是用來定義要對應至參數的 XQuery 表示式。您必須定義每個參數的對應。例如，如果 XSL 轉換具有兩個輸入參數 (分別名為 one 和 two)，變數對應欄位就會有兩個標籤：one 和 two，且每個標籤都會有一個可以輸入 XQuery 表示式的相關聯文字方塊。除了所有輸入變數的對應外，您還必須指定做為轉換作業之「輸入文件」的 XQuery 表示式。該對應是在具有「輸入文件」標籤的文字方塊中指定。

    5. 下列 XQuery 表示式是此欄位之有效輸入的範例：

    $body/*[1]

    $body/po:PurchaseOrder

    注意： 下列變數名稱不是此欄位的有效輸入，而且會導致異常狀況：
    body
  5. 請繼續進行下列任一項工作：
    • 建立用於內置表示式的命名空間。
    • 在 XQuery 編輯器中建立變數結構。
    • 在編輯器工作區文字欄位中建置表示式。
    • 將外部 XQuery 資源連結至內置 XQuery。
    • 按一下驗證。如果成功驗證表示式，則會顯示一則訊息。
    • 按一下測試。請參閱執行 XQuery 測試。
