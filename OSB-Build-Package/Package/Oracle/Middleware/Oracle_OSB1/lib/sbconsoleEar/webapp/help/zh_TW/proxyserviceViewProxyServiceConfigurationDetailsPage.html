檢視代理主機服務 - 組態詳細資訊頁面

檢視代理主機服務 - 組態詳細資訊頁面會顯示代理主機服務的組態詳細資訊。 表 19-19 描述此頁面上顯示的所有特性(特性會根據代理主機服務的詳細資訊而不同)。

列在此頁面上的類別會對應至用於建立和編輯代理主機服務設定的「建立/編輯代理主機服務」頁面 (如建立/編輯代理主機服務 - 頁面參照中所述)。

按一下任一類別名稱旁的「編輯」連結，以顯示關聯的組態頁面。
