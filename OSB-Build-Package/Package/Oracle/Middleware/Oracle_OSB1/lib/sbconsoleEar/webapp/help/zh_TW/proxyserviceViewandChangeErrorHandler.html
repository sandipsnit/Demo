編輯錯誤處理程式

開始之前

這些指示假設您已在編輯訊息流程頁面中編輯訊息流程 (如建立、編輯和檢視訊息流程中所述)。

若要檢視和變更錯誤處理程式

  1. 請執行下列其中一項操作。

    2. 表 23-1 檢視和變更錯誤處理程式 
    若要...
    請完成此步驟...
    檢視和變更代理主機服務訊息流程錯誤處理程式
    按一下代理主機服務圖示，然後按一下編輯服務錯誤處理程式。隨即顯示編輯錯誤處理程式頁面。請參閱新增代理主機服務錯誤處理程式。
    檢視和變更管線錯誤處理程式
    按一下適當的要求管線圖示或回應管線圖示，然後按一下編輯管線錯誤處理程式。隨即顯示編輯錯誤處理程式頁面。請參閱新增管線錯誤處理程式。
    檢視和變更路由節點錯誤處理程式
    按一下適當的路由節點圖示，然後按一下編輯路由錯誤處理程式。隨即顯示編輯錯誤處理程式頁面。請參閱新增路由節點錯誤處理程式。
    檢視和變更階段錯誤處理程式
    按一下適當的階段圖示，然後按一下編輯階段錯誤處理程式。隨即顯示編輯錯誤處理程式頁面。請參閱新增階段錯誤處理程式。

完成作業後

在編輯訊息流程頁面上，繼續建構訊息流程 (如建立、編輯和檢視訊息流程中所述)。
