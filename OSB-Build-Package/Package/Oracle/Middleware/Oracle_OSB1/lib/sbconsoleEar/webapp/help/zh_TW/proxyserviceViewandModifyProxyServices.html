編輯代理主機服務組態

本節說明如何編輯代理主機服務組態。

  1. 如果尚未進行這樣的處理，請按一下建立以建立新的階段作業，或按一下編輯以輸入現有的階段作業。請參閱使用變更中心。
  2. 尋找想要檢視或編輯的代理主機服務。請參閱尋找代理主機服務。
  3. 按一下代理主機服務名稱。檢視代理主機服務 - 組態詳細資訊頁面會顯示選定代理主機服務的組態資訊。
  4. 若要檢視或修改設定，請執行下列任一項操作：
    • 按一下想要檢視或編輯特性之類別名稱 (例如一般組態、傳輸組態等) 旁的編輯圖示。您所能編輯的頁面，視您在建立代理主機服務時所設定的頁面而定。

      • 有關所有這些頁面的清單，請參閱檢視代理主機服務 - 組態詳細資訊頁面。

    • 按一下頁面底端的「編輯」，以顯示建立/編輯代理主機服務 - 一般組態頁面，這是一系列用來設定此代理主機服務的頁面中所顯示的第一頁。
  5. 繼續檢視或編輯 (如建立和設定代理主機服務中所述)。
  6. 在編輯代理主機服務 - 摘要頁面上，按一下「儲存」以確認目前階段作業中的更新。
  7. 若要結束階段作業並將組態建置至執行時期，請按一下「變更中心」下的「啟動」。
