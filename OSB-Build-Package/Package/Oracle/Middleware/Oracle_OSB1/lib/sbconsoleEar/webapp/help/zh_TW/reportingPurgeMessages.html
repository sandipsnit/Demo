清除訊息

使用此頁面可以清除報告資料庫中的選定訊息。

訊息清除會在背景進行，因此進行清除時，並不會鎖定訊息報告摘要頁面。因此，您可以執行需要一些時間完成的清除；而時間的長短則是取決於需要清除的訊息數。不過，如果篩選報告訊息，則清除流程會變慢；此外因為部分資料尚未清除完成，所以「訊息報告摘要」頁面可能會顯示錯誤的資料。

「清除訊息」功能在專案的設計和測試階段可能十分有用。不過，在生產環境中，Oracle 建議由資料庫管理員使用資料庫管理工具來管理您資料庫中的資料 (包括清除)。

  1. 選取「作業」>「清除訊息」。
  2. 執行下列其中一個步驟，如表 27-9 中所示。

    3. 表 27-9 訊息清除選項
    若要...
    請完成這些步驟...
    清除所有訊息
    1. 選取「清除所有訊息」單選按鈕。
    2. 按一下「清除」。
    清除所指定時段內的訊息
    1. 選取「清除訊息自... 目標 ... 」單選按鈕。
    2. 在清除訊息自欄位中，以 MM/DD/YY HH:MM:SS 格式輸入月、日、年和時間。例如，您可以輸入 10/10/07 12:45:00 AM。
    3. 在「目標」欄位中，輸入 MM/DD/YY HH:MM:SS 格式的月、日、年和時間。
    4. 按一下「清除」。
