新增分割連接

開始之前

分割連接需要 .flow 檔，此檔案是在您使用 Eclipse 專用的 Oracle Service Bus 外掛程式建立分割連接時建立，或將分割連接組態 JAR 匯入外掛程式時建立。.flow 檔可能具有代理主機服務、業務服務或外部 WSDL 資源的參照。在啟動分割連接之前，分割連接所依據的資源必須已在伺服器上。

  1. 如果尚未進行這樣的處理，請按一下建立以建立新的階段作業，或按一下編輯以輸入現有的階段作業。請參閱使用變更中心。
  2. 選取專案瀏覽器，然後選取要在其中新增「分割連接」的專案或資料夾。隨即顯示專案/資料夾檢視畫面頁面。
  3. 從「建立資源」下拉清單中，選取「分割連接」，以顯示「建立新的分割連接資源」頁面。

    4. 指定表 9-2 中所示的參數。

    表 9-2 分割連接參數
    特性
    建立或編輯...
    資源名稱
    輸入此分割連接的唯一名稱。

    請遵循資源命名限制的命名指南。
    資源描述
    輸入分割連接的描述。
    分割連接
    按一下瀏覽在系統上找出 .flow 檔案，一般位於 MW_HOME\user_projects\workspaces\your_workspace 下。
    分配原則
    選取 WebLogic Server 工作管理程式的執行處理，以用此執行處理來建立用以執行分割連接的繫線 (並行分支)。如果不存在其他工作管理程式，則會使用預設的工作管理程式。

  4. 按一下「儲存」。會建立分割連接並將它儲存在目前階段作業中。
    5. 注意： 分割連接所依據的資源必須已存在，才能啟動分割連接。
  5. 若要結束階段作業並將組態建置至執行時期，請按一下「變更中心」下的「啟動」。
