新增 JAR

  1. 如果尚未進行這樣的處理，請按一下建立以建立新的階段作業，或按一下編輯以輸入現有的階段作業。請參閱使用變更中心。
  2. 選取專案瀏覽器，然後選取要在其中新增 JAR 的專案或資料夾。隨即顯示專案/資料夾檢視畫面頁面。
  3. 從「建立資源」下拉清單的「公用程式」中選取「JAR」，以顯示「建立新的 JAR 資源」頁面。
  4. 在資源名稱欄位中，輸入此 JAR 的唯一名稱。這是必要欄位。

    請遵循資源命名限制的命名指南。
  5. 在「資源描述」欄位中，輸入 JAR 的描述。
  6. 在「JAR 來源檔案」欄位中，請執行下列其中一項操作：
    • 輸入 JAR 位置的路徑。
    • 按一下「瀏覽」以尋找和匯入現有的 JAR。

      • 這是必要欄位。

  7. 按一下「儲存」。會建立 JAR 資源，並儲存在目前的階段作業中。

    8. 如果載入 JAR 時未發生問題，則可以檢視該 JAR 的內容。若要瞭解如何檢視 JAR 詳細資訊，請參閱編輯 JAR。

  8. 若要結束階段作業並將組態建置至執行時期，請按一下「變更中心」下的「啟動」。
