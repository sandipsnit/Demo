新增 MQ 連線

開始之前

請參閱 Oracle Fusion Middleware Supported System Configurations，網址為：http://www.oracle.com/technology/software/products/ias/files/fusion_certification.html，瞭解 Oracle Service Bus 支援的 WebSphere MQ 版本。WebSphere MQ 從屬端程式庫未與 Oracle Service Bus 安裝程式組合在一起。請從您的 WebSphere MQ 安裝取得 WebSphere MQ 從屬端程式庫。

若要在 Oracle Service Bus 設定及使用 MQ 連線資訊和 MQ 傳輸，您必須將 MQ 從屬端程式庫新增至類別路徑。其中一個選項可將 com.ibm.mq.jar 程式庫新增至 DOMAIN_NAME/lib 目錄，其中 DOMAIN_NAME 是網域所在的目錄名稱，一般是 MW_HOME\user_projects\domains\DOMAIN_NAME。

若要新增 MQ 連線資源

  1. 如果尚未進行這樣的處理，請按一下建立以建立新的階段作業，或按一下編輯以輸入現有的階段作業。請參閱使用變更中心。
  2. 選取專案瀏覽器，然後選取要在其中新增「MQ 連線」資源的專案或資料夾。隨即顯示專案/資料夾檢視畫面頁面。
  3. 從「建立資源」下拉清單的「公用程式」中選取「MQ 連線」，以顯示「建立新的 MQ 連線資源」頁面。

    4. 指定表 8-2 中所顯示的連線參數。

    資源名稱
    輸入此「MQ 連線」資源的唯一名稱。

    請遵循資源命名限制的命名指南。
    資源描述
    輸入 MQ 連線資源的描述。
    連線類型

    • tcp 模式 - 使用 TCP/IP 以連線至與 Oracle Service Bus 位在不同機器上的佇列管理程式。
    • 連結模式 - 使用連結模式以連線至與 Oracle Service Bus 位在相同機器的佇列管理程式。

    輸入 MQ 佇列管理程式的主機名稱。

    輸入 MQ 佇列管理程式監聽器的連接埠號碼。
    MQ 佇列管理程式名稱
    輸入要連線之 MQ 佇列管理程式的名稱。
    佇列管理程式 CCSID
    僅限 tcp mode 連線：
    建立連線時會使用編碼字元集 ID (CCSID)。CCSID 的主要用途是國際化支援。
    若要進一步瞭解，請參閱 http://www.redbooks.ibm.com/redbooks/SG247128/wwhelp/wwhimpl/java/html/wwhelp.htm 上的 WebSphere MQ 基礎架構（英文）。
    MQ 佇列管理程式通道名稱
    僅限 tcp mode 連線：
    輸入佇列管理程式伺服器連線通道名稱。
    需要 SSL
    僅限 tcp mode 連線：
    選取此核取方塊，以使用 SSL 來傳送訊息。如果「未」選取「需要雙向 SSL」選項，則只會啟用伺服器端 SSL。
    密碼組
    此選項只適用於選取「需要 SSL」核取方塊時。
    選取 SSL 使用的密碼組演算法。
    密碼組演算法用來加密和解密 WebSphere MQ 伺服器與 WebSphere MQ 從屬端之間的訊息通訊。因此，與 WebSphere MQ 伺服器之間使用 SSL 通訊時，必須指定密碼組演算法。
    需要雙向 SSL
    此選項只適用於選取「需要 SSL」核取方塊時。
    選取核取方塊來同時啟用從屬端與伺服器端 SSL 認證。
    服務金鑰提供者的參照
    如果已選取「需要雙向 SSL」，則必須提供服務金鑰提供者的參照，以取得適當的金鑰儲存庫和信任儲存庫資訊。
    輸入路徑 (project/folder) 和服務金鑰提供者名稱，或按一下「瀏覽」，從「選取服務金鑰提供者」頁面中選取服務金鑰提供者。
    靜態服務帳號的參照
    僅限 tcp mode 連線：
    進行使用者名稱和密碼認證的必要項目。輸入路徑 (project/folder) 和靜態服務帳號的名稱，或按一下「瀏覽」，從瀏覽器中選取服務帳號。
    WebSphere MQ 版本
    選取 WebSphere MQ 版本：
    • 5.3
    • 6.0
    • 7.0

    注意：IBM 已不再支援 5.3 及 6.0 版本。Oracle Service Bus 已經不再使用這些版本。
    MQ 連線集區大小
    輸入 MQ 連線集區的大小。
    MQ 連線逾時
    輸入時間間隔 (秒)，此時間後便會刪除未使用的連線。預設值為 1800 秒。
    MQ 連線最長等待時間
    輸入等待連線變為可用的最長等待時間 (毫秒)。如果未在這個時間間隔內建立連線，Oracle Service Bus 會發生異常狀況。預設是 3000 毫秒。

  4. 按一下「儲存」。會建立「MQ 連線」資源，並儲存在目前的階段作業中。
  5. 若要結束階段作業並將組態建置至執行時期，請按一下「變更中心」下的「啟動」。
