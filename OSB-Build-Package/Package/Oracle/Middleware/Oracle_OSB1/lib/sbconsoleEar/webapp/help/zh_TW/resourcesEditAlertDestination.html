編輯警示目標

  1. 如果尚未進行這樣的處理，請按一下建立以建立新的階段作業，或按一下編輯以輸入現有的階段作業。請參閱使用變更中心。
  2. 尋找警示目標 (如尋找警示目標或尋找警示目標中所述)。
  3. 按一下警示目標名稱。「檢視警示目標 - 組態」頁面會顯示表 5-2 中所顯示的資訊。

    4. 表 5-2 警示目標組態詳細資訊 
    特性
    描述
    上次修改者
    建立或編輯此警示目標的使用者。
    上次修改時間
    使用者建立或編輯此警示目標的日期和時間。按一下日期和時間連結，以檢視此資源的變更歷史記錄。請參閱檢視變更歷史記錄頁面。
    參照
    此警示目標所參照的物件數。如果有這類參照，請按一下數字連結以檢視物件清單。請參閱檢視資源參照。
    參照者
    參照此警示目標的物件數。如果有這類參照，請按一下數字連結以檢視物件清單。請參閱檢視資源參照。
    描述
    此警示目標的描述 (如果有的話)。
    SNMP 陷阱
    「是」或「否」，表示設定此警示時，是否指定「SNMP 陷阱」選項做為警示的目標。
    報告
    「是」或「否」，表示設定此警示時，是否指定「報告」選項做為警示的目標。
    警示記錄日誌
    「是」或「否」，表示警示記錄日誌是否啟用。
    電子郵件與 JMS
    如果已設定電子郵件收件者和 (或) JMS 目標，則會顯示它們的清單。

  4. 您可以變更所選擇的「SNMP 陷阱」、「報告」，或「警示記錄日誌」設定。
  5. 若要編輯或刪除電子郵件收件者或 JMS 目標，請按一下頁面底端的「編輯」。
    1. 若要編輯電子郵件收件者，請在電子郵件的選項下按一下要編輯之電子郵件收件者的編輯圖示。使用編輯電子郵件收件者頁面，可變更電子郵件收件者資訊。請參閱新增電子郵件收件者。
    2. 若要編輯 JMS 目標，請在 JMS 的選項下按一下想要編輯之 JMS 目標的編輯圖示。使用編輯 JMS 目標頁面，可變更 JMS 目標資訊。請參閱新增 JMS 目標。
    3. 若要刪除一個或所有電子郵件收件者或 JMS 目標，請在「選項」下按一下想要刪除之電子郵件收件者或 JMS 目標的「刪除」圖示。
  6. 按一下「儲存」，確認目前階段作業中的更新。
  7. 若要結束階段作業並將組態建置至執行時期，請按一下「變更中心」下的「啟動」。
