編輯分割連接

  1. 如果尚未進行這樣的處理，請按一下「建立」以建立新的階段作業，或按一下編輯以輸入現有的階段作業。請參閱使用變更中心。
  2. 尋找分割連接 (如尋找分割連接中所述)。
  3. 按一下分割連接名稱。

    4. 「檢視分割連接 - 組態」頁面會顯示表 9-3 中所示的資訊。

    表 9-3 分割連接詳細資訊 
    特性
    描述
    上次修改者
    建立此分割連接或將其匯入至組態中的使用者。
    上次修改時間
    使用者建立此分割連接或將其匯入組態中的日期和時間。按一下日期和時間連結，以檢視此資源的變更歷史記錄。請參閱檢視變更歷史記錄頁面。
    參照
    此分割連接參照的物件數。如果有這類參照，請按一下數字連結以檢視物件清單。請參閱檢視資源參照。
    參照者
    參照此分割連接的物件數。如果有這類參照，請按一下數字連結以檢視物件清單。請參閱檢視資源參照。
    描述
    此分割連接的描述 (若有的話)。

  4. 您可以對分割連接進行下列作業設定：

    5. 表 9-4 分割連接作業設定
    特性
    描述
    日誌
    如果您想要擷取分割連接中的日誌資訊，請選取「啟用記錄」，並選取想要擷取的記錄層級。
    執行追蹤
    如果您想要啟用執行時期追蹤，請選取「已啟用」。

  5. 按一下「更新」，確認目前階段作業中的更新。
  6. 若要結束階段作業並將組態建置至執行時期，請按一下「變更中心」下的「啟動」。
