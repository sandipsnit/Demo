編輯 MQ 連線

本節說明如何編輯現有的 MQ 連線資源。

  1. 如果尚未進行這樣的處理，請按一下「建立」以建立新的階段作業，或按一下編輯以輸入現有的階段作業。請參閱使用變更中心。
  2. 尋找 MQ 連線資源 (如尋找 MQ 連線中所述)。
  3. 按一下 MQ 連線資源名稱。

    4. 「檢視 MQ 連線 - 組態」頁面會顯示表 8-3 中所顯示的資訊。

    表 8-3 MQ 連線詳細資訊 
    特性
    描述
    上次修改者
    建立此 MQ 連線資源或將它匯入組態中的使用者。
    上次修改時間
    使用者建立此 MQ 連線資源或將其匯入組態的日期和時間。按一下日期和時間連結，以檢視此資源的變更歷史記錄。請參閱檢視變更歷史記錄頁面。
    參照
    此 MQ 連線資源參照的物件數。如果有這類參照，請按一下數字連結以檢視物件清單。請參閱檢視資源參照。
    參照者
    參照此 MQ 連線資源的物件數。如果有這類參照，請按一下數字連結以檢視物件清單。請參閱檢視資源參照。
    描述
    此 MQ 連線資源的描述 (如果有的話)。

  4. 若要變更欄位，請按一下編輯。有關欄位的描述，請參閱新增 MQ 連線。

    5. 您不可以變更「資源名稱」欄位。

  5. 按一下「儲存」，確認目前階段作業中的更新。
  6. 若要結束階段作業並將組態建置至執行時期，請按一下「變更中心」下的「啟動」。
