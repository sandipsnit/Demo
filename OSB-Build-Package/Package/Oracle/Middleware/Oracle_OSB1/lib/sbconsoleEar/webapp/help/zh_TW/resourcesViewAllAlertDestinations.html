尋找警示目標

您可以在警示目標摘要頁面中檢視警示目標清單。如果尚未建立警示目標，請從「專案瀏覽器」中建立新的警示目標資源。若要瞭解作法，請參閱新增警示目標。

  1. 請執行下列任一項操作：
    • 選取專案瀏覽器以顯示專案檢視畫面頁面或專案/資料夾檢視畫面頁面。然後瀏覽專案和資料夾，以尋找警示目標。
    • 選擇資源瀏覽器 > 警示目標。警示目標摘要頁面會顯示表 5-1 中所顯示的資訊。
  2. 若要限制清單中的項目數，則可以依據名稱和 (或) 路徑來進行篩選。在「名稱」和「路徑」欄位中，輸入搜尋目標的名稱和路徑，然後按一下「搜尋」。

    3. 路徑是專案名稱和警示目標所在資料夾的名稱。

    按一下「檢視全部」移除搜尋篩選，並顯示所有警示目標。

    表 5-1 警示目標詳細資訊
    特性
    描述
    名稱
    警示目標資源的名稱。此名稱是連到檢視警示目標 - 組態頁面的連結。請參閱編輯警示目標。
    路徑
    路徑是專案名稱和警示目標所在資料夾的名稱。它是連到包含此資源之專案或資料夾的連結。請參閱使用專案和資料夾限定資源名稱。
    選項
    按一下刪除圖示，以刪除特定「警示目標」。請參閱刪除警示目標。
