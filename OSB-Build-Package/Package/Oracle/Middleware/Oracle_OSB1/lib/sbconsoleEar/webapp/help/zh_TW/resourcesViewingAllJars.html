尋找 JAR

  1. 請執行下列任一項操作：
    • 選取專案瀏覽器以顯示專案檢視畫面頁面或專案/資料夾檢視畫面頁面。然後瀏覽專案和資料夾，以尋找 JAR。
    • 選擇資源瀏覽器 > JAR。JAR 的摘要頁面會顯示表 7-1 中所顯示的資訊。有關特性的更詳細描述，請參閱編輯 JAR。
  2. 若要限制清單中的項目數或尋找特定 JAR，請使用名稱和路徑篩選來搜尋 JAR。在「名稱」和「路徑」欄位中，輸入搜尋目標的名稱和路徑，然後按一下「搜尋」。

    3. 路徑是專案名稱和 JAR 所在資料夾的名稱。

    此搜尋支援萬用字元。例如，您可以在「名稱」欄位中輸入 "j*"，以取得開頭為 "j" 的 JAR 清單。

    按一下「檢視全部」移除搜尋篩選，並顯示所有 JAR。

    表 7-1 JAR 詳細資訊
    特性
    描述
    JAR 名稱
    指派給 JAR 的唯一名稱。此名稱是連到檢視 JAR 資源 - JAR 組態頁面的連結。請參閱編輯 JAR。
    路徑
    路徑是專案名稱和 JAR 所在資料夾的名稱。它是連到包含此資源之專案或資料夾的連結。請參閱使用專案和資料夾限定資源名稱。
    類別數
    JAR 所含的類別數。
    大小
    JAR 的位元組大小。
    選項
    按一下刪除圖示，以刪除特定 JAR。請參閱刪除 JAR。
