尋找 MQ 連線

  1. 請執行下列任一項操作：
    • 選取專案瀏覽器以顯示專案檢視畫面頁面或專案/資料夾檢視畫面頁面。然後瀏覽專案和資料夾，以尋找 MQ 連線資源。
    • 選擇資源瀏覽器 > MQ 連線。MQ 連線的摘要頁面會顯示表 8-1 中所顯示的資訊。有關特性的更詳細描述，請參閱編輯 MQ 連線。
  2. 若要限制清單中的項目數，則可以依據名稱和 (或) 路徑來進行篩選。在「名稱」和「路徑」欄位中，輸入搜尋目標的名稱和路徑，然後按一下「搜尋」。
  3. 按一下開啟圖示，以顯示其他搜尋篩選。如果指定了多個搜尋條件，則只會傳回符合全部條件的結果。
    • 依據佇列管理程式名稱進行篩選。輸入任何字串，並在其中包括萬用字元，然後按一下搜尋。
    • 依據佇列管理程式通道名稱進行篩選。輸入任何字串，並在其中包括萬用字元，然後按一下搜尋。
    • 按一下「檢視全部」移除搜尋篩選，並顯示所有 MQ 連線資源。

      • 表 8-1 MQ 連線詳細資訊
      特性
      描述
      MQ 連線名稱
      指派給「MQ 連線」資源的名稱。此名稱是連到 MQ 連線組態頁面的連結。請參閱編輯 MQ 連線。
      路徑
      路徑是專案名稱和 MQ 連線資源所在資料夾的名稱。它是連到包含此資源之專案或資料夾的連結。請參閱使用專案和資料夾限定資源名稱。
      選項
      按一下刪除圖示，以刪除特定「MQ 連線」資源。請參閱刪除 MQ 連線。
