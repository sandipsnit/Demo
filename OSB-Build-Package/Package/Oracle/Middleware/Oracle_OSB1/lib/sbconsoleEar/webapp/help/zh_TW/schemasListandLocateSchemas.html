尋找 XML 綱要

  1. 請執行下列任一項操作：
    • 選取專案瀏覽器以顯示專案檢視畫面頁面或專案/資料夾檢視畫面頁面。然後瀏覽專案和資料夾，以尋找 XML 綱要。
    • 選擇資源瀏覽器 > XML 綱要。XML 綱要的摘要頁面會顯示表 4-1 中所顯示的資訊。有關特性的更詳細描述，請參閱編輯 XML 綱要。
  2. 若要尋找特定 XML 綱要：
    • 按一下開啟圖示，以顯示其他搜尋篩選。如果指定了多個搜尋條件，則只會傳回符合全部條件的結果。
    • 若要限制清單中的項目數，則可以依據名稱、路徑和命名空間來進行篩選。在「名稱」、「路徑」和「命名空間」欄位中，輸入搜尋目標的名稱、路徑和命名空間，然後按一下「搜尋」。

      • 此路徑是專案名稱及 XML 綱要所在資料夾的名稱。

    • 按一下「檢視全部」移除搜尋篩選，並顯示所有 XML 綱要。

      • 表 4-1 XML 綱要資訊
      特性
      描述
      XML 綱要名稱
      指定給 XML 綱要的唯一名稱。此名稱是連到 XML 綱要詳細資訊頁面的連結。請參閱編輯 XML 綱要。
      路徑
      此路徑是專案名稱及 XML 綱要所在資料夾的名稱。它是連到包含此資源之專案或資料夾的連結。請參閱使用專案和資料夾限定資源名稱。
      XML 綱要命名空間
      用來限定 XML 綱要中所包含之任何定義的命名空間。
      選項
      按一下刪除圖示，以刪除特定 XML 綱要。請參閱刪除 XML 綱要。
