解析未解析的 XML 綱要參照

使用「編輯 XML 綱要資源的參照」頁面可以設定 XML 綱要參照的對應 (例如，XML 綱要匯入)，以解析未解析的 XML 綱要參照。

  1. 如果尚未進行這樣的處理，請按一下建立以建立新的階段作業，或按一下編輯以輸入現有的階段作業。請參閱使用變更中心。
  2. 在 XML 綱要的摘要頁面的 XML 綱要名稱欄位中，按一下適當的 XML 綱要名稱，以檢視未解析之參照的詳細資訊。隨即顯示檢視 XML 綱要詳細資訊頁面。
  3. 按一下「編輯參照」，以顯示「編輯 XML 綱要資源的參照」頁面。
  4. 在「資源類型」欄位中，選取「XML 綱要」。
  5. 按一下瀏覽。隨即顯示 XML 綱要瀏覽器。
  6. 在「XML 綱要瀏覽器」中選取 XML 綱要，然後從「定義」窗格中選取定義。
  7. 按一下「提交」。您選取的 XML 綱要會顯示在資源名稱欄位中。
  8. 按一下「儲存」以解析參照。
  9. 在 XML 綱要詳細資訊頁面中，按一下確定。XML 綱要的狀態會顯示為有效。
