新增群組

  1. 以具備 WebLogic Server 管理權限的使用者身分登入「Oracle Service Bus 管理主控台」。只有具備管理角色的使用者才可修改安全組態資料。請參閱 Oracle Fusion Middleware Developer's Guide for Oracle Service Bus 中的 Configuring Administrative Security，網址為 http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539。
  2. 選取「安全組態」>「群組」，以顯示「群組摘要」頁面。
  3. 按一下「新增」。

    4. 您可以從階段作業內部或外部新增群組。

  4. 在群組名稱欄位中，輸入唯一名稱。請注意，您不能輸入空格或特殊字元。這是必要欄位。
  5. 在「認證提供者」欄位中，選取認證提供者。
  6. 在「群組成員資格」欄位中，選取此群組所屬的群組。
    1. 從「可用的群組」欄位中選取群組。
    2. 按一下箭頭，將群組移至「目前群組」欄位中。
  7. 按一下「儲存」，以建立群組。

    8. 「Oracle Service Bus 管理主控台」會儲存群組，且所有階段作業都可立即使用該群組。如果您新增群組時是在階段作業中，然後捨棄該階段作業，「Oracle Service Bus 管理主控台」並不會刪除新群組。
