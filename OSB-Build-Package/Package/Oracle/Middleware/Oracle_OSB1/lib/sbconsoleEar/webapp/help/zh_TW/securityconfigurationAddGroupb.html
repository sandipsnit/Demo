編輯群組

使用「檢視群組詳細資訊」頁面可以檢視和變更特定群組的詳細資訊。

  1. 以擁有 WebLogic Server 管理權限的使用者身份登入「Oracle Service Bus 主控台」。只有管理角色中的使用者才可以修改安全設定資料。請參閱《Oracle Service Bus 安全指南》中的設定管理安全。
  2. 尋找群組。請參閱尋找群組。
  3. 按一下群組名稱。「檢視群組詳細資訊」頁面會顯示表 23-4 中所示的資訊。

    4. 表 23-4 群組詳細資訊 
    特性
    說明
    群組名稱
    此群組的名稱。
    身份驗證提供程式
    包含此群組定義的身份驗證提供程式。
    群組
    此群組所屬的群組。

  4. 若要編輯群組詳細資訊，請按一下「重新設定」以顯示「編輯群組詳細資訊」頁面。

    5. 您可以從階段作業內部或外部編輯群組詳細資訊。

  5. 對「群組成員資格」欄位進行適當的變更。有關欄位的說明，請參閱新增群組。

    6. 您不能變更「群組名稱」欄位。

  6. 按一下「儲存變更」，以更新群組。

    7. 「Oracle Service Bus 主控台」會更新群組詳細資訊，且所有階段作業可立即使用該更新。如果您更新群組時是在階段作業中，然後捨棄該階段作業，「Oracle Service Bus 主控台」並不會刪除該更新。
