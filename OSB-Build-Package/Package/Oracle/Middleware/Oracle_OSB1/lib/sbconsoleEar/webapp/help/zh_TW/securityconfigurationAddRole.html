新增角色

  1. 以具備 WebLogic Server 管理權限的使用者身分登入「Oracle Service Bus 管理主控台」。只有具備管理角色的使用者才可修改安全組態資料。請參閱 Oracle Fusion Middleware Developer's Guide for Oracle Service Bus 中的 Configuring Administrative Security，網址為 http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539。
  2. 選取「安全組態」>「角色」，以顯示「全域角色」頁面。
  3. 按一下「新增」。

    4. 您可以從階段作業內部或外部新增角色。

  4. 在角色名稱欄位中，輸入唯一名稱。這是必要欄位。

    5. 請確定安全角色名稱中沒有空格或 < > 字元。安全角色名稱需區分大小寫。Oracle 慣例是所有安全角色名稱皆為單數型。

  5. 若要建立角色，請按一下「確定」。

    6. 「Oracle Service Bus 管理主控台」會儲存角色，且所有階段作業都可立即使用該角色。如果您新增角色時是在階段作業中，然後捨棄該階段作業，「Oracle Service Bus 管理主控台」並不會刪除新角色。

    按一下「確定」建立角色時，下一步就是要定義套用該角色的條件。

  6. 繼續進行定義角色條件。
