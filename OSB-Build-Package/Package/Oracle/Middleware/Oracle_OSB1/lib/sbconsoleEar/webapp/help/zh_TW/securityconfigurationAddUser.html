新增使用者

  1. 以具備 WebLogic Server 管理權限的使用者身分登入「Oracle Service Bus 管理主控台」。只有具備管理角色的使用者才可修改安全組態資料。請參閱 Oracle Fusion Middleware Developer's Guide for Oracle Service Bus 中的 Configuring Administrative Security，網址為 http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539。
  2. 選取「安全組態」，以顯示「使用者摘要」頁面。
  3. 按一下「新增」，以顯示「建立新的使用者 - 一般組態」頁面。

    4. 您可以從階段作業內部或外部新增使用者。

  4. 在使用者名稱欄位中，輸入唯一名稱。這是必要欄位。
  5. 在密碼欄位中，輸入密碼。這是必要欄位。
    6. 注意： 認證提供者可以限制密碼長度下限。若為 WebLogic 認證提供者中所定義的使用者，則預設的密碼長度下限是 8 個字元。您可以使用「WebLogic Server 管理主控台」來自訂此設定(WebLogic 認證提供者是設定於預設安全範圍中，其名稱為 DefaultAuthenticator)。
  6. 在確認密碼欄位中，輸入您在密碼欄位中輸入的密碼。這是必要欄位。
  7. 在「認證提供者」欄位中，為此使用者選取認證提供者。

    8. 如果在安全範圍中設定多個認證提供者，則它們會顯示在清單中。請選取要用於儲存新使用者相關資訊的認證提供者資料庫。

  8. 在「群組成員資格」欄位中，選取此使用者的群組。
    1. 從「可用的群組」欄位中選取群組。
    2. 按一下箭頭，將群組移至「目前群組」欄位中。
  9. 按一下「儲存」，以建立使用者。

    10. 「Oracle Service Bus 管理主控台」會儲存使用者，且所有階段作業都可立即使用該使用者。如果您新增使用者時是在階段作業中，然後捨棄該階段作業，「Oracle Service Bus 管理主控台」並不會刪除新使用者。
