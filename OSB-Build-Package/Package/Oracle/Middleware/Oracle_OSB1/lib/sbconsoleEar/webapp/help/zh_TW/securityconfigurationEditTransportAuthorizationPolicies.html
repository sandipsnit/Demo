編輯傳輸層級存取原則

使用檢視原則詳細資訊頁面，可以編輯代理主機服務的傳輸層級存取控制原則。此頁面顯示表 24-10 中所示的資訊。

表 24-10 原則詳細資訊 
特性
描述
代理主機服務名稱
顯示您在「安全性」頁面選取「傳輸存取控制」時所針對之代理主機服務名稱的名稱。
提供者
顯示對該安全範圍設定的授權提供者。
原則條件
顯示決定代理主機服務要為哪些使用者處理要求的條件。

  1. 以具備 WebLogic Server 管理權限的使用者身分登入「Oracle Service Bus 管理主控台」。只有具備管理角色的使用者才可修改安全組態資料。請參閱 Oracle Fusion Middleware Developer's Guide for Oracle Service Bus 中的 Configuring Administrative Security，網址為 http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539。
  2. 尋找存取原則。請參閱尋找存取控制原則。
  3. 在安全性頁面的傳輸存取控制欄中，按一下代理主機服務的名稱。隨即顯示原則編輯器。

    4. 您可以從階段作業內部編輯存取控制原則。

  4. 在授權提供者欄位中，選取授權提供者。Oracle 建議您選擇 XACMLAuthorizer。
    5. 注意： Oracle Service Bus 已經不再支援 WebLogic 預設授權提供者。現在 Oracle 改而建議您使用 WebLogic XACML 授權提供者。
  5. 新增原則條件。請參閱新增原則條件。
  6. 在「原則條件」區段中完成輸入條件後，按一下「儲存」。
