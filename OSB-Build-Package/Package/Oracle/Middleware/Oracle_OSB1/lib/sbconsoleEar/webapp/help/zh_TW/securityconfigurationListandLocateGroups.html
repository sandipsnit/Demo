尋找群組

  1. 選取安全組態 > 群組。群組摘要頁面會顯示表 24-3 中所顯示的資訊。有關特性的更詳細描述，請參閱編輯群組。
  2. 若要限制清單中的項目數或尋找特定群組，則可以根據群組名稱來進行篩選。在「名稱」欄位中輸入部分或整個群組名稱，然後按一下「搜尋」。

    3. 您可以使用星號 (*) 萬用字元(不支援其他萬用字元)。

    按一下「檢視全部」移除搜尋篩選，並顯示所有群組。

    表 24-3 群組資訊 
    特性
    描述
    群組名稱
    群組的名稱。此名稱是連到檢視群組詳細資訊頁面的連結。請參閱編輯群組。
    群組成員資格
    此群組所屬的群組。此名稱是連到檢視群組詳細資訊頁面的連結。請參閱編輯群組。
    認證提供者
    包含此群組定義的認證提供者。
    刪除
    按一下刪除圖示，以刪除特定群組。請參閱刪除群組。
