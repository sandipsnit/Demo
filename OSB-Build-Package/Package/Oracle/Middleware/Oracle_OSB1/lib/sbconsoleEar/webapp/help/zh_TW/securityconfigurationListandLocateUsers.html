尋找使用者

  1. 選取安全組態 > 使用者。使用者摘要頁面會顯示表 24-1 中所顯示的資訊。有關特性的更詳細描述，請參閱編輯使用者。
  2. 若要限制清單中的項目數或尋找特定使用者，則可以根據使用者名稱來進行篩選。在「名稱」欄位中輸入部分或整個使用者名稱，然後按一下「搜尋」。

    3. 您可以使用星號 (*) 萬用字元(不支援其他萬用字元)。

    按一下「檢視全部」移除搜尋篩選，並顯示所有使用者。

    表 24-1 使用者資訊
    特性
    描述
    使用者名稱
    指派給使用者的名稱。此名稱是連到檢視使用者詳細資訊頁面的連結。請參閱編輯使用者。
    群組成員資格
    此使用者所屬之群組的名稱。此名稱是連到檢視群組詳細資訊頁面的連結。請參閱編輯群組。
    認證提供者
    此使用者的認證提供者。
    選項
    按一下刪除圖示，以刪除特定使用者。請參閱刪除使用者。
