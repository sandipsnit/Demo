編輯角色

使用「檢視角色詳細資訊」頁面可以檢視和變更特定角色的詳細資訊。

  1. 以具備 WebLogic Server 管理權限的使用者身分登入「Oracle Service Bus 管理主控台」。只有具備管理角色的使用者才可修改安全組態資料。請參閱 Oracle Fusion Middleware Developer's Guide for Oracle Service Bus 中的 Configuring Administrative Security，網址為 http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539。
  2. 尋找角色。請參閱尋找角色。
  3. 按一下角色名稱。「檢視角色詳細資訊」頁面會顯示表 24-8 中所示的資訊。

    4. 表 24-8 角色詳細資訊 
    特性
    描述
    名稱
    角色的名稱。
    角色條件
    決定此角色成員資格的條件。

    您可以從階段作業內部或外部變更角色詳細資訊。

  4. 執行下列其中一個步驟 (如表 24-9 中所述)。

    5. 表 24-9 角色選項
    若要...
    請完成此步驟...
    變更選定表示式的順序。
    按一下「上移」及「下移」。
    合併或取消合併角色條件，並且在表示式之間切換反白及/或陳述式。
    按一下「結合」及「取消結合」。
    制定否定條件，例如，「非群組操作員」會將操作員群組從角色中排除。
    按一下「否定」。
    刪除選取的表示式。
    按一下「移除」。

  5. 按一下「儲存」。

    6. 「Oracle Service Bus 管理主控台」會更新角色，且所有階段作業都可立即使用該更新。如果您更新角色時是在階段作業中，然後捨棄該階段作業，「Oracle Service Bus 管理主控台」並不會刪除更新。
