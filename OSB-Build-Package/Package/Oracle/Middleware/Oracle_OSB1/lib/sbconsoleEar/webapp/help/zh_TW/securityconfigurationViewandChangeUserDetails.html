編輯使用者

使用「檢視使用者詳細資訊」頁面可以檢視和變更特定使用者的詳細資訊。

  1. 以具備 WebLogic Server 管理權限的使用者身分登入「Oracle Service Bus 管理主控台」。只有具備管理角色的使用者才可修改安全組態資料。請參閱 Oracle Fusion Middleware Developer's Guide for Oracle Service Bus 中的 Configuring Administrative Security，網址為 http://www.oracle.com/pls/as111170/lookup?id=OSBDV1539。
  2. 尋找使用者。請參閱尋找使用者。
  3. 按一下使用者名稱。檢視使用者詳細資訊頁面會顯示表 24-2 中所顯示的資訊。

    4. 表 24-2 使用者詳細資訊
    特性
    描述
    使用者名稱
    此使用者的名稱。
    認證提供者
    包含此使用者定義的認證提供者。
    群組成員資格
    此使用者所屬之群組的名稱。

  4. 若要編輯使用者詳細資訊，請按一下「重新設定」以顯示「編輯使用者詳細資訊」頁面。

    5. 您可以從階段作業內部或外部編輯使用者詳細資訊。

  5. 對新密碼、確認密碼及群組成員資格欄位進行適當的變更。有關欄位的描述，請參閱新增使用者。

    6. 您不可以變更「使用者名稱」欄位。

  6. 按一下「儲存變更」，以更新使用者。

    7. 「Oracle Service Bus 管理主控台」會更新使用者詳細資訊，且所有階段作業都可立即使用該更新。如果您更新使用者時是在階段作業中，然後捨棄該階段作業，「Oracle Service Bus 管理主控台」並不會刪除更新。
