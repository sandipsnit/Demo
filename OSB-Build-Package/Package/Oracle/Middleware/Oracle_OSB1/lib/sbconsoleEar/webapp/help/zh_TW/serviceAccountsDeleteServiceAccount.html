刪除服務帳號

當您刪除服務帳號時，使用者名稱、密碼，或是該服務帳號所包含的本機使用者對遠端使用者對應資料也會一併刪除。

  1. 如果尚未進行這樣的處理，請按一下建立以建立新的階段作業，或按一下編輯以輸入現有的階段作業。請參閱使用變更中心。
  2. 如果設定任何業務服務或代理主機服務使用服務帳號，請將該服務帳號從業務服務或代理主機服務中移除。

    3. 請參閱編輯業務服務組態或編輯代理主機服務組態。

  3. 選取「資源瀏覽器」>「服務帳號」，以顯示「服務帳號的摘要」頁面。
  4. 在想要刪除之服務帳號的「選項」欄位中按一下「刪除」圖示。

    5. 將刪除目前階段作業中的服務帳號。如果業務服務或代理主機服務已設定成使用服務帳號，則刪除警告圖示指出您可以確認警告來刪除服務帳號。因為仍有服務參照已刪除服務帳號的未解析參照，所以可能會導致衝突。

  5. 若要結束階段作業並將組態建置至執行時期，請按一下「變更中心」下的「啟動」。
