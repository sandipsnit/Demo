尋找服務帳號

  1. 請執行下列任一項操作：
    • 選取專案瀏覽器以顯示專案檢視畫面頁面或專案/資料夾檢視畫面頁面。然後瀏覽專案和資料夾，以尋找服務帳號。
    • 選擇資源瀏覽器 > 服務帳號。服務帳號的摘要會顯示表 16-1 中所顯示的資訊。
  2. 若要搜尋服務帳號，請在名稱欄位中輸入部分或全部的帳號名稱。您也可以在路徑欄位中輸入部分或整個帳號專案名稱和資料夾。按一下搜尋。

    3. 按一下「檢視全部」移除搜尋篩選，並顯示所有服務帳號。

    表 16-1 服務帳號資訊
    特性
    描述
    服務帳號名稱
    服務帳號的唯一名稱。按一下名稱即可查看檢視服務帳號詳細資訊頁面。請參閱編輯服務帳號。
    路徑
    專案名稱和服務帳號所在資料夾的名稱。按一下名稱，以查看包含此資源的專案或資料夾。請參閱使用專案和資料夾限定資源名稱。
    選項
    包含刪除圖示。如果業務服務或代理主機服務已設定成使用服務帳號，則刪除警告圖示指出您可以確認警告來刪除服務帳號。因為仍有服務參照已刪除服務帳號的未解析參照，所以可能會導致衝突。請參閱刪除服務帳號。
