分割連接

「分割連接」可讓您平行傳送訊息叫用 (除了循序之外) 並聚總各個回應。若要進一步瞭解分割連接，請參閱 Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus 中的 "Improving Service Performance with Split-Join"，網址為：http://www.oracle.com/pls/as111170/lookup?id=OSBAG291。另請參閱 Oracle Fusion Middleware Developer's Guide for Oracle Service Bus 中的 "Working with Split-Join"，網址為：http://www.oracle.com/pls/as111170/lookup?id=OSBDV193 和 "Split-Join User Interface Reference"，網址為：http://www.oracle.com/pls/as111170/lookup?id=OSBDV573。

您可以透過流程傳輸類型業務服務來存取分割連接。如需詳細資訊，請參閱流程傳輸組態頁面。
