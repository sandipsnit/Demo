使用自動匯入狀態

您可以使用自動匯入狀態頁面，將服務的變更與登錄檔中的變更進行同步。當登錄檔中的服務有所變更時，Oracle Service Bus 即會在列出所有未同步之服務的自動匯入狀態頁面上提供變更通知。您就可以同步「Oracle Service Bus 管理主控台」中的服務與 UDDI 登錄檔中對應的服務。

在設定登錄檔時，您可以選取能夠自動同步匯入的服務與 UDDI 登錄檔的啟用自動匯入選項。任何在選擇此選項的狀態下匯入的服務，均會自動保持與 UDDI 登錄檔之間的同步。請參閱設定 UDDI 登錄檔。

在自動同步期間若有任何失敗的情形，均會報告在「自動匯入狀態」頁面上，以便您手動加以更新。

  1. 如果尚未進行這樣的處理，請按一下建立以建立新的階段作業，或按一下編輯以輸入現有的階段作業。請參閱使用變更中心。
  2. 選取「系統管理」>「自動匯入狀態」。
  3. 在「自動匯入狀態」頁面上，從「檢視」下拉清單中選取您要顯示的服務類型。
  4. 針對要與登錄檔中之對應服務同步的代理主機服務，選取其旁邊的核取方塊。
  5. 按一下「同步」。
  6. 在下一頁中，按一下「完成」。
