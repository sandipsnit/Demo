使用自動發佈

您可以使用自動發佈，將代理主機服務自動發佈到登錄檔。若要這樣做，您必須啟用建立/編輯代理主機服務 - 一般組態頁面中的發佈到登錄選項。請參閱一般組態頁面。請注意，只有設定有預設登錄檔時，才會顯示發佈到登錄選項。請參閱設定預設 UDDI 組態。

  1. 如果尚未進行這樣的處理，請按一下建立以建立新的階段作業，或按一下編輯以輸入現有的階段作業。請參閱使用變更中心。
  2. 選取「專案瀏覽器」。
  3. 展開專案資料夾，然後選取您要編輯的代理主機服務。
  4. 按一下位於「檢視代理主機服務」頁面底部的「編輯」。
  5. 選取「發佈到登錄」，然後按一下「儲存」，確認目前階段作業中的更新。
  6. 若要結束階段作業並將組態建置至執行時期，請按一下「變更中心」下的「啟動」。

    7. 服務會進行設定，以發佈到預設登錄檔。

附註：
附註：
  • 當您將服務成功發佈到登錄檔後，用以選取自動發佈到登錄檔之服務的選項便會無法再使用。
附註：
  • 若服務未成功發佈到登錄檔，您可以重新加以發佈。若要重新發佈服務，請在「自動發佈狀態」頁面上選取該服務，然後按一下「發佈」。
附註：
  • 如果啟用發佈到登錄選項，代理主機服務會在建立或編輯後以及階段作業啟動時立即發佈。您可以對使用「本機傳輸」以外的所有代理主機服務使用發佈到登錄選項。
