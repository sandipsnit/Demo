建立自訂檔案

您可以在建立自訂檔案頁面輕鬆地為一組所選取的資源或專案產生自訂檔案。接著，您可以此檔案為基礎，指定環境的實際值以進行所需的修改。

您可以使用自訂檔案來變更環境值，或變更資源內的參照。自訂檔案可以包含對您所選取資源中所有環境值的自訂，包括 EnvValueTypes 類別內定義的複雜環境值類型。此外，它還包含參照自訂類型，可用於變更具有相依性之資源內的資源參照。

可說明自訂類型的自訂 Schem (Customization.xsd) 位於 Oracle Service Bus 安裝中的下列位置：

OSB_ORACLE_HOME\modules\com.bea.common.configfwk_version.jar。

  1. 選取「系統管理」>「建立自訂檔案」。

    2. 建立自訂檔案頁面會顯示組態中的物件清單。每項物件的名稱與類型均會顯示。

  2. 選取要納入自訂檔案中的專案或資源。
    1. 展開專案資料夾。專案中各項資源的名稱與類型均會顯示。
    2. 選取與要納入自訂檔案中之專案或資源的相關核取方塊。
    3. 清除不納入自訂檔案中之專案或資源的相關核取方塊。
  3. 按一下「建立檔案」，建立自訂 XML 檔案。
  4. 在「檔案下載」對話方塊中，按一下「開啟」以開啟自訂檔案，或按一下「儲存」將 XML 檔案儲存到桌面上。
