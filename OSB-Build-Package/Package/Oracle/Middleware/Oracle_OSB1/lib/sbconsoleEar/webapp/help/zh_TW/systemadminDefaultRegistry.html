設定預設 UDDI 組態

您也可以將其中一個設定的登錄檔指定為網域的預設 UDDI 登錄檔。請參閱設定 UDDI 登錄檔。

若要使用自動發佈功能，則必須先設定預設登錄檔。請參閱使用自動發佈。

  1. 如果尚未進行這樣的處理，請按一下建立以建立新的階段作業，或按一下編輯以輸入現有的階段作業。請參閱使用變更中心。
  2. 選取「系統管理」>「UDDI 登錄檔」。
  3. 在「UDDI 預設組態」面板中，按一下「選取預設值」。
  4. 從「預設登錄檔名稱」清單中，選取想要設為預設登錄檔的登錄檔名稱。
  5. 若要設定預設業務實體，請從「業務實體」清單中進行選擇。
  6. 按一下「設為預設值」。
  7. 若要變更預設登錄檔，請按一下「清除選取」，再重複上述步驟。
