刪除 JNDI 提供者

  1. 如果尚未進行這樣的處理，請按一下建立以建立新的階段作業，或按一下編輯以輸入現有的階段作業。請參閱使用變更中心。
  2. 選取「系統管理」>「JNDI 提供者」。
  3. 在 JNDI 提供者摘要頁面中，選取一個「JNDI 提供者」。您也可以使用搜尋選項來搜尋特定的「JNDI 提供者」。請參閱檢視 JNDI 提供者。
  4. 在想要刪除之「JNDI 提供者」的選項欄位中按一下刪除圖示。當其他資源參照此資源時，會顯示「刪除警告」圖示。確認警告後即可刪除資源。因為仍有已刪除資源的未解析參照，所以可能會導致衝突。
  5. 若要結束階段作業並將組態建置至執行時期，請按一下「變更中心」下的「啟動」。
