中斷服務

若不想讓「Oracle Service Bus 管理主控台」中的服務與登錄檔中對應的服務同步，您可以透過下列其中一種方式從登錄檔中斷此服務，以避免進行同步：

  • 以編輯服務組態的方式中斷服務
  • 從自動匯入狀態頁面中斷服務
以編輯服務組態的方式中斷服務
  1. 如果尚未進行這樣的處理，請按一下建立以建立新的階段作業，或按一下編輯以輸入現有的階段作業。請參閱使用變更中心。
  2. 選取「專案瀏覽器」。
  3. 展開專案資料夾，並選取要從登錄檔中的對應服務中斷的業務服務。
  4. 按一下位於「檢視業務服務」頁面底部的「編輯」。
  5. 啟用從登錄中斷選項。此業務服務會從 UDDI 登錄檔中的對應服務中斷。
從自動匯入狀態頁面中斷服務

只有在「自動匯入狀態」頁面取得有關變更的通知時，才能使用「中斷」選項。

  1. 在「系統管理」>「自動匯入狀態」頁面上，選取業務服務旁邊的核取方塊，而這些業務服務是您想要從 UDDI 登錄檔中的對應服務中斷的業務服務。

    2. 只有在登錄檔中存在的原始服務發生變更時，這些服務才會顯示在此頁面。並非每一個服務都會出現在此頁面上。

  2. 按一下「中斷」。
