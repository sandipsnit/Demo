編輯代理伺服器

使用此頁面可以檢視代理伺服器的詳細資訊，以及依需要編輯組態。

  1. 如果尚未進行這樣的處理，請按一下建立以建立新的階段作業，或按一下編輯以輸入現有的階段作業。請參閱使用變更中心。
  2. 尋找代理伺服器。請參閱尋找代理伺服器。
  3. 按一下代理伺服器名稱。

    4. 檢視代理主機伺服器 - 組態頁面會顯示表 30-6 中所示的資訊。

    表 30-6 代理伺服器資訊
    特性
    描述
    上次修改者
    建立或修改此代理伺服器資源的使用者。
    上次修改時間
    使用者建立或修改此代理伺服器資源的日期和時間。按一下日期和時間連結，以檢視此資源的變更歷史記錄。請參閱檢視變更歷史記錄頁面。
    參照
    此代理伺服器所參照的物件數。如果有這類參照，請按一下數字連結以檢視參照清單。請參閱檢視資源參照。
    參照者
    參照此代理伺服器的物件數。如果有這類參照，請按一下數字連結以檢視物件清單。請參閱檢視資源參照。
    描述
    此代理伺服器的描述 (若有提供)。
    主機連接埠參數
    代理伺服器資源中每部代理伺服器的詳細資訊，包括下列資訊：
    • 伺服器主機：代理伺服器的主機名稱或 IP 位址。

      • Oracle Service Bus 代理伺服器的伺服器主機名稱必須與實際代理伺服器的伺服器主機名稱完全相同。

    • 純文字連接埠：代理伺服器的純文字連接埠號碼。
    • SSL 連接埠：代理伺服器 SSL 連接埠號碼。
    使用者名稱
    代理主機認證所使用的使用者名稱。

  4. 按一下「編輯」，以顯示「編輯代理伺服器 - [伺服器名稱]」頁面。

    5. 您無法編輯「名稱」欄位。

  5. 您可以在「描述」欄位中編輯代理伺服器的簡短描述。
  6. 在「主機連接埠參數」區段中，您可以輸入下列資訊以新增新的代理伺服器：
    1. 在伺服器主機欄位中，輸入「代理伺服器」的主機名稱或 IP 位址。這是必要欄位。

      2. Oracle Service Bus 代理伺服器的伺服器主機名稱必須與實際代理伺服器的伺服器主機名稱完全相同。

    2. 在「純文字連接埠」欄位中，輸入代理伺服器的純文字連接埠號碼。
    3. 在 SSL 連接埠欄位中，輸入「代理伺服器」的 SSL 連接埠號碼。您必須輸入純文字或 SSL 連接埠號。
    4. 按一下「新增」。

      5. 您可以為每個代理伺服器資源設定多部代理伺服器。如此可讓 Oracle Service Bus 執行負載平衡，以及為代理伺服器資源提供容錯功能。

      您也可以執行下列動作，以編輯或刪除現有的代理伺服器：

    • 在想要編輯之代理伺服器的選項欄位中按一下編輯圖示。依序編輯伺服器主機欄位、純文字連接埠欄位與 SSL 連接埠欄位，然後按一下更新。
    • 在想要刪除之代理伺服器的「選項」欄位中按一下「刪除」圖示。
  7. 您可以編輯代理主機認證所使用的「使用者名稱」與「密碼」欄位。
  8. 按一下「儲存」，確認目前階段作業中的更新。
  9. 若要結束階段作業並將組態建置至執行時期，請按一下「變更中心」下的「啟動」。
