編輯 SMTP 伺服器

使用此頁面可以檢視 SMTP 伺服器的詳細資訊，以及依需要編輯組態。

  1. 如果尚未進行這樣的處理，請按一下建立以建立新的階段作業，或按一下編輯以輸入現有的階段作業。請參閱使用變更中心。
  2. 尋找 SMTP 伺服器。請參閱尋找 SMTP 伺服器。
  3. 按一下 SMTP 伺服器名稱。

    4. 檢視 SMTP 伺服器 - 組態頁面會顯示表 30-4 中所示的資訊。

    表 30-4 SMTP 伺服器資訊 
    特性
    描述
    上次修改者
    建立或修改此 SMTP 伺服器資源的使用者。
    上次修改時間
    使用者建立或修改此 SMTP 伺服器資源的日期和時間。按一下日期和時間連結，以檢視此資源的變更歷史記錄。請參閱檢視變更歷史記錄頁面。
    參照
    此 SMTP 伺服器所參照的物件數。如果有這類參照，請按一下數字連結以檢視參照清單。請參閱檢視資源參照。
    參照者
    參照此 SMTP 伺服器的物件數。如果有這類參照，請按一下數字連結以檢視物件清單。請參閱檢視資源參照。
    描述
    此 SMTP 伺服器的描述 (若有提供)。
    伺服器 URL
    指向 SMTP 伺服器所在位置的 URL。
    連接埠號碼
    此 SMTP 伺服器的連接埠號碼。其預設值為連接埠 25。
    使用者名稱
    用以存取安全 SMTP 伺服器的使用者名稱。

  4. 按一下「編輯」，以顯示「編輯 SMTP 伺服器 - [伺服器名稱]」頁面。

    5. 您無法編輯「名稱」欄位。

  5. 您可以在「描述」欄位中編輯 SMTP 伺服器的簡短描述。
  6. 您可以在「伺服器 URL」欄位中編輯 SMTP 伺服器的 URL。
  7. 您可以在「連接埠號碼」欄位中編輯連接埠號碼。
  8. 您可以編輯「使用者名稱」與「密碼」欄位。
  9. 按一下「儲存」，確認目前階段作業中的更新。
  10. 若要結束階段作業並將組態建置至執行時期，請按一下「變更中心」下的「啟動」。

如果您重新命名 SMTP 資源，新的名稱會自動更新到任何參照端點 URI 中 SMTP 資源的電子郵件業務服務。
