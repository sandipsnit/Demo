執行自訂檔案

使用此頁面可以執行之前已儲存在系統上的自訂 XML 檔案。自訂檔案提供便捷的方法，可讓您在建置期間將環境從開發修改為暫存區，或在設計期間將環境從暫存區修改為開發。

您必須處於階段作業中才能執行自訂檔案。

  1. 如果尚未進行這樣的處理，請按一下建立以建立新的階段作業，或按一下編輯以輸入現有的階段作業。請參閱使用變更中心。
  2. 選取「系統管理」>「執行自訂檔案」。
  3. 選取要執行的自訂檔案。
    1. 按一下「檔案名稱」欄位中的「瀏覽」，以尋找自訂檔案的存放目錄。
    2. 選取 XML 檔案，然後按一下「開啟」。
  4. 按一下下一步。會顯示下列自訂詳細資訊 (如表 31-2 中所示)。

    5. 表 31-2 自訂詳細資訊
    特性
    描述
    作業
    • 尋找和取代 - 執行字串取代以修改環境值。
    • 對應參照 - 將資源中找到的現有參照對應至其他參照。
    • 設定環境值 - 修改資源的環境值。
    適用於
    要自訂的服務、專案、資料夾或資源的名稱。
    此名稱是連到服務或資源之「組態詳細資訊」頁面的連結，或「專案瀏覽器」中之專案或資料夾的連結。
    描述
    每項自訂的描述。

  5. 若只要自訂目前階段作業中變更的資源、服務、專案或資料夾，請選取「僅限在目前階段作業中變更的項目」核取方塊。

    6. 如果您將自訂只限於目前階段作業中修改的資源，則適用於欄會以目前階段作業中修改的資源進行更新。如果您將自訂限於專案或資料夾，則在該專案或資料夾內的目前階段作業中修改的資源會顯示在適用於欄中。

  6. 按一下「執行」，確認目前階段作業中的更新。

    7. 若要檢視自訂詳細資訊，請在階段作業內選取變更中心下的檢視變更，然後按一下自訂工作。請參閱檢視組態變更。

  7. 若要結束階段作業並將組態建置至執行時期，請按一下「變更中心」下的「啟動」。
