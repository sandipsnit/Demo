匯出資源

可以在匯出資源頁面匯出專案或匯出資源。若您選擇匯出整個專案，則會有一或多個專案可供選取以匯出。您仍然可以展開專案以檢視資料夾與資源，但是只能在專案層級進行選取。此外，您也無法選取「包含相依性」。

無論您是在階段作業之內或之外工作，您都可以匯出專案和資源。如果是在階段作業之內匯出，則資源屬於階段作業資源，且組態可能不完整或有衝突。如果是在階段作業之外匯出，則資源是已啟動的核心資源。

Oracle Service Bus 無法匯出您在「Oracle Service Bus 管理主控台」的「安全組態」模組中建立的使用者、群組或角色。它也無法匯出您在「WebLogic Server 管理主控台」中建立的證明資料對應或其他安全提供者資料。您應該使用「WebLogic Server 管理主控台」匯出這項資料。請參閱 Oracle Fusion Middleware Securing Oracle WebLogic Server，網址為 http://www.oracle.com/pls/as111170/lookup?id=SECMG319。

  1. 選取「系統管理」>「匯出資源」。

    2. 匯出資源頁面會顯示組態中的物件清單。每項物件的名稱、類型、上次修改時間與參照均會顯示。

    對於服務，最後修改的時間戳反映對服務定義所作的修改，以及對服務傳輸或管線組態所作的修改。不過，它不會反映出對定義給該服務的警示所進行的變更。

  2. 選取要「匯出專案」還是「匯出資源」。
    3. 注意： 匯入完整的專案 JAR 檔案後，匯出專案可能會導致資源刪除。有關詳細資訊，請參閱匯入資源。
  3. 選取要匯出的專案或資源。如果選取了「匯出專案」，將無法取消選取個別資源的核取方塊。
    1. 展開專案資料夾。專案中所含之每項資源的名稱、類型、上次修改時間與參照均會顯示。
    2. 選取您要匯出之專案或資源的相關核取方塊。
    3. 清除您不要匯出之專案或資源的相關核取方塊。
  4. 若您選取了匯出資源，請選取或清除包含相依性核取方塊。使用此選項可以匯出此資源所參照的其他任何資源。
  5. 按一下「匯出」，以建立 JAR 檔案並將其匯出。

完成作業後

如果適當，請繼續進行指定匯出安全設定。
