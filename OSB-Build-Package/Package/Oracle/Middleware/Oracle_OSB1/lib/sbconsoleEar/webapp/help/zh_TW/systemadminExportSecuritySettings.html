指定匯出安全設定

如果您的匯出包含服務帳號或服務金鑰提供者，則會顯示「匯出資源 - 安全設定」頁面。

對於每個服務帳號，Oracle Service Bus 會匯出遠端使用者對應的使用者名稱及密碼或本機使用者 (視服務帳號中所儲存的資料而定)。對於每個服務金鑰提供者，Oracle Service Bus 則會匯出 PKI 證明資料對應提供者的金鑰組連結別名，而不會匯出私密金鑰、憑證或金鑰儲存庫中的其他資料。金鑰儲存庫資料必須使用金鑰儲存庫廠商所提供的工具進行匯出。

若要讓 Oracle Service Bus 先對服務帳號、服務金鑰提供者、UDDI 登錄檔、JNDI 提供者或 SMTP 提供者資源中的使用者名稱與密碼資料進行加密，再將其匯出至 JAR 檔案，請執行下列動作：

  1. 選取「保護機密資料」核取方塊。
  2. 輸入並確認密碼。

    3. 若未指定此密碼，則在您或他人嘗試匯入此 JAR 時，Oracle Service Bus 即不會使用加密的使用者名稱與密碼資料來匯入資源。JAR 中所有其他未加密的資源則不需指定密碼即可匯入。

  3. 按一下「完成匯出」按鈕。
  4. 當您按一下「匯出」或「完成匯出」時，會顯示正在處理組態資料快顯視窗。請等待此視窗關閉後，再於「Oracle Service Bus 管理主控台」上執行其他作業。此視窗會在檔案下載對話方塊顯示時自動關閉。
  5. 在「檔案下載」對話方塊中，按一下「開啟」，開啟匯出的 JAR 檔案，或按一下「儲存」，將 JAR 檔案儲存至您的桌面。
