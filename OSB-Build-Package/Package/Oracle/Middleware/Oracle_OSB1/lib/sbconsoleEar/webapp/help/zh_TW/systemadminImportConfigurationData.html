匯入資源

您可以使用匯入資源頁面，匯入先前從另一個 Oracle Service Bus 網域匯出的完整專案 JAR 檔案或資源 JAR 檔案。您可以選擇只匯入已匯出資料的子集。若在匯入系統中已有該項資源存在，則會加以更新。否則將會建立該資源。若要匯入的 JAR 是完整專案 JAR，且在匯入系統中含有匯入之 JAR 檔案所沒有的相同專案資源，則只會排程資源以進行刪除。並不會刪除位在其他專案中的資源。

即使資源是在專案層級匯出，也可以在資源層級匯入資源。例如，即使已將系統資源匯出至完整專案 JAR 檔案中，在匯入時還是可以將它們取消選取。

在匯出組態時，並無法匯出使用者、群組、角色或憑證。因此，您必須在匯入先前匯出的組態時，再次建立這些物件。

注意： 如需更多匯入非 Oracle Service Bus 組態之一部份資源的相關資訊，請參閱從 Zip 檔案載入資源和從 URL 載入資源。
  1. 如果尚未進行這樣的處理，請按一下建立以建立新的階段作業，或按一下編輯以輸入現有的階段作業。請參閱使用變更中心。
  2. 選取「系統管理」>「匯入資源」。
  3. 選取匯出的專案或資源 JAR 檔案。
    1. 按一下「檔案名稱」欄位中的「瀏覽」，以尋找 JAR 檔案的存放目錄。
    2. 選取 JAR 檔案，然後按一下「開啟」。
  4. 按一下下一步。繼續進行選取要匯入的專案和資源。
