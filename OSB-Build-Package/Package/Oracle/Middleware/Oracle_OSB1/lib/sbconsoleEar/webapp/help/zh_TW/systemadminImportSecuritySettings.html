指定匯入安全設定

如果 JAR 檔案包含的資源在匯出時已使用使用者名稱和密碼資料進行加密，則會顯示「匯入資源 - 安全設定」頁面。

  1. 輸入用來加密資料的密碼 (請參閱匯出資源)。

    2. 若您不知道用以加密資源資料的密碼，請按上一步按鈕，然後清除資源旁的核取方塊。您可以匯入其他不需要密碼的未加密資源。

  2. 按一下「完成匯入」。

    3. 就會顯示正在處理組態資料快顯視窗。請等待此視窗關閉後，再於「Oracle Service Bus 管理主控台」上執行其他作業。此視窗會在系統完成匯入物件後自動關閉。此時會顯示匯入資源頁面，其中含有您已成功建置資料的訊息。

完成作業後

繼續進行檢查匯入摘要。
