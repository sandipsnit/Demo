檢視設定的 UDDI 登錄檔

使用此頁面可以指定預設 UDDI 登錄檔 (請參閱設定預設 UDDI 組態)，以及檢視 UDDI 登錄檔。此外，還可以搜尋先前設定的登錄檔。

  1. 在「UDDI 登錄檔」面板的「名稱」欄位中，輸入要尋找的登錄檔名稱，或輸入萬用字元；使用 * 表示多個字元，而 ? 表示單一字元。搜尋會傳回所有滿足名稱條件的記錄。
  2. 按一下「搜尋」。
  3. 按一下「檢視全部」移除搜尋篩選，並檢視所有設定成使用 Oracle Service Bus 的登錄檔。

    4. 表 29-1 顯示 UDDI 登錄檔組態設定。

    表 29-1 UDDI 登錄檔組態設定 
    特性
    描述
    登錄檔名稱
    登錄檔的名稱。在登錄檔首次建立時所指派的名稱。登錄檔的名稱儲存後即無法再編輯。
    詢問 URL
    用以尋找及匯入服務之詢問 API 端點的 URL。
    發佈 URL
    用以發佈服務之發佈 API 端點的 URL。
    安全 URL
    用以取得認證記號而讓您得以發佈至登錄檔之安全性 API 端點的 URL。
    訂閱 URL
    用以訂閱登錄檔變更、建立登錄檔訂閱以及監聽匯入服務有何變更之訂閱 API 端點的 URL。
    使用者名稱
    使用者認證所需的使用者名稱。
    密碼 (確認密碼)
    使用者認證所需的密碼。
    將 tModels 載入至登錄檔
    將 tModels 載入至選定登錄檔。每個登錄檔只需選取此選項一次。
    啟用自動匯入
    自動同步服務與 UDDI 登錄檔。
