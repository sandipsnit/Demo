新增 SMTP 伺服器

您可以在 SMTP 伺服器摘要頁面新增和設定「SMTP 伺服器」資源，並使其成為 Oracle Service Bus 中的可用系統資源。您必須在作用中階段作業中，才能設定或重新設定 SMTP 伺服器資源。

  1. 如果尚未進行這樣的處理，請按一下建立以建立新的階段作業，或按一下編輯以輸入現有的階段作業。請參閱使用變更中心。
  2. 選取「系統管理」>「SMTP 伺服器」。
  3. 按一下「新增」。
  4. 在名稱欄位中，輸入 SMTP 伺服器資源的名稱。這是必要欄位。

    請遵循資源命名限制的命名指南。
  5. 在「描述」欄位中，輸入 SMTP 伺服器資源的簡短描述。
  6. 在伺服器 URL 欄位中，輸入指向「SMTP 伺服器」的 URL。這是必要欄位。
  7. 在連接埠號碼欄位中，輸入「SMTP 伺服器」的連接埠號碼 (預設連接埠為 25)。這是必要欄位。
  8. 如果必須提供使用者名稱和密碼才能存取目標 SMTP 伺服器，請在「使用者名稱」欄位中輸入使用者名稱，並在「密碼」和「確認密碼」欄位中輸入相關的密碼。

    9. 這些是選擇性欄位，只有在 SMTP 伺服器受到保護時才會成為必要欄位。

  9. 按一下「儲存」，以在目前階段作業中建立和儲存 SMTP 伺服器資源。
  10. 若要結束階段作業並將組態建置至執行時期，請按一下「變更中心」下的「啟動」。
