設定預設 SMTP 伺服器

您可以將其中一部設定的 SMTP 伺服器指定為網域的預設伺服器。請參閱新增 SMTP 伺服器。

  1. 如果尚未進行這樣的處理，請按一下建立以建立新的階段作業，或按一下編輯以輸入現有的階段作業。請參閱使用變更中心。
  2. 選取「系統管理」>「SMTP 伺服器」。
  3. 在「預設 SMTP 伺服器」面板中，按一下「選取預設值」。
  4. 從「預設 SMTP 伺服器」下拉清單中，選取 SMTP 伺服器。
  5. 按一下「設為預設值」。

    6. 隨即顯示「SMTP 伺服器摘要」頁面，其中會將選定 SMTP 伺服器顯示為網域的預設 SMTP 伺服器。

  6. 若要結束階段作業並將組態建置至執行時期，請按一下「變更中心」下的「啟動」。
