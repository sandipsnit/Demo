檢視 SMTP 伺服器

使用此頁面可以指定 SMTP 伺服器 (請參閱設定預設 SMTP 伺服器)，以及檢視 SMTP 伺服器清單。

SMTP 伺服器資源可以在設定警示目標資源和電子郵件傳輸型業務服務時使用。請參閱新增警示目標及建立/編輯業務服務 - 電子郵件傳輸組態頁面。

表 30-3 顯示 SMTP 伺服器資訊。

表 30-3 SMTP 伺服器資訊 
特性
描述
名稱
指派給此 SMTP 伺服器資源的唯一名稱。此名稱是連到檢視 SMTP 伺服器 - 組態頁面的連結。請參閱編輯 SMTP 伺服器。
伺服器 URL
指向 SMTP 伺服器的 URL。請參閱新增 SMTP 伺服器。
選項
  • 按一下「重新命名」圖示，以重新命名此 SMTP 伺服器資源。
  • 按一下刪除圖示，以刪除特定的「SMTP 伺服器」。請參閱刪除 SMTP 伺服器。
