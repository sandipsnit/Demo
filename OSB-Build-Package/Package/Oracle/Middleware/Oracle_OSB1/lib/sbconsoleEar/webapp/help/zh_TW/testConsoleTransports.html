瞭解執行時期如何使用測試主控台上的傳輸設定

設定代理主機服務測試資料和設定業務服務測試資料描述在「測試主控台」中測試代理主機服務或業務服務時，如何設定外送要求的傳輸標頭值、傳輸描述資料值及傳輸相關安全資料值。不過，某些您可以在「測試主控台」中設定的規格並不會在執行時期生效。也就是說，在執行測試時，Oracle Service Bus 會在執行時期覆寫或忽略特定標頭或描述資料的值。 表 32-10 說明有所限制的標頭和描述資料。
