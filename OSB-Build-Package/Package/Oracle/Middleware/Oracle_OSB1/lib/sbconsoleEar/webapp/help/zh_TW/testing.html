測試主控台

「Oracle Service Bus 測試主控台」是瀏覽器型的測試環境，可用來驗證及測試您的系統設計。您可以在「測試主控台」中設定測試的物件 (代理主機服務、業務服務或資源)、執行測試並檢視結果。在某些執行處理中，您可以透過程式碼進行追蹤，並檢查訊息在特定追蹤點的狀態。如需瞭解有關「測試主控台」的詳細資訊，請參閱 Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus 中的 Using the Test Console，網址為 http://www.oracle.com/pls/as111170/lookup?id=OSBAG357。

只有 IntegrationAdmin 和 IntegrationDeployer 角色的使用者可以使用「測試主控台」。如需角色的詳細資訊，請參閱 Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus 中的 Roles in Oracle Service Bus，網址為 http://www.oracle.com/pls/as111170/lookup?id=OSBAG463。

注意：如果您收到指出未執行「測試主控台」服務的錯誤，請嘗試將「管理伺服器」監聽位址設為特定的有效位址 (例如 localhost)。請到「Oracle WebLogic Server 主控台」的環境 > 伺服器 > admin_server_name > 組態 > 一般中設定「監聽位址」。
