執行 XQuery 測試

您可以使用下列編輯器來編輯和測試訊息流程中的動作：「XQuery 表示式編輯器」、「XQuery 條件編輯器」和「XPath 表示式編輯器」。「XQuery 表示式編輯器」和「XQuery 條件編輯器」的測試會採用相同的形式。不過，「XPath 表示式編輯器」的情形則不同，因其僅使用一項輸入。

注意： 您必須停用瀏覽器的彈出視窗阻擋程式，XQuery 測試才能夠順利進行。如果您在 IE 瀏覽器中裝有工具列，則可能必須從「選項」功能表下停用彈出視窗阻擋程式，並停用所有設定成封鎖彈出視窗的工具列。

  • 使用 XQuery 表示式和 XQuery 條件編輯器
  • 使用 XPath 表示式編輯器
