檢視代理主機服務測試結果

表 32-2 會描述代理主機服務測試結果。只有在同時選擇「直接呼叫」和「包含追蹤」選項時，才會啟用追蹤。

表 32-2 代理主機服務的測試結果
區段
描述
代理主機服務名稱
要測試之代理主機服務的名稱。
測試主控台動作
「返回」可顯示上一個瀏覽器頁面。
關閉可關閉「測試主控台」。
要求文件
「測試主控台」傳送至代理主機服務的要求訊息。
若「測試主控台」未修改要求訊息，此區段最初將為收合狀態。若 SOAP 服務使用格式標籤進行設定，或套用了「WS 安全性」，則此區段最初將為展開狀態。
如果套用 WS 安全性時，此區段會包含兩則 SOAP 訊息：第一則訊息為「純文字」訊息；第二則為「安全」SOAP 訊息。
回應文件
訊息回應。
若為具有 WS 安全性回應的 SOAP 服務，此區段會包含兩則 SOAP 訊息。第一則 SOAP 訊息為從屬端所接收的安全訊息。第二則 SOAP 訊息為對應的純文字訊息。
回應描述資料
與訊息回應一起傳回的描述資料。
追蹤代理主機服務
追蹤會顯示訊息通過系統時的狀態。如果未選取直接呼叫選項，就不會執行追蹤。有關追蹤的詳細資訊，請參閱追蹤代理主機服務。
