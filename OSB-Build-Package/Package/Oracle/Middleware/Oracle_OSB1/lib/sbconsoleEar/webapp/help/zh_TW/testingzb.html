區段
描述
業務服務名稱
業務服務的名稱。
測試主控台動作
按一下「返回」可顯示上一個瀏覽器頁面。
按一下關閉可關閉「測試主控台」視窗。
要求文件
「測試主控台」傳送至業務服務的要求訊息。
若「測試主控台」未修改要求訊息，此區段最初將為收合狀態。若 SOAP 服務使用格式標籤進行設定，或套用了「WS 安全性」，則此區段最初將為展開狀態。
如果已套用 WS 安全性，則此區段會包含兩則 SOAP 訊息。第一則訊息為純文字訊息。第二則為安全 SOAP 訊息。
回應文件
訊息回應。
若為具有 WS 安全性回應的 SOAP 服務，此區段會包含兩則 SOAP 訊息。第一則 SOAP 訊息為從屬端所接收的安全訊息。第二則 SOAP 訊息為對應的純文字訊息。
回應描述資料
與訊息回應一起傳回的描述資料。

注意： 安全 SOAP 訊息顯示時會加上排版修飾，例如額外的空白字元。由於空白字元會影響文件的語意，因此這則 SOAP 訊息不一定可做為字面資料。例如，數位簽章即可能因為會區分空白字元，而失去其有效性。
