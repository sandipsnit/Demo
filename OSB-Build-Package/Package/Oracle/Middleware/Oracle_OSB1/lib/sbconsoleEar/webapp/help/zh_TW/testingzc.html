測試 MFL 轉換

「訊息格式語言」(MFL) 文件是用來說明二進位資料配置的特殊 XML 文件。MFL 資源支援兩種轉換。每種轉換都只接受一個輸入，並提供單一輸出：XML 至二進位或二進位至 XML。

表 32-5 描述 MFL 組態選項。

MFL 轉換組態
XML 輸入
XML 至二進位轉換時需要有：
MFL 文件的 XML 綱要可推斷而得。文字欄位中會自動輸入範例 XML 文件。
XML 輸入可以是檔案型或文字型。參照檔案的輸入優先於文字輸入。請瀏覽並選擇想要用於測試的檔案。
二進位輸入
二進位至 XML 轉換時需要有：
二進位輸入可以是檔案型或文字型。參照檔案的輸入優先於文字輸入。請瀏覽並選擇想要用於測試的檔案。

您可以測試設計時期或執行時期。

  1. 如果想要測試執行時期，請按一下啟動。而測試設計時期，則請不要啟動階段作業。
  2. 選擇「資源瀏覽器」>「MFL」，以顯示「MFL 檔案的摘要」頁面。
  3. 在動作底下，按一下與想要測試之資源相關聯的啟動測試主控台圖示。「測試主控台」就會開啟測試資源頁面。
  4. 選擇想要測試的轉換。
  5. 設定 MFL 資源的測試資料。有關詳細資訊，請參閱表 32-5。
  6. 按一下「執行」。測試資源頁面會顯示結果。
  7. 若要重新測試，請按一下返回。您可以關閉「測試主控台」、進行修改，然後重新測試資源。
