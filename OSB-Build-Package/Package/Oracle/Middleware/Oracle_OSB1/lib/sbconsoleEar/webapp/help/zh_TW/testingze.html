測試 XQuery 轉換

XQuery 對應可以說明 XML 至 XML、XML 至非 XML 和非 XML 至 XML 對應。XQuery 轉換可接受多項輸入並傳回一項輸出。每項輸入各對應至 XQuery 資源中宣告的一項 XQuery 外部變數。XQuery 輸入變數的值可以是基本值 (字串、整數、日期等)、XML 文件或這些類型的順序。輸出值可以是基本值 (字串、整數、日期等)、XML 文件，或這些類型的順序。

注意： 「測試主控台」不支援在輸入中使用序列。

<param_name> ([] as XML)
param_name 是 XQuery 資源中的 XQuery 變數名稱。
在「測試主控台」中，如果類型為簡單類型，會顯示單行編輯方塊。如果資料為 XML，則會顯示多行編輯方塊。
當變數不具有類型時，會使用組合輸入 (<param_name> ([] as XML))。您必須宣告變數類型。選擇此核取方塊可識別類型為 XML 的參數。
XML 輸入可以是檔案型或文字型。參照檔案的輸入優先於文字輸入。請瀏覽並選擇想要用於測試的檔案。
「測試主控台」中的輸入會根據類型來轉譯，以便讓您更容易瞭解必須輸入的資料類型。當不具有類型時，預設類型為 XML。

您可以測試設計時期或執行時期。

  1. 如果想要測試執行時期，請按一下啟動。而測試設計時期，則請不要啟動階段作業。
  2. 選擇「專案瀏覽器」>「XQuery」，以顯示「XQuery 的摘要」頁面。
  3. 在動作底下，按一下與想要測試之資源相關聯的啟動測試主控台圖示。「測試主控台」就會開啟測試資源頁面。
  4. 在頁面的變數區段中設定資源的測試資料。有關詳細資訊，請參閱表 32-7。
  5. 按一下「執行」。測試資源頁面會顯示結果。
  6. 若要重新測試，請按一下返回。您可以關閉「測試主控台」、進行修改，然後重新測試資源。
