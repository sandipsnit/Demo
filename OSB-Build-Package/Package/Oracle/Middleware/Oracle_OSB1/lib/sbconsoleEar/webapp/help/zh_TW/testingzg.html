使用 XPath 表示式編輯器

您可以使用 XPath 表示式選擇 XML 訊息相關資訊環境變數的子集。您可以在「XPath 表示式編輯器」中使用「測試主控台」測試 XPath 表示式的定義。XPath 表示式會使用單一 XML 文件做為輸入，並產生一系列的 XML 文件、基本類型或兩者做為輸出。

表 32-9 描述 XPath 表示式組態選項。

變數
此區段含有對應至 XML 文件 (XPath 表示式據以測試者) 的單一輸入欄位。
 
XML 輸入可以是檔案型或文字型。參照檔案的輸入優先於文字輸入。請瀏覽並選擇想要用於測試的檔案。

  1. 在編輯管線之訊息流程中的動作時存取「測試主控台」。若要存取「XPath 表示式編輯器」，請參閱建立和編輯內置 XQuery 和 XPath 表示式。
  2. 在頁面的變數區段中設定 XPath 表示式的測試資料。有關詳細資訊，請參閱表 32-9。
  3. 按一下「執行」。測試頁面會顯示結果。
  4. 完成測試之後，可以使用返回按鈕執行新的測試。若要在變更 XPath 表示式後執行新的測試，您必須關閉並重新開啟「測試主控台」才能使變更生效。
