UDDI

「通用描述、探查和整合 (UDDI)」登錄檔是用於企業，以共用 Web 服務。UDDI 提供用來分類您業務的架構、其服務，以及想要公開之服務的技術詳細資訊。

將服務發佈至登錄檔需要瞭解服務類型以及在登錄檔中代表該服務的資料結構。登錄檔項目具有特定相關的特性，而這些特性類型是在登錄檔建立時所定義。您可以將服務發佈至登錄檔，讓它成為可供其他組織搜尋和使用的項目。由 Oracle Service Bus 開發的代理主機服務均可發佈至 UDDI 登錄檔。Oracle Service Bus 可以與任何 UDDI 3.0 版相容登錄檔互動。

另請參閱 Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus 中的 "UDDI"，網址為：http://www.oracle.com/pls/as111170/lookup?id=OSBAG392。
