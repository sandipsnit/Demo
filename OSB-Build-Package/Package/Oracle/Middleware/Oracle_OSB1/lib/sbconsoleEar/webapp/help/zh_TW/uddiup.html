編輯 UDDI 登錄檔

  1. 如果尚未進行這樣的處理，請按一下建立以建立新的階段作業，或按一下編輯以輸入現有的階段作業。請參閱使用變更中心。
  2. 選取「系統管理」>「UDDI 登錄檔」。
  3. 在「UDDI 登錄檔」面板的可用登錄檔清單中，按一下想要編輯之登錄檔的名稱。

    4. 您也可以使用搜尋選項來搜尋特定的登錄檔。請參閱檢視設定的 UDDI 登錄檔。

    UDDI 登錄檔組態頁面會顯示可定義登錄檔的組態特性以及下列登錄檔特有資訊 (如表 29-2 中所示)。

    表 29-2 UDDI 登錄檔資訊
    特性
    描述
    上次修改者
    前次變更此 UDDI 登錄檔資源組態的使用者。
    上次修改時間
    前次進行組態變更的日期和時間。按一下日期和時間連結，以檢視此資源的變更歷史記錄。請參閱檢視變更歷程頁面。
    參照
    此 UDDI 登錄檔所參照的物件數。如果有這類參照，請按一下數字連結以檢視參照清單。請參閱檢視資源參照。
    參照者
    參照此 UDDI 登錄檔的物件數。如果有這類參照，請按一下數字連結以檢視物件清單。請參閱檢視資源參照。
    描述
    登錄檔的描述。

  4. 在頁面底端按一下「編輯」。
  5. 在「UDDI 組態 - 編輯登錄檔」頁面上，編輯登錄檔組態參數。

    6. 表 29-1 中描述這些特性。您無法編輯登錄檔的名稱。

  6. 按一下「儲存」，確認目前階段作業中的更新。
  7. 若要結束階段作業並將組態建置至執行時期，請按一下「變更中心」下的「啟動」。
