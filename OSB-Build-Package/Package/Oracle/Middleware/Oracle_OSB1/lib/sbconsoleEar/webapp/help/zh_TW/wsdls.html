WSDL

WSDL (Web Service Definition Language，Web 服務定義語言) 是 Web 服務的正式描述；在 Oracle Service Bus 中，它可說明代理主機服務或業務服務。WSDL 可用於描述 Web 服務能執行的動作、其所在位置以及其叫用方式。

您可以使用現有 WSDL 資源做為 SOAP 和 XML 服務的基礎。WSDL 文件可供代理主機服務和業務服務用於任何傳輸。請參閱建立/編輯代理主機服務 - 一般組態頁面及建立和設定業務服務。

如需詳細資訊，請參閱 Oracle Fusion Middleware Administrator's Guide for Oracle Service Bus中的 "How WSDL is Used in Oracle Service Bus"，網址為：http://www.oracle.com/pls/as111170/lookup?id=OSBAG114。
