新增 WSDL

  1. 如果尚未進行這樣的處理，請按一下建立以建立新的階段作業，或按一下編輯以輸入現有的階段作業。請參閱使用變更中心。
  2. 選取專案瀏覽器，然後選取要在其中新增 WSDL 的專案或資料夾。隨即顯示專案/資料夾檢視畫面頁面。
  3. 從「建立資源」下拉清單的「介面」中選取「WSDL」，以顯示「建立新的 WSDL 資源」頁面。
  4. 在資源名稱欄位中，輸入此 WSDL 的唯一名稱。這是必要欄位。

    請遵循資源命名限制的命名指南。
  5. 在「資源描述」欄位中，輸入 WSDL 的描述。
  6. 在「WSDL」欄位中，請執行下列其中一項操作：
    • 輸入新 WSDL 的文字。
    • 按一下「瀏覽」以尋找並匯入現有的 WSDL。
    • 從現有的 WSDL 複製及貼上文字至此欄位。

      • 這是必要欄位。

  7. 按一下「儲存」。新的 WSDL 會儲存在目前的階段作業中。

    8. 當您按一下儲存時，如果新的 WSDL 有任何未解析的參照，系統會加以顯示。請參閱檢視未解析的 WSDL 參照。

  8. 若要結束階段作業並將組態建置至執行時期，請按一下「變更中心」下的「啟動」。
