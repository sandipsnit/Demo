尋找 WSDL

  1. 請執行下列任一項操作：
    • 選取專案瀏覽器以顯示專案檢視畫面頁面或專案/資料夾檢視畫面頁面。然後瀏覽專案和資料夾，以尋找 WSDL。
    • 選擇資源瀏覽器 > WSDL。WSDL 的摘要頁面會顯示表 6-1 中所顯示的資訊。有關特性的更詳細描述，請參閱編輯 WSDL。
  2. 若要尋找特定 WSDL：
    • 按一下開啟圖示，以顯示其他搜尋篩選。如果指定了多個搜尋條件，則只會傳回符合全部條件的結果。
    • 若要限制清單中的項目數，則可以依據名稱、路徑和命名空間來進行篩選。在「名稱」、「路徑」和「命名空間」欄位中，輸入搜尋目標的名稱、路徑和命名空間，然後按一下「搜尋」。

      • 路徑是專案名稱和 WSDL 所在資料夾的名稱。

    • 按一下「檢視全部」移除搜尋篩選，並顯示所有 WSDL。

      • 表 6-1 WSDL 詳細資訊
      特性
      描述
      WSDL 名稱
      指派給 WSDL 的唯一名稱。此名稱是連到檢視 WSDL 詳細資訊頁面的連結。請參閱編輯 WSDL。
      路徑
      路徑是專案名稱和 WSDL 所在資料夾的名稱。它是連到包含此資源之專案或資料夾的連結。請參閱使用專案和資料夾限定資源名稱。
      WSDL 命名空間
      WSDL 的位址。它的顯示格式為：
      http://example.com/example/example
      選項
      按一下刪除圖示以刪除特定 WSDL。請參閱刪除 WSDL。
