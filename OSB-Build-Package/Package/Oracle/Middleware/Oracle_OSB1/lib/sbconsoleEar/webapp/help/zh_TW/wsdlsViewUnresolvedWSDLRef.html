WSDL 名稱

WSDL 的位址。它的顯示格式為：

若要尋找特定參照：

  • 重新排序清單。按一下加上水平底線的欄名稱。遞增和遞減箭頭表示排序順序。按一下欄名稱以變更排序順序。
  • 捲動頁面。使用表格上下方的頁面控制項。您可以選取頁碼以前往該頁，也可以使用箭頭按鈕前往下一頁、上一頁、第一頁或最後一頁。
