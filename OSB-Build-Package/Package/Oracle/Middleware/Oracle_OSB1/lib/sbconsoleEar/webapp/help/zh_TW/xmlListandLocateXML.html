尋找 XML 文件

「XML 文件摘要」頁面列出您已新增到 Oracle Service Bus 網域中的 XML 文件。

  1. 請執行下列任一項操作：
    • 選取專案瀏覽器以顯示專案檢視畫面頁面或專案/資料夾檢視畫面頁面。然後瀏覽專案和資料夾，以尋找「XML 文件」。
    • 選取資源瀏覽器 > XML 文件。XML 文件的摘要頁面會顯示表 13-1 中所顯示的資訊。有關特性的更詳細描述，請參閱編輯 XML 文件」。
  2. 若要尋找特定 XML 文件：
    • 若要限制在清單中項目的數量，您可以在按下「搜尋」之前篩選名稱及路徑。

      • 路徑是專案名稱和資源所在資料夾的名稱。

    • 按一下「檢視全部」移除搜尋篩選，並顯示所有 XML 文件。

      • 表 13-1 XML 文件詳細資訊
      特性
      描述
      XML 文件
      指定給 XML 文件的唯一名稱。此名稱是連到檢視 XML 文件頁面的連結。請參閱編輯 XML 文件。
      路徑
      此路徑是專案名稱及 XML 文件所在資料夾的名稱。它是連到包含此資源之專案或資料夾的連結。請參閱使用專案和資料夾限定資源名稱。
      選項
      按一下刪除圖示，以刪除特定 XML 文件。請參閱刪除 XML 文件。
