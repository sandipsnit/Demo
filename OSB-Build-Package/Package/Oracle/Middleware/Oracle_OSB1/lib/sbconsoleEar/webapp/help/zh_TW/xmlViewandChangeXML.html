編輯 XML 文件

使用「編輯 XML 文件」頁面檢視及變更 JCA 連結的詳細資訊。

  1. 如果尚未進行這樣的處理，請按一下建立以建立新的階段作業，或按一下編輯以輸入現有的階段作業。請參閱使用變更中心。
  2. 尋找 XML 文件 (如尋找 XML 文件中所述)。
  3. 按一下 XML 文件名稱。檢視 XML 文件頁面顯示資訊，如表 13-2 中所示。

    4. 表 13-2 XML 文件詳細資訊
    特性
    描述
    上次修改者
    建立這個 XML 文件或將其匯入組態中的使用者。
    上次修改時間
    使用者建立此 XML 文件或將其匯入組態中的日期和時間。按一下日期和時間連結，以檢視此資源的變更歷史記錄。請參閱檢視變更歷史記錄頁面。
    參照
    此 XML 文件參照的物件數。如果有這類參照，請按一下數字連結以檢視物件清單。請參閱檢視資源參照。
    參照者
    參照此 XML 文件的物件數。如果有這類參照，請按一下數字連結以檢視物件清單。例如，若 XML 檔案為 JCA 連結所參照的 TopLink 檔案，當您按一下連結時，就會將 JCA 連結列為參照。請參閱檢視資源參照。
    描述
    此 XML 文件的描述 (若有的話)。

    預設會顯示唯讀的 XML 文件檢視。

  4. 若要編輯 XML 文件，請按一下編輯。請參閱新增 XML 文件取得欄位的描述。
  5. 進行適當的編輯，並按一下「儲存」以確認在目前工作階段中的更新。
  6. 若要結束階段作業並將組態建置至執行時期，請按一下「變更中心」下的「啟動」。
