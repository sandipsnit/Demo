編輯 XQuery 轉換

  1. 如果尚未進行這樣的處理，請按一下建立以建立新的階段作業，或按一下編輯以輸入現有的階段作業。請參閱使用變更中心。
  2. 尋找 XQuery 轉換 (如尋找 XQuery 轉換中所述)。
  3. 按一下 XQuery 轉換名稱。「檢視 XQuery 轉換詳細資訊」頁面會顯示表 10-2 中所示的資訊。

    4. 表 10-2 XQuery 轉換資源詳細資訊 
    特性
    描述
    上次修改者
    建立這個 XQuery 轉換，或者將其匯入組態的使用者。
    上次修改時間
    使用者建立此 XQuery 轉換或將其匯入組態中的日期和時間。按一下日期和時間連結，以檢視此資源的變更歷史記錄。請參閱檢視變更歷史記錄頁面。
    參照
    此 XQuery 轉換參照的物件數。如果有這類參照，請按一下數字連結以檢視物件清單。如需詳細資訊，請參閱檢視資源參照。
    參照者
    參照此 XQuery 轉換的物件數。如果有這類參照，請按一下數字連結以檢視物件清單。例如，如果您已將此 XQuery 轉換指派給特定代理主機服務之「訊息流程」中的變數，則代理主機服務會在您按一下連結時列為參照。如需詳細資訊，請參閱檢視資源參照。
    描述
    此 XQuery 轉換的描述 (若有的話)。
    XQuery
    XQuery 轉換的完整文字。
    XQuery 變數
    針對此 XQuery 轉換選取的 XQuery 變數。

  4. 若要變更欄位，請按一下編輯。有關欄位的描述，請參閱新增 XQuery 轉換。

    5. 您不可以變更「資源名稱」欄位。

  5. 按一下「儲存」，確認目前階段作業中的更新。
  6. 若要結束階段作業並將組態建置至執行時期，請按一下「變更中心」下的「啟動」。
