新增 XSL 轉換

  1. 如果尚未進行這樣的處理，請按一下建立以建立新的階段作業，或按一下編輯以輸入現有的階段作業。請參閱使用變更中心。
  2. 選取「專案瀏覽器」，以顯示專案檢視畫面頁面或專案/資料夾檢視畫面頁面。
  3. 瀏覽至想要新增 XSL 轉換的專案或資料夾。
  4. 從「建立資源」清單中選取「XSLT」，以顯示「建立新的 XSLT」頁面。
  5. 在資源名稱欄位中，輸入此 XSL 轉換的唯一名稱。

    請遵循資源命名限制的命名指南。
  6. 在「資源描述」欄位中，輸入 XSL 轉換的描述。
  7. 在「XSLT」欄位中，執行下列操作之一：
    • 輸入新 XSL 轉換的文字。
    • 按一下「瀏覽」以尋找並匯入現有的 XSL 轉換。
    • 從現有的 XSL 轉換複製文字，並貼入此欄位。
  8. 按一下「儲存」，確認目前階段作業中的更新。
  9. 若要結束階段作業並將組態建置至執行時期，請按一下「變更中心」下的「啟動」。
