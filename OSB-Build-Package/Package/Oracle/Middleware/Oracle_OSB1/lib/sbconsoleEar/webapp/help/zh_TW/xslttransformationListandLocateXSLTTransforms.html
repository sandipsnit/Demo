尋找 XSL 轉換

  1. 請執行下列任一項操作：
    • 選取專案瀏覽器以顯示專案檢視畫面頁面或專案/資料夾檢視畫面頁面。然後瀏覽專案和資料夾，以尋找 XSL 轉換。
    • 選擇資源瀏覽器 > XSLT。XSLT 的摘要頁面會顯示表 14-1 中所示的資訊。
  2. 若要限制清單中的項目數，則可以依據名稱和 (或) 路徑來進行篩選。在「名稱」和「路徑」欄位的「搜尋」下，輸入目標的名稱和 (或) 路徑，然後按一下「搜尋」按鈕。

    3. 路徑是專案名稱和 XSLT 所在資料夾的名稱。

    萬用字元 * 和 ? 皆可使用。搜尋會區分大小寫。

    按一下檢視全部，以顯示網域中的所有 XSLT。這樣會清除前次搜尋的搜尋參數。

    表 14-1 XSL 轉換詳細資訊 
    特性
    描述
    XSLT 名稱
    指派給 XSL 轉換的名稱。按一下名稱，以顯示「檢視 XSL 轉換詳細資訊」頁面。
    請參閱編輯 XSL 轉換。
    路徑
    路徑是專案名稱和 XSL 所在資料夾的名稱，例如，MyProject/MyFolder/MyResource。
    按一下 XSLT 的路徑，以顯示包含它的專案/資料夾檢視畫面頁面。
    動作
    按一下啟動測試主控台圖示來叫用「測試主控台」，您可以用此來驗證和測試服務及轉換的設計。對於 XSL 轉換，無論在階段作業內或外，您都可以使用「測試主控台」。若要進一步瞭解，請參閱測試轉換。
    選項
    如果尚未進行這樣的處理，請按一下建立以建立新的階段作業，或按一下編輯以輸入現有的階段作業。請參閱使用變更中心。
    按一下刪除圖示刪除 XSLT。當其他資源參照此資源時，會顯示刪除警告圖示。確認警告後即可刪除資源。因為仍有已刪除資源的未解析參照，所以可能會導致衝突。有關詳細資訊，請參閱刪除 XSL 轉換。
