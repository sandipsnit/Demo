解析未解析的 XSL 轉換參照

  1. 如果尚未進行這樣的處理，請按一下建立以建立新的階段作業，或按一下編輯以輸入現有的階段作業。請參閱使用變更中心。
  2. 選取「資源瀏覽器」>「XSLT」，以顯示「XSLT 的摘要」頁面。
  3. 在 XSLT 名稱欄中，按一下適當的 XSLT 名稱，以檢視未解析參照的詳細資訊。隨即顯示檢視 XSL 轉換詳細資訊頁面。
  4. 按一下「編輯參照」，以顯示「編輯 XSLT 的參照」頁面。
  5. 在「資源類型」欄位中選取「XSLT」。
  6. 按一下瀏覽。隨即顯示 XSLT 瀏覽器。
  7. 在「XSLT 瀏覽器」中選取 XSLT，再從「相依性」窗格中選取相依性。
  8. 按一下「提交」。您選取的 XSLT 會顯示在資源名稱欄位中。
  9. 按一下「儲存」。隨即顯示檢視 XSLT 詳細資訊頁面。
  10. 按一下確定。XSL 轉換的狀態會顯示為有效。
  11. 若要結束階段作業並將組態建置至執行時期，請按一下「變更中心」下的「啟動」。
