編輯 XSL 轉換

  1. 如果尚未進行這樣的處理，請按一下建立以建立新的階段作業，或按一下編輯以輸入現有的階段作業。請參閱使用變更中心。
  2. 尋找 XSL 轉換 (如尋找 XSL 轉換中所述)。
  3. 按一下 XSL 轉換名稱。檢視 XSL 轉換詳細資訊頁面預設會顯示 XSLT 的結構檢視畫面，其中包含表 14-2 中所示的資訊。

    4. 表 14-2 XSL 轉換資源詳細資訊 
    特性
    描述
    上次修改者
    建立這個 XSL 轉換，或者將其匯入組態的使用者。
    上次修改時間
    使用者建立此 XSL 轉換或將其匯入組態的日期和時間。按一下日期和時間連結，以檢視此資源的變更歷史記錄。請參閱檢視變更歷史記錄頁面。
    參照
    此 XSL 轉換參照的物件數。如果有這類參照，請按一下數字連結以檢視物件清單。如需詳細資訊，請參閱檢視資源參照。
    參照者
    參照此 XSL 轉換的物件數。如果有這類參照，請按一下數字連結以檢視物件清單。例如，如果將此 XSL 轉換指派給特定代理主機服務之訊息流程中的變數，則在您按下連結後，會將代理主機服務列為參照。如需詳細資訊，請參閱檢視資源參照。
    描述
    此 XSL 轉換的描述。
    目標命名空間
    用來限定 XSL 轉換中包含之任何定義的命名空間。
    XSLT 相依性
    與 XSLT 關聯的屬性和群組：
    • XSLT 位置
    狀態
    XSL 轉換的狀態：
    • 有效

      • 將會指定目前 XSL 轉換所包含的所有 XSL 轉換位置，並且這些位置都有效。此外，也會指定巢狀 XSL 轉換 (包含的 XSL 轉換再包含的 XSL 轉換) 的所有位置，並且這些位置都有效。

    • 無效

      • 並未指定目前 XSL 轉換所包含之 XSL 轉換的一或多個位置，或者位置無效。此外，一或多個巢狀 XSL 轉換 (包括的 XSL 轉換隨後包括的 XSL 轉換) 可能未加以指定或可能無效。若要解析無效的 XSL 轉換，請按一下「編輯參照」。若要進一步瞭解，請參閱解析未解析的 XSL 轉換參照。

  4. 按一下文字檢視畫面，以顯示 XSL 轉換詳細資訊的文字檢視畫面。此文字檢視畫面包含表 14-3 中所示的資訊。

    5. 表 14-3 XSL 轉換文字詳細資訊
    特性
    描述
    目標命名空間
    用來限定 XSL 轉換中包含之任何相依性的命名空間。
    XSLT
    XSL 轉換的全文。

  5. 若要變更欄位，請按一下編輯。有關欄位的描述，請參閱新增 XSL 轉換。

    6. 您不可以變更「資源名稱」欄位。

  6. 按一下「儲存」，確認目前階段作業中的更新。
