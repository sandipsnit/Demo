Produkte zur Zelle hinzufügen

Wählen Sie die Produkte, die Sie in diese Zelle aufnehmen möchten. Die Option "Oracle CIE Core" ist standardmäßig aktiviert und kann nicht gelöscht werden. Diese Option fügt der Domain den erforderlichen benutzerdefinierten CIE-Onlineservice hinzu und definiert diesen als Ziel für den OracleAdminServer.

Wenn Sie eine vorhandene Zelle bearbeiten oder einen Rechner mit einer Zelle verbinden, sind Produkte, die bereits in die Zelle aufgenommen wurden, bereits ausgewählt und können nicht deaktiviert werden.


Hinweis:

Normalerweise wählen Sie beim Erstellen einer neuen Zelle auf dem Deployment Manager-Knoten nur die Option Oracle CIE Core. Wählen Sie bei einer Konfiguration mit mehreren Knoten zusätzliche Produkte, wenn Sie den Konfigurationsassistenten auf den Rechnern ausführen, die Sie mit der Zelle im Deployment Manager-Knoten verbinden. Obwohl Sie die Fusion Middleware-Komponenten als Ziele für den Administrationsserver oder die Anwendungsserver auf dem Deployment Manager-Knoten definieren können, sollten Sie sie als Ziele für Anwendungsserver auf den Remote-Knoten definieren.

Aktivieren Sie die Kontrollkästchen für die anderen Produkte, die Sie in die Zelle aufnehmen möchten. Wenn Sie ein Produkt auswählen, das von Ressourcen von einem anderen Produkt oder anderen Produkten in der Liste abhängt, werden diese Produkte automatisch ebenfalls ausgewählt. Wenn Sie umgekehrt das Kontrollkästchen für ein Produkt deaktivieren, von dem ein anderen Produkt abhängt, wird das Kontrollkästchen für das abhängige Produkt ebenfalls deaktiviert.

Die Liste der Produkte (Konfigurationsvorlagen) variiert abhängig von der Fusion Middleware-Produktinstallation, die Sie konfigurieren. Das hier gezeigte Beispiel gilt für eine ....-Installation.

Wenn Sie Produkte zu einer vorhandenen Zelle hinzufügen, sind alle Produkte, die der Zelle zuvor hinzugefügt wurden, bereits ausgewählt und können nicht deaktiviert werden.

Wenn Sie alle in die Zelle aufzunehmenden Produkte ausgewählt haben, klicken Sie auf Weiter, um fortzufahren.


Hinweise:

Wenn eines der ausgewählten Produkte Ressourcen benötigt, die von einem anderen, nicht ausgewählten Produkt bereitgestellt werden, und Sie auf Weiter klicken, wird ein Warndialogfeld angezeigt. Kehren Sie zu dem Bildschirm Produkte zur Zelle hinzufügen zurück, um das Kontrollkästchen des abhängigen Produkts zu aktivieren, bevor Sie fortfahren.
