Zusätzliche Cluster-Member konfigurieren

Auf diesem Bildschirm können Sie dem Cluster zusätzliche Anwendungsserver hinzufügen. Wenn Sie einem Cluster einen neuen Anwendungsserver hinzufügen, werden dessen Attribute vom primären Member des Clusters geclont.

  • Name: Der Name des Anwendungsservers. Der Standardname für einen neuen Server ist new_ClusterMember_n, wobei n bei 1 beginnt und für jeden neuen hinzugefügten Server erhöht wird. Um den Servernamen zu ändern, klicken Sie auf das Feld Name, und geben Sie den neuen Namen ein.

  • Knotenname: Eine Dropdown-Liste mit allen verfügbaren Knoten. Klicken Sie auf die Liste, um den Knoten zu wählen, auf dem dieser Anwendungsserver gehalten wird. Wenn Sie noch keine anderen Rechner mit der aktuellen Zelle verbunden haben, ist nur ein Knoten in dieser Liste verfügbar.

  • Cluster-Name: Eine Dropdown-Liste mit allen verfügbaren Clustern. Wählen Sie das Cluster, dem dieser Server zugewiesen werden soll.

  • Entwicklungsmodus: Aktivieren Sie dieses Kontrollkästchen, um den Anwendungsserver im Entwicklungsmodus auszuführen. Lassen Sie das Kontrollkästchen leer, um den Anwendungsserver im Production-Modus auszuführen. Informationen zu den Unterschieden zwischen dem Entwicklungsmodus und dem Production-Modus in einer WebSphere-Umgebung finden Sie in der WebSphere-Dokumentation.

So fügen Sie ein zusätzliches Cluster-Member hinzu:

  1. Klicken Sie auf Hinzufügen. Der Tabelle wird eine neue Zeile hinzugefügt.

  2. Geben Sie den Namen des Servers ein, und wählen Sie die entsprechende Einstellung für Knoten, Cluster und Entwicklungsmodus.

  3. Wiederholen Sie die Schritte 1 und 2 für jedes Cluster-Member, das Sie hinzufügen möchten.

Um ein Cluster-Member zu löschen, wählen Sie das betreffende Cluster-Member, und klicken Sie auf Löschen. Sie können jeweils immer nur ein Cluster-Member löschen.

Um die Anzeige von dem standardmäßigen tabellarischen Format in ein Registerkartenformat zu ändern (mit einer Registerkarte pro Cluster), klicken Sie auf Anzeige umschalten. Um zum tabellarischen Format zurückzukehren, klicken Sie erneut auf Anzeige umschalten.

Klicken Sie abschließend auf Weiter, um fortzufahren.


Hinweis:

Alle auf diesem Bildschirm vorgenommenen Änderungen werden erst in der Zellenkonfiguration gespeichert, wenn Sie im letzten Bildschirm des Konfigurationsassistenten auf Erstellen klicken. Wenn Sie den Assistenten verlassen oder auf Zurück klicken, um zu einem vorherigen Bildschirm zurückzukehren, müssen Sie die Änderungen erneut vornehmen.
