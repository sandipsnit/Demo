Cluster konfigurieren

Ein Cluster ist eine Gruppe von Anwendungsservern, die gemeinsam für Skalierbarkeit und hohe Verfügbarkeit von Anwendungen sorgen. Durch die Erstellung von Clustern können Sie Server in Gruppen zusammenfassen, sodass sie als einzelner Host für Anwendungen und Ressourcen fungieren.

Auf diesem Bildschirm können Sie ein Cluster hinzufügen und den Anwendungsserver auswählen, der als primäres Cluster-Member für das Cluster dient. Alle weiteren Cluster-Member werden von diesem primären Cluster-Member geclont. Nach der Erstellung eines Clusters können Sie auf dem nächsten Bildschirm weitere Anwendungsserver hinzufügen.

Sie können auf diesem Bildschirm auch Cluster löschen oder die Einstellungen für ein vorhandenes Cluster ändern.

So erstellen Sie einen Cluster:

  1. Klicken Sie auf Hinzufügen. Eine neue Zeile wird in der Tabelle angezeigt. Der Standardname für ein neues Cluster lautet new_Cluster_n, wobei n bei 1 beginnt und für jedes neu hinzugefügte Cluster erhöht wird. Um den Standardnamen zu ändern, geben Sie den gewünschten Namen in die Spalte Name ein. Der Name des Clusters muss für alle Komponenten in der Zelle eindeutig sein.

  2. Wählen Sie einen vorhandenen Anwendungsserver als primäres Member des Clusters, indem Sie den Server aus der Dropdown-Liste Erstes Cluster-Member auswählen. In dieser Liste werden nur Anwendungsserver angezeigt, die noch keinem Cluster zugewiesen wurden.

  3. Wiederholen Sie die Schritte 1 und 2 für jedes weitere Cluster, das Sie erstellen möchten.

  4. Klicken Sie nach Abschluss des Vorgangs auf Weiter. Auf dem Bildschirm Zusätzliche Cluster-Member konfigurieren können Sie dem Cluster weitere Server hinzufügen. Dieser Bildschirm wird angezeigt, wenn Sie auf Weiter klicken.


    Hinweis:

    Alle auf diesem Bildschirm vorgenommenen Änderungen werden erst in der Zellenkonfiguration gespeichert, wenn Sie im letzten Bildschirm des Konfigurationsassistenten auf Erstellen klicken. Wenn Sie den Assistenten verlassen oder auf Zurück klicken, um zu einem vorherigen Bildschirm zurückzukehren, müssen Sie die Änderungen erneut vornehmen.

Um ein Cluster zu löschen, wählen Sie das Cluster, und klicken Sie auf Löschen. Wenn Sie ein Cluster löschen, wird dadurch nicht der Anwendungsserver gelöscht, der als primäres Cluster-Member dient. Er wird lediglich aus dem Cluster entfernt. Sie können jeweils immer nur ein Cluster löschen.

Um die Einstellungen für ein Cluster zu ändern, wählen Sie das Cluster, und nehmen Sie die gewünschten Änderungen vor.

Um die Anzeige von dem standardmäßigen tabellarischen Format in ein Registerkartenformat zu ändern (mit einer Registerkarte pro Cluster), klicken Sie auf Anzeige umschalten. Um zum tabellarischen Format zurückzukehren, klicken Sie erneut auf Anzeige umschalten.

Wenn Sie die Konfiguration der Cluster abgeschlossen haben, klicken Sie auf Weiter, um fortzufahren.

Weitere Informationen zu Clustern in einer WebSphere-Umgebung finden Sie in der WebSphere-Dokumentation.
