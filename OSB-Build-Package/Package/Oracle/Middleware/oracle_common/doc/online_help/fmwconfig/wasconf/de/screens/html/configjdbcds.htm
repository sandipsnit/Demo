JDBC-Datenquellen konfigurieren

Eine JDBC-Datenquelle enthält einen Pool mit Datenbankverbindungen, die erstellt werden, wenn die Datenquelleninstanz erstellt wird, wenn sie bereitgestellt oder als Ziel definiert wird oder wenn der Server hochgefahren wird. Anwendungen suchen eine Datenquelle im JNDI-Baum und fordern dann eine Verbindung an. Benötigen die Anwendungen die Verbindungen nicht länger, geben Sie die Verbindungen an den Connection Pool in der Datenquelle zurück.

Auf diesem Bildschirm können Sie die in der Zelle definierten JDBC-Datenquellen konfigurieren.

In der unteren Bildschirmhälfte sind die JDBC-Datenquellen aufgelistet, die mit den Produkten verknüpft sind, für die Sie die Zelle erstellen.


Hinweis:

Wenn Sie eine oder mehrere der Datenquellen als Oracle RAC-Multidatenquellen konfigurieren möchten, aktivieren Sie das Kontrollkästchen neben dem Namen der jeweiligen Datenquelle, und aktivieren Sie das Kontrollkästchen Konfigurieren Sie gewählte Datenquellen als RAC-Multidatenquellen im nächsten Fensterbereich. Klicken Sie dann auf die Schaltfläche Weiter.

Wählen Sie die Datenquelle(n), für die Sie Einstellungen vornehmen möchten, indem Sie das Kontrollkästchen neben dem Namen der Datenquelle aktivieren.


Hinweise:

Wenn Sie mehrere Datenquellen auswählen, wird möglicherweise der Text Variiert unter Datenquellen in bestimmten Feldern angezeigt. Dies weist darauf hin, dass die aktuellen Werte dieser Felder nicht in allen ausgewählten Datenquellen identisch sind. Wenn Sie die Werte in solchen Feldern ändern, werden die neuen Werte einheitlich auf die gewählten Datenquellen angewendet.

Die Standardwerte von Datenquellenparametern wie Hersteller, Treiber, Hostname und Portnummer sind abhängig von den Werten, die in den Anwendungsvorlagen festgelegt sind.

  • Hersteller: Wählen Sie den Datenbankhersteller.

  • Treiber: Wählen Sie den JDBC-Treiber für den Verbindungsaufbau zur Datenbank. Die Liste enthält gängige JDBC-Treiber für den gewählten Datenbankhersteller.

  • Benutzername: Geben Sie den Benutzernamen für die Datenbankanmeldung ein.

  • Kennwort: Geben Sie das Kennwort für den angegebenen Benutzernamen ein.

  • DBMS/Service: Geben Sie eine DBMS-SID oder einen Servicenamen ein. Der eingegebene Wert hängt vom ausgewählten Treiber ab.

    • Wenn der Name des ausgewählten Oracle Treibers die Wörter "für Instanzverbindungen" enthält, müssen Sie eine SID eingeben.

    • Wenn der Name des ausgewählten Oracle Treibers die Wörter "für Service-Verbindungen" enthält, müssen Sie einen Service-Namen eingeben.

    • Weitere Informationen zur Konfiguration von DataDirect-Treibern finden Sie in der DataDirect-Dokumentation.

  • Hostname: Geben Sie den Namen des Servers ein, der die Datenbank hostet.

  • Port: Geben Sie die Portnummer für die Verbindung mit dem Server ein.

Die von Ihnen angegebenen Werte werden für die gewählte Datenquelle in den entsprechenden Spalten in der Datenquellenliste angezeigt.

Wenn Sie die Konfiguration der JDBC-Datenquellen abgeschlossen haben, klicken Sie auf Weiter, um fortzufahren.
