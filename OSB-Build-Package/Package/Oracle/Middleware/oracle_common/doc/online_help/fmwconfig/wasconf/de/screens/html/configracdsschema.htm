RAC-Datenquell-Komponentenschema konfigurieren

Auf diesem Bildschirm können Sie die Komponentenschemas konfigurieren, die als Oracle RAC-Schemas in der WebSphere-Zelle enthalten sind.

Die Komponentenschemas, die Sie im Bildschirm JDBC-Komponentenschema konfigurieren des Assistenten für die Konfiguration als Oracle RAC-Schemas ausgewählt haben, werden in der unteren Bildschirmhälfte aufgelistet.

Wählen Sie aus der Schemaliste in der unteren Bildschirmhälfte die Schemas, die Sie als Oracle RAC-Schemas konfigurieren möchten. Aktivieren Sie dazu das Kontrollkästchen neben dem jeweiligen Schemanamen.


Hinweis:

Wenn Sie mehrere Schemas auswählen, wird möglicherweise der Text "Variiert unter Komponentenschemas" in bestimmten Feldern angezeigt. Dies weist darauf hin, dass die aktuellen Werte dieser Felder nicht in allen ausgewählten Schemas identisch sind. Wenn Sie die Werte in solchen Feldern ändern, werden die neuen Werte einheitlich auf die gewählten Schemas angewendet.

  • Treiber: Wählen Sie den JDBC-Treiber für die Anmeldung bei der Datenbank.

  • Service-Name: Geben Sie einen Namen für den Oracle RAC-Datenbankservice ein.

  • Benutzername: Geben Sie den Benutzernamen für die Datenbankanmeldung ein.

  • Kennwort: Geben Sie das Kennwort für den angegebenen Benutzernamen ein.

  • Hostname: Geben Sie den Namen des Servers ein, auf dem die Oracle RAC-Datenbankinstanzen gehostet werden.

  • Instanzname: Geben Sie für jede Oracle-Datenbankinstanz den Namen ein.

  • Port: Geben Sie die Portnummern für die Verbindung mit dem Server ein, der die Datenbank hostet.


Hinweis:

Sie müssen für mindestens eine Datenbankinstanz den Hostnamen, Instanznamen und die Portnummer angeben.

Sie können eine Datenbankinstanz hinzufügen, indem Sie auf Hinzufügen klicken. Geben Sie dann den Hostnamen, Instanznamen und Port an.

Die von Ihnen angegebenen Werte werden für die gewählten Schemas in den entsprechenden Spalten in der Schemaliste angezeigt.
