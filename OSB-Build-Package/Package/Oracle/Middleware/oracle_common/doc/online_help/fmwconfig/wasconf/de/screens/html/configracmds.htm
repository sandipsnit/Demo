RAC-Multidatenquellen konfigurieren

Auf diesem Bildschirm können Sie die Datenquellen konfigurieren, die Sie für die Konfiguration als Oracle RAC-Datenquellen gewählt haben.

In der unteren Bildschirmhälfte sind die Datenquellen aufgelistet, die Sie auf dem Bildschirm JDBC-Datenquellen konfigurieren als Oracle RAC-Datenquellen konfiguriert haben.

Wählen Sie die Datenquelle(n), für die Sie Einstellungen vornehmen möchten, indem Sie das Kontrollkästchen neben dem Namen der Datenquelle aktivieren.


Hinweis:

Wenn Sie mehrere Datenquellen auswählen, wird möglicherweise der Text Variiert unter Datenquellen in bestimmten Feldern angezeigt. Dies weist darauf hin, dass die aktuellen Werte dieser Felder nicht in allen ausgewählten Datenquellen identisch sind. Wenn Sie die Werte in solchen Feldern ändern, werden die neuen Werte einheitlich auf die gewählten Datenquellen angewendet.

  • Treiber: Wählen Sie den JDBC-Treiber für die Anmeldung bei der Datenbank.

  • Service-Name: Geben Sie einen Namen für den Oracle RAC-Datenbankservice ein.

  • Benutzername: Geben Sie den Benutzernamen für die Datenbankanmeldung ein.

  • Kennwort: Geben Sie das Kennwort für den angegebenen Benutzernamen ein.

  • Hostname: Geben Sie den Namen des Servers ein, auf dem die Oracle RAC-Datenbankinstanzen gehostet werden.

  • Instanzname: Geben Sie für jede Oracle-Datenbankinstanz den Namen ein.

  • Port: Geben Sie die Portnummern für die Verbindung mit dem Server ein, der die Datenbank hostet.

Sie können eine neue Datenbankinstanz hinzufügen, indem Sie auf Hinzufügen klicken und dann den Hostnamen, Instanznamen und die Portnummer angeben.

Wenn Sie die Konfiguration der RAC-Multidatenquellen abgeschlossen haben, klicken Sie auf Weiter.
