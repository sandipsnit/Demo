Konfigurationszusammenfassung

Prüfen Sie die genauen Konfigurationseinstellungen der Zelle, bevor Sie fortfahren.

Wählen Sie im Bereich Zusammenfassung ein Element, um Details zu diesem Element im Bereich Details auf der rechten Seite anzuzeigen.

Sie können die im Bereich Zusammenfassung angezeigten Elemente eingrenzen, indem Sie eine Filteroption aus der Dropdown-Liste Zusammenfassung auswählen. Beispiel: Um die JDBC-Datenquellen anzuzeigen, die Ihren Clustern und Servern zugewiesen sind, wählen Sie JDBC in der Dropdown-Liste Zusammenfassung.

Wenn Sie die Konfiguration ändern müssen, klicken Sie auf Zurück, um zum entsprechenden Bildschirm zurückzukehren.

Wenn Sie mit der Prüfung der Konfiguration fertig sind, klicken Sie auf:

  • Erstellen (wenn Sie eine neue Zelle erstellen)

  • Erweitern (wenn Sie eine vorhandene Zelle ändern)

  • (wenn Sie einen Rechner mit einer vorhandenen Zelle verbinden)
