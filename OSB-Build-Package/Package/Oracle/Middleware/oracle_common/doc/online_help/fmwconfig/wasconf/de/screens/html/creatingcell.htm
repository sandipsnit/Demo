Zellenfortschritt

Dieser Bildschirm zeigt den Status des Prozesses zum Erstellen einer Zelle, Laden einer Zelle oder Verbinden eines Rechners an. Dieses Beispiel zeigt den Bildschirm Zelle wird erstellt. Der Bildschirmtitel variiert abhängig von der aktuellen Aufgabe:

  • Wenn Sie eine neue Zelle erstellen, lautet der Bildschirmtitel Zelle wird erstellt.

  • Wenn Sie eine vorhandene Zelle aktualisieren, lautet der Bildschirmtitel Zelleninformationen werden geladen.

  • Wenn Sie einen Remote-Rechner mit einer Zelle verbinden, lautet der Bildschirmtitel Rechner verbinden.

Wenn der Fortschrittsbalken 100 % erreicht und die Meldung "Zelleninformationen erfolgreich gelesen. Klicken Sie auf "Weiter"" im unteren Fenster angezeigt wird, klicken Sie auf Weiter, um fortzufahren.
