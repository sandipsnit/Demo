Konfigurationsoption auswählen

Wählen Sie eine der folgenden Optionen. Die angezeigten Optionen hängen von der aktuellen Konfiguration ab.

  • Zelle erstellen und konfigurieren: Wählen Sie diese Option, um eine neue WebSphere-Zelle zu erstellen. Daraufhin wird der Bildschirm Zellen-, Profil- und Knoteninformationen angeben angezeigt.

  • Rechner verbinden und Zelle konfigurieren: Wählen Sie diese Option, um den aktuellen WebSphere-Knoten zu einer vorhandenen Fusion Middleware-Zelle hinzuzufügen. Daraufhin wird der Bildschirm Geben Sie Profil- und Knotennamensinformationen an angezeigt. In diesem Fall müssen Sie den Konfigurationsassistenten auf dem Knoten ausführen, den Sie mit einer Zelle verbinden möchten, die zuvor auf dem Deployment Manager-Knoten erstellt wurde.

  • Wählen und konfigurieren Sie die vorhandene Zelle: Diese Option ist nur verfügbar, wenn Sie bereits eine Zelle erstellt haben. Wählen Sie diese Option, um die Konfiguration einer vorhandenen Zelle zu ändern. Klicken Sie im Listenfeld auf die zu konfigurierende Zelle (die gewählte Zelle wird markiert). Daraufhin wird der Bildschirm Zelleninformationen werden geladen angezeigt. Informationen zu diesem Bildschirm finden Sie unter Zellenfortschritt.
