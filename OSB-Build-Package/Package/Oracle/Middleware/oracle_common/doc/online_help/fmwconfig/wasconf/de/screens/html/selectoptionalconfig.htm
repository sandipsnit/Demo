Optionale Konfiguration wählen

Wählen Sie alle Kategorien (falls zutreffend), für die Sie eine erweiterte Konfiguration vornehmen möchten. Für jede ausgewählte Kategorie wird der entsprechende Konfigurationsbildschirm angezeigt, in dem Sie eine erweiterte Konfiguration ausführen können. Wenn Sie hier keine Elemente auswählen, wird als Nächstes der Bildschirm Konfigurationszusammenfassung angezeigt.


Hinweise:

Die in diesem Bildschirm aufgelisteten Kategorien hängen von den Ressourcen ab, die in den für die Zelle ausgewählten Vorlagen definiert wurden.

  • Anwendungsserver und Cluster: Wählen Sie diese Option, wenn Sie die folgenden Aufgaben ausführen möchten:

    • Anwendungsserver hinzufügen, löschen oder clonen

    • Den Namen eines Anwendungsservers ändern

    • Den Entwicklungsmodus für einen Anwendungsserver ein- oder ausschalten

    • Cluster zur Zelle hinzufügen und Server zu einem Cluster hinzufügen

    • Endpunkte (Ports) für den Administrationsserver oder einen Anwendungsserver konfigurieren

    • Ändern Sie die freigegebenen Dateispeicherverzeichnisse für die Cluster, einschließlich Logverzeichnis, dauerhaftes Speicherverzeichnis und Zwischenspeicherverzeichnis

  • Deployments und Services: Wählen Sie diese Option, wenn Sie die folgenden Aufgaben ausführen möchten:

    • Deployment-Ziele für die Cluster oder Server ändern

    • Serviceziele für die Cluster oder Server ändern

    • Benutzerdefinierte Serviceziele für die Server ändern

    • Ressourcenadapterziele für die Server ändern

  • JDBC: Wählen Sie diese Option, wenn Sie die JDBC-Datenquellziele für die Cluster oder Server ändern möchten.

  • JMS: Wählen Sie diese Option, wenn Sie die JMS-Ressourcenziele für die Cluster oder Server ändern möchten.

Klicken Sie nach der Auswahl der gewünschten Konfigurationsoptionen auf Weiter, um fortzufahren. Der als Nächstes angezeigte Bildschirm hängt von den gewählten Optionen ab.
