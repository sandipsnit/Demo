Geben Sie Deployment Manager-Informationen an

Auf diesem Bildschirm geben Sie den Hostnamen, SOAP-Port (nur für Verbindungsprozess), Administratorbenutzernamen und das Kennwort für den Deployment Manager an. All diese Felder sind Pflichtfelder.

  • Deployment Manager-Hostname: Wenn Sie eine neue Zelle erstellen, wird der Hostname des aktuellen Rechners als Standardwert für dieses Feld verwendet.

    Wenn Sie einen Rechner verbinden, geben Sie den Hostnamen des Deployment Managers an, der den Remote-Knoten verwaltet.

  • SOAP-Port: Dieses Feld wird nur angezeigt, wenn Sie einen Rechner verbinden. Geben Sie den SOAP-Port für die Verbindung mit dem Deployment Manager-Knoten an. Dabei handelt es sich um den SOAP-Port des Deployment Managers, mit dem Sie diesen Rechner verbinden möchten. Der Standardwert lautet normalerweise 8880. Wenn Sie den SOAP-Port für den Deployment Manager nicht wissen, öffnen Sie die folgende Datei auf dem Deployment Manager-Host:

    WAS_HOME/profiles/DepMgr_name/logs/AboutThisProfile.txt

    wobei WAS_HOME das WebSphere-Standardverzeichnis und DepMgr_name der Name des entsprechenden Deployment Manager-Profils auf dem angegebenen Host ist.

  • Admin-Benutzername: Wenn Sie eine neue Zelle erstellen, geben Sie den Administrator-Benutzernamen für die Anmeldung beim Deployment Manager ein.

    Wenn Sie einen Rechner verbinden, geben Sie den Administrator-Benutzernamen ein, der auf dem angegebenen Host für den Deployment Manager konfiguriert wurde.

  • Kennwort: Wenn Sie eine neue Zelle erstellen, geben Sie das Administratorkennwort für die Anmeldung beim Deployment Manager ein.

    Wenn Sie einen Rechner verbinden, geben Sie das Administratorkennwort ein, das auf dem angegebenen Host für den Deployment Manager konfiguriert wurde.

  • Kennwort bestätigen: Dieses Feld wird nur angezeigt, wenn Sie eine neue Zelle erstellen. Geben Sie das Administratorkennwort zur Bestätigung erneut ein. Diese Eingabe muss identisch mit dem Wert im Feld Kennwort sein.

Wenn Sie jedes Feld aufgefüllt haben, klicken Sie auf Weiter, um fortzufahren.
