JDBC Data Sources als Ziel für Cluster oder Server definieren

Auf diesem Bildschirm können Sie JDBC-Datenquellen auf Server oder Cluster ausrichten.

So richten Sie eine JDBC-Datenquelle auf ein Cluster oder einen Server aus:

  1. Wählen Sie im Listenfeld Ziel das Cluster oder den Server, auf das bzw. den Sie eine Datenquelle ausrichten möchten.

    Der Name des gewählten Ziels wird als Titel des Listenfeldes auf der rechten Seite angezeigt.

  2. Aktivieren Sie im Listenfeld target_name die Kontrollkästchen für die Datenquellen, die Sie auf den gewählten Server oder das gewählte Cluster ausrichten möchten.


    Hinweis:

    Wenn Sie einen Server im Listenfeld Ziel wählen, sind einige der Kontrollkästchen im Listenfeld target_name möglicherweise deaktiviert. Dies weist auf Datenquellen hin, die bereits auf das Cluster ausgerichtet sind, das den gewählten Server enthält.

    Wenn Sie Datenquellen gewählt haben, wird der Name des Zielclusters oder -servers in der Spalte Ziel im Listenfeld target_name angezeigt.

  3. Wiederholen Sie die Schritte 1 und 2 nach Bedarf für weitere Cluster und Server.

  4. Wenn Sie alle erforderlichen Einstellungen vorgenommen haben, klicken Sie auf die Schaltfläche Weiter.
