Ressourcenadapter als Ziel für Server definieren

Auf diesem Bildschirm können Sie Ressourcenadapter auf einzelne Server ausrichten. Sie können Ressourcenadapter nicht auf Cluster ausrichten. Server, die Clustern zugewiesen wurden, sind im Listenfeld Ziel enthalten.

Ressourcenadapter, die mit dem Produkt verknüpft sind, für das Sie die Zelle konfigurieren (Beispiel: SOA) werden automatisch auf den Server ausgerichtet, der für dieses Produkt erstellt wurde. In diesem Bildschirm können Sie Ressourcenadapter auf zusätzliche Server ausrichten.

So richten Sie Ressourcenadapter auf Server aus:

  1. Wählen Sie im Listenfeld Ziel den Server, auf den Sie Ressourcenadapter ausrichten möchten.

    Der Name des gewählten Ziels wird als Titel des Listenfeldes auf der rechten Seite angezeigt.

  2. Aktivieren Sie im Listenfeld target_name die Kontrollkästchen für die Ressourcenadapter, die Sie auf den gewählten Server ausrichten möchten.

    Die hier angezeigten Ressourcenadapter hängen von den Produkten ab, die Sie auf dem Bildschirm Produkte zur Zelle hinzufügen weiter vorne im Assistenten ausgewählt haben.

    Wenn Sie Ressourcenadapter gewählt haben, werden die Namen der Zielserver in der Spalte Ziel des Listenfeldes target_name angezeigt.

  3. Wiederholen Sie die Schritte 1 und 2 nach Bedarf für weitere Server.

  4. Wenn Sie alle erforderlichen Einstellungen vorgenommen haben, klicken Sie auf die Schaltfläche Weiter.

Wenn Sie eine vorhandene Zelle neu konfigurieren und einen Server löschen, auf den derzeit Ressourcenadapter ausgerichtet sind, richtet der Konfigurationsassistent diese Ressourcenadapter automatisch wie folgt neu aus:

  • Wenn die Ressourcenadapter ursprünglich nur auf den Server ausgerichtet waren, den Sie jetzt löschen (das heißt, nach dem Löschen des Servers hätten die Ressourcenadapter in der geänderten Zelle keine Ziele mehr), richtet der Konfigurationsassistent automatisch die Ressourcenadapter neu auf alle berechtigen Ziele aus.

    Berechtigte Ziele sind alle Server, die nicht in der Konfigurationsgruppenspezifikation (config-groups.xml) einer enthaltenen Produktvorlage definiert sind. In der Datei config-groups.xml angegebene Server gehören zu der Vorlage und sind daher nicht für automatische Zieldefinition berechtigt.

  • Falls die Ressourcenadapter ursprünglich auf mehrere Ziele ausgerichtet waren (einschließlich Server und Administrationsserver) und einer der als Ziel definierten Server gelöscht wird, werden die verbleibenden Zielverknüpfungen beibehalten. Der Konfigurationsassistent nimmt keine erneute Ausrichtung der Ressourcenadapter vor.
