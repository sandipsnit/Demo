Komponentenschema testen

Auf diesem Bildschirm können Sie die Konfigurationen testen, die Sie auf dem vorherigen Bildschirm für die Komponentenschemas angegeben haben.

Aktivieren Sie die Kontrollkästchen neben den Namen der zu testenden Schemas, und klicken Sie dann auf Verbindungen testen.

Der Assistent testet die Konfiguration für jedes Schema, indem er versucht, die Verbindung mit einem URL aufzunehmen, der mit den bei der Schemakonfiguration definierten Angaben zu Treiber, Host, Port und sonstigen Informationen erstellt wurde.

Das Ergebnis des Testes wird in der Spalte Status angezeigt. Details werden im Bereich Verbindungsergebnis-Log angezeigt.

Wenn die Schematests erfolgreich abgeschlossen wurden, klicken Sie auf Weiter, um fortzufahren.
