JDBC-Datenquellen testen

Auf diesem Bildschirm können Sie die Datenquellverbindungen testen, die Sie auf den Bildschirmen JDBC-Datenquellen konfigurieren und Oracle RAC-Multidatenquellen konfigurieren konfiguriert haben.


Hinweise:

Um die Datenbankverbindungen zu testen, muss die Datenbank, bei der Sie sich anmelden, hochgefahren sein.

Wenn Sie die Verbindungen zu diesem Zeitpunkt nicht testen möchten, wählen Sie keine Datenquellen. Klicken Sie zum Fortfahren auf Weiter, und klicken Sie im daraufhin angezeigten Warndialogfeld auf "OK".

Aktivieren Sie das Kontrollkästchen für jede zu testende Datenquelle, und klicken Sie dann auf Verbindungen testen.

Der Assistent testet die Konfiguration für jede gewählte Datenquelle, indem er versucht, die Verbindung mit einem URL aufzunehmen, der mit den bei der Konfiguration der Datenquelle definierten Angaben zu Treiber, Host, Port und sonstigen Informationen erstellt wurde.

Das Ergebnis des Testes wird in der Spalte Status angezeigt. Details werden im Bereich Verbindungsergebnis-Log angezeigt.

Wenn die Tests erfolgreich abgeschlossen wurden, klicken Sie auf Weiter, um fortzufahren.
