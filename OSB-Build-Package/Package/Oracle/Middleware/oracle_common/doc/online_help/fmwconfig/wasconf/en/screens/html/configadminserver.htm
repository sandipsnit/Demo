Configure the Application Server

This screen appears only for IBM WebSphere AS. Specify the following Application Server settings:

  • Name: The name of the Application Server. It defaults to OracleAdminServer or to the Application Server name that you specified on the Specify Cell, Profile and Node Name Information screen. You cannot change the name.

  • Node name: The name of the Application Server node.

  • Development mode: Select this check box to run the Application Server in development mode. Leave this check box empty to run the Application Server in production mode. For information about the differences between development mode and production mode, see the IBM WebSphere documentation.

When you are done, click Next to continue.
