Configuring Cell

This screen shows the progress of the cell configuration process.

When the cell configuration process completes, click Done to close the Configuration Wizard.

On Windows operating systems, to start the Deployment Manager after closing the Configuration Wizard, select the Start Deployment Manager check box. This check box is not available on UNIX operating systems.
