Configurar Servidor de Administración

Especifique la siguiente configuración del Servidor de Administración:

  • Nombre: Nombre del Servidor de Administración. Está definido por defecto en OracleAdminSever y no se puede cambiar.

  • Nombre del Nodo: Esta lista desplegable contiene todos los nodos disponibles. Seleccione el nodo en el que desea que resida el Servidor de Administración.

  • Modo de Desarrollo: Active esta casilla de control si desea ejecutar el servidor de administración en modo de desarrollo. Deje esta casilla de control sin activar para ejecutar el servidor de administración en modo de producción. Para obtener información acerca de las diferencias entre el modo de desarrollo y el de producción, consulte la documentación de WebSphere.

Cuando haya finalizado, haga clic en Siguiente para continuar.
