Configuración de Celda

En esta pantalla se muestra el progreso del proceso de configuración de la celda.

Cuando finalice el proceso de configuración de la celda, haga clic en Listo para cerrar el Asistente de Configuración.

En los sistemas Windows, si desea iniciar el Gestor de Despliegue después de cerrar el Asistente de Configuración, active la casilla de control Iniciar Gestor de Despliegue. Esta casilla de control no está disponible para los sistemas UNIX.
