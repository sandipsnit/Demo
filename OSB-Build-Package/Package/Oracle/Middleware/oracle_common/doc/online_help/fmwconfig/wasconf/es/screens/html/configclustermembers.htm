Configurar Miembros de Cluster Adicionales

Utilice esta pantalla para agregar servidores de aplicaciones adicionales al cluster. Cuando agrega un servidor de aplicaciones nuevo a un cluster, sus atributos se clonan a partir del miembro primario del cluster.

  • Nombre: Nombre del servidor de aplicaciones. El nombre por defecto de un servidor nuevo es new_ClusterMember_n, donde el valor n empieza en 1 y se va incrementando para cada servidor nuevo que agregue. Para cambiar el nombre del servidor, haga clic en el campo Nombre e introduzca el nuevo nombre.

  • Nombre del Nodo: Lista desplegable de todos los nodos disponibles. Haga clic en la lista para seleccionar el nodo en el que reside este servidor de aplicaciones. Si aún no ha federado otras máquinas en la celda actual, sólo habrá un nodo disponible en esta lista.

  • Nombre del Cluster: Lista desplegable de todos los clusters disponibles. Seleccione el cluster al que desea asignar este servidor.

  • Modo de Desarrollo: Active esta casilla de control para ejecutar el servidor de aplicaciones en modo de desarrollo. Deje esta casilla sin marcar para ejecutar el servidor de aplicaciones en modo de producción. Para obtener información acerca de las diferencias entre el modo de producción y el modo de desarrollo, consulte la documentación de WebSphere.

Para agregar un miembro de cluster adicional:

  1. Haga clic en Agregar. Se agrega una nueva fila a la tabla.

  2. Introduzca el nombre del servidor y seleccione el valor del modo de desarrollo, cluster y nodo adecuados.

  3. Repita los pasos 1 y 2 para cada miembro de cluster que desee agregar.

Para suprimir un miembro de cluster, seleccione el miembro de cluster y haga clic en Suprimir. Sólo puede suprimir un miembro de cluster a la vez.

Para cambiar la visualización de formato tabular por defecto (por un separador por cada cluster), haga clic en Cambiar Visualización. Vuelva a hacer clic en Cambiar Visualización para regresar al formato tabular.

Cuando haya finalizado, haga clic en Siguiente para continuar.


Nota:

Los cambios que realice en esta pantalla no se guardan en la configuración de la celda hasta que no se hace clic en Crear en la última pantalla del Asistente de Configuración. Si sale del asistente o hace clic en Anterior para volver a la pantalla anterior, deberá volver a realizar los cambios.
