Configurar Clusters

Un cluster es un grupo de servidores de aplicaciones que funcionan de forma conjunta para proporcionar escalabilidad y una alta disponibilidad para las aplicaciones. La creación de clusters permite agrupar servidores de modo que funcionen como una sola unidad para alojar aplicaciones y recursos.

Utilice esta pantalla para agregar un cluster y seleccionar el servidor de aplicaciones que será el miembro principal del cluster. Todos los miembros de cluster adicionales se clonan a partir de este miembro de cluster principal. Después de crear un cluster, puede agregar servidores de aplicaciones adicionales al cluster en la pantalla siguiente.

También puede utilizar esta pantalla para suprimir un cluster o modificar la configuración de un cluster existente.

Para crear un cluster:

  1. Haga clic en Agregar. Aparece una línea nueva en la tabla. El nombre por defecto de un cluster nuevo es new_Cluster_n, donde el valor n empieza en 1 y ser va incrementando para cada cluster nuevo que agregue. Para cambiar el nombre por defecto, introduzca el nombre que desee en la columna Nombre. El nombre del cluster debe ser único entre todos los nombres de componentes de la celda.

  2. Seleccione un servidor de aplicaciones ya existente como miembro principal del cluster seleccionando el servidor de la lista desplegable Primer Miembro de Cluster. En esta lista aparecen sólo los servidores de aplicaciones que aún no se hayan asignado a ningún cluster.

  3. Repita los pasos 1 y 2 para cada cluster adicional que desee crear.

  4. Cuando haya finalizado, haga clic en Siguiente. Puede agregar servidores adicionales al cluster en la pantalla Configurar Miembros de Cluster Adicionales, que aparece cuando se hace clic en Siguiente.


    Nota:

    Los cambios que realice en esta pantalla no se guardan en la configuración de la celda hasta que no se hace clic en Crear en la última pantalla del Asistente de Configuración. Si sale del asistente o hace clic en Anterior para volver a la pantalla anterior, deberá volver a realizar los cambios.

Para suprimir un cluster, seleccione el cluster y haga clic en Suprimir. Cuando se suprime un cluster, no se suprime el servidor de aplicaciones que es miembro principal del cluster. Únicamente se elimina del cluster. Sólo se puede suprimir un cluster a la vez.

Para cambiar la configuración de un cluster, seleccione el cluster y realice los cambios correspondientes.

Para cambiar la visualización de formato tabular por defecto (por un separador por cada cluster), haga clic en Cambiar Visualización. Vuelva a hacer clic en Cambiar Visualización para regresar al formato tabular.

Cuando haya terminado de configurar los clusters, haga clic en Siguiente para continuar.

Para obtener más información acerca de los clusters en un entorno WebSphere, consulte la documentación de WebSphere.
