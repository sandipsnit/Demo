Configurar Orígenes de Datos JDBC

Un origen de datos JDBC contiene un pool de conexiones de base de datos que se crean al crear la instancia del origen de datos (al desplegarla o dirigirla o al iniciar el servidor). Las aplicaciones consultan un origen de datos en el árbol de JNDI y, a continuación, solicitan una conexión. Si las aplicaciones ya no necesitan las conexiones, las devuelven al pool de conexiones del origen de datos.

Utilice esta pantalla para configurar los orígenes de datos JDBC definidos en la celda.

Los orígenes de datos JDBC asociados a los productos para los que se va crear la celda se muestran en la mitad inferior de la pantalla.


Nota:

Para configurar uno o más orígenes de datos como orígenes de datos múltiples de Oracle RAC, active la casilla de control junto al nombre de los orígenes de datos necesarios y la casilla de control Configurar los orígenes de datos seleccionados como orígenes de datos múltiples de RAC en el panel siguiente. A continuación, haga clic en el botón Siguiente.

Para seleccionar los orígenes de datos para los que desea especificar la configuración, active la casilla de control junto a cada nombre del origen de datos.


Notas:

Si se seleccionan varios orígenes de datos, puede que se muestre el texto Varía entre los orígenes de datos en determinados campos, lo que indica que los valores actuales de estos campos no son los mismos en los orígenes de datos seleccionados. Si cambia los valores de estos campos, los valores nuevos se aplican de manera uniforme a los orígenes de datos seleccionados.

Los valores por defecto de los parámetros de origen de datos (por ejemplo, proveedor, controlador, nombre del host y número de puerto) dependen de los valores especificados en las plantillas de aplicación.

  • Proveedor: Seleccione el proveedor de bases de datos.

  • Controlador: Seleccione el controlador JDBC que desea utilizar para conectar con la base de datos. La lista incluye los controladores JDBC comunes para el proveedor de base de datos seleccionado.

  • Nombre de Usuario: Introduzca el nombre de usuario para conectarse a la base de datos.

  • Contraseña: Introduzca la contraseña para el nombre de usuario especificado.

  • DBMS/Servicio: Introduzca un nombre de servicio o SID de DBMS. El valor que introduzca depende del controlador que haya seleccionado.

    • Si el nombre del controlador de Oracle que ha seleccionado contiene las palabras "para conexiones de instancia", debe introducir el SID.

    • Si el nombre del controlador de Oracle contiene las palabras "para conexiones de servicio", debe introducir el nombre del servicio.

    • Para obtener más información acerca de la configuración de un controlador de DataDirect, consulte la documentación de DataDirect.

  • Nombre del Host: Introduzca el nombre del servidor que aloja la base de datos.

  • Puerto: Introduzca el número de puerto que se va a utilizar para conectar al servidor.

Los valores especificados se muestran en las columnas correspondientes de la lista de orígenes de datos para el origen de datos seleccionado.

Cuando haya terminado de configurar los orígenes de datos JDBC, haga clic en Siguiente para continuar.
