Configurar Esquema de Componentes de JDBC

Para algunos componentes de Fusion Middleware (por ejemplo, SOA y Web Center Spaces), puede que haya que definir orígenes de datos JDBC como parte del esquema de base de datos del componente, que se carga (durante la instalación ) mediante la Utilidad de Creación de Repositorios (RCU).

Al crear una celda para dichos componentes con el Asistente de Configuración, puede configurar los valores del esquema del componente JDBC (por ejemplo, el controlador de la base de datos, el propietario del esquema, la contraseña, etc.).

Los esquemas de componentes de JDBC asociados a los productos para los que se va a crear la celda (por ejemplo, SOA) se muestran en la mitad inferior de la pantalla.


Nota:

Para configurar uno o varios de los esquemas como orígenes de datos múltiples de Oracle RAC, active las casillas de control junto a los nombres de esquema y la casilla Configurar los esquemas de componentes seleccionados como esquemas de orígenes de datos múltiples de RAC en el panel siguiente. Haga clic en Siguiente cuando haya terminado.

Para seleccionar los esquemas para los que desea especificar la configuración de orígenes de datos, active la casilla de control junto a los nombres de esquema.


Nota:

Si se seleccionan varios esquemas de componentes, puede que se muestre el texto Varía entre los esquemas de componentes en determinados campos, lo que indica que los valores actuales de estos campos no son los mismos en los componentes de esquemas seleccionados. Si cambia los valores de estos campos, los valores nuevos se aplican de manera uniforme a los esquemas de componentes seleccionados.

Los valores por defecto de los parámetros del esquema del componente (por ejemplo, proveedor, controlador, nombre del host y número de puerto) dependen de los valores especificados en las plantillas de aplicación.

  • Proveedor: Seleccione el proveedor de bases de datos.

  • Controlador: Seleccione el controlador JDBC que desea utilizar para conectar con la base de datos. La lista incluye los controladores JDBC comunes para el proveedor de base de datos seleccionado.

  • Propietario del Esquema: Introduzca el nombre de usuario para conectarse a la base de datos.

  • Contraseña del Esquema: Introduzca la contraseña para el propietario del esquema especificado.

  • DBMS/Servicio: Introduzca un nombre de DBMS de base de datos o un nombre de servicio si ha seleccionado un controlador de servicio.

  • Nombre del Host: Introduzca el nombre del servidor que aloja la base de datos.

  • Puerto: Introduzca el número de puerto que se va a utilizar para establecer la conexión con el servidor que aloja la base de datos.

Los valores especificados se muestran en las columnas correspondientes de la lista de esquemas para los esquemas seleccionados.

Cuanto termine de configurar los esquemas de componentes, haga clic en Siguiente para continuar.
