Configurar Orígenes de Datos Múltiples de RAC

Utilice esta pantalla para configurar los orígenes de datos que decide configurar como orígenes de datos de Oracle RAC.

Los orígenes de datos para los que se han configurado los orígenes de datos de Oracle RAC en la pantalla Configurar Orígenes de Datos JDBC se muestran en la mitad inferior de la pantalla.

Para seleccionar los orígenes de datos para los que desea especificar la configuración, active la casilla de control junto a cada nombre del origen de datos.


Nota:

Si se seleccionan varios orígenes de datos, puede que se muestre el texto Varía entre los orígenes de datos en determinados campos, lo que indica que los valores actuales de estos campos no son los mismos en los orígenes de datos seleccionados. Si cambia los valores de estos campos, los valores nuevos se aplican de manera uniforme a los orígenes de datos seleccionados.

  • Controlador: Especifique el controlador JDBC que se utilizará para conectar a la base de datos.

  • Nombre del Servicio: Introduzca un nombre de servicio de base de datos de Oracle RAC.

  • Nombre de Usuario: Introduzca el nombre de usuario para conectarse a la base de datos.

  • Contraseña: Introduzca la contraseña para el nombre de usuario especificado.

  • Nombre del Host: Introduzca el nombre del servidor que aloja las instancias de la base de datos de Oracle RAC.

  • Nombre de la Instancia: Introduzca el nombre de cada instancia de la base de datos Oracle.

  • Puerto: Introduzca el número de puerto que se va a utilizar para conectar con el servidor que aloja la base de datos.

Para agregar una nueva instancia de base de datos, haga clic en Agregar y, a continuación, especifique el nombre de host, el nombre de instancia y el número de puerto.

Cuando haya terminado de configurar los orígenes de datos múltiples de RAC, haga clic en Siguiente.
