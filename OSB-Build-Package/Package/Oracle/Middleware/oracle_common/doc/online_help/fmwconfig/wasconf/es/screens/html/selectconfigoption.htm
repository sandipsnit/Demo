Seleccionar Opción de Configuración

Seleccione una de las siguientes opciones. Las opciones que vea dependerán de su configuración actual.

  • Crear y Configurar Celda: Seleccione esta opción para crear una celda de WebSphere nueva. A continuación, aparece la pantalla Especificar Información de Celda, Perfil y Nombre de Nodo.

  • Federar Máquina y Configurar Celda: Seleccione esta opción para agregar el nodo de WebSphere actual a una celda de Fusion Middleware existente. A continuación, aparece la pantalla Especificar Información de Perfil y Nombre de Nodo. En este caso, deberá ejecutar el Asistente de Configuración en el nodo que desee federar en una celda que se haya creado previamente a partir del nodo del Gestor de Despliegue.

  • Seleccionar y Configurar Celda Existente: Esta opción sólo está disponible si ya ha creado una celda. Para cambiar la configuración de una celda existente, seleccione esta opción. En el cuadro de lista, haga clic en la celda que desee configurar (se resaltará la celda seleccionada). A continuación, aparece la pantalla Cargando Información de Celda. Consulte Progreso de Celda para obtener información acerca de esta pantalla.
