Seleccionar Configuración Opcional

Seleccione todas las categorías (si hay alguna) para las que desea definir la configuración avanzada. Para cada categoría que seleccione, se muestra la pantalla de configuración correspondiente para que pueda definir la configuración avanzada. Si no selecciona ningún elemento en esta pantalla, a continuación aparece la pantalla Resumen de Configuración.


Notas:

Las categorías que se muestran en esta pantalla dependen de los recursos definidos en las plantillas que ha seleccionado para la celda.

  • Servidores de Aplicaciones y Clusters: Seleccione esta opción si desea realizar alguna de las siguientes tareas:

    • Agregar, suprimir o clonar un servidor de aplicaciones

    • Cambiar el nombre de un servidor de aplicaciones

    • Activar o desactivar el modo de desarrollo de un servidor de aplicaciones

    • Agregar clusters a la celda y agregar servidores al cluster

    • Configurar los puntos finales (puertos) para el servidor de administración o para cualquier de un servidor de aplicaciones

    • Cambiar las ubicaciones del directorio compartido del almacén de archivos para los clusters, incluidos el directorio de log, el directorio de almacén permanente y el directorio de almacén temporal

  • Despliegues y Servicios: Seleccione esta opción si desea realizar alguna de las siguientes tareas:

    • Cambiar los destinos de despliegue para los clusters o servidores

    • Cambiar los destinos de servicio para los clusters o servidores

    • Cambiar los destinos de servicio personalizado para los servidores

    • Cambiar los destinos de adaptador de recursos personalizado para los servidores

  • JDBC: Seleccione esta opción si desea cambiar los destinos de origen de datos de JDBC para los clusters o servidores.

  • JMS: Seleccione esta opción si desea cambiar los destinos de origen de datos de JMS para los clusters o servidores.

Tras seleccionar las opciones de configuración pertinentes, haga clic en Siguiente para continuar. La pantalla que se muestra a continuación cambia en función de las opciones seleccionadas.
