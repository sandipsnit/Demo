Especificar Información de Celda, Perfil y Nombre de Nodo

Introduzca los valores adecuados en los siguientes campos. Todos los campos son necesarios. Utilice únicamente caracteres alfanuméricos.

  • Nombre de Celda: Introduzca un nombre único para la nueva celda. Este es el nombre del directorio en el que se creará la celda, bajo los dos directorios siguientes:

    • WAS_HOME/profiles/DepMgrProfile/config/cells, donde WAS_HOME es el directorio de instalación de WebSphere y DepMgrProfile es el nombre del perfil del Gestor de Despliegue.

    • WAS_HOME/profiles/AppSrvrProfile/config/cells, donde WAS_HOME es el directorio de instalación de WebSphere y AppSrvrProfile es el nombre del perfil del servidor de aplicaciones.

    El nombre por defecto es machine_nameCelln, donde machine_name es el nombre de la máquina actual y n es el próximo número de dos dígitos en la secuencia de esa máquina. Por ejemplo, si el nombre de la máquina es machine1 ya existe Cell01, el nombre por defecto de la celda es machine1Cell02.

  • Nombre del Perfil del Gestor de Despliegue: Introduzca un nombre único de perfil del Gestor de Despliegue para esta celda. Este es el nombre del directorio en el que se crea el perfil del Gestor de Despliegue, bajo WAS_HOME/profiles.

    El nombre por defecto es Dmgrn, donde n es el próximo número de dos dígitos en la secuencia. Por ejemplo, si el perfil del Gestor de Despliegue Dmgr01 ya existe, el valor por defecto para este campo es Dmgr02.

  • Nombre del Nodo del Gestor de Despliegue: Introduzca un nombre único para el nodo del Gestor de Despliegue para esta celda.

    El nombre por defecto es machine_nameCellManagern, donde machine_name es el nombre de la máquina actual, y n es el próximo número de dos dígitos en la secuencia de esa máquina. Por ejemplo, si el nombre de la máquina es machine1 y ya existe CellManager01, el nombre por defecto de la celda es machine1CellManager02.

  • Nombre de Perfil de Servidor de Aplicaciones: Introduzca un nombre único para el perfil del servidor de aplicaciones para esta celda. Este es el nombre del directorio en el que se crea el perfil del servidor de aplicaciones, bajo WAS_HOME/profiles.

    El nombre por defecto es Customn, donde n es el próximo número de dos dígitos en la secuencia. Por ejemplo, si el perfil del servidor de aplicaciones Custom01 ya existe, el valor por defecto para este campo es Custom02.

  • Nombre de Nodo de Servidor de Aplicaciones: Introduzca un nombre único para el nodo del servidor de aplicaciones para esta celda.

    El nombre por defecto es machine_nameNoden, donde machine_name es el nombre de la máquina actual y n es el próximo número de dos dígitos en la secuencia de esa máquina. Por ejemplo, si el nombre de la máquina es machine1 y ya existe Node01, el nombre por defecto del nodo es machine1Node02.

Cuando termine de rellenar cada campo, haga clic en Siguiente.
