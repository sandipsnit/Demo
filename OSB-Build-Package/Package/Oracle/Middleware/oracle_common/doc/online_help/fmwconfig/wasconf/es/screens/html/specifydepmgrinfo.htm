Especificar Información del Gestor de Despliegue

Utilice esta pantalla para especificar el nombre de host, el puerto SOAP (sólo en procesos de federación), el nombre del usuario administrador y la contraseña. Todos los campos son necesarios.

  • Nombre de Host del Gestor de Despliegue: Al crear una celda nueva, el valor por defecto de este campo es el nombre de host de la máquina actual.

    Cuando federe una máquina, introduzca el nombre del Gestor de Despliegue que gestionará el nodo remoto.

  • Puerto SOAP: Este campo se muestra únicamente cuando se federa una máquina. Introduzca el puerto SOAP que va a utilizar para conectar al nodo del Gestor de Despliegue. Este es el puerto SOAP del Gestor de Despliegue con el que desea federar esta máquina. Normalmente el valor por defecto es 8880. Si no conoce el puerto SOAP del Gestor de Despliegue, abra el siguiente archivo en el host del Gestor de Despliegue:

    WAS_HOME/profiles/DepMgr_name/logs/AboutThisProfile.txt

    donde WAS_HOME es el directorio raíz de WebSphere y DepMgr_name es el nombre del perfil del Gestor de Despliegue adecuado en el host especificado.

  • Usuario Administrador: Al crear una celda nueva, introduzca el nombre del usuario administrador que desea utilizar para conectarse al Gestor de Despliegue.

    Cuando federe una máquina, introduzca el nombre del usuario administrador que se ha configurado para el Gestor de Despliegue en el host especificado.

  • Contraseña: Al crear una celda nueva, introduzca la contraseña del administrador que desea utilizar para conectarse al Gestor de Despliegue.

    Cuando federe una máquina, introduzca la contraseña del administrador que se ha configurado para el Gestor de Despliegue en el host especificado.

  • Confirmar Contraseña: Este campo se muestra únicamente cuando se crea una celda nueva. Vuelva a introducir la contraseña del administrador para confirmarla. Esta entrada debe coincidir exactamente con la entrada del campo Contraseña.

Cuando haya terminado de rellenar cada campo, haga clic en Siguiente para continuar.
