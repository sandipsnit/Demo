Especificar Información de Nombre de Nodo y Perfil

  • Nombre de Perfil de Servidor de Aplicaciones: Introduzca un nombre de perfil de Servidor de Aplicaciones único para este nodo. Es el nombre del directorio en el que se crea el perfil de Servidor de Aplicaciones, en WAS_HOME/profiles en la máquina en la que está creando el nodo y en el host del Gestor de Despliegue.

    El nombre por defecto es Customn, donde n es el siguiente número de dos dígitos en la secuencia del nodo del Gestor de Despliegue que ha especificado en la pantalla anterior. Por ejemplo, si el perfil del Servidor de Aplicaciones Custom01 ya existe en la celda, el nombre por defecto para este campo será Custom02.

    Si cambia el nombre de perfil por defecto, asegúrese de que es único dentro de la celda.

  • Nombre de Nodo de Servidor de Aplicaciones: Introduzca el nombre del nodo que se utilizará para este Servidor de Aplicaciones.

    El nombre por defecto es nombre_máquinaNoden, donde nombre_máquina es el nombre de la máquina actual y n es el siguiente número de dos dígitos en la secuencia de la máquina actual. Por ejemplo, si el nombre de la máquina es máquina y Node01 ya existe en la máquina, el nombre por defecto será máquina2Node02.

    Si cambia el nombre de nodo por defecto, asegúrese de que es único dentro de la celda.

Cuando haya terminado de rellenar cada campo, haga clic en Siguiente para continuar.
