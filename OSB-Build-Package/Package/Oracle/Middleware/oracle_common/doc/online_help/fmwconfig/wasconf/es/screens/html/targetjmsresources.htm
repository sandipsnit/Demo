Dirigir Recursos de JMS a Clusters o Servidores

Utilice esta pantalla para dirigir recursos de JMS a servidores o clusters.

Para dirigir un recurso de JMS a un cluster o servidor:

  1. En el cuadro de lista Destino, seleccione el cluster o servidor al que desea dirigir un recurso de JMS.

    El nombre del destino seleccionado se muestra como el título del cuadro de lista de la derecha.

  2. En el cuadro de lista con los nombres de destino, active las casillas de control correspondientes a los recursos de JMS que desea dirigir al servidor o cluster seleccionado.


    Nota:

    Si selecciona un servidor en el cuadro de lista Destino, algunas de las casillas de control del cuadro de lista con los nombres de destino pueden estar desactivadas, lo que indica que los recursos de JMS ya se han dirigido al cluster que contiene el servidor seleccionado.

    Tras seleccionar orígenes de datos, el nombre del cluster o servidor de destino se muestra en la columna Destino del cuadro de lista con los nombres de destino.

  3. Repita los pasos 1 y 2 para los demás clusters y servidores según sea necesario.

  4. Después de realizar las selecciones requeridas, haga clic en Siguiente.
