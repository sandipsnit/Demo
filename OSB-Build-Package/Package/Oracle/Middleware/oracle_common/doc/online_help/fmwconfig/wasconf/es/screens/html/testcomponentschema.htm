Probar Esquema de Componentes

Utilice esta pantalla para probar las configuraciones que ha especificado para los esquemas de componentes de la pantalla anterior.

Active las casillas de control junto a los nombres de los esquemas que desea probar y, a continuación, haga clic en Probar Conexiones.

El asistente prueba la configuración de cada esquema mediante un intento de conexión a una URL creada con el controlador, host, puerto y demás información especificada durante la configuración del esquema.

El resultado de la prueba se indica en la columna Estado. Los detalles se muestran en la sección Log de Resultados de Conexión.

Cuando se completen correctamente las pruebas del esquema, haga clic en Siguiente para continuar.
