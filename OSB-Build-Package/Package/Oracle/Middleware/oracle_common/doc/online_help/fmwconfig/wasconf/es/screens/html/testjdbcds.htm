Probar Orígenes de Datos JDBC

Utilice esta pantalla para probar las conexiones del origen de datos que ha configurado en las pantallas Configurar Orígenes de Datos JDBC y Configurar Orígenes de Datos Múltiples de RAC.


Notas:

Para probar las conexiones de base de datos, la base de datos a la que se va a conectar debe estar en ejecución.

Si no desea probar las conexiones en este momento, no seleccione ningún origen de datos. Haga clic en Siguiente para continuar y, a continuación, haga clic en Aceptar en el cuadro de diálogo de advertencia que aparece.

Active la casilla de control de cada origen de datos que desee probar y, a continuación, haga clic en Probar Conexiones.

El asistente prueba la configuración de cada origen de datos seleccionado mediante un intento de conexión a una URL creada con el controlador, host, puerto y demás información especificada durante la configuración del origen de datos.

El resultado de la prueba se indica en la columna Estado. Los detalles se muestran en la sección Log de Resultados de Conexión.

Cuando terminen correctamente las pruebas, haga clic en Siguiente para continuar.
