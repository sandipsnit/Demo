Ajouter des produits à la cellule

Choisissez les produits à inclure dans cette cellule. Par défaut, l'option Oracle CIE Core est sélectionnée et ne peut pas être désélectionnée. Cette option permet d'ajouter le service personnalisé en ligne CIE obligatoire au domaine et de le cibler sur OracleAdminServer.

Si vous modifiez le nom d'une cellule existante ou fédérez un ordinateur à une cellule, les produits déjà inclus dans la cellule sont sélectionnés et ne peuvent pas être désélectionnés.


Remarque :

Généralement, lorsque vous créez une cellule dans le noeud du gestionnaire de déploiement, vous sélectionnez uniquement Oracle CIE Core. Dans une configuration qui inclut plusieurs noeuds, sélectionnez des produits supplémentaires lorsque vous exécutez l'assistant Configuration sur les ordinateurs que vous fédérez à la cellule sur le noeud du gestionnaire de déploiement. Même si vous pouvez cibler les composants Fusion Middleware sur le serveur d'administration ou les serveurs d'applications du noeud du gestionnaire de déploiement, il est préférable de les cibler sur les serveurs d'applications des noeuds distants.

Cochez les cases correspondant aux autres produits à inclure dans la cellule. Si vous sélectionnez un produit dépendant des ressources fournies par d'autres produits de la liste, ces produits sont également sélectionnés automatiquement. De même, si vous désélectionnez la case associée à un produit dont dépend un autre produit, la case à cocher du produit dépendant est également désélectionnée.

Si vous ajoutez des produits à une cellule existante, tous les produits ajoutés précédemment à la cellule sont déjà sélectionnés et ne peuvent pas être effacés.

Une fois que tous les produits à inclure dans la cellule ont été sélectionnés, cliquez sur Suivant pour continuer.


Remarques :

Lorsque vous cliquez sur Suivant, si l'un des produits sélectionnés exige des ressources fournies par un autre produit non sélectionné, une boîte de dialogue d'avertissement apparaît. Revenez à l'écran Ajouter des produits à la cellule pour sélectionner le produit dépendant avant de continuer.
