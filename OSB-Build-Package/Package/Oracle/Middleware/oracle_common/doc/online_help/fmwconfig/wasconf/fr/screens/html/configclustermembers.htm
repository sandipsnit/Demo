Configurer les membres de cluster supplémentaires

Utilisez cet écran pour ajouter des serveurs d'applications supplémentaires à un cluster. Lorsque vous ajoutez un nouveau serveur d'applications à un cluster, ses attributs sont clonés à partir du membre principal du cluster.

  • Nom : nom du serveur d'applications. Le nom par défaut d'un nouveau serveur est new_ClusterMember_n, où n commence à 1 et est incrémenté à chaque nouveau serveur ajouté. Pour modifier le nom du serveur, cliquez sur le champ Nom et saisissez le nouveau nom.

  • Nom du noeud : liste déroulante de tous les noeuds disponibles. Cliquez sur la liste pour sélectionner le noeud sur lequel le serveur d'applications réside. Si vous n'avez pas encore fédéré d'autres ordinateurs à la cellule en cours, un seul noeud est disponible dans cette liste.

  • Nom de cluster : liste déroulante de tous les clusters disponibles. Sélectionnez le cluster auquel affecter ce serveur.

  • Mode Développement : cochez cette case pour exécuter le serveur d'applications en mode développement. Laissez-la vide si vous souhaitez qu'il soit exécuté en mode production. Pour plus d'informations sur les différences entre le mode production et le mode développement dans un environnement WebSphere, reportez-vous à la documentation WebSphere.

Pour ajouter un membre de cluster supplémentaire, procédez comme suit :

  1. Cliquez sur Ajouter. Une nouvelle ligne est ajoutée à la table.

  2. Saisissez le nom du serveur, puis sélectionnez le noeud, le cluster et le paramètre de mode développement appropriés.

  3. Répétez les étapes 1 et 2 pour chaque membre de cluster à ajouter.

Pour supprimer un membre de cluster, sélectionnez-le et cliquez sur Supprimer. Vous ne pouvez supprimer qu'un seul membre à la fois.

Pour passer de l'affichage par défaut sous forme de tableau à un affichage à onglets (un onglet par cluster), cliquez sur Changer d'affichage. Cliquez de nouveau sur Changer d'affichage pour revenir au format tabulaire.

Lorsque vous avez terminé, cliquez sur Suivant pour continuer.


Remarque :

Les modifications apportées sur cet écran ne seront enregistrées dans la configuration de cellule que lorsque vous cliquerez sur Créer sur le dernier écran de l'assistant Configuration. Si vous quittez l'assistant ou cliquez sur Précédent pour revenir à un écran précédent, vous devez effectuer à nouveau les modifications.
