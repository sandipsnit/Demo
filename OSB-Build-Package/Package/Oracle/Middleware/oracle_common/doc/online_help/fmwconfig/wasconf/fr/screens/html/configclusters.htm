Configurer les clusters

Un cluster est un groupe de serveurs d'applications qui fonctionnent simultanément pour fournir évolutivité et haute disponibilité aux applications. En créant des clusters, vous pouvez regrouper des serveurs de sorte qu'ils fonctionnent comme une unité unique pour l'hébergement d'applications et de ressources.

Utilisez cet écran pour ajouter un cluster et sélectionner le serveur d'applications qui fera office de membre principal du cluster. Tous les membres de cluster supplémentaires sont clonés à partir de ce membre principal. Une fois le cluster créé, vous pouvez ajouter d'autres serveurs d'applications au cluster sur l'écran suivant.

Vous pouvez également utiliser cet écran pour supprimer un cluster ou modifier les paramètres d'un cluster existant.

Pour créer un cluster :

  1. Cliquez sur Ajouter. Une nouvelle ligne apparaît dans la table. Le nom par défaut d'un nouveau cluster est new_Cluster_n, où n commence à 1 et est incrémenté à chaque nouveau cluster ajouté. Pour modifier le nom par défaut, saisissez le nom souhaité dans la colonne Nom. Le nom du cluster doit être unique parmi tous les noms de composants dans la cellule.

  2. Sélectionnez un serveur d'applications existant en tant que membre principal du cluster en le sélectionnant dans la liste déroulante Premier membre du cluster. Seuls les serveurs d'applications qui ne sont pas encore affectés à un cluster apparaissent dans cette liste.

  3. Répétez les étapes 1 et 2 pour chaque cluster supplémentaire à créer.

  4. Lorsque vous avez terminé, cliquez sur Suivant. Vous pouvez ajouter des serveurs supplémentaires au cluster sur l'écran Configurer les membres de cluster supplémentaires, qui apparaît lorsque vous cliquez sur Suivant.


    Remarque :

    Les modifications apportées sur cet écran ne seront enregistrées dans la configuration de cellule que lorsque vous cliquerez sur Créer sur le dernier écran de l'assistant Configuration. Si vous quittez l'assistant ou cliquez sur Précédent pour revenir à un écran précédent, vous devez effectuer à nouveau les modifications.

Pour supprimer un cluster, sélectionnez-le, puis cliquez sur Supprimer. Lorsque vous supprimez un cluster, vous ne supprimez pas le serveur d'applications qui est le membre principal du cluster ; ce dernier est simplement enlevé du cluster. Vous ne pouvez supprimer qu'un seul cluster à la fois.

Pour modifier les paramètres d'un cluster, sélectionnez-le et effectuez les modifications nécessaires.

Pour passer de l'affichage par défaut sous forme de tableau à un affichage à onglets (un onglet par cluster), cliquez sur Changer d'affichage. Cliquez de nouveau sur Changer d'affichage pour revenir au format tabulaire.

Une fois la configuration des clusters terminée, cliquez sur Suivant pour continuer.

Pour plus d'informations sur les clusters d'un environnement WebSphere, reportez-vous à la documentation WebSphere.
