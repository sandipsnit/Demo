Configurer les sources de données JDBC

Une source de données JDBC contient un pool de connexions de base de données créées en même temps que l'instance de source de données (lorsqu'elle est déployée ou ciblée, ou au démarrage du serveur). Les applications recherchent une source de données dans l'arborescence JNDI, puis demandent une connexion. Lorsque les applications n'ont plus besoin des connexions, elles les renvoient vers le pool de connexions de la source de données.

Utilisez cet écran pour configurer les sources de données JDBC définies dans la cellule.

Les sources de données JDBC associées aux produits pour lesquels vous créez la cellule sont répertoriées dans la partie inférieure de l'écran.


Remarque :

Pour configurer des sources de données en tant que sources de données multiples Oracle RAC, cochez la case située en regard du nom de la source de données, puis cochez la case Configurer les sources de données sélectionnées en tant que sources de données multiples RAC dans le panneau suivant. Cliquez ensuite sur le bouton Suivant.

Pour sélectionner les sources de données pour lesquelles indiquer des paramètres, cochez la case située en regard de leur nom.


Remarques :

Si vous sélectionnez plusieurs sources de données, le texte Varie selon les sources de données peut apparaître dans certains champs, ce qui indique que les valeurs en cours de ces champs sont différentes dans les sources de données sélectionnées. Si vous modifiez les valeurs de ces champs, les nouvelles valeurs sont appliquées de façon uniforme dans les sources de données sélectionnées.

Les valeurs par défaut des paramètres de source de données tels que Fournisseur, Pilote, Nom d'hôte et Numéro de port dépendent des valeurs indiquées dans les modèles d'application.

  • Fournisseur : sélectionnez le fournisseur de base de données.

  • Pilote : sélectionnez le pilote JDBC à utiliser pour la connexion à la base de données. La liste comprend les pilotes JDBC standard pour le fournisseur de base de données sélectionné.

  • Nom utilisateur : saisissez le nom utilisateur à employer pour la connexion à la base de données.

  • Mot de passe : entrez le mot de passe correspondant au nom utilisateur indiqué.

  • SGBD/Service : entrez un nom de service ou un SID SGBD. La valeur saisie dépend du pilote que vous avez sélectionné.

    • Si le nom du pilote Oracle sélectionné contient les mots "pour connexions d'instance", vous devez entrer le SID.

    • Si le nom du pilote Oracle contient les mots "pour connexions de service", vous devez entrer le nom de service.

    • Pour plus d'informations sur la configuration d'un pilote DataDirect, reportez-vous à la documentation DataDirect.

  • Nom d'hôte : saisissez le nom du serveur hébergeant la base de données.

  • Port : entrez le numéro de port à utiliser pour établir la connexion au serveur.

Les valeurs que vous indiquez sont affichées dans les colonnes appropriées de la liste de sources de données, pour la source de données sélectionnée.

Une fois la configuration des sources de données JDBC terminée, cliquez sur Suivant pour continuer.
