Configurer le schéma de composant de source de données RAC

Utilisez cet écran pour configurer les schémas de composant inclus dans la cellule WebSphere en tant que schémas Oracle RAC.

Les schémas de composant que vous avez choisi de configurer en tant que schémas Oracle RAC sur l'écran Configurer le schéma de composant JDBC de l'assistant sont répertoriés dans la partie inférieure de l'écran.

Dans la liste de schémas figurant dans la partie inférieure de l'écran, sélectionnez les schémas à configurer en tant que schémas Oracle RAC. Pour ce faire, cochez la case figurant en regard de leur nom.


Remarque :

Si vous sélectionnez plusieurs schémas, le texte Varie selon les schémas de composant peut apparaître dans certains champs, ce qui indique que les valeurs en cours de ces champs sont différentes dans les schémas sélectionnés. Si vous modifiez les valeurs de ces champs, les nouvelles valeurs sont appliquées de façon uniforme dans les schémas sélectionnés.

  • Pilote : sélectionnez le pilote JDBC à utiliser pour la connexion à la base de données.

  • Nom de service : saisissez un nom de service Oracle RAC de base de données.

  • Nom utilisateur : saisissez le nom utilisateur à employer pour la connexion à la base de données.

  • Mot de passe : entrez le mot de passe correspondant au nom utilisateur indiqué.

  • Nom d'hôte : saisissez le nom du serveur hébergeant les instances de base de données Oracle RAC.

  • Nom d'instance : entrez le nom de chaque instance de base de données Oracle.

  • Port : entrez les numéros de port à utiliser pour la connexion au serveur qui héberge la base de données.


Remarque :

Vous devez indiquer le nom d'hôte, le nom d'instance et le numéro de port d'au moins une instance de base de données.

Pour ajouter une instance de base de données, cliquez sur Ajouter, puis indiquez le nom d'hôte, le nom de l'instance et le port.

Les valeurs que vous indiquez sont affichées dans les colonnes appropriées de la liste de schémas, pour les schémas sélectionnés.
