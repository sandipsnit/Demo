Configurer les sources de données multiples RAC

Utilisez cet écran pour configurer des sources de données en tant que sources de données Oracle RAC.

Les sources de données que vous avez choisi de configurer en tant que sources de données oracle RAC sur l'écran Configurer les sources de données JDBC sont répertoriées dans la partie inférieure de l'écran.

Pour sélectionner les sources de données pour lesquelles indiquer des paramètres, cochez la case située en regard de leur nom.


Remarque :

Si vous sélectionnez plusieurs sources de données, le texte Varie selon les sources de données peut apparaître dans certains champs, ce qui indique que les valeurs en cours de ces champs sont différentes dans les sources de données sélectionnées. Si vous modifiez les valeurs de ces champs, les nouvelles valeurs sont appliquées de façon uniforme dans les sources de données sélectionnées.

  • Pilote : sélectionnez le pilote JDBC à utiliser pour la connexion à la base de données.

  • Nom de service : saisissez un nom de service de base de données Oracle RAC.

  • Nom utilisateur : saisissez le nom utilisateur à employer pour la connexion à la base de données.

  • Mot de passe : entrez le mot de passe correspondant au nom utilisateur indiqué.

  • Nom d'hôte : saisissez le nom du serveur hébergeant les instances de base de données Oracle RAC.

  • Nom d'instance : entrez le nom de chaque instance de base de données Oracle.

  • Port : entrez les numéros de port à utiliser pour la connexion au serveur qui héberge la base de données.

Pour ajouter une nouvelle instance de base de données, cliquez sur Ajouter, puis indiquez le nom d'hôte, le nom de l'instance et le numéro de port.

Une fois la configuration des sources de données multiples RAC terminée, cliquez sur Suivant.
