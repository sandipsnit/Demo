Récapitulatif de la configuration

Avant de continuer, vérifiez les paramètres de configuration détaillés de votre cellule.

Dans le panneau Récapitulatif, sélectionnez un élément pour lequel afficher des détails dans le panneau Détails à droite.

Vous pouvez limiter le nombre d'éléments affichés dans le panneau Récapitulatif en sélectionnant une option de filtre dans la liste déroulante Vue récapitulative. Par exemple, pour visualiser les sources de données JDBC affectées à chacun des clusters et serveurs, sélectionnez JDBC dans la liste déroulante Vue récapitulative.

Si vous devez modifier la configuration, cliquez sur Précédent pour revenir à l'écran approprié.

Une fois la configuration vérifiée, cliquez sur :

  • Créer (si vous créez une cellule)

  • Etendre (si vous modifiez une cellule existante)

  • (si vous fédérez un ordinateur à une cellule existante)
