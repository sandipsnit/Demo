Cellule - Progression

Cet écran affiche le statut du processus de création ou de chargement d'une cellule, ou de fédération de l'ordinateur. Cet exemple présente l'écran Création d'une cellule. Le titre de l'écran varie en fonction de la tâche en cours :

  • Lors de la création d'une cellule, le titre de l'écran est Création d'une cellule.

  • Lors de la mise à jour d'une cellule existante, le titre de l'écran est Chargement des informations sur la cellule.

  • Lors de la fédération d'un ordinateur distant à une cellule, le titre de l'écran est Fédération de l'ordinateur.

Lorsque la barre de progression atteint 100 % et que le message "Lecture des informations sur la cellule effectuée. Cliquez sur Suivant" apparaît dans la fenêtre inférieure, cliquez sur Suivant pour continuer.
