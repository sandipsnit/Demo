Sélectionner des options de configuration

Sélectionnez l'une des options suivantes. L'option affichée dépend de la configuration en cours.

  • Créer et configurer une cellule : sélectionnez cette option pour créer une cellule WebSphere. L'écran Indiquer le nom de la cellule, du profil et du noeud apparaît ensuite.

  • Fédérer l'ordinateur et configurer la cellule : sélectionnez cette option pour ajouter le noeud WebSphere en cours à une cellule Fusion Middleware existante. L'écran Indiquer le nom du profil et du noeud apparaît ensuite. Dans ce cas, vous devez exécuter l'assistant Configuration sur le noeud que vous souhaitez fédérer à une cellule précédemment créée à partir du noeud du gestionnaire de déploiement.

  • Sélectionner et configurer une cellule existante : cette option est uniquement disponible si vous avez déjà créé une cellule. Pour modifier la configuration d'une cellule existante, sélectionnez cette option. Dans la zone de liste, cliquez sur la cellule à configurer (la cellule sélectionnée sera mise en évidence). L'écran Chargement des informations sur la cellule apparaît ensuite. Pour plus d'informations sur cet écran, reportez-vous à Cellule - Progression.
