Sélectionner une configuration facultative

Sélectionnez toutes les catégories (le cas échéant) pour lesquelles vous souhaitez effectuer une configuration avancée. Pour chaque catégorie sélectionnée, l'écran de configuration approprié est affiché, afin que vous puissiez procéder à cette configuration. Si vous ne sélectionnez aucun élément sur cet écran, l'écran Récapitulatif de la configuration apparaît ensuite.


Remarques :

Les catégories répertoriées sur cet écran dépendent des ressources définies dans les modèles que vous avez sélectionnés pour la cellule.

  • Serveurs d'applications et clusters : sélectionnez cette option si vous souhaitez exécuter l'une des tâches suivantes :

    • Ajouter, supprimer ou cloner un serveur d'applications

    • Modifier le nom d'un serveur d'applications

    • Activer/désactiver le mode développement pour un serveur d'applications

    • Ajouter des clusters à une cellule et ajouter des serveurs à un cluster

    • Configurer des adresses (ports) pour le serveur d'administration ou n'importe quel serveur d'applications

    • Modifier les emplacements de répertoire partagé de stockage des fichiers pour les clusters, notamment le répertoire de journalisation et les répertoires de stockage permanent et temporaire.

  • Déploiements et services : sélectionnez cette option si vous souhaitez effectuer l'une des tâches suivantes :

    • Modifier les cibles de déploiement pour les clusters ou les serveurs

    • Modifier les cibles de service pour les clusters ou les serveurs

    • Modifier les cibles de service personnalisé pour les serveurs

    • Modifier les cibles d'adaptateur de ressources pour les serveurs

  • JDBC : sélectionnez cette option si vous souhaitez modifier les cibles de sources de données JDBC pour les clusters ou les serveurs.

  • JMS : sélectionnez cette option si vous souhaitez modifier les cibles de ressources JMS pour les clusters ou les serveurs.

Une fois les options de configuration appropriées sélectionnées, cliquez sur Suivant pour continuer. L'écran affiché varie en fonction des options que vous avez sélectionnées.
