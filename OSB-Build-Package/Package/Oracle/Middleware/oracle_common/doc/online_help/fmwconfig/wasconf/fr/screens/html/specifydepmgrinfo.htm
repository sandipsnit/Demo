Indiquez les informations concernant le gestionnaire de déploiement

Utilisez cet écran pour indiquer le nom d'hôte, le port SOAP (processus de fédération uniquement), ainsi que le nom utilisateur et mot de passe administrateur pour le gestionnaire de déploiement. Tous les champs sont obligatoires.

  • Nom d'hôte du gestionnaire de déploiement : lors de la création d'une cellule, ce champ utilise par défaut le nom d'hôte de l'ordinateur en cours.

    Lors de la fédération d'un ordinateur, saisissez le nom d'hôte du gestionnaire de déploiement qui gèrera le noeud distant.

  • Port SOAP : ce champ apparaît uniquement lorsque vous fédérez un ordinateur. Saisissez le port SOAP à utiliser pour la connexion au noeud du gestionnaire de déploiement. Il s'agit du port SOAP du gestionnaire de déploiement auquel vous souhaitez fédérer cet ordinateur. La valeur par défaut est généralement 8880. Si vous ne connaissez pas le port SOAP du gestionnaire de déploiement, ouvrez le fichier suivant sur l'hôte de ce gestionnaire :

    WAS_HOME/profiles/DepMgr_name/logs/AboutThisProfile.txt

    où WAS_HOME correspond au répertoire d'origine Oracle Home WebSphere et DepMgr_name désigne le nom du profil de gestionnaire de déploiement approprié sur l'hôte indiqué.

  • Nom utilisateur de l'administrateur : lors de la création d'une cellule, saisissez le nom utilisateur de l'administrateur à utiliser pour la connexion au gestionnaire de déploiement.

    Lors de la fédération d'un ordinateur, saisissez le nom utilisateur administrateur configuré pour le gestionnaire de déploiement sur l'hôte indiqué.

  • Mot de passe : lors de la création d'une cellule, saisissez le mot de passe administrateur à utiliser pour la connexion au gestionnaire de déploiement.

    Lors de la fédération d'un ordinateur, saisissez le mot de passe administrateur configuré pour le gestionnaire de déploiement sur l'hôte indiqué.

  • Confirmer le mot de passe : ce champ apparaît uniquement lors de la création d'une cellule. Ressaisissez le mot de passe administrateur pour le confirmer. Cette entrée doit être identique à celle du champ Mot de passe.

Une fois chaque champ rempli, cliquez sur Suivant pour continuer.
