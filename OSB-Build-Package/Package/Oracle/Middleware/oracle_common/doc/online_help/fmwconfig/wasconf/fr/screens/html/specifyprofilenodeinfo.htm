Indiquer le nom du profil et du noeud

  • Nom du profil du serveur d'applications : saisissez un nom de profil de serveur d'applications unique pour ce noeud. Il s'agit du nom du répertoire dans lequel le profil du serveur d'applications est créé, sous WAS_HOME/profiles sur les deux ordinateurs sur lesquels vous créez le noeud et sur l'hôte du gestionnaire de déploiement.

    Le nom par défaut est Customn, où n correspond au nombre à deux chiffres suivant dans la séquence sur le noeud du gestionnaire de déploiement que vous avez indiqué sur l'écran précédent. Par exemple, si le profil du serveur d'applications Custom01 existe déjà dans la cellule, le nom par défaut de ce champ est Custom02.

    Si vous modifiez le nom de profil par défaut, assurez-vous qu'il est unique dans la cellule.

  • Nom du noeud du serveur d'applications : saisissez le nom de noeud à utiliser pour ce serveur d'applications.

    Le nom par défaut est machine_nameNoden, où machine_name correspond au nom de l'ordinateur en cours et n désigne le nombre à deux chiffres suivant dans la séquence sur cet ordinateur. Par exemple, si le nom de l'ordinateur est machine2 et que Node01 existe déjà sur l'ordinateur, le nom de noeud par défaut est machine2Node02.

    Si vous modifiez le nom de noeud par défaut, assurez-vous qu'il est unique dans la cellule.

Une fois chaque champ rempli, cliquez sur Suivant pour continuer.
