Cibler les adaptateurs de ressources sur les serveurs

Utilisez cet écran pour cibler les adaptateurs de ressources sur des serveurs individuels. Vous ne pouvez pas cibler les adaptateurs de ressources sur des clusters. Les serveurs affectés aux clusters sont inclus dans la zone de liste Target.

Les adaptateurs de ressources associés au produit pour lequel vous configurez la cellule (par exemple, SOA) sont ciblés automatiquement sur le serveur créé pour ce produit. Sur cet écran, vous pouvez cibler les adaptateurs de ressources sur des serveurs supplémentaires.

Pour cibler les adaptateurs de ressources sur des serveurs, procédez comme suit :

  1. Dans la zone de liste Target, sélectionnez le serveur sur lequel vous souhaitez cibler les adaptateurs de ressources.

    Le nom de la cible sélectionnée est affiché en tant que titre de la zone de liste à droite.

  2. Dans la zone de liste target_name, cochez les cases correspondant aux adaptateurs de ressources que vous souhaitez cibler sur le serveur sélectionné.

    Les adaptateurs de ressources affichés ici varient selon les produits sélectionnés précédemment sur l'écran Ajouter des produits à la cellule de l'assistant.

    Une fois que vous avez sélectionné les adaptateurs de ressources, les noms des serveurs ciblés apparaissent dans la colonne Target de la zone de liste target_name.

  3. Répétez les étapes 1 et 2 pour les autres serveurs requis.

  4. Une fois les sélections obligatoires créées, cliquez sur Suivant.

Lorsque vous reconfigurez une cellule existante, si vous supprimez un serveur sur lequel des adaptateurs de ressources sont actuellement ciblés, l'assistant Configuration cible de nouveau automatiquement ces adaptateurs de ressources comme suit :

  • Si les adaptateurs de ressources n'ont été initialement ciblés que sur le serveur que vous supprimez (au terme de la suppression, ils ne sont pas ciblés dans la cellule modifiée), l'assistant Configuration cible de nouveau automatiquement ces adaptateurs de ressources sur toutes les cibles admissibles.

    Une cible admissible est un serveur non défini dans la spécification de groupes de configuration (fichier config-groups.xml) d'un modèle de produit inclus. Les serveurs indiqués dans le fichier config-groups.xml appartiennent essentiellement au modèle et ne peuvent donc pas faire l'objet d'un ciblage automatique.

  • Si les adaptateurs de ressources ont été initialement ciblés sur plusieurs cibles (y compris les serveurs et le serveur d'administration) et que l'un des serveurs ciblés est supprimé, l'assistant Configuration laisse les autres associations de cibles intactes et ne tente pas de cibler de nouveau les adaptateurs de ressources.
