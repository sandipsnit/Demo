Tester le schéma de composant

Utilisez cet écran pour tester les configurations indiquées pour les schémas de composant sur l'écran précédent.

Cochez les cases situées en regard du nom des schémas à tester, puis cliquez sur Tester les connexions.

L'assistant teste la configuration pour chaque schéma, en tentant de se connecter à une URL constituée du pilote, de l'hôte, du port et d'autres informations indiquées lors de la configuration du schéma.

Le résultat du test est indiqué dans la colonne Statut. Des détails sont affichés dans la section Journal des résultats de connexion.

Une fois les tests de schémas effectués, cliquez sur Suivant pour continuer.
