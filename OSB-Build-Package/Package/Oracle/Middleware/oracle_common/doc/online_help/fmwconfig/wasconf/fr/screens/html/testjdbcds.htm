Tester les sources de données JDBC

Utilisez cet écran pour tester les connexions de source de données que vous avez configurées sur les écrans Configurer les sources de données JDBC et Configurer les sources de données multiples RAC.


Remarques :

Pour tester les connexions de base de données, la base de données à laquelle vous vous connectez doit être en cours d'exécution.

Si vous ne souhaitez pas tester les connexions pour l'instant, ne sélectionnez aucune source de données. Cliquez sur Suivant pour continuer, puis sur OK dans la boîte de dialogue d'avertissement qui apparaît.

Cochez les cases associées à chaque source de données à tester, puis cliquez sur Tester les connexions.

L'assistant teste la configuration pour chaque source de données sélectionnée, en tentant de se connecter à une URL constituée du pilote, de l'hôte, du port et d'autres informations indiquées lors de la configuration de la source de données.

Le résultat du test est indiqué dans la colonne Statut. Des détails sont affichés dans la section Journal des résultats de connexion.

Une fois les tests effectués, cliquez sur Suivant pour continuer.
