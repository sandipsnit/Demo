Aggiungi prodotti alla cella

Scegliere prodotti da includere nella cella. Per impostazione predefinita, l'opzione Oracle CIE Core è selezionata e non può essere annullata. Questa opzione consente di aggiungere al dominio il servizio personalizzato obbligatorio CIE Online al dominio e di indicare come destinazione OracleAdminServer.

Se si sta modificando una cella esistente o si sta rendendo federato un computer a una cella, i prodotti già inclusi nella cella sono già selezionati e non possono essere annullati.


Nota:

In genere, quando si crea una nuova cella nel nodo Deployment Manager, si seleziona solo Oracle CIE Core. In una configurazione che include più nodi, selezionare prodotti aggiuntivi quando si esegue la configurazione guidata su computer che si stanno rendendo federati alla cella sul nodo Deployment Manager. Sebbene sia possibile indirizzare i componenti Fusion Middleware al server di amministrazione o agli Application Servers sul nodo Deployment Manager. è consigliabile indirizzarli agli Application Server sui nodi remoti.

Selezionare le caselle corrispondenti agli altri prodotti da includere nella cella. Se si seleziona un prodotto che dipende da risorse fornite da un altro prodotto o da altri prodotti nella lista, tali prodotti sono selezionati automaticamente. Analogamente, se si annulla la selezione di una casella relativa a un prodotto dal quale dipende un altro prodotto, viene annullata la selezione della casella relativa a tale prodotto.

La lista di prodotti (modelli di configurazione) è diversa a seconda dell'installazione del prodotto Fusion Middleware che si sta configurando.

Se si aggiungono prodotti a una esistente, tutti i prodotti già aggiunti alla cella sono già selezionati e non possono essere annullati.

Dopo avere selezionati tutti i prodotti da includere nella cella, fare clic su Successivo per continuare.


Note:

quando si fa clic su Successivo, se uno qualsiasi dei prodotti selezionati richiede delle risorse fornite da un altro prodotto non selezionato, viene visualizzata una finestra di dialogo di avvertenza. Tornare alla schermata Aggiungi prodotti alla cella per selezionare il prodotto dipendente prima di procedere.
