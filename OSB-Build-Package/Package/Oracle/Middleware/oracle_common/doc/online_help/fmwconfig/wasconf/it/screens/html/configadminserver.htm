Configura server di amministrazione

Specificare le impostazioni di configurazione riportate di seguito.

  • Nome - il nome del Server di amministrazione. Per impostazione predefinita è OracleAdminSever e non può essere modificato.

  • Nome nodo: l'elenco a discesa contiene tutti i nodi disponibili. Selezionare il nodo sul quale risiede il Server di amministrazione.

  • Modalità di sviluppo: selezionare questa casella se si desidera eseguire i Server di amministrazione in modalità di sviluppo. Lasciare la casella non selezionata per eseguire il Server di amministrazione in modalità di produzione. Per le informazioni relative alle differenze fra le due modalità, fare riferimento alla documentazione di WebSphere.

Al termine, fare clic su Successivo per continuare.
