Configura membri cluster aggiuntivi

Utilizzare questa schermata per aggiungere Application Server aggiuntivi al cluster. Quando si aggiunge un nuovo Application Server a un cluster, i suoi attributi vengono duplicati dal membro primario del cluster.

  • Nome: il nome dell'Application Server. Il nome predefinito per un nuovo server è new_ClusterMember_n, dove n inizia da 1 e aumenta per ogni nuovo server aggiunto. Per modificare il nome del server, fare clic nel campo Nome e digitare il nuovo nome.

  • Nome nodo: un elenco a discesa di tutti i nodi disponibili. Fare clic sulla lista per selezionare il nodo sul quale risiede l'Application Server. Se non si sono ancora federati altri computer alla cella corrente, nella lista sarà presente un solo nodo.

  • Nome cluster: un elenco a discesa di tutti i cluster disponibili. Selezionare il cluster a cui assegnare il server.

  • Modalità di sviluppo : selezionare questa casella se si desidera eseguire l'Application Server in modalità di sviluppo. Lasciare la casella non selezionata per eseguire l'Application Server in modalità di produzione. Per le informazioni relative alle differenze fra le due modalità in ambiente WebSphere, fare riferimento alla documentazione di WebSphere.

Per aggiungere un membro cluster aggiuntivo:

  1. Fare clic su Aggiungi. Viene aggiunta una nuova riga alla tabella.

  2. Immettere il nome del server e selezionare il nodo, il cluster e l'impostazione della modalità di sviluppo appropriati.

  3. Ripetere i passi 1 e 2 per ciascun membro cluster da aggiungere.

Per eliminare un membro cluster, selezionarlo e fare clic su Elimina. È possibile eliminare un solo membro cluster alla volta.

Per passare dal formato di visualizzazione tabulare predefinito al formato a schede (con una scheda per ogni cluster), fare clic su Cambia visualizzazione. Fare di nuovo clic su Cambia visualizzazione per tornare al formato tabulare.

Al termine, fare clic su Successivo per continuare.


Nota:

Tutte le modifiche apportate in questa schermata non sono salvate nella configurazione della cella fino a quando non si fa clic su Crea nell'ultima schermata della configurazione guidata. Se si esce dalla procedura guidata o si fa clic su Precedente per ritornare a una schermata precedente, è necessario apportare di nuovo le modifiche.
