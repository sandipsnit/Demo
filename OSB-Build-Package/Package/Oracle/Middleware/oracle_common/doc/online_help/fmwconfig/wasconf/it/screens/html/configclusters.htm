Configura cluster

Un cluster è un gruppo di Application Server che vengono eseguiti simultaneamente per fornire scalabilità e alta disponibilità per le applicazioni. La creazione dei cluster consente di raggruppare i server in modo che possano funzionare come una singola unità per ospitare applicazioni e risorse.

Utilizzare questa schermata per aggiungere un cluster e selezionare l'Application Server che sarà il membro cluster primario. Tutti i membri cluster aggiuntivi sono duplicati dal membro cluster primario. Una volta creato un cluster, è possibile aggiungere altri Application Server al cluster nella schermata successiva.

È possibile utilizzare questa schermata anche per eliminare un cluster o modificare le impostazioni di un cluster esistente.

Per creare un cluster, effettuare le operazioni riportate di seguito.

  1. Fare clic su Aggiungi. Viene aggiunta una nuova riga alla tabella. Il nome predefinito per un nuovo cluster è new_Cluster_n, dove n inizia da 1 e aumenta per ogni nuovo cluster aggiunto. Per modificare il valore predefinito, digitare il nome desiderato nella colonna Nome. Il nome del cluster deve essere univoco fra tutti i nomi dei componenti nella cella.

  2. Selezionare un Application Server esistente come membro primario del cluster selezionando il server nell'elenco a discesa Primo membro cluster. La lista contiene solo gli Application Server non ancora assegnati a un cluster.

  3. Ripetere i passi 1 e 2 per ciascun cluster aggiuntivo da creare.

  4. Al termine, fare clic su Successivo. È possibile aggiungere altri server al cluster nella schermata Configura membri cluster aggiuntivi, visualizzata facendo clic su Successivo.


    Nota:

    Tutte le modifiche apportate in questa schermata non sono salvate nella configurazione della cella fino a quando non si fa clic su Crea nell'ultima schermata della configurazione guidata. Se si esce dalla procedura guidata o si fa clic su Precedente per ritornare a una schermata precedente, è necessario apportare di nuovo le modifiche.

Per eliminare un cluster, selezionarlo e fare clic su Elimina. Quando si elimina un cluster, non viene eliminato l'Application Server che è membro cluster primario ma viene solo rimosso dal cluster. È possibile eliminare un solo cluster alla volta.

Per modificare le impostazioni per un cluster, selezionarlo e apportare le modifiche desiderate.

Per passare dal formato di visualizzazione tabulare predefinito al formato a schede (con una scheda per ogni cluster), fare clic su Cambia visualizzazione. Fare di nuovo clic su Cambia visualizzazione per tornare al formato tabulare.

Una volta completata la configurazione dei cluster, fare clic su Successivo per continuare.

Per ulteriori informazioni sui cluster in ambiente WebSphere, fare riferimento alla documentazione di WebSphere.
