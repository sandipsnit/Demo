Configura origini dati multiple RAC

Utilizzare questa schermata per configurare le origini dati come origini dati Oracle RAC.

Le origini dati che si è scelto di configurare come origini dati RAC nella schermata Configura origini dati JDBC) sono elencate nella parte inferiore della schermata.

Scegliere le origini dati per cui si desidera specificare le impostazioni selezionando la casella di controllo accanto al nome dell'origine dati.


Nota:

quando si selezionano più origini dati, in alcuni campi potrebbe essere visualizzato il testo Varia a seconda delle origini dati, indicante che i valori correnti di questi campi non sono gli stessi per tutte le origini dati selezionate. Se si prosegue e si modificano i valori in tali campi, i nuovi valori vengono applicati in modo uniforme alle origini dati selezionate.

  • Driver : selezionare il driver JDBC da utilizzare per connettersi al database.

  • Nome servizio : immettere un nome di servizio per il database Oracle RAC.

  • Nome utente : immettere il nome utente per la connessione al database.

  • Password : immettere la password per il nome utente specificato.

  • Nome host : immettere il nome del server su cui si trovano le istanze di database Oracle RAC.

  • Nome istanza : immettere il nome di ogni istanza di database Oracle.

  • Numero di porta : immettere il numero di porta da usare per la connessione al server su cui si trova il database.

Per aggiungere un'istanza di database, fare clic su Aggiungi e quindi specificare il nome host, il nome dell'istanza e il numero di porta.

Al termine, fare clic su Successivo.
