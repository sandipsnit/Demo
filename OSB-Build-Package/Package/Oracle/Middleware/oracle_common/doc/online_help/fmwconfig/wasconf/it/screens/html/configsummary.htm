Riepilogo della configurazione

Prima di continuare, rivedere le impostazioni di configurazione dettagliate della cella.

Nel riquadro Riepilogo, selezionare una voce per visualizzarne i dettagli nel riquadro Dettagli sulla destra.

È possibile limitare il numero di voci visualizzate nel riquadro Riepilogo selezionando un'opzione di filtro dall'elenco a discesa Vista di riepilogo. Ad esempio, per visualizzare le origini dati JDBC assegnate a ogni cluster e server, selezionare JDBC dall'elenco a discesa Vista di riepilogo.

Se è necessario modificare la configurazione, fare clic su Precedente per tornare alla schermata appropriata.

Al termine della revisione della configurazione, fare clic su:

  • Crea (se si sta creando una nuova cella)

  • Estendi (se si sta modificando una cella esistente)

  • (in caso di federazione di un computer a una cella esistente)
