Avanzamento del processo cella

In questa schermata viene visualizzato lo stato del processo di creazione della cella, di caricamento della cella o di creazione di un computer federato. L'esempio illustra la schermata Creazione della cella. Il titolo della schermata varia a seconda del task corrente, come indicato di seguito.

  • Quando si crea una nuova cella, il titolo della schermata è Creazione della cella.

  • Quando si aggiorna una cella esistente, il titolo della schermata è Caricamento delle informazioni della cella.

  • Quando si federa un computer remoto in una cella, il titolo della schermata è Creazione di un computer federato.

Quando la barra di avanzamento raggiunge il 100% e viene visualizzato il messaggio "Lettura delle informazioni della cella riuscita. Fare clic su Successivo" nella finestra inferiore, fare clic su Successivo per continuare.
