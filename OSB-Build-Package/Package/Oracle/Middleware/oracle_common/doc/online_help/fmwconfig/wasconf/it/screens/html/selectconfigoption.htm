Seleziona opzione di configurazione

Selezionare una delle opzioni riportate di seguito. Le opzioni disponibili dipendono dalla configurazione corrente.

  • Crea e configura cella: selezionare questa opzione per creare una nuova cella WebSphere. Viene visualizzata la schermata Specificare le informazioni per cella, profilo e nome nodo.

  • Rendi federato il computer e configura cella: selezionare questa opzione per aggiungere il nodo WebSphere corrente a una cella Fusion Middleware esistente. Viene visualizzata la schermata Specificare le informazioni per profilo e nome nodo. In questo caso è necessario eseguire la Configurazione guidata sul nodo da federare in una cella creata in precedenza dal nodo Deployment Manager.

  • Selezionare e configurare una cella esistente: questa opzione è disponibile solo quando è già stata creata una cella. Selezionarla se è necessario modificare la configurazione di una cella esistente. Fare clic sulla cella che si desidera configurare nella casella di riepilogo: la cella selezionata verrà evidenziata. Viene visualizzata la schermata Caricamento delle informazioni della cella. Per informazioni su questa schermata, vedere Avanzamento del processo cella.
