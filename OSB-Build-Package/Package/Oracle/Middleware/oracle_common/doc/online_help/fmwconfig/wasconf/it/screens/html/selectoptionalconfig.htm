Selezione configurazione facoltativa

Selezionare tutte le categorie se presenti per le quali si desidera eseguire la configurazione avanzata. Per ogni categoria selezionata viene visualizzata la schermata di configurazione appropriata che consente di eseguire la configurazione avanzata. Se non si seleziona alcun elemento in questa schermata viene visualizzata successivamente la schermata Riepilogo della configurazione.


Note:

le categorie elencate in questa schermata dipendono dalle risorse definite nei modelli selezionati per la cella.

  • Application Server e cluster: selezionare questa opzione se si desidera eseguire uno dei task riportati di seguito.

    • Aggiungere, eliminare o duplicare un Application Server

    • Modificare il nome di un Application Server

    • Attivare e disattivare la modalità di sviluppo per un Application Server

    • Aggiungere cluster alla cella e server a un cluster

    • Configurare gli endpoint (porte) per il server di amministrazione o un Application Server qualsiasi

    • Modificare le posizioni delle directory condivise dell'area di memorizzazione file per i cluster, comprese la directory di log, la directory dell'area di memorizzazione permanente e la directory dell'area di memorizzazione temporanea

  • Distribuzioni e servizi: selezionare questa opzione se si desidera eseguire uno dei task riportati di seguito.

    • Modificare le destinazioni di distribuzione per i cluster o i server

    • Modificare le destinazioni dei servizi per i cluster o i server

    • Modificare le destinazioni dei servizi personalizzati per i server

    • Modificare le destinazioni di Resource Adapter per i server

  • JDBC: selezionare questa opzione se si desidera modificare le destinazioni delle origini dati JDBC per i cluster o i server.

  • JMS: selezionare questa opzione se si desidera modificare le destinazioni delle risorse JMS per i cluster o i server.

Dopo aver selezionato le opzioni di configurazione appropriate, fare clic su Successivo per continuare. La schermata che viene visualizzata varia a seconda delle opzioni selezionate.
