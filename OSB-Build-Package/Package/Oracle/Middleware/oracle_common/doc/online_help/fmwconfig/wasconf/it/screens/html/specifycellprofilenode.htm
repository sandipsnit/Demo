Specificare le informazioni per cella, profilo e nome nodo

Immettere i valori appropriati nei campi riportati di seguito. I campi sono tutti obbligatori. Usare solo caratteri alfanumerici in ciascun campo.

  • Nome cella: immettere un nome univoco per la nuova cella. Si tratta del nome della directory in cui viene creata la cella, sotto le due directory seguenti:

    • WAS_HOME/profiles/DepMgrProfile/config/cells, dove WAS_HOME è la directory di installazione WebSphere e DepMgrProfile è il nome del profilo Deployment Manager.

    • WAS_HOME/profiles/AppSrvrProfile/config/cells, dove WAS_HOME è la directory di installazione WebSphere e AppSrvrProfile è il nome del profili Application Server.

    Il nome predefinito è nome_computerCelln, dove nome_computer è il nome del computer corrente e n è il numero a due cifre successivo nella sequenza del computer. Se ad esempio il nome del computer è computer1 e Cell01 esiste già, il nome cella predefinito sarà computer1Cell02.

  • Nome profilo Deployment Manager: immettere un nome profilo Deployment Manager univoco per la cella. Si tratta del nome della directory di creazione del profilo Deployment Manager, sotto WAS_HOME/profiles.

    Il nome predefinito è Dmgrn, dove n è il numero a due cifre successivo nella sequenza. Se ad esempio il profilo Deployment Manager denominato Dmgr01 esiste già, il valore predefinito per questo campo sarà Dmgr02.

  • Nome nodo Deployment Manager: immettere un nome nodo Deployment Manager univoco per questa cella.

    Il nome predefinito è nome_computerCellManagern, dove nome_computer è il nome del computer corrente e n è il numero a due cifre successivo nella sequenza. Se ad esempio il nome del computer è computer1 e CellManager01 esiste già, il nome nodo predefinito sarà computer1CellManager02.

  • Nome profilo Application Server: immettere un nome profilo Application Server univoco per questa cella. Si tratta del nome della directory di creazione del profilo Application Server, sotto WAS_HOME/profiles.

    Il nome predefinito è Customn, dove n è il numero a due cifre successivo nella sequenza. Se ad esempio il profilo Application Server Custom01 esiste già, il nome predefinito per questo campo sarà Custom02.

  • Nome nodo Application Server: immettere un nome nodo Application Server univoco per questa cella.

    Il nome predefinito è nome_computerNoden, dove nome_computer è il nome del computer corrente e n è il numero a due cifre successivo nella sequenza. Se ad esempio il nome del computer è computer1 e Node01 esiste già, il nome nodo predefinito sarà computer1Node02.

Dopo aver fornito le necessarie informazioni in ogni campo, fare clic su Successivo.
