Specificare le informazioni di Deployment Manager

Utilizzare questa schermata per specificare il nome host, la porta SOAP (solo processo federato) e il nome utente e la password dell'amministratore di Deployment Manager. Tutti i campi sono obbligatori.

  • Nome host Deployment Manager: durante la creazione di una nuova cella, per impostazione predefinita questo campo viene impostato sul nome host del computer corrente.

    Durante la creazione di un computer federato, immettere il nome host di Deployment Manager che gestirà il nodo remoto.

  • Porta SOAP: questo campo viene visualizzato solo durante la creazione di un computer federato. Immettere la porta SOAP da utilizzare per la connessione al nodo di Deployment Manager. Si tratta della porta SOAP di Deployment Manager in cui si desidera creare il computer federato corrente. In genere, il valore predefinito è 8880. Se non si conosce il valore della porta SOAP per Deployment Manager, aprire il seguente file nell'host di Deployment Manager:

    WAS_HOME/profiles/DepMgr_name/logs/AboutThisProfile.txt

    dove WAS_HOME rappresenta la directory home WebSphere e DepMgr_name rappresenta il nome del profilo Deployment Manager appropriato sull'host specificato.

  • Nome utente amministratore: durante la creazione di una nuova cella, immettere il nome utente dell'amministratore che si desidera utilizzare per eseguire il login a Deployment Manager.

    Durante la creazione di un computer federato, immettere il nome utente dell'amministratore configurato per Deployment Manager sull'host specificato.

  • Password: durante la creazione di una nuova cella, immettere la password dell'amministratore che si desidera utilizzare per eseguire il login a Deployment Manager.

    Durante la creazione di un computer federato, immettere la password configurata per Deployment Manager sull'host specificato.

  • Conferma password: questo campo viene visualizzato solo durante la creazione di una nuova cella. Immettere nuovamente la password dell'amministratore per confermarla. Il valore immesso deve corrispondere esattamente a quello immesso nel campo Password.

Dopo aver immesso i valori desiderati in ogni campo, fare clic su Successivo per continuare.
