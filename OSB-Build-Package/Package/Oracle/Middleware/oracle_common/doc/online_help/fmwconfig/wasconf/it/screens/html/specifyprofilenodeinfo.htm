Specificare le informazioni per profilo e nome nodo

  • Nome profilo Application Server: immettere un nome profilo Application Server univoco per questo nodo. Si tratta del nome della directory di creazione del profilo Application Server, sotto WAS_HOME/profiles, sia nel computer in cui si crea il nodo che nell'host di Deployment Manager.

    Il nome predefinito è Customn, dove n è il numero a due cifre successivo nella sequenza del nodo Deployment Manager specificato nella schermata precedente. Se ad esempio il profilo Application Server Custom01 esiste già nella cella, il nome predefinito per questo campo sarà Custom02.

    Se si modifica il nome profilo predefinito, assicurarsi che il nome fornito sia univoco all'interno della cella.

  • Nome nodo Application Server: immettere il nome nodo da usare per questo Application Server.

    Il nome predefinito è nome_computerNoden, dove nome_computer è il nome del computer corrente e n è il numero a due cifre successivo nella sequenza del computer corrente. Se ad esempio il nome del computer è computer2 e Node01 esiste già nel computer, il nome nodo predefinito sarà computer2Node02.

    Se si modifica il nome nodo predefinito, assicurarsi che il nome fornito sia univoco all'interno della cella.

Dopo aver fornito le necessarie informazioni in ogni campo, fare clic su Successivo per continuare.
