Indirizza risorse JMS a cluster o server

Utilizzare questa schermata per indirizzare le risorse JDBC ai server o ai cluster.

Per indirizzare una risorsa JDBC a un server o un cluster, attenersi alla procedura descritta di seguito.

  1. Nella casella di riepilogo Destinazione selezionare il cluster o il server a cui si desidera indirizzare una risorsa JMS.

    Il nome della destinazione selezionata viene visualizzato come titolo della lista sulla destra.

  2. Nella casella di riepilogo nome_destinazione selezionare le caselle di controllo corrispondenti alle risorse JMS che si desidera indirizzare al server o al cluster selezionato.


    Nota:

    Quando si seleziona un server nella casella di riepilogo Destinazione è possibile che alcune caselle di controllo nell'elenco nome_destinazione siano disabilitate, a indicare le risorse JMS già indirizzate al cluster contenente il server selezionato.

    Dopo aver selezionato le origini dati, il nome del cluster o server di destinazione viene visualizzato nella colonna Destinazione della casella di riepilogo nome_destinazione.

  3. Ripetere i passi 1 e 2 per gli altri cluster e server come richiesto.

  4. Dopo aver effettuato le selezioni necessarie, fare clic su Successivo.
