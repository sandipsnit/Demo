Test dello schema dei componenti

Utilizzare questa schermata per eseguire il test delle configurazioni specificate per gli schemi dei componenti nella schermata precedente.

Selezionare le caselle di controllo accanto ai nomi degli schemi da sottoporre a test, quindi fare clic su Test delle connessioni.

La procedura guidata esegue il test della configurazione per ogni schema mediante un tentativo di connessione a un URL creato utilizzando il driver, l'host, la porta e le altre informazioni specificate durante la configurazione dello schema.

Il risultato del test viene visualizzato nella colonna Stato. I dettagli vengono visualizzati nella sezione Log dei risultati della connessione.

Dopo aver completato i test dello scherma, fare clic su Successivo per continuare.
