Test delle origini dati JDBC

Utilizzare questa schermata per eseguire il test delle connessioni all'origine dati configurate nelle schermate Configura origini dati JDBC e Configura origini dati multiple RAC.


Note:

Per eseguire il test delle connessioni al database, è necessario che il database con cui si sta cercando di stabilire la connessione sia in esecuzione.

Se non si desidera eseguire il test delle connessioni, non selezionare alcuna origine dati. Fare clic su Successivo per continuare oppure su OK nella finestra di dialogo di avvertenza visualizzata.

Selezionare la casella di controllo per ogni origine dati che si desidera sottoporre a test, quindi fare clic su Test delle connessioni.

La procedura guidata esegue il test della configurazione per ogni origine dati mediante un tentativo di connessione a un URL creato utilizzando il driver, l'host, la porta e le altre informazioni specificate durante la configurazione dell'origine dati.

Il risultato del test viene visualizzato nella colonna Stato. I dettagli vengono visualizzati nella sezione Log dei risultati della connessione.

Dopo aver completato i test, fare clic su Successivo per continuare.
