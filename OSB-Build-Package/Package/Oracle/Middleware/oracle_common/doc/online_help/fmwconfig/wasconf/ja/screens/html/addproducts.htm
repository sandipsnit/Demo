セルへの製品追加

このセルに含める製品を選択します。デフォルトではOracle CIE Coreオプションが選択され、選択を解除できません。このオプションは、必要なCIEオンライン・カスタム・サービスをドメインに追加し、OracleAdminServerにターゲット指定します。

既存のセルを編集する場合またはマシンをセルにフェデレートする場合、すでにセルに含まれている製品は選択済であり、選択を解除できません。


注意:

通常、デプロイメント・マネージャ・ノードに新しいセルを作成する場合は、Oracle CIE Coreのみ選択します。複数のノードを含む構成では、デプロイメント・マネージャ・ノードのセルにフェデレートするマシンで構成ウィザードを実行するときに追加の製品を選択します。Fusion Middlewareコンポーネントをデプロイメント・マネージャ・ノードの管理サーバーまたはアプリケーション・サーバーにターゲット指定できますが、リモート・ノードのアプリケーション・サーバーにターゲット指定することをお薦めします（これは正しくない可能性があります。製品をセルに追加する場所に意味があるとは思えません）（メモが必要です。この表現を調べる必要があります。製品を選択しないで、既定のアプリケーション・サーバーがリモート・ノードに作成されるようにすることができます）。

セルに含める他の製品に対応するチェック・ボックスを選択します。リストにある別の1つまたは複数の製品によって提供されているリソースに依存する製品を選択した場合は、それらの製品も自動的に選択されます。同様に、別の製品が依存している製品のチェック・ボックスの選択を解除した場合は、依存製品の選択も解除されます。

製品のリスト（構成テンプレート）は、構成しているFusion Middleware製品のインストール環境によって異なります。ここで示す例は、インストール用です（上の画面を実際の画面で置換する必要があります）。

既存のセルに製品を追加する場合、以前にセルに追加されたすべての製品はすでに選択済であり、選択を解除できません。

セルに含めるすべての製品を選択した後で、「次へ」をクリックして続行します。


注意:

「次へ」をクリックしたときに、選択したいずれかの製品に、選択していない別の製品が提供するリソースが必要な場合は、警告ダイアログ・ボックスが表示されます。次に進む前に、「セルへの製品追加」画面に戻って依存関係の製品を選択します。
