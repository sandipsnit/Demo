構成オプションの選択

次のオプションの1つを選択します。表示されるオプションは、現在の構成によって異なります。

  • セルの作成と構成: このオプションは、新しいWebSphereセルを作成する場合に選択します。「セル、プロファイルおよびノード名の情報を指定してください」画面が次に表示されます。

  • マシンのフェデレートとセルの構成: このオプションは、現在のWebSphereノードを既存のFusion Middlewareセルに追加する場合に選択します。「プロファイルおよびノード名の情報を指定してください」画面が次に表示されます。この場合は、デプロイメント・マネージャ・ノードから以前に作成したセルにフェデレートするノードで構成ウィザードを実行する必要があります。

  • 既存のセルの選択と構成: このオプションは、セルをすでに作成してある場合にのみ使用できます。既存のセルの構成を変更するには、このオプションを選択します。リスト・ボックスで、構成するセルをクリックします（選択されたセルはハイライト表示されます）。「セル情報のロード中」画面が次に表示されます。この画面の詳細は、セルの進行状況を参照してください。
