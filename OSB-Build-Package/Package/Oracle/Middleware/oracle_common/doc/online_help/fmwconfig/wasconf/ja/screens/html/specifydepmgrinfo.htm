デプロイメント・マネージャ情報の指定

この画面では、デプロイメント・マネージャのホスト名、SOAPポート（フェデレーション・プロセスのみ）、管理者のユーザー名とパスワードを指定します。すべてのフィールドが必須です。

  • デプロイメント・マネージャ・ホスト名: 新しいセルを作成する場合、このフィールドはデフォルトで現在のマシンのホスト名になります。

    マシンをフェデレートする場合は、リモート・ノードを管理するデプロイメント・マネージャのホスト名を入力します。

  • SOAPポート: このフィールドは、マシンをフェデレートする場合にのみ表示されます。デプロイメント・マネージャ・ノードへの接続に使用するSOAPポートを入力します。これは、このマシンをフェデレートするデプロイメント・マネージャのSOAPポートです。通常、デフォルトは8880です。デプロイメント・マネージャのSOAPポートが不明な場合は、デプロイメント・マネージャ・ホストで次のファイルを開きます。

    WAS_HOME/profiles/DepMgr_name/logs/AboutThisProfile.txt

    WAS_HOMEはWebSphereホーム・ディレクトリで、DepMgr_nameは指定したホスト上の適切なデプロイメント・マネージャ・プロファイルの名前です。

  • 管理ユーザー名: 新しいセルを作成する場合は、デプロイメント・マネージャへのログインに使用する管理者ユーザー名を入力します。

    マシンをフェデレートする場合は、指定したホスト上のデプロイメント・マネージャに対して構成された管理者ユーザー名を入力します。

  • パスワード: 新しいセルを作成する場合は、デプロイメント・マネージャへのログインに使用する管理者パスワードを入力します。

    マシンをフェデレートする場合は、指定したホスト上のデプロイメント・マネージャに対して構成された管理者パスワードを入力します。

  • パスワードの確認: このフィールドは、新しいセルを作成する場合にのみ表示されます。管理者パスワードを再入力して確認します。このエントリは、「パスワード」フィールドのエントリと完全に一致する必要があります。

各フィールドへの入力が終了したら、「次へ」をクリックして続行します。
