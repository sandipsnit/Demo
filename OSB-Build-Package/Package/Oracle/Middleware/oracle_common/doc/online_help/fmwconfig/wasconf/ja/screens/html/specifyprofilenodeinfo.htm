プロファイルおよびノード名の情報を指定してください

  • アプリケーション・サーバー・プロファイル名: このノードの一意のアプリケーション・サーバー・プロファイル名を入力します。これは、ノードを作成するマシンとデプロイメント・マネージャ・ホストの両方のWAS_HOME/profiles の下の、アプリケーション・サーバー・プロファイルが作成されるディレクトリの名前です。

    デフォルト名はCustomnです。nは、前の画面で指定したデプロイメント・マネージャ・ノード上の次の順序を示す2桁の数字です。たとえば、アプリケーション・サーバー・プロファイルCustom01がセルにすでに存在する場合、このフィールドのデフォルトはCustom02です。

    デフォルトのプロファイル名を変更する場合は、セル内で一意であることを確認します。

  • アプリケーション・サーバー・ノード名: このアプリケーション・サーバーに使用するノード名を入力します。

    デフォルト名はmachine_nameNodenです。machine_nameは現在のマシンの名前で、nは現在のマシン上の次の順序を示す2桁の数字です。たとえば、マシン名がmachine2で、Node01がマシンにすでに存在する場合、デフォルトのノード名はmachine2Node02です。

    デフォルトのノード名を変更する場合は、セル内で一意であることを確認します。

各フィールドへの入力が終了したら、「次へ」をクリックして続行します。
