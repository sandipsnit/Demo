JMSリソースのクラスタまたはサーバーへのターゲット設定

この画面では、JMSリソースをサーバーまたはクラスタにターゲット指定します。

JMSリソースをクラスタまたはサーバーにターゲット指定するには、次の手順を実行します。

  1. 「ターゲット」リスト・ボックスで、JMSリソースをターゲット指定するクラスタまたはサーバーを選択します。

    選択したターゲットの名前は右側のリスト・ボックスのタイトルとして表示されます。

  2. target_nameリスト・ボックスで、選択したサーバーまたはクラスタにターゲット指定するJMSリソースに対応するチェック・ボックスを選択します。


    注意:

    「ターゲット」リスト・ボックスでサーバーを選択したときに、target_nameリスト・ボックスの一部のチェック・ボックスが無効になり、選択されたサーバーを含むクラスタにすでに割り当てられているJMSリソースが示される場合があります。

    データ・ソースを選択すると、ターゲットに指定したクラスタまたはサーバーの名前がtarget_nameリスト・ボックスの「ターゲット」列に表示されます。

  3. 必要に応じて、他のクラスタおよびサーバーで手順1と2を繰り返します。

  4. 必要な選択が完了した後、「次へ」をクリックします。
