셀에 제품 추가

이 셀에 포함시킬 제품을 선택합니다. 기본적으로 [Oracle CIE Core] 옵션이 선택되며 지울 수 없습니다. 이 옵션은 필요한 CIE 온라인 사용자 정의 서비스를 도메인에 추가하고 OracleAdminServer로 대상을 지정합니다.

기존 셀을 편집하거나 시스템을 셀에 통합하는 경우 셀에 이미 포함된 제품은 이미 선택되어 있는 것으로 지울 수 없습니다.


주:

일반적으로 배치 관리자 노드에서 새 셀을 생성하는 경우 Oracle CIE Core만 선택합니다. 여러 노드를 포함하는 구성에서는 배치 관리자 노드의 셀에 통합하는 시스템에서 구성 마법사를 실행할 때 추가 제품을 선택합니다. Fusion Middleware 구성 요소를 배치 관리자 노드의 관리 서버나 애플리케이션 서버로 대상을 지정할 수 있지만 원격 노드의 애플리케이션 서버로 대상을 지정하는 것이 좋습니다. (그렇지 않을 수도 있습니다. 제품을 셀에 추가하는 위치는 중요하지 않다고 생각합니다.) (주에 대한 추가 설명이 필요합니다. 통합된 애플리케이션 서버가 원격 노드에 생성되도록 하려면 제품을 선택하지 마십시오.)

셀에 포함시킬 다른 제품에 해당하는 체크 박스를 선택합니다. 목록의 다른 제품에서 제공하는 리소스에 종속된 제품을 선택할 경우 해당 제품도 자동으로 선택됩니다. 마찬가지로 다른 제품이 종속된 제품에 대한 체크 박스를 지울 경우 종속 제품에 대한 체크 박스도 지워집니다.

제품 목록(구성 템플리트)은 구성하려는 Fusion Middleware 제품 설치에 따라 달라집니다. 여기에 표시된 예는 .... 설치입니다. (위의 화면을 실제 화면으로 바꾸십시오.)

기존 셀에 제품을 추가할 때 이전에 추가된 모든 제품은 미리 선택되는 것으로 지울 수 없습니다.

셀에 포함시킬 제품을 모두 선택한 후 다음을 눌러 계속하십시오.


주:

다음을 누를 때 선택한 제품 중 하나라도 선택되지 않은 다른 제품에서 제공하는 리소스가 필요한 경우 경고 대화상자가 표시됩니다. 계속 진행하려면 셀에 제품 추가 화면으로 돌아가 종속 제품을 선택하십시오.
