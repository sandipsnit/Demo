구성 요약

계속 진행하기 전에 셀의 구성 설정 세부 정보를 검토합니다.

요약 창에서 오른쪽의 세부 정보 창에 항목에 대한 세부 정보를 표시할 항목을 선택합니다.

요약 보기 드롭다운 목록에서 필터 옵션을 선택하여 요약 창에 표시되는 항목을 제한할 수 있습니다. 예를 들어, 각 클러스터 및 서버에 지정된 JDBC 데이터 소스를 보려면 요약 보기 드롭다운 목록에서 JDBC를 선택합니다.

구성을 변경해야 하는 경우 이전을 눌러 해당 화면으로 돌아갑니다.

구성 검토를 마쳤으면 다음을 누릅니다.

  • 생성(새 셀을 생성하는 경우)

  • 확장(기존 셀을 수정하는 경우)

  • (시스템을 기존 셀에 통합하는 경우)
