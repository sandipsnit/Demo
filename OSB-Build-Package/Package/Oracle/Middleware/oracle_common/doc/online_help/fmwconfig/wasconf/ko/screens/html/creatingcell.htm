셀 진행률

이 화면에는 셀 생성, 셀 로드 또는 시스템 통합 프로세스의 상태가 표시됩니다. 이 예는 셀 생성 중 화면을 표시합니다. 화면 제목은 현재 작업에 따라 달라집니다.

  • 새 셀을 생성할 때 화면 제목은 셀 생성 중입니다.

  • 기존 셀을 갱신할 때 화면 제목은 셀 정보 로드 중입니다.

  • 원격 시스템을 셀에 통합할 때 화면 제목은 시스템 통합 중입니다.

진행률 막대가 100%에 도달하고 아래쪽 창에 "셀 정보를 성공적으로 읽었습니다. [다음]을 누르십시오." 메시지가 표시되면 다음을 눌러 계속 진행하십시오.
