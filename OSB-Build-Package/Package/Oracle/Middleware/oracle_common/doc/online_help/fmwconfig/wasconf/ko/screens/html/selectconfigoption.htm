구성 옵션 선택

다음 옵션 중 하나를 선택합니다. 표시되는 옵션은 현재 구성에 따라 다릅니다.

  • 셀 생성 및 구성: 새 WebSphere 셀을 생성하려면 이 옵션을 선택합니다. 셀, 프로파일 및 노드 정보 지정 화면이 다음에 나타납니다.

  • 시스템 통합 및 셀 구성: 현재 WebSphere 노드를 기존 Fusion Middleware 셀에 추가하려면 이 옵션을 선택합니다. 프로파일 및 노드 이름 정보 지정 화면이 다음에 나타납니다. 이 경우 이전에 배치 관리자 노드에서 생성한 셀에 통합할 노드에서 구성 마법사를 실행해야 합니다.

  • 기존 셀 선택 및 구성: 이 옵션은 이미 셀을 생성한 경우에만 사용할 수 있습니다. 기존 셀의 구성을 변경하려면 이 옵션을 선택합니다. 목록 상자에서 구성할 셀을 누릅니다(선택한 셀이 강조 표시됩니다). 셀 정보 로드 중 화면이 다음에 나타납니다. 이 화면에 대한 정보는 셀 진행률을 참조하십시오.
