선택적 구성 선택

고급 구성을 수행할 모든 범주를 선택합니다. 선택하는 각 범주에 대해 고급 구성을 수행할 수 있는 해당 구성 화면이 표시됩니다. 이 화면에서 항목을 선택하지 않으면 [구성 요약] 화면이 다음에 표시됩니다.


주:

이 화면에 나열되는 범주는 셀에 대해 선택한 템플리트에서 정의된 리소스에 따라 다릅니다.

  • 애플리케이션 서버 및 클러스터: 다음 작업을 수행하려는 경우 이 옵션을 선택합니다.

    • 애플리케이션 서버 추가, 삭제 또는 복제

    • 애플리케이션 서버 이름 변경

    • 애플리케이션 서버에 대한 개발 모드 사용 또는 사용 안함

    • 셀에 클러스터 추가 및 클러스터에 서버 추가

    • 관리 서버 또는 애플리케이션 서버에 대한 끝점(포트) 구성

    • 로그 디렉토리, 영구 저장소 디렉토리 및 임시 저장소 디렉토리를 포함하여 클러스터에 대한 파일 저장소 공유 디렉토리 위치 변경

  • 배치 및 서비스: 다음 작업을 수행하려는 경우 이 옵션을 선택합니다.

    • 클러스터 또는 서버에 대한 배치 대상 변경

    • 클러스터 또는 서버에 대한 서비스 대상 변경

    • 서버에 대한 사용자 정의 서비스 대상 변경

    • 서버에 대한 리소스 어댑터 대상 변경

  • JDBC: 클러스터 또는 서버에 대한 JDBC 데이터 소스 대상을 변경하려는 경우 이 옵션을 선택합니다.

  • JMS: 클러스터 또는 서버에 대한 JMS 리소스 대상을 변경하려는 경우 이 옵션을 선택합니다.

알맞은 구성 옵션을 선택한 후 다음을 눌러 계속 진행하십시오. 다음에 표시되는 화면은 선택한 옵션에 따라 달라집니다.
