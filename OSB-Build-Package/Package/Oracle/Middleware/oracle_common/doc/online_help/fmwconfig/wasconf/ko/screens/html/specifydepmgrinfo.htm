배치 관리자 정보 지정

이 화면에서는 배치 관리자에 대한 호스트 이름, SOAP 포트(통합 프로세스만 해당), 관리자 사용자 이름 및 비밀번호를 지정할 수 있습니다. 모든 필드는 필수 필드입니다.

  • 배치 관리자 호스트 이름: 새 셀을 생성할 때 이 필드의 기본값은 현재 시스템의 호스트 이름입니다.

    시스템을 통합하는 경우 원격 노드를 관리할 배치 관리자의 호스트 이름을 입력합니다.

  • SOAP 포트: 이 필드는 시스템을 통합하는 경우에만 표시됩니다. 배치 관리자 노드에 접속하기 위해 사용할 SOAP 포트를 입력합니다. 이 시스템을 통합하려는 배치 관리자의 SOAP 포트입니다. 기본값은 일반적으로 8880입니다. 배치 관리자에 대한 SOAP 포트를 알지 못하는 경우 배치 관리자 호스트에서 다음 파일을 엽니다.

    WAS_HOME/profiles/DepMgr_name/logs/AboutThisProfile.txt

    여기서 WAS_HOME은 WebSphere 홈 디렉토리이고 DepMgr_name은 지정된 호스트에서 해당 배치 관리자 프로파일의 이름입니다.

  • 관리자 사용자 이름: 새 셀을 생성하는 경우 배치 관리자에 로그인하는 데 사용할 관리자 사용자 이름을 입력합니다.

    시스템을 통합하는 경우 지정된 호스트에서 배치 관리자에 대해 구성된 관리자 사용자 이름을 입력합니다.

  • 비밀번호: 새 셀을 생성하는 경우 배치 관리자에 로그인하는 데 사용할 관리자 비밀번호를 입력합니다.

    시스템을 통합하는 경우 지정된 호스트에서 배치 관리자에 대해 구성된 관리자 비밀번호를 입력합니다.

  • 비밀번호 확인: 이 필드는 새 셀을 생성하는 경우에만 표시됩니다. 관리자 비밀번호를 다시 입력하여 확인합니다. 이 입력은 비밀번호 필드의 입력과 정확히 일치해야 합니다.

각 필드에 입력을 마쳤으면 다음을 눌러 계속 진행하십시오.
