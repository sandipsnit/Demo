프로파일 및 노드 이름 정보 지정

  • 애플리케이션 서버 프로파일 이름: 이 노드에 대한 고유한 애플리케이션 서버 프로파일 이름을 입력합니다. 노드를 생성하는 시스템과 배치 관리자 호스트 모두의 WAS_HOME/profiles 아래에 애플리케이션 서버 프로파일이 생성되는 디렉토리의 이름입니다.

    기본 이름은 Customn입니다. 여기서 n은 이전 화면에서 지정한 배치 관리자 노드의 다음 두 자리 숫자입니다. 예를 들어, 애플리케이션 서버 프로파일 Custom01이 셀에 존재하는 경우 이 필드에 대한 기본값은 Custom02입니다.

    기본 프로파일 이름을 변경하는 경우 셀 내에서 고유한지 확인합니다.

  • 애플리케이션 서버 노드 이름: 이 애플리케이션 서버에 대해 사용할 노드 이름을 입력합니다.

    기본 이름은 machine_nameNoden입니다. 여기서 machine_name은 현재 시스템의 이름이고, n은 현재 시스템에서 다음 두 자리 숫자입니다. 예를 들어, 시스템 이름이 machine2이고 Node01이 시스템에 존재하는 경우 기본 노드 이름은 machine2Node02입니다.

    기본 노드 이름을 변경하는 경우 셀 내에서 고유한지 확인합니다.

각 필드에 입력을 마쳤으면 다음을 눌러 계속 진행하십시오.
