클러스터 또는 서버에 JMS 리소스 대상 지정

이 화면에서는 서버나 클러스터에 JMS 리소스를 대상으로 지정할 수 있습니다.

클러스터 또는 서버에 JMS 리소스를 대상으로 지정하려면 다음과 같이 하십시오.

  1. 대상 목록 상자에서 JMS 리소스를 대상으로 지정할 클러스터 또는 서버를 선택합니다.

    선택한 대상의 이름이 오른쪽 목록 상자의 제목으로 표시됩니다.

  2. target_name 목록 상자에서 선택한 서버나 클러스터에 대상으로 지정할 JMS 리소스에 해당하는 체크 박스를 선택합니다.


    주:

    대상 목록 상자에서 서버를 선택하면 target_name 목록 상자의 일부 체크 박스가 사용 안함으로 설정되며, 이는 선택한 서버를 포함하는 클러스터에서 이미 JMS 리소스가 대상으로 지정되었음을 나타냅니다.

    데이터 소스를 선택하면 대상으로 지정된 클러스터나 서버의 이름이 target_name 목록 상자의 대상 열에 표시됩니다.

  3. 필요한 경우 다른 클러스터 및 서버에 대해 1단계와 2단계를 반복합니다.

  4. 필수 항목을 선택한 후 다음을 누릅니다.
