구성 요소 스키마 테스트

이 화면에서는 이전 화면에서 구성 요소 스키마에 대해 지정한 구성을 테스트할 수 있습니다.

테스트할 스키마 이름 옆에 있는 체크 박스를 선택한 후 구성 요소 테스트를 누릅니다.

마법사는 스키마를 구성하는 동안 지정한 드라이버, 호스트, 포트 및 기타 정보를 사용하여 구축한 URL에 대해 접속을 시도하여 각 스키마에 대해 구성을 테스트합니다.

테스트 결과는 상태 열에 표시됩니다. 세부 정보는 접속 결과 로그 섹션에 표시됩니다.

스키마 테스트가 성공적으로 완료되면 다음을 눌러 계속 진행하십시오.
