JDBC 데이터 소스 테스트

이 화면에서는 JDBC 데이터 소스 구성 및 RAC 다중 데이터 소스 구성 화면에서 구성한 데이터 소스 접속을 테스트할 수 있습니다.


주:

데이터베이스 접속을 테스트하려면 접속하려는 데이터베이스가 실행되고 있어야 합니다.

지금 접속 테스트를 원하지 않을 경우 데이터 소스를 선택하지 말고 다음을 눌러 계속 진행하십시오. 그런 다음 나타나는 경고 대화상자에서 [확인]을 누릅니다.

테스트하려는 각 데이터 소스에 대한 체크 박스를 선택한 후 접속 테스트를 누릅니다.

마법사는 데이터 소스를 구성하는 동안 지정한 드라이버, 호스트, 포트 및 기타 정보를 사용하여 구축한 URL에 대해 접속을 시도하여 선택한 각 데이터 소스에 대해 구성을 테스트합니다.

테스트 결과는 상태 열에 표시됩니다. 세부 정보는 접속 결과 로그 섹션에 표시됩니다.

테스트가 성공적으로 완료되면 다음을 눌러 계속 진행하십시오.
