Adicionar Produtos à Célula

Escolha os produtos que você deseja incluir nesta célula. Por default, a opção Núcleo do Oracle CIE está selecionada e não pode ser desmarcada. Essa opção inclui o Serviço Personalizado On-line do CIE necessário ao domínio e o direciona para o OracleAdminServer.

Se você estiver editando uma célula existente ou federando uma máquina para uma célula, os produtos que já foram incluídos na célula serão selecionados e não poderão ser desmarcados.


Observação:

Geralmente, quando você está criando uma célula nova no nó do Gerenciador de Implantação, você seleciona apenas o Núcleo do Oracle CIE. Em uma configuração que inclui vários nós, selecione produtos adicionais ao executar o Assistente de Configuração nas máquinas que você está federando para a célula no nó do Gerenciador de Implantação. Embora você possa direcionar os componentes do Fusion Middleware para o Servidor de Administração ou Servidores de Aplicações no nó do Gerenciador de Implantação, é recomendável direcioná-los para Servidores de Aplicações em seus nós remotos. (this may not be true; I don't think it matters where you add the products to the cell) (note needs work, need to find out how to word this; maybe not select prods so that canned app srvrs are created on the remote nodes).

Marque as caixas de seleção que correspondem aos outros produtos que você deseja incluir na célula. Se quiser selecionar um produto que seja dependente dos recursos fornecidos por outro produto ou produtos na lista, esses produtos também serão selecionados automaticamente. Da mesma forma, se você desmarcar a caixa de seleção de·um produto do qual outro produto seja dependente, a caixa de seleção do produto dependente também será desmarcada.

Se você estiver adicionando produtos a uma célula existente, todos os produtos que foram adicionados anteriormente à célula já estarão selecionados e não poderão ser desmarcados.

Depois de selecionar todos os produtos para incluir na célula, clique em Próximo para continuar.


Observações:

Quando você clicar em Próximo, se qualquer um dos produtos selecionados precisar de recursos fornecidos por outro produto que não foi selecionado, uma caixa de diálogo de advertência será exibida. Retorne à tela Adicionar Produtos à Célula para selecionar o produto de dependência antes de continuar.
