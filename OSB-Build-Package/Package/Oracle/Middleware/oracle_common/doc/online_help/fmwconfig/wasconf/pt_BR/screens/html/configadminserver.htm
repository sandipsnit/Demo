Configurar o Servidor de Administração

Especifique as seguintes definições do Servidor de Administração:

  • Nome — O nome do Servidor de Administração. O default é OracleAdminServer e não pode ser alterado.

  • Nome do nó — Esta lista drop-down contém todos os nós disponíveis. Selecione o nó em que você deseja que o Servidor de Administração resida.

  • Modo de desenvolvimento — Marque esta caixa de seleção se quiser executar o Servidor de Administração no modo de desenvolvimento. Deixe esta caixa de seleção desmarcada para executar o Servidor de Administração no modo de produção. Para obter informações sobre as diferenças entre o modo de desenvolvimento e o modo de produção, consulte a documentação do WebSphere.

Quando concluir, clique em Próximo para continuar.
