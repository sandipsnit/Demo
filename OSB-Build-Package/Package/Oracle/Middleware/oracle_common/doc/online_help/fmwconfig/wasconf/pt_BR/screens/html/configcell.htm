Configurando a Célula

Esta tela mostra o andamento do processo de configuração da célula.

Quando o processo de configuração da célula for concluído, clique em Concluído para fechar o Assistente de Configuração.

Em sistemas Windows, se você quiser iniciar o Gerenciador de Implantação depois de fechar o Assistente de Configuração, marque a caixa de seleção Iniciar Gerenciador de Implantação. Essa caixa de seleção não está disponível em sistemas UNIX.
