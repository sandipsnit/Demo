Configurar Membros do Cluster Adicionais

Use esta tela para incluir Servidores de Aplicações adicionais em um cluster. Quando você adiciona um novo Servidor de Aplicações a um cluster, seus atributos são clonados desde o membro principal do cluster.

  • Nome — O nome do Servidor de Aplicações. O nome default para um novo servidor é new_ClusterMember_n, em que n começa em 1 e aumenta conforme cada novo servidor é adicionado. Para alterar o nome do servidor, clique no campo Nome e digite o novo nome.

  • Nome do nó — Uma lista drop-down de todos os nós disponíveis. Clique na lista para selecionar o nó em que este Servidor de Aplicações reside. Caso você ainda não tenha federado outras máquinas na célula atual, apenas um nó estará disponível nesta lista.

  • Nome do cluster — Uma lista drop-down de todos os clusters disponíveis. Selecione o cluster ao qual você deseja designar este servidor.

  • Modo de desenvolvimento — Marque esta caixa de seleção para executar o Servidor de Aplicações no modo de desenvolvimento. Deixe esta caixa de seleção desmarcada para executar o servidor de aplicações no modo de produção. Para obter informações sobre as diferenças entre o modo de produção e o modo de desenvolvimento em um ambiente do WebSphere, consulte a documentação do WebSphere.

Para adicionar outro membro do cluster:

  1. Clique em Adicionar. Uma nova linha é adicionada à tabela.

  2. Informe o nome do servidor e selecione a definição apropriada do nó, cluster e do modo de desenvolvimento.

  3. Repita as etapas 1 e 2 para cada membro do cluster que você deseja adicionar.

Para deletar um membro do cluster, selecione o membro do cluster e clique em Deletar. Você pode deletar apenas um membro do cluster por vez.

Para trocar a exibição do formato tabular default para um formato com guias (com uma guia para cada cluster), clique em Trocar Exibição. Clique em Trocar Exibição novamente para retornar ao formato tabular.

Quando concluir, clique em Próximo para continuar.


Observação:

As alterações feitas nesta tela não são salvas na configuração da célula até você clicar em Criar na última tela do Assistente de Configuração. Se sair do assistente ou clicar em Anterior para retornar a uma tela anterior, você deverá fazer as alterações novamente.
