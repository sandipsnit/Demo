Configurar Pontos Finais

Use esta tela para configurar os seguintes pontos finais (endereços de listening) em cada um dos servidores, incluindo o Servidor de Administração. Todos os pontos finais devem ser exclusivos na célula. Por default, pontos finais exclusivos são designados a um servidor quando você adiciona o servidor à célula. Você não precisa modificá-los, a menos que queira usar uma porta diferente da default que foi configurada em um determinado servidor.

Para obter informações sobre os diversos pontos finais listados nesta tela, consulte a documentação do WebSphere.

Para alterar pontos finais:

  1. Na lista drop-down Nome do Servidor/Nome do Nó, selecione o servidor cujos pontos finais você deseja alterar. A lista contém todos os servidores na célula, incluindo o Servidor de Administração, os Servidores de Aplicações designados aos clusters e os Servidores de Aplicações independentes que existem em todos os nós na célula.

  2. Digite o novo valor do ponto final na linha apropriada. Se você informar um valor que já esteja em uso, o seguinte erro de validação será exibido; clique em OK e informe outro valor para o ponto final.



    Observações:

    Se você planeja executar vários Gerenciadores de Implantação na mesma máquina, certifique-se de que os pontos finais em cada célula associada sejam exclusivos entre todos os nós do Gerenciador de Implantação na máquina.

    Da mesma forma, se você tiver Servidores de Aplicações em máquinas remotas e esses servidores forem designados a células diferentes no host do Gerenciador de Implantação, você deverá garantir que todos os pontos finais de todos os Servidores de Aplicações em uma determinada máquina sejam exclusivos.

    Você não será advertido sobre pontos finais em conflito que existam em células diferentes.

  3. Repita a etapa 2 para cada ponto final que você deseja alterar.

Você também pode desativar um ponto final em qualquer servidor caso não precise dele. Para desativar um ou mais pontos finais em um servidor, selecione o servidor na lista drop-down Nome do Servidor/Nome do Nó e desmarque a caixa de seleção para cada ponto final que você deseja desativar.

Para usar um host comum para todos os pontos finais em um determinado servidor, selecione o servidor na lista drop-down Nome do Servidor/Nome do Nó e depois marque a caixa de seleção Usar host comum para todos os pontos finais. Quando essa caixa de seleção for selecionada, os nomes de host para cada ponto final no servidor selecionado no momento não poderão ser editáveis, com exceção do nome do host Ponto final do administrador.

Quando terminar, clique em Próximo. Se houver conflitos de ponto final, a caixa de diálogo Erro de Validação será exibida e você deverá resolver o conflito para poder continuar.
