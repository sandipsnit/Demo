Configurar Origens de Dados JDBC

Uma origem de dados JDBC contém um pool de conexões com o banco de dados que são criadas quando a instância da origem de dados é criada, quando é implantada ou direcionada, ou na inicialização do servidor. As aplicações pesquisam uma origem de dados na árvore JNDI e depois solicitam uma conexão. Quando as aplicações não precisam mais das conexões, elas retornam as conexões para o pool de conexões na origem de dados.

Use esta tela para configurar as origens de dados JDBC definidas em sua célula.

As origens de dados JDBC associadas aos produtos para os quais você está criando a célula estão listadas na metade inferior da tela.


Observação:

Para configurar uma ou mais origens de dados como várias origens de dados do Oracle RAC, marque a caixa de seleção adjacente ao nome da origem de dados e marque a caixa de seleção Configurar origens de dados selecionadas como várias origens de dados do RAC no painel a seguir; em seguida, clique no botão Próximo.

Selecione a(s) origem(ns) de dados cuja(s) definições você deseja especificar marcando a caixa de seleção adjacente a cada nome de origem de dados.


Observações:

Ao selecionar várias origens de dados, o texto Varia entre as origens de dados pode ser exibido em alguns campos, indicando que os valores atuais destes campos não são os mesmos nas origens de dados selecionadas. Se você alterar os valores em tais campos, os novos valores serão aplicados uniformemente entre as origens de dados selecionadas.

Os valores default dos parâmetros da origem de dados, como fornecedor, driver, nome do host e número da porta, dependem dos valores especificados nos modelos da aplicação.

  • Fornecedor — Selecione o fornecedor do banco de dados.

  • Driver — Selecione o driver JDBC a ser usado para conexão com o banco de dados. A lista inclui drivers JDBC comuns para o fornecedor de banco de dados selecionado.

  • Nome do usuário — Informe o nome do usuário para conexão com o banco de dados.

  • Senha — Informe a senha relativa ao nome de usuário especificado.

  • DBMS/Serviço — Informe um nome do serviço ou SID de DBMS. O valor informado depende do driver selecionado.

    • Se o nome do driver Oracle selecionado contiver as palavras "para conexões da Instância", você deve informar o SID.

    • Se o nome do driver Oracle contiver as palavras "para conexões de Serviço", você deve informar o nome do serviço.

    • Para obter informações sobre configuração de um driver DataDirect, consulte a documentação do DataDirect.

  • Nome do Host — Informe o nome do servidor que está hospedando o banco de dados.

  • Porta — Informe o número da porta a ser usado para conexão com o servidor.

Os valores especificados são exibidos nas colunas apropriadas na lista de origens de dados, para a origem de dados selecionada.

Quando terminar de configurar as origens de dados JDBC, clique em Próximo para continuar.
