Resumo da Configuração

Revise as definições de configuração detalhadas de sua célula antes de continuar.

No painel Resumo, selecione um item para exibir detalhes sobre esse item no painel Detalhes à direita.

Você pode limitar os itens que são exibidos no painel Resumo selecionando uma opção de filtro na lista drop-down View Resumida. Por exemplo, para exibir as origens de dados JDBC que estão designadas a cada cluster e servidor, selecione JDBC na lista drop-down View Resumida.

Se precisar alterar a configuração, clique em Anterior para retornar à tela apropriada.

Ao terminar de revisar a configuração, clique em:

  • Criar (se estiver criando uma nova célula)

  • Estender (se estiver modificando uma célula existente)

  • (se estiver federando uma máquina para uma célula existente)
