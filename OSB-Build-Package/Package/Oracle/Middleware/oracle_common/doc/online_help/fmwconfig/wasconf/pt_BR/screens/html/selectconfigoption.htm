Selecionar Opção de Configuração

Selecione uma das opções a seguir. As opções vistas dependem da configuração atual.

  • Criar e Configurar Célula — Marque esta opção para criar uma nova célula do WebSphere. A tela Especificar Informações sobre Célula, Perfil e Nó aparecerá em seguida.

  • Federar Máquina e Configurar Célula — Selecione esta opção para adicionar o nó atual do WebSphere a uma célula existente do Fusion Middleware. A tela Especificar Informações sobre Nome do Perfil e Nó aparece em seguida. Nesse caso, você deve executar o Assistente de Configuração no nó que deseja federar para uma célula que foi criada anteriormente no nó do Gerenciador de Implantação.

  • Selecionar e Configurar Célula Existente — Essa opção só estará disponível se você já tiver criado uma célula. Para alterar a configuração de uma célula existente, selecione esta opção. Na caixa de listagem, clique na célula que você deseja configurar (a célula selecionada será destacada). A tela Carregando Informações da Célula aparece em seguida. Consulte Andamento da Célula para obter informações sobre esta tela.
