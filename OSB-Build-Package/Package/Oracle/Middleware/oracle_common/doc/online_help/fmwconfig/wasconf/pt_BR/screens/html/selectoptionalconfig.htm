Selecionar Configuração Opcional

Selecione todas as categorias (se houver) cuja configuração avançada você deseja executar. Para cada categoria selecionada, a tela de configuração apropriada é exibida para permitir que você execute a configuração avançada. Se você não selecionar nenhum item desta tela, a tela Resumo da Configuração será exibida em seguida.


Observações:

As categorias que estão listadas nesta tela dependem dos recursos definidos nos modelos selecionados para a célula.

  • Servidores de Aplicações e Clusters — Selecione esta opção se quiser executar qualquer uma das tarefas a seguir:

    • Adicionar, deletar ou clonar um Servidor de Aplicações

    • Alterar o nome de um Servidor de Aplicações

    • Ativar ou desativar o modo de desenvolvimento de um Servidor de Aplicações

    • Adicionar clusters à célula e adicionar servidores a um cluster

    • Configurar pontos de extremidade (portas) para o Servidor de Administração ou qualquer Servidor de Aplicações

    • Alterar as localizações do diretório compartilhado da área de armazenamento de arquivos para seus clusters, incluindo o diretório de log, o diretório de armazenamento permanente e o diretório de armazenamento temporário

  • Implantações e Serviços — Selecione esta opção se quiser executar qualquer uma das tarefas a seguir:

    • Alterar os destinos de implantação de seus clusters ou servidores

    • Alterar os destinos de serviço de seus clusters ou servidores

    • Alterar os destinos de serviço personalizado de seus servidores

    • Alterar os destinos do adaptador de recursos de seus servidores

  • JDBC — Selecione esta opção se quiser alterar os destinos da origem de dados JDBC para seus clusters ou servidores.

  • JMS — Selecione esta opção se quiser alterar os destinos do recurso JMS para seus clusters ou servidores.

Depois de selecionar as opções de configuração apropriadas, clique em Próximo para continuar. A tela exibida em seguida varia de acordo com as opções selecionadas.
