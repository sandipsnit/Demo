Especificar Informações sobre Célula, Perfil e Nó

Informe os valores apropriados nos campos a seguir. Cada campo é obrigatório. Use apenas caracteres alfanuméricos em cada campo.

  • Nome da Célula — Informe um nome exclusivo para a nova célula. Esse é o nome do diretório em que a célula é criada, nos dois diretórios a seguir:

    • WAS_HOME/profiles/DepMgrProfile/config/cells, em que WAS_HOME é o diretório de instalação do WebSphere e DepMgrProfile é o nome do perfil do Gerenciador de Implantação.

    • WAS_HOME/profiles/AppSrvrProfile/config/cells, em que WAS_HOME é o diretório de instalação do WebSphere e AppSrvrProfile é o nome do perfil do Servidor de Aplicações.

    O nome default é machine_nameCelln, em que machine_name é o nome da máquina atual e n é o próximo número de dois dígitos na sequência nessa máquina. Por exemplo, se o nome da máquina for machine1 e Cell01 já existir, o nome da célula default será machine1Cell02.

  • Nome do Perfil do Gerenciador de Implantação — Informe um nome de perfil exclusivo do Servidor de Aplicações para esta célula. Este é o nome do diretório em que o perfil do Gerenciador de Implantação é criado, em WAS_HOME/profiles.

    O nome default é Dmgrn, em que n é o próximo número de dois dígitos na sequência. Por exemplo, se o perfil do Gerenciador de Implantação Dmgr01 já existir, o default para este campo será Dmgr02.

  • Nome do Nó do Gerenciador de Implantação — Informe um nome de nó exclusivo do Gerenciador de Implantação para esta célula.

    O nome default é machine_nameCellManagern, em que machine_name é o nome da máquina atual e n é o próximo número de dois dígitos na sequência. Por exemplo, se o nome da máquina for machine1 e CellManager01 já existir, o nome do nó default será machine1CellManager02.

  • Nome do Perfil do Servidor de Aplicações — Informe um nome de perfil exclusivo do Servidor de Aplicações para esta célula. Este é o nome do diretório em que o perfil do Servidor de Aplicações é criado, em WAS_HOME/profiles.

    O nome default é Customn, em que n é o próximo número de dois dígitos na sequência. Por exemplo, se o perfil do Servidor de Aplicações Custom01 já existir, o default para este campo será Custom02.

  • Nome do Nó do Servidor de Aplicações — Informe um nome de nó exclusivo do Servidor de Aplicações para esta célula.

    O nome default é machine_nameNoden, em que machine_name é o nome da máquina atual e n é o próximo número de dois dígitos na sequência na máquina. Por exemplo, se o nome da máquina for machine1 e Node01 já existir, o nome do nó default será machine1Node02.

Quando terminar de preencher cada campo, clique em Próximo.
