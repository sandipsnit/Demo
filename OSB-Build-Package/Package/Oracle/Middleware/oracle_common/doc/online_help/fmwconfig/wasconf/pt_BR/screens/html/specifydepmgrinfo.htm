Especificar Informações do Gerenciador de Implantação

Use esta tela para especificar o nome do host, a porta SOAP (apenas processo de federação), nome de usuário e senha do administrador para o Gerenciador de Implantação. Todos os campos são obrigatórios.

  • Nome do Host do Gerenciador de Implantação — Durante a criação de uma célula nova, esse campo assume como default o nome do host da máquina atual.

    Ao federar uma máquina, informe o nome do host do Gerenciador de Implantação que gerenciará o nó remoto.

  • Porta SOAP — Este campo só é exibido quando você está federando uma máquina. Informe a porta SOAP a ser usada para conexão com o nó do Gerenciador de Implantação. Essa é a porta SOAP do Gerenciador de Implantação cuja máquina você deseja federar. Geralmente, o default é 8880. Se você não souber qual é a porta SOAP para o Gerenciador de Implantação, abra o arquivo a seguir no host do Gerenciador de Implantação:

    WAS_HOME/profiles/DepMgr_name/logs/AboutThisProfile.txt

    em que WAS_HOME é o diretório home do WebSphere e DepMgr_name é o nome do perfil apropriado do Gerenciador de Implantação no host especificado.

  • Nome de Usuário do Administrador — Ao criar uma célula nova, informe o nome de usuário do administrador que deseja usar para efetuar log-in no Gerenciador de Implantação.

    Ao federar uma máquina, informe o nome de usuário do administrador que foi configurado para o Gerenciador de Implantação no host especificado.

  • Senha — Ao criar uma célula nova, informe a senha do administrador que você deseja usar para efetuar log-in no Gerenciador de Implantação.

    Ao federar uma máquina, informe a senha do administrador que foi configurada para o Gerenciador de Implantação no host especificado.

  • Confirmar Senha — Este campo é exibido apenas durante a criação de uma célula nova. Informe novamente a senha do administrador para confirmar. Essa entrada deve corresponder exatamente à entrada no campo Senha.

Quando terminar de preencher cada campo, clique em Próximo para continuar.
