Especificar Informações sobre Nome do Perfil e Nó

  • Nome do Perfil do Servidor de Aplicações — Informe um nome de perfil exclusivo do Servidor de Aplicações deste nó. Este é o nome do diretório em que o perfil do Servidor de Aplicações é criado, em WAS_HOME/profiles em ambas as máquinas em que você está criando o nó e o host do Gerenciador de Implantação.

    O nome default é Customn, em que n é o próximo número de dois dígitos na sequência no nó do Gerenciador de Implantação especificado na tela anterior. Por exemplo, se o perfil do Servidor de Aplicações Custom01 já existir na célula, o default para este campo será Custom02.

    Se você alterar o nome do perfil default, certifique-se de que ele seja exclusivo na célula.

  • Nome do Nó do Servidor de Aplicações — Informe o nome do nó a ser usado para este Servidor de Aplicações.

    O nome default é machine_nameNoden, em que machine_name é o nome da máquina atual e n é o próximo número de dois dígitos na sequência na máquina atual. Por exemplo, se o nome da máquina for machine2 e Node01 já existir na máquina, o nome do nó default será machine2Node02.

    Se você alterar o nome do nó default, certifique-se de que ele seja exclusivo na célula.

Quando terminar de preencher cada campo, clique em Próximo para continuar.
