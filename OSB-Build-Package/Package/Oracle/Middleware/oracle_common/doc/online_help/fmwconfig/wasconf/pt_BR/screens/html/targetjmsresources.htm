Direcionar Recursos JMS para Clusters ou Servidores

Use esta tela para direcionar recursos JMS para servidores ou clusters.

Para direcionar um recurso JDBC para um cluster ou servidor:

  1. Na caixa de listagem Destino, selecione o cluster ou servidor ao qual você deseja destinar um recurso JMS.

    O nome do destino selecionado é exibido como o título da caixa de listagem à direita.

  2. Na caixa de listagem target_name, marque as caixas de seleção correspondentes aos recursos JMS que você deseja direcionar para o servidor ou cluster selecionado.


    Observação:

    Quando você seleciona um servidor na caixa de listagem Destino, algumas das caixas de seleção na caixa de listagem target_name podem ser desativadas, indicando recursos JMS que já estão direcionados no cluster que contém o servidor selecionado.

    Depois que você seleciona origens de dados, o nome do cluster ou servidor direcionado é exibido na coluna Destino na caixa de listagem target_name.

  3. Repita as etapas 1 e 2 para os outros clusters e servidores, conforme necessário.

  4. Após efetuar as seleções e entradas necessárias, clique em Próximo.
