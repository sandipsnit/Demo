Direcionar Adaptadores de Recursos para Servidores

Use esta tela para direcionar adaptadores de recursos para servidores individuais. Você não pode direcionar adaptadores de recursos para clusters. Os servidores designados para clusters estão incluídos na caixa de listagem Destino.

Os adaptadores de recursos que estão associados ao produto para o qual você está configurando a célula (por exemplo, SOA) são direcionados automaticamente para o servidor criado para aquele produto. Nesta tela, é possível direcionar adaptadores de recursos para servidores adicionais.

Para direcionar adaptadores de recursos para servidores:

  1. Na caixa de listagem Destino, selecione o servidor ao qual você deseja direcionar adaptadores de recursos.

    O nome do destino selecionado é exibido como o título da caixa de listagem à direita.

  2. Na caixa de listagem target_name, marque as caixas de seleção correspondentes aos adaptadores de recursos que você deseja direcionar para o servidor selecionado.

    Os adaptadores de recursos exibidos aqui variam, dependendo dos produtos selecionados na tela Adicionar Produtos à Célula anteriormente no assistente.

    Depois que você seleciona adaptadores de recursos, os nomes dos servidores direcionados são exibidos na coluna Destino na caixa de listagem target_name.

  3. Repita as etapas 1 e 2 para os outros servidores, conforme necessário.

  4. Após efetuar as seleções necessárias, clique em Próximo.

Ao reconfigurar uma célula existente, se você deletar um servidor para o qual os adaptadores de recursos estão direcionados no momento, o Assistente de Configuração redirecionará automaticamente estes adaptadores de recursos conforme a seguir:

  • Se os adaptadores de recursos foram direcionados originalmente somente ao servidor que você está deletando agora (isto é, após você deletar o servidor, os adaptadores de recursos se tornariam não direcionados na célula modificada), então o Assistente de Configuração redireciona automaticamente os adaptadores de recursos a todos os alvos apropriados.

    Um destino apropriado é qualquer servidor que não é definido na especificação dos grupos de configuração (arquivo config-groups.xml) de um modelo de produto incluído. Os servidores que são especificados em config-groups.xml pertencem basicamente ao modelo e, por isso, não são apropriados para direcionamento automático.

  • Se os adaptadores de recursos foram direcionados originalmente a vários destinos (incluindo servidores e o Servidor de Administração) e um dos servidores direcionados for deletado, então o Assistente de Configuração deixará as associações de destino restantes intactas e não tentará redirecionar os adaptadores de recursos.
