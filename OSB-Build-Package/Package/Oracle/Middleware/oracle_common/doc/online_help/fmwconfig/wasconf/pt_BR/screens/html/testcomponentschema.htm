Testar Esquema do Componente

Use esta tela para testar as configurações especificadas para os esquemas de componentes na tela anterior.

Marque as caixas de seleção adjacentes aos nomes dos esquemas a serem testados e, em seguida, clique em Testar Conexões.

O assistente testa a configuração para cada esquema ao tentar se conectar a um URL que seja construído com a utilização do driver, host, porta e outras informações especificadas na configuração do esquema.

O resultado do teste é indicado na coluna Status. Os detalhes são exibidos na seção Log do Resultado da Conexão.

Quando os testes do esquema forem concluídos com sucesso, clique em Próximo para continuar.
