Configurar Origens de Dados JDBC

Use esta tela para testar as conexões com a origem de dados configuradas nas telas Configurar Origens de Dados JDBC e Configurar Várias Origens de Dados do RAC.


Observações:

Para testar as conexões com o banco de dados, o banco de dados com o qual você está estabelecendo conexão deve estar em execução.

Se não quiser testar as conexões no momento, não selecione nenhuma origem de dados. Clique em Próximo para continuar e, em seguida, clique em OK na caixa de diálogo de aviso que aparece.

Marque a caixa de seleção para cada origem de dados que você deseja testar e, em seguida, clique em Testar Conexões.

O assistente testa a configuração para cada origem de dados selecionada ao tentar conectar-se a um URL que seja construído com a utilização de um driver, host, porta e outras informações especificadas na configuração da origem de dados.

O resultado do teste é indicado na coluna Status. Os detalhes são exibidos na seção Log do Resultado da Conexão.

Quando os testes forem concluídos com sucesso, clique em Próximo para continuar.
