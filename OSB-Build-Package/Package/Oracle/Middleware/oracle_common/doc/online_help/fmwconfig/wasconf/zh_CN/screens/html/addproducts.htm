将产品添加到单元

选择您希望包含在此单元中的产品。默认情况下“Oracle CIE 核心”选项会被选中并且无法清除。使用此选项会将所需的 CIE 联机定制服务添加到域，并将其定位到 OracleAdminServer。

如果您正在编辑现有单元，或者正在将计算机与某个单元联合，则已包含在该单元中的产品会被选中并且无法清除。


注：

通常，当您在部署管理器节点上创建新单元时，只选中 Oracle CIE 核心。在一个包含多个节点的配置中，当在正与部署管理器节点上的单元联合的计算机上运行配置向导时可选择其他产品。虽然您可以将 Fusion Middleware 组件定位到部署管理器节点上的管理服务器或应用程序服务器，但是最佳做法是将它们定位到远程节点上的应用程序服务器。(this may not be true; I don't think it matters where you add the products to the cell) (note needs work, need to find out how to word this; maybe not select prods so that canned app srvrs are created on the remote nodes)。

选中与您要包含在单元中的其他产品相对应的复选框。如果您选择的产品依赖于列表中的其他产品所提供的资源，则被依赖的产品也将被自动选中。同样，如果您清除了某个产品的复选框，而这个产品是其他产品所依赖的，则从属产品的复选框也会被清除。

产品列表 (配置模板) 会根据您配置的 Fusion Middleware 产品安装而有所不同。此处显示的示例用于安装...。(need to replace the above screen with a real one)

如果您正在将产品添加到一个现有的单元，则以前添加到该单元中的所有产品会被已选中并且无法清除。

选中要包含在该单元中的所有产品后，单击下一步可继续。


注：

单击下一步时，如果选中的任何产品需要其他产品提供的资源，而该其他产品未被选中，则会显示一个警告对话框。请返回到将产品添加到单元屏幕选择相关产品，然后继续操作。
