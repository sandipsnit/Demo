配置管理服务器

指定以下管理服务器设置：

  • 名称 - 管理服务器的名称。默认为 OracleAdminSever，并且不能更改。

  • 节点名 - 此下拉列表包含所有可用节点。选择您希望管理服务器在其上驻留的节点。

  • 开发模式 - 如果您希望在开发模式下运行管理服务器，请选中此复选框。要在生产模式下运行管理服务器，请保留此复选框为清除状态。有关开发模式和生产模式之间的差异的信息，请参阅 WebSphere 文档。

完成后，单击下一步可继续。
