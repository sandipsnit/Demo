配置集群

集群是一组应用程序服务器，这些服务器一起工作，为应用程序提供可伸缩性和高可用性。通过创建集群，可以对服务器进行分组，使它们作为单个单元运行来承载应用程序和资源。

使用此屏幕可添加集群并选择将作为该集群的主要集群成员的应用程序服务器。所有其他集群成员将从此主要集群成员中克隆。创建集群后，您可以在下一个屏幕上将其他应用程序服务器添加到集群。

还可以使用此屏幕删除集群或修改现有集群的设置。

要创建一个集群，请执行以下操作：

  1. 单击添加。一个新行将显示在表中。新集群的默认名称为 new_Cluster_n，其中 n 从 1 开始，并且每次添加一个新集群都会递增。要更改默认值，请在名称列中键入所需名称。该集群的名称在单元的所有组件名中必须是唯一的。

  2. 通过从第一个集群成员下拉列表中选择一个现有应用程序服务器可以将该应用程序服务器选为集群的主要成员。只有尚未分配到集群的应用程序服务器才会显示在此列表中。

  3. 对每个要创建的其他集群重复步骤 1 和 2。

  4. 完成后，请单击下一步。您可以在配置其他集群成员屏幕 (单击下一步后会显示) 上将其他服务器添加到集群。


    注：

    在您单击最后一个配置向导屏幕上的创建前，在此屏幕上所做的任何更改都不会保存到单元中。如果您退出了向导，或者单击上一步返回到前面的屏幕，则必须再次进行更改。

要删除某个集群，请选择该集群，然后单击删除。删除集群时，不会删除作为主要集群成员的应用程序服务器；该服务器只是从集群中删除了。您一次只能删除一个集群。

要更改某个集群的设置，请选择该集群，然后进行相应的更改。

要将显示方式从默认的表格格式切换到选项卡格式 (每个集群对应一个选项卡)，请单击切换显示。再次单击切换显示则恢复到表格格式。

完成配置集群后，单击下一步可继续。

有关 WebSphere 环境中的集群的详细信息，请参阅 WebSphere 文档。
