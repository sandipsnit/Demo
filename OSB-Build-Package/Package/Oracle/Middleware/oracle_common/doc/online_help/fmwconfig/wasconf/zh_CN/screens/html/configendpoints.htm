配置端点

使用此屏幕可以配置各个服务器 (包括管理服务器) 的以下端点 (监听地址)。所有端点在单元中必须唯一。默认情况下，将服务器添加到单元时会为该服务器分配唯一的端点。除非您要使用指定服务器上配置的默认端口以外的端口，否则无需修改这些端点。

有关此屏幕上列出的各种端点的信息，请参阅 WebSphere 文档。

要更改端点，请执行以下操作：

  1. 从服务器名/节点名下拉列表中选择要更改其端点的服务器。该列表包含单元中的所有服务器，包括管理服务器、分配给集群的应用程序服务器和存在于单元中所有节点上的独立应用程序服务器。

  2. 在相应行上键入新端点值。如果输入的值已在使用中，则将显示以下验证错误；单击确定，然后为端点输入其他值。



    注：

    如果您打算在同一计算机上运行多个部署管理器，则必须确保每个关联的单元中的端点在该计算机上所有部署管理器节点之间都是唯一的。

    同样，您拥有远程计算机上的应用程序服务器，并且这些服务器已分配给部署管理器主机上的不同单元，则必须确保指定计算机上所有应用程序服务器的所有端点都是唯一的。

    您将不会担心存在于不同单元中的端点会发生冲突。

  3. 对每个要更改的端点重复步骤 2。

您还可以禁用任何服务器上的端点 (如果不再需要它)。要禁用服务器上的一个或多个端点，请从服务器名/节点名下拉列表中选择该服务器，然后清除要禁用的每个端点的复选框。

要在指定服务器上为所有端点使用公用主机，请从服务器名/节点名下拉列表中选择该服务器，然后选择为所有端点使用公用主机复选框。选中此复选框后，当前所选服务器上各个端点的主机名将变为可编辑，管理端点主机名除外。

完成后，单击下一步。如果存在任何端点冲突，将显示验证错误对话框，必须解决该冲突才能继续。
