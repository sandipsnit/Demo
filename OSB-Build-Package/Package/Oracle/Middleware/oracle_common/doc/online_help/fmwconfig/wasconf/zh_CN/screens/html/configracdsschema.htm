配置 RAC 数据源组件方案

使用此屏幕可以将包含在 WebSphere 单元中的组件方案配置为 Oracle RAC 方案。

您在此向导的配置 JDBC 组件方案屏幕上选择配置为 Oracle RAC 方案的组件方案将在该屏幕的下半部分列出。

在此屏幕下半部分的方案列表中，通过选中每个方案名称旁的复选框，来选择要配置为 Oracle RAC 方案的方案。


注：

选择多个方案时，某些字段中可能显示各个组件方案之间不同文本，指示在所选方案中这些字段的当前值是不同的。如果仍要更改此类字段中的值，则将在所选方案中统一应用新值。

  • 驱动程序 - 选择要用于连接数据库的 JDBC 驱动程序。

  • 服务名 - 输入数据库 Oracle RAC 服务名。

  • 用户名 - 输入用于连接到数据库的用户名。

  • 口令 - 输入指定用户名的口令。

  • 主机名 - 输入承载 Oracle RAC 数据库实例的服务器的名称。

  • 实例名 - 输入每个 Oracle 数据库实例的名称。

  • 端口 - 输入用于连接到承载数据库的服务器的端口号。


注：

必须指定至少一个数据库实例的主机名、实例名和端口号。

要添加数据库实例，请单击添加。指定主机名、实例名和端口号。

在方案列表中，为选定方案指定的值将显示在相应列中。
