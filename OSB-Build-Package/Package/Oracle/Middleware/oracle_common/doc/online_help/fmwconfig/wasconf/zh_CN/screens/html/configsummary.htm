配置概要

复查单元的详细配置设置，然后继续。

在概要窗格中，选择要在右侧的详细资料窗格中显示其有关详细资料的项。

可以通过从概要视图下拉列表中选择筛选器选项来限制概要窗格中显示的项。例如，要查看已分配给各个集群和服务器的 JDBC 数据源，请从概要视图下拉列表中选择 JDBC。

如果需要更改配置，请单击上一步以返回到相应的屏幕。

完成复查配置后，单击以下项：

  • 创建 (如果创建新单元)

  • 扩展 (如果修改现有单元)

  • (如果将计算机与现有单元联合)
