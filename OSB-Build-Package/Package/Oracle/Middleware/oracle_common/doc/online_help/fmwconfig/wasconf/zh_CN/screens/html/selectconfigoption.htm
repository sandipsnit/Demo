选择配置选项

选择以下选项之一。您看到的选项取决于您的当前配置。

  • 创建和配置单元 - 选择此选项可创建新的 WebSphere 单元。下一步将显示指定单元、概要文件和节点信息屏幕。

  • 联合计算机和配置单元 - 选择此选项可将当前 WebSphere 节点添加到现有 Fusion Middleware 单元。下一步将显示指定概要文件和节点名信息屏幕。在这种情况下，必须在您要与以前从部署管理器节点创建的单元联合的节点上运行配置向导。

  • 选择和配置现有单元 - 仅在您已创建单元时，此选项才可用。要更改现有单元的配置，请选择此选项。从列表框中单击要配置的单元 (所选单元将突出显示)。下一步将显示加载单元信息屏幕。有关该屏幕，请参阅单元进度。
