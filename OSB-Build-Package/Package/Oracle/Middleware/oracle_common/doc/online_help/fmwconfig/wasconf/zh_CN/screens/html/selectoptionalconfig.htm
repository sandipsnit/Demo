选择可选配置

选择要为其执行高级配置的所有类别 (如果存在)。对于所选的每个类别，都会显示可用于执行高级配置的相应配置屏幕。如果不在此屏幕上选择任何项，则下一步将显示配置概要屏幕。


注：

此屏幕中列出的类别取决于在为单元选择的模板中定义的资源。

  • 应用程序服务器和集群 - 如果要执行下列任一任务，请选择此选项：

    • 添加、删除或克隆应用程序服务器

    • 更改应用程序服务器的名称

    • 将应用程序服务器的开发模式切换为“打开”或“关闭”

    • 将集群添加到单元，并将服务器添加到集群

    • 配置管理服务器或任何应用程序服务器的端点 (端口)

    • 更改集群的文件存储共享目录 (包括日志目录、永久存储目录和临时存储目录) 位置

  • 部署和服务 - 如果要执行下列任一任务，请选择此选项：

    • 更改集群或服务器的部署目标

    • 更改集群或服务器的服务目标

    • 更改服务器的定制服务目标

    • 更改服务器的资源适配器目标

  • JDBC - 如果要更改集群或服务器的 JDBC 数据源目标，请选择此选项。

  • JMS - 如果要更改集群或服务器的 JMS 资源目标，请选择此选项。

选择相应的配置选项后，单击下一步可继续。下一步显示的屏幕将因您所选的选项而异。
