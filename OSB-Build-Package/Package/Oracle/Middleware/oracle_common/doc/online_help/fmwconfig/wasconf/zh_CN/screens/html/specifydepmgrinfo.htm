指定部署管理器信息

使用此屏幕可以指定部署管理器的主机名、SOAP 端口 (仅限联合过程)、管理员用户名和口令。所有字段都是必需的。

  • 部署管理器主机名 - 创建新单元时，此字段默认为当前计算机的主机名。

    联合计算机时，输入将管理远程节点的部署管理器的主机名。

  • SOAP 端口 - 仅当联合计算机时才会显示此字段。输入要用于连接到部署管理器节点的 SOAP 端口。这是您要将此计算机联合到的部署管理器的 SOAP 端口。默认值通常为 8880。如果不知道部署管理器的 SOAP 端口，请打开部署管理器主机上的以下文件：

    WAS_HOME/profiles/DepMgr_name/logs/AboutThisProfile.txt

    其中，WAS_HOME 为 WebSphere 主目录，DepMgr_name 为指定主机上相应部署管理器概要文件的名称。

  • 管理员用户名 - 创建新单元时，输入要用于登录部署管理器的管理员用户名。

    联合计算机时，请输入为指定主机上的部署管理器配置的管理员用户名。

  • 口令 - 创建新单元时，输入要用于登录部署管理器的管理员口令。

    联合计算机时，请输入为指定主机上的部署管理器配置的管理员口令。

  • 确认口令 - 仅当创建新单元时才会显示此字段。再次输入管理员口令以确认它。此条目必须与口令字段中的条目完全匹配。

完成填写每个字段后，单击下一步以继续。
