将 JMS 资源定位到集群或服务器

使用此屏幕可以将 JMS 资源定位到服务器或集群。

要将 JMS 资源定位到集群或服务器，请执行以下操作：

  1. 在目标列表框中，选择要将 JMS 资源定位到的集群或服务器。

    所选目标的名称将作为右侧的列表框的标题显示。

  2. 在 target_name 列表框中，选中与要定位到所选服务器或集群的 JMS 资源对应的复选框。


    注：

    在目标列表框中选择服务器时，可能会禁用 target_name 列表框中的某些复选框，指示已将 JMS 资源定位到包含所选服务器的集群。

    完成选择数据源后，所定位的集群或服务器将显示在 target_name 列表框的目标列中。

  3. 如有必要，对其他集群和服务器重复执行步骤 1 和 2。

  4. 做出所需选择之后，单击下一步。
