测试 JDBC 数据源

使用此屏幕可以测试在配置 JDBC 数据源和配置 RAC 多数据源屏幕上配置的数据源连接。


注：

为测试数据库连接，要连接的数据库必须处于运行状态。

如果您不希望在此时测试连接，则不要选择任何数据源。单击下一步以继续，然后在显示的警告对话框上单击“确定”。

选中要测试的每个数据源的复选框，然后单击测试连接。

该向导将通过尝试连接到使用在配置数据源时指定的驱动程序、主机、端口和其他信息构建的 URL，测试每个所选数据源的配置。

状态列中将指明测试结果。详细信息将显示在连接结果日志部分中。

成功完成测试后，单击下一步可继续。
