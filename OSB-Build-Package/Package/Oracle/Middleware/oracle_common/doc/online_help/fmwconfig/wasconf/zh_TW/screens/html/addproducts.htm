將產品新增至封包

選擇您想要併入此封包的產品。預設會選取 Oracle CIE Core 選項，而且無法加以清除。此選項會新增必要的「CIE 線上自訂服務」至網域，並將其定位至OracleAdminServer。

如果您正在編輯現有封包，或正在將機器與封包聯合，則會選取併入到此封包的產品，且無法加以清除。


注意：

通常情況下，當您正在「建置管理程式」節點上建立新封包時，僅會選取 Oracle CIE Core。於包含多個節點組態中，在正與「建置管理程式」節點上的封包聯合的機器上執行「組態精靈」時，請選取其他產品。儘管您可以將 Fusion Middleware 元件定位至「建置管理程式」節點上的「管理伺服器」或「應用程式伺服器」，最佳的實務是將它們定位到您在遠端節點上的「應用程式伺服器」(這也許不是真的，我並不認為在哪裡新增產品至封包會有什麼關係)(注意事項需要進一步工作、必須找出文字表達方式，也許不選取產品，所以才會在遠端節點上建立封裝的應用程式伺服器)。

選取對應至其他想併入封包之產品的核取方塊。如果選取的產品與其他產品或清單中產品所提供的資源相依，則也會自動選取該產品。同樣地，如果清除其他產品相依的產品核取方塊，也會清除相依產品的核取方塊。

產品清單 (組態樣板) 根據您正在設定的 Fusion Middleware 產品安裝而不同。這裡顯示的範例針對的是 .... 安裝。(必須以實例取代上述內容)

如果您正將產品新增至現有封包，將會選取先前新增至封包的所有產品，且無法加以清除。

選取所有要併入封包的產品後，請按下一步，繼續作業。


注意事項：

當您按一下下一步時，如果任一選取產品需要其他未選取之產品提供的資源，則會顯示警告對話方塊。請返回將產品新增至封包畫面以選取相依性產品，再繼續作業。
