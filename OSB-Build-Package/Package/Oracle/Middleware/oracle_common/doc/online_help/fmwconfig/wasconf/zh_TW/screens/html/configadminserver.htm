設定管理伺服器

指定下列「管理伺服器」設定值：

  • 名稱—「管理伺服器」的名稱。預設為 OracleAdminSever，無法變更。

  • 節點名稱—此下拉式清單包含所有可用的節點。選取想要「管理伺服器」駐留的節點。

  • 開發模式—如果要在開發模式中執行「管理伺服器」，請選取此核取方塊。要在生產模式中執行「管理伺服器」，則請清除此核取方塊。有關開發模式與生產模式之間差異的詳細資訊，請參閱 WebSphere 文件。

完成後，請按下一步，繼續作業。
