設定 RAC 多重資料來源

使用此畫面將您選擇的資料來源設為 Oracle RAC 資料來源。

在精靈的設定 JDBC 資料來源畫面中，您選擇要設定為 Oracle RAC 資料來源的資料來源會列在畫面的下半部。

選取資料來源名稱旁的核取方塊，以選取想要指定設定的每個資料來源。


注意：

當您選取多個資料來源時，可能會在特定欄位中顯示因資料來源而異文字，指出在選取之資料來源中，那些欄位的目前值會不同。如果您變更這類欄位中的值，則會將新值統一套用至選取之資料來源。

  • 驅動程式—選取用來連線資料庫的 JDBC 驅動程式。

  • 服務名稱—輸入 Oracle RAC 資料庫服務名稱。

  • 使用者名稱—輸入用來連線資料庫的使用者名稱。

  • 密碼—輸入指定之使用者名稱的密碼。

  • 主機名稱—輸入主控 Oracle RAC 資料庫執行處理之伺服器的名稱。

  • 執行處理名稱—輸入每個 Oracle 資料庫執行處理的名稱。

  • 連接埠—輸入用來連線至主控資料庫之伺服器的連接埠號碼。

若要新增資料庫執行處理，請按一下新增，然後指定主機名稱、執行處理名稱和連接埠號碼。

完成 RAC 多重資料來源設定後，請按下一步。
