組態摘要

先複查詳細的封包組態設定，再繼續進行。

在摘要窗格中選取一個項目，以在右側詳細資訊窗格中顯示該項目的詳細資訊。

您可以從摘要視觀表下拉式清單中選取篩選條件選項，來限制摘要窗格中顯示的項目。例如，若要檢視指派至每個叢集和伺服器的 JDBC 資料來源，請從摘要視觀表下拉式清單中選取 JDBC。

如果您需要變更組態，請按上一步返回適當的畫面。

複查組態完成後，請按一下：

  • 建立 (若要建立新的封包)

  • 延伸 (若要修改現有的封包)

  • (若要將機器與現有的封包聯合)
