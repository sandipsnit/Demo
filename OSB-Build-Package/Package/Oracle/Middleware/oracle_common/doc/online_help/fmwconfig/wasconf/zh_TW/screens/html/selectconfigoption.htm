選取組態選項

請選取下列其中一個選項。所顯示的選項取決於您目前的組態。

  • 建立和設定封包—選取此選項以建立新的 WebSphere 封包。接著會出現指定封包、設定檔和節點資訊畫面。

  • 聯合機器和設定封包—選取此選項將目前的 WebSphere 節點新增至現有的 Fusion Middleware 封包。接著會出現指定設定檔和節點名稱資訊畫面。在此情況中，您必須在要與封包 (先前從「建置管理程式」節點建立) 聯合的節點上執行「組態精靈」。

  • 選取和設定現有封包—只有當您已經建立封包時，才能使用此選項。若要變更現有封包的組態，請選取此選項。從清單方塊中按一下要設定的封包 (會標示出選取的封包)。接著會出現正在載入封包資訊畫面。有關此畫面的詳細資訊，請參閱封包進度。
