指定建置管理程式資訊

使用此畫面可以指定「建置管理程式」的主機名稱、SOAP 連接埠 (僅限聯合流程)、管理員使用者名稱和密碼。全都是必要欄位。

  • 建置管理程式主機名稱—建立新封包時，此欄位會預設為目前機器的主機名稱。

    聯合機器時，請輸入即將管理遠端節點之「建置管理程式」的主機名稱。

  • SOAP 連接埠—只有當您正在聯合機器時，才會顯示此欄位。請輸入用以連線至「建置管理程式」節點的 SOAP 連接埠。這是您想要與此機器聯合之「建置管理程式」的 SOAP 連接埠。預設值一般是 8880。若您不知道「建置管理程式」的 SOAP 連接埠，請在「建置管理程式」主機上開啟下列檔案：

    WAS_HOME/profiles/DepMgr_name/logs/AboutThisProfile.txt

    其中 WAS_HOME 是 WebSphere 本位目錄，而 DepMgr_name 是指定主機上適當的「建置管理程式」設定檔名稱。

  • 管理使用者名稱—建立新封包時，請輸入要用來登入「建置管理程式」的管理員使用者名稱。

    聯合機器時，請輸入為指定主機上之「建置管理程式」設定的管理員使用者名稱。

  • 密碼—建立新封包時，請輸入想要用來登入「建置管理程式」的管理員密碼。

    聯合機器時，請輸入為指定主機上之「建置管理程式」設定的管理員密碼。

  • 確認密碼—只有建立新封包時，才會顯示此欄位。請重新輸入密碼進行確認。此項目必須與密碼欄位中的項目完全一致。

每個欄位都輸入完成後，請按下一步，繼續作業。
