將 JDBC 資料來源定位到叢集或伺服器

使用此畫面將 JDBC 資料來源定位到伺服器或叢集。

若要將 JDBC 資料來源定位到叢集或伺服器：

  1. 在目標清單方塊中，選取想要將資料來源定位到的叢集或伺服器。

    選取之目標的名稱會顯示為右側清單方塊的標題。

  2. 在 target_name 清單方塊中，選取對應至要定位於選取伺服器或叢集之資料來源的核取方塊。


    注意：

    當您在目標清單方塊中選取伺服器時，可能會停用 target_name 清單方塊中的部分核取方塊，以指出已定位至含有選取伺服器之叢集的資料來源。

    選取資料來源之後，已定位叢集或伺服器的名稱會顯示在 target_name 清單方塊中的目標資料欄內。

  3. 依需要針對其他叢集和伺服器，重複步驟 1 和 2。

  4. 進行必要的選取之後，請按一下「下一步」。
