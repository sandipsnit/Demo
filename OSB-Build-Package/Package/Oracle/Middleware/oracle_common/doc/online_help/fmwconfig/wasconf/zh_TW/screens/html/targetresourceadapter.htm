定位資源轉接器至伺服器

使用此畫面可以定位資源轉接器至個別伺服器。您無法定位資源轉接器至叢集。指派到叢集的伺服器包含在目標清單方塊中。

會將與設定封包之產品 (例如，SOA) 相關聯的資源轉接器，自動定位至為該產品所建立的伺服器。在此畫面中，您可以定位資源轉接器至其他伺服器。

若要定位資源轉接器至伺服器：

  1. 在目標清單方塊中，選取想要將資源轉接器定位到的伺服器。

    選取之目標的名稱會顯示為右側清單方塊的標題。

  2. 在 target_name 清單方塊中，選取對應至要定位於選取伺服器之資源轉接器的核取方塊。

    這裡顯示的資源轉接器會根據您先前在精靈的將產品新增至封包畫面中選取的產品而不同。

    選取資源轉接器之後，已定位伺服器的名稱會顯示在 target_name 清單方塊中的目標資料欄內。

  3. 依需要針對其他伺服器，重複步驟 1 和 2。

  4. 進行必要的選取之後，請按下一步。

當您正在重新設定現有的封包時，如果您刪除資源轉接器目前所定位的伺服器，「組態精靈」會依下列方式自動重新定位資源轉接器：

  • 如果資源轉接器之前只定位到您刪除的伺服器 (即在您刪除伺服器之後，資源轉接器在修改後的封包中會變成未定位)，「組態精靈」會自動重新定位資源轉接器至所有合適的目標。

    合適的目標就是未在包含產品樣板之組態群組規格 (config-groups.xml 檔案) 中定義的任何伺服器。config-groups.xml 檔案中指定的伺服器基本上是樣板所擁有，因此不適合用來自動定位。

  • 如果資源轉接器之前定位到多個目標 (包括伺服器和「管理伺服器」)，刪除其中一個所定位的伺服器時，「組態精靈」會維持其他的目標關聯不變，而且不會嘗試重新定位資源轉接器。
