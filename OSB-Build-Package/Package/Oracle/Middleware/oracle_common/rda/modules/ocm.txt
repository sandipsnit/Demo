//   09000.09099      Oracle Configuration Manager Installation Warnings
// ///////////////////////////////////////////////////////////////////////////
//
09000,0, "Oracle Configuration Manager installation not requested"
// *Cause: The installation has not been requested, even though the
//         prerequisites are fulfilled.
// *Action: Rerun the OCM module setup when you want to install OCM.
// *Mnemonic: OCM_NO_REQ
//
09001,0, "The platform is not supported"
// *Cause:  This platform is not currently supported for OCM.
// *Action: None
// *Mnemonic: OCM_NOT_SUP
//
09002,0, "The Oracle Configuration Manager software is not available"
// *Cause: The RDA distribution does not include the OCM software for this
//         platform.
// *Action: Check if an RDA-OCM distribution is available for your platform, and
//          if so, download it.
// *Mnemonic: OCM_NO_SOFT
//
09003,0, "Not owner of the destination home directory"
// *Cause: The current user is not the owner of the destination home
//         directory.
// *Action: When installing Oracle Configuration Manager, run RDA
//          as the Oracle home owner.
// *Mnemonic: OCM_NOT_OWNER
//
09004,0, "Java Development Kit not found"
// *Cause: Java is not found under the $ORACLE_HOME/jdk directory or JAVA_HOME
//         is not set.
// *Action: Set JAVA_HOME to a JDK that is equal to or greater than 1.2.2 and
//          rerun the OCM module setup.
// *Mnemonic: OCM_NO_JDK
//
09005,0, "A setup error occurred"
// *Cause: RDA detected an error during OCM installation and setup.
// *Action: Take appropriate actions based on the information provided with the
//          error and rerun the OCM module setup.
// *Mnemonic: OCM_SETUP
//
09006,0, "Automated setup"
// *Cause: Cannot prompt for OCM questions due to an automated setup.
// *Action: Rerun the OCM module setup in interactive mode.
// *Mnemonic: OCM_AUTO_SETUP
//
09007,0, "Oracle home not found"
// *Cause: The specified Oracle home directory was not found.
// *Action: Ensure that specified Oracle home is correct.
// *Mnemonic: OCM_NO_HOME
//
09008,0, "Multiple OCM distribution kits are present"
// *Cause: Multiple distribution kits were found for the current operating
//         system.
// *Action: Ensure that only one distribution kit is present per operating
//          system, or specify the correct distribution kit in the
//          "OCM_DISTRIBUTION" setting.
// *Mnemonic: OCM_MANY_KITS
//
09009,0, "Middleware home found"
// *Cause: The parent directory of the current Oracle home could be a
//         Middleware home.
// *Action: RDA does not install OCM in such a context. Install OCM manually
//          when appropriate.
// *Mnemonic: OCM_MW_HOME
//
09010,0, "Oracle Configuration Manager setup postponed"
// *Cause: The user declined to complete OCM setup in an Oracle Fusion
//         Middleware environment.
// *Action: Redo the OCM module setup when appropriate.
// *Mnemonic: OCM_POSTPONED
//

09100,0, "Not authorized to execute emCCR"
// *Cause: The current user does not have the privileges for executing emCCR.
// *Action: RDA does not install OCM in such a context. Install OCM manually
//          when appropriate.
// *Mnemonic: OCM_PERM
//
09101,0, "Cannot fork an OCM collection"
// *Cause: The current context has no fork capability.
// *Action: Use a later Perl version and take care that forks are not disabled
//          in the setup file.
// *Mnemonic: OCM_FORK
//
09102,0, "No OCM configuration found"
// *Cause: RDA has found an OCM installation but no valid configuration.
// *Action: Configure OCM when appropriate.
// *Mnemonic: OCM_NO_CONFIG
//
