Server Clustern zuweisen

Hier können Sie Clustern Managed Server zuweisen. Im Listenfeld "Server" werden nur Managed Server aufgelistet. Der Administrationsserver ist nicht aufgeführt, da er keinem Cluster zugewiesen werden kann.

  1. Wählen Sie im Bereich Cluster das Cluster aus, dem Sie einen Managed Server zuweisen möchten.
  2. Sie können Managed Server dem ausgewählten Cluster folgendermaßen zuweisen:
    • Doppelklicken Sie im Bereich Server auf den Namen des Managed Servers.
    • Wählen Sie im Bereich Server den Managed Server aus, und klicken Sie auf die Schaltfläche mit dem Pfeil nach rechts, die sich zwischen den Bereichen Server und Cluster befindet.
    • Klicken Sie im Bereich Server bei gedrückter Umschalttaste auf mehrere Managed Server, um sie auszuwählen, und klicken Sie dann auf die Schaltfläche mit dem Pfeil nach rechts, die sich zwischen den Bereichen Server und Cluster befindet.

    Der Name des zugewiesenen Managed Servers wird daraufhin aus dem Bereich Server entfernt und im Bereich Cluster unter den Namen des Ziel-Clusters eingefügt.

    Hinweis: Im Bereich Server werden nur Managed Server aufgelistet. Der Administrationsserver ist nicht aufgeführt, da er keinem Cluster zugewiesen werden kann.

  3. Wiederholen Sie die Schritte 1 und 2 für jeden Managed Server, den Sie einem Cluster zuweisen möchten.
  4. Prüfen Sie die Cluster-Zuweisungen.

    Falls erforderlich, können Sie einen Managed Server folgendermaßen aus einem Cluster entfernen:

    • Doppelklicken Sie im Bereich Cluster auf den Namen des Managed Servers.
    • Wählen Sie den Managed Server im Bereich Cluster aus, und klicken Sie auf die Schaltfläche mit dem Pfeil nach links, die sich zwischen den Bereichen Server und Cluster befindet.

    Der Name des Managed Servers wird aus dem Bereich Cluster entfernt und im Bereich Server wiederhergestellt.

  5. Wenn Sie die Zuweisung der Managed Server zu den gewünschten Clustern beendet haben, klicken Sie im Assistenten auf die Schaltfläche Weiter.
