Server Rechnern zuweisen

Hier können Sie jedem der von Ihnen definierten Rechner WebLogic Server-Instanzen zuweisen.

  1. Wählen Sie im Bereich Rechner den Windows- oder UNIX-Rechner aus, dem Sie eine WebLogic Server-Instanz zuweisen möchten.
  2. Sie können WebLogic Server-Instanzen einem ausgewählten Rechner folgendermaßen zuweisen:
    • Doppelklicken Sie im Bereich Server auf die WebLogic Server-Instanz.
    • Wählen Sie die entsprechende WebLogic Server-Instanz im Bereich Server aus, und klicken Sie auf die Schaltfläche mit dem Pfeil nach rechts, die sich zwischen den Bereichen Server und Rechner befindet.
    • Klicken Sie bei gedrückter Umschalttaste auf mehrere Server im Bereich Server, um sie auszuwählen, und klicken Sie dann auf die Schaltfläche mit dem Pfeil nach rechts, die sich zwischen den Bereichen Server und Rechner befindet.

    Der Name der zugewiesenen WebLogic Server-Instanz wird daraufhin aus dem Bereich Server entfernt und unter den Namen des Zielrechners im Bereich Rechner eingefügt.

  3. Wiederholen Sie die Schritte 1 und 2 für jede WebLogic Server-Instanz, die Sie einem Rechner zuweisen möchten.
  4. Prüfen Sie die Rechnerzuweisungen.

    Falls erforderlich, können Sie eine WebLogic Server-Instanz folgendermaßen aus einem Rechner entfernen:

    • Doppelklicken Sie im Bereich Rechner auf den Namen der entsprechenden WebLogic Server-Instanz.
    • Wählen Sie die entsprechende WebLogic Server-Instanz im Bereich Rechner aus, und klicken Sie auf die Schaltfläche mit dem Pfeil nach links, die sich zwischen den Bereichen Server und Rechner befindet.

    Der Name der WebLogic Server-Instanz wird aus dem Bereich Rechner entfernt und im Bereich Server wiederhergestellt.

  5. Wenn Sie die Zuweisung der WebLogic Server-Instanzen zu den gewünschten Rechnern beendet haben, klicken Sie auf die Schaltfläche Weiter.

Weitere Informationen finden Sie in der Dokumentations-Library für Oracle Fusion Middleware im Oracle Technology Network.
