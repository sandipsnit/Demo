Erweiterte Konfiguration auswählen

Wählen Sie alle Kategorien (falls zutreffend) aus, für die Sie eine erweiterte Konfiguration ausführen möchten. Für jede ausgewählte Kategorie wird der entsprechende Konfigurationsbildschirm angezeigt, in dem Sie eine erweiterte Konfiguration ausführen können. Wenn Sie hier keine Elemente auswählen, wird als Nächstes der Bildschirm "Konfigurationszusammenfassung" angezeigt.

Hinweis:Die in diesem Bildschirm aufgelisteten Kategorien hängen von den Ressourcen ab, die in der für die Domain ausgewählten Vorlage definiert wurden.

Hinweis: Die Optionen Administrationsserver und RDBMS-Sicherheitsspeicher stehen bei der Erweiterung einer Domain nicht zur Verfügung.

  • Administrationsserver
    Wählen Sie diese Option, um die Einstellungen für den Administrationsserver wie Servername, Listening-Adresse, Listening Port und SSL-Einstellungen zu ändern.
  • Verteiltes JMS-Ziel
    Standardmäßig werden verteilte JMS-Ziele automatisch in gewichtete verteilte Ziele (WDD) konvertiert. Wählen Sie diese Option, um ein verteiltes JMS-Ziel in ein einheitliches verteiltes Ziel (UDD) zu ändern.
  • Managed Server, Cluster und Rechner
    Wählen Sie diese Option, um
    • der Domain Managed Server, Cluster oder Rechner hinzuzufügen
    • vorhandene Managed Server, Cluster oder Rechner zu löschen
    • einem vorhandenen Cluster Managed Server hinzuzufügen
    • Ändern Sie die Einstellungen für vorhandene Managed Server, Cluster oder Rechner.
  • Deployments und Services
    Wählen Sie diese Option, um einzustellen, wie Anwendungs-Deployments und Services als Ziele für Server und Cluster definiert werden.
  • JMS-Dateispeicher
    Wählen Sie diese Option, um die Einstellungen für die JMS-Dateispeicher zu ändern. Sie können für jeden Dateispeicher den Namen, das Verzeichnis und die Policy zum synchronen Schreiben ändern.
  • RDBMS-Sicherheitsspeicher
    Wählen Sie diese Option, um ein externes relationales Datenbankmanagementsystem (RDBMS) als Datenspeicher für verschiedene Sicherheitsprovider zu konfigurieren.

Für jede erweiterte Konfigurationsoption, die Sie auf diesem Bildschirm auswählen, zeigt der Assistent später zusätzliche Bildschirme an, auf denen Sie die entsprechenden erweiterten Konfigurationseinstellungen definieren können.

Aktivieren Sie je nach gewünschter erweiterter Konfiguration die entsprechenden Kontrollkästchen, und klicken Sie dann auf die Schaltfläche Weiter.

Weitere Informationen finden Sie in der Dokumentations-Library für Oracle Fusion Middleware im Oracle Technology Network.
