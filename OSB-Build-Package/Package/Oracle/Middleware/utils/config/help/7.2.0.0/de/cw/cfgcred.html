Zugangsdaten für Keystore konfigurieren

Auf diesem Bildschirm werden für jedes Produkt, für das ein Keystore-Kennwort festgelegt werden muss, die beiden Felder "Keystore-Kennwort eingeben" und "Kennwort bestätigen" angezeigt. Beispiel: Wenn Oracle Information Rights Manager installiert ist, gibt es zu den folgenden zwei Anwendungs-Keystores jeweils eine Gruppe von Kennwortfeldern:

  • Oracle Information Rights Management
  • Oracle Enterprise Manager-Plugin für IRM

  1. Geben Sie das Kennwort ein, das für den Zugriff auf den Keystore der Anwendung erforderlich ist, und bestätigen Sie es.
  2. Wenn Sie alle Keystore-Kennwörter eingegeben haben, klicken Sie auf die Schaltfläche Weiter.

Hinweis: Das Keystore-Kennwort für jedes hier aufgelistete Produkt muss für diese Produkte verwendet werden. Notieren Sie die eingegebenen Kennwörter, und teilen Sie sie allen Personen mit, die zur Verwendung dieser Produkte berechtigt sind.

Weitere Informationen finden Sie in der Dokumentations-Library für Oracle Fusion Middleware im Oracle Technology Network.
