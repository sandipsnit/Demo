Serverstartmodus und JDK konfigurieren

Hier geben Sie an, ob Sie WebLogic im Production-Modus oder im Entwicklungsmodus ausführen möchten. Sie geben außerdem den Speicherort der JDKs an, die für diese Domain verwendet werden.

  1. Wählen Sie den Startup-Modus für die WebLogic-Domain aus.
    • Entwicklungsmodus
      In diesem Modus ist die Sicherheitskonfiguration relativ gelockert, so dass Sie ein automatisches Deployment von Anwendungen durchführen können.
    • Production-Modus
      In diesem Modus ist die Sicherheitskonfiguration relativ streng, sodass für das Deployment von Anwendungen die Eingabe von Benutzername und Kennwort erforderlich ist. Bevor Sie eine Domain in die Produktion überführen, machen Sie sich mit den Verfahren zur Sicherung einer Produktionsumgebung vertraut. Weitere Informationen dazu finden Sie unter Sicherheit der Produktionsumgebung sicherstellen in Produktionsumgebung für Oracle WebLogic Server sichern.

    In der folgenden Tabelle finden Sie Informationen dazu, wie Sie einen Startup-Modus wählen, der Ihren Anforderungen entspricht.

  2. Wählen Sie das JDK aus.

    Wählen Sie im Bereich JDK-Auswahl das JDK für den im Bereich Startup-Modus für WebLogic-Domain ausgewählten Startup-Modus aus. Der Konfigurationsassistent zeigt eine Liste der im Installationsprogramm enthaltenen JDKs an. Sie können eines dieser JDKs oder ein anderes auf Ihrem System installiertes JDK auswählen.

    Hinweise:

    • Wenn Sie ein im Installationsprogramm enthaltenes JDK auswählen, erstellt der Konfigurationsassistent Serverstartskripte, um das JDK aufzurufen. Wenn Sie ein JDK auswählen, das nicht vom Installationsprogramm zur Verfügung gestellt wird, konfiguriert der Konfigurationsassistent keine Startskripte. Sie müssen die Startskripte in diesem Fall manuell ändern.
    • Wählen Sie nur solche JDKs aus, die auf der von Ihnen verwendeten Plattform unterstützt werden. Standardmäßig ist dasjenige JDK ausgewählt, das den Anforderungen Ihres Systems basierend auf der Plattform, auf der Sie die Domain installieren, am besten entspricht.
  3. Wenn Sie alle erforderlichen Einstellungen und Einträge vorgenommen haben, klicken Sie im Assistenten auf die Schaltfläche Weiter.
