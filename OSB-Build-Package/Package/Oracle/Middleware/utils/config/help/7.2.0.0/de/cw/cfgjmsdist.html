JMS-Dateispeicher konfigurieren

Zur Sicherstellung der Abwärtskompatibilität werden verteilte JMS-Ziele automatisch in gewichtete verteilte Ziele (WDD) konvertiert. Um ein verteiltes JMS-Ziel dahingehend zu ändern, dass es sich wie ein einheitliches verteiltes Ziel (UDD) verhält, wählen Sie aus dem Dropdown-Menü "UDD". Oracle empfiehlt die UDD-Einstellung.

Wenn Sie das Verhalten von JMS-Systemressourcen ändern, gilt die Einstellung nur für die aktuelle Session, sofern die JMS-Systemressource nicht einem Cluster zugeordnet wurde. Nachdem die JMS-Systemressource einem Cluster zugeordnet wurde, wird die Einstellung permanent und kann nicht mehr geändert werden.

Weitere Informationen über UDD und WDD finden Sie unter Arten von verteilten Zielen in JMS für Oracle WebLogic Server programmieren.
