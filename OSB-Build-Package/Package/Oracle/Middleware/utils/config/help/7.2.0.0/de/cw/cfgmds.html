RAC-Multidatenquellkomponentenschema konfigurieren

Hier können Sie die in der WebLogic-Domain enthaltenen Komponentenschemas als Oracle RAC-Multidatenquellen konfigurieren.

In der unteren Bildschirmhälfte sind die Komponentenschemas aufgelistet, für die Sie auf dem vorigen Bildschirm des Assistenten (JDBC-Komponentenschema konfigurieren) RAC-Datenquellen konfiguriert haben.

  1. Wählen Sie aus der Schemaliste in der unteren Bildschirmhälfte das Schema aus, das Sie als RAC-Datenquelle konfigurieren wollen. Aktivieren Sie dazu das Kontrollkästchen neben dem jeweiligen Schemanamen.

    Hinweis: Wenn Sie mehrere Datenquellschemas auswählen, wird möglicherweise der Text "Variiert unter Komponentenschemas" in bestimmten Feldern angezeigt. Dies weist darauf hin, dass die aktuellen Werte dieser Felder nicht in allen ausgewählten Schemas identisch sind. Wenn Sie die Werte in solchen Feldern ändern, werden die neuen Werte einheitlich auf die gewählten Schemas angewendet.

  2. Definieren Sie die Einstellungen für die RAC-Datenquellen.

    In der folgenden Tabelle werden die Felder auf diesem Bildschirm beschrieben.

    Feld Beschreibung
    Treiber Wählen Sie den Treiber aus, mit dem Sie sich bei der Datenbank anmelden wollen.
    Service-Name Geben Sie einen Namen für den Oracle RAC-Datenbank-Service ein.
    Benutzername Geben Sie den Anmeldenamen für die Anmeldung bei der Datenbank ein.
    Kennwort Geben Sie das Anmeldekennwort für den angegebenen Benutzernamen ein.
    Host-Name Geben Sie den Namen des Servers ein, auf dem sich die RAC-Datenbankinstanz befindet.
    Instanzname Geben Sie für jede Oracle-Datenbankinstanz den Namen ein.
    Port Geben Sie die Port-Nummer ein, über die Sie sich beim Server anmelden wollen, auf dem sich die Datenbankinstanz befindet.

    Hinweis: Sie müssen für mindestens eine Datenbankinstanz den Hostnamen, Instanznamen und die Port-Nummer angeben.

    Sie können eine Datenbankinstanz hinzufügen, indem Sie auf Hinzufügen klicken und dann den Hostnamen, Datenbankinstanznamen und die Port-Nummer angeben.

    Die von Ihnen angegebenen Werte werden für die gewählten Schemas in den entsprechenden Spalten in der Schemaliste angezeigt.

  3. Wenn Sie alle erforderlichen Einstellungen und Einträge vorgenommen haben, klicken Sie auf die Schaltfläche Weiter, um die Komponentenschemas zu testen.

Weitere Informationen finden Sie in der Dokumentations-Library für Oracle Fusion Middleware im Oracle Technology Network.
