Managed Server konfigurieren

Hier können Sie Managed Server hinzufügen oder löschen. Sie können auch die Einstellungen für einen vorhandenen Managed Server ändern. Pflichtfelder sind mit einem Sternchen (*) markiert.

  1. Prüfen Sie die aktuelle Managed Server-Konfiguration. Die Standardwerte hängen von der gewählten Domain-Quelle ab.

    Hinweis: Der Assistent bietet zwei Ansichten: eine tabellarische Übersicht aller Managed Server und eine Einzelansicht jedes einzelnen Managed Servers, bei der jeder Server über eine eigene Registerkarte dargestellt wird. Sie können den Anzeigemodus zwischen Tabellen- und Registerkartenansicht hin- und herwechseln, indem Sie auf Anzeige umschalten klicken.

  2. Je nach den Erfordernissen Ihrer Domain können Sie Managed Server hinzufügen oder löschen oder die Einstellungen für bestehende Managed Server ändern. Mit Sternchen (*) markierte Felder müssen ausgefüllt werden.

    Weitere Informationen zu den Feldern auf diesem Bildschirm finden Sie auf der Hilfeseite zum Bildschirm Administrationsserver konfigurieren

  3. Wenn Sie alle erforderlichen Einstellungen und Einträge vorgenommen haben, klicken Sie auf die Schaltfläche Weiter.

Weitere Informationen finden Sie in der Dokumentations-Library für Oracle Fusion Middleware im Oracle Technology Network.
