RAC-Multidatenquelle konfigurieren

Hier können Sie die in der Domain enthaltenen Datenquellen als Oracle RAC-Datenquellen konfigurieren.

In der unteren Bildschirmhälfte sind die Datenquellen aufgelistet, die Sie auf dem vorigen Bildschirm des Assistenten (JDBC-Datenquellen konfigurieren) als RAC-Datenquellen konfiguriert haben.

  1. Wählen Sie die Datenquelle aus, für die Sie Datenquelleinstellungen vornehmen möchten, indem Sie das Kontrollkästchen neben dem Namen der Datenquelle aktivieren.

    Hinweis: Wenn Sie mehrere Datenquellen auswählen, wird möglicherweise der Text "Variiert unter Datenquellen" in bestimmten Feldern angezeigt. Dies weist darauf hin, dass die aktuellen Werte dieser Felder nicht in allen ausgewählten Datenquellen identisch sind. Wenn Sie die Werte in solchen Feldern ändern, werden die neuen Werte einheitlich auf die gewählten Datenquellen angewendet.

  2. Definieren Sie die Einstellungen für die RAC-Datenquellen.

    In der folgenden Tabelle werden die Felder auf diesem Bildschirm beschrieben.

    Feld Beschreibung
    Treiber Wählen Sie den JDBC-Treiber aus, mit dem Sie sich bei der Datenbank anmelden wollen.
    Service-Name Geben Sie einen Namen für den Oracle RAC-Datenbank-Service ein.
    Benutzername Geben Sie den Anmeldenamen für die Anmeldung bei der Datenbank ein.
    Kennwort Geben Sie das Anmeldekennwort für den angegebenen Benutzernamen ein.
    Host-Name Geben Sie den Namen des Servers ein, auf dem sich die Oracle RAC-Datenbankinstanz befindet.
    Instanzname Geben Sie für jede Oracle-Datenbankinstanz den Namen ein.
    Port Geben Sie die Port-Nummer ein, über die Sie sich beim Server anmelden wollen, auf dem sich die Datenbankinstanz befindet.

    Hinweis: Sie müssen für mindestens eine Datenbankinstanz den Hostnamen, Instanznamen und die Port-Nummer angeben.

    Sie können eine Datenbankinstanz hinzufügen, indem Sie auf Hinzufügen klicken und dann den Hostnamen, Instanznamen und die Port-Nummer angeben.

    Die von Ihnen angegebenen Werte werden für die gewählten Datenquellen in den entsprechenden Spalten in der Datenquellenliste angezeigt.

  3. Wenn Sie alle erforderlichen Einstellungen und Einträge vorgenommen haben, klicken Sie auf Weiter, um die Verbindungen zu den Datenquellen zu testen.

Weitere Informationen finden Sie in der Dokumentations-Library für Oracle Fusion Middleware im Oracle Technology Network.
