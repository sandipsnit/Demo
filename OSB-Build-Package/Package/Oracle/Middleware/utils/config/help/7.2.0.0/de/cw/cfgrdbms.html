RDBMS-Sicherheitsspeicherdatenbank konfigurieren

Hier können Sie die Einstellungen für den RDBMS-Sicherheitsspeicher definieren.

In der folgenden Tabelle werden die Felder auf diesem Bildschirm beschrieben.

Feld Beschreibung
Datenbanktyp Wählen Sie aus der Dropdown-Liste den Datenbanktyp aus, den Sie als RDBMS-Sicherheitsspeicher verwenden wollen.
Treiber Wählen Sie den Treiber aus, den Sie für die Datenbank verwenden möchten. Welche Treiber zur Verfügung stehen, hängt von dem gewählten Datenbanktyp ab.
Klassenname Keine Maßnahme erforderlich. Der Klassenname wird automatisch basierend auf dem gewählten Treiber angezeigt.
DBMS-SID Geben Sie die SID der Datenbank ein.
DBMS-Host Geben Sie den Namen des Rechners ein, auf dem sich die Datenbank befindet.
DBMS-Port Geben Sie den Port ein, der für die Verbindung mit dem Server verwendet werden soll. Die Standard-Port-Nummer, die mit dem gewählten Datenbanktyp verknüpft ist, wird automatisch angezeigt.
URL Keine Maßnahme erforderlich. Der URL wird automatisch basierend auf dem gewählten Treiber angezeigt.
Benutzername Geben Sie den Anmeldenamen für die Anmeldung bei der Datenbank ein.
Benutzerkennwort Geben Sie das Kennwort für den Zugriff auf die Datenbank ein.

Gültige Werte bestehen aus einer alphanumerischen Zeichenfolge. Bindestrich (-) und Unterstrich ( _ ) werden unterstützt.
Der Wert wird verschlüsselt.
Benutzerkennwort bestätigen Geben Sie das Kennwort erneut ein.
Bekannte Eigenschaften Keine Maßnahme erforderlich. Die bekannten Eigenschaften der Datenbank werden automatisch basierend auf dem gewählten Treiber angezeigt.
Zusätzliche Eigenschaften Geben Sie bei Bedarf zusätzliche Eigenschaften an, die an den Treiber weitergereicht werden sollen.

Hinweis: Sie können die Verbindung zur Datenbank testen, indem Sie auf Verbindung testen klicken. Vor dem Start des Servers müssen Sie die erforderlichen SQL-Skripte für den RDBMS-Sicherheitsspeicher laden. Wenn Sie einen RDBMS-Sicherheitsspeicher in einer geclusterten Domain einsetzen, wird die Verwendung einer JMS-Konfiguration empfohlen (JNDI-Name und JMS-Topic).

Wenn Sie alle erforderlichen Einstellungen und Einträge vorgenommen haben, klicken Sie im Assistenten auf die Schaltfläche Weiter.

Weitere Informationen finden Sie in der Dokumentations-Library für Oracle Fusion Middleware im Oracle Technology Network.
