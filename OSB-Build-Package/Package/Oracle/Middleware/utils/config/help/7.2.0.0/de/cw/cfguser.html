Benutzername und Kennwort des Administrators konfigurieren

Hier können Sie den WebLogic-Standard-Administratoraccount für die Domain definieren. Mit diesem Account wird der Administrationsserver der Domain hochgefahren und eine Verbindung zu ihm hergestellt.

  • Name
    : Der Anmeldebenutzername für den WebLogic-Administratoraccount.

    Der Benutzername darf keine Kommas, Tabulatoren und keines der folgenden Zeichen enthalten:
    <>#|&?(){}

  • Kennwort
    : Beim Kennwort ist die Groß- und Kleinschreibung zu beachten. Es muss mindestens acht alphanumerische Zeichen enthalten, darunter mindestens eine Zahl oder eines der folgenden Zeichen:
    ! " # $ % & ' ( ) * + , - . / : ; < = > ? @ [ \ ] ^ _ ` { | } ~

Wenn Sie alle erforderlichen Einträge vorgenommen haben, klicken Sie auf die Schaltfläche Weiter.

Weitere Informationen finden Sie in der Dokumentations-Library für Oracle Fusion Middleware im Oracle Technology Network.
