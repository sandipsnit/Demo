Datenbankskripte ausführen

Eine Domainvorlage kann eine Gruppe von nach Datenbanktyp organisierten SQL-Dateien enthalten. Wenn die Domainvorlage SQL-Dateien enthält, können Sie sie während der Erstellung der WebLogic-Domain auf dem Bildschirm Datenbankskripte ausführen ausführen. Der Datenbankinhalt für die einzelnen in der WebLogic-Domain definierten Datenquellen wird mit bestehenden SQL- oder Datenbankladedateien festgelegt.

Der Datenbankinhalt für die einzelnen in Ihrer Domain definierten Datenquellen wird mit bestehenden SQL- oder Datenbankladedateien festgelegt.

  1. Wählen Sie im Bereich Verfügbare JDBC-Datenquellen die Datenquelle aus, für die Sie die Skripte ausführen wollen.

    Die ausführbaren Skripte werden im Bereich Verfügbare SQL-Dateien und Optionen für das Laden der Datenbank angezeigt.

  2. Wählen Sie aus der Liste DB-Version die Datenbankversion aus.
  3. Klicken Sie auf Skripte ausführen.

    Alle im Bereich Verfügbare SQL-Dateien und Optionen für das Laden der Datenbank angezeigten Skripte werden für die gewählte Datenquelle ausgeführt. Die Ergebnisse werden im Bereich Ergebnisse angezeigt.

    Sie können die Testdatenausgabe in einer Log-Datei festhalten, indem Sie das Kontrollkästchen Log-Datei aktivieren und den Speicherort der Log-Datei angeben.

  4. Wiederholen Sie die Schritte 1 bis 3 für jede Datenquelle, für die Sie Datenbankskripte ausführen wollen.
  5. Klicken Sie nach der Ausführung der Datenbankskripte auf Weiter.

Weitere Informationen finden Sie in der Dokumentations-Library für Oracle Fusion Middleware im Oracle Technology Network.
