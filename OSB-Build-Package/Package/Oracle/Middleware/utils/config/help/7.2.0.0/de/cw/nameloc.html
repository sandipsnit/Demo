Domainname und Speicherort angeben

Geben Sie folgende Elemente an:

  • Domainname
    : Der Name der Domain. Der Konfigurationsassistent speichert die Datei config.xml sowie alle anderen generierten Komponenten für die Domain in diesem Verzeichnis, innerhalb des angegebenen Domainverzeichnisses. Beispiel: Wenn Sie mydomain eingeben, werden die Domaindateien (standardmäßig) im Verzeichnis MW_HOME\user_projects\domains\mydomain gespeichert.

    Hinweis: Domainnamen dürfen nicht mit einer Zahl beginnen. Durch diese Einschränkung sollen mögliche Probleme mit intern generierten JDBC-Speichertabellennamen, die mit einem Buchstaben anfangen müssen, vermieden werden.

  • Domainspeicherort
    : Der Speicherort für das Domainverzeichnis, standardmäßig MW_HOME\user_projects\domains. Der Domainspeicherort kann sich an einer beliebigen Stelle im lokalen Laufwerk oder Netzwerk befinden und muss nicht unter MW_HOME liegen.

    Unter Windows müssen Sie den Laufwerksbuchstaben in diesen Pfad aufnehmen.

  • Anwendungsspeicherort
    : Dieses Feld wird nur angezeigt, wenn Sie eine Domainvorlage oder ein Produkt mit enthaltenen Anwendungen ausgewählt haben. Dies ist der Speicherort, in dem sich die Anwendungen befinden. Der Standard lautet MW_HOME\user_projects\applications.

    Unter Windows müssen Sie den Laufwerksbuchstaben in diesen Pfad aufnehmen.

Wenn Sie alle erforderlichen Einstellungen und Einträge vorgenommen haben, klicken Sie auf die Schaltfläche Weiter.
