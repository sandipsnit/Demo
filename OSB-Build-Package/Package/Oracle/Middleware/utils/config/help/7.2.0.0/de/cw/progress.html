Domain wird erstellt

Hier wird der Fortschritt der Domainerstellung angezeigt.

Wenn der Vorgang abgeschlossen ist, ist die neue (oder erweiterte) Domain bereit zum Einsatz.

Wenn Sie den Administrationsserver umgehend starten möchten, aktivieren Sie das Kontrollkästchen Admin-Server starten, und klicken Sie auf Fertig. Diese Option steht nur auf Windows-Systemen und nur bei der Erstellung einer neuen Domain zur Verfügung.

Um den Konfigurationsassistenten zu beenden, klicken Sie auf Fertig.

Weitere Informationen zu den verschiedenen Methoden, die Sie zum Starten des Administrationsservers verwenden können, finden Sie unter Server starten und stoppen in Das Hoch- und Herunterfahren von Servern für Oracle WebLogic Server verwalten.

Weitere Informationen finden Sie in der Dokumentations-Library für Oracle Fusion Middleware im Oracle Technology Network.
