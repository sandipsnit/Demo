Konfigurationszusammenfassung

Prüfen Sie die genauen Konfigurationseinstellungen der Domain, bevor Sie fortfahren.

  1. Wählen Sie im Bereich Domainzusammenfassung ein Element aus, um die dazugehörigen Details im Bereich Details zu prüfen. Sie können begrenzte Änderungen vornehmen, indem Sie auf die Schaltfläche Zurück klicken und zu den vorherigen Bildschirmen im Assistenten zurückkehren.

    Hinweis: Sie können die Art der im Bereich Domainzusammenfassung angezeigten Informationen einschränken, indem Sie aus der Liste Zusammenfassung einen Filter auswählen.

  2. Wenn Sie die Domaineinstellungen geprüft haben, klicken Sie auf Erstellen (bzw. Erweitern, wenn Sie eine bestehende Domain erweitern).

Weitere Informationen finden Sie in der Dokumentations-Library für Oracle Fusion Middleware im Oracle Technology Network.
