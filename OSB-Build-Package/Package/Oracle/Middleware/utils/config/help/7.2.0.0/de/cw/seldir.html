Verzeichnis wählen

Auf diesem Bildschirm des Konfigurationsassistenten können Sie die bestehende Domain auswählen, zu der Sie zusätzliche Anwendungen oder Services hinzufügen wollen.

Wählen Sie aus der Navigationsbaumstruktur ein gültiges Domainverzeichnis aus (ein Verzeichnis, das eine Datei config.xml im Verzeichnis config der Domain enthält). Gültige Domainverzeichnisse werden in der Navigationsbaumstruktur mit einem blauen Ordnersymbol markiert.

Wenn Sie die erforderliche Auswahl vorgenommen haben, klicken Sie auf die Schaltfläche Weiter.

Weitere Informationen finden Sie in der Dokumentations-Library für Oracle Fusion Middleware im Oracle Technology Network.
