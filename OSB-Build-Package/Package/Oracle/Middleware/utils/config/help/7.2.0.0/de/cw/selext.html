Erweiterungsquelle wählen

Entscheiden Sie, ob Sie zur Erweiterung der Domain eines der aufgelisteten Produkte auswählen oder eine Erweiterungsvorlage suchen möchten.

  1. Wählen Sie eine der folgenden Optionen:
    • Domain automatisch erweitern, damit die folgenden hinzugefügten Produkte unterstützt werden
      : Aktivieren Sie die Kontrollkästchen zu den entsprechenden anderen Produkten, die Sie in die Domain aufnehmen möchten. Die bereits in der Domain enthaltenen Produkte werden durch grau angezeigte Kontrollkästchen markiert. Wenn Sie ein Produkt wählen, das abhängig von den Ressourcen anderer Produkte in der Liste ist, werden diese Produkte automatisch mit ausgewählt.
    • Domain mit einer vorhandenen Erweiterungsvorlage erweitern
      : Geben Sie im Feld Vorlagenspeicherort den vollständigen Pfad der erforderlichen Vorlage ein, oder klicken Sie auf Durchsuchen, um zum Verzeichnis mit der Vorlage zu navigieren.
  2. Wenn Sie die erforderliche Auswahl vorgenommen haben, klicken Sie auf die Schaltfläche Weiter.

Hinweis: Wenn eines der ausgewählten Produkte Ressourcen benötigt, die von einem anderen, nicht ausgewählten Produkt bereitgestellt werden, und Sie auf Weiter klicken, wird ein Warndialogfeld angezeigt. Kehren Sie zu dem Bildschirm "Erweiterungsquelle wählen" zurück, um das Kontrollkästchen des abhängigen Produkts zu aktivieren, bevor Sie fortfahren.

Informationen zu den hier angezeigten Produktvorlagen finden Sie in Oracle Fusion Middleware Domain Template Reference.
