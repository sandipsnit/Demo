Domainquelle wählen

Wählen Sie, ob eine Domain generiert werden soll, die automatisch für die Unterstützung der ausgewählten Produkte konfiguriert ist, oder ob Sie eine auf der vorhandenen Domainvorlage basierende Domain erstellen möchten. Jedes in der Liste angezeigte Produkt wird einer einzelnen Domain oder Erweiterungsvorlage (JAR-Datei) zugeordnet, mit der die für dieses Produkt benötigten Ressourcen konfiguriert werden.

  1. Wählen Sie eine der folgenden Optionen:
    • Generieren Sie eine Domain, die automatisch konfiguriert wird, um die folgenden Produkte zu unterstützen
      : Standardmäßig ist die Option Grundlegende WebLogic Server-Domain aktiviert. Aktivieren Sie die Kontrollkästchen zu den entsprechenden anderen Produkten, die Sie in die Domain aufnehmen möchten. Wenn Sie ein Produkt wählen, das abhängig von den Ressourcen anderer Produkte in der Liste ist, werden diese Produkte automatisch mit ausgewählt.
    • Erstellen Sie diese Domain mithilfe einer bereits vorhandenen Vorlage
      : Wählen Sie diese Option, wenn Sie eine Domain basierend auf einer vorhandenen Domainvorlage erstellen möchten. Geben Sie in das Feld Vorlagenspeicherort den vollständigen Pfad zu der JAR-Datei der Vorlage ein, oder klicken Sie auf Durchsuchen, um zu dem Verzeichnis zu navigieren, in dem sich die erforderliche Vorlage befindet.
  2. Wenn Sie die erforderliche Einstellung vorgenommen haben, klicken Sie im Assistenten auf die Schaltfläche Weiter.

Hinweis: Wenn eines der ausgewählten Produkte Ressourcen benötigt, die von einem anderen, nicht ausgewählten Produkt bereitgestellt werden, und Sie auf Weiter klicken, wird ein Warndialogfeld angezeigt. Kehren Sie zu dem Bildschirm "Domainquelle wählen" zurück, um das Kontrollkästchen des abhängigen Produkts zu aktivieren, bevor Sie fortfahren.

Informationen zu den hier angezeigten Produktvorlagen finden Sie in Oracle Fusion Middleware Domain Template Reference.
