JDBC-Datenquellen testen

Hier können Sie die Verbindungen zu Datenquellen testen, die Sie auf den Bildschirmen JDBC-Datenquellen konfigurieren und Oracle RAC-Multidatenquellen konfigurieren konfiguriert haben.

Wenn Sie die Verbindungen zu diesem Zeitpunkt nicht testen möchten, wählen Sie keine Datenquellen. Klicken Sie auf Weiter, um fortzufahren.

Hinweis: Um die Datenbankverbindung zu testen, muss die Datenbank, bei der Sie sich anmelden, hochgefahren sein.

  1. Aktivieren Sie die Kontrollkästchen neben den Namen der zu testenden Datenquellen.
  2. Klicken Sie auf Verbindungen testen.

    Der Assistent testet die Konfiguration für jede Datenquelle, indem er mit den von Ihnen bei der Datenquellenkonfiguration definierten Angaben zu Treiber, Host, Port und sonstigen Informationen mit einem URL Verbindung aufzunehmen versucht.

    Das Ergebnis des Tests wird in der Spalte Status angezeigt. Details werden im Bereich Verbindungsergebnis-Log angezeigt.

  3. Nachdem Sie die Verbindungen getestet haben, klicken Sie Weiter.

Weitere Informationen finden Sie in der Dokumentations-Library für Oracle Fusion Middleware im Oracle Technology Network.
