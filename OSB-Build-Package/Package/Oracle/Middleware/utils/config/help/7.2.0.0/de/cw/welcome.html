Willkommen

Auf diesem Bildschirm des Konfigurationsassistenten wählen Sie, ob eine WebLogic-Domain erstellt oder eine bestehende Domain geändert werden soll.

  1. Wählen Sie eine der folgenden Optionen:

    • Neue WebLogic-Domain erstellen

      Um eine neue Domain zu erstellen, wählen Sie diese Option. Normalerweise wird eine neue Domain auf der Grundlage einer WebLogic Server-Basisdomain erstellt, zu der ein Administrationsserver und die Standard-Sicherheits-Realm myrealm gehören. Sie können jedoch zur Erstellung der Domain eine eigene Domainvorlage verwenden. Diese Domainvorlage muss mindestens einen Administrationsserver enthalten.

    • Vorhandene WebLogic-Domain erweitern

      Wählen Sie diese Option, wenn Sie bereits eine WebLogic-Domain haben und dieser Domain Ressourcen hinzufügen möchten.

  2. Wenn Sie die erforderliche Auswahl vorgenommen haben, klicken Sie auf die Schaltfläche Weiter.

Weitere Informationen finden Sie in der Dokumentations-Library für Oracle Fusion Middleware im Oracle Technology Network.
