Configure Key Store Credentials

This screen displays one set of Enter Keystore Password/Confirm Password fields for each product that requires a keystore password to be set. For example, if you installed Oracle Information Rights Manager, there is one set of password fields for each of the following two application keystores:

  • Oracle Information Rights Management
  • Oracle Enterprise Manager Plugin for IRM

  1. Enter and confirm the password that is needed to access the application's keystore.
  2. After entering all keystore passwords, click Next.

Note: The keystore password for each product listed on this screen is required to use these products. Note the passwords you enter and provide them to anyone who is authorized to use these products.
