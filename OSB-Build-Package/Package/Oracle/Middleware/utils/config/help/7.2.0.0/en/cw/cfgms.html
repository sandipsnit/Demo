Configure Managed Servers

Use this screen to add or delete Managed Servers. You can also change the settings for an existing Managed Server. Required fields are indicated by an asterisk (*).

  1. Review the current managed server configuration. Default values might vary based on the domain source you have selected.

    Note: The wizard provides two views: a concise tabular view of all the managed servers, and an individual view of each managed server with each server represented by a tab. To toggle the view mode between the table and tab views, click Switch Display.

  2. Add or delete managed servers, or change the settings for existing managed servers, as required for your domain. The fields marked with an asterisk (*) are mandatory fields.

    For more information about the fields in this screen, see the help page for the Configure the Administration Server screen.

  3. After making the required selections and entries, click Next.
