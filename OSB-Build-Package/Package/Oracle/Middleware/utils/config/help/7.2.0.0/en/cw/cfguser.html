Configure Administrator User Name and Password

Use this screen to define the default WebLogic Administrator account for the domain. This account is used to boot and connect to the domain’s Administration Server.

  • Name
    The login username for the WebLogic Administrator account.

    The user name must not contain commas, tabs, and any of the following characters:
    < > # | & ? ( ) { }

  • User Password
    The password is case sensitive. It must contain at least eight alphanumeric characters, with at least one numeral or one of the following characters:
    ! " # $ % & ' ( ) * + , - . / : ; < = > ? @ [ \ ] ^ _ ` { | } ~

After making the required entries, click Next.
