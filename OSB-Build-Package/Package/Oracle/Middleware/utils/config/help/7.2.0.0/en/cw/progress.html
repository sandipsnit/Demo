Creating Domain

This screen shows the progress of the domain creation.

When the process is complete, the new (or extended) domain is ready for use.

If you want to start the admin server immediately, select the Start Admin Server check box, and click Done. This option is available only for Windows systems, and if you are creating a new domain.

To exit the Configuration Wizard, click Done.

For more information on the various methods you can use to start the Administration Server, see Starting and Stopping Servers in Managing Server Startup and Shutdown for Oracle WebLogic Server.
