Configuration Summary

IReview the detailed configuration settings of your domain before continuing.

  1. Select an item in the Domain Summary pane to review the associated details in the Details pane. You can make limited changes by clicking the Previous button and returning to the previous screens in the wizard.

    Note: You can limit the type of information displayed in the Domain Summary pane by selecting a filter from the Summary View list.

  2. After reviewing the domain settings, click Create (or Extend if you are extending an existing domain).

For more information, see the Oracle Fusion Middleware documentation library on the Oracle Technology Network.
