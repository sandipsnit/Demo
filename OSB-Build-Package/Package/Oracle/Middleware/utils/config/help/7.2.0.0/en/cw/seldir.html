Select Directory

In this screen of the Configuration Wizard, you select the existing domain that you want to update with additional applications or services.

Use the navigation tree to select a valid domain directory (a directory that contains a config.xml file in the config directory of the domain). Valid domain directories are indicated by a blue folder icon in the navigation tree.

After making the required selection, click Next.
