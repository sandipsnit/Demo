Select Extension Source

IChoose whether to extend the domain by selecting one of the listed products, or by browsing to an extension template.

  1. Select one of the following options:
    • Extend my domain automatically to support the following added products
      Select the check boxes that correspond to the other products that you want to include in the domain. The products already included in your domain are indicated by grayed-out check boxes. If you select a product that is dependent on resources provided by another product or products in the list, those products are also selected automatically.
    • Extend my domain using an existing extension template
      Enter the full path of the required template in the Template location field, or click Browse to navigate to the directory containing the template.
  2. After making the required selection, click Next.

Note: When you click Next, if any of the selected products require resources provided by another product that is not selected, a warning dialog box is displayed. Return to the Select Extension Source screen to select the dependency product before proceeding.

For information about the product templates displayed on this screen, see Oracle Fusion Middleware Domain Template Reference.
