Select Domain Source

Choose whether to generate a domain that is configured automatically to support selected products, or to create a domain based on an existing domain template. Each product in the displayed list is associated with a single domain or extension template (a JAR file) which configures the domain resources needed for that product.

  1. Select one of the following options:
    • Generate a domain configured automatically to support the following products
      By default, the Basic WebLogic Server Domain option is selected. Select the check boxes that correspond to the other products that you want to include in the domain. If you select a product that is dependent on resources provided by another product or products in the list, those products are also selected automatically.
    • Base this domain on an existing template
      Select this option if you want to create a domain by using an existing domain template. Enter the full path to the template JAR file in the Template location field, or click Browse to navigate to the directory containing the required template.
  2. After making the required selection, click the Next button in the wizard.

Note: When you click Next, if any of the selected products require resources provided by another product that is not selected, a warning dialog box is displayed. Return to the Select Domain Source screen to select the dependency product before proceeding.

For information about the product templates displayed on this screen, see Oracle Fusion Middleware Domain Template Reference.
