Asignar Servidores a Máquinas

Utilice esta pantalla para asignar instancias de WebLogic Server a cada una de las máquinas que ha definido.

  1. En el panel Máquina, seleccione la máquina de Windows o UNIX a la que desee asignar una instancia de WebLogic Server.
  2. Asigne las instancias de WebLogic Server a la máquina seleccionada de una de las siguientes formas:
    • Haga clic dos veces en la instancia de WebLogic Server en el panel Servidor.
    • Seleccione la instancia de WebLogic Server correspondiente en el panel Servidor y haga clic en el botón de flecha derecha ubicado entre los paneles Servidor y Máquina.
    • Haga clic mientras pulsa Mayús para seleccionar varios servidores en el panel Servidor; a continuación, haga clic en el botón de flecha derecha ubicado entre los paneles Servidor y Máquina.

    El nombre de la instancia de WebLogic Server asignada se elimina del panel Servidor y se agrega debajo del nombre de la máquina de destino en el panel Máquina.

  3. Repita los pasos 1 y 2 para cada instancia de WebLogic Server que desee asignar a una máquina.
  4. Revise las asignaciones de máquina.

    Si es necesario, puede eliminar una instancia de WebLogic Server de una máquina de una de las siguientes formas:

    • Haga clic dos veces en el nombre de la instancia de WebLogic Server correspondiente en el panel Máquina.
    • Seleccione la instancia de WebLogic Server correspondiente en el panel Máquina y haga clic en el botón de flecha izquierda ubicado entre los paneles Servidor y Máquina.

    El nombre de la instancia de WebLogic Server se elimina del panel Máquina y se restaura en el panel Servidor.

  5. Después de asignar instancias de WebLogic Server a máquinas, haga clic en Siguiente.
