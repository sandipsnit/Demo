Seleccionar Configuración Avanzada

Seleccione todas las categorías (si hay alguna) en las cuales desea definir la configuración avanzada. Por cada categoría que seleccione, se muestra la pantalla de configuración correspondiente para que pueda definirla. Si no selecciona ningún elemento en esta pantalla, a continuación aparece la pantalla Resumen de Configuración.

Nota: Las categorías que se muestran en esta pantalla dependen de los recursos definidos en las plantillas que ha seleccionado para el dominio.

Nota: Las opciones Servidor de Administración y Almacén de Seguridad de RDBMS no están disponibles cuando se amplía un dominio.

  • Servidor de Administración
    Seleccione esta opción para modificar la configuración del servidor de administración (por ejemplo, el nombre del servidor, la dirección de recepción, el puerto de recepción y la configuración de SSL).
  • Destino Distribuido de JMS
    Por defecto, los destinos distribuidos de JMS se convierten automáticamente en destinos distribuidos ponderados (WDD). Seleccione esta opción para cambiar un destino distribuido de JMS por un destino distribuido unificado (UDD).
  • Servidores Gestionados, Clusters y Máquinas
    Seleccione esta opción para:
    • Agregar servidores gestionados, clusters o máquinas al dominio
    • Suprimir un servidor gestionado, cluster o máquina existente
    • Agregar servidores gestionados a un cluster existente
    • Modificar la configuración de un servidor gestionado, cluster o máquina existente
  • Despliegues y Servicios
    Seleccione esta opción para personalizar la forma de dirigir los servicios y despliegues de aplicaciones a servidores y clusters.
  • Almacenes de Archivos de JMS
    Selecciones esta opción para cambiar la configuración de los almacenes de archivos de JMS. Puede cambiar el nombre, el directorio y la política de escritura síncrona de cada almacén de archivos.
  • Almacén de Seguridad de RDBMS
    Seleccione esta opción para configurar un sistema de gestión de base de datos relacional (RDBMS) externo como almacén de datos para varios proveedores de seguridad.

Para cada opción de configuración avanzada que seleccione en esta pantalla, el asistente muestra más adelante pantallas adicionales en las que puede definir los valores de configuración avanzada correspondientes.

Active las casillas de control requeridas según la configuración avanzada que desee definir y, a continuación, haga clic en Siguiente.
