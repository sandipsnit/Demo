Configurar Credenciales del Almacén de Claves

En esta pantalla se muestra un juego de campos Introducir Contraseña de Almacén de Claves/Confirmar Contraseña por cada producto que requiere que se defina una contraseña de almacén de claves. Por ejemplo, si ha instalado Oracle Information Rights Management, hay un juego de campos de contraseña por cada uno de los dos almacenes de claves de aplicación siguientes:

  • Oracle Information Rights Management
  • Plugin de Oracle Enterprise Manager para IRM

  1. Introduzca y confirme la contraseña necesaria para acceder al almacén de claves de la aplicación.
  2. Después de introducir todas las contraseñas del almacén de claves, haga clic en Siguiente.

Nota: La contraseña del almacén de claves de cada producto que se indica en esta pantalla es necesaria para utilizar estos productos. Anote las contraseñas que introduzca y proporcióneselas a las personas autorizadas para utilizar estos productos.

Para obtener más información, consulte la biblioteca de documentación de Oracle Fusion Middleware en Oracle Technology Network.
