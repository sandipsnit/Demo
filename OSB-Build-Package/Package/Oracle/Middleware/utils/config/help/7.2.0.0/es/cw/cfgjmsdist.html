Configurar Almacenes de Archivos de JMS

Por defecto, por motivos de compatibilidad con versiones anteriores, los destinos distribuidos de JMS se convierten automáticamente en destinos distribuidos ponderados (WDD). Para cambiar un destino distribuido de JMS para utilizar el comportamiento del destino distribuido uniforme (UDD), seleccione UDD en la lista desplegable. Oracle recomienda utilizar el valor de UDD.

Al cambiar el comportamiento de los recursos del sistema de JMS, el valor sólo se aplica a la sesión actual, a menos que el recurso del sistema de JMS esté asignado a un cluster. Cuando el recurso del sistema de JMS está asignado a un cluster, el valor se convierte en fijo y no se puede cambiar.

Para obtener más información acerca de UDD y WDD, consulte Tipos de Destinos Distribuidos en Programación de JMS para Oracle WebLogic Server.
