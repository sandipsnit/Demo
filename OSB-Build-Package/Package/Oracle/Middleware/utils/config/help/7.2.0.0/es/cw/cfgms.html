Configurar Servidores Gestionados

Utilice esta pantalla para agregar o suprimir servidores gestionados. También puede cambiar la configuración de un servidor gestionado existente. Los campos obligatorios están marcados con un asterisco (*).

  1. Revise la configuración actual de los servidores gestionados. Los valores por defecto pueden variar según el origen de dominio seleccionado.

    Nota: El asistente proporciona dos vistas (una vista tabular concisa de todos los servidores gestionados y una vista individual de cada servidor gestionado con cada servidor representado por un separador). Para alternar entre las vistas de tabla y separador, haga clic en Cambiar Visualización.

  2. Agregue o suprima servidores gestionados, o cambie la configuración de los servidores gestionados existentes, según sea necesario para el dominio. Los campos marcados con un asterisco (*) son obligatorios.

    Para obtener más información acerca de los campos de esta pantalla, consulte la página de ayuda de la pantalla Configurar Servidor de Administración.

  3. Después de realizar las selecciones y entradas requeridas, haga clic en Siguiente.

Para obtener más información, consulte la biblioteca de documentación de Oracle Fusion Middleware en Oracle Technology Network.
