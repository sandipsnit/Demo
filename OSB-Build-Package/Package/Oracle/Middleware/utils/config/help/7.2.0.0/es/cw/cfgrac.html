Configurar Orígenes de Datos Múltiples de RAC

Utilice esta pantalla para configurar los orígenes de datos (que se incluyen en el dominio) como orígenes de datos de Oracle RAC.

Los orígenes de datos para los que se han configurado los orígenes de datos de RAC en la pantalla anterior del asistente (Configurar Orígenes de Datos JDBC) se muestran en la mitad inferior de la pantalla.

  1. Para seleccionar el origen de datos para el que desea especificar la configuración, active la casilla de control junto al nombre del origen de datos.

    Nota: Si se seleccionan varios orígenes de datos, es posible que se muestre el texto “Varía entre los orígenes de datos” en determinados campos, lo que indica que los valores actuales de estos campos no son los mismos en los orígenes de datos seleccionados. Si continúa y cambia los valores de estos campos, los valores nuevos se aplican de manera uniforme a los orígenes de datos seleccionados.

  2. Especifique la configuración de los orígenes de datos de RAC.

    En la siguiente tabla se describen los campos de esta pantalla.

    Campo Descripción
    Controlador Seleccione el controlador JDBC que desee utilizar para conectarse a la base de datos.
    Nombre del Servicio Introduzca un nombre de servicio de base de datos de Oracle RAC.
    Usuario Introduzca el nombre de conexión para conectarse a la base de datos.
    Contraseña Introduzca la contraseña de conexión para el usuario especificado.
    Nombre del Host Introduzca el nombre del servidor que aloja las instancias de base de datos de Oracle RAC.
    Nombre de la Instancia Introduzca el nombre de cada instancia de base de datos de Oracle.
    Puerto Introduzca el número de puerto que se va a utilizar para establecer la conexión con el servidor que aloja la instancia de base de datos.

    Nota: Debe especificar el nombre de host, el nombre de instancia y el número de puerto de una instancia de base de datos como mínimo.

    Para agregar una instancia de base de datos, haga clic en Agregar y, a continuación, especifique el nombre de host, el nombre de instancia y el número de puerto.

    Los valores especificados se muestran en las columnas correspondientes de la lista de orígenes de datos para los orígenes de datos seleccionados.

  3. Después de realizar las selecciones y entradas requeridas, haga clic en Siguiente para probar las conexiones a los orígenes de datos.
