Configurar Base de Datos del Almacén de Seguridad de RDBMS

Utilice esta pantalla para definir la configuración del almacén de seguridad de RDBMS.

En la siguiente tabla se describen los campos de esta pantalla.

Campo Descripción
Tipo de Base de Datos En la lista desplegable, seleccione el tipo de base de datos que desee utilizar como almacén de seguridad de RDBMS.
Controlador Seleccione el controlador que desee utilizar para la base de datos. La lista de controladores disponibles puede variar según el tipo de base de datos seleccionado.
Nombre de Clase No se requiere ninguna acción. El nombre de clase se muestra automáticamente según el controlador seleccionado.
SID de DBMS Introduzca el SID de la base de datos.
Host de DBMS Introduzca el nombre de la máquina que aloja la base de datos.
Puerto de DBMS Introduzca el puerto que se va a utilizar para establecer la conexión con el servidor. El número de puerto por defecto asociado al tipo de base de datos seleccionado se muestra automáticamente.
URL No se requiere ninguna acción. La dirección URL se muestra automáticamente según el controlador seleccionado.
Usuario Introduzca el nombre de conexión para conectarse a la base de datos.
Contraseña de Usuario Introduzca la contraseña para acceder a la base de datos.

Los valores válidos se componen de una cadena de caracteres alfanuméricos. Se admiten los caracteres de guión (-) y subrayado ( _ ).
El valor está cifrado.
Confirmar Contraseña de Usuario Vuelva a introducir la contraseña.
Propiedades Conocidas No se requiere ninguna acción. Las propiedades conocidas de las base de datos se muestran automáticamente según el controlador seleccionado.
Propiedades Adicionales Introduzca la propiedades adicionales (si las hay) que se van a pasar al controlador.

Nota: Para probar la conexión con la base de datos, haga clic en Probar Conexión. Antes de iniciar el servidor, debe cargar los scripts de SQL necesarios para el almacén de seguridad de RDBMS. Si utiliza un almacén de seguridad de RDBMS de un dominio en cluster, se recomienda utilizarlo con la configuración de JMS (nombre y tema de JNDI).

Después de realizar las selecciones y entradas requeridas, haga clic en el botón Siguiente del asistente.
