Configurar Nombre de Usuario y Contraseña del Administrador

Utilice esta pantalla para definir la cuenta de administrador de WebLogic para el dominio. Esta cuenta sirve para iniciar y conectarse al servidor de administración del dominio.

  • Nombre
    Nombre de usuario de inicio de sesión de la cuenta de administrador de WebLogic.

    El nombre de usuario no debe contener comas, separadores y ninguno de los siguientes caracteres:
    <>#|&?(){}.

  • Contraseña de Usuario
    La contraseña es sensible a mayúsculas/minúsculas. Debe contener como mínimo ocho caracteres alfanuméricos y un número o uno de los siguientes caracteres:
    ! " # $ % & ' ( ) * + , - . / : ; < = > ? @ [ \ ] ^ _ ` { | } ~

Después realizar las entradas requeridas, haga clic en Siguiente.
