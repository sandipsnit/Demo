Creando Dominio

Esta pantalla muestra el progreso de creación del dominio.

Una vez completado el proceso, el dominio nuevo (o ampliado) ya está listo para su uso.

Si desea iniciar el servidor de administración de inmediato, active la casilla de control Iniciar Servidor de Administración y haga clic en Done. Esta opción está disponible solo para los sistemas Windows y si va a crear un dominio nuevo.

Para salir del Asistente de Configuración, haga clic en Listo.

Para obtener más información acerca de los diversos métodos que se pueden utilizar para iniciar el servidor de administración, consulte Inicio y Parada de Servidores en Gestión del Inicio y Cierre de Servidores para Oracle WebLogic Server.

Para obtener más información, consulte la biblioteca de documentación de Oracle Fusion Middleware en Oracle Technology Network.
