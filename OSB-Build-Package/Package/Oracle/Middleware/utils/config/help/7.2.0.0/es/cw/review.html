Resumen de Configuración

Revise la configuración detallada del dominio antes de continuar.

  1. Seleccione un elemento en el panel Resumen de Dominio para revisar los detalles relacionados en el panel Detalles. Para realizar cambios limitados, haga clic en el botón Anterior y vuelva a las pantallas anteriores del asistente.

    Nota: Para limitar la información mostrada en el panel Resumen de Dominio, seleccione un filtro de la lista Vista de Resumen.

  2. Después de revisar la configuración del dominio, haga clic en Crear (o Ampliar si va ampliar un dominio).

Para obtener más información, consulte la biblioteca de documentación de Oracle Fusion Middleware en Oracle Technology Network.
