Seleccionar Directorio

En esta pantalla del Asistente de Configuración, puede seleccionar el dominio existente que desee actualizar con aplicaciones o servicios adicionales.

Utilice el árbol de navegación para seleccionar un directorio de dominio válido (un directorio que contenga un archivo config.xml en el directorio config del dominio). Los directorios de dominio válidos están marcados con un icono de carpeta azul en el árbol de navegación.

Después de realizar las selecciones requeridas, haga clic en Siguiente.

Para obtener más información, consulte la biblioteca de documentación de Oracle Fusion Middleware en Oracle Technology Network.
