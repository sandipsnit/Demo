Seleccionar Origen de Dominio

Elija si desea ampliar el dominio seleccionando uno de los productos indicados o desplazándose a una plantilla de ampliación.

  1. Seleccione una de las opciones siguientes:
    • Ampliar mi dominio automáticamente para soportar los siguientes productos agregados
      Active las casillas de control correspondientes a otros productos que desea incluir en el dominio. Los productos que ya están incluidos están marcados con casillas de control atenuadas. Si selecciona un producto que depende de recursos que proporciona otro producto o productos de la lista, esos productos también se seleccionan automáticamente.
    • Ampliar mi dominio con una plantilla de ampliación existente
      Introduzca la ruta de acceso completa de la plantilla requerida en el campo Ubicación de Plantilla o haga clic en Examinar para navegar hasta el directorio que contiene la plantilla.
  2. Después de realizar las selecciones requeridas, haga clic en Siguiente.

Nota: Al hacer clic en Siguiente, si alguno de los productos seleccionados necesita recursos de otro producto que no está seleccionado, aparece un cuadro de diálogo de advertencia. Vuelva a la pantalla Seleccionar Origen de Ampliación para seleccionar el producto de dependencia antes de continuar.

Para obtener información acerca de las plantillas de producto que aparecen en esta pantalla, consulte Referencia de Plantillas de Dominio de Oracle Fusion Middleware.
