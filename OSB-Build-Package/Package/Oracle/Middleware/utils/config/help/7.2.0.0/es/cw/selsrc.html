Seleccionar Origen de Dominio

Seleccione si va a generar un dominio que se configura automáticamente para soportar los productos seleccionados o si va a crear un dominio basado en una plantilla de dominio existente. Cada producto de la lista mostrada está asociado a un solo dominio o a una plantilla de ampliación (un archivo JAR) que configura los recursos de dominio necesarios para ese producto.

  1. Seleccione una de las opciones siguientes:
    • Generar un dominio configurado automáticamente para soportar los siguientes productos
      Por defecto, está seleccionada la opción Dominio Básico de WebLogic Server . Active las casillas de control correspondientes a los demás productos que desea incluir en el dominio. Si selecciona un producto que depende de los recursos que proporciona otro producto o productos de la lista, esos productos también se seleccionan automáticamente.
    • Basar este dominio en una plantilla existente
      Seleccione esta opción si desea crear un dominio mediante una plantilla de dominio existente. Introduzca la ruta de acceso completa al archivo JAR de plantilla en el campo Ubicación de Plantilla o haga clic en Examinar para navegar hasta el directorio que contiene la plantilla requerida.
  2. Después de realizar la selección requerida, haga clic en el botón Siguiente del asistente.

Nota: Cuando haga clic en Siguiente, si alguno de los productos seleccionados necesita recursos que proporciona otro producto que no está seleccionado, aparece un cuadro de diálogo de advertencia. Vuelva a la pantalla Seleccionar Origen de Dominio para seleccionar el producto de dependencia antes de continuar.

Para obtener información acerca de las plantillas de producto que aparecen en esta pantalla, consulte Referencia de Plantillas de Dominio de Oracle Fusion Middleware.
