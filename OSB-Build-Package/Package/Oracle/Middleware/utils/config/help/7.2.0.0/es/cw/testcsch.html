Probar Esquema de Componentes

En esta pantalla del Asistente de Configuración, puede probar las configuraciones especificadas para el esquema de componentes en la pantalla anterior.

  1. Active las casillas de control junto a los nombres de los esquemas que desee probar.
  2. Haga clic en Probar Conexiones.

    El asistente prueba la configuración de cada esquema mediante un intento de conexión con una dirección URL creada con el controlador, host, puerto y demás información especificada durante la configuración del esquema.

    El resultado de la prueba se indica en la columna Estado. Los detalles se muestran en el panel Log de Resultados de Conexión.

  3. Después de probar las conexiones, haga clic en Siguiente.

Para obtener más información, consulte la biblioteca de documentación de Oracle Fusion Middleware en Oracle Technology Network.
