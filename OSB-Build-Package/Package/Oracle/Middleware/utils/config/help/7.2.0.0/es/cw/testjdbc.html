Probar Orígenes de Datos JDBC

Utilice esta pantalla para probar las conexiones del origen de datos que ha configurado en las pantallas Configurar Orígenes de Datos JDBC y Configurar Orígenes de Datos Múltiples de RAC.

Si no desea probar las conexiones en este momento, no seleccione ningún origen de datos. Haga clic en Siguiente para continuar.

Nota: Para probar las conexiones de base de datos, la base de datos a la que se va a conectar debe estar en ejecución.

  1. Active las casillas de control junto a los nombres de los orígenes de datos que desee probar.
  2. Haga clic en Probar Conexiones.

    El asistente prueba la configuración de cada origen de datos mediante un intento de conexión con una dirección URL creada con el controlador, host, puerto y demás información especificada durante la configuración del origen de datos.

    El resultado de la prueba se indica en la columna Estado. Los detalles se muestran en el panel Log de Resultados de Conexión.

  3. Después de probar las conexiones, haga clic en Siguiente.

Para obtener más información, consulte la biblioteca de documentación de Oracle Fusion Middleware en Oracle Technology Network.
