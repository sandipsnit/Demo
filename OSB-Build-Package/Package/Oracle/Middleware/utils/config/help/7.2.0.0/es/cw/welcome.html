Bienvenido

En esta pantalla del Asistente de Configuración, puede decidir si desea crear un dominio de WebLogic o modificar un dominio existente.

  1. Seleccione una de las opciones siguientes:

    • Crear Nuevo Dominio de WebLogic

      Seleccione esta opción si desea crear un dominio nuevo. Normalmente, el nuevo dominio se basará en el dominio básico de WebLogic Server, que incluye un servidor de administración y el dominio de seguridad por defecto myrealm. Sin embargo, puede elegir su propia plantilla de dominio para crear el dominio; esta plantilla debe contener al menos un servidor de administración.

    • Ampliar Domnio de WebLogix Existente

      Seleccione esta opción si ya tiene un dominio de WebLogic y desea agregarle recursos.

  2. Después de realizar las selecciones requeridas, haga clic en Siguiente.

Para obtener más información, consulte la biblioteca de documentación de Oracle Fusion Middleware en Oracle Technology Network.
