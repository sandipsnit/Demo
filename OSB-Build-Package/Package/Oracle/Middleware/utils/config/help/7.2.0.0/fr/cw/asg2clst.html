Affecter des serveurs aux clusters

Utilisez cet écran pour affecter des serveurs gérés aux clusters. Seuls les serveurs gérés sont affichés dans la zone de liste Serveur. Le serveur d'administration n'apparaît pas car il ne peut pas être affecté à un cluster.

  1. Dans le panneau Cluster, sélectionnez le cluster auquel affecter un serveur géré.
  2. Pour affecter des serveurs gérés au cluster sélectionné, procédez de l'une des façons suivantes :
    • Cliquez deux fois sur le nom du serveur géré dans le panneau Serveur.
    • Sélectionnez le serveur géré dans le panneau Serveur et cliquez sur la flèche droite située entre les panneaux Serveur et Cluster.
    • Pour sélectionner plusieurs serveurs gérés à la fois dans le panneau Serveur, appuyez sur la touche Maj tout en effectuant votre sélection ; cliquez ensuite sur la flèche droite située entre les panneaux Serveur et Cluster.

    Le nom du serveur géré affecté disparaît du panneau Serveur et est ajouté sous le nom du cluster cible dans le panneau Cluster.

    Remarque : seuls les serveurs gérés sont répertoriés dans le panneau Serveur. Le serveur d'administration n'apparaît pas car il ne peut pas être affecté à un cluster.

  3. Répétez les étapes 1 et 2 pour chaque serveur géré à affecter à un cluster.
  4. Passez en revue les affectations de cluster.

    Si nécessaire, vous pouvez enlever un serveur géré d'un cluster de l'une des façons suivantes :

    • Cliquez deux fois sur le nom du serveur géré dans le panneau Cluster.
    • Sélectionnez le serveur géré dans le panneau Cluster et cliquez sur la flèche gauche située entre les panneaux Serveur et Cluster.

    Le nom du serveur géré disparaît du panneau Cluster et apparaît dans le panneau Serveur.

  5. Une fois les serveurs gérés affectés aux clusters, cliquez sur le bouton Suivant de l'assistant.

Pour plus d'informations, reportez-vous à la bibliothèque de documentation Oracle Fusion Middleware sur Oracle Technology Network.
