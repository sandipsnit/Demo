Affecter des serveurs à des ordinateurs

Utilisez cet écran pour affecter des instances WebLogic Server à chacun des ordinateurs que vous avez définis.

  1. Dans le panneau Ordinateur, sélectionnez l'ordinateur Windows ou UNIX auquel affecter une instance WebLogic Server.
  2. Pour affecter des instances WebLogic Server à l'ordinateur sélectionné, procédez de l'une des façons suivantes :
    • Cliquez deux fois sur l'instance WebLogic Server dans le panneau Serveur.
    • Sélectionnez l'instance WebLogic Server appropriée dans le panneau Serveur et cliquez sur la flèche droite située entre les panneaux Serveur et Ordinateur.
    • Pour sélectionner plusieurs serveurs à la fois dans le panneau Serveur, appuyez sur la touche Maj tout en effectuant votre sélection ; cliquez ensuite sur la flèche droite située entre les panneaux Serveur et Ordinateur.

    Le nom de l'instance WebLogic Server affectée disparaît du panneau Serveur et est ajouté sous le nom de l'ordinateur cible dans le panneau Ordinateur.

  3. Répétez les étapes 1 et 2 pour chaque instance WebLogic Server à affecter à un ordinateur.
  4. Passez en revue les affectations d'ordinateur.

    Si nécessaire, vous pouvez enlever une instance WebLogic Server d'un ordinateur de l'une des façons suivantes :

    • Cliquez deux fois sur le nom de l'instance WebLogic Server appropriée dans le panneau Ordinateur.
    • Sélectionnez l'instance WebLogic Server appropriée dans le panneau Ordinateur et cliquez sur la flèche gauche située entre les panneaux Serveur et Ordinateur.

    Le nom de l'instance WebLogic Server disparaît du panneau Ordinateur et apparaît dans le panneau Serveur.

  5. Une fois les instances WebLogic Server affectées aux ordinateurs, cliquez sur Suivant.
