Sélectionner la configuration avancée

Sélectionnez toutes les catégories (si elles existent) pour lesquelles vous souhaitez effectuer une configuration avancée. Pour chaque catégorie que vous sélectionnez, l'écran de configuration approprié est affiché, afin que vous puissiez procéder à cette configuration. Si vous ne sélectionnez aucun élément sur cet écran, l'écran Récapitulatif de la configuration apparaît ensuite.

Remarque : les catégories répertoriées sur cet écran dépendent des ressources définies dans les modèles que vous avez sélectionnés pour le domaine.

Remarque : les options Serveur d'administration et Banque de sécurité SGBDR ne sont pas disponibles lorsque vous étendez un domaine.

  • Serveur d'administration
    Sélectionnez cette option pour modifier les paramètres du serveur d'administration, tels que son nom, ses adresse et port d'écoute, et ses paramètres SSL.
  • Destination distribuée JMS
    Par défaut, les destinations distribuées JMS sont converties automatiquement en destinations distribuées pondérées. Sélectionnez cette option pour changer une destination distribuée JMS en destination distribuée unifiée.
  • Serveurs gérés, clusters et ordinateurs
    Sélectionnez cette option pour effectuer les opérations suivantes :
    • Ajouter des serveurs gérés, des clusters ou des ordinateurs au domaine
    • Supprimer un serveur géré, un cluster ou un ordinateur existant
    • Ajouter des serveurs gérés à un cluster existant
    • Modifier les paramètres d'un serveur géré, d'un cluster ou d'un ordinateur existant
  • Déploiements et services
    Sélectionnez cette option pour personnaliser la manière dont les services et déploiements d'application sont ciblés sur les serveurs et les clusters.
  • Emplacements de stockage des fichiers JMS
    Sélectionnez cette option pour modifier les paramètres des emplacements de stockage de vos fichiers JMS. Vous pouvez modifier le nom, le répertoire et la stratégie d'écriture synchrone de chaque emplacement de stockage.
  • Banque de sécurité SGBDR
    Sélectionnez cette option pour configurer un système de gestion de base de données relationnelle (SGBDR) en tant que banque de données pour plusieurs fournisseurs de sécurité.

Pour chaque option de configuration avancée que vous sélectionnez dans cet écran, l'assistant affiche ultérieurement d'autres écrans dans lesquels vous pouvez définir les paramètres de configuration avancée appropriés.

Cochez les cases requises selon la configuration avancée à effectuer, puis cliquez sur Suivant.
