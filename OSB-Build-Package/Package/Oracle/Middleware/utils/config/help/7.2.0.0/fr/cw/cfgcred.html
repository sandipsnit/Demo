Configurer les informations d'identification et de connexion du fichier de clés

Cet écran affiche un ensemble de champs Saisir le mot de passe du fichier de clés/Confirmer le mot de passe pour chaque produit exigeant la définition d'un mot de passe de fichier de clés. Par exemple, si vous avez installé Oracle Information Rights Manager, il existe un ensemble de champs de mot de passe pour chacun des deux fichiers de clés d'application suivants :

  • Oracle Information Rights Management
  • Oracle Enterprise Manager Plugin for IRM

  1. Saisissez et confirmez le mot de passe qui permet d'accéder au fichier de clés de l'application.
  2. Une fois tous les mots de passe de fichier de clés saisis, cliquez sur Suivant.

Remarque : vous devez fournir le mot de passe de fichier de clés de tous les produits répertoriés sur cet écran pour utiliser ces derniers. Relevez les mots de passe entrés et communiquez-les à toute personne autorisée à se servir des produits.

Pour plus d'informations, reportez-vous à la bibliothèque de documentation Oracle Fusion Middleware sur Oracle Technology Network.
