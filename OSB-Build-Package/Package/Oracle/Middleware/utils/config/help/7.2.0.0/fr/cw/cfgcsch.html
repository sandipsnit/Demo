Configurer le schéma de composant JDBC

Pour certains composants Fusion Middleware (par exemple, SOA et Web Center Spaces), les sources de données JDBC peuvent être définies comme éléments du schéma de base de données du composant, chargés (lors de l'installation) via l'utilitaire de création de référentiel (RCU).

Lorsque vous créez un domaine WebLogic pour ces composants à l'aide de l'assistant Configuration, vous pouvez configurer les paramètres de schéma de composant JDBC, tels que le pilote de base de données, le propriétaire de schéma, le mot de passe, etc.

Les schémas de composant JDBC associés aux produits pour lesquels vous créez le domaine (par exemple, SOA) sont répertoriés dans la partie inférieure de l'écran.

Remarque : pour configurer des schémas en tant que sources de données RAC, cochez les cases situées en regard du nom des schémas requis, puis cochez la case Configurer les schémas de composant sélectionnés en tant que schéma de sources de données multiples RAC dans le panneau suivant. Cliquez enfin sur le bouton Suivant.

  1. Pour sélectionner les schémas pour lesquels indiquer des paramètres de source de données, cochez la case située en regard de leur nom.

    Remarque : si vous sélectionnez plusieurs schémas de composant, le texte "Varie selon les schémas de composant" peut apparaître dans certains champs, indiquant que les valeurs en cours de ces champs ne sont pas identiques dans les schémas de composant sélectionnés. Si vous continuez et modifiez les valeurs de ces champs, les nouvelles valeurs sont appliquées de façon uniforme dans les schémas de composant sélectionnés.

    Remarque : les valeurs par défaut des paramètres de schéma de composant tels que Fournisseur, Pilote, Nom d'hôte et Numéro de port dépendent des valeurs indiquées dans les modèles d'application.

  2. Passez en revue les paramètres de configuration en cours et modifiez-les si nécessaire.

    Remarque : les valeurs par défaut des paramètres de schéma tels que Fournisseur, Pilote, Nom d'hôte et Numéro de port dépendent des valeurs indiquées dans les modèles d'application.

    Champ Description
    Fournisseur Sélectionnez le fournisseur de base de données.
    Pilote Sélectionnez le pilote JDBC à utiliser pour la connexion à la base de données.

    La liste comprend les pilotes JDBC correspondant au fournisseur de base de données sélectionné.
    Propriétaire de schéma Entrez le nom de connexion à la base de données.
    Mot de passe du schéma Entrez le mot de passe de connexion du propriétaire de schéma indiqué.
    SGBD/Service Entrez un nom SGBD de base de données ou un nom de service si vous avez sélectionné un pilote de type service.
    Nom d'hôte Entrez le nom du serveur hébergeant la base de données.
    Port Entrez le numéro de port à utiliser pour la connexion au serveur qui héberge la base de données.

    Les valeurs que vous indiquez sont affichées dans les colonnes appropriées de la liste de schémas, pour les schémas sélectionnés.

  3. Après avoir effectué les sélections et entrées requises, cliquez sur le bouton Suivant de l'assistant pour tester les schémas de composant.
