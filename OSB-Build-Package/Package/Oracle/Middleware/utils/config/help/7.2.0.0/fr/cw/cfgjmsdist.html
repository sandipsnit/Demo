Configurer des emplacements de stockage des fichiers JMS

Par défaut, à des fins de compatibilité descendante, les destinations distribuées JMS sont converties automatiquement en destinations distribuées pondérées. Pour modifier une destination distribuée JMS afin qu'elle se comporte comme une destination distribuée uniformément, sélectionnez UDD dans la liste déroulante. Oracle recommande d'employer le paramètre UDD.

Lorsque vous modifiez le comportement d'une ressource système JMS, le paramètre s'applique uniquement à la session en cours, sauf si cette ressource est affectée à un cluster. Une fois qu'elle est affectée à un cluster, ce paramètre devient fixe et et ne peut plus être modifié.

Pour plus d'informations sur les destinations distribuées uniformément et les destinations distribuées pondérées, reportez-vous à Types de destination distribuée dans Programmation de JMS pour Oracle WebLogic Server.
