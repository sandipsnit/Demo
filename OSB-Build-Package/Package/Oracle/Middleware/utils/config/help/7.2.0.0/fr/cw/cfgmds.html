Configurer le schéma de composant de source de données multiples RAC

Utilisez cet écran pour configurer les schémas de composant inclus dans le domaine WebLogic en tant que sources de données multiples Oracle RAC.

Les schémas de composant pour lesquels vous avez choisi de configurer des sources de données RAC dans l'écran précédent de l'assistant (Configurer le schéma de composant JDBC) sont répertoriés dans la partie inférieure de l'écran.

  1. Dans la liste de schémas figurant dans la partie inférieure de l'écran, sélectionnez le schéma à configurer en tant que source de données RAC. Pour ce faire, cochez la case figurant en regard de son nom.

    Remarque : si vous sélectionnez plusieurs schémas de source de données, le texte "Varie selon les schémas de composant" peut apparaître dans certains champs, indiquant que les valeurs en cours de ces champs ne sont pas identiques dans les schémas sélectionnés. Si vous continuez et modifiez les valeurs de ces champs, les nouvelles valeurs sont appliquées de façon uniforme dans les schémas sélectionnés.

  2. Spécifiez les paramètres des sources de données RAC.

    Le tableau suivant décrit les champs de cet écran.

    Champ Description
    Pilote Sélectionnez le pilote à utiliser pour la connexion à la base de données.
    Nom du service Saisissez un nom de service Oracle RAC de base de données.
    Nom utilisateur Entrez le nom de connexion à la base de données.
    Mot de passe Entrez le mot de passe de connexion correspondant au nom utilisateur indiqué.
    Nom d'hôte Saisissez le nom du serveur hébergeant les instances de base de données RAC.
    Nom d'instance Entrez le nom de chaque instance de base de données Oracle.
    Port Entrez le numéro de port à utiliser pour la connexion au serveur qui héberge l'instance de base de données.

    Remarque : vous devez indiquer le nom d'hôte, le nom d'instance et le numéro de port d'au moins une instance de base de données.

    Pour ajouter une instance de base de données, cliquez sur Ajouter, puis indiquez le nom d'hôte, le nom de l'instance de base de données et le numéro de port.

    Les valeurs que vous indiquez sont affichées dans les colonnes appropriées de la liste de schémas, pour les schémas sélectionnés.

  3. Une fois les sélections et entrées obligatoires créées, cliquez sur Suivant pour tester les schémas de composant.
