Configurer les serveurs gérés

Utilisez cet écran pour ajouter ou supprimer des serveurs gérés. Vous pouvez également modifier les paramètres d'un serveur géré existant. Les champs obligatoires sont signalés par un astérisque (*).

  1. Passez en revue la configuration de serveur géré en cours. Les valeurs par défaut peuvent varier selon la source de domaine sélectionnée.

    Remarque : l'assistant propose deux vues : une vue tabulaire concise de tous les serveurs gérés et une vue individuelle de chaque serveur géré, chaque serveur étant représenté par un onglet. Pour passer d'une vue à l'autre, cliquez sur Changer d'affichage.

  2. Ajoutez ou supprimez des serveurs gérés, ou modifiez les paramètres de serveurs gérés existants, selon les besoins de votre domaine. Les champs accompagnés d'un astérisque (*) sont obligatoires.

    Pour plus d'informations sur les champs de cet écran, reportez-vous à la page d'aide de l'écran Configurer le serveur d'administration.

  3. Une fois les sélections et entrées obligatoires créées, cliquez sur Suivant.

Pour plus d'informations, reportez-vous à la bibliothèque de documentation Oracle Fusion Middleware sur Oracle Technology Network.
