Configurer la source de données multiples RAC

Utilisez cet écran pour configurer les sources de données (incluses dans le domaine) en tant que sources de données Oracle RAC.

Les sources de données que vous avez choisi de configurer en tant que sources de données RAC dans l'écran précédent de l'assistant (Configurer les sources de données JDBC) sont répertoriées dans la partie inférieure de l'écran.

  1. Pour sélectionner la source de données pour laquelle indiquer des paramètres, cochez la case située en regard de son nom.

    Remarque : si vous sélectionnez plusieurs sources de données, le texte "Varie selon les sources de données" peut apparaître dans certains champs, indiquant que les valeurs en cours de ces champs ne sont pas identiques dans les sources de données sélectionnées. Si vous continuez et modifiez les valeurs de ces champs, les nouvelles valeurs sont appliquées de façon uniforme dans les sources de données sélectionnées.

  2. Spécifiez les paramètres des sources de données RAC.

    Le tableau suivant décrit les champs de cet écran.

    Champ Description
    Pilote Sélectionnez le pilote JDBC à utiliser pour la connexion à la base de données.
    Nom du service Saisissez un nom de service de base de données Oracle RAC.
    Nom utilisateur Entrez le nom de connexion à la base de données.
    Mot de passe Entrez le mot de passe de connexion correspondant au nom utilisateur indiqué.
    Nom d'hôte Saisissez le nom du serveur hébergeant les instances de base de données Oracle RAC.
    Nom d'instance Entrez le nom de chaque instance de base de données Oracle.
    Port Entrez le numéro de port à utiliser pour la connexion au serveur qui héberge l'instance de base de données.

    Remarque : vous devez indiquer le nom d'hôte, le nom d'instance et le numéro de port d'au moins une instance de base de données.

    Pour ajouter une instance de base de données, cliquez sur Ajouter, puis indiquez le nom d'hôte, le nom de l'instance et le numéro de port.

    Les valeurs que vous indiquez sont affichées dans les colonnes appropriées de la liste de sources de données, pour les sources de données sélectionnées.

  3. Après avoir effectué les sélections et entrées obligatoires, cliquez sur Suivant pour tester les connexions aux sources de données.

Pour plus d'informations, reportez-vous à la bibliothèque de documentation Oracle Fusion Middleware sur Oracle Technology Network.
