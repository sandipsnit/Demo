Configurer la base de données de banque de sécurité SGBDR

Utilisez cet écran pour définir les paramètres de la banque de sécurité SGBDR.

Le tableau suivant décrit les champs de cet écran.

Champ Description
Type de base de données Dans la liste déroulante, sélectionnez le type de base de données à utiliser comme banque de sécurité SGBDR.
Pilote Sélectionnez le pilote à utiliser pour la base de données. La liste des pilotes disponibles varie selon le type de base de données sélectionné.
Nom de classe : Aucune action n'est nécessaire. Le nom de classe apparaît automatiquement en fonction du pilote que vous sélectionnez.
DBMS SID Entrez le SID de la base de données.
DBMS Host Entrez le nom de l'ordinateur qui héberge la base de données.
DBMS Port Entrez le port à utiliser pour la connexion au serveur. Le numéro de port par défaut associé au type de base de données sélectionné apparaît automatiquement.
URL Aucune action n'est nécessaire. L'URL apparaît automatiquement en fonction du pilote que vous sélectionnez.
Nom utilisateur Entrez le nom de connexion à la base de données.
Mot de passe utilisateur Entrez le mot de passe permettant d'accéder à la base de données.

Les valeurs valides sont composées d'une chaîne de caractères alphanumériques. Le tiret (-) et le trait de soulignement (_) sont pris en charge.
La valeur est cryptée.
Confirmer le mot de passe utilisateur Saisissez de nouveau le mot de passe.
Known Properties Aucune action n'est nécessaire. Les propriétés connues de la base de données apparaissent automatiquement en fonction du pilote que vous sélectionnez.
Propriétés supplémentaires Entrez éventuellement d'autres propriétés à transmettre au pilote.

Remarque : pour tester la connexion à la base de données, cliquez sur Tester la connexion. Avant de démarrer le serveur, vous devez charger les scripts SQL nécessaires pour la banque de sécurité SGBDR. Si vous utilisez une banque de sécurité SGBDR dans un domaine inclus dans un cluster, il est recommandé de le faire avec une configuration JMS (nom JNDI et rubrique JMS).

Après avoir effectué les sélections et entrées requises, cliquez sur le bouton Suivant de l'assistant.

Pour plus d'informations, reportez-vous à la bibliothèque de documentation Oracle Fusion Middleware sur Oracle Technology Network.
