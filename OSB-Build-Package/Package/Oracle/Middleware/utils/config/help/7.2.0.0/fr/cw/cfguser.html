Configurer le nom utilisateur et le mot de passe d'administrateur

Utilisez cet écran pour définir le compte d'administrateur WebLogic par défaut du domaine. Ce compte permet la réinitialisation et la connexion au serveur d'administration de ce domaine.

  • Nom
    Nom utilisateur de connexion du compte d'administrateur WebLogic.

    Le nom utilisateur ne doit contenir ni virgules ni tabulations, et aucun des caractères suivants :
    <>#|&?(){}

  • Mot de passe de l'utilisateur
    Le mot de passe distingue les majuscules des minuscules. Il doit contenir au moins huit caractères alphanumériques, dont au moins un chiffre ou l'un des caractères suivants :
    ! " # $ % & ' ( ) * + , - . / : ; < = > ? @ [ \ ] ^ _ ` { | } ~

Une fois les entrées obligatoires créées, cliquez sur Suivant.
