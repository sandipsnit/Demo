Création d'un domaine

Cet écran présente la progression de la création du domaine.

Une fois le processus terminé, le nouveau domaine (ou le domaine étendu) est prêt à l'emploi.

Pour démarrer immédiatement le serveur d'administration, cochez la case Démarrer le serveur d'administration, puis cliquez sur Terminé. Cette option est disponible uniquement si vous créez un domaine sous Windows.

Pour quitter l'assistant Configuration, cliquez sur Terminé.

Pour plus d'informations sur les différentes méthodes permettant de démarrer le serveur d'administration, reportez-vous à Démarrage et arrêt des serveurs dans Gestion du démarrage et de l'arrêt des serveurs pour Oracle WebLogic Server.

Pour plus d'informations, reportez-vous à la bibliothèque de documentation Oracle Fusion Middleware sur Oracle Technology Network.
