Récapitulatif de la configuration

Avant de continuer, vérifiez les paramètres de configuration détaillés de votre domaine.

  1. Sélectionnez un élément dans le panneau Récapitulatif de domaine et vérifiez les détails associés dans le panneau Détails. Vous pouvez apporter des modifications limitées. Pour ce faire, cliquez sur le bouton Précédent et revenez aux écrans précédents de l'assistant.

    Remarque : pour limiter le type d'informations affichées dans le panneau Récapitulatif de domaine, sélectionnez un filtre dans la liste Vue récapitulative.

  2. Après avoir vérifié les paramètres de domaine, cliquez sur Créer (ou sur Etendre si vous étendez un domaine existant).

Pour plus d'informations, reportez-vous à la bibliothèque de documentation Oracle Fusion Middleware sur Oracle Technology Network.
