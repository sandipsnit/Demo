Sélectionner un répertoire

Cet écran de l'assistant Configuration permet de sélectionner le domaine existant à mettre à jour avec des applications ou des services supplémentaires.

Utilisez l'arborescence de navigation pour sélectionner un répertoire de domaine valide (un répertoire contenant un fichier config.xml dans le répertoire config du domaine). Les répertoires de domaine valides sont signalés par une icône représentant un dossier bleu.

Une fois la sélection obligatoire créée, cliquez sur Suivant.

Pour plus d'informations, reportez-vous à la bibliothèque de documentation Oracle Fusion Middleware sur Oracle Technology Network.
