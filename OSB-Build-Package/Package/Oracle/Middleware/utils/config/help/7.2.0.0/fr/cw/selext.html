Sélectionner une source d'extension

Indiquez si le domaine doit être étendu en sélectionnant l'un des produits répertoriés ou en accédant à un modèle d'extension.

  1. Sélectionnez l'une des options suivantes :
    • Etendre mon domaine automatiquement pour prendre en charge les produits ajoutés suivants
      Cochez les cases correspondant aux autres produits à inclure dans le domaine. Les produits déjà inclus sont signalés par des cases à cocher grisées. Si vous sélectionnez un produit dépendant de ressources fournies par d'autres produits de la liste, ces derniers sont également sélectionnés automatiquement.
    • Etendre mon domaine à l'aide d'un modèle d'extension existant
      Entrez le chemin complet du modèle requis dans le champ Emplacement de modèle ou cliquez sur Parcourir pour accéder au répertoire contenant le modèle.
  2. Une fois la sélection obligatoire créée, cliquez sur Suivant.

Remarque : lorsque vous cliquez sur Suivant, si l'un des produits sélectionnés exige des ressources fournies par un autre produit non sélectionné, une boîte de dialogue d'avertissement apparaît. Revenez à l'écran Sélectionner une source d'extension pour sélectionner le produit dépendant avant de continuer.

Pour plus d'informations sur les modèles de produit affichés sur cet écran, reportez-vous à Référence de modèle de domaine Oracle Fusion Middleware.
