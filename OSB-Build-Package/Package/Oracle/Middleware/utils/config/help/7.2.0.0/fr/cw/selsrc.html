Sélectionner une source de domaine

Indiquez si un domaine configuré automatiquement doit être généré pour prendre en charge les produits sélectionnés ou si un domaine doit être créé à partir d'un modèle de domaine existant. Chaque produit répertorié dans la liste affichée est associé à un même modèle de domaine ou d'extension (un fichier JAR) configurant les ressources de domaine nécessaires à ce produit.

  1. Sélectionnez l'une des options suivantes :
    • Générer un domaine configuré automatiquement pour prendre en charge les produits suivants
      Par défaut, l'option Domaine WebLogic Server de base est sélectionnée. Cochez les cases correspondant aux autres produits à inclure dans le domaine. Si vous sélectionnez un produit dépendant des ressources fournies par d'autres produits de la liste, ces derniers sont également sélectionnés automatiquement.
    • Baser le domaine sur un modèle existant
      Sélectionnez cette option si vous souhaitez créer un domaine à l'aide d'un modèle de domaine existant. Entrez le chemin complet du modèle dans le champ Emplacement de modèle ou cliquez sur Parcourir pour accéder au répertoire contenant le modèle requis.
  2. Après avoir effectué la sélection requise, cliquez sur le bouton Suivant de l'assistant.

Remarque : lorsque vous cliquez sur Suivant, si l'un des produits sélectionnés exige des ressources fournies par un autre produit non sélectionné, une boîte de dialogue d'avertissement apparaît. Revenez à l'écran Sélectionner une source de domaine pour sélectionner le produit dépendant avant de continuer.

Pour plus d'informations sur les modèles de produit affichés sur cet écran, reportez-vous à Référence de modèle de domaine Oracle Fusion Middleware.
