Tester le schéma de composant

Cet écran de l'assistant Configuration permet de tester les configurations indiquées pour le schéma de composant dans l'écran précédent.

  1. Cochez les cases situées en regard du nom des schémas à tester.
  2. Cliquez sur Tester les connexions.

    L'assistant teste la configuration pour chaque schéma, en tentant de se connecter à une URL constituée du pilote, de l'hôte, du port et d'autres informations indiquées lors de la configuration du schéma.

    Le résultat du test est indiqué dans la colonne Statut. Des détails sont affichés dans le panneau Journal des résultats de connexion.

  3. Une fois les connexions testées, cliquez sur Suivant.

Pour plus d'informations, reportez-vous à la bibliothèque de documentation Oracle Fusion Middleware sur Oracle Technology Network.
