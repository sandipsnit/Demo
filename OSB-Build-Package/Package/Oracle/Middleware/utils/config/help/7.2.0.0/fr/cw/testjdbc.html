Tester les sources de données JDBC

Utilisez cet écran pour tester les connexions de source de données que vous avez configurées dans les écrans Configurer les sources de données JDBC et Configurer les sources de données multiples Oracle RAC.

Si vous ne souhaitez pas tester les connexions pour l'instant, ne sélectionnez aucune source de données. Cliquez sur Suivant pour continuer.

Remarque : pour tester les connexions de base de données, la base de données à laquelle vous vous connectez doit être en cours d'exécution.

  1. Cochez les cases situées en regard du nom des sources de données à tester.
  2. Cliquez sur Tester les connexions.

    L'assistant teste la configuration pour chaque source de données, en tentant de se connecter à une URL constituée du pilote, de l'hôte, du port et d'autres informations indiquées lors de la configuration de la source de données.

    Le résultat du test est indiqué dans la colonne Statut. Des détails sont affichés dans le panneau Journal des résultats de connexion.

  3. Une fois les connexions testées, cliquez sur Suivant.

Pour plus d'informations, reportez-vous à la bibliothèque de documentation Oracle Fusion Middleware sur Oracle Technology Network.
