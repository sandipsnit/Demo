Bienvenue

Cet écran de l'assistant Configuration vous permet soit de créer un domaine WebLogic, soit de modifier un domaine existant.

  1. Sélectionnez l'une des options suivantes :

    • Créer un domaine WebLogic

      Sélectionnez cette option pour créer un domaine. En général, vous utilisez le domaine de base WebLogic Server à cet effet. Ce dernier inclut un serveur d'administration et le domaine de sécurité par défaut myrealm. Toutefois, vous pouvez choisir d'utiliser votre propre modèle de domaine pour créer le domaine ; ce modèle doit contenir au moins un serveur d'administration.

    • Etendre un domaine WebLogic existant

      Sélectionnez cette option si vous disposez déjà d'un domaine WebLogic et que vous souhaitez y ajouter des ressources.

  2. Une fois la sélection obligatoire créée, cliquez sur Suivant.

Pour plus d'informations, reportez-vous à la bibliothèque de documentation Oracle Fusion Middleware sur Oracle Technology Network.
