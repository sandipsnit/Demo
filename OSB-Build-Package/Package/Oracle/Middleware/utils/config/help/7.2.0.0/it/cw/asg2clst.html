Assegna server a cluster

Utilizzare questa schermata per assegnare server gestiti ai cluster. Nella casella di riepilogo Server vengono visualizzati solo i server gestiti. Il server di amministrazione non viene elencato poiché non può essere assegnato a un cluster.

  1. Nel riquadro Cluster selezionare il cluster a cui si desidera assegnare un server gestito.
  2. Assegnare i server gestiti al cluster selezionato in uno dei modi riportati di seguito.
    • Fare doppio clic sul nome del server gestito nel riquadro Server.
    • Selezionare il server gestito nel riquadro Server e fare clic sul pulsante freccia destra che si trova tra i riquadri Server e Cluster.
    • Tenendo premuto il tasto Maiusc fare clic per selezionare più server gestiti nel riquadro Server, quindi fare clic sul pulsante freccia destra che si trova tra i riquadri Server e Cluster.

    Il nome del server gestito assegnato viene rimosso dal riquadro Server e viene aggiunto sotto il nome del cluster di destinazione nel riquadro Cluster.

    Nota: nel riquadro Server vengono elencati solo i server gestiti. Il server di amministrazione non viene elencato poiché non può essere assegnato a un cluster.

  3. Ripetere i passi 1 e 2 per ogni server gestito che si desidera assegnare a un cluster.
  4. Rivedere le assegnazioni dei cluster.

    Se necessario, è possibile rimuovere un server gestito da un cluster in uno dei modi riportati di seguito.

    • Fare doppio clic sul nome del server gestito nel riquadro Cluster.
    • Selezionare il server gestito nel riquadro Cluster e fare clic sul pulsante freccia sinistra che si trova tra i riquadri Server e Cluster.

    Il nome del server gestito viene rimosso dal riquadro Cluster e ripristinato nel riquadro Server.

  5. Dopo aver assegnato i server gestiti ai cluster, fare clic sul pulsante Successivo nella procedura guidata.
