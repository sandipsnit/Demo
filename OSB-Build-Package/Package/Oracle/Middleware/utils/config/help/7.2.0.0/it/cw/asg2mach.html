Assegna server a computer

Utilizzare questa schermata per assegnare istanze del server WebLogic a ogni computer definito.

  1. Nel riquadro Computer selezionare il computer Windows o UNIX a cui assegnare un'istanza del server WebLogic.
  2. Assegnare le istanze del server WebLogic al computer in uno dei modi riportati di seguito.
    • Fare doppio clic sull'istanza del server WebLogic nel riquadro Server.
    • Selezionare l'istanza del server WebLogic appropriata nel riquadro Server e fare clic sul pulsante freccia destra che si trova tra i riquadri Server e Computer.
    • Tenendo premuto il tasto Maiusc fare clic per selezionare più server nel riquadro Server, quindi fare clic sul pulsante freccia destra che si trova tra i riquadri Server e Computer.

    Il nome dell'istanza del server WebLogic assegnata viene rimosso dal riquadro Server e aggiunto sotto il nome del computer di destinazione nel riquadro Computer.

  3. Ripetere i passi 1 e 2 per ogni istanza del server WebLogic che si desidera assegnare a un computer.
  4. Rivedere le assegnazioni dei computer.

    Se necessario, è possibile rimuovere un'istanza del server WebLogic da un computer in uno dei modi riportati di seguito.

    • Fare doppio clic sul nome dell'istanza del server WebLogic appropriata nel riquadro Computer.
    • Selezionare l'istanza del server WebLogic appropriata nel riquadro Computer e quindi fare clic sul pulsante freccia sinistra che si trova tra i riquadri Server e Computer.

    Il nome dell'istanza del server WebLogic viene rimosso dal riquadro Computer e ripristinato nel riquadro Server.

  5. Dopo aver assegnato le istanze del server WebLogic ai computer, fare clic su Successivo.
