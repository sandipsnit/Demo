Seleziona configurazione avanzata

Selezionare tutte le categorie (se presenti) per le quali si desidera eseguire la configurazione avanzata. Per ogni categoria selezionata viene visualizzata la schermata di configurazione appropriata che consente di eseguire la configurazione avanzata. Se non si seleziona alcun elemento in questa schermata, viene visualizzata successivamente la schermata Riepilogo della configurazione.

Nota: le categorie elencate in questa schermata dipendono dalle risorse definite nei modelli selezionati per il dominio.

Nota: le opzioni Server di amministrazione e Area di memorizzazione della sicurezza RDBMS non sono disponibili quando si esegue l'estensione di un dominio.

  • Server di amministrazione
    Selezionare questa opzione per modificare le impostazioni del server di amministrazione, come ad esempio il nome del server, l'indirizzo di ascolto, la porta di ascolto e le impostazioni SSL.
  • Destinazione distribuita JMS
    Per impostazione predefinita, le destinazioni distribuite JMS vengono convertite in modo automatico in destinazioni distribuite ponderate (WDD). Selezionare questa opzione per modificare una destinazione distribuita JMS in una destinazione distribuita unificata (UDD).
  • Server, cluster e computer gestiti
    Selezionare questa opzione per effettuare le operazioni riportate di seguito.
    • Aggiungere server, cluster o computer gestiti al dominio.
    • Eliminare un server, un cluster o un computer gestito esistente.
    • Aggiungere server gestiti a un cluster esistente.
    • Modificare le impostazioni per un server, un cluster o un computer gestito esistente.
  • Distribuzioni e servizi
    Selezionare questa opzione per personalizzare il modo in cui le distribuzioni e i servizi delle applicazioni vengono indirizzati ai server e ai cluster.
  • Aree di memorizzazione file JMS
    Selezionare questa opzione per modificare le impostazioni per le aree di memorizzazione file JMS. È possibile modificare il nome, la directory e i criteri di scrittura sincrona per ogni area di memorizzazione file.
  • Area di memorizzazione della sicurezza RDBMS
    Selezionare questa opzione per configurare un sistema di gestione dei database relazionali esterni (RDBMS) come data store per vari provider della sicurezza.

Per ogni opzione di configurazione avanzata selezionata in questa schermata, la procedura guidata visualizza successivamente delle schermate aggiuntive in cui è possibile definire le impostazioni di configurazione avanzata rilevanti.

Selezionare le caselle di controllo necessarie in base alla configurazione avanzata che si desidera eseguire, quindi fare clic su Successivo.
