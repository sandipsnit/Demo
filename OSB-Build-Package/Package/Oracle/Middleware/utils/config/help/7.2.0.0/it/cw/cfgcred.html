Configura credenziali del keystore

In questa schermata viene visualizzato un set di campi Immetti password keystore/Conferma password per ogni prodotto che richiede l'impostazione di una password del keystore. Ad esempio, se è installato Oracle Information Rights Management, sarà disponibile un set di campi di password per ciascuno dei due keystore dell'applicazione seguenti:

  • Oracle Information Rights Management
  • Oracle Enterprise Manager Plugin for IRM

  1. Immettere e confermare la password necessaria per accedere al keystore dell'applicazione.
  2. Dopo aver immesso tutte le password del keystore, fare clic su Successivo.

Nota: la password del keystore per ogni prodotto elencato in questa schermata è obbligatoria per utilizzare questi prodotti. Prendere nota delle password immesse e fornirle agli utenti autorizzati a utilizzare questi prodotti.

Per ulteriori informazioni, vedere la libreria della documentazione di Oracle Fusion Middleware su Oracle Technology Network.
