Configura origini dati JDBC

Utilizzare questa schermata per configurare le origini dati JDBC definite nell'origine del dominio.

Un'origine dati JDBC contiene un pool di connessioni al database create quando viene creata l'istanza dell'origine dati (quando viene distribuita o indirizzata o all'avvio del server). Le applicazioni ricercano un'origine dati nella struttura JNDI e quindi richiedono una connessione. Quando le applicazioni non hanno più bisogno delle connessioni, le restituiscono al connection pool nell'origine dati.

Le origini dati JDBC associate ai prodotti per cui si sta creando il dominio sono elencate nella parte inferiore della schermata.

Nota: se si desidera configurare una o più origini dati come origini dati RAC, selezionare le caselle di controllo accanto al nome delle origini dati e la casella di controllo Configurare nel pannello successivo le origini dati selezionate come origini dati multiple RAC. Quindi fare clic sul pulsante Successivo.

  1. Scegliere le origini dati per cui si desidera specificare le impostazioni selezionando la casella di controllo accanto al nome dell'origine dati.

    Nota: quando si selezionano più origini dati, in alcuni campi potrebbe essere visualizzato il testo "Varia a seconda delle origini dati", indicante che i valori correnti di questi campi non sono gli stessi per tutte le origini dati selezionate. Se si prosegue e si modificano i valori in tali campi, i nuovi valori vengono applicati in modo uniforme alle origini dati selezionate.

    Nota: i valori predefiniti dei parametri delle origini dati quali fornitore, driver, nome host e numero di porta dipendono dai valori specificati nei modelli di applicazione.

  2. Rivedere le impostazioni di configurazione correnti e modificarle come richiesto.

    Nella tabella riportata di seguito vengono descritti i campi di questa schermata.

    Nota: i valori predefiniti dei parametri delle origini dati quali fornitore, driver, nome host e numero di porta dipendono dai valori specificati nei modelli di applicazione.

    Campo Descrizione
    Fornitore Selezionare il fornitore del database.
    Driver Selezionare il driver JDBC da usare per connettersi al database.

    La lista include i driver JDBC comuni per il fornitore del database selezionato.
    Nome utente Immettere il nome di login per la connessione al database.
    Password Immettere la password di login per il nome utente specificato.
    DBMS/Servizio Immettere un SID DBMS o un nome di servizio. Il valore immesso dipende dal driver selezionato.
    • Se il nome del driver Oracle selezionato contiene la stringa "per le connessioni all'istanza", è necessario immettere il SID.
    • Se il nome del driver Oracle contiene la stringa "per le connessioni ai servizi", è necessario immettere il nome del servizio.
    • Per informazioni sulla configurazione del driver DataDirect, vedere la documentazione corrispondente.
    Nome host Immettere il nome del server su cui si trova il database.
    Porta Immettere la porta da usare per la connessione al server su cui si trova il database.

    I valori specificati vengono visualizzati nelle colonne appropriate della lista di origini dati per le origini dati selezionate.

  3. Dopo aver effettuato le selezioni e le immissioni necessarie, fare clic sul pulsante Successivo per eseguire il test delle connessioni alle origini dati JDBC.
