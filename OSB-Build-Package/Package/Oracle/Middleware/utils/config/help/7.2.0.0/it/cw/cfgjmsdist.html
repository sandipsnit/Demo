Configura aree memorizzazione file JMS

Per impostazione predefinita, le destinazioni distribuite JMS vengono convertite in modo automatico in destinazioni distribuite ponderate (WDD) per garantire la compatibilità con le versioni precedenti. Per modificare una destinazione distribuita JMS in modo da utilizzarla come destinazione distribuita uniforme (UDD), selezionare UDD dall'elenco a discesa. Oracle raccomanda di utilizzare l'impostazione UDD.

Quando si modifica il funzionamento di una risorsa di sistema JMS, l'impostazione viene applicata solo alla sessione corrente se tale risorsa non è assegnata a un cluster. Dopo aver assegnato la risorsa di sistema JMS a un cluster, l'impostazione diventa effettiva e non può essere modificata.

Per ulteriori informazioni sulle destinazioni distribuite uniformi e ponderate, vedere Tipi di destinazioni distribuite in Programmazione di JMS per Oracle WebLogic Server.
