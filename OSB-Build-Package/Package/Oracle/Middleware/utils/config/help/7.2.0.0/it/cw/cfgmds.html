Configura schema dei componenti per origini dati multiple RAC

Utilizzare questa schermata per configurare gli schemi dei componenti inclusi nel dominio WebLogic come origini dati multiple RAC.

Gli schemi dei componenti per cui si è scelto di configurare le origini dati RAC nella schermata precedente della procedura guidata (Configura schema dei componenti JDBC) sono elencati nella parte inferiore della schermata.

  1. Nella lista di schemi nella parte inferiore della schermata, scegliere lo schema che si desidera configurare come origine dati RAC selezionando la casella di controllo accanto al nome dello schema.

    Nota: quando si selezionano più schemi di origini dati multiple, in alcuni campi potrebbe essere visualizzato il testo "Varia a seconda degli schemi dei componenti", indicante che i valori correnti di questi campi non sono gli stessi per tutti gli schemi selezionati. Se si prosegue e si modificano i valori in tali campi, i nuovi valori vengono applicati in modo uniforme agli schemi selezionati.

  2. Specificare le impostazioni per le origini dati RAC.

    Nella tabella riportata di seguito vengono descritti i campi di questa schermata.

    Campo Descrizione
    Driver Selezionare il driver da usare per connettersi al database.
    Nome servizio Immettere un nome di servizio per il database Oracle RAC.
    Nome utente Immettere il nome di login per la connessione al database.
    Password Immettere la password di login per il nome utente specificato.
    Nome host Immettere il nome del server su cui si trovano le istanze del database RAC.
    Nome istanza Immettere il nome di ogni istanza di database Oracle.
    Porta Immettere il numero di porta da usare per la connessione al server su cui si trova l'istanza di database.

    Nota: è necessario specificare il nome host, il nome istanza e il numero di porta di almeno un'istanza di database.

    Per aggiungere un'istanza di database, fare clic su Aggiungi e quindi specificare il nome host, il nome dell'istanza di database e il numero di porta.

    I valori specificati vengono visualizzati nelle colonne appropriate della lista di schemi per gli schemi selezionati.

  3. Dopo aver effettuato le selezioni e le immissioni necessarie, fare clic su Successivo per eseguire il test degli schemi dei componenti.
