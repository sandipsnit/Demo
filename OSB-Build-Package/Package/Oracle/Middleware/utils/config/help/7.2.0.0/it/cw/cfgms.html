Configura server gestiti

Utilizzare questa schermata per aggiungere o eliminare server gestiti. È anche possibile modificare le impostazioni per un server gestito esistente. I campi obbligatori sono indicati da un asterisco (*).

  1. Rivedere la configurazione corrente dei server gestiti. I valori predefiniti possono variare in base all'origine del dominio selezionata.

    Nota: la procedura guidata fornisce due viste, una vista tabellare concisa di tutti i server gestiti e una singola vista di ogni server gestito con ogni server rappresentato da una scheda. Per spostarsi tra le modalità di visualizzazione delle viste tabellari e a scheda, fare clic su Cambia visualizzazione.

  2. Aggiungere o eliminare i server gestiti o modificare le impostazioni dei server gestiti esistenti, come richiesto dal dominio. I campi contrassegnati da un asterisco (*) sono obbligatori.

    Per ulteriori informazioni sui campi di questa schermata, vedere la pagina della Guida per la schermata Configura server di amministrazione.

  3. Dopo aver effettuato le selezioni e le immissioni necessarie, fare clic su Successivo.

Per ulteriori informazioni, vedere la libreria della documentazione di Oracle Fusion Middleware su Oracle Technology Network.
