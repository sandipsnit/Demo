Configura origini dati multiple RAC

Utilizzare questa schermata per configurare le origini dati (incluse nel dominio) come origini dati Oracle RAC.

Le origini dati che si è scelto di configurare come origini dati RAC nella schermata precedente della procedura guidata (Configura origini dati JDBC) sono elencate nella parte inferiore della schermata.

  1. Scegliere le origini dati per cui si desidera specificare le impostazioni selezionando la casella di controllo accanto al nome dell'origine dati.

    Nota: quando si selezionano più origini dati, in alcuni campi potrebbe essere visualizzato il testo "Varia a seconda delle origini dati", indicante che i valori correnti di questi campi non sono gli stessi per tutte le origini dati selezionate. Se si prosegue e si modificano i valori in tali campi, i nuovi valori vengono applicati in modo uniforme alle origini dati selezionate.

  2. Specificare le impostazioni per le origini dati RAC.

    Nella tabella riportata di seguito vengono descritti i campi di questa schermata.

    Campo Descrizione
    Driver Selezionare il driver JDBC da usare per connettersi al database.
    Nome servizio Immettere un nome di servizio per il database Oracle RAC.
    Nome utente Immettere il nome di login per la connessione al database.
    Password Immettere la password di login per il nome utente specificato.
    Nome host Immettere il nome del server su cui si trovano le istanze di database Oracle RAC.
    Nome istanza Immettere il nome di ogni istanza di database Oracle.
    Porta Immettere il numero di porta da usare per la connessione al server su cui si trova l'istanza di database.

    Nota: è necessario specificare il nome host, il nome istanza e il numero di porta di almeno un'istanza di database.

    Per aggiungere un'istanza di database, fare clic su Aggiungi e quindi specificare il nome host, il nome dell'istanza e il numero di porta.

    I valori specificati vengono visualizzati nelle colonne appropriate della lista di origini dati per le origini dati selezionate.

  3. Dopo aver effettuato le selezioni e le immissioni necessarie, fare clic su Successivo per eseguire il test delle connessioni alle origini dati.
