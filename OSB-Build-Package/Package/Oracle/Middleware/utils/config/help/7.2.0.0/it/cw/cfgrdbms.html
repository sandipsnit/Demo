Configura database area di memorizzazione della sicurezza RDBMS

Utilizzare questa schermata per definire le impostazioni dell'area di memorizzazione della sicurezza RDBMS.

Nella tabella riportata di seguito vengono descritti i campi di questa schermata.

Campo Descrizione
Tipo database Selezionare dall'elenco a discesa il tipo di database che si desidera usare come area di memorizzazione della sicurezza RDBMS.
Driver Selezionare il driver da usare per il database. La lista di driver disponibili varia in base al tipo di database selezionato.
Nome classe Non è richiesta alcuna azione. Il nome della classe viene visualizzato automaticamente in base al driver selezionato.
DBMS SID Immettere il SID del database.
Host DBMS Immettere il nome del computer su cui si trova il database.
Porta DBMS Immettere la porta da usare per la connessione al database. Il numero di porta predefinito associato al tipo di database selezionato viene visualizzato automaticamente.
URL Non è richiesta alcuna azione. L'URL viene visualizzato automaticamente in base al tipo di driver selezionato.
Nome utente Immettere il nome di login per la connessione al database.
Password utente Immettere la password per l'accesso al database.

I valori validi consistono di una stringa di caratteri alfanumerici. Sono supportati i trattini (-) e i caratteri di sottolineatura (_).
Il valore è cifrato.
Password utente di conferma Immettere di nuovo la password.
Proprietà note Non è richiesta alcuna azione. Le proprietà note del database vengono visualizzate automaticamente in base al tipo di driver selezionato.
Proprietà aggiuntive Immettere le proprietà aggiuntive, se presenti, da passare al driver.

Nota: è possibile effettuare il test della connessione al database facendo clic su Test della connessione. Prima di avviare il server, caricare gli script SQL necessari per l'area di memorizzazione della sicurezza RDBMS. Se si usa un'area di memorizzazione della sicurezza RDBMS in un dominio in cluster, si consiglia di usarla con la configurazione JMS (nome JNDI e argomento JMS).

Dopo aver effettuato le selezioni e le immissioni necessarie, fare clic sul pulsante Successivo nella procedura guidata.

Per ulteriori informazioni, vedere la libreria della documentazione di Oracle Fusion Middleware su Oracle Technology Network.
