Configura nome utente e password dell'amministratore

Utilizzare questa schermata per definire l'account predefinito dell'amministratore WebLogic per il dominio. Questo account viene utilizzato per avviare e connettersi al server di amministrazione del dominio.

  • Nome
    Il nome utente di login per l'account dell'amministratore WebLogic.

    Il nome utente non deve contenere virgole, tabulazioni e uno dei seguenti caratteri:
    < > # | & ? ( ) { }

  • Password utente
    La password fa distinzione tra maiuscole e minuscole, deve contenere almeno otto caratteri alfanumerici con almeno un carattere numerico o uno dei seguenti caratteri:
    ! " # $ % & ' ( ) * + , - . / : ; < = > ? @ [ \ ] ^ _ ` { | } ~.

Dopo aver effettuato le selezioni necessarie, fare clic su Successivo.
