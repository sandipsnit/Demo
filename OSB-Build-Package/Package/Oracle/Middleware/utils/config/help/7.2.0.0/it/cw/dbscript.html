Esegui script di database

Un modello di dominio può contenere un set di file SQL organizzati in base al tipo di database. Se il modello di dominio contiene file SQL, è possibile eseguirli durante la creazione del dominio WebLogic nella schermata Esegui script di database. Il contenuto del database per ogni origine dati definita nel dominio WebLogic viene impostato utilizzando file SQL o di caricamento del database preesistenti.

Il contenuto del database per ogni origine dati definita nel dominio viene impostato utilizzando file SQL o di caricamento del database preesistenti.

  1. Nel riquadro Origini dati JDBC disponibili selezionare l'origine dati per cui si desidera eseguire gli script.

    Gli script che possono essere eseguiti vengono visualizzati nel riquadro File SQL disponibili e opzioni di caricamento del database.

  2. Selezionare la versione del database dalla lista Versione del database.
  3. Fare clic su Esegui script.

    Vengono eseguiti tutti gli script visualizzati nel riquadro File SQL disponibili e opzioni di caricamento del database per l'origine dati selezionata e i risultati vengono visualizzati nel riquadro Risultati.

    Se si desidera acquisire l'output del test in un file di log, selezionare la casella di controllo File di log e specificare la posizione del file di log.

  4. Ripetere i passi da 1 a 3 per ogni origine dati per cui si desidera eseguire gli script di database.
  5. Dopo aver eseguito gli script di database, fare clic su Successivo.
