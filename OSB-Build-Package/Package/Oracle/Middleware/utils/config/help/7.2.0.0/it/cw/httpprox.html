Crea applicazioni proxy HTTP

Utilizzare questa schermata per creare un'applicazione proxy HTTP per ogni cluster e specificare il server gestito su cui deve essere distribuita l'applicazione proxy. Un'applicazione proxy HTTP funge da intermediario per le richieste HTTP.

Questa schermata viene visualizzata solo se entrambe le condizioni riportate di seguito sono vere.

  • È presente almeno un server gestito assegnato a un cluster.
  • È presente almeno un server gestito non assegnato ad alcun cluster.
  1. Selezionare la casella di controllo Crea proxy HTTP per il cluster per cui si desidera creare l'applicazione proxy HTTP.

    Nella lista Server proxy viene visualizzata una lista dei server gestiti non assegnati ad alcun cluster.

  2. Nella lista Server proxy selezionare un server gestito su cui deve essere distribuita l'applicazione proxy.

    Nel server gestito selezionato viene creata e distribuita un'applicazione proxy denominata OracleProxy4_<nomecluster>_<nomeserver>.

  3. Ripetere i passi 1 e 2 per ogni cluster per cui si desidera creare applicazioni proxy HTTP.
  4. Dopo aver effettuato le selezioni necessarie, fare clic su Successivo.
