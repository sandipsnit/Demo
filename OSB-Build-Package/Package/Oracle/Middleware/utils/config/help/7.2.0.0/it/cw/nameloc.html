Specifica nome e posizione dominio

Specificare gli elementi riportati di seguito.

  • Nome dominio
    Il nome del dominio. La Configurazione guidata memorizza il file config.xml e tutti gli altri componenti generati per il dominio in questa directory sotto la directory del dominio specificata. Ad esempio, se si immette mydomain, i file del dominio vengono memorizzati per impostazione predefinita in MW_HOME\user_projects\domains\mydomain.

    Nota: il nome del dominio non deve iniziare con un numero. Questa limitazione è intesa a prevenire potenziali conflitti con i nomi delle tabelle dell'area di memorizzazione JDBC generati internamente, che devono iniziare con un carattere alfabetico.

  • Posizione dominio
    La posizione in cui verrà memorizzata la directory del dominio. L'impostazione predefinita è MW_HOME\user_projects\domains. La posizione del dominio può essere qualsiasi posizione sull'unità locale o sulla rete, ad eccezione di MW_HOME.

    In Windows è necessario includere la lettera dell'unità in questo percorso.

  • Posizione applicazione
    Questo campo viene visualizzato solo se è stato selezionato un modello di dominio o un prodotto che contiene le applicazioni. Si tratta della posizione in cui risiederanno le applicazioni. L'impostazione predefinita è MW_HOME\user_projects\applications.

    In Windows è necessario includere la lettera dell'unità in questo percorso.

Dopo aver effettuato le selezioni e le immissioni necessarie, fare clic su Successivo.
