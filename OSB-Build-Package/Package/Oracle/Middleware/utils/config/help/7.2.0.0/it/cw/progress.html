Creazione del dominio

In questa schermata viene visualizzato lo stato di avanzamento della creazione del dominio.

Al termine del processo, il dominio nuovo (o esteso) è pronto per l'uso.

Se si desidera avviare immediatamente il server di amministrazione, selezionare la casella di controllo Avvia server di amministrazione e fare clic su Fine. Questa opzione è disponibile solo per i sistemi Windows e se si sta creando un nuovo dominio.

Per uscire dalla Configurazione guidata, fare clic su Fine.

Per ulteriori informazioni sui vari metodi che è possibile utilizzare per avviare il server di amministrazione, vedere Avvio e interruzione dei server in Gestione dell'avvio e della chiusura del server per Oracle WebLogic Server.

Per ulteriori informazioni, vedere la libreria della documentazione di Oracle Fusion Middleware su Oracle Technology Network.
