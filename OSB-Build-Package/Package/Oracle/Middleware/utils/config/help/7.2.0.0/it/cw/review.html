Riepilogo della configurazione

Prima di continuare, rivedere le impostazioni di configurazione dettagliate del dominio.

  1. Selezionare un elemento nel riquadro Riepilogo dominio per rivedere i dettagli associati nel riquadro Dettagli. È possibile apportare modifiche limitate facendo clic sul pulsante Precedente e tornando alle schermate precedenti della procedura guidata.

    Nota: è possibile limitare il tipo di informazioni visualizzate nel riquadro Riepilogo dominio selezionando un filtro dalla lista Vista di riepilogo.

  2. Dopo aver rivisto le impostazioni del dominio, fare clic su Crea (o Estendi se si sta estendendo un dominio esistente).

Per ulteriori informazioni, vedere la libreria della documentazione di Oracle Fusion Middleware su Oracle Technology Network.
