Seleziona directory

In questa schermata della Configurazione guidata è possibile selezionare il dominio esistente che si desidera aggiornare con applicazioni o servizi aggiuntivi.

Utilizzare la struttura di navigazione per selezionare una directory di dominio valida, ovvero una directory che contiene un file config.xml nella directory config del dominio. Le directory di dominio valide sono indicate da un'icona a forma di cartella di colore blu nella struttura di navigazione.

Dopo aver effettuato la selezione necessaria, fare clic su Successivo.

Per ulteriori informazioni, vedere la libreria della documentazione di Oracle Fusion Middleware su Oracle Technology Network.
