Seleziona origine dominio

Scegliere se estendere il dominio selezionando uno dei prodotti elencati oppure sfogliando un modello di estensione.

  1. Selezionare una delle opzioni riportate di seguito.
    • Estendi automaticamente il dominio per supportare i seguenti prodotti aggiunti
      Selezionare le caselle di controllo corrispondenti agli altri prodotti che si desidera includere nel dominio. I prodotti già inclusi nel dominio vengono indicati da caselle di controllo disattivate. Se si seleziona un prodotto che dipende dalle risorse fornite da altri prodotti riportati nella lista, anche questi prodotti vengono selezionati automaticamente.
    • Estendi il dominio utilizzando un modello di estensione esistente
      Immettere il percorso completo del modello richiesto nel campo Posizione modello o fare clic su Sfoglia per passare alla directory contenente il modello.
  2. Dopo aver effettuato la selezione necessaria, fare clic su Successivo.

Nota: quando si fa clic su Successivo, se uno qualsiasi dei prodotti selezionati richiede delle risorse fornite da un altro prodotto non selezionato, viene visualizzata una finestra di dialogo di avvertenza. Tornare alla schermata Seleziona origine estensione per selezionare il prodotto dipendente prima di procedere.

Per informazioni sui modelli di prodotti visualizzati in questa schermata, vedere Oracle Fusion Middleware Domain Template Reference.
