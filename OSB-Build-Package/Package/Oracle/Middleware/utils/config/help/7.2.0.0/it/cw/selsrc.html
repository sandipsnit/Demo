Seleziona origine dominio

Scegliere se generare un dominio configurato in modo automatico per supportare i prodotti selezionati oppure creare un dominio basato su un modello di dominio esistente. Ogni prodotto riportato nella lista visualizzata è associato a un singolo modello di dominio o di estensione (un file JAR) che consente di configurare le risorse del dominio necessarie per tale prodotto.

  1. Selezionare una delle opzioni riportate di seguito.
    • Genera un dominio configurato automaticamente per supportare i seguenti prodotti
      Per impostazione predefinita, l'opzione Dominio di base del server WebLogic è selezionata. Selezionare le caselle di controllo corrispondenti agli altri prodotti che si desidera includere nel dominio. Se si seleziona un prodotto che dipende dalle risorse fornite da altri prodotti nella lista, anche questi prodotti vengono selezionati automaticamente.
    • Basa questo dominio su un modello esistente
      Selezionare questa opzione se si desidera creare un dominio usando un modello di dominio esistente. Immettere il percorso completo del file JAR modello nel campo Posizione modello o fare clic su Sfoglia per passare alla directory contenente il modello richiesto.
  2. Dopo aver effettuato la selezione necessaria, fare clic sul pulsante Successivo nella procedura guidata.

Nota: quando si fa clic su Successivo, se uno qualsiasi dei prodotti selezionati richiede delle risorse fornite da un altro prodotto non selezionato, viene visualizzata una finestra di dialogo di avvertenza. Tornare alla schermata Seleziona origine dominio per selezionare il prodotto dipendente prima di procedere.

Per informazioni sui modelli di prodotti visualizzati in questa schermata, vedere Oracle Fusion Middleware Domain Template Reference.
