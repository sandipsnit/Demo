Test dello schema dei componenti

In questa schermata della Configurazione guidata è possibile eseguire il test delle configurazioni specificate per lo schema dei componenti nella schermata precedente.

  1. Selezionare le caselle di controllo accanto ai nomi degli schemi su cui si desidera eseguire il test.
  2. Fare clic su Test delle connessioni.

    La procedura guidata esegue il test della configurazione per ogni schema tentando di connettersi a un URL costruito usando il driver, l'host, la porta e altre informazioni specificate durante la configurazione dello schema.

    Il risultato del test viene indicato nella colonna Stato. I dettagli vengono visualizzati nel riquadro Log dei risultati della connessione.

  3. Dopo aver effettuato il test delle connessioni, fare clic su Successivo.

Per ulteriori informazioni, vedere la libreria della documentazione di Oracle Fusion Middleware su Oracle Technology Network.
