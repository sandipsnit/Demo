Test delle origini dati JDBC

Utilizzare questa schermata per eseguire il test delle connessioni alle origini dati configurate nelle schermate Configura origini dati JDBC e Configura origini dati multiple Oracle RAC.

Se non si desidera eseguire il test delle connessioni in questo momento, non selezionare alcuna origine dati. Fare clic su Successivo per continuare.

Nota: per eseguire il test delle connessioni al database, è necessario che il database al quale si sta eseguendo la connessione sia in esecuzione.

  1. Selezionare le caselle di controllo accanto ai nomi delle origini dati su cui si desidera eseguire il test.
  2. Fare clic su Test delle connessioni.

    La procedura guidata esegue il test della configurazione per ogni origine dati tentando di connettersi a un URL costruito usando il driver, l'host, la porta e altre informazioni specificate durante la configurazione dell'origine dati.

    Il risultato del test viene indicato nella colonna Stato. I dettagli vengono visualizzati nel riquadro Log dei risultati della connessione.

  3. Dopo aver effettuato il test delle connessioni, fare clic su Successivo.

Per ulteriori informazioni, vedere la libreria della documentazione di Oracle Fusion Middleware su Oracle Technology Network.
