Benvenuti

In questa schermata della Configurazione guidata è possibile scegliere se creare un dominio WebLogic o modificarne uno esistente.

  1. Selezionare una delle opzioni riportate di seguito.

    • Creare un nuovo dominio WebLogic

      Selezionare questa opzione se si desidera creare un nuovo dominio. In genere, il nuovo dominio si baserà sul dominio di base del server WebLogic, che include un server di amministrazione e il realm di sicurezza predefinito myrealm. Per creare il dominio, è tuttavia possibile scegliere di utilizzare il proprio modello di dominio, che deve contenere almeno un server di amministrazione.

    • Estendere un dominio WebLogic esistente

      Selezionare questa opzione se esiste già un dominio WebLogic e si desidera aggiungere risorse a tale dominio.

  2. Dopo aver effettuato la selezione necessaria, fare clic su Successivo.

Per ulteriori informazioni, vedere la libreria della documentazione di Oracle Fusion Middleware su Oracle Technology Network.
