サーバーのクラスタへの割当

この画面では、管理対象サーバーをクラスタに割り当てます。「サーバー」リスト・ボックスには管理対象サーバーのみが表示されます。管理サーバーはクラスタに割り当てることができないため表示されません。

  1. 「クラスタ」ペインで、管理対象サーバーを割り当てるクラスタを選択します。
  2. 次のいずれかの方法で、指定したクラスタに管理対象サーバーを割り当てます。
    • 「サーバー」ペインの管理対象サーバーの名前をダブルクリックします。
    • 「クラスタ」ペインで管理対象サーバーを選択して「サーバー」ペインと「クラスタ」ペインの間にある右矢印をクリックします。
    • 「サーバー」ペインで[Shift]を押しながらクリックして複数のサーバーを選択し、「サーバー」ペインと「クラスタ」ペインの間にある右矢印をクリックします。

    割り当てられた管理対象サーバーの名前が「サーバー」ペインから削除され、「クラスタ」ペインのターゲット・クラスタの名前の下に追加されます。

    注意: 「サーバー」ペインに表示されるのは管理対象サーバーのみです。管理サーバーは、クラスタに割り当てることができないため表示されません。

  3. クラスタに割り当てる各管理対象サーバーに対して、手順1と2を繰り返します。
  4. クラスタの割当てを確認します。

    必要な場合は、次のいずれかの方法でクラスタから管理対象サーバーを削除できます。

    • 「クラスタ」ペインの管理対象サーバーの名前をダブルクリックします。
    • 「クラスタ」ペインで管理対象サーバーを選択して「サーバー」ペインと「クラスタ」ペインの間にある左矢印をクリックします。

    管理対象サーバーの名前が「クラスタ」ペインから削除され、「サーバー」ペインに戻ります。

  5. 管理対象サーバーをクラスタに割り当てた後、ウィザードで「次へ」ボタンをクリックします。

詳細は、Oracle Technology NetworkのOracle Fusion Middlewareのドキュメント・ライブラリを参照してください。
