サーバーのマシンへの割当

この画面では、定義した各マシンにWebLogic Serverインスタンスを割り当てます。

  1. 「マシン」ペインで、WebLogic Serverインスタンスを割り当てるWindowsマシンまたはUNIXマシンを選択します。
  2. 次のいずれかの方法で、指定したマシンにWebLogic Serverインスタンスを割り当てます。
    • 「サーバー」ペインのWebLogic Serverインスタンスをダブルクリックします。
    • 「サーバー」ペインで適切なWebLogic Serverインスタンスを選択して「サーバー」ペインと「マシン」ペインの間にある右矢印をクリックします。
    • 「サーバー」ペインで[Shift]を押しながらクリックして複数のサーバーを選択し、「サーバー」ペインと「マシン」ペインの間にある右矢印をクリックします。

    割り当てられたWebLogic Serverインスタンスの名前が「サーバー」ペインから削除され、「マシン」ペインのターゲット・マシンの名前の下に追加されます。

  3. マシンに割り当てる各WebLogic Serverインスタンスに対して、手順1と2を繰り返します。
  4. マシンの割当てを確認します。

    必要な場合は、次のいずれかの方法でマシンからWebLogic Serverインスタンスを削除できます。

    • 「マシン」ペインの適切なWebLogic Serverインスタンスの名前をダブルクリックします。
    • 「マシン」ペインで適切なWebLogic Serverインスタンスを選択して「サーバー」ペインと「マシン」ペインの間にある左矢印をクリックします。

    WebLogic Serverインスタンスの名前が「マシン」ペインから削除され、「サーバー」ペインに戻ります。

  5. WebLogic Serverインスタンスのマシンへの割当てが完了した後、「次へ」をクリックします。

詳細は、Oracle Technology NetworkのOracle Fusion Middlewareのドキュメント・ライブラリを参照してください。
