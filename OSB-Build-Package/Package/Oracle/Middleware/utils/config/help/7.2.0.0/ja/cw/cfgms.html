管理対象サーバーの構成

この画面では、管理対象サーバーを追加または削除します。既存の管理対象サーバーの設定を変更することもできます。アスタリスク（*）は必須フィールドを示します。

  1. 現在の管理対象サーバーの構成を確認します。選択したドメイン・ソースによってデフォルト値は異なります。

    注意: 2種類の表示方法があります。すべての管理対象サーバーを簡潔な表形式で表示する方法と、各管理対象サーバーを個別に表示（1つのサーバーを1つのタブに表示）する方法です。表形式とタブ形式の表示モードを切り替えるには、「表示の切替え」をクリックします。

  2. 必要に応じて、管理対象サーバーの追加または削除、または既存の管理対象サーバーの設定を変更します。アスタリスク（*）が付いているフィールドは必須フィールドです。

    この画面のフィールドの詳細は、管理サーバーの構成画面のヘルプ・ページを参照してください。

  3. 必要な選択と入力が完了した後、「次へ」をクリックします。

詳細は、Oracle Technology NetworkのOracle Fusion Middlewareのドキュメント・ライブラリを参照してください。
