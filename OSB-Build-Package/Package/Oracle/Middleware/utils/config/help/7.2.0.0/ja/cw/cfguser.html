管理者のユーザー名とパスワードの構成

この画面では、ドメインのデフォルトのWebLogic Administratorアカウントを定義します。このアカウントを使用して、ドメインの管理サーバーを起動し、接続します。

  • 名前
    WebLogic Administratorアカウントのログイン・ユーザー名。

    ユーザー名にカンマ、タブ、および次の文字を使用することはできません:
    <>#|&?(){}

  • ユーザー・パスワード
    パスワードの大文字と小文字は区別されます。少なくとも1つの数字または次のいずれかの文字を含む8文字以上の英数字である必要があります:
    !"#$%&'()*+,-./:;<=>?@[¥]^_`{|}~

必要な入力が完了した後、「次へ」をクリックします。

詳細は、Oracle Technology NetworkのOracle Fusion Middlewareのドキュメント・ライブラリを参照してください。
