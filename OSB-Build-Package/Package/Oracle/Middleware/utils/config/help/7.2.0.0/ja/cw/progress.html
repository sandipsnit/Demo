ドメインの作成

この画面には、ドメインの作成の進行状況が示されます。

作成プロセス完了後、新しい（または拡張された）ドメインをすぐに使用できます。

管理サーバーをただちに起動する場合は、「管理サーバーの起動」チェック・ボックスをチェックして「完了」をクリックします。このオプションはWindowsシステムで新しいドメインを作成するときにのみ使用できます。

構成ウィザードを終了するには、「完了」をクリックします。

管理サーバーを開始するための様々な方法の詳細は、Oracle WebLogic Serverのサーバーの起動と停止の管理のサーバーの起動と停止を参照してください。

詳細は、Oracle Technology NetworkのOracle Fusion Middlewareのドキュメント・ライブラリを参照してください。
