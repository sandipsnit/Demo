構成サマリー

続行する前に、ドメイン構成の設定の詳細を確認します。

  1. 「ドメインのサマリ」ペインで項目を選択し、関連する詳細を「詳細」ペインで確認します。「前へ」ボタンをクリックして前の画面に戻り、一部変更を加えることができます。

    注意: 「サマリー・ビュー」リストでフィルタを選択すると、「ドメインのサマリー」ペインに表示する情報の種類を制限できます。

  2. ドメインの設定を確認した後、「作成」（既存のドメインを拡張する場合は「拡張」）をクリックします。

詳細は、Oracle Technology NetworkのOracle Fusion Middlewareのドキュメント・ライブラリを参照してください。
