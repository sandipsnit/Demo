ディレクトリの選択

構成ウィザードのこの画面では、追加のアプリケーションまたはサーバーを使用して更新する必要がある既存のドメインを選択します。

ナビゲーション・ツリーを使用して、有効なドメイン・ディレクトリ（ドメインのconfigディレクトリにconfig.xmlファイルが含まれているディレクトリ）を選択します。有効なドメイン・ディレクトリはナビゲーション・ツリー内で青いフォルダ・アイコンで示されます。

必要な選択が完了した後、「次へ」をクリックします。

詳細は、Oracle Technology NetworkのOracle Fusion Middlewareのドキュメント・ライブラリを参照してください。
