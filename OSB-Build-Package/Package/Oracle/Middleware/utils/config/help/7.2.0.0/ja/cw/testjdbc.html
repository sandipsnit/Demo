JDBCデータ・ソースのテスト

この画面では、「JDBCデータ・ソースの構成」およびOracle RACマルチ・データ・ソースの構成画面で構成されたデータ・ソースの接続をテストします。

この時点で接続をテストする必要がない場合は、どのデータ・ソースも選択しないでください。「次へ」をクリックして続行します。

注意: データベースの接続をテストするには、接続するデータベースが実行中である必要があります。

  1. テストするデータ・ソース名の横のチェック・ボックスをチェックします。
  2. 「接続のテスト」をクリックします。

    ウィザードでは、データ・ソースの構成で指定されたドライバ、ホスト、ポート、およびその他の情報に基づいて作成したURLへの接続を試みることにより、各データ・ソースの構成がテストされます。

    テストの結果は「Status」列に表示されます。詳細は「接続結果ログ」に表示されます。

  3. 接続のテストが完了した後、「次へ」をクリックします。

詳細は、Oracle Technology NetworkのOracle Fusion Middlewareのドキュメント・ライブラリを参照してください。
