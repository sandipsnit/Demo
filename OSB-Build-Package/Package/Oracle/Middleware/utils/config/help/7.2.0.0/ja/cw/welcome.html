構成ウィザードのこの画面では、新しいドメインを作成するか、既存のドメインを変更するかを選択します。

  1. 次のいずれかのオプションを選択します。

    • 新しいWebLogicドメインの作成

      新しいドメインを作成するには、このオプションを選択します。通常、新しいドメインはWebLogic Serverの基本ドメインに基づいて作成され、これには管理サーバーとデフォルトのセキュリティ・レルムmyrealmが含まれています。ただし、独自のドメイン・テンプレートを使用してドメインを作成することもできます。ドメイン・テンプレートには少なくとも管理サーバーが含まれている必要があります。

    • 既存のWebLogicドメインの拡張

      WebLogicドメインがすでにあり、そのドメインにリソースを追加する場合は、このオプションを選択します。

  2. 必要な選択が完了した後、「次へ」をクリックします。

詳細は、Oracle Technology NetworkのOracle Fusion Middlewareのドキュメント・ライブラリを参照してください。
