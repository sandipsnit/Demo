클러스터에 서버 할당

이 화면에서는 클러스터에 관리 서버를 할당할 수 있습니다. 관리되는 서버만 [서버] 목록 상자에 표시됩니다. 관리 서버는 클러스터에 할당할 수 없기 때문에 목록에 표시되지 않습니다.

  1. 클러스터 창에서 관리되는 서버를 할당할 클러스터를 선택합니다.
  2. 다음 방법 중 하나를 사용하여 선택한 클러스터에 관리되는 서버를 할당합니다.
    • 서버 창에서 관리되는 서버 이름을 두 번 누릅니다.
    • 서버 창에서 관리되는 서버를 선택하고 서버와 클러스터 창 사이의 오른쪽 화살표 단추를 누릅니다.
    • Shift를 누른 채 눌러 서버 창에서 관리되는 서버를 여러 개 선택하고 서버와 클러스터 창 사이의 오른쪽 화살표 단추를 누릅니다.

    할당된 관리되는 서버의 이름이 서버 창에서 제거되고 클러스터 창의 대상 클러스터 이름 아래에 추가됩니다.

    주: 관리되는 서버만 서버 창에 표시됩니다. 관리 서버는 클러스터에 할당할 수 없기 때문에 목록에 표시되지 않습니다.

  3. 클러스터에 할당할 각 관리되는 서버에 대해 1단계와 2단계를 반복합니다.
  4. 클러스터 할당 내용을 검토합니다.

    필요한 경우, 다음 방법 중 하나를 사용하여 클러스터에서 관리되는 서버를 제거할 수 있습니다.

    • 클러스터 창에서 관리되는 서버 이름을 두 번 누릅니다.
    • 클러스터 창에서 관리되는 서버를 선택하고 서버와 클러스터 창 사이의 왼쪽 화살표 단추를 누릅니다.

    관리되는 서버 이름이 클러스터 창에서 제거되고 서버 창에 다시 추가됩니다.

  5. 관리되는 서버를 클러스터에 할당한 후 마법사의 다음 단추를 누릅니다.
