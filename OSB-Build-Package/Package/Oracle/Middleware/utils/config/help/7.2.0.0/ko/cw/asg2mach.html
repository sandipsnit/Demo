시스템에 서버 할당

이 화면에서는 WebLogic Server 인스턴스를 정의된 각 시스템에 할당할 수 있습니다.

  1. 시스템 창에서 WebLogic Server 인스턴스를 할당할 Windows 또는 UNIX 시스템을 선택합니다.
  2. 다음 방법 중 하나를 사용하여 WebLogic Server 인스턴스를 선택한 시스템에 할당합니다.
    • 서버 창에서 WebLogic Server 인스턴스를 두 번 누릅니다.
    • 서버 창에서 적합한 WebLogic Server 인스턴스를 선택하고 서버와 시스템 창 사이의 오른쪽 화살표 단추를 누릅니다.
    • Shift를 누른 채 서버 창에서 서버를 여러 개 선택하고 서버와 시스템 창 사이의 오른쪽 화살표 단추를 누릅니다.

    할당된 WebLogic Server 인스턴스 이름이 서버 창에서 제거되고 시스템 창의 대상 시스템 이름 아래에 추가됩니다.

  3. 시스템에 할당할 각 WebLogic Server 인스턴스에 대해 1단계와 2단계를 반복합니다.
  4. 시스템 할당 내용을 검토합니다.

    필요한 경우, 다음 방법 중 하나를 사용하여 시스템에서 WebLogic Server 인스턴스를 제거할 수 있습니다.

    • 시스템 창에서 적합한 WebLogic Server 인스턴스 이름을 두 번 누릅니다.
    • 시스템 창에서 적합한 WebLogic Server 인스턴스를 선택하고 서버와 시스템 창 사이의 왼쪽 화살표 단추를 누릅니다.

    WebLogic Server 인스턴스 이름이 시스템 창에서 제거되어 서버 창에 다시 추가됩니다.

  5. WebLogic Server 인스턴스를 시스템에 할당한 후 다음을 누릅니다.
