고급 구성 선택

고급 구성을 수행할 모든 범주를 선택합니다. 선택하는 각 범주에 대해 고급 구성을 수행할 수 있는 해당 구성 화면이 표시됩니다. 이 화면에서 항목을 선택하지 않으면 [구성 요약] 화면이 다음에 표시됩니다.

주: 이 화면에 나열되는 범주는 도메인에 대해 선택한 템플리트에서 정의된 리소스에 따라 다릅니다.

주: 관리 서버 및 RDBMS 보안 저장소 옵션은 도메인을 확장하는 경우 사용할 수 없습니다.

  • 관리 서버
    서버 이름, 수신 주소, 수신 포트 및 SSL 설정과 같은 관리 서버 설정을 수정하려면 이 옵션을 선택합니다.
  • JMS 분배 대상
    기본적으로 JMS 분배 대상은 WDD(가중 분배 대상)로 자동 변환됩니다. JMS 분배 대상을 UDD(균등 분배 대상)로 변경하려면 이 옵션을 선택합니다.
  • 관리되는 서버, 클러스터 및 시스템
    다음의 경우 이 옵션을 선택합니다.
    • 관리되는 서버, 클러스터 또는 시스템을 도메인에 추가
    • 기존 관리되는 서버, 클러스터 또는 시스템 삭제
    • 관리되는 서버를 기존 클러스터에 추가
    • 기존 관리되는 서버, 클러스터 또는 시스템에 대한 설정 수정
  • 배치 및 서비스
    응용 프로그램 배치 및 서비스가 서버 및 클러스터에 지정되는 방식을 사용자 정의하려면 이 옵션을 선택합니다.
  • JMS 파일 저장소
    JMS 파일 저장소에 대한 설정을 변경하려면 이 옵션을 선택합니다. 각 파일 저장소에 대한 이름, 디렉토리 및 동기화 쓰기 정책을 변경할 수 있습니다.
  • RDBMS 보안 저장소
    외부 RDBMS(관계형 데이터베이스 관리 시스템)를 다양한 보안 제공자를 위한 데이터 저장소로 구성하려면 이 옵션을 선택합니다.

이 화면에서 선택한 각 고급 구성 옵션마다 마법사는 관련 고급 구성 설정을 정의할 수 있는 추가 화면을 나중에 표시합니다.
