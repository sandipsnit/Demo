키 저장소 구성 인증서

이 화면에는 키 저장소 비밀번호를 설정해야 하는 각 제품마다 [키 저장소 비밀번호 입력/비밀번호 확인] 필드가 표시됩니다. 예를 들어, Oracle Information Rights Manager를 설치한 경우 다음 두 응용 프로그램 키 저장소에 대해 각각 비밀번호 필드가 있습니다.

  • Oracle Information Rights Management
  • IRM용 Oracle Enterprise Manager 플러그인

  1. 응용 프로그램의 키 저장소에 액세스하는 데 필요한 비밀번호를 입력하고 확인합니다.
  2. 모든 키 저장소 비밀번호를 입력한 후 다음을 누릅니다.

주: 이 화면에 나열된 각 제품에 대한 키 저장소 비밀번호는 해당 제품을 사용하는 데 필요합니다. 입력한 비밀번호를 기억하고 해당 제품을 사용해야 하는 모든 사람에게 제공하십시오.
