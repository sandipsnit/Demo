서버 시작 모드 및 JDK 구성

이 화면에서는 WebLogic을 운영 모드 또는 개발 모드에서 실행할지 지정합니다. 이 도메인에서 사용할 JDK의 위치를 지정할 수도 있습니다.

  1. WebLogic 도메인 시작 모드를 선택합니다.
    • 개발 모드
      이 모드에서는 보안 구성의 수준이 상대적으로 낮으므로 응용 프로그램을 자동 배치할 수 있습니다.
    • 운영 모드
      이 모드에서는 보안 구성의 수준이 상대적으로 높으므로 응용 프로그램을 배치하려면 사용자 이름 및 비밀번호를 입력해야 합니다. 도메인을 운영하기 전에 운영 환경의 보안 절차에 대해 잘 알고 있어야 합니다. 자세한 내용은 Securing a Production Environment for Oracle WebLogic Server의 Ensuring the Security of Your Production Environment를 참조하십시오.

    다음 표에는 요구 사항에 맞는 시작 모드를 선택하는 데 도움이 되는 정보가 나와 있습니다.

    기능 개발 모드 운영 모드
    SSL WebLogic Server 보안 서비스에서 제공하는 데모 디지털 인증서와 데모 키 저장소를 사용할 수 있습니다. 이 인증서를 사용하면 SSL로 보안이 유지되는 환경에서 응용 프로그램이 작동하도록 설계할 수 있습니다.

    자세한 내용은 Securing Oracle WebLogic Server의 Configuring SSL을 참조하십시오.

    		 데모 디지털 인증서와 데모 키 저장소를 사용해서는 안 됩니다. 그럴 경우 경고 메시지가 표시됩니다.
    응용 프로그램 배치 WebLogic Server 인스턴스는 domain_name/autodeploy 디렉토리에 상주하는 응용 프로그램을 자동으로 배치 및 갱신할 수 있습니다.

    단일 서버 개발 환경에서만 이 방법을 사용하는 것이 권장됩니다.

    자세한 내용은 Deploying Applications to Oracle WebLogic Server의 Deploying Applications and Modules with weblogic.deployer를 참조하십시오.

    		 자동 배치 기능이 사용 안함으로 설정되어 있으므로 WebLogic Server 관리 콘솔, weblogic.Deployer 툴 또는 WebLogic Scripting Tool을 사용해야 합니다.
    로그 파일 순환 기본적으로 WebLogic Server 인스턴스를 시작하면 서버는 로컬 서버 로그 파일을 SERVER-NAME.log.n으로 자동으로 이름을 다시 지정(순환)합니다. 나머지 서버 세션의 경우 파일 크기가 500KB에 도달할 때까지 메시지가 로그 파일에 기록됩니다.

    로깅 구성의 유지되는 파일 수 제한 설정 기본값은 true입니다. 이 설정은 서버 인스턴스에서 이전 메시지를 저장하기 위해 생성하는 로그 파일 수를 제한합니다.

    		 파일 크기가 5000KB에 도달하면 서버가 로컬 로그 파일을 순환합니다. 서버가 운영 모드용으로 구성되면 기본적으로 로그 파일의 모든 버전이 그대로 유지됩니다. 관리자가 유지되는 로그 파일의 수를 사용자 정의하려는 경우도 있습니다.
    JDBC 시스템 리소스 기본 용량은 15개 연결입니다. 기본 용량은 25개 연결입니다.
  2. JDK를 선택합니다.

    JDK 선택 사항 창에서 WebLogic 도메인 시작 모드 창에서 선택한 시작 모드용 JDK를 선택합니다. 구성 마법사는 설치 프로그램에 포함된 JDK의 목록을 제공합니다. 이러한 JDK 또는 시스템에 설치된 다른 JDK 중 하나를 선택할 수 있습니다.

    주:

    • 설치 프로그램에 포함된 JDK를 선택하면 구성 마법사는 서버 시작 스크립트를 생성하여 JDK를 호출합니다. JDK에서 제공되지 않는 JDK를 선택하면 구성 마법사는 시작 스크립트를 구성하지 않습니다. 시작 스크립트를 수동으로 변경해야 합니다.
    • 사용 중인 플랫폼에서 지원하는 JDK만 선택하십시오. 기본 선택 사항으로는 도메인을 설치할 플랫폼의 환경 요구 사항에 가장 잘 맞는 JDK가 선택되어 표시됩니다.
  3. 필수 선택 사항을 선택하고 입력한 다음 마법사의 다음 단추를 누릅니다.

자세한 내용은 Oracle Technology Network의 Oracle Fusion Middleware 설명서 라이브러리를 참조하십시오.
