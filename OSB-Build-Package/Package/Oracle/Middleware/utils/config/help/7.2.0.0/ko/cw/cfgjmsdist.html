JMS 파일 저장소 구성

기본적으로 역호환성을 위해 JMS 분배 대상은 WDD(가중 분배 대상)로 자동 변환됩니다. UDD(균등 분배 대상) 동작을 사용하도록 JMS 분배 대상을 변경하려면 드롭다운 목록에서 UDD를 선택합니다. Oracle은 UDD 설정 사용을 권장합니다.

JMS 시스템 리소스 동작을 변경할 경우 JMS 시스템 리소스가 클러스터에 할당되지 않으면 설정은 현재 세션에만 적용됩니다. JMS 시스템 리소스가 클러스터에 할당되면 설정은 고정되고 변경할 수 없습니다.

UDD 및 WDD에 대한 자세한 내용은 Programming JMS for Oracle WebLogic Server의 Types of Distributed Destinations를 참조하십시오.
