관리 서버 구성

이 화면에서는 관리되는 서버를 추가하거나 삭제할 수 있습니다. 기존 관리되는 서버에 대한 설정을 변경할 수도 있습니다. 별표(*)로 표시된 필드는 필수 필드입니다.

  1. 현재 관리되는 서버 구성을 검토합니다. 기본값은 선택한 도메인 소스에 준해 달라질 수 있습니다.

    주: 마법사는 두 가지 보기를 제공합니다. 하나는 모든 관리 서버가 표시되는 간결한 표 형식의 보기이고, 다른 하나는 각각의 관리 서버가 탭으로 표시되는 개별 보기입니다. 보기 모드를 표 보기와 탭 보기 간에 토글하려면 표시 전환을 누릅니다.

  2. 사용자의 도메인에 필요한 대로 관리 서버를 추가 또는 삭제하거나 기존 관리 서버의 설정을 변경합니다. 별표(*)가 있는 필드는 필수 필드입니다.
