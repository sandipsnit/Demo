관리자 사용자 이름 및 비밀번호 구성

이 화면에서는 도메인에 대한 기본 WebLogic 관리자 계정을 정의할 수 있습니다. 이 계정은 도메인 관리 서버를 시작하고 접속하는 데 필요합니다.

  • 이름
    WebLogic 관리자 계정에 대한 로그인 사용자 이름입니다.

    사용자 이름은 콤마, 탭 및 다음 문자를 포함하면 안됩니다.
    < > # | & ? ( ) { }

  • 사용자 비밀번호
    비밀번호는 대소문자를 구분합니다. 비밀번호는 하나 이상의 숫자 또는 다음 문자 중 하나를 포함해서 최소 8자 이상의 영숫자를 사용해야 합니다.
    ! " # $ % & ' ( ) * + , - . / : ; < = > ? @ [ \ ] ^ _ ` { | } ~
