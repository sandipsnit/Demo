데이터베이스 스크립트 실행

도메인 템플리트에는 데이터베이스 유형별로 구성된 SQL 파일이 포함될 수 있습니다. 도메인 템플리트에 SQL 파일이 포함된 경우 WebLogic 도메인을 생성할 때 데이터베이스 스크립트 실행 화면에서 실행할 수 있습니다. WebLogic 도메인에 정의된 각 데이터 소스의 데이터베이스 콘텐츠는 기존 SQL 또는 데이터베이스 로드 파일을 사용하여 설정됩니다.

도메인에 정의된 각 데이터 소스의 데이터베이스 콘텐츠는 기존 SQL 또는 데이터베이스 로드 파일을 사용하여 설정됩니다.

  1. 사용할 수 있는 JDBC 데이터 소스 창에서 스크립트를 실행할 데이터 소스를 선택합니다.

    실행할 수 있는 스크립트는 사용할 수 있는 SQL 파일 및 데이터베이스 로드 옵션 창에 표시됩니다.

  2. DB 버전 목록에서 데이터베이스 버전을 선택합니다.
  3. 스크립트 실행을 누릅니다.

    선택한 데이터 소스에 대해 사용할 수 있는 SQL 파일 및 데이터베이스 로드 옵션 창에 표시되는 모든 스크립트가 실행되며 결과는 결과 창에 표시됩니다.

    테스트 출력을 로그 파일에 캡쳐하려면 로그 파일 체크 박스를 선택하고 로그 파일의 위치를 지정합니다.

  4. 데이터베이스 스크립트를 실행할 각 데이터 소스에 대해 1-3단계를 반복합니다.
  5. 데이터베이스 스크립트를 실행한 후에 다음을 누릅니다.
