HTTP 프록시 응용 프로그램 생성

이 화면에서는 각 클러스터에 대해 HTTP 프록시 응용 프로그램을 생성하고 프록시 응용 프로그램을 배치해야 하는 관리되는 서버를 지정할 수 있습니다. HTTP 프록시 응용 프로그램은 HTTP 요청에 대한 매개 역할을 합니다.

이 화면은 다음 두 조건이 충족될 경우에만 표시됩니다.

  • 적어도 하나의 관리되는 서버가 클러스터에 할당되어 있습니다.
  • 적어도 하나의 관리되는 서버가 어느 클러스터에도 할당되어 있지 않습니다.
  1. HTTP 프록시 응용 프로그램을 생성할 클러스터에 대해 HTTP 프록시 생성 체크 박스를 선택합니다.

    클러스터에 할당되지 않은 관리되는 서버의 목록은 프록시 서버 목록에 표시됩니다.

  2. 프록시 서버 목록에서 프록시 응용 프로그램을 배치해야 하는 관리되는 서버를 선택합니다.

    OracleProxy4_<clustername>_<servername>(으)로 이름이 지정된 프록시 응용 프로그램은 선택한 관리되는 서버에서 생성되고 배치됩니다.

  3. HTTP 프록시 응용 프로그램을 생성할 각 클러스터에 대해 1단계와 2단계를 반복합니다.
