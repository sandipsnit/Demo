도메인 이름 및 위치 지정

다음 항목을 지정합니다.

  • 도메인 이름
    도메인의 이름입니다. 구성 마법사는 config.xml 파일 및 도메인에 대해 생성되는 모든 기타 구성 요소를 지정된 도메인 디렉토리 아래의 이 디렉토리에 저장합니다. 예를 들어, mydomain을 입력할 경우 도메인 파일은 (기본적으로) MW_HOME\user_projects\domains\mydomain에 저장됩니다.

    주: 도메인 이름은 숫자로 시작하면 안됩니다. 이 제한 사항은 내부적으로 생성된 JDBC 저장소 테이블 이름과의 잠재적 문제를 방지하기 위한 것이며, 따라서 도메인 이름은 반드시 영문자로 시작해야 합니다.

  • 도메인 위치
    도메인 디렉토리가 저장될 위치입니다. 기본값은 MW_HOME\user_projects\domains입니다. 도메인 위치는 로컬 드라이브나 네트워크의 어느 위치든 가능합니다. MW_HOME에 있을 필요는 없습니다.

    Windows의 경우 이 경로에 드라이브 문자가 포함되어야 합니다.

  • 응용 프로그램 위치
    이 필드는 응용 프로그램을 포함하는 도메인 템플리트나 제품을 선택한 경우에만 표시됩니다. 응용 프로그램이 상주할 위치입니다. 기본값은 MW_HOME\user_projects\applications입니다.

    Windows의 경우 이 경로에 드라이브 문자가 포함되어야 합니다.
