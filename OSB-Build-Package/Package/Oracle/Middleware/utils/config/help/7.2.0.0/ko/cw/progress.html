도메인 생성

이 화면에서는 도메인 생성 진행 과정을 보여줍니다.

이 프로세스가 완료되면 새 도메인 또는 확장 도메인을 사용할 수 있습니다.

관리 서버를 즉시 시작하려면 관리 서버 시작 체크 박스를 선택하고 완료를 누릅니다. 이 옵션은 새 도메인을 생성하는 경우 Windows 시스템에서만 사용할 수 있습니다.

구성 마법사를 종료하려면 완료를 누릅니다.

관리 서버를 시작하는 데 사용할 수 있는 다양한 방법에 대한 자세한 내용은 Managing Server Startup and Shutdown for Oracle WebLogic Server의 Starting and Stopping Servers를 참조하십시오.

자세한 내용은 Oracle Technology Network의 Oracle Fusion Middleware 설명서 라이브러리를 참조하십시오.
