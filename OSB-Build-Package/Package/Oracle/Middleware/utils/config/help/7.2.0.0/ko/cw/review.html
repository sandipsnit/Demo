구성 요약

계속 진행하기 전에 도메인의 구성 설정 세부 정보를 검토합니다.

  1. 도메인 요약 창에서 항목을 선택하여 세부 정보 창에서 연관된 세부 정보를 검토합니다. 이전 단추를 누르고 마법사의 이전 화면으로 돌아가 제한된 변경 사항을 수행할 수 있습니다.

    주: 요약 보기 목록에서 필터를 선택하여 도메인 요약 창에 표시되는 정보 유형을 제한할 수 있습니다.

  2. 도메인 설정을 검토한 후에 생성(또는 기존 도메인을 확장하는 경우 확장)을 누릅니다.
