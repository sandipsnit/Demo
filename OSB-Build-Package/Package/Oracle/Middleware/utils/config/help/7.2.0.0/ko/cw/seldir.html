디렉토리 선택

구성 마법사의 이 화면에서는 추가 응용 프로그램이나 서비스를 사용하여 갱신할 기존 도메인을 선택할 수 있습니다.

탐색 트리를 사용하여 적합한 도메인 디렉토리를 선택합니다. 이 디렉토리는 도메인의 config 디렉토리에 있는 config.xml 파일을 포함합니다. 적합한 도메인 디렉토리는 탐색 트리에 파란색 폴더 아이콘으로 나타납니다.
