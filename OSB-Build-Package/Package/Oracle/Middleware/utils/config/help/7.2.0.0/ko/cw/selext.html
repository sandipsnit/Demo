확장 소스 선택

나열된 제품 중 하나를 선택하거나 확장 템플리트로 이동하여 도메인을 확장할지 여부를 선택합니다.

  1. 다음 옵션 중 하나를 선택합니다.
    • 다음과 같이 추가된 제품을 지원하도록 내 도메인 자동 확장
      도메인에 포함시킬 다른 제품에 해당하는 체크 박스를 선택합니다. 이미 도메인에 포함된 제품은 회색 체크 박스로 나타납니다. 목록의 다른 제품에서 제공하는 리소스에 종속된 제품을 선택할 경우 해당 제품도 자동으로 선택됩니다.
    • 기존 확장 템플리트를 사용하여 내 도메인 확장
      템플리트 위치 필드에 필수 템플리트의 전체 경로를 입력하거나 찾아보기를 눌러 템플리트가 있는 디렉토리로 이동합니다.
  2. 필수 항목을 선택한 후 다음을 누릅니다.

주: 다음을 누를 때 선택한 제품 중 하나라도 선택되지 않은 다른 제품에서 제공하는 리소스가 필요한 경우 경고 대화상자가 표시됩니다. 계속 진행하기 전에 [확장 소스 선택] 화면으로 돌아가 종속 제품을 선택하십시오.

이 화면에 표시되는 제품 템플리트에 대한 자세한 내용은 Oracle Fusion Middleware Domain Template Reference를 참조하십시오.
