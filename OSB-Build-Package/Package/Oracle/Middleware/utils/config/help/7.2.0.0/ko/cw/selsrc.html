도메인 소스 선택

선택한 제품을 지원하거나 기존 도메인 템플리트를 기준으로 도메인을 생성하도록 자동으로 구성된 도메인을 생성할지 여부를 선택합니다. 표시된 목록의 각 제품은 해당 제품에 대해 필요한 도메인 리소스를 구성하는 단일 도메인 또는 확장 템플리트(JAR 파일)와 연관됩니다.

  1. 다음 옵션 중 하나를 선택합니다.
    • 다음 제품을 지원하도록 자동으로 구성된 도메인 생성
      기본적으로 기본 WebLogic Server 도메인 옵션은 선택되어 있습니다. 도메인에 포함시킬 다른 제품에 해당하는 체크 박스를 선택합니다. 목록의 다른 제품에서 제공하는 리소스에 종속된 제품을 선택할 경우 해당 제품도 자동으로 선택됩니다.
    • 기존 템플리트를 기준으로 이 도메인 생성
      기존 도메인 템플리트를 사용하여 도메인을 생성하려면 이 옵션을 선택합니다. 템플리트 위치 필드에 템플리트 JAR 파일의 전체 경로를 입력하거나 찾아보기를 눌러 필요한 템플리트가 있는 디렉토리로 이동합니다.
  2. 필수 항목을 선택한 후 마법사의 다음 단추를 누릅니다.

주: 다음을 누를 때 선택한 제품 중 하나라도 선택되지 않은 다른 제품에서 제공하는 리소스가 필요한 경우 경고 대화상자가 표시됩니다. 계속 진행하기 전에 [도메인 소스 선택] 화면으로 돌아가 종속 제품을 선택하십시오.

이 화면에 표시되는 제품 템플리트에 대한 자세한 내용은 Oracle Fusion Middleware Domain Template Reference를 참조하십시오.
