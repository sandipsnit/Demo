구성 요소 스키마 테스트

구성 마법사의 이 화면에서는 이전 화면에서 구성 요소 스키마에 지정한 구성을 테스트할 수 있습니다.

  1. 테스트할 스키마 이름 옆의 체크 박스를 선택합니다.
  2. 연결 테스트를 누릅니다.

    마법사는 스키마를 구성하는 동안 지정한 드라이버, 호스트, 포트 및 기타 정보를 사용하여 구축한 URL에 대해 연결을 시도하여 각 스키마에 대해 구성을 테스트합니다.

    테스트 결과는 상태 열에 표시됩니다. 세부 정보는 접속 결과 로그 창에 표시됩니다.

  3. 접속을 테스트한 후 다음을 누릅니다.
