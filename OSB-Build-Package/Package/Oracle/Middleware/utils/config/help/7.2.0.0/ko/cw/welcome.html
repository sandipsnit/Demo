시작

구성 마법사의 이 화면에서는 WebLogic 도메인을 생성할 지 또는 기존 도메인을 수정할 지 여부를 선택할 수 있습니다.

  1. 다음 옵션 중 하나를 선택합니다.

    • 새 WebLogic 도메인 생성

      새 도메인을 생성하려면 이 옵션을 선택합니다. 일반적으로 새 도메인은 관리 서버 및 기본 보안 영역 myrealm을 포함하는 WebLogic Server 기본 도메인을 기준으로 합니다. 하지만 자신의 도메인 템플리트를 사용하여 도메인을 생성할 수 있습니다. 도메인 템플리트는 적어도 하나의 관리 서버를 포함해야 합니다.

    • 기존 WebLogic 도메인 확장

      이미 WebLogic 도메인이 있고 해당 도메인에 리소스를 추가하려면 이 옵션을 선택합니다.
