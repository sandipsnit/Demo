Designar Servidores aos Clusters

Use esta tela para designar Servidores Gerenciados aos clusters. Somente os Servidores Gerenciados são exibidos na caixa de listagem Servidor. O Servidor de Administração não é listado porque não pode ser designado a um cluster.

  1. No painel do Cluster, selecione o cluster em que você deseja designar um Servidor Gerenciado.
  2. Designe Servidores Gerenciados para o cluster selecionado de uma das seguintes maneiras:
    • Clique duas vezes no nome do Servidor Gerenciado no painel do Servidor.
    • Selecione o Servidor Gerenciado no painel do Servidore clique no botão de seta para a direita localizado entre os painéis do Servidore do Cluster.
    • Shift+clique para selecionar vários Servidores Gerenciados no painel do Servidor; em seguida, clique no botão de seta para a direita localizado entre os painéis do Servidor e do Cluster.

    O nome do Servidor Gerenciado designado foi removido do painel do Servidor e adicionado abaixo do nome do cluster alvo, no painel do Cluster.

    Observação: Apenas Servidores Gerenciados estão listados no painel do Servidor. O servidor de administração não está listado porque ele não pode ser designado a um cluster.

  3. Repita as etapas 1 e 2 para cada Servidor Gerenciado que você deseja designar a um cluster.
  4. Revise as designações do cluster.

    Se necessário, você pode remover um Servidor Gerenciado de um cluster de umas das seguintes formas:

    • Clique duas vezes no nome do Servidor Gerenciado no painel do Cluster.
    • Selecione o Servidor Gerenciado no painel do Cluster e clique no botão da seta à esquerda localizado nos painéis do Servidor e do Cluster.

    O nome do Servidor Gerenciado é removido do painel do Cluster e restaurado para o painel do Servidor.

  5. Após a designação dos Servidores Gerenciados para os clusters, clique no botão Próximo no assistente.
