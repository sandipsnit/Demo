Designar Servidores para Máquinas

Use esta tela para designar instâncias do WebLogic Server a cada uma das máquinas definidas.

  1. No painel de Máquina, selecione a máquina do Windows ou do UNIX na qual você deseja designar uma instância do WebLogic Server.
  2. Designe instâncias do WebLogic Server para a máquina selecionada, de uma das seguintes formas:
    • Clique duas vezes na instância do WebLogic Server no painel do Servidor.
    • Selecione a instância do WebLogic Server adequada no painel do Servidor e clique no botão de seta à direita localizado entre os painéis do Servidor e da Máquina.
    • Shift+clique para selecionar vários servidores no painel do Servidor, em seguida, clique no botão de seta à direita localizado entre os painéis do Servidor e da Máquina.

    O nome da instância do WebLogic Server designada é removido do painel Servidor e adicionado, abaixo do nome da máquina de destino, no painel da Máquina.

  3. Repita as etapas 1 e 2 para cada instância do WebLogic Server que você deseja designar para uma máquina.
  4. Consulte as designações das máquinas.

    Se necessário, você pode remover uma instância do WebLogic Server de uma máquina de uma das seguintes formas:

    • Clique duas vezes no nome da instância do WebLogic Server adequada no painel da Máquina.
    • Selecione a instância do WebLogic Server adequada no painel da Máquina e clique no botão de seta à esquerda localizado entre os painéis do Servidor e da Máquina.

    O nome da a instância do WebLogic Server foi removida do painel da Máquina e restaurado no painel do Servidor.

  5. Após designar as instâncias do WebLogic Server às máquinas, clique em Próximo.
