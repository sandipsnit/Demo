Selecionar a Configuração Avançada

Seleciona todas as categorias (se houver) para as quais você deseja executar a configuração avançada. Para cada categoria selecionada, a tela de configuração apropriada é exibida para permitir que você execute a configuração avançada. Se você não selecionar nenhum item desta tela, a tela Resumo da Configuração será exibida em seguida.

Observação: As categorias que estão listadas nesta tela dependem dos recursos definidos nos modelos selecionados para o domínio.

Observação: As opções do Servidor de Administração e do Armazenamento de Segurança RDBMS não ficam disponíveis quando você está estendendo um domínio.

  • Servidor de Administração
    Selecione esta opção para modificar definições do Servidor de Administração, como nome do servidor, endereço de listening, porta de listening e definições de SSL.
  • Destino Distribuído JMS
    Por default, os destinos distribuídos JMS são convertidos automaticamente para Weighted Distributed Destinations (WDD). Selecione esta opção para alterar um destino distribuído JMS para um Unified Distributed Destination (UDD).
  • Servidores Gerenciados, Clusters e Máquinas
    Selecione esta opção para
    • Adicionar Servidores Gerenciados, clusters ou máquinas ao domínio
    • Deletar um Servidor Gerenciado, cluster ou máquina existente
    • Adicionar Servidores Gerenciados a um cluster existente
    • Modificar as definições de um Servidor Gerenciados, cluster ou máquina
  • Implantações e Serviços
    Selecione esta opção para personalizar como as implantações e serviços da aplicação são direcionados aos servidores e clusters.
  • Armazenamentos de Arquivo JMS
    Selecione esta opção para alterar as definições de armazenamento de arquivo JMS. Você pode alterar o nome, diretório e política de gravação síncrona de cada armazenamento de arquivo.
  • Armazenamento de Segurança RDBMS
    Selecione esta opção para configurar um sistema de gerenciamento do banco de dados relacional externo (RDBMS) como um armazenamento de dados para vários provedores de segurança.

Para cada opção de configuração avançada selecionada nesta tela, o assistente exibe telas adicionais depois, nas quais é possível definir as definições de configuração avançadas relevantes.

Selecione as caixas de seleção necessárias dependendo da configuração avançada que você deseja executar e, em seguida, clique em Próximo.
