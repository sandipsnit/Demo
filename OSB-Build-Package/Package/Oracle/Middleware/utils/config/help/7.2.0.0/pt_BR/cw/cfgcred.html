Configurar Credenciais da Área de Armazenamento de Chaves

Esta tela exibe um conjunto de campos de Informar Senha/Confirmar Senha da Área de Armazenamento de Chaves para cada produto que precisa que uma senha da área de armazenamento de chaves seja definida. Por exemplo, se você instalou o Oracle Information Rights Manager, há um conjunto de campos de senha para cada uma das duas áreas de armazenamento de chaves a seguir:

  • Oracle Information Rights Management
  • Plug-in do Oracle Enterprise Manager para IRM

  1. Informe e confirme a senha necessária para acesso à área de armazenamento de chaves da aplicação.
  2. Após informar todas as senhas da área de armazenamento de chaves, clique em Próximo.

Observação: A senha da área de armazenamento de chaves de cada produto listado nesta tela é obrigatória para utilização destes produtos. Anote as senha informadas e forneça-as à pessoa autorizada a usar estes produtos.

Para obter mais informações, consulte a biblioteca de documentação do Oracle Fusion Middleware no Oracle Technology Network.
