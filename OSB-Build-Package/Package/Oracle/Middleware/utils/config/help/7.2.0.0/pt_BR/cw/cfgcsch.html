Configurar Esquema do Componente JDBC

Para alguns componentes do Fusion Middleware (por exemplo, SOA e Web Center Spaces), as origens de dados JDBC podem ser definidas como parte do esquema do banco de dados do componente, que são carregadas (durante a instalação) usando o Repository Creation Utility (RCU).

Quando você cria um domínio do WebLogic para tais componentes usando o Assistente de Configuração, é possível configurar as definições do esquema do componente JDBC, como driver do banco de dados, proprietário do esquema, senha e assim por diante.

Os esquemas do componente JDBC associados com os produtos para os quais você está criando o domínio (por exemplo, SOA) estão listados na metade inferior da tela.

Observação: Se você desejar configurar um ou mais dos esquemas como origens de dados RAC, selecione a caixa de seleção Configure os esquemas de componentes selecionados como esquemas de origens de vários dados RAC no painel a seguir. Em seguida, clique no botão Próximo.

  1. Selecione os esquemas para os quais você deseja especificar as definições de origem de dados, selecionando a caixa de seleção adjacente aos nomes do esquema.

    Observação: Ao selecionar vários esquemas do componente , o texto “Varia entre os esquemas de componentes” pode ser exibido em alguns campos, indicando que os valores atuais destes campos não são os mesmos que estão selecionados nos esquemas do componente. Se você alterar os valores nestes campos, os novos valores serão aplicados uniformemente pelos esquemas do componente selecionados.

    Observação: Os valores default dos parâmetros do esquema do componente, como fornecedor, driver, nome de host e número da porta dependem dos valores que estão especificados nos modelos da aplicação.

  2. Revise as definições de configurações atuais e as modifique conforme o exigido.

    Observação: Os valores default dos parâmetros do esquema, como fornecedor, driver, nome de host e número da porta dependem dos valores que estão especificados nos modelos da aplicação.

    Campo Descrição
    Fornecedor Selecione o fornecedor do banco de dados.
    Driver Selecione o driver JDBC que você deseja usar para conectar-se ao banco de dados.

    Esta lista inclui os drivers JDBC para o fornecedor do banco de dados selecionado.
    Proprietário do Esquema Informe o nome de log-in para conectar-se ao banco de dados.
    Senha do Esquema Informe a senha de log-in para o proprietário do esquema especificado.
    DBMS/Serviço Informe um nome do DBMS do banco de dados ou um nome de serviço, caso seja selecionado um driver do tipo de serviço.
    Nome do Host Informe o nome do servidor que está hospedando o banco de dados.
    Porta Informe o número da porta a ser usado para conectar-se ao servidor que hospeda o banco de dados.

    Os valores que você especificar são exibidos nas colunas apropriadas na lista de esquema, para os esquemas selecionados.

  3. Após a marcação das seleções e entradas selecionadas, clique no botão Próximo no assistente para testar os esquemas do componente.
