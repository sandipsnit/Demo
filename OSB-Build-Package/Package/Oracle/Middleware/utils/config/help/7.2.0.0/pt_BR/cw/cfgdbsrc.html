Configurar as Origens de Dados JDBC

Use esta tela para configurar as origens de dados JDBC definidas em sua origem de domínio.

A origem de dados JDBC contém um pool de conexões do banco de dados que são criadas quando a instância da origem de dados é criada — quando é implantada ou direcionada, ou na inicialização do servidor. As aplicações pesquisam uma origem de dados na árvore JNDI e solicitam uma conexão. Quando as aplicações não precisam mais das conexões, elas retornam as conexões para o pool de conexão na origem de dados.

As origens de dados JDBC associadas com os produtos para os quais você está criando o domínio estão listadas na metade inferior da tela.

Observação: Se você deseja configurar uma ou mais origens de dados como origens de dados RAC, selecione as caixas de seleção adjacentes para o nome das origens de dados exigidos e selecione a caixa de seleção Configure as origens de dados selecionadas como origens de vários dados RAC no painel a seguir. Em seguida, clique no botão Próximo.

  1. Selecione a origem de dados para a qual você deseja especificar definições selecionando a caixa de seleção adjacente para o nome da origem de dados.

    Observação: Ao selecionar várias origens de dados, o texto “Varia entre as origens de dados” pode ser exibido em alguns campos, indicando que os valores atuais destes campos não são os mesmos nas origens de dados selecionados. Se você alterar os valores em tais campos, os novos valores serão aplicados uniformemente pelos campos de dados selecionados.

    Observação: Os valores default dos parâmetros da origem de dados, como fornecedor, driver, nome de host e número da porta dependem dos valores que estão especificados nos modelos da aplicação.

  2. Revise as definições de configurações atuais e as modifique conforme o exigido.

    A tabela a seguir descreve os campos nesta tela

    Observação: Os valores default dos parâmetros da origem de dados, como fornecedor, driver, nome de host e número da porta dependem dos valores que estão especificados nos modelos da aplicação.

    Campo Descrição
    Fornecedor Selecione o fornecedor do banco de dados.
    Driver Selecione o driver JDBC que você deseja usar para conectar-se ao banco de dados.

    Esta lista inclui os drivers JDBC comuns para o fornecedor do banco de dados selecionado.
    Nome do Usuário Informe o nome de log-in para conectar-se ao banco de dados.
    Senha Informe a senha de log-in para o nome de usuário especificado.
    DBMS/Serviço Informe um nome do serviço ou SID de DBMS. O valor informado depende do driver selecionado.
    • Se o nome do driver Oracle selecionado contiver as palavras "para conexões da Instância", você deve informar o SID.
    • Se o nome do driver Oracle contiver as palavras "para conexões de Serviço", você deve informar o nome do serviço.
    • Para obter informações sobre configuração de um driver DataDirect, consulte a documentação do DataDirect.
    Nome do Host Informe o nome do servidor que está hospedando o banco de dados.
    Porta Informe a porta a ser usada para conectar-se ao servidor que hospeda o banco de dados.

    Os valores que você especificar são exibidos nas colunas apropriadas na lista de origem de dados, para as origens de dados selecionados.

  3. Após realizar as seleções e entradas necessárias, clique no botão Próximo para testar as conexões com as origens do banco de dados JDBC.
