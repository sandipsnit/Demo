Configurar Armazenamentos de Arquivo JMS

Por default, para retrocompatibilidade, os destinos distribuídos JMS são convertidos automaticamente para Destino Distribuído Ponderado (WDD). Para alterar um destino distribuído JMS para usar o comportamento Destino Distribuído Uniforme (UDD), selecione UDD na lista drop-down. A Oracle recomenda usar a definição UDD.

Quando você altera o comportamento de recurso do sistema JMS, a definição é aplicada somente à sessão atual, a menos que os recursos sejam designados a um cluster. Depois que o recurso do sistema JMS é designado a um cluster, uma definição se torna fixa e não pode ser alterada.

Para obter mais informações sobre UDD e WDD, consulte Tipos de Destinos Distribuídos em Programando JMS para o Oracle WebLogic Server.
