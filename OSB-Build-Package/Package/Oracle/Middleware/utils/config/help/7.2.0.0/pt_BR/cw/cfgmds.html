Configurar o Esquema do Componente da Origem de Vários Dados RAC

Use esta tela para configurar os esquemas do componente incluídos no domínio do WebLogic como várias origens de dados do Oracle RAC.

Os esquemas do componente para os quais você optou configurar as origens de dados RAC na tela anterior do assistente (Configurar o Esquema do Componente JDBC) estão listados na metade inferior da tela.

  1. Na lista de esquema na metade inferior da tela, selecione o esquema que você deseja configurar como a origem de dados RAC, selecionando a caixa de seleção adjacente ao nome do esquema.

    Observação: Ao selecionar vários esquemas do componente , o texto “Varia entre os esquemas de componentes” pode ser exibido em alguns campos, indicando que os valores atuais destes campos não são os mesmos que estão selecionados nos esquemas do componente. Se você alterar os valores nestes campos, os novos valores serão aplicados uniformemente pelos esquemas do componente selecionados.

  2. Especifique as definições para as origens de dados RAC.

    A tabela a seguir descreve os campos nesta tela

    Campo Descrição
    Driver Selecione o driver que você deseja usar para conectar-se ao banco de dados.
    Nome do Serviço Informe um nome do serviço do banco de dados do Oracle RAC.
    Nome do Usuário Informe o nome de log-in para conectar-se ao banco de dados.
    Senha Informe a senha de log-in para o nome de usuário especificado.
    Nome do Host Informe o nome do servidor que hospeda as instâncias do banco de dados RAC.
    Nome da instância Informe o nome de cada instância do banco de dados Oracle.
    Porta Informe o número da porta a ser usada para conectar-se ao servidor que hospeda a instância do banco de dados.

    Observação: Você deve especificar o nome de host, nome da instância e número da porta de, no mínimo, uma instância do banco de dados.

    Para adicionar a instância do banco de dados, clique em Adicionar e, em seguida, especifique o nome do host, nome da instância do banco de dados e número da porta.

    Os valores que você especificar são exibidos nas colunas apropriadas na lista de esquema, para os esquemas selecionados.

  3. Após efetuar as seleções e entradas necessárias, clique em Próximo para testar os esquemas do componente.
