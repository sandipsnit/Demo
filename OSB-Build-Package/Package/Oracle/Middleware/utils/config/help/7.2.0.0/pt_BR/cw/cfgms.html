Configurar Servidores Gerenciados

Use esta tela para adicionar ou deletar Servidores Gerenciados. Você também pode alterar as definições de um Servidor Gerenciado existente. Os campos obrigatórios são indicados por um asterisco (*).

  1. Revise a configuração do servidor gerenciado atual. Os valores default podem variar baseados na origem do domínio que você selecionou.

    Observação: O assistente fornece duas views: uma view tabular concisa de todos os servidores gerenciados e uma view individual de cada servidor gerenciado com cada servidor representado por uma guia. Para alternar para um modo de view entre as views da tabela e da guia, clique em Trocar Exibição.

  2. Adicionar ou deletar servidores gerenciados ou alterar as definições para servidores gerenciados existentes, conforme o exigido para o seu domínio. Os campos marcados com um asterisco (*) são campos obrigatórios.

    Para mais informações sobre os campos nesta tela, consulte a página de ajuda para a tela Configurar o Servidor de Administração.

  3. Após efetuar as seleções e entradas necessárias, clique em Próximo.

Para obter mais informações, consulte a biblioteca de documentação do Oracle Fusion Middleware no Oracle Technology Network.
