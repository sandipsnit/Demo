Configurar o Banco de Dados do Armazenamento de Segurança RDBMS

Use esta tela para estabelecer as definições do armazenamento de segurança RDBMS.

A tabela a seguir descreve os campos desta tela.

Campo Descrição
Tipo de Banco de Dados Na lista suspensa, selecione o tipo do banco de dados que você deseja utilizar como o armazenamento de segurança RDBMS.
Driver Selecione o driver que você deseja utilizar para o banco de dados. A lista de drivers disponíveis varia, dependendo do tipo do banco de dados que você selecionar.
Nome da Classe Nenhum ação é exigida. O nome de classe é exibido automaticamente baseando-se no driver selecionado.
SID DBMS Informe o SID do banco de dados.
Host DBMS Informe o nome da máquina que hospeda o banco de dados.
Porta DBMS Informe a porta a ser usada para conectar-se ao servidor. O número da porta default associado ao tipo de banco de dados selecionado é exibido automaticamente.
URL Nenhuma ação exigida. O URL é exibido automaticamente baseando-se no driver que você selecionar.
Nome do Usuário Informe o nome de log-in para conectar-se ao banco de dados.
Senha do Usuário Informe a senha para o acesso ao banco de dados.

Os valores válidos consistem em uma string de caracteres alfanuméricos. Os caracteres de hífen (-) e sublinhado ( _ ) são suportados.
O valor é criptografado.
Confirmar Senha do Usuário Informe de novo a senha.
Propriedades Conhecidas Nenhuma ação exigida. As propriedades conhecidas do banco de dados são exibidas automaticamente baseando-se no driver que você selecionar.
Propriedades Adicionais Informe as propriedades adicionais, caso exista alguma, a serem passadas para o driver.

Observação: É possível testar a conexão com o banco de dados ao clicar em Testar Conexão. Antes de inicializar o servidor, você deve carregar os scripts SQL necessários para o armazenamento de segurança RDBMS. Se você utilizar o armazenamento de segurança do RDBMS em um domínio do cluster, é recomendável que você o utilize com a configuração JMS (nome JNDI e tópico JMS).

Após a marcação das seleções e entradas solicitadas, clique no botão Próximo no assistente.
