Configurar o Nome e a Senha do Usuário Administrador

Use esta tela para definir a conta do Administrador do WebLogic default do domínio. Esta conta é usada para inicializar e estabelecer conexão com o Servidor de Administração do domínio.

  • Nome
    O nome do usuário de log-in da conta do Administrador do WebLogic.

    O nome do usuário não deve conter vírgulas, guias e nenhum dos caracteres a seguir:
    <>#|&?(){}

  • Senha do Usuário
    A senha faz distinção entre maiúsculas e minúsculas. Ela deve conter no mínimo oito caracteres alfanuméricos, com pelo menos um numeral ou um dos caracteres a seguir:
    ! " # $ % & ' ( ) * + , - . / : ; < = > ? @ [ \ ] ^ _ ` { | } ~

Após efetuar as seleções e entradas necessárias, clique em Próximo.
