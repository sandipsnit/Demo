Executar Scripts do Banco de Dados

Um modelo de domínio pode conter um conjunto de arquivos SQL organizados por tipo de banco de dados. Se o modelo de domínio contiver arquivos SQL, você pode executá-los ao criar o domínio do WebLogic, na tela Executar Scripts do Banco de Dados. O conteúdo do banco de dados de cada uma das origens de dados definidas no domínio do WebLogic é configurado através da utilização de SQL pré-existente ou de arquivos de carga do banco de dados.

O conteúdo do banco de dados para cada uma das origens de dados definidas no seu domínio é configurada usando os arquivos de carregamento do SQL ou do banco de dados pré-existentes.

  1. No painel Origens de Dados JDBC Disponíveis, selecione a origem de dados para a qual você deseja executar os scripts.

    Os scripts que podem ser executados são exibidos no painel Opções de Carregamento de Arquivos e Banco de Dados SQL Disponíveis.

  2. Selecione a versão do banco de dados na lista Versão do BD.
  3. Clique em Executar Scripts.

    Todos os scripts exibidos no painel Opções de Carregamento de Arquivos e Banco de Dados SQL Disponíveis para a origem de dados selecionada são executados e o resultados são exibidos no painel Resultados.

    Se você desejar capturar a saída de teste em um arquivo de log, selecione a caixa de seleção Arquivo de Log e especifique a localização do arquivo de log.

  4. Repita as etapas 1 a 3 para cada origem de dados em que você desejar executar os scripts do banco de dados.
  5. Após a execução dos scripts do banco de dados, clique em Próximo.
