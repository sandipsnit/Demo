Criar Domínio

Esta tela mostra o andamento da criação do domínio.

Quando o processo for concluído, o novo (ou estendido) domínio está pronto para a utilização.

Se você desejar inicializar o servidor de administração imediatamente, selecione a caixa de seleção Inicializar o Servidor de Administração e clique em Concluído. esta opção só estará disponível para sistemas do Windows e caso você esteja criando um novo domínio.

Para sair do Assistente de Configuração, clique em Concluído.

Para obter informações sobre os vários métodos que você pode usar para iniciar o Servidor de Administração, consulte Iniciando e Interrompendo Servidores em Gerenciando a Inicialização e o Shut-down do Servidor do Oracle WebLogic Server.

Para obter mais informações, consulte a biblioteca de documentação do Oracle Fusion Middleware no Oracle Technology Network.
