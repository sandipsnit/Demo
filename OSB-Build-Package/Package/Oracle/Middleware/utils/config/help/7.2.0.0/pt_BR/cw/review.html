Resumo da Configuração

Revise as definições de configuração detalhadas de seu domínio antes de continuar.

  1. Selecione um item no painel Resumo do Domínio para revisar os detalhes associados no painel Detalhes. É possível fazer alterações limitadas ao clicar no botão Anterior e retornar às telas anteriores no assistente.

    Observação: É possível limitar o tipo de informações exibidas no painel Resumo do Domínio ao selecionar um filtro da lista View Resumida.

  2. Após a revisão das definições do domínio, clique e Criar (ou Estender caso você esteja estendendo um domínio existente).

Para obter mais informações, consulte a biblioteca de documentação do Oracle Fusion Middleware no Oracle Technology Network.
