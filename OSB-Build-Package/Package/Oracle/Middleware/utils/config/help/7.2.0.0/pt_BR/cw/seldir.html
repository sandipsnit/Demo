Selecionar Diretório

Nesta tela do Assistente de Configuração, você seleciona um domínio existente que você deseja atualizar com aplicações ou serviços adicionais.

Use a árvore de navegação para selecionar um diretório de domínio válido (um diretório que contém um arquivo config.xmlno diretório config do domínio). Os diretórios de domínio válidos são indicados por um ícone de pasta azul na árvore de navegação.

Após fazer a seleção necessária, clique em Próximo.
