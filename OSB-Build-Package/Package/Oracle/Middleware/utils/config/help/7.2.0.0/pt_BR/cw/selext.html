Selecionar Origem da Extensão

Escolha se a extensão do domínio deve ser feita através da seleção de um dos produtos listados ou através da busca de um modelo de extensão.

  1. Selecione uma das seguintes opções:
    • Estender meu domínio automaticamente para suportar os seguintes produtos adicionados
      Marque as caixas de seleção que correspondem aos outros produtos que você deseja incluir no domínio. Os produtos já incluídos no domínio são indicados pelas caixas de seleção cinzas. Se você selecionar um produto que depende dos recursos fornecidos por outros produtos da lista, esses produtos também serão selecionados automaticamente.
    • Estender meu domínio usando um modelo de extensão existente
      Informe o caminho completo do modelo solicitado no campo Localização do modelo ou clique em Procurar para navegar até o diretório que contém o modelo.
  2. Após fazer a seleção necessária, clique em Próximo.

Observação: Quando você clicar em Próximo, se qualquer um dos produtos selecionados precisar de recursos fornecidos por outro produto que não foi selecionado, será exibida uma caixa de diálogo de advertência. Retorne à tela Selecionar Origem de Extensão para selecionar o produto de dependência antes de continuar.

Para obter informações sobre os modelos do produto exibidos nesta tela, consulte Referência do Modelo de Domínio do Oracle Fusion Middleware.
