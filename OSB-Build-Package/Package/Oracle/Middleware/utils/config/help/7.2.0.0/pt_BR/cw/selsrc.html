Selecionar a Origem do Domínio

Selecione se deve ser gerado um domínio que é configurado automaticamente para suportar produtos selecionados ou para criar um domínio com base em um modelo de domínio existente. Cada produto da lista exibida é associado a um domínio único ou modelo de extensão (um arquivo JAR) que configura os recursos de domínio necessários para aquele produto.

  1. Selecione uma das seguintes opções:
    • Gerar um domínio configurado automaticamente para suportar os seguintes produtos
      Por default, será selecionada a opção Domínio Básico do WebLogic Server. Marque as caixas de seleção que correspondem a outros produtos que você deseja incluir no domínio. Se você selecionar um produto que depende dos recursos fornecidos por outro produto ou produtos da lista, eles também serão selecionados automaticamente.
    • Basear este domínio em um modelo existente
      Selecione esta opção se quiser criar um domínio usando um modelo de domínio existente. Informe o caminho completo para o arquivo JAR do modelo no campo Localização do modelo, ou clique em Procurar para navegar até o diretório que contém o modelo necessário.
  2. Após a marcação da seleção solicitada, clique no botão Próximo no assistente.

Observação: Quando você clicar em Próximo, se qualquer um dos produtos selecionados precisar de recursos fornecidos por outro produto que não foi selecionado, uma caixa de diálogo de advertência será exibida. Retorne à tela Selecionar Origem do Domínio para selecionar o produto de dependência antes de continuar.

Para obter informações sobre os modelos do produto exibidos nesta tela, consulte Referência do Modelo de Domínio do Oracle Fusion Middleware.
