Testar o Esquema do Componente

Nesta tela do Assistente de Configuração, é possível testar as configurações que você especificou para o esquema do componente na tela anterior.

  1. Selecione as caixas de seleção adjacentes para os nomes dos esquemas que você deseja testar.
  2. Clique em Testar Conexões.

    O assistente testa a configuração para cada esquema ao tentar conectar-se a um URL que seja construído com a utilização de um driver, host, porta e outras informações que você tenha especificado ao configurar o esquema.

    O resultado do teste é indicado na coluna Status. Os detalhes estão exibidos no painel de Log do Resultado da Conexão.

  3. Após testar as conexões, clique em Próximo.

Para obter mais informações, consulte a biblioteca de documentação do Oracle Fusion Middleware no Oracle Technology Network.
