Configurar as Origens de Dados JDBC

Use esta tela para testar as conexões da origem de dados configuradas nas telas Configurar Origem de Dados JDBC e Configurar Várias Origens de Dados do Oracle RAC.

Se não quiser testar as conexões no momento, não selecione nenhuma origem de dados. Clique em Próximo para continuar.

Observação: Para testar as conexões do banco de dados, o banco de dados ao qual você está estabelecendo conexão deve estar em execução.

  1. Selecione as caixas de seleção adjacentes para os nomes das origens de dados que você deseja testar.
  2. Clique em Testar Conexões.

    O assistente testa a configuração para cada origem de dados ao tentar conectar-se a um URL que seja construído com a utilização de um driver, host, porta e outras informações que você tenha especificado ao configurar a origem de dados.

    O resultado do teste é indicado na coluna Status. Os detalhes são exibidos no painel de Log do Resultado da Conexão.

  3. Após testar as conexões, clique em Próximo.

Para obter mais informações, consulte a biblioteca de documentação do Oracle Fusion Middleware no Oracle Technology Network.
