Bem-vindo

Nesta tela do Assistente de Configuração, você seleciona se deseja criar um domínio WebLogic ou modificar um domínio existente.

  1. Selecione uma das seguintes opções:

    • Criar um novo domínio do WebLogic

      Selecione esta opção se quiser criar um novo domínio. Geralmente, o novo domínio terá como base o Domínio Básico do WebLogic Server que inclui um Servidor de Administração e o realm de segurança default myrealm. Você pode, no entanto, optar por usar seu próprio modelo de domínio para criar o domínio; seu modelo de domínio deve conter pelo menos um Servidor de Administração.

    • Estender um domínio do WebLogic existente

      Selecione esta opção se já tiver um domínio do WebLogic e quiser adicionar recursos ao referido domínio.

  2. Após fazer a seleção necessária, clique em Próximo.
