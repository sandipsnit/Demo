向集群分配服务器

使用此屏幕可以将受管服务器分配给集群。“服务器”列表框中只显示受管服务器，而不列出管理服务器，因为无法将管理服务器分配到集群。

  1. 在集群窗格中，选择要为其分配受管服务器的集群。
  2. 按下列方法之一将受管服务器分配到所选集群中：
    • 在服务器窗格中，双击受管服务器的名称。
    • 在服务器窗格中选择受管服务器，然后单击服务器窗格和集群窗格之间的右箭头按钮。
    • 在服务器窗格中，按住 Shift 键并单击选择多个受管服务器，然后单击服务器窗格和集群窗格之间的右箭头按钮。

    分配的受管服务器的名称即从服务器窗格中删除，并添加到集群窗格中目标集群的名称之下。

    注：服务器窗格中只列出受管服务器，而不列出管理服务器，因为无法将其分配到集群。

  3. 针对要分配到集群的每个受管服务器重复执行步骤 1 和 2。
  4. 检查集群分配情况。

    如有必要，可按下列方法之一从集群中删除受管服务器：

    • 在集群窗格中，双击受管服务器的名称。
    • 在集群窗格中选择受管服务器，然后单击服务器窗格和集群窗格之间的左箭头按钮。

    该受管服务器的名称即从集群窗格中删除，并重新出现在服务器窗格中。

  5. 将受管服务器分配到集群后，单击该向导中的下一步按钮。

有关详细信息，请参阅 Oracle 技术网中的 Oracle Fusion Middleware 文档库。
