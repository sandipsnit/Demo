向计算机分配服务器

使用此屏幕可以将 WebLogic Server 实例分配给您定义的每个计算机。

  1. 在计算机窗格中，选择要为其分配 WebLogic Server 实例的 Windows 或 UNIX 计算机。
  2. 按下列方法之一将 WebLogic Server 实例分配到选定计算机：
    • 在服务器窗格中，双击 WebLogic Server 实例。
    • 在服务器窗格中选择相应的 WebLogic Server 实例，然后单击服务器窗格和计算机窗格之间的右箭头按钮。
    • 在服务器窗格中，按住 Shift 键并单击选择多个服务器，然后单击服务器窗格和计算机窗格之间的右箭头按钮。

    分配的 WebLogic Server 实例的名称即从服务器窗格中删除，并添加到计算机窗格中目标计算机的名称下。

  3. 针对要分配到计算机的每个 WebLogic Server 实例重复执行步骤 1 和 2。
  4. 检查计算机分配情况。

    如有必要，可按下列方法之一从计算机中删除 WebLogic Server 实例：

    • 在计算机窗格中，双击相应的 WebLogic Server 实例的名称。
    • 在计算机窗格中选择相应的 WebLogic Server 实例，然后单击服务器窗格和计算机窗格之间的左箭头按钮。

    该 WebLogic Server 实例的名称即从计算机窗格中删除，并重新出现在服务器窗格中。

  5. 将 WebLogic Server 实例分配给各计算机之后，单击下一步。
