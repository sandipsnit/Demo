配置密钥库身份证明

对于需要设置密钥库口令的每个产品，此屏幕会显示一组“输入密钥库口令/确认口令”字段。例如，如果安装了 Oracle Information Rights Manager，则以下两个应用程序密钥库中的每个都有一组对应的口令字段：

  • Oracle Information Rights Management
  • 用于 IRM 的 Oracle Enterprise Manager 插件

  1. 输入并确认访问应用程序密钥库所需的口令。
  2. 输入所有密钥库口令之后，单击下一步。

注：要使用这些产品，必须输入此屏幕中列出的每个产品的密钥库口令。请记下您输入的口令，并将其提供给授权使用这些产品的任何用户。

有关详细信息，请参阅 Oracle 技术网中的 Oracle Fusion Middleware 文档库。
