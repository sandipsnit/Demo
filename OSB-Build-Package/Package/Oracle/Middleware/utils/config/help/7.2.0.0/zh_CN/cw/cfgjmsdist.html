配置 JMS 文件存储

默认情况下，为实现向后兼容，JMS 分布式目标自动转换为加权分布式目标 (WDD)。要更改 JMS 分布式目标以使用统一分布式目标 (UDD) 行为，请从下拉列表中选择 UDD。Oracle 建议使用 UDD 设置。

更改 JMS 系统资源行为时，除非 JMS 系统资源已分配给某个集群，否则只会将设置应用于当前会话。一旦将 JMS 系统资源分配给某个集群，设置将成为固定设置且无法更改。

有关 UDD 和 WDD 的详细信息，请参阅 Programming JMS for Oracle WebLogic Server 中的 Types of Distributed Destinations。
