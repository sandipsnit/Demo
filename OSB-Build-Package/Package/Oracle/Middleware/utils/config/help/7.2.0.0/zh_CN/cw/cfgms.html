配置受管服务器

使用此屏幕可以添加或删除受管服务器。您也可以更改现有受管服务器的设置。星号 (*) 指示必需字段。

  1. 检查当前受管服务器配置。默认值因所选的域源而异。

    注：该向导提供了两种视图：一种是简洁明了的表格式视图，其中包含所有受管服务器；一种是每个受管服务器的单个视图，每个服务器由一个选项卡来表示。要在表视图和选项卡视图之间切换视图模式，请单击切换显示。

  2. 根据域的需要添加或删除受管服务器，或者更改现有受管服务器的设置。带星号 (*) 标记的字段是必填字段。

    有关此屏幕中的字段的详细信息，请参阅配置管理服务器屏幕的帮助页。

  3. 做出所需选择和输入所需内容之后，单击下一步。

有关详细信息，请参阅 Oracle 技术网中的 Oracle Fusion Middleware 文档库。
