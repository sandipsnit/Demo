选择扩展源

选择是否通过选择列出的产品之一，或通过浏览到一个扩展模板来对域进行扩展。

  1. 选择以下选项之一：
    • 自动扩展我的域以便支持下列添加的产品
      选择与要包含在域中的其他产品相对应的复选框。域中已包含的产品由灰显复选框指示。如果选择的产品与列表中其他产品提供的资源相关，则还会自动选中这些其他产品。
    • 使用现有的扩展模板扩展我的域
      在模板位置字段中输入所需模板的完整路径，或者单击浏览导航到包含相应模板的目录。
  2. 做出所需选择之后，单击下一步。

注：单击下一步时，如果选中的任何产品需要其他产品提供的资源，而该其他产品未被选中，则会显示一个警告对话框。请返回到“选择扩展源”屏幕选择相关产品，然后继续操作。

有关此屏幕中显示的产品模板的信息，请参阅 Oracle Fusion Middleware Domain Template Reference。
