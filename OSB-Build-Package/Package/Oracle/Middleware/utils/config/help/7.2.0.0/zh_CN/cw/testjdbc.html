测试 JDBC 数据源

使用此屏幕可以测试在配置 JDBC 数据源和配置 Oracle RAC 多数据源屏幕上配置的数据源连接。

如果此时不需要测试这些连接，请不要选择任何数据源。单击下一步以继续。

注：为测试数据库连接，连接到的数据库必须处于运行状态。

  1. 选中要测试的数据源名称旁的复选框。
  2. 单击测试连接。

    该向导将通过尝试连接到使用在配置数据源时指定的驱动程序、主机、端口和其他信息构建的 URL，测试每个数据源的配置。

    状态列中将指明测试结果。详细信息显示在连接结果日志窗格中。

  3. 测试连接之后，单击下一步。

有关详细信息，请参阅 Oracle 技术网中的 Oracle Fusion Middleware 文档库。
