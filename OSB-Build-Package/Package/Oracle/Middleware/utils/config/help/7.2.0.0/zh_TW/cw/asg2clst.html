將伺服器指派給叢集

使用此畫面將「受管理伺服器」指派至叢集。伺服器清單方塊中只會顯示「受管理伺服器」。因為「管理伺服器」無法指派至叢集，所以不會列出。

  1. 在「叢集」窗格中，選取想要為其指派「受管理伺服器」的叢集。
  2. 以下列其中一種方式，將「受管理伺服器」指派至選取的叢集：
    • 在「伺服器」窗格中，按兩下「受管理伺服器」的名稱。
    • 在「伺服器」窗格中選取「受管理伺服器」，然後按一下位在「伺服器」與「叢集」窗格之間的向右箭頭按鈕。
    • Shift + 按一下以在「伺服器」窗格中選取多個「受管理伺服器」；然後按一下位在「伺服器」與「叢集」窗格之間的向右箭頭按鈕。

    指派的「受管理伺服器」名稱會從「伺服器」窗格中移除，並新增至「叢集」窗格的目標叢集名稱下。

    注意：只有「受管理伺服器」才會列在「伺服器」窗格中。因為管理伺服器無法指派至叢集，所以不會列出。

  3. 針對每個要指派至叢集的「受管理伺服器」，重複步驟 1 和 2。
  4. 複查叢集指派。

    如有必要，您可以使用下列其中一種方式，移除叢集中的「受管理伺服器」：

    • 在「叢集」窗格中，按兩下「受管理伺服器」的名稱。
    • 在「叢集」窗格中選取「受管理伺服器」，然後按一下位在「伺服器」與「叢集」窗格之間的向左箭頭按鈕。

    「受管理伺服器」的名稱會從「叢集」窗格中移除，並回復至「伺服器」窗格。

  5. 將「受管理伺服器」指派至叢集之後，請按一下精靈中的「下一步」按鈕。
