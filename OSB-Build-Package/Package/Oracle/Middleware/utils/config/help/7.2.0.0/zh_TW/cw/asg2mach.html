將伺服器指派給機器

使用此畫面將 WebLogic Server 執行處理指派到您定義的每一部機器。

  1. 在「機器」窗格中，選取要指派 WebLogic Server 執行處理的 Windows 或 UNIX 機器。
  2. 以下列其中一種方式，將 WebLogic Server 執行處理指派至選定機器：
    • 在「伺服器」窗格中，按兩下 WebLogic Server 執行處理。
    • 在「伺服器」窗格中選取適當的 WebLogic Server 執行處理，然後按一下位在「伺服器」與「機器」窗格之間的右箭頭按鈕。
    • Shift+按一下以在「伺服器」窗格中選取多個伺服器；然後按一下位在「伺服器」與「機器」窗格之間的右箭頭按鈕。

    指派的 WebLogic Server 執行處理名稱會從「伺服器」窗格中移除，並新增至「機器」窗格的目標機器名稱下。

  3. 針對每個要指派至機器的 WebLogic Server 執行處理，重複步驟 1 和 2。
  4. 複查機器指派。

    如有必要，可以使用下列其中一種方式，將 WebLogic Server 執行處理從機器移除：

    • 在「機器」窗格中，按兩下適當的 WebLogic Server 執行處理名稱。
    • 在「機器」窗格中選取適當的 WebLogic Server 執行處理，然後按一下位在「伺服器」與「機器」窗格之間的左箭頭按鈕。

    WebLogic Server 執行處理的名稱會從「機器」窗格中移除，並回復至「伺服器」窗格。

  5. 將 WebLogic Server 執行處理指派到機器後，請按一下「下一歩」。
