選取進階組態

選取您要執行進階組態的所有類別 (如果有)。針對您選取的每個類別，會顯示適當的組態畫面，讓您執行進階組態。如果您未在此畫面上選取任何項目，下一個顯示的會是「組態摘要」畫面。

注意：列在此畫面上的類別，取決於您選取之網域樣板所定義的資源。

注意：當您擴充網域時，「管理伺服器」和「RDBMS 安全儲存」選項無法使用。

  • 管理伺服器
    選取此選項以修改「管理伺服器」設定，例如：伺服器名稱、監聽位址、監聽連接埠和 SSL 設定值。
  • JMS 分散式目標
    根據預設，JMS 分散式目標會自動轉換為「加權分散式目標」(WDD)。選取此選項將 JMS 分散式目標變更為「統一分散式目標」(UDD)。
  • 受管理伺服器、叢集和機器
    選取此選項以
    • 新增「受管理伺服器」、叢集或機器至網域
    • 刪除現有的「受管理伺服器」、叢集或機器
    • 新增「受管理伺服器」至現有的叢集
    • 修改現有的「受管理伺服器」、叢集或機器的設定。
  • 建置和服務
    選取此選項以自訂應用程式建置和服務定位到伺服器和叢集的方法。
  • JMS 檔案儲存
    選取此選項以變更您的 JMS 檔案儲存設定。您可以變更每個檔案儲存的名稱、目錄和同步寫入原則。
  • RDBMS 安全儲存
    選取此選項以設定外部關聯式資料庫管理系統 (RDBMS)，作為各種安全提供者的資料存放區。

針對您在此畫面中選取的每個進階組態選項，此精靈之後還會顯示其他畫面，而您可以在這些畫面中定義相關進階組態。

根據想要執行的進階組態來選取必要的核取方塊，然後按一下「下一步」。
