設定金鑰儲存庫證明資料

針對需要設定金鑰儲存庫密碼的每項產品，此畫面會顯示一組「輸入金鑰儲存庫密碼/確認密碼」欄位。例如，如果您安裝了 Oracle Information Rights Manager，畫面就會針對以下兩個應用程式金鑰儲存庫個別顯示一組密碼欄位：

  • Oracle Information Rights Management
  • Oracle Enterprise Manager Plugin for IRM

  1. 輸入並確認密碼，存取應用程式的金鑰儲存庫將需要該密碼。
  2. 輸入所有的金鑰儲存庫密碼後，請按一下「下一歩」。

注意：列在此畫面上的每項產品，都需要金鑰儲存庫密碼，才能使用這些產品。請記下您輸入的密碼，並將密碼提供給有權使用這些產品的所有人。

有關詳細資訊，請參閱 Oracle 技術網路上的 Oracle Fusion Middleware 文件庫。
