設定 JMS 檔案儲存

根據預設，考量回溯相容性問題，JMS 分散式目標會自動轉換為「加權分散式目標」(WDD)。若要將 JMS 分散式目標變更為使用「統一分散式目標」(UDD) 行為，請從下拉式清單中選取 UDD。Oracle 建議使用 UDD 設定。

當您變更 JMS 系統資源行為時，除非將 JMS 系統資源指派至叢集，否則設定僅會套用至目前的階段作業。指派 JMS 系統資源至叢集之後，設定就會變成固定，無法變更。

有關 UDD 和 WDD 的詳細資訊，請參閱「JMS for Oracle WebLogic Server 程式設計」中的分散式目標的類型。
