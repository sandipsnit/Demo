設定受管理伺服器

使用此畫面以新增或刪除「受管理伺服器」。您也可以變更現有的「受管理伺服器」設定。星號（*）表示必要欄位。

  1. 複查目前受管理伺服器組態。預設值可能會根據您選取的網域來源而不同。

    注意：此精靈提供兩種視圖：所有受管理伺服器的簡要表格式視圖，以及每個受管理伺服器的個別視圖，其中每個伺服器都是以頁籤代表。若要切換表格和頁籤視圖這兩種視圖模式，請按一下「切換顯示」。

  2. 新增或刪除受管理伺服器，或變更現有受管理伺服器的設定 (依網域需要)。標上星號 (*) 的欄位是必要欄位。

    有關此畫面中之欄位的詳細資訊，請參閱設定管理伺服器畫面的說明頁面。

  3. 進行必要的選取和輸入之後，請按一下「下一步」。

有關詳細資訊，請參閱 Oracle 技術網路上的 Oracle Fusion Middleware 文件庫。
