設定管理員使用者名稱和密碼

使用此畫面以定義網域的預設「WebLogic 管理員」帳號。此帳號用來啟動並連線至網域的「管理伺服器」。

  • 名稱
    WebLogic 管理員帳號的登入使用者名稱。

    使用者名稱不得包含逗號、Tab 鍵和下列任一字元：
    <>#|&?(){}

  • 使用者密碼
    密碼區分大小寫。密碼至少「必須」包含八個英數字元，其中至少要有一個數字或下列一個字元：
    ! " # $ % & ' ( ) * + , - . / : ; < = > ? @ [ \ ] ^ _ ` { | } ~

進行必要的輸入之後，請按一下「下一步」。

有關詳細資訊，請參閱 Oracle 技術網路上的 Oracle Fusion Middleware 文件庫。
