執行資料庫命令碼

網域樣板可能包含一組依資料庫類型組織的 SQL 檔案。如果網域樣板包含 SQL 檔案，建立 WebLogic 網域時，您可以在「執行資料庫命令碼」畫面中執行該檔案。使用預先存在的 SQL 或資料庫載入檔案，就可以設定 WebLogic 網域中所定義之每個資料來源的資料庫內容。

使用預先存在的 SQL 或資料庫載入檔案，就可以設定網域中所定義之每個資料來源的資料庫內容。

  1. 在「可用的 JDBC 資料來源」窗格中，選取想要為其執行命令碼的資料來源。

    可以執行的命令碼會顯示在「可用 SQL 檔案和資料庫載入選項」窗格中。

  2. 從「DB 版本」清單中選取資料庫版本。
  3. 按一下「執行命令碼」。

    會執行選定資料來源之「可用 SQL 檔案和資料庫載入選項」窗格中顯示的所有命令碼，並會將結果顯示在「結果」窗格中。

    如果您想要將測試輸出捕獲至日誌檔，請選取「日誌檔」核取方塊，並指定日誌檔的位置。

  4. 針對想要執行資料庫命令碼的每個資料來源重複步驟 1 到 3。
  5. 執行資料庫命令碼之後，請按「下一步」。
