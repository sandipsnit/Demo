指定網域名稱和位置

指定下列項目。

  • 網域名稱
    網域的名稱。「組態精靈」會將 config.xml 檔案和所有其他在此目錄中產生的網域元件，儲存在指定的「網域」目錄中。例如，如果您輸入 mydomain，您的網域檔案就會儲存 (根據預設) 在 MW_HOME\user_projects\domains\mydomain 中。

    注意：網域名稱的開頭不能是數字。此限制是要防止與內部產生之 JDBC 儲存表格名稱 (開頭必須為字母字元) 的潛在衝突。

  • 網域位置
    儲存網域目錄的位置。預設為 MW_HOME\user_projects\domains。網域位置可以是您本機磁碟機或網路上的任何位置。它不一定要位於 MW_HOME。

    在 Windows 上，您必須在此路徑上加入磁碟機代號。

  • 應用程式位置
    此欄位僅在您選取的網域樣板或產品包含應用程式時，才會顯示。此為應用程式將來所在的位置。預設為 MW_HOME\user_projects\applications。

    在 Windows 上，您必須在此路徑上加入磁碟機代號。

進行必要的選取和輸入之後，請按一下「下一步」。
