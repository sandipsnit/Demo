建立網域

此畫面顯示網域建立的進度。

流程完成後，新 (或擴充) 網域就備妥可供使用。

如果您想要立即啟動管理伺服器，請選取「啟動管理伺服器」核取方塊，然後按一下「完成」。只有在 Windows 系統中，以及建立新的網域時，才可以使用此選項。

若要結束「組態精靈」，請按一下「完成」。

有關啟動管理伺服器之各種方法的詳細資訊，請參閱「管理 Oracle WebLogic Server伺服器的啟動和關閉」中的啟動和停止伺服器。

有關詳細資訊，請參閱 Oracle 技術網路上的 Oracle Fusion Middleware 文件庫。
