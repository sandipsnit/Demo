選取擴充來源

選取列出的其中一項產品，或是瀏覽擴充樣板，來選擇是否擴充網域。

  1. 請選取下列其中一個選項：
    • 自動擴充我的網域，以支援下列新增的產品
      選取對應至其他想要併入網域之產品的核取方塊。已併入網域中的產品會以灰色核取方塊表示。若您選取的產品與其他產品或清單上的產品所提供的資源相依，也會自動選取該產品。
    • 使用現有擴充樣板擴充我的網域
      在「樣板位置」欄位中輸入必要樣板的完整路徑，或按一下「瀏覽」以瀏覽至含有該樣板的目錄。
  2. 進行必要的選取之後，請按一下「下一步」。

請注意：當您按一下「下一歩」時，如果任何選取的產品需要其他未選取之產品提供資源，警告對話方塊就會顯示。請先返回「選取擴充來源」畫面選取相依性產品，再繼續進行作業。

有關此畫面顯示之產品樣板的詳細資訊，請參閱 Oracle Fusion Middleware 網域樣板參照。
