選取網域來源

選擇要以自動設定來支援選定產品的方式產生網域，還是要根據現有的網域樣板來建立網域。所顯示清單中的每項產品都與單一網域或擴充樣板 (JAR 檔案)(設定該產品所需的網域資源) 相關聯。

  1. 請選取下列其中一個選項：
    • 產生自動設定的網域，以支援下列產品
      預設會選取「基本 WebLogic Server 網域」選項。選取對應至其他想要併入網域之產品的核取方塊。若您選取的產品與其他產品或清單上的產品所提供的資源相依，也會自動選取該產品。
    • 以現有樣板做為這個網域的基礎
      如果您想要使用現有的網域樣板來建立網域，請選取此選項。在「樣板位置」欄位中輸入樣板 JAR 檔案的完整路徑，或按一下「瀏覽」以瀏覽至含有必要樣板的目錄。
  2. 進行必要的選取之後，請按一下精靈中的「下一步」按鈕。

請注意：當您按一下「下一歩」時，如果任何選取的產品需要其他未選取之產品提供資源，警告對話方塊就會顯示。請先返回「選取網域來源」畫面選取相依性產品，再繼續進行作業。

有關此畫面顯示之產品樣板的詳細資訊，請參閱 Oracle Fusion Middleware 網域樣板參照。
