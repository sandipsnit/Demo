測試元件綱要

在「組態精靈」的這個畫面中，您可以測試在前一個畫面中為元件綱要指定的組態。

  1. 選取與想要測試之綱要名稱相鄰的核取方塊。
  2. 按一下「測試連線」。

    此精靈會嘗試連線至 URL，以測試每個綱要的組態，而 URL 是使用您在設定綱要時指定的驅動程式、主機、連接埠和其他資訊所建構。

    測試結果會顯示在「狀態」資料欄。詳細資訊則會顯示在「連線結果記錄」窗格中。

  3. 測試連線之後，請按一下「下一歩」。

有關詳細資訊，請參閱 Oracle 技術網路上的 Oracle Fusion Middleware 文件庫。
