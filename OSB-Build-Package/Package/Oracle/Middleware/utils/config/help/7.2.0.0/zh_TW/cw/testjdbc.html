測試 JDBC 資料來源

使用此畫面以測試您在「設定 JDBC 資料來源」和「設定 Oracle RAC 多重資料來源」畫面上設定的資料來源連線。

如果您目前不想要測試連線，請不要選取任何資料來源。請按一下「下一歩」繼續。

請注意：要測試資料庫連線，就必須執行您正在連線的資料庫。

  1. 選取與想要測試之資料來源名稱相鄰的核取方塊。
  2. 按一下「測試連線」。

    此精靈會嘗試連線至 URL，以測試每個資料來源的組態，而 URL 是使用您在設定資料來源時指定的驅動程式、主機、連接埠和其他資訊所建構。

    測試結果會顯示在「狀態」資料欄。詳細資訊則會顯示在「連線結果記錄」窗格中。

  3. 測試連線之後，請按一下「下一歩」。

有關詳細資訊，請參閱 Oracle 技術網路上的 Oracle Fusion Middleware 文件庫。
