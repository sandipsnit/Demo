歡迎

在「組態精靈」的這個畫面中，您可以選擇是要建立 WebLogic 網域，還是修改現有的網域。

  1. 請選取下列其中一個選項：

    • 建立新的 WebLogic 網域

      如果您要建立新的網域，請選取此選項。通常，新的網域會以「WebLogic Server 基本網域」(包含一個「管理伺服器」和預設安全範圍 myrealm) 為基礎。不過，您可以選擇使用自己的網域樣板來建立網域，您的網域樣板必須至少包含一個「管理伺服器」。

    • 擴充現有的 WebLogic 網域

      如果您已經有 WebLogic 網域，而想要新增資源到該網域，請選取此選項。

  2. 進行必要的選取之後，請按一下「下一步」。

有關詳細資訊，請參閱 Oracle 技術網路上的 Oracle Fusion Middleware 文件庫。
